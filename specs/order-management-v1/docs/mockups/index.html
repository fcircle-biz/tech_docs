注文管理システム モック画面

ダッシュボード

  • ダッシュボード（トップページ） - 売上サマリー、最近の注文

顧客管理

  • 顧客一覧 - 検索、一覧表示、削除
  • 顧客登録 - 新規登録フォーム
  • 顧客編集 - 編集フォーム

商品管理

  • 商品一覧 - カテゴリフィルタ、販売状態切替
  • 商品登録 - 新規登録フォーム
  • 商品編集 - 編集フォーム

注文管理

  • 注文一覧 - ステータスフィルタ
  • 注文作成 - 顧客選択、商品追加、数量変更
  • 注文詳細 - 明細表示、ステータス更新

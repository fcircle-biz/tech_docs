# ユーザー管理システム v2 仕様書

この資料は、認証機能付きユーザー管理システムのチュートリアル作成時の参考として使用します。

## 概要

ログイン・ログアウト機能を備えたシンプルなユーザー管理システムです。
基本的なCRUD操作と認証機能を学習できます。

## 主な機能

### 認証機能
- ユーザー登録（サインアップ）
- ログイン
- ログアウト

### ユーザー管理機能
- ユーザー一覧表示
- ユーザー情報の編集
- ユーザーの削除
- ユーザー検索

## 画面一覧

| 画面名 | 説明 | 認証 |
|--------|------|------|
| ログイン画面 | ユーザー名とパスワードでログイン | 不要 |
| ユーザー登録画面 | 新規アカウント作成 | 不要 |
| ユーザー一覧画面 | 登録ユーザーの一覧表示 | 必要 |
| ユーザー詳細画面 | ユーザー情報の表示 | 必要 |
| ユーザー編集画面 | ユーザー情報の編集 | 必要 |
| ユーザー削除確認画面 | 削除前の確認 | 必要 |

## 学習項目

### 認証
- ログイン処理の流れ
- セッション管理の基本
- パスワードの安全な保存

### データベース設計
- テーブル設計の基本
- データ型と制約の選択

### フロントエンド
- ログインフォームの作成
- 入力値の検証
- エラーメッセージの表示

### バックエンド
- 認証処理の実装
- セッション管理
- アクセス制御

## ファイル構成

- `database.md` - データベース設計
- `authentication.md` - 認証フローの説明

## チュートリアル構成（推奨）

| ステップ | 内容 | 学習ポイント |
|----------|------|--------------|
| 1 | 環境設定 | 開発環境の構築 |
| 2 | データベース設計 | テーブル作成、データ型 |
| 3 | ユーザー登録 | フォーム、バリデーション、パスワードハッシュ |
| 4 | ログイン機能 | 認証処理、セッション開始 |
| 5 | ユーザー一覧 | データ取得、一覧表示、アクセス制御 |
| 6 | ユーザー編集 | 更新処理 |
| 7 | ユーザー削除 | 削除処理、確認画面 |
| 8 | ログアウト・仕上げ | セッション終了、セキュリティ確認 |

## 対象レベル

プログラミング初心者向け。認証の基本概念とCRUD操作を実践的に学べます。

## 設計方針（初心者向けのため）

以下の機能は**含めない**：
- ロール・権限管理
- パスワードリセット
- メール認証
- Remember Me機能
- 2要素認証

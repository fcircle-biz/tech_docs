# Java研修 演習課題集

この練習問題集では、Java研修における理解度を確認し、段階的にプログラミングスキルを身につけるための演習課題を提供しています。

## 対象者
- Java初学者（プログラミング未経験者含む）
- Javaの基礎からオブジェクト指向までを体系的に学びたい方
- 実務でJava開発に携わる予定のエンジニア

## 前提知識
- 基本的なPC操作
- JDK環境のセットアップ完了
- テキストエディタまたはIDEの基本操作

## 学習の進め方

1. **関連ガイドで学習**: まず対応する学習ガイドで基礎を学びます
2. **課題に挑戦**: 各課題の要件を読み、自力でコードを書きます
3. **解答・解説を確認**: 完成後、解答例と照らし合わせて理解を深めます
4. **復習**: 理解が不十分な箇所は関連ガイドに戻って復習します

## 到達目標

この練習問題集を完了すると、以下のスキルを確認・強化できます：

### フェーズ1完了時
- Javaの基本構文（変数、型、演算子）を理解している
- 条件分岐（if文、switch文）を適切に使用できる
- 繰り返し処理（for文、while文）を実装できる
- 配列を宣言・初期化し、要素を操作できる
- メソッドの定義と呼び出しができる
- 処理の責務を適切に分離できる
- 可読性の高いコードを書くことができる

### フェーズ2完了時
- クラスとオブジェクトの概念を理解している
- カプセル化の原則に従ったクラス設計ができる
- 例外処理を適切に実装できる
- 継承とオーバーライドを活用できる
- コレクション（List、Map）を適切に選択・使用できる
- 拡張性を考慮した設計ができる
- Stream APIを使った効率的なデータ処理ができる
- 設計意図を文書化し、説明できる

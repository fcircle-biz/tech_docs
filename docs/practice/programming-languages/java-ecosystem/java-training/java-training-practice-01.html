1

Hello World
「Hello World」をコンソールに表示するプログラムを作成してください。

回答を表示

模範解答:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

ポイント: Javaプログラムはクラス内にmainメソッドを定義し、そこから実行が始まります。System.out.println()は標準出力に文字列を表示するメソッドです。

2

変数と出力
String型で名前、int型で年齢を宣言し、「〇〇は〇歳です」と表示してください。

回答を表示

模範解答:

public class VariableOutput {
    public static void main(String[] args) {
        String name = "田中太郎";
        int age = 25;

        System.out.println(name + "は" + age + "歳です");
    }
}

ポイント: 文字列の連結には+演算子を使用します。int型の変数は自動的に文字列に変換されます。

3

四則演算
2つのint変数を使い、四則演算（+ − × ÷）の結果を表示してください。

回答を表示

模範解答:

public class Arithmetic {
    public static void main(String[] args) {
        int a = 10;
        int b = 3;

        System.out.println("a + b = " + (a + b));
        System.out.println("a - b = " + (a - b));
        System.out.println("a * b = " + (a * b));
        System.out.println("a / b = " + (a / b));
        System.out.println("a % b = " + (a % b));
    }
}

ポイント: int同士の除算は小数点以下が切り捨てられます。余りを求めるには%（剰余演算子）を使用します。

4

条件分岐
int型の数値を用意し、正の数・負の数・0をif文で判定して表示してください。

回答を表示

模範解答:

public class Condition {
    public static void main(String[] args) {
        int number = 5;

        if (number > 0) {
            System.out.println(number + "は正の数です");
        } else if (number < 0) {
            System.out.println(number + "は負の数です");
        } else {
            System.out.println("0です");
        }
    }
}

ポイント: if-else if-else構文で複数の条件を順番にチェックします。条件は上から順に評価され、最初に真になった分岐が実行されます。

5

合否判定
点数（0〜100）を用意し、60点以上なら「合格」、それ以外は「不合格」と表示してください。

回答を表示

模範解答:

public class PassFail {
    public static void main(String[] args) {
        int score = 75;

        if (score >= 60) {
            System.out.println("合格");
        } else {
            System.out.println("不合格");
        }
    }
}

ポイント: >=は「以上」を表す比較演算子です。条件分岐の基本パターンとして覚えておきましょう。

6

for文
for文を使い、1〜100の数字を表示してください。

回答を表示

模範解答:

public class ForLoop {
    public static void main(String[] args) {
        for (int i = 1; i <= 100; i++) {
            System.out.println(i);
        }
    }
}

ポイント: for文は「初期化; 条件; 更新」の3つの部分で構成されます。i++はi = i + 1と同じ意味です。

7

倍数判定
for文を使い、1〜100のうち3の倍数のみを表示してください。

回答を表示

模範解答:

public class Multiples {
    public static void main(String[] args) {
        for (int i = 1; i <= 100; i++) {
            if (i % 3 == 0) {
                System.out.println(i);
            }
        }
    }
}

ポイント: %（剰余演算子）で3で割った余りが0かどうかを判定します。余りが0なら3の倍数です。

8

配列
int配列に5人分の点数を格納し、for文ですべて表示してください。

回答を表示

模範解答:

public class ArrayExample {
    public static void main(String[] args) {
        int[] scores = {85, 72, 90, 68, 95};

        for (int i = 0; i < scores.length; i++) {
            System.out.println((i + 1) + "人目: " + scores[i] + "点");
        }
    }
}

ポイント: 配列のインデックスは0から始まります。配列名.lengthで配列の要素数を取得できます。

9

合計点
配列を使い、合計点を計算して表示してください。

回答を表示

模範解答:

public class TotalScore {
    public static void main(String[] args) {
        int[] scores = {85, 72, 90, 68, 95};
        int total = 0;

        for (int i = 0; i < scores.length; i++) {
            total += scores[i];
        }

        System.out.println("合計点: " + total + "点");
    }
}

ポイント: total += scores[i]はtotal = total + scores[i]と同じ意味です。ループで累積加算するパターンは頻出です。

10

メソッド化
合計点を計算する処理をメソッド化し、mainメソッドから呼び出してください。

回答を表示

模範解答:

public class MethodExample {
    public static void main(String[] args) {
        int[] scores = {85, 72, 90, 68, 95};

        int total = calculateTotal(scores);

        System.out.println("合計点: " + total + "点");
    }

    public static int calculateTotal(int[] scores) {
        int total = 0;
        for (int i = 0; i < scores.length; i++) {
            total += scores[i];
        }
        return total;
    }
}

ポイント: メソッドは処理を再利用可能な形で分離します。staticキーワードを付けることで、mainメソッドから直接呼び出せます。引数で配列を受け取り、戻り値で結果を返します。

1

成績処理
点数配列から各点数・合計点・平均点を表示するプログラムを作成してください。処理は3つ以上のメソッドに分割してください。

回答を表示

模範解答:

public class ScoreProcessor {
    public static void main(String[] args) {
        int[] scores = {85, 72, 90, 68, 95};

        displayScores(scores);
        int total = calculateTotal(scores);
        double average = calculateAverage(scores);

        System.out.println("合計点: " + total + "点");
        System.out.println("平均点: " + average + "点");
    }

    public static void displayScores(int[] scores) {
        System.out.println("=== 各点数 ===");
        for (int i = 0; i < scores.length; i++) {
            System.out.println((i + 1) + "人目: " + scores[i] + "点");
        }
    }

    public static int calculateTotal(int[] scores) {
        int total = 0;
        for (int score : scores) {
            total += score;
        }
        return total;
    }

    public static double calculateAverage(int[] scores) {
        int total = calculateTotal(scores);
        return (double) total / scores.length;
    }
}

ポイント: 各処理を独立したメソッドに分けることで、コードの可読性と保守性が向上します。拡張for文（for (int score : scores)）を使うと配列の走査がシンプルになります。

2

評価判定
平均点に応じて評価を判定するメソッドを作成してください。
90点以上: 優、80点以上: 良、60点以上: 可、それ以外: 不可

回答を表示

模範解答:

public class ScoreEvaluation {
    public static void main(String[] args) {
        int[] scores = {85, 72, 90, 68, 95};

        double average = calculateAverage(scores);
        String evaluation = getEvaluation(average);

        System.out.println("平均点: " + average + "点");
        System.out.println("評価: " + evaluation);
    }

    public static double calculateAverage(int[] scores) {
        int total = 0;
        for (int score : scores) {
            total += score;
        }
        return (double) total / scores.length;
    }

    public static String getEvaluation(double average) {
        if (average >= 90) {
            return "優";
        } else if (average >= 80) {
            return "良";
        } else if (average >= 60) {
            return "可";
        } else {
            return "不可";
        }
    }
}

ポイント: 評価判定ロジックを独立したメソッドに切り出すことで、判定基準の変更が容易になります。条件判定は上から順に評価されるため、大きい値から書くのが一般的です。

3

最大値・最小値
配列から最大値を返すメソッドと最小値を返すメソッドを作成してください。

回答を表示

模範解答:

public class MinMax {
    public static void main(String[] args) {
        int[] scores = {85, 72, 90, 68, 95};

        int max = findMax(scores);
        int min = findMin(scores);

        System.out.println("最高点: " + max + "点");
        System.out.println("最低点: " + min + "点");
    }

    public static int findMax(int[] scores) {
        int max = scores[0];
        for (int score : scores) {
            if (score > max) {
                max = score;
            }
        }
        return max;
    }

    public static int findMin(int[] scores) {
        int min = scores[0];
        for (int score : scores) {
            if (score < min) {
                min = score;
            }
        }
        return min;
    }
}

ポイント: 最大値・最小値を求める際は、配列の最初の要素で初期化してから比較していきます。Integer.MIN_VALUEやInteger.MAX_VALUEで初期化する方法もあります。

4

入力値チェック
点数配列に0から100以外の値が含まれている場合、それらを無視して計算するメソッドを作成してください。

回答を表示

模範解答:

public class ScoreValidation {
    public static void main(String[] args) {
        int[] scores = {85, 72, -5, 90, 105, 68, 95};

        int validCount = countValidScores(scores);
        int total = calculateValidTotal(scores);
        double average = (double) total / validCount;

        System.out.println("有効な点数の数: " + validCount);
        System.out.println("合計点: " + total + "点");
        System.out.println("平均点: " + average + "点");
    }

    public static boolean isValidScore(int score) {
        return score >= 0 && score <= 100;
    }

    public static int countValidScores(int[] scores) {
        int count = 0;
        for (int score : scores) {
            if (isValidScore(score)) {
                count++;
            }
        }
        return count;
    }

    public static int calculateValidTotal(int[] scores) {
        int total = 0;
        for (int score : scores) {
            if (isValidScore(score)) {
                total += score;
            }
        }
        return total;
    }
}

ポイント: 入力値チェックのロジックをisValidScoreメソッドとして共通化することで、複数箇所で同じ判定条件を使い回せます。これはDRY原則（Don't Repeat Yourself）の実践例です。

5

処理の再利用
これまで作成したメソッドを組み合わせて、複数のクラスの成績を比較するプログラムを作成してください。

回答を表示

模範解答:

public class ClassComparison {
    public static void main(String[] args) {
        int[] classA = {85, 72, 90, 68, 95};
        int[] classB = {78, 88, 92, 65, 81};

        System.out.println("=== Aクラス ===");
        displayStatistics(classA);

        System.out.println("\n=== Bクラス ===");
        displayStatistics(classB);

        compareClasses(classA, classB);
    }

    public static void displayStatistics(int[] scores) {
        int total = calculateTotal(scores);
        double average = calculateAverage(scores);
        int max = findMax(scores);
        int min = findMin(scores);

        System.out.println("合計点: " + total + "点");
        System.out.println("平均点: " + average + "点");
        System.out.println("最高点: " + max + "点");
        System.out.println("最低点: " + min + "点");
    }

    public static void compareClasses(int[] classA, int[] classB) {
        double avgA = calculateAverage(classA);
        double avgB = calculateAverage(classB);

        System.out.println("\n=== クラス比較 ===");
        if (avgA > avgB) {
            System.out.println("Aクラスの方が平均点が高い");
        } else if (avgA < avgB) {
            System.out.println("Bクラスの方が平均点が高い");
        } else {
            System.out.println("両クラスの平均点は同じ");
        }
    }

    public static int calculateTotal(int[] scores) {
        int total = 0;
        for (int score : scores) {
            total += score;
        }
        return total;
    }

    public static double calculateAverage(int[] scores) {
        return (double) calculateTotal(scores) / scores.length;
    }

    public static int findMax(int[] scores) {
        int max = scores[0];
        for (int score : scores) {
            if (score > max) {
                max = score;
            }
        }
        return max;
    }

    public static int findMin(int[] scores) {
        int min = scores[0];
        for (int score : scores) {
            if (score < min) {
                min = score;
            }
        }
        return min;
    }
}

ポイント: 再利用可能なメソッドを作成しておくことで、新しい機能を追加する際に既存のメソッドを組み合わせるだけで実装できます。これがメソッド化の大きなメリットです。

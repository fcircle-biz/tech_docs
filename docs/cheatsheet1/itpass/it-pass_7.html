7-2 アローダイアグラム

📌 ポイント: アローダイアグラムはプロジェクトの各作業の順序と依存関係を図式化した工程表です。

アローダイアグラムの要素

⭐
ノード: 作業の開始点と終了点を表す丸や四角（イベント）
⭐
アクティビティ: 作業を表す矢印
⭐
ダミー: 依存関係を示す点線の矢印（実際の作業を伴わない）

クリティカルパス

プロジェクト内で最も長い所要時間を持つ作業経路

クリティカルパス上の作業が遅れるとプロジェクト全体の完了が遅れる

時間計算の基本概念

ES
最早開始時刻
EF
最早終了時刻
LS
最遅開始時刻
LF
最遅終了時刻

EF = ES + 作業時間

LS = LF - 作業時間

フロート = LS - ES（余裕時間）

7-3 順序と組合せ

📌 ポイント: プロジェクト管理では、作業の順序関係や選択肢の組合せを理解することが重要です。

順列と組合せ

🔢
順列: n個から r個を選び、順序を考慮して並べる
🔢
組合せ: n個から r個を選ぶ（順序は考慮しない）

nPr = n!/(n-r)! （順列）

nCr = n!/r!(n-r)! （組合せ）

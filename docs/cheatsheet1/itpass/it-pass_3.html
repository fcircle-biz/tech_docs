3-2 無線LAN

物理的なケーブルを使わずに電波や赤外線などを利用してデータ通信を行う無線LANの技術について学びます。

主な無線LAN規格には、IEEE 802.11シリーズがあります。

無線LANの周波数帯
2.4GHz帯 (IEEE 802.11b/g)
  • 障害物に強い（壁などを通過しやすい）
  • 到達距離が長い
  • 電子レンジやBluetooth機器と干渉しやすい
  • チャンネル数が少なく混雑しやすい
  • 最大速度: 11Mbps(b)、54Mbps(g)
5GHz帯 (IEEE 802.11a)
  • 障害物に弱い（壁などを通過しにくい）
  • 到達距離が短い
  • 干渉が少なく安定した通信が可能
  • チャンネル数が多く混雑しにくい
  • 最大速度: 54Mbps
※IEEE 802.11n以降は両方の周波数帯に対応したデュアルバンド方式が主流
SSID (Service Set Identifier)
無線LANアクセスポイントの識別名。Wi-Fiネットワークの名前として表示される。複数の無線LANが存在する環境でネットワークを区別するために使用される。
  • 最大32文字までの英数字で設定可能
  • ステルス機能：SSIDを非公開にして第三者からの検出を防ぐ
  • 初期設定のままだとセキュリティリスクとなる場合がある
セキュリティ
無線LANのセキュリティプロトコルにはWEP、WPA、WPA2、WPA3などがあり、暗号化強度が異なります。

3-4 IPアドレス

IPアドレスはインターネット上でコンピュータを識別するための住所のようなものです。

IPアドレスにはIPv4（32ビット）とIPv6（128ビット）の2種類があります。

IPv4
192.168.1.1
IPv6
2001:0db8:85a3:0000:0000:8a2e:0370:7334

IPアドレスはグローバルIPアドレスとプライベートIPアドレスに分けられます。

グローバルIPとプライベートIP
グローバルIPアドレス
  • インターネット上で一意に識別される公開アドレス
  • ISPから割り当てられる
  • 世界中どこからでもアクセス可能
  • 数に限りがあり、IPv4では枯渇問題がある
  • 例: 203.0.113.1
プライベートIPアドレス
  • 組織内など閉じたネットワークで使用される非公開アドレス
  • RFC1918で定義された特定の範囲のアドレス:
    • 10.0.0.0 - 10.255.255.255 (10.0.0.0/8)
    • 172.16.0.0 - 172.31.255.255 (172.16.0.0/12)
    • 192.168.0.0 - 192.168.255.255 (192.168.0.0/16)
  • 同じアドレスが異なるネットワークで重複使用可能
  • 例: 192.168.1.100
NAT (Network Address Translation)

プライベートIPアドレスとグローバルIPアドレスの変換を行う技術。家庭用ルーターなどで実装されている。

ポート番号

IPアドレスが「どのコンピュータか」を示すのに対し、ポート番号は「そのコンピュータ上のどのアプリケーションか」を識別します。0〜65535の番号が使われます。

3-7 IoT

IoT（Internet of Things、モノのインターネット）とは、様々な「モノ」がインターネットに接続され、データのやり取りを行う仕組みです。

スマート家電
センサー
ウェアラブル

IoTデバイスはセンサーなどを使用してデータを収集し、インターネットを通じて送信します。収集したデータはビッグデータとして分析され、様々なサービスに活用されます。

IoTのセキュリティリスク
IoTデバイスは多くの場合セキュリティが脆弱なため、DDoS攻撃の踏み台にされたり、情報漏洩の経路となるリスクがあります。

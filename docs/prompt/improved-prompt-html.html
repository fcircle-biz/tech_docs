プログラミング学習教材作成プロンプト

以下のプロンプトをコピーして使用することで、AIがカスタマイズされたプログラミング教材を作成します。プロンプト内のCSSは以下のように指定されています：

body { font-family: 'Helvetica Neue', Arial, sans-serif; line-height: 1.6; margin: 0; padding: 0; color: #333; background-color: #f5f5f5; }

カリキュラム部分を実際の教えたい内容に置き換えてください。

初心者向けのプログラミング学習教材を作成してください。 以下の要件に従って教材を作成してください： 1. 教材はHTMLフォーマットで出力してください 2. 学習しやすいようにナビゲーションメニューを含めてください 3. カリキュラムに基づいて「第1章〜第X章」という形で章立てしてください 4. プログラムコードは<code>タグで囲み、読みやすいように適切なインデントを付けてください 5. レスポンシブデザインでレイアウトが崩れないように設計してください 6. スタイリングにはBootstrap 5を使用してください 7. フォントは「Noto Sans JP」を使用してください 8. 実践的な例と演習問題を含めてください 9. 各章の終わりに理解度確認のための小テストを追加してください 10.図表を用いて分かりやすく説明してください。 11.Claudeの文字制約に収まるようにHTMLコンテンツを考えてください。 【フォーマット参考】 添付HTML 【カリキュラム内容】 添付カリキュラム
カスタマイズのポイント
  • 【カリキュラム内容】の部分は、実際に教えたい内容に置き換えてください
  • 特定のプログラミング言語を教えたい場合は、言語名を明記してください
  • 必要に応じて、対象者のレベル（完全初心者、プログラミング経験者など）を指定すると、より適切な教材ができます
  • 特別な要件（例：特定のプロジェクト作成、特定のライブラリの使用方法など）があれば追記してください

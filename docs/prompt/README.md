## 📖 概要

このディレクトリには、AIを効果的に活用するためのプロンプトテンプレートが含まれています。各プロンプトは以下の特徴を持っています：

- **即座に使用可能**: コピー&ペーストですぐに利用開始
- **高品質出力**: プロフェッショナルレベルの結果を生成
- **詳細な指示**: AIが最適な出力を生成できるよう設計
- **実用的**: ビジネスや学習の現場で実際に使える

## 🎯 使用方法

### 基本的な使い方

1. **プロンプトの選択**
   - 目的に合ったプロンプトのリンクをクリック

2. **コピー**
   - プロンプトページの「コピー」ボタンでテキスト全体をコピー

3. **AIに貼り付け**
   - ChatGPT、Claude、Gemini等のAIアシスタントに貼り付け

4. **情報入力**
   - プロンプトの指示に従って必要な情報を入力

5. **結果の活用**
   - AIが生成した高品質な出力を活用

### Tips

- 初めて使う場合は、プロンプト内の指示をよく読んでください
- 必要に応じてプロンプトをカスタマイズできます
- 複数回試して、最適な結果を得るための調整を行ってください

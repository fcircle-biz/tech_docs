AIプロンプト：企業技術戦略スライド生成

以下のプロンプトをコピーして、AIに高品質なプレゼンテーションスライドを生成させることができます。

以下の文章/記事を、McKinsey/BCG/Bainクラスの超一流コンサルティングファームが作成したような、完璧な16:9プレゼンテーションスライドに変換してください。 ## 目的 ユーザーから提供される情報をもとに、簡潔で情報価値の高い、経営幹部向けの企業技術戦略スライドを作成します。 ## 入力情報 以下の情報を提供してください： - 企業名 - 分析したい技術分野または戦略領域（任意） ## 出力内容 以下の要素を含む一枚のスライドをSVG形式で作成します： 「[企業名] の技術開発戦略」として、以下の項目を含めた資料を作成します： 1. 企業概要（設立年、事業内容、主な製品・サービス） 2. 現在の技術開発の重点分野（例：AI、IoT、再生可能エネルギー、半導体など） 3. 研究開発体制（研究所、R&D部門、提携機関、海外拠点など） 4. 過去の技術的成果や代表的な特許・製品 5. 今後の開発戦略・ロードマップ（公開されている中期計画や技術目標） 6. 競合との差別化要因 7. 技術開発における課題やリスク 8. 参考文献・出典 ## デザイン仕様 1. カラースキーム： - プライマリー： #2563EB（ロイヤルブルー）とそのグラデーション - セカンダリー： #475569（ダークグレー） - アクセント： #f8fafc（ライトグレー背景） - 重要度表示：高= #ef4444（赤）、中= #f59e0b（オレンジ）、低= #10b981（緑） 2. タイポグラフィ： - タイトル：32px、グラデーション効果、太字 - サブタイトル：16px、#475569 - セクション見出し：18px、#1e40af、アイコン付き - 本文：14px、#334155、行間1.4 3. レイアウト： - ヘッダー：左揃えタイトル＋右揃え日付/出典 - 2カラム構成：左側60%、右側40% - カード型コンポーネント：白背景、角丸12px、微細シャドウ - セクション間の適切な余白と階層構造 - 16:9プレゼンテーションスライド 4. ビジュアル要素： - データ表現：シンプルな棒グラフ/円グラフ（必要に応じて） - アイコン：線画スタイル、一貫性のあるデザイン - 企業ロゴ：右下フッターに配置 - 強調要素：キーポイントの視覚的階層化 ## 最終仕上げ - プロフェッショナルなフッター（ロゴ位置+出典） - 16:9比率に最適化された情報密度 - 視線誘導を考慮したレイアウト設計 - 説得力を高める微細なビジュアル要素 ## 注意点 - 一枚のスライドに収めるため、最重要情報のみ選別すること - 専門用語は最小限に抑え、明確に説明すること - 定量的データと定性的分析のバランスを取ること - 客観的な事実と主観的な評価を区別すること - 日本語で出力する場合はレイアウト崩れに注意すること - 情報間の因果関係や相関性を視覚的に表現すること - エグゼクティブの意思決定に役立つ戦略的視点を優先すること - &等の特殊文字は適切にエスケープすること このスライドは、C-levelエグゼクティブへのプレゼンテーションにも耐えうる品質を目指します。 すべての情報を1枚に凝縮しながら、視認性と情報の階層性を犠牲にしないデザインにしてください。 #変換する文章/記事（または分析対象企業に関する情報を以下に入力）：

使用方法

  1. 上記の「コピー」ボタンをクリックしてプロンプト全体をコピーします

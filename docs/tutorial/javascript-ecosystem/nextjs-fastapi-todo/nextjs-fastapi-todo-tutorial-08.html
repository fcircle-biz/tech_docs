8.1 エラー表示の実装

ユーザーにわかりやすくエラーを通知するために、エラーメッセージ表示機能を追加します。API呼び出しが失敗したときや、入力内容に問題があるときに表示されます。

実装 8-1: エラー通知コンポーネントの作成

エラーメッセージを表示するコンポーネントを作成します。

1

ErrorMessage.tsxを作成

frontend/components/ErrorMessage.tsx を作成します。

frontend/components/ErrorMessage.tsx 
// エラーメッセージ表示コンポーネント
interface ErrorMessageProps {
  message: string | null;
  onClose: () => void;
}

export default function ErrorMessage({ message, onClose }: ErrorMessageProps) {
  if (!message) return null;

  return (
    <div className="fixed top-20 right-4 max-w-md bg-red-50 border border-red-200 rounded-lg shadow-lg p-4 animate-slide-in-right">
      <div className="flex items-start gap-3">
        <i className="fas fa-exclamation-circle text-red-500 text-lg mt-0.5"></i>
        <div className="flex-1">
          <p className="text-red-800 font-medium">エラー</p>
          <p className="text-red-700 text-sm mt-1">{message}</p>
        </div>
        <button
          onClick={onClose}
          className="text-red-400 hover:text-red-600 transition-colors"
        >
          <i className="fas fa-times"></i>
        </button>
      </div>
    </div>
  );
}
2

page.tsxにエラー状態を追加

メインページにエラー状態管理を追加します。

frontend/app/page.tsx（一部抜粋） 
// エラーメッセージコンポーネントをインポート
import ErrorMessage from '@/components/ErrorMessage';

export default function Home() {
  const [error, setError] = useState<string | null>(null);

  // API呼び出し時にエラーハンドリング
  const fetchTodos = async () => {
    try {
      const response = await fetch('http://localhost:8000/api/todos');
      if (!response.ok) throw new Error('タスクの取得に失敗しました');
      const data = await response.json();
      setTodos(data);
      setError(null); // 成功時はエラーをクリア
    } catch (err) {
      setError(err instanceof Error ? err.message : '不明なエラーが発生しました');
    }
  };

  return (
    <>
      <ErrorMessage message={error} onClose={() => setError(null)} />
      {/* 既存のコンテンツ */}
    </>
  );
}

ローディング状態を管理

page.tsxでローディング状態を管理し、表示を切り替えます。

frontend/app/page.tsx（一部抜粋） 
import Loading from '@/components/Loading';

export default function Home() {
  const [isLoading, setIsLoading] = useState(true);

  const fetchTodos = async () => {
    setIsLoading(true);
    try {
      const response = await fetch('http://localhost:8000/api/todos');
      if (!response.ok) throw new Error('タスクの取得に失敗しました');
      const data = await response.json();
      setTodos(data);
      setError(null);
    } catch (err) {
      setError(err instanceof Error ? err.message : '不明なエラーが発生しました');
    } finally {
      setIsLoading(false);
    }
  };

  return (
    <div>
      {isLoading ? (
        <Loading />
      ) : (
        <TodoList todos={todos} />
      )}
    </div>
  );
}

実装 8-3: UI改善の適用

1

TodoItemにホバーエフェクトを追加

タスクアイテムにマウスを乗せたときの効果を追加します。

frontend/components/TodoItem.tsx（一部抜粋） 
export default function TodoItem({ todo, onUpdate, onDelete }: TodoItemProps) {
  return (
    <div className="flex items-center gap-3 p-4 bg-white rounded-lg border border-slate-200
                    hover:shadow-md hover:border-emerald-300 transition-all duration-200">
      {/* チェックボックス */}
      <input
        type="checkbox"
        checked={todo.completed}
        onChange={() => onUpdate(todo.id, { completed: !todo.completed })}
        className="w-5 h-5 text-emerald-500 rounded focus:ring-emerald-500 cursor-pointer"
      />

      {/* タスクタイトル */}
      <p className={`flex-1 ${todo.completed ? 'line-through text-slate-400' : 'text-slate-700'}`}>
        {todo.title}
      </p>

      {/* 削除ボタン */}
      <button
        onClick={() => onDelete(todo.id)}
        className="px-3 py-1 text-sm text-red-600 hover:bg-red-50 rounded-lg transition-colors"
      >
        <i className="fas fa-trash"></i>
      </button>
    </div>
  );
}

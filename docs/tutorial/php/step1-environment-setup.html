1.1 Eclipse IDE for PHP Developersの準備

Eclipse IDE for PHP Developersは、PHP開発に特化した統合開発環境です。Webサーバー機能、データベース連携、デバッグ機能が統合されており、初心者にも使いやすい環境を提供します。

実習 1-1: Eclipse IDE for PHP Developersのダウンロードとインストール

公式サイトからEclipse IDE for PHP Developersをダウンロードし、開発環境を構築します。

ダウンロード手順
  1. 公式サイトアクセス: Eclipse Packagesにアクセス
  2. パッケージ選択: 「Eclipse IDE for PHP Developers」を選択
  3. OS選択: 使用するOSに対応したバージョンをダウンロード
  4. 解凍とインストール: ダウンロードしたファイルを解凍し、適切なディレクトリに配置
初期設定
  1. Eclipse起動: eclipse.exe (Windows) または eclipse (Mac/Linux) を実行
  2. ワークスペース設定: プロジェクト用ディレクトリを指定
  3. 文字エンコーディング設定: UTF-8に設定（Window → Preferences → General → Workspace）
  4. PHP設定確認: PHPバージョンと実行環境の確認
推奨設定
  • Text file encoding: UTF-8
  • New text file line delimiter: Unix
  • PHP Version: 8.1 以上

1.2 PostgreSQLデータベースサーバーの準備

PostgreSQLは、高機能で標準規格に準拠したオープンソースのリレーショナルデータベース管理システムです。

実習 1-2: PostgreSQLのインストールと設定

PostgreSQLサーバーをインストールし、PHP開発で使用するための基本設定を行います。

PostgreSQLインストール手順
  1. 公式サイトアクセス: PostgreSQL Downloads
  2. OSに対応したインストーラーをダウンロード
  3. インストール実行: 管理者権限でインストーラーを実行
  4. データベースクラスター初期化とスーパーユーザーパスワード設定
  5. ポート設定: デフォルト5432を使用
接続確認

コマンドラインまたはpgAdminで接続を確認：

# コマンドライン接続例
psql -U postgres -h localhost -p 5432

# 接続成功時の表示例
postgres=#
トラブルシューティング
  • 接続エラー: PostgreSQLサービスが起動しているか確認
  • 認証エラー: pg_hba.confファイルの設定を確認
  • ポートエラー: 他のサービスがポート5432を使用していないか確認

1.3 PHPプロジェクトの作成

Eclipse IDEでPHPプロジェクトを作成し、Webアプリケーション開発の基盤を構築します。

実習 1-3: PHPプロジェクト作成とディレクトリ構成

ユーザー管理システムプロジェクトを作成し、標準的なディレクトリ構成を整えます。

プロジェクト作成手順
  1. 新規プロジェクト作成: File → New → PHP Project
  2. プロジェクト名: 「user-management」と入力
  3. PHPバージョン選択: PHP 8.1 以上を選択
  4. プロジェクト作成完了: Finishをクリック
ディレクトリ構成の作成

プロジェクトエクスプローラーで以下のディレクトリ構成を作成：

user-management/
├── index.php              # トップページ
├── config/
│   └── database.php       # データベース設定
├── includes/
│   ├── header.php         # 共通ヘッダー
│   └── footer.php         # 共通フッター
├── users/
│   ├── list.php           # ユーザー一覧
│   ├── detail.php         # ユーザー詳細
│   ├── create.php         # ユーザー登録
│   ├── edit.php           # ユーザー編集
│   └── delete.php         # ユーザー削除
├── assets/
│   ├── css/
│   │   └── style.css      # スタイルシート
│   └── js/
│       └── script.js      # JavaScript
└── sql/
    └── create_tables.sql  # テーブル作成SQL
ディレクトリ作成方法
  1. プロジェクトを右クリック → New → Folder
  2. フォルダ名を入力（例: config, includes, users等）
  3. 必要なディレクトリをすべて作成

1.5 HTMLフォームとPHPの連携基礎

HTMLフォームからのデータ送信とPHPでの受信処理の基本パターンを学習します。

実習 1-5: 簡単なフォーム処理の実装

名前入力フォームとPHPでの処理を実装し、GET/POSTメソッドの動作を確認します。

form_test.phpファイルの作成

プロジェクトルートに以下の内容で作成：

<?php
// フォームデータの処理
$name = '';
$message = '';

if ($_POST['name'] ?? false) {
    $name = htmlspecialchars($_POST['name'], ENT_QUOTES, 'UTF-8');
    $message = "こんにちは、{$name}さん！";
}
?>
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>フォーム処理テスト</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <div class="row justify-content-center">
            <div class="col-md-6">
                <div class="card">
                    <div class="card-header bg-warning">
                        <h3 class="card-title mb-0">フォーム処理テスト</h3>
                    </div>
                    <div class="card-body">
                        <?php if ($message): ?>
                            <div class="alert alert-success">
                                <?php echo $message; ?>
                            </div>
                        <?php endif; ?>
                        
                        <form method="POST" action="">
                            <div class="mb-3">
                                <label for="name" class="form-label">お名前</label>
                                <input type="text" class="form-control" id="name" name="name" 
                                       value="<?php echo htmlspecialchars($name, ENT_QUOTES, 'UTF-8'); ?>" 
                                       required>
                            </div>
                            <button type="submit" class="btn btn-warning">送信</button>
                        </form>
                        
                        <hr>
                        <h6>送信されたデータ</h6>
                        <pre><?php 
                        echo "POST: " . print_r($_POST, true); 
                        echo "GET: " . print_r($_GET, true); 
                        ?></pre>
                    </div>
                </div>
            </div>
        </div>
    </div>
</body>
</html>
動作確認手順
  1. ファイル保存: form_test.phpを保存
  2. ブラウザアクセス: http://localhost:8080/user-management/form_test.php
  3. フォーム送信テスト: 名前を入力して送信ボタンをクリック
  4. 結果確認: メッセージ表示とPOSTデータの内容を確認
重要なポイント
  • htmlspecialchars(): XSS対策のための出力エスケープ
  • $_POST/$_GET: スーパーグローバル変数でのデータ取得
  • Null合体演算子（??）: 未定義変数のチェック
  • ENT_QUOTES: シングル・ダブルクォートのエスケープ

Bootstrap チートシート

レスポンシブWebデザインの実践リファレンス

Bootstrapとは？

BootstrapはTwitter（現X）が開発した、世界で最も人気のあるCSSフレームワークです。レスポンシブデザイン対応のコンポーネント群とグリッドシステムにより、モバイル対応のWebサイトを素早く構築できます。

グリッドシステム コンポーネント レスポンシブ ユーティリティ

基本セットアップ

CDN経由での読み込みが最も簡単。<head>タグ内にCSSを、</body>の直前にJSを記述します。 

<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Bootstrap JavaScript Bundle (Popper含む) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

● Bootstrap 5.3以降を推奨（IE11サポート終了）

● bundle版にはPopperライブラリが含まれています

グリッドシステム

12カラムグリッド。.containerでコンテナを作成し、.rowで行、.colで列を定義 

<div class="container">
  <div class="row">
    <div class="col-md-6">左カラム</div>
    <div class="col-md-6">右カラム</div>
  </div>
</div>

● .col-{breakpoint}-{size} でレスポンシブ指定

● breakpoint: sm, md, lg, xl, xxl

レスポンシブブレークポイント

Extra Small (xs)

<576px - スマートフォン

Small (sm)

≥576px - 横向きスマホ

Medium (md)

≥768px - タブレット

Large (lg)

≥992px - デスクトップ

Extra Large (xl) / XXL (xxl)

≥1200px / ≥1400px - 大画面

よく使うユーティリティクラス

マージン・パディング 

<!-- マージン -->
.m-3   (全方向)
.mt-3  (top)
.mb-3  (bottom)
.ms-3  (start/left)
.me-3  (end/right)
.mx-3  (左右)
.my-3  (上下)

<!-- パディング -->
.p-3, .pt-3, .pb-3
.ps-3, .pe-3
.px-3, .py-3
サイズ: 0〜5 (0, 0.25rem, 0.5rem, 1rem, 1.5rem, 3rem)

テキスト 

<!-- 配置 -->
.text-start  (左寄せ)
.text-center (中央)
.text-end    (右寄せ)

<!-- 色 -->
.text-primary
.text-success
.text-danger

<!-- 太さ -->
.fw-bold   (太字)
.fw-normal (標準)

表示・非表示 

<!-- レスポンシブ表示 -->
.d-none           (非表示)
.d-block          (ブロック表示)
.d-flex           (フレックス)

<!-- ブレークポイント -->
.d-md-none        (md以上で非表示)
.d-lg-block       (lg以上でブロック)

<!-- Flexbox -->
.justify-content-center
.align-items-center

実務で役立つTips

モバイルファースト

Bootstrapは「モバイルファースト」設計。まず小画面向けに記述し、ブレークポイントで大画面対応を追加します。

カスタマイズ

Sass変数を上書きしてカスタマイズ可能。CDN版では上書きCSSで対応。独自スタイルは最後に読み込みます。

Bootstrap Icons

公式アイコンライブラリ「Bootstrap Icons」が利用可能。Font Awesomeと組み合わせも可能です。

バージョン注意

Bootstrap 4→5で一部クラス名変更（例: .ml-3 → .ms-3）。移行時はドキュメント確認必須です。

3 コンポーネント配置
4 レスポンシブ調整
5 カスタマイズ

公式ドキュメント: https://getbootstrap.com/

9.1 認証の基礎

認証（Authentication）は、ユーザーが本人であることを確認するプロセスです。ASP.NETでは、フォーム認証、Windows認証、パスポート認証などの方法を提供しています。

9.1.1 フォーム認証の設定

web.configでの認証設定：

<configuration>
  <system.web>
    <authentication mode="Forms">
      <forms loginUrl="~/Login.aspx" 
             defaultUrl="~/Default.aspx" 
             timeout="30" 
             name=".ASPXAUTH" 
             cookieless="false" 
             requireSSL="false" />
    </authentication>
    
    <authorization>
      <deny users="?" />  <!-- 匿名ユーザーを拒否 -->
    </authorization>
  </system.web>
</configuration>

9.1.2 簡単なログインページの実装

Login.aspx：

Login.aspx.vb：

Imports System.Web.Security

Partial Public Class Login
    Inherits System.Web.UI.Page

    Protected Sub btnLogin_Click(sender As Object, e As EventArgs) Handles btnLogin.Click
        ' 簡単な認証チェック（実際の実装ではデータベースなどを使用）
        If ValidateUser(txtUsername.Text, txtPassword.Text) Then
            ' 認証チケットを作成
            FormsAuthentication.RedirectFromLoginPage(txtUsername.Text, False)
        Else
            lblMessage.Text = "ユーザー名またはパスワードが正しくありません。"
        End If
    End Sub

    Private Function ValidateUser(username As String, password As String) As Boolean
        ' デモ用の簡単な認証（実際の実装では暗号化したパスワードをチェック）
        Return (username = "admin" AndAlso password = "password123") OrElse
               (username = "user" AndAlso password = "user123")
    End Function
End Class

9.2 メンバーシップ システム

ASP.NET メンバーシップ システムは、ユーザー管理機能を提供するフレームワークです。データベースでのユーザー情報管理を簡単に実装できます。

9.2.1 メンバーシッププロバイダーの設定

web.configでのメンバーシップ設定：

<configuration>
  <connectionStrings>
    <add name="DefaultConnection" 
         connectionString="Data Source=.\SQLEXPRESS;AttachDbFilename=|DataDirectory|\Database.mdf;Integrated Security=True;User Instance=True" 
         providerName="System.Data.SqlClient" />
  </connectionStrings>
  
  <system.web>
    <membership defaultProvider="SqlProvider">
      <providers>
        <clear />
        <add name="SqlProvider" 
             type="System.Web.Security.SqlMembershipProvider" 
             connectionStringName="DefaultConnection"
             applicationName="WebApp"
             enablePasswordRetrieval="false"
             enablePasswordReset="true"
             requiresQuestionAndAnswer="false"
             requiresUniqueEmail="true"
             minRequiredPasswordLength="6"
             minRequiredNonalphanumericCharacters="0" />
      </providers>
    </membership>
    
    <roleManager enabled="true" defaultProvider="SqlRoleProvider">
      <providers>
        <clear />
        <add name="SqlRoleProvider" 
             type="System.Web.Security.SqlRoleProvider" 
             connectionStringName="DefaultConnection"
             applicationName="WebApp" />
      </providers>
    </roleManager>
  </system.web>
</configuration>

9.2.2 ユーザー登録機能の実装

Register.aspx.vb：

Imports System.Web.Security

Partial Public Class Register
    Inherits System.Web.UI.Page

    Protected Sub btnRegister_Click(sender As Object, e As EventArgs) Handles btnRegister.Click
        Try
            ' 新しいユーザーを作成
            Dim status As MembershipCreateStatus
            Dim newUser As MembershipUser = Membership.CreateUser(
                txtUsername.Text,
                txtPassword.Text,
                txtEmail.Text,
                Nothing,
                Nothing,
                True,
                status)

            If status = MembershipCreateStatus.Success Then
                ' デフォルトロールを割り当て
                Roles.AddUserToRole(txtUsername.Text, "User")
                
                lblMessage.Text = "ユーザー登録が完了しました。"
                lblMessage.ForeColor = Drawing.Color.Green
                
                ' ログインページにリダイレクト
                Response.Redirect("Login.aspx")
            Else
                lblMessage.Text = GetErrorMessage(status)
                lblMessage.ForeColor = Drawing.Color.Red
            End If

        Catch ex As Exception
            lblMessage.Text = "エラーが発生しました: " & ex.Message
            lblMessage.ForeColor = Drawing.Color.Red
        End Try
    End Sub

    Private Function GetErrorMessage(status As MembershipCreateStatus) As String
        Select Case status
            Case MembershipCreateStatus.DuplicateEmail
                Return "このメールアドレスは既に使用されています。"
            Case MembershipCreateStatus.DuplicateUserName
                Return "このユーザー名は既に使用されています。"
            Case MembershipCreateStatus.InvalidPassword
                Return "パスワードが要件を満たしていません。"
            Case MembershipCreateStatus.InvalidEmail
                Return "メールアドレスの形式が正しくありません。"
            Case Else
                Return "ユーザー登録に失敗しました。"
        End Select
    End Function
End Class
実習 9-1: メンバーシップ システムの実装

メンバーシップ システムを使用したユーザー管理機能を実装してみましょう。

手順
  1. aspnet_regsql.exe を使用してメンバーシップデータベースを作成
  2. web.config でメンバーシップとロール管理を設定
  3. ユーザー登録ページを作成
  4. ログインページを作成
  5. 管理者ページでユーザー一覧を表示
管理者ページでのユーザー一覧表示
Protected Sub Page_Load(sender As Object, e As EventArgs) Handles Me.Load
    If Not IsPostBack Then
        LoadUsers()
    End If
End Sub

Private Sub LoadUsers()
    Dim users As MembershipUserCollection = Membership.GetAllUsers()
    Dim userList As New List(Of Object)()
    
    For Each user As MembershipUser In users
        userList.Add(New With {
            .UserName = user.UserName,
            .Email = user.Email,
            .IsOnline = user.IsOnline,
            .LastLoginDate = user.LastLoginDate,
            .IsApproved = user.IsApproved
        })
    Next
    
    GridViewUsers.DataSource = userList
    GridViewUsers.DataBind()
End Sub

9.4 認可とロールベース セキュリティ

認可（Authorization）は、認証されたユーザーがリソースにアクセスする権限があるかを確認するプロセスです。

9.4.1 フォルダレベルでのアクセス制御

管理者フォルダのweb.config：

<?xml version="1.0"?>
<configuration>
  <system.web>
    <authorization>
      <allow roles="Admin" />
      <deny users="*" />
    </authorization>
  </system.web>
</configuration>

9.4.2 コードレベルでのロールチェック

Protected Sub Page_Load(sender As Object, e As EventArgs) Handles Me.Load
    ' ロールチェック
    If User.IsInRole("Admin") Then
        pnlAdminPanel.Visible = True
        LoadAdminData()
    ElseIf User.IsInRole("Manager") Then
        pnlManagerPanel.Visible = True
        LoadManagerData()
    Else
        pnlUserPanel.Visible = True
        LoadUserData()
    End If
End Sub

Protected Sub btnDeleteUser_Click(sender As Object, e As EventArgs)
    ' 管理者権限をチェック
    If Not User.IsInRole("Admin") Then
        lblMessage.Text = "この操作を実行する権限がありません。"
        Return
    End If
    
    ' ユーザー削除処理
    Dim username As String = GridViewUsers.SelectedValue.ToString()
    Membership.DeleteUser(username)
    LoadUsers()
End Sub
実習 9-2: ロールベース アクセス制御の実装

異なる権限レベルを持つユーザー向けのアクセス制御システムを構築しましょう。

ロール管理クラスの作成
Public Class RoleManager
    Public Shared Sub CreateRoles()
        If Not Roles.RoleExists("Admin") Then
            Roles.CreateRole("Admin")
        End If
        
        If Not Roles.RoleExists("Manager") Then
            Roles.CreateRole("Manager")
        End If
        
        If Not Roles.RoleExists("User") Then
            Roles.CreateRole("User")
        End If
    End Sub
    
    Public Shared Sub AssignUserToRole(username As String, roleName As String)
        If Not Roles.IsUserInRole(username, roleName) Then
            Roles.AddUserToRole(username, roleName)
        End If
    End Sub
    
    Public Shared Function GetUserRoles(username As String) As String()
        Return Roles.GetRolesForUser(username)
    End Function
End Class

9.5.1 パスワードハッシュ化の実装

Imports System.Security.Cryptography
Imports System.Text

Public Class PasswordHelper
    Public Shared Function HashPassword(password As String, salt As String) As String
        Using sha256 As SHA256 = SHA256.Create()
            Dim saltedPassword As String = password + salt
            Dim hashBytes As Byte() = sha256.ComputeHash(Encoding.UTF8.GetBytes(saltedPassword))
            Return Convert.ToBase64String(hashBytes)
        End Using
    End Function
    
    Public Shared Function GenerateSalt() As String
        Dim saltBytes(31) As Byte
        Using rng As RNGCryptoServiceProvider = New RNGCryptoServiceProvider()
            rng.GetBytes(saltBytes)
        End Using
        Return Convert.ToBase64String(saltBytes)
    End Function
    
    Public Shared Function VerifyPassword(password As String, hash As String, salt As String) As Boolean
        Dim computedHash As String = HashPassword(password, salt)
        Return computedHash = hash
    End Function
End Class

9.5.2 入力値検証とXSS対策

Public Class SecurityHelper
    Public Shared Function SanitizeInput(input As String) As String
        If String.IsNullOrEmpty(input) Then
            Return String.Empty
        End If
        
        ' HTMLエンコード
        Dim sanitized As String = HttpUtility.HtmlEncode(input)
        
        ' 危険なスクリプトタグを除去
        sanitized = Regex.Replace(sanitized, "<script.*?>.*?</script>", "", RegexOptions.IgnoreCase)
        
        Return sanitized
    End Function
    
    Public Shared Function ValidateEmail(email As String) As Boolean
        Dim emailPattern As String = "^[^@\s]+@[^@\s]+\.[^@\s]+$"
        Return Regex.IsMatch(email, emailPattern)
    End Function
    
    Public Shared Function IsStrongPassword(password As String) As Boolean
        ' 最低8文字、大文字・小文字・数字・特殊文字を含む
        Return password.Length >= 8 AndAlso
               Regex.IsMatch(password, "[A-Z]") AndAlso
               Regex.IsMatch(password, "[a-z]") AndAlso
               Regex.IsMatch(password, "[0-9]") AndAlso
               Regex.IsMatch(password, "[^A-Za-z0-9]")
    End Function
End Class

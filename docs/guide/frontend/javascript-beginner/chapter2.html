第2章: 数字と文字を扱おう

🎯 この章の実践プロジェクト

簡単な計算機を作成します

変数を使って数値を保存し、計算結果を表示する電卓プログラムを作ってみましょう！

この章で学ぶこと
  • 変数の概念（箱の考え方）
  • 数値の扱い方
  • 文字列の扱い方
  • 基本的な四則演算

変数って何？

2.1 変数は「箱」のようなもの

変数とは、データを入れておく「箱」のようなものです。 この箱には名前を付けて、後で中身を取り出したり、新しい値を入れたりできます。

例えば：

  • 「名前」という箱に「太郎」を入れる
  • 「年齢」という箱に「25」を入れる
  • 「好きな食べ物」という箱に「ラーメン」を入れる

2.2 変数を作ってみよう

JavaScriptでは、letという言葉を使って変数を作ります。

実習 2-1: 最初の変数

開発者ツールのコンソールに以下を入力してください：

let myName = "太郎"; console.log(myName);

「太郎」と表示されましたか？これで「myName」という箱に「太郎」を入れて、それを表示しました。

変数の作り方の基本形：

let 変数名 = 値;

数値を扱ってみよう

2.3 数値の基本

JavaScriptでは、数値をそのまま書くことができます。 小数点も使えます。

実習 2-2: 数値を扱う

以下を試してみましょう：

let age = 25; let height = 170.5; let temperature = -3; console.log(age); console.log(height); console.log(temperature);

2.4 四則演算をしてみよう

JavaScriptでは、数学で使う計算記号を使って計算ができます。

計算 記号 例
足し算 + 5 + 3 = 8
引き算 - 10 - 4 = 6
掛け算 * 6 * 7 = 42
割り算 / 15 / 3 = 5
実習 2-3: 計算してみよう

以下を試してみましょう：

let a = 10; let b = 3; console.log(a + b); // 13 console.log(a - b); // 7 console.log(a * b); // 30 console.log(a / b); // 3.333...

2.5 計算結果を変数に保存

計算結果を変数に保存して、後で使うことができます。

実習 2-4: 計算結果を保存
let price = 100; let quantity = 5; let total = price * quantity; console.log("商品の値段: " + price + "円"); console.log("数量: " + quantity + "個"); console.log("合計: " + total + "円");

文字列を扱ってみよう

2.6 文字列の基本

文字列は、文字や文章を表すデータです。 JavaScriptでは、文字列を"（ダブルクォート）や'（シングルクォート）で囲みます。

実習 2-5: 文字列を作る
let firstName = "太郎"; let lastName = "田中"; let hobby = 'プログラミング'; console.log(firstName); console.log(lastName); console.log(hobby);

2.7 文字列を繋げてみよう

+記号を使って、文字列を繋げることができます。

実習 2-6: 文字列を繋げる
let firstName = "太郎"; let lastName = "田中"; let fullName = lastName + firstName; console.log(fullName); // 田中太郎 let greeting = "こんにちは、" + fullName + "さん"; console.log(greeting); // こんにちは、田中太郎さん

2.8 テンプレートリテラル（便利な文字列）

バッククォート（`）を使うと、文字列の中に変数を直接埋め込めます。

実習 2-7: テンプレートリテラル
let name = "太郎"; let age = 25; let message = `私の名前は${name}で、年齢は${age}歳です`; console.log(message);

こちらの方が読みやすくて便利ですね！

実践プロジェクト：簡単な計算機

🎯 プロジェクト: 簡単な計算機を作ろう

変数と四則演算を使って、簡単な計算機プログラムを作りましょう。

ステップ1: 基本的な計算機

2つの数値を使って、四則演算を行います：

let num1 = 15; let num2 = 4; console.log("=== 簡単な計算機 ==="); console.log(`${num1} + ${num2} = ${num1 + num2}`); console.log(`${num1} - ${num2} = ${num1 - num2}`); console.log(`${num1} * ${num2} = ${num1 * num2}`); console.log(`${num1} / ${num2} = ${num1 / num2}`);
ステップ2: 買い物の計算

実際の買い物を想定した計算をしてみましょう：

let itemName = "りんご"; let price = 150; let quantity = 5; let tax = 0.1; // 10%の税金 let subtotal = price * quantity; let taxAmount = subtotal * tax; let total = subtotal + taxAmount; console.log("=== 買い物計算 ==="); console.log(`商品名: ${itemName}`); console.log(`単価: ${price}円`); console.log(`数量: ${quantity}個`); console.log(`小計: ${subtotal}円`); console.log(`税額: ${taxAmount}円`); console.log(`合計: ${total}円`);
ステップ3: 身長とBMIの計算

身長と体重からBMIを計算してみましょう：

let name = "太郎"; let height = 170; // cm let weight = 65; // kg let heightInMeters = height / 100; let bmi = weight / (heightInMeters * heightInMeters); console.log("=== BMI計算 ==="); console.log(`名前: ${name}`); console.log(`身長: ${height}cm`); console.log(`体重: ${weight}kg`); console.log(`BMI: ${bmi.toFixed(1)}`);
理解度確認クイズ
  1. 変数を宣言するときに使う言葉は何ですか？
  2. JavaScriptで掛け算を行う記号は何ですか？
  3. 文字列を囲むために使う記号は何ですか？（2つ答えてください）
  4. 文字列を繋げるために使う記号は何ですか？
  5. テンプレートリテラルで使う記号は何ですか？
答え
  1. let
  2. *（アスタリスク）
  3. "（ダブルクォート）と'（シングルクォート）
  4. +（プラス）
  5. `（バッククォート）

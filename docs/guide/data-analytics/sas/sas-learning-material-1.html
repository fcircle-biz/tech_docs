1.1 SASとは

SAS（Statistical Analysis System）は、1976年に開発されたデータ分析・統計解析のためのソフトウェアパッケージです。特に以下の分野で広く使用されています：

  • 製薬・バイオ統計：臨床試験データの解析
  • 金融・保険：リスク分析、信用評価
  • 政府・公共機関：統計調査、政策立案
  • 学術研究：社会科学、経済学研究
SASの特徴

SASは単なる統計ソフトウェアではなく、データ管理から高度な分析、レポート作成まで包括的に対応できるプラットフォームです。堅牢性と信頼性に優れ、大容量データの処理にも適しています。

1.2 SAS環境のセットアップ

学習目的には、無料で利用できるSAS OnDemand for Academicsが最適です。以下の手順でセットアップを行います：

実習 1-1: SAS OnDemand for Academicsのセットアップ

無料のSAS学習環境にアクセスしましょう。

手順
  1. SAS公式サイト（https://www.sas.com/ja_jp/software/on-demand-for-academics.html）にアクセス
  2. 「今すぐ開始」をクリックしてSASプロファイルアカウントを作成
  3. 学術機関のメールアドレスまたは学習目的であることを証明
  4. アカウント承認後、SAS OnDemand for Academicsにログイン
  5. SAS Studioを起動してクラウドベースのSAS環境にアクセス
確認方法

SAS Studioの画面が表示され、サンプルプログラムが実行できれば正常にセットアップ完了です。

1.3 SASプログラムの基本構造

SASプログラムは主に以下の2つのステップで構成されます：

  1. データステップ（DATA STEP）：データの読み込み、変換、加工
  2. プロシージャステップ（PROC STEP）：分析、集計、出力

基本的な記述ルール

  • 各ステートメントは;（セミコロン）で終了
  • RUN;でステップの実行
  • QUIT;でプロシージャの終了
  • コメントは/* コメント */または* コメント;
実習 1-2: 最初のSASプログラム

簡単なSASプログラムを作成して実行してみましょう。

実行例
/* 最初のSASプログラム */
data sample;
    input name $ age height;
    datalines;
田中 25 170
佐藤 30 165
鈴木 28 175
;
run;

proc print data=sample;
run;
期待される結果

3行のデータが表形式で出力されます。これでSASの基本的な動作を確認できました。

1.4 データステップの基本

データステップはDATA文で始まり、RUN;文で実行されます。基本構文は以下の通りです：

DATA データセット名;
    /* データの読み込み・変換処理 */
    INPUT 変数名1 変数名2 ...;
    DATALINES;
    データ行1
    データ行2
    ...
    ;
RUN;

重要なキーワード

  • DATA：新しいデータセットを作成
  • INPUT：変数の定義と読み込み
  • DATALINES：データ行の開始
  • $：文字変数の指定

1.5 変数とデータセット

SASにおけるデータの基本概念を理解しましょう：

変数（Variable）

  • 数値変数：数値データを格納（例：age, height）
  • 文字変数：文字データを格納（例：name$）

観測値（Observation）

データの1行に相当し、一つの対象（個体、事例）に関する情報を含みます。

データセット（Dataset）

変数と観測値で構成される二次元テーブル。SASライブラリに保存されます。

実習 1-3: 変数とデータセットの操作

異なる種類の変数を持つデータセットを作成してみましょう。

実行例
data employees;
    input emp_id name $ department $ salary hire_date :date9.;
    format hire_date date9.;
    datalines;
1001 田中太郎 営業部 350000 01JAN2020
1002 佐藤花子 経理部 400000 15MAR2019
1003 鈴木一郎 技術部 450000 10JUL2021
;
run;

proc print data=employees;
    title "従業員データ";
run;
期待される結果

従業員IDは数値、名前と部署は文字、給与は数値、入社日は日付形式で表示されます。

  3. データステップを実行するためのキーワードは何ですか？
  4. SASにおけるデータの1行を何と呼びますか？

解答例：1) セミコロン(;) 2) ドル記号($) 3) RUN; 4) 観測値(Observation) 5) /* */ と * ;

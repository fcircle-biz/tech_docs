1.1 R言語とは

Rは統計計算とデータ分析に特化したプログラミング言語です。 1993年にロス・イハカとロバート・ジェントルマンによって開発され、現在では世界中のデータサイエンティスト、 統計学者、研究者に広く使用されています。

Rの特徴

  • オープンソース: 無料で利用できる
  • 統計機能: 豊富な統計解析機能を標準装備
  • データ可視化: 高品質なグラフとチャートの作成が可能
  • 拡張性: 18,000以上のパッケージが利用可能
  • クロスプラットフォーム: Windows、Mac、Linuxで動作
  • コミュニティ: 活発な開発者・利用者コミュニティ

Rが使用される分野

  • データサイエンス・機械学習
  • 統計解析・バイオインフォマティクス
  • 金融分析・リスク管理
  • マーケティング分析
  • 学術研究・論文作成

1.2 RStudioとは

RStudioはR言語のための統合開発環境（IDE）です。 Rの標準的なコンソールよりも使いやすく、データ分析を効率的に行うための機能が充実しています。

RStudioの主な機能

  • ソースエディタ: シンタックスハイライト、コード補完
  • コンソール: R命令の実行とリアルタイム結果表示
  • 環境ペイン: 変数とオブジェクトの管理
  • ファイルブラウザ: プロジェクトファイルの管理
  • プロット表示: グラフとチャートのプレビュー
  • パッケージ管理: ライブラリのインストールと管理
実習 1-1: RとRStudioのインストール

まずはR言語とRStudioをインストールして、基本環境を整えましょう。

手順
  1. Rのインストール
    • CRAN公式サイトにアクセス
    • お使いのOS（Windows/Mac/Linux）を選択
    • 最新版のRをダウンロードしてインストール
  2. RStudioのインストール
    • RStudio公式サイトにアクセス
    • 「RStudio Desktop」の無料版をダウンロード
    • インストーラーを実行してインストール
  3. 動作確認
    • RStudioを起動
    • コンソールにR.versionと入力してEnterキーを押す
    • Rのバージョン情報が表示されることを確認
期待される結果

RStudioが正常に起動し、Rのバージョン情報が表示されれば成功です。

1.3 RStudioの基本操作

RStudioのインターフェースは4つの主要なペインに分かれています：

1. ソースエディタ（左上）

  • Rスクリプトファイル（.R）の編集
  • R Markdownファイルの編集
  • コードの実行（Ctrl+Enter / Cmd+Enter）

2. コンソール（左下）

  • R命令の直接実行
  • 実行結果の表示
  • エラーメッセージの表示

3. 環境・履歴ペイン（右上）

  • Environment: 作成された変数とオブジェクトの一覧
  • History: 実行したコマンドの履歴
  • Connections: データベース接続の管理

4. ファイル・プロット・パッケージペイン（右下）

  • Files: ファイルブラウザ
  • Plots: 作成したグラフの表示
  • Packages: インストール済みパッケージの管理
  • Help: ヘルプドキュメントの表示

1.4 Rの基本的な計算

Rを電卓として使用することから始めてみましょう。コンソールに以下のコードを入力してみてください：

基本的な算術演算

# 足し算
2 + 3

# 引き算
10 - 4

# 掛け算
5 * 6

# 割り算
15 / 3

# べき乗
2^3

# 余り
17 %% 5

# 整数除算
17 %/% 5

変数への代入

# 変数に値を代入
x <- 10
y <- 5

# 変数を使った計算
x + y
x * y

# 結果を新しい変数に代入
result <- x * y
result
重要なポイント
  • Rでは<-を代入演算子として使用（=も使用可能）
  • #はコメント記号（この後の文字は実行されない）
  • 変数名は大文字と小文字を区別する
  • 変数名は文字または.で始まり、数字は2文字目以降で使用可能
実習 1-2: 基本的な計算と変数操作

RStudioのコンソールを使用して、基本的な計算と変数操作を実践してみましょう。

手順
  1. RStudioのコンソールに以下のコードを一行ずつ入力
  2. 各行でEnterキーを押して実行
  3. 結果を確認し、Environmentペインで変数の状態をチェック
実行例
# 売上データの計算例
価格 <- 1500
数量 <- 120
税率 <- 0.1

# 税抜き売上
売上_税抜き <- 価格 * 数量
売上_税抜き

# 税込み売上
売上_税込み <- 売上_税抜き * (1 + 税率)
売上_税込み

# 年間予測売上（12ヶ月分）
年間売上予測 <- 売上_税込み * 12
年間売上予測
期待される結果
  • 売上_税抜き: 180000
  • 売上_税込み: 198000
  • 年間売上予測: 2376000
  • Environmentペインに4つの変数が表示される

1.5 パッケージの管理

Rの強力な機能の一つは、豊富なパッケージエコシステムです。 パッケージは特定の機能や分析手法を提供する追加ライブラリです。

パッケージのインストール

# パッケージのインストール
install.packages("ggplot2")

# 複数パッケージの同時インストール
install.packages(c("dplyr", "readr", "tidyr"))

パッケージの読み込み

# パッケージを読み込んで使用可能にする
library(ggplot2)

# または
require(ggplot2)

インストール済みパッケージの確認

# インストール済みパッケージの一覧
installed.packages()

# 読み込み済みパッケージの確認
sessionInfo()

1.6 ヘルプ機能の使用

Rには充実したヘルプシステムが組み込まれています。関数の使い方がわからない時に活用しましょう。

ヘルプの呼び出し方法

# 関数のヘルプを表示
help(mean)

# 短縮記法
?mean

# 例を含むヘルプ
example(mean)

# キーワード検索
help.search("regression")

# または
??regression
実習 1-3: パッケージとヘルプの活用

よく使用されるパッケージをインストールし、ヘルプ機能を使用してRの学習環境を整えましょう。

手順
  1. 以下のパッケージをインストール
  2. パッケージを読み込んで動作確認
  3. ヘルプ機能を使用して関数の使い方を調べる
実行例
# データ操作用パッケージのインストール
install.packages(c("dplyr", "ggplot2", "readr"))

# パッケージの読み込み
library(dplyr)
library(ggplot2)

# 組み込みデータセットの確認
data()

# mtcarsデータセットを使用
data(mtcars)
head(mtcars)

# mean関数のヘルプを確認
?mean

# mtcarsの燃費の平均値を計算
mean(mtcars$mpg)
期待される結果
  • パッケージが正常にインストール・読み込まれる
  • mtcarsデータセットの最初の6行が表示される
  • ヘルプペインにmean関数の説明が表示される
  • 燃費の平均値（約20.09）が計算される
理解度確認クイズ
  1. R言語の主な特徴を3つ挙げて説明してください。
  2. RStudioの4つの主要なペインの名前と役割を説明してください。
  3. Rにおける代入演算子は何ですか？また、変数名の命名ルールを説明してください。
  4. パッケージをインストールするコマンドと、読み込むコマンドを書いてください。
  5. mean関数のヘルプを表示するコマンドを2通り書いてください。
  6. 以下の計算結果を予測してください：
    x <- 8; y <- 3; result <- x %% y; result
解答例
  1. オープンソース（無料）、統計解析に特化、豊富なパッケージエコシステム
  2. ソースエディタ（スクリプト編集）、コンソール（命令実行）、環境ペイン（変数管理）、ファイル・プロット・パッケージペイン（ファイル管理・グラフ表示）
  3. <-（=も可）、文字または.で始まり、大文字小文字を区別、数字は2文字目以降
  4. install.packages("パッケージ名")、library(パッケージ名)
  5. help(mean)、?mean
  6. 2（8を3で割った余り）

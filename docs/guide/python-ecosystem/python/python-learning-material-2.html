第2章: 基本的な文法と変数

基本的な文法と変数

この章で学ぶこと
  • 変数の概念と命名規則を理解する
  • 基本的なデータ型（数値、文字列、ブール値）を学ぶ
  • 演算子の種類と使い方を習得する
  • 型変換の方法を学ぶ
  • コメントの書き方を理解する

2.1 変数とは

変数は、データを格納するための「入れ物」のようなものです。 Pythonでは、変数に値を代入することで、その値を後で使用できます。

変数の基本的な使い方

# 変数への代入
name = "太郎"
age = 25
height = 175.5

# 変数の使用
print(name)      # 太郎
print(age)       # 25
print(height)    # 175.5

変数の命名規則

  • 英字、数字、アンダースコア（_）が使用可能
  • 数字で始めることはできない
  • 大文字と小文字は区別される
  • 予約語（if、for、while等）は使用できない
  • スネークケース（snake_case）が推奨される
良い変数名の例
  • user_name, total_price, student_count
  • is_valid, has_permission, can_edit
避けるべき変数名
  • 1name（数字で開始）
  • for（予約語）
  • x, data（意味が不明確）

2.2 データ型の基礎

主要なデータ型

データ型 説明 例
int 整数 42, -10, 0
float 浮動小数点数 3.14, -2.5, 1.0
str 文字列 "Hello", 'Python', "123"
bool ブール値 True, False

データ型の確認

# type()関数でデータ型を確認
number = 42
print(type(number))       # <class 'int'>

price = 99.99
print(type(price))        # <class 'float'>

message = "Hello, Python!"
print(type(message))      # <class 'str'>

is_student = True
print(type(is_student))   # <class 'bool'>

2.3 文字列の操作

文字列の作成

# シングルクォートとダブルクォート
greeting = "こんにちは"
name = 'Python'

# 複数行の文字列
long_text = """これは
複数行にわたる
文字列です"""

文字列の連結と繰り返し

# 文字列の連結
first_name = "山田"
last_name = "太郎"
full_name = first_name + " " + last_name
print(full_name)  # 山田 太郎

# 文字列の繰り返し
pattern = "Python! " * 3
print(pattern)    # Python! Python! Python!

文字列フォーマット

# f-string（推奨方法）
name = "田中"
age = 30
message = f"私の名前は{name}で、{age}歳です。"
print(message)

# format()メソッド
template = "{}さんは{}歳です"
result = template.format("佐藤", 25)
print(result)
実習 2-1: 変数と文字列操作

自己紹介プログラムを作成しましょう。

手順
  1. 自分の名前、年齢、趣味を変数に代入する
  2. f-stringを使って自己紹介文を作成する

# 個人情報の変数定義
my_name = "鈴木花子"

2.4 演算子

算術演算子

演算子 説明 例 結果
+ 加算 5 + 3 8
- 減算 10 - 4 6
* 乗算 7 * 2 14
/ 除算 15 / 4 3.75
// 整数除算 15 // 4 3
% 剰余 15 % 4 3
** べき乗 2 ** 3 8

比較演算子

# 比較演算子の例
a = 10
b = 5

print(a > b)     # True
print(a < b)     # False
print(a == b)    # False
print(a != b)    # True
print(a >= b)    # True
print(a <= b)    # False

論理演算子

# 論理演算子の例
x = True
y = False

print(x and y)   # False
print(x or y)    # True
print(not x)     # False
print(not y)     # True

2.5 型変換

明示的な型変換

# 文字列から数値への変換
number_str = "123"
number_int = int(number_str)
number_float = float(number_str)

print(number_int)    # 123
print(number_float)  # 123.0

# 数値から文字列への変換
age = 25
age_str = str(age)
print("私は" + age_str + "歳です")

# ブール値への変換
print(bool(1))       # True
print(bool(0))       # False
print(bool(""))      # False（空文字列）
print(bool("Hello")) # True
型変換の注意点
  • 不正な文字列を数値に変換しようとするとエラーが発生
  • 例: int("abc") → ValueError
  • 小数点を含む文字列をintに変換する場合は、先にfloatに変換
実習 2-2: 計算プログラム

円の面積と円周を計算するプログラムを作成しましょう。

手順
  1. 円の半径を変数に代入する
  2. 円周率（3.14159）を定数として定義する
  3. 面積と円周を計算する
  4. 結果を分かりやすく出力する
コード例
# 円の計算プログラム
radius = 5.0          # 半径
pi = 3.14159         # 円周率

# 計算
area = pi * radius ** 2      # 面積 = π × r²
circumference = 2 * pi * radius  # 円周 = 2 × π × r

# 結果の出力
print(f"半径: {radius}")
print(f"面積: {area:.2f}")
print(f"円周: {circumference:.2f}")

# 型の確認
print(f"半径の型: {type(radius)}")
print(f"面積の型: {type(area)}")
期待される結果
半径: 5.0
面積: 78.54
円周: 31.42
半径の型: <class 'float'>
面積の型: <class 'float'>

2.6 コメント

コメントの書き方

# これは一行コメントです
name = "Python"  # 行末コメント

"""
これは複数行コメントです。
詳細な説明を書くときに使います。
関数やクラスの説明にも使用されます。
"""

def greet():
    """
    挨拶を表示する関数
    
    Returns:
        str: 挨拶メッセージ
    """
    return "こんにちは、Python!"

良いコメントの書き方

  • なぜを説明する（何をしているかは明らか）
  • 複雑なロジックの説明
  • TODO やFIXMEなどの注意事項
  • 設計上の決定の背景
理解度確認クイズ
  1. 問題1: 次のうち、有効な変数名はどれですか？
    • a) user_name
    • b) 2nd_value
    • c) for
    • d) userName
  2. 問題2: 10 // 3の結果は何ですか？
  3. 問題3: 文字列 "123" を整数に変換する関数は何ですか？
  4. 問題4: f-stringを使って、名前が"太郎"、年齢が25の場合の出力文を書いてください。
  5. 問題5: bool("")の結果は何ですか？
解答を見る

解答:

  1. a) user_name と d) userName（ただしPythonではuser_nameが推奨）
  2. 3（整数除算）
  3. int("123")
  4. f"私の名前は{name}で、{age}歳です"
  5. False（空文字列はFalse）

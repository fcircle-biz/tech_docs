C++の便利機能 - STL入門

C++標準テンプレートライブラリ（STL）の基礎を学びます。vector（可変長配列）、string（文字列クラス）、範囲for文など、C++をより便利に使うための機能を習得します。

この章で学ぶこと

  • STL（標準テンプレートライブラリ）とは何かを理解する
  • vectorの基本的な使い方をマスターする
  • stringクラスの便利な機能を学ぶ
  • 範囲for文でコードをシンプルに書けるようになる

13.1 STLとは何か

STL（Standard Template Library：標準テンプレートライブラリ）は、C++の標準ライブラリに含まれる、よく使う機能をまとめた部品集です。 料理に例えると、「毎回レシピを一から作らなくても、既に用意された便利な調理器具や下ごしらえ済みの食材を使える」ようなものです。

STLには主に以下の3つの要素があります：

コンテナ

データを格納する「入れ物」です。vector（配列のような入れ物）、list（リスト）、map（辞書のような入れ物）などがあります。

イテレータ

コンテナの中身を順番に見ていくための「ポインタのようなもの」です。

アルゴリズム

コンテナに対して操作を行う関数です。ソート（並べ替え）、検索、変換などができます。

ポイント

この章では、特に初心者がよく使う「vector（可変長配列）」と「string（文字列クラス）」を中心に学びます。これらをマスターすれば、C/C++プログラミングが格段に楽になります！

13.2 vector - 可変長配列

vectorとは

vectorは、「サイズを自動で変更できる配列」です。C言語の配列は宣言時にサイズが固定されますが、vectorは要素を追加すると自動的に大きくなります。

flowchart LR A["C言語の配列
int arr[5] = サイズ固定"] B["C++のvector
サイズ可変＋自動拡張"] C["メリット
・要素追加が簡単
・サイズ自動管理
・範囲外アクセス検出"] A --> B B --> C

vectorの基本的な使い方

vector_basic.cpp 
#include <iostream>
#include <vector>  // vectorを使うために必要
using namespace std;

int main() {
    // 空のvectorを作成（整数型）
    vector<int> numbers;

    // 要素を追加（push_back）
    numbers.push_back(10);
    numbers.push_back(20);
    numbers.push_back(30);

    // サイズを取得
    cout << "要素数: " << numbers.size() << endl;

    // 要素にアクセス（配列と同じ）
    cout << "1番目の要素: " << numbers[0] << endl;
    cout << "2番目の要素: " << numbers[1] << endl;

    return 0;
}

重要ポイント

vector<int> の <int> は「整数型のvector」という意味です。これを「テンプレート」と呼びます。vector<double> なら小数型、vector<string> なら文字列型のvectorになります。

よく使うvectorのメンバ関数

メンバ関数 説明 使用例
push_back() 末尾に要素を追加 v.push_back(10);
pop_back() 末尾の要素を削除 v.pop_back();
size() 要素数を取得 int n = v.size();
empty() 空かどうか判定 if (v.empty())
clear() 全要素を削除 v.clear();
at() 範囲チェック付きアクセス int x = v.at(0);

実習 13-1: vectorで成績管理

生徒の点数をvectorに格納し、平均点を計算するプログラムを作成してみましょう。

grade_manager.cpp 
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> scores;  // 点数を格納するvector

    // 点数を追加
    scores.push_back(85);
    scores.push_back(92);
    scores.push_back(78);
    scores.push_back(88);
    scores.push_back(95);

    // 合計点を計算
    int total = 0;
    for (int i = 0; i < scores.size(); i++) {
        total += scores[i];
    }

    // 平均点を計算
    double average = (double)total / scores.size();

    cout << "生徒数: " << scores.size() << "人" << endl;
    cout << "平均点: " << average << "点" << endl;

    return 0;
}

13.3 string - 文字列クラス

C言語の文字列とC++の文字列

C言語では文字列をchar str[] = "Hello";のような文字配列で扱いましたが、これには以下のような不便さがありました：

  • 文字列の長さを自分で管理する必要がある
  • 文字列の結合や比較が面倒（strcpy、strcmp などの関数を使う）
  • バッファオーバーフロー（配列の範囲外アクセス）のリスクがある

C++のstringクラスは、これらの問題を解決し、文字列を簡単に扱えるようにしてくれます。

string_basic.cpp 
#include <iostream>
#include <string>  // stringを使うために必要
using namespace std;

int main() {
    // 文字列の作成
    string name = "Yamada";
    string greeting = "Hello, ";

    // 文字列の結合（+演算子で簡単！）
    string message = greeting + name;
    cout << message << endl;  // Hello, Yamada

    // 文字列の長さ
    cout << "文字数: " << name.length() << endl;

    // 文字列の比較（==演算子で簡単！）
    if (name == "Yamada") {
        cout << "山田さんですね！" << endl;
    }

    return 0;
}

よく使うstringのメンバ関数

メンバ関数 説明 使用例
length() 文字列の長さを取得 int len = s.length();
empty() 空文字列か判定 if (s.empty())
substr() 部分文字列を取得 s.substr(0, 5);
find() 文字列を検索 s.find("abc");
replace() 文字列を置換 s.replace(0, 3, "XYZ");
append() 文字列を末尾に追加 s.append("end");

実習 13-2: 文字列操作プログラム

ユーザーの名前を入力し、挨拶メッセージを表示するプログラムを作成してみましょう。

greeting.cpp 
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;

    // 名前を入力
    cout << "あなたの名前を入力してください: ";
    cin >> name;

    // 挨拶メッセージを作成
    string message = "こんにちは、" + name + "さん！";
    cout << message << endl;

    // 名前の文字数を表示
    cout << "あなたの名前は" << name.length() << "文字です。" << endl;

    return 0;
}

13.4 範囲for文 - 要素の巡回を簡単に

従来のfor文の不便さ

vectorの全要素を処理する場合、従来はこのように書いていました：

traditional_for.cpp 
vector<int> numbers = {10, 20, 30, 40, 50};

// 従来のfor文（インデックスを使う）
for (int i = 0; i < numbers.size(); i++) {
    cout << numbers[i] << endl;
}

範囲for文（Range-based for）

C++11から導入された範囲for文を使うと、もっとシンプルに書けます：

range_based_for.cpp 
vector<int> numbers = {10, 20, 30, 40, 50};

// 範囲for文（シンプル！）
for (int num : numbers) {
    cout << num << endl;
}

重要ポイント

for (int num : numbers) は「numbersの各要素をnumという変数で順番に取り出す」という意味です。インデックスを使わなくてよいので、コードが読みやすくなります。

参照を使った範囲for文

要素を変更したい場合や、大きなオブジェクトをコピーしたくない場合は、参照（&）を使います：

range_for_reference.cpp 
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> numbers = {1, 2, 3, 4, 5};

    // 参照を使って各要素を2倍にする
    for (int& num : numbers) {
        num *= 2;  // 元の配列が書き換わる
    }

    // 結果を表示（読み取り専用なので const & を使う）
    for (const int& num : numbers) {
        cout << num << " ";  // 2 4 6 8 10
    }
    cout << endl;

    return 0;
}

実習 13-3: 範囲for文で文字列処理

文字列のvectorを作成し、範囲for文で全文字列を表示してみましょう。

fruit_list.cpp 
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    // 果物のリストを作成
    vector<string> fruits = {"りんご", "バナナ", "ぶどう", "みかん"};

    // 範囲for文で表示
    cout << "好きな果物リスト:" << endl;
    for (const string& fruit : fruits) {
        cout << "・" << fruit << endl;
    }

    return 0;
}

理解度確認クイズ

  1. Q1. STLとは何の略ですか？また、STLの3つの主要な要素を挙げてください。

    回答を表示

    STLは「Standard Template Library（標準テンプレートライブラリ）」の略です。3つの主要な要素は以下の通りです：
    ① コンテナ（データを格納する入れ物）
    ② イテレータ（コンテナの中身を順番に見ていくためのもの）
    ③ アルゴリズム（コンテナに対して操作を行う関数）

  2. Q2. vectorの特徴を、C言語の配列と比較して説明してください。

    回答を表示

    vectorの特徴：
    ・サイズが可変（要素を追加すると自動的に大きくなる）
    ・push_back()で簡単に要素を追加できる
    ・size()で要素数を簡単に取得できる
    ・at()を使えば範囲外アクセスを検出できる
    C言語の配列は宣言時にサイズが固定され、これらの機能がありません。

  3. Q3. 次のコードの出力結果を答えてください。

    vector<int> v = {10, 20, 30};
v.push_back(40);
v.pop_back();
cout << v.size();
    回答を表示

    出力結果は「3」です。
    最初にvは{10, 20, 30}の3要素です。push_back(40)で{10, 20, 30, 40}の4要素になりますが、その後pop_back()で末尾の40が削除され、{10, 20, 30}の3要素に戻ります。

  4. Q4. 範囲for文の利点を説明してください。

    回答を表示

    範囲for文の利点：
    ① コードが短く、読みやすい（インデックス変数が不要）
    ② インデックスの範囲外アクセスのミスを防げる
    ③ どんなコンテナでも同じ書き方で使える
    例: for (int num : numbers) という書き方で、numbersの各要素を順番に取り出せます。

この章のまとめ

  • STL（標準テンプレートライブラリ）は、C++でよく使う機能をまとめた便利な部品集である
  • vectorは可変長配列で、push_back()、size()、at()などの便利な機能がある
  • stringクラスは文字列操作を簡単にし、+演算子での結合や==演算子での比較ができる
  • 範囲for文を使うと、コンテナの全要素を簡潔に処理できる

ファイル入出力

テキストファイルの読み書き方法を学びます。C言語スタイル（fopen/fclose）とC++スタイル（fstream）の両方を理解し、データの永続化ができるようになります。

この章で学ぶこと

  • ファイル入出力の概念とデータの永続化を理解する
  • C言語スタイル（fopen/fclose/fprintf/fscanf）でファイルを読み書きできる
  • C++スタイル（fstream/ifstream/ofstream）でファイルを読み書きできる
  • エラー処理とファイル操作のベストプラクティスを学ぶ

14.1 ファイル入出力とは何か - データの永続化

これまで作成したプログラムは、実行が終了するとメモリ上のデータはすべて消えてしまいました。 ファイル入出力を使うと、データをハードディスクやSSDに保存して、プログラムを終了しても残しておくことができます。 これを「データの永続化」と呼びます。

たとえば、ゲームのセーブデータ、設定ファイル、ログファイル、文書データなど、多くのアプリケーションでファイル入出力が使われています。

（ディスク上）")] C --> D["ファイル入力
（読み込み）"] D --> E["プログラム
（メモリ上のデータ）"]

ポイント

ファイル入出力には大きく分けて「テキストファイル」と「バイナリファイル」の2種類があります。この章ではテキストファイル（文字データ）の扱い方を中心に学びます。

14.2 C言語スタイルのファイル操作（fopen/fclose）

C言語では、stdio.hヘッダーに定義されているfopen関数でファイルを開き、fclose関数でファイルを閉じます。

fopenの基本書式

基本構文 
FILE *fopen(const char *filename, const char *mode);

fopenは、ファイルを開いてFILE型へのポインタを返します。 ファイルが開けなかった場合はNULLを返すので、エラー処理が必要です。

ファイルオープンモード

モード 意味 ファイルが存在しない場合
"r" 読み込み専用 エラー（NULL）
"w" 書き込み専用（上書き） 新規作成
"a" 追記モード 新規作成
"r+" 読み書き両用 エラー（NULL）
"w+" 読み書き両用（上書き） 新規作成
"a+" 読み書き両用（追記） 新規作成

注意

"w"モードでファイルを開くと、既存のファイルの内容はすべて削除されます。既存データを残したい場合は"a"（追記）モードを使いましょう。

14.3 ファイルへの書き込み（fprintf、fputs）

ファイルへの書き込みには、fprintfやfputsを使います。使い方はprintfやputsとほぼ同じです。

write_file.c 
#include <stdio.h>

int main() {
    FILE *fp;  // ファイルポインタ

    // ファイルを書き込みモードで開く
    fp = fopen("output.txt", "w");

    // ファイルが開けたか確認
    if (fp == NULL) {
        printf("ファイルを開けませんでした\n");
        return 1;  // エラー終了
    }

    // ファイルに書き込み
    fprintf(fp, "こんにちは、ファイル!\n");
    fprintf(fp, "数値: %d\n", 42);
    fputs("文字列を書き込み\n", fp);

    // ファイルを閉じる（重要！）
    fclose(fp);

    printf("ファイルに書き込みました\n");
    return 0;
}

重要ポイント

ファイルを開いたら、必ずfcloseで閉じることが重要です。閉じないと、データが正しく書き込まれなかったり、メモリリークの原因になります。

14.4 ファイルからの読み込み（fscanf、fgets）

ファイルからデータを読み込むには、fscanfやfgetsを使います。 また、ファイルの終わりを検出するfeof関数も重要です。

read_file.c 
#include <stdio.h>

int main() {
    FILE *fp;
    char buffer[256];  // 読み込み用バッファ

    // ファイルを読み込みモードで開く
    fp = fopen("output.txt", "r");

    if (fp == NULL) {
        printf("ファイルを開けませんでした\n");
        return 1;
    }

    // ファイルの終わりまで1行ずつ読み込む
    while (fgets(buffer, sizeof(buffer), fp) != NULL) {
        printf("%s", buffer);  // 読み込んだ行を表示
    }

    fclose(fp);
    return 0;
}

fscanfでフォーマット読み込み

read_numbers.c 
#include <stdio.h>

int main() {
    FILE *fp = fopen("data.txt", "r");
    if (fp == NULL) return 1;

    int num;
    // ファイルから整数を読み込む
    while (fscanf(fp, "%d", &num) == 1) {
        printf("読み込んだ数値: %d\n", num);
    }

    fclose(fp);
    return 0;
}

ヒント

fgetsは改行文字も含めて読み込みます。一方、fscanfは指定したフォーマットに従ってデータを解析します。用途に応じて使い分けましょう。

14.5 C++スタイルのファイル操作（fstream）

C++では、<fstream>ヘッダーで提供されるストリームクラスを使います。 iostreamと同じように、<<や>>演算子でファイルの読み書きができます。

主なクラス

ifstream

ファイルからの入力（読み込み）専用

ofstream

ファイルへの出力（書き込み）専用

fstream

読み書き両用

14.6 ofstreamでファイルに書き込む

write_cpp.cpp 
#include <iostream>
#include <fstream>  // ファイルストリーム
#include <string>

int main() {
    // ofstreamオブジェクトを作成
    std::ofstream outFile("output.txt");

    // ファイルが開けたか確認
    if (!outFile) {
        std::cerr << "ファイルを開けませんでした" << std::endl;
        return 1;
    }

    // coutと同じようにファイルに書き込む
    outFile << "こんにちは、C++のファイル操作!" << std::endl;
    outFile << "数値: " << 42 << std::endl;
    outFile << "小数: " << 3.14 << std::endl;

    // ファイルは自動的に閉じられる（デストラクタ）
    // 明示的に閉じる場合: outFile.close();

    std::cout << "ファイルに書き込みました" << std::endl;
    return 0;
}

重要ポイント

C++のfstreamは、オブジェクトがスコープを抜けると自動的にファイルを閉じます（RAII: Resource Acquisition Is Initialization）。これにより、閉じ忘れのミスを防げます。

14.7 ifstreamでファイルを読み込む

read_cpp.cpp 
#include <iostream>
#include <fstream>
#include <string>

int main() {
    std::ifstream inFile("output.txt");

    if (!inFile) {
        std::cerr << "ファイルを開けませんでした" << std::endl;
        return 1;
    }

    std::string line;
    // 1行ずつ読み込む
    while (std::getline(inFile, line)) {
        std::cout << line << std::endl;
    }

    return 0;
}

データをパースして読み込む

read_data.cpp 
#include <iostream>
#include <fstream>
#include <string>

int main() {
    std::ifstream inFile("scores.txt");
    if (!inFile) return 1;

    std::string name;
    int score;

    // 名前とスコアをペアで読み込む
    while (inFile >> name >> score) {
        std::cout << name << ": " << score << "点" << std::endl;
    }

    return 0;
}

14.8 エラー処理とファイル操作のベストプラクティス

ファイル操作のチェックリスト

必ずエラーチェック

ファイルが開けたか確認し、失敗時の処理を実装する

必ずファイルを閉じる

C言語ではfclose、C++ではスコープを抜けるか明示的にclose

適切なモードを選択

上書き・追記・読み込みなど、目的に応じたモードを使う

バッファオーバーフロー対策

fgetsなどでバッファサイズを必ず指定する

エラー処理の例（C++）

error_handling.cpp 
#include <iostream>
#include <fstream>

int main() {
    std::ifstream inFile("data.txt");

    // ファイルオープンの確認
    if (!inFile.is_open()) {
        std::cerr << "エラー: ファイルが開けません" << std::endl;
        return 1;
    }

    int value;
    while (inFile >> value) {
        // データの読み込みと処理
        std::cout << value << std::endl;
    }

    // 読み込みエラーのチェック
    if (inFile.bad()) {
        std::cerr << "読み込みエラーが発生しました" << std::endl;
    } else if (inFile.eof()) {
        std::cout << "ファイルの終わりに達しました" << std::endl;
    }

    return 0;
}

14.9 実習：成績データの保存と読み込み

実習 14-1: 成績管理プログラム（C++版）

生徒の名前と点数をファイルに保存し、後で読み込んで表示するプログラムを作成しましょう。

手順

  1. 1 3人分の名前と点数をキーボードから入力する
  2. 2 データを"scores.txt"というファイルに保存する
  3. 3 ファイルからデータを読み込んで画面に表示する
ヒントを表示

• ofstreamで書き込み、ifstreamで読み込みます

• データ形式: "名前 点数" を1行に1人ずつ

• while (inFile >> name >> score) でペアで読み込めます

score_manager.cpp（解答例） 
#include <iostream>
#include <fstream>
#include <string>

int main() {
    // 1. データの入力と保存
    std::ofstream outFile("scores.txt");
    if (!outFile) {
        std::cerr << "ファイルを開けませんでした" << std::endl;
        return 1;
    }

    std::cout << "=== 成績入力 ===" << std::endl;
    for (int i = 0; i < 3; i++) {
        std::string name;
        int score;
        std::cout << (i+1) << "人目の名前: ";
        std::cin >> name;
        std::cout << "点数: ";
        std::cin >> score;

        // ファイルに保存
        outFile << name << " " << score << std::endl;
    }
    outFile.close();  // 明示的に閉じる

    // 2. データの読み込みと表示
    std::ifstream inFile("scores.txt");
    if (!inFile) {
        std::cerr << "ファイルを開けませんでした" << std::endl;
        return 1;
    }

    std::cout << "\n=== 成績一覧 ===" << std::endl;
    std::string name;
    int score;
    int total = 0;
    int count = 0;

    while (inFile >> name >> score) {
        std::cout << name << ": " << score << "点" << std::endl;
        total += score;
        count++;
    }

    if (count > 0) {
        std::cout << "平均点: " << (total / count) << "点" << std::endl;
    }

    return 0;
}

理解度確認クイズ

  1. Q1. fopenで"w"モードを指定した場合、既存ファイルの内容はどうなりますか？

    回答を表示

    既存ファイルの内容はすべて削除され、新しいデータで上書きされます。既存データを残したい場合は"a"（追記）モードを使用します。

  2. Q2. C言語でファイルを扱う際、必ず行わなければならない処理は何ですか？

    回答を表示

    ① fopenでファイルが正しく開けたかNULLチェックを行う
    ② 処理が終わったら必ずfcloseでファイルを閉じる
    この2点を守らないと、データの書き込み失敗やメモリリークの原因になります。

  3. Q3. C++のifstreamとofstreamの違いを説明してください。

    回答を表示

    ifstream: ファイルから入力（読み込み）するためのストリームクラス
    ofstream: ファイルへ出力（書き込み）するためのストリームクラス
    両方が必要な場合はfstreamクラスを使います。

  4. Q4. C++のfstreamオブジェクトがスコープを抜けたときに何が起こりますか？

    回答を表示

    デストラクタが自動的に呼ばれ、ファイルが自動的に閉じられます。これをRAII（Resource Acquisition Is Initialization）と呼び、リソースの確実な解放を保証します。

この章のまとめ

  • ファイル入出力により、プログラム終了後もデータを保存できる（永続化）
  • C言語ではfopen/fclose、fprintf/fscanfを使ってファイル操作を行う
  • C++ではfstream（ofstream/ifstream）を使い、coutやcinと同様の操作ができる
  • ファイルを開く際は必ずエラーチェックを行い、終了時には必ずファイルを閉じる
  • C++のfstreamはRAIIにより、スコープを抜けると自動的にファイルを閉じる

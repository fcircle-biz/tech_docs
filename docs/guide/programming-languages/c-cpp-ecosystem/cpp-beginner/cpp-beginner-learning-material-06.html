第6章

繰り返し処理 - ループを使いこなす

for文、while文、do-while文を使って、同じ処理を繰り返す方法を学びます。九九の表示や合計計算など、実用的な例を通じて繰り返し処理のパワーを体験します。

この章で学ぶこと

  • 繰り返し処理（ループ）の概念を理解し、なぜ必要なのかを知る
  • for文、while文、do-while文の違いと使い分けを習得する
  • 九九表示、合計計算など実用的なプログラムが書けるようになる

6.1 繰り返し処理とは

繰り返し処理（ループ）とは、同じ処理を何度も実行する仕組みです。 プログラミングにおいて最も重要な概念の一つで、「順次処理」「条件分岐」「繰り返し処理」の3つの基本構造の一つです。

たとえば、「1から10までの数字を表示する」場合、繰り返し処理を使わなければ、以下のように10行も書く必要があります：

繰り返し処理を使わない場合（非効率） 
printf("1\n");
printf("2\n");
printf("3\n");
printf("4\n");
printf("5\n");
printf("6\n");
printf("7\n");
printf("8\n");
printf("9\n");
printf("10\n");

しかし、繰り返し処理を使えば、たった数行で同じことができます：

繰り返し処理を使った場合（効率的） 
// iが1から10まで繰り返す
for (int i = 1; i <= 10; i++) {
    printf("%d\n", i);  // iの値を表示
}
flowchart TD A["開始"] --> B["カウンタ初期化
（i = 1）"] B --> C{"条件判定
i <= 10?"} C -->|はい| D["処理を実行
（printf）"] D --> E["カウンタを増やす
（i++）"] E --> C C -->|いいえ| F["ループ終了"]

ポイント

繰り返し処理を使うことで、コードが短く読みやすくなり、「1から1000まで」のような大量の繰り返しも簡単に書けるようになります。

6.2 for文 - 回数が決まった繰り返し

for文は、繰り返す回数があらかじめ決まっている場合に最もよく使われるループ構文です。 「10回繰り返す」「配列の要素分だけ繰り返す」といった場面で活躍します。

for文の基本構文

for文の構文 
for (初期化式; 継続条件式; 更新式) {
    // 繰り返し実行する処理
}

for文の3つの要素

  • 初期化式: ループの最初に1回だけ実行。カウンタ変数の初期値を設定する
  • 継続条件式: 各ループの前に評価。trueならループ継続、falseなら終了
  • 更新式: 各ループの最後に実行。カウンタ変数を更新する

例題1: 1から5までの数字を表示

basic_for.c 
#include <stdio.h>

int main() {
    // 1から5までの数字を表示
    for (int i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }

    return 0;
}

実行結果

1
2
3
4
5

例題2: 1から10までの合計を計算

sum_numbers.c 
#include <stdio.h>

int main() {
    int sum = 0;  // 合計を格納する変数

    // 1から10までを足していく
    for (int i = 1; i <= 10; i++) {
        sum = sum + i;  // sumにiを加算
    }

    printf("1から10までの合計: %d\n", sum);

    return 0;
}

実行結果

1から10までの合計: 55

重要ポイント

for文のカウンタ変数（i）は、ループの中でのみ有効です。これを「ローカル変数」と呼びます。また、i++は「i = i + 1」の省略形で、iを1増やす意味です。

6.3 while文 - 条件が満たされる間繰り返す

while文は、繰り返す回数が事前に決まっていない場合に使います。 「条件が真（true）である間ずっと繰り返す」という動作をします。

while文の基本構文

while文の構文 
while (継続条件式) {
    // 条件が真の間、繰り返し実行する処理
}
flowchart TD A["開始"] --> B{"継続条件式を評価"} B -->|true| C["処理を実行"] C --> B B -->|false| D["ループ終了"]

例題1: 1から5までの数字を表示（while版）

basic_while.c 
#include <stdio.h>

int main() {
    int i = 1;  // カウンタ変数を初期化

    // iが5以下の間繰り返す
    while (i <= 5) {
        printf("%d\n", i);
        i++;  // カウンタを1増やす
    }

    return 0;
}

例題2: 正の数の合計を計算（0で終了）

sum_until_zero.c 
#include <stdio.h>

int main() {
    int num;
    int sum = 0;

    printf("正の数を入力してください（0で終了）:\n");

    scanf("%d", &num);  // 最初の数を入力

    // 入力が0でない間繰り返す
    while (num != 0) {
        sum += num;  // sumにnumを加算（sum = sum + num と同じ）
        printf("現在の合計: %d\n", sum);
        scanf("%d", &num);  // 次の数を入力
    }

    printf("最終合計: %d\n", sum);

    return 0;
}

for文とwhile文の使い分け

for文: 繰り返す回数が決まっている場合（例: 配列の要素を全て処理）
while文: 特定の条件が満たされるまで繰り返す場合（例: ユーザー入力が0になるまで）

6.4 do-while文 - 最低1回は実行する

do-while文は、while文と似ていますが、最低でも1回は必ず実行される点が異なります。 条件判定が「後」に行われるため、メニュー選択やバリデーションチェックなどでよく使われます。

do-while文の基本構文

do-while文の構文 
do {
    // 処理（最低1回は実行される）
} while (継続条件式);
flowchart LR subgraph while文 A1["開始"] --> B1{"条件判定"} B1 -->|true| C1["処理実行"] C1 --> B1 B1 -->|false| D1["終了"] end subgraph do-while文 A2["開始"] --> C2["処理実行
（必ず1回実行）"] C2 --> B2{"条件判定"} B2 -->|true| C2 B2 -->|false| D2["終了"] end

例題: 1から10の範囲で数値を入力させる

input_validation.c 
#include <stdio.h>

int main() {
    int num;

    // 正しい範囲の数値が入力されるまで繰り返す
    do {
        printf("1から10までの数値を入力してください: ");
        scanf("%d", &num);

        if (num < 1 || num > 10) {
            printf("エラー: 範囲外です。もう一度入力してください。\n");
        }
    } while (num < 1 || num > 10);

    printf("入力された数値: %d\n", num);

    return 0;
}

do-while文の活用場面

「最低1回は実行したい」場合に便利です。例えば、メニュー画面を最初に必ず表示したり、ユーザー入力を少なくとも1回は受け取りたい場合などに使います。

6.5 ネストしたループ（二重ループ）

ループの中にループを入れることを「ネスト（入れ子）」と呼びます。 九九の表や2次元配列の処理など、繰り返しの中でさらに繰り返しが必要な場合に使います。

例題: 九九の表を表示する

multiplication_table.c 
#include <stdio.h>

int main() {
    printf("=== 九九の表 ===\n\n");

    // 外側のループ: 段（1の段〜9の段）
    for (int i = 1; i <= 9; i++) {
        printf("%dの段: ", i);

        // 内側のループ: 各段の計算（1〜9を掛ける）
        for (int j = 1; j <= 9; j++) {
            printf("%d ", i * j);
        }

        printf("\n");  // 段が終わったら改行
    }

    return 0;
}

実行結果

=== 九九の表 ===

1の段: 1 2 3 4 5 6 7 8 9
2の段: 2 4 6 8 10 12 14 16 18
3の段: 3 6 9 12 15 18 21 24 27
...（9の段まで続く）

ネストループの動作イメージ

i = 1のとき: 内側のループがj = 1, 2, 3...9まで実行される

i = 2のとき: 内側のループが再びj = 1, 2, 3...9まで実行される

...これをi = 9まで繰り返す

合計実行回数: 9 × 9 = 81回

6.6 break文とcontinue文 - ループの制御

ループの途中で特別な処理をしたい場合、break文とcontinue文を使います。

break文

ループを途中で終了します。ループから完全に抜け出します。 

for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;  // i=5で終了
    }
    printf("%d ", i);
}
// 出力: 1 2 3 4

continue文

現在の繰り返しをスキップして、次の繰り返しに進みます。 

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;  // i=3をスキップ
    }
    printf("%d ", i);
}
// 出力: 1 2 4 5

実用例: 偶数だけを表示

even_numbers.c 
#include <stdio.h>

int main() {
    printf("1から10までの偶数:\n");

    for (int i = 1; i <= 10; i++) {
        // 奇数の場合はスキップ
        if (i % 2 != 0) {
            continue;
        }
        printf("%d ", i);
    }

    printf("\n");
    return 0;
}

理解度確認クイズ

  1. Q1. for文の3つの要素（初期化式、継続条件式、更新式）は何のために必要ですか？

    回答を表示

    初期化式: カウンタ変数の開始値を設定する（例: i = 0）
    継続条件式: ループを続けるかどうかを判定する（例: i < 10）
    更新式: 各ループの最後にカウンタを更新する（例: i++）
    この3つが揃うことで、「何回繰り返すか」を正確に制御できます。

  2. Q2. while文とdo-while文の違いは何ですか？

    回答を表示

    while文: 最初に条件判定を行うため、条件が最初からfalseの場合、1回も実行されません。
    do-while文: 処理を実行してから条件判定を行うため、最低でも1回は必ず実行されます。
    「最低1回は実行したい」場合はdo-while文を使います。

  3. Q3. break文とcontinue文の違いを説明してください。

    回答を表示

    break文: ループを完全に終了し、ループの外に抜け出します。
    continue文: 現在の繰り返しだけをスキップし、次の繰り返しに進みます。ループ自体は終了しません。

  4. Q4. 二重ループ（ネストしたループ）で、外側が3回、内側が4回繰り返す場合、合計で何回処理が実行されますか？

    回答を表示

    合計 12回（3 × 4 = 12）実行されます。
    外側のループが1回実行されるごとに、内側のループが4回実行されるため、3 × 4 = 12回になります。

この章のまとめ

  • 繰り返し処理（ループ）は、同じ処理を効率的に繰り返す仕組みである
  • for文は回数が決まった繰り返しに、while文は条件が満たされる間の繰り返しに使う
  • do-while文は最低1回は必ず実行されるため、メニュー選択などで便利
  • break文でループを終了、continue文で現在の繰り返しをスキップできる
  • ネストしたループ（二重ループ）で、九九表や2次元配列の処理ができる

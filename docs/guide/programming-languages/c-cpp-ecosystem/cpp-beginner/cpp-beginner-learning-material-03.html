変数と基本データ型

データを保存する「変数」という箱の概念を学びます。整数（int）、小数（float/double）、文字（char）など、C言語の基本的なデータ型と変数の宣言方法を、身近な例を使って解説します。

この章で学ぶこと

  • 変数とは何か、なぜ必要なのかを理解する
  • 整数型（int）、小数型（float/double）、文字型（char）の使い方を学ぶ
  • 変数の宣言と初期化、値の代入方法をマスターする
  • printfを使って変数の値を表示できるようになる

3.1 変数とは何か - 「箱」のたとえ

変数（へんすう）とは、データを保存しておく「箱」のようなものです。 プログラムの中で数値や文字を一時的に覚えておきたいとき、変数という名前のついた箱に入れておくことができます。

たとえば、あなたの部屋に「本」「文房具」「洋服」を入れる箱があるように、プログラムでも「年齢」「名前」「合計金額」などのデータを入れる箱（変数）を用意します。 箱には必ず名前がついていて、「この箱には何が入っているか」がわかるようになっています。

flowchart LR A["変数名: age
（箱の名前）"] --> B["値: 20
（箱の中身）"] C["変数名: name
（箱の名前）"] --> D["値: Taro
（箱の中身）"] E["変数名: price
（箱の名前）"] --> F["値: 1980.5
（箱の中身）"]

ポイント

変数は「データを一時的に記憶する仕組み」です。プログラムが終了すると変数の中身も消えてしまいます。ずっと残しておきたいデータはファイルに保存する必要があります（後の章で学びます）。

3.2 整数型（int）- 数を数える

整数型（int）は、小数点を含まない数（0, 1, 2, -5, 100など）を保存する変数です。 「人数を数える」「年齢を記録する」「点数を保存する」など、日常でよく使う数を扱うときに使います。

変数の宣言と初期化

変数を使うには、まず「この名前の箱を用意します」と宣言する必要があります。これを変数の宣言と呼びます。

variable_example.c 
#include <stdio.h>

int main() {
    // 変数の宣言（箱を用意する）
    int age;

    // 変数に値を代入（箱に数値を入れる）
    age = 20;

    // 宣言と同時に初期化（箱を用意して、最初から値を入れる）
    int score = 85;

    // 変数の値を表示
    printf("年齢: %d歳\n", age);
    printf("点数: %d点\n", score);

    return 0;
}

実行結果 

年齢: 20歳
点数: 85点

重要ポイント

%dは「ここに整数を表示してね」という指示です。printfで変数の値を表示するときは、このフォーマット指定子を使います。

3.3 小数型（float/double）- 小数を扱う

小数型は、小数点を含む数（3.14, -0.5, 98.6など）を保存する変数です。 C言語にはfloat（単精度浮動小数点数）とdouble（倍精度浮動小数点数）の2種類がありますが、 一般的には精度が高いdoubleを使うことが多いです。

float_example.c 
#include <stdio.h>

int main() {
    // float型（単精度）- 小数点以下6〜7桁の精度
    float temperature = 36.5;

    // double型（倍精度）- 小数点以下15桁程度の精度
    double pi = 3.141592653589793;
    double price = 1980.99;

    // 変数の値を表示（%fは小数用のフォーマット指定子）
    printf("体温: %.1f度\n", temperature);
    printf("円周率: %.15f\n", pi);
    printf("価格: %.2f円\n", price);

    return 0;
}

実行結果 

体温: 36.5度
円周率: 3.141592653589793
価格: 1980.99円

ポイント

%.1fは「小数点以下1桁まで表示」、%.2fは「小数点以下2桁まで表示」という意味です。小数点以下の桁数を指定できます。

データ型 精度 メモリサイズ 用途
float 約6〜7桁 4バイト メモリ節約が重要な場合
double 約15〜16桁 8バイト 一般的な用途（推奨）

3.4 文字型（char）- 1文字を扱う

文字型（char）は、1文字だけを保存する変数です。 'A'、'a'、'1'、'@'のように、シングルクォート（'）で囲んで書きます。

char_example.c 
#include <stdio.h>

int main() {
    // 文字型変数の宣言と初期化
    char grade = 'A';
    char initial = 'T';
    char symbol = '@';

    // 文字を表示（%cは文字用のフォーマット指定子）
    printf("成績: %c\n", grade);
    printf("イニシャル: %c\n", initial);
    printf("記号: %c\n", symbol);

    // 文字は内部的には数値（ASCIIコード）で保存されている
    printf("文字 'A' の数値表現: %d\n", grade);

    return 0;
}

実行結果 

成績: A
イニシャル: T
記号: @
文字 'A' の数値表現: 65

重要ポイント

文字は内部的にはASCIIコードという数値で保存されています。'A'は65、'B'は66...という具合です。%dで表示すると、その数値が見えます。

つまずきポイント

1文字は'A'（シングルクォート）、文字列は"Hello"（ダブルクォート）です。混同しやすいので注意しましょう。文字列については第7章で詳しく学びます。

3.5 変数の命名規則 - 良い名前のつけ方

変数名にはルールと推奨される書き方があります。ルールを守らないとエラーになり、推奨される書き方に従うと読みやすいコードになります。

必須ルール（守らないとエラー）

  • 数字で始めてはいけない（1age は✗）
  • 予約語は使えない（int、return など）
  • 記号は使えない（アンダースコア_のみOK）
  • スペースを入れてはいけない

推奨される書き方

  • 意味のある名前をつける（age、total_price）
  • 小文字で書き、単語をアンダースコアで区切る
  • 短すぎず長すぎず（a より age が良い）
  • 英語の名前を使う（ローマ字は避ける）
悪い例 良い例 理由
a age 意味が明確
1name name1 数字で始まらない
user-name user_name ハイフンではなくアンダースコア
nenrei age ローマ字ではなく英語

3.6 基本データ型まとめ

データ型 用途 例 printf指定子
int 整数 -5, 0, 42, 1000 %d
float 小数（単精度） 3.14, -0.5, 98.6 %f
double 小数（倍精度） 3.141592653589793 %f または %lf
char 1文字 'A', 'x', '9', '@' %c

理解度確認クイズ

  1. Q1. 変数とは何ですか？簡単に説明してください。

    回答を表示

    変数とは、データを一時的に保存しておく「箱」のようなものです。変数には名前がついていて、その名前を使ってデータを出し入れできます。プログラムが終了すると、変数の中身も消えてしまいます。

  2. Q2. 次のうち、正しい変数名はどれですか？
    ① 1age ② user_name ③ total-price ④ int

    回答を表示

    正解は② user_name です。
    ① 数字で始まるのでエラー
    ③ ハイフン（-）は使えない（アンダースコア_のみOK）
    ④ intは予約語なので使えない

  3. Q3. 整数を保存する変数の型は何ですか？

    回答を表示

    int型です。整数（小数点を含まない数）を保存するときに使います。

  4. Q4. 3.14という小数を保存したい場合、どの型を使いますか？

    回答を表示

    float型またはdouble型を使います。一般的には精度が高いdouble型が推奨されます。

  5. Q5. printfで整数を表示するときのフォーマット指定子は何ですか？

    回答を表示

    %dです。例: printf("年齢: %d歳\n", age);

  6. Q6. 'A'と"A"の違いは何ですか？

    回答を表示

    'A'（シングルクォート）は1文字（char型）、"A"（ダブルクォート）は文字列です。char型には'A'のように書き、文字列は第7章で学びます。

この章のまとめ

  • 変数は「データを一時的に保存する箱」で、名前をつけて管理する
  • int型は整数、float/double型は小数、char型は1文字を保存する
  • 変数名は数字で始められず、予約語やほとんどの記号は使えない
  • printfのフォーマット指定子は %d（整数）、%f（小数）、%c（文字）

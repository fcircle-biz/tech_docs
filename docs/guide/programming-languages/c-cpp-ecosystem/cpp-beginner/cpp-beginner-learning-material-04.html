第4章

演算子と式

足し算、引き算などの算術演算子、大小比較の比較演算子、条件を組み合わせる論理演算子を学びます。電卓のように計算を行い、結果を変数に保存する方法を習得します。

この章で学ぶこと

  • 算術演算子を使って計算を行い、結果を変数に代入できるようになる
  • 比較演算子を使って値の大小や等価性を判定できるようになる
  • 論理演算子を使って複数の条件を組み合わせることができる
  • 演算子の優先順位を理解し、意図通りの計算結果を得られるようになる

4.1 式とは何か

プログラミングにおける「式（expression）」とは、値を計算して結果を返すものです。 たとえば、算数で「3 + 5」と書くと答えは8になりますよね。これと同じように、C/C++でも「3 + 5」という式を書くと、計算結果の8が得られます。

式は、変数と演算子を組み合わせて作ります。以下のようなものが式の例です：

式の例 
// 算術式
int a = 10 + 5;        // 10と5を足した結果（15）をaに代入
int b = 20 - 3;        // 20から3を引いた結果（17）をbに代入
int c = 4 * 3;         // 4と3をかけた結果（12）をcに代入

// 複数の変数を使った式
int x = 10;
int y = 20;
int sum = x + y;       // xとyの合計（30）をsumに代入

ポイント

式は「値を生み出す計算」です。プログラムは式を評価（計算）して、結果を変数に代入したり、画面に表示したりします。

4.2 算術演算子 - 計算の基本

算術演算子は、足し算・引き算・かけ算・割り算などの基本的な計算を行うための記号です。 電卓と同じように、C/C++でも数値を計算できます。

基本的な算術演算子

演算子 意味 使用例 結果
+ 足し算 10 + 5 15
- 引き算 10 - 5 5
* かけ算 10 * 5 50
/ 割り算 10 / 5 2
% 余り 10 % 3 1
arithmetic.cpp 
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3; // 変数の宣言と初期化

    cout << "a + b = " << (a + b) << endl; // 13
    cout << "a - b = " << (a - b) << endl; // 7
    cout << "a * b = " << (a * b) << endl; // 30
    cout << "a / b = " << (a / b) << endl; // 3（整数の割り算）
    cout << "a % b = " << (a % b) << endl; // 1（余り）

    return 0;
}

整数の割り算に注意

int型同士の割り算は、小数点以下が切り捨てられます。
例: 10 / 3 = 3（小数点以下の.333...は切り捨て）
小数を含む結果が欲しい場合は、double型を使いましょう。

4.3 代入演算子とインクリメント/デクリメント

複合代入演算子

変数の値を更新するとき、x = x + 5;のように書くこともできますが、より短く書ける複合代入演算子があります。

演算子 意味 使用例 同等の式
+= 加算して代入 x += 5; x = x + 5;
-= 減算して代入 x -= 3; x = x - 3;
*= 乗算して代入 x *= 2; x = x * 2;
/= 除算して代入 x /= 2; x = x / 2;

インクリメント/デクリメント演算子

変数の値を1増やす、または1減らす操作はプログラムでよく使うため、専用の演算子が用意されています。

演算子 意味 使用例 同等の式
++ 1増やす（インクリメント） x++; または ++x; x = x + 1;
-- 1減らす（デクリメント） x--; または --x; x = x - 1;
assignment.cpp 
#include <iostream>
using namespace std;

int main() {
    int count = 10; // カウンター変数

    count += 5;     // count = 15 に更新
    cout << "count += 5: " << count << endl;

    count++;        // count = 16 に増加
    cout << "count++: " << count << endl;

    count--;        // count = 15 に減少
    cout << "count--: " << count << endl;

    return 0;
}

4.4 比較演算子 - 値を比べる

比較演算子は、2つの値を比べて「正しい（true）」か「間違い（false）」を判定します。 次の章で学ぶ条件分岐（if文）で、この比較演算子が活躍します。

比較演算子の一覧

演算子 意味 使用例 結果
== 等しい 5 == 5 true（正しい）
!= 等しくない 5 != 3 true（正しい）
< 小さい 3 < 5 true（正しい）
> 大きい 5 > 3 true（正しい）
<= 以下（小さいまたは等しい） 5 <= 5 true（正しい）
>= 以上（大きいまたは等しい） 3 >= 5 false（間違い）
comparison.cpp 
#include <iostream>
using namespace std;

int main() {
    int age = 18;

    cout << "age == 18: " << (age == 18) << endl; // 1（true）
    cout << "age != 20: " << (age != 20) << endl; // 1（true）
    cout << "age < 20: " << (age < 20) << endl;   // 1（true）
    cout << "age >= 18: " << (age >= 18) << endl; // 1（true）

    return 0;
}

よくある間違い

等しいかどうかの比較は==（イコール2つ）です。
=（イコール1つ）は代入の意味なので、注意しましょう。
例: if (x == 5) ← 正しい / if (x = 5) ← 間違い

4.5 論理演算子 - 条件を組み合わせる

論理演算子は、複数の条件を組み合わせて判定するときに使います。 たとえば「年齢が18歳以上で、かつ免許を持っている」のように、2つの条件を同時に満たす必要がある場合に活躍します。

論理演算子の一覧

演算子 意味 使用例 結果
&& かつ（AND） (5 > 3) && (10 > 8) true（両方とも正しい）
|| または（OR） (5 > 3) || (10 < 8) true（どちらか一方が正しい）
! 否定（NOT） !(5 > 3) false（正しいを否定すると間違い）
flowchart TD A["条件1: age >= 18"] --> B{"AND演算 &&"} C["条件2: hasLicense == true"] --> B B -->|"両方true"| D["結果: true
運転可能"] B -->|"片方または両方false"| E["結果: false
運転不可"]
logical.cpp 
#include <iostream>
using namespace std;

int main() {
    int age = 20;
    bool hasLicense = true; // 免許を持っているか

    // AND演算子: 両方の条件が満たされる必要がある
    bool canDrive = (age >= 18) && hasLicense;
    cout << "運転可能? " << canDrive << endl; // 1（true）

    // OR演算子: どちらか一方が満たされればOK
    bool isStudent = (age < 18) || (age >= 18 && age <= 22);
    cout << "学生割引対象? " << isStudent << endl; // 1（true）

    return 0;
}

論理演算子の覚え方

&&（AND）: 「〜かつ〜」両方の条件が必要
||（OR）: 「〜または〜」どちらか一方でOK
!（NOT）: 「〜ではない」条件を反転

4.6 演算子の優先順位

複数の演算子を使った式では、算数と同じように計算の順番があります。 たとえば、2 + 3 * 4という式では、かけ算が先に行われるため、結果は14になります。

演算子の優先順位（高い順）

  1. 1.
    カッコ () - 最優先で計算される
  2. 2.
    単項演算子 ++ -- !
  3. 3.
    乗除算 * / %
  4. 4.
    加減算 + -
  5. 5.
    比較演算子 < > <= >=
  6. 6.
    等価演算子 == !=
  7. 7.
    論理AND &&
  8. 8.
    論理OR ||
  9. 9.
    代入演算子 = += -= - 最後に評価される
priority.cpp 
#include <iostream>
using namespace std;

int main() {
    // かけ算が先に計算される
    int result1 = 2 + 3 * 4;     // 2 + 12 = 14
    cout << "2 + 3 * 4 = " << result1 << endl;

    // カッコで優先順位を変更
    int result2 = (2 + 3) * 4;   // 5 * 4 = 20
    cout << "(2 + 3) * 4 = " << result2 << endl;

    return 0;
}

重要ポイント

優先順位を覚えるのが難しいときは、カッコ () を使って明示的に計算順序を指定しましょう。 コードも読みやすくなります。

実習 4-1: 簡単な計算機プログラムを作ろう

ユーザーから2つの数値を入力してもらい、それらの足し算、引き算、かけ算、割り算の結果を表示するプログラムを作成してみましょう。

手順

  1. 1 2つの整数変数 a と b を宣言し、ユーザーから入力を受け取る
  2. 2 算術演算子を使って、4つの計算結果を求める
  3. 3 結果をコンソールに表示する
解答例を表示 
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cout << "1つ目の数値を入力: ";
    cin >> a;
    cout << "2つ目の数値を入力: ";
    cin >> b;

    cout << "足し算: " << (a + b) << endl;
    cout << "引き算: " << (a - b) << endl;
    cout << "かけ算: " << (a * b) << endl;
    cout << "割り算: " << (a / b) << endl;

    return 0;
}

理解度確認クイズ

  1. Q1. 10 / 3 の結果はいくつですか？（int型同士の割り算の場合）

    回答を表示

    答え: 3
    int型同士の割り算では、小数点以下が切り捨てられます。10 ÷ 3 = 3.333... ですが、3のみが残ります。

  2. Q2. x += 5; は、どの式と同じ意味ですか？

    回答を表示

    答え: x = x + 5;
    += は複合代入演算子で、変数の値に右辺の値を加えて、その結果を変数に代入します。

  3. Q3. (5 > 3) && (10 < 8) の結果は true と false のどちらですか？

    回答を表示

    答え: false
    &&（AND演算子）は、両方の条件が true のときだけ true になります。
    (5 > 3) は true ですが、(10 < 8) は false なので、全体として false になります。

  4. Q4. 2 + 3 * 4 の計算結果はいくつですか？

    回答を表示

    答え: 14
    かけ算は足し算より優先順位が高いため、3 * 4 = 12 が先に計算され、その後 2 + 12 = 14 となります。

この章のまとめ

  • 式は「値を計算して結果を返すもの」で、変数と演算子を組み合わせて作る
  • 算術演算子（+、-、*、/、%）を使って基本的な計算ができる
  • 比較演算子（==、!=、<、>、<=、>=）を使って値を比較し、true/falseで判定できる
  • 論理演算子（&&、||、!）を使って複数の条件を組み合わせることができる
  • 演算子には優先順位があり、カッコ () を使って計算順序を明示できる

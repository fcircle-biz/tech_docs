15.2 演習1: 四則演算電卓プログラム

概要

ユーザーから2つの数値と演算子（+、-、*、/）を入力してもらい、計算結果を表示する電卓プログラムを作ります。 条件分岐、演算子、関数、例外処理（エラーチェック）を組み合わせます。

実習 15-1: 電卓プログラムの作成

四則演算（加算・減算・乗算・除算）ができる簡単な電卓を作成します。

手順

  1. 1 新しいファイル calculator.cpp を作成します
  2. 2 以下のコードを入力します
calculator.cpp 
#include <iostream>
using namespace std;

// 四則演算を行う関数
double calculate(double a, double b, char op) {
    switch(op) {
        case '+': return a + b;
        case '-': return a - b;
        case '*': return a * b;
        case '/':
            if (b == 0) {
                cout << "エラー: ゼロ除算はできません" << endl;
                return 0;
            }
            return a / b;
        default:
            cout << "エラー: 無効な演算子です" << endl;
            return 0;
    }
}

int main() {
    double num1, num2;
    char op;

    cout << "=== 四則演算電卓 ===" << endl;
    cout << "1つ目の数値を入力: ";
    cin >> num1;

    cout << "演算子を入力 (+, -, *, /): ";
    cin >> op;

    cout << "2つ目の数値を入力: ";
    cin >> num2;

    double result = calculate(num1, num2, op);
    cout << "結果: " << num1 << " " << op << " "
         << num2 << " = " << result << endl;

    return 0;
}
  1. 3 コンパイルして実行します: g++ calculator.cpp -o calculator
  2. 4 実行して、いろいろな計算を試してみましょう

コードのポイント

  • calculate() 関数で演算を分離し、再利用可能にしています
  • switch文で演算子による条件分岐を実装
  • ゼロ除算のエラーチェックを実装し、安全性を確保

拡張アイデア

余力があれば、計算履歴を保存する機能や、繰り返し計算できるループ機能を追加してみましょう。

15.3 演習2: 数当てゲーム

概要

コンピュータがランダムに選んだ1〜100の数値を、プレイヤーが当てるゲームを作ります。 プレイヤーが数値を入力すると、「もっと大きい」「もっと小さい」「正解！」とヒントを出します。 ループ、条件分岐、乱数生成を使います。

実習 15-2: 数当てゲームの作成

ランダムな数値を当てる簡単なゲームを作成します。

手順

  1. 1 新しいファイル guessing_game.cpp を作成します
  2. 2 以下のコードを入力します
guessing_game.cpp 
#include <iostream>
#include <cstdlib>  // rand(), srand()用
#include <ctime>    // time()用
using namespace std;

int main() {
    // 乱数の初期化（プログラム起動時刻を種に使う）
    srand(time(0));

    // 1〜100のランダムな数値を生成
    int answer = rand() % 100 + 1;
    int guess;
    int attempts = 0;  // 試行回数カウント

    cout << "=== 数当てゲーム ===" << endl;
    cout << "1から100の間の数値を当ててください！" << endl;

    // 正解するまでループ
    while (true) {
        cout << "予想する数値を入力: ";
        cin >> guess;
        attempts++;

        if (guess < answer) {
            cout << "もっと大きい数です！" << endl;
        } else if (guess > answer) {
            cout << "もっと小さい数です！" << endl;
        } else {
            cout << "正解！" << endl;
            cout << "試行回数: " << attempts << "回" << endl;
            break;  // ループを抜ける
        }
    }

    return 0;
}
  1. 3 コンパイルして実行します: g++ guessing_game.cpp -o game
  2. 4 ゲームを楽しみましょう！何回で当てられるかチャレンジしてみてください

コードのポイント

  • srand(time(0)) で乱数を初期化（毎回違う数値になる）
  • rand() % 100 + 1 で1〜100の範囲の乱数を生成
  • while(true) で無限ループを作り、正解で break で抜ける

拡張アイデア

難易度を選べるようにしたり（範囲を1〜10、1〜100、1〜1000など）、もう一度遊ぶか聞く機能を追加してみましょう。

15.4 演習3: 名簿管理プログラム

概要

名前と年齢を管理する簡単なアプリを作ります。クラス（Person）、STLのvector、ファイル入出力を組み合わせた、これまでの学習の総まとめです。 登録、一覧表示、ファイル保存・読み込み機能を実装します。

実習 15-3: 名簿管理プログラムの作成

クラス、vector、ファイル入出力を使った名簿管理アプリを作成します。

手順

  1. 1 新しいファイル roster.cpp を作成します
  2. 2 以下のコードを入力します
roster.cpp 
#include <iostream>
#include <vector>
#include <fstream>
#include <string>
using namespace std;

// Personクラス
class Person {
public:
    string name;
    int age;

    Person(string n, int a) : name(n), age(a) {}

    void display() const {
        cout << "名前: " << name << ", 年齢: " << age << endl;
    }
};

// 名簿管理クラス
class Roster {
private:
    vector<Person> people;

public:
    // 人物を追加
    void addPerson(const string& name, int age) {
        people.push_back(Person(name, age));
        cout << name << " さんを登録しました。" << endl;
    }

    // 全員表示
    void displayAll() const {
        if (people.empty()) {
            cout << "名簿は空です。" << endl;
            return;
        }
        cout << "\n=== 名簿一覧 ===" << endl;
        for (size_t i = 0; i < people.size(); i++) {
            cout << i + 1 << ". ";
            people[i].display();
        }
    }

    // ファイルに保存
    void saveToFile(const string& filename) {
        ofstream ofs(filename);
        if (!ofs) {
            cout << "ファイルが開けません。" << endl;
            return;
        }
        for (const auto& p : people) {
            ofs << p.name << "," << p.age << endl;
        }
        ofs.close();
        cout << filename << " に保存しました。" << endl;
    }

    // ファイルから読み込み
    void loadFromFile(const string& filename) {
        ifstream ifs(filename);
        if (!ifs) {
            cout << "ファイルが開けません。" << endl;
            return;
        }
        people.clear();
        string name;
        int age;
        while (getline(ifs, name, ',') >> age) {
            ifs.ignore();  // 改行を無視
            people.push_back(Person(name, age));
        }
        ifs.close();
        cout << filename << " から読み込みました。" << endl;
    }
};

int main() {
    Roster roster;
    int choice;

    while (true) {
        cout << "\n=== 名簿管理システム ===" << endl;
        cout << "1. 人物を追加" << endl;
        cout << "2. 一覧表示" << endl;
        cout << "3. ファイルに保存" << endl;
        cout << "4. ファイルから読み込み" << endl;
        cout << "5. 終了" << endl;
        cout << "選択 (1-5): ";
        cin >> choice;

        if (choice == 1) {
            string name;
            int age;
            cout << "名前: ";
            cin.ignore();  // 前の入力のバッファクリア
            getline(cin, name);
            cout << "年齢: ";
            cin >> age;
            roster.addPerson(name, age);
        } else if (choice == 2) {
            roster.displayAll();
        } else if (choice == 3) {
            roster.saveToFile("roster.txt");
        } else if (choice == 4) {
            roster.loadFromFile("roster.txt");
        } else if (choice == 5) {
            cout << "終了します。" << endl;
            break;
        } else {
            cout << "無効な選択です。" << endl;
        }
    }

    return 0;
}
  1. 3 コンパイルして実行します: g++ roster.cpp -o roster
  2. 4 メニューから機能を選んで、名簿管理を試してみましょう

コードのポイント

  • Personクラスでデータをカプセル化
  • Rosterクラスで名簿管理機能をまとめて実装
  • vector<Person> で可変長の人物リストを管理
  • ofstream/ifstream でファイルへの保存・読み込みを実装
  • メニュー方式のUI（ループとif-else）でユーザビリティを向上

拡張アイデア

電話番号やメールアドレスのフィールドを追加したり、名前で検索する機能、削除機能などを追加してみましょう。

理解度確認クイズ

  1. Q1. 電卓プログラムで、switch文を使った理由は何ですか？

    回答を表示

    演算子（+、-、*、/）という複数の選択肢を、1つの変数（op）の値に応じて分岐させるためです。if-else ifでも書けますが、switch文の方が読みやすく、各演算子の処理が明確になります。

  2. Q2. 数当てゲームで、srand(time(0))を実行する理由は何ですか？

    回答を表示

    乱数のシード（種）を現在時刻で初期化するためです。これにより、プログラムを実行するたびに異なる乱数が生成されます。srandを呼ばないと、毎回同じ乱数列になってしまいます。

  3. Q3. 名簿管理プログラムで、vector<Person>を使った理由は何ですか？

    回答を表示

    名簿に登録する人数は実行時に変わるため、固定長の配列では対応できません。vectorは可変長配列なので、push_backで自由に要素を追加でき、人数が増減しても柔軟に対応できます。

  4. Q4. これまでの学習で、最も難しかった概念は何ですか？それをどう克服しましたか？

    考えてみましょう

    この問いは自由回答です。多くの学習者にとって「ポインタ」が最難関ですが、繰り返し図を描いたり、小さなプログラムで実験することで理解が深まります。難しい概念は一度で理解しようとせず、時間をかけて慣れていくことが大切です。

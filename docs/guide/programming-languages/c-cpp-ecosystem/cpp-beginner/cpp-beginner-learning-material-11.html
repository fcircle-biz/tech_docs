C++への移行 - モダンな機能を使おう

C言語からC++へ移行します。iostreamによる入出力、参照（&）の概念、名前空間など、C++ならではの便利な機能を学びます。なぜC++がC言語の「進化版」と呼ばれるのかを理解します。

この章で学ぶこと

  • C言語とC++の違いを理解し、C++の便利な機能を知る
  • iostreamによる入出力（cout、cin）の使い方を習得する
  • 参照（&）の概念と使い方を理解する
  • 名前空間（namespace）の仕組みと使い方を理解する

11.1 C言語とC++の違い

これまで学んできたC言語の知識を基礎にして、いよいよC++の世界に足を踏み入れます。 C++は、C言語をベースにして「オブジェクト指向」という新しい考え方や、より便利な機能を追加した言語です。

「C++」という名前の由来は、C言語のインクリメント演算子「++」から来ています。 つまり、「C言語を1段階進化させた言語」という意味が込められています。

重要な考え方

C++は「C言語の上位互換」と言われることがあります。つまり、C言語のコードはほとんどC++でもそのまま動きます。 これまで学んだprintf()やscanf()も使えますが、C++ではより便利な機能が用意されています。

11.2 iostreamによる入出力（cout、cin）

C言語ではprintf()とscanf()を使って入出力を行いましたが、C++ではcout（シーアウト）とcin（シーイン）という、 より使いやすい入出力の仕組みがあります。

coutによる出力

coutは「Console OUTput（コンソール出力）」の略で、画面に文字や数値を表示するために使います。 printf()のように書式指定子（%d、%fなど）を覚える必要がなく、直感的に使えます。

C言語スタイル

// C言語スタイル
int age = 25;
printf("私は%d歳です\n", age);

C++スタイル

// C++スタイル
int age = 25;
cout << "私は" << age << "歳です" << endl;

coutの仕組み

「<<」（ストリーム挿入演算子）は、右側のデータを左側の出力先（cout）に「流し込む」イメージです。
endlは「end of line（行の終わり）」の略で、改行とバッファのフラッシュを行います（\nと似ていますが、より確実です）。

cinによる入力

cinは「Console INput（コンソール入力）」の略で、キーボードからデータを受け取るために使います。 scanf()のように書式指定子やアドレス演算子（&）を使う必要がありません。

C言語スタイル

// C言語スタイル
int number;
scanf("%d", &number);

C++スタイル

// C++スタイル
int number;
cin >> number;

実習 11-1: C++スタイルの入出力を試す

C++の入出力を実際に使ってみましょう。

hello_cpp.cpp 
#include <iostream>  // iostreamヘッダをインクルード
using namespace std;  // std名前空間を使用

int main() {
    // 名前を入力してもらう
    string name;
    cout << "あなたの名前を入力してください: ";
    cin >> name;

    // 年齢を入力してもらう
    int age;
    cout << "あなたの年齢を入力してください: ";
    cin >> age;

    // 結果を表示
    cout << "こんにちは、" << name << "さん！" << endl;
    cout << "あなたは" << age << "歳ですね。" << endl;

    return 0;
}

コンパイルと実行

# コンパイル（C++コンパイラを使う）
g++ hello_cpp.cpp -o hello_cpp

# 実行
./hello_cpp

実行例:
あなたの名前を入力してください: 太郎
あなたの年齢を入力してください: 20
こんにちは、太郎さん！
あなたは20歳ですね。

11.3 参照（&）の概念

C++には、ポインタとは別に参照（reference）という便利な仕組みがあります。 参照は「変数の別名」を作る機能で、ポインタよりも安全で使いやすいです。

参照の基本

reference_basic.cpp 
#include <iostream>
using namespace std;

int main() {
    int original = 10;    // 元の変数
    int& ref = original;  // originalの参照（別名）を作成

    cout << "original = " << original << endl;  // 10
    cout << "ref = " << ref << endl;            // 10

    // 参照を通じて値を変更
    ref = 20;

    cout << "original = " << original << endl;  // 20（変わった！）
    cout << "ref = " << ref << endl;            // 20

    return 0;
}

ポイント

参照は「変数の別名」です。refはoriginalの別名なので、refを変更するとoriginalも変わります。 ポインタのように*や->を使う必要がなく、普通の変数と同じように扱えます。

関数の引数としての参照

参照の最も便利な使い方は、関数の引数として使うことです。 C言語ではポインタを使って「値渡し」ではなく「参照渡し」を実現しましたが、C++では参照を使うとより簡潔に書けます。

C言語（ポインタ使用）

// 値を2倍にする関数
void doubleValue(int *p) {
    *p = *p * 2;
}

int main() {
    int num = 5;
    doubleValue(&num);  // アドレスを渡す
    printf("%d\n", num);  // 10
    return 0;
}

C++（参照使用）

// 値を2倍にする関数
void doubleValue(int &n) {
    n = n * 2;
}

int main() {
    int num = 5;
    doubleValue(num);  // 普通に渡す
    cout << num << endl;  // 10
    return 0;
}

重要ポイント

参照を使うと、ポインタのように*や&を使わずに、元の変数を直接変更できます。 コードがシンプルになり、ポインタのようなNULLポインタの危険性もありません。

11.4 名前空間（namespace）

C++では名前空間（namespace）という仕組みを使って、関数や変数の名前の衝突を防ぎます。 これは、大規模なプログラムで同じ名前の関数が複数あっても混乱しないようにする仕組みです。

std名前空間

C++の標準ライブラリ（cout、cin、stringなど）は、すべてstd名前空間の中に入っています。 そのため、本来はstd::coutのように書く必要があります。

namespace_example.cpp 
#include <iostream>

// using namespace std; を使わない場合
int main() {
    std::cout << "Hello, World!" << std::endl;
    // std::を毎回つける必要がある
    return 0;
}

毎回std::を書くのは面倒なので、 using namespace std;を使うと省略できます。

using_namespace.cpp 
#include <iostream>
using namespace std;  // std名前空間を使うと宣言

int main() {
    cout << "Hello, World!" << endl;
    // std::を省略できる
    return 0;
}

独自の名前空間を作る

自分でも名前空間を作ることができます。これは大きなプログラムで名前の衝突を避けるのに役立ちます。

custom_namespace.cpp 
#include <iostream>
using namespace std;

// Math名前空間を定義
namespace Math {
    int add(int a, int b) {
        return a + b;
    }

    int multiply(int a, int b) {
        return a * b;
    }
}

int main() {
    // Math名前空間の関数を呼び出す
    cout << "5 + 3 = " << Math::add(5, 3) << endl;
    cout << "5 * 3 = " << Math::multiply(5, 3) << endl;
    return 0;
}

11.5 C++の便利な機能まとめ

iostream

cout、cinによる直感的な入出力。書式指定子を覚える必要がなく、型に応じて自動的に正しく表示されます。

参照（&）

変数の別名を作る機能。ポインタより安全で使いやすく、関数の引数として渡すときに便利です。

名前空間

関数や変数の名前の衝突を防ぐ仕組み。大規模なプログラムで整理された構造を保つのに役立ちます。

string型

C言語のchar配列よりも便利な文字列型。長さを気にせず、+演算子で連結できます（次章で詳しく学びます）。

実習 11-2: 参照を使った値の交換

参照を使って2つの変数の値を交換する関数を作りましょう。

swap_values.cpp 
#include <iostream>
using namespace std;

// 参照を使った値の交換関数
void swap(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;

    cout << "交換前: x = " << x << ", y = " << y << endl;

    swap(x, y);  // 参照渡しで値を交換

    cout << "交換後: x = " << x << ", y = " << y << endl;

    return 0;
}

期待される出力:
交換前: x = 10, y = 20
交換後: x = 20, y = 10

理解度確認クイズ

  1. Q1. C++で画面に文字を出力するために使う標準的な方法は何ですか？

    回答を表示

    cout（シーアウト）を使います。例: cout << "Hello" << endl;
    C言語のprintf()も使えますが、C++ではcoutを使うのが標準的です。

  2. Q2. 参照（reference）とポインタの主な違いは何ですか？

    回答を表示

    参照は変数の別名で、普通の変数と同じように使えます。ポインタのように*や&を使う必要がなく、より安全で直感的です。 また、参照は必ず何かを参照している必要があり、NULLにはできません。

  3. Q3. 「using namespace std;」は何のために使いますか？

    回答を表示

    std名前空間を使うことを宣言します。これにより、std::coutをcoutと省略して書けるようになります。 C++の標準ライブラリの機能をより簡単に使うための便利な宣言です。

  4. Q4. C言語とC++の両方が使える理由は何ですか？

    回答を表示

    C++はC言語をベースに開発されたため、C言語のコードはほぼそのままC++でも動きます（上位互換性）。 そのため、printf()やscanf()などのC言語の関数も、C++のプログラムで使うことができます。

この章のまとめ

  • C++はC言語をベースに、オブジェクト指向など便利な機能を追加した言語である
  • iostream（cout、cin）を使うと、printf/scanfより直感的に入出力ができる
  • 参照（&）は変数の別名を作る機能で、ポインタより安全で使いやすい
  • 名前空間（namespace）により、関数や変数の名前の衝突を防げる
  • C言語の知識はC++でも活かせるが、C++にはより便利な機能がたくさんある

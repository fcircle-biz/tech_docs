C/C++の世界へようこそ

プログラミングとは何か、C言語とC++とはどんな言語かを学びます。その歴史と「なぜ今でも使われているのか」を理解し、開発環境（VSCode + コンパイラ）のインストールまでを丁寧に解説します。

この章で学ぶこと

  • プログラミングとは何か、コンピュータがどのように動くのかを理解する
  • C言語とC++の歴史、特徴、そして今でも使われる理由を知る
  • 開発環境（VSCode + コンパイラ）のセットアップができるようになる

1.1 プログラミングとは何か

プログラミングとは、コンピュータに「やってほしいこと」を正確に伝える作業です。 私たちが日本語や英語で会話するように、コンピュータと会話するには「プログラミング言語」という特別な言葉が必要です。

たとえば、料理をするときのレシピを想像してください。レシピには「1. 玉ねぎを切る」「2. フライパンで炒める」「3. 調味料を加える」のように、順番に手順が書かれています。 プログラミングも同じで、コンピュータに「何を、どの順番でやるか」を細かく指示することで、様々な仕事をさせることができます。

ポイント

プログラミング言語はたくさんありますが（Python、Java、JavaScriptなど）、それぞれ得意な分野が違います。C/C++は「コンピュータの動く仕組みを深く理解できる」「処理速度が速い」という特徴があります。

1.2 C言語とC++の歴史

C言語の誕生（1972年）

C言語は、1972年にアメリカのベル研究所でデニス・リッチー氏によって開発されました。 最初の目的は、Unix（ユニックス）というオペレーティングシステム（OS）を作ることでした。 当時のプログラミング言語は、特定のコンピュータでしか動かないものが多かったのですが、C言語は「どんなコンピュータでも動く移植性の高い言語」として設計されました。

C++の誕生（1983年）

C++は、1983年にビャーネ・ストロヴストルップ氏がC言語を拡張して開発しました。 C言語の「速さ」や「低レベル制御」の強みはそのままに、「オブジェクト指向プログラミング」という新しい考え方を取り入れました。 これにより、大規模で複雑なプログラムを整理して書きやすくなりました。

重要ポイント

C言語は50年以上前に誕生した言語ですが、今でもOSやデバイスドライバ、組み込みシステムなど「高速で動く必要がある」分野で現役バリバリです。

1.3 なぜ今でもC/C++を学ぶのか

「こんなに古い言語を学ぶ必要があるの？」と思うかもしれません。しかし、C/C++には以下のような強力な理由があります：

1. 圧倒的な処理速度

C/C++で書かれたプログラムは、コンピュータのハードウェアに近いレベルで動作するため、PythonやJavaScriptなどの高レベル言語と比べて非常に高速です。

2. メモリを細かく制御できる

ポインタや動的メモリ確保により、コンピュータのメモリ（RAM）を直接操作できます。これにより効率的なプログラムが書けます。

3. OSやゲームエンジンの基盤

Windows、Linux、macOSのようなOS、Unreal EngineやUnityの一部は、C/C++で書かれています。

4. コンピュータの仕組みが深く理解できる

C/C++を学ぶことで、メモリ、CPU、プログラムの実行の仕組みなど、コンピュータサイエンスの基礎が身につきます。

1.4 開発環境のセットアップ

C/C++プログラミングを始めるには、以下の2つのソフトウェアが必要です：

1. エディタ（VSCode）

プログラムを書くための高機能なテキストエディタです。無料で使えて、拡張機能が豊富です。

2. コンパイラ（GCC/Clang/MSVC）

人間が書いたC/C++のコードを、コンピュータが理解できる機械語に変換するソフトです。

OSごとのセットアップ手順

Windows 10/11

  1. 1
    Visual Studio Codeをインストール

    https://code.visualstudio.com/ からダウンロードしてインストール

  2. 2
    MinGW-w64をインストール

    MinGW-w64 または MSYS2 からGCCコンパイラをインストール

  3. 3
    VSCodeにC/C++拡張機能をインストール

    VSCodeを開き、左側の拡張機能アイコンから「C/C++」を検索してインストール

macOS

  1. 1
    Visual Studio Codeをインストール

    https://code.visualstudio.com/ からダウンロードしてインストール

  2. 2
    Xcode Command Line Toolsをインストール

    ターミナルを開き、xcode-select --install を実行

  3. 3
    VSCodeにC/C++拡張機能をインストール

    VSCodeを開き、左側の拡張機能アイコンから「C/C++」を検索してインストール

Linux (Ubuntu/Debian)

  1. 1
    Visual Studio Codeをインストール

    https://code.visualstudio.com/ からダウンロードしてインストール

  2. 2
    GCCをインストール

    ターミナルで sudo apt update と sudo apt install build-essential を実行

  3. 3
    VSCodeにC/C++拡張機能をインストール

    VSCodeを開き、左側の拡張機能アイコンから「C/C++」を検索してインストール

確認方法

コンパイラが正しくインストールされたか確認するには、ターミナル（Windowsの場合はコマンドプロンプト）で以下を実行してください：
gcc --version または g++ --version
バージョン番号が表示されれば成功です！

理解度確認クイズ

  1. Q1. プログラミング言語が必要な理由は何ですか？

    回答を表示

    コンピュータは0と1（機械語）しか理解できません。人間が理解しやすい形でコンピュータに指示を出すために、プログラミング言語が必要です。プログラミング言語で書かれたコードは、コンパイラによって機械語に翻訳されます。

  2. Q2. C言語が誕生した年と、その目的は何ですか？

    回答を表示

    C言語は1972年に誕生しました。目的は、Unixオペレーティングシステムを開発するためでした。また、どんなコンピュータでも動く「移植性の高い言語」を目指して設計されました。

  3. Q3. C/C++を学ぶメリットを3つ挙げてください。

    回答を表示

    ① 圧倒的な処理速度（ハードウェアに近いレベルで動作）
    ② メモリを細かく制御できる（ポインタなど）
    ③ コンピュータの仕組みが深く理解できる（メモリ、CPU、実行の仕組みなど）
    ※ その他、OSやゲームエンジンの基盤として使われているなども正解です。

  4. Q4. C/C++プログラミングを始めるために必要な2つのソフトウェアは何ですか？

    回答を表示

    ① エディタ（Visual Studio Codeなど）- プログラムを書くためのツール
    ② コンパイラ（GCC、Clang、MSVCなど）- 人間が書いたコードを機械語に翻訳するツール

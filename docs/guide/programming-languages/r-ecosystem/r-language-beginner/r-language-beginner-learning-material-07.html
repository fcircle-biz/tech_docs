データフレーム入門 - 表形式でデータを管理する

Excelの表のようにデータを扱う「データフレーム」を学びます。実際のデータ分析で最も使う形式です。

この章で学ぶこと

  • データフレームとは何かを理解する
  • データフレームを作成する方法を学ぶ
  • 列と行を取り出す基本操作をマスターする
  • データフレームの便利な関数を使えるようになる

7.1 データフレームとは何か

データフレームとは、表形式でデータを管理するためのデータ構造です。Excelの表のように、行と列で整理されたデータを扱います。

ベクトルは同じ種類のデータを一列に並べたものでしたが、データフレームは異なる種類のデータを含む複数の列を持つ表を扱えます。

データフレームのイメージ

名前 年齢 身長（cm） 血液型
田中 25 170 A
佐藤 30 165 B
鈴木 28 175 O

このような表形式のデータを、Rでは「データフレーム」として扱います

ベクトルとデータフレームの違い

ベクトル：同じ種類のデータを1列に並べたもの（例：数値だけ、文字だけ）
データフレーム：異なる種類のデータを含む複数の列を持つ表（例：名前（文字）+ 年齢（数値）+ 身長（数値））

7.2 データフレームの作成方法

データフレームは、data.frame()関数を使って作成します。各列をベクトルとして指定します。

基本的な作成方法

data_frame_basic.R 
# データフレームを作成
students <- data.frame(
  name = c("田中", "佐藤", "鈴木"),
  age = c(25, 30, 28),
  height = c(170, 165, 175),
  blood_type = c("A", "B", "O")
)

# データフレームを表示
print(students)

実行結果

   name age height blood_type
1  田中  25    170          A
2  佐藤  30    165          B
3  鈴木  28    175          O

作成時のポイント

列の長さは揃える

すべての列（ベクトル）は同じ長さでなければなりません。長さが異なるとエラーになります。

列名は分かりやすく

列名は英数字とアンダースコア（_）を使うのが一般的です。日本語も使えますが、英語が推奨されます。

7.3 データフレームの基本操作

列を取り出す

データフレームから特定の列を取り出すには、ドル記号（$）を使います。

column_access.R 
# 名前の列を取り出す
students$name
# 結果: "田中" "佐藤" "鈴木"

# 年齢の列を取り出す
students$age
# 結果: 25 30 28

行を取り出す

特定の行を取り出すには、角かっこ [ ] を使います。[行番号, ]のように指定します。

row_access.R 
# 1行目を取り出す
students[1, ]
# 結果: 田中 25 170 A

# 2行目を取り出す
students[2, ]
# 結果: 佐藤 30 165 B

特定のセルを取り出す

特定の行と列を指定してセルの値を取り出すには、[行番号, 列番号]を使います。

cell_access.R 
# 1行目の2列目（年齢）を取り出す
students[1, 2]
# 結果: 25

# 3行目の1列目（名前）を取り出す
students[3, 1]
# 結果: "鈴木"

列名でも指定できます

列番号の代わりに列名を使うこともできます：students[1, "age"]

7.4 データフレームの便利な関数

データフレームを扱う際によく使う便利な関数をご紹介します。

str() - 構造を確認

データフレームの構造（列数、行数、各列のデータ型）を確認できます。 

str(students)
# 'data.frame':	3 obs. of  4 variables:
#  $ name      : chr  "田中" "佐藤" "鈴木"
#  $ age       : num  25 30 28
#  $ height    : num  170 165 175
#  $ blood_type: chr  "A" "B" "O"

head() / tail() - 先頭/末尾を表示

データフレームの先頭や末尾の数行だけを表示します。大きなデータを確認するときに便利です。 

# 先頭2行を表示
head(students, 2)

# 末尾2行を表示
tail(students, 2)

nrow() / ncol() - 行数/列数を取得

データフレームの行数と列数を取得します。 

# 行数を取得
nrow(students)  # 結果: 3

# 列数を取得
ncol(students)  # 結果: 4

names() - 列名を確認

データフレームのすべての列名を確認できます。 

names(students)
# 結果: "name" "age" "height" "blood_type"

print(products)
  2. 2 価格の列だけを取り出して表示してください（products$price）
  3. 3 2行目のデータを表示してください（products[2, ]）
  4. 4 データフレームの構造を確認してください（str(products)）

理解度確認クイズ

  1. Q1. データフレームとは何ですか？

  2. Q2. データフレームから特定の列を取り出すにはどうすればよいですか？

    回答を表示

    nrow(データフレーム名)とすることで、データフレームの行数を取得できます。列数を調べる場合はncol()を使います。

第7章のまとめ

  • データフレームとは、表形式でデータを管理するための構造で、Excelの表のようなものです
  • data.frame()関数を使ってデータフレームを作成します
  • ドル記号（$）を使って列を取り出し、角かっこ [ ] を使って行やセルを取り出します
  • str()、nrow()、ncol()など、便利な関数でデータフレームを操作できます
  • データフレームは実際のデータ分析で最もよく使われる形式です

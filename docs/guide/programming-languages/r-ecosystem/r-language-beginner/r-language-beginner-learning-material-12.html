12.1 CSVファイルとは

CSV（Comma Separated Values）ファイルは、カンマ（,）で区切られたテキスト形式のデータファイルです。ExcelやGoogleスプレッドシートなどで作成したデータを保存する際によく使われます。

CSVファイルの例

例：売上データ（sales_data.csv）の中身

sales_data.csv 
商品名,価格,販売個数
りんご,120,50
バナナ,80,100
みかん,150,30

このCSVファイルをExcelで開くと、次のような表として表示されます：

商品名 価格 販売個数
りんご 120 50
バナナ 80 100
みかん 150 30

CSVファイルの特徴

  • シンプル：テキスト形式なので、どんなプログラムでも読み込める
  • 互換性が高い：Excel、R、Python、データベースなど、さまざまなツールで扱える
  • 軽量：ファイルサイズが小さく、メール添付やダウンロードが簡単

12.2 CSVファイルの読み込み - read.csv()

Rでは、read.csv()関数を使ってCSVファイルを読み込むことができます。

基本的な使い方

CSVファイルの読み込み 
# CSVファイルを読み込み、データフレームに格納
sales_data <- read.csv("sales_data.csv")

# データの内容を確認
print(sales_data)

実行結果 

  商品名 価格 販売個数
1 りんご  120       50
2 バナナ   80      100
3 みかん  150       30

ファイルパスの指定方法

CSVファイルの場所を指定する方法はいくつかあります：

方法1 作業ディレクトリに置く（推奨）

Rの作業ディレクトリにCSVファイルを置くと、ファイル名だけで読み込めます。

作業ディレクトリの確認と設定 
# 現在の作業ディレクトリを確認
getwd()

# 作業ディレクトリを変更（Windows）
setwd("C:/Users/YourName/Documents/R_projects")

# 作業ディレクトリを変更（Mac）
setwd("/Users/YourName/Documents/R_projects")

# ファイル名だけで読み込み
data <- read.csv("sales_data.csv")

方法2 フルパスで指定

ファイルの完全なパスを指定する方法です。

フルパス指定 
# Windowsの場合（スラッシュ / を使う）
data <- read.csv("C:/Users/YourName/Documents/data.csv")

# Macの場合
data <- read.csv("/Users/YourName/Documents/data.csv")

方法3 ファイル選択ダイアログを使う

ダイアログでファイルを選択する方法です（初心者におすすめ）。

ファイル選択ダイアログ 
# ファイル選択ダイアログを表示
file_path <- file.choose()

# 選択したファイルを読み込み
data <- read.csv(file_path)

read.csv()のオプション

オプション 説明 例
header = TRUE 1行目を列名として扱う（デフォルト） header = FALSE
sep = "," 区切り文字を指定 sep = "\t"
na.strings = "" 欠損値として扱う文字列を指定 na.strings = "NA"
encoding = "" 文字エンコーディングを指定 encoding = "UTF-8"
オプションを使った読み込み 
# ヘッダーなし、タブ区切りのファイルを読み込み
data <- read.csv("data.csv", header = FALSE, sep = "\t")

# 欠損値を"NA"として扱う
data <- read.csv("data.csv", na.strings = "NA")

12.3 データの確認と操作

CSVファイルを読み込んだ後は、データの内容を確認し、必要に応じて操作します。

データの確認方法

データ確認の基本コマンド 
# データを読み込み
sales_data <- read.csv("sales_data.csv")

# 先頭6行を表示
head(sales_data)

# 末尾6行を表示
tail(sales_data)

# データ構造を確認
str(sales_data)

# 要約統計量を表示
summary(sales_data)

実践：データの操作

実習：売上データを読み込んで分析してみよう

以下の手順で、売上データを読み込み、基本的な分析をしてみましょう。

ステップ1: データを読み込む 
# CSVファイルを読み込み
sales_data <- read.csv("sales_data.csv")

# データの内容を確認
print(sales_data)

# 売上金額 = 価格 × 販売個数
sales_data$売上金額 <- sales_data$価格 * sales_data$販売個数

12.4 CSVファイルの書き出し - write.csv()

Rで分析した結果を、write.csv()関数を使ってCSVファイルに保存できます。

基本的な使い方

CSVファイルの書き出し 
# データフレームをCSVファイルに保存
write.csv(sales_data, "sales_result.csv")

# 行番号なしで保存
write.csv(sales_data, "sales_result.csv", row.names = FALSE)

row.names = FALSE の意味

row.names = FALSEを指定すると、行番号（1, 2, 3...）がCSVファイルに保存されません。通常はこのオプションを使うことをおすすめします。

write.csv()のオプション

オプション 説明 例
row.names 行番号を含めるか（FALSEを推奨） row.names = FALSE
na 欠損値の表記方法 na = ""
fileEncoding 文字エンコーディング fileEncoding = "UTF-8"
オプションを使った書き出し 
# 行番号なし、欠損値を空白で保存
write.csv(sales_data,
          "sales_result.csv",
          row.names = FALSE,
          na = "")

12.5 実践：CSVファイルを使ったデータ分析

実習：月別売上データの分析

月別の売上データを読み込み、分析して、結果をCSVファイルに保存してみましょう。

月別売上データ（monthly_sales.csv）の例 
月,売上
1月,120000
2月,150000
3月,180000
4月,160000
5月,190000
6月,200000
完全なコード例 
# ステップ1: データを読み込む
monthly_sales <- read.csv("monthly_sales.csv")
print("元のデータ:")
print(monthly_sales)

# ステップ2: 分析を実行
# 平均売上を計算
avg_sales <- mean(monthly_sales$売上)
print(paste("平均売上:", avg_sales, "円"))

# 最高売上を計算
max_sales <- max(monthly_sales$売上)
print(paste("最高売上:", max_sales, "円"))

# 最低売上を計算
min_sales <- min(monthly_sales$売上)
print(paste("最低売上:", min_sales, "円"))

# ステップ3: 目標達成率を追加
target <- 170000  # 目標売上
monthly_sales$目標達成率 <- (monthly_sales$売上 / target) * 100

# ステップ4: 結果を新しいCSVファイルに保存
write.csv(monthly_sales,
          "sales_analysis_result.csv",
          row.names = FALSE)

print("分析結果をsales_analysis_result.csvに保存しました")

保存されたファイルの内容 

月,売上,目標達成率
1月,120000,70.58824
2月,150000,88.23529
3月,180000,105.88235
4月,160000,94.11765
5月,190000,111.76471
6月,200000,117.64706

12.6 よくあるエラーと対処法

トラブルシューティング

エラー1: ファイルが見つからない

Error: cannot open file 'data.csv': No such file or directory

原因：指定したファイルが存在しないか、パスが間違っている

対処法：

  • ファイル名のスペルを確認する
  • getwd()で作業ディレクトリを確認する
  • list.files()でファイル一覧を確認する
  • フルパスで指定してみる

エラー2: 文字化けが発生する

原因：ファイルの文字エンコーディングが正しく認識されていない

対処法：

# UTF-8で読み込み
data <- read.csv("data.csv", fileEncoding = "UTF-8")

# Shift-JISで読み込み（Windowsで作成したファイル）
data <- read.csv("data.csv", fileEncoding = "Shift-JIS")

エラー3: 列名が正しく認識されない

原因：CSVファイルにヘッダー行がない、または余分な行がある

対処法：

# ヘッダーなしとして読み込み
data <- read.csv("data.csv", header = FALSE)

# 読み込み後に列名を手動で設定
colnames(data) <- c("商品名", "価格", "個数")

エラー4: 数値が文字型として読み込まれる

原因：数値に余分な文字（カンマ、円マークなど）が含まれている

対処法：

# カンマを削除して数値に変換
data$価格 <- as.numeric(gsub(",", "", data$価格))

# データ型を確認
str(data)

理解度確認クイズ

  1. Q1. CSVファイルとは何ですか？

    回答を表示

    回答：カンマ（,）で区切られたテキスト形式のデータファイル

    CSV（Comma Separated Values）は、データをカンマで区切って保存したテキストファイルです。ExcelやRなど、さまざまなツールで扱うことができます。

  2. Q2. CSVファイルを読み込む関数は何ですか？

    回答を表示

    回答：read.csv()

    read.csv("ファイル名")の形式で使用します。この関数はCSVファイルを読み込み、データフレームとして返します。

  3. Q3. write.csv()で行番号を含めないようにするオプションは？

    回答を表示

    回答：row.names = FALSE

    write.csv(data, "file.csv", row.names = FALSE)のように指定すると、行番号（1, 2, 3...）がCSVファイルに保存されません。

  4. Q4. 作業ディレクトリを確認するコマンドは？

    回答を表示

    回答：getwd()

    getwd()を実行すると、現在の作業ディレクトリのパスが表示されます。ファイルが見つからない場合は、まずこのコマンドで確認しましょう。

第12章のまとめ

  • CSVファイルは、カンマで区切られたテキスト形式のデータファイルです
  • read.csv()関数でCSVファイルを読み込み、データフレームとして扱えます
  • ファイルの場所は、作業ディレクトリ、フルパス、ファイル選択ダイアログで指定できます
  • 読み込んだデータは、head()、str()、summary()などで確認できます
  • write.csv()関数で分析結果をCSVファイルに保存できます
  • エラーが発生したら、ファイルパス、文字エンコーディング、データ型を確認しましょう

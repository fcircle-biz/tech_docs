第8章

条件分岐 - 「もし〜ならば」の処理

「もし条件に合うなら、こうする」という処理の書き方を学びます。プログラムに判断させる方法です。

この章で学ぶこと

  • if文の基本構文を理解する
  • 比較演算子と論理演算子を使いこなす
  • if-else、if-else if-elseの使い分けができる
  • ifelse関数でベクトル全体に条件分岐を適用できる

8.1 条件分岐とは

条件分岐とは、「もし〜ならば、こうする」という判断をプログラムに行わせる仕組みです。

日常生活でも私たちは常に条件分岐を使っています：

  • もし雨が降っていれば傘を持つ、そうでなければ持たない
  • もしテストが60点以上なら合格、そうでなければ不合格
  • もし購入金額が5000円以上なら送料無料、そうでなければ送料500円

プログラミングでも同じように、条件によって異なる処理を実行できます。

条件分岐のイメージ

8.2 if文の基本

R言語で条件分岐を行うにはif文を使います。基本的な構文は次のとおりです：

基本構文 
# if文の基本構文
if (条件) {
  # 条件が TRUE のときに実行される処理
}

構文のポイント

  • 条件は丸括弧 ( ) で囲みます
  • 実行する処理は波括弧 { } で囲みます
  • 条件がTRUEのときだけ{ }内の処理が実行されます

実例1: 合格判定

example1.R 
# テストの点数
score <- 75

# もし60点以上なら「合格」と表示
if (score >= 60) {
  print("合格です！")
}

# 実行結果: "合格です！" と表示される

実例2: 天気による判断

example2.R 
# 天気
weather <- "雨"

# もし雨なら傘を持つ
if (weather == "雨") {
  print("傘を持っていきましょう")
}

# 実行結果: "傘を持っていきましょう" と表示される

8.3 比較演算子

条件を判定するときに使う比較演算子を覚えましょう。比較演算子は2つの値を比べてTRUEまたはFALSEを返します。

主な比較演算子

演算子 意味 例 結果
== 等しい 5 == 5 TRUE
!= 等しくない 5 != 3 TRUE
> より大きい 10 > 5 TRUE
< より小さい 3 < 8 TRUE
>= 以上（等しいか大きい） 5 >= 5 TRUE
<= 以下（等しいか小さい） 3 <= 5 TRUE

つまずきポイント

等しいかどうかを調べるときは== (イコール2つ)を使います。= (イコール1つ)は代入演算子なので間違えないようにしましょう。

比較演算子の実例

comparison.R 
# 比較演算子の例
x <- 10

# xが10と等しいか
x == 10  # TRUE

# xが5より大きいか
x > 5    # TRUE

# xが20以下か
x <= 20  # TRUE

# xが15と等しくないか
x != 15  # TRUE

8.4 if-else文

条件がTRUEのときとFALSEのとき、両方の場合で異なる処理をしたいときはif-else文を使います。

if-else構文 
# if-else文の構文
if (条件) {
  # 条件が TRUE のときの処理
} else {
  # 条件が FALSE のときの処理
}

実例: 合格・不合格判定

if_else_example.R 
# テストの点数
score <- 55

# もし60点以上なら合格、そうでなければ不合格
if (score >= 60) {
  print("合格です！")
} else {
  print("不合格です。次回頑張りましょう。")
}

# 実行結果: "不合格です。次回頑張りましょう。" と表示される

実例: 年齢による判定

age_check.R 
# 年齢
age <- 17

# 20歳以上かどうかで判定
if (age >= 20) {
  print("成人です")
} else {
  print("未成年です")
}

# 実行結果: "未成年です" と表示される

8.5 if-else if-else文

3つ以上の場合分けをしたいときは、else ifを使って条件を追加できます。

if-else if-else構文 
# if-else if-else文の構文
if (条件1) {
  # 条件1が TRUE のときの処理
} else if (条件2) {
  # 条件1が FALSE で、条件2が TRUE のときの処理
} else {
  # すべての条件が FALSE のときの処理
}

実例: 成績評価

grade_evaluation.R 
# テストの点数
score <- 72

# 点数に応じて成績を判定
if (score >= 80) {
  grade <- "優"
} else if (score >= 70) {
  grade <- "良"
} else if (score >= 60) {
  grade <- "可"
} else {
  grade <- "不可"
}

print(paste("あなたの成績は", grade, "です"))
# 実行結果: "あなたの成績は 良 です"

8.6 論理演算子

複数の条件を組み合わせたいときは論理演算子を使います。

論理演算子

演算子 意味 例
&& かつ（AND）- 両方ともTRUE (x > 5) && (x < 10)
|| または（OR）- どちらかTRUE (x < 5) || (x > 10)
! でない（NOT）- 真偽を反転 !(x == 5)

実例: AND演算子（&&）

and_example.R 
# 気温と天気
temperature <- 28
weather <- "晴れ"

# 気温が25度以上かつ晴れなら外出
if (temperature >= 25 && weather == "晴れ") {
  print("外出日和です！")
} else {
  print("家で過ごしましょう")
}

# 実行結果: "外出日和です！" と表示される

実例: OR演算子（||）

or_example.R 
# 曜日
day <- "土曜日"

# 土曜日または日曜日なら休日
if (day == "土曜日" || day == "日曜日") {
  print("休日です！")
} else {
  print("平日です")
}

# 実行結果: "休日です！" と表示される

8.7 ifelse関数 - ベクトル全体への条件分岐

ifelse関数を使うと、ベクトルの各要素に対して一度に条件分岐を適用できます。

ifelse構文 
# ifelse関数の構文
ifelse(条件, TRUEのときの値, FALSEのときの値)

実例: 複数の点数を一度に判定

ifelse_example.R 
# 5人分のテスト点数
scores <- c(45, 70, 85, 55, 90)

# 各点数が60点以上かどうかを判定
results <- ifelse(scores >= 60, "合格", "不合格")

print(results)
# 実行結果: "不合格" "合格" "合格" "不合格" "合格"

実例: 成績評価をベクトルに適用

nested_ifelse.R 
# 複数の学生の点数
scores <- c(85, 72, 55, 90, 68)

# ifelseをネストして3段階評価
grades <- ifelse(scores >= 80, "優",
                 ifelse(scores >= 60, "良", "不可"))

print(grades)
# 実行結果: "優" "良" "不可" "優" "良"

重要ポイント

通常のif文は1つの値しか扱えませんが、ifelse関数はベクトル全体に一度に適用できます。データフレームの列の値を条件によって変換するときに非常に便利です。

実習 8-1: 気温による服装アドバイス

気温に応じて適切な服装をアドバイスするプログラムを作成しましょう。

手順

  1. 1 変数temperatureに気温（数値）を代入してください
  2. 2 if-else if-else文を使って次のように判定してください：
    • • 25度以上: "半袖で大丈夫です"
    • • 15度以上25度未満: "長袖を着ましょう"
    • • 15度未満: "上着を持っていきましょう"
  3. 3 結果を表示してください
解答例を表示 
# 気温
temperature <- 18

# 服装アドバイス
if (temperature >= 25) {
  print("半袖で大丈夫です")
} else if (temperature >= 15) {
  print("長袖を着ましょう")
} else {
  print("上着を持っていきましょう")
}

# 実行結果: "長袖を着ましょう"

理解度確認クイズ

  1. Q1. 等しいかどうかを判定する比較演算子はどれですか？

    回答を表示

    回答：== (イコール2つ)

    等しいかどうかを判定するときは == を使います。= (イコール1つ)は代入演算子なので、条件判定には使えません。

  2. Q2. 「AかつB」という条件を表す論理演算子はどれですか？

    回答を表示

    回答：&& (AND演算子)

    && は、両方の条件が TRUE のときだけ TRUE を返します。例: (x > 5) && (x < 10) は、xが5より大きく、かつ10より小さいときに TRUE になります。

  3. Q3. ifelse関数の利点は何ですか？

    回答を表示

    回答：ベクトル全体に一度に条件分岐を適用できる

    通常のif文は1つの値しか処理できませんが、ifelse関数はベクトルの各要素すべてに条件分岐を適用できます。これにより、複数のデータを効率的に処理できます。

第8章のまとめ

  • 条件分岐とは、条件によってプログラムの処理を変えることです
  • if文で条件がTRUEのときだけ処理を実行できます
  • 比較演算子（==, !=, >, <, >=, <=）で値を比較できます
  • if-else文でTRUEとFALSEの両方の場合に異なる処理を書けます
  • else ifを使うと3つ以上の場合分けができます
  • 論理演算子（&&, ||, !）で複数の条件を組み合わせられます
  • ifelse関数でベクトル全体に条件分岐を一度に適用できます

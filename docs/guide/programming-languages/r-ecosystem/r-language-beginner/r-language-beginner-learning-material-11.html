第11章

グラフ作成入門 - データを見える化する

棒グラフ、折れ線グラフ、散布図など、基本的なグラフの作り方を学びます。データを視覚的に理解する力を養います。

この章で学ぶこと

  • 基本的なグラフの種類と用途を理解する
  • 棒グラフの作成方法を学ぶ
  • 折れ線グラフの作成方法を学ぶ
  • 散布図の作成方法を学ぶ
  • グラフをカスタマイズする方法を知る

11.1 グラフ作成の基礎

データをグラフ（可視化）で表現すると、数値だけでは見えにくい傾向やパターンが一目でわかるようになります。R言語は、グラフ作成に非常に優れた機能を持っています。

なぜグラフが重要なのか

視覚的理解

数値の羅列では見えない傾向や異常値が、グラフにすると一目でわかります。

コミュニケーション

分析結果を他の人に説明する際、グラフがあると理解してもらいやすくなります。

発見のツール

データを可視化することで、予想外のパターンや関係性を発見できることがあります。

重要ポイント

グラフの種類は、「何を伝えたいか」によって選びます。比較なら棒グラフ、変化なら折れ線グラフ、関係性なら散布図といった具合です。

11.2 棒グラフの作成

棒グラフは、カテゴリ別の数値を比較するのに最適なグラフです。例えば、「各支店の売上高」や「商品別の販売数」などを表示するのに使います。

基本的な棒グラフの作成

Rではbarplot()関数を使って棒グラフを作成します。

実習 11-1: 支店別売上の棒グラフ

各支店の売上高を棒グラフで表示してみましょう。

barplot_example.R 
# 支店別の売上データ
sales <- c(150, 230, 180, 200, 170)

# 支店名
branches <- c("東京", "大阪", "名古屋", "福岡", "札幌")

# 棒グラフを作成
barplot(sales,
        names.arg = branches,  # 横軸のラベル
        main = "支店別売上高",  # グラフタイトル
        xlab = "支店",          # 横軸ラベル
        ylab = "売上（万円）",  # 縦軸ラベル
        col = "skyblue")        # 棒の色

barplot()関数のパラメータ

names.arg 横軸に表示する名前（ラベル）
main グラフのタイトル
xlab 横軸のラベル
ylab 縦軸のラベル
col 棒の色（"red", "blue", "green"など）

棒グラフのカスタマイズ

各棒に異なる色を付けることもできます。

colorful_barplot.R 
# 各棒に異なる色を設定
colors <- c("red", "blue", "green", "orange", "purple")

barplot(sales,
        names.arg = branches,
        main = "支店別売上高（カラフル版）",
        xlab = "支店",
        ylab = "売上（万円）",
        col = colors)  # 色のベクトルを指定

11.3 折れ線グラフの作成

折れ線グラフは、時間的な変化や推移を表示するのに最適です。例えば、「月別の気温変化」や「年別の売上推移」を表示するのに使います。

基本的な折れ線グラフの作成

Rではplot()関数にtype = "l"を指定して折れ線グラフを作成します。

実習 11-2: 月別売上の折れ線グラフ

1月から12月までの売上推移を折れ線グラフで表示してみましょう。

lineplot_example.R 
# 月別売上データ
monthly_sales <- c(120, 130, 145, 160, 150, 170,
                   180, 175, 165, 155, 140, 200)

# 月のラベル
months <- 1:12

# 折れ線グラフを作成
plot(months, monthly_sales,
     type = "l",                  # 折れ線グラフ（line）
     main = "月別売上推移",       # タイトル
     xlab = "月",                 # 横軸ラベル
     ylab = "売上（万円）",       # 縦軸ラベル
     col = "blue",                # 線の色
     lwd = 2)                     # 線の太さ

点とラインの両方を表示

折れ線グラフに点も表示すると、各データポイントが見やすくなります。

line_with_points.R 
# 折れ線グラフと点の両方を表示
plot(months, monthly_sales,
     type = "b",              # 両方（both）を表示
     main = "月別売上推移",
     xlab = "月",
     ylab = "売上（万円）",
     col = "darkgreen",
     lwd = 2,
     pch = 19)                # 点の形状（19は塗りつぶし円）

typeパラメータの種類

type = "l" 線のみ（line）
type = "p" 点のみ（points）
type = "b" 両方（both）
type = "o" 点の上に線を重ねる（overplotted）

11.4 散布図の作成

散布図は、2つの変数の関係性を探るのに使います。例えば、「勉強時間とテスト点数の関係」や「気温とアイス販売数の関係」などを調べるのに適しています。

基本的な散布図の作成

散布図もplot()関数で作成できます（デフォルトが散布図です）。

実習 11-3: 勉強時間と点数の散布図

勉強時間とテスト点数の関係を散布図で調べてみましょう。

scatter_example.R 
# 勉強時間（時間）
study_time <- c(2, 3, 4, 5, 3, 6, 7, 4, 5, 8)

# テスト点数（点）
test_score <- c(60, 65, 70, 75, 68, 80, 85, 72, 78, 90)

# 散布図を作成
plot(study_time, test_score,
     main = "勉強時間とテスト点数の関係",
     xlab = "勉強時間（時間）",
     ylab = "テスト点数（点）",
     col = "red",
     pch = 19)  # 塗りつぶし円

散布図から読み取れること

この例では、右上がりの傾向が見えます。つまり、勉強時間が長いほどテスト点数が高いという関係（正の相関）があることがわかります。

11.5 グラフのカスタマイズ

グラフをより見やすく、美しくするためのカスタマイズ方法を学びましょう。

複数のグラフを並べて表示

par()関数を使うと、複数のグラフを並べて表示できます。

multiple_plots.R 
# 2行2列でグラフを配置
par(mfrow = c(2, 2))

# グラフ1: 棒グラフ
barplot(sales, names.arg = branches, main = "棒グラフ")

# グラフ2: 折れ線グラフ
plot(months, monthly_sales, type = "l", main = "折れ線グラフ")

# グラフ3: 散布図
plot(study_time, test_score, main = "散布図")

# グラフ4: ヒストグラム
hist(test_score, main = "ヒストグラム")

# 設定を元に戻す
par(mfrow = c(1, 1))

グラフの色と記号のカスタマイズ

よく使う色の指定方法

色名で指定:

col = "red", col = "blue", col = "green" など

カラーパレットを使う:

rainbow(5) - 虹色の5色, heat.colors(5) - 暖色系の5色

よく使う点の形状（pch）

pch = 1 白抜きの円
pch = 19 塗りつぶしの円
pch = 15 塗りつぶしの四角
pch = 17 塗りつぶしの三角

グラフをファイルに保存

作成したグラフを画像ファイルとして保存できます。

save_graph.R 
# PNG画像として保存
png("sales_graph.png", width = 800, height = 600)

# グラフを作成
barplot(sales,
        names.arg = branches,
        main = "支店別売上高",
        col = rainbow(5))

# ファイルを閉じる（これで保存完了）
dev.off()

# 保存完了メッセージ
print("グラフをsales_graph.pngとして保存しました")

ファイル形式

png()以外にも、以下の形式で保存できます：

  • • pdf() - PDFファイル（レポートに最適）
  • • jpeg() - JPEG画像
  • • svg() - SVGベクター画像（拡大しても綺麗）

理解度確認クイズ

  1. Q1. 時間的な変化を表示するのに最適なグラフは何ですか？

    回答を表示

    回答：折れ線グラフ

    折れ線グラフは時系列データ（時間の経過に伴う変化）を表示するのに最適です。月別の売上推移や年間の気温変化などを視覚化するのに使います。

  2. Q2. 散布図から何がわかりますか？

    回答を表示

    回答：2つの変数の関係性（相関）

    散布図を見ることで、2つの変数に正の相関（一方が増えるともう一方も増える）、負の相関（一方が増えるともう一方は減る）、または相関なし（関係性がない）のいずれかがあるかを視覚的に判断できます。

  3. Q3. barplot()関数で棒の色を変更するパラメータは何ですか？

    回答を表示

    回答：col（color）

    col = "red"のように指定します。色名（"red", "blue"など）や、複数の色をベクトルで指定することもできます。

第11章のまとめ

  • グラフを使うと、データの傾向やパターンが視覚的に理解しやすくなります
  • 棒グラフはbarplot()関数で作成し、カテゴリ別の量を比較するのに使います
  • 折れ線グラフはplot(type = "l")で作成し、時間的な変化を表示します
  • 散布図はplot()関数で作成し、2つの変数の関係性を探ります
  • 色、サイズ、形状などをカスタマイズして、より見やすいグラフを作成できます
  • グラフはpng()やpdf()関数を使ってファイルに保存できます

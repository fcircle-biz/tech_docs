変数とオブジェクト - データに名前をつけて保存する

計算結果やデータに名前をつけて保存し、後から使い回す方法を学びます。プログラミングの基本となる重要な概念です。

この章で学ぶこと

  • 変数とは何かを理解する
  • 変数に値を代入する方法を学ぶ
  • 変数を使って計算する方法を理解する
  • オブジェクトの概念を知る
  • 変数の命名規則を学ぶ

5.1 変数とは何か

変数（へんすう）とは、データに名前をつけて保存する入れ物です。

たとえば、日常生活で「タンスの引き出しにラベルを貼って、服を整理する」ことを想像してください。「Tシャツ」「靴下」などのラベルを貼ると、どこに何があるかすぐにわかります。

変数も同じです。計算結果や値に名前をつけておくと、後から簡単に使えます。

変数のイメージ

flowchart LR A["計算結果: 100"] -->|名前をつける| B["変数名: kekka"] B -->|後から呼び出せる| C["100を使って計算"] style A fill:#fef3c7,stroke:#f59e0b,stroke-width:2px style B fill:#dbeafe,stroke:#3b82f6,stroke-width:2px style C fill:#d1fae5,stroke:#10b981,stroke-width:2px

変数を使う理由

  • 計算結果を保存：一度計算した結果を何度も使える
  • わかりやすくなる：数字だけより、意味のある名前の方が理解しやすい
  • 修正が簡単：値を変えたいとき、一箇所変えるだけでよい

5.2 変数への代入

変数に値を保存することを「代入（だいにゅう）」と言います。R言語では、代入演算子<-または=を使います。

基本的な代入の書き方

基本的な代入 
# 変数に値を代入する
x <- 10  # xという変数に10を代入

# 変数の中身を表示
x  # 10と表示される

代入演算子の読み方

x <- 10は、「xに10を代入する」または「xには10を入れる」と読みます。

矢印<-の向きに注意してください。値（10）が変数（x）の方向に流れ込むイメージです。

実習：変数を使ってみよう

ハンズオン：変数への代入

実際にRStudioで変数を使ってみましょう。

変数への代入の例 
# 年齢を変数に代入
nenrei <- 25

# 身長を変数に代入（cm）
shincho <- 170

# 変数の中身を確認
nenrei    # 25と表示
shincho   # 170と表示

5.3 変数を使った計算

変数に保存した値を使って、計算ができます。また、計算結果を別の変数に保存することもできます。

変数を使った計算 
# 価格を変数に代入
kakaku <- 1000

# 税率を変数に代入
zeiritsu <- 0.1

# 税込価格を計算
zeikomi <- kakaku * (1 + zeiritsu)

# 結果を表示
zeikomi  # 1100と表示される

変数を使うメリット

上の例で、もし税率が変わったら？zeiritsuの値を1箇所変えるだけで、すべての計算に反映されます。 

# 税率が変わった場合
zeiritsu <- 0.08  # ここを変えるだけ！

# 計算式は変更不要
zeikomi <- kakaku * (1 + zeiritsu)

変数の値を更新する

変数の値は、後から上書きすることができます。

変数の更新 
# 最初に値を代入
kazu <- 5
kazu  # 5と表示

# 値を上書き
kazu <- 10
kazu  # 10と表示（前の値5は消えている）

# 自分自身を使って更新
kazu <- kazu + 3  # kazuに3を足して、再びkazuに代入
kazu  # 13と表示

5.4 オブジェクトとは

R言語では、変数に保存されるデータを「オブジェクト」と呼びます。

オブジェクトには、いろいろな種類があります：

数値オブジェクト

数値を保存するオブジェクト 

x <- 10
y <- 3.14

文字オブジェクト

文字列を保存するオブジェクト 

namae <- "田中"
aisatsu <- "こんにちは"

論理値オブジェクト

TRUE（真）またはFALSE（偽）を保存 

seikou <- TRUE
shippai <- FALSE

ベクトルオブジェクト

複数の値をまとめて保存（次章で学習） 

suuji <- c(1, 2, 3, 4, 5)

変数とオブジェクトの関係

変数は「名前」、オブジェクトは「中身」と考えるとわかりやすいです。
「xという名前の入れ物に、10という数値オブジェクトを入れる」というイメージです。

5.5 変数の命名規則

変数名をつけるときには、いくつかのルールと推奨事項があります。

必ず守るルール（文法）

数字で始めてはいけない

1kazu <- 10  # エラー！
kazu1 <- 10  # OK

スペースや特殊記号は使えない

my data <- 10  # エラー！
my-data <- 10  # エラー！（マイナス記号と誤認）
my_data <- 10  # OK（アンダースコアは使える）

予約語は使えない

R言語が特別な意味で使っている単語（TRUE、FALSE、if、forなど）は変数名に使えません。 

TRUE <- 10   # エラー！
if <- 10     # エラー！

推奨される命名方法

わかりやすい変数名をつけよう

良い例：意味がわかる名前

uriage <- 100000    # 売上
kosuu <- 50         # 個数
heikin <- 75.5      # 平均

悪い例：意味不明な名前

a <- 100000
x <- 50
zzz <- 75.5

大文字・小文字の区別

R言語では、大文字と小文字は別の文字として扱われます。 

kazu <- 10
Kazu <- 20
KAZU <- 30

# これらは全て別の変数として扱われる

5.6 よくあるエラーと対処法

トラブルシューティング

エラー: object 'xxx' not found

原因：変数が存在しない、または変数名のスペルミス 

kazu <- 10
kaz  # エラー！（スペルミス）

対処法：変数名のスペルを確認する。大文字・小文字も正確に。

エラー: unexpected symbol

原因：変数名にスペースや禁止文字が含まれている 

my data <- 10  # エラー！

対処法：スペースをアンダースコアに変える（my_data）

代入したのに何も表示されない

これはエラーではありません：Rでは代入だけでは結果が表示されません 

x <- 10  # 何も表示されない（これは正常）

# 中身を見るには変数名を入力
x  # 10と表示される

ハンズオン：変数を使った計算

以下の課題に挑戦してみましょう。

課題1: 基本的な代入

あなたの年齢をnenreiという変数に代入し、表示してください。

課題2: 計算結果の保存

商品の価格が800円、個数が5個のとき、合計金額を計算してgoukeiという変数に保存してください。

課題3: 変数を使った計算

円周率（3.14）をpiに、半径（5）をhankeiに代入し、円の面積（半径×半径×円周率）を計算してmensekiに保存してください。

解答例を表示 
# 課題1
nenrei <- 25  # あなたの年齢を入力
nenrei

# 課題2
kakaku <- 800
kosuu <- 5
goukei <- kakaku * kosuu
goukei  # 4000と表示

# 課題3
pi <- 3.14
hankei <- 5
menseki <- hankei * hankei * pi

理解度確認クイズ

  1. Q1. 変数とは何ですか？

    回答を表示

    回答：データに名前をつけて保存する入れ物

    変数を使うことで、計算結果や値を保存して、後から何度も使うことができます。変数名をつけることで、コードもわかりやすくなります。

  2. Q2. x <- 10という命令の意味を説明してください

    回答を表示

    回答：「xという変数に10を代入する」

    代入演算子<-を使って、右側の値（10）を左側の変数（x）に保存します。これ以降、xと書くだけで10を呼び出せます。

  3. Q3. 変数名をつけるときに守るべきルールを3つ挙げてください

    回答を表示

    回答：
    1. 数字で始めてはいけない（1kazuは×、kazu1は○）
    2. スペースや特殊記号は使えない（my dataは×、my_dataは○）
    3. 予約語は使えない（TRUE、if、forなどは×）

第5章のまとめ

  • 変数は、データに名前をつけて保存する入れ物です
  • 代入演算子<-または=を使って、変数に値を保存します
  • 変数を使うことで、計算結果を保存して何度も使えます
  • オブジェクトは、変数に保存されるデータのことです（数値、文字、論理値など）
  • 変数名は、数字で始めない、スペースを使わない、予約語を避けるなどのルールがあります
  • わかりやすい変数名をつけることで、コードが読みやすくなります

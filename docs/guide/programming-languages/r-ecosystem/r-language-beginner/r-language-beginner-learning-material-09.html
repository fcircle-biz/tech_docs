繰り返し処理 - 同じ作業を自動化する

同じ処理を何度も繰り返す方法を学びます。手作業では面倒な処理を一瞬で行えるようになります。

この章で学ぶこと

  • 繰り返し処理とは何かを理解する
  • for文を使った繰り返し処理の書き方
  • while文を使った条件付き繰り返し
  • ベクトルに対する繰り返し処理の実践

9.1 繰り返し処理とは何か

繰り返し処理（ループ）とは、同じ処理を何度も繰り返し実行することです。プログラミングの中で最も重要な概念の一つです。

例えば、次のような場面で繰り返し処理が役立ちます：

  • 1から100までの数字を順番に表示する
  • ベクトルの各要素に同じ計算を適用する
  • 複数のファイルを順番に処理する
  • 条件を満たすまで計算を繰り返す

なぜ繰り返し処理が重要なのか？

繰り返し処理を使わないと、同じコードを何度も書く必要があります。例えば、1から100までの数字を表示するのに100行のコードを書くのは非効率です。繰り返し処理を使えば、わずか数行で実現できます。これがプログラミングによる自動化の本質です。

9.2 for文の基本

for文は、決まった回数だけ処理を繰り返すときに使います。R言語で最もよく使われる繰り返し処理の方法です。

for文の基本構文

R 
# for文の基本形
for (変数 in ベクトル) {
  # ここに繰り返し実行する処理を書く
}

基本例：1から5まで表示

R 
# 1から5までの数字を順番に表示
for (i in 1:5) {
  print(i)
}

# 実行結果：
# [1] 1
# [1] 2
# [1] 3
# [1] 4
# [1] 5

コードの読み方

  • for (i in 1:5) は「iという変数に1から5までの値を順番に代入して繰り返す」という意味です
  • 1:5 は c(1, 2, 3, 4, 5) と同じです
  • 中括弧 {} の中に繰り返したい処理を書きます

ベクトルの要素を順番に処理

R 
# 果物の名前を順番に表示
fruits <- c("りんご", "バナナ", "オレンジ")

for (fruit in fruits) {
  print(fruit)
}

# 実行結果：
# [1] "りんご"
# [1] "バナナ"
# [1] "オレンジ"

実習：for文を使ってみよう

以下のコードをRStudioで実行して、for文の動きを確認しましょう。

R 
# 1から10までの数字を表示
for (i in 1:10) {
  print(i)
}

# 各数字の2倍を計算して表示
for (i in 1:5) {
  result <- i * 2
  print(result)
}

9.3 for文での計算

for文の中で計算を行い、結果を蓄積することができます。

例1：合計を計算

R 
# 1から10までの合計を計算
total <- 0  # 合計を格納する変数を初期化

for (i in 1:10) {
  total <- total + i  # 現在の合計にiを加える
}

print(total)  # 結果：55

例2：ベクトルの各要素を2倍にする

R 
# 元のベクトル
numbers <- c(2, 4, 6, 8, 10)
doubled <- c()  # 結果を格納する空のベクトル

# 各要素を2倍にして新しいベクトルに追加
for (num in numbers) {
  doubled <- c(doubled, num * 2)
}

print(doubled)
# 結果：4 8 12 16 20

9.4 while文 - 条件付き繰り返し

while文は、条件が真（TRUE）である限り処理を繰り返すときに使います。

while文の基本構文

R 
# while文の基本形
while (条件) {
  # 条件が真の間、繰り返し実行する処理
}

基本例：カウントダウン

R 
# 5から1までカウントダウン
count <- 5

while (count > 0) {
  print(count)
  count <- count - 1  # カウントを1減らす
}

print("発射！")

# 実行結果：
# [1] 5
# [1] 4
# [1] 3
# [1] 2
# [1] 1
# [1] "発射！"

無限ループに注意！

while文では、条件が永遠に真のままになってしまう「無限ループ」に注意が必要です。必ず条件が偽（FALSE）になるように設計しましょう。

悪い例（無限ループ）：

# これは永遠に終わらない！
count <- 5
while (count > 0) {
  print(count)
  # countを減らしていないので永遠に5 > 0のまま
}

for文とwhile文の使い分け

for文

繰り返す回数が決まっているときに使う

例：1から100まで、ベクトルの全要素に対して

while文

条件を満たす間繰り返すときに使う

例：合計が100を超えるまで、誤差が0.001以下になるまで

9.5 ネストされた繰り返し

for文やwhile文の中に、さらに別の繰り返し処理を入れることができます。これをネスト（入れ子）と言います。

例：九九の表を作成

R 
# 九九の表を作成
for (i in 1:3) {  # 外側のループ
  for (j in 1:3) {  # 内側のループ
    result <- i * j
    print(paste(i, "×", j, "=", result))
  }
}

# 実行結果：
# [1] "1 × 1 = 1"
# [1] "1 × 2 = 2"
# [1] "1 × 3 = 3"
# [1] "2 × 1 = 2"
# [1] "2 × 2 = 4"
# [1] "2 × 3 = 6"
# [1] "3 × 1 = 3"
# [1] "3 × 2 = 6"
# [1] "3 × 3 = 9"

ネストの仕組み

外側のループが1回実行されるごとに、内側のループが全て実行されます。上の例では、iが1のとき、jが1、2、3と変化し、次にiが2のとき、またjが1、2、3と変化します。

実習：繰り返し処理の練習

以下の問題に挑戦してみましょう。

  1. 問題1：1から20までの偶数だけを表示するfor文を書いてください。
  2. 問題2：ベクトル c(5, 10, 15, 20, 25) の各要素を3倍にして表示してください。
  3. 問題3：while文を使って、1から始めて2倍ずつ増やしていき、100を超えたら終了するプログラムを書いてください。

理解度確認クイズ

  1. Q1. for文とwhile文の違いは何ですか？

    回答を表示

    回答：
    for文：繰り返す回数が決まっているときに使います。ベクトルの要素数や特定の範囲など、事前に回数がわかっている場合に適しています。

    while文：条件が真である限り繰り返すときに使います。繰り返す回数が事前にわからず、ある条件を満たすまで処理を続ける場合に適しています。

  2. Q2. 次のコードの実行結果は何ですか？

    sum <- 0
for (i in c(2, 4, 6)) {
  sum <- sum + i
}
print(sum)
    回答を表示

    回答：12

    解説：sumは0から始まり、2、4、6を順番に加算します。0 + 2 = 2、2 + 4 = 6、6 + 6 = 12となります。

  3. Q3. while文で注意すべきことは何ですか？

    回答を表示

    回答：無限ループに注意する

    while文では、条件が永遠に真のままだと処理が終わらない「無限ループ」になってしまいます。必ず、繰り返しの中で条件を偽にする処理（カウンタを増やす、減らすなど）を含めるようにしましょう。

よくあるエラーと対処法

エラー：unexpected symbol

原因：中括弧 {} が正しく閉じられていない

対処法：for文やwhile文の開始の { に対して、必ず閉じる } があるか確認しましょう。

問題：プログラムが止まらない

原因：無限ループになっている

対処法：RStudioで Esc キーを押すか、コンソールの「停止」ボタンをクリックして処理を中断します。while文の条件を見直して、必ず偽になるように修正しましょう。

第9章のまとめ

  • 繰り返し処理（ループ）は、同じ処理を何度も繰り返し実行する仕組みです
  • for文は、繰り返す回数が決まっているときに使います（例：ベクトルの全要素に対して）
  • while文は、条件が真である限り繰り返すときに使います（無限ループに注意）
  • 繰り返し処理の中に別の繰り返し処理を入れるネスト（入れ子）も可能です
  • 繰り返し処理を使うことで、手作業では面倒な大量の処理を一瞬で自動化できます

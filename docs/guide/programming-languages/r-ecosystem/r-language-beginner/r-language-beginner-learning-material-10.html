関数の使い方 - 便利な機能を活用する

Rに用意されている便利な機能「関数」の使い方を学びます。平均、合計など、よく使う関数を紹介します。

この章で学ぶこと

  • 関数とは何かを理解する
  • 関数の基本的な使い方を学ぶ
  • よく使う基本関数を知る
  • ヘルプ機能の使い方を覚える

10.1 関数とは何か

関数とは、特定の処理をまとめて、何度も使えるようにした便利な道具です。

たとえば、料理で「野菜を切る」という作業を考えてみましょう。包丁という道具を使えば、簡単に野菜を切ることができます。プログラミングの関数も同じように、複雑な処理を簡単に実行できる道具なのです。

関数のイメージ

flowchart LR A[データを入れる
入力] -->|関数に渡す| B{関数
処理} B -->|計算や処理| C[結果を返す

関数のメリット

  • コードが短くなる：複雑な処理を1行で実行できる
  • 読みやすくなる：何をしているか分かりやすい
  • 間違いが減る：テスト済みの処理を使える
  • 効率的：同じ処理を何度も書かなくて済む

10.2 関数の基本的な使い方

関数を使うには、関数名(引数)という形式で書きます。

関数の基本構造

R 
# 基本的な形式
# 結果 <- 関数名(引数1, 引数2, ...)

# 例: sum関数で合計を計算
result <- sum(1, 2, 3, 4, 5)
print(result)  # 15が表示される
関数名： 実行したい処理の名前（例：sum, mean, maxなど）
引数： 関数に渡すデータ（カッコの中に書く）
戻り値： 関数が返す結果（変数に保存できる）

実習：基本的な関数を使ってみよう

R 
# 数値のベクトルを作成
numbers <- c(10, 20, 30, 40, 50)

# 合計を計算
total <- sum(numbers)
print(total)  # 150

# 平均を計算
average <- mean(numbers)
print(average)  # 30

# 最大値を計算
maximum <- max(numbers)
print(maximum)  # 50

10.3 よく使う数学関数

Rには、データ分析でよく使う数学関数がたくさん用意されています。

sum() - 合計を計算

数値の合計値を計算します。

R 
sum(1, 2, 3, 4, 5)  # 15
sum(c(10, 20, 30))  # 60

mean() - 平均を計算

数値の平均値を計算します。

R 
mean(c(10, 20, 30))  # 20
mean(c(5, 10, 15, 20))  # 12.5

median() - 中央値を計算

データを並べたときの真ん中の値を返します。

R 
median(c(1, 2, 3, 4, 5))  # 3
median(c(10, 20, 30, 40))  # 25（2つの真ん中の平均）

max() / min() - 最大値・最小値

データの中で最も大きい値、最も小さい値を返します。

R 
max(c(10, 50, 30, 20))  # 50
min(c(10, 50, 30, 20))  # 10

round() - 四捨五入

小数を指定した桁数で四捨五入します。

R 
round(3.14159)      # 3（整数に四捨五入）
round(3.14159, 2)   # 3.14（小数第2位まで）
round(3.14159, 4)   # 3.1416（小数第4位まで）

sqrt() / abs() - 平方根・絶対値

平方根（ルート）と絶対値を計算します。

R 
sqrt(16)   # 4（16の平方根）
sqrt(100)  # 10

abs(-5)    # 5（-5の絶対値）
abs(3)     # 3

10.4 よく使う文字列関数

文字列（テキスト）を操作する関数も便利です。

nchar() - 文字数を数える

文字列の文字数を返します。

R 
nchar("こんにちは")  # 5
nchar("R言語")       # 3

toupper() / tolower() - 大文字・小文字変換

英字を大文字または小文字に変換します。

R 
toupper("hello")  # "HELLO"
tolower("WORLD")  # "world"

paste() - 文字列を結合

複数の文字列をつなげて1つにします。

R 
paste("こんにちは", "世界")  # "こんにちは 世界"
paste("R", "言語", sep = "")  # "R言語"（区切りなし）
paste("第", 1, "章", sep = "")  # "第1章"

substr() - 文字列の一部を取り出す

文字列の指定した位置から文字を抽出します。

R 
text <- "R言語入門"
substr(text, 1, 3)  # "R言語"（1文字目から3文字目まで）
substr(text, 4, 5)  # "入門"（4文字目から5文字目まで）

10.5 データ確認関数

データの内容や構造を確認する関数も重要です。

length() - 要素数を確認

ベクトルの要素数を返します。

R 
vec <- c(10, 20, 30, 40, 50)
length(vec)  # 5（要素が5個）

head() / tail() - 最初・最後のデータを表示

データの最初の部分または最後の部分を表示します。

R 
numbers <- 1:100  # 1から100までの数値

head(numbers)     # 最初の6個を表示
head(numbers, 3)  # 最初の3個を表示

tail(numbers)     # 最後の6個を表示
tail(numbers, 3)  # 最後の3個を表示

str() - データ構造を確認

データの型や構造を詳しく表示します。

R 
vec <- c(10, 20, 30)
str(vec)
# num [1:3] 10 20 30（数値型ベクトル、3要素）

summary() - 統計サマリーを表示

データの基本統計量（最小値、最大値、平均など）を表示します。

R 
numbers <- c(10, 20, 30, 40, 50)
summary(numbers)
# Min. 1st Qu. Median  Mean 3rd Qu.  Max.
#   10      20     30    30      40    50

10.6 ヘルプ機能の使い方

Rには、関数の使い方を調べるためのヘルプ機能が備わっています。

ヘルプの3つの方法

R 
# 方法1: help関数を使う
help(mean)

# 方法2: ?マークを使う（より簡単）
?mean

# 方法3: example関数で使用例を見る
example(mean)

ヘルプを見ると、関数の説明、引数の意味、使用例などが表示されます。分からない関数があったら、まずヘルプを見てみましょう！

実習：ヘルプ機能を使ってみよう

R 
# 1. medianのヘルプを見る
?median

# 2. sumの使用例を見る
example(sum)

# 3. roundのヘルプを見て、使い方を確認
?round
# → digitsという引数で小数点以下の桁数を指定できることが分かる

ヘルプページは英語ですが、使用例（Examples）を見るだけでも十分参考になります。

10.7 関数を使う際の注意点

注意点1: NA（欠損値）の扱い

データにNA（欠損値）が含まれていると、多くの関数は結果もNAを返します。

R 
# NAが含まれるデータ
data <- c(10, 20, NA, 40, 50)

mean(data)  # NA（結果もNAになる）

# 解決策: na.rm = TRUE でNAを除外
mean(data, na.rm = TRUE)  # 30（10+20+40+50の平均）

注意点2: 引数の順序と名前

引数は順序通りに書くか、名前を指定して書くことができます。

R 
# 順序通りに書く方法
round(3.14159, 2)  # 3.14

# 名前を指定する方法（分かりやすい）
round(x = 3.14159, digits = 2)  # 3.14

# 名前を指定すれば順序は自由
round(digits = 2, x = 3.14159)  # 3.14

Rの便利な機能: ベクトル化

多くの関数は、ベクトル全体に対して一度に処理を実行できます（ベクトル化）。

R 
# ベクトルの各要素に対して処理が実行される
numbers <- c(1.1, 2.7, 3.3, 4.9)
round(numbers)  # c(1, 3, 3, 5)

# 平方根も同様
sqrt(c(4, 9, 16, 25))  # c(2, 3, 4, 5)

for文を使わなくても、ベクトル全体に処理を適用できるのがRの強みです！

実践課題：関数を組み合わせて使おう

以下のテストの点数データについて、各種統計値を計算してみましょう。

R 
# テストの点数データ
scores <- c(85, 92, 78, 95, 88, 76, 89, 91, 84, 90)

# 課題1: 平均点を計算
avg_score <- mean(scores)
print(paste("平均点:", avg_score))

# 課題2: 最高点と最低点を表示
print(paste("最高点:", max(scores)))
print(paste("最低点:", min(scores)))

# 課題3: 中央値を計算
median_score <- median(scores)
print(paste("中央値:", median_score))

# 課題4: 平均点を小数第1位で四捨五入
rounded_avg <- round(avg_score, 1)
print(paste("平均点（小数第1位）:", rounded_avg))

# 課題5: データの要素数を確認
print(paste("データ数:", length(scores)))

# 課題6: summary関数で全体像を把握
summary(scores)

理解度確認クイズ

  1. Q1. 関数とは何ですか？

    回答を表示

    回答：特定の処理をまとめて、何度も使えるようにした便利な道具

    関数を使うことで、複雑な処理を簡単に実行でき、コードが短く読みやすくなります。Rには数多くの便利な関数が用意されています。

  2. Q2. 次のコードの出力は何ですか？ mean(c(10, 20, 30, 40))

    回答を表示

    回答：25

    mean()は平均を計算する関数です。(10 + 20 + 30 + 40) ÷ 4 = 100 ÷ 4 = 25となります。

  3. Q3. データにNA（欠損値）が含まれている場合、mean()関数で正しく平均を計算するにはどうすればよいですか？

    回答を表示

    回答：na.rm = TRUE を指定する

    例: mean(data, na.rm = TRUE)
    na.rm（NA remove）をTRUEにすることで、NAを除外して平均を計算できます。

第10章のまとめ

  • 関数は、特定の処理をまとめた便利な道具で、複雑な処理を簡単に実行できます
  • 関数の基本形式は関数名(引数)で、引数を渡すと結果（戻り値）が返されます
  • 数学関数（sum, mean, median, max, min, round, sqrt, absなど）で基本的な計算ができます
  • 文字列関数（nchar, toupper, tolower, paste, substrなど）で文字列を操作できます
  • データ確認関数（length, head, tail, str, summaryなど）でデータの概要を把握できます
  • ?関数名やhelp()でヘルプを見て、関数の使い方を調べることができます
  • NAが含まれるデータではna.rm = TRUEを指定する必要があります

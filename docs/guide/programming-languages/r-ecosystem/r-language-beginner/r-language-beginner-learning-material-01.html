1.1 プログラミングとは何か

プログラミングとは、コンピュータに指示を出して、自分のやりたいことを自動でやってもらうことです。

たとえば、次のようなことができます：

  • 大量のデータを一瞬で計算する
  • データをグラフにして見やすく表示する
  • 毎日同じ作業を自動で繰り返す
  • データの中から必要な情報を探し出す

プログラミングのイメージ

プログラミングを学ぶと、手作業で1時間かかる作業を数秒で終わらせることができるようになります。これは、あなたの時間を大幅に節約し、より価値のある仕事に集中できることを意味します。

1.2 R言語とは

R言語は、データ分析と統計処理に特化したプログラミング言語です。世界中の研究者、データサイエンティスト、企業のデータ分析担当者が使っています。

R言語の歴史

1993年 ニュージーランドのオークランド大学でロス・イハカとロバート・ジェントルマンが開発を開始
2000年 最初の安定版がリリースされ、世界中で使われるようになる
現在 データサイエンス分野で最も人気のある言語の一つとして広く使われている

R言語の3つの特徴

無料で使える

誰でも無料でダウンロードして使えます。高価なソフトウェアを買う必要がありません。

データ分析に強い

統計分析やグラフ作成が簡単にできます。専門的な計算も少ないコードで実現できます。

豊富な拡張機能

世界中の人が作った便利な機能（パッケージ）を使って、簡単に高度な分析ができます。

1.3 R言語でできること

R言語を使うと、以下のようなことができます：

データの読み込みと整理

ExcelやCSVファイルのデータを読み込んで、必要な部分だけを取り出したり、並び替えたりできます。

例：「東京支店の2023年の売上データだけを抽出」

統計的な計算

平均、合計、最大値、最小値などの基本的な計算から、相関分析や回帰分析などの高度な統計計算まで簡単にできます。

例：「過去5年間の売上の平均値を計算」

美しいグラフ作成

棒グラフ、折れ線グラフ、散布図など、さまざまな種類のグラフを作成できます。色やデザインも自由にカスタマイズできます。

例：「月別売上の推移を折れ線グラフで表示」

機械学習とAI

データから傾向を見つけたり、将来の予測をしたりする機械学習の手法も使えます。

例：「過去のデータから来月の売上を予測」

1.4 R言語が使われている場面

R言語は、さまざまな業界や分野で実際に使われています：

ビジネス分析

売上データの分析、顧客の購買傾向の把握、マーケティング効果の測定など

医学・生物学研究

臨床試験のデータ分析、遺伝子データの解析、疫学研究など

金融・経済

株価予測、リスク分析、経済指標の分析、投資戦略の開発など

学術研究

心理学、社会学、教育学など、さまざまな分野でのデータ分析と統計処理

データの種類を学ぼう - 数値・文字・論理値

Rで扱うデータには種類があります。数値、文字、真偽（TRUE/FALSE）の違いと使い分けを理解します。

この章で学ぶこと

  • データ型の基本概念を理解する
  • 数値型・文字型・論理型の違いを理解する
  • データ型の確認方法を習得する
  • データ型の変換方法を学ぶ

4.1 データ型とは何か

プログラミングでは、データに「種類」があります。これをデータ型（データタイプ）と呼びます。

データ型とは、データがどんな性質を持っているかを表すものです。数字なのか、文字なのか、真偽（正しい/間違い）なのかを区別します。

データ型の分類

なぜデータ型が必要なのか？

コンピュータは、データの種類によって処理方法を変えます。たとえば、数値なら計算ができますが、文字では計算できません。データ型を正しく理解することで、エラーを減らし、思い通りのプログラムを書けるようになります。

4.2 数値型（numeric）

数値型は、数字を扱うためのデータ型です。計算に使える数値を表します。

数値型の特徴

  • 整数：小数点のない数（例：10, -5, 0）
  • 小数：小数点を含む数（例：3.14, -0.5, 100.0）
  • 計算可能：足し算、引き算、掛け算、割り算ができる
R 
# 数値型の例
x <- 100        # 整数
y <- 3.14       # 小数
z <- -5         # 負の数

# 計算ができる
x + y           # 103.14
x * 2           # 200
y / 2           # 1.57

整数と小数の違い

R言語では、整数も小数も同じ「数値型（numeric）」として扱われます。特別に指定しない限り、すべての数値は小数として保存されます。

R 
# 100と入力しても、内部的には100.0として保存される
x <- 100
class(x)  # "numeric"と表示される

4.3 文字型（character）

文字型は、文字や文章を扱うためのデータ型です。名前、住所、メッセージなど、テキスト情報を保存します。

文字型の特徴

  • クォーテーション：ダブルクォート（"）またはシングルクォート（'）で囲む
  • 計算不可：数字に見えても、クォートで囲むと文字として扱われる
  • 結合可能：文字同士を繋げることができる
R 
# 文字型の例
name <- "田中太郎"       # ダブルクォートで囲む
city <- '東京'           # シングルクォートでも可
message <- "こんにちは"

# 数字も文字として扱える
zip_code <- "100-0001"  # 郵便番号（文字型）

# 文字の結合
paste(name, "さん", sep = "")  # "田中太郎さん"

注意：数字と文字の違い

同じ「100」でも、数値型と文字型では全く違うものになります。

R 
# 数値型の100
num <- 100
num * 2      # 200（計算できる）

# 文字型の100
text <- "100"
text * 2     # エラー！文字は計算できない

4.4 論理型（logical）

論理型は、「正しい」または「間違い」を表すデータ型です。YesかNoか、条件に合うか合わないかを判断する時に使います。

論理型の値

TRUE（真）

条件が正しい、成立している状態を表します。「はい」「合っている」という意味です。

FALSE（偽）

条件が間違い、成立していない状態を表します。「いいえ」「違っている」という意味です。

R 
# 論理型の例
is_student <- TRUE    # 学生である
is_adult <- FALSE     # 大人ではない

# 比較演算で論理型が返される
10 > 5                # TRUE（10は5より大きい）
3 == 3                # TRUE（3は3と等しい）
5 < 2                 # FALSE（5は2より小さくない）
"東京" == "大阪"       # FALSE（東京と大阪は等しくない）

論理型の使い道

論理型は、後の章で学ぶ「条件分岐（if文）」で大活躍します。「もし条件がTRUEなら〜をする」というプログラムを書く時に必須です。

R 
# 条件分岐の例（第8章で詳しく学びます）
age <- 20
is_adult <- age >= 20  # 20歳以上ならTRUE

if (is_adult) {
  print("大人です")     # is_adultがTRUEなら実行
}

4.5 データ型の確認方法

変数に保存されているデータがどの型なのかを確認する方法を学びます。

class()関数

class()関数を使うと、データの型を確認できます。

R 
# データ型の確認
x <- 100
class(x)              # "numeric"と表示

name <- "田中"
class(name)           # "character"と表示

is_true <- TRUE
class(is_true)        # "logical"と表示

is.～()関数

特定の型かどうかを確認する関数も用意されています。結果はTRUEまたはFALSEで返されます。

R 
# 型の判定
x <- 100
is.numeric(x)     # TRUE（数値型である）
is.character(x)   # FALSE（文字型ではない）
is.logical(x)     # FALSE（論理型ではない）

name <- "田中"
is.character(name)  # TRUE（文字型である）

4.6 データ型の変換

データ型を別の型に変換する方法を学びます。これを型変換（キャスト）と呼びます。

as.～()関数

as.numeric()、as.character()、as.logical()関数を使って型変換ができます。

R 
# 文字型 → 数値型
text_num <- "123"
num <- as.numeric(text_num)  # 123（数値型）
num * 2                       # 246（計算できる）

# 数値型 → 文字型
x <- 100
text <- as.character(x)      # "100"（文字型）

# 数値型 → 論理型
as.logical(0)    # FALSE（0はFALSE）
as.logical(1)    # TRUE（0以外はTRUE）
as.logical(100)  # TRUE

変換できないケース

数値に変換できない文字列を変換しようとすると、NA（欠損値）になります。

R 
# 変換できない例
text <- "こんにちは"
as.numeric(text)  # NA（数値に変換できない）
# 警告メッセージが表示される

実習：データ型を使ってみよう

RStudioで実際にデータ型を操作してみましょう。以下のコードを入力して実行してください。

R 
# 実習1: さまざまなデータ型を作成
age <- 25                    # 数値型
name <- "山田太郎"            # 文字型
is_student <- TRUE           # 論理型

# 実習2: データ型を確認
class(age)
class(name)
class(is_student)

# 実習3: 型変換を試す
text_age <- as.character(age)  # 25 → "25"
class(text_age)

# 実習4: 比較演算で論理型を作る
result <- age >= 20          # TRUE
class(result)

理解度確認クイズ

  1. Q1. R言語の3つの基本データ型は何ですか？

    回答を表示

    回答：数値型（numeric）、文字型（character）、論理型（logical）

    それぞれ、計算に使う数値、テキスト情報、真偽値を表します。

  2. Q2. 文字型のデータを作る時、何で囲む必要がありますか？

    回答を表示

    回答：ダブルクォート（"）またはシングルクォート（'）

    例："東京"、'こんにちは'

  3. Q3. 論理型が取りうる値は何ですか？

    回答を表示

    回答：TRUEまたはFALSE

    TRUE（真）は条件が正しい状態、FALSE（偽）は条件が間違っている状態を表します。

  4. Q4. データ型を確認する関数は何ですか？

    回答を表示

    回答：class()関数

    例：class(x)でxのデータ型を確認できます。

よくあるエラーと対処法

Error: non-numeric argument to binary operator

原因：文字型のデータで計算しようとした

対処法：as.numeric()で数値型に変換してから計算する

Warning: NAs introduced by coercion

原因：数値に変換できない文字列をas.numeric()で変換しようとした

対処法：元のデータが数値として有効かを確認する

第4章のまとめ

  • R言語には数値型・文字型・論理型の3つの基本データ型があります
  • 数値型は計算に使える数値、文字型はテキスト情報、論理型は真偽値を表します
  • 文字型は必ずクォーテーション（"または'）で囲む必要があります
  • class()関数でデータ型を確認できます
  • as.numeric()、as.character()、as.logical()関数でデータ型を変換できます

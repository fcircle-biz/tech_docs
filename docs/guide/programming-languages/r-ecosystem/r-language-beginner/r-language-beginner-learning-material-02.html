第2章

RStudioのインストールと使い方

R言語を快適に使うためのツール「RStudio」のインストール方法と基本的な画面の見方を学びます。

この章で学ぶこと

  • RStudioとは何かを理解する
  • RとRStudioの違いを知る
  • RとRStudioをインストールする
  • RStudioの画面構成を理解する
  • 基本的な操作方法を習得する

2.1 RStudioとは何か

RStudioは、R言語をより快適に使うための総合開発環境（IDE）です。

総合開発環境（IDE）とは？

IDEとは「Integrated Development Environment」の略で、プログラミングに必要な機能（コード入力、実行、デバッグ、グラフ表示など）を一つにまとめたソフトウェアのことです。RStudioを使うことで、R言語をより効率的に学習・使用できます。

RStudioを使うメリット

コード入力が楽になる

入力候補の表示、自動補完、文法チェックなどの機能があり、タイプミスを防げます。

グラフがすぐ見られる

作成したグラフが画面内にすぐ表示され、確認や保存が簡単にできます。

データが見やすい

読み込んだデータを表形式で見たり、変数の一覧を確認したりできます。

ファイル管理が簡単

スクリプトファイルの保存・読み込みが簡単にでき、プロジェクト管理も可能です。

2.2 RとRStudioの違い

初心者がよく混乱するポイントですが、RとRStudioは別のソフトウェアです。

flowchart LR A[R言語
プログラミング言語本体] -->|必須| B[RStudio
R言語を使いやすくするツール] B -->|より快適に| C[効率的な学習・開発] style A fill:#dbeafe,stroke:#3b82f6,stroke-width:2px style B fill:#fef3c7,stroke:#f59e0b,stroke-width:2px style C fill:#d1fae5,stroke:#10b981,stroke-width:2px

R言語（本体）

  • プログラミング言語そのもの
  • 計算やグラフ作成などの機能を提供
  • 単体でも使えるが画面が簡素

RStudio（IDE）

  • R言語を使いやすくするツール
  • 見やすい画面、便利な機能を提供
  • R言語がインストールされていないと動かない

たとえ話

R言語を「車のエンジン」、RStudioを「快適な運転席と計器パネル」に例えると分かりやすいです。エンジン（R言語）だけでも動きますが、快適な運転席（RStudio）があると操作がずっと楽になります。

2.3 RとRStudioのインストール

RStudioを使うには、まずR言語本体をインストールし、その後RStudioをインストールする必要があります。

注意：インストール順序が重要

必ず「R言語」→「RStudio」の順にインストールしてください。逆にするとRStudioが正しく動作しません。

ステップ1: R言語のインストール

R言語のダウンロードとインストール

  1. 1 CRAN（R公式サイト）にアクセスします
  2. 2 お使いのOS（Windows/Mac/Linux）を選択します
  3. 3 Windowsの場合: 「Download R for Windows」→「base」→「Download R-X.X.X for Windows」をクリック
  4. 4 Macの場合: 「Download R for macOS」→お使いのMacのCPU（Intel/Apple Silicon）に合ったファイルをクリック
  5. 5 ダウンロードしたファイルをダブルクリックし、画面の指示に従ってインストールします

ステップ2: RStudioのインストール

RStudioのダウンロードとインストール

  1. 1 RStudio公式サイトにアクセスします
  2. 2 「Download RStudio Desktop」ボタンをクリックします
  3. 3 お使いのOSに合ったファイルが自動で選ばれるので、ダウンロードします
  4. 4 ダウンロードしたファイルをダブルクリックし、画面の指示に従ってインストールします
  5. 5 インストール完了後、RStudioを起動します

インストール成功の確認

RStudioを起動すると、以下のような画面が表示されれば成功です：

  • 複数のパネル（ペイン）に分割された画面が表示される
  • 左下のパネルに「Console」というタブがある
  • Consoleに「R version X.X.X」というメッセージが表示される

2.4 RStudioの画面構成

RStudioの画面は、4つの主要なパネル（ペイン）に分かれています。それぞれのパネルには異なる役割があります。

各パネルの詳細

1. ソースエディタ（左上）

Rのスクリプトファイル（.Rファイル）を編集するエリアです。保存して繰り返し使えるコードを書きます。

使い方：

  • コードを書いて保存できます
  • 選択した行を実行できます（Ctrl+Enter / Cmd+Enter）
  • 最初は表示されていない場合があります（File → New File → R Scriptで表示）

2. コンソール（左下）

Rのコードを実行し、結果が表示されるエリアです。ここに直接コードを入力することもできます。

使い方：

  • コードを直接入力してEnterで実行
  • 実行結果がすぐに表示されます
  • 「>」記号がある行にコードを入力します

3. 環境／履歴（右上）

作成した変数やデータの一覧、過去に実行したコマンドの履歴が表示されます。

タブの種類：

  • Environment: 現在使用中の変数やデータの一覧
  • History: 過去に実行したコマンドの履歴
  • Connections: データベース接続（上級者向け）

4. ファイル／グラフ／パッケージ／ヘルプ（右下）

様々な情報を表示する多機能エリアです。タブを切り替えて使います。

タブの種類：

  • Files: ファイルやフォルダの一覧
  • Plots: 作成したグラフが表示されます
  • Packages: インストール済みのパッケージ一覧
  • Help: 関数のヘルプドキュメント
  • Viewer: HTMLなどの表示（上級者向け）

2.5 基本的な操作方法

実際にRStudioを使う上で知っておくべき基本操作を学びましょう。

新しいスクリプトファイルを作成する

実習 2-1: 最初のスクリプトファイルを作成しよう

  1. 1 メニューバーから「File」→「New File」→「R Script」を選択
  2. 2 左上にソースエディタが表示されます
  3. 3 試しに「# 私の最初のRスクリプト」と入力してみましょう（#は コメント行です）
  4. 4 「File」→「Save」で保存（ファイル名: first_script.R）

コードを実行する

RStudioでコードを実行する方法は2つあります：

1

コンソールに直接入力

コンソール（左下）に直接コードを入力してEnterキーを押す

用途: 簡単な計算やテスト
2

スクリプトから実行

ソースエディタ（左上）でコードを書き、行を選択して Ctrl+Enter（Mac: Cmd+Enter）

用途: 複雑な処理、保存して再利用

よく使うショートカットキー

覚えておくと便利なショートカット

コード実行 Ctrl + Enter
ファイル保存 Ctrl + S
新しいスクリプト Ctrl + Shift + N
コンソールクリア Ctrl + L

Macの場合は「Ctrl」を「Cmd」に読み替えてください

画面のカスタマイズ

RStudioの画面は自分の好みに合わせてカスタマイズできます。

外観のカスタマイズ方法

「Tools」→「Global Options」→「Appearance」で、文字サイズやテーマ（配色）を変更できます。

  • Editor font size: 文字の大きさを調整
  • Editor theme: 背景色や文字色を変更（ダークモードなど）

理解度確認クイズ

  1. Q1. RとRStudioの正しい関係を説明してください

    回答を表示

    回答：Rはプログラミング言語本体で、RStudioはR言語を快適に使うための開発環境（ツール）です。

    RStudioを使うには、まずR言語をインストールする必要があります。R言語だけでも使えますが、RStudioを使うとコード入力補完、グラフ表示、ファイル管理など便利な機能が使えるようになります。

  2. Q2. RStudioの4つのパネルの役割を説明してください

    回答を表示

    回答：
    1. ソースエディタ（左上）：スクリプトファイルを編集するエリア
    2. コンソール（左下）：コードを実行し、結果を表示するエリア
    3. 環境／履歴（右上）：変数の一覧やコマンド履歴を表示
    4. ファイル／グラフ／パッケージ／ヘルプ（右下）：ファイル一覧、グラフ、ヘルプなど様々な情報を表示

  3. Q3. スクリプトエディタからコードを実行するショートカットキーは？

    回答を表示

    回答：Ctrl + Enter（Macの場合は Cmd + Enter）

    このショートカットキーを使うと、選択した行やカーソルがある行のコードがコンソールで実行されます。長いスクリプトを書く際に何度も使う非常に重要なショートカットです。

第2章のまとめ

  • RStudioは、R言語を快適に使うための統合開発環境（IDE）です
  • RとRStudioは別のソフトウェアで、インストールは「R言語」→「RStudio」の順に行います
  • RStudioの画面は4つのパネルに分かれ、それぞれ異なる役割を持っています
  • コードを実行する方法は、コンソールに直接入力する方法と、スクリプトから実行する方法があります
  • Ctrl+Enter（Cmd+Enter）でスクリプトのコードを実行できます

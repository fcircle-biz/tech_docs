第5章

関数の基礎

関数の定義と呼び出し、引数と戻り値、スコープの概念、組み込み関数の活用方法を学びます。

この章で学ぶこと

  • 関数とは何か、なぜ必要なのかを理解する
  • 関数の定義と呼び出しの方法を習得する
  • 引数と戻り値の使い方を学ぶ
  • 変数のスコープ（有効範囲）を理解する
  • PHPの便利な組み込み関数を活用する

5.1 関数とは

関数（function）とは、特定の処理をまとめて名前をつけたものです。 一度定義すれば、何度でも繰り返し使うことができます。

関数を使うと、同じコードを何度も書く必要がなくなり、プログラムが読みやすく、管理しやすくなります。

関数は「道具箱」のようなもの

料理で「みじん切りにする」という手順を毎回詳しく書くのは大変です。 「みじん切り」という名前の道具（関数）を用意しておけば、「みじん切りする」と言うだけで済みます。 プログラミングでも同じように、よく使う処理を関数にまとめておくと便利です。

関数を使うメリット

  • コードの再利用: 同じ処理を何度も書かなくて済む
  • 読みやすさ向上: 処理のまとまりが分かりやすくなる
  • 保守性向上: 修正が必要な時、1箇所直せば全体に反映される
  • テストしやすい: 個別の機能をテストできる

関数の概念図

flowchart LR A[入力: 引数] --> B[関数: 処理を実行] B --> C[出力: 戻り値]

5.2 関数の定義と呼び出し

PHPで関数を作るには、functionキーワードを使います。

基本的な関数の定義

basic_function.php 
<?php
// 関数の定義
function sayHello() {
    echo "こんにちは！
";
}

// 関数の呼び出し
sayHello();  // 出力: こんにちは！
sayHello();  // 出力: こんにちは！（何度でも呼び出せる）
?>

関数の命名規則

関数名は英数字とアンダースコア（_）が使えます。数字で始めることはできません。 分かりやすい名前をつけることが重要です（例: calculateTotal, send_email）。

実習 5-1: 簡単な挨拶関数を作る

自分の名前を表示する関数を作成しましょう

showMyName()という関数を作り、自分の名前を表示してみましょう。

practice_5_1.php 
<?php
function showMyName() {
    echo "私の名前は太郎です。
";
}

showMyName();  // 関数を呼び出す
?>

5.3 引数（パラメータ）

引数（ひきすう）とは、関数に渡す値のことです。 引数を使うと、同じ関数でも異なる結果を得ることができます。

引数を使った関数

function_with_parameter.php 
<?php
// 引数を1つ受け取る関数
function greet($name) {
    echo "こんにちは、{$name}さん！
";
}

greet("太郎");  // 出力: こんにちは、太郎さん！
greet("花子");  // 出力: こんにちは、花子さん！
?>

複数の引数を持つ関数

multiple_parameters.php 
<?php
// 複数の引数を受け取る関数
function introduce($name, $age) {
    echo "{$name}さんは{$age}歳です。
";
}

introduce("太郎", 25);  // 出力: 太郎さんは25歳です。
introduce("花子", 30);  // 出力: 花子さんは30歳です。
?>

デフォルト値（初期値）

引数にデフォルト値を設定すると、省略された場合にその値が使われます。

default_parameter.php 
<?php
// デフォルト値を持つ引数
function greetWithTime($name, $time = "おはよう") {
    echo "{$time}ございます、{$name}さん！
";
}

greetWithTime("太郎");           // 出力: おはようございます、太郎さん！
greetWithTime("花子", "こんばんは");  // 出力: こんばんはございます、花子さん！
?>

5.4 戻り値（return文）

戻り値とは、関数が処理結果として返す値のことです。 return文を使って値を返します。

戻り値を持つ関数

return_value.php 
<?php
// 2つの数値を足し算して結果を返す
function add($a, $b) {
    return $a + $b;  // 計算結果を返す
}

$result = add(5, 3);  // 関数の戻り値を変数に代入
echo "5 + 3 = {$result}
";  // 出力: 5 + 3 = 8

// 直接使うこともできる
echo "10 + 20 = " . add(10, 20) . "
";  // 出力: 10 + 20 = 30
?>

echoとreturnの違い

echoは画面に出力するだけで、値を返しません。 returnは値を返すので、その結果を変数に代入したり、他の処理で使ったりできます。

実習 5-2: 消費税計算関数を作る

税込価格を計算する関数を作成しましょう

税抜価格を受け取り、税込価格を返す関数を作ってください（消費税率10%）。

practice_5_2.php 
<?php
function calcTax($price) {
    return $price * 1.10;  // 税込価格を返す
}

$basePrice = 1000;
$withTax = calcTax($basePrice);
echo "税抜{$basePrice}円 → 税込{$withTax}円
";
?>

5.5 スコープ（変数の有効範囲）

スコープとは、変数が使える範囲のことです。 PHPでは、関数の外で定義した変数と、関数の中で定義した変数は別物として扱われます。

ローカル変数とグローバル変数

scope.php 
<?php
$globalVar = "グローバル変数";  // グローバルスコープ

function testScope() {
    $localVar = "ローカル変数";  // ローカルスコープ
    echo $localVar . "
";     // OK: ローカル変数は使える
    // echo $globalVar;          // エラー: グローバル変数は使えない
}

testScope();
echo $globalVar . "
";  // OK: グローバル変数は使える
// echo $localVar;         // エラー: ローカル変数は使えない
?>

スコープの概念図

flowchart TB subgraph Global[グローバルスコープ（プログラム全体）] G1[グローバル変数] subgraph Func1[関数A] L1[ローカル変数A] end subgraph Func2[関数B] L2[ローカル変数B] end end

global キーワード（非推奨）

関数内でグローバル変数を使いたい場合、globalキーワードを使うことができますが、 コードが複雑になるため推奨されません。代わりに引数で渡す方が良いです。

better_way.php 
<?php
// 推奨される方法: 引数で渡す
$userName = "太郎";

function showGreeting($name) {
    echo "こんにちは、{$name}さん！
";
}

showGreeting($userName);  // 引数として渡す
?>

スコープのベストプラクティス

関数は引数で必要な値を受け取り、戻り値で結果を返すようにしましょう。 グローバル変数に頼らない設計にすることで、バグを減らし、テストしやすいコードになります。

5.6 組み込み関数の活用

PHPには、あらかじめ用意された便利な組み込み関数がたくさんあります。 これらを活用することで、効率的にプログラムを作ることができます。

よく使う組み込み関数

関数名 説明 使用例
strlen() 文字列の長さを返す strlen("Hello")
strtoupper() 大文字に変換 strtoupper("hello")
strtolower() 小文字に変換 strtolower("HELLO")
round() 四捨五入 round(3.14159, 2)
abs() 絶対値を返す abs(-10)
date() 日付を取得・フォーマット date("Y-m-d")

組み込み関数の使用例

built_in_functions.php 
<?php
// 文字列関数
$text = "Hello World";
echo "文字数: " . strlen($text) . "
";        // 出力: 11
echo "大文字: " . strtoupper($text) . "
";    // 出力: HELLO WORLD
echo "小文字: " . strtolower($text) . "
";    // 出力: hello world

// 数値関数
$price = 1234.5678;
echo "四捨五入: " . round($price, 2) . "
";  // 出力: 1234.57
echo "絶対値: " . abs(-100) . "
";            // 出力: 100

// 日付関数
echo "今日の日付: " . date("Y年m月d日") . "
";
?>

公式ドキュメントを活用しよう

PHPには数千の組み込み関数があります。全てを覚える必要はありません。 必要な時にPHP公式ドキュメントで検索する習慣をつけましょう。

理解度確認クイズ

  1. Q1. 関数を使うメリットを3つ挙げてください。

    回答を表示

    1. コードの再利用ができる
    2. 読みやすさが向上する
    3. 保守性が向上する（修正が1箇所で済む）
    ※その他、テストしやすいなどのメリットもあります。

  2. Q2. 引数と戻り値の違いを説明してください。

    回答を表示

    引数: 関数に渡す入力値。関数を呼び出すときに指定する。
    戻り値: 関数が返す出力値。return文で返される。

  3. Q3. ローカル変数とグローバル変数の違いは何ですか？

    回答を表示

    ローカル変数: 関数の中で定義された変数。その関数内でのみ使える。
    グローバル変数: 関数の外で定義された変数。プログラム全体で使えるが、関数内からは直接アクセスできない。

  4. Q4. echoとreturnの違いを説明してください。

    回答を表示

    echo: 画面に出力するだけで、値を返さない。
    return: 関数の呼び出し元に値を返す。返された値は変数に代入したり、他の処理で使ったりできる。

第5章のまとめ

  • 関数は、特定の処理をまとめて名前をつけたもので、コードの再利用や保守性向上に役立ちます。
  • 関数はfunctionキーワードで定義し、関数名を書いて呼び出します。
  • 引数を使うと、関数に値を渡すことができ、同じ関数でも異なる結果を得られます。
  • 戻り値はreturn文で返され、関数の処理結果を受け取ることができます。
  • スコープは変数の有効範囲で、ローカル変数は関数内のみ、グローバル変数はプログラム全体で使えます。
  • PHPには便利な組み込み関数が多数用意されており、これらを活用すると効率的にプログラムを作成できます。

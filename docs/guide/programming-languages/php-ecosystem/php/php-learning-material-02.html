開発環境の構築

XAMPP/MAMPを使用したPHP開発環境の構築方法、エディタの設定、最初のPHPプログラムの作成と実行方法を習得します。

この章で学ぶこと

  • 開発環境とは何かを理解する
  • XAMPP/MAMPのインストール方法を習得する
  • Visual Studio Codeの基本設定を行う
  • 最初のPHPプログラムを作成・実行する

2.1 開発環境とは

開発環境とは、プログラムを作成し、動作確認するために必要なソフトウェアやツールのセットのことです。 料理に例えると、キッチンや調理器具、食材を揃えることに相当します。

PHPを学習するためには、以下の3つの要素が必要になります：

Apache] B[PHP本体

MySQL] D[エディタ
VS Code] end A --> E[PHPプログラムを実行] B --> E C --> F[データを保存・管理] D --> G[コードを書く]

XAMPP/MAMPが便利な理由

通常、Webサーバー・PHP・MySQLをそれぞれ個別にインストールする必要がありますが、 XAMPP（Windows/Linux）やMAMP（Mac）を使うと、これらを一度にまとめてインストールできます。 初心者にとって、環境構築の手間が大幅に削減されます。

2.2 必要なソフトウェア

この学習ガイドでは、以下のソフトウェアを使用します。すべて無料でダウンロード・利用できます。

XAMPP / MAMP

Webサーバー・PHP・MySQLをまとめてインストールできるパッケージソフトウェア。

対応OS:
Windows/Linux → XAMPP
macOS → MAMP

Visual Studio Code

Microsoft社が提供する無料の高機能コードエディタ。拡張機能が豊富で、PHP開発に最適。

対応OS:
Windows / macOS / Linux

Webブラウザ

PHPプログラムの動作確認に使用します。Chrome、Firefox、Safari等の最新版を推奨。

推奨: Google Chrome

文字コード設定

PHPファイルは必ずUTF-8（BOM無し）で保存します。

重要: 文字化け防止のため必須

2.3 XAMPP/MAMPのインストール

お使いのOSに応じて、XAMPPまたはMAMPをインストールします。

実習 2-1: XAMPP のインストール（Windows/Linux）

手順

  1. 1 公式サイトにアクセス: https://www.apachefriends.org/jp/index.html
  2. 2 お使いのOS用インストーラーをダウンロード（PHP 8.x版を推奨）
  3. 3 ダウンロードしたインストーラーを実行
  4. 4 インストール先はデフォルト（C:\xampp）のままでOK
  5. 5 インストール完了後、XAMPP Control Panelを起動
  6. 6 「Apache」と「MySQL」の「Start」ボタンをクリックして起動

実習 2-2: MAMP のインストール（macOS）

手順

  1. 1 公式サイトにアクセス: https://www.mamp.info/en/downloads/
  2. 2 「MAMP」（無料版）のインストーラーをダウンロード
  3. 3 .pkgファイルを開いてインストール
  4. 4 インストール完了後、MAMPアプリを起動
  5. 5 「Start」ボタンをクリックしてサーバーを起動

動作確認

ブラウザで以下のURLにアクセスして、Apacheの起動を確認します： 

http://localhost/

期待される結果

XAMPPまたはMAMPのウェルカムページ（ダッシュボード）が表示されればOKです。

トラブルシューティング

Apacheが起動しない（ポート80が使用中）

原因: 別のアプリケーション（Skype、IISなど）がポート80を使用しています。

解決方法: XAMPP/MAMPの設定でポート番号を変更（例: 8080）するか、競合するアプリケーションを停止してください。

http://localhost/ にアクセスできない

原因: Apacheが起動していない、またはファイアウォールでブロックされています。

解決方法: XAMPP/MAMPのコントロールパネルでApacheが「実行中」になっているか確認。ファイアウォール設定でApacheを許可してください。

2.4 Visual Studio Code のセットアップ

Visual Studio Code（VS Code）は、世界中の開発者に愛用されている無料のコードエディタです。 PHP開発を快適に行うための設定を行いましょう。

実習 2-3: VS Code のインストールと設定

手順

  1. 1 公式サイトからダウンロード: https://code.visualstudio.com/
  2. 2 インストーラーを実行してインストール
  3. 3 VS Codeを起動し、左サイドバーの拡張機能アイコンをクリック
  4. 4 以下の拡張機能を検索してインストール：
    • 「PHP Intelephense」（PHP補完）
    • 「Japanese Language Pack」（日本語化）
  5. 5 VS Codeを再起動

推奨設定: 自動保存をオンにする

ファイル → 自動保存 をチェックすると、編集内容が自動で保存されるため、保存忘れによるトラブルを防げます。

2.5 最初のPHPプログラム作成

いよいよ、最初のPHPプログラムを作成します。プログラミングの世界では、最初に「Hello, World!」というメッセージを表示するプログラムを作ることが伝統となっています。

実習 2-4: Hello World プログラムの作成

最初のPHPプログラムを作成し、ブラウザで表示してみましょう。

手順

  1. 1 ドキュメントルートフォルダを確認：
    XAMPP: C:\xampp\htdocs
    MAMP: /Applications/MAMP/htdocs
  2. 2 htdocsフォルダ内に「php_lesson」という新しいフォルダを作成
  3. 3 VS Codeで「php_lesson」フォルダを開く
  4. 4 新しいファイルを作成し、「hello.php」という名前で保存
  5. 5 以下のコードを入力
hello.php 
<?php
// これはコメントです（プログラムに影響しない説明文）
echo "Hello, World!";
?>

コードの説明

  • <?php: PHPコードの開始を示すタグ
  • // ...: コメント（プログラムに影響しない説明文）
  • echo: 画面に文字を表示する命令
  • "Hello, World!": 表示する文字列（ダブルクォートで囲む）
  • ;: 命令の終わりを示す記号（セミコロン）
  • ?>: PHPコードの終了を示すタグ

重要ポイント

PHPファイルは必ず.phpという拡張子で保存してください。 .txtや.htmlでは、PHPコードが実行されません。

2.6 プログラムの実行と動作確認

作成したPHPプログラムをブラウザで実行してみましょう。

動作確認

ブラウザで以下のURLにアクセスします： 

http://localhost/php_lesson/hello.php

期待される結果 

Hello, World!

ブラウザに「Hello, World!」と表示されれば成功です！🎉

PHPの実行の仕組み

sequenceDiagram participant B as ブラウザ participant A as Apacheサーバー participant P as PHP participant F as hello.php B->>A: hello.phpにアクセス A->>F: ファイルを読み込み A->>P: PHPコードを実行依頼 P->>P: echo命令を実行 P-->>A: 実行結果（HTML） A-->>B: 結果を返す B->>B: Hello, World! を表示

PHPはサーバーサイドで実行されます。 ブラウザがPHPファイルにアクセスすると、Apacheサーバーが PHPインタープリタを呼び出し、 コードを実行してから結果をブラウザに返します。

つまずきポイント

PHPファイルをダブルクリックして開くと、PHPコードがそのまま表示されてしまいます。 必ずブラウザのアドレスバーに http://localhost/... と入力してアクセスしてください。

理解度確認クイズ

  1. Q1. 開発環境とは何ですか？ 簡単に説明してください。

    回答を表示

    開発環境とは、プログラムを作成し、動作確認するために必要なソフトウェアやツールのセットのことです。 PHP開発では、Webサーバー（Apache）、PHP本体、データベース（MySQL）、エディタが必要です。

  2. Q2. XAMPP/MAMPを使うメリットは何ですか？

    回答を表示

    通常は個別にインストールする必要がある Webサーバー・PHP・MySQLを、一度にまとめてインストールできます。 初心者にとって、環境構築の手間が大幅に削減され、簡単に開発を始められます。

  3. Q3. PHPファイルの拡張子は何ですか？

    回答を表示

    .php です。
    .txt や .html ではPHPコードが実行されません。

  4. Q4. PHPプログラムを実行する正しい方法はどれですか？

    A) ファイルをダブルクリックして開く
    B) ブラウザのアドレスバーに http://localhost/... と入力してアクセス
    C) メモ帳で開く

    回答を表示

    正解は B です。
    PHPはサーバーサイドで実行されるため、必ずWebサーバー（Apache）経由でアクセスする必要があります。 ファイルを直接開くと、PHPコードがそのまま表示されてしまいます。

第2章のまとめ

  • 開発環境は、プログラム作成に必要なソフトウェアのセットで、Webサーバー・PHP・データベース・エディタが含まれます。
  • XAMPP/MAMPを使うと、必要な環境をまとめてインストールでき、初心者でも簡単に開発を始められます。
  • Visual Studio Codeは無料で高機能なエディタで、拡張機能を追加することでPHP開発を快適に行えます。
  • PHPファイルは.php拡張子で保存し、http://localhost/...経由でブラウザからアクセスして実行します。
  • echo命令を使うと、画面に文字列を表示できます。命令の最後には必ずセミコロン（;）を付けます。

第6章

配列の操作

配列の作成と操作、連想配列、多次元配列、配列関数を使った実践的なデータ処理方法を習得します。

この章で学ぶこと

  • 配列とは何か、その仕組みを理解する
  • インデックス配列と連想配列の違いを知る
  • 多次元配列を扱えるようになる
  • 配列関数を使った実践的なデータ処理を習得する

6.1 配列とは

配列（はいれつ）とは、複数のデータを1つの変数にまとめて管理するための仕組みです。 これまで学んできた変数は、1つの値しか保存できませんでしたが、配列を使うと複数の値を順番に保存できます。

例えば、生徒の名前を管理する場合、変数を使うと以下のように複数の変数が必要になります：

変数を使った例 
<?php
// 変数を使った管理（非効率）
$student1 = "田中太郎";
$student2 = "佐藤花子";
$student3 = "鈴木一郎";
// ... 生徒が増えると管理が大変！
?>

配列を使うと、これらのデータを1つの変数でまとめて管理できます：

配列を使った例 
<?php
// 配列を使った管理（効率的）
$students = ["田中太郎", "佐藤花子", "鈴木一郎"];
echo $students[0]; // 田中太郎
echo $students[1]; // 佐藤花子
echo $students[2]; // 鈴木一郎
?>

配列は「引き出し付きの箱」に似ています

配列は、番号が振られた引き出しがたくさんある箱のようなものです。 各引き出しにデータを入れておき、番号（インデックス）を指定して取り出すことができます。

配列のイメージ

flowchart LR A["変数 $students"] --> B["インデックス 0
田中太郎"] A --> C["インデックス 1
佐藤花子"] A --> D["インデックス 2
鈴木一郎"]

配列のメリット

  • 管理がシンプル: 複数のデータを1つの変数で管理できる
  • 繰り返し処理に便利: foreach文などでまとめて処理できる
  • データの追加・削除が簡単: 要素数が増えても柔軟に対応できる

6.2 配列の作成と基本操作

PHPで配列を作成する方法はいくつかあります。ここでは基本的な作成方法と操作を学びます。

配列の作成方法

array_basic.php 
<?php
// 方法1: 短い構文（推奨）
$fruits = ["りんご", "バナナ", "オレンジ"];

// 方法2: array()関数
$colors = array("赤", "青", "緑");

// 空の配列を作成
$empty = [];

// 値の取得（インデックスは0から始まる）
echo $fruits[0]; // りんご
echo $fruits[1]; // バナナ
?>

インデックスは0から始まる

配列の最初の要素は0番目です。 2番目は1、3番目は2となります。 これは多くのプログラミング言語で共通のルールです。

配列への要素の追加

array_add.php 
<?php
$fruits = ["りんご", "バナナ"];

// 末尾に追加（推奨）
$fruits[] = "オレンジ";
$fruits[] = "ぶどう";

// 特定のインデックスに代入
$fruits[5] = "メロン"; // インデックス5に代入

print_r($fruits);
// Array ( [0]=>りんご [1]=>バナナ [2]=>オレンジ [3]=>ぶどう [5]=>メロン )
?>

配列の要素数を取得

array_count.php 
<?php
$fruits = ["りんご", "バナナ", "オレンジ"];

// count()関数で要素数を取得
$count = count($fruits);
echo "果物は{$count}種類あります"; // 果物は3種類あります

// 配列が空かどうか確認
if (empty($fruits)) {
    echo "配列は空です";
}
?>

6.3 連想配列

連想配列は、数値のインデックスの代わりに、文字列のキー（鍵）を使ってデータを管理する配列です。 「キー」と「値」のペアでデータを保存するため、データの意味をわかりやすく表現できます。

assoc_array.php 
<?php
// 連想配列の作成
$user = [
    "name" => "田中太郎",
    "age" => 25,
    "email" => "tanaka@example.com"
];

// 値の取得
echo $user["name"];  // 田中太郎
echo $user["age"];   // 25
?>

連想配列の構造

flowchart LR A["変数 $user"] --> B["キー: name
値: 田中太郎"] A --> C["キー: age
値: 25"] A --> D["キー: email
値: tanaka@example.com"]

連想配列の操作

assoc_operations.php 
<?php
$user = ["name" => "田中太郎", "age" => 25];

// 新しいキーと値を追加
$user["city"] = "東京";

// 値の更新
$user["age"] = 26;

// foreach文で全要素を表示
foreach ($user as $key => $value) {
    echo "{$key}: {$value}\n";
}
?>

連想配列の活用例

product.php 
<?php
// 商品情報を連想配列で管理
$product = [
    "id" => 101,
    "name" => "ノートパソコン",
    "price" => 89800,
    "stock" => 15
];

// 価格表示
echo "商品名: {$product['name']}\n";
echo "価格: ¥" . number_format($product['price']) . "\n";
echo "在庫数: {$product['stock']}台\n";
?>

連想配列を使うべき場面

データに意味のある名前を付けたい場合は連想配列を使いましょう。 「ユーザー情報」「商品情報」「設定データ」など、複数の項目を持つデータを扱う場合に適しています。

6.4 多次元配列

多次元配列とは、配列の中に配列を入れた構造のことです。 複雑なデータ構造を表現する場合に便利です。例えば、複数の生徒の情報をまとめて管理する場合などに使います。

multidimensional.php 
<?php
// 2次元配列の例（生徒情報）
$students = [
    ["name" => "田中太郎", "score" => 85],
    ["name" => "佐藤花子", "score" => 92],
    ["name" => "鈴木一郎", "score" => 78]
];

// 最初の生徒の名前
echo $students[0]["name"]; // 田中太郎

// 2番目の生徒の点数
echo $students[1]["score"]; // 92
?>

2次元配列の構造イメージ

flowchart TB A["変数 $students"] --> B["インデックス 0"] A --> C["インデックス 1"] A --> D["インデックス 2"] B --> B1["name: 田中太郎"] B --> B2["score: 85"] C --> C1["name: 佐藤花子"] C --> C2["score: 92"] D --> D1["name: 鈴木一郎"] D --> D2["score: 78"]

多次元配列の走査

loop_multidim.php 
<?php
$students = [
    ["name" => "田中太郎", "score" => 85],
    ["name" => "佐藤花子", "score" => 92],
    ["name" => "鈴木一郎", "score" => 78]
];

// 全生徒の情報を表示
foreach ($students as $student) {
    echo "{$student['name']}さん: {$student['score']}点\n";
}
?>

インデックスの指定に注意

多次元配列では、$students[0]["name"]のように、 外側から順番にインデックスやキーを指定します。順序を間違えるとエラーになるので注意しましょう。

6.5 配列関数の活用

PHPには配列を操作するための便利な組み込み関数がたくさん用意されています。 これらを活用することで、効率的にデータ処理ができます。

よく使う配列関数

関数名 説明
count() 配列の要素数を取得
array_push() 配列の末尾に要素を追加
array_pop() 配列の末尾から要素を取り出す
in_array() 配列に特定の値が含まれるか確認
array_merge() 複数の配列を結合
sort() 配列を昇順にソート

配列関数の使用例

array_functions.php 
<?php
$fruits = ["りんご", "バナナ", "オレンジ"];

// 末尾に要素を追加
array_push($fruits, "ぶどう");

// 特定の値が含まれるか確認
if (in_array("バナナ", $fruits)) {
    echo "バナナがあります\n";
}

// 配列をソート
sort($fruits);
print_r($fruits);
?>

配列のマッピングとフィルタリング

array_map_filter.php 
<?php
$numbers = [1, 2, 3, 4, 5];

// 各要素を2倍にする（array_map）
$doubled = array_map(function($n) {
    return $n * 2;
}, $numbers);
print_r($doubled); // [2, 4, 6, 8, 10]

// 偶数だけ抽出（array_filter）
$evens = array_filter($numbers, function($n) {
    return $n % 2 == 0;
});
print_r($evens); // [2, 4]
?>

実践例：生徒の成績処理

score_processing.php 
<?php
$scores = [85, 92, 78, 65, 90];

// 平均点の計算
$average = array_sum($scores) / count($scores);
echo "平均点: " . round($average, 1) . "点\n";

// 最高点・最低点
echo "最高点: " . max($scores) . "点\n";
echo "最低点: " . min($scores) . "点\n";

// 合格者（60点以上）をカウント
$passed = array_filter($scores, fn($s) => $s >= 60);
echo "合格者数: " . count($passed) . "人\n";
?>

理解度確認クイズ

  1. Q1. 配列のインデックスは何番から始まりますか？

    回答を表示

    配列のインデックスは0から始まります。最初の要素は$array[0]でアクセスします。

  2. Q2. 連想配列とは何ですか？ 通常の配列との違いを説明してください。

    回答を表示

    連想配列は、数値のインデックスではなく文字列のキーを使ってデータを管理する配列です。 「キー」と「値」のペアでデータを保存するため、データの意味がわかりやすくなります。 例：$user["name"] = "田中太郎"

  3. Q3. 配列の要素数を取得する関数は何ですか？

    回答を表示

    count()関数を使います。 例：$count = count($array);

  4. Q4. 配列に特定の値が含まれているか確認する関数は何ですか？

    回答を表示

    in_array()関数を使います。 例：if (in_array("バナナ", $fruits)) { ... }

第6章のまとめ

  • 配列は複数のデータを1つの変数にまとめて管理する仕組みで、インデックス（番号）は0から始まります。
  • 連想配列は文字列のキーを使ってデータを管理し、データの意味をわかりやすく表現できます。
  • 多次元配列は配列の中に配列を入れた構造で、複雑なデータを表現できます。
  • PHPにはcount()、in_array()、array_push()、sort()など、配列を操作する便利な関数が多数用意されています。
  • 配列を使うことで、繰り返し処理やデータ管理が効率的になり、実用的なプログラムを作成できます。

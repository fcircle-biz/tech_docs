PHPとプログラミングの基礎

プログラミングとは何か、PHPの特徴と用途、Web開発における役割など、プログラミングが初めての方でも理解できる基礎知識を学びます。

この章で学ぶこと

  • プログラミングとは何かを理解する
  • PHPの特徴と用途を知る
  • Web開発におけるPHPの役割を理解する
  • PHPでできることを知る

1.1 プログラミングとは何か

プログラミングとは、コンピュータに「何をどのように処理するか」を指示する作業のことです。 人間が日常的に使う言葉（日本語や英語）ではなく、コンピュータが理解できる特別な言葉（プログラミング言語）を使って指示を書きます。

例えば、「1から10までの数字を足し算して結果を表示する」という処理をコンピュータにやらせたい場合、 私たちは頭の中で「1+2+3+...+10 = 55」と計算できますが、コンピュータはそのままでは理解できません。 プログラミング言語を使って、具体的な手順を書く必要があります。

プログラミングは「レシピ」に似ています

料理のレシピには「材料を切る」「鍋に入れる」「火にかける」など、手順が書かれています。 プログラミングも同じで、コンピュータに「この順番でこれをやってください」と指示を書くのです。

プログラミングでできること

  • 計算を自動化する: 複雑な計算を瞬時に実行
  • データを管理する: 大量の情報を整理して保存・検索
  • Webサイトを作る: インターネット上で動く便利なサービスを構築
  • 繰り返し作業を自動化する: 単調な作業をコンピュータに任せる

1.2 PHPとは

PHP（ピー・エイチ・ピー）は、Web開発に特化したプログラミング言語です。 正式名称は「PHP: Hypertext Preprocessor」で、主にWebサーバー上で動作するプログラムを作るために使われます。

PHPは1995年に誕生し、現在では世界中の多くのWebサイトで使われています。 有名なサービスでは、Facebook（初期）、Wikipedia、WordPressなどがPHPで作られています。

PHPの歴史

timeline title PHPの発展 1995 : PHP 1.0登場（個人の趣味プロジェクト） 1997 : PHP 3.0（本格的なWeb言語へ） 2000 : PHP 4.0（パフォーマンス向上） 2004 : PHP 5.0（オブジェクト指向対応） 2015 : PHP 7.0（大幅な高速化） 2020 : PHP 8.0（最新機能追加）

1.3 PHPの特徴

PHPには初心者にとって学びやすい、いくつかの大きな特徴があります。

学習しやすい

文法がシンプルで、英語に近い書き方ができます。 初心者でも数日で簡単なプログラムが書けるようになります。

無料で使える

PHPは完全に無料で使えるオープンソースソフトウェアです。 高額な開発ツールを買う必要がありません。

サーバーで動く

Webサーバー上で実行されるため、動的なWebページを作れます。 訪問者ごとに異なる内容を表示することも可能です。

データベース連携

MySQLなどのデータベースと簡単に連携できます。 会員情報や商品データを効率的に管理できます。

1.4 Web開発におけるPHPの役割

Webサイトは大きく分けて「フロントエンド」と「バックエンド」という2つの部分から構成されています。 PHPはバックエンドで活躍する言語です。

flowchart TB subgraph Frontend[フロントエンド（ユーザーが見る部分）] A[HTML: ページの構造] B[CSS: デザイン・装飾] C[JavaScript: 動的な動き] end subgraph Backend[バックエンド（裏側の処理）] D[PHP: ビジネスロジック] E[MySQL: データ保存] end Frontend --> Backend D --> E

フロントエンドとバックエンドの違い

項目 フロントエンド バックエンド（PHP）
実行場所 ブラウザ（ユーザーのPC） Webサーバー
主な役割 見た目・操作性 データ処理・ロジック
使用言語 HTML, CSS, JavaScript PHP, Python, Ruby等
具体例 ボタンのデザイン、画像表示 ログイン認証、検索処理

例えば、ECサイト（オンラインショップ）で商品を検索する場合：

  1. ユーザーが検索ボックスに「スニーカー」と入力（フロントエンド）
  2. PHPがデータベースから「スニーカー」に該当する商品を検索（バックエンド）
  3. 検索結果がHTMLとして生成され、ブラウザに表示（フロントエンド）

1.5 PHPでできること

PHPを使うと、様々な種類のWebアプリケーションを作ることができます。 ここでは、具体的な例を見ていきましょう。

ECサイト

商品の表示、カート機能、決済処理など、オンラインショップに必要な機能を実装できます。

SNS・コミュニティ

ユーザー登録、ログイン、投稿機能、コメント機能など、交流サイトを構築できます。

CMS・ブログ

WordPress のようなコンテンツ管理システムや、個人ブログサイトを作成できます。

業務システム

顧客管理、在庫管理、売上集計など、社内で使う管理ツールを開発できます。

お問い合わせフォーム

メール送信機能付きのフォームを作成し、訪問者からの問い合わせを受け付けられます。

API開発

他のシステムと連携するためのAPIを作成し、データをやり取りできます。

PHPの強み

PHPは特に「動的なWebページ」の作成が得意です。 ユーザーごとに異なる内容を表示したり、データベースの情報をリアルタイムで反映させたりすることができます。

理解度確認クイズ

  1. Q1. プログラミングとは何ですか？ 簡単に説明してください。

    回答を表示

    プログラミングとは、コンピュータに「何をどのように処理するか」を指示する作業のことです。 プログラミング言語という特別な言葉を使って、コンピュータが理解できる形で命令を書きます。

  2. Q2. PHPの特徴を3つ挙げてください。

    回答を表示

    1. 学習しやすい（文法がシンプル）
    2. 無料で使える（オープンソース）
    3. サーバーで動く（動的なWebページを作成可能）
    ※その他、データベース連携が簡単などの特徴もあります。

  3. Q3. フロントエンドとバックエンドの違いは何ですか？

    回答を表示

    フロントエンド: ユーザーが直接見て操作する部分。ブラウザで実行され、見た目や操作性を担当（HTML, CSS, JavaScript）。
    バックエンド: Webサーバー上で動く裏側の処理。データの処理やビジネスロジックを担当（PHP, Pythonなど）。

  4. Q4. PHPで作れるWebアプリケーションの例を2つ挙げてください。

    回答を表示

    例：
    1. ECサイト（オンラインショップ）
    2. SNS・コミュニティサイト
    ※その他、CMS・ブログ、業務システム、お問い合わせフォーム、API開発なども該当します。

第1章のまとめ

  • プログラミングは、コンピュータに指示を出すための作業で、特別な言語（プログラミング言語）を使って書きます。
  • PHPは、Web開発に特化した無料のプログラミング言語で、世界中で広く使われています。
  • PHPの特徴は、学習しやすい、無料で使える、サーバーで動く、データベース連携が簡単などがあります。
  • PHPはバックエンドで活躍し、データ処理やビジネスロジックを担当します。
  • PHPでは、ECサイト、SNS、CMS、業務システム、お問い合わせフォーム、APIなど、様々なWebアプリケーションを作ることができます。

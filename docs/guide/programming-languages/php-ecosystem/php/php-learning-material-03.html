変数とデータ型

変数の概念、データ型（文字列、数値、真偽値）、型変換、定数の使い方など、プログラミングの基本要素を理解します。

この章で学ぶこと

  • 変数とは何かを理解する
  • 変数の宣言方法と命名規則を習得する
  • 主要なデータ型（文字列、数値、真偽値）を理解する
  • 型変換の方法を学ぶ
  • 定数の使い方を習得する

3.1 変数とは

変数とは、データを一時的に保存しておくための「箱」のようなものです。 プログラムの中で、計算結果や入力された値などを記憶しておきたい時に使います。

例えば、「りんごの価格」を保存したい場合、$priceという名前の変数を作り、 その中に「150」という数値を入れることができます。 後でその変数を参照すれば、保存した値を取り出すことができます。

変数は「ラベル付きの箱」

変数には名前（ラベル）をつけて、その中にデータ（値）を入れることができます。 必要な時にラベルを呼び出せば、中のデータを取り出して使えます。

変数のイメージ

flowchart LR A[変数名: $price] -->|格納| B["値: 150"] C[変数名: $name] -->|格納| D["値: 'りんご'"] E[変数名: $is_available] -->|格納| F["値: true"]

3.2 変数の宣言と命名規則

PHPでは、変数名の先頭にドルマーク（$）を付けて変数を宣言します。 =（代入演算子）を使って、変数に値を代入します。

変数の基本的な宣言 
<?php
// 変数の宣言と代入
$name = "山田太郎";  // 文字列を代入
$age = 25;          // 数値を代入
$height = 175.5;    // 小数を代入

// 変数の値を出力
echo $name;  // 出力: 山田太郎
echo $age;   // 出力: 25
?>

変数の命名規則

変数名には一定のルールがあります。以下の規則を守って命名しましょう。

ルール 説明 例
$で始める 必ず先頭に$を付ける $name
英字または_で始める $の後は英字またはアンダースコア $_count
英数字と_を使用 2文字目以降は英数字とアンダースコアが使える $user_name_1
大文字小文字を区別 $nameと$Nameは別の変数 $name ≠ $Name
予約語は使えない PHPの特別な単語は変数名にできない $class (NG)

良い変数名の付け方

  • 内容がわかりやすい名前をつける（$user_age は良い、$x は悪い）
  • 英単語を使う（$price、$total_amountなど）
  • 複数単語はアンダースコアで区切る（$user_name、$item_count）

3.3 データ型の種類

データ型とは、変数に格納できるデータの種類のことです。 PHPには様々なデータ型がありますが、初心者がまず覚えるべき主要な型は以下の3つです。

文字列型

テキストデータを扱う型です。シングルクォート（'）またはダブルクォート（"）で囲みます。

$name = "太郎";

数値型

整数や小数を扱う型です。計算に使用できます。

$age = 25;

真偽値型

true（真）またはfalse（偽）のいずれかの値を持つ型です。

$is_active = true;

PHPの主要なデータ型

flowchart TD A[PHPのデータ型] --> B[スカラー型] A --> C[複合型] A --> D[特殊型] B --> E[文字列: string] B --> F[整数: integer] B --> G[浮動小数点数: float] B --> H[真偽値: boolean] C --> I[配列: array] C --> J[オブジェクト: object] D --> K[NULL] D --> L[リソース: resource]

※この章では、スカラー型（文字列、整数、浮動小数点数、真偽値）を中心に学習します。

3.4 文字列型（String）

文字列型は、テキストデータを扱うデータ型です。 シングルクォート（'）またはダブルクォート（"）で囲んで表現します。

文字列の基本 
<?php
// シングルクォートで囲む
$message1 = 'こんにちは';

// ダブルクォートで囲む
$message2 = "こんにちは";

// 文字列の連結（.演算子を使用）
$first_name = "太郎";
$last_name = "山田";
$full_name = $last_name . $first_name;  // "山田太郎"

echo $full_name;
?>

シングルクォートとダブルクォートの違い

項目 シングルクォート（'） ダブルクォート（"）
変数の展開 展開されない 展開される
エスケープシーケンス 一部のみ有効 すべて有効
処理速度 わずかに速い わずかに遅い
シングルクォートとダブルクォートの実例 
<?php
$name = "太郎";

// シングルクォート：変数が展開されない
echo '私の名前は$nameです';  // 出力: 私の名前は$nameです

// ダブルクォート：変数が展開される
echo "私の名前は$nameです";  // 出力: 私の名前は太郎です

// 改行文字の違い
echo 'こんにちは\n世界';  // 出力: こんにちは\n世界
echo "こんにちは\n世界";  // 出力: こんにちは（改行）世界
?>

使い分けのポイント

変数を含めたい場合はダブルクォート、単純な文字列はシングルクォートを使うのが一般的です。

3.5 数値型（Integer / Float）

数値型には、整数を扱うInteger型と、 小数を扱うFloat型（浮動小数点数）があります。

数値型の基本 
<?php
// 整数型（Integer）
$age = 25;
$count = 100;
$negative = -50;  // 負の数も使える

// 浮動小数点数型（Float）
$height = 175.5;
$price = 1980.0;
$pi = 3.14159;

// 四則演算
$a = 10;
$b = 3;
echo $a + $b;  // 加算: 13
echo $a - $b;  // 減算: 7
echo $a * $b;  // 乗算: 30
echo $a / $b;  // 除算: 3.333...
echo $a % $b;  // 剰余（余り）: 1
?>

主な算術演算子

演算子 名称 例 結果
+ 加算 $a + $b $aと$bの和
- 減算 $a - $b $aと$bの差
* 乗算 $a * $b $aと$bの積
/ 除算 $a / $b $aを$bで割った商
% 剰余 $a % $b $aを$bで割った余り
** べき乗 $a ** $b $aの$b乗

3.6 真偽値型（Boolean）

真偽値型（Boolean）は、true（真）またはfalse（偽）のいずれかの値のみを持つデータ型です。 主に条件判定に使用されます。

真偽値の基本 
<?php
// 真偽値の代入
$is_active = true;   // 真
$is_deleted = false; // 偽

// 比較演算の結果は真偽値
$a = 10;
$b = 20;
$result = ($a < $b);  // true（10は20より小さい）

// 真偽値の使用例
if ($is_active) {
    echo "アクティブです";
}
?>

比較演算子

演算子 名称 例 説明
== 等しい $a == $b $aと$bが等しい
=== 厳密に等しい $a === $b 値と型が両方等しい
!= 等しくない $a != $b $aと$bが等しくない
< より小さい $a < $b $aが$bより小さい
> より大きい $a > $b $aが$bより大きい
<= 以下 $a <= $b $aが$b以下
>= 以上 $a >= $b $aが$b以上

==と===の違いに注意

==は値のみを比較しますが、===は値と型の両方を比較します。 例: 5 == "5"はtrue、5 === "5"はfalse

3.7 型変換

型変換とは、あるデータ型を別のデータ型に変換することです。 PHPでは、自動型変換と明示的な型変換の2種類があります。

自動型変換

PHPは文脈に応じて自動的にデータ型を変換します。

自動型変換の例 
<?php
// 文字列と数値の連結
$result = "価格: " . 1500;  // "価格: 1500"（数値が文字列に変換）

// 文字列を数値として計算
$a = "10";
$b = "5";
$sum = $a + $b;  // 15（文字列が数値に自動変換）

// 真偽値と文字列
$active = true;
echo "ステータス: " . $active;  // "ステータス: 1"（trueが1に変換）
?>

明示的な型変換（キャスト）

変換したい型を(型名)の形式で明示的に指定できます。

明示的な型変換の例 
<?php
// 文字列を整数に変換
$str = "123";
$num = (int)$str;  // 123

// 浮動小数点数を整数に変換（小数点以下切り捨て）
$float = 3.99;
$int = (int)$float;  // 3

// 整数を文字列に変換
$number = 456;
$string = (string)$number;  // "456"

// 真偽値に変換
$value = 1;
$bool = (bool)$value;  // true
?>

型変換関数

関数 説明 例
intval() 整数に変換 intval("123") → 123
floatval() 浮動小数点数に変換 floatval("3.14") → 3.14
strval() 文字列に変換 strval(456) → "456"
boolval() 真偽値に変換 boolval(1) → true

3.8 定数

定数は、一度値を設定したら変更できない特別な変数です。 プログラム全体で共通して使う値（税率、円周率など）を定義するときに使います。

定数の定義と使用 
<?php
// define関数で定数を定義
define("TAX_RATE", 0.1);  // 消費税率 10%
define("SITE_NAME", "私のサイト");

// 定数の使用（$マークは不要）
$price = 1000;
$tax = $price * TAX_RATE;  // 100
$total = $price + $tax;    // 1100

echo SITE_NAME;  // "私のサイト"

// constキーワードでも定義可能
const MAX_USERS = 100;
echo MAX_USERS;
?>

変数と定数の違い

項目 変数 定数
記号 $を付ける $は不要
値の変更 何度でも変更可能 変更不可
命名規則 小文字が一般的 大文字が一般的
用途 変化する値 固定値

定数を使うべき場面

  • 税率や手数料など、プログラム全体で使う固定値
  • 円周率や数学定数
  • 設定値やシステム定数

実習: 変数とデータ型を使ってみよう

以下のプログラムを作成して、変数とデータ型の動作を確認しましょう。

practice.php 
<?php
// 商品情報を変数で管理
$product_name = "ノートパソコン";
$price = 89800;
$stock = 15;
$is_available = true;

// 消費税を定数で定義
define("TAX_RATE", 0.1);

// 税込価格を計算
$tax = $price * TAX_RATE;
$total_price = $price + $tax;

// 結果を表示
echo "商品名: " . $product_name . "\n";
echo "本体価格: " . $price . "円\n";
echo "消費税: " . $tax . "円\n";
echo "税込価格: " . $total_price . "円\n";
echo "在庫数: " . $stock . "個\n";

if ($is_available) {
    echo "ステータス: 販売中\n";
} else {
    echo "ステータス: 在庫切れ\n";
}
?>

動作確認

実習プログラムを実行すると、以下のような結果が表示されます：

商品名: ノートパソコン 本体価格: 89800円 消費税: 8980円 税込価格: 98780円 在庫数: 15個 ステータス: 販売中

理解度確認クイズ

  1. Q1. 変数名として正しいものを選んでください。

    回答を表示

    正しい例: $user_name、$age、$_count
    間違い例: user_name（$がない）、$1st_name（数字で始まる）、$class（予約語）

  2. Q2. シングルクォートとダブルクォートの違いは何ですか？

    回答を表示

    シングルクォート（'）: 変数が展開されず、そのまま表示される
    ダブルクォート（"）: 変数が展開され、値が表示される

  3. Q3. 次のコードの実行結果は何ですか？
    $a = "10"; $b = "5"; echo $a + $b;

    回答を表示

    15が出力されます。文字列が自動的に数値に変換され、加算されます。

  4. Q4. 変数と定数の違いを説明してください。

    回答を表示

    変数: $を付けて宣言し、値を何度でも変更できる
    定数: defineまたはconstで定義し、一度設定したら値を変更できない

第3章のまとめ

  • 変数は、データを一時的に保存するための「箱」で、$マークを付けて宣言します。
  • 変数名は英字またはアンダースコアで始め、わかりやすい名前をつけることが重要です。
  • 主要なデータ型は、文字列型、数値型（整数・浮動小数点数）、真偽値型の3つです。
  • シングルクォート（'）は変数を展開せず、ダブルクォート（"）は変数を展開します。
  • 型変換には自動変換と明示的変換（キャスト）があり、データ型を変更できます。
  • 定数は、一度設定したら変更できない固定値で、defineまたはconstで定義します。

第8章

複数のデータを扱おう（配列とオブジェクト）

たくさんのデータを効率的に管理する配列と、関連する情報をまとめるオブジェクトの使い方を、実例を通じて学習します。

この章で学ぶこと

  • 配列を使って複数のデータを効率的に管理する方法を理解する
  • 配列の基本操作（追加・削除・取得・長さ）を身につける
  • オブジェクトを使って関連する情報をまとめる方法を学ぶ
  • 配列とオブジェクトを組み合わせて実用的なデータを扱えるようになる

8.1 配列とは何か

これまで変数を使って、1つの値を保存してきました。例えば、let name = "田中";のように、 1つの変数には1つの値しか入れられませんでした。

しかし、実際のプログラムでは複数のデータをまとめて扱いたい場面がたくさんあります。 例えば、クラスの生徒全員の名前や、買い物リストの商品など、関連するデータをまとめて管理したいときです。

配列のイメージ

配列は、番号の付いたロッカーのようなものです。 それぞれのロッカー（要素）には番号（インデックス）が付いていて、 番号を指定することで、そのロッカーに入っているデータを取り出すことができます。

flowchart LR A["配列: students"] --> B["0番目
田中"] A --> C["1番目
佐藤"] A --> D["2番目
鈴木"] A --> E["3番目
高橋"]

重要ポイント

配列の番号（インデックス）は0から始まります。最初の要素が0番目、2番目の要素が1番目となります。 これは最初は混乱しやすいですが、プログラミングでは一般的な決まりです。

8.2 配列の基本操作

配列の作成

配列を作るには、角カッコ [ ] を使います。カッコの中に、カンマで区切ってデータを並べます。

配列の作成例 
// 買い物リストの配列
let shoppingList = ["りんご", "バナナ", "牛乳", "パン"];

// 数値の配列
let scores = [85, 92, 78, 95, 88];

// 空の配列（後からデータを追加する場合）
let emptyArray = [];

配列の要素を取得する

配列から特定の要素を取り出すには、インデックス（番号）を角カッコで指定します。

配列の要素取得 
let fruits = ["りんご", "バナナ", "オレンジ", "ぶどう"];

console.log(fruits[0]);  // りんご（最初の要素）
console.log(fruits[1]);  // バナナ（2番目の要素）
console.log(fruits[2]);  // オレンジ（3番目の要素）

// 変数に代入することもできる
let firstFruit = fruits[0];
console.log(firstFruit);  // りんご

配列の長さを調べる

配列に何個のデータが入っているかを調べるには、.lengthプロパティを使います。

配列の長さ 
let fruits = ["りんご", "バナナ", "オレンジ"];

console.log(fruits.length);  // 3（3つの要素がある）

// 最後の要素を取得する（インデックスは0始まりなので、長さから1を引く）
let lastFruit = fruits[fruits.length - 1];
console.log(lastFruit);  // オレンジ

配列に要素を追加する

配列にデータを追加するには、主に2つの方法があります。

配列への要素追加 
let fruits = ["りんご", "バナナ"];

// pushメソッド: 配列の最後に追加
fruits.push("オレンジ");
console.log(fruits);  // ["りんご", "バナナ", "オレンジ"]

// unshiftメソッド: 配列の最初に追加
fruits.unshift("いちご");
console.log(fruits);  // ["いちご", "りんご", "バナナ", "オレンジ"]

配列から要素を削除する

配列からデータを削除する方法もいくつかあります。

配列からの要素削除 
let fruits = ["りんご", "バナナ", "オレンジ", "ぶどう"];

// popメソッド: 配列の最後の要素を削除して返す
let lastItem = fruits.pop();
console.log(lastItem);  // ぶどう
console.log(fruits);    // ["りんご", "バナナ", "オレンジ"]

// shiftメソッド: 配列の最初の要素を削除して返す
let firstItem = fruits.shift();
console.log(firstItem);  // りんご
console.log(fruits);     // ["バナナ", "オレンジ"]

8.3 配列をループで処理する

配列の最大の利点は、繰り返し処理（ループ）と組み合わせることで、 たくさんのデータを効率的に処理できることです。

for文で配列を処理する

for文で配列を処理 
let fruits = ["りんご", "バナナ", "オレンジ", "ぶどう"];

// 配列の全要素を表示する
for (let i = 0; i < fruits.length; i++) {
    console.log((i + 1) + "番目: " + fruits[i]);
}

// 出力:
// 1番目: りんご
// 2番目: バナナ
// 3番目: オレンジ
// 4番目: ぶどう

for...of文で配列を処理する（より簡単）

for...of文を使うと、インデックスを気にせず配列の要素を1つずつ処理できます。

for...of文で配列を処理 
let fruits = ["りんご", "バナナ", "オレンジ", "ぶどう"];

// 配列の要素を1つずつ取り出す
for (let fruit of fruits) {
    console.log("好きな果物: " + fruit);
}

// 出力:
// 好きな果物: りんご
// 好きな果物: バナナ
// 好きな果物: オレンジ
// 好きな果物: ぶどう

for文とfor...of文の使い分け

インデックス（番号）が必要な場合はfor文を使い、 要素の値だけが必要な場合はfor...of文を使うと便利です。

8.4 オブジェクトとは

配列は複数のデータを番号で管理していました。 一方、オブジェクトは、名前（キー）と値のペアでデータを管理します。

オブジェクトのイメージ

オブジェクトは、人物カードのようなものです。 名前、年齢、住所など、関連する情報がまとまって1つのカードになっています。 それぞれの情報には名前（ラベル）が付いていて、その名前で情報を取り出せます。

flowchart TB A["オブジェクト: person"] --> B["name
田中太郎"] A --> C["age
25"] A --> D["city
東京都"] A --> E["hobby
読書"]

オブジェクトの作成

オブジェクトを作るには、波カッコ { } を使います。 カッコの中に、キー: 値の形式で、カンマで区切って書きます。

オブジェクトの作成 
// 人物の情報をオブジェクトで管理
let person = {
    name: "田中太郎",
    age: 25,
    city: "東京都",
    hobby: "読書"
};

// 商品の情報をオブジェクトで管理
let product = {
    name: "ノートパソコン",
    price: 89800,
    stock: 15,
    isAvailable: true
};

オブジェクトの値を取得する

オブジェクトから値を取り出すには、主に2つの方法があります。

オブジェクトの値取得 
let person = {
    name: "田中太郎",
    age: 25,
    city: "東京都"
};

// 方法1: ドット記法（よく使われる）
console.log(person.name);  // 田中太郎
console.log(person.age);   // 25

// 方法2: ブラケット記法（キー名が変数の場合などに使う）
console.log(person["city"]);  // 東京都

// 変数に代入することもできる
let personName = person.name;
console.log(personName);  // 田中太郎

オブジェクトの値を変更・追加する

オブジェクトの値変更・追加 
let person = {
    name: "田中太郎",
    age: 25
};

// 既存の値を変更
person.age = 26;
console.log(person.age);  // 26

// 新しいプロパティを追加
person.hobby = "プログラミング";
console.log(person.hobby);  // プログラミング

console.log(person);
// { name: "田中太郎", age: 26, hobby: "プログラミング" }

8.5 配列とオブジェクトの組み合わせ

実際のプログラムでは、配列とオブジェクトを組み合わせて使うことが非常に多いです。 例えば、複数の人物情報を管理したい場合、オブジェクトの配列を使います。

オブジェクトの配列

オブジェクトの配列 
// 生徒の情報を配列で管理
let students = [
    { name: "田中太郎", age: 18, score: 85 },
    { name: "佐藤花子", age: 17, score: 92 },
    { name: "鈴木一郎", age: 18, score: 78 }
];

// 最初の生徒の情報を取得
console.log(students[0].name);   // 田中太郎
console.log(students[0].score);  // 85

// 全生徒の情報を表示
for (let student of students) {
    console.log(student.name + "さん: " + student.score + "点");
}
// 出力:
// 田中太郎さん: 85点
// 佐藤花子さん: 92点
// 鈴木一郎さん: 78点

実用的な例: 商品管理システム

商品管理の例 
// 商品データ
let products = [
    { id: 1, name: "ノートパソコン", price: 89800, stock: 5 },
    { id: 2, name: "マウス", price: 2980, stock: 20 },
    { id: 3, name: "キーボード", price: 5980, stock: 0 }
];

// 在庫がある商品だけを表示
console.log("在庫あり商品:");
for (let product of products) {
    if (product.stock > 0) {
        console.log(product.name + " - " + product.price + "円");
    }
}

// 在庫切れ商品を表示
console.log("\n在庫切れ商品:");
for (let product of products) {
    if (product.stock === 0) {
        console.log(product.name);
    }
}

実習 8-2: 生徒成績管理プログラムを作ろう

配列とオブジェクトを組み合わせて、生徒の成績を管理するプログラムを作りましょう。

要件

  1. 1 3人以上の生徒データ（名前、数学の点数、英語の点数）をオブジェクトの配列で作成
  2. 2 全生徒の名前と2科目の平均点を表示する
  3. 3 平均点が80点以上の生徒だけを表示する

第8章のまとめ

  • 配列は複数のデータを番号（インデックス）で管理する。インデックスは0から始まる
  • 配列の基本操作: push（追加）、pop（削除）、length（長さ）
  • 配列はfor文やfor...of文と組み合わせて、繰り返し処理に使える
  • オブジェクトは、名前（キー）と値のペアで関連データをまとめて管理する
  • オブジェクトの値はドット記法（obj.key）で取得できる
  • 配列とオブジェクトの組み合わせで、複雑なデータを効率的に管理できる

理解度確認クイズ

  1. Q1. 配列 let colors = ["赤", "青", "緑"]; の2番目の要素を取得するコードは？

    回答を表示

    colors[1]
    配列のインデックスは0から始まるため、2番目の要素は1番のインデックスです。

  2. Q2. 配列の最後に要素を追加するメソッドは？

    回答を表示

    push()メソッド
    例: arr.push("新しい要素");

  3. Q3. オブジェクトの値を取得する2つの方法を説明してください。

    回答を表示

    ① ドット記法: obj.key（よく使われる）
    ② ブラケット記法: obj["key"]（キー名が変数の場合などに使う）

  4. Q4. 配列とオブジェクトの主な違いは何ですか？

    回答を表示

    配列: 複数のデータを番号（インデックス）で管理する
    オブジェクト: 複数のデータを名前（キー）と値のペアで管理する
    配列は順序が重要な場合、オブジェクトは関連データをまとめる場合に使います。

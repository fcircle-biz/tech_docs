開発環境を整えよう

TypeScriptでプログラムを書くための道具（Node.js、Visual Studio Code）の準備を画像付きで丁寧に解説します。最も簡単な方法で環境を整え、すぐにプログラミングを始められるようにします。

この章で学ぶこと

  • 開発環境とは何か、なぜ必要なのかを理解する
  • Node.jsのインストール方法を習得する
  • Visual Studio Codeのインストールと基本的な使い方を習得する

2.1 開発環境って何？

「開発環境」という言葉を聞くと難しく感じるかもしれませんが、実は「プログラムを書くための道具セット」のことです。

例えば、絵を描くときには筆や絵の具、キャンバスが必要ですよね。 プログラミングでも同じように、コードを書くためのエディタ、コードを実行するための環境など、いくつかの道具が必要になります。

インストールする順番

この章では、①Node.js → ②Visual Studio Code の順にインストールします。 Node.jsには「npm」というパッケージ管理ツールが含まれており、これを使ってTypeScript本体を後でインストールします。

2.2 Node.jsをインストールしよう

Node.js（ノード・ジェイエス）は、JavaScriptやTypeScriptのプログラムをパソコン上で実行するための環境です。

Node.jsとは？

もともとJavaScriptはWebブラウザの中でしか動かせませんでした。 Node.jsは、JavaScriptをブラウザの外、つまりパソコン上で直接実行できるようにしたツールです。

TypeScriptのコードをJavaScriptに変換したり、実際に実行したりするのに使います。

インストール手順

1

公式サイトにアクセス

Webブラウザで以下のURLにアクセスします：

https://nodejs.org/

自動的に日本語ページが表示されます。英語の場合は、ページ右上の言語選択で「日本語」を選択してください。

2

LTS版をダウンロード

公式サイトには「LTS」と「Current」の2つのバージョンが表示されています。

どちらを選ぶ？

  • LTS版（推奨） - Long Term Support（長期サポート）の略。安定していて初心者におすすめ
  • Current版 - 最新機能が使えるが、変更が多い

「○○.○○.○○ LTS」と書かれた緑色のボタンをクリックしてダウンロードします。

3

インストーラーを実行

ダウンロードしたファイルをダブルクリックして、インストーラーを起動します。

インストール時のポイント

  • 基本的に「次へ」をクリックしていけばOK
  • 「Add to PATH」にチェックが入っていることを確認（通常は自動でチェック済み）
  • インストールには管理者権限が必要な場合があります

インストール確認

正しくインストールできたか確認しましょう。

Windows の場合

  1. 1. スタートメニューで「cmd」または「コマンド プロンプト」と検索
  2. 2. 「コマンド プロンプト」を起動
  3. 3. 以下のコマンドを入力してEnterキーを押す

macOS / Linux の場合

  1. 1. 「ターミナル」アプリを起動
  2. 2. 以下のコマンドを入力してEnterキーを押す
バージョン確認コマンド 
node -v

「v18.0.0」のようなバージョン番号が表示されればインストール成功です！

用語解説：コマンドプロンプト / ターミナル

コマンドプロンプト（Windows）やターミナル（macOS/Linux）は、文字だけでコンピュータに命令を出すツールです。 プログラミングでは頻繁に使うので、少しずつ慣れていきましょう。

2.3 Visual Studio Codeをインストールしよう

Visual Studio Code（ビジュアル スタジオ コード）、略してVS Codeは、 プログラムを書くための高機能なエディタです。

Visual Studio Codeとは？

Microsoft社が開発した無料のコードエディタです。TypeScriptとの相性が非常に良く、以下のような便利な機能があります：

  • コードの自動補完（入力候補を表示してくれる）
  • エラーをリアルタイムで指摘してくれる
  • 色分けされて見やすい
  • 拡張機能で機能を追加できる

インストール手順

1

公式サイトにアクセス

Webブラウザで以下のURLにアクセスします：

https://code.visualstudio.com/
2

OS版をダウンロード

ページ中央の大きな「Download for ○○」ボタンをクリックします。 自動的にお使いのOS（Windows、macOS、Linuxなど）に合ったバージョンが選ばれます。

異なるOSをダウンロードしたい場合

ボタンの下の「other downloads」リンクから選択できます。

3

インストーラーを実行

ダウンロードしたファイルをダブルクリックして、インストーラーを起動します。

インストール時の推奨設定（Windows）

  • 「PATHへの追加」にチェック
  • 「エクスプローラーのファイル コンテキスト メニューに [Code で開く] アクションを追加する」にチェック
  • 「エクスプローラーのディレクトリ コンテキスト メニューに [Code で開く] アクションを追加する」にチェック

これらの設定により、フォルダを右クリックして簡単にVS Codeで開けるようになります。

2.4 Visual Studio Codeの初期設定

VS Codeをインストールしたら、より使いやすくするために基本的な設定を行いましょう。

日本語化（必要に応じて）

VS Codeは初期状態では英語表記ですが、日本語化することができます。

  1. 1. VS Codeを起動し、左側のアクティビティバーから「拡張機能」アイコン（四角が4つ並んだマーク）をクリック
  2. 2. 検索ボックスに「Japanese Language Pack」と入力
  3. 3. 「Japanese Language Pack for Visual Studio Code」を見つけて「Install」をクリック
  4. 4. VS Codeを再起動すると日本語表示になります

推奨拡張機能のインストール

TypeScript開発を快適にする拡張機能をインストールしましょう。

ESLint

コードの品質をチェックしてくれるツール

文法的に正しくても、より良い書き方を提案してくれます

Prettier

コードを自動で整形してくれるツール

インデント（字下げ）やスペースを統一してくれます

拡張機能は後からでもOK

拡張機能のインストールは必須ではありません。まずは基本機能でTypeScriptを書き始めて、 必要に応じて後から追加していくのも良い方法です。

基本操作

フォルダを開く

プログラムを書く前に、作業用のフォルダを開きます。

  • • メニュー: 「ファイル」→「フォルダーを開く」
  • • ショートカット: Ctrl+K → Ctrl+O （macOS: Cmd+K → Cmd+O）

新しいファイルを作成

コードを書くためのファイルを作成します。

  • • サイドバーの「エクスプローラー」で右クリック → 「新しいファイル」
  • • ショートカット: Ctrl+N （macOS: Cmd+N）

ターミナルを開く

VS Code内でコマンドを実行するためのターミナルを開きます。

  • • メニュー: 「ターミナル」→「新しいターミナル」
  • • ショートカット: Ctrl+` （バッククォート）

統合ターミナルが便利

VS Code内にターミナルが統合されているので、わざわざ別のウィンドウでコマンドプロンプトやターミナルを開く必要がありません。 コードを書きながらすぐにコマンドを実行できるので、とても効率的です。

理解度確認クイズ

  1. Q1. Node.jsとは何ですか？

    回答を表示

    JavaScriptやTypeScriptのプログラムをパソコン上で実行するための環境です。 もともとブラウザの中でしか動かせなかったJavaScriptを、パソコン上で直接実行できるようにしたツールです。

  2. Q2. Node.jsをインストールする際、「LTS版」と「Current版」のどちらを選ぶべきですか？

    回答を表示

    「LTS版」を選ぶべきです。LTSはLong Term Support（長期サポート）の略で、 安定していて初心者におすすめです。Current版は最新機能が使えますが、変更が多いため学習には向きません。

  3. Q3. Visual Studio Codeの「統合ターミナル」の利点は何ですか？

    回答を表示

    VS Code内にターミナルが統合されているので、別のウィンドウでコマンドプロンプトやターミナルを開く必要がありません。 コードを書きながらすぐにコマンドを実行できるので、作業効率が大幅に向上します。

よくあるトラブルと解決方法

「node -v」を実行しても認識されない

解決方法：

  • • インストール後、コマンドプロンプト/ターミナルを再起動してください
  • • それでも認識されない場合は、パソコンを再起動してみてください
  • • PATHへの追加に失敗している可能性があるので、Node.jsの再インストールを検討してください

VS Codeが日本語化されない

解決方法：

  • • Japanese Language Packをインストール後、VS Codeを完全に終了して再起動してください
  • • Ctrl+Shift+P（macOS: Cmd+Shift+P）でコマンドパレットを開き、「Configure Display Language」と入力
  • • 「ja」を選択して再起動してください

Windows Defenderに警告される

解決方法：

  • • Node.jsやVS Codeは安全なソフトウェアです
  • • 警告が出た場合は「詳細情報」→「実行」を選択してください
  • • 公式サイトからダウンロードしたことを確認してください

基本的な型を使いこなそう

数値(number)、文字列(string)、真偽値(boolean)などの基本的な型と、それらを使った計算や文字の操作方法を、実例を通じて学びましょう。型があることで間違いを防げることを実感します。

この章で学ぶこと

  • number型での数値計算と演算子の使い方を理解する
  • string型での文字操作と便利なメソッドを習得する
  • boolean型での真偽判定と論理演算を理解する
  • 型があることで間違いを防げることを実感する

5.1 数値型（number）：計算の基本

number型は、数値を扱うための型です。TypeScriptでは、整数も小数も同じnumber型として扱います。

まずは基本的な数値の扱い方を見てみましょう：

数値型の基本 
// 整数
let age: number = 25;
let temperature: number = -5;

// 小数
let height: number = 170.5;
let pi: number = 3.14159;

// 大きな数値
let population: number = 1000000;

console.log(age);        // 25
console.log(height);     // 170.5

算術演算子

数値型では、さまざまな計算ができます。これを「算術演算」と呼びます。

演算子 意味 例 結果
+ 足し算 5 + 3 8
- 引き算 10 - 4 6
* 掛け算 6 * 7 42
/ 割り算 20 / 4 5
% 余り 10 % 3 1
** べき乗 2 ** 3 8
算術演算の例 
// 買い物の合計金額を計算
let apple: number = 150;
let banana: number = 100;
let total: number = apple + banana;
console.log("合計金額:", total, "円");  // 合計金額: 250 円

// 消費税を計算（10%）
let price: number = 1000;
let taxRate: number = 0.1;
let tax: number = price * taxRate;
console.log("消費税:", tax, "円");      // 消費税: 100 円

// 割り勘を計算
let bill: number = 3000;
let people: number = 4;
let perPerson: number = bill / people;
console.log("1人あたり:", perPerson, "円"); // 1人あたり: 750 円

演算の優先順位

計算の順序は数学と同じで、掛け算・割り算が先に実行され、その後に足し算・引き算が実行されます。 順序を変えたい場合は()を使います。
例：2 + 3 * 4は14、(2 + 3) * 4は20

5.2 文字列型（string）：テキストを扱おう

string型は、文字や文章を扱うための型です。文字列は"（ダブルクォート）、 '（シングルクォート）、または`（バッククォート）で囲みます。

文字列型の基本 
// ダブルクォート
let message1: string = "こんにちは";

// シングルクォート
let message2: string = 'TypeScript';

// バッククォート（テンプレートリテラル）
let name: string = "太郎";
let greeting: string = `こんにちは、${name}さん！`;

console.log(message1);  // こんにちは
console.log(greeting);  // こんにちは、太郎さん！

文字列の連結

文字列を結合する方法は2つあります：+演算子を使う方法と、 テンプレートリテラル（バッククォート）を使う方法です。

文字列の連結 
let firstName: string = "山田";
let lastName: string = "太郎";

// 方法1: + 演算子で連結
let fullName1: string = firstName + " " + lastName;
console.log(fullName1);  // 山田 太郎

// 方法2: テンプレートリテラル（推奨）
let fullName2: string = `${firstName} ${lastName}`;
console.log(fullName2);  // 山田 太郎

// 数値を含む場合
let age: number = 25;
let intro: string = `私は${fullName2}です。${age}歳です。`;
console.log(intro);  // 私は山田 太郎です。25歳です。

便利な文字列メソッド

文字列には、便利な操作を行うための「メソッド」が用意されています。メソッドとは、データに付随する機能のことです。

1

length

文字列の長さを取得

"TypeScript".length // 10
2

toUpperCase()

大文字に変換

"hello".toUpperCase() // HELLO
3

toLowerCase()

小文字に変換

"HELLO".toLowerCase() // hello
4

includes()

特定の文字が含まれるか

"TypeScript".includes("Script") // true
文字列メソッドの活用例 
let email: string = "user@example.com";

// 文字列の長さを確認
console.log(email.length);  // 16

// 大文字に変換
let upperEmail: string = email.toUpperCase();
console.log(upperEmail);  // USER@EXAMPLE.COM

// 特定の文字が含まれているか確認
if (email.includes("@")) {
  console.log("正しいメールアドレス形式です");
}

// 文字列の一部を取得
let domain: string = email.slice(5);  // "@example.com"
console.log(domain);

5.3 真偽値型（boolean）：はい/いいえを扱う

boolean型は、「はい」か「いいえ」の2つの値だけを持つ型です。 true（真）またはfalse（偽）のどちらかの値を取ります。

真偽値型の基本 
// 真偽値の宣言
let isRaining: boolean = true;
let isSunny: boolean = false;

console.log(isRaining);  // true
console.log(isSunny);    // false

// 実際の使用例
let hasUmbrella: boolean = true;
if (hasUmbrella) {
  console.log("傘を持っています");
}

比較演算子

比較演算子を使うと、2つの値を比較した結果として真偽値が得られます。

演算子 意味 例 結果
=== 等しい 5 === 5 true
!== 等しくない 5 !== 3 true
> より大きい 10 > 5 true
< より小さい 3 < 10 true
>= 以上 5 >= 5 true
<= 以下 3 <= 5 true
比較演算の例 
let age: number = 20;
let minAge: number = 18;

// 成人かどうかを判定
let isAdult: boolean = age >= minAge;
console.log(isAdult);  // true

// テストの合格判定
let score: number = 85;
let passingScore: number = 60;
let isPassed: boolean = score >= passingScore;
console.log(isPassed);  // true

// 文字列の比較
let password: string = "secret123";
let inputPassword: string = "secret123";
let isCorrect: boolean = password === inputPassword;
console.log(isCorrect);  // true

論理演算子

複数の条件を組み合わせる時に使う演算子です。

&&

かつ（AND）

両方ともtrueの時だけtrue

true && true // true true && false // false
||

または（OR）

どちらかがtrueならtrue

true || false // true false || false // false
!

否定（NOT）

trueとfalseを反転

!true // false !false // true
論理演算の例 
let age: number = 20;
let hasLicense: boolean = true;

// AND演算子：両方の条件を満たす
let canDrive: boolean = age >= 18 && hasLicense;
console.log(canDrive);  // true

// OR演算子：どちらかの条件を満たす
let isWeekend: boolean = false;
let isHoliday: boolean = true;
let canRest: boolean = isWeekend || isHoliday;
console.log(canRest);   // true

// NOT演算子：条件を反転
let isRaining: boolean = false;
let isSunny: boolean = !isRaining;
console.log(isSunny);   // true

5.4 型があることで間違いを防ぐ

TypeScriptの型システムは、間違った操作をしようとすると、プログラムを実行する前にエラーを教えてくれます。 これが「型安全性」と呼ばれる大きなメリットです。

間違った型の使用例

let age: number = 25;

// エラー：数値型に文字列を代入できない
age = "二十五";  // Type 'string' is not assignable to type 'number'.

let name: string = "太郎";

// エラー：文字列に数値を足すことはできない（意図しない結果になる）
let result: number = name + 100;  // Type 'string' is not assignable to type 'number'.

TypeScriptは、このような間違いをコードを書いている時点で指摘してくれます。

正しい型の使用例

let age: number = 25;

// 正しい：数値型同士の計算
let nextYearAge: number = age + 1;
console.log(nextYearAge);  // 26

let name: string = "太郎";

// 正しい：文字列同士の連結
let fullMessage: string = name + "さん";
console.log(fullMessage);  // 太郎さん

// 正しい：テンプレートリテラルで数値を文字列に含める
let introduction: string = `私は${age}歳です`;
console.log(introduction);  // 私は25歳です

型による安全性

TypeScriptの型システムは、「このデータは数値だから、数値としてしか使えない」というルールを強制します。 これにより、うっかりミスによるバグを大幅に減らすことができます。 プログラムを実行する前にエラーが分かるので、安心してコードを書き進められます。

5.5 実習：基本的な型を組み合わせて使ってみよう

これまで学んだ3つの基本的な型を組み合わせた実践的な例を見てみましょう。

実習：商品情報を管理するプログラム

オンラインショップの商品情報を扱うプログラムを作成してみましょう。

商品情報管理プログラム 
// 商品情報
let productName: string = "TypeScript入門書";
let price: number = 2980;
let inStock: boolean = true;
let quantity: number = 5;

// 消費税率
const TAX_RATE: number = 0.1;

// 税込価格を計算
let tax: number = price * TAX_RATE;
let priceWithTax: number = price + tax;

// 在庫状況のメッセージを作成
let stockMessage: string;
if (inStock) {
  stockMessage = `在庫あり（${quantity}冊）`;
} else {
  stockMessage = "在庫切れ";
}

// 商品情報を表示
console.log("=== 商品情報 ===");
console.log(`商品名: ${productName}`);
console.log(`価格: ${price}円（税抜）`);
console.log(`税込価格: ${priceWithTax}円`);
console.log(`在庫状況: ${stockMessage}`);

// 複数購入時の合計金額を計算
let buyQuantity: number = 2;
let totalPrice: number = priceWithTax * buyQuantity;
console.log(`${buyQuantity}冊の合計: ${totalPrice}円`);

実行結果

=== 商品情報 ===
商品名: TypeScript入門書
価格: 2980円（税抜）
税込価格: 3278円
在庫状況: 在庫あり（5冊）
2冊の合計: 6556円

コードのポイント解説

  • string型で商品名を管理
  • number型で価格や数量を計算
  • boolean型で在庫の有無を判定
  • テンプレートリテラルで読みやすいメッセージを作成
  • constで変更されない値（税率）を定義

理解度確認クイズ

  1. Q1. 次のうち、number型で扱えないものはどれですか？

    回答を表示

    正解：文字列（"こんにちは"など）
    number型は整数・小数・負の数を扱えますが、文字列はstring型で扱います。

  2. Q2. テンプレートリテラルで使用する記号は何ですか？

    回答を表示

    正解：バッククォート（`）
    テンプレートリテラルは ` で囲み、${} 内に変数や式を埋め込むことができます。

  3. Q3. 「age >= 20 && hasLicense === true」はどういう意味ですか？

    回答を表示

    正解：「年齢が20以上」かつ「ライセンスを持っている」
    &&演算子は両方の条件が真の時だけ、全体が真になります。

この章のまとめ

  • number型は数値を扱い、算術演算子（+、-、*、/など）で計算ができる
  • string型は文字列を扱い、テンプレートリテラルで変数を埋め込める
  • boolean型はtrue/falseの2つの値を持ち、条件判定に使われる
  • 比較演算子（===、!==、>、<など）で値を比較し、真偽値を得られる
  • 論理演算子（&&、||、!）で複数の条件を組み合わせられる
  • 型があることで、間違った操作を実行前に検出できる

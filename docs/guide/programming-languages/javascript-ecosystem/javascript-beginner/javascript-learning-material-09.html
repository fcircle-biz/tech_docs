Webページを動かそう（DOM操作の基礎）

JavaScriptでWebページの要素を取得・変更・追加する方法を学び、動的なWebページを作る基礎を身につけます。

この章で学ぶこと

  • DOMとは何か、Webページがどう構成されているかを理解する
  • JavaScriptでWebページの要素を取得する方法を学ぶ
  • 要素の内容やスタイルを変更する方法を習得する
  • 新しい要素を追加・削除する基本操作をマスターする

9.1 DOMとは何か

DOM（ドム、Document Object Modelの略）とは、 Webページの構造を「木」のような形で表したものです。 JavaScriptは、このDOMを通じてWebページの要素にアクセスし、変更することができます。

木の構造で考えてみよう

家系図を想像してください。一番上に祖父母がいて、その下に親がいて、その下に子どもがいます。 Webページも同じように、親要素の中に子要素がある、という階層構造になっています。 この構造全体をDOMと呼びます。

簡単なHTMLとそのDOM構造：

HTML 
<!DOCTYPE html>
<html>
  <head>
    <title>私のページ</title>
  </head>
  <body>
    <h1>こんにちは</h1>
    <p>これは段落です</p>
  </body>
</html>

このHTMLのDOM構造を図で表すと：

graph TD A["html"] --> B["head"] A --> C["body"] B --> D["title"] D --> E["私のページ"] C --> F["h1"] C --> G["p"] F --> H["こんにちは"] G --> I["これは段落です"]

JavaScriptは、このDOMの木構造をたどって、特定の要素を見つけたり、内容を変更したりすることができます。

9.2 要素を取得する方法

Webページの要素を変更するには、まずその要素を取得する必要があります。 よく使われる取得方法を見ていきましょう。

1 getElementById（アイディーで取得）

id属性を指定して要素を取得します。idはページ内で1つだけの目印です。

HTML 
<p id="message">こんにちは</p>
JavaScript 
// id="message"の要素を取得
const element = document.getElementById('message');
console.log(element); // <p id="message">こんにちは</p> が表示される

2 querySelector（セレクターで取得）

CSSのセレクター（選択方法）を使って要素を取得します。最初に見つかった1つ目の要素だけを取得します。

HTML 
<div class="box">ボックス1</div>
<div class="box">ボックス2</div>
JavaScript 
// class="box"の最初の要素を取得
const box = document.querySelector('.box');
console.log(box); // <div class="box">ボックス1</div> が表示される

3 querySelectorAll（全部取得）

条件に合う要素をすべて取得します。結果は配列のようなリストになります。

JavaScript 
// class="box"の要素をすべて取得
const boxes = document.querySelectorAll('.box');
console.log(boxes.length); // 2 が表示される（2つある）

// for文で1つずつ処理できる
for (let i = 0; i < boxes.length; i++) {
  console.log(boxes[i]);
}

どれを使えばいい？

  • 1つだけ取得したい → getElementById または querySelector
  • 複数まとめて取得したい → querySelectorAll
  • 迷ったら → querySelector系が万能で便利です

9.3 要素の内容を変更する

取得した要素の中身（テキスト）を変更してみましょう。

textContent（テキストを変更）

textContentプロパティを使うと、要素の中のテキストを読み取ったり、書き換えたりできます。

JavaScript 
// id="message"の要素を取得
const message = document.getElementById('message');

// 今の内容を確認
console.log(message.textContent); // 「こんにちは」

// 内容を変更
message.textContent = 'こんばんは';
// → 画面の表示が「こんばんは」に変わる！

innerHTML（HTML込みで変更）

innerHTMLプロパティを使うと、HTMLタグごと書き換えることができます。

JavaScript 
// id="content"の要素を取得
const content = document.getElementById('content');

// HTMLタグごと変更（太字と改行を含む）
content.innerHTML = '<strong>重要</strong><br>お知らせです';
// → 「重要」が太字で表示され、改行される

注意：textContent と innerHTML の使い分け

textContent: 単純にテキストだけを変更したいとき。安全です。
innerHTML: HTMLタグを含めて変更したいとき。ただし、ユーザーからの入力をそのまま入れると危険な場合があるので注意。

9.4 スタイル（見た目）を変更する

要素の色やサイズなど、見た目を変更することもできます。

JavaScript 
// id="box"の要素を取得
const box = document.getElementById('box');

// 背景色を変更
box.style.backgroundColor = 'lightblue';

// 文字色を変更
box.style.color = 'navy';

// 文字サイズを変更
box.style.fontSize = '24px';

// 幅と高さを変更
box.style.width = '200px';
box.style.height = '100px';

CSSプロパティの書き方の違い

CSS では background-color と書きますが、 JavaScript では backgroundColor のように、ハイフンを取ってキャメルケース（単語の最初を大文字にする書き方）にします。

9.5 要素を追加・削除する

JavaScriptで新しい要素を作成したり、不要な要素を削除したりできます。

要素を作成して追加する

JavaScript 
// 新しいp要素を作成
const newParagraph = document.createElement('p');

// 作成した要素にテキストを設定
newParagraph.textContent = 'これは新しい段落です';

// id="container"の要素に追加
const container = document.getElementById('container');
container.appendChild(newParagraph);
// → 画面に新しい段落が表示される！

要素を削除する

JavaScript 
// 削除したい要素を取得
const oldElement = document.getElementById('old');

// 要素を削除
oldElement.remove();
// → 画面から要素が消える！

実習：カウンターを作ろう

ボタンをクリックすると数字が増えるカウンターを作ってみましょう。

HTML 
<!DOCTYPE html>
<html>
<body>
  <h1 id="count">0</h1>
  <button onclick="increment()">+1する</button>

  <script>
    function increment() {
      // 現在の数字を取得
      const countElement = document.getElementById('count');
      const currentCount = parseInt(countElement.textContent);

      // 1増やす
      const newCount = currentCount + 1;

      // 画面に表示
      countElement.textContent = newCount;
    }
  </script>
</body>
</html>

試してみよう：

  • 「-1する」ボタンも追加してみましょう
  • 数字が10を超えたら色を変えてみましょう

第9章のまとめ

  • DOMは、Webページの構造を木のように表したもの
  • 要素の取得にはgetElementById, querySelector, querySelectorAllを使う
  • textContentでテキストを変更、innerHTMLでHTMLごと変更できる
  • styleプロパティで見た目（色、サイズなど）を変更できる
  • createElementとappendChildで要素を追加、removeで削除できる

理解度確認クイズ

  1. Q1. DOMとは何ですか？簡単に説明してください。

    回答を表示

    DOMとは、Webページの構造を木のような階層で表したものです。 JavaScriptはこのDOMを通じて、Webページの要素にアクセスしたり変更したりできます。

  2. Q2. id="title"の要素を取得するコードを書いてください。

    回答を表示

    const title = document.getElementById('title');
    または
    const title = document.querySelector('#title');

  3. Q3. textContent と innerHTML の違いは何ですか？

    回答を表示

    textContent: 要素の中のテキストだけを扱います。HTMLタグは無視されます。
    innerHTML: HTMLタグを含めて要素の中身を扱います。HTMLタグも解釈されて表示されます。

  4. Q4. 要素の背景色を赤色に変更するコードを書いてください（要素は変数 element に入っているとします）。

    回答を表示

    element.style.backgroundColor = 'red';

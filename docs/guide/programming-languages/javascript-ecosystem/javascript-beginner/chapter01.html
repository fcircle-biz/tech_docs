プログラミングとは何か

1.1 プログラミングって何？

プログラミングとは、コンピューターに指示を出すことです。 料理のレシピのように、コンピューターに「何を」「どのように」「いつ」やるかを教えることです。

例えば：

  • 「画面に"こんにちは"と表示して」
  • 「1から10まで数えて」
  • 「ボタンが押されたら音を出して」

これらの指示を、コンピューターが理解できる言葉（プログラミング言語）で書いたものがプログラムです。

1.2 JavaScriptって何？

JavaScriptは、Webページを動かすためのプログラミング言語です。 YouTubeの動画プレイヤー、Googleの検索機能、SNSのいいねボタンなど、 Web上で動いているほぼすべてのものにJavaScriptが使われています。

JavaScriptの特徴：

  • 🌐 ブラウザでそのまま動く
  • 📱 パソコンでもスマホでも動く
  • 🔧 特別なソフトをインストールしなくても始められる
  • 🎨 ゲームやアニメーションも作れる

最初のプログラムを書いてみよう

1.3 開発者ツールを開いてみよう

まずは、ブラウザの開発者ツールを開いてみましょう。これがあなたのプログラミングの練習場所になります。

実習 1-1: 開発者ツールを開く
  1. Google ChromeやFirefoxを開く
  2. キーボードでF12キーを押す
  3. 画面の下や右に開発者ツールが表示される
  4. 「Console」タブをクリック

できましたか？ これがあなたのプログラミング環境です！

1.4 console.logを使ってみよう

console.log()
は、プログラムからメッセージを表示する基本的な方法です。 コンソールに入力して、実際に動かしてみましょう。

実習 1-2: 初めてのメッセージ

開発者ツールのコンソールに以下を入力して、Enterキーを押してください：

console.log("Hello, JavaScript!");

メッセージが表示されましたか？おめでとうございます！これがあなたの最初のJavaScriptプログラムです。

他にも試してみましょう：

console.log("私の名前は太郎です");
console.log("今日は良い天気ですね");
console.log("プログラミングは楽しいです");

1.5 alert()を使ってみよう

alert()
は、ポップアップウィンドウでメッセージを表示します。 ユーザーに重要な情報を伝えるときに使います。

実習 1-3: アラートの表示

コンソールに以下を入力してください：

alert("こんにちは、JavaScript!");

ポップアップが表示されましたか？「OK」ボタンを押すと消えます。

実践プロジェクト：自己紹介プログラム

🎯 プロジェクト: 自己紹介プログラムを作ろう

あなたの情報を使って、自己紹介プログラムを作成しましょう。

ステップ1: 基本的な自己紹介

以下のコードを参考に、あなたの情報に書き換えてください：

console.log("自己紹介をします");
console.log("私の名前は田中太郎です");
console.log("年齢は25歳です");
console.log("趣味は読書とプログラミングです");
ステップ2: アラートも追加

最後にアラートも追加してみましょう：

alert("自己紹介が完了しました！");
完成形

すべてをまとめると、このようになります：

console.log("自己紹介をします");
console.log("私の名前は田中太郎です");
console.log("年齢は25歳です");
console.log("趣味は読書とプログラミングです");
alert("自己紹介が完了しました！");
理解度確認クイズ
  1. コンソールにメッセージを表示するために使う関数は何ですか？
  2. ポップアップウィンドウを表示するために使う関数は何ですか？
  3. 開発者ツールを開くキーボードショートカットは何ですか？
  4. JavaScriptが主に使われる場所はどこですか？
答え
  1. console.log()
  2. alert()
  3. F12キー
  4. Webページ・Webアプリケーション

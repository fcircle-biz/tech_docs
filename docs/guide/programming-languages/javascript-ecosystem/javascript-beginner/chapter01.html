プログラミングとは何か

1.1 プログラミングって何？

プログラミングとは、コンピューターに指示を出すことです。 料理のレシピのように、コンピューターに「何を」「どのように」「いつ」やるかを教えることです。

例えば：

  • 「画面に"こんにちは"と表示して」
  • 「1から10まで数えて」
  • 「ボタンが押されたら音を出して」

これらの指示を、コンピューターが理解できる言葉（プログラミング言語）で書いたものがプログラムです。

1.2 JavaScriptって何？

JavaScriptは、Webページを動かすためのプログラミング言語です。 YouTubeの動画プレイヤー、Googleの検索機能、SNSのいいねボタンなど、 Web上で動いているほぼすべてのものにJavaScriptが使われています。

JavaScriptの特徴：

  • 🌐 ブラウザでそのまま動く
  • 📱 パソコンでもスマホでも動く
  • 🔧 特別なソフトをインストールしなくても始められる
  • 🎨 ゲームやアニメーションも作れる

最初のプログラムを書いてみよう

1.3 開発者ツールを開いてみよう

まずは、ブラウザの開発者ツールを開いてみましょう。これがあなたのプログラミングの練習場所になります。

実習 1-1: 開発者ツールを開く
  1. Google ChromeやFirefoxを開く
  2. キーボードでF12キーを押す
  3. 画面の下や右に開発者ツールが表示される
  4. 「Console」タブをクリック

できましたか？ これがあなたのプログラミング環境です！

1.4 console.logを使ってみよう

console.log()
は、プログラムからメッセージを表示する基本的な方法です。 コンソールに入力して、実際に動かしてみましょう。

実習 1-2: 初めてのメッセージ

開発者ツールのコンソールに以下を入力して、Enterキーを押してください：

console.log("Hello, JavaScript!");

メッセージが表示されましたか？おめでとうございます！これがあなたの最初のJavaScriptプログラムです。

他にも試してみましょう：

console.log("私の名前は太郎です");
console.log("今日は良い天気ですね");
console.log("プログラミングは楽しいです");

1.5 alert()を使ってみよう

alert()
は、ポップアップウィンドウでメッセージを表示します。 ユーザーに重要な情報を伝えるときに使います。

実習 1-3: アラートの表示

コンソールに以下を入力してください：

alert("こんにちは、JavaScript!");

ポップアップが表示されましたか？「OK」ボタンを押すと消えます。

1.6 コメントを書いてみよう

コメントは、プログラムの中に書く「メモ」や「説明」です。 コンピューターはコメントを無視するので、プログラムの動作には影響しません。 自分や他の人が後でコードを見たときに理解しやすくするために使います。

コメントの種類：

① 単一行コメント（//）

1行だけのコメントを書くときに使います。//の後ろに書いた文字は、すべてコメントになります。

// これはコメントです
console.log("こんにちは");  // この行の後ろにもコメントが書けます

② 複数行コメント（/* */）

複数行にわたるコメントを書くときに使います。/*で始めて*/で終わります。

/*
  これは複数行の
  コメントです。
  何行でも書けます。
*/
console.log("プログラムが実行されます");
実習 1-4: コメントを使ってみよう

コンソールに以下のコードを入力して、実行してみましょう：

// 自己紹介プログラム
console.log("私の名前は太郎です");  // 名前を表示

/*
  次は年齢と趣味を表示します
*/
console.log("年齢は20歳です");
console.log("趣味はプログラミングです");

コメント部分は実行されず、console.logの3行だけが実行されることを確認してください。

💡 コメントを書く理由
  • 後で自分がコードを見返したときに、何をしていたか思い出しやすい
  • 他の人がコードを読むときに、理解しやすくなる
  • 複雑な処理の説明を書ける
  • 一時的にコードを無効化したいとき（デバッグ時など）に使える

実践プロジェクト：自己紹介プログラム

🎯 プロジェクト: 自己紹介プログラムを作ろう

あなたの情報を使って、自己紹介プログラムを作成しましょう。

ステップ1: 基本的な自己紹介

以下のコードを参考に、あなたの情報に書き換えてください：

console.log("自己紹介をします");
console.log("私の名前は田中太郎です");
console.log("年齢は25歳です");
console.log("趣味は読書とプログラミングです");
ステップ2: アラートも追加

最後にアラートも追加してみましょう：

alert("自己紹介が完了しました！");
完成形（コメント付き）

学んだコメントも活用して、このようになります：

// 自己紹介プログラム
// 作成日：2025年1月

console.log("自己紹介をします");
console.log("私の名前は田中太郎です");  // 名前を表示
console.log("年齢は25歳です");  // 年齢を表示
console.log("趣味は読書とプログラミングです");  // 趣味を表示

/*
  すべての情報を表示した後、
  完了メッセージを出します
*/
alert("自己紹介が完了しました！");

コメントを使うと、後で見返したときにコードが理解しやすくなります！

理解度確認クイズ

  2. ポップアップウィンドウを表示するために使う関数は何ですか？
  3. 開発者ツールを開くキーボードショートカットは何ですか？
  4. JavaScriptが主に使われる場所はどこですか？
  5. 1行だけのコメントを書くときに使う記号は何ですか？
  6. 複数行のコメントを書くときに使う記号は何ですか？
答え
  1. console.log()
  2. alert()
  3. F12キー
  4. Webページ・Webアプリケーション

補足：フロントエンドとバックエンドって何？

フロントエンドとは

フロントエンドは、ユーザーが直接見て操作する部分です。 ブラウザ上で動くすべてのものがフロントエンドに含まれます。

フロントエンドの例：

  • 📄 Webページのデザインや見た目（HTML/CSS）
  • 🖱️ ボタンをクリックしたときの動き（JavaScript）
  • 🎨 画像やアニメーションの表示
  • 📱 スマホ画面への対応（レスポンシブデザイン）
  • ✅ フォームの入力チェック（メールアドレスの形式など）

フロントエンドで使う主な技術：

  • HTML：Webページの構造を作る
  • CSS：見た目やデザインを整える
  • JavaScript：動きや機能を追加する ← 今回学ぶのはこれ！

バックエンドとは

バックエンドは、サーバー側で動く、ユーザーからは見えない裏側の処理です。 データの保存、計算、認証など、重要な処理を担当します。

バックエンドの例：

  • 💾 データベースへのデータ保存・取得
  • 🔐 ログイン認証（パスワードチェック）
  • 💳 決済処理
  • 📧 メール送信
  • 🔢 複雑な計算処理
  • 🗂️ ファイルのアップロード・管理

バックエンドで使う主な技術：

  • サーバー言語：Java, Python, PHP, Ruby, Node.js（JavaScript）など
  • データベース：MySQL, PostgreSQL, MongoDBなど
  • サーバー：Apache, Nginxなど

実は、JavaScriptはフロントエンドとバックエンドの両方で使えます！

💡 初心者はまずフロントエンドから！

このガイドでは、まずフロントエンドのJavaScriptから学びます。 ブラウザだけあれば動かせるので、初心者に最適です。 将来的に、フロントエンドを理解してからバックエンドを学ぶと、Webアプリ全体を作れるようになります！

実際のWebサイトの動き

例えば、Amazonで商品を買うときの流れを見てみましょう：

  1. 【フロントエンド】 商品ページを表示（HTML/CSS/JavaScript）
  2. 【フロントエンド】 「カートに追加」ボタンをクリック（JavaScript）
  3. 【バックエンド】 サーバーに商品情報を送信
  4. 【バックエンド】 データベースにカート情報を保存
  5. 【フロントエンド】 「カートに追加しました」と画面に表示（JavaScript）
  6. 【フロントエンド】 購入手続き画面を表示（HTML/CSS/JavaScript）
  7. 【バックエンド】 決済処理を実行
  8. 【バックエンド】 注文情報をデータベースに保存
  9. 【バックエンド】 確認メールを送信
  10. 【フロントエンド】 「ご注文ありがとうございました」と表示

このように、フロントエンドとバックエンドが協力して、1つのWebアプリケーションを作っています。

第3章: 条件によって処理を変えよう

🎯 この章の実践プロジェクト

じゃんけんゲームを作成します

条件分岐を使って、プレイヤーの選択とコンピューターの選択を比較し、勝敗を判定するゲームを作りましょう！

この章で学ぶこと
  • if文の基本
  • 比較演算子
  • else、else if文
  • 条件の組み合わせ

条件分岐って何？

3.1 条件分岐とは

条件分岐とは、状況に応じて異なる処理を実行することです。 日常生活でも、いつも条件分岐を使っています。

例えば：

  • 「雨が降っていたら傘を持つ」
  • 「テストの点数が80点以上なら合格」
  • 「お金が足りなければ買い物を諦める」

プログラミングでも同じように、条件によって処理を変えることができます。

3.2 if文の基本

JavaScriptでは

if
文を使って条件分岐を書きます。 

if (条件) {
    // 条件が真の場合に実行される処理
}
実習 3-1: 最初のif文

開発者ツールのコンソールに以下を入力してください：

let weather = "雨";

if (weather === "雨") {
    console.log("傘を持って出かけよう");
}

「傘を持って出かけよう」と表示されましたか？

比較演算子を覚えよう

3.3 比較演算子とは

比較演算子は、2つの値を比較するための記号です。 結果は

true
（真）または
false
（偽）になります。

演算子 意味 例 結果
=== 等しい 5 === 5 true
!== 等しくない 5 !== 3 true
> より大きい 8 > 5 true
< より小さい 3 < 7 true
>= 以上 10 >= 10 true
<= 以下 5 <= 8 true
実習 3-2: 比較演算子を試す
let score = 85;

if (score >= 80) {
    console.log("合格です！");
}

if (score < 60) {
    console.log("不合格です...");
}

3.4 else文を使おう

else
文を使うと、条件が偽の場合の処理を書くことができます。

実習 3-3: else文
let age = 17;

if (age >= 20) {
    console.log("成人です");
} else {
    console.log("未成年です");
}

3.5 else if文で複数の条件

else if
文を使うと、複数の条件を順番にチェックできます。

実習 3-4: else if文
let score = 75;

if (score >= 90) {
    console.log("優秀！");
} else if (score >= 80) {
    console.log("良好");
} else if (score >= 70) {
    console.log("普通");
} else {
    console.log("頑張ろう");
}

条件を組み合わせよう

3.6 論理演算子

論理演算子を使うと、複数の条件を組み合わせることができます。

演算子 意味 例
&& AND（かつ） age >= 20 && age < 65
|| OR（または） weather === "雨" || weather === "雪"
! NOT（〜でない） !isWeekend
実習 3-5: 論理演算子
let age = 25;
let hasLicense = true;

if (age >= 18 && hasLicense) {
    console.log("車を運転できます");
} else {
    console.log("車を運転できません");
}

let weather = "雨";
if (weather === "雨" || weather === "雪") {
    console.log("悪天候です");
}

実践プロジェクト：じゃんけんゲーム

🎯 プロジェクト: じゃんけんゲームを作ろう

条件分岐を使って、じゃんけんゲームを作成しましょう。

ステップ1: 基本的な勝敗判定

まずは、固定された手での勝敗判定を作ります：

let player = "グー";
let computer = "チョキ";

console.log("=== じゃんけんゲーム ===");
console.log(`あなた: ${player}`);
console.log(`コンピューター: ${computer}`);

if (player === computer) {
    console.log("結果: あいこ");
} else if (
    (player === "グー" && computer === "チョキ") ||
    (player === "チョキ" && computer === "パー") ||
    (player === "パー" && computer === "グー")
) {
    console.log("結果: あなたの勝ち！");
} else {
    console.log("結果: あなたの負け...");
}
ステップ2: ランダムな手を生成

コンピューターがランダムに手を選ぶようにします：

let choices = ["グー", "チョキ", "パー"];
let randomIndex = Math.floor(Math.random() * 3);
let computer = choices[randomIndex];

let player = "グー";  // 好きな手に変更してください

console.log("=== じゃんけんゲーム ===");
console.log(`あなた: ${player}`);
console.log(`コンピューター: ${computer}`);

if (player === computer) {
    console.log("結果: あいこ");
} else if (
    (player === "グー" && computer === "チョキ") ||
    (player === "チョキ" && computer === "パー") ||
    (player === "パー" && computer === "グー")
) {
    console.log("結果: あなたの勝ち！🎉");
} else {
    console.log("結果: あなたの負け...😢");
}
ステップ3: 改良版じゃんけんゲーム

より詳しい結果表示を追加しましょう：

function playRockPaperScissors(playerChoice) {
    let choices = ["グー", "チョキ", "パー"];
    let randomIndex = Math.floor(Math.random() * 3);
    let computer = choices[randomIndex];
    
    console.log("=== じゃんけんゲーム ===");
    console.log(`あなた: ${playerChoice}`);
    console.log(`コンピューター: ${computer}`);
    
    if (playerChoice === computer) {
        console.log("結果: あいこ 🤝");
        console.log("もう一度チャレンジ！");
    } else if (
        (playerChoice === "グー" && computer === "チョキ") ||
        (playerChoice === "チョキ" && computer === "パー") ||
        (playerChoice === "パー" && computer === "グー")
    ) {
        console.log("結果: あなたの勝ち！🎉");
        console.log("おめでとうございます！");
    } else {
        console.log("結果: あなたの負け... 😢");
        console.log("次回は頑張りましょう！");
    }
}

// ゲームを実行（好きな手を選んでください）
playRockPaperScissors("グー");
理解度確認クイズ
  1. 条件分岐を書くために使う文は何ですか？
  2. 「等しい」を表す比較演算子は何ですか？
  3. 「かつ」を表す論理演算子は何ですか？
  4. 条件が偽の場合の処理を書くために使う文は何ですか？
  5. 複数の条件を順番にチェックするために使う文は何ですか？
答え
  1. if文
  2. ===
  3. &&
  4. else文
  5. else if文

第5章

条件によって処理を変えよう（if文）

「もし〜なら」という条件分岐を使って、状況に応じて異なる処理を行う方法を、日常生活の判断に例えながら学習します。

この章で学ぶこと

  • if文とは何か、どのような場面で使うのかを理解する
  • 比較演算子を使って条件を書く方法を学ぶ
  • if、if-else、if-else if-elseの使い分けを理解する
  • 論理演算子を使って複雑な条件を書けるようになる

5.1 if文とは

if文とは、「もし〜ならば、〇〇をする」という条件によって処理を変えるための命令文です。 私たちは日常生活の中で、無意識にたくさんの条件判断をしています。

日常生活での条件判断

  • もし雨が降っていたら、傘を持って出かける
  • もし信号が青なら、横断歩道を渡る
  • もし時間が8時を過ぎていたら、急いで家を出る

プログラムでも同じように、条件によって処理を変えることができます。

JavaScriptでは、この「もし〜ならば」という条件判断をif文で表現します。 if文を使うと、状況に応じてプログラムの動きを変えることができるようになります。

5.2 if文の基本的な書き方

if文の基本的な書き方は次のようになります。

if文の基本形 
if (条件) {
    // 条件が成り立つときに実行される処理
}

if文の書き方のポイント

  • ifというキーワードの後に、半角の丸カッコ()で条件を囲みます
  • 条件が成り立つときに実行したい処理を、波カッコ{}で囲みます
  • 波カッコの中の処理は、インデント（字下げ）をして見やすくします

では、実際の例を見てみましょう。

example1.js 
// 年齢を変数に入れる
let age = 20;

// もし年齢が18以上なら、メッセージを表示
if (age >= 18) {
    console.log("あなたは成人です");
}

このプログラムでは、変数ageが18以上の場合に、 「あなたは成人です」というメッセージが表示されます。 もしageが17以下なら、何も表示されません。

5.3 比較演算子

if文の条件を書くときに使う比較演算子について学びましょう。 比較演算子は、2つの値を比べて、その結果が「正しい（true）」か「正しくない（false）」かを判断します。

主な比較演算子

演算子 意味 例 結果
=== 等しい 5 === 5 true（正しい）
!== 等しくない 5 !== 3 true（正しい）
> より大きい 10 > 5 true（正しい）
< より小さい 3 < 5 true（正しい）
>= 以上 5 >= 5 true（正しい）
<= 以下 3 <= 5 true（正しい）

注意: ==と===の違い

JavaScriptには==と===の2種類の等価演算子があります。 ===は値だけでなく型も厳密に比較するため、より安全です。 初心者のうちは必ず===を使うようにしましょう。

比較演算子の例 
// 点数を変数に入れる
let score = 85;

// もし点数が80より大きければ、「優秀です」と表示
if (score > 80) {
    console.log("優秀です");
}

// もし点数が60以上なら、「合格です」と表示
if (score >= 60) {
    console.log("合格です");
}

5.4 if-else文（条件が成り立たない場合の処理）

if-else文を使うと、「もし〜ならばAをする、そうでなければBをする」という2つの選択肢を書くことができます。

if-else文の基本形 
if (条件) {
    // 条件が成り立つときの処理
} else {
    // 条件が成り立たないときの処理
}

実際の例を見てみましょう。

example2.js 
// 天気を変数に入れる
let weather = "晴れ";

// もし天気が「雨」なら傘を持つ、そうでなければ傘は不要
if (weather === "雨") {
    console.log("傘を持っていきましょう");
} else {
    console.log("傘は不要です");
}
flowchart TD Start([プログラム開始]) Start --> Check{"天気 === 雨?"} Check -->|はい| Rain["傘を持っていきましょう"] Check -->|いいえ| NoRain["傘は不要です"] Rain --> End([プログラム終了]) NoRain --> End

日常生活に例えると

「もし時間があるなら映画を見に行く、そうでなければ家でゆっくりする」という判断と同じです。 if-else文を使うと、どちらか一方の処理が必ず実行されます。

5.5 if-else if-else文（3つ以上の条件分岐）

3つ以上の選択肢がある場合は、else ifを使って条件を追加できます。

if-else if-else文の基本形 
if (条件1) {
    // 条件1が成り立つときの処理
} else if (条件2) {
    // 条件1は成り立たないが、条件2が成り立つときの処理
} else {
    // どの条件も成り立たないときの処理
}

実際の例として、点数に応じて成績を判定するプログラムを作ってみましょう。

example3.js 
// テストの点数
let score = 75;

// 点数に応じて成績を判定
if (score >= 80) {
    console.log("成績: A（優秀です）");
} else if (score >= 60) {
    console.log("成績: B（合格です）");
} else {
    console.log("成績: C（もう少し頑張りましょう）");
}
flowchart TD Start([プログラム開始]) Start --> Check1{"score >= 80?"} Check1 -->|はい| A["成績: A"] Check1 -->|いいえ| Check2{"score >= 60?"} Check2 -->|はい| B["成績: B"] Check2 -->|いいえ| C["成績: C"] A --> End([プログラム終了]) B --> End C --> End

条件の順序が大切

if-else if-else文では、上から順番に条件をチェックします。 最初に成り立った条件の処理だけが実行され、それ以降の条件はチェックされません。 そのため、条件の順序を間違えると、期待通りに動かないことがあります。

5.6 論理演算子（複数の条件を組み合わせる）

論理演算子を使うと、複数の条件を組み合わせて、より複雑な判断ができるようになります。

主な論理演算子

1 AND演算子（かつ）: &&

両方の条件が成り立つときにtrueになります。

age >= 18 && hasLicense === true

→「年齢が18以上」かつ「免許を持っている」

2 OR演算子（または）: ||

どちらか一方の条件が成り立つときにtrueになります。

day === "土曜日" || day === "日曜日"

→「土曜日」または「日曜日」

3 NOT演算子（否定）: !

条件の結果を反対にします。trueをfalseに、falseをtrueにします。

!(score < 60)

→「点数が60未満ではない」（つまり60以上）

実際の例を見てみましょう。

example4.js 
// 年齢と免許の有無
let age = 20;
let hasLicense = true;

// 年齢が18以上かつ免許を持っている場合
if (age >= 18 && hasLicense) {
    console.log("車を運転できます");
} else {
    console.log("車を運転できません");
}

// 土日の判定
let day = "土曜日";

// 土曜日または日曜日の場合
if (day === "土曜日" || day === "日曜日") {
    console.log("週末です。ゆっくり休みましょう");
} else {
    console.log("平日です。頑張りましょう");
}

実習 5-1: 年齢判定プログラムを作ろう

ユーザーの年齢を変数に入れて、年齢に応じたメッセージを表示するプログラムを作成しましょう。

手順

  1. 1 HTMLファイルを作成し、<script>タグの中にJavaScriptを書きます
  2. 2 変数ageを作成し、好きな年齢を入れます
  3. 3 if-else if-else文を使って、以下の条件で判定します：
    - 20歳以上: 「成人です」
    - 13歳以上20歳未満: 「中高生です」
    - 13歳未満: 「小学生以下です」
  4. 4 ブラウザで開き、コンソールに正しいメッセージが表示されるか確認します
  5. 5 年齢を色々な値に変えて、動作を確認してみましょう
解答例を表示 
let age = 15;

if (age >= 20) {
    console.log("成人です");
} else if (age >= 13) {
    console.log("中高生です");
} else {
    console.log("小学生以下です");
}

実習 5-2: 入場料金計算プログラム

遊園地の入場料金を計算するプログラムを作成しましょう。 年齢と学生かどうかで料金が変わります。

料金設定

  • 18歳以上で学生ではない: 3000円
  • 18歳以上で学生: 2000円（学割）
  • 12歳以上18歳未満: 1500円（中高生料金）
  • 12歳未満: 500円（子供料金）

手順

  1. 1 変数ageとisStudentを作成します
  2. 2 if-else if-else文と論理演算子を使って、料金を判定します
  3. 3 判定結果をコンソールに表示します
解答例を表示 
let age = 20;
let isStudent = true;

if (age >= 18 && !isStudent) {
    console.log("入場料金: 3000円");
} else if (age >= 18 && isStudent) {
    console.log("入場料金: 2000円（学割）");
} else if (age >= 12) {
    console.log("入場料金: 1500円（中高生料金）");
} else {
    console.log("入場料金: 500円（子供料金）");
}

第5章のまとめ

  • if文は、条件によって処理を変えるための命令文
  • 比較演算子（===、!==、>、<、>=、<=）で条件を書く
  • if-else文で「〜ならAをする、そうでなければBをする」という2択を表現
  • if-else if-else文で3つ以上の選択肢を表現
  • 論理演算子（&&、||、!）で複数の条件を組み合わせられる
  • 条件の順序が大切。上から順番にチェックされる

理解度確認クイズ

  1. Q1. 次のコードで、コンソールに何が表示されますか？

    let temperature = 25;

if (temperature > 30) {
    console.log("暑い");
} else {
    console.log("涼しい");
}
    回答を表示

    「涼しい」が表示されます。
    temperature（25）は30より大きくないため、else節が実行されます。

  2. Q2. 「年齢が20以上かつ免許を持っている」という条件を正しく書いているのはどれですか？

    A) age >= 20 || hasLicense === true

    B) age >= 20 && hasLicense === true

    C) age > 20 && hasLicense === true

    回答を表示

    正解: B
    「かつ」という条件なので、AND演算子（&&）を使います。 また、「20以上」なので >= を使います。

  3. Q3. 次のコードで、score が 75 のとき、何が表示されますか？

    let score = 75;

if (score >= 90) {
    console.log("A");
} else if (score >= 70) {
    console.log("B");
} else if (score >= 60) {
    console.log("C");
} else {
    console.log("D");
}
    回答を表示

    「B」が表示されます。
    75は90以上ではないので最初の条件は不成立。
    75は70以上なので、2番目の条件が成立し、「B」が表示されます。
    それ以降の条件はチェックされません。

  4. Q4. === と == の違いについて、正しい説明はどれですか？

    A) どちらも同じなので、どちらを使っても良い

    B) === は値と型の両方を厳密に比較する。初心者は === を使うべき

    C) == の方が厳密に比較する

    回答を表示

    正解: B
    === は値だけでなく型も厳密に比較するため、より安全です。 初心者のうちは必ず === を使うようにしましょう。

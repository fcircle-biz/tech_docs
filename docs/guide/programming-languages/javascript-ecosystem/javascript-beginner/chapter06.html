第6章: Webページを操作しよう

JavaScriptでWebページを操作する

6.1 DOM（Document Object Model）とは

DOMとは、HTMLの要素をJavaScriptから操作するための仕組みです。 WebページのボタンやテキストなどをJavaScriptから自由に変更できます。

できることの例：

  • 📝 テキストの内容を変更
  • 🎨 色やスタイルを変更
  • ➕ 新しい要素を追加
  • ❌ 要素を削除
  • 🖱️ クリックなどのイベントに反応

6.2 HTML要素を取得する

JavaScriptでHTML要素を操作するには、まず要素を「取得」する必要があります。

方法 説明 例
getElementById IDで要素を取得 document.getElementById('myId')
getElementsByClassName クラス名で要素を取得 document.getElementsByClassName('myClass')
querySelector CSSセレクターで要素を取得 document.querySelector('.myClass')
デモ用HTML

以下のHTMLコードを使って、実際に要素を操作してみましょう：

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>JavaScript DOM練習</title>
</head>
<body>
    <h1 id="title">ようこそ！</h1>
    <p id="message">ここにメッセージが表示されます</p>
    <button id="changeButton">クリックして変更</button>
    
    <script>
        // ここにJavaScriptコードを書きます
    </script>
</body>
</html>
実習 6-1: 要素を取得する

上記のHTMLを使って、以下のJavaScriptコードを実行してみましょう：

// 要素を取得
let titleElement = document.getElementById('title');
let messageElement = document.getElementById('message');
let buttonElement = document.getElementById('changeButton');

// 取得した要素を確認
console.log(titleElement);
console.log(messageElement);
console.log(buttonElement);

6.3 要素の内容を変更する

取得した要素の内容を変更してみましょう。

実習 6-2: テキストを変更する
// 要素の内容を変更
let titleElement = document.getElementById('title');
titleElement.textContent = '新しいタイトル';

let messageElement = document.getElementById('message');
messageElement.textContent = 'JavaScriptでテキストが変更されました！';

// HTMLを変更することもできます
messageElement.innerHTML = '<strong>太字のテキスト</strong>';

6.4 スタイルを変更する

要素のスタイル（色、サイズなど）も変更できます。

実習 6-3: スタイルを変更する
let titleElement = document.getElementById('title');

// 色を変更
titleElement.style.color = 'red';

// 背景色を変更
titleElement.style.backgroundColor = 'yellow';

// フォントサイズを変更
titleElement.style.fontSize = '40px';

// 複数のスタイルを一度に適用
titleElement.style.cssText = 'color: blue; font-size: 50px; text-align: center;';

イベントを処理しよう

6.5 クリックイベント

ユーザーがボタンをクリックしたときに、特定の処理を実行できます。

実習 6-4: クリックイベントを処理する
let buttonElement = document.getElementById('changeButton');

// クリックされたときの処理を設定
buttonElement.addEventListener('click', function() {
    let messageElement = document.getElementById('message');
    messageElement.textContent = 'ボタンがクリックされました！';
    messageElement.style.color = 'green';
});

6.6 複数のイベント

クリック以外にも様々なイベントに反応できます。

実習 6-5: 様々なイベント
let buttonElement = document.getElementById('changeButton');

// マウスが乗ったとき
buttonElement.addEventListener('mouseenter', function() {
    buttonElement.style.backgroundColor = 'lightblue';
});

// マウスが離れたとき
buttonElement.addEventListener('mouseleave', function() {
    buttonElement.style.backgroundColor = '';
});

// ダブルクリックしたとき
buttonElement.addEventListener('dblclick', function() {
    alert('ダブルクリックされました！');
});

フォームを操作しよう

6.7 入力フォームの操作

ユーザーからの入力を受け取り、それを処理できます。

フォーム用のHTML
<div>
    <label for="nameInput">名前:</label>
    <input type="text" id="nameInput" placeholder="お名前を入力してください">
</div>
<div>
    <label for="ageInput">年齢:</label>
    <input type="number" id="ageInput" placeholder="年齢を入力してください">
</div>
<button id="submitButton">送信</button>
<p id="result"></p>
実習 6-6: フォームの値を取得する
let submitButton = document.getElementById('submitButton');

submitButton.addEventListener('click', function() {
    // 入力値を取得
    let nameInput = document.getElementById('nameInput');
    let ageInput = document.getElementById('ageInput');
    let resultElement = document.getElementById('result');
    
    let name = nameInput.value;
    let age = ageInput.value;
    
    // 入力値をチェック
    if (name === '' || age === '') {
        resultElement.textContent = '名前と年齢を入力してください';
        resultElement.style.color = 'red';
        return;
    }
    
    // 結果を表示
    resultElement.textContent = `こんにちは、${name}さん！あなたは${age}歳ですね。`;
    resultElement.style.color = 'green';
});

ステップ2: クイズデータとゲームロジック

次に、クイズの問題データとゲームロジックを実装します：

// クイズデータ
const questions = [
    {
        question: "JavaScriptでコンソールに出力するための関数は？",
        options: ["console.log()", "print()", "echo()", "output()"],
        correct: 0
    },
    {
        question: "変数を宣言するときに使うキーワードは？",
        options: ["var", "let", "const", "すべて正しい"],
        correct: 3
    },
    {
        question: "配列の最初の要素のインデックスは？",
        options: ["1", "0", "-1", "最初の要素による"],
        correct: 1
    },
    {
        question: "関数を定義するときに使うキーワードは？",
        options: ["function", "def", "func", "method"],
        correct: 0
    },
    {
        question: "条件分岐に使う文は？",
        options: ["for", "while", "if", "switch"],
        correct: 2
    }
];

// ゲーム状態
let currentQuestion = 0;
let score = 0;
let selectedAnswer = null;

// DOM要素の取得
const questionElement = document.getElementById('question');
const optionsElement = document.getElementById('options');
const nextButton = document.getElementById('nextButton');
const quizArea = document.getElementById('quizArea');
const resultArea = document.getElementById('resultArea');
const finalResult = document.getElementById('finalResult');
const restartButton = document.getElementById('restartButton');
ステップ3: 問題表示とユーザー操作

問題を表示し、ユーザーの選択を処理する機能を実装します：

// 問題を表示する関数
function displayQuestion() {
    const question = questions[currentQuestion];
    questionElement.textContent = `問題 ${currentQuestion + 1}: ${question.question}`;
    
    // 選択肢をクリア
    optionsElement.innerHTML = '';
    
    // 選択肢を表示
    question.options.forEach((option, index) => {
        const optionElement = document.createElement('div');
        optionElement.className = 'option';
        optionElement.textContent = option;
        optionElement.addEventListener('click', () => selectAnswer(index));
        optionsElement.appendChild(optionElement);
    });
    
    // 次へボタンを隠す
    nextButton.style.display = 'none';
    selectedAnswer = null;
}

// 答えを選択する関数
function selectAnswer(answerIndex) {
    selectedAnswer = answerIndex;
    const question = questions[currentQuestion];
    const options = optionsElement.querySelectorAll('.option');
    
    // すべての選択肢をリセット
    options.forEach(option => {
        option.classList.remove('correct', 'incorrect');
    });
    
    // 正解の選択肢を緑色に
    options[question.correct].classList.add('correct');
    
    // 間違った選択肢を赤色に
    if (answerIndex !== question.correct) {
        options[answerIndex].classList.add('incorrect');
    } else {
        score++;
    }
    
    // 選択肢をクリック無効にする
    options.forEach(option => {
        option.style.pointerEvents = 'none';
    });
    
    // 次へボタンを表示
    nextButton.style.display = 'block';
}
ステップ4: ゲーム進行と結果表示

最後に、ゲームの進行と結果表示を実装します：

// 次の問題に進む
function nextQuestion() {
    currentQuestion++;
    
    if (currentQuestion < questions.length) {
        displayQuestion();
    } else {
        showResult();
    }
}

// 結果を表示する
function showResult() {
    quizArea.style.display = 'none';
    resultArea.style.display = 'block';
    
    const percentage = Math.round((score / questions.length) * 100);
    let message = '';
    
    if (percentage >= 80) {
        message = '🎉 素晴らしい！JavaScript マスターです！';
    } else if (percentage >= 60) {
        message = '👍 良くできました！もう少しで完璧です！';
    } else {
        message = '📚 もう少し勉強が必要です。頑張りましょう！';
    }
    
    finalResult.innerHTML = `
        <div class="score">スコア: ${score} / ${questions.length} (${percentage}%)</div>
        <div>${message}</div>
    `;
}

// ゲームを再開する
function restartGame() {
    currentQuestion = 0;
    score = 0;
    selectedAnswer = null;
    quizArea.style.display = 'block';
    resultArea.style.display = 'none';
    displayQuestion();
}

// イベントリスナーの設定
nextButton.addEventListener('click', nextQuestion);
restartButton.addEventListener('click', restartGame);

// ゲーム開始
displayQuestion();
理解度確認クイズ
  1. HTMLの要素をJavaScriptから操作するための仕組みを何と呼びますか？
  2. IDで要素を取得するメソッドは何ですか？
  3. 要素のテキスト内容を変更するプロパティは何ですか？
  4. ボタンのクリックを検出するために使うメソッドは何ですか？
  5. 入力フォームの値を取得するプロパティは何ですか？
答え
  1. DOM（Document Object Model）
  2. getElementById
  3. textContent
  4. addEventListener
  5. value

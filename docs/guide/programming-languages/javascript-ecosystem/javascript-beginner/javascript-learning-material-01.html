プログラミングとJavaScriptを知ろう

プログラミングとは何か、JavaScriptで何ができるのかを、身近な例を使って分かりやすく説明します。コンピュータと対話する第一歩を踏み出しましょう。

この章で学ぶこと

  • プログラミングとは何か、なぜ必要なのかを理解する
  • JavaScriptがどんな言語で、何ができるのかを知る
  • プログラミングの基本的な考え方（順次処理、条件分岐、繰り返し）を身近な例で理解する
  • これからの学習の進め方と心構えを知る

1.1 プログラミングとは何か

プログラミングとは、コンピュータに「何をしてほしいか」を指示することです。 難しく聞こえるかもしれませんが、実は私たちの日常生活でもよく似たことをしています。

身近な例で考えてみよう

自動販売機を使うとき、あなたは「お金を入れる → ボタンを押す」という順番で操作します。 自動販売機は、この指示通りに「お金が入ったか確認する → 押されたボタンの飲み物を出す」という動きをします。 これがプログラミングの考え方です。

コンピュータは、人間の言葉をそのまま理解することができません。 そこで、プログラミング言語という特別な言葉を使って、コンピュータに指示を出します。 JavaScriptは、そんなプログラミング言語の一つです。

1.2 JavaScriptとは

JavaScriptは、Webブラウザ（インターネットを見るためのソフト）で動くプログラミング言語です。 Google ChromeやSafari、Microsoft Edgeなど、みなさんが普段使っているブラウザには、 最初からJavaScriptを動かす機能が入っています。

JavaScriptが初心者におすすめな理由

  • 特別なソフトをインストールしなくても、今すぐ始められる
  • ブラウザがあれば、どのパソコンでも動かせる
  • 書いたプログラムの結果が画面ですぐに確認できる
  • 世界中で使われていて、情報がたくさんある

ちなみに、JavaScriptという名前に「Java」という言葉が入っていますが、 「Java」という別のプログラミング言語とはまったく別のものです。 名前が似ているだけなので、混同しないように注意しましょう。

1.3 JavaScriptで何ができるのか

JavaScriptを使うと、Webページを動的（ダイナミック）にすることができます。 「動的」とは、「動きがある」「変化する」という意味です。

クリックで反応するボタン

ボタンを押すと、画面に文字が表示されたり、色が変わったりする仕組みを作れます。

画像のスライドショー

写真が自動で次々と切り替わるスライドショーを作ることができます。

入力フォームのチェック

メールアドレスやパスワードが正しく入力されているか、その場で確認できます。

簡単なゲーム

クイズゲームやパズルゲームなど、ブラウザで遊べるゲームを作ることができます。

JavaScriptはWebページだけじゃない

JavaScriptは、もともとWebページを動かすために作られた言語ですが、 今ではスマートフォンアプリやサーバー（Webサイトの裏側で動くコンピュータ）のプログラムも JavaScriptで作ることができます。一つの言語で色々なことができる、とても便利な言語です。

1.4 プログラミングの基本的な考え方

どんなプログラムも、基本的には次の3つの考え方の組み合わせで作られています。

1 順次処理（じゅんじしょり）

上から順番に、一つずつ処理を実行するという考え方です。

例：カップラーメンを作る手順

  1. カップの中身を確認する
  2. お湯を注ぐ
  3. フタを閉める
  4. 3分待つ
  5. フタを開ける

この順番を変えると、美味しいカップラーメンは作れませんね。プログラムも同じです。

2 条件分岐（じょうけんぶんき）

「もし〜ならAをする、そうでなければBをする」という考え方です。

例：天気によって持ち物を決める

  • もし雨が降っていたら → 傘を持っていく
  • もし晴れていたら → 帽子を持っていく

プログラムでも、状況に応じて違う処理を実行することができます。

3 繰り返し（くりかえし）

同じ処理を何度も繰り返すという考え方です。

例：お米を洗う作業

  • お米に水を入れる
  • お米をかき混ぜる
  • 水を捨てる
  • ↑ この作業を「水が透明になるまで」繰り返す

プログラムでも、条件を満たすまで同じ処理を繰り返すことができます。

これらの3つの考え方を図にすると、次のようになります：

1.5 この学習の進め方

プログラミングは、理論だけ読んでもなかなか身につきません。 一番大切なのは、実際に手を動かして、自分でコードを書いてみることです。

学習のポイント

1

まずは真似から始める： 最初は例に出てくるコードをそのまま入力してみましょう。動かすことが大事です。

2

エラーは怖くない： プログラムを書いていると、必ずエラー（間違い）が出ます。これは誰でも同じです。エラーメッセージをよく読んで、原因を探す練習をしましょう。

3

自分で変えてみる： 例のコードが動いたら、数字や文字を変えてみましょう。「こうしたらどうなるかな？」という好奇心が、上達の近道です。

4

焦らず一歩ずつ： この教材は、基礎から順番に学べるように作られています。わからないところがあっても大丈夫。前の章に戻って復習しながら進みましょう。

第1章のまとめ

  • プログラミングとは、コンピュータに指示を出すこと
  • JavaScriptは、Webブラウザで動くプログラミング言語で、初心者でも始めやすい
  • JavaScriptで動的なWebページやゲームが作れる
  • プログラミングの基本は順次処理、条件分岐、繰り返しの3つ
  • 学習のコツは実際に手を動かすこととエラーを恐れないこと

理解度確認クイズ

  1. Q1. プログラミングとは何ですか？簡単に説明してください。

    回答を表示

    プログラミングとは、コンピュータに「何をしてほしいか」を指示することです。 プログラミング言語という特別な言葉を使って、コンピュータが理解できる形で命令を書きます。

  2. Q2. JavaScriptが初心者におすすめな理由を2つ挙げてください。

    回答を表示

    ① 特別なソフトをインストールしなくても、Webブラウザがあればすぐに始められる
    ② 書いたプログラムの結果がブラウザの画面ですぐに確認できる
    （他にも「世界中で使われていて情報が豊富」「色々なことができる」なども正解です）

  3. Q3. プログラミングの3つの基本的な考え方を答えてください。

    回答を表示

    ① 順次処理（上から順番に処理を実行する）
    ② 条件分岐（条件によって処理を変える）
    ③ 繰り返し（同じ処理を何度も繰り返す）

  4. Q4. 「JavaScriptとJavaは同じプログラミング言語である」という文は正しいですか？

    回答を表示

    正しくありません。JavaScriptとJavaは、名前が似ていますが、まったく別のプログラミング言語です。

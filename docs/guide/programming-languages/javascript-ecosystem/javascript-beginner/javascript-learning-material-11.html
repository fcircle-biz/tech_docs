第11章

時間差で処理しよう（非同期処理入門）

タイマー機能やアニメーションなど、時間を制御する処理の基本を、実例を作りながら理解します。

この章で学ぶこと

  • 非同期処理とは何か、なぜ必要なのかを理解する
  • setTimeout を使って一定時間後に処理を実行する方法を学ぶ
  • setInterval を使って一定間隔で繰り返し処理を実行する方法を学ぶ
  • タイマーを停止する方法とカウントダウンアプリを作成する

11.1 非同期処理とは何か

これまで学んできたプログラムは、上から順番に一つずつ実行されるものでした。 しかし、実際のWebアプリケーションでは「○秒後に実行する」「一定間隔で繰り返す」といった 時間を制御する処理が必要になることがあります。

日常の例で理解しよう

料理をするとき、「お湯を沸かす」→「麺を茹でる（3分）」→「その間に具材を切る」 というように、待ち時間の間に別の作業をすることがありますね。 プログラムでも同じように、「○秒待つ」間に他の処理を行ったり、 一定時間ごとに処理を繰り返したりすることができます。

JavaScriptで時間を扱う処理には、主に以下の2つがあります：

setTimeout

一定時間後に1回だけ処理を実行します。 例：「3秒後にメッセージを表示する」

setInterval

一定間隔で繰り返し処理を実行します。 例：「1秒ごとに時刻を更新する」

○秒後に1回実行"] B --> D["setInterval
○秒ごとに繰り返し実行"] C --> E["指定時間経過後に実行"] D --> F["指定間隔で何度も実行"]

11.2 setTimeout で一定時間後に処理を実行する

setTimeout は、 指定した時間（ミリ秒）が経過した後に、1回だけ処理を実行する関数です。

時間の単位に注意

JavaScriptでは時間をミリ秒（1/1000秒）で指定します。 1秒 = 1000ミリ秒、3秒 = 3000ミリ秒となります。

基本的な書き方

setTimeout の基本形 
// 書き方: setTimeout(実行したい処理, 待ち時間（ミリ秒）);

// 例：3秒後にメッセージを表示
setTimeout(function() {
  console.log('3秒経ちました！');
}, 3000);  // 3000ミリ秒 = 3秒

console.log('この文字は先に表示されます');
// 実行順序: 先に「この文字は先に表示されます」が表示され、
// その3秒後に「3秒経ちました！」が表示されます

処理の実行順序

setTimeout で指定した処理は「予約」されるだけで、その後のコードはすぐに実行されます。 つまり、プログラムは待たずに次の行に進みます。

実習 11-1: 時間差でメッセージを表示する

ボタンをクリックすると、2秒後にメッセージが画面に表示されるプログラムを作ってみましょう。

手順

  1. 1 以下のHTMLとJavaScriptコードを入力します
timer_test.html 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>タイマーテスト</title>
</head>
<body>
  <button id="startBtn">カウントダウン開始</button>
  <p id="message"></p>

  <script>
    const btn = document.getElementById('startBtn');
    const message = document.getElementById('message');

    btn.addEventListener('click', function() {
      message.textContent = '2秒後にメッセージが表示されます...';

      // 2秒後に実行される処理
      setTimeout(function() {
        message.textContent = '時間になりました！';
      }, 2000);  // 2000ミリ秒 = 2秒
    });
  </script>
</body>
</html>
  1. 2 ブラウザで開いて「カウントダウン開始」ボタンをクリックし、2秒後にメッセージが変わることを確認しましょう

11.3 setInterval で一定間隔で繰り返し処理を実行する

setInterval は、 指定した時間間隔で繰り返し処理を実行する関数です。 デジタル時計やアニメーションなど、定期的に更新が必要な処理に使います。

基本的な書き方

setInterval の基本形 
// 書き方: setInterval(繰り返し実行したい処理, 間隔（ミリ秒）);

// 例：1秒ごとにメッセージを表示
setInterval(function() {
  console.log('1秒経過');
}, 1000);  // 1000ミリ秒 = 1秒

// 実行結果: 「1秒経過」が1秒ごとに繰り返し表示され続けます

注意: 止め方を知っておこう

setInterval は一度実行すると、止めるまで永遠に繰り返し続けます。 必要に応じて停止できるようにしておきましょう（次のセクションで学びます）。

実用例：デジタル時計

デジタル時計の例 
// 1秒ごとに現在時刻を表示
setInterval(function() {
  const now = new Date();  // 現在の日時を取得
  const hours = now.getHours();    // 時
  const minutes = now.getMinutes();  // 分
  const seconds = now.getSeconds();  // 秒

  // 時刻を「時:分:秒」形式で表示
  console.log(hours + ':' + minutes + ':' + seconds);
}, 1000);  // 1秒ごとに実行

11.4 タイマーを停止する

setTimeout や setInterval で開始したタイマーを途中で止めたい場合は、 clearTimeout や clearInterval を使います。

タイマーIDを保存する

setTimeout や setInterval を実行すると、タイマーIDという番号が返されます。 このIDを使ってタイマーを停止できます。

タイマーIDの保存 
// タイマーIDを変数に保存
const timerId = setTimeout(function() {
  console.log('実行されました');
}, 3000);

// このtimerIdを使って後から停止できる

clearTimeout - setTimeout を停止

clearTimeout の使い方 
// 5秒後に実行される予定の処理
const timerId = setTimeout(function() {
  console.log('この文字は表示されません');
}, 5000);

// 1秒後にタイマーをキャンセル
setTimeout(function() {
  clearTimeout(timerId);  // タイマーを停止
  console.log('タイマーを停止しました');
}, 1000);

clearInterval - setInterval を停止

clearInterval の使い方 
let count = 0;

// 1秒ごとにカウントアップ
const timerId = setInterval(function() {
  count++;
  console.log(count + '秒経過');

  // 5回実行したら停止
  if (count >= 5) {
    clearInterval(timerId);  // タイマーを停止
    console.log('カウント終了');
  }
}, 1000);

実習 11-2: カウントダウンタイマーを作ろう

10から0までカウントダウンし、0になったら「終了！」と表示するタイマーを作ってみましょう。

countdown.html 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>カウントダウンタイマー</title>
  <style>
    #display { font-size: 48px; font-weight: bold; }
  </style>
</head>
<body>
  <h1>カウントダウンタイマー</h1>
  <div id="display">10</div>
  <button id="startBtn">開始</button>

  <script>
    let count = 10;  // カウント開始値
    let timerId = null;  // タイマーID

    const display = document.getElementById('display');
    const startBtn = document.getElementById('startBtn');

    startBtn.addEventListener('click', function() {
      // ボタンを無効化（連続クリック防止）
      startBtn.disabled = true;
      count = 10;  // カウントをリセット

      // 1秒ごとにカウントダウン
      timerId = setInterval(function() {
        count--;  // カウントを1減らす
        display.textContent = count;  // 画面に表示

        // 0になったら停止
        if (count <= 0) {
          clearInterval(timerId);  // タイマー停止
          display.textContent = '終了！';
          startBtn.disabled = false;  // ボタンを再び有効化
        }
      }, 1000);  // 1秒ごと
    });
  </script>
</body>
</html>

コードの解説

  • count-- でカウントを1ずつ減らします
  • display.textContent = count で画面に数字を表示します
  • count <= 0 になったら clearInterval でタイマーを停止します
  • startBtn.disabled でボタンの有効/無効を切り替えます

動作確認

ブラウザでHTMLファイルを開き、「開始」ボタンをクリックしてください。

期待される結果

  • 「10」から始まり、1秒ごとに数字が減っていく
  • 「0」になったら「終了！」と表示される
  • カウントダウン中は「開始」ボタンが押せない
  • 終了後、もう一度「開始」ボタンを押すとまたカウントダウンが始まる

11.5 応用例：デジタル時計を作ろう

setInterval を使って、リアルタイムで更新されるデジタル時計を作ってみましょう。

digital_clock.html 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>デジタル時計</title>
  <style>
    #clock { font-size: 64px; font-weight: bold; color: #333; }
  </style>
</head>
<body>
  <h1>現在時刻</h1>
  <div id="clock">00:00:00</div>

  <script>
    const clockDisplay = document.getElementById('clock');

    // 時刻を更新する関数
    function updateClock() {
      const now = new Date();  // 現在時刻を取得
      const h = String(now.getHours()).padStart(2, '0');    // 時（2桁）
      const m = String(now.getMinutes()).padStart(2, '0');  // 分（2桁）
      const s = String(now.getSeconds()).padStart(2, '0');  // 秒（2桁）

      clockDisplay.textContent = h + ':' + m + ':' + s;
    }

    // 最初に1回実行
    updateClock();

    // 1秒ごとに時刻を更新
    setInterval(updateClock, 1000);
  </script>
</body>
</html>

padStart について

padStart(2, '0') は、文字列を2桁にして、足りない場合は先頭に「0」を追加します。 例：「9」→「09」、「15」→「15」（そのまま）

第11章のまとめ

  • setTimeoutは指定時間後に1回だけ処理を実行する
  • setIntervalは指定間隔で繰り返し処理を実行する
  • 時間はミリ秒で指定する（1秒 = 1000ミリ秒）
  • clearTimeout / clearIntervalでタイマーを停止できる
  • タイマーIDを保存しておくことで、後から停止・制御が可能になる

理解度確認クイズ

  1. Q1. setTimeout と setInterval の違いを説明してください。

    回答を表示

    setTimeout は指定時間後に1回だけ処理を実行します。 setInterval は指定間隔で繰り返し処理を実行し続けます。

  2. Q2. 5秒後に「こんにちは」と表示するコードを書いてください（時間の単位に注意）。

    回答を表示

    setTimeout(function() { console.log('こんにちは'); }, 5000);
    5秒 = 5000ミリ秒なので、第2引数は5000になります。

  3. Q3. setInterval で開始したタイマーを停止するには、どの関数を使いますか？

    回答を表示

    clearInterval(タイマーID) を使います。
    タイマーIDは、setInterval実行時に返される値を変数に保存しておく必要があります。

  4. Q4. なぜタイマーIDを変数に保存しておく必要がありますか？

    回答を表示

    後からタイマーを停止したい時に、clearTimeout や clearInterval にタイマーIDを渡す必要があるためです。 タイマーIDを保存しておかないと、どのタイマーを停止すればいいかわからなくなります。

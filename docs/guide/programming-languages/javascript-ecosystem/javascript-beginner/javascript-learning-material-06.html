第6章

同じ処理を繰り返そう（for文・while文）

繰り返し処理を使って効率的にプログラムを書く方法を、カウントダウンや九九の表などを作りながら実践的に学びます。

この章で学ぶこと

  • 繰り返し処理とは何か、なぜ必要なのかを理解する
  • for文を使って決まった回数だけ処理を繰り返す方法を学ぶ
  • while文を使って条件を満たす間だけ処理を繰り返す方法を学ぶ
  • break文とcontinue文を使った繰り返しの制御方法を知る
  • 繰り返し処理を使って九九の表やカウントダウンを作る

6.1 繰り返し処理とは

繰り返し処理（ループ処理）とは、同じ処理を何度も自動的に実行する仕組みです。 プログラミングでは、この繰り返し処理を使うことで、同じコードを何度も書く必要がなくなり、 効率的にプログラムを作ることができます。

日常生活での繰り返し

朝の支度で「歯を磨く → 顔を洗う → 服を着替える」という作業を考えてみましょう。 もし家族4人分の朝食を作るとしたら、「卵を割る → フライパンで焼く → お皿に盛る」という 作業を4回繰り返しますね。これと同じように、プログラムでも同じ処理を繰り返すことができます。

JavaScriptには、繰り返し処理を行うためのいくつかの方法がありますが、 この章では最も基本的なfor文とwhile文を学びます。

更新] Update --> Check Check -->|いいえ| End([繰り返し終了])

6.2 for文で決まった回数繰り返す

for文は、「あらかじめ決まった回数だけ処理を繰り返したい」ときに使います。 例えば、「1から10まで数える」「クラスの30人全員に連絡する」といった場合に便利です。

for文の基本構文

for文の基本形 
for (初期化; 条件式; 増減式) {
  // 繰り返したい処理
}

for文の3つの部品

1

初期化（しょきか）

カウンターの初期値を設定します（例: let i = 0）

2

条件式（じょうけんしき）

繰り返しを続ける条件を書きます（例: i < 5）

3

増減式（ぞうげんしき）

1回の処理が終わるたびにカウンターを更新します（例: i++）

例1: 1から5まで表示する

for-example1.html 
// 1から5までの数字を表示
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
// 出力:
// 1
// 2
// 3
// 4
// 5

i++ とは？

i++は、 i = i + 1の省略形です。 変数iの値を1増やすという意味です。プログラミングでは非常によく使う書き方なので覚えておきましょう。

例2: 偶数だけを表示する

for-example2.html 
// 1から10までの偶数を表示
for (let i = 2; i <= 10; i = i + 2) {
  console.log(i);
}
// 出力:
// 2
// 4
// 6
// 8
// 10

実習 6-1: カウントダウンを作ろう

for文を使って、10から1までカウントダウンするプログラムを作ってみましょう。

手順

  1. 1 HTMLファイルを作成し、以下のコードを書いてみましょう
countdown.html 
// 10から1までカウントダウン
for (let i = 10; i >= 1; i--) {
  console.log(i);
}
console.log("発射！");

6.3 while文で条件を満たす間繰り返す

while文は、「条件が満たされている間、ずっと処理を繰り返す」ときに使います。 for文との違いは、繰り返す回数が最初から決まっていない場合に使うことです。

while文の基本構文

while文の基本形 
while (条件式) {
  // 条件が真の間、繰り返す処理
}

for文とwhile文の使い分け

  • for文: 繰り返す回数が決まっている（例: 1から10まで、配列の全要素）
  • while文: 繰り返す回数が決まっていない（例: 正しい入力があるまで、目標値に達するまで）

例1: while文で1から5まで表示

while-example1.html 
// while文で1から5まで表示
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;  // 忘れずにカウンターを増やす
}
// 出力:
// 1
// 2
// 3
// 4
// 5

無限ループに注意！

while文では、条件式が常に真（true）になってしまうと、永遠に繰り返しが続いてしまいます。 これを無限ループと呼びます。必ず繰り返しの中でカウンターを更新するか、 条件が偽（false）になる仕組みを入れましょう。

例2: 合計が100を超えるまで足す

while-example2.html 
// 1から順番に足して、合計が100を超えるまで繰り返す
let sum = 0;
let num = 1;

while (sum <= 100) {
  sum = sum + num;
  console.log(num + "を足しました。合計: " + sum);
  num++;
}
console.log("100を超えました！");

6.4 繰り返しを途中で制御する（break / continue）

繰り返し処理の途中で、「この条件の時だけループを抜ける」「この処理だけスキップする」といった 細かい制御をしたい場合があります。そんな時に使うのがbreak文とcontinue文です。

break文: ループを途中で終了する

break-example.html 
// 1から10まで数えるが、7で止める
for (let i = 1; i <= 10; i++) {
  if (i === 7) {
    console.log("7が出たので終了します");
    break;  // ここでループを抜ける
  }
  console.log(i);
}
// 出力:
// 1
// 2
// 3
// 4
// 5
// 6
// 7が出たので終了します

continue文: その回だけスキップする

continue-example.html 
// 1から10まで、3の倍数だけスキップして表示
for (let i = 1; i <= 10; i++) {
  if (i % 3 === 0) {
    continue;  // 3の倍数の時はスキップ
  }
  console.log(i);
}
// 出力:
// 1
// 2
// 4
// 5
// 7
// 8
// 10

break文とcontinue文の違い：

ループを完全に終了"] BreakNode --> End([繰り返し終了]) Check -->|continue発動| ContinueNode["continue
次の繰り返しへスキップ"] ContinueNode --> Loop Check -->|通常処理| Process[処理を実行] Process --> Loop Loop -->|終了| End

実習 6-2: 九九の表を作ろう

for文を2重に使って（ネストして）、九九の表を作ってみましょう。

手順

  1. 1 以下のコードを書いて実行してみましょう
multiplication-table.html 
// 九九の表を作る
for (let i = 1; i <= 9; i++) {
  let line = "";  // 1行分の文字列
  for (let j = 1; j <= 9; j++) {
    let result = i * j;
    line = line + result + " ";
  }
  console.log(line);
}
  1. 2 コンソールに九九の表が表示されることを確認しましょう

6.5 実践例: 星の模様を描く

繰り返し処理を使って、コンソールに星の模様を描いてみましょう。 これは、プログラミングでよく使われる練習問題です。

例: ピラミッドを作る

star-pyramid.html 
// 星でピラミッドを作る
for (let i = 1; i <= 5; i++) {
  let stars = "";
  for (let j = 1; j <= i; j++) {
    stars = stars + "★";
  }
  console.log(stars);
}
// 出力:
// ★
// ★★
// ★★★
// ★★★★
// ★★★★★

第6章のまとめ

  • 繰り返し処理を使うと、同じ処理を何度も効率的に実行できる
  • for文は繰り返す回数が決まっている場合に使う（例: 1から10まで）
  • while文は条件を満たす間だけ繰り返す場合に使う
  • break文でループを途中で終了、continue文でその回だけスキップできる
  • for文をネスト（入れ子）にすると、表や模様などを作れる
  • 無限ループにならないよう、条件式とカウンターの更新に注意する

理解度確認クイズ

  1. Q1. for文の3つの部品（初期化、条件式、増減式）について、それぞれ何をするものか説明してください。

    回答を表示

    ① 初期化: カウンター変数の初期値を設定する（例: let i = 0）
    ② 条件式: 繰り返しを続ける条件を指定する（例: i < 10）
    ③ 増減式: 1回の処理が終わるたびにカウンターを更新する（例: i++）

  2. Q2. for文とwhile文の使い分けについて説明してください。

    回答を表示

    for文は繰り返す回数があらかじめ決まっている場合に使います（例: 1から10まで数える）。 while文は繰り返す回数が決まっておらず、条件を満たす間だけ繰り返したい場合に使います （例: 正しい入力があるまで、目標値に達するまで）。

  3. Q3. break文とcontinue文の違いを説明してください。

    回答を表示

    break文はループを完全に終了します（ループから抜け出す）。 continue文は現在の繰り返しだけをスキップして、次の繰り返しに進みます。

  4. Q4. 次のコードの出力結果を答えてください。

    for (let i = 0; i < 4; i++) {
  console.log(i * 2);
}
    回答を表示

    0
    2
    4
    6

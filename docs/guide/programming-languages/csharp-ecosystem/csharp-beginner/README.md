# C#入門 学習ガイドライン

このガイドラインでは、プログラミング完全未経験者を対象に、C#（シーシャープ）の基礎を丁寧かつ段階的に学習するためのカリキュラムを提供しています。Visual Studioの使い方から始まり、本格的なコンソールアプリケーションの開発までを一歩ずつ学んでいきます。

## 前提条件

### 必要な環境
- Windows 10 または Windows 11（64ビット版推奨）
- Visual Studio 2022 Community Edition（無料）
- インターネット接続（インストール時に必要）
- 4GB以上のRAM、10GB以上の空きディスク容量

### 参考リソース
- [C# 公式ドキュメント（日本語）](https://docs.microsoft.com/ja-jp/dotnet/csharp/)
- [Visual Studio ダウンロードページ](https://visualstudio.microsoft.com/ja/downloads/)
- [.NET 公式サイト](https://dotnet.microsoft.com/ja-jp/)
- [C# 入門チュートリアル](https://docs.microsoft.com/ja-jp/dotnet/csharp/tour-of-csharp/)

### 前提知識
- **必須**: 基本的なパソコン操作（マウス操作、キーボード入力、ファイル保存など）
- **推奨**: なし（プログラミング未経験者を対象としています）

### [2. Visual Studioをインストールしよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-02.html)

### [3. Visual Studioの基本操作を覚えよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-03.html)
Visual Studioの画面構成、各パネルの役割、プロジェクトの作成方法を学びます。開発環境に慣れることで、今後のプログラミング学習がスムーズになります。

### [5. 変数とデータ型を理解しよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-05.html)

足し算、引き算などの算術演算子や、文字列の連結など、プログラムでの計算方法を学びます。ユーザーからの入力を受け取り、計算結果を表示するプログラムを作成します。

### [7. 条件分岐で処理を分けよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-07.html)

### [8. 繰り返し処理をマスターしよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-08.html)
for文、while文、do-while文、foreach文を使った繰り返し処理を学びます。同じ処理を効率的に繰り返す方法と、無限ループを避けるためのポイントを理解します。

### [9. 配列とリストでデータを管理しよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-09.html)
複数のデータをまとめて扱う配列とListの基本を学びます。要素の追加・削除・検索など、実践的なデータ操作を体験します。

### [10. メソッドで処理をまとめよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-10.html)
メソッド（関数）の作成方法を学びます。引数と戻り値の概念を理解し、処理を再利用可能な形にまとめる技術を身につけます。

### [11. クラスとオブジェクト指向の基礎を学ぼう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-11.html)
C#の特徴であるオブジェクト指向プログラミングの基本概念を学びます。クラス、オブジェクト、プロパティ、コンストラクタについて、身近な例を使って分かりやすく解説します。

### [12. 例外処理とエラーへの対応を学ぼう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-12.html)
try-catch文を使った例外処理の基本を学びます。プログラムの実行中に発生するエラーに適切に対応し、堅牢なプログラムを作成する方法を理解します。

### [13. 総合演習：シンプルな家計簿アプリを作ろう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/csharp-ecosystem/csharp-beginner/csharp-beginner-learning-material-13.html)

## 学習の進め方

1. **環境構築を最優先に**: 第1-3章でVisual Studioのセットアップと基本操作を習得し、プログラミングができる環境を整えましょう
2. **実際に手を動かす**: サンプルコードは必ず自分で入力してください。コピー&ペーストではなく、タイピングすることで構文が身につきます
3. **エラーを恐れない**: エラーはプログラミング学習の一部です。エラーメッセージを読み、原因を考えることで理解が深まります
4. **章の順番を守る**: 各章は前の章の知識を前提としています。必ず順番通りに進めてください
5. **復習と練習**: 各章の最後にある練習問題に取り組み、理解を確認してから次の章に進みましょう
6. **分からないことは調べる**: 公式ドキュメントや参考リソースを活用して、疑問点を解消しましょう

## 推奨学習期間

- **環境構築コース** (1-3章): 1週間
  - Visual Studioのインストールと基本操作に慣れる期間
- **基礎文法コース** (4-8章): 3-4週間
  - 変数、演算子、条件分岐、繰り返しなど基本的なプログラミング概念を習得
- **データ構造・関数コース** (9-10章): 2週間
  - 配列、リスト、メソッドを使った本格的なプログラミングを学習
- **オブジェクト指向入門コース** (11-12章): 2週間
  - クラスの基礎と例外処理を理解
- **総合演習コース** (13章): 1週間
  - 学んだ知識を総動員してアプリケーションを作成

**合計: 9-10週間（1日1-2時間の学習を想定）**

## 学習目標

このガイドを完了すると、以下のスキルを身につけることができます：

- プログラミングの基本概念（変数、条件分岐、繰り返し、配列、メソッド）を理解し、C#のコードで表現できる
- Visual Studioを使ってC#プロジェクトを作成・編集・実行できる
- エラーメッセージを読んで、簡単なバグを自分で修正できる
- クラスとオブジェクトの基本概念を理解し、簡単なクラスを設計できる
- 例外処理を使って、エラーに強いプログラムを作成できる
- 学んだ知識を組み合わせて、簡単なコンソールアプリケーションを自作できる

## 次のステップ

このガイドライン完了後は、以下の学習に進むことをお勧めします：

- **C#中級編**: オブジェクト指向の3大要素（継承、カプセル化、ポリモーフィズム）、インターフェース、ジェネリクスなど高度な機能を学習
- **WPF / Windows Forms**: C#を使ったGUIデスクトップアプリケーション開発に挑戦
- **ASP.NET Core**: C#を使ったWebアプリケーション開発を学習
- **データベース連携**: Entity FrameworkやADO.NETを使ったデータベースアプリケーションの開発
- **LINQ**: C#のデータ操作機能を使った効率的なプログラミング

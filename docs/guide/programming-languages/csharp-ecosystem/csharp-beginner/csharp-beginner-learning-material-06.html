第6章

演算子と計算をしてみよう

足し算、引き算などの算術演算子や、文字列の連結など、プログラムでの計算方法を学びます。ユーザーからの入力を受け取り、計算結果を表示するプログラムを作成します。

この章で学ぶこと

  • 算術演算子（+、-、*、/、%）を使った計算方法を理解する
  • 代入演算子と複合代入演算子の使い方を学ぶ
  • インクリメント・デクリメント演算子の動作を理解する
  • ユーザーからの入力を受け取り、計算結果を表示するプログラムを作成する

6.1 算術演算子の基本

プログラムで計算を行うには、演算子を使います。 演算子とは、足し算や引き算などの計算を指示する記号のことです。

基本的な算術演算子

C#では、以下の算術演算子を使って計算ができます。

算術演算子一覧

演算子 意味 例 結果
+ 足し算 5 + 3 8
- 引き算 10 - 7 3
* 掛け算 4 * 6 24
/ 割り算 20 / 5 4
% 剰余（あまり） 17 % 5 2

実際に計算してみよう

算術演算子を使ったプログラムを作成してみましょう。

算術演算子の例 
// 基本的な算術演算
int a = 10;
int b = 3;

int sum = a + b;       // 足し算: 13
int diff = a - b;      // 引き算: 7
int product = a * b;   // 掛け算: 30
int quotient = a / b;  // 割り算: 3（整数同士の割り算は整数）
int remainder = a % b; // 剰余: 1（10 ÷ 3 = 3あまり1）

Console.WriteLine($"足し算: {sum}");
Console.WriteLine($"引き算: {diff}");
Console.WriteLine($"掛け算: {product}");
Console.WriteLine($"割り算: {quotient}");
Console.WriteLine($"あまり: {remainder}");

整数の割り算に注意

int型同士の割り算は、小数点以下が切り捨てられます。
例: 10 / 3 は 3 になります（3.333...ではない）。
小数点を含む結果が必要な場合は、double型を使います。

6.2 演算子の優先順位

算数と同じように、プログラムでも計算の順番があります。 掛け算・割り算は足し算・引き算より先に計算されます。

演算子の優先順位 
// 優先順位の例
int result1 = 5 + 3 * 2;  // 結果: 11（3 * 2 = 6、その後 5 + 6 = 11）
int result2 = (5 + 3) * 2; // 結果: 16（カッコが優先されて 8 * 2 = 16）

Console.WriteLine($"5 + 3 * 2 = {result1}");
Console.WriteLine($"(5 + 3) * 2 = {result2}");

ヒント

計算の順番が分かりにくいときは、カッコ（ ）を使うと分かりやすくなります。
カッコ内の計算は最優先で実行されます。

6.3 代入演算子と複合代入演算子

=は代入演算子です。 右辺の値を左辺の変数に代入（格納）します。

複合代入演算子

計算と代入を同時に行う複合代入演算子があります。 コードを短く書けるので便利です。

複合代入演算子一覧

演算子 意味 例 同じ意味のコード
+= 加算代入 x += 5 x = x + 5
-= 減算代入 x -= 3 x = x - 3
*= 乗算代入 x *= 2 x = x * 2
/= 除算代入 x /= 4 x = x / 4
%= 剰余代入 x %= 3 x = x % 3
複合代入演算子の例 
// 複合代入演算子の使用例
int score = 100;

score += 20;  // score = score + 20 と同じ（scoreは120になる）
Console.WriteLine($"得点（+20点後）: {score}");

score -= 10;  // score = score - 10 と同じ（scoreは110になる）
Console.WriteLine($"得点（-10点後）: {score}");

score *= 2;   // score = score * 2 と同じ（scoreは220になる）
Console.WriteLine($"得点（2倍後）: {score}");

6.4 インクリメント・デクリメント演算子

変数の値を1だけ増やすまたは1だけ減らすことは、 プログラムでよく行う操作です。そのための専用の演算子があります。

インクリメント・デクリメント演算子

演算子 意味 例 同じ意味のコード
++ インクリメント（1増やす） count++ count = count + 1
-- デクリメント（1減らす） count-- count = count - 1
インクリメント・デクリメントの例 
// インクリメント・デクリメントの例
int count = 0;

count++;  // countを1増やす（1になる）
Console.WriteLine($"count: {count}");

count++;  // countを1増やす（2になる）
Console.WriteLine($"count: {count}");

count--;  // countを1減らす（1になる）
Console.WriteLine($"count: {count}");

前置と後置の違い

++count（前置）とcount++（後置）には、 他の式と組み合わせたときに違いがありますが、初心者のうちは後置（count++）を使うのが一般的です。

6.5 文字列の連結

+演算子は、数値だけでなく文字列の連結にも使えます。 複数の文字列をつなげて、1つの長い文字列を作ることができます。

文字列の連結 
// 文字列の連結
string firstName = "太郎";
string lastName = "山田";

// +演算子で文字列を連結
string fullName = lastName + firstName;
Console.WriteLine(fullName);  // 出力: 山田太郎

// 文字列と数値を連結
int age = 20;
string message = "私は" + age + "歳です。";
Console.WriteLine(message);  // 出力: 私は20歳です。

重要ポイント

文字列と数値を+でつなぐと、数値が自動的に文字列に変換されて連結されます。
これを暗黙的な型変換と言います。

6.6 ユーザー入力の受け取りと型変換

実用的なプログラムでは、ユーザーからの入力を受け取ることが多くあります。 Console.ReadLine()を使うと、ユーザーが入力した文字列を受け取れます。

Console.ReadLine()の基本

ユーザー入力の受け取り 
// ユーザー入力の受け取り
Console.Write("あなたの名前を入力してください: ");
string name = Console.ReadLine();  // ユーザーが入力した文字列を受け取る

Console.WriteLine($"こんにちは、{name}さん！");

数値として受け取る方法

Console.ReadLine()は常にstring型を返します。 計算に使うには、int型やdouble型に変換する必要があります。

型変換の例 
// 文字列を数値に変換する
Console.Write("年齢を入力してください: ");
string input = Console.ReadLine();
int age = int.Parse(input);  // string型をint型に変換

Console.WriteLine($"あなたは{age}歳ですね。");

// より簡潔に書く方法
Console.Write("身長を入力してください（cm）: ");
int height = int.Parse(Console.ReadLine());

Console.WriteLine($"身長は{height}cmです。");

注意

int.Parse()は、変換できない文字列（例: "abc"）を渡すとエラーになります。
後の章で学ぶ例外処理を使うと、エラーを安全に処理できます。

実習 6-1: 2つの数値を足すプログラム

ユーザーから2つの数値を入力してもらい、その合計を表示するプログラムを作成しましょう。

手順

  1. 1 Visual Studioで新しいコンソールアプリプロジェクトを作成
  2. 2 「1つ目の数値を入力してください」と表示し、ユーザーの入力を受け取る
  3. 3 「2つ目の数値を入力してください」と表示し、ユーザーの入力を受け取る
  4. 4 両方の入力をint型に変換し、合計を計算
  5. 5 「合計は〇〇です」と結果を表示
解答例 
// 2つの数値を足すプログラム
Console.Write("1つ目の数値を入力してください: ");
int num1 = int.Parse(Console.ReadLine());

Console.Write("2つ目の数値を入力してください: ");
int num2 = int.Parse(Console.ReadLine());

int sum = num1 + num2;
Console.WriteLine($"合計は{sum}です。");

実習 6-2: 四則演算計算機

ユーザーから2つの数値を入力してもらい、足し算・引き算・掛け算・割り算の結果をすべて表示するプログラムを作成しましょう。

解答例 
// 四則演算計算機
Console.WriteLine("== 四則演算計算機 ==");
Console.Write("1つ目の数値: ");
int a = int.Parse(Console.ReadLine());

Console.Write("2つ目の数値: ");
int b = int.Parse(Console.ReadLine());

Console.WriteLine($"\n--- 計算結果 ---");
Console.WriteLine($"{a} + {b} = {a + b}");
Console.WriteLine($"{a} - {b} = {a - b}");
Console.WriteLine($"{a} × {b} = {a * b}");
Console.WriteLine($"{a} ÷ {b} = {a / b}");
Console.WriteLine($"{a} ÷ {b} の余り = {a % b}");

理解度確認クイズ

  1. Q1. 17 % 5 の計算結果は何ですか？

    回答を表示

    答え: 2
    %は剰余演算子（あまり）です。17 ÷ 5 = 3あまり2 なので、結果は2になります。

  2. Q2. x += 10 と同じ意味のコードはどれですか？

    回答を表示

    答え: x = x + 10
    +=は複合代入演算子で、変数に値を加算して代入します。

  3. Q3. Console.ReadLine() が返す値の型は何ですか？

    回答を表示

    答え: string型
    Console.ReadLine()は常にstring型を返します。 数値として計算したい場合は、int.Parse()などで変換が必要です。

  4. Q4. 5 + 3 * 2 の計算結果は何ですか？

    回答を表示

    答え: 11
    掛け算が優先されるので、まず3 * 2 = 6が計算され、 次に5 + 6 = 11になります。

  • 複合代入演算子（+=、-=など）で計算と代入を同時に行える
  • インクリメント・デクリメント演算子（++、--）で値を1増減できる
  • Console.ReadLine()でユーザー入力を受け取り、int.Parse()で数値に変換できる

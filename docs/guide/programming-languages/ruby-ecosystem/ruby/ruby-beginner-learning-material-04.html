演算と文字列操作 - 計算と文字を扱おう

電卓のような計算機能と、文字列の結合・分割などの操作方法を学びます。日常的な計算やテキスト処理をプログラムで自動化する方法を習得します。

この章で学ぶこと

  • 四則演算（足し算、引き算、掛け算、割り算）の基本
  • 剰余演算と累乗の使い方
  • 文字列の連結と式展開（変数の埋め込み）
  • 文字列の基本メソッド（大文字・小文字変換、長さの取得など）

4.1 四則演算の基本 - 電卓のように計算しよう

Rubyでは、電卓と同じように数値の計算ができます。プログラミングで最も基本的な操作の一つが算術演算です。

基本的な演算子

Rubyで使える主な算術演算子は以下の通りです：

演算子 意味 例 結果
+ 足し算 10 + 5 15
- 引き算 10 - 5 5
* 掛け算 10 * 5 50
/ 割り算 10 / 5 2

実際に計算してみよう

以下のプログラムで、実際に四則演算を試してみましょう。

calculation.rb 
# 四則演算の例
puts 10 + 5   # 足し算
puts 10 - 5   # 引き算
puts 10 * 5   # 掛け算
puts 10 / 5   # 割り算

# 複数の演算を組み合わせることもできる
result = (10 + 5) * 2
puts result   # 30

演算の優先順位

数学と同じように、掛け算と割り算は足し算と引き算より先に計算されます。順序を変えたい場合は、カッコ（）を使います。

4.2 様々な演算子 - 割り算と余りの計算

四則演算以外にも、プログラミングでよく使う便利な演算子があります。

剰余演算（割り算の余り）

%演算子を使うと、割り算の余りを求めることができます。これは「偶数か奇数か」の判定などによく使われます。

modulo.rb 
# 剰余演算の例
puts 10 % 3   # 1（10÷3=3余り1）
puts 15 % 4   # 3（15÷4=3余り3）
puts 20 % 5   # 0（20÷5=4余り0、割り切れる）

# 偶数か奇数かの判定に便利
number = 7
if number % 2 == 0
  puts "#{number}は偶数です"
else
  puts "#{number}は奇数です"
end

累乗（べき乗）

**演算子を使うと、累乗（べき乗）を計算できます。たとえば「2の3乗」は2 ** 3と書きます。

power.rb 
# 累乗の例
puts 2 ** 3   # 8（2×2×2）
puts 5 ** 2   # 25（5×5）
puts 10 ** 3  # 1000（10×10×10）

整数と小数の割り算

整数同士の割り算は、結果も整数になります。小数点以下が必要な場合は、10.0 / 3のように、どちらか一方を小数にします。

division.rb 
# 整数と小数の割り算
puts 10 / 3     # 3（整数同士なので小数点以下切り捨て）
puts 10.0 / 3   # 3.333...（小数を含むので正確な結果）
puts 10 / 3.0   # 3.333...（どちらか一方が小数でOK）

4.3 文字列の連結 - 文字をつなげよう

文字列（テキスト）同士をつなげることを連結と言います。Rubyでは+演算子を使って文字列を連結できます。

string_concat.rb 
# 文字列の連結
first_name = "太郎"
last_name = "山田"

# +演算子で連結
full_name = last_name + first_name
puts full_name   # 山田太郎

# 文字列を複数つなげる
greeting = "こんにちは、" + full_name + "さん！"
puts greeting    # こんにちは、山田太郎さん！

注意：数値と文字列の連結

文字列と数値を+で直接つなげることはできません。数値を文字列に変換する必要があります（次のセクションで解説）。

4.4 式展開 - 文字列の中に変数を埋め込む

Rubyには、文字列の中に変数や計算結果を直接埋め込める式展開という便利な機能があります。#{}を使います。

interpolation.rb 
# 式展開の例
name = "田中"
age = 25

# #{}の中に変数を書く
puts "私の名前は#{name}です"
puts "私は#{age}歳です"

# #{}の中で計算もできる
price = 1000
puts "消費税込みの価格は#{price * 1.1}円です"

# 複数の変数を埋め込む
puts "#{name}さんは#{age}歳で、来年は#{age + 1}歳になります"

+連結 vs 式展開

同じ結果を得るのに、+連結と式展開の2つの方法があります。式展開の方が読みやすく、Rubyらしい書き方です。

comparison.rb 
name = "佐藤"
age = 30

# +連結（長くて読みにくい）
message1 = name + "さんは" + age.to_s + "歳です"

# 式展開（短くて読みやすい）★推奨
message2 = "#{name}さんは#{age}歳です"

puts message1  # 佐藤さんは30歳です
puts message2  # 佐藤さんは30歳です

シングルクォートとダブルクォート

式展開はダブルクォート（"）の中でのみ動作します。シングルクォート（'）では式展開されません。

4.5 文字列のメソッド - 便利な文字列操作

Rubyの文字列には、様々な便利なメソッド（機能）が用意されています。よく使うものを見ていきましょう。

length / size

文字列の長さ（文字数）を取得します。 

"こんにちは".length  # 5

upcase

すべての文字を大文字に変換します。 

"hello".upcase  # "HELLO"

downcase

すべての文字を小文字に変換します。 

"HELLO".downcase  # "hello"

reverse

文字列を逆順にします。 

"Ruby".reverse  # "ybuR"

実際に使ってみよう

string_methods.rb 
# 文字列メソッドの例
message = "Hello Ruby"

puts message.length      # 10（文字数）
puts message.upcase      # HELLO RUBY（大文字）
puts message.downcase    # hello ruby（小文字）
puts message.reverse     # ybuR olleH（逆順）

# メソッドは連続して使える（メソッドチェーン）
puts "ruby".upcase.reverse   # "YBUR"

元の文字列は変わらない

これらのメソッドは、元の文字列を変更せず、新しい文字列を返します。元の文字列を変更したい場合は、upcase!のように末尾に!をつけます。

4.6 実用例 - 消費税計算プログラムを作ろう

これまで学んだ演算と文字列操作を組み合わせて、実用的なプログラムを作ってみましょう。

実習 4-1: 消費税計算プログラム

商品の価格を入力すると、消費税込みの価格を計算して表示するプログラムを作ります。

tax_calculator.rb 
# 消費税計算プログラム
price = 1500          # 商品の価格（円）
tax_rate = 0.1        # 消費税率（10%）

# 消費税額を計算
tax = price * tax_rate

# 税込み価格を計算
total_price = price + tax

# 結果を表示（式展開を使用）
puts "商品価格: #{price}円"
puts "消費税: #{tax.to_i}円"
puts "税込み価格: #{total_price.to_i}円"

さらに改良しよう

複数の商品の合計金額と消費税を計算するプログラムに拡張してみましょう。

理解度確認クイズ

  1. Q1. 15 % 4の結果は何ですか？

    回答を表示

    答え：3。15÷4=3余り3なので、余りである3が結果になります。

  2. Q2. 文字列の連結に+を使う方法と、式展開#{}を使う方法のどちらが推奨されますか？理由も説明してください。

    回答を表示

    答え：式展開#{}が推奨されます。理由は、コードが短く読みやすく、数値を文字列に変換する手間（to_s）が不要で、Rubyらしい書き方だからです。

  3. Q3. "Ruby".upcase.reverseの結果は何ですか？

    回答を表示

    答え："YBUR"。まずupcaseで"RUBY"になり、次にreverseで逆順にされて"YBUR"になります。

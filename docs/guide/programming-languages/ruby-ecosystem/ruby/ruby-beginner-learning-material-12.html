12.2 プログラム1: じゃんけんゲーム

最初は、誰もが知っている「じゃんけん」をプログラムで作ってみましょう。このゲームには、配列、ランダム、条件分岐、繰り返しなど、多くの基礎要素が含まれています。

プログラムの設計

まず、プログラムが「何をすべきか」を整理します。じゃんけんゲームに必要な機能は以下の通りです：

  • プレイヤーに「グー・チョキ・パー」を選ばせる
  • コンピュータがランダムに手を選ぶ
  • 勝敗を判定する
  • 結果を表示する
  • もう一度遊ぶか聞く

プログラムの実装

janken_game.rb 
# じゃんけんゲーム
def janken_game
  # 手の種類を配列で定義
  hands = ["グー", "チョキ", "パー"]

  puts "=== じゃんけんゲームへようこそ！ ==="

  loop do
    # プレイヤーの手を入力
    puts "\n手を選んでください（0:グー, 1:チョキ, 2:パー）"
    print "> "
    player_choice = gets.to_i

    # 0-2以外の入力はエラー
    if player_choice < 0 || player_choice > 2
      puts "0、1、2のいずれかを入力してください"
      next
    end

    # コンピュータの手をランダムに選択
    computer_choice = rand(3)

    # 結果表示
    puts "\nあなた: #{hands[player_choice]}"
    puts "コンピュータ: #{hands[computer_choice]}"

    # 勝敗判定
    if player_choice == computer_choice
      puts "結果: 引き分け！"
    elsif (player_choice == 0 && computer_choice == 1) ||
          (player_choice == 1 && computer_choice == 2) ||
          (player_choice == 2 && computer_choice == 0)
      puts "結果: あなたの勝ち！"
    else
      puts "結果: あなたの負け..."
    end

    # 続けるか聞く
    puts "\nもう一度遊びますか？（y/n）"
    print "> "
    answer = gets.chomp
    break if answer != "y"
  end

  puts "\nゲームを終了します。ありがとうございました！"
end

プログラムのポイント

配列のインデックスを使って手を管理し、`rand(3)`でランダムな数を生成しています。条件分岐では「勝つパターン」を明確に書くことで、判定ロジックが分かりやすくなっています。

  2. 2 ターミナルで`ruby janken_game.rb`を実行

12.3 プログラム2: 簡単な計算機

次は、四則演算（足し算、引き算、掛け算、割り算）ができる電卓プログラムを作ってみましょう。メソッドを活用して、機能ごとに処理を分けます。

プログラムの実装

calculator.rb 
# 簡単な計算機プログラム
def calculator
  puts "=== 簡単な計算機 ==="

  loop do
    # 1つ目の数を入力
    puts "\n1つ目の数を入力してください"
    print "> "
    num1 = gets.to_f

    # 演算子を選択
    puts "\n演算子を選んでください（+, -, *, /）"
    print "> "
    operator = gets.chomp

    # 2つ目の数を入力
    puts "\n2つ目の数を入力してください"
    print "> "
    num2 = gets.to_f

    # 計算実行
    result = case operator
    when "+"
      num1 + num2
    when "-"
      num1 - num2
    when "*"
      num1 * num2
    when "/"
      if num2 == 0
        puts "エラー: 0で割ることはできません"
        next
      end
      num1 / num2
    else
      puts "エラー: 無効な演算子です"
      next
    end

    # 結果表示
    puts "\n計算結果: #{num1} #{operator} #{num2} = #{result}"

    # 続けるか聞く
    puts "\n続けて計算しますか？（y/n）"
    print "> "
    answer = gets.chomp
    break if answer != "y"
  end

  puts "\n計算機を終了します"
end

12.4 プログラム3: ToDoリスト

最後は、タスクを管理できる「ToDoリスト」を作ってみましょう。配列を使ってタスクを管理し、ファイルに保存して次回も使えるようにします。

機能一覧

  • タスクを追加する
  • タスク一覧を表示する
  • タスクを削除する
  • タスクをファイルに保存する
  • 起動時にファイルから読み込む

プログラムの実装

todo_list.rb 
# ToDoリスト管理プログラム
class TodoList
  def initialize
    @tasks = []
    @filename = "todos.txt"
    load_tasks
  end

  # タスク一覧を表示
  def show_tasks
    if @tasks.empty?
      puts "\nタスクはありません"
    else
      puts "\n=== タスク一覧 ==="
      @tasks.each_with_index do |task, index|
        puts "#{index + 1}. #{task}"
      end
    end
  end

  # タスクを追加
  def add_task
    print "\n追加するタスクを入力してください: "
    task = gets.chomp
    if task.empty?
      puts "タスクが入力されていません"
    else
      @tasks << task
      save_tasks
      puts "タスクを追加しました: #{task}"
    end
  end

  # タスクを削除
  def delete_task
    show_tasks
    return if @tasks.empty?

    print "\n削除するタスク番号を入力してください: "
    num = gets.to_i

    if num > 0 && num <= @tasks.length
      deleted = @tasks.delete_at(num - 1)
      save_tasks
      puts "タスクを削除しました: #{deleted}"
    else
      puts "無効な番号です"
    end
  end

  # ファイルに保存
  def save_tasks
    File.open(@filename, "w") do |file|
      @tasks.each { |task| file.puts(task) }
    end
  end

  # ファイルから読み込み
  def load_tasks
    if File.exist?(@filename)
      @tasks = File.readlines(@filename, chomp: true)
    end
  end

  # メニュー表示
  def show_menu
    puts "\n=== ToDoリスト ==="
    puts "1. タスク一覧"
    puts "2. タスク追加"
    puts "3. タスク削除"
    puts "4. 終了"
    print "> "
  end

  # メインループ
  def run
    loop do
      show_menu
      choice = gets.to_i

      case choice
      when 1
        show_tasks
      when 2
        add_task
      when 3
        delete_task
      when 4
        puts "\nToDoリストを終了します"
        break
      else
        puts "1〜4を入力してください"
      end
    end
  end
end

# プログラム起動
todo = TodoList.new
todo.run

クラスを活用した設計

このプログラムでは`TodoList`クラスを使って、タスク管理に必要な機能をまとめています。ファイルの読み書きも組み込むことで、プログラムを終了してもデータが残るようになっています。

12.5 プログラム4: おみくじプログラム

最後に、楽しいおみくじプログラムを作ってみましょう。ハッシュを使って運勢とメッセージを管理します。

omikuji.rb 
# おみくじプログラム
def omikuji
  # 運勢のデータ（ハッシュの配列）
  fortunes = [
    { level: "大吉", message: "最高の1日になるでしょう！", emoji: "🌟" },
    { level: "中吉", message: "良いことがありそうです", emoji: "😊" },
    { level: "小吉", message: "地道な努力が実を結びます", emoji: "🌱" },
    { level: "吉", message: "穏やかな1日になりそうです", emoji: "☀️" },
    { level: "末吉", message: "小さな幸せを見つけましょう", emoji: "🍀" },
    { level: "凶", message: "焦らず落ち着いて過ごしましょう", emoji: "🌧️" }
  ]

  puts "=== おみくじプログラム ==="

  loop do
    puts "\nEnterキーを押しておみくじを引いてください"
    gets

    # ランダムに運勢を選択
    result = fortunes.sample

    # 結果表示（演出付き）
    puts "\n占い中..."
    sleep(1)
    puts "."
    sleep(0.5)
    puts ".."
    sleep(0.5)
    puts "..."
    sleep(0.5)

    puts "\n" + "=" * 40
    puts "  #{result[:emoji]}  あなたの運勢は...【#{result[:level]}】  #{result[:emoji]}"
    puts "=" * 40
    puts "\nメッセージ: #{result[:message]}"
    puts "=" * 40

    # 続けるか聞く
    puts "\nもう一度引きますか？（y/n）"
    print "> "
    answer = gets.chomp
    break if answer != "y"
  end

  puts "\nおみくじを終了します。良い1日を！"
end

# おみくじ起動
omikuji

ハッシュの配列活用

複数のデータ（運勢、メッセージ、絵文字）をハッシュにまとめ、それを配列で管理しています。`sample`メソッドを使うとランダムに要素を取得できます。`sleep`メソッドで演出を加えることもできます。

12.6 プログラムを改造してみよう

ここまでで4つのプログラムを作成しました。次は、これらのプログラムを自分なりに改造してみましょう。改造のアイデアをいくつか紹介します。

じゃんけんの改造

  • 勝敗の記録を保存して統計を表示する
  • 3回勝負にして最終勝者を決める
  • あいこの回数をカウントする

計算機の改造

  • 累乗（べき乗）の計算を追加
  • 計算履歴を保存して表示する
  • 平方根や三角関数を追加

ToDoリストの改造

  • タスクに期限を設定できるようにする
  • 完了/未完了の状態を管理する
  • 優先度を設定して並び替える

おみくじの改造

  • 恋愛運、金運、健康運など項目別に占う
  • 名前を入力して個別のメッセージを表示
  • ラッキーカラーやラッキーナンバーを追加

改造のコツ

まずは小さな改造から始めましょう。「メッセージの文言を変える」「新しい選択肢を1つ追加する」など、できそうなことから試してみてください。動いたら次の改造に進みます。

理解度確認クイズ

  1. Q1. じゃんけんプログラムで、コンピュータがランダムに手を選ぶために使ったメソッドは何ですか？

    回答を表示

    `rand(3)`メソッドを使いました。これは0から2までのランダムな整数を生成します。配列のインデックスとして使うことで、ランダムに手を選べます。

  2. Q2. ToDoリストプログラムで、タスクをファイルに保存するために使ったメソッドは何ですか？

    回答を表示

    `File.open`メソッドを使って、書き込みモード("w")でファイルを開き、配列の各要素を`puts`で書き込みました。また、読み込みには`File.readlines`を使いました。

  3. Q3. おみくじプログラムで、配列からランダムに1つの要素を取得するために使ったメソッドは何ですか？

    回答を表示

    `sample`メソッドを使いました。配列に対して`.sample`を呼ぶと、配列からランダムに1つの要素を取得できます。簡単で便利なメソッドです。

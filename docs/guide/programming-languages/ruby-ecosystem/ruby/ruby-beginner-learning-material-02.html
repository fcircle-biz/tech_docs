環境構築 - Rubyをインストールして最初のプログラムを動かそう

Rubyのインストール方法を詳しく解説し、「Hello, World!」という最初のプログラムを作成・実行します。プログラムが動く喜びを体験しましょう。

この章で学ぶこと

  • Rubyのインストール方法（Windows/Mac/Linux）を理解する
  • インストールが正しく完了したか確認する方法を学ぶ
  • 最初のプログラム「Hello, World!」を実行する
  • Rubyプログラムをファイルに保存して実行する方法を習得する

2.1 環境構築の前に知っておくこと

プログラミングを始める前に、まずRubyをコンピュータにインストールする必要があります。これは、料理を始める前に調理器具を揃えるのと同じように、プログラミングに必要な道具を準備する作業です。

この章では、お使いのパソコンのOSに合わせて、Rubyをインストールする手順を丁寧に説明します。難しそうに感じるかもしれませんが、手順通りに進めれば10〜15分程度で完了します。

OSって何？

OS（Operating System）とは、パソコンを動かす基本ソフトのことです。Windows、macOS（Mac）、Linuxなどがあります。自分のパソコンがどのOSか分からない場合は、Windowsの場合「設定」→「システム」から、Macの場合は画面左上の「りんごマーク」→「このMacについて」から確認できます。

2.2 Windows環境でのRubyインストール

Windowsをお使いの場合は、RubyInstallerという専用のインストーラーを使うのが最も簡単です。

手順1: RubyInstallerのダウンロード

  1. 1 Webブラウザで「RubyInstaller for Windows」と検索するか、https://rubyinstaller.org/ にアクセスします
  2. 2 「Download」ボタンをクリックします
  3. 3 「WITH DEVKIT」と書かれた最新版（例: Ruby+Devkit 3.3.x (x64)）を選んでダウンロードします

DEVKITって必要なの？

DEVKITは、将来的に拡張機能を使う際に必要になる開発ツールです。初心者には難しく感じるかもしれませんが、「WITH DEVKIT」版を選んでおけば、後で困ることがありません。

手順2: RubyInstallerのインストール

  1. 1 ダウンロードした「rubyinstaller-devkit-x.x.x-x-x64.exe」ファイルをダブルクリックして実行します
  2. 2 「I accept the License」にチェックを入れて「Next」をクリックします
  3. 3 インストール先はデフォルトのまま（変更不要）で「Install」をクリックします
  4. 4 インストール完了後、「Run 'ridk install'」にチェックが入っていることを確認して「Finish」をクリックします
  5. 5 黒い画面（コマンドプロンプト）が表示されたら、何も入力せずEnterキーを押します（自動的に必要なコンポーネントがインストールされます）
  6. 6 処理が完了したら、もう一度Enterキーを押して画面を閉じます

2.3 Mac環境でのRubyインストール

Macには最初からRubyがインストールされていますが、バージョンが古い場合があります。最新バージョンをインストールするために、Homebrewというツールを使います。

手順1: Homebrewのインストール（未インストールの場合）

まず、Homebrewがインストールされているか確認します。「ターミナル」アプリを起動して、以下のコマンドを入力してください。

ターミナル 
# Homebrewのバージョンを確認
brew --version

バージョン番号が表示されれば、Homebrewは既にインストールされています。「command not found」と表示された場合は、以下のコマンドでHomebrewをインストールします。

ターミナル 
# Homebrewのインストール
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

ターミナルの開き方

Macで「ターミナル」を開くには、Launchpad（ロケットのアイコン）から「その他」→「ターミナル」をクリックするか、Spotlight検索（Command + スペースキー）で「ターミナル」と入力して開きます。

手順2: Rubyのインストール

Homebrewを使ってRubyをインストールします。ターミナルで以下のコマンドを実行してください。

ターミナル 
# Rubyのインストール
brew install ruby

インストールが完了するまで、数分かかる場合があります。コーヒーでも飲みながら気長に待ちましょう。

手順3: パスの設定

インストール後、新しいRubyを優先的に使うようにパスを設定します。使っているシェル（zshまたはbash）に応じて、以下のコマンドを実行してください。

ターミナル（zshの場合） 
# zshの場合（macOS Catalina以降のデフォルト）
echo 'export PATH="/usr/local/opt/ruby/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
ターミナル（bashの場合） 
# bashの場合
echo 'export PATH="/usr/local/opt/ruby/bin:$PATH"' >> ~/.bash_profile
source ~/.bash_profile

2.4 Linux環境でのRubyインストール

Linuxをお使いの場合は、ディストリビューションのパッケージマネージャーを使う方法が一般的です。ここでは代表的なUbuntu/Debianでの手順を紹介します。

Ubuntu/Debianの場合

ターミナルを開いて、以下のコマンドを順番に実行してください。

ターミナル 
# パッケージリストの更新
sudo apt update

# Rubyと関連ツールのインストール
sudo apt install ruby-full

Fedora/CentOS/RHELの場合

ターミナル 
# Rubyのインストール
sudo dnf install ruby

rbenvを使う方法も人気

より柔軟にRubyのバージョンを管理したい場合は、rbenvやRVMといったバージョン管理ツールを使う方法もあります。初心者の方は、まずは上記のシンプルな方法で始めて、慣れてきたら検討しましょう。

2.5 インストールの確認

Rubyが正しくインストールされたか確認しましょう。コマンドプロンプト（Windows）またはターミナル（Mac/Linux）を開いて、以下のコマンドを入力してください。

コマンドプロンプト / ターミナル 
# Rubyのバージョン確認
ruby --version

以下のような出力が表示されれば、インストール成功です！

実行結果の例 

ruby 3.3.0 (2023-12-25 revision 5124f9ac75) [x64-mingw-ucrt]

Windowsでコマンドプロンプトの開き方

Windowsキー + Rを押して、「cmd」と入力してEnterキーを押すか、スタートメニューから「コマンドプロンプト」で検索して開きます。

2.6 最初のプログラム「Hello, World!」を動かそう

それでは、記念すべき最初のRubyプログラムを実行してみましょう！プログラミングの世界では、最初に「Hello, World!」と画面に表示するプログラムを作るのが伝統です。

対話型実行環境（irb）を使う

Rubyにはirb（Interactive Ruby）という、対話形式でコードを試せる便利な機能があります。コマンドプロンプトまたはターミナルで、以下のコマンドを入力してください。

コマンドプロンプト / ターミナル 
# irbの起動
irb

「irb(main):001:0>」のような表示が出たら、irbが起動しています。次に、以下のコードを入力してEnterキーを押してください。

irb内で実行 
# 画面に文字を表示する
puts "Hello, World!"

動作確認

上記のコードを入力してEnterキーを押すと、以下のように表示されるはずです。

期待される結果 

Hello, World!
=> nil

「Hello, World!」と表示されましたか？おめでとうございます！これがあなたの最初のRubyプログラムです。putsは「画面に文字を表示する」という命令で、"Hello, World!"は表示する文字列です。

irbを終了するには、「exit」と入力してEnterキーを押すか、Ctrl + Dキーを押してください。

「=> nil」って何？

「=> nil」は、irbがコードの実行結果を表示しているものです。「nil」は「何もない」という意味で、putsは画面に文字を表示するだけで特に値を返さないため、nilが表示されます。気にしなくて大丈夫です。

2.7 プログラムをファイルに保存して実行してみよう

irbは便利ですが、プログラムを保存することができません。次は、プログラムをファイルに保存して実行する方法を学びましょう。

手順1: 作業用フォルダの作成

まず、Rubyのプログラムを保存するためのフォルダを作りましょう。デスクトップに「ruby_practice」というフォルダを作成してください。

手順2: プログラムファイルの作成

メモ帳（Windows）またはテキストエディット（Mac）を開いて、以下のコードを入力します。

hello.rb 
# 最初のRubyプログラム
puts "Hello, World!"
puts "ようこそ、Rubyの世界へ！"

# 簡単な計算もできます
puts "2 + 3 = #{2 + 3}"

このファイルを「hello.rb」という名前で、先ほど作った「ruby_practice」フォルダに保存してください。

ファイル名の注意点

Rubyのプログラムファイルは、必ず「.rb」という拡張子を付けてください。また、ファイル名は半角英数字で、スペースは含めないようにしましょう（例: hello.rb、my_program.rb）。

手順3: プログラムの実行

コマンドプロンプト（Windows）またはターミナル（Mac/Linux）を開いて、作業用フォルダに移動します。

Windowsの場合 
# デスクトップのruby_practiceフォルダに移動
cd Desktop\ruby_practice
Mac/Linuxの場合 
# デスクトップのruby_practiceフォルダに移動
cd ~/Desktop/ruby_practice

次に、以下のコマンドでプログラムを実行します。

コマンドプロンプト / ターミナル 
# Rubyプログラムの実行
ruby hello.rb

以下のように表示されれば成功です！

実行結果 

Hello, World!
ようこそ、Rubyの世界へ！
2 + 3 = 5

実習 2-1: 自己紹介プログラムを作ろう

自分の名前と好きなものを表示するプログラムを作成してみましょう。

手順

  1. 1 「my_profile.rb」という名前で新しいファイルを作成します
  2. 2 putsを使って、あなたの名前と好きなものを表示するコードを書きます（例: 「私の名前は○○です」「好きなものは○○です」）
  3. 3 「ruby my_profile.rb」コマンドで実行して、正しく表示されることを確認します

トラブルシューティング

「ruby: command not found」エラーが出る

原因: Rubyがインストールされていないか、パスが通っていません。

解決方法: もう一度インストール手順を確認してください。特にMacの場合は、パスの設定（2.3節 手順3）を確認しましょう。

「No such file or directory」エラーが出る

原因: ファイルが見つからない、またはフォルダの移動ができていません。

解決方法: ①ファイル名が正しいか確認（hello.rb等）、②cdコマンドで正しいフォルダに移動できているか確認（lsコマンドでファイル一覧を表示して確認できます）。

日本語が文字化けする

原因: ファイルの文字エンコーディングがUTF-8になっていない可能性があります。

解決方法: ファイルを保存する際に、文字コードを「UTF-8」に設定してください。メモ帳の場合は「名前を付けて保存」で「エンコード」を「UTF-8」に変更します。

理解度確認クイズ

  1. Q1. Rubyが正しくインストールされたか確認するコマンドは何ですか？

    回答を表示

    「ruby --version」コマンドです。このコマンドを実行すると、インストールされているRubyのバージョン番号が表示されます。

  2. Q2. 画面に文字を表示するために使うRubyの命令は何ですか？

    回答を表示

    「puts」です。例えば「puts "Hello, World!"」と書くと、「Hello, World!」と画面に表示されます。

  3. Q3. Rubyのプログラムファイルに付ける拡張子は何ですか？

    回答を表示

    「.rb」です。例えば、「hello.rb」「my_program.rb」のようにファイル名を付けます。

  4. Q4. 保存したRubyプログラム（hello.rb）を実行するコマンドは何ですか？

    回答を表示

    「ruby hello.rb」です。ファイルが保存されているフォルダに移動してから、このコマンドを実行します。

1.1 Pythonデータ分析とは

「データ分析」とは、大量のデータから意味のある情報を引き出す作業です。たとえば、「先月の売上が落ちた原因を数字で探る」「どの商品が最も利益に貢献しているかを調べる」といった問いに、データをもとに答えを出すのがデータ分析の目的です。

Pythonはデータ分析の分野で世界中のエンジニア・研究者から選ばれているプログラミング言語です。その理由は、強力なライブラリが充実しており、少ないコードで複雑なデータ操作を実現できるからです。

なぜPythonが選ばれるのか

Pythonは文法がシンプルで学びやすく、データ分析に特化したライブラリ（pandas, NumPy, matplotlib）が豊富に揃っています。また、機械学習（scikit-learn）や深層学習（TensorFlow, PyTorch）とも連携しやすく、データ分析から機械学習まで一貫して学べる環境が整っています。

1.2 主要ライブラリの役割と関係

Pythonデータ分析の世界では、主に3つのライブラリが中心的な役割を果たします。それぞれが異なる目的を持ちながら、互いに連携してデータ分析を支えています。

NumPy

高速な数値計算の基盤。配列（ndarray）を使った効率的な演算を提供します。

主な用途

  • ・行列・ベクトル演算
  • ・統計計算（平均・標準偏差）
  • ・pandasの内部エンジン

pandas

表形式データ操作の主役。ExcelのようなDataFrameでデータの読込・加工・集計を行います。

主な用途

  • ・CSV/Excelの読み書き
  • ・データの選択・フィルタ
  • ・グループ集計・結合

matplotlib

データ可視化の定番ライブラリ。折れ線・棒・散布図などさまざまなグラフを作成できます。

主な用途

  • ・折れ線・棒・散布図
  • ・ヒストグラム・箱ひげ図
  • ・グラフの保存・カスタマイズ

ライブラリの関係図

NumPyはpandasの土台となる数値計算エンジンです。pandasはNumPyの配列を内部で使いながら、表形式のデータ操作に特化した機能を提供します。

1.3 ライブラリのインストール

データ分析に必要なライブラリをインストールしましょう。Pythonの標準パッケージ管理ツール pip を使います。

仮想環境の使用を推奨

プロジェクトごとにライブラリのバージョンを管理するため、仮想環境（venv）の使用を推奨します。ただし、学習目的であればグローバル環境へのインストールでも問題ありません。

ターミナル / コマンドプロンプト 
# データ分析に必要なライブラリをまとめてインストール
pip install pandas numpy matplotlib jupyter

# インストール確認（バージョンを表示）
pip show pandas numpy matplotlib

インストールが完了したら、Pythonで正しくインポートできるか確認しましょう。

インポート確認 
# ライブラリのインポートと版確認
import numpy as np        # np というエイリアス（別名）で使用
import pandas as pd       # pd というエイリアスで使用
import matplotlib.pyplot as plt  # plt というエイリアスで使用

# バージョン確認
print(f"NumPy: {np.__version__}")
print(f"pandas: {pd.__version__}")
print(f"matplotlib: {plt.matplotlib.__version__}")

出力例

NumPy: 1.26.4
pandas: 2.2.1
matplotlib: 3.8.3

慣例としてのエイリアス

numpy を np、pandas を pd、matplotlib.pyplot を plt と略すのは、世界中のデータサイエンティストが使う慣例です。他の人のコードを読む際も役立つので、この書き方に慣れておきましょう。

1.4 Jupyter Notebookとは

Jupyter Notebookは、データ分析に最適な対話型の実行環境です。コードと実行結果、説明文、グラフをひとつのファイルにまとめて管理できます。

セル単位で実行

プログラムをコードセル単位で実行できます。エラーがあっても部分的に修正して再実行できるので、試行錯誤がしやすいです。

結果をその場で確認

データの形や内容、グラフをコードのすぐ下に表示します。データの変化を目で確認しながら進められます。

メモと一緒に管理

Markdownセルを使ってコードに説明文を付けられます。分析のプロセスと考察を一緒に残せます。

共有しやすい

.ipynb形式のファイルを共有すれば、他の人が同じ環境で再現できます。GitHub上でも表示可能です。

1.5 Jupyter Notebookの起動と基本操作

起動方法

jupyter notebook

覚えておきたいショートカット

1.6 最初のデータ分析コードを書いてみよう

実際にJupyter Notebookに入力して実行してみましょう。pandas・NumPy・matplotlibをすべて使った簡単な例です。

first_analysis.ipynb 
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 売上データを作成（各月の売上金額）
months = ['1月', '2月', '3月', '4月', '5月', '6月']
sales  = [120, 135, 160, 150, 175, 200]   # 単位: 万円

# pandas の DataFrame（表形式）に変換
df = pd.DataFrame({'月': months, '売上': sales})
print(df)

# NumPy で統計計算
print(f"\n平均売上: {np.mean(sales):.1f}万円")
print(f"最大売上: {np.max(sales)}万円")

# matplotlib でグラフを描画
plt.figure(figsize=(8, 4))
plt.plot(months, sales, marker='o', color='steelblue')
plt.title('月別売上推移')
plt.xlabel('月')
plt.ylabel('売上（万円）')
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

出力例（テキスト部分）

     月   売上
0    1月  120
1    2月  135
2    3月  160
3    4月  150
4    5月  175
5    6月  200

平均売上: 156.7万円
最大売上: 200万円

このように、数十行のコードで「データ作成 → 表確認 → 統計計算 → グラフ表示」の一連の流れが完成します。このガイドでは、この各ステップをより深く、実務で使えるレベルに掘り下げて学んでいきます。

理解度確認クイズ

  1. Q1. pandas・NumPy・matplotlibのそれぞれの主な役割を説明してください。

    回答を表示
    • NumPy: 高速な数値計算の基盤。配列（ndarray）による行列演算や統計計算を担当。pandasの内部エンジンでもある。
    • pandas: CSV・ExcelなどのデータをDataFrame（表形式）で読み込み、フィルタ・集計・結合などのデータ操作を担当。
    • matplotlib: 折れ線グラフ・棒グラフ・散布図などのグラフを描画してデータを可視化する。

  2. Q2. Jupyter Notebookで「セルを実行して次のセルへ移動する」ショートカットは何ですか？

    回答を表示

    Shift + Enter です。セルのコードを実行して、自動的に次のセルへ移動します。セルを実行してその場に留まる場合は Ctrl + Enter を使います。

  3. Q3. import pandas as pd と書く理由を答えてください。

    回答を表示

    as pd はエイリアス（別名）を付ける記法です。以降のコードで pandas.DataFrame の代わりに pd.DataFrame と短く書けるようになります。これは世界共通の慣例であり、他の人のコードとの一貫性を保つためにも重要です。

よくあるトラブルと解決方法

ModuleNotFoundError: No module named 'pandas'

原因: ライブラリがインストールされていない、または異なるPython環境にインストールされている。

解決方法: ターミナルで pip install pandas を実行する。Jupyter内から実行する場合は !pip install pandas と記述する。

Jupyter Notebookが起動しない / ブラウザが開かない

原因: ポートが既に使用されている、またはデフォルトブラウザが設定されていない。

解決方法: ターミナルに表示される URL（例: http://localhost:8888）をブラウザのアドレスバーに直接貼り付けて開く。

グラフが表示されない（matplotlib）

原因: Jupyter Notebook特有の設定が必要な場合がある。

解決方法: コードの冒頭に %matplotlib inline を追加するか、コードの最後に plt.show() を明示的に呼び出す。

Streamlitはじめの一歩 - 最初のWebアプリ

いよいよStreamlitを使ってWebアプリ開発を始めます。この章では、Streamlitのインストールから、ブラウザにタイトルや文章を表示する最初のWebアプリまでを学びます。

この章で学ぶこと

  • Streamlitをパソコンにインストールする
  • st.title、st.header、st.subheaderで見出しを表示する
  • st.writeで文章やデータを表示する
  • st.markdownで装飾された文章を表示する
  • streamlit runコマンドでアプリを起動する

6.1 Streamlitをインストールしよう

Streamlitは、Pythonのパッケージ管理ツール「pip」を使ってインストールします。コマンドラインを開いて、以下のコマンドを実行してください。

コマンドプロンプト / ターミナル 
pip install streamlit

インストールには数分かかります。以下のような表示が出ればインストール成功です。

実行結果例 
Successfully installed streamlit-1.xx.x

pipとは？

pipは、Pythonのパッケージ（便利なツールやライブラリ）をインストール・管理するためのツールです。Pythonをインストールすると自動的に付属してきます。

インストール確認

正しくインストールできたか確認しましょう。以下のコマンドを実行してください。

コマンドプロンプト / ターミナル 
streamlit --version

バージョン番号が表示されればインストール成功です。

実行結果例 
Streamlit, version 1.xx.x

6.2 最初のStreamlitアプリを作ろう

それでは、最初のStreamlitアプリを作成しましょう。テキストエディタを開いて、以下のコードを入力してください。

app.py 
# Streamlitをインポート
import streamlit as st

# タイトルを表示
st.title("私の最初のStreamlitアプリ")

# テキストを表示
st.write("こんにちは、Streamlitの世界へようこそ！")

ファイル名をapp.pyとして保存してください。

アプリを起動しよう

保存したファイルがあるフォルダで、コマンドラインを開いて以下のコマンドを実行してください。

コマンドプロンプト / ターミナル 
streamlit run app.py

すると、自動的にブラウザが開いて、以下のような画面が表示されます。

おめでとうございます！

ブラウザに「私の最初のStreamlitアプリ」というタイトルと、「こんにちは、Streamlitの世界へようこそ！」というメッセージが表示されていれば成功です。これが、あなたが作った最初のWebアプリです！

アプリの停止方法

Streamlitアプリを停止するには、コマンドラインで Ctrl + C（MacはCommand + C）を押してください。

6.3 コードの意味を理解しよう

先ほど書いたコードを1行ずつ見ていきましょう。

1. Streamlitのインポート

import streamlit as st

この行は、「Streamlitライブラリを使えるようにする」という宣言です。as stは「Streamlitをstという短い名前で呼べるようにする」という意味です。これにより、毎回streamlit.title()と長く書く代わりに、st.title()と短く書けます。

2. タイトルの表示

st.title("私の最初のStreamlitアプリ")

st.title()は、大きな見出し（タイトル）を表示する関数です。カッコの中に表示したい文字を書きます。

3. テキストの表示

st.write("こんにちは、Streamlitの世界へようこそ！")

st.write()は、テキストやデータを表示する万能な関数です。文字列だけでなく、数値、リスト、辞書、データフレームなど、様々なものを表示できます。

6.4 さまざまな表示方法を試そう

Streamlitには、テキストを表示するための様々な関数が用意されています。それぞれ試してみましょう。

見出しの階層

見出しには大きさの段階があります。

app.py 
import streamlit as st

# 一番大きな見出し
st.title("これがタイトルです")

# 大見出し
st.header("これが大見出しです")

# 中見出し
st.subheader("これが中見出しです")

# 通常のテキスト
st.write("これが通常のテキストです")

見出しの使い分け

st.title()はページ全体のタイトル、st.header()は大きなセクションの見出し、st.subheader()は小さなセクションの見出しとして使います。

Markdownで装飾

st.markdown()を使うと、文字を太字にしたり、色を変えたりできます。

app.py 
import streamlit as st

# Markdownで装飾されたテキスト
st.markdown("**太字のテキスト**")
st.markdown("*斜体のテキスト*")
st.markdown("~~取り消し線~~")
st.markdown(":red[赤い文字]")
st.markdown(":blue[青い文字]")
書き方 表示
**太字** 太字
*斜体* 斜体
~~取り消し線~~ 取り消し線
:red[赤い文字] 赤い文字

実習 6-1: 自己紹介アプリを作ろう

学んだ関数を使って、自己紹介アプリを作成しましょう。

課題

以下の要素を含むアプリを作成してください。

  • タイトル（st.title）: 「自己紹介」
  • 大見出し（st.header）: 「基本情報」
  • 通常のテキスト（st.write）: あなたの名前と年齢
  • 中見出し（st.subheader）: 「趣味」
  • Markdown（st.markdown）: 趣味を箇条書きで表示

ヒント（コード例）

import streamlit as st

st.title("自己紹介")
st.header("基本情報")
st.write("名前: 田中太郎")
st.write("年齢: 25歳")

st.subheader("趣味")
st.markdown("""
- 読書
- プログラミング
- サイクリング
""")

チャレンジ：Markdownで文字を太字や色付きにしてみましょう。

6.5 st.writeの万能性

st.write()は非常に便利な関数で、文字列だけでなく、様々なデータ型を自動的に適切な形で表示してくれます。

app.py 
import streamlit as st

# 文字列
st.write("これは文字列です")

# 数値
st.write(12345)

# 計算結果
st.write("10 + 20 =", 10 + 20)

# リスト
st.write([1, 2, 3, 4, 5])

# 辞書
st.write({"名前": "田中", "年齢": 25})

このように、st.write()は引数に何を渡しても、適切な形で表示してくれる万能な関数です。

flowchart LR A["st.write()"] --> B[文字列] A --> C[数値] A --> D[リスト] A --> E[辞書] A --> F[データフレーム] B --> G[適切な形式で表示] C --> G D --> G E --> G F --> G

実習 6-2: 計算アプリを作ろう

st.writeを使って、簡単な計算結果を表示するアプリを作りましょう。

課題

以下の計算結果を表示するアプリを作成してください。

  • タイトル: 「簡単計算機」
  • 100 + 50の計算結果
  • 200 - 75の計算結果
  • 10 × 8の計算結果
  • 100 ÷ 4の計算結果

ヒント（コード例）

import streamlit as st

st.title("簡単計算機")

st.write("100 + 50 =", 100 + 50)
st.write("200 - 75 =", 200 - 75)
st.write("10 × 8 =", 10 * 8)
st.write("100 ÷ 4 =", 100 / 4)

注意：Pythonでは掛け算は *、割り算は / を使います。

理解度確認クイズ

  1. Q1. Streamlitアプリを起動するコマンドは何ですか？

    回答を表示

    正解：streamlit run ファイル名.py
    例えば、app.pyを起動する場合は streamlit run app.py とコマンドラインで実行します。

  2. Q2. 大きな見出しを表示するために使う関数は何ですか？

    回答を表示

    正解：st.title()
    st.title("タイトル") とすることで、ページの一番大きな見出しを表示できます。

  3. Q3. Markdownで太字にするための記号は何ですか？

    回答を表示

    正解：**テキスト**
    例えば、st.markdown("**太字**") とすることで、太字のテキストを表示できます。

トラブルシューティング

「ModuleNotFoundError: No module named 'streamlit'」と表示される

原因：Streamlitがインストールされていません。
解決策：pip install streamlit を実行してインストールしてください。

ブラウザが自動的に開かない

原因：自動起動が無効になっている可能性があります。
解決策：コマンドラインに表示されるURL（http://localhost:8501）をブラウザに直接入力してください。

コードを変更したのに反映されない

原因：Streamlitが変更を検知していません。
解決策：ブラウザの「Rerun」ボタンをクリックするか、Ctrl + Cでアプリを停止して、再度 streamlit run app.py を実行してください。

第6章のまとめ

  • pip install streamlitでStreamlitをインストールできる
  • streamlit run ファイル名.pyでアプリを起動できる
  • st.title()でタイトル、st.header()で大見出し、st.subheader()で中見出しを表示できる
  • st.write()は万能な表示関数で、様々なデータ型を自動的に適切な形で表示してくれる
  • st.markdown()でテキストを太字や色付きにできる

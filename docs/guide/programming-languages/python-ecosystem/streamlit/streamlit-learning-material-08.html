第8章

ユーザーとの対話 - ボタンと入力フォーム

ボタンを押したら計算する、テキストを入力したら処理する、といったユーザーとの対話的な機能を実装します。Streamlitが提供する豊富な入力コンポーネントを使いこなしましょう。

この章で学ぶこと

  • st.button - ボタンをクリックしたときの処理
  • st.text_input - テキスト入力フォーム
  • st.number_input - 数値入力フォーム
  • st.selectbox - ドロップダウンリスト
  • st.slider - スライダー（範囲選択）
  • st.checkbox - チェックボックス

8.1 ボタン（st.button）

ボタンは、ユーザーがクリックすることで何か処理を実行させるための基本的なコンポーネントです。

基本的な使い方

button_basic.py 
import streamlit as st

# タイトル
st.title("ボタンの基本")

# ボタンを表示（押されたらTrueが返る）
if st.button("クリックしてください"):
    st.write("ボタンが押されました！")
    st.success("処理が実行されました")

ボタンの動作の仕組み

st.button() はボタンがクリックされた瞬間だけTrueを返し、次の再実行では再びFalseに戻ります。つまり、ボタンクリックは一時的なイベントとして扱われます。

ボタンの処理フロー

flowchart TD A[アプリ起動] --> B[ボタン表示] B --> C{ボタンクリック?} C -->|はい| D[処理実行] C -->|いいえ| E[待機状態] D --> F[結果表示] F --> B E --> B

実習 8-1: シンプルな挨拶ボタン

名前を入力してボタンを押すと挨拶メッセージを表示するアプリを作成してみましょう。

コード例

import streamlit as st

# タイトル
st.title("挨拶アプリ")

# テキスト入力
name = st.text_input("あなたの名前を入力してください")

# ボタン
if st.button("挨拶する"):
    if name:
        st.write(f"こんにちは、{name}さん！")
        st.balloons()  # 風船のアニメーション
    else:
        st.warning("名前を入力してください")

実行方法

streamlit run button_greeting.py

名前を入力してボタンを押すと、挨拶メッセージと風船のアニメーションが表示されます。入力が空の場合は警告メッセージが表示されます。

8.2 テキスト入力（st.text_input）

テキスト入力は、ユーザーが自由に文字列を入力できるフォームです。

基本的な使い方

text_input_basic.py 
import streamlit as st

st.title("テキスト入力の基本")

# 基本的なテキスト入力
name = st.text_input("名前")
st.write(f"入力された名前: {name}")

# プレースホルダー付き
email = st.text_input(
    "メールアドレス",
    placeholder="example@example.com"
)

# パスワード入力（文字が隠れる）
password = st.text_input(
    "パスワード",
    type="password"
)

テキストエリア（複数行入力）

text_area_example.py 
import streamlit as st

# 複数行のテキスト入力
comment = st.text_area(
    "コメント",
    placeholder="ご意見をお聞かせください",
    height=150
)

st.write(f"文字数: {len(comment)}")

8.3 数値入力（st.number_input）

数値入力は、数値のみを入力できるフォームです。最小値・最大値・増減幅を設定できます。

number_input_example.py 
import streamlit as st

st.title("数値入力の基本")

# 整数入力
age = st.number_input(
    "年齢",
    min_value=0,
    max_value=120,
    value=25,  # デフォルト値
    step=1     # 増減の単位
)

# 小数入力
height = st.number_input(
    "身長（cm）",
    min_value=0.0,
    max_value=300.0,
    value=170.0,
    step=0.1
)

st.write(f"年齢: {age}歳")
st.write(f"身長: {height}cm")

実習 8-2: BMI計算機

身長と体重を入力してBMI（体格指数）を計算するアプリを作成します。

コード例

import streamlit as st

st.title("BMI計算機")

# 身長と体重の入力
height_cm = st.number_input(
    "身長（cm）",
    min_value=50.0,
    max_value=250.0,
    value=170.0
)

weight = st.number_input(
    "体重（kg）",
    min_value=10.0,
    max_value=300.0,
    value=60.0
)

# 計算ボタン
if st.button("計算する"):
    # メートルに変換
    height_m = height_cm / 100
    # BMI計算
    bmi = weight / (height_m ** 2)

    st.write(f"あなたのBMI: {bmi:.1f}")

    # 判定
    if bmi < 18.5:
        st.info("低体重です")
    elif 18.5 <= bmi < 25:
        st.success("普通体重です")
    elif 25 <= bmi < 30:
        st.warning("肥満（1度）です")
    else:
        st.error("肥満（2度以上）です")

8.4 セレクトボックス（st.selectbox）

セレクトボックスは、あらかじめ用意された選択肢の中から1つを選ぶドロップダウンリストです。

selectbox_example.py 
import streamlit as st

st.title("セレクトボックスの使い方")

# 基本的なセレクトボックス
color = st.selectbox(
    "好きな色を選んでください",
    ["赤", "青", "緑", "黄色", "紫"]
)

st.write(f"選択された色: {color}")

# 複数選択（マルチセレクト）
hobbies = st.multiselect(
    "趣味を選んでください（複数選択可）",
    ["読書", "映画", "音楽", "スポーツ", "料理", "旅行"]
)

st.write(f"選択された趣味: {hobbies}")

8.5 スライダー（st.slider）

スライダーは、バーを左右に動かして値を選択するコンポーネントです。視覚的に値を調整できます。

slider_example.py 
import streamlit as st

st.title("スライダーの使い方")

# 整数スライダー
age = st.slider("年齢", 0, 100, 25)
st.write(f"年齢: {age}歳")

# 小数スライダー
temperature = st.slider(
    "温度（°C）",
    -10.0, 40.0, 20.0,
    step=0.5
)
st.write(f"温度: {temperature}°C")

# 範囲選択スライダー
time_range = st.slider(
    "時間帯を選択",
    0, 24, (9, 18)  # デフォルト: 9時〜18時
)
st.write(f"選択された時間帯: {time_range[0]}時〜{time_range[1]}時")

8.6 チェックボックス（st.checkbox）

チェックボックスは、ON/OFF（True/False）を切り替えるコンポーネントです。

checkbox_example.py 
import streamlit as st

st.title("チェックボックスの使い方")

# 基本的なチェックボックス
agree = st.checkbox("利用規約に同意します")

if agree:
    st.success("同意されました")
    st.button("次へ進む")
else:
    st.warning("同意が必要です")

# データ表示の切り替え
show_data = st.checkbox("詳細データを表示")

if show_data:
    st.write("詳細データ:")
    st.write({
        "名前": "山田太郎",
        "年齢": 30,
        "職業": "エンジニア"
    })

実習 8-3: アンケートフォーム

これまで学んだ入力コンポーネントを組み合わせて、総合的なアンケートフォームを作成します。

コード例

import streamlit as st

st.title("顧客満足度アンケート")

# 基本情報
st.header("基本情報")
name = st.text_input("お名前")
age = st.number_input("年齢", min_value=10, max_value=100, value=30)

# サービス満足度
st.header("サービス評価")
satisfaction = st.slider(
    "サービスの満足度",
    1, 5, 3,
    help="1: 不満 〜 5: 大変満足"
)

# 利用頻度
frequency = st.selectbox(
    "利用頻度",
    ["毎日", "週に数回", "月に数回", "初めて"]
)

# 改善点
st.header("ご意見")
feedback = st.text_area(
    "改善してほしい点",
    placeholder="自由にご記入ください"
)

# メール配信
newsletter = st.checkbox("メールマガジンを受け取る")

# 送信ボタン
if st.button("送信"):
    if name:
        st.success("アンケートを送信しました！")
        st.write("【送信内容】")
        st.write(f"- 名前: {name}")
        st.write(f"- 年齢: {age}")
        st.write(f"- 満足度: {satisfaction}/5")
        st.write(f"- 利用頻度: {frequency}")
        st.write(f"- 改善点: {feedback}")
        st.write(f"- メール配信: {'希望する' if newsletter else '希望しない'}")
        st.balloons()
    else:
        st.error("名前を入力してください")

8.7 入力コンポーネント一覧

コンポーネント 用途 戻り値

st.text_input 1行テキスト入力 文字列
st.text_area 複数行テキスト入力 文字列
st.number_input 数値入力 数値
st.selectbox 1つ選択 選択された値
st.multiselect 複数選択 リスト
st.slider 値の範囲選択 数値またはタプル
st.checkbox ON/OFF切り替え True/False
st.radio ラジオボタン選択 選択された値

st.time_input 時刻選択 時刻オブジェクト

理解度確認クイズ

  1. Q1. st.buttonの戻り値として正しいものはどれですか？

    回答を表示

    正解：ボタンがクリックされた瞬間だけTrueを返し、次の再実行ではFalseに戻る
    st.button()はボタンがクリックされた瞬間だけTrueを返す一時的なイベントです。状態を保持したい場合はst.session_stateを使用します。

  2. Q2. 複数行のテキストを入力したい場合、どのコンポーネントを使いますか？

    回答を表示

    正解：st.text_area()
    st.text_area()は複数行のテキスト入力に対応しています。st.text_input()は1行のみです。

  3. Q3. スライダーで範囲（開始〜終了）を選択するにはどうしますか？

    回答を表示

    正解：valueにタプル（開始値, 終了値）を指定する
    例：st.slider("時間帯", 0, 24, (9, 18)) のように、デフォルト値をタプルで指定すると範囲選択モードになります。

  • st.text_input / st.text_areaでテキスト入力を受け付けられる

  • st.selectbox / st.multiselectで選択肢から選ばせられる
  • st.sliderで視覚的に値を調整できる
  • st.checkboxでON/OFFの切り替えができる

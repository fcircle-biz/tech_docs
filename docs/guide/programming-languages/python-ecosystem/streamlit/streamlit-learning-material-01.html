1.1 Pythonとは何か？

Python（パイソン）は、世界中で広く使われているプログラミング言語です。特に初心者にとって学びやすい言語として知られています。

プログラミング言語とは、「コンピュータへの命令を書くための言葉」のことです。普段私たちが日本語や英語を使って会話するように、プログラミング言語を使ってコンピュータに指示を出します。

なぜPythonは初心者に優しいのか？

Pythonは「読みやすさ」を重視して設計された言語です。英語のような自然な書き方ができるため、プログラムを読むだけで何をしているのかが理解しやすいのです。

Pythonでできること

Webアプリ開発

ブラウザで動く便利なアプリケーションを作成

データ分析

数値データを集計・グラフ化して分析

AI・機械学習

画像認識や自然言語処理などのAI開発

業務自動化

繰り返し作業を自動化して効率化

1.2 Streamlitとは何か？

Streamlit（ストリームリット）は、Pythonを使ってWebアプリケーションを簡単に作成できるツール（ライブラリ）です。

通常、Webアプリを作るには複数のプログラミング言語（HTML、CSS、JavaScriptなど）を学ぶ必要があります。しかし、Streamlitを使えばPythonだけでWebアプリが作れます。

Streamlitの魅力

わずか数行のコードでグラフを表示したり、ボタンやスライダーなどの操作部品を配置したりできます。データ分析の結果を美しく見せる「ダッシュボード」も簡単に作成可能です。

Streamlitでできることのイメージ

flowchart LR A[Pythonコード] --> B[Streamlit] B --> C[グラフ表示] B --> D[ボタン・入力フォーム] B --> E[データ表示] B --> F[Webアプリとして公開]

1.3 Pythonをインストールしよう

それでは実際にPythonをパソコンにインストールしましょう。以下、Windows・Mac・Linuxそれぞれの手順を説明します。

インストール前の確認

すでにPythonがインストールされているか確認するには、コマンドプロンプト（またはターミナル）を開いて python --version と入力してください。バージョンが表示されれば既にインストール済みです。

Windowsの場合

  1. 1 Python公式サイトにアクセスし、「Download Python」ボタンをクリック
  2. 2 ダウンロードしたファイル（.exe）をダブルクリックして実行
  3. 3 重要：「Add Python to PATH」に必ずチェックを入れる
  4. 4 「Install Now」をクリックしてインストール開始

Macの場合

  1. 1 Python公式サイトにアクセスし、「Download Python」ボタンをクリック
  2. 2 ダウンロードしたファイル（.pkg）をダブルクリックして実行
  3. 3 画面の指示に従ってインストールを完了

インストール確認

インストールが完了したら、正しくインストールされたか確認しましょう。

コマンドプロンプト / ターミナル 
python --version

このコマンドを実行して、以下のようにバージョン番号が表示されればインストール成功です。

実行結果例 
Python 3.12.0

1.4 コマンドライン（黒い画面）に慣れよう

プログラミングでは、「コマンドライン」と呼ばれる文字だけの画面を使うことがよくあります。最初は不安かもしれませんが、基本的な操作は簡単です。

コマンドラインの開き方

Windows

スタートメニューから「コマンドプロンプト」または「PowerShell」を検索して起動

Mac / Linux

「ターミナル」アプリケーションを起動（Spotlightで「ターミナル」と検索）

基本コマンド

コマンド 意味 例
pwd 現在いるフォルダを表示 pwd
ls (Mac/Linux)
dir (Windows)		 現在のフォルダ内のファイル一覧を表示 ls
cd フォルダ名 指定したフォルダに移動 cd Documents
mkdir フォルダ名 新しいフォルダを作成 mkdir python_projects

実習 1-1: 最初のPythonプログラム「Hello World!」

プログラミングの世界では、最初に「Hello World!」と表示するプログラムを作るのが伝統です。一緒にやってみましょう！

手順

  1. 1 作業用のフォルダを作成します。コマンドラインで以下を実行してください。 
mkdir streamlit_learning
cd streamlit_learning
  1. 2 テキストエディタ（メモ帳など）を開き、以下のコードを入力してください。 
# 最初のPythonプログラム
print("Hello World!")
  1. 3 ファイル名を hello.py として保存（streamlit_learningフォルダ内に保存）
  2. 4 コマンドラインで以下のコマンドを実行 
python hello.py

期待される結果

Hello World!

おめでとうございます！これで最初のPythonプログラムを実行できました。print() はPythonで文字を表示するための命令です。

1.5 コードの意味を理解しよう

先ほど実行したコードを1行ずつ見ていきましょう。

hello.py 
# 最初のPythonプログラム
print("Hello World!")

1行目: コメント

# 最初のPythonプログラム

# で始まる行はコメントと呼ばれます。プログラムの実行には影響せず、人間がコードを理解しやすくするためのメモです。

2行目: print関数

print("Hello World!")

print() は画面に文字を表示する命令（関数）です。カッコの中に表示したい文字を "（ダブルクォーテーション）で囲んで書きます。

ヒント: 文字列の囲み方

Pythonでは文字列（テキスト）を "..."（ダブルクォート）または '...'（シングルクォート）で囲みます。どちらを使っても構いません。

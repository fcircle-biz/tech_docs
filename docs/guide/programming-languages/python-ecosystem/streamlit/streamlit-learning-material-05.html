関数 - 処理をまとめて名前をつける

よく使う処理を「関数」としてまとめることで、プログラムを整理しやすくする方法を学びます。料理のレシピのように、手順に名前をつけて再利用できるようになります。

この章で学ぶこと

  • 関数とは何か、なぜ便利なのかを理解する
  • def文を使って独自の関数を定義する
  • 引数と戻り値を使ってデータをやり取りする
  • 組み込み関数を活用して効率的にプログラムを書く

5.1 関数とは何か？

関数（かんすう）は、「よく使う処理をまとめて名前をつけたもの」です。一度関数を作っておけば、その名前を呼ぶだけで何度でも同じ処理を実行できます。

例えば、料理のレシピに「みじん切りにする」という手順があるとします。これを関数として定義しておけば、「玉ねぎをみじん切りにする」「ニンジンをみじん切りにする」のように、何度も同じ手順を繰り返し使えます。

関数のイメージ

flowchart LR A["入力（引数）"] --> B["関数（処理のかたまり）"] B --> C["出力（戻り値）"]

関数の3つのメリット

  • 1. 再利用性: 同じコードを何度も書かずに済む
  • 2. 可読性: プログラムが読みやすくなる
  • 3. 保守性: 修正が必要なときは関数の中だけ直せばよい

これまでに使った関数

実は、すでに多くの関数を使っています。

  • print() - 画面に文字を表示する関数
  • len() - リストや文字列の長さを調べる関数
  • range() - 数値の範囲を作る関数

これらは組み込み関数といって、Pythonに最初から用意されている関数です。この章では、自分で関数を作る方法を学びます。

5.2 関数を定義しよう（def文）

自分で関数を作るには、def文を使います。

関数の基本構文

関数定義の基本形 
def 関数名():
    # ここに実行したい処理を書く
    pass

最も簡単な関数の例

hello.py 
# greet関数を定義
def greet():
    print("こんにちは！")
    print("Pythonの世界へようこそ！")

# 関数を呼び出す
greet()
greet()  # 何度でも呼び出せる

このプログラムを実行すると、以下のように表示されます。

実行結果 
こんにちは！
Pythonの世界へようこそ！
こんにちは！
Pythonの世界へようこそ！

ポイント

関数を定義するだけでは実行されません。greet()のように関数名に()をつけて呼び出すことで、初めて実行されます。

実習 5-1: 自己紹介関数を作ろう

自己紹介を表示する関数を作成してみましょう。

手順

  1. 1 以下のコードを introduce.py として保存してください。 
# 自己紹介関数を定義
def introduce():
    print("私の名前は太郎です")
    print("趣味はプログラミングです")
    print("よろしくお願いします！")

# 関数を呼び出す
introduce()
  1. 2 実行してみましょう。 
python introduce.py

自分の名前や趣味に書き換えて、オリジナルの自己紹介関数を作ってみましょう！

5.3 引数 - 関数にデータを渡す

引数（ひきすう）は、関数に渡すデータのことです。引数を使うことで、同じ関数でも異なるデータで処理できます。

引数の基本

greet_name.py 
# 引数nameを受け取る関数
def greet(name):
    print(f"こんにちは、{name}さん！")

# 関数を呼び出す（引数に値を渡す）
greet("太郎")
greet("花子")
greet("次郎")
実行結果 
こんにちは、太郎さん！
こんにちは、花子さん！
こんにちは、次郎さん！

複数の引数を受け取る

関数は複数の引数を受け取ることもできます。引数はカンマ（,）で区切ります。

calc_area.py 
# 長方形の面積を計算する関数
def calc_rectangle_area(width, height):
    area = width * height
    print(f"幅{width}cm × 高さ{height}cm = 面積{area}cm²")

# 関数を呼び出す
calc_rectangle_area(5, 10)
calc_rectangle_area(8, 3)
実行結果 
幅5cm × 高さ10cm = 面積50cm²
幅8cm × 高さ3cm = 面積24cm²

引数の順序

引数は定義した順序で渡されます。calc_rectangle_area(5, 10)では、5がwidth、10がheightに渡されます。

5.4 戻り値 - 関数から結果を受け取る

戻り値（もどりち）は、関数が処理した結果を呼び出し元に返す値です。return文を使って戻り値を指定します。

戻り値の基本

add.py 
# 2つの数値を足し算する関数
def add(a, b):
    result = a + b
    return result  # 結果を返す

# 関数を呼び出して結果を受け取る
answer = add(10, 20)
print(f"10 + 20 = {answer}")

# 直接printでも使える
print(f"5 + 8 = {add(5, 8)}")
実行結果 
10 + 20 = 30
5 + 8 = 13

returnの有無による違い

returnなし

画面に表示するだけ。結果を変数に保存できない。

returnあり

結果を返す。変数に保存して後で使える。

returnの重要性

関数の結果を後で使いたい場合は、必ずreturnを使って値を返しましょう。これにより、関数を組み合わせて複雑な処理を作ることができます。

実習 5-2: 消費税計算関数を作ろう

価格を受け取って、消費税込みの金額を返す関数を作成しましょう。

# 消費税込み価格を計算する関数
def calc_tax_included(price):
    tax_rate = 0.1  # 消費税率10%
    total = price + (price * tax_rate)
    return total

# 関数を呼び出す
item1 = 1000
item2 = 2500

total1 = calc_tax_included(item1)
total2 = calc_tax_included(item2)

print(f"{item1}円 → {int(total1)}円（税込）")
print(f"{item2}円 → {int(total2)}円（税込）")

実行結果

1000円 → 1100円（税込）
2500円 → 2750円（税込）

消費税率を引数にして、税率を自由に変更できる関数に改造してみましょう！

5.5 便利な組み込み関数

Pythonには最初から用意されている組み込み関数がたくさんあります。よく使うものを覚えておくと、プログラミングがとても楽になります。

主要な組み込み関数

関数 機能 例
len() 長さを取得 len([1,2,3]) → 3
max() 最大値を取得 max(5, 3, 8) → 8
min() 最小値を取得 min(5, 3, 8) → 3
sum() 合計を計算 sum([1,2,3]) → 6
int() 整数に変換 int("123") → 123
str() 文字列に変換 str(123) → "123"
type() データ型を確認 type(123) → int

組み込み関数の使用例

builtin_functions.py 
# テストの点数リスト
scores = [85, 92, 78, 95, 88]

# 組み込み関数を使って集計
print(f"受験者数: {len(scores)}人")
print(f"最高点: {max(scores)}点")
print(f"最低点: {min(scores)}点")
print(f"合計点: {sum(scores)}点")
print(f"平均点: {sum(scores) / len(scores)}点")
実行結果 
受験者数: 5人
最高点: 95点
最低点: 78点
合計点: 438点
平均点: 87.6点

理解度確認クイズ

  1. Q1. 関数を定義するときに使うキーワードは何ですか？

    回答を表示

    正解：def
    Pythonではdef 関数名():の形式で関数を定義します。

  2. Q2. 次のコードの実行結果は何ですか？
    def double(x): return x * 2
    print(double(5))

    回答を表示

    正解：10
    関数doubleは引数を2倍にして返します。double(5)は5 * 2 = 10を返すので、10が表示されます。

  3. Q3. リスト[10, 20, 30]の合計を計算する組み込み関数は何ですか？

    回答を表示

    正解：sum()
    sum([10, 20, 30])は60を返します。sum関数はリストや数値の合計を計算する組み込み関数です。

第5章のまとめ

  • 関数は処理をまとめて名前をつけたもの
  • def 関数名():で関数を定義する
  • 引数でデータを関数に渡し、戻り値で結果を受け取る
  • return文で戻り値を指定する
  • Pythonにはlen()、max()、sum()などの便利な組み込み関数がある

データを見える化 - グラフと表の表示

数値データをグラフや表で視覚的に表示する方法を学びます。棒グラフ、折れ線グラフ、円グラフなど、様々な可視化手法を習得しましょう。

この章で学ぶこと

  • st.dataframeとst.tableを使った表の表示
  • st.line_chartで折れ線グラフを作成
  • st.bar_chartで棒グラフを作成
  • st.metricで数値指標を表示

9.1 データ可視化とは？

データ可視化（データビジュアライゼーション）とは、数値データをグラフや表などの視覚的な形式で表現することです。

数字だけを見るよりも、グラフで表示したほうが直感的に理解しやすくなります。例えば、売上データを棒グラフにすれば、どの月が好調だったかが一目でわかります。

データ可視化の利点

データを視覚化すると、傾向やパターンが見つけやすくなり、意思決定のスピードが上がります。また、他の人に説明するときにも、グラフがあると伝わりやすくなります。

Streamlitで使える主なデータ可視化機能

flowchart TB A[Streamlitデータ可視化] --> B[表の表示] A --> C[グラフの作成] A --> D[指標の表示] B --> E["st.dataframe（操作可能な表）"] B --> F["st.table（静的な表）"] C --> G["st.line_chart（折れ線）"] C --> H["st.bar_chart（棒グラフ）"] C --> I["st.area_chart（面グラフ）"] D --> J["st.metric（KPI表示）"]

9.2 表の表示 - st.dataframeとst.table

まずは、データを表形式で表示する方法を学びましょう。Streamlitには2つの表表示方法があります。

st.dataframe - 操作可能な表

st.dataframe() は、データを操作可能な表として表示します。列の並べ替えやスクロールができます。

app.py 
import streamlit as st
import pandas as pd

# データフレーム作成
df = pd.DataFrame({
    '商品名': ['りんご', 'バナナ', 'オレンジ', 'ぶどう'],
    '価格': [150, 100, 200, 350],
    '在庫数': [30, 50, 20, 15]
})

st.title("商品管理表")
st.dataframe(df)

pandasとは？

pandasは、表形式のデータを扱うためのPythonライブラリです。DataFrameというデータ構造を使って、Excelのような表データを簡単に操作できます。

st.table - 静的な表

st.table() は、データを静的な表として表示します。操作はできませんが、シンプルで見やすい表を作れます。

app.py 
import streamlit as st
import pandas as pd

# 辞書データから表を作成
data = {
    '月': ['1月', '2月', '3月'],
    '売上': [100, 150, 120]
}

st.title("月別売上")
st.table(data)

st.dataframeとst.tableの違い

機能 st.dataframe st.table
並べ替え 可能 不可
スクロール 可能 全行表示
見た目 インタラクティブ シンプル
用途 大量データ、探索 少量データ、レポート

実習 9-1: 学生の成績表を作成しよう

学生の氏名と各科目の成績を表形式で表示するアプリを作成してみましょう。

import streamlit as st
import pandas as pd

st.title("学生成績表")

# 成績データ
scores = pd.DataFrame({
    '氏名': ['田中太郎', '佐藤花子', '鈴木一郎'],
    '数学': [85, 92, 78],
    '英語': [78, 88, 90],
    '国語': [90, 85, 82]
})

st.dataframe(scores)

ヒント: このプログラムを実行して、表の列名をクリックしてみましょう。データが並べ替えられます！

9.3 折れ線グラフ - st.line_chart

折れ線グラフは、時間の経過に伴うデータの変化を表示するのに適しています。例えば、気温の推移や売上の変動などを表現できます。

app.py 
import streamlit as st
import pandas as pd

st.title("月別売上推移")

# 月別売上データ
sales_data = pd.DataFrame({
    '売上': [100, 120, 150, 135, 180, 200]
}, index=['1月', '2月', '3月', '4月', '5月', '6月'])

st.line_chart(sales_data)

indexパラメータとは？

DataFrameのindexは、各行に付ける「ラベル」のことです。グラフでは横軸（X軸）のラベルとして表示されます。

複数の線を表示

複数のデータ系列を同時に表示することもできます。

app.py 
import streamlit as st
import pandas as pd

st.title("店舗別売上比較")

# 複数店舗の売上データ
data = pd.DataFrame({
    '東京店': [100, 120, 150, 135, 180, 200],
    '大阪店': [90, 110, 130, 140, 160, 190]
}, index=['1月', '2月', '3月', '4月', '5月', '6月'])

st.line_chart(data)

実習 9-2: 気温の変化グラフを作成しよう

1週間の最高気温と最低気温を折れ線グラフで表示してみましょう。

import streamlit as st
import pandas as pd

st.title("週間気温推移")

# 気温データ
temp = pd.DataFrame({
    '最高気温': [25, 27, 26, 24, 28, 29, 27],
    '最低気温': [18, 19, 20, 17, 20, 21, 19]
}, index=['月', '火', '水', '木', '金', '土', '日'])

st.line_chart(temp)

確認ポイント: 2本の線が表示され、最高気温と最低気温の推移が一目でわかります。

9.4 棒グラフ - st.bar_chart

棒グラフは、カテゴリー別のデータを比較するのに適しています。商品別の売上や、地域別の人口などを視覚化できます。

app.py 
import streamlit as st
import pandas as pd

st.title("商品別売上")

# 商品別売上データ
products = pd.DataFrame({
    '売上': [150, 200, 120, 180]
}, index=['商品A', '商品B', '商品C', '商品D'])

st.bar_chart(products)

棒グラフの使い所

棒グラフは、「どの項目が一番大きいか」を一目で比較できるため、ランキングや比較を表現するのに最適です。

グループ別の棒グラフ

複数のカテゴリーを同時に比較することもできます。

app.py 
import streamlit as st
import pandas as pd

st.title("地域別売上比較")

# 地域別・商品別売上データ
data = pd.DataFrame({
    '商品A': [100, 120, 90],
    '商品B': [150, 130, 140]
}, index=['東京', '大阪', '名古屋'])

st.bar_chart(data)

実習 9-3: 科目別テストの平均点を表示しよう

各科目のクラス平均点を棒グラフで表示してみましょう。

import streamlit as st
import pandas as pd

st.title("科目別平均点")

# 平均点データ
scores = pd.DataFrame({
    '平均点': [78, 85, 72, 88, 80]
}, index=['数学', '英語', '国語', '理科', '社会'])

st.bar_chart(scores)

確認ポイント: 棒の高さを比較することで、どの科目の平均点が高いか低いかが一目でわかります。

9.5 数値指標の表示 - st.metric

st.metric()は、重要な数値指標（KPI: Key Performance Indicator）を大きく目立つように表示します。前回との差分も表示できます。

app.py 
import streamlit as st

st.title("売上ダッシュボード")

# メトリックを3列に配置
col1, col2, col3 = st.columns(3)

# 今月の売上
col1.metric("今月の売上", "250万円", "+20万円")

# 顧客数
col2.metric("顧客数", "1,250人", "+150人")

# 平均購入額
col3.metric("平均購入額", "2,000円", "-100円")

st.metricのパラメータ

  • label: 指標の名前
  • value: 現在の値
  • delta: 前回からの変化量（オプション）

deltaの表示色

deltaの値がプラスの場合は緑色（上昇）、マイナスの場合は赤色（下降）で表示されます。この色は自動的に設定されますが、意味を反転させることもできます。

app.py 
import streamlit as st

st.title("サーバーモニタリング")

# エラー率（低いほうが良い指標）
st.metric(
    "エラー率",
    "2.5%",
    "-0.5%",
    delta_color="inverse"  # 減少を良い変化として表示
)

実習 9-4: 健康管理ダッシュボードを作成しよう

体重、歩数、睡眠時間の3つの指標をメトリックで表示してみましょう。

import streamlit as st

st.title("健康管理ダッシュボード")

col1, col2, col3 = st.columns(3)

# 体重
col1.metric("体重", "65.2kg", "-0.5kg")

# 歩数
col2.metric("今日の歩数", "8,500歩", "+1,200歩")

# 睡眠時間
col3.metric("睡眠時間", "7.5時間", "+0.5時間")

確認ポイント: 3つの指標が横並びで表示され、それぞれの変化量が色付きで表示されます。

9.6 グラフと表を組み合わせる

実際のアプリケーションでは、グラフと表を組み合わせて使うことが多いです。グラフで全体像を把握し、表で詳細を確認する、という使い方が効果的です。

app.py 
import streamlit as st
import pandas as pd

st.title("売上分析レポート")

# データ作成
sales = pd.DataFrame({
    '売上': [100, 150, 130, 180, 200]
}, index=['1月', '2月', '3月', '4月', '5月'])

# グラフ表示
st.subheader("売上推移")
st.line_chart(sales)

# 表で詳細表示
st.subheader("詳細データ")
st.dataframe(sales)

総合実習: 店舗売上ダッシュボードを作成しよう

学んだすべての機能を使って、店舗の売上を分析するダッシュボードを作成してみましょう。

import streamlit as st
import pandas as pd

st.title("店舗売上ダッシュボード")

# KPI表示
col1, col2, col3 = st.columns(3)
col1.metric("今月の売上", "350万円", "+30万円")
col2.metric("来客数", "1,800人", "+200人")
col3.metric("客単価", "1,944円", "-56円")

# 月別売上推移
st.subheader("月別売上推移")
monthly = pd.DataFrame({
    '売上': [280, 300, 320, 310, 350]
}, index=['1月', '2月', '3月', '4月', '5月'])
st.line_chart(monthly)

# 商品別売上
st.subheader("商品別売上")
products = pd.DataFrame({
    '売上': [80, 120, 90, 60]
}, index=['商品A', '商品B', '商品C', '商品D'])
st.bar_chart(products)

# 詳細データ
st.subheader("詳細データ")
st.dataframe(monthly)

完成！ KPI表示、折れ線グラフ、棒グラフ、表を組み合わせた本格的なダッシュボードが完成しました。

理解度確認クイズ

  1. Q1. st.dataframeとst.tableの違いは何ですか？

    回答を表示

    正解：st.dataframeは操作可能（並べ替え、スクロール）、st.tableは静的な表示
    st.dataframeは大量データの探索に適しており、st.tableは少量データのレポートに適しています。

  2. Q2. 時間の経過に伴うデータの変化を表示するのに最適なグラフは？

    回答を表示

    正解：折れ線グラフ（st.line_chart）
    折れ線グラフは、時系列データの傾向や変化を視覚的に理解するのに最適です。

  3. Q3. st.metricで表示される変化量（delta）がマイナスのとき、通常は何色で表示されますか？

    回答を表示

    正解：赤色
    デフォルトでは、マイナスは赤色（下降）で表示されます。ただし、delta_color="inverse"を指定することで、色の意味を反転できます。

トラブルシューティング

エラー: No module named 'pandas'

原因: pandasがインストールされていません。
解決策: pip install pandas を実行してください。

グラフが表示されない

原因: データの形式が正しくない可能性があります。
解決策: データがDataFrame形式になっているか確認してください。print(type(data))で型を確認できます。

グラフの軸ラベルが文字化けする

原因: フォントの設定問題です。
解決策: Streamlitは自動的に日本語を処理しますが、ブラウザのフォント設定を確認してください。

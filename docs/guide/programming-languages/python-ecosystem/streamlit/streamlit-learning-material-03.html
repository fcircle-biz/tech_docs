条件分岐 - 「もし〜なら」の処理

プログラムを賢くする条件分岐を学びます。「もし点数が60点以上なら合格」のように、状況に応じて異なる処理を実行する方法を習得しましょう。

この章で学ぶこと

  • if文の基本的な書き方と使い方
  • 比較演算子（大きい・小さい・等しい）の使い方
  • else文とelif文による複数条件の処理
  • 条件分岐を使った実用的なプログラムの作成

3.1 条件分岐とは何か？

条件分岐とは、「もし〜なら〜する」という判断をプログラムに組み込むことです。日常生活でも私たちは常に条件分岐をしています。

日常生活の例

  • もし雨が降っているなら、傘を持っていく
  • もし気温が30度以上なら、冷房をつける
  • もし試験の点数が60点以上なら、合格とする

Pythonではif文を使って、このような条件による分岐を表現します。

条件分岐のイメージ

flowchart TD A[処理開始] --> B{条件を満たす?} B -->|はい| C[条件が真の場合の処理] B -->|いいえ| D[条件が偽の場合の処理] C --> E[処理終了] D --> E

3.2 if文の基本的な書き方

if文の基本的な書き方を見てみましょう。

if文の基本形 
# if文の基本形
if 条件式:
    # 条件が真の場合に実行される処理
    処理内容

重要なポイント

  • • if文の行末には必ずコロン(:)を付ける
  • • 条件が真の場合に実行する処理は、必ずインデント（字下げ）する
  • • インデントは通常、スペース4つまたはTab1つを使う

簡単な例

simple_if.py 
# 点数を変数に代入
score = 75

# 60点以上なら合格と表示
if score >= 60:
    print("合格です！")

この例では、scoreが60以上の場合に「合格です！」と表示されます。>=は「以上」を意味する比較演算子です。

3.3 比較演算子を使った条件式

条件式では比較演算子を使って、値を比較します。

演算子 意味 例 結果
== 等しい 5 == 5 True
!= 等しくない 5 != 3 True
> より大きい 7 > 5 True
< より小さい 3 < 5 True
>= 以上 5 >= 5 True
<= 以下 3 <= 5 True

よくある間違い

等しいかどうかを比較するときは==（イコール2つ）を使います。=（イコール1つ）は代入の意味なので注意しましょう。

実習 3-1: 年齢判定プログラム

年齢を入力して、成人かどうかを判定するプログラムを作ってみましょう。

# 年齢を変数に代入
age = 20

# 20歳以上なら成人と表示
if age >= 20:
    print("成人です")

# 18歳以上なら選挙権があると表示
if age >= 18:
    print("選挙権があります")

実行してみよう

  1. 1 上のコードをage_check.pyとして保存
  2. 2 python age_check.pyを実行
  3. 3 ageの値を15、18、25などに変更して再度実行し、結果の違いを確認

ヒント
age = 15の場合、どちらのメッセージも表示されません。age = 18の場合は「選挙権があります」だけが表示されます。

3.4 else文 - 条件が偽の場合の処理

else文を使うと、条件が満たされなかった場合の処理を書くことができます。「もし〜なら〜、そうでなければ〜」という分岐です。

if-else文の基本形 
if 条件式:
    # 条件が真の場合の処理
    処理A
else:
    # 条件が偽の場合の処理
    処理B

具体例

pass_or_fail.py 
# 試験の点数
score = 55

# 60点以上なら合格、そうでなければ不合格
if score >= 60:
    print("合格です！")
else:
    print("不合格です。次回頑張りましょう。")

処理の流れ

flowchart TD A[score = 55] --> B{score >= 60?} B -->|真 全角カッコTrue全角カッコ| C["print 全角カッコ合格です!全角カッコ"] B -->|偽 全角カッコFalse全角カッコ| D["print 全角カッコ不合格です...全角カッコ"] C --> E[終了] D --> E

3.5 elif文 - 複数の条件を判定する

elif文（else ifの略）を使うと、3つ以上の条件を順番に判定できます。

if-elif-else文の基本形 
if 条件式1:
    # 条件1が真の場合の処理
    処理A
elif 条件式2:
    # 条件1が偽で、条件2が真の場合の処理
    処理B
else:
    # すべての条件が偽の場合の処理
    処理C

具体例: 成績評価

grade_evaluation.py 
# 試験の点数
score = 75

# 点数に応じて評価を変える
if score >= 80:
    print("評価: 優")
elif score >= 70:
    print("評価: 良")
elif score >= 60:
    print("評価: 可")
else:
    print("評価: 不可")

この例では、scoreが75なので「評価: 良」が表示されます。条件は上から順番にチェックされ、最初に真となった条件の処理だけが実行されます。

elif文の動作

elif文は何個でも書けます。条件は上から順番に評価され、最初に真になった処理だけが実行されます。すべての条件が偽の場合、最後のelse文が実行されます。

実習 3-2: 温度による服装アドバイス

気温に応じて服装のアドバイスを表示するプログラムを作成します。

# 今日の気温（摂氏）
temperature = 18

# 気温に応じて服装をアドバイス
if temperature >= 30:
    print("とても暑いです。半袖で過ごしましょう。")
elif temperature >= 25:
    print("暑いです。薄着がおすすめです。")
elif temperature >= 20:
    print("過ごしやすい気温です。長袖シャツが良いでしょう。")
elif temperature >= 15:
    print("少し涼しいです。軽い上着があると良いでしょう。")
else:
    print("寒いです。コートやジャケットを着ましょう。")

課題

temperatureの値を5, 18, 26, 32などに変更して、それぞれどのメッセージが表示されるか確認してください。

確認ポイント
条件が上から順番にチェックされることを理解しましょう。temperature = 32の場合、最初の条件（>= 30）が真になるため、「とても暑いです」のメッセージだけが表示されます。

3.6 複数の条件を組み合わせる

論理演算子を使うと、複数の条件を組み合わせた判定ができます。

演算子 意味 例
and 両方とも真の場合に真 age >= 18 and age < 65
or どちらか一方が真の場合に真 day == "土" or day == "日"
not 条件を反転 not is_holiday

具体例: and演算子

discount.py 
# 年齢と会員かどうか
age = 65
is_member = True

# 60歳以上かつ会員なら割引適用
if age >= 60 and is_member:
    print("シニア会員割引が適用されます")
else:
    print("通常価格です")

具体例: or演算子

weekend.py 
# 今日の曜日
day = "土"

# 土曜日または日曜日なら休日
if day == "土" or day == "日":
    print("今日は休日です")
else:
    print("今日は平日です")

実習 3-3: 映画館の料金計算

年齢と曜日に応じて映画館の料金を表示するプログラムを作成しましょう。

料金設定

  • • 3歳未満: 無料
  • • 3歳以上12歳未満（小学生）: 1000円
  • • 12歳以上18歳未満（中高生）: 1500円
  • • 18歳以上60歳未満（一般）: 1800円
  • • 60歳以上（シニア）: 1200円
  • • ただし、水曜日は全員1100円（レディースデー）
# 年齢と曜日を設定
age = 25
day = "水"

# 水曜日は特別料金
if day == "水":
    price = 1100
    print(f"本日はレディースデー！料金: {price}円")
# 年齢に応じた料金設定
elif age < 3:
    price = 0
    print(f"3歳未満のお客様は無料です。料金: {price}円")
elif age < 12:
    price = 1000
    print(f"小学生料金: {price}円")
elif age < 18:
    price = 1500
    print(f"中高生料金: {price}円")
elif age < 60:
    price = 1800
    print(f"一般料金: {price}円")
else:
    price = 1200
    print(f"シニア料金: {price}円")

課題

  1. 1 age = 8, day = "金"として実行
  2. 2 age = 65, day = "月"として実行
  3. 3 age = 30, day = "水"として実行

応用課題
土日は200円増しにする条件を追加してみましょう。論理演算子orを使って実装できます。

理解度確認クイズ

  1. Q1. 次のコードで「大人」と表示されるageの値はどれですか？

    if age >= 20:
    print("大人")
else:
    print("子供")
    回答を表示

    正解：20以上の値（例: 20, 25, 30など）
    >= は「以上」を意味するため、ageが20ちょうどの場合も「大人」と表示されます。

  2. Q2. score = 75のとき、次のコードで何が表示されますか？

    if score >= 90:
    print("A")
elif score >= 70:
    print("B")
elif score >= 60:
    print("C")
else:
    print("D")
    回答を表示

    正解：B
    scoreは75なので、最初の条件（>= 90）は偽、2番目の条件（>= 70）が真となり、「B」が表示されます。

  3. Q3. age = 15, is_student = Trueのとき、次のコードで何が表示されますか？

    if age >= 18 and is_student:
    print("学生割引")
else:
    print("通常料金")
    回答を表示

    正解：通常料金
    and演算子は両方の条件が真の場合のみ真となります。age >= 18は偽（15 < 18）なので、全体として偽となり、elseの処理が実行されます。

第3章のまとめ

  • if文で条件による処理の分岐ができる
  • 比較演算子（==, !=, >, <, >=, <=）で値を比較できる
  • else文で条件が偽の場合の処理を書ける
  • elif文で複数の条件を順番に判定できる
  • 論理演算子（and, or, not）で複数条件を組み合わせられる

Pythonの超基礎 - 計算と文字の扱い方

プログラミングの基本となる「変数」「データ型」「計算」を、身近な例を使って学びます。Pythonで計算させたり、文字を表示させたりする方法を習得しましょう。

この章で学ぶこと

  • 変数を使ってデータに名前をつけて保存する方法
  • 数値と文字列（テキスト）の違いとそれぞれの使い方
  • Pythonで四則演算などの計算をする方法
  • print文でさまざまなデータを表示する方法

2.1 変数とは？ - データに名前をつける

変数（へんすう）とは、データを入れておく「箱」のようなものです。この箱に名前をつけて、後から何度でも使えるようにします。

例えば、「age（年齢）」という名前の箱に「25」という数値を入れたり、「name（名前）」という箱に「太郎」という文字を入れたりできます。

変数の作り方

variable_example.py 
# 変数に数値を代入
age = 25

# 変数に文字列を代入
name = "太郎"

# 変数の内容を表示
print(age)   # 25と表示される
print(name)  # 太郎と表示される

変数の命名ルール

変数名には英字、数字、アンダースコア（_）が使えます。ただし、数字から始めることはできません。また、日本語も使えますが、英語で書く習慣をつけると良いでしょう。

変数のイメージ図

flowchart LR A["age = 25"] --> B["箱の名前: age
中身: 25"] C["name = '太郎'"] --> D["箱の名前: name
中身: 太郎"]

2.2 データ型 - 数値と文字列の違い

Pythonでは、データに「型（かた）」があります。主に使う型は以下の2つです。

整数型（int）

整数型（int）は、小数点を含まない数値です。足し算や掛け算などの計算ができます。

int型の例 
# 整数を変数に代入
count = 100
year = 2024
temperature = -5

print(count)        # 100
print(year)         # 2024
print(temperature)  # -5

文字列型（str）

文字列型（str）は、文字や言葉を表すデータです。ダブルクォート（"）またはシングルクォート（'）で囲みます。

str型の例 
# 文字列を変数に代入
greeting = "こんにちは"
city = '東京'
message = "今日はいい天気ですね"

print(greeting)  # こんにちは
print(city)      # 東京
print(message)   # 今日はいい天気ですね

小数型（float）

小数型（float）は、小数点を含む数値です。

float型の例 
# 小数を変数に代入
price = 1980.5
height = 170.5
pi = 3.14

print(price)   # 1980.5
print(height)  # 170.5
print(pi)      # 3.14

よくある間違い

age = "25" のようにクォートで囲むと、これは数値ではなく文字列になります。計算したい場合は age = 25 のようにクォートを付けないでください。

2.3 計算をしてみよう

Pythonは電卓のように計算ができます。四則演算はもちろん、便利な演算子がたくさんあります。

基本的な演算子

演算子 意味 例 結果
+ 足し算 5 + 3 8
- 引き算 10 - 4 6
* 掛け算 7 * 2 14
/ 割り算 15 / 3 5.0
// 割り算（整数部分のみ） 17 // 5 3
% 余り 17 % 5 2
** べき乗 2 ** 3 8

計算のサンプルコード

calculation.py 
# 足し算
total = 100 + 50
print(total)  # 150

# 引き算
difference = 200 - 75
print(difference)  # 125

# 掛け算
product = 12 * 8
print(product)  # 96

# 割り算
quotient = 100 / 4
print(quotient)  # 25.0

計算の順序

Pythonでは数学と同じように、掛け算・割り算が足し算・引き算より優先されます。カッコ（）を使って計算の順序を変えることもできます。例: (2 + 3) * 4 は20になります。

実習 2-1: 買い物の合計金額を計算しよう

変数と計算を使って、買い物の合計金額を計算するプログラムを作ってみましょう。

手順

  1. 1 テキストエディタを開き、以下のコードを入力してください。 
# 商品の価格を変数に代入
apple_price = 150    # りんご1個の価格
banana_price = 120   # バナナ1房の価格
milk_price = 200     # 牛乳1本の価格

# 合計金額を計算
total = apple_price + banana_price + milk_price

# 結果を表示
print("りんご:", apple_price, "円")
print("バナナ:", banana_price, "円")
print("牛乳:", milk_price, "円")
print("合計:", total, "円")
  1. 2 ファイル名を shopping.py として保存
  2. 3 コマンドラインで実行 
python shopping.py

期待される結果

りんご: 150 円
バナナ: 120 円
牛乳: 200 円
合計: 470 円

価格を変えて、何度か実行してみましょう。変数の値を変えると、自動的に合計も変わることが確認できます。

2.4 文字列の操作

文字列は結合したり、繰り返したりすることができます。

文字列の結合

+ 演算子で文字列同士をつなげることができます。

文字列の結合 
# 文字列を結合
first_name = "太郎"
last_name = "山田"
full_name = last_name + first_name
print(full_name)  # 山田太郎

# 空白を入れて結合
greeting = "こんにちは、" + full_name + "さん"
print(greeting)  # こんにちは、山田太郎さん

文字列の繰り返し

* 演算子で文字列を繰り返すことができます。

文字列の繰り返し 
# 文字列を繰り返す
star = "★"
stars = star * 5
print(stars)  # ★★★★★

# 線を引く
line = "-" * 20
print(line)  # --------------------

f文字列（フォーマット文字列）

f文字列を使うと、変数の値を文字列の中に簡単に埋め込めます。文字列の前に f をつけ、変数を {変数名} で囲みます。

f文字列の使用例 
# f文字列の使用例
name = "太郎"
age = 25
height = 175.5

# f文字列で変数を埋め込む
message = f"{name}さんは{age}歳で、身長は{height}cmです"
print(message)  # 太郎さんは25歳で、身長は175.5cmです

f文字列の便利さ

f文字列を使うと、+ で文字列を繋ぐよりも読みやすく、書きやすいコードになります。現代のPythonコードではf文字列がよく使われます。

2.5 print関数の使い方

第1章で学んだ print() 関数をもう少し詳しく見てみましょう。

複数の値を表示する

複数の値を表示 
# カンマで区切って複数の値を表示
name = "太郎"
age = 25
print("名前:", name, "年齢:", age)  # 名前: 太郎 年齢: 25

# 計算結果も一緒に表示
x = 10
y = 20
print("合計:", x + y)  # 合計: 30

改行を制御する

改行の制御 
# 通常のprint（改行される）
print("1行目")
print("2行目")

# 改行しないprint
print("横に", end=" ")
print("並ぶ", end=" ")
print("表示")  # 横に 並ぶ 表示

理解度確認クイズ

  1. Q1. x = 10 という文で、x は何と呼ばれますか？

    回答を表示

    正解：変数
    変数はデータを保存しておく「箱」のようなもので、好きな名前をつけて何度でも使うことができます。

  2. Q2. 17 % 5 の計算結果はいくつですか？

    回答を表示

    正解：2
    % は余りを求める演算子です。17を5で割ると3余り2なので、答えは2です。

  3. Q3. "Hello" * 3 の結果は何ですか？

    回答を表示

    正解：HelloHelloHello
    文字列に対して * 演算子を使うと、その文字列を指定回数繰り返します。

第2章のまとめ

  • 変数はデータに名前をつけて保存する仕組み
  • データには型があり、整数型（int）、小数型（float）、文字列型（str）などがある
  • Pythonで四則演算（+、-、*、/）や余り（%）、べき乗（**）などの計算ができる
  • 文字列は結合（+）や繰り返し（*）ができる
  • f文字列を使うと変数を文字列に簡単に埋め込める

テンプレート - 動的なHTMLページを作る

Djangoテンプレート言語（DTL）、変数の表示とフィルタ、テンプレートタグによる制御、テンプレートの継承とインクルードを習得します。

この章で学ぶこと

  • Djangoテンプレート言語（DTL）の基本構文と役割を理解する
  • 変数の表示方法とフィルタを使ったデータの加工方法を習得する
  • forループ・if文などのテンプレートタグの使い方を学ぶ
  • テンプレート継承により共通レイアウトを効率的に管理する方法を理解する

5.1 Djangoテンプレートとは

Djangoテンプレートは、動的なHTMLページを生成するための仕組みです。ビュー（View）から受け取ったデータをHTML内に埋め込み、ユーザーに表示するコンテンツを柔軟に変化させることができます。

Djangoテンプレート言語（DTL: Django Template Language）は、HTMLの中にPythonのような記法でロジックやデータを埋め込むための専用の構文です。シンプルで覚えやすく、セキュリティにも配慮された設計になっています。

テンプレートの処理フロー

エンジン] C[テンプレート
ファイル] -->|HTMLテンプレート| B B -->|データ埋め込み| D[動的HTML] D -->|HTTPレスポンス| E[ユーザー] style A fill:#dbeafe,stroke:#3b82f6 style B fill:#fef3c7,stroke:#f59e0b style C fill:#dcfce7,stroke:#10b981 style D fill:#fce7f3,stroke:#ec4899 style E fill:#e0f2fe,stroke:#0ea5e9

Djangoテンプレートの3つの基本構文

1. 変数: {{ "{{" }} variable {{ "}}" }}

ビューから渡されたデータを表示します

2. タグ: {% raw %}{% tag %}{% endraw %}

制御構造（ループ、条件分岐など）を実装します

3. フィルタ: {{ "{{" }} variable|filter {{ "}}" }}

変数の値を加工・変換します

5.2 変数の表示とコンテキストデータ

ビューからテンプレートにデータを渡すには、コンテキストデータを使用します。コンテキストは辞書形式で、テンプレート内で変数として利用できます。

ビューからコンテキストデータを渡す

views.py 
from django.shortcuts import render

def article_detail(request, article_id):
    # コンテキストデータを辞書形式で定義
    context = {
        'title': 'Djangoテンプレートの使い方',
        'author': '山田太郎',
        'publish_date': '2025-01-15',
        'views': 1234,
    }
    # テンプレートにコンテキストを渡す
    return render(request, 'article.html', context)

テンプレートで変数を表示する

templates/article.html 
<!DOCTYPE html>
<html>
<head>
    <title>{{ "{{" }} title {{ "}}" }}</title>
</head>
<body>
    <h1>{{ "{{" }} title {{ "}}" }}</h1>
    <p>著者: {{ "{{" }} author {{ "}}" }}</p>
    <p>公開日: {{ "{{" }} publish_date {{ "}}" }}</p>
    <p>閲覧数: {{ "{{" }} views {{ "}}" }}回</p>
</body>
</html>

オブジェクトの属性やメソッドにアクセス

ドット記法を使用して、オブジェクトの属性やメソッド、辞書のキー、リストのインデックスにアクセスできます。

{{ "{{" }} user.username {{ "}}" }} {{ "{{" }} article.get_category {{ "}}" }} {{ "{{" }} data.key {{ "}}" }} {{ "{{" }} items.0 {{ "}}" }}

5.3 テンプレートフィルタでデータを加工する

フィルタは、変数の値を加工・変換するための機能です。パイプ記号（|）を使って変数に適用します。

よく使われるフィルタ

lower / upper

文字列を小文字/大文字に変換

{{ "{{" }} title|lower {{ "}}" }} {{ "{{" }} name|upper {{ "}}" }}

date

日付を指定フォーマットで表示

{{ "{{" }} publish_date|date:"Y年m月d日" {{ "}}" }}

例: 2025年01月15日

length

リストや文字列の長さを取得

{{ "{{" }} articles|length {{ "}}" }}

例: 5（記事数）

default

値が空の場合のデフォルト値を指定

{{ "{{" }} bio|default:"未入力" {{ "}}" }}

bioが空なら「未入力」を表示

truncatewords

指定した単語数で切り詰め

{{ "{{" }} content|truncatewords:30 {{ "}}" }}

最初の30語のみ表示

safe

HTMLエスケープを無効化

{{ "{{" }} html_content|safe {{ "}}" }}

HTMLタグをそのまま表示

フィルタの連鎖（チェーン）

複数のフィルタを連続して適用できます。左から順番に処理されます。

{{ "{{" }} title|lower|truncatewords:5 {{ "}}" }}

1. titleを小文字化 → 2. 最初の5単語に切り詰め

safeフィルタの使用に注意

safeフィルタは、信頼できるHTML以外には使用しないでください。ユーザー入力をそのまま表示すると、XSS（クロスサイトスクリプティング）攻撃の危険があります。

5.4 テンプレートタグで制御構造を実装する

テンプレートタグは、ループや条件分岐などの制御構造をテンプレート内で実装するための機能です。{% raw %}{% %}{% endraw %}で囲んで記述します。

forタグ - リスト要素をループ処理

テンプレート 
<ul>
    {% raw %}{% for article in articles %}{% endraw %}
        <li>
            <h3>{{ "{{" }} article.title {{ "}}" }}</h3>
            <p>{{ "{{" }} article.author {{ "}}" }} - {{ "{{" }} article.date {{ "}}" }}</p>
        </li>
    {% raw %}{% endfor %}{% endraw %}
</ul>

forループで使える特殊変数

{{ "{{" }} forloop.counter {{ "}}" }}

現在の反復回数（1から始まる）

{{ "{{" }} forloop.counter0 {{ "}}" }}

現在の反復回数（0から始まる）

{{ "{{" }} forloop.first {{ "}}" }}

最初の反復ならTrue

{{ "{{" }} forloop.last {{ "}}" }}

最後の反復ならTrue

emptyタグ - リストが空の場合の処理

テンプレート 
<ul>
    {% raw %}{% for article in articles %}{% endraw %}
        <li>{{ "{{" }} article.title {{ "}}" }}</li>
    {% raw %}{% empty %}{% endraw %}
        <li>記事がありません</li>
    {% raw %}{% endfor %}{% endraw %}
</ul>

ifタグ - 条件分岐

テンプレート 
{% raw %}{% if user.is_authenticated %}{% endraw %}
    <p>ようこそ、{{ "{{" }} user.username {{ "}}" }}さん</p>
    <a href="/logout/">ログアウト</a>
{% raw %}{% else %}{% endraw %}
    <a href="/login/">ログイン</a>
{% raw %}{% endif %}{% endraw %}

ifタグで使える比較演算子と論理演算子

比較演算子

==, !=, <, >, <=, >=

論理演算子

and, or, not
{% raw %}{% if age >= 20 and has_license %}{% endraw %} 運転可能です {% raw %}{% endif %}{% endraw %}

5.5 テンプレートの継承 - 共通レイアウトを再利用する

テンプレート継承は、複数のページで共通するレイアウト（ヘッダー、フッター、ナビゲーションなど）を効率的に管理する仕組みです。親テンプレート（base.html）を作成し、子テンプレートでそれを継承します。

テンプレート継承の仕組み

flowchart TD A[base.html
親テンプレート] -->|継承| B[home.html
子テンプレート] A -->|継承| C[about.html
子テンプレート] A -->|継承| D[contact.html
子テンプレート] E[共通部分
ヘッダー・フッター] -.含まれる.- A F[個別コンテンツ
トップページ内容] -.上書き.- B G[個別コンテンツ
会社概要内容] -.上書き.- C H[個別コンテンツ
お問い合わせ内容] -.上書き.- D style A fill:#fef3c7,stroke:#f59e0b style B fill:#dbeafe,stroke:#3b82f6 style C fill:#dbeafe,stroke:#3b82f6 style D fill:#dbeafe,stroke:#3b82f6

ステップ1: 親テンプレートを作成

templates/base.html 
<!DOCTYPE html>
<html>
<head>
    <title>{% raw %}{% block title %}{% endraw %}サイトタイトル{% raw %}{% endblock %}{% endraw %}</title>
</head>
<body>
    <header>
        <h1>マイサイト</h1>
        <nav>ナビゲーション</nav>
    </header>

    <main>
        {% raw %}{% block content %}{% endraw %}
        <!-- ここに各ページの内容が入ります -->
        {% raw %}{% endblock %}{% endraw %}
    </main>

    <footer>
        <p>© 2025 マイサイト</p>
    </footer>
</body>
</html>

ステップ2: 子テンプレートで継承

templates/home.html 
{% raw %}{% extends "base.html" %}{% endraw %}

{% raw %}{% block title %}{% endraw %}ホーム - マイサイト{% raw %}{% endblock %}{% endraw %}

{% raw %}{% block content %}{% endraw %}
    <h2>ようこそ！</h2>
    <p>これはホームページです。</p>
{% raw %}{% endblock %}{% endraw %}

継承のポイント

  • {% raw %}{% extends %}{% endraw %}は必ずテンプレートの最初の行に記述
  • 親テンプレートの{% raw %}{% block %}{% endraw %}を子テンプレートで上書き
  • blockタグ外の内容は親テンプレートのまま

5.6 テンプレートのインクルード - 部品を再利用する

インクルードは、テンプレートの一部を別ファイルに分割し、複数のページで再利用する機能です。共通の部品（サイドバー、コメント欄など）をモジュール化できます。

部品テンプレートを作成

templates/sidebar.html 
<aside>
    <h3>人気記事</h3>
    <ul>
        {% raw %}{% for article in popular_articles %}{% endraw %}
            <li>{{ "{{" }} article.title {{ "}}" }}</li>
        {% raw %}{% endfor %}{% endraw %}
    </ul>
</aside>

メインテンプレートでインクルード

templates/article_detail.html 
{% raw %}{% extends "base.html" %}{% endraw %}

{% raw %}{% block content %}{% endraw %}
    <article>
        <h2>{{ "{{" }} article.title {{ "}}" }}</h2>
        <p>{{ "{{" }} article.content {{ "}}" }}</p>
    </article>

    <!-- サイドバーをインクルード -->
    {% raw %}{% include "sidebar.html" %}{% endraw %}
{% raw %}{% endblock %}{% endraw %}

withパラメータでデータを渡す

テンプレート 
<!-- インクルード先に変数を渡す -->
{% raw %}{% include "comment.html" with comment=article.latest_comment %}{% endraw %}

継承とインクルードの使い分け

テンプレート継承（extends）

  • • ページ全体の骨格を共有
  • • 1つの親テンプレートのみ継承可能
  • • 用途: ヘッダー・フッター等の共通レイアウト

インクルード（include）

  • • 部品テンプレートを埋め込み
  • • 複数の部品を何度でも使用可能
  • • 用途: サイドバー、コメント欄等の部品

実習: ブログ記事一覧ページの作成

テンプレートの継承とフィルタを使って、ブログ記事一覧ページを作成してみましょう。

ステップ1: ビューでコンテキストデータを準備

# views.py
def article_list(request):
    articles = [
        {'title': 'Python入門', 'date': '2025-01-10', 'views': 1500},
        {'title': 'Django基礎', 'date': '2025-01-12', 'views': 2300},
        {'title': 'テンプレート活用', 'date': '2025-01-15', 'views': 800},
    ]
    return render(request, 'articles.html', {'articles': articles})

ステップ2: テンプレートで一覧表示

{% raw %}{% extends "base.html" %}{% endraw %}

{% raw %}{% block content %}{% endraw %}
<h2>記事一覧</h2>
{% raw %}{% for article in articles %}{% endraw %}
    <div>
        <h3>{{ "{{" }} forloop.counter {{ "}}" }}. {{ "{{" }} article.title {{ "}}" }}</h3>
        <p>公開日: {{ "{{" }} article.date|date:"Y年m月d日" {{ "}}" }}</p>
        <p>閲覧数: {{ "{{" }} article.views {{ "}}" }}回</p>
    </div>
{% raw %}{% empty %}{% endraw %}
    <p>記事がありません</p>
{% raw %}{% endfor %}{% endraw %}
{% raw %}{% endblock %}{% endraw %}

理解度確認クイズ

  1. Q1. テンプレートで変数を表示する際、どの記号で囲みますか？

    回答を表示

    二重中括弧 {{ "{{" }} {{ "}}" }} で囲みます。例: {{ "{{" }} title {{ "}}" }}

  2. Q2. 日付を「2025年01月15日」形式で表示するにはどのフィルタを使いますか？

    回答を表示

    dateフィルタを使用します。
    記述例: {{ "{{" }} publish_date|date:"Y年m月d日" {{ "}}" }}

  3. Q3. テンプレート継承で親テンプレートを指定するタグは？

    回答を表示

    {% raw %}{% extends "親テンプレート名" %}{% endraw %}タグを使用します。必ずテンプレートの最初の行に記述する必要があります。

  4. Q4. リストが空の場合のメッセージを表示するために、forタグと組み合わせて使うタグは？

    回答を表示

    {% raw %}{% empty %}{% endraw %}タグです。forループ内で使用し、イテレート対象のリストが空の場合に実行される内容を記述します。

第5章のまとめ

  • Djangoテンプレート言語（DTL）には、変数表示・タグ・フィルタの3つの基本構文があり、動的なHTMLページを生成できます。
  • フィルタを使うことで、日付フォーマット、文字列変換、長さ取得など、変数の値を柔軟に加工できます。
  • forタグとifタグにより、ループ処理や条件分岐をテンプレート内で実装できます。
  • テンプレート継承を使えば、共通レイアウトを効率的に管理し、コードの重複を減らせます。
  • インクルードにより、サイドバーやコメント欄などの部品を再利用可能なモジュールとして管理できます。

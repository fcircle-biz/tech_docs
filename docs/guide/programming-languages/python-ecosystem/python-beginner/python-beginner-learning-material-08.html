第8章

辞書 - データに名前を付けて整理しよう

電話帳や住所録のように、名前（キー）と値（バリュー）のペアでデータを管理する辞書型の使い方を学びます。効率的なデータ管理方法を身につけます。

この章で学ぶこと

  • 辞書（dict）とは何か、リストとの違いを理解する
  • 辞書の作成方法と基本的な使い方を習得する
  • キーと値の追加・取得・削除・更新の方法を学ぶ
  • 辞書の便利なメソッドと実用的な活用方法を習得する

8.1 辞書とは？リストとの違い

辞書（dict）は、キー（key）と値（value）のペアでデータを管理するデータ構造です。 電話帳で名前から電話番号を探すように、キーを指定することで対応する値を取り出すことができます。

身近な例で理解する

電話帳を例にしてみましょう。「田中」という名前（キー）で検索すると「090-1234-5678」（値）が見つかります。 辞書も同じように、キーを使って対応する値を素早く取り出せます。 リストは番号（インデックス）で管理しますが、辞書は意味のある名前（キー）で管理できるのが特徴です。

リストと辞書の違い

graph TB subgraph リスト L1["fruits[0] = 'りんご'"] L2["fruits[1] = 'バナナ'"] L3["fruits[2] = 'みかん'"] L1 --> L2 --> L3 end subgraph 辞書 D1["'田中': '090-1234-5678'"] D2["'佐藤': '080-9876-5432'"] D3["'鈴木': '070-1111-2222'"] end

リストは順序を持つデータの集まりで、0から始まるインデックス番号でアクセスします。 一方、辞書は順序を気にせず、キーという名前でデータにアクセスできます。 「3番目の要素」ではなく「田中さんの電話番号」という自然な形でデータを扱えます。

8.2 辞書の作成と基本操作

辞書の作成方法

辞書は{}（波括弧）を使って作成します。 キー: 値の形式でペアを定義します。

dict_basic.py - 辞書の作成 
# 空の辞書を作成
phonebook = {}

# 最初から要素を入れて作成
phonebook = {
    "田中": "090-1234-5678",
    "佐藤": "080-9876-5432",
    "鈴木": "070-1111-2222"
}

# 辞書の内容を表示
print(phonebook)
# 出力: {'田中': '090-1234-5678', '佐藤': '080-9876-5432', '鈴木': '070-1111-2222'}

値の取得

辞書から値を取り出すには、キーを指定します。存在しないキーにアクセスするとエラーになるので注意が必要です。

dict_get.py - 値の取得 
phonebook = {
    "田中": "090-1234-5678",
    "佐藤": "080-9876-5432"
}

# 値を取得（方法1: []を使う）
print(phonebook["田中"])  # 出力: 090-1234-5678

# 値を取得（方法2: get()メソッドを使う - 安全）
print(phonebook.get("佐藤"))  # 出力: 080-9876-5432
print(phonebook.get("山田"))  # 出力: None（エラーにならない）
print(phonebook.get("山田", "未登録"))  # 出力: 未登録（デフォルト値）

重要ポイント

phonebook["山田"]は存在しないキーなのでエラーになりますが、 phonebook.get("山田")はエラーにならずNoneを返します。 安全にデータを取得したい場合はget()メソッドを使いましょう。

値の追加と更新

新しいキーに値を代入すると追加、既存のキーに値を代入すると更新されます。

dict_update.py - 追加と更新 
phonebook = {
    "田中": "090-1234-5678"
}

# 新しい要素を追加
phonebook["山田"] = "080-5555-6666"
print(phonebook)
# 出力: {'田中': '090-1234-5678', '山田': '080-5555-6666'}

# 既存の要素を更新
phonebook["田中"] = "090-9999-0000"
print(phonebook)
# 出力: {'田中': '090-9999-0000', '山田': '080-5555-6666'}

値の削除

del文やpop()メソッドで要素を削除できます。

dict_delete.py - 削除 
phonebook = {
    "田中": "090-1234-5678",
    "佐藤": "080-9876-5432",
    "鈴木": "070-1111-2222"
}

# del文で削除
del phonebook["佐藤"]
print(phonebook)
# 出力: {'田中': '090-1234-5678', '鈴木': '070-1111-2222'}

# pop()で削除（削除した値を取得できる）
phone = phonebook.pop("鈴木")
print(f"削除した電話番号: {phone}")  # 出力: 削除した電話番号: 070-1111-2222
print(phonebook)  # 出力: {'田中': '090-1234-5678'}

8.3 辞書の便利なメソッド

辞書には、キーや値を取り出したり、存在確認をしたりする便利なメソッドが用意されています。

dict_methods.py - 辞書のメソッド 
student = {
    "名前": "田中太郎",
    "年齢": 20,
    "学年": 3,
    "専攻": "情報工学"
}

# keys() - すべてのキーを取得
print(student.keys())
# 出力: dict_keys(['名前', '年齢', '学年', '専攻'])

# values() - すべての値を取得
print(student.values())
# 出力: dict_values(['田中太郎', 20, 3, '情報工学'])

# items() - キーと値のペアを取得
print(student.items())
# 出力: dict_items([('名前', '田中太郎'), ('年齢', 20), ('学年', 3), ('専攻', '情報工学')])

# in演算子 - キーの存在確認
if "名前" in student:
    print("名前が登録されています")

# len() - 要素数を取得
print(f"要素数: {len(student)}")  # 出力: 要素数: 4

辞書のループ処理

for文を使って辞書の全要素を処理できます。

dict_loop.py - 辞書のループ 
scores = {
    "国語": 85,
    "数学": 92,
    "英語": 78
}

# キーだけループ
for subject in scores:
    print(subject)

# キーと値を同時にループ（推奨）
for subject, score in scores.items():
    print(f"{subject}: {score}点")

# 出力:
# 国語: 85点
# 数学: 92点
# 英語: 78点

8.4 辞書の実用例

例1: 商品在庫管理システム

inventory.py - 在庫管理 
# 商品の在庫管理
inventory = {
    "りんご": 50,
    "バナナ": 30,
    "みかん": 20
}

# 商品を販売（在庫を減らす）
product = "りんご"
quantity = 5
inventory[product] -= quantity
print(f"{product}を{quantity}個販売しました")
print(f"残り在庫: {inventory[product]}個")

# 新商品を追加
inventory["ぶどう"] = 15
print(f"新商品を追加しました: {inventory}")

例2: ユーザー情報管理

user_info.py - ユーザー管理 
# ユーザー情報を辞書で管理
user = {
    "username": "tanaka123",
    "email": "tanaka@example.com",
    "age": 25,
    "premium": True
}

# ユーザー情報を表示
print(f"ユーザー名: {user['username']}")
print(f"メール: {user['email']}")

# プレミアム会員かチェック
if user.get("premium", False):
    print("プレミアム会員です！")
else:
    print("一般会員です")

例3: 単語の出現回数カウント

word_count.py - 単語カウント 
# 文章内の単語出現回数をカウント
text = "りんご バナナ りんご みかん バナナ りんご"
words = text.split()

# 単語の出現回数を辞書で管理
word_count = {}

for word in words:
    if word in word_count:
        word_count[word] += 1  # 既存の単語ならカウント増加
    else:
        word_count[word] = 1   # 新しい単語なら1を設定

print(word_count)
# 出力: {'りんご': 3, 'バナナ': 2, 'みかん': 1}

8.5 リストと辞書の使い分け

リストと辞書はそれぞれ得意な用途があります。適切に使い分けることで、効率的なプログラムを書けます。

リストを使う場面

  • • 順序が重要な場合（並び順に意味がある）
  • • 同じ値が複数回出現する場合
  • • インデックス番号でアクセスしたい場合
  • • 例: 買い物リスト、成績の履歴

辞書を使う場面

  • • 名前でデータを管理したい場合
  • • キーが一意（重複しない）の場合
  • • 高速な検索が必要な場合
  • • 例: 電話帳、ユーザー情報、設定値
comparison.py - 使い分けの例 
# リストが適している例: ToDoリスト（順序が大事）
todo_list = ["資料作成", "メール返信", "会議準備"]
print(f"次のタスク: {todo_list[0]}")

# 辞書が適している例: ユーザー設定（名前で管理）
settings = {
    "theme": "dark",
    "font_size": 14,
    "notifications": True
}
print(f"テーマ: {settings['theme']}")

  • 1. 新しい連絡先を追加する機能
  • 2. 名前で電話番号を検索する機能
  • 3. 連絡先を削除する機能
  • 4. 全ての連絡先を表示する機能

理解度確認クイズ

  1. Q1. 辞書とリストの最も大きな違いは何ですか？

    回答を表示

    リストは数値のインデックス（0, 1, 2...）でデータにアクセスしますが、 辞書は意味のある名前（キー）でデータにアクセスできます。 また、リストは順序を持ちますが、辞書は順序を気にせずキーでデータを管理します。

  2. Q2. 辞書から安全に値を取得するメソッドは何ですか？

    回答を表示

    get()メソッドです。存在しないキーにアクセスしてもエラーにならず、Noneまたは指定したデフォルト値を返します。 例: phonebook.get("山田", "未登録")

  3. Q3. 辞書のすべてのキーと値を同時にループ処理するにはどうしますか？

    回答を表示

    items()メソッドを使います。
    for key, value in dictionary.items():
        print(key, value)
    この形式で、キーと値を同時に取得できます。

  4. Q4. 辞書に新しい要素を追加する方法は？

    回答を表示

    dictionary[新しいキー] = 値 の形式で追加できます。 例: phonebook["山田"] = "080-1234-5678" 既存のキーに代入すると値が更新されます。

第8章のまとめ

  • 辞書は、キーと値のペアでデータを管理する便利なデータ構造
  • {}（波括弧）で作成し、「キー: 値」の形式でデータを定義する
  • get()メソッドを使うと、存在しないキーでもエラーにならず安全に値を取得できる
  • keys()、values()、items()メソッドで辞書の内容を効率的に処理できる
  • 順序が重要ならリスト、名前で管理したいなら辞書を使い分ける

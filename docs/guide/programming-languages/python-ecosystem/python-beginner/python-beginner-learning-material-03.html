変数とデータ型 - 値を入れる「箱」の使い方

プログラムで扱うデータを保存する「変数」という箱の概念を学びます。数値、文字列、真偽値など、基本的なデータの種類と使い方を身近な例で理解します。

この章で学ぶこと

  • 変数とは何か、なぜ必要なのかを身近な例で理解する
  • 変数の命名規則と正しい使い方を学ぶ
  • 基本的なデータ型（整数、小数、文字列、真偽値）の違いを理解する
  • 型の確認方法と型変換の方法を習得する

3.1 変数とは？「箱」に値を入れる

変数とは、プログラムの中でデータを一時的に保管しておく「箱」のようなものです。 この箱には名前（ラベル）を付けることができ、その名前を使って中身のデータを取り出したり、新しいデータに入れ替えたりできます。

日常生活での例え

引き出しに「靴下」「ハンカチ」「ベルト」とラベルを貼って整理するのと同じです。 プログラムでも「age」「name」「price」といった名前の箱（変数）を用意して、 それぞれに年齢、名前、価格といったデータを入れて管理します。

variable_basic.py - 変数の基本 
# 変数に値を代入する
name = "太郎"
age = 20
height = 170.5

# 変数を使って表示する
print(name)      # 太郎
print(age)       # 20
print(height)    # 170.5

# 変数を組み合わせて使う
print(f"{name}さんは{age}歳です")

上のコードでは、 name、 age、 height という3つの箱（変数）を用意して、それぞれに値を入れています。 =は「代入」という意味で、 「右側の値を左側の箱に入れる」という操作を表しています。

変数の仕組み

graph LR A[プログラム] --> B["変数 name"] A --> C["変数 age"] A --> D["変数 height"] B --> B1["値: 太郎"] C --> C1["値: 20"] D --> D1["値: 170.5"]

変数を使う理由

変数を使わずに毎回値を直接書くこともできますが、変数を使うことで「同じ値を何度も使う」「値を後で変更する」 「意味のある名前を付けて分かりやすくする」といったメリットがあります。

3.2 変数の命名規則と使い方

変数名には、いくつかのルールと推奨される書き方があります。 これらを守ることで、エラーを防ぎ、他の人（未来の自分も含めて）が読みやすいコードを書くことができます。

使える文字

英字（a-z, A-Z）、数字（0-9）、アンダースコア（_）が使えます。ただし、数字から始めることはできません。

大文字と小文字を区別

nameとNameは別の変数として扱われます。

意味のある名前を付ける

xよりtotal_priceのように、何を表すか分かる名前が推奨されます。

スネークケース

複数の単語を組み合わせる場合は、アンダースコアで区切るuser_nameが一般的です。

良い例 

# 分かりやすい変数名
user_name = "田中"
total_price = 1500
is_active = True
student_count = 30

悪い例 

# 分かりにくい、またはエラーになる
x = "田中"          # 何を表すか不明
1st_name = "太郎"  # 数字から始まる（エラー）
user-name = "鈴木" # ハイフン使用（エラー）
class = "A組"      # 予約語（エラー）

予約語に注意

Pythonにはif、for、 class、defなど、 特別な意味を持つ「予約語」があります。これらは変数名として使えません。

3.3 データ型の種類

変数に入れるデータには、数値や文字列など様々な種類（型）があります。 Pythonは自動的に型を判断してくれますが、それぞれの型の特徴を理解することは重要です。

基本的なデータ型

graph TB Root[データ型] --> Num[数値型] Root --> Str[文字列型 str] Root --> Bool[真偽値型 bool] Num --> Int[整数型 int] Num --> Float[小数型 float] Int --> IntEx["例: 10, -5, 0"] Float --> FloatEx["例: 3.14, -0.5, 2.0"] Str --> StrEx["例: 'Hello', '太郎'"] Bool --> BoolEx["例: True, False"]

整数型（int）

小数点を含まない数値。年齢、個数、順番など、数えられるものに使います。 

age = 20
count = -5
zero = 0

小数型（float）

小数点を含む数値。身長、体重、気温、価格など、細かい値を扱うときに使います。 

height = 170.5
pi = 3.14
price = 1980.0

文字列型（str）

文字や文章を扱うデータ。シングルクォート（'）かダブルクォート（"）で囲みます。 

name = "太郎"
message = 'こんにちは'
address = "東京都"

真偽値型（bool）

True（真）またはFalse（偽）の2つの値だけを持つ型。条件判定に使います。 

is_student = True
has_license = False
is_adult = True

実習 3-1: 様々なデータ型を使ってみよう

異なるデータ型の変数を作成して、それぞれの値を表示してみましょう。

data_types.py 
# 様々なデータ型を使う
name = "山田花子"        # 文字列
age = 25               # 整数
height = 165.3         # 小数
is_student = False     # 真偽値

# 全て表示
print(f"名前: {name}")
print(f"年齢: {age}歳")
print(f"身長: {height}cm")
print(f"学生: {is_student}")

3.4 型の確認と変換

変数がどの型なのかを確認したり、ある型から別の型に変換したりすることができます。 これは、プログラムでデータを扱う際に非常に重要なスキルです。

型の確認（type関数）

type() 関数を使うと、変数の型を確認できます。

check_type.py - 型の確認 
# 型を確認する
name = "太郎"
age = 20
height = 170.5
is_student = True

print(type(name))       # 
print(type(age))        # 
print(type(height))     # 
print(type(is_student)) #

型変換

ある型のデータを別の型に変換することを型変換（キャスト）と呼びます。 主な型変換関数は以下の通りです。

1 文字列→整数 

text = "100"
number = int(text)  # 100（整数）
print(type(number)) #

2 文字列→小数 

text = "3.14"
number = float(text)  # 3.14（小数）
print(type(number))   #

3 数値→文字列 

age = 20
text = str(age)    # "20"（文字列）
print(type(text))  #

4 整数→小数 

count = 10
number = float(count)  # 10.0（小数）
print(type(number))    #

なぜ型変換が必要？

例えば、ユーザーからの入力は常に文字列として受け取られます。 「年齢を入力してください」と聞いて「20」と入力された場合、これは文字列の"20"です。 数値として計算するには、int()で整数に変換する必要があります。

conversion_example.py - 型変換の実用例 
# ユーザー入力を想定（実際はinput関数で受け取る）
age_text = "20"        # 文字列として受け取る
price_text = "1980"    # 文字列として受け取る

# 型変換して計算に使う
age = int(age_text)
price = int(price_text)

# 計算
next_year_age = age + 1
total = price * 2

print(f"来年は{next_year_age}歳です")
print(f"合計金額: {total}円")

型変換の注意点

変換できない値を変換しようとするとエラーになります。 例: int("こんにちは")はエラー。 数値に変換できる文字列かどうか確認してから変換しましょう。

理解度確認クイズ

  1. Q1. 変数とは何ですか？なぜ必要なのですか？

    回答を表示

    変数とは、データを一時的に保管しておく「箱」のようなものです。 同じ値を何度も使う、値を後で変更する、意味のある名前を付けて分かりやすくする、といった理由で必要です。

  2. Q2. 変数名として使えないものはどれですか？ ① my_name ② 1st_name ③ user_age ④ total_price

    回答を表示

    ② 1st_name が使えません。変数名は数字から始めることができないためです。

  3. Q3. 基本的なデータ型を4つ挙げて、それぞれの例を示してください。

    回答を表示

    ① 整数型（int）: 20, -5, 0
    ② 小数型（float）: 3.14, 170.5, -0.5
    ③ 文字列型（str）: "太郎", "こんにちは"
    ④ 真偽値型（bool）: True, False

  4. Q4. 文字列"100"を整数に変換するにはどうすればいいですか？

    回答を表示

    int("100")で変換できます。 例: number = int("100")とすると、numberには整数の100が入ります。

第3章のまとめ

  • 変数は、データを保管する「箱」で、名前を付けて管理できる
  • 変数名は英字、数字、アンダースコアが使え、数字から始めることはできない
  • 基本的なデータ型には整数（int）、小数（float）、文字列（str）、真偽値（bool）がある
  • type()関数で型を確認でき、int()、float()、str()などで型変換ができる

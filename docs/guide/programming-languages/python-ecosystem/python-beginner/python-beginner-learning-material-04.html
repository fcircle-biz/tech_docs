4.1 算術演算 - Pythonを電卓として使おう

Pythonは電卓のように計算ができます。プログラミングの世界では、計算記号のことを演算子（えんざんし）と呼びます。 まずは基本的な四則演算から見ていきましょう。

主な算術演算子

演算子 意味 例 結果
+ 足し算 10 + 5 15
- 引き算 10 - 5 5
* 掛け算 10 * 5 50
/ 割り算（小数） 10 / 3 3.333...
// 割り算（整数） 10 // 3 3
% 余り（あまり） 10 % 3 1
** 累乗（べき乗） 2 ** 3 8（2の3乗）
calculation.py - 基本的な計算 
# 基本的な四則演算
print(10 + 5)    # 足し算: 15
print(10 - 5)    # 引き算: 5
print(10 * 5)    # 掛け算: 50
print(10 / 3)    # 割り算（小数）: 3.333...

# 特殊な演算
print(10 // 3)   # 整数の割り算: 3
print(10 % 3)    # 余り: 1
print(2 ** 3)    # 累乗（2の3乗）: 8

# 変数を使った計算
price = 1000
tax_rate = 0.1
total = price + (price * tax_rate)
print(f"税込価格: {total}円")  # 税込価格: 1100.0円

割り算の注意点

/ は常に小数（float型）を返します。整数だけが欲しい場合は // を使いましょう。余り（あまり）を求めるには % を使います（偶数・奇数の判定などに便利です）。

4.2 比較演算子と論理演算子 - 条件を判定しよう

比較演算子は、2つの値を比べて「正しい（True）」か「間違っている（False）」を判定します。 論理演算子は、複数の条件を組み合わせるときに使います。

比較演算子

演算子 意味 例
== 等しい 5 == 5
!= 等しくない 5 != 3
> より大きい 5 > 3
< より小さい 3 < 5
>= 以上 5 >= 5
<= 以下 3 <= 5

論理演算子

演算子 意味 例
and かつ（両方真） a and b
or または（どちらか真） a or b
not 否定（真偽を反転） not a
comparison.py - 比較と論理演算 
# 比較演算子の例
age = 20
print(age >= 20)  # True（20歳以上？）
print(age == 18)  # False（18歳と同じ？）
print(age != 18)  # True（18歳ではない？）

# 論理演算子の例
score = 85
is_passed = score >= 60 and score <= 100
print(f"合格: {is_passed}")  # 合格: True

# 複数条件の組み合わせ
weather = "晴れ"
temperature = 25
is_good_day = weather == "晴れ" and temperature >= 20
print(f"お出かけ日和: {is_good_day}")  # お出かけ日和: True

つまずきポイント

比較には ==（イコール2つ）を使います。 =（イコール1つ）は代入（値を入れる）なので、混同しないよう注意しましょう。

4.3 文字列の基本操作 - 文字を自在に扱おう

文字列（もじれつ）は、文字が連なったデータです。Pythonでは文字列を使って、名前、メッセージ、住所など 様々なテキスト情報を扱います。ここでは文字列の基本的な操作方法を学びます。

文字列操作の基本

graph TB String[文字列操作] --> Concat[結合] String --> Repeat[繰り返し] String --> Slice[スライス] String --> Method[メソッド] Concat --> ConcatEx["'Hello' + ' ' + 'World'"] Repeat --> RepeatEx["'Python' * 3"] Slice --> SliceEx["text[0:5]"] Method --> MethodEx["upper, lower, split など"]
string_basics.py - 文字列の基本操作 
# 文字列の結合（+ 演算子）
first_name = "太郎"
last_name = "山田"
full_name = last_name + " " + first_name
print(full_name)  # 山田 太郎

# 文字列の繰り返し（* 演算子）
message = "おめでとう！"
print(message * 3)  # おめでとう！おめでとう！おめでとう！

# 文字列の長さ
text = "Python"
print(len(text))  # 6（文字数）

# スライス（部分取得）
text = "Python Programming"
print(text[0:6])    # Python（0文字目から5文字目まで）
print(text[7:])     # Programming（7文字目から最後まで）
print(text[-4:])    # ming（後ろから4文字）

ヒント

スライスの [0:6] は「0番目から6番目の手前まで」という意味です。 Pythonでは文字の位置（インデックス）は0から始まります。最初の文字は0番目、2番目の文字は1番目です。

4.4 便利な文字列メソッド - テキスト処理を自動化

メソッドとは、文字列が持っている便利な機能のことです。 文字列.メソッド名() という形で使うことで、大文字に変換したり、分割したり、様々な処理が簡単にできます。

よく使う文字列メソッド

メソッド 説明 例 結果
upper() 大文字に変換 "hello".upper() HELLO
lower() 小文字に変換 "HELLO".lower() hello
strip() 前後の空白を削除 " hi ".strip() hi
split() 区切り文字で分割 "a,b,c".split(",") ['a', 'b', 'c']
replace() 文字列を置換 "cat".replace("c","b") bat
startswith() 指定文字で始まるか "Python".startswith("Py") True
endswith() 指定文字で終わるか "Python".endswith("on") True
find() 文字列の位置を検索 "Python".find("th") 2
string_methods.py - 文字列メソッドの活用 
# 大文字・小文字変換
name = "python programming"
print(name.upper())  # PYTHON PROGRAMMING
print(name.title())  # Python Programming（各単語の先頭を大文字に）

# 空白の削除
user_input = "  太郎  "
clean_name = user_input.strip()
print(f"[{clean_name}]")  # [太郎]

# 文字列の分割
csv_data = "りんご,みかん,バナナ"
fruits = csv_data.split(",")
print(fruits)  # ['りんご', 'みかん', 'バナナ']

# 文字列の置換
message = "こんにちは、○○さん"
personalized = message.replace("○○", "太郎")
print(personalized)  # こんにちは、太郎さん

# 文字列の検索
email = "example@gmail.com"
if "@" in email:
    print("有効なメールアドレス形式です")

# 文字列の結合（join）
words = ["Python", "は", "楽しい"]
sentence = "".join(words)
print(sentence)  # Pythonは楽しい

実習 4-1: 簡単な計算機プログラム

ユーザーから2つの数値を入力してもらい、四則演算の結果を表示するプログラムを作成してみましょう。

手順

  1. 1 input()関数で2つの数値を文字列として受け取る
  2. 2 int()またはfloat()で数値型に変換
  3. 3 足し算、引き算、掛け算、割り算の結果を計算して表示
calculator.py - 参考例 
# 簡単な計算機プログラム
num1 = float(input("1つ目の数値: "))
num2 = float(input("2つ目の数値: "))

print(f"{num1} + {num2} = {num1 + num2}")
print(f"{num1} - {num2} = {num1 - num2}")
print(f"{num1} × {num2} = {num1 * num2}")
print(f"{num1} ÷ {num2} = {num1 / num2}")

理解度確認クイズ

  1. Q1. 10を3で割った余りを求める演算子は何ですか？

    回答を表示

    %（剰余演算子）です。10 % 3 は 1 になります。 この演算子は偶数・奇数判定（n % 2 == 0なら偶数）などによく使われます。

  2. Q2. 比較演算子で「等しい」を表すのは「=」ですか？

    回答を表示

    いいえ、==（イコール2つ）です。 =（イコール1つ）は代入（変数に値を入れる）のために使います。 比較と代入を混同しないように注意しましょう。

  3. Q3. 文字列 "Python" を3回繰り返すコードを書いてください。

    回答を表示

    "Python" * 3 で「PythonPythonPython」が得られます。 文字列に対して * 演算子を使うと、その回数分繰り返されます。

  4. Q4. 文字列メソッド upper() と lower() の違いを説明してください。

    回答を表示

    upper()は文字列を全て大文字に変換し、 lower()は全て小文字に変換します。 例: "Hello".upper() → "HELLO"、 "Hello".lower() → "hello"

第4章のまとめ

  • 算術演算子（+, -, *, /, //, %, **）を使って、様々な計算ができる
  • 比較演算子（==, !=, >, <, >=, <=）で値を比較し、True/Falseを得られる
  • 論理演算子（and, or, not）で複数の条件を組み合わせられる
  • 文字列の結合（+）、繰り返し（*）、スライス（[開始:終了]）で自在に文字列を操作できる
  • 文字列メソッド（upper, lower, split, replace など）で便利なテキスト処理ができる

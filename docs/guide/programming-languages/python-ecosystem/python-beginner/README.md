# Python入門 学習ガイドライン

このガイドラインでは、プログラミング完全未経験の方でもPythonを基礎から学習できるよう、段階的にステップを踏んで解説しています。専門用語を極力避け、日常的な例えを使いながら、実践的なプログラミングスキルを身につけることができます。

## 前提条件

### 必要な環境
- Windows、Mac、またはLinuxが動作するコンピュータ
- インターネット接続（Pythonのダウンロードや情報検索のため）
- テキストエディタ（メモ帳でも可、後の章でVS Codeを紹介）

### 参考リソース
- [Python公式サイト（日本語）](https://www.python.org/doc/)
- [Python公式チュートリアル（日本語）](https://docs.python.org/ja/3/tutorial/)
- [Python.jp](https://www.python.jp/) - 日本のPythonコミュニティ

### 前提知識
- **必須**: 特になし（コンピュータの基本操作ができれば十分）
- **推奨**: ファイルやフォルダの操作ができること、キーボードでの文字入力に慣れていること

## 学習コンテンツ

### [1. Pythonとは？プログラミングを始めよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-01.html)
プログラミングとは何か、Pythonでできることは何か、なぜPythonから始めるのが良いのかを、身近な例を使いながら解説します。プログラミングの楽しさと可能性について理解できます。

### [2. 環境構築 - Pythonをインストールして最初のプログラムを動かそう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-02.html)
Pythonのインストール方法を画像付きで詳しく解説し、「Hello, World!」という最初のプログラムを作成・実行します。プログラムが動く喜びを体験できます。

### [3. 変数とデータ型 - 値を入れる「箱」の使い方](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-03.html)
プログラムで扱うデータを保存する「変数」という箱の概念を学びます。数値、文字列、真偽値など、基本的なデータの種類と使い方を身近な例で理解します。

### [4. 演算と文字列操作 - 計算と文字の扱い方](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-04.html)
電卓のような計算機能と、文字列の結合・分割などの操作方法を学びます。日常的な計算やテキスト処理をプログラムで自動化する方法を習得します。

### [5. 条件分岐 - プログラムに判断をさせよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-05.html)
「もし〜なら〜する」という条件判断（if文）を使って、状況に応じて動作を変えるプログラムを作ります。年齢確認や点数判定など、実用的な例で学習します。

### [6. 繰り返し処理 - 同じ作業を自動化しよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-06.html)
forループとwhileループを使って、繰り返し処理を自動化する方法を学びます。100回同じことを書く代わりに、3行で済ませる効率的なプログラミングを体験します。

### [7. リストとタプル - 複数のデータをまとめて管理](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-07.html)
買い物リストや成績表のように、関連するデータをまとめて扱う方法を学びます。データの追加、削除、検索など、実用的な操作方法を習得します。

### [8. 辞書 - データに名前を付けて整理しよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-08.html)
電話帳や住所録のように、名前（キー）と値（バリュー）のペアでデータを管理する辞書型の使い方を学びます。効率的なデータ管理方法を身につけます。

### [9. 関数の基本 - 処理をパーツ化して再利用しよう](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-09.html)
よく使う処理を「関数」としてまとめ、何度でも簡単に呼び出せるようにする方法を学びます。プログラムを部品化して、効率的に開発する技術を習得します。

### [10. ファイルの読み書き - データを保存して次回も使えるように](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-10.html)
テキストファイルやCSVファイルの読み込みと書き込み方法を学びます。プログラムで作成したデータを保存し、後で再利用する実用的な技術を習得します。

### [11. エラーへの対処法 - つまずきポイントの攻略](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-11.html)
初心者がよく遭遇するエラーメッセージの読み方と解決方法を学びます。エラーは敵ではなく、問題解決のヒントであることを理解し、デバッグ力を身につけます。

### [12. 簡単なプログラムを作ってみよう - 実践編](https://fcircle-biz.github.io/tech_docs/guide/programming-languages/python-ecosystem/python-beginner/python-beginner-learning-material-12.html)
これまでに学んだ知識を組み合わせて、じゃんけんゲーム、ToDoリスト、簡単な計算機などの実用的なプログラムを作成します。総合的な実践力を身につけます。

## 学習の進め方

1. **まずは動かしてみる**
   各章の最初にあるサンプルコードをそのまま入力して実行してみましょう。動くことを確認してから、詳しい説明を読むと理解しやすくなります。

2. **エラーを恐れない**
   エラーメッセージが出ても心配いりません。エラーは「ここが違うよ」という親切なお知らせです。第11章で詳しく対処法を学びます。

3. **小さく試す**
   新しい機能を学んだら、すぐに小さなプログラムで試してみましょう。「これを変えたらどうなるだろう？」と実験することが上達への近道です。

4. **復習を大切に**
   新しい章に進む前に、前の章の練習問題をもう一度解いてみましょう。繰り返すことで知識が定着します。

5. **質問メモを作る**
   分からないところは飛ばさず、メモに残しておきましょう。後の章で理解できることも多く、成長を実感できます。

## 学習目標

このガイドを完了すると、以下のスキルを身につけることができます：

- **基本的なプログラムが書ける**: 変数、条件分岐、繰り返しを使った簡単なプログラムを自力で作成できる
- **エラーメッセージが読める**: エラーが発生しても慌てず、メッセージから原因を特定して修正できる
- **データを扱える**: リスト、辞書、ファイルを使って、実用的なデータ処理ができる
- **関数で処理を整理できる**: 長いプログラムを関数に分割して、読みやすく保守しやすいコードが書ける
- **簡単なアプリが作れる**: 学んだ知識を組み合わせて、実用的な小規模プログラムを開発できる
- **次の学習ステップが分かる**: Pythonの基礎を習得し、Web開発やデータ分析など、次の専門分野に進む準備ができる

第6章

繰り返し処理 - 同じ作業を自動化しよう

forループとwhileループを使って、繰り返し処理を自動化する方法を学びます。100回同じことを書く代わりに、3行で済ませる効率的なプログラミングを体験します。

この章で学ぶこと

  • 繰り返し処理の必要性と、プログラミングにおける効率化の考え方
  • forループを使った回数指定の繰り返しと、range()関数の活用
  • whileループを使った条件付き繰り返しと、無限ループへの対処
  • break、continueを使ったループ制御の実践的なテクニック

6.1 繰り返し処理とは？なぜ必要なのか

プログラミングの大きな利点の一つは、同じ作業を何度も自動的に繰り返せることです。 例えば、「おはよう」を100回表示したいとき、コードを100行書く必要はありません。 たった3行のループで実現できます。これが繰り返し処理（ループ）の力です。

日常生活での繰り返し

料理で野菜を切る作業や、洗濯物を干す作業は「同じ動作を繰り返す」行為です。 プログラミングでも同様に、「同じ処理を指定回数繰り返す」「条件を満たす間繰り返す」といった 自動化が可能になります。人間が手作業で100回繰り返すのは大変ですが、 コンピュータなら一瞬で完了します。

繰り返し処理の基本的な流れ

graph TD A[開始] --> B[繰り返し条件チェック] B -->|条件を満たす| C[処理を実行] C --> B B -->|条件を満たさない| D[ループ終了] D --> E[次の処理へ]

Pythonには主に2種類のループがあります。 forループ（回数を決めて繰り返す）と whileループ（条件が満たされる間繰り返す）です。 それぞれに適した使い方があり、状況に応じて使い分けます。

6.2 forループ - 回数を指定して繰り返す

forループは、リストやrange()などの「繰り返し可能なもの」から 要素を一つずつ取り出して処理します。最も基本的で頻繁に使われるループです。

基本的なforループの構文

basic_for.py - forループの基本 
# 5回繰り返す
for i in range(5):
    print(f"{i}回目の実行です")

# 実行結果:
# 0回目の実行です
# 1回目の実行です
# 2回目の実行です
# 3回目の実行です
# 4回目の実行です

重要ポイント

range(5)は 0から4までの5個の数字を生成します（0, 1, 2, 3, 4）。 Pythonでは多くの場合、数え始めは0からです。

range()関数の応用

range_examples.py - range()の様々な使い方 
# 1から5まで繰り返す（開始と終了を指定）
for i in range(1, 6):
    print(i)  # 1, 2, 3, 4, 5

# 0から10まで2ずつ増やす（ステップ指定）
for i in range(0, 11, 2):
    print(i)  # 0, 2, 4, 6, 8, 10

# 10から1まで逆順にカウントダウン
for i in range(10, 0, -1):
    print(i)  # 10, 9, 8, ..., 1

リストを使ったforループ

list_loop.py - リストの要素を繰り返す 
# 果物リストの各要素を表示
fruits = ["りんご", "バナナ", "オレンジ"]
for fruit in fruits:
    print(f"好きな果物: {fruit}")

# 実行結果:
# 好きな果物: りんご
# 好きな果物: バナナ
# 好きな果物: オレンジ

6.3 whileループ - 条件を満たす間繰り返す

whileループは、条件が真（True）である限り処理を繰り返します。 「何回繰り返すかわからないが、特定の条件まで続ける」場合に便利です。

basic_while.py - whileループの基本 
# カウントダウンプログラム
count = 5
while count > 0:
    print(f"残り{count}秒")
    count -= 1  # countを1減らす
print("発射！")

# 実行結果:
# 残り5秒
# 残り4秒
# 残り3秒
# 残り2秒
# 残り1秒
# 発射！

つまずきポイント

whileループで条件が永遠に真のままだと無限ループになります。 必ず「条件が偽になる」処理を含めましょう。無限ループになった場合は Ctrl + C でプログラムを強制終了できます。

実用例：数字当てゲーム

guess_game.py - whileループの実用例 
# 正解の数字
secret = 7
guess = 0

# 正解するまで繰り返す
while guess != secret:
    guess = int(input("1〜10の数字を当ててください: "))
    if guess < secret:
        print("もっと大きいです")
    elif guess > secret:
        print("もっと小さいです")

print("正解です！")

6.4 break と continue - ループを制御する

ループの途中で処理を中断したり、スキップしたりする命令が breakとcontinueです。 これらを使うことで、より柔軟なループ制御ができます。

break - ループを途中で終了

break_example.py - breakの使い方 
# 特定の値が見つかったらループを終了
numbers = [1, 3, 5, 7, 9, 11, 13]
for num in numbers:
    if num > 10:
        print("10より大きい数字が見つかりました")
        break  # ここでループ終了
    print(num)

# 実行結果: 1, 3, 5, 7, 9 が表示され、
# "10より大きい数字が見つかりました" の後、ループ終了

continue - 次の繰り返しへスキップ

continue_example.py - continueの使い方 
# 偶数だけスキップして奇数のみ表示
for i in range(1, 11):
    if i % 2 == 0:  # 偶数の場合
        continue  # 以降の処理をスキップして次の繰り返しへ
    print(f"{i}は奇数です")

# 実行結果:
# 1は奇数です
# 3は奇数です
# 5は奇数です
# 7は奇数です
# 9は奇数です

break と continue の動作の違い

graph TD A[ループ開始] --> B[条件判定] B -->|break実行| C[ループ完全終了] B -->|continue実行| D[残りの処理をスキップ] D --> E[次の繰り返しへ] E --> B B -->|通常処理| F[処理実行] F --> E C --> G[ループ外の処理へ]

6.5 ネストしたループ - ループの中にループ

ループの中に別のループを入れることをネスト（入れ子）と呼びます。 九九の表や座標のパターンなど、二重の繰り返しが必要な場合に使います。

nested_loop.py - ネストしたループの例 
# 九九の表を作成
for i in range(1, 10):  # 外側のループ：1から9
    for j in range(1, 10):  # 内側のループ：1から9
        print(f"{i}×{j}={i*j}", end="  ")
    print()  # 改行

# 実行結果（一部）:
# 1×1=1  1×2=2  1×3=3  ...
# 2×1=2  2×2=4  2×3=6  ...
# ...

ヒント

print()関数の end=" " パラメータを使うと、改行せずに横に並べて表示できます。 デフォルトはend="\n"（改行）です。

実習 6-1: 合計値を計算するプログラム

forループを使って、1から100までの数字の合計を計算するプログラムを作成してみましょう。

手順

  1. 1 変数totalを0で初期化する
  2. 2 for i in range(1, 101)でループを作る
  3. 3 ループ内でtotal += iを実行
  4. 4 ループ終了後、totalの値を表示（答え: 5050）
解答例を表示 
total = 0
for i in range(1, 101):
    total += i
print(f"1から100までの合計: {total}")

理解度確認クイズ

  1. Q1. range(3, 10, 2)が生成する数値を答えてください。

    回答を表示

    3, 5, 7, 9（3から始まり、10未満まで、2ずつ増加）

  2. Q2. forループとwhileループの使い分けの基準を説明してください。

    回答を表示

    forループは繰り返し回数が決まっている場合（リストの要素分、5回など）に使用。 whileループは条件が満たされる間繰り返す場合（ユーザー入力が正しくなるまで、など）に使用します。

  3. Q3. breakとcontinueの違いを説明してください。

    回答を表示

    breakはループを完全に終了してループの外に抜けます。 continueは現在の繰り返しをスキップして次の繰り返しに進みます。

  4. Q4. 無限ループを防ぐために注意すべきことは何ですか？

    回答を表示

    whileループでは、必ずループ内で条件を偽にする処理（カウンタの増減、フラグの変更など）を含める必要があります。 また、条件式が正しく設定されているか確認することも重要です。

第6章のまとめ

  • 繰り返し処理は同じ作業を自動化し、プログラミングの効率を劇的に向上させる
  • forループは回数が決まっている繰り返しに使い、range()やリストと組み合わせる
  • whileループは条件を満たす間繰り返すが、無限ループに注意が必要
  • break、continueを使うことで、ループの柔軟な制御が可能になる

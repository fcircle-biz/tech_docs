第5章

条件分岐 - プログラムに判断をさせよう

「もし〜なら〜する」という条件判断（if文）を使って、状況に応じて動作を変えるプログラムを作ります。年齢確認や点数判定など、実用的な例で学習します。

この章で学ぶこと

  • if文の基本的な構文と使い方を理解する
  • elif、elseを使った複数分岐の条件判断を学ぶ
  • 比較演算子と論理演算子を使った条件式の作成方法を習得する
  • 実用的な条件分岐プログラムを作成できるようになる

5.1 条件分岐とは何か？

条件分岐とは、「もし〜なら〜する、そうでなければ〜する」という判断をプログラムに行わせる仕組みです。 これは日常生活でも頻繁に行っている意思決定と同じです。

身近な例で理解する

朝起きて外を見て、「もし雨が降っていたら傘を持っていく、晴れていたら傘は持たない」と判断しますね。 これが条件分岐です。プログラムでも全く同じように、「もし〜なら」という条件を設定して、 その条件が成り立つかどうかで処理内容を変えることができます。

プログラムにおける条件分岐では、if文という命令を使います。 Pythonのif文は、とても直感的で読みやすい文法になっています。

条件分岐の基本的な流れ

flowchart TD A[プログラム開始] --> B{条件を判定} B -->|条件が成立 True| C[処理A を実行] B -->|条件が不成立 False| D[処理B を実行] C --> E[次の処理へ] D --> E

5.2 if文の基本構文

Pythonのif文は、以下のような構文で書きます。重要なのはインデント（字下げ）です。 Pythonでは、インデントでブロック（処理のまとまり）を表現します。

if文の基本構文 
# if文の基本形
if 条件式:
    # 条件がTrueの時に実行される処理
    処理内容

重要ポイント

① 条件式の後ろにコロン（:）を必ず付ける
② 実行したい処理はインデント（4つのスペース）で字下げする
③ インデントのレベルが同じ行が同じブロックとして扱われる

実践例：年齢確認プログラム

age_check.py - 年齢確認 
# 年齢確認プログラム
age = 20

if age >= 20:
    print("あなたは成人です")
    print("お酒を購入できます")

このプログラムでは、age >= 20 という条件式がTrue（真）の時だけ、インデントされた2行が実行されます。 ageが20以上なら「成人です」と「お酒を購入できます」の両方が表示されます。

5.3 elif と else - 複数の条件分岐

「条件Aが成り立たなければ、条件Bを確認する」という複数の条件判断には、elif（else if の略）を使います。 また、「どの条件にも当てはまらない場合」の処理にはelseを使います。

if-elif-elseの判断フロー

flowchart TD A[開始] --> B{条件A?} B -->|True| C[処理A] B -->|False| D{条件B?} D -->|True| E[処理B] D -->|False| F{条件C?} F -->|True| G[処理C] F -->|False| H[else の処理] C --> I[次の処理へ] E --> I G --> I H --> I

実践例：点数による成績判定

grade_check.py - 成績判定 
# 点数による成績判定
score = 85

if score >= 90:
    print("評価：A（優秀）")
elif score >= 70:
    print("評価：B（良好）")
elif score >= 60:
    print("評価：C（合格）")
else:
    print("評価：D（不合格）")

この例では、点数が90以上ならA、70以上90未満ならB、60以上70未満ならC、60未満ならDと判定されます。 上から順に条件を確認し、最初にTrueになった処理だけが実行されます。

ヒント

elif は何個でも書けますが、else は最後に1つだけ書けます。 また、else は省略することもできます（全ての条件が不成立の時は何もしない、という場合）。

5.4 比較演算子と論理演算子

条件式を作るには、値を比較する比較演算子と、複数の条件を組み合わせる論理演算子を使います。

比較演算子一覧

演算子 意味 使用例 結果
== 等しい 5 == 5 True
!= 等しくない 5 != 3 True
> より大きい 5 > 3 True
< より小さい 5 < 3 False
>= 以上 5 >= 5 True
<= 以下 5 <= 3 False

つまずきポイント

「等しい」は==（イコール2つ）です。=（イコール1つ）は「代入」なので混同しないように！ if age = 20: はエラーになります。 正しくはif age == 20: です。

論理演算子一覧

演算子 意味 使用例
and かつ（両方とも真） age >= 20 and age < 65
or または（どちらか真） age < 18 or age >= 65
not 〜でない（否定） not is_rainy
logical_operators.py - 論理演算子の使用例 
# and: 両方の条件が真の時だけ真
age = 25
if age >= 20 and age < 65:
    print("一般成人料金です")

# or: どちらかの条件が真なら真
if age < 18 or age >= 65:
    print("割引料金が適用されます")

# not: 条件の否定
is_member = False
if not is_member:
    print("会員登録をおすすめします")

5.5 実践的な条件分岐の例

ここでは、日常的な場面を想定した実用的なプログラム例をいくつか紹介します。

例1：BMI判定プログラム

bmi_check.py - BMI判定 
# BMI判定プログラム
height = 1.70  # 身長（メートル）
weight = 65    # 体重（kg）

# BMIを計算
bmi = weight / (height ** 2)

# 判定
if bmi < 18.5:
    print(f"BMI: {bmi:.1f} - やせ型")
elif bmi < 25:
    print(f"BMI: {bmi:.1f} - 標準")
elif bmi < 30:
    print(f"BMI: {bmi:.1f} - 肥満（軽度）")
else:
    print(f"BMI: {bmi:.1f} - 肥満（高度）")

例2：パスワード強度チェック

password_check.py - パスワード強度 
# パスワード強度チェック
password = "MyP@ssw0rd"
length = len(password)

# 数字と記号が含まれているか確認
has_digit = any(c.isdigit() for c in password)
has_special = any(not c.isalnum() for c in password)

# 判定
if length >= 12 and has_digit and has_special:
    print("パスワード強度: 強")
elif length >= 8 and has_digit:
    print("パスワード強度: 中")
elif length >= 6:
    print("パスワード強度: 弱")
else:
    print("パスワードが短すぎます")

例3：うるう年判定

leap_year.py - うるう年判定 
# うるう年判定プログラム
year = 2024

# うるう年の条件
# ① 4で割り切れる
# ② ただし100で割り切れる年は除く
# ③ ただし400で割り切れる年は含む

if year % 400 == 0:
    print(f"{year}年はうるう年です")
elif year % 100 == 0:
    print(f"{year}年はうるう年ではありません")
elif year % 4 == 0:
    print(f"{year}年はうるう年です")
else:
    print(f"{year}年はうるう年ではありません")

5.6 ネストした条件分岐

if文の中に別のif文を入れることをネスト（入れ子）といいます。 複雑な条件判断が必要な場合に使いますが、深くネストしすぎると読みにくくなるので注意が必要です。

ネストした条件分岐の構造

flowchart TD A[開始] --> B{条件1?} B -->|True| C{条件2?} B -->|False| D[処理C] C -->|True| E[処理A] C -->|False| F[処理B] E --> G[終了] F --> G D --> G
ticket_price.py - チケット料金計算（ネスト版） 
# チケット料金計算（年齢と会員種別で判定）
age = 15
is_member = True

if age < 18:
    # 子供料金
    if is_member:
        price = 500  # 会員子供料金
    else:
        price = 800  # 一般子供料金
else:
    # 大人料金
    if is_member:
        price = 1200  # 会員大人料金
    else:
        price = 1800  # 一般大人料金

print(f"料金: {price}円")

注意

ネストは2〜3段までに抑えましょう。それ以上深くなる場合は、 論理演算子（and, or）を使って1つの条件式にまとめるか、 後の章で学ぶ関数を使って処理を分割する方が読みやすくなります。

実習 5-1: じゃんけんゲームを作ろう

条件分岐を使って、コンピュータとじゃんけんをするプログラムを作成してみましょう。

手順

  1. 1 プレイヤーの手を入力（0:グー、1:チョキ、2:パー）
  2. 2 コンピュータの手をランダムに決定（ヒント: import random を使用）
  3. 3 if文を使って勝敗を判定し、結果を表示する
サンプルコードを表示 
# じゃんけんゲーム
import random

# プレイヤーの手を入力
player = int(input("手を選んでください (0:グー 1:チョキ 2:パー): "))

# コンピュータの手をランダムに決定
computer = random.randint(0, 2)

# 手の名前
hands = ["グー", "チョキ", "パー"]
print(f"あなた: {hands[player]}")
print(f"コンピュータ: {hands[computer]}")

# 勝敗判定
if player == computer:
    print("引き分けです！")
elif (player == 0 and computer == 1) or \
     (player == 1 and computer == 2) or \
     (player == 2 and computer == 0):
    print("あなたの勝ちです！")
else:
    print("コンピュータの勝ちです！")

理解度確認クイズ

  1. Q1. if文を書く際に、条件式の後ろに必ず付ける記号は何ですか？

    回答を表示

    コロン（:）です。例: if age >= 20:

  2. Q2. 「等しい」を判定する比較演算子は何ですか？「代入」との違いも説明してください。

    回答を表示

    ==（イコール2つ）です。=（イコール1つ）は代入を表します。 比較: if x == 5:（xが5と等しいかチェック）
    代入: x = 5（xに5を代入）

  3. Q3. 次のコードの実行結果を予測してください。
    age = 16
    if age >= 18:
        print("成人")
    elif age >= 13:
        print("中高生")
    else:
        print("子供")

    回答を表示

    「中高生」と表示されます。
    理由: age = 16 なので、最初の条件（age >= 18）は False、 次の条件（age >= 13）が True になるため、「中高生」が実行されます。

  4. Q4. 「AかつB」を表す論理演算子と、「AまたはB」を表す論理演算子は何ですか？

    回答を表示

    「AかつB」→ and
    「AまたはB」→ or
    例: if age >= 20 and age < 65:

第5章のまとめ

  • if文を使うと、条件によって実行する処理を変えることができる
  • elifで複数の条件を順に判定し、elseでどれにも当てはまらない場合の処理を書く
  • 比較演算子（==, !=, >, <, >=, <=）で値を比較し、論理演算子（and, or, not）で条件を組み合わせる
  • 条件式の後ろにはコロン（:）、実行する処理にはインデントが必須

関数の基本 - 処理をパーツ化して再利用しよう

よく使う処理を「関数」としてまとめ、何度でも簡単に呼び出せるようにする方法を学びます。プログラムを部品化して、効率的に開発する技術を習得します。

この章で学ぶこと

  • 関数とは何か、なぜ必要なのかを理解する
  • 関数の定義方法と呼び出し方を学ぶ
  • 引数と戻り値を使って、柔軟な関数を作る
  • デフォルト引数を使った便利な関数を作成する

9.1 関数とは何か？

関数（function）とは、繰り返し使う処理をひとまとめにして、名前を付けたものです。 これは料理のレシピに似ています。「カレーを作る」というレシピを一度書いておけば、 何度でも「カレーを作る」と呼び出すだけで同じ料理を作れるようになります。

関数を使う理由

同じコードを何度も書く代わりに、一度だけ関数として定義しておけば、 必要な時に呼び出すだけで使えます。これにより、コードが短くなり、 修正も一箇所で済むため、プログラムの保守がとても楽になります。

関数の基本的な仕組み

flowchart LR A[プログラム] -->|関数を呼び出す| B[関数: 処理のかたまり] B -->|結果を返す| A B --> C[処理1] B --> D[処理2] B --> E[処理3]

Pythonでは、defキーワードを使って関数を定義します。 関数名の後ろに()を付け、 最後に:で終わります。

9.2 関数の定義と呼び出し

最もシンプルな関数から始めましょう。引数も戻り値もない、挨拶を表示するだけの関数です。

basic_function.py - 基本的な関数 
# 関数を定義する
def greet():
    print("こんにちは！")
    print("Pythonの関数を学習中です")

# 関数を呼び出す
greet()  # 関数名()で呼び出し

# 何度でも呼び出せる
greet()
greet()

重要ポイント

関数は定義しただけでは実行されません。 greet()のように 関数名に()を付けて 呼び出すことで初めて実行されます。

9.3 引数 - 関数に情報を渡す

引数（ひきすう）は、関数に渡す「材料」のようなものです。 カレーを作る時に「辛さ」や「量」を指定できるように、関数にも情報を渡すことができます。

parameter_function.py - 引数を持つ関数 
# 引数を1つ受け取る関数
def greet_person(name):
    print(f"こんにちは、{name}さん！")

# 異なる名前で呼び出せる
greet_person("太郎")  # こんにちは、太郎さん！
greet_person("花子")  # こんにちは、花子さん！
greet_person("健太")  # こんにちは、健太さん！

複数の引数を受け取ることもできます。引数はカンマ,で区切ります。

multiple_parameters.py - 複数の引数 
# 2つの引数を受け取る関数
def introduce(name, age):
    print(f"私の名前は{name}です")
    print(f"年齢は{age}歳です")

# 関数を呼び出す
introduce("太郎", 20)  # 私の名前は太郎です 年齢は20歳です
introduce("花子", 18)  # 私の名前は花子です 年齢は18歳です

9.4 戻り値 - 関数から結果を受け取る

戻り値（もどりち）は、関数が処理した結果を返すものです。 returnキーワードを使って、 計算結果や処理結果を呼び出し元に返すことができます。

return_function.py - 戻り値を返す関数 
# 2つの数値を足し算する関数
def add(a, b):
    result = a + b
    return result  # 計算結果を返す

# 関数の戻り値を変数に代入できる
sum1 = add(5, 3)
print(f"5 + 3 = {sum1}")  # 5 + 3 = 8

sum2 = add(10, 20)
print(f"10 + 20 = {sum2}")  # 10 + 20 = 30

# 直接print()の中で使うこともできる
print(f"100 + 200 = {add(100, 200)}")  # 100 + 200 = 300

引数と戻り値の流れ

flowchart LR A[呼び出し元] -->|引数: 5, 3| B[関数 add] B -->|処理: 5 + 3| C[result = 8] C -->|return 8| A A -->|変数に代入| D[sum1 = 8]

ヒント

returnを実行すると、 その時点で関数の処理が終了し、呼び出し元に戻ります。 returnの後ろのコードは実行されません。

9.5 デフォルト引数 - 省略可能な引数

引数にデフォルト値を設定すると、その引数を省略して関数を呼び出すことができます。 省略された場合は、デフォルト値が使われます。

default_parameter.py - デフォルト引数 
# デフォルト引数を持つ関数
def greet_with_time(name, time="朝"):
    print(f"{time}、{name}さん！")

# 引数を2つ指定する場合
greet_with_time("太郎", "夜")  # 夜、太郎さん！

# 第2引数を省略した場合（デフォルト値が使われる）
greet_with_time("花子")  # 朝、花子さん！
greet_with_time("健太")  # 朝、健太さん！

注意

デフォルト引数は、通常の引数の後ろに書く必要があります。 def func(a=1, b)のように デフォルト引数の後ろに通常の引数を書くとエラーになります。

9.6 関数を使うメリット

関数を使うことで、プログラムの品質が大きく向上します。主なメリットを見ていきましょう。

再利用性

一度書いたコードを何度でも使い回せます。同じ処理を100回書く代わりに、 関数を100回呼び出すだけで済みます。

可読性

複雑な処理に名前を付けることで、コードが読みやすくなります。 「calculate_tax()」という関数名を見れば、何をしているか一目瞭然です。

保守性

処理を修正する時、関数の中身を変えるだけで、 その関数を使っている全ての場所に変更が反映されます。

デバッグ

問題が起きた時、どの関数で起きているかが特定しやすくなります。 処理が小さく分かれているため、原因の切り分けが容易です。

実習 9-1: BMI計算関数を作ろう

体重と身長からBMI（体格指数）を計算する関数を作成してみましょう。

手順

  1. 1 calculate_bmiという名前の関数を定義する
  2. 2 引数として体重（kg）と身長（m）を受け取る
  3. 3 BMI = 体重 ÷ (身長 × 身長) を計算して返す
  4. 4 関数を呼び出して、結果を表示する
解答例を表示 
# BMI計算関数
def calculate_bmi(weight, height):
    bmi = weight / (height * height)
    return bmi

# 関数を呼び出す
my_bmi = calculate_bmi(60, 1.7)
print(f"あなたのBMIは {my_bmi:.2f} です")

# 別の値でも計算できる
bmi2 = calculate_bmi(70, 1.75)
print(f"BMI: {bmi2:.2f}")

理解度確認クイズ

  1. Q1. 関数を定義するために使うキーワードは何ですか？

    回答を表示

    defキーワードを使います。 例: def 関数名():

  2. Q2. 関数から値を返すために使うキーワードは何ですか？

    回答を表示

    returnキーワードを使います。 例: return result

  3. Q3. 関数に渡す情報を何と呼びますか？

    回答を表示

    引数（ひきすう）と呼びます。英語では parameter または argument と言います。

  4. Q4. 関数を使うメリットを3つ挙げてください。

    回答を表示

    ① 再利用性: 同じコードを何度も書かなくて済む
    ② 可読性: 処理に名前を付けることでコードが読みやすくなる
    ③ 保守性: 修正が一箇所で済み、管理が楽になる

第9章のまとめ

  • 関数は繰り返し使う処理をまとめたもので、defキーワードで定義する
  • 引数を使って関数に情報を渡し、戻り値で結果を受け取ることができる
  • デフォルト引数を使うと、省略可能な引数を持つ関数を作成できる
  • 関数を使うことで、再利用性・可読性・保守性が向上し、プログラムの品質が高まる

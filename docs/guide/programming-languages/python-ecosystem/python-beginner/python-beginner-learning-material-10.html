第10章

ファイルの読み書き - データを保存して次回も使えるように

テキストファイルやCSVファイルの読み込みと書き込み方法を学びます。プログラムで作成したデータを保存し、後で再利用する実用的な技術を習得します。

この章で学ぶこと

  • ファイルの開く・閉じるという基本操作を理解する
  • テキストファイルの読み込み・書き込み方法を習得する
  • with文を使った安全なファイル操作の重要性を学ぶ
  • CSVファイルの読み書きができるようになる
  • 実践的なファイル操作プログラムを作成できるようになる

10.1 ファイル操作の基本概念

これまで学んできたプログラムは、実行が終わると作成したデータは消えてしまいました。 例えば、変数に保存したリストや辞書のデータは、プログラムを終了すると失われます。 ファイル操作を使うと、データを永続的に保存し、次回プログラムを実行した時に再利用できるようになります。

身近な例で理解する

メモ帳で文章を書いて保存し、後で開いて編集できるのと同じように、 Pythonプログラムもファイルにデータを書き込んで保存し、後で読み込むことができます。 例えば、ゲームのセーブデータ、設定ファイル、ログファイルなど、 プログラムが終了してもデータを残しておきたい場面は数多くあります。

ファイル操作の基本的な流れ

graph LR A[プログラム開始] --> B[ファイルを開く] B --> C{読み込み/書き込み?} C -->|読み込み| D[データを読み取る] C -->|書き込み| E[データを書き込む] D --> F[ファイルを閉じる] E --> F F --> G[プログラム終了]

ファイル操作には、必ず「開く → 処理する → 閉じる」という3つのステップがあります。 これは、本を読む時に「本を開く → 読む → 本を閉じる」という流れと同じです。 ファイルを開きっぱなしにすると、他のプログラムが使えなくなったり、 データが正しく保存されなかったりする問題が起こる可能性があります。

10.2 テキストファイルの読み込み

まずは、既に存在するテキストファイルを読み込む方法を学びましょう。 open()関数を使って ファイルを開き、その内容を読み取ることができます。

read_file.py - ファイルの基本的な読み込み 
# ファイルを開く（読み込みモード "r"）
file = open("sample.txt", "r", encoding="utf-8")

# ファイル全体を読み込む
content = file.read()
print(content)

# ファイルを閉じる（重要！）
file.close()

read()

ファイル全体を一つの文字列として読み込みます。小さいファイル向け。

readline()

1行ずつ読み込みます。大きなファイルを少しずつ処理する時に便利。

readlines()

全ての行をリストとして読み込みます。各行を個別に処理したい時に。

read_lines.py - 1行ずつ読み込む 
# ファイルを開く
file = open("sample.txt", "r", encoding="utf-8")

# 1行ずつ読み込んで表示
for line in file:
    print(line.strip())  # strip()で改行文字を削除

# ファイルを閉じる
file.close()

ポイント

encoding="utf-8" を指定することで、日本語を含むファイルも正しく読み込めます。指定しないと文字化けする可能性があります。

10.3 テキストファイルの書き込み

次に、ファイルにデータを書き込む方法を学びましょう。 書き込みモードでファイルを開き、write()メソッドを使います。

write_file.py - ファイルへの書き込み 
# ファイルを開く（書き込みモード "w"）
file = open("output.txt", "w", encoding="utf-8")

# データを書き込む
file.write("こんにちは、Python!\n")
file.write("ファイルに書き込みました。\n")

# ファイルを閉じる（これで保存される）
file.close()

print("ファイルに書き込みました！")

ファイルを開くモードの種類

モード 意味 説明
r 読み込み専用 ファイルが存在しないとエラー
w 書き込み専用 既存のファイルを上書き（内容が消える）
a 追記モード ファイルの末尾に追加（既存内容は残る）
r+ 読み書き両用 既存ファイルを読み書き両方できる

注意点

"w"モードでファイルを開くと、既存のファイルの内容は完全に削除されます。 データを残したい場合は"a"（追記）モードを使いましょう。

append_file.py - ファイルへの追記 
# 追記モード "a" で開く
file = open("log.txt", "a", encoding="utf-8")

# ファイルの末尾に追加
file.write("新しいログエントリ\n")

# ファイルを閉じる
file.close()

10.4 with文を使った安全なファイル操作

これまでの例では、close()を 忘れずに呼ぶ必要がありました。しかし、エラーが発生した場合など、close()が実行されない可能性もあります。 with文を使うと、自動的にファイルを閉じてくれるため、より安全です。

with_statement.py - with文を使った安全な操作 
# with文を使った読み込み（推奨）
with open("sample.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)
# ここでファイルは自動的に閉じられる

# with文を使った書き込み
with open("output.txt", "w", encoding="utf-8") as file:
    file.write("with文で書き込みました\n")
# ここでファイルは自動的に閉じられる

ベストプラクティス

ファイル操作は必ずwith文を使いましょう。 エラーが発生してもファイルが確実に閉じられるため、データの損失やリソースの無駄遣いを防げます。 プロのPython開発者は、ほぼ100%の確率でwith文を使います。

with文による自動的なファイルクローズ

graph LR A[with文開始] --> B[ファイルを開く] B --> C[処理を実行] C --> D{エラー発生?} D -->|YES| E[自動的にファイルを閉じる] D -->|NO| E E --> F[with文終了]

10.5 CSVファイルの扱い方

CSV（Comma-Separated Values）は、カンマ区切りでデータを保存する形式です。 Excelなどの表計算ソフトと互換性があり、データの保存や交換に広く使われています。 Pythonには、CSVファイルを簡単に扱うためのcsvモジュールが用意されています。

CSVファイルの例

例えば、以下のような成績データをCSVで保存できます：

名前,数学,英語,国語
太郎,85,90,78
花子,92,88,95
次郎,78,85,82
read_csv.py - CSVファイルの読み込み 
import csv

# CSVファイルを読み込む
with open("scores.csv", "r", encoding="utf-8") as file:
    reader = csv.reader(file)

    # 1行ずつ読み込む
    for row in reader:
        print(row)  # リスト形式で表示される
        # 例: ['太郎', '85', '90', '78']
write_csv.py - CSVファイルの書き込み 
import csv

# CSVファイルに書き込む
with open("output.csv", "w", encoding="utf-8", newline="") as file:
    writer = csv.writer(file)

    # ヘッダー行を書き込む
    writer.writerow(["名前", "年齢", "都市"])

    # データ行を書き込む
    writer.writerow(["太郎", "20", "東京"])
    writer.writerow(["花子", "22", "大阪"])
    writer.writerow(["次郎", "21", "福岡"])

print("CSVファイルを作成しました！")
dict_csv.py - 辞書形式でCSVを扱う 
import csv

# 辞書形式でCSVを読み込む（列名でアクセス可能）
with open("scores.csv", "r", encoding="utf-8") as file:
    reader = csv.DictReader(file)

    for row in reader:
        print(f"{row['名前']}さんの数学の点数: {row['数学']}")
        # 例: 太郎さんの数学の点数: 85

10.6 実践例：シンプルなメモ帳アプリを作ろう

これまで学んだファイル操作の知識を使って、メモを保存・表示できる簡単なメモ帳アプリを作ってみましょう。

実習 10-1: メモ帳アプリの作成

メモの追加・表示・全削除ができるシンプルなメモ帳アプリを作成します。

完成イメージ

--- メモ帳アプリ ---
1. メモを追加
2. メモを表示
3. メモを全削除
4. 終了
選択してください:
memo_app.py - メモ帳アプリの完成版 
def add_memo():
    """メモを追加する"""
    memo = input("メモを入力してください: ")
    with open("memo.txt", "a", encoding="utf-8") as file:
        file.write(memo + "\n")
    print("メモを保存しました！")

def show_memos():
    """保存されたメモを表示する"""
    try:
        with open("memo.txt", "r", encoding="utf-8") as file:
            memos = file.readlines()
            if memos:
                print("\n--- 保存されたメモ ---")
                for i, memo in enumerate(memos, 1):
                    print(f"{i}. {memo.strip()}")
            else:
                print("メモはまだありません。")
    except FileNotFoundError:
        print("メモはまだありません。")

def clear_memos():
    """全てのメモを削除する"""
    confirm = input("本当に全て削除しますか？ (yes/no): ")
    if confirm.lower() == "yes":
        with open("memo.txt", "w", encoding="utf-8") as file:
            pass  # 空のファイルを作成（全削除）
        print("全てのメモを削除しました。")

# メインプログラム
while True:
    print("\n--- メモ帳アプリ ---")
    print("1. メモを追加")
    print("2. メモを表示")
    print("3. メモを全削除")
    print("4. 終了")

    choice = input("選択してください: ")

    if choice == "1":
        add_memo()
    elif choice == "2":
        show_memos()
    elif choice == "3":
        clear_memos()
    elif choice == "4":
        print("メモ帳アプリを終了します。")
        break
    else:
        print("1〜4の数字を入力してください。")

動作確認

プログラムを実行し、以下の操作を試してみましょう：

  1. 1. 「1」を選択してメモを2〜3個追加する
  2. 2. 「2」を選択して保存されたメモが表示されることを確認
  3. 3. 「4」で一度終了し、もう一度起動して「2」を選択
  4. 4. プログラムを終了してもメモが残っていることを確認

トラブルシューティング

FileNotFoundError: ファイルが見つからない

原因: 読み込もうとしたファイルが存在しない、またはパスが間違っている。

解決方法: ファイルが正しい場所にあるか確認し、try-except文でエラーをキャッチして適切に処理しましょう。

UnicodeDecodeError: 文字化けが発生する

原因: ファイルのエンコーディング（文字コード）が一致していない。

解決方法: open()関数でencoding="utf-8"を明示的に指定しましょう。

PermissionError: ファイルにアクセスできない

原因: ファイルが他のプログラムで使用中、または権限がない。

解決方法: ファイルを閉じているか確認し、管理者権限が必要な場合は適切な権限で実行しましょう。

理解度確認クイズ

  1. Q1. ファイル操作の基本的な3つのステップを説明してください。

    回答を表示

    ① ファイルを開く（open関数）
    ② データを読み込む/書き込む（read/write）
    ③ ファイルを閉じる（close、またはwith文で自動）
    この3ステップを守ることで、安全にファイルを操作できます。

  2. Q2. "w"モードと"a"モードの違いを説明してください。

    回答を表示

    "w"モード（書き込み）: 既存のファイル内容を完全に上書き（削除）して新しく書き込む。
    "a"モード（追記）: 既存のファイル内容を保持したまま、末尾に新しい内容を追加する。
    既存データを残したい場合は"a"モードを使用します。

  3. Q3. with文を使うメリットは何ですか？

    回答を表示

    with文を使うと、ブロックを抜ける時に自動的にファイルが閉じられます。 これにより、close()を書き忘れたり、エラーが発生してもファイルが確実に閉じられるため、 データの損失やリソースリークを防げます。プロのPythonプログラマーが推奨する方法です。

  4. Q4. CSVファイルとは何ですか？どのような用途に使われますか？

    回答を表示

    CSV（Comma-Separated Values）は、カンマ区切りでデータを保存するテキスト形式です。 Excelなどの表計算ソフトと互換性があり、顧客リスト、売上データ、成績表など、 表形式のデータを保存・交換する際に広く使われます。 Pythonのcsvモジュールを使えば簡単に扱えます。

第10章のまとめ

  • ファイル操作は「開く → 処理 → 閉じる」の3ステップが基本
  • read()でファイル全体、readline()で1行ずつ、readlines()で全行をリストとして読み込める
  • "w"モードは上書き、"a"モードは追記として使い分ける
  • with文を使うと自動的にファイルが閉じられるため、安全で推奨される
  • csvモジュールを使えば、表形式のデータを簡単に読み書きできる

環境構築 - Pythonをインストールして最初のプログラムを動かそう

Pythonのインストール方法を画像付きで詳しく解説し、「Hello, World!」という最初のプログラムを作成・実行します。プログラムが動く喜びを体験できます。

この章で学ぶこと

  • Windows・Mac・LinuxでのPythonインストール方法を理解する
  • Pythonのバージョン確認方法とインストール確認手順を習得する
  • 初めてのプログラム「Hello, World!」を作成して実行する
  • 対話モードとスクリプトファイルの違いと使い分けを学ぶ

2.1 Pythonのダウンロード

Pythonをインストールする前に、まずは公式サイトから最新版のPythonをダウンロードしましょう。 この章では、各OS（Windows、Mac、Linux）ごとのインストール手順を詳しく解説します。

バージョンについて

2025年現在、Python 3.13が最新版です。Python 2は2020年にサポートが終了しているため、 必ずPython 3系（3.9以上推奨）をインストールしてください。 本ガイドでは、Python 3.13を使用して解説します。

2.2 Windowsへのインストール

Windows版Pythonのインストールは、ダウンロードしたインストーラーを実行するだけで簡単に完了します。 ただし、重要な設定があるので、必ず以下の手順に従ってください。

実習: Windowsにインストールしよう

  1. 1 ダウンロードしたインストーラー（python-3.x.x.exe）をダブルクリック
  2. 2 重要！ 「Add Python 3.x to PATH」に必ずチェックを入れる
  3. 3 「Install Now」をクリックしてインストール開始
  4. 4 「Setup was successful」と表示されたら完了

注意: PATHの設定を忘れずに！

「Add Python 3.x to PATH」のチェックを忘れると、コマンドプロンプトでPythonを実行できません。 もしチェックを忘れた場合は、一度アンインストールして再インストールするか、 手動でPATHを設定する必要があります。

2.3 macOS・Linuxへのインストール

macOSの場合

macOSには標準でPythonが入っていますが、古いバージョン（Python 2.x）の場合があるため、 最新のPython 3をインストールすることをお勧めします。

Terminal - Homebrewを使ったインストール（推奨） 
# Homebrewがインストールされていない場合は、まずHomebrewをインストール
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Pythonをインストール
brew install python3

# バージョン確認
python3 --version

Linuxの場合

多くのLinuxディストリビューションには最初からPython 3がインストールされています。 バージョンが古い場合や、インストールされていない場合は、パッケージマネージャーを使ってインストールします。

Ubuntu/Debian系

# パッケージリスト更新
sudo apt update

# Python3インストール
sudo apt install python3 python3-pip

# バージョン確認
python3 --version

CentOS/RHEL系

# Python3インストール
sudo yum install python3

# バージョン確認
python3 --version

2.4 インストール確認 - バージョンチェック

Pythonが正しくインストールされたか確認しましょう。 コマンドプロンプト（Windows）またはターミナル（Mac/Linux）を開いて、以下のコマンドを実行します。

動作確認: バージョンチェック

Windows

# コマンドプロンプトで実行
python --version

# または
py --version

Mac/Linux

# ターミナルで実行
python3 --version

正しい出力例：

Python 3.13.0

このように「Python 3.x.x」と表示されればインストール成功です！

トラブルシューティング

❌ 「'python' は、内部コマンドまたは外部コマンドとして認識されていません」

→ PATHが設定されていません。Pythonを再インストールして「Add Python to PATH」にチェックを入れてください。

❌ 「Python 2.7.x」と表示される（Mac/Linux）

→ pythonではなくpython3コマンドを使用してください。

❌ コマンドプロンプトが開けない（Windows）

→ Windowsキー + R → 「cmd」と入力してEnter、またはスタートメニューから「コマンドプロンプト」を検索

2.5 最初のプログラム「Hello, World!」

インストールが確認できたら、さっそくプログラムを動かしてみましょう！ プログラミングの世界では、最初に「Hello, World!」と表示するプログラムを作るのが伝統です。

方法1: 対話モードで実行

対話モード（インタラクティブモード）は、1行ずつコードを入力してすぐに実行できるモードです。 簡単な動作確認や計算に便利です。

実習: 対話モードで Hello, World!

  1. 1 コマンドプロンプト（またはターミナル）を開く
  2. 2 python（Mac/Linuxはpython3）と入力してEnter
  3. 3 以下のコードを1行ずつ入力してEnter
Python対話モード 
# >>> はPythonの対話モードのプロンプト（自動表示されます）
>>> print("Hello, World!")
Hello, World!

>>> print("ようこそ、Pythonの世界へ！")
ようこそ、Pythonの世界へ！

# 終了するには exit() を入力
>>> exit()

方法2: スクリプトファイルで実行

スクリプトファイル（.pyファイル）にコードを書いて保存すれば、何度でも実行できます。 実際の開発ではこちらの方法を使います。

実習: スクリプトファイルで Hello, World!

  1. 1 メモ帳（またはテキストエディタ）を開く
  2. 2 以下のコードを入力
hello.py - 初めてのPythonスクリプト 
# 初めてのPythonプログラム
print("Hello, World!")
print("私はPythonを学んでいます")
print("プログラミングは楽しい！")
  1. 3 ファイル名をhello.pyとして保存（デスクトップなど分かりやすい場所に）
  2. 4 コマンドプロンプトで保存した場所に移動して実行 
# Windowsの例（デスクトップに保存した場合）
cd Desktop
python hello.py

# Mac/Linuxの例
cd ~/Desktop
python3 hello.py

実行結果：

Hello, World!
私はPythonを学んでいます
プログラミングは楽しい！

2.6 コードエディタの紹介

メモ帳でもPythonのコードは書けますが、プログラミング専用のエディタを使うと、 コードが色分けされて読みやすくなったり、自動補完機能が使えたりして、学習効率が格段に上がります。

IDLE（Python付属）

Pythonインストール時に一緒にインストールされる公式エディタです。 すぐに使い始められるので、初心者におすすめです。

初心者向け 無料 設定不要

Visual Studio Code（推奨）

Microsoftが開発した高機能エディタ。Python拡張機能をインストールすることで、 強力なコード補完やデバッグ機能が使えます。

高機能 無料 拡張性高

PyCharm

JetBrains社のPython専用IDE（統合開発環境）。 本格的なPython開発に最適ですが、少し重いです。

プロ向け Community版無料 高機能

Google Colab

ブラウザ上で動くPython環境。インストール不要で、すぐに始められます。 データ分析やAI学習に特に便利です。

クラウド 無料 インストール不要

おすすめの学習ステップ

最初はIDLEでPythonの基本に慣れる → Visual Studio Codeに移行して本格的に学習 → 将来的にPyCharmで大規模開発、というステップがおすすめです。

理解度確認クイズ

  1. Q1. Windowsにインストールする際、必ずチェックを入れるべき項目は何ですか？

    回答を表示

    「Add Python 3.x to PATH」にチェックを入れる必要があります。 これを忘れると、コマンドプロンプトでPythonコマンドが認識されません。

  2. Q2. インストール後、Pythonが正しくインストールされたか確認するコマンドは何ですか？

    回答を表示

    Windows: python --version または py --version
    Mac/Linux: python3 --version
    「Python 3.x.x」と表示されれば成功です。

  3. Q3. 対話モードとスクリプトファイルの違いは何ですか？それぞれどのような場面で使いますか？

    回答を表示

    対話モード：1行ずつ入力してすぐ実行。簡単な動作確認や計算に便利。
    スクリプトファイル：.pyファイルに保存して実行。実際のプログラム開発で使用。 何度も実行したいプログラムはスクリプトファイルにします。

  4. Q4. print()関数は何をする命令ですか？実際に使ってみた感想を書いてください。

    回答を表示

    print()関数は、括弧内の文字列や値を画面に表示する命令です。 "Hello, World!"と表示されたときは、プログラムが動いた実感があって嬉しいですね！

第2章のまとめ

  • Pythonのインストールは、公式サイトからダウンロードして実行するだけ（WindowsはPATH設定を忘れずに）
  • バージョン確認コマンド（python --version）でインストールが成功したか確認できる
  • Hello, World!プログラムで、print()関数の使い方と実行方法を学んだ
  • 対話モードとスクリプトファイルの2つの実行方法があり、用途に応じて使い分ける
  • コードエディタを使うと、プログラミングがより効率的になる（IDLE、VS Codeなど）

次の第3章では、プログラムで扱うデータを保存する「変数」という概念を学びます。 数値や文字列などのデータ型について、身近な例で理解していきましょう！

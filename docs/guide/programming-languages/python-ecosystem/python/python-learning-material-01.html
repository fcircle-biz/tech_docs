Python入門と開発環境構築

この章で学ぶこと
  • Pythonの特徴と用途を理解する
  • Python開発環境をセットアップする
  • 最初のPythonプログラムを作成し実行する
  • インタープリターとスクリプトの実行方法を学ぶ
  • Python開発に必要なツールの基本操作を習得する

1.1 Pythonとは何か

Pythonは1991年にGuido van Rossumによって開発されたプログラミング言語です。 シンプルで読みやすい構文が特徴で、初心者にも学びやすい言語として人気があります。

Pythonの主な特徴

  • 読みやすい構文: インデントによるブロック構造で、コードが自然に整理される
  • 豊富なライブラリ: 標準ライブラリと外部ライブラリが充実している
  • クロスプラットフォーム: Windows、macOS、Linuxで動作する
  • オープンソース: 無料で使用でき、コミュニティが活発
  • 多目的: Web開発、データ分析、AI、自動化など幅広い用途

Pythonの用途

  • Webアプリケーション開発（Django、Flask）
  • データ分析・機械学習（pandas、NumPy、scikit-learn）
  • 自動化スクリプト
  • デスクトップアプリケーション
  • ゲーム開発

1.2 Python環境のセットアップ

Pythonのインストール

Python 3.8以降のバージョンをインストールします。 公式サイト（python.org）からダウンロードできます。

注意事項

Python 2.xは既にサポートが終了しているため、必ずPython 3.xをインストールしてください。

インストール手順（Windows）

  1. Python公式サイトにアクセス
  2. 「Download Python 3.x.x」ボタンをクリック
  3. ダウンロードしたインストーラーを実行
  4. 「Add Python to PATH」にチェックを入れる
  5. 「Install Now」をクリック

インストール確認

コマンドプロンプトまたはターミナルで以下のコマンドを実行します：

python --version

正しくインストールされていれば、Pythonのバージョンが表示されます。

1.3 開発環境の準備

推奨エディタ・IDE

  • Visual Studio Code: 無料、軽量、拡張機能が豊富
  • PyCharm: Python専用IDE、高機能
  • IDLE: Python標準の統合開発環境
  • Jupyter Notebook: データ分析に最適

Visual Studio Codeの設定

  1. VS Codeをインストール
  2. Python拡張機能をインストール
  3. フォルダを開いて新しいPythonファイルを作成
  4. インタープリターが正しく設定されているか確認
実習 1-1: 最初のPythonプログラム

「Hello, Python!」を出力する最初のプログラムを作成しましょう。

手順
  1. テキストエディタまたはIDEを開く
  2. 新しいファイルを作成し、「hello.py」として保存
  3. 以下のコードを入力する
  4. ファイルを保存し、実行する
コード例
print("Hello, Python!")
print("Python学習を始めました！")

# 変数を使った例
name = "Python初心者"
print(f"こんにちは、{name}さん！")
実行方法
python hello.py
期待される結果
Hello, Python!
Python学習を始めました！
こんにちは、Python初心者さん！

1.4 Pythonインタープリターの使い方

対話型インタープリター

コマンドラインでpythonと入力すると、対話型インタープリターが起動します。

$ python
Python 3.x.x (...)
>>> print("Hello, World!")
Hello, World!
>>> 2 + 3
5
>>> exit()

スクリプトの実行

Pythonファイル（.py）を作成して実行することで、プログラムを保存・再利用できます。

実習 1-2: インタープリターの基本操作

対話型インタープリターを使って、基本的な計算を行いましょう。

手順
  1. コマンドラインでpythonを実行
  2. 以下の計算を順番に実行する
  3. 結果を確認する
  4. exit()でインタープリターを終了
実行例
>>> 10 + 5
15
>>> 20 * 3
60
>>> "Python" + "学習"
'Python学習'
>>> len("プログラミング")
7

1.5 開発ツールの基本

pipパッケージマネージャー

Pythonライブラリのインストールに使用するツールです。

# pipのバージョン確認
pip --version

# ライブラリのインストール例
pip install requests

仮想環境

プロジェクトごとに独立したPython環境を作成できます。

# 仮想環境の作成
python -m venv myproject

# 仮想環境の有効化（Windows）
myproject\Scripts\activate

# 仮想環境の有効化（macOS/Linux）
source myproject/bin/activate

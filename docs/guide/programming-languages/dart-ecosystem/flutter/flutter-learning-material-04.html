Dart言語の基礎（前編）

Flutterで使用するDart言語の基本を学びます。変数、データ型、演算子、文字列操作など、プログラミングの基礎概念を丁寧に解説します。

この章で学ぶこと

  • 変数の宣言と代入の方法を理解する
  • 基本的なデータ型（整数、小数、文字列、真偽値）を使い分ける
  • 算術演算子や比較演算子の使い方を学ぶ
  • 文字列の結合や補間（変数埋め込み）を実践する

4.1 Dart言語とは

Dart（ダート）は、Googleが開発したプログラミング言語で、Flutterアプリの開発に使用されます。JavaやJavaScriptと似た文法を持ち、初心者にも学びやすい設計になっています。

型安全

データの型を明確に定義し、バグを事前に防ぐ

オブジェクト指向

クラスとオブジェクトで構造化された設計

学びやすい文法

JavaやJavaScriptに似た親しみやすい構文

高速実行

AOTコンパイルによるネイティブ並みの速度

4.2 変数とは

変数は、データを一時的に保存するための「箱」のようなものです。箱に名前（変数名）をつけて、その中にデータを入れたり、取り出したりできます。

変数のイメージ

graph LR A["変数名: age"] --> B["箱の中身: 25"] C["変数名: name"] --> D["箱の中身: 太郎"] E["変数名: isStudent"] --> F["箱の中身: true"] style A fill:#dbeafe style C fill:#d1fae5 style E fill:#fef3c7

変数宣言の基本形

Dart
// 型を指定した宣言
int age = 25;        // 整数型（25歳）
String name = '太郎';  // 文字列型（名前）
bool isStudent = true; // 真偽値型（学生かどうか）

// var を使った宣言（型推論）
var height = 170;    // int型と自動推論
var city = '東京';    // String型と自動推論

重要ポイント

  • セミコロン（;）が必要: Dartでは文の最後に必ずセミコロンをつけます
  • varとデータ型: varを使うと型を自動推論、型名を書くと明示的に指定
  • 文字列はシングルクォート: Dartでは文字列を'...'または"..."で囲みます

4.3 基本的なデータ型

Dartには様々なデータ型がありますが、まずは以下の4つの基本型を覚えましょう。

1 int（整数型）

整数（小数点を含まない数値）を表現します。年齢、個数、点数など。

Dart
int age = 25;        // 年齢
int count = 100;     // カウント数
int score = 80;      // 点数
int temperature = -5; // 気温（マイナスも可能）

2 double（浮動小数点型）

小数点を含む数値を表現します。身長、体重、価格など。

Dart
double height = 170.5;   // 身長（cm）
double weight = 65.3;    // 体重（kg）
double price = 1980.0;   // 価格（円）
double pi = 3.14159;     // 円周率

3 String（文字列型）

文字や文章を表現します。名前、メッセージ、住所など。

Dart
String name = '山田太郎';         // 名前
String city = '東京';            // 都市名
String message = 'こんにちは！';  // メッセージ
String email = "test@example.com"; // ダブルクォートも可

4 bool（真偽値型）

真（true）か偽（false）の2つの値のみを持ちます。フラグや条件判定に使用。

Dart
bool isStudent = true;     // 学生かどうか
bool hasLicense = false;   // 免許を持っているか
bool isRaining = true;     // 雨が降っているか
bool isLoggedIn = false;   // ログインしているか

4.4 演算子

演算子は、計算や比較を行うための記号です。ここでは主要な演算子を紹介します。

算術演算子

数値の計算に使用します。

Dart
int a = 10;
int b = 3;

int sum = a + b;      // 加算: 13
int diff = a - b;     // 減算: 7
int product = a * b;  // 乗算: 30
double quotient = a / b; // 除算: 3.333...
int remainder = a % b;   // 剰余（余り）: 1

比較演算子

2つの値を比較し、真（true）か偽（false）を返します。

Dart
int x = 10;
int y = 20;

bool isEqual = x == y;     // 等しい: false
bool isNotEqual = x != y;  // 等しくない: true
bool isGreater = x > y;    // より大きい: false
bool isLess = x < y;       // より小さい: true
bool isGreaterOrEqual = x >= y; // 以上: false
bool isLessOrEqual = x <= y;    // 以下: true

論理演算子

複数の条件を組み合わせるために使用します。

Dart
bool isStudent = true;
bool hasLicense = false;

// AND演算（かつ）: 両方がtrueの場合のみtrue
bool canDrive = isStudent && hasLicense; // false

// OR演算（または）: どちらかがtrueならtrue
bool canEnter = isStudent || hasLicense; // true

// NOT演算（否定）: trueとfalseを反転
bool isNotStudent = !isStudent; // false

4.5 文字列操作

Dartでは文字列を簡単に操作できます。文字列の結合や変数の埋め込みなど、よく使う操作を学びましょう。

文字列の結合

+演算子で文字列を連結できます。

Dart
String firstName = '山田';
String lastName = '太郎';

// 文字列の結合
String fullName = firstName + lastName;  // '山田太郎'
String greeting = 'こんにちは、' + fullName + 'さん！';
// 結果: 'こんにちは、山田太郎さん！'

文字列補間（変数埋め込み）

$変数名または${式}で文字列内に変数を埋め込めます。

Dart
String name = '太郎';
int age = 25;

// $変数名 で埋め込み
String message = 'こんにちは、$nameさん！';
// 結果: 'こんにちは、太郎さん！'

// ${式} で式の結果を埋め込み
String info = '私は$nameで、${age}歳です。';
// 結果: '私は太郎で、25歳です。'

// 計算式も埋め込める
String future = '来年は${age + 1}歳になります。';
// 結果: '来年は26歳になります。'

複数行文字列

トリプルクォート（'''...'''）で複数行の文字列を表現できます。

Dart
String multiLine = '''
これは
複数行の
文字列です。
''';

// 変数も埋め込める
String name = '太郎';
String message = '''
こんにちは、$nameさん。
今日もよろしくお願いします。
''';

1

変数とデータ型の練習

DartPad（https://dartpad.dev/）を開き、以下のコードを入力して実行してみましょう。

Dart
void main() {
  // 自分の情報を変数に格納
  String myName = 'あなたの名前';
  int myAge = 25;
  double myHeight = 170.5;
  bool isStudent = true;

  // 表示
  print('名前: $myName');
  print('年齢: $myAge歳');

やってみよう: 変数の値を自分の情報に書き換えて実行してみましょう。

2

計算と文字列補間

商品の価格計算プログラムを作成してみましょう。

Dart
void main() {
  // 商品情報
  String productName = 'ノートパソコン';
  int price = 80000;
  int quantity = 2;

  // 合計金額を計算
  int totalPrice = price * quantity;

  // 結果を表示
  print('商品名: $productName');
  print('単価: ${price}円');
  print('数量: $quantity個');
  print('合計: ${totalPrice}円');
}

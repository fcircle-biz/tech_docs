Dart言語の関数とクラス

関数の定義と呼び出し、クラスとオブジェクトの基本概念を学びます。Flutterのウィジェットを理解するための土台を作ります。

この章で学ぶこと

  • 関数の定義と呼び出し、引数と戻り値の概念を理解する
  • 名前付き引数とオプション引数の使い方を学ぶ
  • クラスとオブジェクトの基本概念を理解する
  • コンストラクタ、メソッド、プロパティの使い方を習得する

6.1 関数とは

関数（Function）は、特定の処理をまとめて名前をつけたものです。一度定義すれば、何度でも呼び出して使用できます。

関数を使う理由

  • コードの再利用: 同じ処理を何度も書かなくて済む
  • 可読性の向上: 複雑な処理に名前をつけることで理解しやすくなる
  • メンテナンス性: 修正が必要な場合、1箇所を直せば全体に反映される

関数の基本構文

Dart 
// 基本的な関数の定義
void greet() {
  print('こんにちは！');
}

// 関数の呼び出し
void main() {
  greet(); // 出力: こんにちは！
  greet(); // 出力: こんにちは！（何度でも呼び出せる）
}

voidは「戻り値なし」を意味します。関数名の後ろの()内に引数を書きます（この例では引数なし）。

6.2 引数と戻り値

関数には引数（入力）と戻り値（出力）を設定できます。

引数のある関数

Dart 
// 引数を受け取る関数
void greetPerson(String name) {
  print('こんにちは、${name}さん！');
}

void main() {
  greetPerson('太郎'); // 出力: こんにちは、太郎さん！
  greetPerson('花子'); // 出力: こんにちは、花子さん！
}

戻り値のある関数

Dart 
// 戻り値を返す関数（int型を返す）
int add(int a, int b) {
  return a + b; // 計算結果を返す
}

void main() {
  int result = add(3, 5);
  print('結果: $result'); // 出力: 結果: 8
}

6.3 名前付き引数とオプション引数

Dartには、引数を柔軟に扱うための便利な機能があります。

名前付き引数

Dart 
// 名前付き引数（中括弧で囲む）
void introduce({required String name, required int age}) {
  print('名前: $name、年齢: $age歳');
}

void main() {
  // 引数の順序を入れ替えても動作する
  introduce(name: '太郎', age: 20);
  introduce(age: 25, name: '花子');
}

requiredは「必須」を意味します。名前付き引数は呼び出し時に引数名を指定するため、順序を気にしなくて良いのが利点です。

オプション引数（デフォルト値）

Dart 
// デフォルト値を設定（オプション引数）
void greet({String name = 'ゲスト', String message = 'こんにちは'}) {
  print('$message、${name}さん！');
}

void main() {
  greet(); // 出力: こんにちは、ゲストさん！
  greet(name: '太郎'); // 出力: こんにちは、太郎さん！
  greet(name: '花子', message: 'おはよう'); // 出力: おはよう、花子さん！
}

Flutterでの活用例

Flutterのウィジェットは名前付き引数を多用します。例えばContainer(width: 100, height: 50, color: Colors.blue)のように、引数名を明示することで可読性が向上します。

6.4 クラスとは

クラス（Class）は、データ（プロパティ）と処理（メソッド）をまとめた設計図です。クラスからオブジェクト（インスタンス）を作成して使用します。

クラスとオブジェクトの関係

クラスは「たい焼きの型」、オブジェクトは「実際のたい焼き」に例えられます。1つの型（クラス）から複数のたい焼き（オブジェクト）を作れます。

クラスの基本構文

Dart 
// クラスの定義
class Person {
  // プロパティ（データ）
  String name;
  int age;

  // コンストラクタ（オブジェクト作成時に呼ばれる）
  Person(this.name, this.age);

  // メソッド（処理）
  void introduce() {
    print('私は${name}です。${age}歳です。');
  }
}

void main() {
  // オブジェクトの作成
  var taro = Person('太郎', 20);
  var hanako = Person('花子', 25);

  // メソッドの呼び出し
  taro.introduce(); // 出力: 私は太郎です。20歳です。
  hanako.introduce(); // 出力: 私は花子です。25歳です。
}

6.5 コンストラクタ

コンストラクタ（Constructor）は、オブジェクトを作成するときに呼ばれる特別なメソッドです。初期化処理を行います。

名前付きコンストラクタ

Dart 
class Point {
  double x;
  double y;

  // 通常のコンストラクタ
  Point(this.x, this.y);

  // 名前付きコンストラクタ（原点を作成）
  Point.origin()
      : x = 0,
        y = 0;
}

void main() {
  var p1 = Point(3, 4);
  var p2 = Point.origin(); // 原点(0, 0)を作成
  print('p1: (${p1.x}, ${p1.y})'); // 出力: p1: (3.0, 4.0)
  print('p2: (${p2.x}, ${p2.y})'); // 出力: p2: (0.0, 0.0)
}

Flutterでの活用

FlutterではEdgeInsets.all(8)やEdgeInsets.symmetric(vertical: 10)のように、名前付きコンストラクタを頻繁に使います。

6.6 プロパティとメソッド

クラスにはプロパティ（データ）とメソッド（処理）を定義できます。

ゲッター・セッター

Dart 
class Rectangle {
  double width;
  double height;

  Rectangle(this.width, this.height);

  // ゲッター（計算プロパティ）
  double get area => width * height;

  // セッター（値の設定時に処理を挟む）
  set setWidth(double value) {
    if (value > 0) {
      width = value;
    }
  }
}

void main() {
  var rect = Rectangle(10, 5);
  print('面積: ${rect.area}'); // 出力: 面積: 50.0

  rect.setWidth = 20;
  print('面積: ${rect.area}'); // 出力: 面積: 100.0
}

実習: Carクラスの作成

以下の仕様に従って、車を表すクラスを作成してみましょう。

仕様:

  • プロパティ: name（車名）、speed（速度）
  • メソッド: accelerate()（速度+10）、brake()（速度-10）、showInfo()（情報表示）
解答例を見る
Dart 
class Car {
  String name;
  int speed;

  Car(this.name, {this.speed = 0});

  void accelerate() {
    speed += 10;
    print('${name}が加速しました。現在の速度: ${speed}km/h');
  }

  void brake() {
    if (speed >= 10) {
      speed -= 10;
      print('${name}が減速しました。現在の速度: ${speed}km/h');
    }
  }

  void showInfo() {
    print('車名: $name、速度: ${speed}km/h');
  }
}

void main() {
  var myCar = Car('プリウス');
  myCar.showInfo(); // 出力: 車名: プリウス、速度: 0km/h
  myCar.accelerate(); // 出力: プリウスが加速しました。現在の速度: 10km/h
  myCar.accelerate(); // 出力: プリウスが加速しました。現在の速度: 20km/h
  myCar.brake(); // 出力: プリウスが減速しました。現在の速度: 10km/h
}

第6章のまとめ

  • 関数は処理をまとめて名前をつけたもので、再利用性とメンテナンス性を向上させる
  • 関数には引数（入力）と戻り値（出力）を設定できる
  • 名前付き引数は中括弧で囲み、呼び出し時に引数名を指定する
  • クラスはオブジェクトの設計図で、データ（プロパティ）と処理（メソッド）をまとめる
  • コンストラクタはオブジェクト作成時に呼ばれる特別なメソッド
  • Flutterのウィジェットもクラスで、名前付き引数を多用する

ウィジェットの基本

Flutterの核心であるウィジェットの概念を学びます。StatelessWidgetとStatefulWidgetの違い、基本的なウィジェット（Text、Container、Icon等）の使い方を理解します。

この章で学ぶこと

  • Flutterにおけるウィジェットの役割と基本概念を理解する
  • StatelessWidgetとStatefulWidgetの違いを学ぶ
  • 基本的なウィジェット（Text、Container、Icon、Image等）の使い方をマスターする
  • ウィジェットツリーとウィジェットの組み合わせ方を理解する

7.1 ウィジェットとは

Widget（ウィジェット）は、Flutterアプリケーションを構成する基本的な構成要素です。画面に表示されるすべての要素（ボタン、テキスト、画像、レイアウトなど）はウィジェットで表現されます。

Flutterの設計哲学は「Everything is a Widget（すべてはウィジェット）」であり、UIの構築はウィジェットを組み合わせるツリー構造で行います。

ウィジェットの特徴

  • イミュータブル（不変）: 一度作成されたウィジェットは変更できない
  • 宣言的UI: 「今どうあるべきか」を宣言することでUIを構築
  • 再利用可能: カスタムウィジェットを作成して何度も利用できる
  • ツリー構造: ウィジェットを入れ子にしてUIを組み立てる

ウィジェットツリーの構造

graph TD A["MaterialApp
（アプリ全体）"] --> B["Scaffold
（画面の骨組み）"] B --> C["AppBar
（ヘッダー）"] B --> D["Body

（縦並び）"] E --> F["Text

（アイコン）"]

このように、親ウィジェットの中に子ウィジェットを配置することでUIを構築します。

7.2 StatelessWidgetとStatefulWidget

Flutterには大きく分けて2種類のウィジェットがあります。

StatelessWidget（状態を持たないウィジェット）

StatelessWidgetは、作成後に内容が変更されないウィジェットです。一度表示されたら、そのまま変わらない静的なコンテンツに使用します。

使用例: 固定のテキスト、アイコン、画像など
StatelessWidgetの例 
// StatelessWidgetの定義
class MyText extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Text(
      'こんにちは、Flutter！',
      style: TextStyle(fontSize: 20),
    );
  }
}

StatefulWidget（状態を持つウィジェット）

StatefulWidgetは、時間とともに変化する状態を持つウィジェットです。ユーザーの操作やデータの変更に応じて表示内容が更新されます。

使用例: カウンター、入力フォーム、チェックボックスなど
StatefulWidgetの基本構造 
// StatefulWidgetは2つのクラスで構成される
class Counter extends StatefulWidget {
  @override
  _CounterState createState() => _CounterState();
}

class _CounterState extends State<Counter> {
  int count = 0; // 状態（変化する値）

  @override
  Widget build(BuildContext context) {
    return Text('カウント: $count');
  }
}
特徴 StatelessWidget StatefulWidget
状態の保持 しない する
再描画 親から再構築されたときのみ setState()で自由に更新可能
使用場面 静的コンテンツ 動的コンテンツ
例 固定テキスト、アイコン カウンター、フォーム

7.3 基本的なウィジェット - Text

Textウィジェットは、画面にテキストを表示するための基本的なウィジェットです。

Textウィジェットの基本 
// シンプルなテキスト
Text('Hello, Flutter!'),

// スタイル付きテキスト
Text(
  'スタイル付きテキスト',
  style: TextStyle(
    fontSize: 24,        // 文字サイズ
    fontWeight: FontWeight.bold, // 太字
    color: Colors.blue,  // 文字色
  ),
),

TextStyleの主なプロパティ

  • fontSize - 文字サイズ（double型）
  • fontWeight - 太さ（FontWeight.normal, FontWeight.boldなど）
  • color - 文字色（Colors.red, Colors.blueなど）
  • fontStyle - スタイル（FontStyle.normal, FontStyle.italicなど）
  • letterSpacing - 文字間隔
  • decoration - 装飾（下線、取り消し線など）

7.4 基本的なウィジェット - Container

Containerは、ボックス（箱）を表現するウィジェットで、余白、背景色、枠線、角丸などを設定できます。レイアウトの基本となる重要なウィジェットです。

Containerウィジェットの例 
Container(
  width: 200,           // 幅
  height: 100,          // 高さ
  padding: EdgeInsets.all(16), // 内側の余白
  margin: EdgeInsets.all(8),   // 外側の余白
  decoration: BoxDecoration(
    color: Colors.blue,  // 背景色
    borderRadius: BorderRadius.circular(12), // 角丸
    border: Border.all(color: Colors.black, width: 2), // 枠線
  ),
  child: Text('Container内のテキスト'), // 子ウィジェット
),

Containerの主なプロパティ

  • width / height - 幅と高さ
  • padding - 内側の余白
  • margin - 外側の余白
  • decoration - 装飾（背景色、枠線、影など）
  • alignment - 子ウィジェットの配置
  • child - 内部に配置するウィジェット（1つのみ）

余白（EdgeInsets）の指定方法

余白の指定パターン 
EdgeInsets.all(16)        // 上下左右すべて16
EdgeInsets.symmetric(horizontal: 16, vertical: 8) // 水平16、垂直8
EdgeInsets.only(left: 8, top: 16) // 個別指定
EdgeInsets.fromLTRB(8, 16, 8, 16) // 左、上、右、下の順

7.5 基本的なウィジェット - Icon

Iconウィジェットは、アイコンを表示するためのウィジェットです。FlutterにはMaterial Iconsが標準で組み込まれており、数千種類のアイコンを使用できます。

Iconウィジェットの例 
// シンプルなアイコン
Icon(Icons.favorite),

// スタイル付きアイコン
Icon(
  Icons.favorite,
  size: 48,           // サイズ
  color: Colors.red,  // 色
),

よく使うアイコンの例

Icons.home

Icons.person

Icons.settings

Icons.favorite

Icons.search

Icons.shopping_cart

Icons.email

Icons.notifications

7.6 基本的なウィジェット - Image

Imageウィジェットは、画像を表示するためのウィジェットです。ローカル画像、ネットワーク画像、アセット画像など、さまざまなソースから画像を読み込めます。

Imageウィジェットの例 
// ネットワーク画像
Image.network('https://example.com/image.png'),

// アセット画像（プロジェクト内の画像）
Image.asset('assets/images/logo.png'),

// サイズ指定
Image.network(
  'https://example.com/image.png',
  width: 200,
  height: 200,
  fit: BoxFit.cover, // 画像の表示方法
),

BoxFit（画像の表示方法）

  • BoxFit.cover - 領域を埋めるように拡大（はみ出た部分は切り取り）
  • BoxFit.contain - 全体が見えるように縮小
  • BoxFit.fill - 領域に合わせて伸縮（アスペクト比無視）
  • BoxFit.fitWidth - 幅に合わせる
  • BoxFit.fitHeight - 高さに合わせる

7.7 実習: 基本ウィジェットを組み合わせる

実習: プロフィールカードを作成

Container、Text、Icon、Imageを組み合わせて、簡単なプロフィールカードを作成してみましょう。

プロフィールカードのコード 
import 'package:flutter/material.dart';

class ProfileCard extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Container(
      padding: EdgeInsets.all(20),
      margin: EdgeInsets.all(16),
      decoration: BoxDecoration(
        color: Colors.white,
        borderRadius: BorderRadius.circular(16),
        boxShadow: [
          BoxShadow(color: Colors.black12, blurRadius: 10),
        ],
      ),
      child: Column(
        children: [
          Icon(Icons.person, size: 80, color: Colors.blue),
          Text('山田太郎', style: TextStyle(fontSize: 24)),
          Text('Flutterエンジニア', style: TextStyle(color: Colors.grey)),
        ],
      ),
    );
  }
}

ヒント

  • Columnウィジェット（次章で詳しく学習）を使うと、複数のウィジェットを縦に並べられます
  • BoxShadowで影をつけると、カードらしい立体感が出ます
  • 色やサイズを変更して、自分なりのデザインにカスタマイズしてみましょう

第7章のまとめ

  • Widgetは、Flutterアプリを構成する基本的な構成要素であり、「Everything is a Widget」が設計哲学
  • StatelessWidgetは状態を持たない静的なウィジェット、StatefulWidgetは状態を持つ動的なウィジェット

レイアウトの基礎

Row、Column、Stack、Containerを使った画面レイアウトの組み立て方を学びます。Flexboxに似た配置の考え方を理解します。

この章で学ぶこと

  • Columnウィジェットを使った縦方向のレイアウト構築
  • Rowウィジェットを使った横方向のレイアウト構築
  • Stackウィジェットを使った重ね合わせレイアウト
  • mainAxisAlignmentとcrossAxisAlignmentによる配置制御

8.1 レイアウトとは

レイアウトとは、画面上にウィジェット（テキスト、ボタン、画像など）をどのように配置するかを決めることです。Flutterでは、レイアウトもウィジェットで表現されます。

主なレイアウトウィジェットには以下の3つがあります:

Column

縦方向に並べる

Row

横方向に並べる

Stack

重ねて配置する

レイアウトウィジェットの配置イメージ

8.2 Columnウィジェット（縦方向配置）

Columnは、子ウィジェットを縦方向（上から下へ）に並べるレイアウトウィジェットです。

基本的なColumn - main.dart 
// Columnで縦に並べる
Column(
  children: [
    Text('1つ目のテキスト'),  // 一番上
    Text('2つ目のテキスト'),  // 真ん中
    Text('3つ目のテキスト'),  // 一番下
  ],
)

mainAxisAlignment（主軸の配置）

Columnの場合、主軸は縦方向です。mainAxisAlignmentで縦方向の配置を制御します。

mainAxisAlignmentの種類

MainAxisAlignment.start（上揃え・デフォルト）

子ウィジェットを上に詰めて配置します。

MainAxisAlignment.center（中央揃え）

子ウィジェットを縦方向の中央に配置します。

MainAxisAlignment.end（下揃え）

子ウィジェットを下に詰めて配置します。

MainAxisAlignment.spaceBetween（均等配置）

最初と最後を端に配置し、残りを均等に間隔を空けて配置します。

mainAxisAlignmentの使用例 
// 中央揃え
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  children: [
    Text('上のテキスト'),
    Text('真ん中のテキスト'),
    Text('下のテキスト'),
  ],
)

crossAxisAlignment（交差軸の配置）

Columnの場合、交差軸は横方向です。crossAxisAlignmentで横方向の配置を制御します。

8.3 Rowウィジェット（横方向配置）

Rowは、子ウィジェットを横方向（左から右へ）に並べるレイアウトウィジェットです。Columnと使い方はほぼ同じですが、方向が異なります。

基本的なRow 
// Rowで横に並べる
Row(
  children: [
    Icon(Icons.home),     // 左端
    Text('ホーム'),        // 真ん中
    Icon(Icons.settings), // 右端
  ],
)

RowとColumnの違い

Rowの主軸は横方向、Columnの主軸は縦方向です。それ以外の使い方はほぼ同じです。

Rowでの均等配置 
// 左右端に配置して真ん中に余白
Row(
  mainAxisAlignment: MainAxisAlignment.spaceBetween,
  children: [
    Text('左'),
    Text('中央'),
    Text('右'),
  ],
)

8.4 Stackウィジェット（重ね合わせ配置）

Stackは、子ウィジェットを重ねて配置するレイアウトウィジェットです。最初に書いたウィジェットが一番下（背景）、最後に書いたウィジェットが一番上（前面）に表示されます。

基本的なStack 
// 画像の上にテキストを重ねる
Stack(
  children: [
    Image.network('画像URL'),        // 背景
    Text('画像の上のテキスト'),       // 前面
  ],
)

Positionedウィジェット

Stack内の子ウィジェットをPositionedで囲むと、上下左右からの距離を指定して位置を制御できます。

Positionedで位置を指定 
// 右上にテキストを配置
Stack(
  children: [
    Container(color: Colors.blue, width: 200, height: 200),
    Positioned(
      top: 10,    // 上から10ピクセル
      right: 10,  // 右から10ピクセル
      child: Text('右上'),
    ),
  ],
)

8.5 実践例：レイアウトを組み合わせる

実際のアプリでは、Column、Row、Stackを組み合わせて複雑なレイアウトを作ります。

実習 8-1: プロフィール画面のレイアウト

Column、Row、Stackを組み合わせて、プロフィール画面のようなレイアウトを作成します。

プロフィール画面レイアウト 
// プロフィール画面のレイアウト例
Column(
  children: [
    // プロフィール画像と名前を重ねる
    Stack(
      children: [
        Container(
          height: 200,
          color: Colors.blue,
        ),
        Positioned(
          bottom: 10,
          left: 10,
          child: Text(
            '田中太郎',
            style: TextStyle(color: Colors.white, fontSize: 24),
          ),
        ),
      ],
    ),
    // 詳細情報を縦に並べる
    Padding(
      padding: EdgeInsets.all(16),
      child: Column(
        children: [
          Row(
            children: [
              Icon(Icons.email),
              SizedBox(width: 8),
              Text('taro@example.com'),
            ],
          ),
          SizedBox(height: 8),
          Row(
            children: [
              Icon(Icons.phone),
              SizedBox(width: 8),
              Text('090-1234-5678'),
            ],
          ),
        ],
      ),
    ),
  ],
)

Q1

子ウィジェットを縦方向に並べるレイアウトウィジェットはどれですか？

解答を見る
正解: B
Columnは子ウィジェットを縦方向（上から下へ）に並べます。Rowは横方向、Stackは重ねて配置します。
Q2

Columnで子ウィジェットを縦方向の中央に配置するには、どのプロパティを使いますか？

解答を見る
正解: B
Columnの主軸は縦方向なので、mainAxisAlignmentで縦方向の配置を制御します。MainAxisAlignment.centerを指定すると中央揃えになります。
Q3

Stack内で子ウィジェットの位置を細かく指定するために使うウィジェットは何ですか？

解答を見る
正解: A
Positionedウィジェットを使うと、上下左右からの距離を指定してStack内の子ウィジェットを配置できます。

第8章のまとめ

  • Columnは子ウィジェットを縦方向（上から下）に並べる
  • Rowは子ウィジェットを横方向（左から右）に並べる
  • Stackは子ウィジェットを重ねて配置する
  • mainAxisAlignmentで主軸方向の配置を制御（Column=縦、Row=横）
  • crossAxisAlignmentで交差軸方向の配置を制御
  • PositionedをStack内で使うと、上下左右からの距離を指定して配置できる
  • レイアウトウィジェットを組み合わせることで複雑なUIを構築できる

Material Designの活用

AppBar、Drawer、BottomNavigationBar、Cardなど、Material Designウィジェットを活用した美しいUI作成を学びます。

この章で学ぶこと

  • Material Designの基本概念と設計原則を理解する
  • AppBarを使った画面上部のナビゲーションバー作成
  • Drawerを使ったサイドメニューの実装
  • BottomNavigationBarを使った画面下部のタブナビゲーション
  • Cardウィジェットを活用した情報の整理表示

13.1 Material Designとは

Material Design（マテリアルデザイン）は、Googleが開発したデザインシステムです。物理的な素材（紙、インク）のメタファーを使い、一貫性のある美しいUIを作成するための設計原則とガイドラインを提供しています。

奥行きと階層

影（elevation）を使って要素の重なりを表現

鮮やかな色彩

プライマリカラー、セカンダリカラーの体系的な管理

滑らかなアニメーション

自然でわかりやすい動きと遷移

統一されたコンポーネント

ボタン、カード、ダイアログなど標準部品が充実

13.2 AppBarの使い方

AppBarは、アプリ画面上部に表示されるナビゲーションバーです。タイトル、アクションボタン、戻るボタンなどを配置できます。

基本的なAppBar

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        // AppBarを配置
        appBar: AppBar(
          title: Text('ホーム画面'),  // タイトル
          backgroundColor: Colors.blue,  // 背景色
        ),
        body: Center(
          child: Text('メインコンテンツ'),
        ),
      ),
    );
  }
}

アクションボタン付きAppBar

AppBar(
  title: Text('メッセージ'),
  backgroundColor: Colors.teal,
  // 左側のアイコン
  leading: IconButton(
    icon: Icon(Icons.menu),
    onPressed: () {
      print('メニューボタンが押されました');
    },
  ),
  // 右側のアクションボタン
  actions: [
    IconButton(
      icon: Icon(Icons.search),
      onPressed: () {
        print('検索ボタンが押されました');
      },
    ),
    IconButton(
      icon: Icon(Icons.more_vert),
      onPressed: () {
        print('その他ボタンが押されました');
      },
    ),
  ],
)

AppBarの主要プロパティ

  • title - タイトルテキスト
  • backgroundColor - 背景色
  • leading - 左側のウィジェット（通常はメニューボタン）
  • actions - 右側のアクションボタンリスト
  • elevation - 影の高さ（デフォルト4）

13.3 Drawerの使い方

Drawerは、画面の左端または右端から引き出せるサイドメニューです。設定、プロフィール、メニュー一覧などを配置するのに適しています。

Drawerの実装例

class MyHomePage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('ホーム'),
      ),
      // Drawerを配置
      drawer: Drawer(
        child: ListView(
          padding: EdgeInsets.zero,  // パディングをゼロに
          children: [
            // ヘッダー部分
            DrawerHeader(
              decoration: BoxDecoration(
                color: Colors.blue,
              ),
              child: Column(
                crossAxisAlignment: CrossAxisAlignment.start,
                children: [
                  CircleAvatar(
                    radius: 30,
                    child: Icon(Icons.person, size: 40),
                  ),
                  SizedBox(height: 10),
                  Text(
                    'ユーザー名',
                    style: TextStyle(color: Colors.white, fontSize: 18),
                  ),
                ],
              ),
            ),
            // メニュー項目
            ListTile(
              leading: Icon(Icons.home),
              title: Text('ホーム'),
              onTap: () {
                Navigator.pop(context);  // Drawerを閉じる
              },
            ),
            ListTile(
              leading: Icon(Icons.settings),
              title: Text('設定'),
              onTap: () {
                Navigator.pop(context);
                // 設定画面へ遷移する処理
              },
            ),
            ListTile(
              leading: Icon(Icons.logout),
              title: Text('ログアウト'),
              onTap: () {
                Navigator.pop(context);
                // ログアウト処理
              },
            ),
          ],
        ),
      ),
      body: Center(
        child: Text('メインコンテンツ'),
      ),
    );
  }
}

Drawerの開閉

AppBarの左端にあるメニューアイコンをタップするとDrawerが開きます。プログラムから開く場合はScaffold.of(context).openDrawer()を使用します。

13.4 BottomNavigationBarの使い方

BottomNavigationBarは、画面下部に表示されるタブナビゲーションです。主要な画面を素早く切り替えるのに適しています。

BottomNavigationBarの実装例

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  int _selectedIndex = 0;  // 選択されているタブのインデックス

  // 各タブに表示する画面
  final List<Widget> _pages = [
    Center(child: Text('ホーム画面')),
    Center(child: Text('検索画面')),
    Center(child: Text('お気に入り画面')),
  ];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('タブナビゲーション'),
      ),
      body: _pages[_selectedIndex],  // 選択されたページを表示
      bottomNavigationBar: BottomNavigationBar(
        currentIndex: _selectedIndex,  // 現在選択されているタブ
        onTap: (index) {
          setState(() {
            _selectedIndex = index;  // タブ選択を更新
          });
        },
        items: [
          BottomNavigationBarItem(
            icon: Icon(Icons.home),
            label: 'ホーム',
          ),
          BottomNavigationBarItem(
            icon: Icon(Icons.search),
            label: '検索',
          ),
          BottomNavigationBarItem(
            icon: Icon(Icons.favorite),
            label: 'お気に入り',
          ),
        ],
      ),
    );
  }
}

13.5 Cardウィジェットの活用

Cardは、関連する情報をまとめて表示するための角丸で影付きのコンテナです。リスト項目、プロフィールカード、商品カードなど幅広い用途に使えます。

Cardの基本例

Card(
  elevation: 4,  // 影の高さ
  margin: EdgeInsets.all(16),
  child: Padding(
    padding: EdgeInsets.all(16),
    child: Column(
      crossAxisAlignment: CrossAxisAlignment.start,
      children: [
        Text(
          '商品タイトル',
          style: TextStyle(fontSize: 20, fontWeight: FontWeight.bold),
        ),
        SizedBox(height: 8),
        Text('商品の説明文がここに入ります。'),
        SizedBox(height: 12),
        ElevatedButton(
          onPressed: () {},
          child: Text('詳細を見る'),
        ),
      ],
    ),
  ),
)

Cardのリスト表示例

ListView.builder(
  itemCount: 5,
  itemBuilder: (context, index) {
    return Card(
      margin: EdgeInsets.symmetric(horizontal: 16, vertical: 8),
      child: ListTile(
        leading: CircleAvatar(
          child: Text('${index + 1}'),
        ),
        title: Text('アイテム ${index + 1}'),
        subtitle: Text('サブタイトル'),
        trailing: Icon(Icons.arrow_forward_ios),
        onTap: () {
          print('アイテム ${index + 1} がタップされました');
        },
      ),
    );
  },
)

13.6 実習課題

実習: Material Designアプリの作成

AppBar、Drawer、BottomNavigationBar、Cardを組み合わせた本格的なアプリを作成しましょう。

  • AppBarにタイトルと検索アイコンを配置
  • Drawerにユーザー情報と3つのメニュー項目を配置

  • ホーム画面にCardを使ったリスト表示を実装

Q1

Material Designの「elevation」とは何ですか？

解答を見る

第13章のまとめ

  • Material Designは、Googleが提供する統一されたデザインシステム
  • AppBarは画面上部のナビゲーションバーで、タイトルやアクションボタンを配置
  • Drawerは左端から引き出せるサイドメニューで、設定やユーザー情報を配置
  • BottomNavigationBarは画面下部のタブナビゲーションで、主要画面の切り替えに使用
  • Cardは関連情報をまとめて表示する角丸・影付きのコンテナ
  • これらのMaterial Designウィジェットを組み合わせることで、統一感のある美しいUIを効率的に作成できる

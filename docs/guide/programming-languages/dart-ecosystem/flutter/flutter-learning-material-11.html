リスト表示とスクロール

ListView、GridViewを使ったリスト表示、スクロール可能なコンテンツの作成方法を学びます。

この章で学ぶこと

  • ListViewを使った基本的なリスト表示の方法
  • ListView.builderによる効率的な大量データの表示
  • GridViewを使ったグリッド形式の表示
  • スクロール可能なコンテンツの作成方法

11.1 リスト表示の基本概念

モバイルアプリでは、複数のアイテムをリスト形式で表示することが非常に多くあります。例えば、メールアプリのメール一覧、SNSのタイムライン、ECサイトの商品一覧などです。

Flutterでは、ListViewとGridViewという2つの主要なウィジェットを使ってリスト表示を実現します。

ListView

垂直方向に並んだ要素を表示するリスト

GridView

グリッド形式（格子状）で要素を表示するリスト

リスト表示の概念

graph LR A["リスト表示"] --> B["ListView
（縦並び）"] A --> C["GridView
（格子状）"] B --> B1["アイテム1"] B --> B2["アイテム2"] B --> B3["アイテム3"] C --> C1["アイテム1 | アイテム2"] C --> C2["アイテム3 | アイテム4"]

11.2 ListViewの基本

ListViewは、子要素を垂直方向にスクロール可能な形で表示するウィジェットです。画面に収まらない場合は自動的にスクロール可能になります。

基本的なListViewの使い方

ListViewの中に複数のウィジェット（ListTile等）を配置します。

lib/main.dart 
import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: SimpleListScreen(),
    );
  }
}

class SimpleListScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('シンプルなリスト')),
      body: ListView(
        children: [
          ListTile(
            leading: Icon(Icons.person),
            title: Text('山田太郎'),
            subtitle: Text('営業部'),
          ),
          ListTile(
            leading: Icon(Icons.person),
            title: Text('佐藤花子'),
            subtitle: Text('開発部'),
          ),
          ListTile(
            leading: Icon(Icons.person),
            title: Text('鈴木次郎'),
            subtitle: Text('総務部'),
          ),
        ],
      ),
    );
  }
}

ListTileの構造

ListTileは、リストアイテムを表示するための便利なウィジェットです。

  • leading - 左側に表示するアイコンや画像
  • title - メインのタイトルテキスト
  • subtitle - サブタイトルテキスト
  • trailing - 右側に表示するアイコンやボタン
  • onTap - タップ時の処理

11.3 ListView.builderによる効率的なリスト

大量のデータを表示する場合、通常のListViewでは全アイテムを一度に作成してしまい、パフォーマンスが悪化します。ListView.builderを使うと、画面に表示される分だけアイテムを生成するため、効率的です。

ListView.builderの利点

  • 効率的: 画面に表示される分だけウィジェットを生成（遅延読み込み）
  • メモリ節約: 数千件のデータでもメモリ消費が少ない
  • 高速: スクロールがスムーズ

ListView.builderの使用例

lib/main.dart 
import 'package:flutter/material.dart';

class BuilderListScreen extends StatelessWidget {
  // ダミーデータ（100件）
  final List<String> items = List.generate(100, (i) => 'アイテム ${i + 1}');

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('ListView.builder')),
      body: ListView.builder(
        itemCount: items.length, // アイテムの総数
        itemBuilder: (context, index) {
          // indexに対応するアイテムを生成
          return ListTile(
            leading: CircleAvatar(child: Text('${index + 1}')),
            title: Text(items[index]),
            trailing: Icon(Icons.chevron_right),
            onTap: () {
              print('タップ: ${items[index]}');
            },
          );
        },
      ),
    );
  }
}

itemBuilderパラメータ

itemBuilderは、インデックス番号（index）を受け取り、そのインデックスに対応するウィジェットを返す関数です。

  • context: BuildContext（通常そのまま使用）
  • index: 現在のアイテムのインデックス番号（0から始まる）

11.4 GridViewの基本

GridViewは、アイテムをグリッド（格子状）に表示するウィジェットです。写真ギャラリーや商品一覧など、複数列で表示したい場合に便利です。

GridView.countの使用例

GridView.countは、列数を指定してグリッドを作成します。

lib/main.dart 
import 'package:flutter/material.dart';

class SimpleGridScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('GridView.count')),
      body: GridView.count(
        crossAxisCount: 2, // 列数（横方向のアイテム数）
        padding: EdgeInsets.all(16),
        crossAxisSpacing: 10, // 横方向の間隔
        mainAxisSpacing: 10, // 縦方向の間隔
        children: [
          Container(
            color: Colors.red,
            child: Center(child: Text('1')),
          ),
          Container(
            color: Colors.blue,
            child: Center(child: Text('2')),
          ),
          Container(
            color: Colors.green,
            child: Center(child: Text('3')),
          ),
          Container(
            color: Colors.orange,
            child: Center(child: Text('4')),
          ),
        ],
      ),
    );
  }
}

GridView.builderの使用例

ListView.builderと同様、効率的に大量データを表示できます。

lib/main.dart 
import 'package:flutter/material.dart';

class BuilderGridScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('GridView.builder')),
      body: GridView.builder(
        gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
          crossAxisCount: 3, // 列数
          crossAxisSpacing: 8,
          mainAxisSpacing: 8,
        ),
        itemCount: 50, // アイテム総数
        itemBuilder: (context, index) {
          return Container(
            color: Colors.blue[100 * ((index % 9) + 1)],
            child: Center(
              child: Text('${index + 1}'),
            ),
          );
        },
      ),
    );
  }
}

11.5 スクロール制御

ListViewやGridViewのスクロール動作をカスタマイズできます。

スクロール方向の変更

scrollDirectionプロパティで、水平スクロールに変更できます。

水平スクロールの例 
ListView.builder(
  scrollDirection: Axis.horizontal, // 水平スクロール
  itemCount: 10,
  itemBuilder: (context, index) {
    return Container(
      width: 150,
      margin: EdgeInsets.all(8),
      color: Colors.blue,
      child: Center(child: Text('$index')),
    );
  },
)

スクロール挙動の制御

physicsプロパティでスクロールの物理挙動を変更できます。

  • BouncingScrollPhysics(): iOS風のバウンススクロール
  • ClampingScrollPhysics(): Android風の端でグロー表示
  • NeverScrollableScrollPhysics(): スクロール無効

実習: タスクリストアプリの作成

これまで学んだリスト表示の知識を使って、タスクリストアプリを作成してみましょう。

lib/main.dart 
import 'package:flutter/material.dart';

void main() => runApp(TaskApp());

class TaskApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(home: TaskListScreen());
  }
}

class TaskListScreen extends StatefulWidget {
  @override
  _TaskListScreenState createState() => _TaskListScreenState();
}

class _TaskListScreenState extends State<TaskListScreen> {
  final List<String> tasks = ['買い物', 'メールチェック', '掃除'];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('タスクリスト')),
      body: ListView.builder(
        itemCount: tasks.length,
        itemBuilder: (context, index) {
          return ListTile(
            leading: Icon(Icons.check_box_outline_blank),
            title: Text(tasks[index]),
            trailing: IconButton(
              icon: Icon(Icons.delete, color: Colors.red),
              onPressed: () {
                setState(() {
                  tasks.removeAt(index);
                });
              },
            ),
          );
        },
      ),
    );
  }
}

理解度チェッククイズ

Q1

大量データを効率的に表示するために使うべきウィジェットは？

解答を見る
正解: B
ListView.builderは画面に表示される分だけウィジェットを生成するため、大量データでも効率的に表示できます。
Q2

GridView.countのcrossAxisCountプロパティは何を指定しますか？

解答を見る
正解: B
crossAxisCountは列数（横方向のアイテム数）を指定します。例えば2を指定すると、2列のグリッドになります。
Q3

水平スクロールのリストを作成するために設定するプロパティは？

解答を見る
正解: B
scrollDirection: Axis.horizontalを指定すると、垂直スクロールではなく水平スクロールのリストになります。

第11章のまとめ

  • ListViewは垂直方向にスクロール可能なリストを表示するウィジェット
  • ListView.builderを使うと大量データを効率的に表示できる（遅延読み込み）
  • GridViewはグリッド形式でアイテムを表示するウィジェット
  • ListTileはリストアイテムを簡単に作成できる便利なウィジェット
  • scrollDirectionプロパティで水平スクロールに変更可能
  • physicsプロパティでスクロール挙動をカスタマイズできる

ユーザー入力とボタン

ボタン（ElevatedButton、TextButton等）の使い方、TextFieldによるテキスト入力、ユーザー操作への応答方法を学びます。

この章で学ぶこと

  • Flutterのボタンウィジェット（ElevatedButton、TextButton、OutlinedButton）の使い方
  • TextFieldを使ったテキスト入力の基本
  • TextEditingControllerを使った入力値の取得・制御
  • ユーザー操作への応答とイベント処理の基本

9.1 Flutterのボタンウィジェット

Flutterには複数のボタンウィジェットが用意されており、それぞれ異なる見た目と用途を持っています。Material Designのガイドラインに従ったボタンを簡単に実装できます。

ElevatedButton

立体的な影のあるボタン。主要なアクションに使用

TextButton

フラットなテキストボタン。副次的なアクションに使用

OutlinedButton

枠線のあるボタン。中間的な重要度のアクションに使用

ボタンウィジェットの種類

ElevatedButtonの基本

elevated_button_example.dart 
// ElevatedButton - 立体的な影のあるボタン
ElevatedButton(
  onPressed: () {
    // ボタンが押された時の処理
    print('ボタンが押されました');
  },
  child: Text('送信する'),
)

// スタイルをカスタマイズしたElevatedButton
ElevatedButton(
  onPressed: () {
    print('カスタムボタンが押されました');
  },
  style: ElevatedButton.styleFrom(
    backgroundColor: Colors.blue,  // 背景色
    foregroundColor: Colors.white, // テキスト色
    padding: EdgeInsets.symmetric(horizontal: 32, vertical: 16),
  ),
  child: Text('カスタムボタン'),
)

TextButtonとOutlinedButton

other_buttons_example.dart 
// TextButton - フラットなテキストボタン
TextButton(
  onPressed: () {
    print('テキストボタンが押されました');
  },
  child: Text('キャンセル'),
)

// OutlinedButton - 枠線のあるボタン
OutlinedButton(
  onPressed: () {
    print('アウトラインボタンが押されました');
  },
  child: Text('編集'),
)

// IconButton - アイコンのみのボタン
IconButton(
  icon: Icon(Icons.favorite),
  onPressed: () {
    print('お気に入りボタンが押されました');
  },
)

ボタンの無効化

onPressedにnullを設定すると、ボタンが無効化され、押せなくなります。これは入力が未完了の時などに便利です。

9.2 TextFieldによるテキスト入力

TextFieldは、ユーザーがテキストを入力するためのウィジェットです。メールアドレス、パスワード、検索キーワードなど、さまざまな入力フォームで使用されます。

TextFieldの基本

textfield_basic.dart 
// 基本的なTextField
TextField(
  decoration: InputDecoration(
    labelText: '名前',           // ラベル
    hintText: '山田太郎',        // プレースホルダー
    border: OutlineInput Border(), // 枠線
  ),
)

// パスワード入力用のTextField
TextField(
  obscureText: true,  // 入力文字を隠す
  decoration: InputDecoration(
    labelText: 'パスワード',
    border: OutlineInputBorder(),
    prefixIcon: Icon(Icons.lock),  // 先頭にアイコン
  ),
)

// 複数行入力のTextField
TextField(
  maxLines: 5,  // 最大行数
  decoration: InputDecoration(
    labelText: 'コメント',
    hintText: 'ご意見をお聞かせください',
    border: OutlineInputBorder(),
  ),
)

TextFieldの主な装飾オプション

  • 1 labelText: フィールド上部に表示されるラベル
  • 2 hintText: 入力前に表示されるヒントテキスト
  • 3 prefixIcon / suffixIcon: フィールドの前後に表示するアイコン
  • 4 border: 枠線のスタイル（OutlineInputBorder、UnderlineInputBorder等）
  • 5 errorText: エラーメッセージの表示

9.3 TextEditingControllerで入力値を取得

TextFieldに入力された値を取得するには、TextEditingControllerを使用します。これにより、入力値の取得、設定、クリアなどの操作が可能になります。

TextEditingControllerの基本

textfield_controller.dart 
import 'package:flutter/material.dart';

class InputExample extends StatefulWidget {
  @override
  _InputExampleState createState() => _InputExampleState();
}

class _InputExampleState extends State<InputExample> {
  // TextEditingControllerを作成
  final TextEditingController _controller = TextEditingController();

  @override
  void dispose() {
    // 使用後は必ずdisposeする
    _controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        // TextFieldにcontrollerを設定
        TextField(
          controller: _controller,
          decoration: InputDecoration(
            labelText: '名前を入力',
            border: OutlineInputBorder(),
          ),
        ),
        SizedBox(height: 16),
        ElevatedButton(
          onPressed: () {
            // 入力値を取得
            String inputText = _controller.text;
            print('入力された値: $inputText');
          },
          child: Text('入力内容を表示'),
        ),
      ],
    );
  }
}

TextEditingControllerの主なメソッド

  • .text : 現在の入力値を取得
  • .text = '値' : 入力値を設定
  • .clear() : 入力値をクリア
  • .dispose() : リソースを解放（必須）

disposeを忘れないこと

TextEditingControllerは使用後に必ずdispose()メソッドを呼び出してリソースを解放する必要があります。これを忘れるとメモリリークの原因になります。

9.4 実習 - シンプルな入力フォームの作成

実習 9-1: 名前とメールアドレスの入力フォーム

ユーザーが名前とメールアドレスを入力し、送信ボタンを押すと入力内容がコンソールに表示されるアプリを作成します。

手順

  1. 1 新しいFlutterプロジェクトを作成するか、既存のプロジェクトを開く
  2. 2 名前用とメールアドレス用の2つのTextEditingControllerを作成
  3. 3 TextFieldを2つ配置し、それぞれにcontrollerを設定
  4. 4 送信ボタンを配置し、押された時に入力値をコンソールに出力
  5. 5 アプリを実行して動作確認

サンプルコード

main.dart 
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: '入力フォーム',
      home: InputFormPage(),
    );
  }
}

class InputFormPage extends StatefulWidget {
  @override
  _InputFormPageState createState() => _InputFormPageState();
}

class _InputFormPageState extends State<InputFormPage> {
  // 名前とメールアドレス用のController
  final TextEditingController _nameController = TextEditingController();
  final TextEditingController _emailController = TextEditingController();

  @override
  void dispose() {
    // リソースを解放
    _nameController.dispose();
    _emailController.dispose();
    super.dispose();
  }

  void _submitForm() {
    // 入力値を取得
    String name = _nameController.text;
    String email = _emailController.text;

    // コンソールに出力
    print('名前: $name');
    print('メールアドレス: $email');
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('入力フォーム'),
      ),
      body: Padding(
        padding: EdgeInsets.all(16.0),
        child: Column(
          children: [
            // 名前入力フィールド
            TextField(
              controller: _nameController,
              decoration: InputDecoration(
                labelText: '名前',
                hintText: '山田太郎',
                border: OutlineInputBorder(),
                prefixIcon: Icon(Icons.person),
              ),
            ),
            SizedBox(height: 16),
            // メールアドレス入力フィールド
            TextField(
              controller: _emailController,
              decoration: InputDecoration(
                labelText: 'メールアドレス',
                hintText: 'example@example.com',
                border: OutlineInputBorder(),
                prefixIcon: Icon(Icons.email),
              ),
            ),
            SizedBox(height: 24),
            // 送信ボタン
            ElevatedButton(
              onPressed: _submitForm,
              child: Text('送信'),
              style: ElevatedButton.styleFrom(
                padding: EdgeInsets.symmetric(horizontal: 32, vertical: 16),
              ),
            ),
          ],
        ),
      ),
    );
  }
}

動作確認

アプリを実行し、名前とメールアドレスを入力して送信ボタンを押してください。

期待される結果 

コンソール出力:
名前: 山田太郎
メールアドレス: yamada@example.com

9.5 キーボードタイプと入力制限

TextFieldでは、入力内容に応じて適切なキーボードタイプを指定したり、入力可能な文字を制限したりすることができます。

キーボードタイプの指定

keyboard_types.dart 
// メールアドレス入力用キーボード
TextField(
  keyboardType: TextInputType.emailAddress,
  decoration: InputDecoration(labelText: 'メールアドレス'),
)

// 数字入力用キーボード
TextField(
  keyboardType: TextInputType.number,
  decoration: InputDecoration(labelText: '電話番号'),
)

// URL入力用キーボード
TextField(
  keyboardType: TextInputType.url,
  decoration: InputDecoration(labelText: 'ウェブサイト'),
)

// 複数行テキスト入力
TextField(
  keyboardType: TextInputType.multiline,
  maxLines: 5,
  decoration: InputDecoration(labelText: 'コメント'),
)

主なTextInputType

  • TextInputType.text : 通常のテキスト入力（デフォルト）
  • TextInputType.number : 数字キーボード
  • TextInputType.emailAddress : メールアドレス入力用キーボード
  • TextInputType.phone : 電話番号入力用キーボード
  • TextInputType.url : URL入力用キーボード
  • TextInputType.multiline : 複数行テキスト入力

入力アクション（Enterキーの動作）

text_input_action.dart 
// 次のフィールドへ移動
TextField(
  textInputAction: TextInputAction.next,
  decoration: InputDecoration(labelText: '名前'),
)

// 完了（送信）
TextField(
  textInputAction: TextInputAction.done,
  onSubmitted: (value) {
    // Enterキーが押された時の処理
    print('入力完了: $value');
  },
  decoration: InputDecoration(labelText: 'メールアドレス'),
)

解答を見る

TextFieldの入力値を取得するために使用するクラスは何ですか？

解答を見る
正解: B

Q3

ボタンを無効化（押せなくする）するにはどうすればよいですか？

解答を見る

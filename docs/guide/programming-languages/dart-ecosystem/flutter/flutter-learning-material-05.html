Dart言語の基礎（後編）

条件分岐（if文）、繰り返し（for文、while文）、リスト、マップなど、プログラムの流れを制御する構文を学びます。

この章で学ぶこと

  • リスト（配列）の作成と操作方法を学ぶ
  • マップ（辞書）を使ったキーと値の管理を理解する

5.1 条件分岐（if文）

条件分岐は、「もし〜ならば」という条件に応じて処理を変える仕組みです。プログラムの中で最も基本的な制御構造の1つです。

if文の基本構文

条件式がtrueの場合のみ、中括弧{}内のコードが実行されます。

Dart 
void main() {
  int age = 18;

  // 単純なif文
  if (age >= 18) {
    print('成人です');
  }

  // if-else文
  if (age >= 20) {
    print('お酒が飲めます');
  } else {
    print('まだお酒は飲めません');
  }
}

if-else ifによる複数条件の分岐

複数の条件を順番にチェックする場合は、else ifを使います。

Dart 
void main() {
  int score = 75;

  if (score >= 90) {
    print('評価: A');
  } else if (score >= 80) {
    print('評価: B');
  } else if (score >= 70) {
    print('評価: C');
  } else {
    print('評価: D');
  }
}

よく使う比較演算子

演算子 意味 例
== 等しい age == 20
!= 等しくない age != 20
> より大きい age > 20
< より小さい age < 20
>= 以上 age >= 20
<= 以下 age <= 20

論理演算子で複数条件を組み合わせる

複数の条件を同時にチェックしたい場合は、論理演算子を使います。

Dart 
void main() {
  int age = 25;
  bool hasLicense = true;

  // AND演算子（両方trueの場合のみtrue）
  if (age >= 18 && hasLicense) {
    print('運転できます');
  }

  // OR演算子（どちらかがtrueならtrue）
  if (age < 18 || !hasLicense) {
    print('運転できません');
  }
}

5.2 switch文による分岐

switch文は、1つの値に対して複数のケースを判定する場合に便利です。if-else ifを連続して書くよりも読みやすくなります。

Dart 
void main() {
  String grade = 'B';

  switch (grade) {
    case 'A':
      print('優秀です');
      break;
    case 'B':
      print('良好です');
      break;
    case 'C':
      print('普通です');
      break;
    default:
      print('評価不明');
  }
}

注意: breakを忘れずに

各caseの最後にはbreak;を書きましょう。breakがないと次のcaseも実行されてしまいます。

5.3 繰り返し処理（for文）

for文は、決まった回数だけ処理を繰り返す場合に使います。「1から10まで数える」といった処理に最適です。

基本的なfor文

Dart 
void main() {
  // 1から5まで表示
  for (int i = 1; i <= 5; i++) {
    print('カウント: $i');
  }
}

for文の構造

graph LR A["for (初期化; 条件; 更新)"] --> B["初期化: int i = 1"] A --> C["条件: i <= 5"] A --> D["更新: i++"] B --> E["最初に1回だけ実行"] C --> F["毎回チェック"] D --> G["1ループ終了ごとに実行"]

for-in文（コレクション用）

リストの要素を順番に取り出す場合は、for-in文が便利です。

Dart 
void main() {
  List fruits = ['りんご', 'バナナ', 'オレンジ'];

  for (String fruit in fruits) {
    print('好きな果物: $fruit');
  }
}

5.4 while文とdo-while文

while文は、条件がtrueの間だけ処理を繰り返します。繰り返し回数が事前にわからない場合に便利です。

while文

Dart 
void main() {
  int count = 1;

  while (count <= 5) {
    print('カウント: $count');
    count++; // カウントを増やす
  }
}

do-while文

do-while文は、最初に必ず1回処理を実行してから条件をチェックします。

Dart 
void main() {
  int count = 1;

  do {
    print('カウント: $count');
    count++;
  } while (count <= 5);
}

whileとdo-whileの違い

while文

条件を先にチェック → 条件がfalseなら1回も実行されない

do-while文

処理を先に実行 → 条件が最初からfalseでも1回は実行される

5.5 リスト（配列）の操作

リスト（List）は、複数のデータを順番に格納するデータ構造です。配列とも呼ばれます。

リストの作成と基本操作

Dart 
void main() {
  // リストの作成
  List numbers = [1, 2, 3, 4, 5];

  // 要素の取得（インデックスは0から始まる）
  print(numbers[0]); // 1
  print(numbers[2]); // 3

  // 要素の追加
  numbers.add(6);
  print(numbers); // [1, 2, 3, 4, 5, 6]

  // リストの長さ
  print('要素数: ${numbers.length}');
}

リストの便利なメソッド

メソッド 説明 例
add() 末尾に要素を追加 list.add(10)
remove() 指定要素を削除 list.remove(5)
contains() 要素が含まれるか確認 list.contains(3)
clear() 全要素を削除 list.clear()
length 要素数を取得 list.length
Dart 
void main() {
  List tasks = ['買い物', '掃除', '洗濯'];

  // 要素を追加
  tasks.add('料理');

  // 全要素を表示
  for (String task in tasks) {
    print('タスク: $task');
  }

  // 特定の要素が含まれるか確認
  if (tasks.contains('買い物')) {
    print('買い物が含まれています');
  }
}

5.6 マップ（辞書）の操作

マップ（Map）は、キーと値のペアでデータを管理するデータ構造です。辞書（Dictionary）とも呼ばれます。

マップの作成と基本操作

Dart 
void main() {
  // マップの作成
  Map scores = {
    'Alice': 85,
    'Bob': 92,
    'Charlie': 78,
  };

  // 値の取得
  print(scores['Alice']); // 85

  // 値の追加・更新
  scores['David'] = 88;
  scores['Alice'] = 90; // 更新

  print(scores);
}

マップの便利な操作

Dart 
void main() {
  Map capitals = {
    '日本': '東京',
    'アメリカ': 'ワシントンD.C.',
    'フランス': 'パリ',
  };

  // 全てのキーを取得
  print(capitals.keys);

  // 全ての値を取得
  print(capitals.values);

  // キーの存在確認
  if (capitals.containsKey('日本')) {
    print('日本の首都: ${capitals['日本']}');
  }
}

マップをループで処理する

Dart 
void main() {
  Map ages = {
    'Alice': 25,
    'Bob': 30,
    'Charlie': 22,
  };

  // forEach()メソッドで全要素を処理
  ages.forEach((name, age) {
    print('$name は $age 歳です');
  });
}

int x = 10;
if (x > 5) {

} else {

解答を見る
正解: A

正解: B

第5章のまとめ

  • if文で条件分岐、switch文で複数ケースの判定ができる
  • for文は決まった回数の繰り返し、while文は条件が満たされる間の繰り返し
  • リスト（List）は順序付きのデータ集合で、インデックスは0から始まる
  • マップ（Map）はキーと値のペアでデータを管理する辞書型データ構造
  • 論理演算子（&&、||、!）で複数条件を組み合わせられる
  • これらの制御構文とデータ構造はFlutterアプリ開発の基礎となる重要な要素

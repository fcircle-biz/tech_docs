第3章

データの取得(SELECT文)

SELECT文を使用してデータベースからデータを取得する方法を学びます。ResultSetオブジェクトを使ったデータの読み取り、様々なデータ型の取得方法、条件付きクエリの実行方法を習得します。

この章で学ぶこと

  • SELECT文を使用したデータベースからのデータ取得方法
  • ResultSetオブジェクトを使ったデータの読み取りと操作
  • 様々なデータ型（文字列、数値、日付）の取得方法
  • WHERE句を使った条件付きデータ取得の実装

3.1 SELECT文の基本

SELECT文は、データベースからデータを取得するための最も基本的なSQL文です。JDBCでSELECT文を実行するには、前章で学んだStatementオブジェクトを使用します。取得したデータはResultSetオブジェクトに格納され、このオブジェクトを通じてデータにアクセスできます。

SELECT文の実行フロー

1. Connectionオブジェクトを取得
2. StatementオブジェクトをConnectionから作成
3. executeQuery()メソッドでSELECT文を実行
4. ResultSetオブジェクトで結果を受け取る
5. ResultSetから各行のデータを順に読み取る 

sequenceDiagram
    participant Java as Javaプログラム
    participant Conn as Connection
    participant Stmt as Statement
    participant DB as データベース
    participant RS as ResultSet

    Java->>Conn: createStatement()
    Conn->>Stmt: Statementオブジェクト返却
    Java->>Stmt: executeQuery(SQL文)
    Stmt->>DB: SELECT文実行
    DB->>RS: 検索結果を格納
    RS->>Java: ResultSetオブジェクト返却
    loop 各行の処理
        Java->>RS: next()で次の行へ移動
        Java->>RS: getXxx()でデータ取得
    end
BasicSelectExample.java 
import java.sql.*;

public class BasicSelectExample {
    public static void main(String[] args) {
        // データベース接続情報
        String url = "jdbc:postgresql://localhost:5432/mydb";
        String user = "postgres";
        String password = "password";

        // データベース接続と検索処理
        try (Connection conn = DriverManager.getConnection(url, user, password);
             Statement stmt = conn.createStatement()) {

            // SELECT文を実行してResultSetを取得
            String sql = "SELECT id, name, email FROM users";
            ResultSet rs = stmt.executeQuery(sql);

            // 結果セットから1行ずつデータを取得
            while (rs.next()) {
                // カラム名を指定してデータを取得
                int id = rs.getInt("id");
                String name = rs.getString("name");
                String email = rs.getString("email");

                // 取得したデータを表示
                System.out.println("ID: " + id +
                                 ", 名前: " + name +
                                 ", Email: " + email);
            }

            rs.close(); // ResultSetをクローズ

        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}

ポイント

  • executeQuery(): SELECT文を実行する際に使用するメソッド
  • ResultSet: 検索結果を保持するオブジェクト
  • next(): ResultSet内のカーソルを次の行に移動させる（最初はデータの前にいるため必ず呼び出す）
  • getXxx(): 各カラムのデータを取得するメソッド（Xxxはデータ型に対応）

3.2 ResultSetからのデータ取得

ResultSetオブジェクトは、データベースから取得した結果セットを表します。カーソルという概念を使って、1行ずつデータにアクセスします。カーソルは最初はデータの前にあり、next()メソッドで次の行に移動します。next()メソッドは、次の行が存在すればtrueを、存在しなければfalseを返します。

データ型別の取得メソッド

データ型 取得メソッド 説明
整数 getInt() INT型のデータを取得
長整数 getLong() BIGINT型のデータを取得
文字列 getString() VARCHAR, TEXT型のデータを取得
浮動小数点 getDouble() DOUBLE型のデータを取得
真偽値 getBoolean() BOOLEAN型のデータを取得
日付 getDate() DATE型のデータを取得
日時 getTimestamp() TIMESTAMP型のデータを取得

カラムの指定方法

ResultSetからデータを取得する際、カラムを指定する方法は2つあります。

ColumnSpecification.java 
// 方法1: カラム名で指定（推奨）
String name = rs.getString("name");
int age = rs.getInt("age");

// 方法2: カラム番号で指定（1から始まる）
String name = rs.getString(1);  // 1番目のカラム
int age = rs.getInt(2);         // 2番目のカラム

推奨事項

カラム名での指定を推奨します。カラム番号での指定は、テーブル構造の変更に弱く、コードの可読性も低くなります。カラム名を使用することで、コードの保守性と理解しやすさが向上します。

DataTypeExample.java 
import java.sql.*;

public class DataTypeExample {
    public static void main(String[] args) {
        String url = "jdbc:postgresql://localhost:5432/mydb";
        String sql = "SELECT id, name, price, available, created_at FROM products";

        try (Connection conn = DriverManager.getConnection(url, "user", "pass");
             Statement stmt = conn.createStatement();
             ResultSet rs = stmt.executeQuery(sql)) {

            while (rs.next()) {
                // 各データ型の取得例
                int id = rs.getInt("id");                    // 整数型
                String name = rs.getString("name");          // 文字列型
                double price = rs.getDouble("price");        // 浮動小数点型
                boolean available = rs.getBoolean("available"); // 真偽値型
                Timestamp createdAt = rs.getTimestamp("created_at"); // 日時型

                System.out.printf("ID: %d, 商品名: %s, 価格: %.2f円, " +
                                "在庫あり: %s, 登録日: %s%n",
                                id, name, price, available, createdAt);
            }

        } catch (SQLException e) {
            System.err.println("データ取得エラー: " + e.getMessage());
        }
    }
}

3.3 条件付きデータ取得（WHERE句）

実際のアプリケーションでは、特定の条件に合致するデータのみを取得することがほとんどです。SQL文にWHERE句を追加することで、条件に合うデータだけを取得できます。

基本的なWHERE句の使用

WhereClauseExample.java 
import java.sql.*;

public class WhereClauseExample {
    public static void main(String[] args) {
        String url = "jdbc:postgresql://localhost:5432/mydb";

        try (Connection conn = DriverManager.getConnection(url, "user", "pass");
             Statement stmt = conn.createStatement()) {

            // 例1: 特定の値と一致するデータを取得
            String sql1 = "SELECT * FROM users WHERE age = 25";
            ResultSet rs1 = stmt.executeQuery(sql1);

            System.out.println("=== 年齢が25歳のユーザー ===");
            while (rs1.next()) {
                System.out.println("名前: " + rs1.getString("name") +
                                 ", 年齢: " + rs1.getInt("age"));
            }
            rs1.close();

            // 例2: 範囲指定でデータを取得
            String sql2 = "SELECT * FROM users WHERE age >= 20 AND age <= 30";
            ResultSet rs2 = stmt.executeQuery(sql2);

            System.out.println("\n=== 20歳以上30歳以下のユーザー ===");
            while (rs2.next()) {
                System.out.println("名前: " + rs2.getString("name") +
                                 ", 年齢: " + rs2.getInt("age"));
            }
            rs2.close();

            // 例3: 部分一致検索（LIKE句）
            String sql3 = "SELECT * FROM users WHERE name LIKE '%田%'";
            ResultSet rs3 = stmt.executeQuery(sql3);

            System.out.println("\n=== 名前に「田」が含まれるユーザー ===");
            while (rs3.next()) {
                System.out.println("名前: " + rs3.getString("name"));
            }
            rs3.close();

        } catch (SQLException e) {
            System.err.println("データベースエラー: " + e.getMessage());
        }
    }
}

並び替え（ORDER BY句）

取得したデータを特定の順序で並び替えるには、ORDER BY句を使用します。

OrderByExample.java 
// 昇順で並び替え（ASC: Ascending = 昇順、デフォルト）
String sql1 = "SELECT * FROM products ORDER BY price ASC";

// 降順で並び替え（DESC: Descending = 降順）
String sql2 = "SELECT * FROM products ORDER BY price DESC";

// 複数カラムで並び替え
String sql3 = "SELECT * FROM products ORDER BY category ASC, price DESC";

集計関数の使用

SQLの集計関数を使用して、データの合計、平均、最大値、最小値、件数などを取得できます。

AggregateFunctionExample.java 
import java.sql.*;

public class AggregateFunctionExample {
    public static void main(String[] args) {
        String url = "jdbc:postgresql://localhost:5432/mydb";

        try (Connection conn = DriverManager.getConnection(url, "user", "pass");
             Statement stmt = conn.createStatement()) {

            // 集計関数を使ったSQL文
            String sql = "SELECT COUNT(*) as user_count, " +
                        "AVG(age) as avg_age, " +
                        "MAX(age) as max_age, " +
                        "MIN(age) as min_age " +
                        "FROM users";

            ResultSet rs = stmt.executeQuery(sql);

            if (rs.next()) {
                int count = rs.getInt("user_count");      // 件数
                double avgAge = rs.getDouble("avg_age");  // 平均
                int maxAge = rs.getInt("max_age");        // 最大値
                int minAge = rs.getInt("min_age");        // 最小値

                System.out.println("=== ユーザー統計情報 ===");
                System.out.println("総ユーザー数: " + count + "人");
                System.out.printf("平均年齢: %.1f歳%n", avgAge);
                System.out.println("最高年齢: " + maxAge + "歳");
                System.out.println("最低年齢: " + minAge + "歳");
            }

            rs.close();

        } catch (SQLException e) {
            System.err.println("データベースエラー: " + e.getMessage());
        }
    }
}

主な集計関数

  • COUNT(*) - 行数をカウント
  • SUM(カラム名) - 合計を計算
  • AVG(カラム名) - 平均を計算
  • MAX(カラム名) - 最大値を取得
  • MIN(カラム名) - 最小値を取得

実習 3-1: 書籍検索プログラムの作成

書籍データベースから特定の条件に合う書籍を検索し、結果を表示するプログラムを作成してみましょう。

前提条件

以下のテーブルが存在することを想定します： 

CREATE TABLE books (
    id SERIAL PRIMARY KEY,
    title VARCHAR(200) NOT NULL,
    author VARCHAR(100) NOT NULL,
    price INTEGER NOT NULL,
    published_year INTEGER NOT NULL
);

課題

  1. 1 2020年以降に出版された書籍を検索する
  2. 2 価格が2000円以下の書籍を価格の安い順に表示する
  3. 3 著者名に「山田」が含まれる書籍を検索する

実装例

import java.sql.*;

public class BookSearchProgram {
    private static final String URL = "jdbc:postgresql://localhost:5432/bookstore";
    private static final String USER = "postgres";
    private static final String PASS = "password";

    public static void main(String[] args) {
        // 課題1: 2020年以降の書籍を検索
        searchRecentBooks(2020);

        // 課題2: 2000円以下の書籍を安い順に表示
        searchCheapBooks(2000);

        // 課題3: 著者名に「山田」が含まれる書籍を検索
        searchBooksByAuthor("山田");
    }

    // 2020年以降の書籍を検索
    private static void searchRecentBooks(int year) {
        String sql = "SELECT * FROM books WHERE published_year >= " + year;

        try (Connection conn = DriverManager.getConnection(URL, USER, PASS);
             Statement stmt = conn.createStatement();
             ResultSet rs = stmt.executeQuery(sql)) {

            System.out.println("=== " + year + "年以降に出版された書籍 ===");
            while (rs.next()) {
                displayBook(rs);
            }

        } catch (SQLException e) {
            e.printStackTrace();
        }
    }

    // 指定価格以下の書籍を安い順に検索
    private static void searchCheapBooks(int maxPrice) {
        String sql = "SELECT * FROM books WHERE price <= " + maxPrice +
                    " ORDER BY price ASC";

        try (Connection conn = DriverManager.getConnection(URL, USER, PASS);
             Statement stmt = conn.createStatement();
             ResultSet rs = stmt.executeQuery(sql)) {

            System.out.println("\n=== " + maxPrice + "円以下の書籍（安い順） ===");
            while (rs.next()) {
                displayBook(rs);
            }

        } catch (SQLException e) {
            e.printStackTrace();
        }
    }

    // 著者名で書籍を検索
    private static void searchBooksByAuthor(String authorName) {
        String sql = "SELECT * FROM books WHERE author LIKE '%" + authorName + "%'";

        try (Connection conn = DriverManager.getConnection(URL, USER, PASS);
             Statement stmt = conn.createStatement();
             ResultSet rs = stmt.executeQuery(sql)) {

            System.out.println("\n=== 著者「" + authorName + "」の書籍 ===");
            while (rs.next()) {
                displayBook(rs);
            }

        } catch (SQLException e) {
            e.printStackTrace();
        }
    }

    // 書籍情報を表示する共通メソッド
    private static void displayBook(ResultSet rs) throws SQLException {
        System.out.printf("[%d] %s / %s (%d年) - %d円%n",
            rs.getInt("id"),
            rs.getString("title"),
            rs.getString("author"),
            rs.getInt("published_year"),
            rs.getInt("price")
        );
    }
}

理解度確認クイズ

  1. Q1. ResultSetオブジェクトのnext()メソッドの役割は何ですか？

    回答を表示

    next()メソッドは、ResultSet内のカーソルを次の行に移動させる役割を持ちます。次の行が存在すればtrueを、存在しなければfalseを返します。ResultSetは最初データの前にカーソルがあるため、データにアクセスする前に必ずnext()を呼び出す必要があります。通常はwhile文と組み合わせて使用します。

  2. Q2. ResultSetからデータを取得する際、カラム名指定とカラム番号指定のどちらが推奨されますか？その理由も説明してください。

    回答を表示

    カラム名での指定が推奨されます。
    理由：
    ① コードの可読性が高く、どのカラムのデータを取得しているか明確
    ② テーブル構造の変更（カラムの追加や順序変更）に強い
    ③ メンテナンス性が向上する
    カラム番号指定は、SELECT文のカラム順序に依存するため、変更に弱く、コードの保守性が低下します。

  3. Q3. SELECT文を実行する際に使用するStatementメソッドは何ですか？また、その戻り値の型は何ですか？

    回答を表示

    executeQuery()メソッドを使用します。
    戻り値の型はResultSetです。
    使用例：ResultSet rs = stmt.executeQuery(sql);
    このメソッドはSELECT文専用で、取得した結果セットをResultSetオブジェクトとして返します。

  4. Q4. SQLのWHERE句で部分一致検索を行う場合、どの演算子を使用しますか？また、ワイルドカードの記号は何ですか？

    回答を表示

    LIKE演算子を使用します。
    ワイルドカードは%（パーセント）を使用します。
    使用例：
    ・WHERE name LIKE '%田%' - 「田」を含む
    ・WHERE name LIKE '田%' - 「田」で始まる
    ・WHERE name LIKE '%田' - 「田」で終わる

  5. Q5. 次のコードの問題点を指摘してください。

    ResultSet rs = stmt.executeQuery("SELECT * FROM users");
    String name = rs.getString("name");
    System.out.println(name);
    回答を表示

    問題点：next()メソッドを呼び出していない
    ResultSetは最初、データの前にカーソルがあります。データにアクセスする前に必ずnext()を呼び出してカーソルを最初の行に移動させる必要があります。

    修正例：
    ResultSet rs = stmt.executeQuery("SELECT * FROM users");
    if (rs.next()) { // または while (rs.next())
        String name = rs.getString("name");
        System.out.println(name);
    }

この章のまとめ

  • SELECT文の実行にはexecuteQuery()メソッドを使用し、結果はResultSetオブジェクトに格納される
  • ResultSetのnext()メソッドでカーソルを次の行に移動し、データにアクセスする
  • getXxx()メソッド（getString, getInt, getDouble等）を使って、カラムのデータを取得する
  • カラムの指定はカラム名での指定が推奨される（可読性と保守性の向上）
  • WHERE句を使用して条件に合致するデータのみを取得できる
  • ORDER BY句でデータの並び順を制御できる（ASC: 昇順、DESC: 降順）
  • 集計関数（COUNT, SUM, AVG, MAX, MIN）を使用してデータの統計情報を取得できる

データを扱う - 変数と型

プログラムでデータを保存・利用するための「変数」と、データの種類を表す「型」について学びましょう。

この章で学ぶこと

  • 変数とは何か、なぜ必要なのかを理解する
  • データの種類を表す「型」の概念を学ぶ
  • 計算や文字列操作の基本的な演算子を使えるようになる

2.1 変数って何だろう？

プログラムでは、数値や文字などのデータを一時的に保存して、後から使いたいことがよくあります。 そんなとき使うのが変数（variable：バリアブル）です。

変数とは？

データを入れておく「名前付きの箱」のようなものです。 箱に名前をつけておくと、「あの箱に入れたデータを見せて」と言えば、 コンピュータが中身を取り出してくれます。

たとえ話: ラベル付きの箱

📦

「年齢」という箱

25

📦

「名前」という箱

"田中"

箱にラベル（変数名）を貼っておくと、「年齢の箱を見せて」と言えばすぐ見つかります。

変数を使うメリット

  • データに名前をつけられる - 何のデータか分かりやすくなる
  • 何度も使い回せる - 同じデータを複数の場所で使える
  • 値を後から変えられる - 計算結果を更新したりできる

2.2 変数を作ってみよう

Javaで変数を作るには、宣言という作業が必要です。 「こういう種類のデータを入れる、こういう名前の箱を作ります」とコンピュータに伝えるのです。

変数の宣言と代入 
// 変数の宣言（箱を用意）
int age;

// 変数に値を代入（箱にデータを入れる）
age = 25;

// 宣言と代入を同時に行う（よく使う書き方）
int score = 100;

変数名のルール

  • ・英字、数字、アンダースコア(_)、ドル記号($)が使える
  • ・数字から始めることはできない（例: 1name は NG）
  • ・予約語（intやclassなど）は使えない
  • ・大文字と小文字は区別される（age と Age は別物）

2.3 データ型を知ろう

Javaでは、変数に入れるデータの種類（型：type）を 必ず指定する必要があります。これは「この箱には整数しか入れられません」のように、 データの形式を決めておくことで、間違いを防ぐための仕組みです。

基本データ型（プリミティブ型）

Javaには8種類の基本データ型がありますが、まずは最もよく使う4つを覚えましょう。

型名 扱うデータ 例 覚え方
int 整数 100, -50, 0 integer（整数）の略
double 小数 3.14, -0.5 倍精度浮動小数点数
boolean 真偽値 true, false Yes/Noの二択
char 1文字 'A', 'あ' character（文字）の略

参照型：String（文字列）

複数の文字をまとめて扱いたいとき（名前や住所など）は、 String（ストリング）型を使います。

様々なデータ型の変数 
// 整数型
int age = 25;

// 小数型
double height = 170.5;

// 真偽値型
boolean isStudent = true;

// 文字列型（ダブルクォーテーションで囲む）
String name = "田中太郎";

charとStringの違い

  • ・charは1文字だけ。シングルクォート（'A'）で囲む
  • ・Stringは複数文字OK。ダブルクォート（"ABC"）で囲む

2.4 計算してみよう - 演算子

変数に入れたデータを使って計算することができます。 計算に使う記号を演算子（operator）といいます。

算術演算子

演算子 意味 例 結果
+ 足し算 10 + 3 13
- 引き算 10 - 3 7
* 掛け算 10 * 3 30
/ 割り算 10 / 3 3（整数同士は整数結果）
% 余り（剰余） 10 % 3 1
Calculation.java 
public class Calculation {
    public static void main(String[] args) {
        int a = 10;
        int b = 3;

        System.out.println("足し算: " + (a + b));  // 13
        System.out.println("引き算: " + (a - b));  // 7
        System.out.println("掛け算: " + (a * b));  // 30
        System.out.println("割り算: " + (a / b));  // 3
        System.out.println("余り: " + (a % b));    // 1
    }
}

文字列の連結

+演算子は、文字列同士をつなげる（連結する）こともできます。

StringConcat.java 
String firstName = "太郎";
String lastName = "田中";

// 文字列の連結
String fullName = lastName + firstName;
System.out.println(fullName);  // 田中太郎

// 文字列と数値の連結
int age = 25;
System.out.println("年齢は" + age + "歳です");  // 年齢は25歳です

実習 2-1: 自己紹介プログラムを作ろう

変数を使って、自分の情報を表示するプログラムを作成しましょう。

手順

  1. 1 VSCodeで「SelfIntro.java」という新しいファイルを作成
  2. 2 以下のコードを参考に、自分の情報に書き換えて入力
SelfIntro.java 
public class SelfIntro {
    public static void main(String[] args) {
        // 自分の情報を変数に格納
        String name = "山田花子";
        int age = 20;
        double height = 160.5;
        boolean isStudent = true;

        // 情報を表示
        System.out.println("=== 自己紹介 ===");
        System.out.println("名前: " + name);
        System.out.println("年齢: " + age + "歳");
        System.out.println("身長: " + height + "cm");
        System.out.println("学生ですか: " + isStudent);
    }
}

期待される結果

=== 自己紹介 ===
名前: 山田花子
年齢: 20歳
身長: 160.5cm
学生ですか: true

理解度確認クイズ

  1. Q1. 整数を扱うデータ型は何ですか？

    回答を表示

    int型です。 integer（整数）の略です。

  2. Q2. 10 % 3 の結果は何ですか？

    回答を表示

    1です。 %は割り算の余りを求める演算子で、10÷3=3余り1なので1が答えです。

  3. Q3. 文字列を扱う型は何ですか？また、文字列は何で囲みますか？

    回答を表示

    String型です。 文字列はダブルクォーテーション（"）で囲みます。 例: "Hello"

2.5 この章のまとめ

  • 変数はデータを入れる「名前付きの箱」。後から使い回せる。
  • データ型は変数に入れるデータの種類を決める。int、double、boolean、Stringなどがある。
  • 変数を作るには「型 変数名 = 値;」の形式で書く。
  • 演算子（+、-、*、/、%）を使って計算ができる。
  • +演算子は文字列の連結にも使える。

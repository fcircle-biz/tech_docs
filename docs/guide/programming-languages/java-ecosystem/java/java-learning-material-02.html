第2章

変数とデータ型 - データを扱う基本

プログラムでデータを扱うための「変数」の概念と、Javaで使用できるさまざまなデータ型について学びます。変数は、プログラミングにおける最も基本的で重要な概念の一つです。

この章で学ぶこと

  • 変数の概念と役割を理解する
  • 基本データ型（プリミティブ型）を使い分けられる
  • 変数の宣言、初期化、代入ができる
  • 型変換（キャスト）の仕組みを理解する

2.1 変数とは何か

変数とは、プログラムの中でデータを一時的に保管しておく「入れ物」のようなものです。例えば、数学の授業で「x = 5」と書くとき、「x」という名前を使って「5」という値を覚えていますよね。プログラミングでも同じように、データに名前を付けて、後から使えるようにします。

graph LR A[データ
例: 25] --> B[変数名
例: age] B --> C[メモリに保存] C --> D[後から使用] style A fill:#fed7aa,stroke:#fb923c style B fill:#fed7aa,stroke:#fb923c style C fill:#fed7aa,stroke:#fb923c style D fill:#fed7aa,stroke:#fb923c

変数には、次の3つの要素があります：

  • データ型：どんな種類のデータを入れるか（整数、小数、文字など）
  • 変数名：データを識別するための名前
  • 値：実際に保存されているデータ

変数の宣言と初期化

VariableExample.java 
public class VariableExample {
    public static void main(String[] args) {
        // 変数の宣言：「int型のageという名前の変数を作る」
        int age;

        // 変数への代入：「ageに25という値を入れる」
        age = 25;

        // 宣言と同時に初期化（よく使われる書き方）
        int height = 175;

        // 変数の値を使用
        System.out.println("年齢: " + age + "歳");
        System.out.println("身長: " + height + "cm");
    }
}

ポイント

変数は使用する前に必ず宣言する必要があります。また、値を入れていない変数は使えないため、初期化（最初の値を設定すること）も重要です。

変数名のつけ方（命名規則）

変数名は自由につけられますが、いくつかのルールと推奨される書き方があります：

ルール 良い例 悪い例
英字、数字、アンダースコア（_）のみ使用 age, user_name, count1 年齢, user-name, 1count
最初の文字は英字または_ total, _temp 1total, @value
小文字で始め、複数単語はキャメルケース userName, totalPrice UserName, total_price
意味のある名前をつける studentAge, itemCount a, x, temp123
予約語は使用不可 count, result int, class, if

2.2 基本データ型（プリミティブ型）

Javaには、基本的なデータを扱うための8つのプリミティブ型があります。それぞれの型には、扱えるデータの種類と範囲が決まっています。

整数型

整数（小数点以下がない数値）を扱う型です。扱える数値の大きさによって4つの型があります。

型 サイズ 値の範囲 使用例
byte 8ビット -128 〜 127 小さな整数、バイナリデータ
short 16ビット -32,768 〜 32,767 中程度の整数
int 32ビット -2,147,483,648 〜 2,147,483,647 最も一般的に使用
long 64ビット -9,223,372,036,854,775,808 〜 9,223,372,036,854,775,807 非常に大きな整数
IntegerTypes.java 
public class IntegerTypes {
    public static void main(String[] args) {
        // byte: 小さな整数
        byte temperature = 25;  // 気温など

        // short: 中程度の整数
        short year = 2025;

        // int: 通常の整数（最もよく使う）
        int population = 1500000;

        // long: 非常に大きな整数（数値の最後にLまたはlをつける）
        long distanceToSun = 149600000000L;  // 地球から太陽までの距離（km）

        System.out.println("気温: " + temperature + "℃");
        System.out.println("年: " + year);
        System.out.println("人口: " + population + "人");
        System.out.println("太陽までの距離: " + distanceToSun + " km");
    }
}

浮動小数点型

小数を扱う型です。精度（細かさ）によって2つの型があります。

型 サイズ 精度 使用例
float 32ビット 約6〜7桁 通常の小数
double 64ビット 約15桁 最も一般的に使用
FloatingPointTypes.java 
public class FloatingPointTypes {
    public static void main(String[] args) {
        // float: 通常の小数（数値の最後にFまたはfをつける）
        float weight = 65.5F;  // 体重（kg）

        // double: より精度の高い小数（最もよく使う）
        double pi = 3.141592653589793;
        double price = 1980.5;

        System.out.println("体重: " + weight + " kg");
        System.out.println("円周率: " + pi);
        System.out.println("価格: " + price + "円");
    }
}

文字型

1文字を扱う型です。シングルクォート（'）で囲みます。

CharType.java 
public class CharType {
    public static void main(String[] args) {
        // char: 1文字（シングルクォートで囲む）
        char grade = 'A';
        char initial = '太';  // 日本語も使用可能
        char symbol = '$';

        System.out.println("成績: " + grade);
        System.out.println("イニシャル: " + initial);
        System.out.println("記号: " + symbol);
    }
}

論理型

真（true）か偽（false）の2つの値だけを扱う型です。条件判定でよく使います。

BooleanType.java 
public class BooleanType {
    public static void main(String[] args) {
        // boolean: trueまたはfalse
        boolean isRaining = true;
        boolean hasLicense = false;

        System.out.println("雨が降っている: " + isRaining);
        System.out.println("免許を持っている: " + hasLicense);
    }
}

2.3 文字列型（String）

Stringは、複数の文字をまとめて扱うための型です。プリミティブ型ではなく、参照型（クラス）ですが、非常によく使うため、特別な書き方ができます。

StringExample.java 
public class StringExample {
    public static void main(String[] args) {
        // 文字列はダブルクォート（"）で囲む
        String name = "田中太郎";
        String city = "東京";

        // 文字列の連結（+演算子を使用）
        String message = "私の名前は" + name + "です。";
        String address = city + "に住んでいます。";

        System.out.println(message);
        System.out.println(address);

        // 基本的な文字列メソッド
        System.out.println("文字列の長さ: " + name.length() + "文字");
        System.out.println("大文字変換: " + "hello".toUpperCase());
        System.out.println("小文字変換: " + "WORLD".toLowerCase());
    }
}

つまずきポイント：文字列の比較

文字列を比較するときは==ではなく、equals()メソッドを使います。==は文字列の内容ではなく、メモリ上の場所を比較してしまいます。

文字列比較の例 
// ❌ 間違った書き方
String str1 = "Hello";
String str2 = new String("Hello");
if (str1 == str2) {  // falseになることがある
    System.out.println("同じ");
}

// ✅ 正しい書き方
if (str1.equals(str2)) {  // 常にtrueになる
    System.out.println("同じ");
}

2.4 型変換（キャスト）

データ型を別の型に変換することを型変換またはキャストと呼びます。型変換には、自動で行われる「暗黙的な型変換」と、明示的に指定する「明示的な型変換」があります。

暗黙的な型変換（自動型変換）

小さいデータ型から大きいデータ型への変換は、自動的に行われます。これを暗黙的な型変換といいます。

graph LR A[byte
8bit] --> B[short
16bit] B --> C[int
32bit] C --> D[long
64bit] C --> E[float
32bit] E --> F[double
64bit] style A fill:#fed7aa,stroke:#fb923c style B fill:#fed7aa,stroke:#fb923c style C fill:#fed7aa,stroke:#fb923c style D fill:#fed7aa,stroke:#fb923c style E fill:#fed7aa,stroke:#fb923c style F fill:#fed7aa,stroke:#fb923c
ImplicitCasting.java 
public class ImplicitCasting {
    public static void main(String[] args) {
        // int → long（自動変換）
        int num1 = 100;
        long num2 = num1;  // intをlongに自動変換
        System.out.println("num2: " + num2);

        // int → double（自動変換）
        int score = 85;
        double average = score;  // intをdoubleに自動変換
        System.out.println("average: " + average);  // 85.0と表示される

        // 計算時の自動変換
        int a = 10;
        double b = 3.5;
        double result = a + b;  // intがdoubleに自動変換されて計算
        System.out.println("result: " + result);  // 13.5
    }
}

明示的な型変換（キャスト）

大きいデータ型から小さいデータ型への変換や、精度を落とす変換は、明示的に指定する必要があります。これを明示的な型変換またはキャストといいます。

ExplicitCasting.java 
public class ExplicitCasting {
    public static void main(String[] args) {
        // double → int（キャストが必要）
        double price = 1980.75;
        int roundedPrice = (int) price;  // 小数点以下を切り捨て
        System.out.println("元の価格: " + price + "円");
        System.out.println("切り捨て後: " + roundedPrice + "円");

        // long → int（キャストが必要）
        long bigNumber = 100000L;
        int smallNumber = (int) bigNumber;
        System.out.println("smallNumber: " + smallNumber);

        // 整数の割り算で小数を得る
        int a = 10;
        int b = 3;

        // ❌ 整数÷整数 = 整数（小数点以下切り捨て）
        double result1 = a / b;
        System.out.println("result1: " + result1);  // 3.0（3.333...ではない）

        // ✅ どちらかをdoubleに変換してから計算
        double result2 = (double) a / b;
        System.out.println("result2: " + result2);  // 3.3333333333333335
    }
}

つまずきポイント：整数の割り算

整数同士の割り算は、結果も整数になります（小数点以下切り捨て）。小数を得たい場合は、どちらかの値をdoubleにキャストする必要があります。

文字列と数値の変換

文字列と数値の相互変換も頻繁に行います。

StringNumberConversion.java 
public class StringNumberConversion {
    public static void main(String[] args) {
        // 文字列 → 数値
        String strAge = "25";
        int age = Integer.parseInt(strAge);  // Stringをintに変換
        System.out.println("年齢: " + age + "歳");

        String strPrice = "1980.5";
        double price = Double.parseDouble(strPrice);  // Stringをdoubleに変換
        System.out.println("価格: " + price + "円");

        // 数値 → 文字列
        int count = 100;
        String strCount = String.valueOf(count);  // intをStringに変換
        System.out.println("カウント: " + strCount);

        // より簡単な方法（文字列連結を使用）
        String strCount2 = count + "";  // 空文字列と連結
        System.out.println("カウント2: " + strCount2);
    }
}

実習 2-1: 自己紹介プログラム

様々なデータ型の変数を使って、自己紹介情報を表示するプログラムを作成しましょう。

手順

  1. 1 名前（String）、年齢（int）、身長（double）、血液型（char）、学生かどうか（boolean）を表す変数を宣言し、値を代入する
  2. 2 各変数の値を表示する
  3. 3 年齢を文字列に変換して、「私は〇〇歳です」というメッセージを作成する

実装例

SelfIntroduction.java 
public class SelfIntroduction {
    public static void main(String[] args) {
        // 自己紹介情報を変数に格納
        String name = "山田花子";
        int age = 20;
        double height = 165.5;
        char bloodType = 'A';
        boolean isStudent = true;

        // 情報を表示
        System.out.println("===== 自己紹介 =====");
        System.out.println("名前: " + name);
        System.out.println("年齢: " + age + "歳");
        System.out.println("身長: " + height + "cm");
        System.out.println("血液型: " + bloodType + "型");
        System.out.println("学生: " + isStudent);

        // 年齢を使ったメッセージ
        String message = "私は" + age + "歳です。";
        System.out.println(message);
    }
}

期待される結果

===== 自己紹介 =====

名前: 山田花子

年齢: 20歳

身長: 165.5cm

血液型: A型

学生: true

私は20歳です。

チャレンジ

  • 生まれた年を計算して表示する（現在年 - 年齢）
  • 来年の年齢を計算して表示する
  • 身長をインチに変換して表示する（1インチ = 2.54cm）

理解度確認クイズ

  1. Q1. 次のコードの実行結果はどうなりますか？
    int a = 10; int b = 3; double result = a / b;

    回答を表示

    答え：3.0
    整数同士の割り算なので、結果は整数になります（小数点以下切り捨て）。その後、doubleに代入されるため、3.0と表示されます。3.333...を得るには、(double) a / bまたはa / (double) bとキャストする必要があります。

  2. Q2. 文字列の比較に==ではなくequals()メソッドを使う理由は何ですか？

    回答を表示

    ==は、文字列の内容ではなく、メモリ上の場所（参照）を比較するためです。2つの文字列が同じ内容でも、メモリ上の異なる場所に格納されている場合、==はfalseを返します。equals()メソッドは文字列の内容を比較するため、正しく判定できます。

  3. Q3. byte型で扱える値の範囲は-128から127までですが、byte b = 200;とするとどうなりますか？

    回答を表示

    コンパイルエラーが発生します。
    200はbyte型の範囲（-128〜127）を超えているため、コンパイラがエラーを出します。この場合は、int型またはshort型を使用する必要があります。範囲を超える値を無理やり代入すると、オーバーフローが発生し、予期しない値になります。

  4. Q4. 変数名として適切でないものはどれですか？
    A) userName B) _count C) 1stPlace D) totalPrice

    回答を表示

    答え：C) 1stPlace
    変数名は数字で始めることができません。最初の文字は英字またはアンダースコア（_）でなければなりません。正しくはfirstPlaceまたはplace1stのようにします。

  5. Q5. long bigNum = 10000000000;と書くとエラーになります。正しい書き方は？

    回答を表示

    答え：long bigNum = 10000000000L;
    long型のリテラル（値）には、最後にLまたはlをつける必要があります。つけないと、Javaはその値をint型として解釈しようとし、int型の範囲を超えているためエラーになります。

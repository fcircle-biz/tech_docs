第2章

変数とデータ型 - データを扱う基本

プログラムでデータを扱うための「変数」の概念と、Javaで使用できるさまざまなデータ型について学びます。

この章で学ぶこと

  • 変数の概念と役割を理解する
  • 基本データ型（プリミティブ型）を使い分けられる
  • 変数の宣言、初期化、代入ができる
  • 型変換（キャスト）の仕組みを理解する

2.1 変数とは何か

変数を「データの入れ物」として理解する

変数とは、プログラムの中でデータを一時的に保存するための「入れ物」です。現実世界で物を箱に入れて保管するのと同じように、プログラムではデータを変数に入れて管理します。

例えば、人の年齢を扱うプログラムを作る場合、「age」という名前の変数を作って、そこに「25」という数値を保存できます。後でこの年齢を変更したり、計算に使ったりすることができます。

変数の3つの要素

  • 変数名: 変数を識別するための名前（例: age, name）
  • データ型: 変数に入れられるデータの種類（例: int, String）
  • 値: 変数に実際に保存されているデータ（例: 25, "山田太郎"）

変数名のつけ方（命名規則）

Javaでは、変数名をつける際にいくつかのルールとマナーがあります。

必ず守るべきルール

  • • 先頭は英字、アンダースコア（_）、ドル記号（$）のみ
  • • 2文字目以降は英字、数字、_、$が使える
  • • 予約語（Java で特別な意味を持つ語）は使えない
  • • 大文字と小文字は区別される（age と Age は別物）

推奨される命名マナー

  • • 変数名は小文字で始める（キャメルケース）
  • • 意味のある名前をつける（x より age）
  • • 複数の単語は最初の語以外を大文字に（studentAge）
  • • 日本語は使わない（使えるが推奨しない）
変数名の例 
// 良い変数名の例
int age;              // 年齢
String studentName;   // 生徒の名前（複数単語）
double totalPrice;    // 合計金額

// 悪い変数名の例
int a;                // 意味不明（何を表すか分からない）
String StudentName;   // 大文字で始まっている
double total_price;   // アンダースコア（Java では非推奨）

変数の宣言と初期化

変数を使うには、まず「宣言」が必要です。宣言とは、「この名前の変数を使います」とJavaに伝えることです。

flowchart LR A[変数の宣言] --> B[初期化
初期値を代入] B --> C[使用
計算や表示] C --> D[値の変更
新しい値を代入] style A fill:#fff7ed,stroke:#f97316 style B fill:#fed7aa,stroke:#f97316 style C fill:#d1fae5,stroke:#10b981 style D fill:#dbeafe,stroke:#3b82f6
VariableExample.java 
public class VariableExample {
    public static void main(String[] args) {
        // 変数の宣言
        int age;

        // 変数に値を代入（初期化）
        age = 25;

        // 宣言と初期化を同時に行う（推奨）
        String name = "山田太郎";

        // 変数の値を表示
        System.out.println("名前: " + name);
        System.out.println("年齢: " + age);

        // 変数の値を変更
        age = 26;
        System.out.println("更新後の年齢: " + age);
    }
}

変数のスコープ（有効範囲）の基礎

変数には「スコープ」という有効範囲があります。変数は宣言されたブロック {} の中でのみ使用できます。

スコープの例 
public class ScopeExample {
    public static void main(String[] args) {
        int x = 10;  // main メソッド内で有効

        {
            int y = 20;  // このブロック内でのみ有効
            System.out.println(x);  // OK: x はアクセス可能
            System.out.println(y);  // OK: y はアクセス可能
        }

        System.out.println(x);  // OK: x はまだ有効
        // System.out.println(y);  // エラー: y はスコープ外
    }
}

2.2 基本データ型（プリミティブ型）

Javaには、基本的なデータを扱うための8つのプリミティブ型（基本データ型）があります。これらは、数値、文字、真偽値など、プログラムの基本要素を表現するために使います。

整数型: byte, short, int, long

整数型は、小数点を含まない数値（整数）を扱うためのデータ型です。扱える数値の大きさによって4つの型があります。

型名 サイズ 範囲 用途
byte 8bit -128 ~ 127 バイト単位のデータ処理
short 16bit -32,768 ~ 32,767 小さな整数値
int 32bit 約-21億 ~ 21億 整数の標準型（最もよく使う）
long 64bit 約-922京 ~ 922京 非常に大きな整数値
整数型の使用例 
// 整数型の宣言と初期化
int age = 25;              // 年齢（通常はintを使う）
long population = 125000000L;  // 日本の人口（Lを付ける）
short year = 2024;         // 年号
byte score = 100;          // テストの点数（0-100）

System.out.println("年齢: " + age);
System.out.println("人口: " + population);

整数型の選び方

迷ったら int を使いましょう。特別な理由がない限り、整数は int で十分です。非常に大きな数（例: タイムスタンプ、IDなど）を扱う場合のみ long を使います。

浮動小数点型: float, double

浮動小数点型は、小数を含む数値を扱うためのデータ型です。

型名 サイズ 精度 用途
float 32bit 約7桁 単精度浮動小数点
double 64bit 約15桁 小数の標準型（最もよく使う）
浮動小数点型の使用例 
// 浮動小数点型の宣言と初期化
double pi = 3.14159;           // 円周率（通常はdoubleを使う）
float height = 170.5f;         // 身長（fを付ける）
double price = 1980.0;         // 価格

System.out.println("円周率: " + pi);
System.out.println("身長: " + height + "cm");

文字型: char

char型は、1文字を扱うためのデータ型です。シングルクォーテーション（'）で囲んで表します。

文字型の使用例 
// 文字型の宣言と初期化
char grade = 'A';           // 成績
char initial = '山';        // 日本語も1文字ならOK
char symbol = '$';          // 記号

System.out.println("成績: " + grade);
System.out.println("イニシャル: " + initial);

論理型: boolean

boolean型は、真（true）か偽（false）の2つの値だけを扱うデータ型です。条件判定で非常によく使います。

論理型の使用例 
// 論理型の宣言と初期化
boolean isStudent = true;       // 学生かどうか
boolean hasLicense = false;     // 免許を持っているか
boolean isAdult = age >= 20;    // 成人かどうか（条件式の結果）

System.out.println("学生ですか: " + isStudent);
System.out.println("成人ですか: " + isAdult);

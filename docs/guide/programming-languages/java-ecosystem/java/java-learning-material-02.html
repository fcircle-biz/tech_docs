変数とデータ型の基礎

データを扱うための基本概念である変数とデータ型について学びます。整数、小数、文字列などの様々なデータ型と、それらを組み合わせた計算や処理方法を理解します。

この章で学ぶこと

  • 変数の概念と役割、変数を使うメリット
  • Javaの基本データ型（整数型、浮動小数点型、文字型、真偽型）
  • 変数の宣言、初期化、代入の方法
  • 文字列（String）の扱い方と基本操作

2.1 変数とは何か

変数とは、プログラムの中でデータを一時的に保存するための「箱」のようなものです。変数には名前（変数名）をつけることで、後からその値を参照したり、変更したりできます。

例えば、ゲームのスコアを保存したり、ユーザーが入力した名前を記録したりする際に変数を使います。変数を使うことで、プログラムが柔軟に様々なデータを処理できるようになります。

graph LR A[変数名: score] -->|保存| B[値: 100] C[変数名: name] -->|保存| D[値: 太郎] E[変数名: isActive] -->|保存| F[値: true]

変数を使う利点

変数を使うことで、同じ値を何度も書く必要がなくなります。また、値を一箇所で変更すれば、その変数を使っている全ての場所に反映されるため、プログラムの保守性が向上します。

2.2 Javaのデータ型

Javaは静的型付け言語であり、変数を作成する際には必ず「どのような種類のデータを保存するか」を指定する必要があります。これをデータ型といいます。

基本データ型（プリミティブ型）

Javaには8つの基本データ型があります。以下は特によく使われる4つです。

データ型 説明 例
int 整数（-2,147,483,648 〜 2,147,483,647） 100, -50, 0
double 浮動小数点数（小数を含む実数） 3.14, -0.5, 2.0
boolean 真偽値（trueまたはfalse） true, false
char 文字（単一文字） 'A', 'あ', '1'

参照型: String（文字列）

文字列（複数の文字の連なり）を扱うにはString型を使います。Stringは基本データ型ではなく参照型ですが、非常によく使われるため、特別な扱いを受けています。

文字列の例 
// 文字列変数の例
String greeting = "こんにちは";
String message = "Java学習中です！";

2.3 変数の宣言と初期化

変数を使うには、まず宣言（変数を作成すること）を行い、次に初期化（最初の値を設定すること）を行います。

変数宣言の基本構文

基本構文 
// データ型 変数名;
int age;

// データ型 変数名 = 初期値;
int score = 100;

実例: 様々なデータ型の変数

VariableExample.java 
public class VariableExample {
    public static void main(String[] args) {
        // 整数型変数
        int studentCount = 30;

        // 浮動小数点型変数
        double temperature = 25.5;

        // 真偽型変数
        boolean isRaining = false;

        // 文字型変数
        char grade = 'A';

        // 文字列型変数
        String message = "Hello, Java!";

        // 変数の値を画面に表示
        System.out.println("生徒数: " + studentCount);
        System.out.println("気温: " + temperature + "度");
        System.out.println("雨が降っている: " + isRaining);
        System.out.println("成績: " + grade);
        System.out.println(message);
    }
}

変数名の命名規則

  • • 変数名は英数字、アンダースコア（_）、ドル記号（$）を使用可能
  • • 数字から始めることはできない（例: 1score は不可）
  • • Javaの予約語（int、class、publicなど）は使用不可
  • • 慣例として、変数名は小文字で始め、複数単語の場合はキャメルケース（例: studentCount）を使用

2.4 変数の代入と更新

一度宣言した変数には、後から別の値を代入（更新）することができます。代入には代入演算子 = を使用します。

変数の更新例 
public class AssignmentExample {
    public static void main(String[] args) {
        // 変数の宣言と初期化
        int score = 80;
        System.out.println("初期スコア: " + score);

        // 変数に新しい値を代入
        score = 95;
        System.out.println("更新後のスコア: " + score);

        // 変数を使った計算結果を代入
        score = score + 5;
        System.out.println("5点加算後: " + score);
    }
}

代入の方向

score = score + 5; は、「scoreの現在値に5を加えた結果を、scoreに代入する」という意味です。= は「等しい」ではなく「右辺の値を左辺に代入する」という意味です。

2.5 文字列（String）の基本操作

文字列型（String）は、Javaで最もよく使われるデータ型の一つです。文字列には様々な操作が用意されています。

文字列の連結

+ 演算子を使って、複数の文字列を連結できます。

文字列の連結 
public class StringExample {
    public static void main(String[] args) {
        String firstName = "太郎";
        String lastName = "山田";

        // 文字列の連結
        String fullName = lastName + firstName;
        System.out.println("名前: " + fullName);

        // 数値と文字列の連結
        int age = 20;
        String info = fullName + "さんは" + age + "歳です。";
        System.out.println(info);
    }
}

よく使う文字列メソッド

文字列メソッドの例 
public class StringMethods {
    public static void main(String[] args) {
        String message = "Hello, Java!";

        // 文字列の長さ取得
        int length = message.length();
        System.out.println("文字数: " + length);

        // 大文字に変換
        String upper = message.toUpperCase();
        System.out.println("大文字: " + upper);

        // 小文字に変換
        String lower = message.toLowerCase();
        System.out.println("小文字: " + lower);
    }
}

実習 2-1: 自己紹介プログラム

変数を使って自己紹介情報を保存し、画面に表示するプログラムを作成しましょう。

課題

名前、年齢、好きな食べ物を変数に保存し、「私の名前は○○です。年齢は△△歳で、好きな食べ物は□□です。」という形式で表示してください。

実装例

public class SelfIntroduction {
    public static void main(String[] args) {
        // 自己紹介情報を変数に保存
        String name = "太郎";
        int age = 20;
        String favoriteFood = "ラーメン";

        // 自己紹介文を作成して表示
        String intro = "私の名前は" + name + "です。";
        intro = intro + "年齢は" + age + "歳で、";
        intro = intro + "好きな食べ物は" + favoriteFood + "です。";

        System.out.println(intro);
    }
}

期待される結果

私の名前は太郎です。年齢は20歳で、好きな食べ物はラーメンです。

チャレンジ: 趣味や出身地など、さらに情報を追加してみましょう。

理解度確認クイズ

  1. Q1. 整数を保存するためのデータ型は何ですか？

    回答を表示

    int（またはlong、short、byte）
    最も一般的に使われるのはint型です。

  2. Q2. 次のコードの実行結果はどうなりますか？ int x = 5; x = x + 3;

    回答を表示

    変数xの値は8になります。
    x = x + 3;は「xの現在値（5）に3を加えた結果（8）をxに代入する」という意味です。

  3. Q3. 文字列を連結する際に使う演算子は何ですか？

    回答を表示

    + 演算子を使います。
    例: "Hello" + " " + "World" → "Hello World"

  4. Q4. 真偽値（trueまたはfalse）を保存するデータ型は何ですか？

    回答を表示

    boolean型です。
    条件判定や状態フラグなどに使用されます。

この章のまとめ

  • 変数はデータを保存する「箱」で、名前をつけて後から参照・更新できる
  • Javaの基本データ型には、int（整数）、double（小数）、boolean（真偽値）、char（文字）がある
  • 変数宣言は「データ型 変数名 = 初期値;」の形式で行う
  • 文字列はString型で扱い、+演算子で連結できる
  • 変数名はキャメルケースで命名し、予約語や数字始まりは避ける

第6章

配列 - 複数のデータをまとめて管理

同じ種類のデータを複数まとめて扱う「配列」の概念と使い方を学びます。

この章で学ぶこと

  • 配列の概念と必要性を理解する
  • 配列の宣言、生成、初期化ができる
  • 配列の要素にアクセスし操作できる
  • 拡張for文（for-each）を使える
  • 多次元配列の基本を理解する

6.1 配列とは何か

配列（Array）とは、同じ型のデータを複数まとめて管理するためのデータ構造です。これまでは1つの変数に1つの値しか格納できませんでしたが、配列を使えば多数のデータを効率的に扱うことができます。

配列を使う利点

配列を使わない場合、5人の学生の点数を保存するには：

int score1 = 85;
int score2 = 92;
int score3 = 78;
int score4 = 88;
int score5 = 95;

配列を使えば：

int[] scores = {85, 92, 78, 88, 95};

100人分のデータでも、配列なら簡潔に扱えます！

配列の重要な概念

インデックス（添え字）

配列の各要素にアクセスするための番号。0から始まる点に注意！

配列の長さ

配列に格納できる要素の数。lengthプロパティで取得できます。

インデックスは0から始まる！

配列の最初の要素はarray[0]でアクセスします。長さ5の配列のインデックスは0, 1, 2, 3, 4です。array[5]にアクセスするとエラー（ArrayIndexOutOfBoundsException）になります。

6.2 配列の作成と初期化

Javaで配列を使うには、「宣言」→「生成」→「初期化」の3つのステップが必要です。ただし、これらをまとめて1行で書くこともできます。

方法1: 宣言と生成を分けて書く

配列の宣言と生成 
// ステップ1: 宣言（配列変数を用意）
int[] numbers;  // または int numbers[];

// ステップ2: 生成（メモリ上に配列を作成）
numbers = new int[5];  // 要素数5の配列を作成

// ステップ3: 初期化（各要素に値を代入）
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;

方法2: 宣言と生成を同時に書く

宣言と生成を1行で 
// 宣言と生成を同時に
int[] numbers = new int[5];

// 各要素に値を代入
numbers[0] = 10;
numbers[1] = 20;
// ...

方法3: 初期化子を使う（最も簡潔）

初期化子を使った作成 
// 宣言、生成、初期化を1行で
int[] numbers = {10, 20, 30, 40, 50};

// 文字列配列の例
String[] fruits = {"りんご", "バナナ", "オレンジ"};

// double配列の例
double[] prices = {150.5, 200.0, 99.99};

デフォルト値について

newで配列を生成すると、各要素には自動的にデフォルト値が設定されます：

  • 数値型（int, double など）: 0 または 0.0
  • boolean型: false
  • 参照型（String など）: null

様々な型の配列

ArrayTypes.java 
public class ArrayTypes {
    public static void main(String[] args) {
        // int型配列
        int[] ages = new int[3];
        ages[0] = 20;
        ages[1] = 25;
        ages[2] = 30;

        // String型配列
        String[] names = {"太郎", "花子", "次郎"};

        // double型配列
        double[] heights = {170.5, 165.2, 180.0};

        // boolean型配列
        boolean[] flags = {true, false, true, false};

        // char型配列
        char[] grades = {'A', 'B', 'C', 'D', 'F'};
    }
}

6.3 配列の操作

配列を作成したら、要素へのアクセス、更新、全体の処理など、様々な操作ができます。

要素へのアクセスと更新

配列の要素操作 
int[] scores = {85, 92, 78, 88, 95};

// 要素の取得（読み取り）
int firstScore = scores[0];   // 85
int thirdScore = scores[2];   // 78

// 要素の更新（書き込み）
scores[0] = 90;  // 最初の要素を90に変更
scores[2] = 80;  // 3番目の要素を80に変更

// 配列の長さを取得
int length = scores.length;  // 5（要素数）

// 最後の要素にアクセス
int lastScore = scores[scores.length - 1];  // 95

配列の長さ（length）

配列にはlengthプロパティがあり、配列の要素数を取得できます。これはメソッドではなくプロパティなので、括弧()は不要です。

lengthの使用例 
int[] numbers = {10, 20, 30, 40, 50};

System.out.println("要素数: " + numbers.length);  // 5

// 範囲外アクセスを防ぐ
int index = 10;
if (index < numbers.length) {
    System.out.println(numbers[index]);
} else {
    System.out.println("インデックスが範囲外です");
}

ArrayIndexOutOfBoundsException

配列の範囲外エラー

配列のインデックスが範囲外の場合、ArrayIndexOutOfBoundsExceptionが発生します：

int[] arr = {1, 2, 3};
System.out.println(arr[3]); // エラー！有効なインデックスは0〜2

常に0 <= index < array.lengthの範囲でアクセスしましょう。

6.4 配列とループの組み合わせ

配列とループを組み合わせることで、多数のデータを効率的に処理できます。配列の全要素を処理する方法として、通常のfor文と拡張for文（for-each文）の2種類があります。

通常のfor文を使う方法

通常のfor文 
int[] scores = {85, 92, 78, 88, 95};

// 全要素を表示
for (int i = 0; i < scores.length; i++) {
    System.out.println("scores[" + i + "] = " + scores[i]);
}

// 合計を計算
int sum = 0;
for (int i = 0; i < scores.length; i++) {
    sum += scores[i];
}
System.out.println("合計: " + sum);  // 438

// 平均を計算
double average = (double) sum / scores.length;
System.out.println("平均: " + average);  // 87.6

拡張for文（for-each文）を使う方法

拡張for文は、配列の全要素を順番に処理する場合に便利です。インデックスを意識する必要がなく、よりシンプルに書けます。

拡張for文（for-each） 
int[] scores = {85, 92, 78, 88, 95};

// 構文: for (要素の型 変数名 : 配列) { }
for (int score : scores) {
    System.out.println(score);
}

// 合計を計算
int sum = 0;
for (int score : scores) {
    sum += score;
}
System.out.println("合計: " + sum);

// String配列の例
String[] fruits = {"りんご", "バナナ", "オレンジ"};
for (String fruit : fruits) {
    System.out.println(fruit + "が好きです");
}

通常のfor文を使う場合

  • インデックスが必要な場合
  • 要素を変更したい場合
  • 特定の範囲だけ処理したい場合
  • 逆順で処理したい場合

拡張for文を使う場合

  • 全要素を順番に読み取る場合
  • インデックスが不要な場合
  • コードをシンプルにしたい場合
  • 配列のサイズを気にしたくない場合

配列の実用例

ArrayOperations.java 
public class ArrayOperations {
    public static void main(String[] args) {
        int[] numbers = {45, 23, 67, 12, 89, 34, 56};

        // 最大値を探す
        int max = numbers[0];
        for (int num : numbers) {
            if (num > max) {
                max = num;
            }
        }
        System.out.println("最大値: " + max);  // 89

        // 最小値を探す
        int min = numbers[0];
        for (int num : numbers) {
            if (num < min) {
                min = num;
            }
        }
        System.out.println("最小値: " + min);  // 12

        // 特定の値を検索
        int target = 67;
        boolean found = false;
        int index = -1;

        for (int i = 0; i < numbers.length; i++) {
            if (numbers[i] == target) {
                found = true;
                index = i;
                break;
            }
        }

        if (found) {
            System.out.println(target + "は" + index + "番目にあります");
        } else {
            System.out.println(target + "は見つかりませんでした");
        }
    }
}

6.5 多次元配列

配列の要素として別の配列を持つことができます。これを多次元配列といいます。最もよく使われるのは2次元配列で、表形式のデータ（スプレッドシートのようなもの）を扱う際に便利です。

2次元配列の作成

2次元配列の宣言と初期化 
// 方法1: サイズを指定して作成
int[][] matrix = new int[3][4];  // 3行4列の2次元配列

// 方法2: 初期化子を使って作成
int[][] scores = {
    {85, 92, 78},  // 1行目（生徒1の3科目の点数）
    {88, 95, 82},  // 2行目（生徒2の3科目の点数）
    {76, 88, 91}   // 3行目（生徒3の3科目の点数）
};

// 要素へのアクセス
int score = scores[0][1];  // 1行目、2列目 → 92

// 要素の更新
scores[1][2] = 85;  // 2行目、3列目を85に変更

2次元配列の処理

TwoDimensionalArray.java 
public class TwoDimensionalArray {
    public static void main(String[] args) {
        int[][] scores = {
            {85, 92, 78},
            {88, 95, 82},
            {76, 88, 91}
        };

        // 全要素を表示（多重ループ）
        for (int i = 0; i < scores.length; i++) {        // 行ループ
            for (int j = 0; j < scores[i].length; j++) { // 列ループ
                System.out.print(scores[i][j] + "\t");
            }
            System.out.println();
        }

        // 拡張for文で全要素を表示
        for (int[] row : scores) {
            for (int score : row) {
                System.out.print(score + "\t");
            }
            System.out.println();
        }

        // 各生徒の合計点を計算
        for (int i = 0; i < scores.length; i++) {
            int sum = 0;
            for (int j = 0; j < scores[i].length; j++) {
                sum += scores[i][j];
            }
            System.out.println("生徒" + (i + 1) + "の合計: " + sum);
        }

        // 行数と列数を取得
        int rows = scores.length;           // 行数: 3
        int cols = scores[0].length;        // 列数: 3
        System.out.println(rows + "行" + cols + "列");
    }
}

2次元配列のサイズ取得

  • array.length - 行数（外側の配列の要素数）
  • array[0].length - 列数（最初の行の要素数）

理解度確認クイズ

  1. Q1. 配列のインデックスは何から始まりますか？また、要素数5の配列の最後のインデックスは？

    回答を表示

    答え: 0から始まり、最後は4
    配列のインデックスは0から始まります。要素数が5の場合、有効なインデックスは0, 1, 2, 3, 4です。最後の要素はarray[array.length - 1]でアクセスできます。

  2. Q2. 配列をnew int[10]で作成した場合、各要素の初期値は何ですか？

    回答を表示

    答え: 0
    int型配列をnewで生成すると、すべての要素は自動的に0で初期化されます。double型なら0.0、boolean型ならfalse、参照型（Stringなど）ならnullになります。

  3. Q3. 通常のfor文と拡張for文（for-each）の使い分けはどうすればよいですか？

    回答を表示

    答え:
    拡張for文: 全要素を順番に読み取るだけの場合（インデックス不要）
    通常のfor文: インデックスが必要、要素を変更する、特定範囲だけ処理する、逆順処理が必要な場合

  4. Q4. 2次元配列int[][] matrix = new int[3][4];の行数と列数は？

    回答を表示

    答え: 3行4列
    最初の数字[3]が行数、2番目の数字[4]が列数です。行数はmatrix.length、列数はmatrix[0].lengthで取得できます。

  5. Q5. 次のコードで何が出力されますか？
    String[] arr = {"A", "B", "C"};
    System.out.println(arr.length);

    回答を表示

    答え: 3
    lengthは配列の要素数を返すプロパティです。この配列には"A", "B", "C"の3つの要素があるため、3が出力されます。注意：lengthはメソッドではないため括弧()は不要です。

この章のまとめ

  • 配列は同じ型のデータを複数まとめて管理するデータ構造で、インデックス（0から始まる）でアクセスする
  • 配列の作成方法は「宣言 → 生成 → 初期化」の3ステップだが、初期化子を使えば1行で書ける
  • lengthプロパティで配列の要素数を取得でき、範囲外アクセスに注意する
  • 配列とループを組み合わせることで、大量のデータを効率的に処理できる
  • 拡張for文は全要素を順番に読み取る場合に便利で、コードがシンプルになる
  • 2次元配列は配列の配列で、表形式のデータを扱う際に使用する

第6章

配列とコレクション入門

複数のデータをまとめて扱うための配列について学びます。配列の宣言、初期化、要素へのアクセス、多次元配列、そして拡張forループによる走査方法を理解します。

この章で学ぶこと

  • 配列とは何か、なぜ必要かを理解する
  • 配列の宣言、初期化、要素へのアクセス方法
  • for文と拡張for文を使った配列の走査（すべての要素へのアクセス）
  • 多次元配列（2次元配列）の基礎と実用例

6.1 配列とは何か

これまで、1つのデータを1つの変数に格納してきました。しかし、実際のプログラムでは複数の関連するデータをまとめて扱いたい場合が多くあります。例えば、「クラスの生徒10人分のテスト点数」や「1週間分の気温データ」などです。

配列（Array）は、同じ型のデータを複数まとめて格納できる入れ物です。配列を使うことで、似たようなデータを効率的に管理し、ループ処理と組み合わせて効率的に処理できます。

graph LR subgraph 通常の変数 A1[score1 = 85] A2[score2 = 90] A3[score3 = 78] end subgraph 配列 B[scores] B --> B1["scores[0] = 85"] B --> B2["scores[1] = 90"] B --> B3["scores[2] = 78"] end

配列のメリット

配列を使うことで、大量のデータを個別の変数として管理する必要がなくなり、コードが簡潔になります。また、ループ処理と組み合わせることで、効率的にデータを処理できます。

6.2 配列の宣言と初期化

配列を使うには、まず宣言と初期化を行います。配列の宣言には複数の方法があります。

方法1: サイズを指定して宣言

要素数（配列のサイズ）を指定して配列を作成します。この場合、各要素は初期値（数値型なら0、booleanならfalse、参照型ならnull）で初期化されます。

ArrayDeclaration.java 
// int型の配列を宣言（要素数5）
int[] scores = new int[5];

// 初期値は0で埋められる
System.out.println(scores[0]);  // 0
System.out.println(scores[1]);  // 0

方法2: 初期値を指定して宣言

配列の要素に直接値を指定して初期化できます。この方法では、要素数は自動的に決まります。

ArrayInitialization.java 
// 初期値を指定して配列を作成（要素数3）
int[] scores = {85, 90, 78};

// 文字列配列の例
String[] fruits = {"りんご", "バナナ", "オレンジ"};

System.out.println(scores[0]);  // 85
System.out.println(fruits[1]);  // バナナ

配列のインデックスに関する重要な注意

  • • 配列のインデックス（添え字）は0から始まります
  • • 要素数5の配列なら、インデックスは0, 1, 2, 3, 4です（5ではありません）
  • • 存在しないインデックスにアクセスするとArrayIndexOutOfBoundsExceptionエラーが発生します

6.3 配列要素へのアクセスと操作

配列の各要素には、インデックス（添え字）を使ってアクセスします。インデックスは角括弧[]で囲んで指定します。

ArrayAccess.java 
public class ArrayAccess {
    public static void main(String[] args) {
        // 配列の宣言と初期化
        int[] scores = {85, 90, 78, 92, 88};

        // 配列の長さ（要素数）を取得
        System.out.println("要素数: " + scores.length);  // 5

        // 各要素にアクセス
        System.out.println("1番目の点数: " + scores[0]);  // 85
        System.out.println("3番目の点数: " + scores[2]);  // 78

        // 要素の値を変更
        scores[1] = 95;
        System.out.println("変更後2番目: " + scores[1]);  // 95
    }
}

lengthプロパティ

配列名に.lengthを付けることで、配列の要素数を取得できます。これはループ処理で配列のすべての要素を処理する際に非常に便利です。

6.4 配列の走査（ループ処理）

配列の全要素を処理するには、ループ（繰り返し処理）を使います。Javaには2つの主な方法があります。

方法1: 通常のforループ（インデックスベース）

インデックスを使って各要素にアクセスする方法です。要素の値だけでなく、インデックスも必要な場合に便利です。

ArrayForLoop.java 
public class ArrayForLoop {
    public static void main(String[] args) {
        int[] scores = {85, 90, 78, 92, 88};

        // 通常のforループで全要素を表示
        for (int i = 0; i < scores.length; i++) {
            System.out.println((i + 1) + "番目: " + scores[i]);
        }
    }
}

方法2: 拡張forループ（for-each文）

配列のすべての要素を順番に処理する場合、拡張forループ（for-each文）を使うとコードがシンプルになります。

ArrayEnhancedFor.java 
public class ArrayEnhancedFor {
    public static void main(String[] args) {
        String[] fruits = {"りんご", "バナナ", "オレンジ", "ぶどう"};

        // 拡張forループで全要素を表示
        for (String fruit : fruits) {
            System.out.println("果物: " + fruit);
        }
    }
}

拡張forループの使いどころ

拡張forループは「配列のすべての要素を順番に読み取りたい」場合に最適です。インデックスが不要で、コードが読みやすくなります。ただし、要素の値を変更したい場合や、特定のインデックスだけ処理したい場合は通常のforループを使います。

6.5 配列の実用例

配列を使った実践的なプログラム例を見てみましょう。

ScoreCalculator.java 
public class ScoreCalculator {
    public static void main(String[] args) {
        // テスト点数の配列
        int[] scores = {85, 90, 78, 92, 88};

        // 合計点を計算
        int sum = 0;
        for (int score : scores) {
            sum += score;
        }

        // 平均点を計算
        double average = (double) sum / scores.length;

        // 最高点を見つける
        int max = scores[0];
        for (int score : scores) {
            if (score > max) {
                max = score;
            }
        }

        // 結果を表示
        System.out.println("合計点: " + sum);
        System.out.println("平均点: " + average);
        System.out.println("最高点: " + max);
    }
}

6.6 多次元配列（2次元配列）

配列の各要素にさらに配列を格納することができます。これを多次元配列といいます。最も一般的なのは2次元配列で、表（テーブル）のようなデータを表現できます。

graph TB A["2次元配列 scores[3][4]"] A --> R0["行0: {80, 75, 90, 85}"] A --> R1["行1: {88, 92, 78, 84}"] A --> R2["行2: {76, 81, 87, 89}"] R0 --> C0["scores[0][0]=80"] R0 --> C1["scores[0][1]=75"] R1 --> C2["scores[1][0]=88"] R2 --> C3["scores[2][3]=89"]

2次元配列は、行と列を持つ表形式のデータを扱う際に便利です。例えば、クラスの複数生徒の複数科目の点数などです。

TwoDimensionalArray.java 
public class TwoDimensionalArray {
    public static void main(String[] args) {
        // 3人の生徒、4科目の点数
        int[][] scores = {
            {80, 75, 90, 85},  // 生徒1の点数
            {88, 92, 78, 84},  // 生徒2の点数
            {76, 81, 87, 89}   // 生徒3の点数
        };

        // 各生徒の平均点を計算
        for (int i = 0; i < scores.length; i++) {
            int sum = 0;
            for (int j = 0; j < scores[i].length; j++) {
                sum += scores[i][j];
            }
            double avg = (double) sum / scores[i].length;
            System.out.println("生徒" + (i+1) + "の平均: " + avg);
        }
    }
}

2次元配列のインデックス

2次元配列ではarray[行][列]の形式でアクセスします。scores[1][2]は「2行目3列目の要素」を指します（インデックスは0から始まるので）。

実習 6-1: 週間天気データの管理

配列を使って1週間分の最高気温を管理し、平均気温や最高・最低気温を求めるプログラムを作成しましょう。

課題

1週間分（7日間）の最高気温を配列で管理し、以下を計算・表示するプログラムを作成してください。

  • 週間の平均気温
  • 週間の最高気温
  • 週間の最低気温

実装例

public class WeatherData {
    public static void main(String[] args) {
        // 1週間の最高気温データ
        int[] temperatures = {28, 30, 32, 29, 27, 31, 33};

        // 平均気温を計算
        int sum = 0;
        for (int temp : temperatures) {
            sum += temp;
        }
        double average = (double) sum / temperatures.length;

        // 最高気温と最低気温を見つける
        int max = temperatures[0];
        int min = temperatures[0];
        for (int temp : temperatures) {
            if (temp > max) max = temp;
            if (temp < min) min = temp;
        }

        // 結果表示
        System.out.println("平均気温: " + average + "度");
        System.out.println("最高気温: " + max + "度");
        System.out.println("最低気温: " + min + "度");
    }
}

理解度確認クイズ

  1. Q1. 配列のインデックスは何番から始まりますか？

    回答を表示

    配列のインデックスは0から始まります。
    例えば要素数5の配列なら、インデックスは0, 1, 2, 3, 4となります。

  2. Q2. 配列の要素数（長さ）を取得するにはどうすればよいですか？

    回答を表示

    配列名.lengthを使います。
    例: int[] arr = {1, 2, 3}; arr.length → 3

  3. Q3. 拡張forループ（for-each文）の構文として正しいものはどれですか？

    回答を表示

    for (データ型 変数名 : 配列名) { ... }
    例: for (int num : numbers) { System.out.println(num); }

  4. Q4. 2次元配列int[][] data = new int[3][4];の要素数（行数と列数）は？

    回答を表示

    行数: 3、列数: 4
    data.length → 3（行数）
    data[0].length → 4（列数）

この章のまとめ

  • 配列は同じ型のデータを複数まとめて格納できる入れ物で、効率的なデータ管理を可能にする
  • 配列のインデックスは0から始まり、要素数は.lengthで取得できる
  • 通常のforループでインデックスと値の両方にアクセスでき、拡張forループで簡潔に全要素を走査できる
  • 2次元配列を使うことで、表形式のデータを効率的に扱える
  • 配列とループを組み合わせることで、合計・平均・最大値などの集計処理が簡単に実装できる

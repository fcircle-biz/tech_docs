Javaプログラミングの世界へようこそ

プログラミングとは何か、Javaとはどんな言語かを理解し、開発環境を整えて最初のプログラムを動かしましょう。

この章で学ぶこと

  • プログラミングとは何か、コンピュータに指示を出す仕組みを理解する
  • Javaという言語の特徴と、なぜ世界中で使われているかを知る
  • VSCode + JDKで開発環境を構築し、「Hello World」を実行する

1.1 プログラミングって何だろう？

あなたは毎日、スマートフォンやパソコンを使っていますよね。 LINEでメッセージを送ったり、YouTubeで動画を見たり、ゲームを楽しんだり...。 これらはすべてプログラム（program）というものによって動いています。

プログラムとは？

コンピュータにやってほしいことを、順番に書いた「指示書」のことです。 料理のレシピのように、「まず〇〇して、次に〇〇して...」と手順を書いておくと、コンピュータがその通りに動いてくれます。

プログラミング（programming）とは、このプログラムを作る作業のことです。 つまり、コンピュータへの指示書を書く行為がプログラミングです。

なぜコンピュータには「指示書」が必要なの？

人間は「いい感じにやっておいて」と言われればなんとなく対応できますが、 コンピュータはそうはいきません。コンピュータはとても正確に動く反面、 「具体的に何をどうすればいいのか」を細かく教えてあげないと動けないのです。

たとえ話: カレーを作る指示

人間への指示

「カレーを作っておいて」

これだけで通じる

コンピュータへの指示

  • 1. 野菜を冷蔵庫から取り出す
  • 2. 野菜を一口大に切る
  • 3. 鍋に油を入れて火をつける
  • 4. 野菜を炒める...

一つ一つ具体的に書く必要がある

1.2 プログラミング言語とは

コンピュータは実は「0」と「1」の並びしか理解できません（これを機械語といいます）。 しかし、人間が「01100001 01100100 01100100...」のような数字の羅列でプログラムを書くのは非常に大変です。

そこで生まれたのがプログラミング言語（programming language）です。 プログラミング言語は、人間にとって読み書きしやすい言葉で書いたものを、 コンピュータが理解できる形に変換（翻訳）してくれる仕組みを持っています。

世の中にはたくさんの言語がある

日本語、英語、中国語...と人間の言葉にも種類があるように、 プログラミング言語にもたくさんの種類があります。 それぞれに得意なことや特徴があり、目的によって使い分けられています。

1.3 Javaってどんな言語？

Java（ジャバ）は1995年にサン・マイクロシステムズ社が開発したプログラミング言語です。 現在はオラクル社が管理しており、世界中の企業や開発者に使われています。

Javaの3つの大きな特徴

どこでも動く

「Write Once, Run Anywhere」がスローガン。 Windows、Mac、Linuxなど、どのコンピュータでも同じプログラムが動きます。

安全で信頼性が高い

銀行のシステムや企業の基幹システムなど、 ミスが許されない場面で多く使われています。

仲間が多い

世界中に開発者がたくさんいるため、 困ったときに情報を見つけやすく、学習しやすい環境があります。

名前の由来

Javaという名前は、開発チームがよく飲んでいたコーヒーの産地であるインドネシアのジャワ島から来ています。 そのため、Javaのロゴにはコーヒーカップが描かれています。

Javaはどこで使われている？

あなたの身近なところでもJavaは活躍しています。

  • Androidアプリ - スマートフォンの多くのアプリがJavaで作られています
  • 企業の業務システム - 銀行、保険、通販サイトなどの裏側で動いています
  • ゲーム - 有名なゲーム「Minecraft」もJavaで作られています
  • ICカード - クレジットカードやSuicaなどのチップ内でも動いています

1.4 Javaが動く仕組み（JVMとは）

Javaが「どこでも動く」と言われる秘密は、JVM（Java Virtual Machine：ジャバ仮想マシン）にあります。

JVMって何？

JVMは、Javaのプログラムを動かすための「仮想的なコンピュータ」です。 Windows用、Mac用、Linux用のJVMがそれぞれ用意されていて、 同じJavaプログラムをどのJVMに渡しても同じように動いてくれます。

ポイント

  • ・ソースコード：人間が書いたプログラム（.javaファイル）

  • ・JVM：バイトコードを各OSで実行してくれる仮想マシン

1.5 開発環境を準備しよう

Javaでプログラムを書いて動かすには、以下の2つが必要です。

JDK

Java Development Kitの略。 Javaのプログラムを作って動かすための道具一式です。 コンパイラ（翻訳機）やJVMが含まれています。

※ 無料で使えるOpenJDKを使用します

VSCode

Visual Studio Codeの略。 プログラムを書くための高機能なエディタ（編集ソフト）です。 無料で使え、多くの開発者に愛用されています。

※ Extension Pack for Javaを追加してJava対応にします

実習 1-1: JDKをインストールしよう

まずはJavaの開発ツール（JDK）をインストールします。

手順（Windowsの場合）

  1. 1 Webブラウザで https://adoptium.net/ にアクセス
  2. 2 「Latest LTS Release」ボタンをクリックしてダウンロード
    （LTS = Long Term Supportの略で、長期間サポートされる安定版）
  3. 3 ダウンロードした.msiファイルをダブルクリック
  4. 4 インストーラの指示に従って「Next」を押していく
    （「Set JAVA_HOME variable」にチェックが入っていることを確認）
  5. 5 「Install」をクリックし、完了まで待つ

インストール確認

コマンドプロンプト（Windowsキー + Rで「cmd」と入力してEnter）で以下を実行： 

java -version

期待される結果

openjdk version "21.0.x" 2024-xx-xx
OpenJDK Runtime Environment Temurin-21.0.x+xx (build 21.0.x+xx)
OpenJDK 64-Bit Server VM Temurin-21.0.x+xx (build 21.0.x+xx, mixed mode)

バージョン番号が表示されればインストール成功です！

実習 1-2: VSCodeとJava拡張機能をインストールしよう

次に、プログラムを書くためのエディタを用意します。

Step 1: VSCodeのインストール

  1. 1 https://code.visualstudio.com/ にアクセス
  2. 2 「Download for Windows」をクリック
  3. 3 ダウンロードしたファイルを実行してインストール

Step 2: Java拡張機能のインストール

  1. 1 VSCodeを起動
  2. 2 左側のアイコンから「Extensions」(四角が4つのアイコン)をクリック
  3. 3 検索ボックスに「Extension Pack for Java」と入力
  4. 4 検索結果から「Extension Pack for Java」(Microsoft社製)を選択
    （緑色のJavaアイコンが目印）
  5. 5 「Install」ボタンをクリック

これで開発環境の準備は完了です！

1.6 最初のプログラムを書こう - Hello World

プログラミングの世界では、新しい言語を学ぶとき、最初に「Hello World」と表示するプログラムを作る習慣があります。 これは1970年代から続く伝統で、「ちゃんと環境が動いているか」を確認する意味もあります。

実習 1-3: Hello Worldを作ろう

いよいよ最初のJavaプログラムを書いて実行します！

手順

  1. 1 VSCodeを起動し、「File」→「Open Folder」で作業用フォルダを開く
    （例：デスクトップに「java-practice」フォルダを作成して開く）
  2. 2 「File」→「New File」で新しいファイルを作成
  3. 3 ファイル名を「Hello.java」として保存
    （Ctrl + Sで保存ダイアログを表示）
  4. 4 以下のコードを入力：
Hello.java 
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
  1. 5 ファイルを保存（Ctrl + S）
  2. 6 コードの上に表示される「Run」をクリック
    （または右クリック→「Run Java」）

期待される結果

Hello World

画面下部の「TERMINAL」パネルに「Hello World」と表示されれば成功です！

1.7 コードを読み解こう

先ほど書いたプログラムの意味を一行ずつ見ていきましょう。 今は完全に理解できなくても大丈夫です。これから学んでいくうちに分かるようになります。

Hello.java（解説付き） 
public class Hello {                    // クラスの始まり
    public static void main(String[] args) {  // プログラムの開始地点
        System.out.println("Hello World");     // 画面に文字を表示
    }                                          // mainメソッドの終わり
}                                              // クラスの終わり

public class Hello

「Hello」という名前のクラス（プログラムの入れ物）を作っています。 Javaではすべてのプログラムはクラスの中に書きます。
※ ファイル名（Hello.java）とクラス名（Hello）は一致させる必要があります。

public static void main(String[] args)

mainメソッドと呼ばれる、プログラムの「開始地点」です。 Javaでプログラムを実行すると、必ずこのmainメソッドから処理が始まります。
※ この一行はおまじないのように覚えてしまいましょう。

System.out.println("Hello World");

画面（コンソール）に文字を表示する命令です。 printlnは「print line」の略で、「一行出力する」という意味です。
※ 文末のセミコロン（;）は「この命令はここまで」という区切りを示します。

{ }（波カッコ）

プログラムの「ここからここまで」という範囲を示します。 開き波カッコ { と閉じ波カッコ } は必ずペアで使います。

よくあるエラー

  • ・セミコロン（;）を忘れている → 文の最後には必ず ; を付ける
  • ・波カッコ { } が足りない、または多い → ペアになっているか確認
  • ・大文字小文字が違う → JavaはSystemとsystemを区別します
  • ・全角文字を使っている → プログラムは半角英数字で書く

理解度確認クイズ

  1. Q1. プログラミング言語とは何ですか？

    回答を表示

    人間が読み書きしやすい言葉でプログラムを書き、それをコンピュータが理解できる形に変換（翻訳）してくれる仕組みを持った言語のことです。

  2. Q2. JVMとは何の略ですか？またその役割は？

    回答を表示

    Java Virtual Machine（ジャバ仮想マシン）の略です。 Javaのバイトコード（.classファイル）を各OS上で実行してくれる仮想的なコンピュータです。 JVMがあるおかげで、同じJavaプログラムがどのOSでも動きます。

  3. Q3. System.out.println()は何をする命令ですか？

    回答を表示

    画面（コンソール）に文字を一行表示する命令です。 括弧の中に書いた文字列が画面に出力されます。

  4. Q4. Javaのプログラムはどこから実行が始まりますか？

    回答を表示

    mainメソッド（public static void main(String[] args)）から実行が始まります。 Javaプログラムには必ずmainメソッドが必要です。

1.8 この章のまとめ

  • プログラムはコンピュータへの指示書。プログラミングはその指示書を書くこと。
  • プログラミング言語は、人間とコンピュータの翻訳をしてくれるもの。
  • Javaは「どこでも動く」「安全」「仲間が多い」言語で、世界中で広く使われている。
  • JVM（Java仮想マシン）のおかげで、同じプログラムが異なるOSで動く。
  • 開発環境はJDK（道具）とVSCode（エディタ）で構築。
  • Javaプログラムはmainメソッドから始まり、System.out.println()で画面に文字を表示できる。

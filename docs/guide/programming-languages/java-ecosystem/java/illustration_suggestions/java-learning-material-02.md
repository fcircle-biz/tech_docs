## 優先度別サマリー

### 高優先度
- **2.1 変数って何だろう？**: メモリ上の変数のイメージ図（メモリアドレス、変数名、格納値の関係）
- **2.3 charとStringの違い**: プリミティブ型と参照型の根本的な違いをメモリ構造で示す図

### 中優先度
- **2.2 変数を作ってみよう**: 宣言→代入のメモリ変化を示すシーケンス図
- **2.3 データ型を知ろう**: プリミティブ型のメモリサイズと格納範囲の比較図
- **2.4 演算子**: 整数割り算と小数割り算の結果の違いを視覚的に示す比較図

### 低優先度
- **2.4 文字列の連結**: 文字列連結時のStringオブジェクト生成を示す概念図（初級者には高度すぎる可能性がある）

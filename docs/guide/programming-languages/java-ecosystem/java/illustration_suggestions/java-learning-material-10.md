## 優先度別サマリー

### 高優先度
- **クラス階層図**: 親クラスと子クラスのis-a関係を示す階層構造図
- **継承によるメンバー引き継ぎ**: 子クラスが親クラスのフィールド・メソッドを受け継ぐメモリレイアウト図
- **メソッドオーバーライドの動的ディスパッチ**: 実行時のメソッド選択の仕組みを示すシーケンス図
- **コンストラクタチェーン**: 子→親のコンストラクタ呼び出し順序を示すシーケンス図

### 中優先度
- **superキーワードの動作**: super.methodName()による親クラスメソッド呼び出しの流れを示すフローチャート

### 低優先度
- なし

## 優先度別サマリー

### 高優先度
- **正常系と異常系の処理フロー**: 例外発生時とそうでない場合の制御の流れを並列表示するフローチャート
- **例外のスロー→キャッチ**: メソッド呼び出しスタック上での例外伝播を示すシーケンス図
- **例外クラス階層図**: Throwableを頂点とする例外クラスの継承関係を示す階層図

### 中優先度
- **チェック例外と非チェック例外**: 両者の特徴と処理義務の違いを示す比較表
- **finally句の実行保証**: returnがあってもfinallyが実行されるフローチャート

### 低優先度
- なし

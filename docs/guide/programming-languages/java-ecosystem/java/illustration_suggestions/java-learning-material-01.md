## 優先度別サマリー

### 高優先度
- **1.2 プログラミング言語とは**: ソースコード→コンパイル→機械語→実行の詳細フロー図で、プログラムがどのように翻訳され実行されるかを段階的に示す
- **1.4 Javaが動く仕組み（JVMとは）**: JVMアーキテクチャの内部構造図で、バイトコードがJVM上でどのように各OS向けに実行されるかを示す

### 中優先度
- **1.1 プログラミングって何だろう？**: プログラムの役割を示す概念図（アプリケーション→プログラム→コンピュータの関係）
- **1.3 Javaってどんな言語？**: Java活用事例のイラスト集（スマホ、銀行システム、ゲーム、ICカードなど）

### 低優先度
- **1.5 開発環境を準備しよう**: 開発フロー全体図（エディタ→コンパイラ→JVMの関係）
- **1.6 Windowsでの環境構築**: インストール全体フロー図（既存の説明で十分な場合は不要）

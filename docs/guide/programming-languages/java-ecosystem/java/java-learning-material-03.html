演算子と式 - 計算と比較の方法

Javaで使用できる様々な演算子を学び、数値の計算や条件の比較ができるようになります。プログラムの基本となる演算処理をマスターしましょう。

この章で学ぶこと

  • 算術演算子を使った計算ができる
  • 比較演算子と論理演算子を理解する
  • 代入演算子と複合代入演算子を使える
  • 演算子の優先順位を理解する

3.1 算術演算子 - 四則演算の基本

算術演算子は、数値の計算を行うための演算子です。プログラミングにおいて最も基本的な演算で、数学と同じように四則演算（足し算・引き算・掛け算・割り算）を行えます。

基本的な算術演算子

演算子 説明 例 結果
+ 加算（足し算） 10 + 5 15
- 減算（引き算） 10 - 5 5
* 乗算（掛け算） 10 * 5 50
/ 除算（割り算） 10 / 5 2
% 剰余（割り算の余り） 10 % 3 1
ArithmeticOperators.java 
public class ArithmeticOperators {
    public static void main(String[] args) {
        // 基本的な算術演算
        int a = 10;
        int b = 3;

        System.out.println("a + b = " + (a + b));  // 13（加算）
        System.out.println("a - b = " + (a - b));  // 7（減算）
        System.out.println("a * b = " + (a * b));  // 30（乗算）
        System.out.println("a / b = " + (a / b));  // 3（除算：整数同士の割り算）
        System.out.println("a % b = " + (a % b));  // 1（剰余：10を3で割った余り）

        // 剰余演算子の活用例：偶数・奇数の判定
        int number = 7;
        if (number % 2 == 0) {
            System.out.println(number + "は偶数です");
        } else {
            System.out.println(number + "は奇数です");  // この行が実行される
        }
    }
}

整数同士の割り算の注意点

整数同士で割り算を行うと、結果も整数になります。小数点以下は切り捨てられるため、期待した結果にならないことがあります。 

int result1 = 10 / 3;      // 3（小数点以下切り捨て）
double result2 = 10 / 3;   // 3.0（整数同士の割り算後にdoubleに変換）
double result3 = 10.0 / 3; // 3.3333...（正しい小数の結果）
double result4 = (double)10 / 3; // 3.3333...（キャストを使用）

増減演算子（インクリメント・デクリメント）

変数の値を1増やす、または1減らす処理は頻繁に使用します。Javaでは、これらの操作を簡潔に書ける特別な演算子が用意されています。

演算子 説明 例
++変数 前置インクリメント（先に1増やす） ++i
変数++ 後置インクリメント（使用後に1増やす） i++
--変数 前置デクリメント（先に1減らす） --i
変数-- 後置デクリメント（使用後に1減らす） i--
IncrementDecrement.java 
public class IncrementDecrement {
    public static void main(String[] args) {
        // 前置と後置の違いを理解する
        int x = 5;
        System.out.println("初期値: x = " + x);  // 5

        // 後置インクリメント：使用してから1増やす
        int y = x++;
        System.out.println("x++の結果: y = " + y + ", x = " + x);  // y=5, x=6

        // 前置インクリメント：1増やしてから使用
        x = 5;  // リセット
        y = ++x;
        System.out.println("++xの結果: y = " + y + ", x = " + x);  // y=6, x=6

        // 単独で使う場合は前置も後置も同じ
        int count = 0;
        count++;  // これは count = count + 1; と同じ
        System.out.println("count = " + count);  // 1
    }
}

3.2 比較演算子 - 大小や等しさの判定

比較演算子は、2つの値を比較して、その結果を真（true）または偽（false）として返します。条件分岐やループ処理で頻繁に使用される重要な演算子です。

比較演算子一覧

演算子 説明 例 結果
== 等しい 5 == 5 true
!= 等しくない 5 != 3 true
> より大きい 5 > 3 true
< より小さい 5 < 3 false
>= 以上（等しいか大きい） 5 >= 5 true
<= 以下（等しいか小さい） 5 <= 3 false
ComparisonOperators.java 
public class ComparisonOperators {
    public static void main(String[] args) {
        int score = 85;
        int passingScore = 60;

        // 比較演算子の使用例
        System.out.println("score == 85: " + (score == 85));        // true
        System.out.println("score != 100: " + (score != 100));      // true
        System.out.println("score > 80: " + (score > 80));          // true
        System.out.println("score < 90: " + (score < 90));          // true
        System.out.println("score >= 85: " + (score >= 85));        // true
        System.out.println("score <= 60: " + (score <= 60));        // false

        // 実用的な例：合格判定
        boolean isPassed = score >= passingScore;
        System.out.println("合格: " + isPassed);  // true
    }
}

代入演算子（=）と比較演算子（==）の混同に注意

プログラミング初心者が最も頻繁に間違えるポイントの1つです。=は代入、==は比較です。 

int x = 5;     // 正しい：変数xに5を代入
if (x == 5) {  // 正しい：xが5と等しいか比較
    // 処理
}

// }

3.3 論理演算子 - 複数の条件を組み合わせる

論理演算子は、複数の条件を組み合わせて判定するための演算子です。「AかつB」「AまたはB」「Aでない」といった論理的な判断を行えます。

論理演算子一覧

演算子 説明 例 結果
&& 論理AND（かつ） true && true true
|| 論理OR（または） true || false true
! 論理NOT（否定） !true false

論理演算子の真理値表

AND演算子（&&）

両方がtrueのときだけtrue

A B A && B
true true true
true false false
false true false
false false false

OR演算子（||）

どちらかがtrueならtrue

A B A || B
true true true
true false true
false true true
false false false
LogicalOperators.java 
public class LogicalOperators {
    public static void main(String[] args) {
        int age = 25;
        boolean hasLicense = true;
        boolean hasExperience = false;

        // AND演算子（&&）：両方の条件が真のときだけ真
        boolean canDrive = age >= 18 && hasLicense;
        System.out.println("運転できる: " + canDrive);  // true

        // OR演算子（||）：どちらかの条件が真なら真
        boolean canApply = hasLicense || hasExperience;
        System.out.println("応募できる: " + canApply);  // true

        // NOT演算子（!）：真偽を反転
        boolean isMinor = !(age >= 18);
        System.out.println("未成年: " + isMinor);  // false

        // 複雑な条件の組み合わせ
        int temperature = 28;
        boolean isSunny = true;

        // 「晴れていて、かつ気温が25度以上30度以下」
        boolean perfectWeather = isSunny && (temperature >= 25 && temperature <= 30);
        System.out.println("最高の天気: " + perfectWeather);  // true
    }
}

短絡評価（ショートサーキット）

Javaの論理演算子は「短絡評価」を行います。これは、結果が確定した時点で残りの評価を省略する仕組みです。

  • &&の場合： 左側がfalseなら、右側は評価されない（どちらにしてもfalse）
  • ||の場合： 左側がtrueなら、右側は評価されない（どちらにしてもtrue） 
int x = 5;
// 左側がfalseなので、x++は実行されない
if (false && x++ > 0) {
    // 実行されない
}
System.out.println(x);  // 5（x++は実行されていない）

3.4 代入演算子 - 値の代入と複合代入

代入演算子は変数に値を格納するための演算子です。基本的な代入演算子=に加えて、演算と代入を同時に行う複合代入演算子があります。

複合代入演算子一覧

演算子 説明 例 同等の記述
= 代入 x = 5 -
+= 加算代入 x += 3 x = x + 3
-= 減算代入 x -= 3 x = x - 3
*= 乗算代入 x *= 3 x = x * 3
/= 除算代入 x /= 3 x = x / 3
%= 剰余代入 x %= 3 x = x % 3
AssignmentOperators.java 
public class AssignmentOperators {
    public static void main(String[] args) {
        // 基本的な代入
        int score = 100;
        System.out.println("初期スコア: " + score);  // 100

        // 複合代入演算子の使用例
        score += 50;  // score = score + 50 と同じ
        System.out.println("ボーナス加算後: " + score);  // 150

        score -= 30;  // score = score - 30 と同じ
        System.out.println("ペナルティ減算後: " + score);  // 120

        score *= 2;   // score = score * 2 と同じ
        System.out.println("2倍後: " + score);  // 240

        score /= 4;   // score = score / 4 と同じ
        System.out.println("4分の1後: " + score);  // 60

        score %= 7;   // score = score % 7 と同じ
        System.out.println("7で割った余り: " + score);  // 4

        // 実用例：累積計算
        int total = 0;
        total += 100;  // 1回目の購入
        total += 250;  // 2回目の購入
        total += 80;   // 3回目の購入
        System.out.println("合計購入金額: " + total + "円");  // 430円
    }
}

3.5 演算子の優先順位 - 計算の順序を理解する

複数の演算子を含む式では、演算子の優先順位によって計算される順序が決まります。数学と同様に、掛け算・割り算は足し算・引き算より先に計算されます。

主要な演算子の優先順位（高い順）

優先順位 演算子 説明
1（最高） ( ) 括弧
2 ++ -- インクリメント・デクリメント
3 ! 論理否定
4 * / % 乗算・除算・剰余
5 + - 加算・減算
6 < <= > >= 大小比較
7 == != 等価比較
8 && 論理AND
9 || 論理OR
10（最低） = += -= *= /= %= 代入演算子
OperatorPrecedence.java 
public class OperatorPrecedence {
    public static void main(String[] args) {
        // 優先順位の例
        int result1 = 10 + 5 * 2;  // 20（*が先に計算される）
        System.out.println("10 + 5 * 2 = " + result1);

        // 括弧を使って優先順位を変更
        int result2 = (10 + 5) * 2;  // 30（括弧内が先に計算される）
        System.out.println("(10 + 5) * 2 = " + result2);

        // 複雑な式
        int result3 = 100 - 50 / 5 + 10 * 2;
        // 計算順序: 50/5=10, 10*2=20, 100-10=90, 90+20=110
        System.out.println("100 - 50 / 5 + 10 * 2 = " + result3);  // 110

        // 論理演算子の優先順位
        boolean result4 = 5 > 3 && 10 < 20;  // true && true = true
        System.out.println("5 > 3 && 10 < 20 = " + result4);

        // 括弧で明示的に優先順位を示す（推奨）
        boolean result5 = (age >= 18) && (hasLicense == true);
        // 読みやすく、意図が明確

        // 複雑な条件式には括弧を使う
        int age = 25;
        boolean hasLicense = true;
        int experience = 3;

        boolean canRentCar = (age >= 21) && (hasLicense) && (experience >= 1);
        System.out.println("レンタカー可能: " + canRentCar);
    }
}

読みやすいコードを書くヒント

  • 優先順位に自信がない場合は、迷わず括弧()を使いましょう
  • 複雑な式は複数行に分割し、中間結果を変数に格納すると理解しやすくなります
  • 論理演算を含む条件式には括弧を使って、意図を明確にしましょう

実習 3-1: BMI計算プログラム

体重と身長からBMI（Body Mass Index）を計算し、肥満度を判定するプログラムを作成します。算術演算子と比較演算子を使って実装しましょう。

手順

  1. 1 体重（kg）と身長（cm）を変数に格納する
  2. 2 身長をメートルに変換する（100で割る）
  3. 3 BMI = 体重 ÷ (身長 × 身長) の式で計算する
  4. 4 BMI値と肥満度判定（18.5未満：やせ、25未満：標準、25以上：肥満）を表示する

実装例

public class BMICalculator {
    public static void main(String[] args) {
        // 体重と身長の入力
        double weight = 65.0;  // kg
        double heightCm = 170.0;  // cm

        // 身長をメートルに変換
        double heightM = heightCm / 100.0;

        // BMI計算
        double bmi = weight / (heightM * heightM);

        // 結果表示
        System.out.println("体重: " + weight + "kg");
        System.out.println("身長: " + heightCm + "cm");
        System.out.println("BMI: " + bmi);

        // 判定（次章で学ぶif文を使った実装例）
        System.out.println("BMI判定: 次章で判定ロジックを実装します");
    }
}

実習 3-2: 簡単な電卓プログラム

2つの数値に対して四則演算を行う電卓プログラムを作成します。全ての算術演算子を使ってみましょう。

実装例

public class SimpleCalculator {
    public static void main(String[] args) {
        double num1 = 25.0;
        double num2 = 4.0;

        System.out.println("=== 簡易電卓 ===");
        System.out.println("数値1: " + num1);
        System.out.println("数値2: " + num2);
        System.out.println();

        // 四則演算の実行
        System.out.println("加算: " + num1 + " + " + num2 + " = " + (num1 + num2));
        System.out.println("減算: " + num1 + " - " + num2 + " = " + (num1 - num2));
        System.out.println("乗算: " + num1 + " * " + num2 + " = " + (num1 * num2));
        System.out.println("除算: " + num1 + " / " + num2 + " = " + (num1 / num2));
        System.out.println("剰余: " + num1 + " % " + num2 + " = " + (num1 % num2));
    }
}

理解度確認クイズ

  1. Q1. 以下のコードを実行したときの結果は何ですか？

    int x = 10;
int y = x++ + ++x;
System.out.println(y);
    回答を表示

    答え: 22
    解説: x++は使用後に増加するので10を返し、xは11になります。次に++xは先に増加するので12になり、12を返します。したがって、10 + 12 = 22です。

  2. Q2. 整数同士の割り算 7 / 2 の結果は何ですか？

    回答を表示

    答え: 3
    解説: 整数同士の割り算では、結果も整数になります。小数点以下は切り捨てられるため、7 ÷ 2 = 3.5 ではなく 3 になります。3.5を得たい場合は、少なくとも一方をdouble型にする必要があります（例: 7.0 / 2）。

  3. Q3. 論理演算 true && false || true の結果は何ですか？

    回答を表示

    答え: true
    解説: &&演算子は||演算子より優先順位が高いため、まず true && false が評価されて false になります。次に false || true が評価されて true になります。

  4. Q4. x += 5 と同等の記述はどれですか？

    回答を表示

    答え: x = x + 5
    解説: 複合代入演算子 += は、変数に値を加算して結果を再び変数に代入する演算を1つにまとめたものです。コードを簡潔に書けるため、頻繁に使用されます。

  5. Q5. 剰余演算子 % の使い道として適切なものはどれですか？

    回答を表示

    答え: 偶数・奇数の判定、一定範囲でのループ、周期的な処理など
    解説: 剰余演算子は割り算の余りを求めます。num % 2 == 0 で偶数判定、index % length で配列の範囲内にインデックスを収める、などの用途があります。

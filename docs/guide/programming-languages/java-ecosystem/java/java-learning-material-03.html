プログラムに判断させる - 条件分岐

「もし～ならば」という条件によってプログラムの動きを変える方法を学びます。

この章で学ぶこと

  • 条件分岐の概念と、なぜ必要かを理解する
  • if文、else文、else if文の書き方を覚える
  • 比較演算子と論理演算子を使った条件の書き方を学ぶ

3.1 条件分岐って何？

私たちは日常生活で「もし雨が降ったら傘を持っていく」「もし暑かったらエアコンをつける」のように、 状況に応じて行動を変えています。プログラムも同じように、条件によって動作を変えることができます。 これを条件分岐といいます。

3.2 if文の基本

Javaで条件分岐を行うにはif文を使います。 「もし〇〇ならば、△△を実行する」という形で書きます。

if文の基本形 
int score = 80;

// もしscoreが60以上ならば
if (score >= 60) {
    System.out.println("合格です！");
}

比較演算子

演算子 意味 例
==等しいa == b
!=等しくないa != b
>より大きいa > b
<より小さいa < b
>=以上a >= b
<=以下a <= b

注意: == と = の違い

=は代入、==は比較です。 間違えやすいので注意しましょう！

3.3 if-else文

「条件に合えば○○、合わなければ△△」というように、条件に合わない場合の処理も書きたいときは elseを使います。

if-else文 
int score = 45;

if (score >= 60) {
    System.out.println("合格です！");
} else {
    System.out.println("不合格です。もう一度頑張りましょう。");
}

3.4 else if文で複数条件

3つ以上の選択肢がある場合はelse ifを使います。

Grade.java 
int score = 85;

if (score >= 90) {
    System.out.println("評価: A（素晴らしい！）");
} else if (score >= 80) {
    System.out.println("評価: B（よくできました）");
} else if (score >= 70) {
    System.out.println("評価: C（まずまずです）");
} else if (score >= 60) {
    System.out.println("評価: D（合格ラインです）");
} else {
    System.out.println("評価: F（要再試験）");
}

3.5 論理演算子で条件を組み合わせる

複数の条件を組み合わせたい場合は論理演算子を使います。

演算子 意味 説明
&&AND（かつ）両方の条件が成り立つとき
||OR（または）どちらか一方でも成り立つとき
!NOT（否定）条件を反転させる
論理演算子の例 
int age = 20;
boolean hasLicense = true;

// AND: 両方の条件を満たす場合
if (age >= 18 && hasLicense) {
    System.out.println("車を運転できます");
}

// OR: どちらかの条件を満たす場合
int dayOfWeek = 6;  // 土曜日
if (dayOfWeek == 6 || dayOfWeek == 0) {
    System.out.println("週末です！");
}

実習 3-1: おみくじプログラム

1〜5のランダムな数字に応じて運勢を表示するプログラムを作りましょう。

Omikuji.java 
import java.util.Random;

public class Omikuji {
    public static void main(String[] args) {
        Random random = new Random();
        int fortune = random.nextInt(5) + 1;  // 1〜5の乱数

        System.out.println("今日の運勢は...");
        if (fortune == 1) {
            System.out.println("大吉！最高の一日になります！");
        } else if (fortune == 2) {
            System.out.println("中吉！良いことがありそう");
        } else if (fortune == 3) {
            System.out.println("小吉！まずまずの運勢");
        } else if (fortune == 4) {
            System.out.println("末吉！油断大敵");
        } else {
            System.out.println("凶！慎重に行動しましょう");
        }
    }
}

理解度確認クイズ

  1. Q1. if文で等しいかどうかを比較するときに使う演算子は？

    回答を表示

    ==（イコール2つ）です。 =（イコール1つ）は代入なので注意！

  2. Q2. 「AかつB」を表す論理演算子は？

    回答を表示

    &&（AND演算子）です。 両方の条件が成り立つ場合にtrueになります。

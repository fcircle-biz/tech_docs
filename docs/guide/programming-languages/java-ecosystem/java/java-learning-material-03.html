第3章

演算子と式 - 計算と比較の方法

Javaで使用できる様々な演算子を学び、数値の計算や条件の比較ができるようになります。

この章で学ぶこと

  • 算術演算子を使った計算ができる
  • 比較演算子と論理演算子を理解する
  • 代入演算子と複合代入演算子を使える
  • 演算子の優先順位を理解する

3.1 演算子とは何か

演算子は、変数や値を使って計算や比較を行うための記号です。プログラミングでは、数学の計算と同じように四則演算（足し算・引き算・掛け算・割り算）ができるほか、値を比較したり、論理的な判断をしたりすることができます。

演算子の種類

Javaには主に以下の演算子があります：

  • 算術演算子: 数値の計算を行う（+、-、*、/、%）
  • 比較演算子: 値を比較してtrueまたはfalseを返す（==、!=、<、>）
  • 論理演算子: 複数の条件を組み合わせる（&&、||、!）
  • 代入演算子: 変数に値を代入する（=、+=、-=）

3.2 算術演算子 - 数値の計算

算術演算子は、数値の四則演算を行うための演算子です。数学で使う計算記号とほぼ同じですが、掛け算は*、割り算は/を使います。

基本的な算術演算子

演算子 名前 例 結果
+ 加算（足し算） 5 + 3 8
- 減算（引き算） 5 - 3 2
* 乗算（掛け算） 5 * 3 15
/ 除算（割り算） 10 / 3 3（整数同士の割り算）
% 剰余（割り算の余り） 10 % 3 1
ArithmeticExample.java 
public class ArithmeticExample {
    public static void main(String[] args) {
        // 基本的な四則演算
        int a = 10;
        int b = 3;

        System.out.println("a + b = " + (a + b));  // 13
        System.out.println("a - b = " + (a - b));  // 7
        System.out.println("a * b = " + (a * b));  // 30
        System.out.println("a / b = " + (a / b));  // 3（整数除算）
        System.out.println("a % b = " + (a % b));  // 1（余り）

        // 浮動小数点数の割り算
        double c = 10.0;
        double d = 3.0;
        System.out.println("c / d = " + (c / d));  // 3.3333...
    }
}

整数同士の割り算の注意点

整数型（int）同士の割り算は、結果も整数になります。小数部分は切り捨てられます。

10 / 3 → 3（3.333...ではない！）
小数の結果が欲しい場合は、どちらか一方をdouble型にキャストします：
(double)10 / 3 → 3.3333...

増減演算子（インクリメント・デクリメント）

変数の値を1増やしたり、1減らしたりする特別な演算子です。ループ処理などで頻繁に使われます。

IncrementExample.java 
public class IncrementExample {
    public static void main(String[] args) {
        int count = 5;

        // 後置インクリメント（使った後に+1）
        System.out.println(count++);  // 5を表示してから+1
        System.out.println(count);    // 6

        // 前置インクリメント（先に+1してから使う）
        System.out.println(++count);  // +1してから7を表示

        // デクリメント（-1）も同様
        System.out.println(count--);  // 7を表示してから-1
        System.out.println(--count);  // -1してから5を表示
    }
}

前置と後置の違い

++count（前置）は先に+1してから値を使います。
count++（後置）は値を使ってから+1します。
単独で使う場合は同じですが、式の中で使う場合は動作が異なります。

3.3 比較演算子 - 値を比較する

比較演算子は、2つの値を比較して、その結果を真（true）または偽（false）で返します。条件分岐（次章で学習）で判断を行う際に必須の演算子です。

演算子 意味 例 結果
== 等しい 5 == 5 true
!= 等しくない 5 != 3 true
< より小さい 5 < 10 true
> より大きい 5 > 10 false
<= 以下 5 <= 5 true
>= 以上 5 >= 10 false
ComparisonExample.java 
public class ComparisonExample {
    public static void main(String[] args) {
        int age = 20;
        int minAge = 18;

        // 比較演算子の使用例
        boolean isAdult = age >= minAge;  // 20 >= 18 → true
        System.out.println("成人ですか？: " + isAdult);

        boolean isSame = age == 20;  // 20 == 20 → true
        System.out.println("年齢は20歳ですか？: " + isSame);

        boolean isDifferent = age != 30;  // 20 != 30 → true
        System.out.println("年齢は30歳ではありませんか？: " + isDifferent);

        // 数値の大小比較
        int score = 75;
        boolean isHighScore = score > 80;  // false
        System.out.println("80点以上ですか？: " + isHighScore);
    }
}

=（代入）と==（比較）の違い

初心者が最もよく間違える部分です：
= は代入演算子（値を変数に入れる）
== は比較演算子（値が等しいか判定する）
必ず使い分けましょう！

3.4 論理演算子 - 複数の条件を組み合わせる

論理演算子は、複数の条件を組み合わせて、より複雑な判断を行うための演算子です。「AかつB」「AまたはB」「Aでない」といった論理的な判断ができます。

演算子 意味 説明
&& AND（かつ） 両方の条件がtrueの場合にtrue
|| OR（または） どちらか一方がtrueの場合にtrue
! NOT（否定） trueとfalseを反転

論理演算子の真理値表

論理演算子の動作を真理値表で確認しましょう。

AND演算（&&）

A B A && B
true true true
true false false
false true false
false false false

OR演算（||）

A B A || B
true true true
true false true
false true true
false false false
LogicalExample.java 
public class LogicalExample {
    public static void main(String[] args) {
        int age = 25;
        boolean hasLicense = true;

        // AND演算（&&）: 両方の条件を満たす必要がある
        boolean canDrive = age >= 18 && hasLicense;
        System.out.println("運転できますか？: " + canDrive);  // true

        // OR演算（||）: どちらか一方を満たせばOK
        int dayOfWeek = 6;  // 土曜日
        boolean isWeekend = dayOfWeek == 6 || dayOfWeek == 7;
        System.out.println("週末ですか？: " + isWeekend);  // true

        // NOT演算（!）: 条件を反転
        boolean isWeekday = !isWeekend;
        System.out.println("平日ですか？: " + isWeekday);  // false

        // 複雑な条件の組み合わせ
        int score = 85;
        boolean isPassed = score >= 60 && score <= 100;
        System.out.println("合格ですか？: " + isPassed);  // true
    }
}

短絡評価（ショートサーキット）

&&や||は短絡評価を行います：
• &&: 左側がfalseなら右側を評価しない（どうせ結果はfalse）
• ||: 左側がtrueなら右側を評価しない（どうせ結果はtrue）
これにより処理が効率的になり、エラーを防ぐこともできます。

3.5 代入演算子と複合代入演算子

代入演算子は変数に値を格納するための演算子です。基本的な=に加えて、演算と代入を同時に行う複合代入演算子もあります。

演算子 例 同じ意味
= x = 5 xに5を代入
+= x += 3 x = x + 3
-= x -= 3 x = x - 3
*= x *= 3 x = x * 3
/= x /= 3 x = x / 3
%= x %= 3 x = x % 3
AssignmentExample.java 
public class AssignmentExample {
    public static void main(String[] args) {
        int score = 100;
        System.out.println("初期スコア: " + score);

        // 複合代入演算子の使用例
        score += 50;  // score = score + 50; と同じ
        System.out.println("50点追加: " + score);  // 150

        score -= 30;  // score = score - 30; と同じ
        System.out.println("30点減点: " + score);  // 120

        score *= 2;   // score = score * 2; と同じ
        System.out.println("2倍に: " + score);     // 240

        score /= 4;   // score = score / 4; と同じ
        System.out.println("4分の1に: " + score);  // 60
    }
}

複合代入演算子の利点

複合代入演算子を使うと、コードが短く読みやすくなります。
count += 1 は count = count + 1 より簡潔で、意図も明確です。

3.6 演算子の優先順位

複数の演算子を組み合わせた式では、どの演算子から先に計算されるかが重要です。数学と同じように、掛け算・割り算は足し算・引き算より先に計算されます。

優先順位 演算子 説明
高 ( ) 括弧
↓ ++, -- インクリメント、デクリメント
↓ !, +（正）, -（負） 単項演算子
↓ *, /, % 乗算、除算、剰余
↓ +, - 加算、減算
↓ <, >, <=, >= 大小比較
↓ ==, != 等価比較
↓ && 論理AND
↓ || 論理OR
低 =, +=, -=, *=, /= 代入演算子
PrecedenceExample.java 
public class PrecedenceExample {
    public static void main(String[] args) {
        // 演算子の優先順位の例
        int result1 = 10 + 5 * 2;  // *が先に計算される
        System.out.println(result1);  // 20（10 + 10）

        // 括弧で優先順位を変更
        int result2 = (10 + 5) * 2;  // 括弧内が先に計算される
        System.out.println(result2);  // 30（15 * 2）

        // 複雑な式
        int a = 5;
        int b = 10;
        boolean result3 = a < b && b > 0;  // 比較が先、その後AND
        System.out.println(result3);  // true

        // 読みやすさのために括弧を使う（推奨）
        boolean result4 = (a < b) && (b > 0);
        System.out.println(result4);  // true
    }
}

読みやすいコードを書くコツ

優先順位を覚える必要はありません。迷ったら括弧( )を使いましょう。
括弧を使うことで、コードの意図が明確になり、バグも防げます。

実習 3-1: BMI計算プログラム

体重と身長からBMI（Body Mass Index）を計算し、肥満度を判定するプログラムを作成しましょう。

手順

  1. 1 体重（kg）と身長（m）を変数に格納する
  2. 2 BMI = 体重 / (身長 * 身長) の式で計算する
  3. 3 BMIが25以上かどうかを判定する変数を作成
  4. 4 結果を表示する

実行例

BMICalculator.java 
public class BMICalculator {
    public static void main(String[] args) {
        // 体重と身長の設定
        double weight = 70.0;  // kg
        double height = 1.75;  // m

        // BMIの計算
        double bmi = weight / (height * height);

        // 肥満度の判定
        boolean isOverweight = bmi >= 25.0;

        // 結果の表示
        System.out.println("体重: " + weight + "kg");
        System.out.println("身長: " + height + "m");
        System.out.println("BMI: " + bmi);
        System.out.println("肥満ですか？: " + isOverweight);
    }
}

期待される結果

体重: 70.0kg
身長: 1.75m
BMI: 22.857142857142858
肥満ですか？: false

実習 3-2: 簡単な電卓プログラム

2つの数値に対して四則演算を行い、結果を表示するプログラムを作成しましょう。

実行例

SimpleCalculator.java 
public class SimpleCalculator {
    public static void main(String[] args) {
        int num1 = 15;
        int num2 = 4;

        // 四則演算の実行
        int sum = num1 + num2;
        int difference = num1 - num2;
        int product = num1 * num2;
        int quotient = num1 / num2;      // 整数除算
        int remainder = num1 % num2;     // 余り
        double decimalResult = (double)num1 / num2;  // 小数除算

        // 結果の表示
        System.out.println("数値1: " + num1);
        System.out.println("数値2: " + num2);
        System.out.println("加算: " + sum);
        System.out.println("減算: " + difference);
        System.out.println("乗算: " + product);
        System.out.println("除算（整数）: " + quotient);
        System.out.println("余り: " + remainder);
        System.out.println("除算（小数）: " + decimalResult);
    }
}

理解度確認クイズ

  1. Q1. 次のコードの実行結果は何でしょうか？
    int result = 17 / 5;

    回答を表示
    答え: 3
    整数同士の割り算では、小数部分が切り捨てられます。17 ÷ 5 = 3.4 ですが、整数型では3になります。

  2. Q2. true && false の結果は？

    回答を表示
    答え: false
    AND演算（&&）は両方の条件がtrueの時だけtrueになります。一方がfalseなので結果はfalseです。

  3. Q3. int x = 10; x += 5; 実行後のxの値は？

    回答を表示
    答え: 15
    x += 5 は x = x + 5 と同じ意味です。10 + 5 = 15になります。

  4. Q4. 10 % 3 の結果は何でしょうか？

    回答を表示
    答え: 1
    %は剰余演算子で、割り算の余りを返します。10 ÷ 3 = 3 余り 1 なので、結果は1です。

  5. Q5. 次の式の実行順序は？
    int result = 2 + 3 * 4;

    回答を表示
    答え: 14（先に3 * 4 = 12、次に2 + 12 = 14）
    乗算（*）は加算（+）より優先順位が高いため、先に3 * 4が計算されます。

この章のまとめ

  • 算術演算子: +、-、*、/、% を使って数値の計算ができます
  • 比較演算子: ==、!=、<、> などで値を比較し、true/falseを得られます
  • 論理演算子: &&（AND）、||（OR）、!（NOT）で複数の条件を組み合わせられます
  • 代入演算子: =、+=、-= などで変数に値を代入できます
  • 演算子の優先順位: 迷ったら括弧を使って明示しましょう
  • 整数除算の注意: int同士の割り算は小数部分が切り捨てられます

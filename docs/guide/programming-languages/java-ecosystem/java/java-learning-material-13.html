便利な道具を使いこなす - Java標準ライブラリ

Javaに最初から用意されている便利な機能（標準ライブラリ）を学び、実践的なプログラム作成に役立てましょう。

この章で学ぶこと

  • Javaの標準ライブラリとは何か、どのような便利な機能が用意されているかを理解する
  • 文字列操作（Stringクラス）の便利なメソッドを使いこなす方法を学ぶ
  • 日付時間処理（LocalDate、LocalTime）や数学計算（Mathクラス）の実用的な使い方を習得する

13.1 Java標準ライブラリとは？

プログラミングをしていると、「文字列を操作したい」「日付を扱いたい」「数学的な計算をしたい」といった よくある作業に出会います。これらを毎回自分で一から作るのは大変です。

そこでJavaには最初から標準ライブラリ（Standard Library）という 「便利な道具セット」が用意されています。これは料理でいえば「包丁」「フライパン」「計量カップ」のような 基本的な調理器具にあたります。

標準ライブラリとは？

Javaのプログラムで頻繁に使う機能が、あらかじめ「クラス」や「メソッド」の形で提供されているものです。 これらはJDKをインストールすれば誰でも無料で使えます。

代表的な標準ライブラリ

パッケージ/クラス 用途 例
java.lang.String 文字列操作 文字列の長さ、検索、置換など
java.time 日付・時刻処理 今日の日付、時間の計算など
java.lang.Math 数学計算 平方根、最大値、乱数など
java.util.Random 乱数生成 サイコロ、抽選、ゲームなど

13.2 文字列を操る - Stringクラス

Stringクラスは、文字列を扱うための最も基本的で重要なクラスです。 文字列の長さを調べたり、一部を切り出したり、大文字小文字を変換したりと、たくさんの便利なメソッドが用意されています。

主要なStringメソッド

メソッド 説明 例
length() 文字列の長さを返す "Hello".length() → 5
charAt(int) 指定位置の文字を取得 "Java".charAt(0) → 'J'
substring(int) 部分文字列を切り出す "Hello".substring(1, 4) → "ell"
toUpperCase() 大文字に変換 "java".toUpperCase() → "JAVA"
toLowerCase() 小文字に変換 "JAVA".toLowerCase() → "java"
contains(String) 指定文字列を含むか判定 "Java".contains("av") → true
replace(char, char) 文字を置換 "Java".replace('a', 'o') → "Jovo"
trim() 前後の空白を削除 " Java ".trim() → "Java"
StringExample.java 
public class StringExample {
    public static void main(String[] args) {
        String message = "Hello Java";

        // 文字列の長さを取得
        System.out.println("長さ: " + message.length());  // 10

        // 大文字に変換
        System.out.println("大文字: " + message.toUpperCase());  // HELLO JAVA

        // 文字列に特定の文字が含まれるか確認
        if (message.contains("Java")) {
            System.out.println("Javaという文字列が含まれています");
        }

        // 文字列を置換
        String newMessage = message.replace("Java", "World");
        System.out.println("置換後: " + newMessage);  // Hello World
    }
}

実習 13-1: 文字列操作を試してみよう

Stringクラスの便利なメソッドを使って、文字列をいろいろ操作してみましょう。

課題

ユーザーが入力したメールアドレスが正しい形式かを簡易的にチェックするプログラムを作成してください。

EmailChecker.java 
public class EmailChecker {
    public static void main(String[] args) {
        String email = "user@example.com";

        // @マークが含まれているかチェック
        if (!email.contains("@")) {
            System.out.println("エラー: @が含まれていません");
            return;
        }

        // 空白が含まれていないかチェック
        if (email.contains(" ")) {
            System.out.println("エラー: 空白は使えません");
            return;
        }

        // .comまたは.jpで終わっているかチェック
        if (email.endsWith(".com") || email.endsWith(".jp")) {
            System.out.println("✓ 正しいメールアドレスの形式です");
        } else {
            System.out.println("エラー: .comまたは.jpで終わっていません");
        }
    }
}

実行例

✓ 正しいメールアドレスの形式です

13.3 日付と時刻を扱う - java.timeパッケージ

日付や時刻を扱うとき、Java 8以降ではjava.timeパッケージの クラスを使います。これらは使いやすく、間違えにくい設計になっています。

主要なクラス

LocalDate

日付のみを扱うクラス。年・月・日の情報を持ちます。

例: 2025-11-30

LocalTime

時刻のみを扱うクラス。時・分・秒の情報を持ちます。

例: 14:30:00

LocalDateTime

日付と時刻の両方を扱うクラス。

例: 2025-11-30 14:30:00

LocalDateの基本操作

DateExample.java 
import java.time.LocalDate;

public class DateExample {
    public static void main(String[] args) {
        // 今日の日付を取得
        LocalDate today = LocalDate.now();
        System.out.println("今日: " + today);  // 2025-11-30

        // 特定の日付を作成
        LocalDate birthday = LocalDate.of(2000, 5, 15);
        System.out.println("誕生日: " + birthday);  // 2000-05-15

        // 日付の計算
        LocalDate tomorrow = today.plusDays(1);  // 1日後
        LocalDate nextMonth = today.plusMonths(1);  // 1ヶ月後
        LocalDate lastYear = today.minusYears(1);  // 1年前

        System.out.println("明日: " + tomorrow);
        System.out.println("来月の今日: " + nextMonth);
        System.out.println("去年の今日: " + lastYear);
    }
}

LocalTimeの基本操作

TimeExample.java 
import java.time.LocalTime;

public class TimeExample {
    public static void main(String[] args) {
        // 現在の時刻を取得
        LocalTime now = LocalTime.now();
        System.out.println("現在時刻: " + now);  // 14:30:45.123

        // 特定の時刻を作成
        LocalTime lunchTime = LocalTime.of(12, 0);  // 12:00
        System.out.println("ランチタイム: " + lunchTime);

        // 時刻の計算
        LocalTime afterTwoHours = now.plusHours(2);  // 2時間後
        System.out.println("2時間後: " + afterTwoHours);
    }
}

immutable（不変）という特徴

LocalDateやLocalTimeは「immutable（イミュータブル）」、つまり一度作ったら内容を変更できません。 plusDays()などのメソッドは、元のオブジェクトを変更するのではなく、新しいオブジェクトを返します。

実習 13-2: 誕生日から経過日数を計算しよう

LocalDateを使って、誕生日から今日まで何日経過したかを計算するプログラムを作成しましょう。

AgeCalculator.java 
import java.time.LocalDate;
import java.time.Period;

public class AgeCalculator {
    public static void main(String[] args) {
        // 誕生日を設定（年、月、日）
        LocalDate birthday = LocalDate.of(2000, 5, 15);

        // 今日の日付を取得
        LocalDate today = LocalDate.now();

        // 期間を計算
        Period period = Period.between(birthday, today);

        // 結果を表示
        System.out.println("誕生日: " + birthday);
        System.out.println("今日: " + today);
        System.out.println("経過期間: " + period.getYears() + "年 "
                          + period.getMonths() + "ヶ月 "
                          + period.getDays() + "日");
    }
}

実行例

誕生日: 2000-05-15
今日: 2025-11-30
経過期間: 25年 6ヶ月 15日

13.4 数学計算を行う - Mathクラス

Mathクラスには、数学的な計算を行うためのメソッドがたくさん用意されています。 すべてstaticメソッドなので、 インスタンスを作らずに直接呼び出せます。

主要なMathメソッド

メソッド 説明 例
abs(x) 絶対値を返す Math.abs(-5) → 5
max(a, b) 2つの値の大きい方を返す Math.max(10, 20) → 20
min(a, b) 2つの値の小さい方を返す Math.min(10, 20) → 10
sqrt(x) 平方根を返す Math.sqrt(16) → 4.0
pow(a, b) aのb乗を返す Math.pow(2, 3) → 8.0
round(x) 四捨五入 Math.round(3.6) → 4
random() 0.0以上1.0未満の乱数 Math.random() → 0.7531...
MathExample.java 
public class MathExample {
    public static void main(String[] args) {
        // 絶対値
        int abs = Math.abs(-10);
        System.out.println("絶対値: " + abs);  // 10

        // 最大値・最小値
        int max = Math.max(50, 30);
        int min = Math.min(50, 30);
        System.out.println("最大値: " + max);  // 50
        System.out.println("最小値: " + min);  // 30

        // 平方根
        double sqrt = Math.sqrt(25);
        System.out.println("平方根: " + sqrt);  // 5.0

        // べき乗（2の10乗）
        double pow = Math.pow(2, 10);
        System.out.println("2の10乗: " + pow);  // 1024.0

        // 円周率
        System.out.println("π: " + Math.PI);  // 3.141592653589793
    }
}

乱数の生成

Math.random()は0.0以上1.0未満の小数を返します。 これを工夫することで、サイコロやくじ引きなど、さまざまな乱数を作れます。

DiceRoll.java 
public class DiceRoll {
    public static void main(String[] args) {
        // 1～6のサイコロを振る
        int dice = (int)(Math.random() * 6) + 1;
        System.out.println("サイコロの目: " + dice);

        // 1～100のランダムな整数
        int number = (int)(Math.random() * 100) + 1;
        System.out.println("ランダムな数: " + number);
    }
}

乱数の範囲を調整する公式

min～maxの範囲の整数を生成したいときは：
(int)(Math.random() * (max - min + 1)) + min

実習 13-3: じゃんけんゲームを作ろう

Math.random()を使って、コンピュータとじゃんけんをするプログラムを作成しましょう。

Janken.java 
public class Janken {
    public static void main(String[] args) {
        // 0:グー, 1:チョキ, 2:パー
        int player = 0;  // プレイヤーはグー
        int computer = (int)(Math.random() * 3);  // コンピュータはランダム

        // 手を表示
        String[] hands = {"グー", "チョキ", "パー"};
        System.out.println("あなた: " + hands[player]);
        System.out.println("コンピュータ: " + hands[computer]);

        // 勝敗判定
        if (player == computer) {
            System.out.println("引き分け！");
        } else if ((player == 0 && computer == 1) ||
                   (player == 1 && computer == 2) ||
                   (player == 2 && computer == 0)) {
            System.out.println("あなたの勝ち！");
        } else {
            System.out.println("コンピュータの勝ち！");
        }
    }
}

実行例

あなた: グー
コンピュータ: チョキ
あなたの勝ち！

13.5 その他の便利なクラス

StringBuilder - 文字列を効率よく結合

大量の文字列を連結する場合、通常の+演算子よりも StringBuilderを使う方が効率的です。

StringBuilderExample.java 
public class StringBuilderExample {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder();

        // 文字列を次々と追加
        sb.append("Hello");
        sb.append(" ");
        sb.append("Java");
        sb.append("!");

        // 文字列に変換
        String result = sb.toString();
        System.out.println(result);  // Hello Java!
    }
}

Random - より高機能な乱数生成

Randomクラスは、Math.random()よりも細かい制御ができる乱数生成クラスです。

RandomExample.java 
import java.util.Random;

public class RandomExample {
    public static void main(String[] args) {
        Random random = new Random();

        // 整数の乱数（0～99）
        int num = random.nextInt(100);
        System.out.println("整数: " + num);

        // true/falseのランダム
        boolean bool = random.nextBoolean();
        System.out.println("真偽値: " + bool);

        // 小数の乱数
        double d = random.nextDouble();
        System.out.println("小数: " + d);
    }
}

理解度確認クイズ

  1. Q1. "Hello World".length()の結果はいくつですか？

    回答を表示

    11です。スペースも1文字としてカウントされます（H-e-l-l-o- -W-o-r-l-d）。

  2. Q2. LocalDate.now()は何を返しますか？

    回答を表示

    今日の日付（LocalDateオブジェクト）を返します。実行した時点の年月日が含まれます。

  3. Q3. Math.max(10, 20)の結果はいくつですか？

    回答を表示

    20です。Math.max()は2つの引数のうち大きい方の値を返します。

  4. Q4. 1～10の範囲の整数乱数を生成するには？

    回答を表示

    (int)(Math.random() * 10) + 1 とします。Math.random()は0.0以上1.0未満なので、10を掛けて0～9にし、1を足すことで1～10になります。

13.6 この章のまとめ

  • 標準ライブラリは、Javaに最初から用意されている便利な道具セット。よく使う機能がクラスやメソッドとして提供されている。
  • Stringクラスには、文字列の長さ取得、検索、置換、大文字小文字変換など豊富なメソッドがある。
  • java.timeパッケージ（LocalDate、LocalTime）を使えば、日付や時刻の操作が簡単にできる。
  • Mathクラスには、絶対値、最大値、平方根、べき乗、乱数などの数学関数が揃っている。
  • StringBuilderは大量の文字列連結に便利。Randomクラスはより高機能な乱数生成ができる。

繰り返し処理 - ループ

同じ処理を何度も繰り返す方法を学びます。for文、while文を使って効率的にプログラムを書きましょう。

この章で学ぶこと

  • 繰り返し処理（ループ）の概念を理解する
  • for文を使って決まった回数だけ繰り返す方法
  • while文を使って条件が満たされる間繰り返す方法

4.1 ループって何？

「1から10まで表示する」プログラムを書くとき、System.out.println()を10回書くのは大変です。 ループ（繰り返し）を使えば、同じ処理を何度でも簡単に繰り返せます。

4.2 for文 - 回数が決まっている繰り返し

「何回繰り返すか」が決まっているときはfor文を使います。

for文の基本 
// 1から5まで表示
for (int i = 1; i <= 5; i++) {
    System.out.println(i + "回目");
}
// 出力: 1回目, 2回目, 3回目, 4回目, 5回目

for文の3つの部品

  • int i = 1 : 初期化（最初の値）
  • i <= 5 : 条件（繰り返す条件）
  • i++ : 増分（ループごとの変化）
九九の3の段 
// 九九の3の段を表示
for (int i = 1; i <= 9; i++) {
    int result = 3 * i;
    System.out.println("3 x " + i + " = " + result);
}

4.3 while文 - 条件が続く間繰り返す

「条件を満たしている間ずっと繰り返す」ときはwhile文を使います。

while文の基本 
int count = 0;

while (count < 5) {
    System.out.println("カウント: " + count);
    count++;  // これを忘れると無限ループ！
}

無限ループに注意！

while文の条件が永遠にtrueのままだと、プログラムが止まらなくなります。 ループ内で必ず条件が変わるようにしましょう。

4.4 breakとcontinue

ループの途中で抜けたり、スキップしたりするにはbreakとcontinueを使います。

breakとcontinueの例 
// break: 5で終了
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;  // ループを抜ける
    }
    System.out.println(i);  // 1, 2, 3, 4 のみ表示
}

// continue: 3をスキップ
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;  // 次のループへ
    }
    System.out.println(i);  // 1, 2, 4, 5 が表示（3なし）
}

実習 4-1: 合計を計算しよう

1から100までの合計を計算するプログラムを作りましょう。

Sum.java 
public class Sum {
    public static void main(String[] args) {
        int sum = 0;  // 合計を入れる変数

        for (int i = 1; i <= 100; i++) {
            sum = sum + i;  // sumにiを足していく
        }

        System.out.println("1から100の合計: " + sum);
    }
}

期待される出力: 1から100の合計: 5050

理解度確認クイズ

  1. Q1. for文の3つの部品は何ですか？

    回答を表示

    初期化（int i = 0）、条件（i < 10）、増分（i++）の3つです。

  2. Q2. ループを途中で抜けるキーワードは？

    回答を表示

    breakです。 continueは次のループにスキップするだけで、ループ自体は継続します。

4.5 この章のまとめ

  • ループを使うと同じ処理を効率的に繰り返せる
  • for文は回数が決まっている繰り返しに使う
  • while文は条件が続く間繰り返すときに使う
  • breakでループを抜け、continueで次の周へスキップできる

繰り返し処理（ループ）

for文、while文、do-while文を使った繰り返し処理について学びます。同じ処理を効率的に実行する方法や、ループの制御（break、continue）についても理解します。

この章で学ぶこと

  • 繰り返し処理とは何か、なぜ必要なのかを理解する
  • for文を使った回数指定の繰り返し処理
  • while文とdo-while文を使った条件による繰り返し処理
  • break文とcontinue文によるループ制御
  • ネストしたループ（二重ループ）の使い方と注意点

5.1 繰り返し処理とは

繰り返し処理（ループ）は、同じ処理を何度も実行する仕組みです。プログラミングでは、同じコードを何度も書くのではなく、ループを使って効率的に処理を行います。

例えば、「1から100までの数字を表示する」という処理を考えてみましょう。ループを使わない場合、System.out.printlnを100回書く必要がありますが、ループを使えば数行で実現できます。

繰り返し処理が使われる場面

配列やリストの要素を処理する、一定回数の処理を繰り返す、条件を満たすまで処理を続ける、など様々な場面で活用されます。データ処理や計算処理において必須の概念です。

5.2 for文による繰り返し

for文は、繰り返し回数が決まっている場合に最もよく使われるループです。カウンタ変数を使って、指定した回数だけ処理を繰り返します。

for文の基本構文

for文の構文 
for (初期化; 条件式; 更新処理) {
    // 繰り返し実行される処理
}
  • 初期化: カウンタ変数の初期値を設定（最初に一度だけ実行）
  • 条件式: ループを続けるかどうかの判定（各反復の前に評価）
  • 更新処理: カウンタ変数の更新（各反復の後に実行）

基本的な使用例

1から10までの数字を表示 
public class ForExample {
    public static void main(String[] args) {
        // 1から10まで表示
        for (int i = 1; i <= 10; i++) {
            System.out.println(i);
        }
    }
}

様々なfor文の書き方

様々なカウント方法 
// 10から1まで逆順にカウント
for (int i = 10; i >= 1; i--) {
    System.out.println(i);
}

// 2ずつ増やす（偶数のみ表示）
for (int i = 2; i <= 10; i += 2) {
    System.out.println(i);
}

// 0から開始（配列のインデックス向け）
for (int i = 0; i < 5; i++) {
    System.out.println("インデックス: " + i);
}

5.3 while文による繰り返し

while文は、条件が真である間繰り返しを続けるループです。繰り返し回数が事前に決まっていない場合によく使われます。

while文の基本構文

while文の構文 
while (条件式) {
    // 条件が真の間繰り返される処理
}

while文の使用例

カウントダウンプログラム 
public class WhileExample {
    public static void main(String[] args) {
        int count = 5;

        // countが0より大きい間繰り返す
        while (count > 0) {
            System.out.println("カウント: " + count);
            count--; // カウントを減らす
        }
        System.out.println("終了!");
    }
}

無限ループに注意

while文では、条件が永遠に真のままだと無限ループになります。必ず条件が偽になる仕組みを入れましょう。 

// 悪い例：無限ループ
int x = 1;
while (x > 0) {
    System.out.println(x); // xは減らないので永遠に続く
}

5.4 do-while文による繰り返し

do-while文は、最低1回は必ず実行され、その後条件を判定して繰り返すかどうかを決めるループです。while文とは条件判定のタイミングが異なります。

do-while文の基本構文

do-while文の構文 
do {
    // 必ず最低1回は実行される処理
} while (条件式);

whileとdo-whileの違い

動作の違いを比較 
// while文：条件が最初から偽なら1回も実行されない
int x = 0;
while (x > 0) {
    System.out.println("実行されない");
}

// do-while文：最低1回は実行される
int y = 0;
do {
    System.out.println("1回は実行される");
} while (y > 0);

5.5 break文とcontinue文

break文とcontinue文は、ループの流れを制御するための特別な命令です。条件に応じてループを途中で終了したり、一部の処理をスキップしたりできます。

break文：ループを抜ける

break文は、ループを即座に終了して次の処理に進みます。特定の条件で繰り返しを中断したい場合に使います。

break文の使用例 
public class BreakExample {
    public static void main(String[] args) {
        // 1から10まで探して、5を見つけたら終了
        for (int i = 1; i <= 10; i++) {
            System.out.println(i);
            if (i == 5) {
                System.out.println("5を見つけました！");
                break; // ここでループ終了
            }
        }
        System.out.println("ループ終了");
    }
}

continue文：次の反復へ

continue文は、現在の反復をスキップして次の反復に進みます。特定の条件の時だけ処理を飛ばしたい場合に使います。

continue文の使用例 
public class ContinueExample {
    public static void main(String[] args) {
        // 1から10までのうち、偶数だけ表示
        for (int i = 1; i <= 10; i++) {
            if (i % 2 != 0) {
                continue; // 奇数の場合はスキップ
            }
            System.out.println(i); // 偶数のみ表示
        }
    }
}

breakとcontinueの使い分け

  • • break: 目的のデータを見つけたら終了、エラーが発生したら中断など
  • • continue: 無効なデータをスキップ、特定条件の時だけ処理を省略など

5.6 ネストしたループ（二重ループ）

ループの中にさらにループを入れることをネスト（入れ子）といいます。二次元的なデータ処理や、複雑なパターンの出力によく使われます。

基本的な二重ループ

九九の表を作成 
public class MultiplicationTable {
    public static void main(String[] args) {
        // 九九の表を表示
        for (int i = 1; i <= 9; i++) {
            for (int j = 1; j <= 9; j++) {
                System.out.print(i * j + "\t");
            }
            System.out.println(); // 行の終わりで改行
        }
    }
}

星形パターンの出力

直角三角形パターン 
public class StarPattern {
    public static void main(String[] args) {
        // 直角三角形を描画
        for (int i = 1; i <= 5; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}

実習 5-1: 合計と平均を計算するプログラム

for文を使って、1から100までの整数の合計と平均を計算するプログラムを作成しましょう。

手順

  1. 1 合計を保存する変数sumを0で初期化
  2. 2 for文で1から100まで繰り返し、各数値をsumに加算
  3. 3 ループ終了後、合計と平均（合計÷100）を表示

実装例

public class SumAndAverage {
    public static void main(String[] args) {
        int sum = 0; // 合計を保存

        // 1から100まで加算
        for (int i = 1; i <= 100; i++) {
            sum += i;
        }

        // 平均を計算（小数点以下も表示）
        double average = sum / 100.0;

        System.out.println("合計: " + sum);
        System.out.println("平均: " + average);
    }
}

チャレンジ: 1から100までの偶数だけの合計を計算してみましょう（continueまたは条件分岐を活用）。

理解度確認クイズ

  1. Q1. for文とwhile文の使い分けについて正しいものはどれですか？

    回答を表示

    for文は繰り返し回数が決まっている場合、while文は条件によって回数が変わる場合に適しています。
    例: 配列の全要素を処理→for文、ユーザー入力が正しくなるまで繰り返す→while文

  2. Q2. 次のコードの出力結果は何ですか？

    for (int i = 0; i < 3; i++) {
    if (i == 1) continue;
    System.out.print(i + " ");
}
    回答を表示

    出力: 0 2
    i=1の時にcontinueが実行されるため、1は表示されずスキップされます。

  3. Q3. while文とdo-while文の最も大きな違いは何ですか？

    回答を表示

    do-while文は条件が最初から偽でも最低1回は実行されます。while文は条件が最初から偽なら1回も実行されません。
    do-while文は「まず実行してから判定」、while文は「判定してから実行」という違いがあります。

  4. Q4. 無限ループを作るコードの例として正しいものはどれですか？

    回答を表示

    while (true) { ... } や for (;;) { ... } など。
    無限ループは意図的に使う場合もありますが（サーバープログラムなど）、通常はbreak文などで抜ける仕組みが必要です。

この章のまとめ

  • 繰り返し処理（ループ）は同じ処理を効率的に実行するための仕組み
  • for文は繰り返し回数が決まっている場合に使い、初期化・条件式・更新処理の3要素で構成される
  • while文は条件が真の間繰り返し、do-while文は最低1回は必ず実行される
  • break文でループを途中終了、continue文で現在の反復をスキップできる
  • ネストしたループ（二重ループ）で二次元的なデータ処理やパターン出力が可能

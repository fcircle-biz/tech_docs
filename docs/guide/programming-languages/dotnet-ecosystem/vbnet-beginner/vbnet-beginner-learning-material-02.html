第2章

Visual Studioをインストールしよう

開発に必要なVisual Studioのダウンロードからインストールまでを、一つずつ丁寧に進めます。インストール時のオプション選択についても詳しく説明します。

この章で学ぶこと

  • Visual Studioとは何か、どのエディションを使うべきかを理解する
  • 自分のパソコンがインストール要件を満たしているか確認する
  • Visual Studio 2022のダウンロードとインストールを完了する
  • インストール後の初回起動と基本設定を行う

2.1 Visual Studioとは

Visual Studio（ビジュアル スタジオ）は、マイクロソフトが提供する統合開発環境（IDE）です。プログラムを書く、実行する、デバッグする、といった開発作業に必要な機能が、すべて一つのソフトウェアにまとまっています。

メモ帳でもプログラムを書くことはできますが、Visual Studioを使うと、以下のような便利な機能が使えます。

コード補完（インテリセンス）

入力途中で候補が表示され、タイプミスを防げます。初心者にとって非常に心強い機能です。

デバッグ機能

プログラムの実行を一時停止して、変数の中身を確認しながらエラーの原因を調べられます。

ビジュアルデザイナー

ボタンやテキストボックスをドラッグ＆ドロップで配置し、見た目を直感的に作れます。

リアルタイムエラー表示

コードを書いている最中に、間違いがあれば赤い波線で教えてくれます。

ポイント

Visual Studioは、プログラミングに必要な道具がすべて揃った「プログラミング専用のソフトウェア」です。これがあれば、すぐにVB.NETの学習を始められます。

2.2 どのエディションを選ぶか

Visual Studioには、3つのエディション（バージョン）があります。このガイドでは、Visual Studio 2022 Community Edition（無料版）を使用します。

Visual Studioの3つのエディション

Community Edition（推奨）

無料

個人開発者、学生、小規模チーム（5人まで）が無料で使える完全版です。学習目的やオープンソース開発には制限なく使用できます。

機能的には有料版とほぼ同じで、VB.NETの学習には十分すぎる性能です。

Professional Edition

有料

業務用の有料版です。チームコラボレーション機能や高度なデバッグツールが追加されています。企業での開発に使用されます。

Enterprise Edition

有料

大規模企業向けの最上位版です。アーキテクチャ分析、コードカバレッジなど、超大規模開発に必要な機能が含まれます。

重要ポイント

学習目的であれば、Community Editionで十分です。無料でありながら、プロの開発者も使う本格的な機能がすべて揃っています。

2.3 システム要件の確認

Visual Studioをインストールする前に、自分のパソコンが必要な条件を満たしているか確認しましょう。

Visual Studio 2022の推奨環境

オペレーティングシステム

Windows 10（バージョン 1909以降）または Windows 11（64ビット版）

プロセッサ

1.8 GHz以上のプロセッサ（4コア以上推奨）

メモリ

最低 4GB、推奨 16GB以上のRAM

ディスク容量

最低 20GB、推奨 50GB以上の空き容量

ディスプレイ

解像度 1366 x 768以上（推奨 1920 x 1080以上）

パソコンのスペック確認方法

Windowsで「設定」→「システム」→「バージョン情報」を開くと、OSのバージョン、プロセッサ、メモリの情報を確認できます。

2.4 Visual Studioのダウンロード

それでは、Visual Studio 2022 Community Editionをダウンロードしましょう。以下の手順に従って進めてください。

実習 2-1: Visual Studioをダウンロードする

公式サイトからインストーラーをダウンロードします。

手順

  1. 1 Webブラウザ（EdgeやChromeなど）を開き、アドレスバーに以下のURLを入力してEnterキーを押します。
    https://visualstudio.microsoft.com/ja/
  2. 2 Visual Studioの公式サイトが表示されます。ページ上部にある「ダウンロード」ボタンをクリックします。
  3. 3 3つのエディションが表示されます。「Community」の下にある「無料ダウンロード」ボタンをクリックします。
  4. 4 ブラウザの下部またはダウンロードフォルダに、「VisualStudioSetup.exe」というファイルがダウンロードされます（サイズは約3MB程度）。
  5. 5 ダウンロードが完了したら、次のステップに進みます。

注意

ダウンロードした「VisualStudioSetup.exe」は、本体ではなくインストーラー（小さなプログラム）です。実際のVisual Studioは、次のステップでこのインストーラーを実行することでダウンロード・インストールされます。

2.5 Visual Studioのインストール

ダウンロードしたインストーラーを実行し、Visual Studioをパソコンにインストールします。インストールには10〜30分程度かかることがあります。

実習 2-2: Visual Studioをインストールする

インストーラーを実行し、必要なコンポーネントを選択してインストールします。

手順

  1. 1 ダウンロードした「VisualStudioSetup.exe」をダブルクリックして実行します。
  2. 2 「ユーザーアカウント制御」ダイアログが表示されたら、「はい」をクリックします。これにより、インストーラーが起動します。
  3. 3 「Visual Studio Installer」が起動し、初回は準備作業が自動で行われます。数分待ちます。
  4. 4 「ワークロード」選択画面が表示されます。ここで、どの機能をインストールするかを選びます。
  5. 5 画面に表示されるワークロードの中から、「.NETデスクトップ開発」にチェックを入れます。これで、VB.NETを使ったWindowsアプリケーション開発に必要な機能がすべてインストールされます。
  6. 6 画面右下にある「インストール」ボタンをクリックします。
  7. 7 ライセンス条項が表示されるので、内容を確認して「同意する」をクリックします。
  8. 8 インストールが開始されます。進捗バーが表示され、必要なファイルが自動でダウンロード・インストールされます。この処理には、インターネット回線速度やパソコンの性能によって10〜30分程度かかります。
  9. 9 インストールが完了すると、「インストールが完了しました」というメッセージが表示されます。「起動」ボタンをクリックして、Visual Studioを起動します。

ヒント：ワークロードとは

「ワークロード」とは、開発の目的ごとにまとめられた機能セットのことです。たとえば「.NETデスクトップ開発」を選ぶと、Windowsアプリ開発に必要なツールがまとめてインストールされます。後から追加することもできますが、最初に必要なものを選んでおくとスムーズです。

2.6 初回起動と基本設定

インストールが完了したら、Visual Studioを起動して初期設定を行います。

初回起動の流れ

1

サインイン画面

Microsoftアカウントでサインインを求められます。アカウントがある場合はサインインし、ない場合は「後で行う」を選択できます。サインインすると、設定が複数のデバイスで同期されて便利ですが、必須ではありません。

2

開発設定の選択

「開発設定」のドロップダウンから「Visual Basic」を選択します。「配色テーマ」は、好みに応じて「青」「明るい」「濃色」「青（ハイコントラスト）」から選べます（後から変更可能です）。

3

Visual Studio起動

「Visual Studioの開始」ボタンをクリックすると、Visual Studioのスタートページが表示されます。ここから新しいプロジェクトを作成したり、既存のプロジェクトを開いたりできます。

重要ポイント

初回起動時に「Visual Basic」を開発設定として選択することで、VB.NET開発に最適化されたUIや機能配置になります。これにより、学習がスムーズに進みます。

よくあるトラブルと対処法

インストーラーのダウンロードが進まない

原因: インターネット接続が不安定、またはファイアウォールがダウンロードをブロックしている可能性があります。

解決方法: インターネット接続を確認し、セキュリティソフトやファイアウォールの設定で、Visual Studioのダウンロードを許可してください。別のネットワーク（モバイルホットスポットなど）で試すのも有効です。

インストール中にエラーが出る

原因: ディスク容量不足、または既存のVisual Studioと競合している可能性があります。

解決方法: ディスクの空き容量を確認し、不要なファイルを削除してください。既存のVisual Studioがある場合は、Visual Studio Installerから一度アンインストールしてから再インストールしてください。

「.NETデスクトップ開発」が見つからない

原因: ワークロード選択画面で、他のタブを見ている可能性があります。

解決方法: 画面上部のタブが「ワークロード」になっているか確認してください。「個別のコンポーネント」や「言語パック」ではなく、「ワークロード」タブから「.NETデスクトップ開発」を探してください。

起動時に「ライセンスの有効期限が切れました」と表示される

原因: Community Editionは30日間の試用期間があり、その後Microsoftアカウントでサインインが必要です。

解決方法: Microsoftアカウント（無料）を作成し、Visual Studioでサインインしてください。サインインすることで、無制限に使用できるようになります。

理解度確認クイズ

  1. Q1. Visual Studioの役割は何ですか？

    回答を表示

    Visual Studioは、プログラムを書く、実行する、デバッグするといった開発作業に必要な機能がすべてまとまった「統合開発環境（IDE）」です。コード補完、デバッグ機能、ビジュアルデザイナーなど、開発を効率化する多くの機能を提供します。

  2. Q2. このガイドで使用するVisual Studioのエディションと、その理由を説明してください。

    回答を表示

    Community Editionを使用します。理由は、無料でありながら有料版とほぼ同じ機能を持ち、学習目的には十分すぎる性能を持っているためです。個人開発者や学生が制限なく使用できます。

  3. Q3. インストール時に選択すべきワークロードは何ですか？また、ワークロードとは何ですか？

    回答を表示

    「.NETデスクトップ開発」を選択します。
    ワークロードとは、開発の目的ごとにまとめられた機能セットのことです。「.NETデスクトップ開発」を選ぶことで、VB.NETを使ったWindowsアプリケーション開発に必要なツールがすべてインストールされます。

第2章のまとめ

  • Visual Studioは、プログラミングに必要な機能がすべて揃った統合開発環境です。
  • Community Editionは無料で、学習や個人開発に最適です。
  • インストール時は「.NETデスクトップ開発」ワークロードを選択します。
  • 初回起動時に「Visual Basic」を開発設定として選択すると、VB.NET開発に最適化されます。

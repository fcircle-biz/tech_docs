変数とデータ型を理解しよう

データを一時的に保存する「変数」の概念と、数値や文字列などのデータ型について学びます。実際にプログラムで変数を使い、値の保存と取り出しを体験します。

この章で学ぶこと

  • 変数とは何か、なぜ必要なのかを理解する
  • VB.NETの主なデータ型（Integer、Double、String、Boolean、Date）を学ぶ
  • 変数の宣言方法と使い方を実践する
  • 適切なデータ型の選び方と命名ルールを理解する

5.1 変数とは何か

プログラミングにおいて、変数とは「データを一時的に保存しておく箱」のようなものです。この箱には名前をつけることができ、必要なときにデータを入れたり、取り出したりできます。

たとえば、ユーザーが入力した年齢を覚えておいて、後で計算に使いたい場合、その年齢を変数に保存します。変数がなければ、データを何度も入力し直す必要があり、プログラムは実用的になりません。

ポイント

変数は「データに名前をつけて、後で使えるようにする仕組み」です。この仕組みがあることで、プログラムは複雑な処理を実現できます。

5.2 変数の宣言と使い方

VB.NETで変数を使うには、まず「この変数を使います」と宣言する必要があります。これを変数宣言と呼びます。

基本的な変数宣言の構文

変数宣言の基本構文 
' 基本構文: Dim 変数名 As データ型
Dim age As Integer         ' 整数を保存する変数
Dim name As String          ' 文字列を保存する変数
Dim price As Double         ' 小数を保存する変数

変数宣言の要素

Dim

「変数を宣言します」という意味のキーワード（Dimension = 次元・領域の略）

変数名

変数につける名前。後でこの名前を使ってデータを取り出します

As

「～として」という意味のキーワード

データ型

この変数に保存するデータの種類（整数、文字列など）

変数への値の代入

変数を宣言したら、次に値を入れます。これを代入と呼びます。

変数への代入例 
' 変数を宣言
Dim age As Integer
Dim name As String

' 値を代入
age = 25
name = "田中太郎"

' 宣言と同時に代入することもできます
Dim score As Integer = 85

重要ポイント

等号（=）は「等しい」という意味ではなく、「右側の値を左側の変数に入れる」という意味です。数学の等号とは違う使い方なので注意しましょう。

5.3 VB.NETの主なデータ型

データ型とは、変数に保存できるデータの種類を指定するものです。適切なデータ型を選ぶことで、メモリを効率的に使い、プログラムのミスを防ぐことができます。

Integer（整数型）

整数を保存

小数点のない整数を保存します。年齢、個数、回数など、整数で表せるデータに使います。

例:

Dim age As Integer = 30
Dim count As Integer = 100

範囲: -2,147,483,648 ～ 2,147,483,647

Double（実数型）

小数を保存

小数点を含む数値を保存します。価格、割合、計算結果など、小数が必要なデータに使います。

例:

Dim price As Double = 1980.5
Dim rate As Double = 0.08

非常に大きな範囲の実数を扱えます

String（文字列型）

文字列を保存

文字や文章を保存します。名前、住所、メッセージなど、テキストデータに使います。値はダブルクォーテーション（"）で囲みます。

例:

Dim name As String = "山田太郎"
Dim message As String = "こんにちは"

約20億文字まで保存可能

Boolean（真偽値型）

True/Falseを保存

True（真）またはFalse（偽）のどちらかを保存します。フラグ、チェック状態など、2つの状態を表すデータに使います。

例:

Dim isActive As Boolean = True
Dim hasError As Boolean = False

値はTrueまたはFalseのみ

Date（日付型）

日付・時刻を保存

日付と時刻を保存します。誕生日、予定日、タイムスタンプなど、日時データに使います。

例:

Dim birthday As Date = #1/15/1990#
Dim today As Date = Date.Today

#記号で囲んで日付を指定

5.4 実習: 変数を使ってみよう

実習 5-1: さまざまなデータ型の変数を使う

異なるデータ型の変数を宣言して、値を代入し、コンソールに表示してみましょう。

手順

  1. 1 Visual Studioで新しいコンソールアプリケーションプロジェクトを作成
  2. 2 以下のコードを入力
Module1.vb 
Module Module1
    Sub Main()
        ' 整数型の変数
        Dim age As Integer = 25
        Console.WriteLine("年齢: " & age & "歳")

        ' 実数型の変数
        Dim height As Double = 170.5
        Console.WriteLine("身長: " & height & "cm")

        ' 文字列型の変数
        Dim name As String = "田中太郎"
        Console.WriteLine("名前: " & name)

        ' 真偽値型の変数
        Dim isStudent As Boolean = True
        Console.WriteLine("学生: " & isStudent)

        ' 日付型の変数
        Dim birthday As Date = #3/15/2000#
        Console.WriteLine("誕生日: " & birthday)

        ' キー入力待機
        Console.ReadLine()
    End Sub
End Module
  1. 3 F5キーを押してプログラムを実行

動作確認

プログラムを実行すると、以下のような結果が表示されるはずです。

期待される結果 

年齢: 25歳
身長: 170.5cm
名前: 田中太郎
学生: True
誕生日: 2000/03/15

5.5 変数の命名ルール

変数名は自由につけられますが、読みやすく分かりやすいコードを書くために、いくつかのルールと慣習があります。

必ず守るべきルール

1. 使える文字

アルファベット（A-Z, a-z）、数字（0-9）、アンダースコア（_）のみ

OK: age, userName, score_1

2. 先頭は文字またはアンダースコア

数字から始めることはできません

NG: 1age, 99score

3. 予約語は使えない

VB.NETが特別な意味で使う単語（Dim、Sub、Ifなど）は変数名にできません

NG: Dim, If, Sub

4. 大文字・小文字を区別しない

age、Age、AGEはすべて同じ変数として扱われます

推奨される命名規則

意味のある名前をつける

変数が何を表すのか、名前から分かるようにします

良い例: studentAge 悪い例: a

キャメルケースを使う

複数の単語を組み合わせるときは、2つ目以降の単語の先頭を大文字にします

例: firstName, totalPrice, isActive

短すぎず、長すぎない

一般的に2～20文字程度が読みやすいとされています

つまずきポイント

日本語の変数名も使えますが、チーム開発や将来的なことを考えると、英語での命名を推奨します。ただし、学習段階では分かりやすさを優先して日本語を使っても構いません。

5.6 型変換

異なるデータ型の変数同士を組み合わせて計算したり、ユーザーからの入力（文字列）を数値に変換したりする場合、型変換が必要になります。

主な型変換関数

CInt() - 整数に変換

文字列や小数を整数型（Integer）に変換します

Dim num As Integer = CInt("123")

CDbl() - 実数に変換

文字列や整数を実数型（Double）に変換します

Dim price As Double = CDbl("1980.5")

CStr() - 文字列に変換

数値や日付を文字列型（String）に変換します

Dim text As String = CStr(123)

CBool() - 真偽値に変換

数値や文字列を真偽値型（Boolean）に変換します

Dim flag As Boolean = CBool(1) ' Trueになる

型変換の実例

型変換の使用例 
Module Module1
    Sub Main()
        ' 文字列を整数に変換
        Dim ageText As String = "25"
        Dim age As Integer = CInt(ageText)
        Console.WriteLine("年齢: " & age)

        ' 整数を文字列に変換
        Dim count As Integer = 100
        Dim message As String = "商品数: " & CStr(count) & "個"
        Console.WriteLine(message)

        ' 文字列を実数に変換
        Dim priceText As String = "1980.5"
        Dim price As Double = CDbl(priceText)
        Console.WriteLine("価格: " & price & "円")

        Console.ReadLine()
    End Sub
End Module

注意

数値に変換できない文字列（例: "abc"）をCInt()やCDbl()で変換しようとすると、エラーが発生します。ユーザー入力を変換する場合は、エラー処理が必要です（後の章で学習します）。

理解度確認クイズ

  1. Q1. 変数とは何ですか？簡単に説明してください。

    回答を表示

    変数とは、データを一時的に保存しておく「箱」のようなものです。名前をつけることで、後でそのデータを取り出したり、計算に使ったりできます。

  2. Q2. 以下のデータに適したデータ型を選んでください。
    (a) 商品の価格（1980.5円）
    (b) ユーザーの名前（"山田太郎"）
    (c) 在庫数（50個）
    (d) ログイン状態（ログイン中/未ログイン）

    回答を表示

    (a) Double型（小数を扱うため）
    (b) String型（文字列を扱うため）
    (c) Integer型（整数を扱うため）
    (d) Boolean型（True/Falseの2状態を扱うため）

  3. Q3. 次の変数名のうち、VB.NETで使えないものはどれですか？
    (a) userName
    (b) 1stPlace
    (c) total_score
    (d) If

    回答を表示

    (b) 1stPlace → 数字から始まっているため使えません
    (d) If → VB.NETの予約語（条件分岐で使うキーワード）のため使えません

    (a) userName と (c) total_score は使えます。

第5章のまとめ

  • 変数は、データを一時的に保存する「箱」で、名前をつけて後で使うことができます。
  • VB.NETの主なデータ型には、Integer（整数）、Double（実数）、String（文字列）、Boolean（真偽値）、Date（日付）があります。
  • 変数を宣言するには「Dim 変数名 As データ型」の構文を使います。
  • 変数名は意味のある名前をつけ、キャメルケースを使うことが推奨されます。
  • 型変換関数（CInt、CDbl、CStrなど）を使って、データ型を変換できます。

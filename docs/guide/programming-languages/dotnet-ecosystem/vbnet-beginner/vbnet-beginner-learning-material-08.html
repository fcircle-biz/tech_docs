第8章

繰り返し処理をマスターしよう

For文、While文、Do Loop文を使った繰り返し処理を学びます。同じ処理を効率的に繰り返す方法と、無限ループを避けるためのポイントを理解します。

この章で学ぶこと

  • 繰り返し処理の概念と、プログラムで使う意味を理解する
  • For文を使った回数指定の繰り返しをマスターする
  • While文とDo Loop文の違いを理解し、使い分けられるようになる
  • 無限ループを避けるためのポイントを学ぶ

8.1 繰り返し処理とは

繰り返し処理（ループ）とは、同じ処理を何度も実行するための仕組みです。たとえば、「1から10までの数字を表示する」というプログラムを考えてみましょう。繰り返し処理を使わないと、以下のように同じコードを10回書く必要があります。

繰り返しを使わない場合（非効率） 
' 繰り返しを使わない場合
Console.WriteLine(1)
Console.WriteLine(2)
Console.WriteLine(3)
' ... 10まで続く（コードが長い！）

繰り返し処理を使うと、わずか数行のコードで同じ処理を効率的に実行できます。

ポイント

繰り返し処理は、同じコードを何度も書かずに済むため、プログラムを短く、読みやすくできます。プログラマーにとって必須のスキルです。

8.2 For文の基本

For文は、繰り返す回数が決まっているときに使う構文です。「iが1から10まで、1ずつ増えながら繰り返す」といった処理を簡潔に書けます。

For文の基本構文

For 変数 = 開始値 To 終了値 Step 増分
    ' 繰り返したい処理
Next 変数
  • 変数: 繰り返しの回数を管理する変数（カウンタ変数）
  • 開始値: 変数の最初の値
  • 終了値: 変数がこの値以下の間、繰り返しを続ける
  • Step 増分: 変数の増加量（省略時は1）

実習 8-1: For文で1から10まで表示する

For文を使って、1から10までの数字を表示するプログラムを作成しましょう。

手順

  1. 1 新しいコンソールアプリケーションを作成します
  2. 2 以下のコードを入力します
Module1.vb 
Module Module1
    Sub Main()
        ' 1から10まで繰り返す
        For i As Integer = 1 To 10
            Console.WriteLine(i)
        Next i

        Console.ReadLine()
    End Sub
End Module

動作確認

F5キーを押してプログラムを実行します。

重要ポイント

For文のカウンタ変数（ここではi）は、繰り返しごとに自動的に増えていきます。手動で「i = i + 1」と書く必要はありません。

8.3 Step を使った増分の変更

Stepキーワードを使うと、カウンタ変数の増加量を変更できます。たとえば、2ずつ増やすことで偶数だけを表示したり、マイナスにすることでカウントダウンしたりできます。

例1: 2ずつ増やす（偶数） 

For i As Integer = 2 To 10 Step 2
    Console.WriteLine(i)
Next i
' 出力: 2, 4, 6, 8, 10

例2: カウントダウン（-1ずつ） 

For i As Integer = 10 To 1 Step -1
    Console.WriteLine(i)
Next i
' 出力: 10, 9, 8, ..., 1

ヒント

Stepを省略すると、自動的に「Step 1」として扱われます。ほとんどの場合は1ずつ増やすので、省略して書くことが多いです。

8.4 While文の基本

While文は、条件がTrueである限り繰り返しを続ける構文です。「あらかじめ回数が決まっていない」ときに便利です。たとえば、「ユーザーが正しい値を入力するまで繰り返す」といった処理に向いています。

While文の基本構文

While 条件式
    ' 条件式がTrueの間、繰り返したい処理
End While

実習 8-2: While文で1から10まで表示する

While文を使って、1から10までの数字を表示するプログラムを作成しましょう。

Module1.vb 
Module Module1
    Sub Main()
        Dim i As Integer = 1

        ' iが10以下の間、繰り返す
        While i <= 10
            Console.WriteLine(i)
            i = i + 1  ' カウンタを自分で増やす必要あり
        End While

        Console.ReadLine()
    End Sub
End Module

注意

While文では、カウンタ変数を自分で増やす必要があります。「i = i + 1」を忘れると、iが永遠に1のまま、無限ループになってしまいます。

8.5 Do Loop文の基本

Do Loop文は、While文と似ていますが、条件判定のタイミングを選べます。条件を最初に判定する（Do While～Loop）か、最後に判定する（Do～Loop While）かで動作が変わります。

Do While～Loop

最初に条件を判定してから処理を実行します。条件がFalseなら、1回も実行されません。

Do While 条件式
    ' 処理
Loop

Do～Loop While

処理を実行してから条件を判定します。条件がFalseでも、最低1回は実行されます。

Do
    ' 処理
Loop While 条件式

実習 8-3: Do Loop文でパスワード入力を求める

正しいパスワードが入力されるまで繰り返し入力を求めるプログラムを作成しましょう。

Module1.vb 
Module Module1
    Sub Main()
        Dim correctPassword As String = "VB2025"
        Dim input As String = ""

        ' 正しいパスワードが入力されるまで繰り返す
        Do While input <> correctPassword
            Console.Write("パスワードを入力してください: ")
            input = Console.ReadLine()

            If input <> correctPassword Then
                Console.WriteLine("パスワードが違います。")
            End If
        Loop

        Console.WriteLine("ログインしました！")
        Console.ReadLine()
    End Sub
End Module

重要ポイント

Do Loop文は、「少なくとも1回は実行したい」場合に「Do～Loop While」を、「条件が満たされないなら1回も実行しない」場合に「Do While～Loop」を選びます。

8.6 Exit文による途中終了

繰り返し処理の途中で、特定の条件が満たされたら繰り返しを抜けたいことがあります。そのときに使うのがExit文です。

Exit文の種類

  • Exit For

    For文のループを抜けます

  • Exit While

    While文のループを抜けます

  • Exit Do

    Do Loop文のループを抜けます

例: Exit Forで途中終了 
For i As Integer = 1 To 100
    Console.WriteLine(i)

    ' iが5になったらループを抜ける
    If i = 5 Then
        Exit For
    End If
Next i
' 出力: 1, 2, 3, 4, 5（6以降は表示されない）

ヒント

Exit文を使うと、条件を満たした時点ですぐにループを抜けられます。無駄な処理を省いて、プログラムを効率的にできます。

8.7 無限ループに注意

繰り返し処理で最も注意すべきなのが無限ループです。これは、終了条件が永遠に満たされず、プログラムが止まらなくなる状態のことです。

無限ループの例（悪い例）

Dim i As Integer = 1

While i <= 10
    Console.WriteLine(i)
    ' iを増やすのを忘れている！
End While
' iが永遠に1のまま、ループが終わらない

無限ループを避けるためのチェックリスト

  • While文やDo Loop文では、カウンタ変数を必ず増やす（または減らす）処理を書く
  • 終了条件が必ず満たされるようにする
  • For文を使える場合は、For文を優先する（自動でカウンタが増えるため安全）
  • 万が一無限ループに入ったら、Ctrl + C または Visual Studioの「停止」ボタンで強制終了

つまずきポイント

初心者が最もつまずくのは、While文やDo Loop文でカウンタ変数の増減を忘れることです。繰り返しごとに変数が変化しているか、必ず確認しましょう。

8.8 繰り返し処理の使い分け

For文、While文、Do Loop文、それぞれに適した使い場面があります。状況に応じて使い分けることで、読みやすく、バグの少ないプログラムを書けます。

For文

使う場面:

  • ・繰り返す回数が決まっているとき
  • ・配列の要素を順番に処理するとき

例: 1から100までの数字を表示

While文

使う場面:

  • ・条件を最初に判定したいとき
  • ・繰り返し回数が事前に分からないとき

例: ユーザーが正しい値を入力するまで繰り返す

Do Loop文

使う場面:

  • ・必ず1回は実行したいとき
  • ・条件判定のタイミングを選びたいとき

例: メニューを表示してユーザーの選択を待つ

重要ポイント

迷ったら、まずFor文で書けないか考えましょう。For文が最も安全で、無限ループになりにくいです。

理解度確認クイズ

  1. Q1. For文で「1から50まで、5ずつ増やす」繰り返しを書くには、どのように書けばよいですか？

    回答を表示

    For i As Integer = 1 To 50 Step 5
    Step 5 を指定することで、iが1, 6, 11, 16, ...と5ずつ増えていきます。

  2. Q2. While文で無限ループを避けるために、必ずやるべきことは何ですか？

    回答を表示

    カウンタ変数を増やす（または減らす）処理を忘れずに書くことです。たとえば「i = i + 1」などの処理がないと、条件が永遠に満たされず、ループが終わりません。

  3. Q3. 「ユーザーがメニューから0を選ぶまで繰り返す」場合、For文、While文、Do Loop文のどれが最適ですか？

    回答を表示

    Do Loop文（特にDo～Loop While）が最適です。メニューは必ず1回は表示する必要があり、繰り返す回数は事前に分からないためです。While文でも可能ですが、Do Loop文の方が「最低1回実行」という意図が明確になります。

第8章のまとめ

  • 繰り返し処理を使うと、同じコードを何度も書かずに効率的にプログラムを書けます。
  • For文は回数が決まっているとき、While文とDo Loop文は条件で繰り返すときに使います。
  • Exit文を使うと、特定の条件でループを途中終了できます。
  • While文やDo Loop文では、カウンタ変数の増減を忘れると無限ループになるので注意が必要です。
  • 迷ったら、まずFor文で書けないか考えると安全です。

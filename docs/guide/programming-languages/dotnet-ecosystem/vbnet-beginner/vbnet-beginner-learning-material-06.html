演算子と計算をしてみよう

足し算、引き算などの算術演算子や、文字列の連結など、プログラムでの計算方法を学びます。ユーザーからの入力を受け取り、計算結果を表示するプログラムを作成します。

この章で学ぶこと

  • 算術演算子を使った四則演算とその他の計算方法
  • 文字列連結演算子を使った文字列の結合
  • 演算子の優先順位と正しい計算順序の理解
  • ユーザー入力を受け取り、計算結果を表示する実践的なプログラムの作成

6.1 算術演算子とは

算術演算子とは、数値の計算を行うための記号です。私たちが普段使っている「+」「-」「×」「÷」などの計算記号を、プログラムでは専用の記号で表現します。VB.NETで使える主な算術演算子を見ていきましょう。

主な算術演算子

+ 加算（足し算）

2つの数値を足し合わせます。例: 5 + 3 = 8

- 減算（引き算）

左の数値から右の数値を引きます。例: 10 - 4 = 6

* 乗算（掛け算）

2つの数値を掛け合わせます。例: 3 * 4 = 12

/ 除算（割り算・小数）

左の数値を右の数値で割ります。例: 10 / 4 = 2.5

\ 整数除算（割り算・整数）

小数点以下を切り捨てた整数の結果。例: 10 \ 4 = 2

Mod 剰余（余り）

割り算の余りを求めます。例: 10 Mod 3 = 1

^ 累乗（べき乗）

左の数値を右の数値で累乗します。例: 2 ^ 3 = 8

注意

掛け算は「×」ではなく「*」（アスタリスク）、割り算は「÷」ではなく「/」（スラッシュ）を使います。これはプログラミング言語の共通ルールです。

6.2 算術演算子を使った計算

実際に算術演算子を使った計算のコードを見ていきましょう。変数を使って、計算結果を保存できます。

算術演算子の基本例 
Module Module1
    Sub Main()
        ' 変数を宣言
        Dim a As Integer = 10
        Dim b As Integer = 3
        Dim result As Integer

        ' 加算（足し算）
        result = a + b
        Console.WriteLine("10 + 3 = " & result)  ' 結果: 13

        ' 減算（引き算）
        result = a - b
        Console.WriteLine("10 - 3 = " & result)  ' 結果: 7

        ' 乗算（掛け算）
        result = a * b
        Console.WriteLine("10 * 3 = " & result)  ' 結果: 30

        ' 整数除算（割り算・整数）
        result = a \ b
        Console.WriteLine("10 \ 3 = " & result)  ' 結果: 3

        ' 剰余（余り）
        result = a Mod b
        Console.WriteLine("10 Mod 3 = " & result)  ' 結果: 1

        Console.ReadLine()
    End Sub
End Module

ポイント

「&」記号は文字列を連結する演算子です。計算結果を文字列と組み合わせて表示できます。

実行結果

10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 \ 3 = 3
10 Mod 3 = 1

小数点を含む除算

通常の割り算「/」を使うと、小数点以下も含めた結果が得られます。このとき、変数のデータ型をDoubleにする必要があります。

小数点を含む除算 
Module Module1
    Sub Main()
        Dim a As Double = 10
        Dim b As Double = 3
        Dim result As Double

        ' 通常の除算（小数点を含む）
        result = a / b
        Console.WriteLine("10 / 3 = " & result)  ' 結果: 3.333...

        Console.ReadLine()
    End Sub
End Module

6.3 文字列連結演算子

文字列連結演算子「&」は、複数の文字列を1つにつなげるために使います。メッセージを表示するときに、変数の値を文字列と組み合わせて表示する場合に便利です。

文字列連結の例 
Module Module1
    Sub Main()
        ' 文字列変数を宣言
        Dim firstName As String = "太郎"
        Dim lastName As String = "山田"

        ' 文字列を連結
        Dim fullName As String = lastName & firstName
        Console.WriteLine("フルネーム: " & fullName)  ' 結果: 山田太郎

        ' 数値と文字列を連結
        Dim age As Integer = 25
        Console.WriteLine("名前: " & fullName & "、年齢: " & age & "歳")

        Console.ReadLine()
    End Sub
End Module

重要ポイント

「&」は数値と文字列の連結にも使えます。数値は自動的に文字列に変換されます。「+」も連結に使えますが、意図しない計算が発生する可能性があるため、文字列連結には「&」を使うことが推奨されます。

6.4 演算子の優先順位

複数の演算子を組み合わせて計算するとき、演算子の優先順位が重要になります。数学と同じように、掛け算や割り算は足し算や引き算より先に計算されます。

演算子の優先順位（高い順）

  1. 累乗: ^ （最優先）
  2. 符号: +（正）、-（負）
  3. 乗算・除算: *、/、\、Mod
  4. 加算・減算: +、-
  5. 文字列連結: & （最後）
優先順位の例 
Module Module1
    Sub Main()
        Dim result As Integer

        ' 掛け算が先に計算される
        result = 5 + 3 * 2
        Console.WriteLine("5 + 3 * 2 = " & result)  ' 結果: 11（3*2=6、5+6=11）

        ' カッコを使って計算順序を変更
        result = (5 + 3) * 2
        Console.WriteLine("(5 + 3) * 2 = " & result)  ' 結果: 16（5+3=8、8*2=16）

        ' 複雑な計算
        result = 10 + 2 * 3 - 4
        Console.WriteLine("10 + 2 * 3 - 4 = " & result)  ' 結果: 12

        Console.ReadLine()
    End Sub
End Module

ポイント

計算の順序を明確にするために、カッコ「( )」を使いましょう。カッコ内の計算が最優先で実行されます。

6.5 ユーザー入力と計算

実用的なプログラムでは、ユーザーから数値を入力してもらい、その値を使って計算します。Console.ReadLine()で入力を受け取り、Integer.Parse()やDouble.Parse()で数値に変換します。

ユーザー入力を使った計算 
Module Module1
    Sub Main()
        ' ユーザーに1つ目の数値を入力してもらう
        Console.Write("1つ目の数値を入力してください: ")
        Dim input1 As String = Console.ReadLine()
        Dim num1 As Integer = Integer.Parse(input1)

        ' ユーザーに2つ目の数値を入力してもらう
        Console.Write("2つ目の数値を入力してください: ")
        Dim input2 As String = Console.ReadLine()
        Dim num2 As Integer = Integer.Parse(input2)

        ' 計算結果を表示
        Console.WriteLine("--- 計算結果 ---")
        Console.WriteLine("足し算: " & (num1 + num2))
        Console.WriteLine("引き算: " & (num1 - num2))
        Console.WriteLine("掛け算: " & (num1 * num2))
        Console.WriteLine("割り算: " & (num1 / num2))

        Console.ReadLine()
    End Sub
End Module

実行例

1つ目の数値を入力してください: 20
2つ目の数値を入力してください: 4
--- 計算結果 ---
足し算: 24
引き算: 16
掛け算: 80
割り算: 5

注意

Console.ReadLine()は必ず文字列（String型）を返します。数値として計算するには、Integer.Parse()やDouble.Parse()で変換する必要があります。

実習課題

課題1: 消費税を含む金額を計算

商品の税抜き価格をユーザーに入力してもらい、消費税10%を加えた金額を表示するプログラムを作成してください。

ヒントを表示
  • 税抜き価格を入力: Console.ReadLine()
  • 文字列を数値に変換: Integer.Parse()
  • 消費税額 = 税抜き価格 * 0.1
  • 税込み価格 = 税抜き価格 + 消費税額

課題2: 平均点を計算

3つのテストの点数を入力してもらい、平均点を小数点第1位まで表示するプログラムを作成してください。

ヒントを表示
  • 変数のデータ型をDoubleにする
  • 平均 = (点数1 + 点数2 + 点数3) / 3
  • Math.Round(平均, 1)で小数点第1位まで四捨五入

課題3: 円の面積を計算

円の半径を入力してもらい、円の面積を計算して表示するプログラムを作成してください。（円周率は3.14として計算）

ヒントを表示
  • 円の面積 = 半径 * 半径 * 3.14
  • 累乗演算子「^」を使う場合: 半径 ^ 2 * 3.14
  • 変数のデータ型をDoubleにする

よくあるエラーと対処法

エラー: "型 'String' の値を 'Integer' に変換できません"

原因: 数値以外の文字（アルファベットなど）が入力されている

対処法: 入力時に数値のみを入力してください。次章で学ぶエラー処理を使えば、入力チェックができます。

エラー: "0 による除算が試行されました"

原因: 割り算の右側（除数）が0になっている

対処法: 0以外の数値を入力してください。プログラム側で0チェックを追加することもできます。

計算結果が予想と違う

原因: 演算子の優先順位、または整数除算と通常の除算の違い

対処法: カッコ「( )」を使って計算順序を明確にする。小数点が必要な場合は「/」を使い、データ型をDoubleにする。

理解度確認クイズ

  1. Q1. 次のコードの実行結果は何ですか？ Console.WriteLine(10 + 5 * 2)

    回答を表示

    答え: 20
    理由: 掛け算が先に計算されます。5 * 2 = 10、その後 10 + 10 = 20となります。

  2. Q2. 「\」と「/」の違いは何ですか？

    回答を表示

    答え:
    「\」は整数除算で、小数点以下を切り捨てた整数の結果を返します（例: 10 \ 3 = 3）。
    「/」は通常の除算で、小数点を含む結果を返します（例: 10 / 3 = 3.333...）。

  3. Q3. 文字列連結に使う演算子は何ですか？また、なぜ「+」ではなくその演算子を使うべきですか？

    回答を表示

    答え: 「&」演算子
    理由: 「+」は数値の加算と文字列の連結の両方の意味を持つため、意図しない計算が発生する可能性があります。「&」を使うことで、明確に文字列連結であることを示せます。

  4. Q4. Console.ReadLine()で入力された値を数値として使うには、どうすればよいですか？

    回答を表示

    答え:
    Integer.Parse()またはDouble.Parse()を使って、文字列を数値に変換する必要があります。例: Dim num As Integer = Integer.Parse(Console.ReadLine())

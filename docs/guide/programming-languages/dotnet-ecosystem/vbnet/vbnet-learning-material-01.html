第1章: VB.NET入門と開発環境構築

VB.NETの世界へようこそ

  • 初めてのVB.NETプログラムを作成・実行する

1.1 VB.NETとは何か

Visual Basic .NET（VB.NET）は、Microsoftが開発したプログラミング言語です。「Visual Basic」という名前の通り、視覚的で分かりやすい文法を持っており、プログラミング初心者にとって親しみやすい言語として設計されています。

VB.NETの歴史と背景

VB.NETは、従来のVisual Basic（VB6）を.NET Frameworkに対応させた現代的なプログラミング言語です。2002年に初めてリリースされ、現在も継続的にアップデートが行われています。

sequenceDiagram participant VB6 as Visual Basic 6 participant NET as .NET Framework participant VBNET as VB.NET participant Modern as 現代のVB.NET VB6->>NET: 2000年頃: .NET登場 NET->>VBNET: 2002年: VB.NET誕生 VBNET->>Modern: 継続的進化 Note over VB6,Modern: プログラミング言語の進化

VB.NETの主な特徴

  • 読みやすい構文: 英語に近い自然な文法でコードが理解しやすい
  • 強力な型システム: データの型安全性を保証し、バグを減らす
  • 豊富なライブラリ: .NET Frameworkの膨大なライブラリを活用可能
  • RAD（Rapid Application Development）: 素早いアプリケーション開発が可能
  • マルチプラットフォーム: Windows、Linux、macOSで動作する

1.2 .NET Frameworkとは

.NET Frameworkは、Microsoftが開発したアプリケーション実行環境です。VB.NETプログラムを動かすための土台となる重要なシステムです。

.NET Frameworkの仕組み

.NET Frameworkは、プログラムの実行を管理し、メモリ管理、セキュリティ、ネットワーク通信など、アプリケーションが必要とする基本機能を提供します。

.NET Framework の主要コンポーネント

  • CLR（Common Language Runtime）: プログラムの実行を管理
  • Base Class Library: 基本的なクラスとメソッドの集合
  • ASP.NET: Webアプリケーション開発フレームワーク
  • Windows Forms: デスクトップアプリケーション開発フレームワーク
  • ADO.NET: データベース接続とデータ操作

1.3 Visual Studio開発環境の構築

VB.NETでプログラム開発を行うには、Visual Studioという統合開発環境（IDE）を使用します。Visual Studioは、コードの記述、デバッグ、テスト、配布まで、ソフトウェア開発に必要なすべての機能を統合した強力なツールです。

Visual Studioエディションの比較

エディション 価格 主な用途 特徴
Community 無料 学習・個人開発 基本機能すべて利用可能
Professional 有料 商用開発 高度なデバッグ・テスト機能
Enterprise 高額 大規模開発 チーム開発・アーキテクチャツール

推奨: 学習目的であれば、無料のVisual Studio Communityで十分です。すべての基本機能が利用でき、本格的なアプリケーション開発も可能です。

1.4 初めてのVB.NETプロジェクト

Visual Studioを使用して、最初のVB.NETプログラムを作成してみましょう。プログラミングの世界では、新しい言語を学ぶときに「Hello World」プログラムを作るのが伝統です。

プロジェクトとは

プロジェクトとは、アプリケーションを構成するすべてのファイル（ソースコード、設定ファイル、画像など）をまとめて管理する単位です。Visual Studioでは、プロジェクトを作成してからプログラミングを開始します。

実習 1-1: Hello World プログラムの作成

VB.NETで最初のプログラムを作成し、プログラミングの基本的な流れを体験しましょう。

手順
  1. Visual Studioを起動
  2. 「新しいプロジェクトの作成」を選択
  3. 「コンソール アプリケーション」を選択
  4. 言語として「Visual Basic」を選択
  5. プロジェクト名を「HelloWorld」に設定
  6. 「作成」をクリック
作成されるコード例
Module Program
    Sub Main(args As String())
        Console.WriteLine("Hello World!")
        Console.ReadKey()
    End Sub
End Module
期待される結果

プログラムを実行すると、コンソールウィンドウに「Hello World!」と表示され、キーを押すまで待機状態になります。

コードの詳細解説

  • Module Program: プログラムの開始点となるモジュールの宣言
  • Sub Main: プログラムが実行される最初のサブルーチン（メソッド）
  • Console.WriteLine: コンソールに文字列を出力するメソッド
  • Console.ReadKey: ユーザーのキー入力を待つメソッド
  • End Sub / End Module: ブロックの終了を示すキーワード

1.5 Visual Studioの基本操作

Visual Studioは多機能な開発環境ですが、最初は基本的な操作方法を覚えることから始めましょう。

主要な画面構成要素

  • コードエディタ: プログラムコードを記述する中央の画面
  • ソリューションエクスプローラー: プロジェクトのファイル一覧
  • プロパティウィンドウ: 選択したオブジェクトの設定
  • エラー一覧: コンパイルエラーや警告の表示
  • 出力ウィンドウ: ビルド結果やデバッグ情報

よく使用するショートカットキー

操作 ショートカット 説明
ビルド Ctrl + Shift + B プロジェクトをコンパイル
実行 F5 デバッグモードで実行
デバッグなし実行 Ctrl + F5 通常実行
保存 Ctrl + S 現在のファイル保存
検索 Ctrl + F コード内検索

各段階の詳細

  1. 要件分析: アプリケーションが満たすべき機能や条件を明確にする
  2. 設計: プログラムの構造、データフロー、ユーザーインターフェースを設計する
  3. コーディング: VB.NETでプログラムコードを実際に記述する
  4. テスト: プログラムが期待通りに動作するかを確認する
  5. デバッグ: エラーやバグを発見し、修正する
  6. デプロイ: 完成したアプリケーションを配布・インストールする
実習 1-2: 簡単な計算プログラム

Hello Worldよりも少し複雑な、簡単な計算を行うプログラムを作成してみましょう。

目標

2つの数字を足し算して結果を表示するプログラムを作成する

サンプルコード
Module Program
    Sub Main(args As String())
        ' 変数の宣言と初期化
        Dim number1 As Integer = 10
        Dim number2 As Integer = 20
        Dim result As Integer
        
        ' 計算の実行
        result = number1 + number2
        
        ' 結果の表示
        Console.WriteLine("数字1: " + number1.ToString())
        Console.WriteLine("数字2: " + number2.ToString())
        Console.WriteLine("合計: " + result.ToString())
        
        ' 終了待機
        Console.WriteLine("何かキーを押してください...")
        Console.ReadKey()
    End Sub
End Module
期待される結果
数字1: 10
数字2: 20
合計: 30
何かキーを押してください...
理解度確認クイズ
  1. VB.NETとは何の略称でしょうか？
    a) Visual Business .NET
    b) Visual Basic .NET
    c) Variable Base .NET
    d) Verified Binary .NET
  2. .NET Frameworkの主な役割は何でしょうか？
    a) コードエディタ機能の提供
    b) プログラムの実行環境とライブラリの提供
    c) データベース管理
    d) ネットワーク接続のみ
  3. VB.NETプログラムの実行開始点となるのは？
    a) Sub Start
    b) Function Begin
    c) Sub Main
    d) Module Entry
  4. Console.WriteLine()の役割は？
    a) ファイルの読み込み
    b) コンソールに文字列を出力
    c) ユーザー入力の受け取り
    d) プログラムの終了
解答を表示
  1. b) Visual Basic .NET - Visual BasicをベースにしたMicrosoftの.NET対応プログラミング言語です。
  2. b) プログラムの実行環境とライブラリの提供 - .NET Frameworkはプログラムの実行管理と豊富なライブラリを提供します。
  3. c) Sub Main - VB.NETプログラムは必ずSub Mainから実行が開始されます。
  4. b) コンソールに文字列を出力 - Console.WriteLine()はコンソールウィンドウに文字列を表示するためのメソッドです。

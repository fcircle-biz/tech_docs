DoCmd オブジェクトによるAccess操作

Access VBAの中核であるDoCmdオブジェクトを使用して、クエリの実行、フォームやレポートの操作、データのインポート・エクスポートなどを自動化する方法を学びます。

この章で学ぶこと

  • DoCmdオブジェクトの役割と基本構文を理解する
  • クエリの実行（RunSQL、OpenQuery）の実装方法
  • フォームやレポートの開閉操作（OpenForm、OpenReport）
  • データのインポート・エクスポート（TransferSpreadsheet、OutputTo）
  • メール送信やマクロ実行などの実用的な自動化

6.1 DoCmdオブジェクトとは？

DoCmdオブジェクトは、Access VBAにおいて最も重要なオブジェクトの1つで、Accessの様々な操作を自動化するためのメソッドを提供します。

「Do Command（コマンドを実行する）」という名前の通り、Accessで手動で行う操作（クエリ実行、フォームを開く、データをエクスポートするなど）を、VBAコードから実行することができます。

DoCmdでできる主な操作

  • • クエリの実行（選択クエリ、アクションクエリ）
  • • フォーム・レポートの開閉
  • • データのインポート・エクスポート（Excel、CSV、テキストファイル）
  • • PDF出力、メール送信
  • • マクロの実行

6.2 クエリの実行（RunSQL、OpenQuery）

DoCmdオブジェクトを使うと、VBAからSQLクエリを実行したり、保存されたクエリを開いたりすることができます。

RunSQL メソッド（アクションクエリの実行）

DoCmd.RunSQLは、アクションクエリ（INSERT、UPDATE、DELETE）を実行するメソッドです。

VBA - RunSQLでデータを追加 
' 顧客テーブルに新規顧客を追加
Sub 顧客を追加する()
    Dim SQL As String

    ' INSERT文を作成
    SQL = "INSERT INTO 顧客テーブル (顧客名, 電話番号, メールアドレス) " & _
          "VALUES ('山田太郎', '03-1234-5678', 'yamada@example.com')"

    ' SQLを実行
    DoCmd.RunSQL SQL

    MsgBox "顧客を追加しました！", vbInformation
End Sub
VBA - RunSQLでデータを更新 
' 顧客の電話番号を更新
Sub 電話番号を更新する()
    Dim SQL As String

    SQL = "UPDATE 顧客テーブル " & _
          "SET 電話番号 = '03-9999-8888' " & _
          "WHERE 顧客名 = '山田太郎'"

    DoCmd.RunSQL SQL
    MsgBox "電話番号を更新しました！", vbInformation
End Sub

注意：RunSQLは選択クエリには使えません

RunSQLはINSERT、UPDATE、DELETE専用です。SELECT（データ取得）を実行したい場合は、次に説明するOpenQueryを使います。

OpenQuery メソッド（保存されたクエリを開く）

Accessで作成・保存したクエリを実行する場合はDoCmd.OpenQueryを使います。

VBA - OpenQueryで保存クエリを実行 
' 保存されたクエリ「売上集計クエリ」を実行
Sub クエリを実行する()
    DoCmd.OpenQuery "売上集計クエリ", acViewNormal
    MsgBox "クエリを実行しました！", vbInformation
End Sub

6.3 フォームとレポートの操作

OpenForm メソッド（フォームを開く）

ボタンクリック時にフォームを開く処理は、業務アプリケーションで最も頻繁に使われる操作の1つです。

VBA - OpenFormでフォームを開く 
' 顧客登録フォームを開く（基本）
Sub 顧客フォームを開く()
    DoCmd.OpenForm "顧客登録フォーム"
End Sub

' 特定の顧客だけを表示（フィルタ付き）
Sub 特定顧客を表示()
    DoCmd.OpenForm "顧客登録フォーム", , , "顧客ID = 101"
End Sub

OpenReport メソッド（レポートを開く・印刷）

レポートを画面プレビュー表示したり、直接印刷したりすることができます。

VBA - OpenReportでレポート操作 
' レポートをプレビュー表示
Sub レポートをプレビュー()
    DoCmd.OpenReport "売上レポート", acViewPreview
End Sub

' レポートを直接印刷
Sub レポートを印刷()
    DoCmd.OpenReport "売上レポート", acViewNormal
End Sub

Close メソッド（フォーム・レポートを閉じる）

VBA - Closeでフォームを閉じる 
' 指定したフォームを閉じる
Sub フォームを閉じる()
    DoCmd.Close acForm, "顧客登録フォーム"
End Sub

' 現在のフォームを閉じる（フォームのボタンから実行）
Sub 自分自身を閉じる()
    DoCmd.Close
End Sub

6.4 データのインポート・エクスポート

TransferSpreadsheet（Excelとの連携）

Excelファイルからデータをインポートしたり、AccessのテーブルデータをExcelにエクスポートしたりできます。

VBA - ExcelファイルをAccessにインポート 
' Excelファイルから顧客データをインポート
Sub Excelをインポート()
    DoCmd.TransferSpreadsheet _
        acImport, _
        acSpreadsheetTypeExcel12Xml, _
        "顧客テーブル", _
        "C:\data\顧客リスト.xlsx", _
        True  ' 先頭行をフィールド名として使用

    MsgBox "Excelファイルをインポートしました！", vbInformation
End Sub
VBA - AccessデータをExcelにエクスポート 
' 顧客テーブルをExcelファイルにエクスポート
Sub Excelにエクスポート()
    DoCmd.TransferSpreadsheet _
        acExport, _
        acSpreadsheetTypeExcel12Xml, _
        "顧客テーブル", _
        "C:\export\顧客リスト.xlsx", _
        True

    MsgBox "Excelファイルにエクスポートしました！", vbInformation
End Sub

OutputTo（PDF出力）

レポートをPDFファイルとして出力する場合はDoCmd.OutputToを使います。

VBA - レポートをPDF出力 
' 売上レポートをPDF出力
Sub レポートをPDF出力()
    DoCmd.OutputTo _
        acOutputReport, _
        "売上レポート", _
        acFormatPDF, _
        "C:\reports\売上レポート.pdf"

    MsgBox "PDFファイルを作成しました！", vbInformation
End Sub

6.5 その他の便利なメソッド

SendObject（メール送信）

VBA - レポートをメール添付で送信 
' レポートをPDF添付でメール送信
Sub レポートをメール送信()
    DoCmd.SendObject _
        acSendReport, _
        "売上レポート", _
        acFormatPDF, _
        "manager@example.com", _
        , _
        , _
        "月次売上レポート", _
        "今月の売上レポートを送付します。"
End Sub

RunCommand（Accessのコマンド実行）

VBA - Accessのコマンドを実行 
' データベースを最適化・修復
Sub データベースを最適化()
    DoCmd.RunCommand acCmdCompactDatabase
    MsgBox "最適化が完了しました！", vbInformation
End Sub

実習：DoCmdを使った自動化プログラム

  1. 1 新しいAccessデータベース（.accdb）を作成し、「顧客テーブル」を作成
  2. 2 標準モジュールを作成し、RunSQLで顧客データを追加するプロシージャを作成
  3. 3 簡単なフォームを作成し、OpenFormメソッドで開くプロシージャを作成
  4. 4 作成したテーブルをExcelファイルにエクスポートする処理を実装

理解度確認クイズ

  1. Q1. DoCmdオブジェクトの役割は何ですか？

    回答を表示

    Accessの様々な操作（クエリ実行、フォーム・レポートの開閉、データのインポート・エクスポートなど）をVBAコードから自動化するためのオブジェクト

  2. Q2. RunSQLメソッドで実行できるクエリの種類は何ですか？

    回答を表示

    アクションクエリ（INSERT、UPDATE、DELETE）のみ。SELECTクエリは実行できません。

  3. Q3. Excelファイルをインポート・エクスポートする際に使うメソッドは何ですか？

    回答を表示

    DoCmd.TransferSpreadsheet

  4. Q4. レポートをPDFファイルとして出力する際に使うメソッドは何ですか？

    回答を表示

    DoCmd.OutputTo（オブジェクトタイプにacOutputReport、出力形式にacFormatPDFを指定）

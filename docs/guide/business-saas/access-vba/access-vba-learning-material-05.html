条件分岐と繰り返し処理

If～Then～Else文による条件分岐、Select Case文による多分岐処理、For～Next、Do～Loopによる繰り返し処理を学び、プログラムの制御構造を習得します。

この章で学ぶこと

  • If文を使った条件分岐の基本と応用
  • Select Case文による多分岐処理の実装方法
  • For～Nextループによる繰り返し処理
  • Do～Loopループと条件判定の組み合わせ
  • ネストした制御構造の書き方と実用例

5.1 条件分岐とは？

プログラムの基本的な制御構造のひとつが条件分岐です。条件によって処理を分けることで、状況に応じた動作を実現します。

例えば「在庫が10個以下なら発注処理を実行する」「入力された値が数値でなければエラーメッセージを表示する」といった、日常業務でよく使われる判断ロジックを実装できます。

5.2 If～Then～Else文の基本

If文はVBAで最も頻繁に使われる条件分岐の構文です。「もし～ならば」という条件判定を行います。

If文の基本構文

If 条件式 Then
  条件が真の場合の処理
Else
  条件が偽の場合の処理
End If

在庫チェックの例 
Sub 在庫チェック()
    Dim 在庫数 As Integer
    在庫数 = 8  ' 現在の在庫数

    ' 在庫が10個以下かチェック
    If 在庫数 <= 10 Then
        MsgBox "在庫が少なくなっています。発注が必要です。"
    Else
        MsgBox "在庫は十分です。"
    End If
End Sub

ElseIfによる複数条件

3つ以上の条件に分岐したい場合は、ElseIfを使います。

成績判定の例 
Sub 成績判定()
    Dim 点数 As Integer
    Dim 評価 As String

    点数 = 75

    ' 複数条件で評価を判定
    If 点数 >= 80 Then
        評価 = "優"
    ElseIf 点数 >= 60 Then
        評価 = "良"
    ElseIf 点数 >= 40 Then
        評価 = "可"
    Else
        評価 = "不可"
    End If

    MsgBox "点数: " & 点数 & " → 評価: " & 評価
End Sub

5.3 Select Case文による多分岐処理

条件分岐が多くなる場合、Select Case文を使うとコードが読みやすくなります。特定の値によって処理を切り替える際に便利です。

曜日判定の例 
Sub 曜日メッセージ()
    Dim 今日 As Integer
    今日 = Weekday(Date)  ' 1=日曜, 2=月曜...7=土曜

    ' 曜日に応じたメッセージを表示
    Select Case 今日
        Case 1  ' 日曜日
            MsgBox "今日は日曜日です。お休みの方も多いでしょう。"
        Case 2  ' 月曜日
            MsgBox "今日は月曜日です。一週間の始まりです！"
        Case 7  ' 土曜日
            MsgBox "今日は土曜日です。週末ですね。"
        Case Else  ' その他
            MsgBox "今日は平日です。お仕事頑張りましょう！"
    End Select
End Sub

範囲指定とカンマ区切り

Select Case文では、Toによる範囲指定や、カンマ区切りで複数の値をまとめて判定できます。

年齢区分判定の例 
Sub 年齢区分()
    Dim 年齢 As Integer
    Dim 区分 As String

    年齢 = 35

    ' 年齢範囲で区分を判定
    Select Case 年齢
        Case 0 To 12
            区分 = "子供"
        Case 13 To 19
            区分 = "ティーンエイジャー"
        Case 20 To 64
            区分 = "成人"
        Case Is >= 65
            区分 = "シニア"
    End Select

    MsgBox "年齢 " & 年齢 & "歳 → " & 区分
End Sub

5.4 For～Nextによる繰り返し処理

For～Next文は、指定回数だけ処理を繰り返すための構文です。カウンタ変数を使って、1から10まで、あるいは配列の要素数分など、決まった回数処理を実行します。

1から10まで表示する例 
Sub 繰り返し処理()
    Dim i As Integer
    Dim 結果 As String

    ' 1から10まで繰り返す
    For i = 1 To 10
        結果 = 結果 & i & " "
    Next i

    MsgBox "1から10まで: " & 結果
End Sub

Stepによる増分の指定

Stepキーワードを使うと、カウンタの増分を指定できます。2ずつ増やしたり、逆に減らしたりできます。

偶数だけ表示する例 
Sub 偶数表示()
    Dim i As Integer

    ' 2, 4, 6, 8, 10を表示
    For i = 2 To 10 Step 2
        Debug.Print i  ' イミディエイトウィンドウに出力
    Next i
End Sub

5.5 Do～Loopによる条件付き繰り返し

Do～Loop文は、条件が満たされている間、または満たされるまで処理を繰り返します。繰り返し回数が事前に分からない場合に使います。

Do While ～ Loop

条件が真の間、処理を繰り返す（前判定）

Do While 条件
  処理
Loop

Do ～ Loop Until

条件が真になるまで処理を繰り返す（後判定）

Do
  処理
Loop Until 条件
合計が100を超えるまで繰り返す例 
Sub 合計計算()
    Dim 合計 As Integer
    Dim 加算値 As Integer

    合計 = 0
    加算値 = 10

    ' 合計が100以上になるまで繰り返す
    Do Until 合計 >= 100
        合計 = 合計 + 加算値
        Debug.Print "現在の合計: " & 合計
    Loop

    MsgBox "最終合計: " & 合計
End Sub

無限ループに注意

Do～Loop文では、条件が永遠に満たされない場合、無限ループに陥る可能性があります。必ず終了条件が成立するように設計してください。

5.6 ネストした制御構造

条件分岐やループを組み合わせて、より複雑な処理を実装できます。これをネスト（入れ子）と呼びます。

九九の表を作成する例 
Sub 九九の表()
    Dim i As Integer, j As Integer
    Dim 結果 As String

    ' 外側のループ: 1の段から9の段
    For i = 1 To 9
        結果 = i & "の段: "

        ' 内側のループ: 1から9まで掛ける
        For j = 1 To 9
            結果 = 結果 & (i * j) & " "
        Next j

        Debug.Print 結果
    Next i

    MsgBox "イミディエイトウィンドウに出力しました"
End Sub

理解度確認クイズ

  1. Q1. If文で複数の条件に分岐する際に使うキーワードは何ですか？

    回答を表示

    ElseIf（エルスイフ）を使います。

  2. Q2. Select Case文で、Case 1, 3, 5 のように複数値をまとめて指定できますか？

    回答を表示

    はい、カンマ区切りで複数の値をまとめて指定できます。

  3. Q3. For i = 10 To 1 Step -1 というループは何を行いますか？

    回答を表示

    10から1まで、1ずつ減らしながら繰り返します（カウントダウン）。

  4. Q4. Do While ～ Loop と Do ～ Loop Until の違いは何ですか？

    回答を表示

    Do Whileは条件が真の間繰り返し（前判定）、Do～Loop Untilは条件が真になるまで繰り返します（後判定）。

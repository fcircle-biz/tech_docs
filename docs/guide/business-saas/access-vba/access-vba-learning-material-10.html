レポートの自動生成とカスタマイズ

VBAによるレポートの動的生成、レポートセクションの制御、グループ化と集計、PDF出力やExcelエクスポートなど、レポート出力の自動化を習得します。

この章で学ぶこと

  • VBAを使ったレポートの動的な生成・プレビュー・印刷
  • レポートセクション（ヘッダー、詳細、フッター）の制御方法
  • レポートのグループ化と集計機能の実装
  • PDF出力、Excelエクスポート、メール送信などの出力自動化

10.1 Accessレポートの基礎知識

レポートヘッダー/フッター

レポート全体の最初と最後に1回だけ表示されます。タイトルや総合計に使用します。

ページヘッダー/フッター

各ページの上部・下部に表示されます。列見出しやページ番号に使用します。

グループヘッダー/フッター

グループ化したデータの開始・終了時に表示されます。部門名や小計に使用します。

詳細セクション

各レコードごとに繰り返し表示されます。実際のデータを表示する部分です。

10.3 VBAでレポートを開く

VBAからDoCmd.OpenReportメソッドを使って、レポートをプレビュー・印刷できます。

レポートをプレビュー表示 
Sub OpenReportPreview()
    ' レポートをプレビュー表示
    DoCmd.OpenReport "売上レポート", acViewPreview
End Sub
レポートを直接印刷 
Sub PrintReportDirectly()
    ' レポートを直接印刷（プレビューなし）
    DoCmd.OpenReport "月次売上レポート", acViewNormal

    ' または印刷ビューを指定
    DoCmd.OpenReport "月次売上レポート", acViewReport
End Sub

acView定数の種類

  • • acViewPreview - プレビュー表示（画面で確認）
  • • acViewNormal - 直接印刷
  • • acViewReport - レポートビュー（編集不可表示）
  • • acViewDesign - デザインビュー（デザイン編集）

10.4 フィルタ条件を指定してレポートを開く

レポートを開く際にWhere条件を指定することで、特定のデータだけを表示できます。

特定の顧客のレポートを表示 
Sub OpenReportWithFilter()
    Dim customerID As Long
    customerID = 1001

    ' Where条件を指定してレポートを開く
    DoCmd.OpenReport "顧客別売上レポート", _
                     acViewPreview, , _
                     "顧客ID = " & customerID
End Sub
期間指定レポート 
Sub OpenReportByDateRange()
    Dim startDate As Date
    Dim endDate As Date

    startDate = #1/1/2025#
    endDate = #1/31/2025#

    ' 日付範囲を指定
    DoCmd.OpenReport "売上レポート", _
                     acViewPreview, , _
                     "売上日 BETWEEN #" & startDate & "# AND #" & endDate & "#"
End Sub

10.5 PDF出力とExcelエクスポート

DoCmd.OutputToメソッドを使えば、レポートをPDFやExcel形式で自動出力できます。

レポートをPDF出力 
Sub ExportReportToPDF()
    Dim outputPath As String

    ' 出力先パスを指定
    outputPath = "C:\Reports\売上レポート_" & Format(Date, "yyyymmdd") & ".pdf"

    ' PDF形式で出力
    DoCmd.OutputTo acOutputReport, "売上レポート", acFormatPDF, outputPath, False

    MsgBox "PDFを出力しました: " & outputPath, vbInformation
End Sub
レポートをExcel出力 
Sub ExportReportToExcel()
    Dim outputPath As String

    outputPath = "C:\Reports\売上データ.xlsx"

    ' Excel形式で出力（xlsx）
    DoCmd.OutputTo acOutputReport, "売上レポート", acFormatXLSX, outputPath, False

    MsgBox "Excelファイルを出力しました", vbInformation
End Sub

出力形式の種類

  • • acFormatPDF - PDF形式（最も一般的）
  • • acFormatXLSX - Excel 2007以降（.xlsx）
  • • acFormatXLS - Excel 97-2003（.xls）
  • • acFormatHTML - HTML形式
  • • acFormatRTF - リッチテキスト形式

レポート開始時の処理 
Private Sub Report_Open(Cancel As Integer)
    ' レポートが開かれる直前に実行される
    ' ユーザーに確認を求めることができる

    Dim result As VbMsgBoxResult
    result = MsgBox("レポートを開きますか？", vbYesNo + vbQuestion)

    If result = vbNo Then
        Cancel = True  ' レポートを開くのをキャンセル
    End If
End Sub
条件付き書式（色変更） 
Private Sub 詳細_Format(Cancel As Integer, FormatCount As Integer)
    ' 売上金額が100万円以上なら背景を黄色に
    If Me.売上金額 >= 1000000 Then
        Me.売上金額.BackColor = vbYellow
        Me.売上金額.ForeColor = vbRed
    Else
        Me.売上金額.BackColor = vbWhite
        Me.売上金額.ForeColor = vbBlack
    End If
End Sub

10.7 グループ化と集計

レポートのグループ化機能を使えば、部門別・月別などの小計を自動的に表示できます。

グループフッターで小計を表示 
' レポートのデザインビューで、グループフッターに以下のテキストボックスを配置
' コントロールソース: =Sum([売上金額])
' 名前: txt小計

Private Sub グループフッター_Format(Cancel As Integer, FormatCount As Integer)
    ' グループごとの小計を太字で表示
    Me.txt小計.FontBold = True
    Me.txt小計.FontSize = 12
End Sub

集計関数の使い方

  • • =Sum([フィールド名]) - 合計
  • • =Avg([フィールド名]) - 平均
  • • =Count([フィールド名]) - 件数
  • • =Max([フィールド名]) - 最大値
  • • =Min([フィールド名]) - 最小値

10.8 レポートをメールで自動送信

DoCmd.SendObjectメソッドを使えば、レポートをPDF添付でメール送信できます。

レポートをPDF添付でメール送信 
Sub SendReportByEmail()
    ' レポートをPDF添付でメール送信
    DoCmd.SendObject _
        ObjectType:=acSendReport, _
        ObjectName:="月次売上レポート", _
        OutputFormat:=acFormatPDF, _
        To:="customer@example.com", _
        Subject:="月次売上レポート", _
        MessageText:="月次売上レポートを送付いたします。", _
        EditMessage:=True  ' メール送信前に編集可能
End Sub

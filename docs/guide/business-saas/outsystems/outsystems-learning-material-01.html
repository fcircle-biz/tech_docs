OutSystemsの世界へようこそ - ローコード開発入門

ローコード開発とは何か、OutSystemsで何ができるのかを学び、開発環境を整えて最初のアプリケーションを作成しましょう。

この章で学ぶこと

  • ローコード開発とは何か、従来の開発方法との違いを理解する
  • OutSystemsの特徴と、どんなアプリケーションが作れるのかを知る
  • Personal Environmentのセットアップと Service Studio のインストールを行う
  • 最初のアプリケーション「Hello OutSystems」を作成して動かす

1.1 ローコード開発って何だろう？

普段、スマートフォンでアプリを使ったり、パソコンでWebサイトを見たりしていますよね。 これらのアプリケーションは、プログラマーがプログラムコード（コンピュータへの指示）を書くことで作られています。

従来の開発方法

これまでのアプリケーション開発では、プログラマーが一行一行、コードを手で入力していました。 例えば、「ボタンをクリックしたらメッセージを表示する」という単純な機能でも、数十行のコードを書く必要がありました。

従来の方法（JavaScript の例） 
// ボタンを取得
const button = document.getElementById('myButton');

// クリックイベントを設定
button.addEventListener('click', function() {
    // メッセージを表示
    alert('こんにちは、OutSystems!');
});

// 画面にボタンを配置するHTMLコード
// スタイルを調整するCSSコード
// ...など、複数のファイルを編集する必要がある

ローコード開発とは

ローコード開発（Low-Code Development）は、 「Low（少ない）Code（コード）」という名前の通り、プログラムコードをほとんど書かずにアプリケーションを作る開発方法です。

ローコード開発のイメージ

PowerPointでスライドを作るときのように、ドラッグ＆ドロップやクリック操作で部品を配置していくだけで、 アプリケーションが完成します。まるでレゴブロックを組み立てるような感覚です。

ローコード開発のメリット

開発スピードが速い

コードを書く代わりに視覚的に操作するため、アプリケーションを短期間で作成できます。

プログラミング初心者でもOK

難しいコードを覚えなくても、マウス操作でアプリケーションを作れます。

変更・修正が簡単

画面を見ながら編集できるので、デザインや機能の変更がすぐにできます。

品質が安定している

プラットフォームが自動的に最適なコードを生成するため、バグが少なく安定したアプリが作れます。

1.2 OutSystemsとは - 世界で選ばれるローコードプラットフォーム

OutSystems（アウトシステムズ）は、 世界中で使われているエンタープライズ向けローコード開発プラットフォームです。 2001年にポルトガルで創業され、現在では世界中の大企業や政府機関で採用されています。

OutSystemsの特徴

ビジュアル開発環境

Service Studioという開発ツールで、ドラッグ＆ドロップでアプリを作成できます。 画面を見ながら直感的に開発できるので、プログラミング経験がなくても大丈夫です。

フルスタック開発

Webアプリ、モバイルアプリ（iOS/Android）、データベース、API連携まで、 すべてOutSystemsだけで開発できます。複数のツールを使い分ける必要がありません。

自動デプロイメント

作ったアプリはボタン一つで本番環境に公開できます。 従来の開発で必要だった複雑なサーバー設定や公開作業が自動化されています。

エンタープライズグレード

大企業向けに設計されており、セキュリティ、スケーラビリティ、パフォーマンスが高いレベルで保証されています。 金融機関や官公庁でも採用される信頼性があります。

OutSystemsで作れるアプリケーション

1.3 Personal Environmentのセットアップ

OutSystemsで開発を始めるには、まずPersonal Environment（個人用無料環境）のアカウントを作成します。 この環境では、学習や個人的な開発に必要な機能がすべて使えます。

Personal Environmentとは

OutSystemsが提供する無料のクラウド開発環境です。 クレジットカード不要で、学習や個人開発に必要な機能がすべて使えます。

実習 1-1: Personal Environmentアカウント作成

OutSystemsの無料アカウントを作成して、開発環境を準備しましょう。

手順

  1. 1 Webブラウザで https://www.outsystems.com/free-edition/ にアクセスします
  2. 2 「Start for free」ボタンをクリックします
  3. 3 メールアドレス、名前、パスワードを入力して「Sign up」をクリックします
  4. 4 登録したメールアドレスに確認メールが届くので、メール内のリンクをクリックして認証します
  5. 5 認証が完了すると、Personal Environment が自動的に作成されます（数分かかることがあります）

環境のURL

作成された環境には以下のようなURLでアクセスできます：

https://あなたの名前-personal.outsystemscloud.com/

注意

Personal Environmentは90日間使用がない場合、自動的に削除されます。 定期的にログインして、環境を維持しましょう。

1.4 Service Studioのインストール

Service Studioは、OutSystemsのアプリケーションを開発するための統合開発環境（IDE）です。 Windows 10/11またはmacOSで動作し、無料でダウンロードできます。

実習 1-2: Service Studioのインストール

開発ツールをダウンロードしてインストールしましょう。

Windows の場合

  1. 1 Personal Environment のポータル画面にログインします
  2. 2 画面上部の「Download Service Studio」ボタンをクリックします
  3. 3 ダウンロードした .exe ファイルをダブルクリックして実行します
  4. 4 インストーラーの指示に従って「Next」をクリックしていき、インストールを完了します

macOS の場合

  1. 1 Personal Environment のポータル画面にログインします
  2. 2 「Download Service Studio」をクリックし、Mac版を選択します
  3. 3 ダウンロードした .dmg ファイルを開き、Service Studioをアプリケーションフォルダにドラッグします

インストールが完了したら、Service Studio を起動して Personal Environment のURLとログイン情報を入力してください。

Service Studio の画面構成

Service Studioを起動すると、以下のようなエリアで構成された画面が表示されます：

  • ツールボックス: ウィジェット（画面部品）やロジック要素が並んでいます
  • キャンバス: 画面やロジックをビジュアルに組み立てる作業エリアです
  • プロパティ: 選択した要素の設定を変更できます
  • エレメントツリー: アプリの構造が階層表示されます

1.5 最初のアプリケーション「Hello OutSystems」を作ろう

それでは、いよいよ最初のアプリケーションを作成してみましょう。 「Hello OutSystems」というメッセージを表示するシンプルなWebアプリを作成します。

実習 1-3: Hello OutSystems アプリを作る

ドラッグ＆ドロップで最初のアプリケーションを作成しましょう。

手順

  1. 1

    新しいアプリケーションを作成

    Service Studioを起動し、「New Application」をクリックします。 テンプレートから「Reactive Web App」を選択し、アプリ名に「HelloOutSystems」と入力して「Create App」をクリックします。

  2. 2

    モジュールを作成

    モジュール名に「HelloModule」と入力し、「Create Module」をクリックします。 Service Studioのメイン画面が開きます。

  3. 3

    画面を追加

    左側のツリーで「Interface」→「UI Flows」→「MainFlow」を右クリックし、 「Add Screen」を選択します。画面テンプレートから「Blank」を選び、画面名に「HelloScreen」と入力して「Create Screen」をクリックします。

  4. 4

    テキストを配置

    左側のツールボックスから「Text」ウィジェットを探し、 中央のキャンバス（画面デザインエリア）にドラッグ＆ドロップします。

  5. 5

    メッセージを設定

    配置したテキストをダブルクリックし、「Hello OutSystems!」と入力します。 右側のプロパティで、フォントサイズを大きくしたり、色を変更したりできます。

  6. 6

    公開（Publish）

    画面上部の「1-Click Publish」ボタンをクリックします。 数秒でアプリケーションがクラウド環境に公開されます。

  7. 7

    アプリを開く

    公開が完了したら、「Open in Browser」ボタンをクリックします。 ブラウザで「Hello OutSystems!」というメッセージが表示されたら成功です！

おめでとうございます！

これであなたも OutSystems 開発者の仲間入りです。 コードを一行も書かずに、Webアプリケーションを作成して公開できました。

理解度確認クイズ

  1. Q1. ローコード開発の主なメリットを3つ挙げてください。

    回答を表示

    1. 開発スピードが速い - 視覚的な操作で短期間で開発できる
    2. プログラミング初心者でも開発できる - コードを書かなくても開発可能
    3. 変更・修正が簡単 - 画面を見ながら直感的に編集できる

  2. Q2. OutSystemsの開発ツールの名前は何ですか？

    回答を表示

    Service Studio（サービス・スタジオ）です。 ドラッグ＆ドロップで画面を作成できるビジュアル開発環境です。

  3. Q3. Personal Environmentは何日間使用しないと削除されますか？

    回答を表示

    90日間使用しないと自動的に削除されます。 定期的にログインして、環境を維持しましょう。

  4. Q4. OutSystemsで作成したアプリケーションを公開するには、何をクリックしますか？

    回答を表示

    「1-Click Publish」ボタンをクリックします。 これだけで、アプリケーションがクラウド環境に自動的にデプロイされます。

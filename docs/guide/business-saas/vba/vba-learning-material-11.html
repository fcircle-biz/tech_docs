第11章

ファイル操作と外部データ連携

テキストファイルやCSVファイルの読み書き、他のExcelファイルとの連携、フォルダ内ファイルの一括処理方法を習得します。

この章で学ぶこと

  • テキストファイルの読み込みと書き込みの基本操作
  • CSVファイルのインポートとエクスポート方法
  • 他のExcelファイルとのデータ連携テクニック
  • FileSystemObjectを使った高度なファイル操作

11.1 ファイル操作の基本概念

VBAでは、Excelブック以外のファイル（テキストファイル、CSVファイル、他のExcelファイルなど）を読み書きできます。これにより、外部システムとのデータ連携や、複数ファイルの一括処理が可能になります。

VBAで扱える主なファイル形式

  • テキストファイル（.txt）: 単純なテキストデータ、ログファイルなど
  • CSVファイル（.csv）: カンマ区切りのデータ、システム間のデータ交換
  • Excelファイル（.xlsx, .xlsm）: 他のワークブックとのデータ連携
  • その他: XMLファイル、JSONファイルなども読み書き可能（上級）

ファイル操作の基本的な流れ

重要な注意点

ファイルを開いたら、必ず閉じることが重要です。ファイルを閉じ忘れると、他のプログラムがそのファイルにアクセスできなくなったり、データが正しく保存されなかったりします。

11.2 テキストファイルの読み込み

まずは、テキストファイルを読み込んでExcelシートに表示する方法を学びます。OpenステートメントとLine Inputを使います。

実習 11-1: テキストファイルを読み込んでシートに表示

テキストファイルの内容を1行ずつ読み込み、A列に表示します。

事前準備

  1. CドライブにC:\tempフォルダを作成
  2. メモ帳でsample.txtを作成し、以下の内容を保存：
こんにちは
VBAファイル操作の練習です
テキストファイルを読み込んでいます
標準モジュール 
Sub テキストファイル読み込み()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim lineText As String
    Dim rowNum As Long

    ' ファイルパスを指定
    filePath = "C:\temp\sample.txt"

    ' 空いているファイル番号を取得
    fileNumber = FreeFile

    ' ファイルを開く（読み込みモード）
    Open filePath For Input As #fileNumber

    ' 行番号の初期化
    rowNum = 1

    ' ファイルの終わりまで繰り返し
    Do While Not EOF(fileNumber)
        ' 1行読み込み
        Line Input #fileNumber, lineText
        ' セルに書き込み
        Cells(rowNum, 1).Value = lineText
        ' 次の行へ
        rowNum = rowNum + 1
    Loop

    ' ファイルを閉じる
    Close #fileNumber

    MsgBox "ファイルの読み込みが完了しました。", vbInformation
End Sub

コードの解説

  • FreeFile: 使用可能なファイル番号を自動取得
  • Open For Input: ファイルを読み込みモードで開く
  • EOF(): End Of File（ファイル終端）判定
  • Line Input: 1行ずつテキストを読み込む

11.3 テキストファイルの書き込み

次に、Excelシートのデータをテキストファイルに書き出す方法を学びます。Open For OutputとPrint #を使います。

標準モジュール 
Sub テキストファイル書き込み()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim lastRow As Long
    Dim i As Long

    ' ファイルパスを指定
    filePath = "C:\temp\output.txt"

    ' 空いているファイル番号を取得
    fileNumber = FreeFile

    ' ファイルを開く（書き込みモード）
    Open filePath For Output As #fileNumber

    ' A列の最終行を取得
    lastRow = Cells(Rows.Count, 1).End(xlUp).Row

    ' A列の全データをファイルに書き込み
    For i = 1 To lastRow
        Print #fileNumber, Cells(i, 1).Value
    Next i

    ' ファイルを閉じる
    Close #fileNumber

    MsgBox "ファイルへの書き込みが完了しました。", vbInformation
End Sub

書き込みモードの種類

  • For Output: 新規作成モード（既存ファイルは上書き）
  • For Append: 追記モード（既存ファイルの末尾に追加）

11.4 CSVファイルの操作

CSV（Comma-Separated Values）ファイルは、カンマで区切られたデータ形式で、システム間のデータ交換によく使われます。VBAでCSVファイルを読み込み、複数列のデータを処理する方法を学びます。

実習 11-2: CSVファイルを読み込んでExcelに展開

CSV形式のデータを読み込み、カンマで分割して複数列に配置します。

事前準備

メモ帳でC:\temp\data.csvを作成し、以下の内容を保存：

商品名,単価,数量
りんご,150,10
バナナ,100,20
みかん,80,15
標準モジュール 
Sub CSV読み込み()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim lineText As String
    Dim dataArray As Variant
    Dim rowNum As Long
    Dim i As Long

    ' ファイルパスを指定
    filePath = "C:\temp\data.csv"
    fileNumber = FreeFile

    ' ファイルを開く
    Open filePath For Input As #fileNumber

    rowNum = 1

    ' ファイルの終わりまで繰り返し
    Do While Not EOF(fileNumber)
        Line Input #fileNumber, lineText
        ' カンマで分割して配列に格納
        dataArray = Split(lineText, ",")
        ' 配列の各要素をセルに書き込み
        For i = 0 To UBound(dataArray)
            Cells(rowNum, i + 1).Value = dataArray(i)
        Next i
        rowNum = rowNum + 1
    Loop

    Close #fileNumber

    MsgBox "CSVファイルの読み込みが完了しました。", vbInformation
End Sub

Split関数の活用

Split(lineText, ",")は、文字列をカンマで分割して配列に変換します。この配列をループで回すことで、各データを別々のセルに配置できます。

CSVファイルへの書き出し

逆に、Excelのデータをカンマ区切りでCSVファイルに出力することもできます。

標準モジュール 
Sub CSV書き出し()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim lastRow As Long, lastCol As Long
    Dim i As Long, j As Long
    Dim lineText As String

    filePath = "C:\temp\export.csv"
    fileNumber = FreeFile

    ' 最終行・列を取得
    lastRow = Cells(Rows.Count, 1).End(xlUp).Row
    lastCol = Cells(1, Columns.Count).End(xlToLeft).Column

    ' ファイルを開く
    Open filePath For Output As #fileNumber

    ' 全行をCSV形式で書き込み
    For i = 1 To lastRow
        lineText = ""
        For j = 1 To lastCol
            lineText = lineText & Cells(i, j).Value
            If j < lastCol Then lineText = lineText & ","
        Next j
        Print #fileNumber, lineText
    Next i

    Close #fileNumber

    MsgBox "CSVファイルへの書き出しが完了しました。", vbInformation
End Sub

11.5 他のExcelファイルとの連携

他のExcelファイルを開いてデータを読み込んだり、データを書き込んだりする方法を学びます。Workbooks.Openメソッドを使います。

標準モジュール 
Sub 他のExcelファイルからデータ取得()
    Dim sourceWb As Workbook
    Dim sourceWs As Worksheet
    Dim targetWs As Worksheet
    Dim lastRow As Long

    ' 現在のシートを対象シートとして設定
    Set targetWs = ThisWorkbook.Worksheets("Sheet1")

    ' 他のExcelファイルを開く
    Set sourceWb = Workbooks.Open("C:\temp\source.xlsx")
    Set sourceWs = sourceWb.Worksheets(1)

    ' データの最終行を取得
    lastRow = sourceWs.Cells(Rows.Count, 1).End(xlUp).Row

    ' データをコピー
    sourceWs.Range("A1:C" & lastRow).Copy _
        Destination:=targetWs.Range("A1")

    ' ファイルを閉じる（保存しない）
    sourceWb.Close SaveChanges:=False

    MsgBox "データのコピーが完了しました。", vbInformation
End Sub

注意点

他のワークブックを開いたら、必ずCloseメソッドで閉じましょう。開きっぱなしにすると、メモリを圧迫したり、ファイルがロックされたりします。

11.6 FileSystemObjectで高度なファイル操作

FileSystemObject（FSO）は、ファイルやフォルダの存在チェック、フォルダ内のファイル一覧取得など、より高度なファイル操作を行うためのオブジェクトです。

FSOを使うための準備

FSOを使用するには、参照設定が必要です（通常は不要ですが、一部の環境で必要な場合があります）。

  1. VBEで「ツール」→「参照設定」を開く
  2. 「Microsoft Scripting Runtime」にチェック
  3. OKをクリック

ファイルの存在チェック

標準モジュール 
Sub ファイル存在チェック()
    Dim fso As Object
    Dim filePath As String

    ' FileSystemObjectを作成
    Set fso = CreateObject("Scripting.FileSystemObject")

    filePath = "C:\temp\sample.txt"

    ' ファイルが存在するかチェック
    If fso.FileExists(filePath) Then
        MsgBox "ファイルが存在します。", vbInformation
    Else
        MsgBox "ファイルが見つかりません。", vbExclamation
    End If

    ' オブジェクトを解放
    Set fso = Nothing
End Sub

フォルダ内のファイル一覧取得

特定のフォルダ内にあるすべてのファイルを一覧表示できます。これは複数ファイルの一括処理に便利です。

標準モジュール 
Sub フォルダ内ファイル一覧()
    Dim fso As Object
    Dim folder As Object
    Dim file As Object
    Dim rowNum As Long

    ' FileSystemObjectを作成
    Set fso = CreateObject("Scripting.FileSystemObject")

    ' フォルダを取得
    Set folder = fso.GetFolder("C:\temp")

    rowNum = 1

    ' フォルダ内の各ファイルを処理
    For Each file In folder.Files
        Cells(rowNum, 1).Value = file.Name
        Cells(rowNum, 2).Value = file.Size
        Cells(rowNum, 3).Value = file.DateLastModified
        rowNum = rowNum + 1
    Next file

    Set fso = Nothing

    MsgBox "ファイル一覧の取得が完了しました。", vbInformation
End Sub

実習 11-3: フォルダ内の全Excelファイルを一括処理

指定フォルダ内のすべてのExcelファイルを開き、A1セルの値を取得するマクロを作成します。

標準モジュール 
Sub 全Excelファイル一括処理()
    Dim fso As Object
    Dim folder As Object
    Dim file As Object
    Dim wb As Workbook
    Dim rowNum As Long

    Set fso = CreateObject("Scripting.FileSystemObject")
    Set folder = fso.GetFolder("C:\temp")

    rowNum = 1

    ' フォルダ内の各ファイルを処理
    For Each file In folder.Files
        ' Excelファイルのみ処理
        If Right(file.Name, 5) = ".xlsx" Then
            ' ファイルを開く
            Set wb = Workbooks.Open(file.Path)
            ' A1セルの値を取得
            Cells(rowNum, 1).Value = file.Name
            Cells(rowNum, 2).Value = wb.Worksheets(1).Range("A1").Value
            ' ファイルを閉じる
            wb.Close SaveChanges:=False
            rowNum = rowNum + 1
        End If
    Next file

    Set fso = Nothing

    MsgBox "一括処理が完了しました。", vbInformation
End Sub

動作確認チェックリスト

  • テキストファイルを正しく読み込み、内容がシートに表示される
  • CSVファイルが複数列に正しく展開される
  • 他のExcelファイルからデータが正しくコピーされる
  • フォルダ内のファイル一覧が正しく表示される

よくあるトラブルと対処法

エラー: ファイルが見つかりません

原因: ファイルパスが間違っているか、ファイルが存在しない
対処法: ファイルパスを確認し、ファイルが指定フォルダに存在することを確認。FSOのFileExistsで事前チェック

エラー: ファイルは既に開いています

原因: 同じファイルを他のプログラムが開いている
対処法: 他のExcelやテキストエディタで開いているファイルを閉じてから実行

ファイルを閉じたはずなのにロックされたまま

原因: オブジェクトが解放されていない
対処法: Set オブジェクト = Nothingで明示的に解放

理解度確認クイズ

  1. Q1. テキストファイルを読み込むときに使う主なステートメントは何ですか？

    回答を表示

    Open For Inputでファイルを開き、Line Inputで1行ずつ読み込みます。読み込み後はCloseでファイルを閉じます。

  2. Q2. CSVファイルの1行を複数のセルに分割するために使う関数は何ですか？

    回答を表示

    Split関数を使います。例: dataArray = Split(lineText, ",")でカンマ区切りの文字列を配列に変換できます。

  3. Q3. FileSystemObjectの利点を2つ挙げてください。

    回答を表示

    ① ファイルやフォルダの存在チェックが簡単にできる（FileExistsメソッド）
    ② フォルダ内のファイル一覧を簡単に取得でき、一括処理がしやすい

  4. Q4. 他のExcelファイルを開いた後、必ずやるべきことは何ですか？

    回答を表示

    Closeメソッドでファイルを閉じることです。開いたままにするとメモリを圧迫し、ファイルがロックされてしまいます。

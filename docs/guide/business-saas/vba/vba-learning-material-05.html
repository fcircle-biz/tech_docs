第5章

条件分岐（If文）で賢い処理を作る

If～Then～Else文の基本構造、複数条件の組み合わせ（And/Or）、Select Case文による多分岐処理を実務的な例題で習得します。

この章で学ぶこと

  • If～Then～Else文の基本構造を理解し、条件に応じた処理分岐を実装できる
  • 複数条件の組み合わせ（And/Or）を使った複雑な条件式を作成できる
  • Select Case文による多分岐処理を実装できる
  • 実務的な場面で適切な条件分岐の方法を選択できる

5.1 条件分岐とは？

条件分岐とは、ある条件が成り立つかどうかによって、実行する処理を変えることです。日常生活でも「もし雨なら傘を持っていく」「そうでなければ傘は不要」というように、条件に応じて行動を変えていますね。VBAでも同じように、プログラムの流れを条件によって変えることができます。

条件分岐の活用例

  • 成績判定: 点数が60点以上なら「合格」、未満なら「不合格」と表示
  • 在庫管理: 在庫数が10個未満なら警告メッセージを表示
  • 割引計算: 購入金額が1万円以上なら10%割引を適用
  • データ検証: 入力値が空白なら「入力してください」と促す

5.2 If～Then文の基本

最もシンプルな条件分岐がIf～Then文です。「もし〜ならば、〜を実行する」という意味になります。

基本構文

If～Then文の基本構文 
If 条件式 Then
    ' 条件が成立したときに実行する処理
End If

具体例: 点数判定

点数が60点以上なら「合格」と表示 
Sub 点数判定()
    Dim 点数 As Integer

    ' セルA1から点数を取得
    点数 = Range("A1").Value

    ' 60点以上なら合格と表示
    If 点数 >= 60 Then
        Range("B1").Value = "合格"
    End If
End Sub

比較演算子

条件式では以下の比較演算子を使います：

  • =: 等しい
  • <>: 等しくない
  • >: より大きい
  • <: より小さい
  • >=: 以上
  • <=: 以下

5.3 If～Then～Else文（2択分岐）

条件が成り立たない場合の処理も指定したいときは、Elseを使います。「もし〜ならば〜、そうでなければ〜」という2択の分岐になります。

基本構文

If～Then～Else文の基本構文 
If 条件式 Then
    ' 条件が成立したときの処理
Else
    ' 条件が成立しなかったときの処理
End If

具体例: 合格・不合格判定

60点以上なら「合格」、未満なら「不合格」 
Sub 合格判定()
    Dim 点数 As Integer

    点数 = Range("A1").Value

    If 点数 >= 60 Then
        Range("B1").Value = "合格"
        Range("B1").Font.Color = RGB(0, 128, 0)  ' 緑色
    Else
        Range("B1").Value = "不合格"
        Range("B1").Font.Color = RGB(255, 0, 0)  ' 赤色
    End If
End Sub

5.4 If～ElseIf～Else文（多分岐）

3つ以上の選択肢がある場合は、ElseIfを使って複数の条件を順番にチェックします。

基本構文

ElseIfを使った多分岐 
If 条件式1 Then
    ' 条件式1が成立したときの処理
ElseIf 条件式2 Then
    ' 条件式2が成立したときの処理
ElseIf 条件式3 Then
    ' 条件式3が成立したときの処理
Else
    ' すべての条件が成立しなかったときの処理
End If

具体例: 成績評価

点数に応じてA～Fの評価を付ける 
Sub 成績評価()
    Dim 点数 As Integer
    Dim 評価 As String

    点数 = Range("A1").Value

    If 点数 >= 90 Then
        評価 = "A（優秀）"
    ElseIf 点数 >= 80 Then
        評価 = "B（良好）"
    ElseIf 点数 >= 70 Then
        評価 = "C（普通）"
    ElseIf 点数 >= 60 Then
        評価 = "D（可）"
    Else
        評価 = "F（不可）"
    End If

    Range("B1").Value = 評価
End Sub

ポイント

条件は上から順番にチェックされ、最初に成立した条件の処理が実行されます。そのため、条件の順番が重要です。例えば、「80点以上」の条件を「90点以上」より先に書くと、90点以上の場合も「B」と判定されてしまいます。

5.5 複数条件の組み合わせ（And/Or）

複数の条件を同時にチェックしたい場合、AndやOrを使って条件を組み合わせることができます。

And（かつ）

すべての条件が成立する必要がある

If A And B Then

AもBも両方Trueのときだけ実行

Or（または）

いずれかの条件が成立すればよい

If A Or B Then

AまたはBのどちらかがTrueなら実行

具体例1: And演算子（割引判定）

購入金額が1万円以上かつ会員なら10%割引 
Sub 割引判定()
    Dim 購入金額 As Long
    Dim 会員フラグ As Boolean

    購入金額 = Range("A1").Value
    会員フラグ = Range("B1").Value  ' TRUE/FALSE

    ' 両方の条件を満たす場合のみ割引
    If 購入金額 >= 10000 And 会員フラグ = True Then
        Range("C1").Value = "10%割引適用"
    Else
        Range("C1").Value = "割引なし"
    End If
End Sub

具体例2: Or演算子（休日判定）

土曜日または日曜日なら「休日」と表示 
Sub 休日判定()
    Dim 曜日 As Integer

    ' 曜日を取得（1=日曜, 2=月曜, ..., 7=土曜）
    曜日 = Weekday(Range("A1").Value)

    ' 土曜(7)または日曜(1)なら休日
    If 曜日 = 1 Or 曜日 = 7 Then
        Range("B1").Value = "休日"
    Else
        Range("B1").Value = "平日"
    End If
End Sub

具体例3: 複雑な組み合わせ

AndとOrを組み合わせた条件 
Sub 送料判定()
    Dim 購入金額 As Long
    Dim 地域 As String

    購入金額 = Range("A1").Value
    地域 = Range("B1").Value

    ' 1万円以上、または北海道・沖縄以外なら送料無料
    If 購入金額 >= 10000 Or (地域 <> "北海道" And 地域 <> "沖縄") Then
        Range("C1").Value = "送料無料"
    Else
        Range("C1").Value = "送料500円"
    End If
End Sub

注意

AndとOrを組み合わせる場合、括弧 ( ) を使って優先順位を明確にすることをお勧めします。括弧を使わないと、意図しない条件判定になる可能性があります。

5.6 Select Case文（多分岐の別方法）

同じ変数に対して複数の値を比較する場合、Select Case文を使うとコードが読みやすくなります。ElseIfを何度も書くよりもスッキリします。

基本構文

Select Case文の基本構文 
Select Case 変数
    Case 値1
        ' 値1のときの処理
    Case 値2
        ' 値2のときの処理
    Case 値3
        ' 値3のときの処理
    Case Else
        ' どれにも該当しないときの処理
End Select

具体例: 曜日メッセージ

曜日に応じたメッセージを表示 
Sub 曜日メッセージ()
    Dim 曜日 As Integer
    Dim メッセージ As String

    曜日 = Weekday(Date)  ' 今日の曜日を取得

    Select Case 曜日
        Case 1  ' 日曜日
            メッセージ = "今日は日曜日です"
        Case 2  ' 月曜日
            メッセージ = "今週も頑張りましょう"
        Case 6  ' 金曜日
            メッセージ = "明日は休みです"
        Case 7  ' 土曜日
            メッセージ = "今日は土曜日です"
        Case Else  ' その他の平日
            メッセージ = "今日も一日頑張りましょう"
    End Select

    MsgBox メッセージ
End Sub

範囲指定や複数値の指定

Select Caseの応用テクニック 
Sub 成績評価2()
    Dim 点数 As Integer

    点数 = Range("A1").Value

    Select Case 点数
        Case Is >= 90
            Range("B1").Value = "A（優秀）"
        Case Is >= 80
            Range("B1").Value = "B（良好）"
        Case 70 To 79  ' 範囲指定
            Range("B1").Value = "C（普通）"
        Case 60, 61, 62, 63, 64, 65, 66, 67, 68, 69  ' 複数値指定
            Range("B1").Value = "D（可）"
        Case Else
            Range("B1").Value = "F（不可）"
    End Select
End Sub

Select Caseで使えるパターン

  • Case 値: 特定の値と一致
  • Case Is >= 値: 比較演算子を使う
  • Case 値1 To 値2: 範囲指定
  • Case 値1, 値2, 値3: 複数の値を列挙

5.7 実務的な例題

実習 5-1: 在庫警告システムを作る

在庫数に応じて警告メッセージを表示するマクロを作成します。在庫が10個未満なら「要発注」、10～30個なら「注意」、31個以上なら「十分」と表示します。

手順

  1. 1 セルA1に在庫数（例: 25）を入力
  2. 2 VBEで新しいモジュールを作成
  3. 3 以下のコードを入力してF5で実行
在庫警告マクロ 
Sub 在庫警告()
    Dim 在庫数 As Integer
    Dim メッセージ As String
    Dim 色 As Long

    在庫数 = Range("A1").Value

    If 在庫数 < 10 Then
        メッセージ = "要発注"
        色 = RGB(255, 0, 0)  ' 赤
    ElseIf 在庫数 <= 30 Then
        メッセージ = "注意"
        色 = RGB(255, 165, 0)  ' オレンジ
    Else
        メッセージ = "十分"
        色 = RGB(0, 128, 0)  ' 緑
    End If

    Range("B1").Value = メッセージ
    Range("B1").Font.Color = 色
End Sub

実習 5-2: BMI判定プログラムを作る

身長と体重からBMIを計算し、やせ・普通・肥満を判定するマクロを作成します。

BMI判定基準

  • • BMI < 18.5: やせ
  • • 18.5 ≦ BMI < 25: 普通
  • • 25 ≦ BMI: 肥満
BMI判定マクロ 
Sub BMI判定()
    Dim 身長 As Double  ' cm
    Dim 体重 As Double  ' kg
    Dim BMI As Double
    Dim 判定 As String

    ' A1に身長、B1に体重を入力
    身長 = Range("A1").Value
    体重 = Range("B1").Value

    ' BMI計算（身長をmに変換）
    BMI = 体重 / ((身長 / 100) ^ 2)

    ' 判定
    If BMI < 18.5 Then
        判定 = "やせ"
    ElseIf BMI < 25 Then
        判定 = "普通"
    Else
        判定 = "肥満"
    End If

    ' 結果表示
    Range("C1").Value = Format(BMI, "0.0")
    Range("D1").Value = 判定
End Sub

理解度確認クイズ

  1. Q1. 以下のコードで、変数Aが10のとき、セルB1には何が表示されますか？

    If A > 5 Then
      Range("B1").Value = "大きい"
    Else
      Range("B1").Value = "小さい"
    End If
    回答を表示

    「大きい」が表示されます。Aは10なので、条件式「A > 5」はTrueとなり、Then以下の処理が実行されます。

  2. Q2. And演算子とOr演算子の違いを説明してください。

    回答を表示

    And: すべての条件が成立する（True）必要があります。1つでもFalseがあれば全体がFalseになります。
    Or: いずれか1つでも条件が成立すれば全体がTrueになります。すべてFalseの場合のみ全体がFalseになります。

  3. Q3. Select Case文が便利な場面を1つ挙げてください。

    回答を表示

    同じ変数に対して複数の値を比較する場合に便利です。例えば、曜日の判定、月の判定、成績評価など、ElseIfを何度も書くよりもSelect Caseを使った方がコードが読みやすくなります。

  4. Q4. 以下の条件式で、変数Aが15、変数Bが20のとき、全体の結果はTrueですかFalseですか？

    If A > 10 And B < 15 Then
    回答を表示

    Falseです。「A > 10」はTrue（15 > 10）ですが、「B < 15」はFalse（20 < 15は成立しない）です。And演算子はすべての条件がTrueである必要があるため、全体としてFalseになります。

セルとワークシートの操作

RangeオブジェクトとCellsプロパティを使ったセルの操作、値の取得と設定、書式設定の自動化、ワークシートの操作方法を学びます。

4.1 Rangeオブジェクトの基本

VBAでExcelのセルを操作するには、Rangeオブジェクトを使います。Rangeオブジェクトは、1つまたは複数のセル範囲を表すオブジェクトで、値の読み書きや書式設定など、さまざまな操作が可能です。

Rangeオブジェクトの基本的な使い方

セルを参照する基本パターン 
Sub セル参照の基本()
    ' 単一セルの参照
    Range("A1").Value = "こんにちは"

    ' セル範囲の参照
    Range("A1:B3").Value = "一括入力"

    ' 複数の離れたセルを参照
    Range("A1,C3,E5").Value = "飛び飛び"
End Sub

Range指定のポイント

  • Range("A1"): A1セルを参照
  • Range("A1:B3"): A1からB3までの範囲を参照
  • Range("A1,C3"): 複数の離れたセルを参照（カンマ区切り）
  • .Valueプロパティで値にアクセス

4.2 Cellsプロパティ - 番号でセルを指定

Cellsプロパティは、行番号と列番号でセルを指定する方法です。ループ処理などでセルを順番に操作する場合に特に便利です。

Cellsプロパティの基本 
Sub Cells使用例()
    ' Cells(行番号, 列番号) で指定
    Cells(1, 1).Value = "A1セル"    ' 1行目、1列目（A1）
    Cells(2, 3).Value = "C2セル"    ' 2行目、3列目（C2）

    ' 変数を使った柔軟な指定が可能
    Dim i As Integer
    For i = 1 To 5
        Cells(i, 1).Value = i & "行目"  ' A列に行番号を表示
    Next i
End Sub

RangeとCellsの使い分け

Range: 決まったセルを参照する場合に便利
例: Range("A1").Value = "タイトル"
Cells: ループ処理や動的なセル参照に便利
例: Cells(i, 1).Value = データ

4.3 セルの値と書式の操作

セルに対して、値の読み書きだけでなく、書式設定（色、フォント、罫線など）も自動化できます。

値の取得と設定

値の読み書き 
Sub 値の操作()
    ' 値の設定
    Range("A1").Value = 100
    Range("A2").Value = "文字列"
    Range("A3").Value = Date  ' 今日の日付

    ' 値の取得
    Dim セルの値 As Variant
    セルの値 = Range("A1").Value
    MsgBox "A1の値は " & セルの値

    ' セルの値を使った計算
    Range("A4").Value = Range("A1").Value * 2
End Sub

書式設定の自動化

セルの書式設定 
Sub 書式設定()
    ' フォント設定
    Range("A1").Font.Size = 14
    Range("A1").Font.Bold = True
    Range("A1").Font.Color = RGB(255, 0, 0)  ' 赤色

    ' 背景色設定
    Range("A2").Interior.Color = RGB(255, 255, 0)  ' 黄色

    ' 罫線設定
    Range("A1:B3").Borders.LineStyle = xlContinuous

    ' 数値書式
    Range("C1").NumberFormat = "#,##0"  ' 3桁区切り
End Sub

実習 4-1: 成績表を書式設定するマクロ

セルに成績データを入力し、条件に応じて書式を設定します。

成績表書式設定 
Sub 成績表書式設定()
    ' ヘッダー作成
    Range("A1").Value = "氏名"
    Range("B1").Value = "点数"
    Range("A1:B1").Font.Bold = True

    ' サンプルデータ入力
    Range("A2").Value = "田中"
    Range("B2").Value = 85
    Range("A3").Value = "佐藤"
    Range("B3").Value = 65

    ' 点数に応じた書式設定
    If Range("B2").Value >= 80 Then
        Range("B2").Interior.Color = RGB(144, 238, 144)  ' 緑（合格）
    Else
        Range("B2").Interior.Color = RGB(255, 182, 193)  ' 赤（要努力）
    End If
End Sub

4.4 ワークシートの操作

複数のシートを扱う場合、シートの追加・削除・名前変更・アクティブ化などの操作が必要になります。

シートの参照方法

シート参照の基本 
Sub シート参照()
    ' シート名で参照
    Worksheets("Sheet1").Range("A1").Value = "データ"

    ' インデックス番号で参照（左から1, 2, 3...）
    Worksheets(1).Range("A1").Value = "1番目のシート"

    ' アクティブシートを参照
    ActiveSheet.Range("A1").Value = "現在のシート"
End Sub

シートの追加・削除

シート操作 
Sub シート操作()
    ' 新しいシートを追加
    Worksheets.Add

    ' シート名を変更
    ActiveSheet.Name = "新規シート"

    ' 特定のシートを削除（確認なし）
    Application.DisplayAlerts = False  ' 警告を非表示
    Worksheets("削除対象").Delete
    Application.DisplayAlerts = True   ' 警告を再表示

    ' シートをアクティブにする
    Worksheets("Sheet1").Activate
End Sub

シート削除時の注意

Worksheets.Deleteは元に戻せません。削除前にApplication.DisplayAlerts = Falseで警告を無効にすると、確認なしで削除されるので注意してください。

4.5 実践例: 月別売上シートの自動作成

ここまで学んだ知識を組み合わせて、12ヶ月分の売上シートを自動作成するマクロを作ります。

月別シート自動作成 
Sub 月別シート作成()
    Dim i As Integer
    Dim 月名 As String

    ' 1月から12月までループ
    For i = 1 To 12
        月名 = i & "月"

        ' 新しいシートを追加
        Worksheets.Add After:=Worksheets(Worksheets.Count)
        ActiveSheet.Name = 月名

        ' ヘッダー作成
        Range("A1:C1").Value = Array("日付", "商品名", "売上")
        Range("A1:C1").Font.Bold = True
        Range("A1:C1").Interior.Color = RGB(200, 220, 255)

        ' 日付を自動入力（1日〜31日）
        Dim j As Integer
        For j = 1 To 31
            Cells(j + 1, 1).Value = i & "/" & j
        Next j
    Next i

    MsgBox "12ヶ月分のシートを作成しました！"
End Sub

動作確認

  1. 上記コードをVBEに貼り付けて実行（F5）
  2. 1月〜12月の12個のシートが自動作成される
  3. 各シートには日付列とヘッダーが自動設定される

手作業で12回繰り返す作業が、マクロ実行1回で完了します！

理解度確認クイズ

  1. Q1. セルA1からB5までの範囲を参照するRangeオブジェクトの書き方は？

    回答を表示

    Range("A1:B5")です。開始セルと終了セルをコロン（:）でつなぎます。

  2. Q2. Cells(3, 2)はどのセルを指しますか？

    回答を表示

    B3セルです。Cells(行番号, 列番号)なので、3行目の2列目（B列）となります。

  3. Q3. セルA1のフォントを太字にするコードを書いてください。

    回答を表示

    Range("A1").Font.Bold = True

  4. Q4. ワークシートを削除する際に確認ダイアログを表示しないようにするには？

    回答を表示

    削除前にApplication.DisplayAlerts = Falseを設定し、削除後にApplication.DisplayAlerts = Trueで元に戻します。

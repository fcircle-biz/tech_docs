VBAとは？プログラミングの第一歩

VBAの概要とExcel自動化のメリットを理解し、開発環境（VBE）の基本操作とはじめてのマクロ記録を体験します。プログラミング未経験者でも安心して始められる内容です。

1.1 VBAとは何か？

VBA（Visual Basic for Applications）は、Microsoft Officeアプリケーション（Excel、Word、Access、PowerPointなど）に組み込まれたプログラミング言語です。Excelの場合、繰り返し行う作業を自動化したり、複雑な処理を簡単に実行できるようにしたりするための強力なツールです。

VBAの特徴

  • Excelに統合: 追加ソフト不要で、Excelがあればすぐに使える
  • 初心者にやさしい: Visual Basicベースで文法がシンプル、日本語コメントも使える
  • マクロ記録機能: 操作を記録してコード生成できるため、学習の入り口が広い
  • 業務直結: 日常的なExcel作業をすぐに自動化できる実用性

1.2 VBAでできること・自動化のメリット

VBAを使うと、今まで手作業で行っていた業務を大幅に効率化できます。以下のような作業が自動化可能です。

データ整形・加工

大量のデータの書式統一、不要な空白削除、データの並べ替えなどを一瞬で処理

レポート・請求書作成

定型フォーマットへのデータ自動入力、複数シート一括作成、PDF出力まで自動化

複雑な集計・分析

ピボットテーブル自動作成、条件に応じた集計、グラフの自動生成

複数ファイル連携

複数のExcelファイルからデータ収集、CSVインポート、他システムへのデータ出力

自動化のメリット

  • 時間短縮: 30分かかる作業が数秒で完了
  • ミス削減: 手作業のコピペミスや入力ミスがなくなる
  • 品質向上: 常に同じ処理で一貫性のある結果
  • 再利用性: 一度作れば何度でも使える

1.3 VBE（Visual Basic Editor）の起動と基本画面

VBAのコードを書くには、VBE（Visual Basic Editor）という専用のエディタを使います。まずはVBEの起動方法と基本的な画面構成を確認しましょう。

開発タブの表示

Excelでは、デフォルトで「開発」タブが非表示になっています。まずはこれを有効化します。

開発タブの表示手順

  1. Excelを起動し、ファイル → オプションをクリック
  2. 左側メニューからリボンのユーザー設定を選択
  3. 右側の「メインタブ」一覧で「開発」にチェックを入れる
  4. OKをクリック

これでExcelのリボンに「開発」タブが表示されます。

VBEの起動方法

ショートカット: Alt + F11

または、「開発」タブ → 「Visual Basic」ボタンをクリック

VBEの主要ウィンドウ

  • プロジェクトエクスプローラー（左上）

  • プロパティウィンドウ（左下）

1.4 はじめてのマクロ記録

VBAを学ぶ第一歩として、マクロ記録機能を使ってみましょう。マクロ記録は、Excelの操作を記録してVBAコードに変換してくれる便利な機能です。

実習 1-1: セルに文字を入力するマクロを記録する

マクロ記録機能を使って、セルA1に「こんにちは」と入力する操作を記録します。

手順

  1. 1 Excelで新しいブックを開く
  2. 2 「開発」タブ → 「マクロの記録」ボタンをクリック
  3. 3 マクロ名を「挨拶マクロ」と入力し、OKをクリック（記録開始）
  4. 4 セルA1をクリックして選択
  5. 5 「こんにちは」と入力し、Enterキーを押す
  6. 6 「開発」タブ → 「記録終了」ボタンをクリック

これで、セルA1に「こんにちは」と入力する操作がVBAコードとして記録されました！

1.5 記録したマクロのコードを見てみよう

記録したマクロがどのようなVBAコードになっているのか確認してみましょう。

コードの確認方法

  1. Alt + F11でVBEを起動
  2. プロジェクトエクスプローラーから「標準モジュール」→「Module1」をダブルクリック
  3. コードウィンドウに記録されたVBAコードが表示されます
Module1（自動生成されたコード例） 
Sub 挨拶マクロ()
'
' 挨拶マクロ Macro
'

'
    Range("A1").Select
    ActiveCell.FormulaR1C1 = "こんにちは"
    Range("A2").Select
End Sub

コードの読み方

  • Sub 挨拶マクロ(): マクロの開始宣言。この名前で呼び出せる
  • Range("A1").Select: セルA1を選択する命令
  • ActiveCell.FormulaR1C1 = "こんにちは": 選択中のセルに「こんにちは」を入力
  • End Sub: マクロの終了

マクロの実行方法

記録したマクロを実行するには、以下のいずれかの方法を使います。

方法1: マクロ一覧から実行

  1. 「開発」タブ → 「マクロ」
  2. 「挨拶マクロ」を選択
  3. 「実行」をクリック

方法2: VBEから実行

  1. VBEでコードウィンドウを開く
  2. マクロ内にカーソルを置く
  3. F5キーを押す

理解度確認クイズ

  1. Q1. VBAとは何の略称ですか？また、どのような用途で使われますか？

    回答を表示

    Visual Basic for Applicationsの略です。Microsoft Officeアプリケーション（Excel、Word、Accessなど）で繰り返し作業の自動化や複雑な処理を簡単に実行するために使われるプログラミング言語です。

  2. Q2. VBEを起動するキーボードショートカットは何ですか？

    回答を表示

    Alt + F11です。このショートカットでVisual Basic Editorが開きます。

  3. Q3. マクロ記録機能の利点を2つ挙げてください。

    回答を表示

    ① プログラミング初心者でも操作を記録するだけでVBAコードを生成できる
    ② 生成されたコードを見ることで、VBAの書き方を学習できる

  4. Q4. VBEの「コードウィンドウ」は何をする場所ですか？

    回答を表示

    VBAコードを実際に記述・編集する場所です。マクロ記録で自動生成されたコードもここに表示され、手動で修正や新規作成ができます。

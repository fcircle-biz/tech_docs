第10章

ユーザーフォームで対話型ツールを作る

ユーザーフォームの作成と基本コントロールの配置、イベント処理の実装、入力検証とデータ受け渡しの方法を学びます。

この章で学ぶこと

  • ユーザーフォームの役割と作成方法を理解する
  • テキストボックス、ラベル、ボタンなど基本コントロールの配置と設定方法を習得する
  • イベント処理（クリック、変更など）の実装方法を学ぶ
  • 入力検証とエラーメッセージの表示方法を理解する
  • フォームとワークシート間のデータ受け渡しを実装できるようになる

10.1 ユーザーフォームとは？

ユーザーフォームは、VBAでカスタムの入力画面を作成する機能です。ユーザーとのやり取りを簡単にし、データ入力やツールの操作をより直感的に行えるようにします。

ユーザーフォームでできること

  • データ入力: テキストボックスやドロップダウンリストで情報を入力
  • 選択操作: チェックボックスやオプションボタンで選択
  • ボタン操作: 処理を実行するボタンを配置
  • エラー表示: 入力ミスをその場で確認・修正

重要ポイント

ユーザーフォームを使うことで、Excelに慣れていない人でも簡単に操作できるツールを作成できます。業務ツールの使いやすさが大幅に向上します。

10.2 ユーザーフォームの作成と基本操作

まずは、VBEでユーザーフォームを作成する基本手順を学びましょう。

実習 10-1: 最初のユーザーフォームを作成する

簡単な入力フォームを作成し、VBAでフォームを表示する方法を体験します。

手順

  1. 1 VBEを開く（Alt + F11）
  2. 2 メニューバーから「挿入」→「ユーザーフォーム」をクリック
  3. 3 空のフォームとツールボックスが表示される
  4. 4 フォーム上で右クリック→「プロパティ」を選択
  5. 5 プロパティウィンドウで「Caption」を「データ入力フォーム」に変更

これで最初のユーザーフォームが作成されました！

フォームを表示するコード

作成したフォームを表示するには、標準モジュールに以下のコードを記述します。

標準モジュール 
Sub フォーム表示()
    ' UserForm1 を表示
    UserForm1.Show
End Sub

ヒント

UserForm1.Showでフォームを表示、UserForm1.Hideでフォームを非表示にできます。

10.3 基本コントロールの配置

フォームに配置できる主要なコントロールと、その使い方を学びます。

主要なコントロール

Label（ラベル）

説明文や項目名を表示する。編集不可のテキスト表示用。

TextBox（テキストボックス）

ユーザーが文字や数値を入力する欄。最も基本的な入力コントロール。

CommandButton（ボタン）

クリックすると処理を実行する。OKボタン、キャンセルボタンなど。

ComboBox（ドロップダウン）

リストから選択する入力方式。選択肢が多い場合に便利。

CheckBox（チェックボックス）

オン/オフの選択。複数選択可能。

OptionButton（ラジオボタン）

複数の選択肢から1つだけ選ぶ。排他的な選択。

実習 10-2: コントロールを配置して名前入力フォームを作る

ラベル、テキストボックス、ボタンを配置して、シンプルな入力フォームを作成します。

手順

  1. 1 ツールボックスから「Label」を選択し、フォーム上に配置
  2. 2 プロパティで「Caption」を「お名前:」に変更
  3. 3 「TextBox」を配置し、オブジェクト名を「txtName」に変更
  4. 4 「CommandButton」を配置し、「Caption」を「登録」に、オブジェクト名を「btnRegister」に変更
  5. 5 もう1つ「CommandButton」を配置し、「Caption」を「キャンセル」に、オブジェクト名を「btnCancel」に変更

ヒント

コントロールのオブジェクト名は、プロパティウィンドウの「(オブジェクト名)」で変更できます。わかりやすい名前を付けると、コードを書くときに便利です。

10.4 イベント処理の実装

ユーザーがボタンをクリックしたり、テキストを入力したりしたときに実行される処理をイベント処理と呼びます。

主要なイベント

  • Click（クリック）
    ボタンなどがクリックされたときに発生
  • Change（変更）
    テキストボックスの内容が変更されたときに発生
  • Initialize（初期化）
    フォームが最初に読み込まれるときに発生

ボタンのClickイベント実装

「登録」ボタンがクリックされたときの処理を実装します。ボタンをダブルクリックすると、イベントプロシージャが自動生成されます。

UserForm1（登録ボタンのClickイベント） 
Private Sub btnRegister_Click()
    ' 入力された名前を取得
    Dim userName As String
    userName = txtName.Value

    ' セルA1に名前を書き込む
    Range("A1").Value = userName

    ' メッセージを表示
    MsgBox "「" & userName & "」を登録しました", vbInformation

    ' フォームを閉じる
    Me.Hide
End Sub

キャンセルボタンのClickイベント

UserForm1（キャンセルボタンのClickイベント） 
Private Sub btnCancel_Click()
    ' フォームを閉じる（何もしない）
    Me.Hide
End Sub

コードの解説

  • txtName.Value: テキストボックスに入力された値を取得
  • Me.Hide: 現在のフォーム自身を非表示にする
  • vbInformation: 情報アイコン付きメッセージボックス

10.5 入力検証の実装

ユーザーが正しい形式でデータを入力しているかチェックし、誤りがあればエラーメッセージを表示する入力検証を実装しましょう。

未入力チェック

入力検証の例 
Private Sub btnRegister_Click()
    ' 入力された名前を取得
    Dim userName As String
    userName = Trim(txtName.Value) ' 前後の空白を削除

    ' 未入力チェック
    If userName = "" Then
        MsgBox "お名前を入力してください", vbExclamation
        txtName.SetFocus ' カーソルを移動
        Exit Sub ' 処理を中断
    End If

    ' セルA1に名前を書き込む
    Range("A1").Value = userName

    ' メッセージを表示
    MsgBox "「" & userName & "」を登録しました", vbInformation

    ' フォームを閉じる
    Me.Hide
End Sub

数値入力のチェック

数値を入力するフォームの場合、入力内容が数値かどうかをチェックします。

数値チェックの例 
Private Sub btnRegister_Click()
    ' 年齢を取得
    Dim age As String
    age = Trim(txtAge.Value)

    ' 未入力チェック
    If age = "" Then
        MsgBox "年齢を入力してください", vbExclamation
        txtAge.SetFocus
        Exit Sub
    End If

    ' 数値チェック
    If Not IsNumeric(age) Then
        MsgBox "年齢は数値で入力してください", vbExclamation
        txtAge.SetFocus
        Exit Sub
    End If

    ' 登録処理
    Range("B1").Value = age
    MsgBox "年齢を登録しました", vbInformation
    Me.Hide
End Sub

重要ポイント

IsNumeric()関数で数値判定、Trim()関数で前後の空白削除、SetFocusメソッドでカーソルを該当コントロールに移動できます。

10.6 フォームとワークシート間のデータ受け渡し

実務では、フォームからワークシートにデータを書き込んだり、逆にワークシートのデータをフォームに読み込んだりする処理が必要です。

フォームからワークシートへの書き込み

複数項目をシートに追加する例 
Private Sub btnRegister_Click()
    Dim lastRow As Long

    ' 最終行を取得
    lastRow = Cells(Rows.Count, 1).End(xlUp).Row + 1

    ' データを書き込む
    Cells(lastRow, 1).Value = txtName.Value   ' 名前
    Cells(lastRow, 2).Value = txtAge.Value    ' 年齢
    Cells(lastRow, 3).Value = cmbDept.Value   ' 部署

    MsgBox "データを登録しました", vbInformation

    ' フォームをクリア
    txtName.Value = ""
    txtAge.Value = ""
    cmbDept.Value = ""
    txtName.SetFocus
End Sub

ワークシートからフォームへの読み込み

フォーム表示時に、ワークシートのデータをフォームに読み込むには、Initializeイベントを使います。

フォーム初期化時にドロップダウンを設定 
Private Sub UserForm_Initialize()
    ' コンボボックスに選択肢を追加
    With cmbDept
        .AddItem "営業部"
        .AddItem "開発部"
        .AddItem "総務部"
        .AddItem "経理部"
    End With

    ' デフォルト値を設定
    cmbDept.ListIndex = 0 ' 最初の項目を選択
End Sub

コードの解説

  • Cells(Rows.Count, 1).End(xlUp).Row: 最終行を取得
  • .AddItem: コンボボックスに項目を追加
  • .ListIndex = 0: 最初の項目を選択状態にする

10.7 実践例: 社員登録フォームを作る

これまで学んだ内容を総合して、実用的な社員登録フォームを作成します。

実習 10-3: 社員登録フォームの作成

名前、年齢、部署、メールアドレスを入力するフォームを作成し、入力検証を実装します。

フォームの構成

  • Label: 「氏名:」「年齢:」「部署:」「メール:」
  • TextBox: txtName, txtAge, txtEmail
  • ComboBox: cmbDept（部署選択用）
  • CommandButton: btnRegister（登録）、btnCancel（キャンセル）

完成コード例

UserForm1（完成版） 
' フォーム初期化
Private Sub UserForm_Initialize()
    With cmbDept
        .AddItem "営業部"
        .AddItem "開発部"
        .AddItem "総務部"
    End With
End Sub

' 登録ボタン
Private Sub btnRegister_Click()
    ' 入力検証
    If Trim(txtName.Value) = "" Then
        MsgBox "氏名を入力してください"
        txtName.SetFocus
        Exit Sub
    End If

    ' 最終行に追加
    Dim lastRow As Long
    lastRow = Cells(Rows.Count, 1).End(xlUp).Row + 1
    Cells(lastRow, 1).Value = txtName.Value
    Cells(lastRow, 2).Value = txtAge.Value
    Cells(lastRow, 3).Value = cmbDept.Value
    Cells(lastRow, 4).Value = txtEmail.Value

    MsgBox "登録完了！"
    Me.Hide
End Sub

' キャンセル
Private Sub btnCancel_Click()
    Me.Hide
End Sub

理解度確認クイズ

  1. Q1. ユーザーフォームを表示するコマンドは何ですか？

    回答を表示

    UserForm1.Showでフォームを表示できます。非表示にするにはUserForm1.Hideを使います。

  2. Q2. テキストボックスに入力された値を取得するには、どのプロパティを使いますか？

    回答を表示

    .Valueプロパティです。例: txtName.Value

  3. Q3. 入力内容が数値かどうかをチェックする関数は何ですか？

    回答を表示

    IsNumeric()関数です。使用例: If IsNumeric(txtAge.Value) Then

  4. Q4. フォームが最初に表示されるときに実行されるイベントは何ですか？

    回答を表示

    UserForm_Initializeイベントです。フォームの初期設定（コンボボックスへの項目追加など）に使います。

繰り返し処理（ループ）で効率化

For～Next、For Each、Do～Loop文を使った繰り返し処理の実装方法と、大量データの一括処理テクニックを学びます。

この章で学ぶこと

  • For～Next文を使った指定回数の繰り返し処理
  • For Each文によるコレクション・配列の反復処理
  • Do～Loop文を使った条件付きループの実装
  • Exit For/Exit Doによるループの途中抜けとネストしたループの活用

6.1 ループ（繰り返し処理）とは何か？

ループ（繰り返し処理）とは、同じ処理を何度も繰り返し実行する仕組みです。Excelで大量のデータを処理する際に欠かせない技術で、手作業では何時間もかかる作業を数秒で完了させることができます。

ループが活躍する場面

  • 大量データの一括処理: 1000行のデータを1行ずつチェック・加工
  • 複数シートの処理: すべてのワークシートに同じ処理を適用
  • 条件を満たすまで繰り返す: 特定の値が見つかるまで検索
  • 一定回数の処理: 請求書を10枚作成するなど

6.2 For～Next文：指定回数の繰り返し

For～Next文は、指定した回数だけ処理を繰り返すループです。カウンター変数を使って「1から10まで」「2行目から100行目まで」のように範囲を指定します。

基本構文

For～Next文の基本形 
For カウンター変数 = 開始値 To 終了値 Step 増分
    ' 繰り返す処理
Next カウンター変数

For文の構成要素

  • カウンター変数: ループの回数を数える変数（通常はi, j, kなどを使用）
  • 開始値 To 終了値: ループの範囲（例: 1 To 10）
  • Step 増分: 1回ごとの増加量（省略時は1ずつ増加）

実習 6-1: For～Next文で1から10までの数値を表示する

For～Next文を使って、セルA1からA10に1から10までの数値を順番に入力します。

実習6-1のコード 
Sub 数値を表示()
    Dim i As Integer

    ' 1から10まで繰り返す
    For i = 1 To 10
        Cells(i, 1).Value = i  ' i行目のA列に数値iを入力
    Next i

    MsgBox "1から10まで入力完了！"
End Sub

実行すると、A1に「1」、A2に「2」…A10に「10」が入力されます！

Step指定の活用

Stepを使うと、増加量を変更したり、逆順に処理したりできます。

Step指定の例 
Sub Stepの例()
    Dim i As Integer

    ' 2ずつ増加（2, 4, 6, 8, 10）
    For i = 2 To 10 Step 2
        Cells(i / 2, 1).Value = i
    Next i

    ' 逆順に処理（10から1まで）
    For i = 10 To 1 Step -1
        Cells(11 - i, 2).Value = i
    Next i
End Sub

6.3 実践例：大量データの一括処理

For～Next文を使って、データの一括処理を実装してみましょう。

実習 6-2: 消費税額を自動計算する

B列に入力された価格（税抜）に対して、C列に消費税額（10%）を自動計算します。

実習6-2のコード 
Sub 消費税計算()
    Dim i As Integer
    Dim 価格 As Long
    Dim 消費税額 As Long

    ' 2行目から100行目まで処理
    For i = 2 To 100
        価格 = Cells(i, 2).Value  ' B列の価格を取得

        ' 価格が0でない場合のみ計算
        If 価格 > 0 Then
            消費税額 = 価格 * 0.1
            Cells(i, 3).Value = 消費税額  ' C列に消費税額を入力
        End If
    Next i

    MsgBox "消費税計算完了！"
End Sub

ループの範囲は、実際のデータ行数に合わせて調整してください。空白行をスキップするために If文 を活用しています。

6.4 For Each文：コレクション・セル範囲の反復処理

For Each文は、コレクション（ワークシート、セル範囲など）の各要素に対して順番に処理を実行します。範囲が動的に変わる場合や、すべての要素を処理したい場合に便利です。

基本構文

For Each文の基本形 
For Each 要素変数 In コレクション
    ' 各要素に対する処理
Next 要素変数

実習 6-3: For Each文でセル範囲を処理する

A1:A10の範囲にある数値をすべて2倍にします。

実習6-3のコード 
Sub セル範囲を2倍にする()
    Dim セル As Range

    ' A1:A10の各セルに対して処理
    For Each セル In Range("A1:A10")
        If IsNumeric(セル.Value) Then  ' 数値の場合のみ
            セル.Value = セル.Value * 2
        End If
    Next セル

    MsgBox "2倍にする処理完了！"
End Sub

For Each文は、セルの個数を気にせず範囲全体を処理できます！

ワークシートのループ処理

For Each文は、すべてのワークシートに対して同じ処理を実行する際にも活用できます。

全シートの処理例 
Sub 全シートにタイトル追加()
    Dim ws As Worksheet

    ' ブック内のすべてのシートに対して処理
    For Each ws In ThisWorkbook.Worksheets
        ws.Range("A1").Value = "データ一覧"
        ws.Range("A1").Font.Bold = True
    Next ws

    MsgBox "全シートにタイトル追加完了！"
End Sub

6.5 Do～Loop文：条件付きループ

Do～Loop文は、条件を満たしている間、または条件を満たすまで処理を繰り返します。回数が不明な場合や、特定の条件まで処理したい場合に使用します。

4つのパターン

Do While ～ Loop

条件がTrueの間繰り返す（前判定）

Do While 条件
    ' 処理
Loop

Do Until ～ Loop

条件がTrueになるまで繰り返す（前判定）

Do Until 条件
    ' 処理
Loop

Do ～ Loop While

条件がTrueの間繰り返す（後判定）

Do
    ' 処理
Loop While 条件

Do ～ Loop Until

条件がTrueになるまで繰り返す（後判定）

Do
    ' 処理
Loop Until 条件

実習 6-4: Do Until文でデータの最終行を探す

A列のデータが空白になるまで、各セルに連番を振ります。

実習6-4のコード 
Sub 連番を振る()
    Dim 行 As Integer

    行 = 2  ' 2行目から開始

    ' A列が空白になるまで繰り返す
    Do Until Cells(行, 1).Value = ""
        Cells(行, 2).Value = 行 - 1  ' B列に連番を入力
        行 = 行 + 1
    Loop

    MsgBox (行 - 2) & "件のデータに連番を振りました！"
End Sub

Do Until文は、データの件数が不明な場合に便利です！

6.6 Exit文とネストしたループ

ループ処理をより柔軟に制御する方法を学びます。

Exit For / Exit Do：ループの途中抜け

条件を満たしたらループを途中で終了する場合に使用します。

Exit Forの使用例 
Sub 特定の値を探す()
    Dim i As Integer

    For i = 1 To 100
        If Cells(i, 1).Value = "完了" Then
            MsgBox i & "行目に「完了」を発見！"
            Exit For  ' ループを抜ける
        End If
    Next i
End Sub

ネストしたループ（二重ループ）

ループの中にさらにループを入れる「ネスト」を使うと、行と列の両方を処理できます。

二重ループの例 
Sub 九九の表を作成()
    Dim 行 As Integer
    Dim 列 As Integer

    ' 外側のループ（行）
    For 行 = 1 To 9
        ' 内側のループ（列）
        For 列 = 1 To 9
            Cells(行, 列).Value = 行 * 列
        Next 列
    Next 行

    MsgBox "九九の表を作成しました！"
End Sub

ネストの深さに注意

ループを何重にもネストすると、処理速度が極端に遅くなります。できるだけシンプルな構造を心がけましょう。

理解度確認クイズ

  1. Q1. For～Next文で10から1まで逆順に処理するには、どう書けばよいですか？

    回答を表示

    For i = 10 To 1 Step -1 と書きます。Step -1で1ずつ減少させます。

  2. Q2. For Each文とFor～Next文の違いは何ですか？

    回答を表示

    For～Next文は回数（開始値～終了値）を指定して繰り返します。For Each文はコレクション（セル範囲、ワークシートなど）の各要素に対して順番に処理を行います。要素数が不明な場合や、全要素を処理したい場合はFor Each文が便利です。

  3. Q3. Do While文とDo Until文の違いを説明してください。

    回答を表示

    Do While文は「条件がTrueの間」繰り返します。Do Until文は「条件がTrueになるまで」繰り返します。While = 〜の間、Until = 〜まで、と覚えましょう。

  4. Q4. Exit Forはどのような場面で使用しますか？

    回答を表示

    ループの途中で特定の条件を満たしたときに、ループを終了したい場合に使用します。例えば、データの中から特定の値を検索し、見つかったらそこでループを抜ける場合などです。

第8章: VBA基礎

8.1 VBAとは何か

VBA（Visual Basic for Applications）は、Microsoft Office製品で使用できるプログラミング言語です。繰り返し作業の自動化や複雑な処理の簡略化に威力を発揮します。

VBAでできること
  • 作業の自動化：繰り返し処理を一瞬で実行
  • 複雑な計算：関数だけでは難しい処理を実現
  • ユーザーインターフェース：ボタンやフォームで操作性向上
  • 他アプリ連携：Word、PowerPointとの連携
  • データ処理：大量データの効率的な処理
段階 方法 特徴 適用場面
第1段階 手動操作 時間がかかる、ミスが発生しやすい シンプルな作業、一回限りの作業
第2段階 関数による自動計算 計算は自動、操作は手動 定型的な計算、データ集計
第3段階 マクロ記録 操作の記録・再生、単純作業に最適 繰り返し作業、定型操作
第4段階 VBAプログラミング 完全自動化、柔軟な処理が可能 複雑なロジック、条件分岐、エラー処理
VBA学習前の注意事項
  • セキュリティ：VBAマクロには潜在的なセキュリティリスクがある
  • 互換性：Excel以外の環境では動作しない
  • 学習コスト：プログラミングの基礎知識が必要
  • 保守性：コードの管理とメンテナンスが必要

8.2 開発環境の準備

VBAを使用するために必要な設定と環境構築を行いましょう。

実習 8-1: 開発タブの表示とVBE起動

VBA開発に必要な環境を準備しましょう。

手順
  1. 開発タブを表示：
    • ファイル → オプション → リボンのユーザー設定
    • 「開発」にチェックを入れてOK
    • リボンに「開発」タブが表示されることを確認
  2. マクロのセキュリティ設定：
    • 開発タブ → マクロのセキュリティ
    • 「警告を表示してすべてのマクロを無効にする」を選択
    • OK をクリック
  3. Visual Basic Editor（VBE）の起動：
    • 開発タブ → Visual Basic
    • VBEが開くことを確認
    • 主要な画面構成を確認（プロジェクトエクスプローラ、プロパティウィンドウ、コードウィンドウ）

8.3 マクロの記録機能

最も簡単なVBA学習方法は、マクロの記録機能を使って操作を記録し、生成されたコードを理解することです。

マクロ記録の流れ
  1. 開発タブ → マクロの記録 をクリック
  2. マクロ名を入力（日本語可）
  3. 保存先を選択（通常は「作業中のブック」）
  4. 記録開始後、自動化したい操作を実行
  5. 開発タブ → 記録終了 をクリック
実習 8-2: マクロ記録で挨拶文を自動入力

定型的な挨拶文の入力をマクロで自動化しましょう。

手順
  1. マクロ記録開始：
    • 開発タブ → マクロの記録
    • マクロ名：「挨拶文入力」
    • ショートカットキー：Ctrl+Shift+G
    • OK をクリック
  2. 操作の記録：
    • セルA1に「いつもお世話になっております。」
    • セルA2に「株式会社○○の田中です。」
    • セルA3に「本日はよろしくお願いいたします。」
    • 各入力後にEnterキーを押下
  3. 記録終了：開発タブ → 記録終了
  4. マクロの実行テスト：
    • 新しいワークシートに移動
    • Ctrl+Shift+G でマクロ実行
    • 挨拶文が自動入力されることを確認

8.4 VBAコードの基本理解

記録されたマクロのコードを見て、VBAの基本構造を理解しましょう。

実習 8-3: VBAコードの確認と編集

記録されたマクロのコードを確認し、簡単な編集を行いましょう。

手順
  1. VBAコードの確認：
    • 開発タブ → マクロ
    • 「挨拶文入力」を選択 → 編集
    • VBEでコードを確認
  2. コードの基本構造理解： 
    Sub 挨拶文入力()
' 
' 挨拶文入力 Macro
'
    Range("A1").Select
    ActiveCell.FormulaR1C1 = "いつもお世話になっております。"
    Range("A2").Select
    ActiveCell.FormulaR1C1 = "株式会社○○の田中です。"
    Range("A3").Select
    ActiveCell.FormulaR1C1 = "本日はよろしくお願いいたします。"
End Sub
  3. コードの改良： 
    Sub 挨拶文入力改良版()
    ' セル選択なしで直接値を設定
    Range("A1").Value = "いつもお世話になっております。"
    Range("A2").Value = "株式会社○○の田中です。"
    Range("A3").Value = "本日はよろしくお願いいたします。"
    
    ' ユーザーに完了を通知
    MsgBox "挨拶文の入力が完了しました。"
End Sub

8.5 マクロの実行とファイル保存

作成したマクロの実行方法とマクロ有効ブックでの保存方法を学びましょう。

実行方法 操作手順 メリット 適用場面
ショートカットキー Ctrl+Shift+設定キー 素早い実行、キーボードオペレーション 頻繁に使用するマクロ
開発タブから実行 開発→マクロ→選択→実行 マクロ一覧から選択可能 複数のマクロから選択実行
ボタンに割り当て 挿入→図形→ボタン作成 直感的な操作、他人でも使いやすい 他人と共有するブック
VBEから実行 F5キーで実行 デバッグ機能、コード編集と同時 デバッグ、テスト実行
マクロファイルの保存形式
  • .xlsm：マクロ有効ブック（推奨）
  • .xlsb：マクロ有効バイナリブック
  • .xltm：マクロ有効テンプレート
  • .xlsx形式ではマクロは保存されないので注意
実習 8-4: ボタンマクロと.xlsm保存

ボタンを作成してマクロを割り当て、適切な形式で保存しましょう。

手順
  1. ボタンの作成：
    • 挿入タブ → 図形 → 四角形を挿入
    • 図形を右クリック → テキストの編集 → 「挨拶文入力」
    • 図形を右クリック → マクロの登録
    • 「挨拶文入力改良版」を選択 → OK
  2. ボタンの動作確認：
    • 作成したボタンをクリック
    • マクロが実行されることを確認
  3. マクロ有効ブックで保存：
    • Ctrl+S で保存ダイアログを開く
    • ファイル名：「VBA学習用.xlsm」
    • ファイルの種類：「Excelマクロ有効ブック (*.xlsm)」
    • 保存をクリック
  4. 再読み込みテスト：
    • ファイルを閉じて再度開く
    • セキュリティ警告が表示された場合「マクロを有効にする」をクリック
    • ボタンが正常に動作することを確認

8.6 VBA学習の次のステップ

基礎を習得したら、より高度なVBAプログラミングに挑戦しましょう。

VBA学習の発展方向
  • 変数と制御構造：If文、For文、Do文の活用
  • ユーザーフォーム：入力フォームやダイアログの作成
  • エラー処理：On Error文による安定したプログラム
  • ファイル操作：複数ファイルの一括処理
  • データベース連携：ADOを使った外部データアクセス
  • 他Office連携：Word、PowerPointとの連携
理解度確認クイズ
  1. VBAを使用するために必要なリボンタブは？
    a) 挿入タブ b) 開発タブ c) 数式タブ d) データタブ
  2. マクロ有効ブックの拡張子は？
    a) .xlsx b) .xls c) .xlsm d) .xlsb
  3. マクロ記録を停止するには？
    a) Escキーを押す b) 開発タブ→記録終了 c) ファイルを閉じる d) Excelを再起動
  4. VBEでマクロを実行するキーは？
    a) F1 b) F5 c) F9 d) F12
  5. VBAでセルに値を設定する正しい構文は？
    a) Range("A1") = "値" b) Range("A1").Value = "値" c) Cell("A1") = "値" d) A1 = "値"
解答:
1. b) 開発タブ
2. c) .xlsm
3. b) 開発タブ→記録終了
4. b) F5
5. b) Range("A1").Value = "値"

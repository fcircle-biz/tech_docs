第5章: オブジェクト操作

図形・画像・テキストボックスで表現力を高める

この章で学ぶこと

5.1 図形の挿入と基本編集

Excelでは、表やデータだけでなく、図形を使って視覚的に分かりやすい資料を作成できます。基本的な図形操作を覚えましょう。

図形カテゴリ 代表的な図形 用途 活用場面
基本図形 四角形・円・三角形 基本的な図解・装飾 組織図、フレーム、強調表示
矢印 直線・ブロック矢印 方向・流れの表示 フローチャート、プロセス図
フローチャート図形 プロセス・判断・開始終了 業務フローの可視化 業務手順書、システム設計図
吹き出し コメント・説明用 注釈・説明の追加 グラフの解説、重要事項の強調
星とリボン 装飾・強調用 視覚的なアクセント 表彰状、キャンペーン資料
実習 5-1: 基本図形の挿入と編集

組織図を作成して図形操作の基本を学びましょう。

手順
  1. 挿入タブ → 図 → 図形 → 基本図形 → 四角形を選択
  2. ワークシート上でドラッグして四角形を描画
  3. 図形を右クリック → テキストの編集 → 「部長」と入力
  4. 同様に「課長」「主任」「一般社員」の四角形を作成
  5. 挿入タブ → 図形 → 線 → 直線で各図形を接続
  6. 図形の書式設定：
    • 部長の四角形を選択
    • 図形の書式タブ → 図形のスタイル → 青色系を選択
    • 他の図形も色分けして階層を表現

5.2 画像の挿入と調整

表やグラフに画像を追加することで、より魅力的で理解しやすい資料を作成できます。

画像の挿入方法
  • ファイルから：挿入タブ → 画像 → このデバイス
  • オンライン画像：挿入タブ → 画像 → オンライン画像
  • アイコン：挿入タブ → アイコン（Microsoft 365）
  • スクリーンショット：挿入タブ → スクリーンショット
実習 5-2: 画像の挿入と調整

売上レポートに画像を追加して、視覚的魅力を高めましょう。

手順
  1. 挿入タブ → アイコン → 「ビジネス」カテゴリから適切なアイコンを選択
  2. 画像を選択した状態で、図の書式タブの機能を確認：
    • トリミング機能
    • 色の調整
    • 効果の追加
  3. 適切なサイズに調整し、表の見出し部分に配置

5.3 配置と整列

複数のオブジェクトを美しく配置するための整列機能と間隔調整を学びましょう。

配置機能 主要機能 具体的な操作 用途
整列 オブジェクトの位置合わせ 左揃え・右揃え・上揃え・下揃え・中央揃え 複数オブジェクトを一直線に並べる
配布 オブジェクト間の間隔調整 左右に整列・上下に整列・等間隔配置 オブジェクト間の距離を均等にする
回転 オブジェクトの向き変更 90度回転・180度回転・自由回転・反転 図形の向きや角度を調整する
実習 5-3: オブジェクトの配置と整列

複数の図形を使ってフローチャートを作成し、整列機能を活用しましょう。

手順
  1. フローチャート用の図形を5つ作成
  2. 全ての図形を選択（Ctrl+クリック）
  3. 図形の書式タブ → 配置 → 左右に整列
  4. 同様に上下方向の配置も調整
  5. グループ化：全選択状態で右クリック → グループ化
理解度確認クイズ
  1. 図形にテキストを追加する方法は？
    a) 図形をダブルクリック b) 図形を右クリック→テキストの編集 c) 挿入タブ→テキスト d) 書式タブ→テキスト
  2. 複数のオブジェクトを一つにまとめる機能は？
    a) 結合 b) 統合 c) グループ化 d) マージ
  3. オブジェクトの重なり順序を変更するには？
    a) 配置メニュー b) 書式メニュー c) 整列メニュー d) 前面・背面メニュー
解答:
1. b) 図形を右クリック→テキストの編集
2. c) グループ化
3. d) 前面・背面メニュー
第5章のまとめ

オブジェクト操作により、Excelの表現力が大幅に向上します。図形、画像、テキストボックスを効果的に配置することで、プロフェッショナルな資料を作成できるようになりました。

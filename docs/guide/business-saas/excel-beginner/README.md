# Excel入門 学習ガイドライン

このガイドラインでは、パソコン初心者の方を対象に、Microsoft Excelの基本的な使い方をゼロからやさしく学べるカリキュラムを提供しています。「パソコンはあまり得意ではない」「Excelを触ったことがない」という方でも安心して取り組めるよう、一つひとつの操作を丁寧に解説しています。

## 前提条件
### 必要な環境
- Microsoft Excel（Excel 2016以降、またはMicrosoft 365）
- パソコン（Windows 10/11、またはMac）
- マウスまたはトラックパッド

### 参考リソース
- [Microsoft Excel ヘルプセンター](https://support.microsoft.com/ja-jp/excel)
- [Excel初心者向けガイド（Microsoft公式）](https://support.microsoft.com/ja-jp/office/excel-%E3%82%AF%E3%82%A4%E3%83%83%E3%82%AF-%E3%82%B9%E3%82%BF%E3%83%BC%E3%83%88-422a1582-0d9b-4a2e-9c26-4b6ed9e21f64)

### 前提知識
- **必須**: パソコンの電源の入れ方、マウスの使い方（クリック、ダブルクリック）がわかる方
- **推奨**: 文字の入力ができる方（ローマ字入力またはかな入力）

## 学習コンテンツ

### [1. Excelを始めよう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-01.html)
Excelとは何か、どんなことができるソフトなのかを紹介します。Excelの起動方法から画面の見方まで、はじめの一歩をやさしく解説します。

**学習内容:**
- Excelとは？ - 表計算ソフトってなに？
- Excelでできること（表作り、計算、グラフなど）
- Excelの起動方法（Windowsの場合）
- 画面の名前を覚えよう（リボン、セル、シートなど）
- マウスの動かし方とクリックのおさらい

### [2. セルに文字や数字を入力しよう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-02.html)
Excelの基本となる「セル」への入力方法を学びます。文字や数字の入れ方、間違えたときの直し方、入力の確定方法など、基本中の基本を身につけます。

**学習内容:**
- セルってなに？ - Excelのマス目を理解しよう
- セルの選び方（クリックで選択）
- 文字を入力してみよう
- 数字を入力してみよう
- 入力を確定する（Enterキー）
- 間違えたときの直し方（BackspaceキーとDeleteキー）
- 入力をやり直す（元に戻す）方法

### [3. ファイルを保存しよう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-03.html)
せっかく作った表をなくさないために、ファイルの保存方法を覚えましょう。保存の仕方、保存場所の選び方、ファイル名の付け方などを学びます。

**学習内容:**
- 保存はなぜ大切？ - パソコンの電源が切れても大丈夫
- はじめて保存する（名前を付けて保存）
- 上書き保存する（同じファイルに保存）
- ファイル名の付け方のコツ
- どこに保存する？ - デスクトップとドキュメントフォルダ
- 保存したファイルを開く方法
- 新しいファイルを作る方法

### [4. かんたんな表を作ってみよう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-04.html)
実際に表を作りながら、データの入力と罫線（けいせん）の引き方を練習します。買い物リストやスケジュール表など、身近な例で学びます。

**学習内容:**
- 表を作る流れを知ろう
- 複数のセルにデータを入力する
- 罫線（けいせん）ってなに？ - 表の線のこと
- 罫線を引いてみよう
- 罫線の種類（太い線、細い線、点線）
- 罫線を消す方法
- かんたんな買い物リストを作ってみよう

### [5. 計算をしてみよう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-05.html)
Excelの最大の特徴である「自動計算」の基本を学びます。足し算、引き算、掛け算、割り算を数式を使って行う方法を覚えましょう。

**学習内容:**
- 数式ってなに？ - Excelに計算をお願いする方法
- 数式の入力ルール（=から始める）
- 足し算をしてみよう（+の使い方）
- 引き算をしてみよう（-の使い方）
- 掛け算をしてみよう（*の使い方）
- 割り算をしてみよう（/の使い方）
- セルを使った計算（A1+B1のような書き方）
- 数式のコピーで計算を楽にしよう

### [6. よく使う関数を覚えよう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-06.html)
Excelには便利な「関数」という機能があります。合計を出すSUM関数、平均を出すAVERAGE関数など、仕事でよく使う関数の使い方を学びます。

**学習内容:**
- 関数ってなに？ - 計算を楽にする魔法の言葉
- 合計を求める（SUM関数）
- SUM関数の入力方法
- 平均を求める（AVERAGE関数）
- 最大値を求める（MAX関数）
- 最小値を求める（MIN関数）
- 個数を数える（COUNT関数）
- オートSUMボタンで楽々入力

### [7. 表を見やすくしよう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-07.html)
作った表をもっと見やすくする方法を学びます。文字を大きくしたり、色を付けたり、配置を変えたりして、きれいな表に仕上げましょう。

**学習内容:**
- 文字のサイズを変える
- 文字を太くする（太字）
- 文字の色を変える
- セルに色を付ける（背景色）
- 文字の位置を変える（左寄せ、中央、右寄せ）
- 列の幅を広げる・縮める
- 行の高さを変える
- セルを結合する（複数のセルを1つにする）

### [8. データを並べ替え・絞り込もう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-08.html)
たくさんのデータがあるときに便利な「並べ替え」と「フィルター」機能を学びます。五十音順や数字の大きい順に並べたり、特定の条件でデータを絞り込んだりできます。

**学習内容:**
- 並べ替えってなに？ - データを順番に整理する
- 昇順（小さい順・あいうえお順）で並べ替え
- 降順（大きい順）で並べ替え
- フィルターってなに？ - 見たいデータだけ表示する
- フィルターの設定方法
- フィルターでデータを絞り込む
- フィルターを解除する

### [9. 表を印刷しよう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-09.html)
作った表を紙に印刷する方法を学びます。印刷プレビューで確認しながら、きれいに印刷するための設定方法を覚えましょう。

**学習内容:**
- 印刷の前に確認！印刷プレビュー
- 用紙の向きを変える（縦・横）
- 用紙のサイズを選ぶ（A4など）
- 表を1ページに収める方法
- 余白（よはく）を調整する
- ヘッダーとフッターを追加する
- 印刷範囲を指定する
- 印刷を実行する

### [10. グラフを作ってみよう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-10.html)
数字だけではわかりにくいデータも、グラフにすると一目でわかるようになります。棒グラフや円グラフなど、基本的なグラフの作り方を学びます。

**学習内容:**
- グラフってなに？ - 数字を絵で見せる
- グラフの種類（棒グラフ、円グラフ、折れ線グラフ）
- グラフを作るための準備（データの選択）
- 棒グラフを作ってみよう
- 円グラフを作ってみよう
- グラフのタイトルを変える
- グラフの大きさを変える
- グラフの位置を移動する

### [11. 実践！業務で使える表を作ろう](https://fcircle-biz.github.io/tech_docs/guide/business-saas/excel-beginner/excel-beginner-learning-material-11.html)
これまで学んだことを活かして、実際の仕事で使える表を作ってみましょう。家計簿、出勤表、売上集計表など、実用的な例で総復習します。

**学習内容:**
- 学んだことのおさらい
- かんたん家計簿を作ってみよう
- 出勤管理表を作ってみよう
- 売上集計表を作ってみよう
- よく使うショートカットキーまとめ
- 困ったときのQ&A
- さらにExcelを学ぶには

## 学習の進め方

1. **順番通りに進めましょう**: 各章は前の章で学んだことをもとに進みます。はじめから順番に学習してください
2. **実際に手を動かしましょう**: 読むだけでなく、必ず自分のパソコンでExcelを開いて同じ操作をしてみてください
3. **何度でも練習しましょう**: 1回でできなくても大丈夫です。わからなければ戻って何度も練習しましょう
4. **わからない言葉はメモしましょう**: 知らない言葉が出てきたらメモして、少しずつ覚えていきましょう
5. **休憩を取りながら進めましょう**: 疲れたら休憩！無理せず自分のペースで進めてください

## 学習目標

このガイドを完了すると、以下のことができるようになります：

- Excelを起動して、画面の各部分の名前がわかる
- セルに文字や数字を入力し、保存できる
- 罫線を引いて、見やすい表を作れる
- 四則演算（足し算・引き算・掛け算・割り算）の数式を入力できる
- SUM関数やAVERAGE関数を使って合計や平均を計算できる
- 文字の大きさや色、セルの背景色を変更できる
- データを並べ替えたり、フィルターで絞り込んだりできる
- 表をきれいに印刷できる
- かんたんなグラフを作成できる
- 仕事で使える基本的な表を自分で作れる

第3章: 財務諸表の構造

この章で学ぶこと
  • 貸借対照表（B/S）の構造と読み方を理解する
  • 損益計算書（P/L）の構造と5つの利益を学ぶ
  • キャッシュフロー計算書（C/F）の概要を把握する
  • 3つの財務諸表のつながりを理解する
  • 実在企業の決算書を読み解く力を養う

3.1 財務諸表とは

財務諸表（Financial Statements）とは、企業の財政状態や経営成績を表す公式な報告書です。 主に以下の3つの表から構成されます。

flowchart TD A[財務諸表] --> B["貸借対照表
(B/S)"] A --> C["損益計算書
(P/L)"] A --> D["キャッシュフロー計算書
(C/F)"] B --> E[財政状態] C --> F[経営成績] D --> G[資金の流れ]
財務諸表 英語名 何を表すか
貸借対照表 Balance Sheet (B/S) ある時点での企業の財政状態（資産・負債・純資産）
損益計算書 Profit and Loss statement (P/L) 一定期間の経営成績（収益・費用・利益）
キャッシュフロー計算書 Cash Flow statement (C/F) 一定期間の資金の流れ（現金の増減）
3つの表の役割を身近な例で理解

家計に例えると：

  • 貸借対照表: 今、財布にいくら入っているか、貯金はいくらか、借金はいくらあるか（現在の状態）
  • 損益計算書: 今月の収入と支出はいくらで、いくら貯金できたか（期間の成績）
  • キャッシュフロー計算書: 今月、実際に現金がいくら動いたか（資金の流れ）

3.2 貸借対照表（B/S: Balance Sheet）

貸借対照表は、ある特定の日時点での企業の財政状態を表します。 企業が何を持っていて（資産）、どこから資金を調達したか（負債と純資産）を示します。

B/Sの基本構造

資産 = 負債 + 純資産

この等式を貸借対照表等式または会計等式と呼びます。 左側（借方）の資産と、右側（貸方）の負債+純資産は必ず一致します（バランスする）。

flowchart LR A[貸借対照表] --> B["資産
(左側・借方)"] A --> C["負債+純資産
(右側・貸方)"] B --> D["企業が持っているもの
(運用形態)"] C --> E["資金の調達源泉
(調達形態)"]
資産の部

企業が保有する経済的価値のあるものです。流動資産と固定資産に分類されます。

分類 説明 具体例
流動資産 1年以内に現金化される資産 現金及び預金、売掛金、受取手形、商品、短期貸付金
固定資産 1年を超えて使用する資産 建物、土地、機械装置、車両、投資有価証券、のれん
負債の部

企業が負っている債務・支払義務です。流動負債と固定負債に分類されます。

分類 説明 具体例
流動負債 1年以内に返済する負債 買掛金、支払手形、短期借入金、未払金、未払法人税等
固定負債 1年を超えて返済する負債 長期借入金、社債、退職給付引当金
純資産の部

資産から負債を引いた正味の財産です。企業の「実質的な価値」を表します。

分類 説明
株主資本 資本金、資本剰余金、利益剰余金（過去から累積した利益）
評価・換算差額等 その他有価証券評価差額金、繰延ヘッジ損益など
貸借対照表のサンプル
株式会社サンプル商事 貸借対照表
（2024年3月31日現在、単位：千円）

  利益剰余金          100,000

負債・純資産合計   300,000
B/Sから読み取れる重要指標
  • 流動比率 = 流動資産 ÷ 流動負債 × 100 = 100,000 ÷ 40,000 × 100 = 250%
    → 短期的な支払能力。200%以上が理想的。
  • 自己資本比率 = 純資産 ÷ 総資産 × 100 = 200,000 ÷ 300,000 × 100 = 66.7%
    → 財務の健全性。40%以上が健全とされる。

3.3 損益計算書（P/L: Profit and Loss statement）

損益計算書は、一定期間（通常1年間）の経営成績を表します。 企業がどれだけ収益を上げ、どれだけ費用を使い、最終的にどれだけ利益を得たかを示します。

P/Lの基本構造

利益 = 収益 - 費用

損益計算書は上から下に流れるように読みます。段階的に5つの利益を計算していきます。

5つの利益

日本の損益計算書では、異なる5つの利益を段階的に計算します。それぞれの利益が企業の収益力を多角的に示します。

flowchart TD A[売上高] --> B[売上原価を引く] B --> C["1. 売上総利益
(粗利)"] C --> D[販売費及び一般管理費を引く] D --> E["2. 営業利益
(本業の儲け)"] E --> F[営業外損益を加減] F --> G["3. 経常利益
(通常の儲け)"] G --> H[特別損益を加減] H --> I["4. 税引前当期純利益"] I --> J[法人税等を引く] J --> K["5. 当期純利益
(最終利益)"]
利益の種類 計算式 何を示すか
1. 売上総利益
（粗利）		 売上高 - 売上原価 商品やサービス自体の利益率。仕入や製造コストを引いた残り。
2. 営業利益 売上総利益 - 販管費 本業で稼いだ利益。人件費や家賃などの経費を引いた後の儲け。
3. 経常利益 営業利益 + 営業外損益 通常の事業活動全体での利益。受取利息や支払利息を含む。
4. 税引前当期純利益 経常利益 + 特別損益 臨時的な損益（固定資産売却損益など）を含めた税引前の利益。
5. 当期純利益 税引前当期純利益 - 法人税等 最終的に企業に残る利益。株主への配当原資となる。

（内訳：給与60,000、家賃20,000、その他40,000）

営業外収益

営業外費用

P/Lから読み取れる重要指標
  • 売上高総利益率 = 売上総利益 ÷ 売上高 × 100 = 200,000 ÷ 500,000 × 100 = 40%
    → 商品・サービスの利益率。業界により異なるが、高いほど良い。
  • 売上高営業利益率 = 営業利益 ÷ 売上高 × 100 = 80,000 ÷ 500,000 × 100 = 16%
    → 本業の収益力。10%以上あれば優良。
  • 売上高経常利益率 = 経常利益 ÷ 売上高 × 100 = 79,000 ÷ 500,000 × 100 = 15.8%
    → 総合的な収益力。5%以上が目安。

3.4 キャッシュフロー計算書（C/F: Cash Flow statement）

キャッシュフロー計算書は、一定期間の現金の流れ（キャッシュフロー）を表します。 利益が出ていても現金が不足すれば倒産するため、資金繰りを把握する上で非常に重要です。

なぜキャッシュフローが重要か？

「勘定合って銭足らず」という言葉があります。

  • 損益計算書では利益が出ているのに、現金が足りず支払いができない状態を指します
  • 売掛金が多く回収が遅れたり、在庫が過剰だったりすると、現金不足に陥ります
  • キャッシュフロー計算書で実際の資金の動きを確認することが不可欠です
3つのキャッシュフロー

キャッシュフロー計算書は、現金の増減を3つの活動に分けて表示します。

分類 説明 主な項目
営業活動による
キャッシュフロー		 本業の営業活動で得た現金の増減 税引前当期純利益、減価償却費、売掛金・棚卸資産の増減、買掛金の増減
投資活動による
キャッシュフロー		 設備投資や有価証券の売買による現金の増減 有形固定資産の取得・売却、投資有価証券の取得・売却
財務活動による
キャッシュフロー		 資金調達や返済による現金の増減 借入金の借入・返済、社債の発行・償還、配当金の支払
flowchart TD A[期首現金残高] --> B[営業活動によるCF] B --> C[投資活動によるCF] C --> D[財務活動によるCF] D --> E[期末現金残高] B -->|プラスが理想| F[本業で現金を稼ぐ] C -->|マイナスが通常| G[将来への投資] D -->|状況による| H[資金調達と返済]

C/Fから読み取れるポイント
  • 営業CF（+74,000）: 本業で現金を稼いでいる → 健全
  • 投資CF（△22,000）: 設備投資にお金を使っている → 将来への投資
  • 財務CF（△5,000）: 借入より返済・配当が多い → 財務健全化
  • フリーCF = 営業CF + 投資CF = 74,000 - 22,000 = 52,000 → 自由に使える現金がある

3.5 3つの財務諸表のつながり

3つの財務諸表は独立しているのではなく、互いに連動しています。このつながりを理解することで、 企業の全体像がより明確に見えてきます。

flowchart TD A[損益計算書
P/L] -->|当期純利益| B[貸借対照表
B/S
利益剰余金] B -->|現金残高| C[キャッシュフロー計算書
C/F
期末現金残高] A -->|減価償却費等| C C -->|期末現金| B
3表のつながりの具体例
  1. P/L → B/S: 損益計算書の当期純利益が、貸借対照表の利益剰余金に加算される
  2. B/S → C/F: 貸借対照表の現金残高が、キャッシュフロー計算書の期末現金残高と一致する
  3. P/L → C/F: 損益計算書の税引前当期純利益が、キャッシュフロー計算書の営業CFの出発点となる
  4. 減価償却費: P/Lでは費用として計上されるが、現金は出ていかないのでC/Fでは加算される
3表の読み方のコツ
財務諸表 まず見るべきポイント
P/L 営業利益と当期純利益 → 本業で稼いでいるか、最終的に利益が残っているか
B/S 自己資本比率と流動比率 → 財務が健全か、倒産リスクは低いか
C/F 営業CFとフリーCF → 本業で現金を稼いでいるか、投資余力はあるか
実習 3-1: 財務諸表を読み解く

次の簡略化された財務諸表から、企業の状態を分析してみましょう。

A社の財務データ（単位：百万円）

損益計算書:

  • 売上高: 1,000
  • 売上原価: 600
  • 販管費: 300
  • 営業外収益: 10
  • 営業外費用: 20
  • 法人税等: 27

貸借対照表:

  • 流動資産: 400（うち現金100）
  • 固定資産: 600
  • 流動負債: 200
  • 固定負債: 300
  • 純資産: 500
課題
  1. 5つの利益（売上総利益、営業利益、経常利益、税引前当期純利益、当期純利益）を計算してください
  2. 売上高営業利益率を計算してください
  3. 自己資本比率と流動比率を計算してください
  4. この会社の財務状態について、良い点と懸念点を挙げてください
解答例
解答を見る

1. 5つの利益の計算

  • 売上総利益 = 1,000 - 600 = 400百万円
  • 営業利益 = 400 - 300 = 100百万円
  • 経常利益 = 100 + 10 - 20 = 90百万円
  • 税引前当期純利益 = 90百万円（特別損益なし）
  • 当期純利益 = 90 - 27 = 63百万円

2. 売上高営業利益率

100 ÷ 1,000 × 100 = 10%

3. 財務指標

  • 自己資本比率 = 500 ÷ 1,000 × 100 = 50%
  • 流動比率 = 400 ÷ 200 × 100 = 200%

4. 財務状態の分析

良い点:

  • 自己資本比率50%は健全水準（40%以上が目安）
  • 流動比率200%で短期支払能力が十分（200%以上が理想）
  • 営業利益率10%は優良水準
  • 黒字経営で当期純利益63百万円を確保

懸念点:

  • 売上総利益率40%から営業利益率10%まで下がっており、販管費が高い（販管費率30%）
  • 営業外費用が営業外収益より多く、支払利息等の負担がある
  • 固定資産が総資産の60%を占め、設備集約型ビジネスの可能性（減価償却負担に注意）
理解度確認クイズ
  1. 貸借対照表（B/S）と損益計算書（P/L）の違いを説明してください。
    解答例を見る

    貸借対照表は、ある特定の時点（期末など）での企業の財政状態（資産・負債・純資産）を示します。ストック（残高）の情報です。
    損益計算書は、一定期間（1年間など）の経営成績（収益・費用・利益）を示します。フロー（流れ）の情報です。

  2. 損益計算書の5つの利益を順番に挙げてください。
    解答例を見る

    1. 売上総利益（粗利）
    2. 営業利益
    3. 経常利益
    4. 税引前当期純利益
    5. 当期純利益

  3. 営業利益と経常利益の違いは何ですか？
    解答例を見る

    営業利益は本業（営業活動）だけで得た利益です。
    経常利益は営業利益に営業外損益（受取利息、支払利息など）を加減したもので、 通常の事業活動全体での利益を示します。

  4. キャッシュフロー計算書の3つの区分を挙げてください。
    解答例を見る

    1. 営業活動によるキャッシュフロー
    2. 投資活動によるキャッシュフロー
    3. 財務活動によるキャッシュフロー

  5. 「勘定合って銭足らず」とはどのような状態ですか？
    解答例を見る

    損益計算書では利益が出ているにもかかわらず、現金が不足して支払いができない状態を指します。 売掛金の回収遅れや在庫過多により、帳簿上は利益でも手元の現金が足りなくなることがあります。 だからこそキャッシュフロー計算書で実際の資金の流れを確認することが重要です。

第3章のまとめ
  • 財務諸表は企業の財政状態と経営成績を示す公式な報告書である
  • 貸借対照表（B/S）は特定時点の財政状態（資産＝負債＋純資産）を表す
  • 損益計算書（P/L）は一定期間の経営成績を5つの利益で段階的に示す
  • キャッシュフロー計算書（C/F）は現金の流れを3つの活動に分けて示す
  • 3つの財務諸表は連動しており、P/Lの利益→B/Sの純資産、B/Sの現金→C/Fの期末残高とつながる
  • 利益が出ていても現金が不足すれば倒産するため、C/Fの確認が不可欠

次章では、これらの財務諸表がどのように作成されるのか、決算と会計処理の流れを詳しく学びます。

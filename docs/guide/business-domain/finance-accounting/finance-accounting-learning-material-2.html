第2章: 取引と仕訳の基礎

この章で学ぶこと
  • 会計の5要素（資産・負債・純資産・収益・費用）の意味を理解する
  • 借方・貸方のルールを理解し、仕訳が切れるようになる
  • 複式簿記の基本原理（貸借平均の原則）を学ぶ
  • 実務でよく使う勘定科目を覚える
  • 代表的な取引の仕訳パターンを習得する

2.1 会計の5要素

会計では、すべての取引を5つの要素に分類して記録します。 この5要素は財務諸表の基礎となる非常に重要な概念です。

flowchart TD A[会計の5要素] --> B["資産
(Assets)"] A --> C["負債
(Liabilities)"] A --> D["純資産
(Equity)"] A --> E["収益
(Revenue)"] A --> F["費用
(Expenses)"] B --> G[貸借対照表] C --> G D --> G E --> H[損益計算書] F --> H
1. 資産（Assets）

企業が保有する経済的価値のあるものです。将来、お金を生み出すか、事業に役立つものを指します。

資産の具体例
  • 現金・預金: 手元にある現金や銀行預金
  • 売掛金: 商品を販売したがまだ回収していない代金（ツケ）
  • 商品・製品: 販売予定の在庫
  • 建物・土地: 事業用の不動産
  • 機械装置・車両: 事業で使う設備
  • 投資有価証券: 他社の株式や債券
2. 負債（Liabilities）

企業が負っている債務・支払義務です。将来、お金を返さなければならないものを指します。

負債の具体例
  • 買掛金: 商品を仕入れたがまだ支払っていない代金（ツケ）
  • 借入金: 銀行から借りたお金
  • 未払金: サービスや経費でまだ支払っていないもの
  • 社債: 会社が発行した債券（返済義務がある）
  • 未払法人税等: 確定したがまだ納めていない税金
3. 純資産（Equity / Net Assets）

資産から負債を引いた正味の財産です。会社の「正味の価値」を表します。

純資産の計算式

純資産 = 資産 - 負債

純資産の具体例
  • 資本金: 株主が出資したお金
  • 利益剰余金: 会社が稼いだ利益の累積（配当せずに残した分）
  • 資本剰余金: 資本金以外の出資
4. 収益（Revenue）

企業活動によって得られた収入・稼ぎです。企業の純資産を増加させる要因です。

収益の具体例
  • 売上高: 商品やサービスの販売による収入（本業の稼ぎ）
  • 受取利息: 預金や貸付金から得られる利息
  • 受取配当金: 保有株式から得られる配当
  • 雑収入: その他の収入（保険金収入、スクラップ売却益など）
5. 費用（Expenses）

収益を得るために使ったお金・消費したものです。企業の純資産を減少させる要因です。

費用の具体例
  • 売上原価: 販売した商品の仕入れ原価
  • 給与: 従業員への給料
  • 家賃: 事務所・店舗の賃料
  • 水道光熱費: 電気代、ガス代、水道代
  • 旅費交通費: 出張費、交通費
  • 減価償却費: 建物や機械の価値減少分
  • 支払利息: 借入金の利息
重要な関係式

資産 = 負債 + 純資産 （貸借対照表等式）
利益 = 収益 - 費用 （損益計算の基本式）
純資産の増加 = 利益 （配当を除く）

2.2 借方・貸方のルール

複式簿記では、すべての取引を借方（かりかた）と貸方（かしかた）に分けて記録します。 これは「左側」と「右側」を意味する専門用語で、お金の借り貸しとは直接関係ありません。

借方と貸方の基本配置
借方（かりかた）
Debit / Dr. 		貸方（かしかた）
Credit / Cr.
左側 右側

重要: 借方は「左」、貸方は「右」と覚えましょう。日本語では「借りる」「貸す」という漢字ですが、 実際の借金の意味とは異なります。

借方・貸方の増減ルール

5要素ごとに、借方に書くと増えるのか、貸方に書くと増えるのかが決まっています。 このルールは会計の絶対的なルールで、世界共通です。

要素 借方（左側） 貸方（右側）
資産 増加 ↑ 減少 ↓
負債 減少 ↓ 増加 ↑
純資産 減少 ↓ 増加 ↑
収益 減少 ↓ 発生 ↑
費用 発生 ↑ 減少 ↓
覚え方のコツ
  • 資産と費用は借方（左）で増える → 「資費（しひ）は左」
  • 負債・純資産・収益は貸方（右）で増える → 「負純収（ふじゅんしゅう）は右」
  • 資産が増えたら借方、減ったら貸方に記入
  • 負債が増えたら貸方、減ったら借方に記入

2.3 仕訳の基本構造

仕訳（しわけ）とは、取引を借方と貸方に分けて記録することです。 すべての取引は必ず借方と貸方の金額が一致します。これを貸借平均の原則と呼びます。

仕訳の基本形式
（借方）勘定科目 金額 / （貸方）勘定科目 金額
仕訳の3つのルール
  1. 借方合計 = 貸方合計: 必ず左右の金額が一致する
  2. 取引を2つの側面から見る: すべての取引には「原因」と「結果」がある
  3. 勘定科目を使う: 取引を適切な勘定科目で分類する

2.4 基本的な仕訳例

例1: 現金で商品を販売した

取引内容: 商品10万円を現金で販売した

考え方:

  • 現金（資産）が10万円増えた → 資産の増加 → 借方
  • 売上（収益）が10万円発生した → 収益の発生 → 貸方
（借方）現金 100,000 / （貸方）売上 100,000
例2: 掛けで商品を仕入れた

取引内容: 商品8万円を掛け（後払い）で仕入れた

考え方:

  • 仕入（費用）が8万円発生した → 費用の発生 → 借方
  • 買掛金（負債）が8万円増えた → 負債の増加 → 貸方
（借方）仕入 80,000 / （貸方）買掛金 80,000
例3: 銀行から借入をした

取引内容: 銀行から100万円を借り入れ、普通預金口座に入金された

考え方:

  • 普通預金（資産）が100万円増えた → 資産の増加 → 借方
  • 借入金（負債）が100万円増えた → 負債の増加 → 貸方
（借方）普通預金 1,000,000 / （貸方）借入金 1,000,000
例4: 給与を現金で支払った

取引内容: 従業員に給与30万円を現金で支払った

考え方:

  • 給与（費用）が30万円発生した → 費用の発生 → 借方
  • 現金（資産）が30万円減った → 資産の減少 → 貸方
（借方）給与 300,000 / （貸方）現金 300,000
例5: 買掛金を現金で支払った

取引内容: 以前の仕入代金（買掛金）5万円を現金で支払った

考え方:

  • 買掛金（負債）が5万円減った → 負債の減少 → 借方
  • 現金（資産）が5万円減った → 資産の減少 → 貸方
（借方）買掛金 50,000 / （貸方）現金 50,000
例6: 減価償却費の計上

取引内容: 建物の減価償却費5万円を計上した

考え方:

  • 減価償却費（費用）が5万円発生した → 費用の発生 → 借方
  • 減価償却累計額（資産のマイナス）が5万円増えた → 資産の減少 → 貸方
（借方）減価償却費 50,000 / （貸方）減価償却累計額 50,000

2.5 主要な勘定科目

取引を記録する際に使う分類名を勘定科目（かんじょうかもく）と呼びます。 会社によって細かい名称は異なりますが、基本的な勘定科目は共通しています。

土地 事業用の土地
車両運搬具 営業車・トラックなど
備品 パソコン、机、椅子など

勘定科目 説明

実習 2-1: 仕訳の練習問題

次の取引について、仕訳を考えてみましょう。

  1. 商品15万円を掛け（後払い）で販売した
  2. 事務所の家賃10万円を現金で支払った
  3. 売掛金20万円が普通預金に入金された
  4. パソコン（備品）8万円を現金で購入した
  5. 銀行への借入金の返済として5万円を普通預金から支払った
解答例
解答を見る

1. 商品15万円を掛けで販売した

（借方）売掛金 150,000 / （貸方）売上 150,000

売掛金（資産）が増加→借方、売上（収益）が発生→貸方

2. 家賃10万円を現金で支払った

（借方）地代家賃 100,000 / （貸方）現金 100,000

地代家賃（費用）が発生→借方、現金（資産）が減少→貸方

3. 売掛金20万円が普通預金に入金された

（借方）普通預金 200,000 / （貸方）売掛金 200,000

普通預金（資産）が増加→借方、売掛金（資産）が減少→貸方

4. パソコン8万円を現金で購入した

（借方）備品 80,000 / （貸方）現金 80,000

備品（資産）が増加→借方、現金（資産）が減少→貸方

5. 借入金5万円を普通預金から返済した

（借方）借入金 50,000 / （貸方）普通預金 50,000

借入金（負債）が減少→借方、普通預金（資産）が減少→貸方

理解度確認クイズ
  1. 会計の5要素を挙げてください。それぞれ英語名も答えてください。
    解答例を見る

    1. 資産（Assets）
    2. 負債（Liabilities）
    3. 純資産（Equity / Net Assets）
    4. 収益（Revenue）
    5. 費用（Expenses）

  2. 資産が増えた場合、借方と貸方のどちらに記入しますか？
    解答例を見る

    借方（左側）に記入します。資産は借方で増加、貸方で減少します。

  3. 「売掛金」とは何ですか？それは5要素のどれに分類されますか？
    解答例を見る

    売掛金とは、商品やサービスを販売したがまだ代金を回収していない債権（ツケ）です。 5要素では「資産」に分類されます。

  4. 貸借平均の原則とは何ですか？
    解答例を見る

    仕訳において、借方の合計金額と貸方の合計金額が必ず一致するという原則です。 複式簿記の基本となる重要なルールで、取引を2つの側面から記録することで 記帳の正確性を担保します。

  5. 次の仕訳は正しいですか？理由も説明してください。
    商品10万円を現金で仕入れた → （借方）現金 100,000 / （貸方）仕入 100,000
    解答例を見る

    間違っています。
    正しい仕訳は: （借方）仕入 100,000 / （貸方）現金 100,000

    理由: 仕入（費用）が発生したので借方、現金（資産）が減少したので貸方に記入します。 出題の仕訳は借方と貸方が逆になっています。

第2章のまとめ
  • 会計の5要素は資産・負債・純資産・収益・費用である
  • 借方（左）と貸方（右）の増減ルールを理解することが仕訳の基礎
  • 資産と費用は借方で増加、負債・純資産・収益は貸方で増加
  • すべての仕訳は借方合計=貸方合計となる（貸借平均の原則）
  • 取引を適切な勘定科目で分類して記録する
  • 仕訳は「増減の方向」と「5要素の分類」を考えれば解ける

次章では、仕訳から作成される財務諸表（貸借対照表・損益計算書・キャッシュフロー計算書）の 構造と読み方を学びます。

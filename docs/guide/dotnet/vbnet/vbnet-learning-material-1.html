第1章: VB.NET入門と開発環境構築

VB.NET入門と開発環境構築

この章で学ぶこと
  • VB.NETとは何か、その特徴を理解する
  • .NET Frameworkの基本概念を学ぶ
  • Visual Studioの開発環境を構築する
  • 初回プロジェクトを作成し、Hello Worldプログラムを実行する
  • VB.NETの基本的なプロジェクト構造を理解する

1.1 VB.NETとは

VB.NET（Visual Basic .NET）は、Microsoft社が開発したプログラミング言語で、.NET Frameworkまたは.NET Core上で動作するオブジェクト指向言語です。

VB.NETの特徴

  • 英語に近い構文: 自然言語に近い記述で、初心者にもわかりやすい
  • 強力な型システム: コンパイル時の型チェックによる安全性
  • .NET生態系: 豊富なライブラリとフレームワークの活用
  • マルチプラットフォーム: .NET Coreにより、Windows、Linux、macOSで動作

1.2 .NET Frameworkの基本概念

.NET Frameworkは、Windows向けのアプリケーション開発プラットフォームです。VB.NETで作成されたプログラムは、.NET Framework上で実行されます。

主要コンポーネント

  • CLR（共通言語ランタイム）: プログラムの実行環境
  • BCL（基底クラスライブラリ）: 基本的な機能を提供するクラス群
  • ASP.NET: Webアプリケーション開発フレームワーク
  • Windows Forms/WPF: デスクトップアプリケーション開発

1.3 Visual Studioの開発環境構築

VB.NET開発には、統合開発環境（IDE）のVisual Studioを使用します。

実習 1-1: Visual Studioのインストール

Visual Studio Community版（無料）をインストールし、VB.NET開発環境を構築しましょう。

手順
  1. Visual Studio公式サイト（https://visualstudio.microsoft.com/ja/）にアクセス
  2. 「Visual Studio Community」をダウンロード
  3. インストーラーを実行し、「.NET デスクトップ開発」ワークロードを選択
  4. 「Visual Basic」言語パックを確認してインストール
  5. インストール完了後、Visual Studioを起動
確認項目
  • Visual Studioが正常に起動すること
  • 新規プロジェクト作成時にVB.NETテンプレートが表示されること

1.4 初回プロジェクトの作成

Visual Studioを使用して、初回のVB.NETプロジェクトを作成します。

実習 1-2: Hello Worldプログラムの作成

VB.NETの基本的なプログラムを作成し、実行してみましょう。

手順
  1. Visual Studioを起動し、「新しいプロジェクトの作成」を選択
  2. 「コンソール アプリ (.NET Framework)」を選択（Visual Basic）
  3. プロジェクト名を「HelloWorldApp」に設定
  4. 以下のコードを入力：
プログラムコード
Module Module1
    Sub Main()
        Console.WriteLine("Hello, VB.NET World!")
        Console.WriteLine("VB.NETプログラミングへようこそ！")
        Console.ReadLine() ' エンターキー待機
    End Sub
End Module
実行方法
  1. F5キーを押すか、「デバッグ」→「デバッグの開始」を選択
  2. コンソールウィンドウに文字が表示されることを確認
  3. エンターキーを押してプログラムを終了
期待される結果
Hello, VB.NET World!
VB.NETプログラミングへようこそ！

1.5 VB.NETプロジェクトの基本構造

作成したプロジェクトの構造を理解しましょう。

主要ファイル

  • Module1.vb: メインのソースコードファイル
  • App.config: アプリケーション設定ファイル
  • AssemblyInfo.vb: アセンブリ情報

VB.NETの基本構文要素

  • Module: プロシージャをグループ化する単位
  • Sub Main(): プログラムのエントリーポイント
  • Console.WriteLine(): コンソールへの出力
  • Console.ReadLine(): ユーザー入力の読み取り
重要な注意点

VB.NETでは大文字・小文字を区別しませんが、一貫性のあるコーディングスタイルを心がけましょう。

1.6 開発環境の設定とカスタマイズ

Visual Studioの開発効率を向上させるための設定を行います。

実習 1-3: Visual Studio設定のカスタマイズ

開発効率を向上させるためのVisual Studio設定を行いましょう。

推奨設定
  1. 「ツール」→「オプション」→「テキストエディター」→「Visual Basic」
  2. 「行番号」を有効にする
  3. 「空白の表示」を有効にする（推奨）
  4. 「インデント」をタブではなくスペース4文字に設定
  5. 「フォントとサイズ」を見やすいものに調整
理解度確認クイズ
  1. VB.NETが動作するプラットフォームの名前は何ですか？
  2. VB.NETプログラムのエントリーポイントとなるメソッド名は何ですか？
  3. コンソールに文字列を出力するために使用するメソッド名は何ですか？
  4. VB.NETの特徴として、大文字・小文字の区別について正しいものはどれですか？
  5. Visual Studioで新しいVB.NETプロジェクトを作成する際、どのテンプレートを選択しますか？
解答例
  1. .NET Framework または .NET Core
  2. Sub Main()
  3. Console.WriteLine()
  4. 大文字・小文字を区別しない
  5. コンソール アプリ (.NET Framework)

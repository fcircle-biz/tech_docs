第2章: VB.NET基本文法

VB.NET基本文法

この章で学ぶこと
  • VB.NETの変数宣言と初期化方法を理解する
  • 基本的なデータ型を学習し、適切に使い分ける
  • 定数の定義と使用方法を習得する
  • 各種演算子の使用方法を理解する
  • コメントの書き方と命名規則を身につける

2.1 変数の宣言と初期化

VB.NETでは、Dimステートメントを使用して変数を宣言します。

基本的な変数宣言

' 変数宣言の基本構文
Dim 変数名 As データ型

' 具体例
Dim userName As String
Dim age As Integer
Dim isStudent As Boolean

変数の初期化

' 宣言と同時に初期化
Dim userName As String = "田中太郎"
Dim age As Integer = 25
Dim isStudent As Boolean = True

' 後から値を代入
Dim score As Double
score = 85.5
実習 2-1: 変数の宣言と使用

様々なデータ型の変数を宣言し、値を代入・表示するプログラムを作成しましょう。

プログラムコード
Module Module1
    Sub Main()
        ' 変数の宣言と初期化
        Dim studentName As String = "佐藤花子"
        Dim studentAge As Integer = 20
        Dim gpa As Double = 3.75
        Dim isGraduated As Boolean = False
        
        ' 変数の値を表示
        Console.WriteLine("学生名: " & studentName)
        Console.WriteLine("年齢: " & studentAge & "歳")
        Console.WriteLine("GPA: " & gpa)
        Console.WriteLine("卒業済み: " & isGraduated)
        
        Console.ReadLine()
    End Sub
End Module
期待される結果
学生名: 佐藤花子
年齢: 20歳
GPA: 3.75
卒業済み: False

2.2 データ型

VB.NETには様々なデータ型が用意されています。適切なデータ型を選択することで、メモリ効率と実行速度を最適化できます。

基本データ型一覧

データ型 説明 値の範囲 例
Boolean 真偽値 True または False True, False
Byte 8ビット符号なし整数 0 ～ 255 255
Short 16ビット符号付き整数 -32,768 ～ 32,767 1000
Integer 32ビット符号付き整数 -2,147,483,648 ～ 2,147,483,647 1000000
Long 64ビット符号付き整数 非常に大きな範囲 9223372036854775807
Single 単精度浮動小数点数 32ビット浮動小数点 3.14F
Double 倍精度浮動小数点数 64ビット浮動小数点 3.14159
Decimal 固定小数点数 高精度の小数 123.45D
Char 単一の文字 Unicode文字 "A"c
String 文字列 可変長文字列 "Hello"
Date 日付と時刻 日付・時刻情報 #12/25/2023#
実習 2-2: 各種データ型の使用

異なるデータ型の変数を使用して、計算処理を行うプログラムを作成しましょう。

プログラムコード
Module Module1
    Sub Main()
        ' 各種データ型の変数
        Dim productName As String = "ノートパソコン"
        Dim unitPrice As Decimal = 89800D
        Dim quantity As Integer = 3
        Dim discount As Double = 0.1
        Dim orderDate As Date = #12/25/2023#
        Dim isInStock As Boolean = True
        
        ' 計算処理
        Dim subtotal As Decimal = unitPrice * quantity
        Dim discountAmount As Decimal = subtotal * discount
        Dim total As Decimal = subtotal - discountAmount
        
        ' 結果表示
        Console.WriteLine("=== 注文明細 ===")
        Console.WriteLine("商品名: " & productName)
        Console.WriteLine("単価: ¥" & unitPrice.ToString("N0"))
        Console.WriteLine("数量: " & quantity & "台")
        Console.WriteLine("小計: ¥" & subtotal.ToString("N0"))
        Console.WriteLine("割引: ¥" & discountAmount.ToString("N0"))
        Console.WriteLine("合計: ¥" & total.ToString("N0"))
        Console.WriteLine("注文日: " & orderDate.ToString("yyyy/MM/dd"))
        Console.WriteLine("在庫有り: " & isInStock)
        
        Console.ReadLine()
    End Sub
End Module

2.3 定数の定義

Constキーワードを使用して、プログラム実行中に変更されない値を定義できます。

' 定数の定義
Const TAX_RATE As Double = 0.1      ' 消費税率
Const COMPANY_NAME As String = "ABC株式会社"
Const MAX_RETRY_COUNT As Integer = 3

' 使用例
Dim price As Decimal = 1000D
Dim priceWithTax As Decimal = price * (1 + TAX_RATE)

2.4 演算子

VB.NETでは様々な演算子を使用して、データの計算や比較を行います。

算術演算子

演算子 説明 例 結果
+ 加算 5 + 3 8
- 減算 5 - 3 2
* 乗算 5 * 3 15
/ 除算（実数） 10 / 3 3.333...
\ 整数除算 10 \ 3 3
Mod 余り 10 Mod 3 1
^ べき乗 2 ^ 3 8

比較演算子

演算子 説明 例 結果
= 等しい 5 = 5 True
<> 等しくない 5 <> 3 True
< 未満 3 < 5 True
> より大きい 5 > 3 True
<= 以下 3 <= 5 True
>= 以上 5 >= 3 True

論理演算子

演算子 説明 例
And 論理積（両方がTrueの場合のみTrue） True And False → False
Or 論理和（いずれかがTrueの場合True） True Or False → True
Not 論理否定（TrueとFalseを反転） Not True → False

文字列演算子

' 文字列の連結
Dim firstName As String = "太郎"
Dim lastName As String = "田中"
Dim fullName As String = lastName & firstName  ' "田中太郎"

' または + 演算子も使用可能
Dim greeting As String = "こんにちは、" + fullName + "さん"
実習 2-3: 演算子を使用した計算プログラム

様々な演算子を使用して、数学的計算を行うプログラムを作成しましょう。

プログラムコード
Module Module1
    Sub Main()
        ' 基本データの設定
        Dim num1 As Integer = 15
        Dim num2 As Integer = 4
        Dim radius As Double = 5.0
        Const PI As Double = 3.14159
        
        Console.WriteLine("=== 基本演算 ===")
        Console.WriteLine(num1 & " + " & num2 & " = " & (num1 + num2))
        Console.WriteLine(num1 & " - " & num2 & " = " & (num1 - num2))
        Console.WriteLine(num1 & " * " & num2 & " = " & (num1 * num2))
        Console.WriteLine(num1 & " / " & num2 & " = " & (num1 / num2))
        Console.WriteLine(num1 & " \ " & num2 & " = " & (num1 \ num2))
        Console.WriteLine(num1 & " Mod " & num2 & " = " & (num1 Mod num2))
        Console.WriteLine("2 ^ 3 = " & (2 ^ 3))
        
        Console.WriteLine()
        Console.WriteLine("=== 円の面積計算 ===")
        Dim area As Double = PI * radius ^ 2
        Console.WriteLine("半径 " & radius & " の円の面積: " & area.ToString("F2"))
        
        Console.WriteLine()
        Console.WriteLine("=== 比較演算 ===")
        Console.WriteLine(num1 & " > " & num2 & " は " & (num1 > num2))
        Console.WriteLine(num1 & " = " & num2 & " は " & (num1 = num2))
        
        Console.ReadLine()
    End Sub
End Module

2.5 コメントの書き方

コメントはプログラムの理解性を向上させる重要な要素です。

コメントの種類

' 単一行コメント（アポストロフィを使用）
Dim userName As String = "田中"  ' 行末コメント

REM 古い形式の単一行コメント（推奨されない）

''' XML ドキュメントコメント（関数やクラスの説明用）
''' <summary>
''' このメソッドは挨拶メッセージを表示します
''' </summary>
Sub ShowGreeting()
    Console.WriteLine("こんにちは")
End Sub
コメントのベストプラクティス
  • コードが「何をするか」ではなく「なぜそうするか」を説明する
  • 複雑なロジックや重要な判断について説明する
  • 定期的にコメントを見直し、コードとの整合性を保つ

2.6 命名規則とコーディング規約

保守性の高いコードを書くために、適切な命名規則を守ることが重要です。

VB.NET命名規則

  • 変数・フィールド: camelCase（例：userName, studentAge）
  • 定数: UPPER_CASE（例：MAX_VALUE, TAX_RATE）
  • プロシージャ・メソッド: PascalCase（例：ShowMessage, CalculateTotal）
  • クラス・モジュール: PascalCase（例：Student, OrderManager）

推奨される命名例

' 良い命名例
Dim customerName As String          ' 顧客名を表す変数
Dim totalAmount As Decimal          ' 合計金額
Const MAX_RETRY_COUNT As Integer = 3    ' 最大再試行回数

Sub CalculateTotalPrice()           ' 合計価格を計算するメソッド
    ' 処理内容
End Sub

' 避けるべき命名例
Dim a As String                     ' 意味が不明
Dim data1 As Integer               ' 具体性に欠ける
Dim temp As String                 ' 一時的でも意味のある名前を
理解度確認クイズ
  1. VB.NETで変数を宣言するために使用するキーワードは何ですか？
  2. 文字列を格納するデータ型の名前は何ですか？
  3. 定数を宣言するために使用するキーワードは何ですか？
  4. 10 \ 3 の結果は何ですか？
  5. 10 Mod 3 の結果は何ですか？
  6. 文字列を連結するために使用する演算子は何ですか？
  7. 単一行コメントを書くために使用する記号は何ですか？
  8. True And False の結果は何ですか？
解答例
  1. Dim
  2. String
  3. Const
  4. 3（整数除算）
  5. 1（余り）
  6. &（アンパサンド）または +
  7. '（アポストロフィ）
  8. False

第4章: 決算と会計処理の流れ

この章で学ぶこと
  • 日々の取引記帳から財務諸表作成までの一連の流れを理解する
  • 決算整理仕訳の主要項目を学ぶ（減価償却、前払・未払、引当金、棚卸）
  • 試算表と精算表の役割を理解する
  • 税務申告と会計の違いを把握する
  • 会計ソフトでの実務イメージを持つ

4.1 決算の流れ（決算サイクル）

決算（けっさん）とは、一会計期間の最後に、帳簿を締めて財務諸表を作成し、 企業の財政状態と経営成績を確定させる作業です。通常は年に1回（年度決算）行われますが、 上場企業は四半期ごとにも決算を行います。

(仕訳入力)"] --> B["2. 月次試算表作成
(残高確認)"] B --> C["3. 決算整理仕訳
(期末調整)"] C --> D["4. 精算表作成
(整理後残高)"] D --> E["5. 財務諸表作成
(B/S, P/L, C/F)"] E --> F["6. 決算報告・承認
(株主総会等)"] F --> G["7. 税務申告
(法人税申告)"]
決算サイクルの詳細
  1. 日々の取引記帳: 売上、仕入、経費などの取引を日次で仕訳入力
  2. 月次試算表作成: 毎月末に残高を集計し、借方・貸方が一致するか確認
  3. 決算整理仕訳: 期末に減価償却費、前払費用、未払費用などを調整
  4. 精算表作成: 決算整理後の残高を整理し、財務諸表の元データを作成
  5. 財務諸表作成: 貸借対照表、損益計算書、キャッシュフロー計算書、株主資本等変動計算書を作成
  6. 決算報告と承認: 取締役会で決算承認後、株主総会で最終承認を得る
  7. 税務申告: 確定した決算書をもとに法人税申告書を作成し、税務署に提出
年次決算のスケジュール例（3月決算の場合）
時期 作業内容
3月31日 会計年度終了（決算日）
4月1日～4月15日 決算整理仕訳、試算表作成、財務諸表作成
4月下旬 監査法人による監査（上場企業）
5月中旬 取締役会で決算承認、決算短信発表（上場企業）
5月末 法人税申告書提出（申告期限は決算日から2ヶ月以内）
6月下旬 株主総会で決算承認、配当支払決定

4.2 決算整理仕訳とは

決算整理仕訳とは、期末に発生主義に基づいて正しい期間損益を計算するために行う調整仕訳です。 日々の取引では記録されない項目を、決算時にまとめて処理します。

なぜ決算整理仕訳が必要か？
  • 日々の取引では現金の動きを中心に記録するが、発生主義では現金が動かなくても計上すべき費用・収益がある
  • 固定資産の価値減少（減価償却）は日々計算せず、決算時に一括計上する
  • 前払費用や未払費用など、期間をまたぐ取引を適切に期間配分する
  • 回収不能リスクのある売掛金に対して引当金を設定する
flowchart TD A[決算整理仕訳] --> B[減価償却] A --> C[前払費用・前受収益] A --> D[未払費用・未収収益] A --> E[貸倒引当金] A --> F[棚卸資産の評価]

4.3 減価償却

減価償却（げんかしょうきゃく）とは、建物や機械など長期間使用する固定資産の価値減少を、 使用期間に応じて費用として配分する会計処理です。

減価償却の考え方

例: 500万円のトラックを購入した場合

  • 購入時に500万円全額を費用計上するのは不適切（5年間使えるのに1年目だけ費用が過大になる）
  • 5年間（耐用年数）使えるなら、500万円 ÷ 5年 = 年間100万円ずつ費用化する
  • これにより、費用収益対応の原則に従い、各年度の利益が適切に計算できる
減価償却の計算方法
方法 計算式 特徴
定額法 （取得原価 - 残存価額） ÷ 耐用年数 毎年同じ金額を償却。建物に使用されることが多い。
定率法 期首帳簿価額 × 償却率 初期に多く償却。機械装置、車両に使用されることが多い。
減価償却の仕訳

取引内容: 建物の減価償却費50万円を計上する

（借方）減価償却費 500,000 / （貸方）減価償却累計額 500,000
減価償却累計額とは？

減価償却累計額は、過去から累積した減価償却費の合計額です。 貸借対照表では固定資産からマイナスする形（評価勘定）で表示されます。

例:

  • 建物（取得原価）: 5,000万円
  • 減価償却累計額: △500万円
  • 建物（帳簿価額）: 4,500万円 ← 貸借対照表に表示される金額

4.4 前払費用・前受収益

前払費用と前受収益は、現金の支払い・受取りと、費用・収益の認識時期がずれる場合に使う勘定科目です。

前払費用（Prepaid Expenses）

前払費用は、既に支払ったが、まだサービスを受けていない部分を資産として計上します。

前払費用の例

3月31日決算の会社が、2月1日に1年分の保険料12万円（2月～翌1月分）を支払った場合：

  • 2月～3月（2ヶ月分）: 2万円 → 当期の費用
  • 4月～翌1月（10ヶ月分）: 10万円 → 次期の費用（前払費用として資産計上）

決算整理仕訳: 保険料10万円のうち、次期分を前払費用に振り替える

（借方）前払費用 100,000 / （貸方）保険料 100,000
前受収益（Unearned Revenue）

前受収益は、既に受け取ったが、まだサービスを提供していない部分を負債として計上します。

前受収益の例

3月31日決算の会社が、1月1日に1年分の家賃収入120万円（1月～12月分）を受け取った場合：

  • 1月～3月（3ヶ月分）: 30万円 → 当期の収益
  • 4月～12月（9ヶ月分）: 90万円 → 次期の収益（前受収益として負債計上）

4.5 未払費用・未収収益

未払費用と未収収益は、現金の支払い・受取りは次期だが、 サービスは当期に受けた（提供した）場合に使う勘定科目です。

未払費用（Accrued Expenses）

未払費用は、既にサービスを受けたが、まだ支払っていない部分を負債として計上します。

未払費用の例

3月31日決算の会社で、3月分の給与30万円を4月25日に支払う場合：

  • 3月分の給与は3月の費用として計上すべき（発生主義）
  • しかし現金支払いは4月なので、3月31日時点で未払費用として負債計上

決算整理仕訳: 3月分の給与30万円を未払費用として計上

（借方）給与 300,000 / （貸方）未払費用 300,000
未収収益（Accrued Revenue）

未収収益は、既にサービスを提供したが、まだ受け取っていない部分を資産として計上します。

未収収益の例

3月31日決算の会社で、貸付金の利息年間1万円を12月31日に一括で受け取る場合：

  • 1月～3月分の利息2,500円は3月期の収益
  • しかし実際の受取りは12月なので、3月31日時点で未収収益として資産計上

4.6 貸倒引当金

貸倒引当金（かしだおれひきあてきん）は、売掛金や貸付金が回収できないリスクに備えて、 あらかじめ計上しておく引当金です。

貸倒引当金の考え方
  • 売掛金1,000万円があるが、過去の経験から2%程度は回収できない可能性がある
  • 実際に倒産が確定してから費用計上するのでは、当期の利益が過大評価される
  • 将来の損失を見込んで、事前に引当金として費用計上する（保守主義の原則）

決算整理仕訳: 売掛金1,000万円に対し、2%の貸倒引当金を設定

（借方）貸倒引当金繰入 200,000 / （貸方）貸倒引当金 200,000
貸借対照表での表示

貸倒引当金は、売掛金からマイナスする形で表示されます：

  • 売掛金: 10,000,000円
  • 貸倒引当金: △200,000円
  • 売掛金（純額）: 9,800,000円 ← 実質的な回収見込額

4.7 棚卸資産の評価

棚卸資産（たなおろししさん）とは、商品や製品、仕掛品、原材料などの在庫のことです。 決算時に実際の在庫数量を確認（棚卸）し、適切に評価します。

棚卸の必要性
  • 帳簿上の在庫と実際の在庫が一致しない場合がある（盗難、破損、記帳ミス）
  • 売上原価を正確に計算するために、期末在庫の確定が必要
  • 売上原価 = 期首在庫 + 当期仕入 - 期末在庫
在庫評価方法
評価方法 説明
先入先出法
(FIFO)		 先に仕入れたものから先に売れたと仮定する方法。物価上昇時には期末在庫が高くなり、売上原価は低くなる。
移動平均法 仕入れの都度、平均単価を計算する方法。実務で広く使われる。
総平均法 期末に当期の総仕入額 ÷ 総仕入数量で平均単価を計算する方法。
棚卸減耗損と商品評価損
  • 棚卸減耗損: 帳簿の在庫数量より実際の在庫が少ない場合の損失（盗難、破損等）
  • 商品評価損: 在庫の市場価値が取得原価より下がった場合の損失（陳腐化、流行遅れ）

4.8 税務申告との関係

会計上の利益と税務上の所得は必ずしも一致しません。企業は会計基準に従って決算書を作成した後、 税法に基づいて課税所得を計算し直す必要があります。

flowchart TD A["会計上の利益
(税引前当期純利益)"] --> B[別表調整] B --> C["税務上の所得
(課税所得)"] C --> D["法人税額の計算
(税率をかける)"] B --> E["加算: 損金不算入項目
(交際費、寄附金等)"] B --> F["減算: 益金不算入項目
(受取配当金等)"]
主な損金不算入項目（税務上、費用として認められない項目）
項目 説明
交際費 一定額を超える接待交際費は損金不算入（中小企業は800万円まで損金算入可能）
寄附金 一定額を超える寄附金は損金不算入
役員給与 不相当に高額な部分や事前確定でない賞与は損金不算入
法人税等 法人税、住民税、事業税は損金不算入
罰金・科料 交通違反の罰金などは損金不算入
別表調整とは？

法人税申告書には「別表」と呼ばれる様式があり、会計上の利益と税務上の所得の違いを調整します。 特に別表四（所得金額の計算）で、損金不算入項目を加算し、益金不算入項目を減算して課税所得を計算します。

4.9 会計ソフトでの実務イメージ

現代の企業では、ほとんどの場合会計ソフトを使って日々の記帳と決算処理を行います。 クラウド型の会計ソフトが普及し、小規模企業でも効率的な会計処理が可能になっています。

主な会計ソフト
ソフト名 特徴 対象
freee 直感的なUI、銀行連携、自動仕訳機能が強力 小規模企業、個人事業主
マネーフォワード クラウド会計 経費精算やワークフローと連携、拡張性が高い 小～中規模企業
弥生会計 老舗の会計ソフト、税理士の利用率が高い 中小企業
勘定奉行 大企業向け、カスタマイズ性が高い 中～大企業
SAP、Oracle ERP グローバル企業向け、財務会計・管理会計を統合 大企業、グループ企業
会計ソフトの主な機能
  • 銀行連携: 銀行口座やクレジットカードと連携し、取引を自動取込
  • 自動仕訳: AI・機械学習で取引を自動的に仕訳提案
  • 領収書スキャン: スマホで領収書を撮影すると自動で経費計上
  • 決算書自動作成: 仕訳データから貸借対照表・損益計算書を自動生成
  • 消費税申告: 消費税の申告書も自動作成
  • 電子申告連携: e-Taxと連携し、オンラインで税務申告
会計ソフトを使っても会計知識は必要

会計ソフトは便利ですが、自動仕訳が常に正しいとは限りません。 勘定科目の選択ミスや、決算整理仕訳は自動化できない部分も多く、 基本的な会計知識を持っていないと正しい決算書が作れません。 本教材で学んだ知識があってこそ、会計ソフトを効果的に活用できます。

実習 4-1: 決算整理仕訳の練習

次の決算整理事項について、仕訳を考えてみましょう。（決算日: 3月31日）

  1. 建物の減価償却費80万円を計上する
  2. 1月1日に支払った年間保険料36万円（1月～12月分）のうち、次期分を前払費用に振り替える
  3. 3月分の給与50万円を4月に支払う予定。未払費用として計上する
  4. 売掛金500万円に対し、3%の貸倒引当金を設定する（前期末の貸倒引当金残高はゼロ）
解答例
解答を見る

1. 減価償却費の計上

（借方）減価償却費 800,000 / （貸方）減価償却累計額 800,000

2. 前払費用の振替

1月～3月（3ヶ月分）: 9万円 → 当期の費用
4月～12月（9ヶ月分）: 27万円 → 次期の費用（前払費用）

（借方）前払費用 270,000 / （貸方）保険料 270,000

3. 未払費用の計上

（借方）給与 500,000 / （貸方）未払費用 500,000

4. 貸倒引当金の設定

500万円 × 3% = 15万円

（借方）貸倒引当金繰入 150,000 / （貸方）貸倒引当金 150,000
理解度確認クイズ
  1. 決算の流れを6つのステップで説明してください。
    解答例を見る

    1. 日々の取引記帳（仕訳入力）
    2. 月次試算表作成（残高確認）
    3. 決算整理仕訳（期末調整）
    4. 精算表作成（整理後残高）
    5. 財務諸表作成（B/S、P/L、C/F）
    6. 決算報告・承認（株主総会等）

  2. 減価償却とは何ですか？なぜ必要ですか？
    解答例を見る

    減価償却とは、長期間使用する固定資産の価値減少を、使用期間に応じて費用として配分する会計処理です。 これが必要な理由は、購入時に全額費用計上すると最初の年度だけ費用が過大になり、 費用収益対応の原則に反するためです。耐用年数にわたって配分することで、 各年度の利益を適切に計算できます。

  3. 前払費用と未払費用の違いを説明してください。
    解答例を見る

    前払費用: 既に支払ったが、まだサービスを受けていない部分（資産）
    例: 年間保険料を前払いした場合の次期分

    未払費用: 既にサービスを受けたが、まだ支払っていない部分（負債）
    例: 当月分の給与を翌月に支払う場合の当月分

  4. 貸倒引当金を設定する理由を説明してください。
    解答例を見る

    売掛金や貸付金が将来回収できないリスクに備えるためです。 実際に倒産が確定してから費用計上するのでは、当期の利益が過大評価されてしまいます。 保守主義の原則に基づき、将来の損失を見込んで事前に引当金として費用計上することで、 より適切な期間損益を計算できます。

  5. 会計上の利益と税務上の所得が異なる理由を説明してください。
    解答例を見る

    会計は企業の経営成績を正しく表すことを目的としますが、 税務は公平な課税を目的としているため、基準が異なります。 例えば、交際費や寄附金は会計上は費用ですが、税務上は一定額を超えると 損金不算入（税金計算上は費用として認められない）となります。 そのため、会計上の利益に損金不算入項目を加算し、益金不算入項目を減算して、 税務上の所得（課税所得）を計算する必要があります。

第4章のまとめ
  • 決算は日々の記帳から試算表、決算整理仕訳、財務諸表作成、承認・申告という流れで進む
  • 決算整理仕訳は発生主義に基づき正しい期間損益を計算するために不可欠
  • 減価償却は固定資産の価値減少を使用期間に応じて費用配分する処理
  • 前払費用・未払費用は期間をまたぐ取引を適切に期間配分する
  • 貸倒引当金は将来の回収不能リスクに備える保守的な会計処理
  • 棚卸資産は決算時に実地棚卸を行い、適切に評価する
  • 会計上の利益と税務上の所得は異なり、別表調整が必要
  • 会計ソフトは便利だが、基本的な会計知識があってこそ正しく活用できる

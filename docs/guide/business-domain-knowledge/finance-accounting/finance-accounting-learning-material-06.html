決算処理と会計サイクル

日々の取引記帳から決算書作成までの一連の流れを学びます。決算整理仕訳の主要項目（減価償却、引当金、経過勘定）と、会計ソフトを使った実務のイメージを理解します。

この章で学ぶこと

  • 会計サイクル（日次→月次→四半期→年次決算）の流れを理解する
  • 決算整理仕訳の主要項目（減価償却、引当金、経過勘定）を習得する
  • 試算表から精算表、財務諸表作成までのプロセスを学ぶ
  • 会計上の利益と課税所得の違い、クラウド会計ソフトの活用を知る

6.1 会計サイクルとは

会計サイクル（Accounting Cycle）とは、日々の取引を記録してから財務諸表を作成し、次の期間に引き継ぐまでの一連の会計処理の流れのことです。このサイクルは期間ごとに繰り返され、企業の経営状況を定期的に把握するために不可欠なプロセスです。

会計サイクルの重要性

定期的な会計サイクルを回すことで、経営状況をタイムリーに把握でき、早期の問題発見や意思決定の質向上につながります。特に月次決算は、年次決算を待たずに毎月の業績を把握する上で重要です。

6.2 決算の流れと精算表の作成

年次決算では、日々記帳した取引データを基に、決算整理仕訳を行い、最終的な財務諸表を作成します。このプロセスを整理するために精算表（せいさんひょう）というワークシートを使用します。

精算表とは

精算表は、試算表の数値に決算整理仕訳を反映させ、最終的な財務諸表の数値を導き出すための集計表です。以下のような列構成になっています。

重要ポイント

精算表の「損益計算書」列と「貸借対照表」列は、それぞれP/LとB/Sの元となる数値を示します。損益計算書列の借方合計と貸方合計の差額が当期純利益（または純損失）になります。

6.3 決算整理仕訳の主要項目

決算整理仕訳とは、期末に行う特別な仕訳で、発生主義に基づき正確な期間損益を計算するために必要な調整です。以下の主要項目を学びます。

1. 減価償却

建物や機械設備などの固定資産は、長期間にわたって使用されるため、購入時に全額費用計上するのではなく、使用可能年数に応じて毎年少しずつ費用化します。これが減価償却です。

定額法

毎年同じ金額を減価償却

計算式：
減価償却費 = （取得原価 - 残存価額）÷ 耐用年数
例：機械200万円、耐用年数5年、残存価額0円
→ 毎年 200万円 ÷ 5年 = 40万円

定率法

期首簿価に一定率をかける（初年度が大きい）

計算式：
減価償却費 = 期首簿価 × 償却率
例：機械200万円、償却率0.4
1年目: 200万円 × 0.4 = 80万円
2年目: 120万円 × 0.4 = 48万円
減価償却の仕訳例 
【仕訳】
(借方) 減価償却費      400,000  | (貸方) 減価償却累計額  400,000

# 説明
# ・減価償却費（費用）を計上し、P/Lで利益を減少させる
# ・減価償却累計額（資産のマイナス勘定）を増やし、B/Sで資産を減少させる
# ・取得原価200万円 - 減価償却累計額40万円 = 簿価160万円（B/S表示額）

2. 貸倒引当金の設定

売掛金や受取手形の中には、取引先の倒産などにより回収できない可能性があるものがあります。この将来の損失に備えて、期末に貸倒引当金を設定します。

計算例：

期末売掛金残高: 10,000,000円
貸倒実績率: 2%
→ 貸倒引当金繰入額 = 10,000,000円 × 2% = 200,000円

貸倒引当金の仕訳例 
【仕訳】
(借方) 貸倒引当金繰入  200,000  | (貸方) 貸倒引当金      200,000

# 説明
# ・貸倒引当金繰入（費用）を計上し、P/Lで利益を減少させる
# ・貸倒引当金（資産のマイナス勘定）を設定し、B/Sで売掛金を間接的に減額
# ・売掛金1,000万円 - 貸倒引当金20万円 = 純額980万円（実質的な回収見込額）

3. 経過勘定（前払・前受・未払・未収）

取引の発生と現金の収支にズレがある場合、発生主義に基づき適切な期間に収益・費用を計上するための調整を行います。

前払費用

すでに支払ったが、次期の費用となる部分

例：3月に4月分の家賃10万円を支払った
→ 3月末に前払費用として資産計上

未払費用

すでに発生したが、まだ支払っていない費用

例：3月分の給与50万円を4月に支払う
→ 3月末に未払費用として負債計上

前受収益

すでに受け取ったが、次期の収益となる部分

例：3月に4月分の家賃20万円を受け取った
→ 3月末に前受収益として負債計上

未収収益

すでに発生したが、まだ受け取っていない収益

例：3月分の受取利息5千円を4月に受け取る
→ 3月末に未収収益として資産計上

4. 棚卸資産の評価

商品や製品、原材料などの在庫は、期末に実地棚卸を行い、帳簿上の数量と実際の在庫数量を照合します。また、仕入原価の計算方法（評価方法）を選択する必要があります。

評価方法 特徴 メリット・デメリット
先入先出法
（FIFO）		 先に仕入れた商品から先に売れたと仮定 物価上昇時は利益が大きく、在庫は時価に近い
移動平均法 仕入のたびに平均単価を再計算 平準化された原価、計算が複雑
総平均法 期間全体の平均単価を使用 計算がシンプル、期末まで確定しない

6.4 試算表の作成と検証

試算表（Trial Balance）とは、各勘定科目の残高を一覧表にまとめたもので、仕訳や転記が正しく行われているかを確認するために作成します。

残高試算表のイメージ

勘定科目 借方残高 貸方残高
現金 500,000 -
売掛金 3,000,000 -
買掛金 - 1,500,000
資本金 - 10,000,000
合計 45,000,000 45,000,000

※ 借方合計と貸方合計が一致していることを確認

試算表の重要性

試算表で借方合計と貸方合計が一致しない場合、どこかで仕訳ミスや転記ミスが発生しています。早期に発見して修正することで、決算作業を円滑に進められます。

6.5 税務申告との関係

会計上の利益と、税務申告で使用する課税所得は必ずしも一致しません。これは、会計基準と税法でルールが異なるためです。

調整項目 内容 影響
損金不算入
（加算）		 会計上は費用だが、税務上は認められない項目
（例：交際費の一部、寄付金限度超過額）		 課税所得が増加
→ 税金増
益金不算入
（減算）		 会計上は収益だが、税務上は課税対象外
（例：受取配当金の一部）		 課税所得が減少
→ 税金減

税務調整の理解

税務申告書では、会計上の利益に対して「加算項目」と「減算項目」を調整し、課税所得を計算します。この差異を理解することで、節税対策や税務リスク管理に役立ちます。

6.6 クラウド会計ソフトの活用

近年、中小企業やスタートアップではクラウド会計ソフトが広く利用されています。銀行口座やクレジットカードとの自動連携により、日々の記帳業務を大幅に効率化できます。

freee会計

  • 直感的なUI、初心者向け
  • 自動仕訳提案が充実
  • 請求書・給与計算連携

マネーフォワード

  • 会計事務所での利用が多い
  • 分析レポート機能が強力
  • 複数企業管理に対応

弥生会計

  • 老舗ブランド、シェアNo.1
  • サポート体制が充実
  • デスクトップ版も選択可

クラウド会計のメリット

  • 自動仕訳：銀行・クレカ明細から自動で仕訳を生成
  • リアルタイム集計：試算表や財務諸表をいつでも確認可能
  • 税務申告書作成：決算書から直接e-Tax連携できる

第6章のまとめ

  • 会計サイクルは日次→月次→四半期→年次の流れで繰り返され、定期的な経営状況把握に不可欠
  • 決算では試算表から精算表を作成し、決算整理仕訳を反映させて財務諸表を確定する
  • 主要な決算整理仕訳には、減価償却（定額法・定率法）、貸倒引当金、経過勘定（前払・前受・未払・未収）、棚卸資産の評価がある
  • 試算表で借方合計と貸方合計が一致することを確認し、仕訳ミスや転記ミスを早期発見する
  • 会計上の利益と課税所得は異なり、税務調整（損金不算入・益金不算入）により法人税を計算する
  • クラウド会計ソフト（freee、マネーフォワード、弥生）により、記帳業務を自動化・効率化できる

理解度確認クイズ

  1. Q1. 決算整理仕訳の目的は何ですか？

    回答を表示

    回答：決算整理仕訳の目的は、発生主義に基づき正確な期間損益を計算することです。日々の仕訳だけでは反映されない期末の調整（減価償却、引当金、経過勘定など）を行い、会計期間に対応する収益と費用を適切に対応させることで、企業の真実な財政状態と経営成績を財務諸表に表します。

  2. Q2. 減価償却の定額法と定率法の違いを説明してください。

    回答を表示

    回答：
    定額法：毎年同じ金額を減価償却する方法。計算式は（取得原価 - 残存価額）÷ 耐用年数。費用が平準化されるため、P/Lが安定する。
    定率法：期首簿価に一定率をかけて減価償却する方法。初年度の償却費が大きく、年々減少していく。早期に費用を多く計上できるため、節税効果が期待できる。

  3. Q3. 会計上の利益と課税所得が異なる理由を説明してください。

    回答を表示

    回答：会計上の利益は会計基準に基づいて計算されるのに対し、課税所得は税法に基づいて計算されるため、両者のルールが異なるからです。例えば、交際費の一部は会計上は費用として認められますが、税法上は損金不算入（課税所得の計算では費用として認められない）とされる場合があります。このような差異を税務調整により修正し、課税所得を算出します。

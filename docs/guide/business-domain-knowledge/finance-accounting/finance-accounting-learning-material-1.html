第1章: 会計の基本概念（導入）

この章で学ぶこと
  • 会計とは何か、その役割と目的を理解する
  • 財務会計と管理会計の違いを説明できる
  • 会計サイクルの全体像を把握する
  • 現金主義と発生主義の違いを理解する
  • 継続企業の前提（ゴーイングコンサーン）と会計期間の概念を学ぶ

1.1 会計とは何か

会計とは、企業の経済活動を記録・計算・整理し、その結果を報告することで、企業の財政状態や経営成績を明らかにする仕組みです。 簡単に言えば、「企業活動をお金の面から見える化する技術」と言えます。

会計の3つの役割
  1. 記録（Recording）: 日々の取引を正確に記録する
  2. 計算（Calculating）: 記録した取引から利益や資産を計算する
  3. 報告（Reporting）: 計算結果を財務諸表などで関係者に報告する

会計は、企業だけでなく、非営利団体や政府機関でも利用されています。本教材では主に企業会計（ビジネス会計）を扱いますが、 その基本原則は他の組織にも応用できます。

1.2 財務会計と管理会計

会計は大きく分けて財務会計（Financial Accounting）と管理会計（Management Accounting）の2つに分類されます。 それぞれ目的や対象者が異なります。

flowchart TD A[会計] --> B["財務会計
(外部報告)"] A --> C["管理会計
(内部管理)"] B --> D["株主・投資家
債権者・税務当局"] C --> E["経営者・管理者
社内の意思決定"]
比較項目 財務会計 管理会計
目的 外部のステークホルダーへの情報提供 内部の経営判断・業績評価
対象者 株主、投資家、債権者、税務当局 経営者、管理者、部門長
法的規制 法律・会計基準に従う（義務） 企業が自由に設計できる
報告頻度 四半期・年次決算（定期的） 必要に応じて随時（日次・週次・月次）
主な成果物 財務諸表（B/S、P/L、C/F） 予算管理表、部門別損益、原価計算書
具体例で理解する

財務会計の例:

  • 上場企業が年度末に有価証券報告書を金融庁に提出する
  • 銀行から融資を受ける際に決算書を提出する
  • 税務署に法人税の申告をする

管理会計の例:

  • 営業部門が毎月の売上目標と実績を比較する
  • 製造部門が製品ごとの原価を計算して利益率を改善する
  • 経営会議で次年度の予算計画を策定する

1.3 ステークホルダーと情報ニーズ

企業を取り巻く様々なステークホルダー（利害関係者）が、それぞれ異なる目的で会計情報を必要としています。

ステークホルダー 関心事項 必要な情報
株主・投資家 投資した資金は増えるか？配当はあるか？ 利益、ROE、配当金、株価
債権者（銀行等） 貸したお金は返ってくるか？利息は払えるか？ 自己資本比率、流動比率、キャッシュフロー
経営者 どの事業が儲かっているか？どこを改善すべきか？ 部門別損益、原価、予実差異、KPI
税務当局 適正な税金が納められているか？ 課税所得、損金算入の妥当性
従業員 会社は安定しているか？給与は増えるか？ 売上高、利益、成長性
取引先 継続的に取引できるか？倒産リスクはないか？ 安全性指標、支払能力

1.4 会計サイクル（取引→記帳→決算→報告）

企業の会計業務は、会計サイクルと呼ばれる一連のプロセスを繰り返します。 このサイクルを理解することが、会計全体の流れを把握する第一歩です。

flowchart LR A["1. 取引発生
(商品販売・仕入等)"] --> B["2. 記帳
(仕訳・帳簿記入)"] B --> C["3. 試算表作成
(月次チェック)"] C --> D["4. 決算整理
(減価償却等)"] D --> E["5. 財務諸表作成
(B/S、P/L、C/F)"] E --> F["6. 報告・開示
(株主・税務署へ)"] F --> A
会計サイクルの詳細
  1. 取引発生: 日々の事業活動（売上、仕入、経費支払い等）が発生
  2. 記帳（仕訳）: 取引を会計ルールに基づいて記録（第2章で詳しく学習）
  3. 試算表作成: 月次で帳簿が合っているか確認
  4. 決算整理: 期末に減価償却や引当金計上などの調整処理を実施（第4章で学習）
  5. 財務諸表作成: B/S（貸借対照表）、P/L（損益計算書）、C/F（キャッシュフロー計算書）を作成（第3章で学習）
  6. 報告・開示: ステークホルダーに財務情報を提供

このサイクルは、日次→月次→四半期→年次のスパンで繰り返されます。 特に年次決算は重要で、1年間の経営成績を確定させ、株主総会での承認を得ます。

1.5 現金主義と発生主義

会計では、取引をいつ認識するか（いつ帳簿に記録するか）という認識基準が非常に重要です。 主な方法として現金主義と発生主義があります。

現金主義（Cash Basis）

現金の入出金があった時点で収益や費用を認識する方法です。 シンプルで分かりやすいですが、企業の実態を正確に表せないという問題があります。

現金主義の例

6月1日に商品を販売し、代金10万円は7月31日に入金される場合：

  • 現金主義: 7月31日に売上10万円を計上（現金が入った日）
  • 6月の売上はゼロ、7月の売上が10万円となる
  • → 実際には6月に商品を引き渡しているのに、6月の業績に反映されない
発生主義（Accrual Basis）

経済事象が発生した時点で収益や費用を認識する方法です。 現代の企業会計では、この発生主義が原則として採用されています。

発生主義の例

6月1日に商品を販売し、代金10万円は7月31日に入金される場合：

  • 発生主義: 6月1日に売上10万円を計上（商品を引き渡した日）
  • 6月の売上が10万円、7月は売掛金の回収のみ
  • → 実際の経済活動（商品引渡し）が正しく反映される
比較項目 現金主義 発生主義
認識タイミング 現金の入出金時 経済事象の発生時
メリット シンプル、分かりやすい、現金管理に適する 企業の実態を正確に表せる、期間損益が適切
デメリット 売掛金・買掛金を考慮できない、期間損益が歪む 複雑、未払・未収の管理が必要
適用例 小規模個人事業、家計簿 企業会計（法人）、上場企業
注意

日本の企業会計では発生主義が原則です。 現金主義は個人事業主の青色申告などで一部認められていますが、企業規模が大きくなると発生主義での記帳が必須となります。

1.6 実現主義と費用収益対応の原則

実現主義（Realization Principle）

発生主義の中でも、特に収益の認識については実現主義が適用されます。 実現主義とは、「商品やサービスを提供し、対価（代金）を受け取る権利が確定した時点」で収益を認識する考え方です。

実現の2要件
  1. 財貨の移転またはサービスの提供: 商品を引き渡した、サービスを提供した
  2. 対価の成立: 代金を受け取るか、受け取る権利が確定した（現金または売掛金）

この2つが満たされた時点で、初めて売上として計上できます。

費用収益対応の原則（Matching Principle）

収益とそれを得るために使った費用を同じ期間に計上するという原則です。 これにより、適切な期間損益計算が可能になります。

費用収益対応の例

9月に商品100万円分を仕入れ、10月に150万円で販売した場合：

  • 10月の売上: 150万円（実現主義により商品引渡し時）
  • 10月の売上原価: 100万円（対応する費用も10月に計上）
  • 10月の利益: 50万円（収益と費用を同じ期間で対応）

もし仕入を9月に計上し、売上を10月に計上すると、9月は赤字、10月は過大利益となり、正しい評価ができません。

1.7 継続企業の前提（ゴーイングコンサーン）

会計では、企業が将来にわたって事業を継続することを前提としています。 これを継続企業の前提（Going Concern Assumption）と呼びます。

継続企業の前提が意味すること
  • 企業はすぐに清算・倒産することなく、長期にわたって営業を続ける
  • 設備投資や在庫は、将来の事業のために使われる（すぐに売却しない）
  • この前提があるからこそ、減価償却や繰延資産などの会計処理が成立する
継続企業の前提が疑われる場合

企業が重大な債務超過や連続赤字などで倒産リスクが高い場合、「継続企業の前提に関する注記」を財務諸表に記載する必要があります。 この注記がある企業は、投資家から警戒されます。

会計期間の概念

企業活動は継続的ですが、一定期間ごとに区切って業績を測定する必要があります。 この区切られた期間を会計期間と呼びます。

主な会計期間
  • 事業年度（会計年度）: 通常1年間（例：4月1日～翌3月31日）
  • 四半期: 3ヶ月ごと（上場企業は四半期報告が義務）
  • 月次: 毎月の業績管理（管理会計で使用）

日本では4月～3月の決算期が多いですが、企業は自由に決算期を設定できます。 上場企業の約6割が3月決算、その他は12月決算なども多いです。

実習 1-1: 会計の基本概念を身近な例で考える

あなたが運営する小さなオンラインショップを例に、会計の基本概念を考えてみましょう。

シナリオ

あなたは手作りアクセサリーをオンラインで販売しています。以下の取引が発生しました：

  • 4月10日: 材料費5,000円を現金で支払い
  • 4月15日: アクセサリーを完成させる
  • 4月20日: 顧客に10,000円で販売（商品発送）
  • 4月30日: 顧客から代金10,000円が銀行口座に入金される
課題
  1. 現金主義の場合、4月の売上はいくらですか？また、費用はいくらですか？
  2. 発生主義の場合、4月の売上はいくらですか？また、費用はいくらですか？
  3. どちらの方法が実態を正しく表していると思いますか？その理由を説明してください。
  4. この取引で、実現主義の2要件（財貨の移転と対価の成立）が満たされたのはいつですか？
考察のヒント
  • 現金主義では、現金の動きがあった日を基準にする
  • 発生主義では、経済事象（商品発送、材料購入）が起きた日を基準にする
  • 費用収益対応の原則を考慮すると、売上と売上原価は同じ月に計上すべき
理解度確認クイズ
  1. 財務会計と管理会計の主な違いは何ですか？（3つ挙げてください）
    解答例を見る

    1. 目的: 財務会計は外部報告、管理会計は内部管理
    2. 対象者: 財務会計は株主・投資家等、管理会計は経営者・管理者
    3. 法的規制: 財務会計は法律・会計基準に従う、管理会計は自由設計

  2. 会計サイクルの6つのステップを順番に並べてください。
    解答例を見る

    1. 取引発生 → 2. 記帳（仕訳） → 3. 試算表作成 → 4. 決算整理 → 5. 財務諸表作成 → 6. 報告・開示

  3. 次の取引について、発生主義ではいつ売上を計上しますか？
    3月25日に商品を発送し、4月10日に代金が入金された。
    解答例を見る

    3月25日に売上を計上します。発生主義では、商品を引き渡した時点（経済事象の発生時）で 収益を認識するためです。現金の入金日（4月10日）ではありません。

  4. 継続企業の前提（ゴーイングコンサーン）とは何ですか？
    解答例を見る

    企業が将来にわたって事業を継続することを前提とする会計の基本的な考え方です。 企業はすぐに清算・倒産することなく、長期的に営業を続けるという前提に基づいて、 減価償却や在庫評価などの会計処理が行われます。

  5. 費用収益対応の原則とは何ですか？簡単な例を挙げて説明してください。
    解答例を見る

    収益とそれを得るために使った費用を同じ期間に計上する原則です。
    例: 9月に100万円で仕入れた商品を10月に150万円で販売した場合、 売上150万円と売上原価100万円の両方を10月に計上します。 こうすることで、10月の適正な利益50万円が計算できます。

第1章のまとめ
  • 会計は企業の経済活動を記録・計算・報告する仕組みである
  • 財務会計は外部報告用、管理会計は内部管理用と目的が異なる
  • ステークホルダーごとに必要な会計情報が異なる
  • 会計サイクルは「取引→記帳→決算→報告」の流れを繰り返す
  • 現金主義より発生主義の方が企業の実態を正確に表せる
  • 実現主義により、商品引渡し時に収益を認識する
  • 継続企業の前提に基づき、長期的視点で会計処理を行う

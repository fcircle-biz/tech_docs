貸借対照表（B/S）の構造と読み方

企業の財政状態を表す貸借対照表（バランスシート）の構造を学びます。資産・負債・純資産の各区分の内容を理解し、B/Sから企業の財務健全性を読み取る方法を習得します。

この章で学ぶこと

  • 貸借対照表の基本構造と「資産 = 負債 + 純資産」の関係を理解する
  • 資産の部の構成（流動資産、固定資産、繰延資産）を学ぶ
  • 負債の部の構成（流動負債、固定負債）と純資産の部を理解する
  • 実在企業のB/Sから財務健全性を読み取る演習を行う

3.1 貸借対照表（B/S）とは何か

貸借対照表（Balance Sheet、B/S）は、企業のある時点における財政状態を表す財務諸表です。英語でBalance Sheetと呼ばれることから「バランスシート」とも呼ばれます。

B/Sが示すもの

貸借対照表は、「企業がどのように資金を調達し（右側）、その資金をどのように運用しているか（左側）」を一覧表にしたものです。決算日時点での財産状況を「スナップショット」のように表します。

貸借対照表の基本構造

貸借対照表は、左側に資産、右側に負債と純資産を配置します。この配置には重要な意味があります。

貸借対照表等式

貸借対照表は常に以下の関係が成り立ちます：
資産 = 負債 + 純資産
これを貸借対照表等式といい、複式簿記の根幹をなす原則です。

3.2 資産の部：企業の財産

資産とは、企業が保有する将来的に経済的利益をもたらすものです。資産は、現金化しやすさ（流動性）によって分類されます。

流動資産

流動資産は、1年以内に現金化される資産です。企業の短期的な支払能力を示す重要な指標となります。

勘定科目 内容 具体例
現金及び預金 手元資金と銀行預金 現金、普通預金、定期預金
売掛金 商品販売後の未回収代金 掛け売りの売上代金
棚卸資産 販売目的で保有する在庫 商品、製品、原材料、仕掛品
有価証券 短期で売却予定の株式・債券 売買目的有価証券

固定資産

固定資産は、長期にわたって事業で使用する資産です。有形・無形・投資の3つに分類されます。

有形固定資産

物理的な形がある資産

  • 建物
  • 機械装置
  • 土地
  • 車両運搬具

無形固定資産

物理的形がない資産

  • ソフトウェア
  • 特許権・商標権
  • のれん
  • 営業権

投資その他

長期投資目的の資産

  • 投資有価証券
  • 長期貸付金
  • 敷金・保証金
  • 関係会社株式

繰延資産とは

繰延資産は、すでに支払った費用のうち、将来にわたって効果が及ぶものを資産として計上するものです。創立費、開業費、株式交付費、開発費などが該当し、数年にわたって償却（費用化）していきます。

3.3 負債の部：他人資本

負債とは、企業が将来返済しなければならない義務です。「他人資本」とも呼ばれ、返済期限によって流動負債と固定負債に分類されます。

流動負債と固定負債の違い

流動負債

1年以内に返済義務が到来する負債

  • 買掛金：商品仕入の未払代金
  • 短期借入金：1年以内返済の借入
  • 未払金：経費や固定資産の未払い
  • 未払法人税等：税金の未払い

固定負債

返済期限が1年を超える負債

  • 長期借入金：銀行からの長期借入
  • 社債：企業が発行した債券
  • 退職給付引当金：将来の退職金
  • リース債務：長期リース契約

3.4 純資産の部：自己資本

純資産とは、返済義務のない企業の正味の財産です。「自己資本」とも呼ばれ、企業の財務健全性を示す重要な指標です。

株主資本の構成

項目 内容 変動要因
資本金 株主からの出資額 増資、減資
資本剰余金 資本取引による剰余金 資本金の組入れ、自己株式処分差益
利益剰余金 過去の利益の蓄積 当期純利益の加算、配当金の減算
自己株式 買い戻した自社株（控除項目） 自己株式の取得・処分

利益剰余金の重要性

利益剰余金は、企業が過去から積み上げてきた利益の蓄積です。損益計算書（P/L）の当期純利益がここに加算され、配当金が差し引かれます。利益剰余金が大きいほど、企業の内部留保が厚く、財務的に安定していると言えます。

このB/Sから読み取れること

  • 資産合計 = 負債＋純資産が成り立っている（2,500 = 1,200 + 1,300）
  • 流動比率 = 200%（流動資産1,200 ÷ 流動負債600）で、短期的な支払能力は良好
  • 自己資本比率 = 52%（純資産1,300 ÷ 資産合計2,500）で、財務健全性は普通レベル
  • 利益剰余金800百万円があり、過去から利益を蓄積している安定企業

第3章のまとめ

  • 貸借対照表（B/S）は、ある時点の企業の財政状態を示し、「資産 = 負債 + 純資産」の関係が常に成り立つ
  • 資産は流動資産（1年以内に現金化）と固定資産（長期保有）に分類される
  • 負債は返済義務のあるお金で、流動負債（1年以内返済）と固定負債（長期返済）に分類される
  • 純資産は返済不要の自己資本で、資本金と利益剰余金が主要な構成要素である
  • B/Sから流動比率や自己資本比率などを計算し、企業の財務健全性を評価できる

理解度確認クイズ

  1. Q1. 貸借対照表等式「資産 = 負債 + 純資産」が成り立つのはなぜですか？

    回答を表示

    回答：貸借対照表の左側（資産）は「調達した資金の運用形態」を、右側（負債＋純資産）は「資金の調達源泉」を表しています。企業が調達した資金（負債と純資産）は、必ず何らかの資産として運用されるため、常に両者は一致します。これが複式簿記の根幹原理です。

  2. Q2. 流動資産と固定資産の違いを、具体例を使って説明してください。

    回答を表示

    回答：流動資産は1年以内に現金化される資産で、現金預金、売掛金、棚卸資産（商品在庫）などが該当します。一方、固定資産は長期にわたって事業で使用する資産で、建物、機械、土地、ソフトウェアなどが該当します。流動資産は短期的な支払能力を、固定資産は事業の継続性を示します。

  3. Q3. 純資産（自己資本）が大きい企業は、なぜ財務的に健全だと言えるのですか？

    回答を表示

    回答：純資産は返済義務のない自己資本であり、株主からの出資や過去の利益の蓄積です。純資産が大きいということは、借入金に頼らず自己資金で事業を運営できていることを意味し、経済環境の変化や業績悪化時にも倒産リスクが低いと評価されます。自己資本比率（純資産÷資産）が高いほど財務健全性が高いとされています。

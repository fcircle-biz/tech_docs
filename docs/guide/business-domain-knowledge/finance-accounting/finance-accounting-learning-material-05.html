キャッシュ・フロー計算書（C/F）の理解

企業の資金の流れを表すキャッシュ・フロー計算書の構造を学びます。3つの活動区分（営業・投資・財務）の意味を理解し、「利益が出ているのに資金繰りが苦しい」といった現象の原因を読み解けるようになります。

この章で学ぶこと

  • キャッシュ・フロー計算書の役割と利益と現金の違いを理解する
  • 3つの活動区分（営業・投資・財務）の意味と読み方を学ぶ
  • 間接法と直接法の違い、フリー・キャッシュ・フローの概念を把握する
  • B/S・P/L・C/Fの三表のつながりと黒字倒産の仕組みを理解する

5.1 キャッシュ・フロー計算書とは

キャッシュ・フロー計算書（Cash Flow Statement、略してC/F）は、企業の一定期間における現金及び現金同等物の増減を表す財務諸表です。損益計算書が「利益」を示すのに対し、キャッシュ・フロー計算書は「実際の現金の流れ」を表します。

利益と現金は違う！

損益計算書では黒字（利益が出ている）でも、現金が不足して倒産することがあります。これを「黒字倒産」といいます。売掛金の回収遅延や過剰な在庫、設備投資により手元現金が枯渇することが原因です。

なぜ利益と現金がズレるのか？

企業会計では発生主義を採用しているため、売上は「現金受領時」ではなく「商品を納品した時」に計上されます。このため、利益と実際の現金残高にはタイムラグが生じます。

損益計算書（P/L）

発生主義に基づく「利益」

例：12月に商品100万円を販売（代金は翌月回収）
→ 12月の売上：100万円
→ 利益が出る

CF計算書（C/F）

実際の「現金の増減」

同じケース：
→ 12月の現金増加：0円
→ 翌1月に100万円回収
→ 手元現金がない！

現金及び現金同等物とは

CF計算書で扱う「現金」には、手元現金だけでなく「現金同等物」も含まれます。現金同等物とは、容易に換金可能で価値変動リスクの少ない資産（3ヶ月以内の定期預金、譲渡性預金など）のことです。

5.2 キャッシュ・フローの3つの活動区分

キャッシュ・フロー計算書は、企業の活動を営業活動、投資活動、財務活動の3つに区分して現金の流れを示します。

営業活動CF

本業での資金創出力

  • 商品・サービスの販売による現金収入
  • 商品の仕入、人件費、経費の支払い
プラス：健全な経営
マイナス：本業で資金を失っている

投資活動CF

設備投資・資産売買

  • 固定資産の売却、有価証券の売却
  • 設備投資、有価証券の購入
マイナス：積極的な投資（成長期）
プラス：資産売却（資金調達）

財務活動CF

資金調達・返済

  • 借入金の借入、社債の発行、増資
  • 借入金の返済、配当金の支払い
プラス：外部から資金調達
マイナス：返済・株主還元

重要ポイント

営業活動CFは必ずプラスであるべきです。本業で現金を稼げない企業は、借入や資産売却でしのいでいる状態であり、長期的には持続不可能です。

5.3 間接法と直接法の違い

営業活動によるキャッシュ・フローを計算する方法には、間接法と直接法の2種類があります。日本企業の多くは間接法を採用しています。

間接法（日本で主流）

損益計算書の当期純利益から調整

計算手順：

① 当期純利益（P/Lから）

② ＋減価償却費（現金支出なし）

③ ±売掛金・買掛金の増減

④ ±棚卸資産の増減

⑤ ±その他の調整

  • 作成が簡単、P/Lとの関係が明確

直接法（参考）

現金収支を直接集計

計算手順：

① 営業収入（現金ベース）

② －仕入支出

③ －人件費支出

④ －経費支出

⑤ －法人税等の支払

  • 現金の流れが直感的に分かる

どちらの方法も営業CFの合計は同じ

間接法と直接法は表示方法が異なるだけで、最終的な営業活動によるキャッシュ・フローの金額は同じになります。間接法は作成が簡単なため、多くの企業で採用されています。

5.4 フリー・キャッシュ・フロー（FCF）

フリー・キャッシュ・フロー（Free Cash Flow、FCF）とは、企業が自由に使える現金のことです。営業活動で稼いだ現金から、事業維持に必要な設備投資を差し引いた金額を指します。

FCFの計算式

フリー・キャッシュ・フロー（FCF）

=

営業活動によるCF － 投資活動によるCF

※投資活動CFは通常マイナスなので、実質的には「営業CF ＋ |投資CF|」

FCFがプラス

本業で稼いだ現金が設備投資を上回る

  • 借入の返済ができる
  • 株主に配当を支払える
  • 成長投資に回せる
→ 財務的に余裕がある状態

FCFがマイナス

設備投資が営業CFを上回る

  • 借入や増資で資金調達が必要
  • 配当の支払いが難しい
  • 成長期の企業では正常
→ 外部資金に依存している状態

重要ポイント

FCFがプラスであることは、企業が「自力で成長できる力」を持っていることを示します。成長期の企業では一時的にマイナスになることもありますが、成熟企業では継続的にプラスであるべきです。

5.6 財務三表のつながり（B/S・P/L・C/F）

貸借対照表（B/S）、損益計算書（P/L）、キャッシュ・フロー計算書（C/F）は、それぞれ独立した財務諸表ではなく、相互に関連しています。

B/S

貸借対照表

ある時点の財政状態
「資産」「負債」「純資産」

→ ストック情報（残高）

P/L

損益計算書

一定期間の経営成績
「収益」「費用」「利益」

→ フロー情報（期間成績）

C/F

CF計算書

一定期間の現金増減
「営業」「投資」「財務」

→ フロー情報（現金の流れ）

三表のつながりを理解しよう

P/Lの当期純利益 → B/Sの利益剰余金に蓄積される
C/Fの現金増減 → B/Sの現金残高に反映される
P/Lの当期純利益 → C/Fの営業活動CFの起点（間接法）

5.7 黒字倒産の仕組みとキャッシュの重要性

黒字倒産とは、損益計算書上は利益が出ているにもかかわらず、手元の現金が不足して支払いができなくなり倒産することです。利益と現金は別物であることを象徴する現象です。

黒字倒産が起きる主な原因

売掛金の回収遅延

売上は計上したが現金が入ってこない

例：
・12月に1000万円を売上計上
・回収は3ヶ月後の3月
・その間の仕入代金は現金払い
→ 手元現金が枯渇

過剰な在庫

商品を仕入れたが売れず在庫になる

例：
・500万円の商品を仕入（現金支払）
・商品が売れず在庫のまま
→ 現金だけ減る
（P/L上は在庫なので費用計上されず）

過大な設備投資

大型設備投資で現金が一気に流出

例：
・新工場建設に2億円支出
・P/Lでは減価償却で少しずつ費用化
→ 支払いは一括
→ 現金が一気に減少

急成長による資金不足

売上急拡大で運転資金が足りない

例：
・売上が倍増し仕入も増加
・仕入代金の支払いは即時
・売上代金の回収は後日
→ 一時的に資金ショート

黒字倒産を防ぐには

  • キャッシュ・フロー計算書を定期的にチェック：営業CFがプラスか確認
  • 売掛金の回収サイト短縮：早期回収交渉、前払い制度導入
  • 在庫管理の徹底：過剰在庫を持たない、適正在庫を維持
  • 資金繰り表の作成：月次・週次で現金の入出金予測を立てる
  • 借入枠の確保：当座貸越、コミットメントラインなど緊急資金源を用意

キャッシュは企業の血液

「利益は意見、キャッシュは事実（Profit is opinion, cash is fact）」という格言があります。会計処理によって利益は変動しますが、現金残高は事実です。企業存続のためには、キャッシュ・フロー管理が不可欠です。

第5章のまとめ

  • キャッシュ・フロー計算書は企業の現金の流れを示す財務諸表であり、利益と現金の違いを明らかにする
  • 3つの活動区分（営業・投資・財務）により、現金がどのように増減したかを把握できる
  • 営業活動CFは必ずプラスであるべきで、本業で現金を稼げない企業は持続不可能
  • フリー・キャッシュ・フロー（FCF）は企業が自由に使える現金を示し、財務的余裕の指標となる
  • CFパターン分析により、企業が成長期・成熟期・衰退期のどの段階にあるかを読み取れる
  • B/S・P/L・C/Fの三表は相互に関連しており、総合的に見ることで企業の実態を把握できる
  • 黒字倒産は利益が出ていても現金不足で起こる現象であり、キャッシュ管理の重要性を示している

  1. Q1. 営業活動CF、投資活動CF、財務活動CFのそれぞれの意味を説明してください。

    回答を表示

    回答：
    営業活動CF：本業での資金創出力を示す。商品販売・仕入・経費支払いなどの現金増減
    投資活動CF：設備投資や資産売却による現金増減。固定資産・有価証券の売買
    財務活動CF：資金調達と返済の状況。借入・返済・配当支払いなど

  2. Q2. フリー・キャッシュ・フロー（FCF）とは何か、計算式と意味を説明してください。

    回答を表示

    回答：FCFは企業が自由に使える現金のこと。
    計算式：FCF = 営業活動によるCF － 投資活動によるCF
    意味：本業で稼いだ現金から、事業維持に必要な設備投資を差し引いた残りの現金。FCFがプラスなら借入返済や配当支払いが可能で、財務的余裕があることを示す。

  3. Q3. 黒字倒産とは何か、なぜ起きるのか説明してください。

    回答を表示

    回答：黒字倒産とは、損益計算書上は利益が出ているにもかかわらず、手元現金が不足して支払いができなくなり倒産すること。
    原因：①売掛金の回収遅延（売上は計上したが現金未回収）、②過剰在庫（仕入代金は支出したが商品が売れず在庫のまま）、③過大な設備投資（一括支払いで現金が一気に流出）、④急成長による運転資金不足など。発生主義会計では利益と現金のタイミングがズレるため起こる。

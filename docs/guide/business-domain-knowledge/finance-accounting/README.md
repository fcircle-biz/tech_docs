# 財務・管理会計入門 学習ガイドライン

このガイドラインでは、財務・管理会計の基礎を入門者向けに段階的に学習するためのカリキュラムを提供しています。経理部門以外の方でも、企業活動を数字で理解し、財務諸表を正しく読み取る力を身につけることができます。ビジネスパーソンとして「数字で語れる」スキルを習得することを目指します。

## 前提条件

### 必要な環境
- 表計算ソフト（Microsoft Excel、Google Sheets等）
- 電卓または表計算ソフトの計算機能
- テキストエディタまたはノートアプリ（演習用）
- Webブラウザ（参考資料閲覧用）

### 参考リソース
- [企業会計基準委員会（ASBJ）](https://www.asb.or.jp/) - 日本の会計基準を策定する機関
- [日本公認会計士協会](https://jicpa.or.jp/) - 会計・監査に関する情報
- [金融庁：EDINET](https://disclosure2.edinet-fsa.go.jp/) - 有価証券報告書の閲覧
- [東京証券取引所](https://www.jpx.co.jp/) - 決算短信、適時開示情報
- [国税庁](https://www.nta.go.jp/) - 法人税、確定申告関連情報
- [中小企業庁：経営力向上計画](https://www.chusho.meti.go.jp/) - 中小企業の経営指標

### 前提知識
- **必須**: 基本的な四則演算（加減乗除、割合の計算）
- **必須**: Excelの基本操作（セル入力、簡単な数式作成）
- **推奨**: 企業の組織構造や業務フローの基本的理解
- **推奨**: 損益計算書や貸借対照表を見たことがある（読めなくてもOK）

## 学習コンテンツ

### [1. 会計の基礎概念と役割](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-01.html)
会計とは何か、なぜビジネスパーソンに会計知識が必要なのかを学びます。財務会計（外部報告）と管理会計（内部管理）の違い、会計情報の利用者（ステークホルダー）、会計の基本原則（発生主義、継続企業の前提、重要性の原則）について理解します。

**学習内容:**
- 会計の定義と2つの目的（説明責任と意思決定支援）
- 財務会計と管理会計の違いと相互関係
- ステークホルダー別の情報ニーズ（株主、債権者、経営者、従業員、税務当局）
- 会計の基本原則：発生主義 vs 現金主義
- 継続企業の前提（ゴーイングコンサーン）
- 会計期間と期間損益計算
- 会計基準の役割（日本基準、IFRS）

### [2. 簿記の基本と仕訳の実践](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-02.html)
複式簿記の仕組みと、会計の5要素（資産・負債・純資産・収益・費用）について学びます。借方・貸方のルールを理解し、実務でよく発生する取引の仕訳パターンを演習を通じて習得します。

**学習内容:**
- 複式簿記の原理と貸借平均の原則
- 会計の5要素の定義と分類
  - 資産：現金、売掛金、商品、建物、備品など
  - 負債：買掛金、借入金、未払金など
  - 純資産：資本金、利益剰余金
  - 収益：売上高、受取利息、雑収入
  - 費用：仕入、給与、家賃、減価償却費
- 借方・貸方のルールと仕訳の構造
- 主要な勘定科目の一覧と使い分け
- 基本的な仕訳パターン20例の演習
- 仕訳帳から総勘定元帳への転記

### [3. 貸借対照表（B/S）の構造と読み方](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-03.html)
企業の財政状態を表す貸借対照表（バランスシート）の構造を学びます。資産・負債・純資産の各区分の内容を理解し、B/Sから企業の財務健全性を読み取る方法を習得します。

**学習内容:**
- 貸借対照表の基本構造と「資産 = 負債 + 純資産」の関係
- 資産の部の構成
  - 流動資産：現金預金、売掛金、棚卸資産（1年以内に現金化）
  - 固定資産：有形固定資産、無形固定資産、投資その他の資産
  - 繰延資産の概念
- 負債の部の構成
  - 流動負債：買掛金、短期借入金、未払金（1年以内に返済）
  - 固定負債：長期借入金、社債、退職給付引当金
- 純資産の部の構成
  - 株主資本：資本金、資本剰余金、利益剰余金、自己株式
  - その他の包括利益累計額
- 実在企業のB/Sを読む演習

### [4. 損益計算書（P/L）の構造と読み方](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-04.html)
企業の経営成績を表す損益計算書の構造を学びます。売上高から当期純利益までの各段階の利益（5つの利益）の意味を理解し、P/Lから企業の収益力を読み取る方法を習得します。

**学習内容:**
- 損益計算書の基本構造（報告式）
- 5つの利益と各段階の意味
  - 売上総利益（粗利）：売上高 - 売上原価
  - 営業利益：本業での儲け
  - 経常利益：通常の企業活動での利益
  - 税引前当期純利益：特別損益を含む利益
  - 当期純利益：最終的な利益
- 売上原価の計算方法（期首在庫 + 仕入 - 期末在庫）
- 販売費及び一般管理費（販管費）の主要項目
- 営業外損益と特別損益の違い
- 実在企業のP/Lを読む演習
- B/SとP/Lのつながり（利益剰余金）

### [5. キャッシュ・フロー計算書（C/F）の理解](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-05.html)
企業の資金の流れを表すキャッシュ・フロー計算書の構造を学びます。3つの活動区分（営業・投資・財務）の意味を理解し、「利益が出ているのに資金繰りが苦しい」といった現象の原因を読み解けるようになります。

**学習内容:**
- キャッシュ・フロー計算書の役割（利益と現金の違い）
- 3つの活動区分
  - 営業活動によるCF：本業での資金創出力
  - 投資活動によるCF：設備投資、有価証券売買
  - 財務活動によるCF：借入、返済、配当支払い
- 間接法と直接法の違い
- フリー・キャッシュ・フロー（FCF）の概念と計算
- CFパターン分析（成長企業、成熟企業、衰退企業）
- B/S・P/L・C/Fの三表のつながり
- 黒字倒産の仕組みとキャッシュの重要性

### [6. 決算処理と会計サイクル](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-06.html)
日々の取引記帳から決算書作成までの一連の流れを学びます。決算整理仕訳の主要項目（減価償却、引当金、経過勘定）と、会計ソフトを使った実務のイメージを理解します。

**学習内容:**
- 会計サイクル（日次→月次→四半期→年次決算）
- 決算の流れと精算表の作成
- 決算整理仕訳の主要項目
  - 減価償却（定額法・定率法）
  - 貸倒引当金の設定
  - 前払費用・前受収益の処理
  - 未払費用・未収収益の計上
  - 棚卸資産の評価（先入先出法、移動平均法）
- 試算表の作成と検証
- 税務申告との関係（会計上の利益と課税所得の違い）
- クラウド会計ソフトの活用（freee、マネーフォワード、弥生）

### [7. 財務分析の基本（収益性・安全性）](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-07.html)
財務諸表を使った経営分析の基本指標を学びます。収益性分析（どれだけ効率的に利益を上げているか）と安全性分析（財務の健全性）の主要指標を理解し、計算・分析できるようになります。

**学習内容:**
- 財務分析の目的と4つの視点
- 収益性分析の指標
  - 売上高総利益率（粗利率）
  - 売上高営業利益率
  - 売上高経常利益率
  - ROE（自己資本利益率）
  - ROA（総資産利益率）
  - ROEの分解（デュポン分析）
- 安全性分析の指標
  - 流動比率（短期的な支払能力）
  - 当座比率（より厳格な指標）
  - 自己資本比率（財務の健全性）
  - 負債比率、固定比率、固定長期適合率
  - インタレスト・カバレッジ・レシオ
- 業界平均値との比較方法

### [8. 財務分析の応用（効率性・成長性）](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-08.html)
効率性分析（資産の活用度）と成長性分析（企業の成長力）を学びます。複数の指標を組み合わせた総合分析と、同業他社との比較分析（ケーススタディ）を通じて実践力を養います。

**学習内容:**
- 効率性分析の指標
  - 総資産回転率
  - 売上債権回転率、回収期間
  - 棚卸資産回転率、在庫期間
  - 買入債務回転率
  - CCC（キャッシュ・コンバージョン・サイクル）
- 成長性分析の指標
  - 売上高成長率
  - 経常利益成長率
  - 総資産成長率
  - 従業員一人当たり売上高
- 総合分析とレーダーチャート
- ケーススタディ：同業2社の比較分析
- 有価証券報告書の読み方（セグメント情報、リスク情報）

### [9. 管理会計の基礎（原価計算と予算管理）](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-09.html)
経営の意思決定に活用する管理会計の基礎を学びます。原価計算の基本（製造原価、変動費・固定費）と予算管理（予算策定、予実差異分析）について理解します。

**学習内容:**
- 管理会計と財務会計の違い
- 原価計算の基礎
  - 製造原価の3要素（材料費、労務費、経費）
  - 直接費と間接費の区分
  - 変動費と固定費の分類
  - 製造間接費の配賦
- 予算管理の基礎
  - 予算の種類（売上予算、経費予算、資本予算）
  - 予算策定プロセス（トップダウン vs ボトムアップ）
  - 月次予実管理と差異分析
  - 差異分析の手法（数量差異、価格差異）
- PDCAサイクルと予算統制
- 部門別損益計算と貢献利益

### [10. ビジネスでの会計知識活用](https://fcircle-biz.github.io/tech_docs/guide/business-domain-knowledge/finance-accounting/finance-accounting-learning-material-10.html)
学んだ会計知識を実務でどう活用するかを学びます。損益分岐点分析、投資判断、経営層への報告・プレゼンテーションのコツなど、ビジネスの現場で役立つスキルを習得します。

**学習内容:**
- 損益分岐点分析（BEP分析）
  - 損益分岐点売上高の計算
  - 損益分岐点比率と安全余裕率
  - グラフによる可視化
  - 値下げ・増産の意思決定への応用
- 投資判断の基礎
  - 回収期間法
  - 投資利益率（ROI）
  - NPV（正味現在価値）の概念
- 経営層への報告・プレゼンテーション
  - データの可視化テクニック
  - 効果的なグラフの選び方
  - KPI設定と追跡
  - ストーリーテリング：数字に意味を持たせる
- 経営者の視点で数字を見る
- 会計知識を活かしたキャリアアップ

## 学習の進め方

1. **理論と実践の両立**: 各章で学んだ概念を、実際の企業の決算書や自社のデータで確認します
2. **演習問題に取り組む**: 各章の練習問題やミニテストで理解度を確認しながら進めます
3. **実在企業の分析**: EDINETや決算短信を使って、実際の上場企業を分析します
4. **Excelで実践**: 財務諸表の作成や財務分析を、Excelで実際に計算・グラフ化します
5. **ディスカッション**: 学習仲間や同僚と財務データについて議論し、理解を深めます
6. **自部門への応用**: 学んだ知識を自部門の予実管理や報告資料作成に活かします

## 学習目標

このガイドを完了すると、以下のスキルを身につけることができます：

- 会計の基本概念（財務会計・管理会計の違い、発生主義など）を説明できる
- 基本的な仕訳を切ることができ、複式簿記の仕組みを理解できる
- 貸借対照表（B/S）を読み、企業の財政状態を把握できる
- 損益計算書（P/L）を読み、企業の収益力を評価できる
- キャッシュ・フロー計算書（C/F）から資金の流れを理解できる
- 決算処理の流れと決算整理仕訳の主要項目を理解できる
- 主要な財務指標（ROE、ROA、流動比率、自己資本比率など）を計算・分析できる
- 財務諸表を使って企業の経営状態を総合的に評価できる
- 原価計算の基本と変動費・固定費の分類ができる
- 予算管理と予実差異分析のプロセスを実践できる
- 損益分岐点分析を使った意思決定ができる
- 財務データを使った効果的なプレゼンテーションができる
- 経営層・上司とのコミュニケーションで財務的な視点を持てる

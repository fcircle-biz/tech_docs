第8章: 社外文書の作成

会社の顔として信頼される文書を作成しよう

この章で学ぶこと
  • 社外文書の種類と特徴を理解する
  • ビジネスレターの正式な書式を習得する
  • 頭語と結語、時候の挨拶の使い方を学ぶ
  • 状況に応じた社外文書（依頼状、お詫び状、礼状等）の作成方法
  • 見積書・請求書などの商取引文書の基本
  • 封筒の宛名書きと送付状の作成

8.1 社外文書とは

社外文書は、会社の公式な意思表示として取引先や顧客に送る文書です。社内文書と異なり、より形式的で丁寧な表現が求められます。

flowchart TD A[社外文書の特徴] --> B[会社代表としての文書] A --> C[正式なビジネスマナー必須] A --> D[法的効力を持つ場合あり] A --> E[企業イメージに直結] B --> B1[個人ではなく組織の意思] C --> C1[敬語・形式の厳守] D --> D1[契約書・覚書等] E --> E1[信頼関係構築の基礎]

社外文書の種類

分類 文書例 主な用途
通知・案内文 新製品案内、イベント通知 情報提供・お知らせ
依頼文 見積依頼、資料提供依頼 相手に行動を求める
お詫び状 納期遅延のお詫び、品質問題のお詫び 謝罪と今後の対応説明
督促状・催促状 支払督促、回答催促 未履行事項への対応要請
礼状 訪問お礼、取引開始のお礼 感謝の意を伝える
商取引文書 見積書、請求書、契約書 取引の正式記録

8.2 ビジネスレターの基本形式

正式な社外文書（ビジネスレター）には、決まった書式があります。以下の構成要素を順に記載します。

flowchart TD A[ビジネスレターの構成] --> B["①発信日付"] A --> C["②宛先"] A --> D["③発信者"] A --> E["④件名"] A --> F["⑤頭語と前文"] A --> G["⑥主文"] A --> H["⑦末文と結語"] A --> I["⑧別記・添付"] B --> B1[和暦または西暦] C --> C1[会社名・役職・氏名] D --> D1[自社情報] E --> E1[文書の主題] F --> F1[拝啓・時候の挨拶] G --> G1[本題の内容] H --> H1[締めの言葉・敬具] I --> I1[詳細項目の記載]

標準的なビジネスレターの例

令和7年11月1日 株式会社ABC商事 営業部長 山田太郎 様 株式会社XYZ 代表取締役 鈴木一郎 新製品発表会のご案内 拝啓 晩秋の候、貴社におかれましてはますますご清栄のこととお慶び申し上げます。 平素は格別のご高配を賜り、厚く御礼申し上げます。 さて、このたび弊社では新製品「ABC-2000」を開発いたしました。 つきましては、下記のとおり新製品発表会を開催いたしますので、 ご多用中誠に恐縮ではございますが、ぜひご来場賜りますよう ご案内申し上げます。 敬具 記 1. 日時 令和7年12月5日（木） 14:00～16:00 2. 場所 ホテル◯◯ 2階ホール （東京都千代田区〇〇1-2-3） 3. 内容 新製品「ABC-2000」発表 デモンストレーション 質疑応答 以上 ※ご出欠につきましては、11月20日までにご連絡くださいますよう お願い申し上げます。 【お問い合わせ先】 株式会社XYZ 企画部 佐藤 TEL: 03-1234-5678 E-mail: info@xyz-corp.co.jp

8.3 頭語と結語の使い方

ビジネスレターでは、文書の冒頭に頭語、文末に結語を使用します。頭語と結語は必ずセットで使用し、文書の格式を表します。

頭語 結語 用途
拝啓 敬具 一般的なビジネス文書（最も使用頻度が高い）
謹啓 謹白（謹言） より丁重な文書、重要な案件
拝復 敬具 返信の文書
前略 草々 時候の挨拶を省略する場合（急ぎの用件）
急啓（急白） 草々（不一） 緊急を要する文書
頭語と結語の注意点
  • 頭語と結語は必ずペアで使用する（拝啓には敬具、謹啓には謹白）
  • メールでは頭語と結語を省略することが一般的
  • 「前略」を使う場合は本当に急ぎの場合のみ
  • お詫び状では「謹啓・謹白」を使用するのが一般的

8.4 時候の挨拶

頭語の後には、時候の挨拶を入れます。季節に応じた挨拶で、相手への気遣いを表現します。

月 時候の挨拶 続く定型文
1月 新春の候、初春の候 貴社におかれましてはますますご清栄のこととお慶び申し上げます。
平素は格別のご高配を賜り、厚く御礼申し上げます。
2月 立春の候、余寒の候
3月 早春の候、春分の候
4月 陽春の候、桜花の候
5月 新緑の候、薫風の候
6月 初夏の候、梅雨の候
7月 盛夏の候、酷暑の候
8月 残暑の候、晩夏の候
9月 初秋の候、秋分の候
10月 秋冷の候、紅葉の候
11月 晩秋の候、向寒の候
12月 初冬の候、歳末の候

8.5 依頼文の書き方

取引先に何かを依頼する際の文書は、相手の立場を尊重し、丁寧にお願いする姿勢が重要です。

令和7年11月10日 株式会社DEF製作所 購買部長 田中次郎 様 株式会社XYZ 営業部長 佐藤花子 見積書ご提出のお願い 拝啓 晩秋の候、貴社におかれましてはますますご清栄のこととお慶び申し上げます。 平素は格別のご高配を賜り、厚く御礼申し上げます。 さて、このたび弊社では新規プロジェクトを立ち上げることとなり、 貴社製品の導入を検討しております。 つきましては、下記製品につきまして、見積書をご提出いただきたく お願い申し上げます。 ご多用中誠に恐縮ではございますが、何卒よろしくお願い申し上げます。 敬具 記 1. 製品名 ABC-2000シリーズ 2. 数量 50台 3. 納期希望 令和8年1月末 4. 見積書提出期限 令和7年11月30日 以上

依頼文のポイント

  • 理由を明確に: なぜ依頼するのか、背景を説明する
  • 具体的な内容: 何を、いつまでに、どのように依頼するか明示
  • 相手への配慮: 「ご多用中恐縮ですが」などの言葉を添える
  • 期限の明示: 回答や提出の期限を明確にする

8.6 お詫び状の書き方

トラブルや不手際が発生した際のお詫び状は、誠意を持って迅速に送ることが重要です。

令和7年11月5日 株式会社GHI商事 総務部長 高橋三郎 様 株式会社XYZ 代表取締役 鈴木一郎 納期遅延に関するお詫び 謹啓 晩秋の候、貴社におかれましてはますますご清栄のこととお慶び申し上げます。 平素は格別のご高配を賜り、厚く御礼申し上げます。 さて、このたびご注文いただきました製品（注文番号：XYZ-12345）につきまして、 当初お約束しておりました11月10日の納期に間に合わない事態となりました。 弊社の生産管理体制の不備により、貴社に多大なるご迷惑をおかけすることとなり、 誠に申し訳ございません。 つきましては、下記のとおりご報告申し上げますとともに、 深くお詫び申し上げます。 謹白 記 1. 遅延理由 部品供給元の製造トラブルにより、必要部品の調達が遅延したため 2. 新しい納期 令和7年11月20日 3. 今後の対応 ・部品調達先の複数化を実施 ・生産計画の見直しと在庫管理の強化 ・お客様への進捗報告体制の整備 4. お詫びの措置 納期遅延のお詫びとして、製品価格の5%を減額させていただきます 以上 今後このようなことがないよう、管理体制を強化してまいります。 何卒ご理解とご容赦を賜りますようお願い申し上げます。

お詫び状の必須要素

  1. 謝罪の言葉: 冒頭で明確に謝罪する
  2. 問題の説明: 何が起きたのか事実を正確に説明
  3. 原因の究明: なぜ起きたのか原因を明らかにする
  4. 今後の対応: 再発防止策を具体的に示す
  5. 補償措置: 可能であれば具体的な補償を提示

8.7 礼状（お礼状）の書き方

取引開始、訪問後、契約成立など、感謝を伝える礼状は、タイミングよく送ることで信頼関係が深まります。

令和7年11月3日 株式会社JKL物産 営業部長 伊藤四郎 様 株式会社XYZ 営業部長 佐藤花子 ご訪問のお礼 拝啓 晩秋の候、貴社におかれましてはますますご清栄のこととお慶び申し上げます。 さて、先日は突然のお願いにもかかわらず、 お時間を割いてご面談の機会を賜り、誠にありがとうございました。 貴社の経営方針や今後の事業展開について詳しくお聞かせいただき、 弊社としても大変参考になりました。 今後とも、貴社のお役に立てるサービスをご提供できるよう、 鋭意努力してまいります。 まずは略儀ながら書中をもってお礼申し上げます。 今後ともよろしくお願い申し上げます。 敬具

礼状のポイント

  • 迅速な送付: 当日または翌日には送る
  • 具体的な内容: 何に対するお礼か明確にする
  • 今後の関係構築: 将来への期待を示す
  • 簡潔に: 長々と書かず、要点をまとめる

8.8 見積書と請求書の基本

商取引において必須の文書である見積書と請求書は、正確性と明確性が特に重要です。

見積書の基本項目

項目 内容
タイトル 「御見積書」
見積番号 管理用の一意の番号
発行日 見積書作成日
有効期限 見積価格が有効な期間
宛先 顧客の会社名・担当者名
発行者情報 自社名・住所・連絡先
明細 品名・数量・単価・金額
合計金額 小計・消費税・合計
備考 納期・支払条件等

請求書の基本項目

請求書は見積書とほぼ同じ項目構成ですが、以下の点が異なります。

  • タイトル: 「御請求書」または「請求書」
  • 請求日: 請求書発行日
  • 支払期限: 「○月○日までにお支払いください」
  • 振込先情報: 銀行名・支店名・口座番号・口座名義
  • インボイス番号: 適格請求書発行事業者の場合は登録番号

8.9 封筒の宛名書きと送付状

重要な社外文書を郵送する際は、封筒の宛名書きと送付状にも正しいマナーがあります。

封筒の宛名書きのルール

宛先 敬称 記載例
個人宛 様 株式会社ABC商事
営業部長 山田太郎 様
部署宛 御中 株式会社ABC商事
営業部 御中
会社宛 御中 株式会社ABC商事 御中
部署内の個人 様 株式会社ABC商事
営業部 山田太郎 様
封筒宛名書きの注意点
  • 「御中」と「様」を同時に使用しない（例：営業部御中 山田様 ✗）
  • 役職名の後には「様」をつける（例：部長様 ✗ → 部長 山田太郎様 ○）
  • 「殿」は目上の人から目下の人に使う敬称（一般的には「様」を使用）
  • 縦書き封筒では右から左に、横書き封筒では上から下に記載

送付状（添え状）の例

令和7年11月10日 株式会社ABC商事 営業部長 山田太郎 様 株式会社XYZ 企画部 佐藤花子 TEL: 03-1234-5678 FAX: 03-1234-5679 資料送付のご案内 拝啓 晩秋の候、貴社におかれましてはますますご清栄のこととお慶び申し上げます。 平素は格別のご高配を賜り、厚く御礼申し上げます。 さて、先日ご依頼いただきました資料を下記のとおりお送りいたします。 ご査収のほど、よろしくお願い申し上げます。 まずは略儀ながら書中をもってご連絡申し上げます。 敬具 記 1. 製品カタログ 1部 2. 価格表 1部 3. 会社案内 1部 以上
実習 8-1: ビジネスレターの作成

次のシチュエーションで、正式なビジネスレターを作成してみましょう。

状況: 新しい取引先（DEF株式会社 購買部長 中村様）に、自社の新サービスを案内する文書を作成してください。

含めるべき要素
  • 適切な頭語と結語
  • 11月の時候の挨拶
  • サービスの概要説明
  • 別記（記書き）でサービスの詳細
  • 問い合わせ先情報
実習 8-2: お詫び状の作成

次のトラブルに対するお詫び状を作成してみましょう。

状況: 納品した製品に不具合があり、顧客（GHI株式会社 総務部長 木村様）から連絡があった。無償交換と点検を実施する予定。

含めるべき要素
  • 明確な謝罪の言葉
  • 不具合の内容と原因
  • 対応方法（無償交換・点検）
  • 再発防止策
理解度確認クイズ
  1. 「拝啓」に対応する結語は何ですか？
  2. お詫び状で一般的に使用される頭語と結語の組み合わせは何ですか？
  3. 11月の時候の挨拶を1つ挙げてください。
  4. 封筒の宛名で、部署宛に使用する敬称は何ですか？
  5. 見積書と請求書の主な違いを2つ挙げてください。
解答例
  1. 敬具
  2. 謹啓・謹白（または謹言）
  3. 晩秋の候、向寒の候、霜降の候など
  4. 御中
  5. ①見積書には有効期限があるが請求書には支払期限がある、②請求書には振込先情報が記載される
この章のまとめ
  • 社外文書は会社の公式な意思表示であり、形式とマナーが重要
  • ビジネスレターは決まった構成（日付・宛先・発信者・件名・本文・別記）に従う
  • 頭語と結語は必ずペアで使用し、文書の格式を表す
  • 時候の挨拶で季節感と相手への配慮を示す
  • 依頼文は理由を明確にし、相手への配慮を忘れない
  • お詫び状は謝罪・原因・対応・再発防止を明確に記載
  • 礼状は迅速に送り、具体的な感謝の内容を伝える
  • 見積書・請求書は正確性が最重要、必要項目を漏れなく記載
  • 封筒の宛名書きでは「御中」と「様」の使い分けに注意

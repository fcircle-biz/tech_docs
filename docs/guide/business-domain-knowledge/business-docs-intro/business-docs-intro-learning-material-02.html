敬語と言葉遣いの基本

ビジネスで使う敬語の基本（尊敬語・謙譲語・丁寧語）、使い分けのポイント、クッション言葉の活用を学びます。

この章で学ぶこと

  • 敬語の3種類（尊敬語・謙譲語・丁寧語）を理解する
  • 尊敬語と謙譲語の見分け方と正しい使い方を身につける
  • 間違いやすい敬語表現を知り、正しく使えるようになる
  • 社内向け・社外向けの敬語の使い分けを学ぶ
  • クッション言葉やビジネス定型表現を活用できるようになる

2.1 敬語の3種類 - 尊敬語・謙譲語・丁寧語

敬語には尊敬語、謙譲語、丁寧語の3種類があります。それぞれの役割を理解することで、正しく使い分けられるようになります。

flowchart TD A[敬語] --> B[尊敬語] A --> C[謙譲語] A --> D[丁寧語] B --> E["相手の行動を高める
（例：おっしゃる、いらっしゃる）"] C --> F["自分の行動をへりくだる
（例：申し上げる、伺う）"] D --> G["丁寧に表現する
（例：です、ます）"]

尊敬語

相手の行動や状態を高める表現

例：

  • • 言う → おっしゃる
  • • 行く → いらっしゃる
  • • 見る → ご覧になる

謙譲語

自分の行動をへりくだって表現

例：

  • • 言う → 申し上げる
  • • 行く → 伺う
  • • 見る → 拝見する

丁寧語

言葉を丁寧に表現する

例：

  • • だ → です
  • • する → します
  • • 食べる → 食べます

2.2 尊敬語と謙譲語の見分け方

尊敬語と謙譲語の使い分けは最初は難しく感じますが、「誰の行動か」を考えれば簡単です。

見分け方のポイント

✓ 尊敬語を使う場面

相手（上司、お客様など）の行動を表すとき
→ 相手を立てるので「尊敬語」

✓ 謙譲語を使う場面

自分（または自社）の行動を表すとき
→ 自分をへりくだるので「謙譲語」

基本形 尊敬語（相手の行動） 謙譲語（自分の行動）
言う おっしゃる 申し上げる
聞く お聞きになる 伺う、お聞きする
見る ご覧になる 拝見する
行く いらっしゃる 伺う、参る
知る ご存じ 存じ上げる
食べる 召し上がる いただく

2.3 間違いやすい敬語の正しい使い方

ビジネスでよく使われる敬語の中には、間違って使われているものがたくさんあります。代表的な間違いを知って、正しく使えるようになりましょう。

よくある間違い1: 二重敬語

❌ 間違い

「おっしゃられる」「ご覧になられる」

※「おっしゃる」+「られる」のように、敬語を重ねています

✅ 正しい

「おっしゃる」「ご覧になる」

よくある間違い2: 尊敬語と謙譲語の混同

❌ 間違い

「部長が申されました」（謙譲語を相手に使っている）

✅ 正しい

「部長がおっしゃいました」（尊敬語を使う）

よくある間違い3: ファミコン言葉（バイト敬語）

❌ 間違い

  • • 「〜になります」（例：こちらが資料になります）
  • • 「よろしかったでしょうか」
  • • 「〜のほう」（例：資料のほうをご覧ください）

✅ 正しい

  • • 「〜です」（例：こちらが資料です）
  • • 「よろしいでしょうか」
  • • 「資料をご覧ください」

注意

二重敬語や間違った敬語は、一見丁寧に見えますが、実は正しい敬語を知っている人からは違和感を持たれます。正しい形を覚えて使いましょう。

2.4 社内向け・社外向けの敬語の違い

社内の人と社外の人では、敬語の使い方が異なります。特に注意が必要なのは、社外の人に対して、自社の上司のことを話すときです。

身内の扱い方のルール

社外の人に対しては、自社の人（上司であっても）は「身内」として扱います。そのため、尊敬語ではなく謙譲語や普通の表現を使います。

社内での会話

「部長がおっしゃいました」
(上司を立てるため尊敬語)

社外への報告

「部長の田中が申しました」
(身内なので謙譲語)

場面 社内での表現 社外での表現
上司が言った 部長がおっしゃいました 部長の田中が申しました
上司が見た 部長がご覧になりました 部長の田中が確認しました
上司が来る 部長がいらっしゃいます 部長の田中が参ります

ポイント

社外の人に対しては、自社の社長であっても「社長の山田が申しました」のように謙譲語を使います。これは、お客様を一番立てるためのルールです。

2.5 クッション言葉で印象アップ

クッション言葉とは、依頼や断りなどをするときに、本題の前に添える言葉です。相手への配慮を示し、印象を柔らかくする効果があります。

依頼するとき

  • • 恐れ入りますが、資料をお送りいただけますか
  • • お手数ですが、ご確認をお願いいたします
  • • ご多忙のところ恐縮ですが、ご回答いただけますでしょうか
  • • 差し支えなければ、ご意見をお聞かせください

断るとき

  • • 申し訳ございませんが、今回は見送らせていただきます
  • • あいにくですが、その日は都合がつきません
  • • 残念ながら、ご期待に添えません
  • • せっかくですが、今回は辞退させていただきます

質問するとき

  • • 失礼ですが、お名前を教えていただけますか
  • • 差し支えなければ、理由をお聞かせいただけますか
  • • よろしければ、ご意見をお聞かせください
  • • お伺いしたいのですが、ご都合はいかがでしょうか

お願い・提案するとき

  • • もしよろしければ、こちらの案もご検討ください
  • • ご検討いただければ幸いですが、別の方法もございます
  • • 可能であれば、今週中にお返事をいただけますと助かります
  • • お時間があれば、一度お会いしてお話しできればと存じます

2.6 ビジネスでよく使う定型表現

ビジネスでは、場面ごとによく使われる定型表現があります。これらを覚えておくと、文書作成がスムーズになります。

場面 定型表現
始まりの挨拶（社内） お疲れ様です
始まりの挨拶（社外） いつもお世話になっております
お礼 ご対応いただきありがとうございます / お忙しい中ありがとうございました
お詫び ご迷惑をおかけして申し訳ございません / お手数をおかけいたしました
確認依頼 ご確認のほどよろしくお願いいたします
返信依頼 ご返信をお待ちしております
締めの言葉 引き続きよろしくお願いいたします / 今後ともよろしくお願いいたします

2.7 書き言葉と話し言葉の違い

ビジネス文書では、書き言葉を使います。普段の会話で使う話し言葉とは異なる表現を使うので注意しましょう。

話し言葉（避ける） 書き言葉（文書で使う）
でも、だけど しかし、しかしながら
だから したがって、そのため
ちょっと 少し、やや
たくさん 多数、多く
すごく 非常に、大変
やっぱり やはり
〜とか 〜など

注意

「〜みたいな」「〜って感じ」「〜っていうか」などの口語表現は、ビジネス文書では絶対に使わないようにしましょう。

実習 2-1: 敬語を正しく使ってみよう

以下の文章を、正しい敬語に直してみましょう。

問題1（社内メール）

「部長が会議資料を見られました。」

解答を見る

正解：「部長が会議資料をご覧になりました。」
※「見られる」は尊敬語として間違いではありませんが、「ご覧になる」の方がより丁寧です。

問題2（お客様へのメール）

「部長の山田がおっしゃっていました。」

解答を見る

正解：「部長の山田が申しておりました。」
※社外の人に対しては、自社の人は身内扱いになるため、謙譲語を使います。

問題3（依頼メール）

「資料を送ってください。」

解答を見る

正解例：「恐れ入りますが、資料をお送りいただけますでしょうか。」
※クッション言葉を添えて、丁寧に依頼します。

理解度確認クイズ

  1. Q1. 敬語の3種類は何ですか？

    回答を表示

    尊敬語、謙譲語、丁寧語の3種類です。

  2. Q2. 社外の人に自社の上司のことを伝えるときは、どんな敬語を使いますか？

    回答を表示

    謙譲語を使います。社外の人に対しては、自社の人（上司であっても）は「身内」として扱うため、尊敬語ではなく謙譲語や普通の表現を使います。

  3. Q3. 「おっしゃられる」という表現が間違っている理由は何ですか？

    回答を表示

    二重敬語だからです。「おっしゃる」自体が既に尊敬語なので、さらに「られる」を付けると敬語を重ねることになります。正しくは「おっしゃる」です。

この章のまとめ

  • 敬語には尊敬語（相手の行動を高める）、謙譲語（自分の行動をへりくだる）、丁寧語（丁寧に表現する）の3種類がある
  • 尊敬語と謙譲語は「誰の行動か」で見分ける。相手の行動なら尊敬語、自分の行動なら謙譲語
  • 二重敬語は避ける。「おっしゃられる」ではなく「おっしゃる」が正しい
  • 社外の人に対しては、自社の上司も身内扱いにして謙譲語を使う
  • クッション言葉（恐れ入りますが、お手数ですが等）で印象を柔らかくする
  • ビジネス文書では書き言葉を使う。「でも」→「しかし」「だから」→「したがって」

7.1 ユーザーフォームの基本

ユーザーフォームを使用することで、使いやすいGUIを作成できます。

実習 7-1: 基本的なユーザーフォーム

以下の手順でユーザーフォームを作成します：

  1. VBEで「挿入」→「ユーザーフォーム」を選択
  2. ツールボックスからコントロールを配置
  3. プロパティウィンドウで設定を変更
フォームのコード例：
Private Sub CommandButton1_Click()
    MsgBox "ボタンがクリックされました！"
End Sub

Private Sub UserForm_Initialize()
    Me.Caption = "データ入力フォーム"
    Me.Width = 300
    Me.Height = 200
End Sub

7.2 主要なコントロール

  • Label: テキストの表示
  • TextBox: テキストの入力
  • CommandButton: ボタン
  • ListBox: リスト選択
  • ComboBox: ドロップダウンリスト
  • CheckBox: チェックボックス
  • OptionButton: ラジオボタン
実習 7-2: 顧客情報入力フォーム

顧客情報を入力するフォームを作成します：

フォームのコード：
Private Sub CommandButton1_Click()
    ' 登録ボタンの処理
    Dim ws As Worksheet
    Dim 最終行 As Long
    
    Set ws = ThisWorkbook.Worksheets("顧客データ")
    最終行 = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row + 1
    
    ' データの検証
    If TextBox1.Value = "" Then
        MsgBox "顧客名を入力してください", vbExclamation
        TextBox1.SetFocus
        Exit Sub
    End If
    
    ' データの登録
    ws.Cells(最終行, 1).Value = TextBox1.Value  ' 顧客名
    ws.Cells(最終行, 2).Value = TextBox2.Value  ' 電話番号
    ws.Cells(最終行, 3).Value = ComboBox1.Value ' 地域
    ws.Cells(最終行, 4).Value = CheckBox1.Value ' VIP顧客
    
    MsgBox "顧客情報を登録しました", vbInformation
    
    ' フォームのクリア
    Call フォームクリア
End Sub

Private Sub CommandButton2_Click()
    ' キャンセルボタンの処理
    Unload Me
End Sub

Private Sub UserForm_Initialize()
    ' フォームの初期化
    Me.Caption = "顧客情報入力"
    
    ' コンボボックスの設定
    ComboBox1.AddItem "北海道"
    ComboBox1.AddItem "東北"
    ComboBox1.AddItem "関東"
    ComboBox1.AddItem "中部"
    ComboBox1.AddItem "関西"
    ComboBox1.AddItem "中国"
    ComboBox1.AddItem "四国"
    ComboBox1.AddItem "九州"
End Sub

Private Sub フォームクリア()
    TextBox1.Value = ""
    TextBox2.Value = ""
    ComboBox1.Value = ""
    CheckBox1.Value = False
    TextBox1.SetFocus
End Sub

7.3 イベント処理

ユーザーの操作に応じて適切な処理を実行します。

実習 7-3: 動的なフォーム操作
Private Sub ComboBox1_Change()
    ' 地域選択時の処理
    Select Case ComboBox1.Value
        Case "関東"
            Label4.Caption = "都市部割引適用"
            Label4.ForeColor = RGB(0, 128, 0)
        Case "北海道", "九州"
            Label4.Caption = "遠隔地送料加算"
            Label4.ForeColor = RGB(255, 0, 0)
        Case Else
            Label4.Caption = "標準料金"
            Label4.ForeColor = RGB(0, 0, 0)
    End Select
End Sub

Private Sub TextBox1_Enter()
    ' テキストボックスにフォーカスが当たった時
    TextBox1.BackColor = RGB(255, 255, 200)
End Sub

Private Sub TextBox1_Exit(ByVal Cancel As MSForms.ReturnBoolean)
    ' テキストボックスからフォーカスが外れた時
    TextBox1.BackColor = RGB(255, 255, 255)
    
    ' 入力チェック
    If Len(TextBox1.Value) < 2 Then
        MsgBox "顧客名は2文字以上で入力してください", vbExclamation
        Cancel = True
    End If
End Sub

7.4 フォームとワークシートの連携

フォームで入力されたデータをワークシートに保存・読み込みします。

実習 7-4: データ検索・編集フォーム
Private Sub CommandButton3_Click()
    ' 検索ボタンの処理
    Dim ws As Worksheet
    Dim 検索範囲 As Range
    Dim 見つかったセル As Range
    
    Set ws = ThisWorkbook.Worksheets("顧客データ")
    Set 検索範囲 = ws.Range("A:A")
    
    Set 見つかったセル = 検索範囲.Find(TextBox3.Value, LookIn:=xlValues)
    
    If Not 見つかったセル Is Nothing Then
        ' データを表示
        TextBox1.Value = 見つかったセル.Value
        TextBox2.Value = 見つかったセル.Offset(0, 1).Value
        ComboBox1.Value = 見つかったセル.Offset(0, 2).Value
        CheckBox1.Value = 見つかったセル.Offset(0, 3).Value
        
        ' 編集モードに変更
        CommandButton1.Caption = "更新"
        CommandButton1.Tag = 見つかったセル.Row
    Else
        MsgBox "該当する顧客が見つかりませんでした", vbInformation
    End If
End Sub

Private Sub CommandButton1_Click()
    ' 登録/更新ボタンの処理
    If CommandButton1.Caption = "更新" Then
        ' 更新処理
        Dim ws As Worksheet
        Dim 行番号 As Long
        
        Set ws = ThisWorkbook.Worksheets("顧客データ")
        行番号 = CLng(CommandButton1.Tag)
        
        ws.Cells(行番号, 1).Value = TextBox1.Value
        ws.Cells(行番号, 2).Value = TextBox2.Value
        ws.Cells(行番号, 3).Value = ComboBox1.Value
        ws.Cells(行番号, 4).Value = CheckBox1.Value
        
        MsgBox "顧客情報を更新しました", vbInformation
        
        ' モードをリセット
        CommandButton1.Caption = "登録"
        CommandButton1.Tag = ""
    Else
        ' 新規登録処理（前の実習と同じ）
        ' ...
    End If
End Sub

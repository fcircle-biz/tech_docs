第1章: 基本操作と表示設定

Excelの基本操作をマスターしよう

1.1 Microsoft Excelとは

Microsoft Excelは、世界中で広く使用されているスプレッドシートソフトウェアです。データの入力、計算、分析、グラフ作成など、ビジネスから個人利用まで幅広く活用されています。

Excelの主な特徴
  • 表計算機能：数値の計算や関数を使用した複雑な計算が可能
  • データ管理：大量のデータを効率的に整理・管理
  • グラフ・チャート：データを視覚的に分かりやすく表示
  • データベース機能：フィルタや並び替えによるデータ抽出
  • 自動化：VBAマクロによる作業の自動化
Excelでの作業フロー
1. データ入力 → 2. 計算処理 → 3. 分析・集計 →
6. 意思決定 ← 5. レポート作成 ← 4. グラフ化

1.2 Excelの起動と画面構成

Excelを効率的に使用するためには、まず画面構成を理解することが重要です。各部分の名称と役割を覚えましょう。

Excelの起動方法
  • スタートメニューから：スタート → Microsoft Excel
  • ショートカットから：デスクトップのExcelアイコンをダブルクリック
  • ファイルから：.xlsx、.xlsファイルをダブルクリック
  • 検索から：Windowsキー + S → 「Excel」と入力
画面構成の理解
メイン画面構成要素
画面要素 説明
リボン 操作メニュー（ホーム、挿入、ページレイアウトなど）
名前ボックス 現在選択中のセルの場所を表示
数式バー セルに入力されている数式や値を表示
ワークシート データを入力するセル領域
シートタブ 複数のワークシートを切り替え
ステータスバー 現在の状態や計算結果を表示
主要リボンタブ
タブ名 主な機能
ホーム 書式設定、コピー・貼り付け
挿入 グラフ、図形、関数の挿入
ページレイアウト 印刷設定、テーマ設定
ワークシート要素
要素 説明
セル A1、B2などの個別データ領域
行番号 1、2、3...（縦方向の識別子）
列番号 A、B、C...（横方向の識別子）
実習 1-1: Excelの起動と画面確認

実際にExcelを起動して、画面構成を確認してみましょう。

手順
  1. スタートメニューまたはデスクトップからExcelを起動します
  2. 「新しいブック」を選択して空白のワークブックを開きます
  3. 以下の要素を画面上で確認してください：
    • リボン（画面上部のメニュー）
    • 名前ボックス（左上のA1と表示されている部分）
    • 数式バー（名前ボックスの右側の入力欄）
    • セル領域（格子状の表示領域）
    • シートタブ（画面下部のSheet1表示）
  4. セルA1をクリックして選択状態にします
  5. 異なるセル（例：C3）をクリックして、名前ボックスの表示が変わることを確認します
確認ポイント

セルを選択すると名前ボックスにセル番地（A1、C3など）が表示され、そのセルが青い枠で囲まれることを確認してください。これがExcelの基本的な操作です。

1.3 ファイル操作の基本

Excelでの作業を始める前に、ファイルの作成、保存、開く操作を確実に覚えましょう。適切なファイル管理は、効率的な作業の基礎となります。

新しいファイルの作成
  • 方法1：Ctrl + N（最も高速）
  • 方法2：ファイルタブ → 新規 → 空白のブック
  • 方法3：Excel起動時のスタート画面で「新しいブック」を選択
ファイルの保存
重要：保存のタイミング

データの紛失を防ぐため、作業の節目では必ず保存しましょう。Ctrl + S を定期的に押す習慣をつけることが大切です。

  • 初回保存：Ctrl + S または ファイルタブ → 名前を付けて保存
  • 上書き保存：Ctrl + S（最も頻繁に使用）
  • 別名で保存：F12キー または ファイルタブ → 名前を付けて保存
ファイル形式の理解
主要なファイル形式
  • .xlsx：Excel 2007以降の標準形式（推奨）
  • .xls：Excel 2003以前の形式
  • .xlsm：マクロ有効ブック
  • .csv：カンマ区切りファイル（他ソフトとの互換性）
実習 1-2: ファイルの作成と保存

実際にファイルを作成し、適切な場所に保存してみましょう。

手順
  1. Ctrl + N で新しいワークブックを作成します
  2. セルA1に「Excel学習用ファイル」と入力します
  3. セルA2に今日の日付を入力します
  4. Ctrl + S を押して保存ダイアログを開きます
  5. デスクトップまたは適切なフォルダを選択します
  6. ファイル名を「Excel基礎練習_第1章」に設定します
  7. ファイルの種類が「Excelブック (*.xlsx)」になっていることを確認します
  8. 「保存」ボタンをクリックします
  9. タイトルバーにファイル名が表示されることを確認します
期待される結果

画面上部のタイトルバーに「Excel基礎練習_第1章.xlsx - Excel」と表示され、ファイルが正常に保存されたことが確認できます。

1.4 コピー＆ペーストの各種方法

Excelでは、データをコピー＆ペーストする際に、値だけ、書式だけ、数式だけなど、様々な方法があります。目的に応じて適切な方法を選択することで、効率的にデータを操作できます。

基本的なコピー＆ペースト操作
  • コピー：Ctrl + C
  • 切り取り：Ctrl + X
  • 貼り付け：Ctrl + V
  • 形式を選択して貼り付け：Ctrl + Alt + V
形式を選択して貼り付けの種類
貼り付け形式 説明 使用場面
すべて 数式 + 書式 + コメント等すべて 完全にコピーしたい場合
数式 計算式のみ（結果ではなく式自体） 計算ロジックだけを移したい場合
値 計算結果の値のみ（式は貼り付けない） 計算結果だけを固定したい場合
書式 文字色、背景色、罫線等の見た目のみ デザインだけを統一したい場合
コメント セルに付けられたコメントのみ 注釈情報だけを移したい場合
入力規則 データ入力の制限ルールのみ 入力制限を他のセルにも適用したい場合
実習 1-3: 各種コピー＆ペーストの練習

異なるコピー＆ペースト方法を体験して、それぞれの違いを理解しましょう。

手順
  1. セルA1に「100」と入力します
  2. セルA2に「200」と入力します
  3. セルA3に数式「=A1+A2」を入力します（結果は300になります）
  4. セルA3を赤い背景色に設定します（ホームタブ → フォントの背景色）
  5. セルA3を選択してCtrl + C でコピーします
  6. セルC1を選択してCtrl + V で通常の貼り付けを行います
  7. セルC2を選択して、Ctrl + Alt + V を押します
  8. 「値」を選択してOKを押します
  9. セルC3を選択して、再度Ctrl + Alt + V を押します
  10. 「書式」を選択してOKを押します
結果の比較
  • C1：数式「=A1+A2」と赤い背景がコピーされる
  • C2：値「300」のみがコピーされ、数式や書式はなし
  • C3：赤い背景のみがコピーされ、値や数式はなし

1.5 行・列の幅調整と罫線設定

データを見やすく表示するためには、行の高さや列の幅を適切に調整し、罫線で表をきれいに整える必要があります。これらの操作を覚えることで、プロフェッショナルな表を作成できます。

列幅の調整方法
  • 手動調整：列番号の境界をドラッグ
  • 自動調整：列番号の境界をダブルクリック
  • 数値指定：列を右クリック → 列の幅
  • 複数列同時：複数列を選択してから調整
行の高さの調整方法
  • 手動調整：行番号の境界をドラッグ
  • 自動調整：行番号の境界をダブルクリック
  • 数値指定：行を右クリック → 行の高さ
罫線の基本設定
罫線設定の手順
  1. 罫線を設定したい範囲を選択
  2. ホームタブ → 罫線ボタン（格子状のアイコン）をクリック
  3. 目的に応じて罫線の種類を選択
実習 1-4: 表の体裁を整える

簡単な売上表を作成して、行列の調整と罫線設定を実践しましょう。

手順
  1. 以下のデータを入力します：
    • A1: 商品名, B1: 単価, C1: 数量, D1: 金額
    • A2: りんご, B2: 150, C2: 10, D2: =B2*C2
    • A3: みかん, B3: 100, C3: 15, D3: =B3*C3
    • A4: ばなな, B4: 200, C4: 8, D4: =B4*C4
  2. A列の列幅を広げて商品名が見やすくなるよう調整します
  3. B列からD列は数値が見やすい幅に調整します
  4. 範囲A1:D4を選択します
  5. ホームタブ → 罫線 → 「格子」を選択します
  6. A1:D1（見出し行）を選択して太字にします
  7. 見出し行の背景色を薄い青色に設定します
期待される結果

きれいに罫線が引かれ、見出しが目立つ見やすい表が完成します。列幅も内容に応じて適切に調整されているはずです。

1.6 表示設定と画面表示の最適化

Excelでは、作業効率を高めるために様々な表示設定を変更できます。表示倍率の調整、目盛線の表示/非表示、ページビューの切り替えなどを覚えましょう。

表示倍率の変更
  • ズームスライダー：画面右下のスライダーを操作
  • 表示タブ：表示タブ → ズーム → 任意の倍率を選択
  • ショートカット：Ctrl + マウスホイール
  • 選択範囲に合わせる：範囲選択後、表示タブ → 選択範囲に合わせる
画面表示の切り替え
表示モード 説明 用途 ショートカット
標準 通常の作業画面 データ入力・編集作業 表示タブ → 標準
ページレイアウト 印刷時のレイアウト確認 印刷プレビューと調整 表示タブ → ページレイアウト
改ページプレビュー 改ページ位置の確認・調整 印刷ページの境界設定 表示タブ → 改ページプレビュー
目盛線とヘッダーの表示制御
  • 目盛線：表示タブ → 目盛線のチェックを外す
  • 見出し：表示タブ → 見出しのチェックを外す
  • 数式バー：表示タブ → 数式バーのチェックを外す
実習 1-5: 表示設定のカスタマイズ

様々な表示設定を変更して、自分の作業スタイルに合った画面設定を見つけましょう。

手順
  1. 現在のズーム倍率を確認します（画面右下）
  2. ズームスライダーを使って150%に拡大してみます
  3. Ctrl + マウスホイールで倍率を変更してみます
  4. 表示タブをクリックします
  5. 「ページレイアウト」をクリックして表示を変更します
  6. 「標準」に戻します
  7. 「目盛線」のチェックを外して、格子線を非表示にします
  8. 再度「目盛線」をチェックして、格子線を表示します
  9. 「見出し」のチェックを外して、行番号・列番号を非表示にします
  10. 「見出し」を再度チェックして元に戻します
習得すべきポイント
  • 作業内容に応じて適切な表示倍率を選択できる
  • 印刷レイアウトを確認したい場合はページレイアウト表示が有効
  • 目盛線を非表示にすることで、よりすっきりした表示が可能
理解度確認クイズ
  1. Excelでファイルを上書き保存する最も簡単な方法は？
    a) ファイルタブ→保存 b) Ctrl + S c) F12キー d) Alt + S
  2. 「値のみ」を貼り付けしたい場合のショートカットキーは？
    a) Ctrl + V b) Ctrl + C c) Ctrl + Alt + V d) Ctrl + Shift + V
  3. 列幅を自動調整（内容に合わせて最適な幅にする）するには？
    a) 列番号をクリック b) 列番号の境界をクリック c) 列番号の境界をダブルクリック d) 列番号を右クリック
  4. Excel 2016以降の標準ファイル形式は？
    a) .xls b) .xlsx c) .xlsm d) .csv
  5. Excelの画面で、選択しているセルの番地を表示するのは？
    a) 数式バー b) ステータスバー c) 名前ボックス d) リボン
解答:
1. b) Ctrl + S
2. c) Ctrl + Alt + V（その後「値」を選択）
3. c) 列番号の境界をダブルクリック
4. b) .xlsx
5. c) 名前ボックス

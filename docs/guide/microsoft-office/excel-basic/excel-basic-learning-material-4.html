4.1 ページレイアウトビューの活用

Excelでは、画面上でのデータ作業と実際の印刷結果が異なる場合があります。ページレイアウトビューを使用することで、印刷時の見た目を確認しながら作業できます。

表示モードの種類と特徴
表示モード 特徴1 特徴2 特徴3 適用場面
標準 通常の編集作業に最適 画面表示に最適化 印刷イメージは確認不可 データ入力・編集メイン作業
ページレイアウト 印刷時の見た目で表示 ヘッダー・フッターが見える 用紙サイズに合わせた表示 印刷前の最終レイアウト確認
改ページプレビュー 改ページ位置を確認・調整 ページ境界が明確に表示 大量データの印刷計画に最適 複数ページにわたる資料の調整
ページレイアウトビューのメリット
なぜページレイアウトビューを使うのか？
  • WYSIWYG（What You See Is What You Get）：印刷結果を正確に確認
  • レイアウト調整：余白やヘッダー・フッターを視覚的に編集
  • 用紙イメージ：実際の用紙サイズでの表示
  • 印刷前確認：印刷プレビューを見る前の事前チェック
実習 4-1: ページレイアウトビューの基本操作

表示モードを切り替えて、それぞれの特徴を体験しましょう。

手順
  1. サンプルデータを準備：
    • A1:F20程度の売上データ表を作成
    • 見出し行には「日付」「商品名」「単価」「数量」「金額」「備考」
  2. 表示モードの切り替えを体験：
    • 現在の標準ビューを確認
    • 表示タブ → ページレイアウト をクリック
    • 画面の変化を観察（用紙の境界、ルーラーの表示など）
  3. 各表示モードでの作業性を確認：
    • ページレイアウトビューでデータを編集
    • 改ページプレビューに切り替え
    • 標準ビューに戻る
  4. ページレイアウトビューでの特別機能：
    • ヘッダー部分（「ヘッダーを追加するにはクリックしてください」）をクリック
    • 「売上実績表」と入力して他の場所をクリック
    • ヘッダーが表示されることを確認
学習ポイント

ページレイアウトビューでは、印刷時と同じ見た目で作業できるため、レイアウト調整が直感的に行えます。

4.2 用紙サイズと印刷の向きの設定

適切な用紙設定は、印刷レイアウトの基礎となります。データの内容に応じて最適な用紙サイズと向きを選択しましょう。

主な用紙サイズ
  • A4：最も標準的（210mm × 297mm）
  • A3：A4の2倍の面積（297mm × 420mm）
  • Letter：北米標準（216mm × 279mm）
  • B4：日本独自サイズ（257mm × 364mm）
  • Legal：法務文書用（216mm × 356mm）
印刷の向きの選択指針
縦向き（Portrait）と横向き（Landscape）の使い分け
  • 縦向き：
    • 文書、レポート、縦に長いリスト
    • 列数が少ない（A-F列程度まで）
    • 行数が多いデータ
  • 横向き：
    • 表、スプレッドシート、グラフ
    • 列数が多い（G列以降まである）
    • 横長のレイアウトが必要
用紙設定の変更方法
  1. ページレイアウトタブ → サイズ → 希望する用紙サイズを選択
  2. ページレイアウトタブ → 印刷の向き → 縦向きまたは横向きを選択
  3. ファイルタブ → 印刷 → 設定から詳細設定も可能
実習 4-2: 用紙設定の最適化

データの内容に応じて適切な用紙設定を行いましょう。

手順
  1. 幅の広いデータ表を作成：
    • A1:J15程度のデータ（10列×15行）
    • 月別売上データなど、横に広がるデータを想定
  2. 現在の印刷イメージを確認：
    • Ctrl + P で印刷プレビューを表示
    • 右端の列が切れていることを確認
    • Esc で印刷プレビューを閉じる
  3. 用紙の向きを変更：
    • ページレイアウトタブ → 印刷の向き → 横向き
    • 再度 Ctrl + P で印刷プレビューを確認
    • 全ての列が収まることを確認
  4. 用紙サイズの変更も試行：
    • ページレイアウトタブ → サイズ → A3
    • より余裕のあるレイアウトになることを確認
    • A4に戻す
  5. 最適な設定を決定：
    • データの内容と印刷目的を考慮
    • 最も見やすく、実用的な設定を選択
判断基準

用紙設定は、データの量・レイアウト・印刷コスト・配布先のことを総合的に考慮して決定しましょう。

4.3 余白の調整とセンタリング設定

適切な余白設定は、印刷物の美観と実用性を大きく左右します。余白の調整とページ中央への配置方法を学びましょう。

余白設定の種類
  • 標準：上下 1.9cm、左右 1.7cm
  • 広い：上下 2.5cm、左右 2.5cm
  • 狭い：上下 1.3cm、左右 1.3cm
  • ユーザー設定：各辺を個別に指定
センタリング（中央揃え）オプション
センタリング方向 配置効果 適用データ 使用場面
水平方向 左右の中央に配置 横長のデータに有効 表やグラフを用紙の真ん中に配置
垂直方向 上下の中央に配置 縦長のデータに有効 リストや縦向きのレポート
両方向 用紙の中央に配置 小さなデータに最適 タイトルや小規模な集計表
余白調整のアクセス方法
  • 方法1：ページレイアウトタブ → 余白 → 余白オプションを選択
  • 方法2：ファイルタブ → 印刷 → 設定 → ページ設定 → 余白タブ
  • 方法3：ページレイアウトビューでルーラーを直接ドラッグ
余白設定時の注意点
  • プリンタの制限：各プリンタには最小余白の制限がある
  • バインダー穴：資料をファイリングする場合は左余白を広めに
  • 読みやすさ：余白が狭すぎると読みにくくなる
  • コスト考慮：余白を狭くしすぎると印刷品質が下がる場合がある
実習 4-3: 余白とセンタリングの調整

小さなデータ表を美しく中央配置で印刷するための設定を行いましょう。

手順
  1. 小さなサンプルデータを作成：
    • A1:D8程度のコンパクトな表
    • 例：月別来客数集計表など
  2. 現在の印刷イメージを確認：
    • Ctrl + P で印刷プレビューを表示
    • データが左上に偏って配置されていることを確認
  3. センタリング設定を適用：
    • 印刷プレビューの「設定」→「ページ設定」をクリック
    • 「余白」タブを選択
    • 「ページ中央」で「水平」と「垂直」両方にチェック
    • OK をクリック
  4. 結果を確認：
    • データが用紙の中央に配置されていることを確認
    • 見栄えの改善を実感
  5. 余白の微調整も試行：
    • 「余白」→「ユーザー設定の余白」
    • 左余白を3cmに設定（ファイリング用）
    • 変化を確認
活用場面

小さなデータや単独の表を印刷する際は、センタリング設定により、プロフェッショナルな見た目にできます。

4.4 ヘッダー・フッターによる文書の体裁向上

ヘッダー・フッターは、文書のタイトル、日付、ページ番号などの重要情報を全ページに表示するために使用します。ビジネス文書では必須の機能です。

ヘッダー・フッターで表示できる情報
よく使用される要素
  • 文書タイトル：レポート名、表のタイトル
  • 作成者情報：部署名、作成者名
  • 日付情報：作成日、印刷日、更新日
  • ページ情報：ページ番号、総ページ数
  • ファイル情報：ファイル名、ファイルパス
  • カスタム要素：会社ロゴ、署名画像
ヘッダー・フッターの構成
セクション位置 配置 代表的な内容1 代表的な内容2 代表的な内容3
左セクション 左寄せ表示 作成者名 部署名 カスタムテキスト
中央セクション 中央寄せ表示 文書タイトル 会社名 画像・ロゴ
右セクション 右寄せ表示 日付 ページ番号 時刻
セクション配置の使い分け
  • 左セクション: 作成者や所属情報（読み手が最初に見る部分）
  • 中央セクション: 文書の主題やタイトル（最も目立つ位置）
  • 右セクション: 日付やページ番号（参照情報）
ヘッダー・フッターの設定方法
  1. ページレイアウトビューから：ヘッダー・フッター領域を直接クリック
  2. 挿入タブから：挿入タブ → ヘッダーとフッター
  3. ページ設定から：ページレイアウトタブ → ページ設定グループの矢印 → ヘッダー/フッタータブ
実習 4-4: プロフェッショナルなヘッダー・フッターの作成

ビジネス文書らしいヘッダー・フッターを設定しましょう。

手順
  1. ページレイアウトビューに切り替え
  2. ヘッダーの設定：
    • 上部の「ヘッダーを追加するにはクリックしてください」をクリック
    • ヘッダーツールの「デザイン」タブが表示されることを確認
    • 左セクションに「営業部」と入力
    • 中央セクションをクリックして「月次売上実績表」と入力
    • 右セクションをクリックして「ヘッダーとフッターツール」→「現在の日付」をクリック
  3. フッターの設定：
    • 下部の「フッターを追加するにはクリックしてください」をクリック
    • 左セクションに「&[ファイル名]」と手動入力（ファイル名を自動表示）
    • 中央セクションに「機密文書」と入力
    • 右セクションに「ヘッダーとフッターツール」→「ページ番号」をクリック
  4. 書式設定の調整：
    • ヘッダーの文字をすべて選択してフォントサイズを12ptに変更
    • 中央セクションの文字を太字に設定
  5. 結果の確認：
    • 他のセルをクリックしてヘッダー・フッター編集を終了
    • 印刷プレビューで全体の見た目を確認
カスタム要素の活用

&[要素名]の形式で様々な自動要素を挿入できます：&[日付]、&[時刻]、&[ページ数]、&[総ページ数]など

4.5 改ページの挿入・削除と印刷範囲の設定

大量のデータを適切にページ分けし、必要な部分のみを印刷するための技術を学びましょう。

改ページの種類
  • 自動改ページ：Excelが自動的に挿入（薄い破線で表示）
  • 手動改ページ：ユーザーが手動で挿入（太い実線で表示）
改ページの挿入・削除方法
操作種別 アクセス方法 操作手順 効果
改ページ挿入 ページレイアウトタブ → 区切り 改ページを挿入したい位置のセルを選択 → 改ページの挿入 指定位置で強制的にページを分割
改ページ削除 ページレイアウトタブ → 区切り 削除したい改ページの直後のセルを選択 → 改ページの削除 手動改ページを無効化
改ページプレビュー 表示タブ → 改ページプレビュー 表示モードを改ページプレビューに変更 ページ境界を視覚的に確認・調整
改ページ操作の注意点
  • 自動改ページ: 削除はできません（手動改ページのみ削除可能）
  • 挿入位置: セルを選択した位置の上側と左側に改ページが挿入されます
  • プレビューモード: 改ページプレビューではドラッグで改ページ位置を調整できます
印刷範囲の設定
印刷範囲設定の用途
  • 部分印刷：必要なデータ部分のみを印刷
  • 複数範囲：離れた複数の範囲を同時に印刷
  • 定期印刷：毎回同じ範囲を印刷する場合
  • コスト削減：不要な空白ページの印刷を回避
印刷範囲の設定手順
  1. 印刷したい範囲を選択
  2. ページレイアウトタブ → 印刷範囲 → 印刷範囲の設定
  3. 印刷プレビューで確認
  4. 解除する場合：ページレイアウトタブ → 印刷範囲 → 印刷範囲のクリア
実習 4-5: 改ページと印刷範囲の実践

大きなデータ表を適切にページ分けし、必要な部分のみを印刷設定しましょう。

手順
  1. 大きなサンプルデータを作成：
    • A1:F50程度の大きなデータ表
    • 月別売上データを50行程度作成
  2. 改ページプレビューで現状確認：
    • 表示タブ → 改ページプレビュー
    • ページ境界（青い線）を確認
    • データが何ページに分かれるか確認
  3. 手動改ページを挿入：
    • 標準ビューに戻る
    • 25行目のセルA25を選択
    • ページレイアウトタブ → 区切り → 改ページの挿入
    • 再度改ページプレビューで境界線の変化を確認
  4. 印刷範囲を設定：
    • 標準ビューで範囲A1:F25を選択
    • ページレイアウトタブ → 印刷範囲 → 印刷範囲の設定
    • Ctrl + P で印刷プレビューを確認
    • 選択した範囲のみが印刷対象になっていることを確認
  5. 複数範囲の印刷範囲設定：
    • 印刷範囲をクリア
    • A1:F10を選択 → Ctrlキーを押しながらA30:F40を追加選択
    • 印刷範囲の設定を実行
    • 印刷プレビューで2つの範囲が別々のページに表示されることを確認
応用ポイント

改ページプレビューでは、ページ境界線をドラッグして移動できます。視覚的に最適な改ページ位置を調整しましょう。

4.6 印刷タイトルと拡大縮小印刷の活用

複数ページにわたる大きな表では、各ページに見出し行・列を繰り返し印刷し、必要に応じて拡大縮小して最適なレイアウトを実現しましょう。

印刷タイトルの重要性
印刷タイトルが必要な理由
  • 理解しやすさ：各ページで列の意味が分かる
  • 独立性：各ページが単独で理解可能
  • プロフェッショナル：ビジネス文書として完成度が高い
  • 効率性：データの確認作業が迅速
拡大縮小印刷のオプション
  • 拡大/縮小：パーセンテージで指定（50%〜400%）
  • 次のページ数に合わせて印刷：指定したページ数に収める
  • 1ページに収める：すべてを1ページに強制収束
拡大縮小印刷の注意点
  • 可読性：縮小しすぎると文字が読めなくなる
  • バランス：レイアウトが崩れる場合がある
  • 印刷品質：プリンタの解像度限界を考慮
実習 4-6: 印刷タイトルと拡大縮小の完全活用

大きな表で印刷タイトルを設定し、適切な拡大縮小で最適な印刷レイアウトを実現しましょう。

手順
  1. 大きなデータ表を準備：
    • A1:H50の年間売上データ表
    • A1:H1に列見出し（日付、商品、単価、数量、金額、担当者、地域、備考）
  2. 印刷タイトルの設定：
    • ページレイアウトタブ → 印刷タイトル
    • 「タイトル行」で「$1:$1」を入力（1行目を各ページで繰り返し）
    • 「タイトル列」で「$A:$A」を入力（A列を各ページで繰り返し）
    • OK をクリック
  3. 印刷プレビューで確認：
    • Ctrl + P で印刷プレビュー表示
    • ページを切り替えて、各ページに見出し行が表示されることを確認
  4. 拡大縮小印刷の設定：
    • 印刷プレビューの設定で「ページ設定」を選択
    • 「ページ」タブ → 「次のページ数に合わせて印刷」を選択
    • 横1ページ、縦2ページに設定
    • OK をクリックして結果を確認
  5. 最適化の調整：
    • 文字が小さすぎる場合は縦ページ数を増やす
    • 余白を狭くして印刷領域を広げる
    • 最も読みやすい設定を見つける
  6. 設定の保存確認：
    • ファイルを保存
    • 設定が保持されることを確認
プロのコツ

印刷タイトルと拡大縮小を組み合わせることで、大量データでも各ページが独立して理解できる美しい印刷物を作成できます。

理解度確認クイズ
  1. 列数が多い表を印刷する場合、適切な印刷の向きは？
    a) 縦向き（Portrait） b) 横向き（Landscape） c) どちらでも同じ d) 自動調整
  2. 小さなデータ表を用紙の中央に配置するための設定は？
    a) 余白を大きくする b) フォントサイズを大きくする c) ページ中央の水平・垂直をチェック d) 印刷倍率を上げる
  3. ヘッダー・フッターの構成は何個のセクションに分かれている？
    a) 2個 b) 3個 c) 4個 d) 5個
  4. 手動で挿入した改ページの表示は？
    a) 薄い破線 b) 太い実線 c) 点線 d) 二重線
  5. 複数ページの表で各ページに見出し行を繰り返し印刷する機能は？
    a) ヘッダー b) フッター c) 印刷タイトル d) 印刷範囲
解答:
1. b) 横向き（Landscape）
2. c) ページ中央の水平・垂直をチェック

第4章のまとめ

この章では、プロフェッショナルな印刷レイアウトの実現方法について学習しました：

  • ページレイアウトビュー：印刷イメージを確認しながらの作業で精度向上
  • 用紙設定：データの性質に応じた用紙サイズと向きの適切な選択
  • 余白とセンタリング：美しいレイアウトと実用性を両立する設定
  • ヘッダー・フッター：文書の体裁向上と情報の自動表示
  • 改ページと印刷範囲：効率的なページ分けと部分印刷の活用
  • 印刷タイトルと拡大縮小：大量データでも読みやすい印刷レイアウト

これらの技術により、どのようなデータでもプロフェッショナルで読みやすい印刷物を作成できます。ペーパーレス化が進む中でも、重要な資料の印刷品質は業務の信頼性に直結します。

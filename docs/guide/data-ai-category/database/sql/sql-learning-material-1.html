第1章: データベースとSQLの基礎概念

データベースとSQLの基礎概念

この章で学ぶこと
  • リレーショナルデータベースの基本概念を理解する
  • データベース管理システム（DBMS）の役割を学ぶ
  • SQLの基本的な概念と分類を理解する
  • PostgreSQLのインストールと学習環境の構築
  • データベース操作の基本的なコマンドライン操作

1.1 リレーショナルデータベースとは

リレーショナルデータベースは、データをテーブルという形式で管理するデータベースモデルです。 テーブルは行（レコード）と列（カラム）で構成され、Excelの表と似ている構造です。

erDiagram CUSTOMERS { int customer_id PK string name string email date created_at } ORDERS { int order_id PK int customer_id FK decimal total_amount date order_date } CUSTOMERS ||--o{ ORDERS : places

主要な特徴

  • テーブル構造: データを行と列で整理
  • リレーション: 複数のテーブル間の関係を定義
  • 制約: データの整合性を保つルール
  • 正規化: データの重複を排除する設計手法

1.2 データベース管理システム（DBMS）

DBMS（Database Management System）は、データベースの作成、管理、操作を行うソフトウェアです。 主要なリレーショナルDBMS（RDBMS）には以下があります：

商用DBMS
  • Oracle Database
  • Microsoft SQL Server
  • IBM DB2
オープンソースDBMS
  • PostgreSQL（今回使用）
  • MySQL
  • MariaDB

1.3 SQLの基礎知識

SQL（Structured Query Language）は、リレーショナルデータベースを操作するための標準的な言語です。 SQLは以下の4つのカテゴリに分類されます：

DDL（Data Definition Language）

データベースの構造を定義

CREATE, ALTER, DROP
DML（Data Manipulation Language）

データの操作

SELECT, INSERT, UPDATE, DELETE
DCL（Data Control Language）

アクセス制御

GRANT, REVOKE
TCL（Transaction Control Language）

トランザクション制御

BEGIN, COMMIT, ROLLBACK

1.4 PostgreSQLの環境構築

このコースでは、PostgreSQLを使用してSQL学習を進めます。 PostgreSQLは高機能で実用的なオープンソースDBMSです。

実習 1-1: PostgreSQLのインストール

PostgreSQLをインストールし、基本的な接続を確認します。

手順
  1. PostgreSQL公式サイトからインストーラをダウンロード
  2. インストールを実行（デフォルト設定でOK）
  3. スーパーユーザーパスワードを設定
  4. コマンドライン（psql）で接続確認
実行例
# PostgreSQLに接続
psql -U postgres -h localhost

# バージョン確認
SELECT version();
期待される結果

PostgreSQLのバージョン情報が表示され、正常に接続できることを確認

実習 1-2: 学習用データベースの作成

SQL学習用の専用データベースを作成します。

手順
  1. psqlでPostgreSQLに接続
  2. 学習用データベース「sql_learning」を作成
  3. 作成したデータベースに接続
  4. データベースの一覧表示で確認
実行例
-- 学習用データベースの作成
CREATE DATABASE sql_learning;

-- データベース一覧の確認
\l

-- 作成したデータベースに接続
\c sql_learning

-- 接続先データベースの確認
SELECT current_database();
期待される結果

「sql_learning」データベースが作成され、正常に接続できることを確認

1.5 基本的なSQL文の実行

データベースに接続できたら、簡単なSQL文を実行してみましょう。 まずは、システム情報を取得するクエリから始めます。

実習 1-3: 基本的なクエリの実行

PostgreSQLでシステム情報を取得する基本的なクエリを実行します。

実行例
-- 現在の日時を取得
SELECT NOW();

-- 現在のユーザー名を取得
SELECT USER;

-- データベース名を取得
SELECT current_database();

-- 簡単な計算
SELECT 100 + 200 AS calculation_result;

-- 文字列の結合
SELECT 'Hello, ' || 'SQL!' AS greeting;
注意点
  • SQL文はセミコロン（;）で終了する必要があります
  • SQLのキーワード（SELECT、FROMなど）は大文字小文字を区別しません
  • 文字列はシングルクォート（'）で囲みます
  • PostgreSQLでは文字列の結合に||演算子を使用します
理解度確認クイズ
  1. 問題1: リレーショナルデータベースでデータを格納する基本単位は何ですか？
    ヒント: 行と列で構成される構造
  2. 問題2: SQLの4つの分類（DDL、DML、DCL、TCL）のうち、データの検索や更新を行うのはどれですか？
  3. 問題3: PostgreSQLで文字列を結合する演算子は何ですか？
  4. 問題4: 以下のSQL文の実行結果を予想してください： 
    SELECT 'PostgreSQL' || ' は' || '素晴らしい' || 'です！';
  5. 問題5: データベースを作成するSQL文は、SQLの分類でいうとどのカテゴリに属しますか？
解答を表示
解答:
  1. テーブル（Table）
  2. DML（Data Manipulation Language）
  3. ||（パイプ2つ）
  4. PostgreSQL は素晴らしいです！
  5. DDL（Data Definition Language）

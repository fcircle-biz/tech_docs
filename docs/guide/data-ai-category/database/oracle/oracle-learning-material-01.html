1.1 Oracle Databaseとは

Oracle Database（オラクル データベース）は、Oracle Corporation が開発・販売しているリレーショナルデータベース管理システム（RDBMS）です。世界中の大企業や政府機関で広く採用されており、データベース市場において最も高いシェアを誇っています。

銀行の勘定システム、航空券予約システム、ECサイトの商品管理など、ミッションクリティカルな業務システムで使用されており、「データを安全に確実に管理する」ことに特化した製品です。

RDBMSとは

Relational Database Management System（リレーショナル・データベース・マネジメント・システム）の略で、「表形式でデータを管理し、表同士を関連付けて扱えるデータベース管理システム」を指します。Oracle以外にも、PostgreSQL、MySQL、SQL Serverなどが代表的なRDBMSです。

1.2 Oracle Databaseの歴史と特徴

歴史

Oracle Databaseは1977年にラリー・エリソン氏らによって創業されたOracle Corporation（当時はSoftware Development Laboratories）によって開発されました。世界初の商用SQLデータベースとして、1979年にOracle Version 2がリリースされました（Version 1は社内テスト版のため非公開）。

timeline title Oracle Database 主要バージョンの歴史 1979 : Oracle V2 : 世界初の商用SQLデータベース 1988 : Oracle 6 : PL/SQL導入 1992 : Oracle 7 : 高度なトランザクション管理 2001 : Oracle 9i : インターネット対応強化 2007 : Oracle 11g : グリッド・コンピューティング 2013 : Oracle 12c : マルチテナント機能 2018 : Oracle 18c : 自律型データベース 2023 : Oracle 23ai : AI統合・JSON強化

主な特徴

  • 高い信頼性と安全性: データの整合性を保つACID特性（原子性、一貫性、独立性、永続性）を完全にサポート
  • 高性能: 大量データの処理に強く、複雑なクエリも高速に実行できる
  • スケーラビリティ: 小規模から大規模システムまで対応可能
  • PL/SQL: データベース内でプログラムを実行できる独自言語を搭載
  • 充実したサポート: エンタープライズ向けの手厚い技術サポート体制

1.3 Oracle Database Freeについて

Oracle Database Free（旧Oracle Database XE）は、Oracleが提供する無料版のデータベースです。学習、開発、小規模な本番環境での利用が可能です。

Oracle Database Freeの特徴

項目 Oracle Database Free Enterprise Edition（参考）
ライセンス費用 無料 有料（数百万円〜）
メモリ使用量 最大2GB 制限なし
ディスク容量 最大12GB 制限なし
CPU 最大2スレッド 制限なし
基本機能 ほぼすべて利用可能 すべて利用可能
商用利用 可能（制限内） 可能

重要ポイント

Oracle Database Freeは学習目的だけでなく、リソース制限内であれば小規模な商用システムでも利用できます。基本的なSQL操作やPL/SQLの学習には十分な機能を備えています。

環境別の特徴

Windows環境

  • Oracle Database Freeを直接インストール
  • シンプルな構成で初心者向け
  • インストール手順が比較的簡単
  • 推奨: 8GB以上のRAM、20GB以上の空き容量

Mac環境

  • Docker Desktopを使用
  • コンテナでOracleを実行
  • 環境の削除・再構築が容易
  • 推奨: 8GB以上のRAM、30GB以上の空き容量

Mac環境でDockerを使う理由

Oracle Database FreeはMacOS向けのネイティブインストーラーを提供していないため、Mac環境ではDockerコンテナを利用してOracleを実行します。これにより、仮想環境内でLinux版のOracleを動かすことができます。

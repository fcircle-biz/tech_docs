確率分布と正規分布 - データの「形」を知る

二項分布、ポアソン分布の基礎と、統計学で最も重要な正規分布について学びます。標準化（z値への変換）と標準正規分布表の使い方を習得します。

この章で学ぶこと

  • 確率分布とは何か、データの「形」の意味を理解する
  • 二項分布の基本概念と具体的な活用例を学ぶ
  • 正規分布の性質と「68-95-99.7ルール」を理解する
  • 標準化（z値）の計算方法と実践的な使い方を習得する

6.1 確率分布とは - データの「形」を理解する

前章で確率の基本を学びましたが、この章では「確率分布」という、データの「形」を表す概念を学びます。 確率分布を理解することで、データ全体の特徴やパターンを把握できるようになります。

確率分布とは

確率分布とは、ある事象が起こる確率がどのように分布しているかを表したものです。 ヒストグラムをイメージすると分かりやすいでしょう。データがどの範囲に多く集まっているか、 どのような「形」をしているかを可視化したものです。

身近な例: サイコロの確率分布

サイコロを1回振ったとき、1〜6の目が出る確率はそれぞれ1/6（約16.7%）です。 これをグラフにすると、すべて同じ高さの棒グラフになります。これが一様分布と呼ばれる確率分布です。 

サイコロの確率分布（一様分布）

確率
 |
 | ■  ■  ■  ■  ■  ■
 | ■  ■  ■  ■  ■  ■
 +--------------------→ 目の数
   1  2  3  4  5  6

各目が出る確率: 1/6 = 約16.7%

離散型と連続型の確率分布

確率分布には大きく分けて2つの種類があります。

離散型確率分布

とびとびの値をとる確率分布。サイコロの目、コインの表裏、不良品の個数など。

例: 二項分布、ポアソン分布

連続型確率分布

連続した値をとる確率分布。身長、体重、気温、測定値など。

例: 正規分布、指数分布

6.2 二項分布 - 「成功/失敗」を繰り返す

二項分布は、「成功か失敗か」という2つの結果しかない試行を複数回繰り返したときの確率分布です。 コイン投げ、品質検査、アンケート調査など、ビジネスでも頻繁に登場します。

二項分布の3つの条件

  1. 各試行は成功か失敗かの2択である
  2. 成功する確率は毎回同じである
  3. 各試行は独立している（前の結果が次に影響しない）

具体例: コイン投げ

コインを10回投げたとき、表が何回出るか？

コインを10回投げて、表が出る確率は1/2です。このとき、「10回中5回表が出る」確率が最も高く、 0回や10回など極端な結果は確率が低くなります。

期待値： 10回 × 1/2 = 5回

※ 実際には4回や6回になることも多い

ビジネスでの活用例

品質検査での二項分布

製造ラインで不良率が2%だとします。100個の製品を検査したとき、不良品が何個見つかるかは二項分布に従います。

期待値： 100個 × 2% = 2個

※ 実際には0〜5個程度のばらつきがある

10個以上見つかったら異常と判断

6.3 正規分布（ガウス分布） - 統計学で最も重要な分布

正規分布は、統計学で最も重要な確率分布です。テストの点数、身長、体重、測定誤差など、 自然界やビジネスのデータの多くは正規分布に従います。別名ガウス分布とも呼ばれます。

正規分布の特徴

  • • 左右対称の釣鐘型（ベルカーブ）
  • • 平均値を中心に分布している
  • • 平均値付近のデータが最も多い
  • • 平均から離れるほどデータは少なくなる

身近な例: テストの点数分布

100人が受けたテストの点数

平均点が60点のとき、60点付近の人が最も多く、0点や100点の人は極端に少なくなります。 このようなデータの分布が正規分布です。

分布のイメージ：

• 平均60点付近: 約68%の人（最も多い）

• 40〜80点の範囲: 約95%の人

• 0〜20点や80〜100点: 各2.5%程度（少ない）

正規分布を決める2つのパラメータ

正規分布は、平均値μ（ミュー）と標準偏差σ（シグマ）の2つだけで完全に決まります。

平均値 μ（ミュー）

分布の中心位置を決めます。テストの平均点、身長の平均など。

例: μ = 60点（テストの平均点）

標準偏差 σ（シグマ）

分布の広がりを決めます。σが大きいほどデータのばらつきが大きい。

例: σ = 10点（点数のばらつき）

正規分布は平均μを中心に左右対称で、μ±σの範囲に約68%のデータが含まれます。

6.4 68-95-99.7ルール - 正規分布の重要な性質

正規分布には「68-95-99.7ルール」という非常に重要な性質があります。 これを覚えておくと、データの分布を直感的に理解できます。

68-95-99.7ルール（経験則）

1

平均μ ± 1σの範囲

約68%のデータが含まれる

例: 平均60点、σ=10点 → 50〜70点に約68%

2

平均μ ± 2σの範囲

約95%のデータが含まれる

例: 平均60点、σ=10点 → 40〜80点に約95%

3

平均μ ± 3σの範囲

約99.7%のデータが含まれる

例: 平均60点、σ=10点 → 30〜90点に約99.7%

このルールの意味

平均から±3σ（標準偏差3個分）を超えるデータはわずか0.3%しかありません。 そのため、±3σを超えるデータは「異常値」として扱われることが多いです。

実例: 品質管理での活用

製品の重量管理（3σルール）

ある製品の重量が平均500g、標準偏差10gの正規分布に従うとします。

正常範囲： 500g ± 30g（3σ） = 470〜530g

• この範囲に99.7%の製品が含まれる

470g未満や530g超の製品は異常と判断し、原因調査

6.5 標準化（z値への変換） - データを「共通の物差し」で測る

平均や標準偏差が異なるデータ同士を比較するとき、標準化という手法を使います。 標準化とは、データを「平均0、標準偏差1」の共通の物差しに変換することです。 変換後の値をz値（z score）と呼びます。

z値の計算式

z = (x - μ) / σ

  • • x: 元のデータの値
  • • μ: 平均値
  • • σ: 標準偏差
  • • z: 標準化後の値（z値）

z値の意味

z値は「平均からどれだけ離れているか」を標準偏差の何個分かで表した値です。

z = 0

平均値と同じ

z = +1

平均より標準偏差1個分上

z = -2

平均より標準偏差2個分下

具体例: テストの点数を標準化する

テストで80点を取ったとき

平均60点、標準偏差10点のテストで、80点を取った場合のz値を計算します。

z = (80 - 60) / 10 = 20 / 10 = 2.0

解釈： z = 2.0 は、平均より標準偏差2個分上の成績です。 68-95-99.7ルールから、上位約2.5%に入る優秀な成績だと分かります。

標準化のメリット: 異なるテストの成績を比較する

平均や難易度が異なる2つのテストの成績を、z値で比較できます。

どちらのテストの成績が良い？

数学テスト

あなた: 80点

平均: 60点、σ: 10点

z = (80-60)/10 = 2.0

英語テスト

あなた: 75点

平均: 50点、σ: 15点

z = (75-50)/15 = 1.67

結論： 数学テストの方がz値が高いため、相対的に良い成績です。 生の点数では英語が75点で数学が80点ですが、平均や難易度を考慮すると数学の方が優秀だと分かります。

身近な例: 偏差値とは

偏差値 = z値を見やすくしたもの

偏差値は、z値をより直感的に理解できるように変換したものです。

偏差値 = 50 + 10 × z

  • • 偏差値50 = 平均（z = 0）
  • • 偏差値60 = 平均より標準偏差1個分上（z = 1）
  • • 偏差値70 = 平均より標準偏差2個分上（z = 2）

実習: z値を計算してみよう

以下のデータで、あなたのz値を計算してみましょう。

問題：

ある会社の社員100人の月の残業時間を調査したところ、平均20時間、標準偏差5時間でした。 あなたの残業時間が15時間だったとき、z値を計算してください。

解答を表示

計算過程：

z = (x - μ) / σ

z = (15 - 20) / 5

z = -5 / 5 = -1.0

解釈： z = -1.0 は、平均より標準偏差1個分少ない残業時間です。 68-95-99.7ルールから、あなたは残業が少ない方から約16%の位置にいます（平均以下の34%のうち、平均から-1σまでの範囲外）。

第6章のまとめ

確率分布： ある事象が起こる確率がどのように分布しているかを表したもの

二項分布： 「成功/失敗」の2択を繰り返す試行の確率分布。品質検査などで活用

正規分布： 左右対称の釣鐘型。平均μと標準偏差σで完全に決まる

68-95-99.7ルール： μ±1σに68%、μ±2σに95%、μ±3σに99.7%のデータが含まれる

標準化（z値）： データを「平均0、標準偏差1」に変換し、異なるデータを比較可能にする

理解度確認クイズ

  1. Q1. 正規分布で、平均μ±2σの範囲に含まれるデータは全体の約何%でしょうか？

    回答を表示

    正解： 約95%です。これは「68-95-99.7ルール」の一部で、 平均から標準偏差2個分の範囲（μ±2σ）に約95%のデータが含まれます。

  2. Q2. テストの平均点が50点、標準偏差が10点のとき、70点を取った人のz値はいくつでしょうか？

    回答を表示

    正解： z = (70 - 50) / 10 = 2.0です。 これは平均より標準偏差2個分上の成績で、上位約2.5%に入る優秀な結果です。

  3. Q3. 二項分布が適用できる条件として正しくないものはどれでしょうか？

    A. 各試行は成功か失敗かの2択である

    B. 成功する確率は毎回変化する

    C. 各試行は独立している

    回答を表示

    正解： B「成功する確率は毎回変化する」が正しくありません。 二項分布では、成功する確率は毎回同じである必要があります。

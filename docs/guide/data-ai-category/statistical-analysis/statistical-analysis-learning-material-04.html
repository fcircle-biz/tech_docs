データの可視化 - グラフで伝える技術

ヒストグラム、散布図、箱ひげ図など、各グラフの特徴と適切な使い分けを学びます。データの特性に合わせた効果的な可視化手法を身につけます。

この章で学ぶこと

  • データ可視化の重要性と目的を理解する
  • ヒストグラム・散布図・箱ひげ図の読み方と作り方を習得する
  • データの種類に応じた適切なグラフを選択できるようになる
  • 効果的な可視化のポイントを理解し実践できる

4.1 データ可視化の重要性 - なぜグラフが必要なのか

データを数字の羅列だけで見ても、そこに隠れたパターンや傾向を見つけるのは困難です。 グラフによる可視化は、複雑なデータを一目で理解できる形に変換する強力な手段です。

直感的な理解

数値表よりもグラフの方が、パターンや傾向を瞬時に把握できる

異常値の発見

外れ値や異常なデータポイントを視覚的に発見しやすい

効果的なコミュニケーション

専門知識がない人にも、データの意味を分かりやすく伝えられる

洞察の発見

数値だけでは気づかない関係性や法則性を発見できる

「1枚の図は1000の言葉に値する」

適切なグラフを使えば、複雑なデータの特徴を一瞬で伝えることができます。 逆に、不適切なグラフは誤解を招く可能性もあるため、正しい選択が重要です。

4.2 ヒストグラム - データの分布を見る

ヒストグラムは、データの分布（どの範囲にどれだけのデータが集まっているか）を視覚化するグラフです。 横軸にデータの範囲、縦軸に度数（その範囲に含まれるデータの個数）を表示します。

ヒストグラムとは

ヒストグラムの特徴

  • • 連続的な数値データの分布を表現
  • • データを階級（区間）に分けて度数を棒グラフで表示
  • • データの「かたまり具合」や「ばらつき」が一目で分かる
  • • 正規分布や偏った分布などの形状を確認できる

具体例：テストの点数分布

階級（区間）の幅に注意

階級の幅を狭くしすぎると細かすぎて傾向が見えにくく、 広くしすぎると大雑把になり詳細が分からなくなります。 データ量に応じて、5～20個程度の階級に分けるのが一般的です。

4.3 散布図 - 2つの変数の関係を見る

散布図は、2つの変数の関係を視覚化するグラフです。 横軸と縦軸にそれぞれ異なる変数を配置し、データを点として表示します。 相関関係（2つの変数の関連性）を一目で把握できるのが特徴です。

散布図とは

散布図の特徴

  • • 2つの変数の関係性（相関）を視覚化
  • • 正の相関、負の相関、相関なしの3パターンを判別
  • • 外れ値（他のデータから大きく離れた点）を発見しやすい
  • • 直線的な関係だけでなく、曲線的な関係も発見できる

具体例：勉強時間と点数の関係

20人の勉強時間とテスト点数

横軸に勉強時間（時間）、縦軸に点数を配置すると：

観察されるパターン：

• 勉強時間が長い人ほど点数が高い傾向がある（右上がりの分布）
• 完全に一直線ではないが、おおまかに正の相関が見られる
• 勉強時間5時間で90点の生徒がいる一方、3時間で80点の生徒もいる
→ 勉強時間だけで決まるわけではないが、影響はある

分かること： 勉強時間と点数には正の相関がある。 つまり、一方が増えると他方も増える傾向がある。

相関と因果関係は違う

散布図で相関が見られても、それが因果関係（原因と結果）とは限りません。 例えば「アイスの売上と水難事故の件数」には正の相関がありますが、 アイスが事故の原因ではなく、両方とも「暑い夏」という共通要因によるものです。

4.4 箱ひげ図 - データの全体像を把握する

箱ひげ図（ボックスプロット）は、データの分布を5つの数値（最小値、第1四分位数、中央値、第3四分位数、最大値）で要約したグラフです。 複数のグループを比較するときに特に便利です。

箱ひげ図とは

箱ひげ図の構成要素

  • • 箱: データの中央50%が含まれる範囲（第1四分位数～第3四分位数）
  • • 箱の中の線: 中央値（データを半分に分ける値）
  • • ひげ: データの広がり（最小値と最大値）
  • • 外れ値: 極端に離れたデータ点（個別に表示）

箱ひげ図の読み方

箱ひげ図が教えてくれること

箱の位置 → データの中心がどこにあるか

箱が上にあれば全体的に値が大きい、下にあれば小さい

箱の高さ → データのばらつき具合

箱が長いとばらつきが大きい、短いとばらつきが小さい

ひげの長さ → データの範囲の広さ

ひげが長いと最小値と最大値の差が大きい

箱内の線の位置 → データの偏り

線が箱の中心にあれば左右対称、偏っていれば分布に偏りがある

具体例：3つのクラスのテスト結果比較

A組、B組、C組の箱ひげ図を比較

3つのクラスのテスト結果を箱ひげ図で並べて表示すると：

観察されること：

  • • A組の箱が最も高い位置 → 平均的に点数が高い
  • • B組の箱が最も長い → ばらつきが大きい（点数の差が大きい）
  • • C組の箱は短く中央値の線も中心 → ばらつきが小さく均等
  • • B組に外れ値（極端に低い点数）が見られる

分かること： 箱ひげ図を使うことで、複数のグループを一目で比較し、 それぞれの特徴（中央値、ばらつき、外れ値）を把握できます。

4.6 効果的な可視化のポイント

グラフを作るだけでは不十分です。伝わるグラフを作るためには、 いくつかのポイントを押さえる必要があります。

良いグラフを作るための7つのポイント

1

タイトルを明確に

「何のグラフか」が一目で分かるタイトルを付ける

例: 「グラフ1」→「2024年度 部門別売上比較」

2

軸ラベルと単位を明記

横軸・縦軸が何を表すか、単位は何かを必ず表示

例: 縦軸「売上（百万円）」、横軸「月」

3

色使いを工夫する

色は意味を持たせて使う（赤=警告、緑=良好など）

カラーユニバーサルデザインにも配慮

4

適切なスケールを選ぶ

縦軸を0から始めないと、差が誇張されて見える

意図的に誤解を招くグラフにしない

5

情報を詰め込みすぎない

1つのグラフに1つのメッセージが基本

複雑になる場合は、複数のグラフに分ける

6

注釈を活用する

重要なポイントに矢印や注釈を加えて理解を助ける

例: 「この月に新製品発売」など

7

データの出典を明記

データの信頼性を示すため、出典元を記載

例: 「出典: 総務省統計局 2024年度調査」

誤解を招くグラフの例

  • • 縦軸が0から始まっていない： 差が実際より大きく見える
  • • 3D効果の濫用： 円グラフの手前が大きく見えて誤解を招く
  • • 過度な装飾： データより見た目が目立ち、本質が伝わらない

第4章のまとめ

データ可視化の重要性： 数値だけでは見えないパターンや関係性を視覚的に把握できる

ヒストグラム： データの分布を表示し、ばらつきや偏りを確認できる

散布図： 2つの変数の相関関係（正の相関・負の相関・相関なし）を判別できる

箱ひげ図： データの分布を5つの数値で要約し、複数グループを比較できる

グラフの使い分け： データの種類と目的に応じて適切なグラフを選択する

効果的な可視化： タイトル、軸ラベル、色使い、スケールに注意して伝わるグラフを作る

理解度確認クイズ

  1. Q1. ヒストグラムが適しているのはどのような場面でしょうか？

    回答を表示

    正解： データの分布（どの範囲にどれだけデータが集まっているか）を見たいときに適しています。 例えば、テストの点数分布、年齢分布、商品の価格帯分布などです。 ヒストグラムを見ることで、データが正規分布しているか、偏っているか、複数の山があるかなどが分かります。

  2. Q2. 散布図で「正の相関」が見られるとき、それはどういう意味でしょうか？

    回答を表示

    正解： 正の相関とは、一方の変数が増えると他方の変数も増える傾向があることを意味します。 散布図では右上がりの分布として現れます。例えば「勉強時間と点数」「身長と体重」などです。 ただし、相関があるからといって因果関係（原因と結果）があるとは限らないことに注意が必要です。

  3. Q3. 箱ひげ図の「箱」は何を表していますか？

    回答を表示

    正解： 箱は、データの中央50%が含まれる範囲を表しています。 具体的には、第1四分位数（下位25%）から第3四分位数（上位25%）までの範囲です。 箱の高さが大きいとデータのばらつきが大きく、小さいとばらつきが小さいことを示します。 箱の中の線は中央値を表しています。

  4. Q4. 円グラフを使うべきでない場面はどれでしょうか？

    回答を表示

    正解： 円グラフは全体に占める割合（構成比）を示すのに適していますが、 以下の場面では使うべきではありません：
    • カテゴリが多すぎる場合（7個以上）→ 棒グラフの方が見やすい
    • 時系列の変化を見たい場合 → 折れ線グラフが適切
    • 正確な数値の比較をしたい場合 → 棒グラフの方が正確

確率の基礎 - 不確実性を数値化する

確率の基本概念と計算方法を、サイコロやトランプといった身近な例で学びます。加法定理・乗法定理、条件付き確率の考え方を理解し、データの不確実性を数値化できるようになります。

この章で学ぶこと

  • 確率とは何か、基本的な考え方を理解する
  • 加法定理と乗法定理を身近な例で学ぶ
  • 条件付き確率の概念を理解する
  • 確率を使った意思決定の基礎を身につける

5.1 確率とは - 不確実性を数値で表す

確率とは、ある出来事が起こる可能性を0から1の数値で表したものです。 0は「絶対に起こらない」、1は「必ず起こる」を意味し、0.5は「50%の確率で起こる」ことを表します。

確率の表現方法

  • • 小数表記: 0.5（0から1の範囲）
  • • パーセント表記: 50%（0%から100%の範囲）
  • • 分数表記: 1/2（数学的表現）

確率の基本式

確率は次の式で計算できます：

確率 P

P = 起こってほしい事象の数 ÷ すべての可能性の数

※ すべての可能性が等しく起こりうる場合（同様に確からしい）

例: サイコロを振る

サイコロで「3」が出る確率は？

すべての可能性： 1, 2, 3, 4, 5, 6 の6通り

「3」が出る場合： 1通り

確率： P = 1/6 ≒ 0.167 = 16.7%

サイコロを100回振れば、理論的には約16〜17回「3」が出ることが期待されます。

5.2 加法定理 - 「または」の確率

加法定理は、「AまたはBが起こる確率」を計算するときに使います。 「または」というのは、どちらか一方、または両方が起こることを意味します。

加法定理の公式

AまたはBが起こる確率

P(A または B) = P(A) + P(B) - P(A かつ B)

※ P(A かつ B) は両方が同時に起こる確率（重複分を引く）

なぜ重複分を引くのか？

P(A) と P(B) を単純に足すと、AとBが両方起こる場合を2回カウントしてしまうため、 1回分を引いて調整する必要があります。

例: トランプカードを1枚引く

「ハート」または「絵札」が出る確率は？

全カード数： 52枚

ハートのカード（A）： 13枚 → P(A) = 13/52

絵札（J, Q, K）のカード（B）： 12枚 → P(B) = 12/52

ハートかつ絵札： 3枚（ハートのJ, Q, K） → P(A かつ B) = 3/52

P(A または B) = 13/52 + 12/52 - 3/52 = 22/52 ≒ 0.423 = 42.3%

カードを1枚引いたとき、約42%の確率で「ハート」または「絵札」が出ます。

排反事象の場合

排反事象とは、同時に起こることがない事象のことです。 この場合、P(A かつ B) = 0 なので、公式はシンプルになります。

排反事象の加法定理

P(A または B) = P(A) + P(B)

例: サイコロで「1が出る」または「2が出る」 → 同時には起こらない

5.3 乗法定理 - 「かつ」の確率

乗法定理は、「AかつBが起こる確率」を計算するときに使います。 「かつ」は、両方の事象が同時に起こることを意味します。

乗法定理の公式

AかつBが起こる確率

P(A かつ B) = P(A) × P(B|A)

※ P(B|A) は「Aが起こったときにBが起こる確率」（条件付き確率）

独立事象の場合

独立事象とは、一方の結果が他方に影響を与えない事象のことです。 この場合、公式はさらにシンプルになります。

独立事象の乗法定理

P(A かつ B) = P(A) × P(B)

例: サイコロを2回振る → 1回目の結果は2回目に影響しない

例: サイコロを2回振る

1回目も2回目も「6」が出る確率は？

1回目に「6」が出る確率（A）： P(A) = 1/6

2回目に「6」が出る確率（B）： P(B) = 1/6

独立事象なので：

P(A かつ B) = 1/6 × 1/6 = 1/36 ≒ 0.028 = 2.8%

2回連続で「6」が出る確率は約2.8%と、かなり低いことが分かります。

例: 非復元抽出（袋から玉を取り出す）

袋の中に赤玉3個、白玉2個。2回連続で赤玉を引く確率は？

全体： 5個の玉

1回目に赤玉を引く確率（A）： P(A) = 3/5

1回目に赤玉を引いた後、残りは赤2個、白2個の計4個

2回目に赤玉を引く確率（B|A）： P(B|A) = 2/4 = 1/2

P(A かつ B) = 3/5 × 1/2 = 3/10 = 0.3 = 30%

非復元抽出では1回目の結果が2回目に影響するため、独立ではありません。

5.4 条件付き確率 - 「〜のとき」の確率

条件付き確率は、「ある事象Aが起こったという条件のもとで、事象Bが起こる確率」を表します。 記号では P(B|A) と書き、「AのときにBが起こる確率」と読みます。

条件付き確率の公式

Aが起こったときにBが起こる確率

P(B|A) = P(A かつ B) ÷ P(A)

※ P(A) > 0 の場合

条件付き確率の直感的な理解

「Aが起こった」という情報によって、全体の可能性がP(A)に絞り込まれます。 その中でBも起こる確率を求めるため、P(A かつ B) を P(A) で割ります。

例: カード引きの条件付き確率

「赤いカード」を引いたとき、それが「ハート」である確率は？

全カード数： 52枚

赤いカード（A）： ハート13枚 + ダイヤ13枚 = 26枚 → P(A) = 26/52

ハートのカード（B）： 13枚

赤いカードかつハート： 13枚 → P(A かつ B) = 13/52

P(B|A) = (13/52) ÷ (26/52) = 13/26 = 1/2 = 50%

赤いカードと分かった時点で、ハートかダイヤのどちらか半々なので50%になります。

実生活での条件付き確率

医療検査

「検査結果が陽性」のとき「実際に病気である」確率（偽陽性の問題）

スパムフィルター

「特定の単語が含まれる」とき「スパムメールである」確率

天気予報

「気圧が低い」とき「雨が降る」確率

購買分析

「商品Aを購入した」とき「商品Bも購入する」確率

5.5 確率を使った意思決定

確率の知識は、日常生活やビジネスでの意思決定に役立ちます。 不確実な状況でも、確率を使って合理的な判断ができるようになります。

例: 期待値を使った意思決定

くじ引きの期待値計算

問

1回500円のくじ引きがあります。1等（確率1%）で5万円、2等（確率5%）で5千円、 3等（確率10%）で千円が当たります。このくじは引くべきでしょうか？

期待値の計算：

  • • 1等: 50,000円 × 0.01 = 500円
  • • 2等: 5,000円 × 0.05 = 250円
  • • 3等: 1,000円 × 0.10 = 100円
  • • ハズレ: 0円 × 0.84 = 0円

期待値 = 500 + 250 + 100 = 850円

→ 500円で期待値850円なので、理論的には「引くべき」と判断できます。

注意点

期待値は「長期的に平均すると」の値です。1回だけ引く場合は必ず得するわけではありません。 また、リスク許容度（どれだけ損失を受け入れられるか）も考慮する必要があります。

練習問題

  1. 1 サイコロを1回振ったとき、「3以下の目」が出る確率を求めてください。
  2. 2 コインを3回投げて、すべて表が出る確率を求めてください。
  3. 3 トランプ52枚から1枚引いたとき、「スペード」または「エース」が出る確率を求めてください。
解答を表示

問題1の解答:

3以下の目 = {1, 2, 3} の3通り

確率 = 3/6 = 1/2 = 0.5 = 50%

問題2の解答:

各回で表が出る確率 = 1/2

独立事象なので: (1/2) × (1/2) × (1/2) = 1/8 = 0.125 = 12.5%

問題3の解答:

スペード: 13枚, エース: 4枚, スペードのエース: 1枚

P(スペード または エース) = 13/52 + 4/52 - 1/52 = 16/52 = 4/13 ≒ 0.308 = 30.8%

第5章のまとめ

確率の基本： 起こる可能性を0から1の数値で表したもの。P = 起こる事象 ÷ すべての可能性

加法定理： 「AまたはB」の確率 = P(A) + P(B) - P(A かつ B)

乗法定理： 「AかつB」の確率 = P(A) × P(B|A)、独立なら P(A) × P(B)

条件付き確率： P(B|A) = P(A かつ B) ÷ P(A)、Aが起こったときにBが起こる確率

実用例： 期待値を使った意思決定、医療検査、スパムフィルター、購買分析など

理解度確認クイズ

  1. Q1. 加法定理で「P(A かつ B)」を引く理由は何でしょうか？

    回答を表示

    正解： P(A) と P(B) を単純に足すと、AとBが両方起こる場合を2回カウントしてしまうため、 重複分である P(A かつ B) を1回分引いて調整する必要があります。

  2. Q2. 独立事象とは何でしょうか？具体例を挙げて説明してください。

    回答を表示

    正解： 独立事象とは、一方の結果が他方に影響を与えない事象のことです。
    例: サイコロを2回振る場合、1回目の結果は2回目に影響しません。
    逆に非独立の例: 袋から玉を取り出す（非復元）場合、1回目の結果が2回目の確率に影響します。

  3. Q3. 条件付き確率 P(B|A) を日常的な言葉で説明してください。

    回答を表示

    正解： 「事象Aが起こったという条件のもとで、事象Bが起こる確率」のことです。
    例えば、「赤いカードを引いた（事象A）という条件のもとで、それがハートである（事象B）確率」のように、 ある情報が得られた後の確率を表します。

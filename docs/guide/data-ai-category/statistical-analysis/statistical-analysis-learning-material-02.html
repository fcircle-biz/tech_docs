データの種類と収集方法 - 正しいデータの見極め方

量的データと質的データの違い、データの尺度（名義・順序・間隔・比例）を理解します。また、サンプリング手法とバイアスの問題についても学びます。

この章で学ぶこと

  • 量的データと質的データの違いを理解する
  • 4つの尺度（名義・順序・間隔・比例）の特徴を学ぶ
  • サンプリング手法（無作為抽出・層別抽出など）を知る
  • バイアス（偏り）の問題と対処法を認識する

2.1 データとは何か - 数字と言葉で表される情報

統計学で扱う「データ」とは、観察や測定によって得られた事実や情報のことです。 データには大きく分けて「量的データ」と「質的データ」の2種類があります。

データの基本

データは「数値で表せるもの」と「カテゴリーで表せるもの」に分かれます。 どちらのタイプかによって、使える統計手法が変わってきます。

2.2 量的データと質的データの違い

量的データ（数値データ）

数値で測定できるデータのことで、計算（足し算、平均など）ができるのが特徴です。

連続データ

小数点を含む値で表せるデータ

  • • 身長: 165.3cm、172.8cm
  • • 体重: 55.2kg、68.5kg
  • • 温度: 23.5℃、18.7℃
  • • 時間: 1.5時間、3.2時間

離散データ

整数で数えるデータ

  • • 人数: 5人、20人
  • • 個数: 3個、15個
  • • 回数: 2回、10回
  • • 年齢: 25歳、40歳

質的データ（カテゴリーデータ）

カテゴリーや属性で分類できるデータのことで、計算（足し算、平均など）ができないのが特徴です。

質的データの例

性別: 男性、女性、その他

血液型: A型、B型、O型、AB型

職業: 会社員、学生、自営業

満足度: とても満足、満足、不満

よくある間違い

「満足度: 5点、4点、3点…」のように数値で表しても、これは質的データです。 「5点 + 3点 = 8点」のような計算に意味がないため、平均を取る際は注意が必要です。

2.3 4つの尺度 - データの「ものさし」を理解する

データには「どのように測定されたか」によって、4つの尺度があります。 尺度によって使える統計手法が異なるため、正しく理解することが重要です。

4つの尺度の階層

右に行くほど情報量が多く、使える統計手法が増えます

1. 名義尺度（Nominal Scale）

区別・分類するだけのデータ

カテゴリーを区別するだけで、順序や大小関係がないデータです。

具体例:

  • • 性別: 男性、女性
  • • 血液型: A型、B型、O型、AB型
  • • 都道府県: 東京都、大阪府、北海道…
  • • 商品カテゴリ: 食品、衣類、家電…

できる計算: 個数を数える、割合を計算する

2. 順序尺度（Ordinal Scale）

順番がある（でも間隔は不明）

カテゴリーに順序・大小関係があるが、間隔が等しくないデータです。

具体例:

  • • 満足度: とても満足 ＞ 満足 ＞ 普通 ＞ 不満
  • • 成績: A ＞ B ＞ C ＞ D
  • • 順位: 1位、2位、3位…
  • • 企業規模: 大企業 ＞ 中企業 ＞ 小企業

注意: 「とても満足」と「満足」の差 ≠ 「満足」と「普通」の差（間隔が不明）

3. 間隔尺度（Interval Scale）

差に意味がある（でも絶対零点なし）

順序があり、値の差（間隔）に意味があるが、絶対的なゼロ点がないデータです。

具体例:

  • • 温度（℃）: 20℃と10℃の差 = 30℃と20℃の差（等間隔）
  • • 西暦年: 2020年と2010年の差 = 10年
  • • IQスコア: 100と110の差 = 110と120の差

できない計算: 「20℃は10℃の2倍暑い」とは言えない（0℃は絶対的なゼロではない）

4. 比例尺度（Ratio Scale）

比率に意味がある（絶対零点あり）

間隔尺度の性質に加えて、絶対的なゼロ点があるため、比率の計算ができるデータです。

具体例:

  • • 身長: 180cmは90cmの2倍（0cmは絶対的なゼロ）
  • • 体重: 60kgは30kgの2倍
  • • 売上金額: 100万円は50万円の2倍
  • • 時間: 2時間は1時間の2倍

できる計算: 四則演算すべて（足す、引く、掛ける、割る）

順序尺度 満足度、成績

2.4 サンプリング手法 - 一部から全体を推測する技術

すべてのデータを集めるのは不可能な場合が多いため、一部を抽出（サンプリング）して分析します。 サンプリングの方法によって、結果の信頼性が大きく変わります。

サンプリングの目的

母集団（全体）の特性を、標本（サンプル）から推測することです。 例: 全国民1億人の意識調査は不可能 → 1,000人に調査して全体を推測

1. 無作為抽出（ランダムサンプリング）

すべての対象が等しく選ばれる確率を持つ

母集団のすべてのメンバーが同じ確率で選ばれるようにする方法です。 最も基本的で、偏りのないサンプリングです。

具体例:

  • • 全社員1,000人から、乱数表を使って100人を選ぶ
  • • 抽選で当選者を決める
  • • Excelの RAND() 関数でランダムに選択

2. 層別抽出（層化サンプリング）

グループ（層）に分けてから抽出

母集団を特性ごとにグループ分けし、各グループから適切な割合でサンプルを抽出します。

具体例:

社員満足度調査で、部署ごとの割合を反映させる

  • • 営業部（500人）→ 50人抽出
  • • 開発部（300人）→ 30人抽出
  • • 管理部（200人）→ 20人抽出

3. 系統抽出（システマティックサンプリング）

一定間隔で抽出

母集団を並べて、一定の間隔（k番目ごと）で抽出する方法です。

具体例:

  • • 1,000人から100人抽出する場合、10人ごとに1人選ぶ
  • • 製造ラインで10個に1個を検品する

4. 便宜的抽出（Convenience Sampling）

手近な対象を選ぶ（非確率的）

簡便だが偏りが生じやすい方法です。統計的な推測には適していません。

具体例:

  • • 街頭インタビュー（通りがかった人に質問）
  • • SNSでアンケート（フォロワーに依頼）

注意: 特定の層に偏る可能性が高い

2.5 バイアス（偏り）の問題と対処法

データ収集で最も注意すべきはバイアス（偏り）です。 偏ったデータから導いた結論は、現実と大きくズレてしまいます。

よくあるバイアスの例

1. 選択バイアス

サンプルの選び方に偏りがある状態

例: 「スマホアプリのアンケート」→ スマホを使える人しか回答できない

2. 無回答バイアス

特定の人だけが回答しない状態

例: 「仕事満足度調査」→ 不満がある人ほど回答しない傾向

3. 質問バイアス

質問の仕方が回答に影響を与える状態

悪い例: 「この素晴らしい商品に満足していますか？」
良い例: 「この商品をどう評価しますか？」

4. 時期バイアス

特定の時期のデータだけを使う状態

例: 「12月の売上だけ」で年間売上を予測する → クリスマス商戦で偏る

バイアスを防ぐ方法

無作為抽出を使う

すべての対象が等しく選ばれる確率を持つようにする

サンプルサイズを増やす

より多くのデータを集めることで、偏りの影響を減らす

層別抽出を使う

各グループから適切な割合でサンプルを集める

中立的な質問にする

誘導的な表現を避け、客観的な質問文にする

2.6 データ収集の実践 - ビジネスでの活用例

実践例: 新商品の市場調査

新しいスマートフォンアプリをリリースする前に、ターゲットユーザーの意見を収集したい場合

1

データの種類を決める

質的データ: 使いやすさの評価（順序尺度）、量的データ: 利用意向（1-10点）

2

サンプリング方法を選ぶ

層別抽出: 年齢層（20代、30代、40代）ごとに均等にサンプル抽出

3

バイアスを避ける

中立的な質問文を使い、誘導しない（「この便利なアプリを〜」は避ける）

4

データを収集・分析

Excelで集計し、年齢層ごとの評価の違いを比較

第2章のまとめ

データの分類: 量的データ（数値で計算可能）と質的データ（カテゴリーで分類）

4つの尺度: 名義（区別のみ）、順序（順番あり）、間隔（差に意味）、比例（比率計算可能）

サンプリング: 無作為抽出が基本、層別抽出で偏りを防ぐ

バイアス対策: 選択バイアス、質問バイアス、時期バイアスに注意し、中立的なデータ収集を心がける

理解度確認クイズ

  1. Q1. 次のうち、「比例尺度」に該当するデータはどれでしょうか？

    a) 温度（℃）、b) 満足度（5段階）、c) 売上金額、d) 血液型

    回答を表示

    正解: c) 売上金額
    比例尺度は絶対的なゼロ点があり、比率計算ができるデータです。売上金額は「0円」が絶対的なゼロで、「100万円は50万円の2倍」と言えます。 温度（℃）は間隔尺度（0℃は絶対的なゼロではない）、満足度は順序尺度、血液型は名義尺度です。

  2. Q2. 無作為抽出（ランダムサンプリング）の最大の利点は何でしょうか？

    回答を表示

    正解: 偏りのない代表的なサンプルを得られること
    無作為抽出では、母集団のすべてのメンバーが等しい確率で選ばれるため、特定のグループに偏らない公平なサンプルを得ることができます。 これにより、サンプルから母集団全体を推測する際の信頼性が高まります。

  3. Q3. 「この素晴らしい商品に満足していますか？」という質問は、どのようなバイアスを含んでいるでしょうか？

    回答を表示

    正解: 質問バイアス（誘導的な質問）
    「素晴らしい」という肯定的な表現が含まれており、回答者を「満足している」方向に誘導してしまいます。 中立的な質問にするには「この商品をどう評価しますか？」のように、誘導的な表現を避けるべきです。

記述統計の基礎 - データの特徴をつかむ

平均値、中央値、最頻値といった代表値の違いと使い分けを学びます。また、分散と標準偏差を通じてデータのばらつきを理解し、Excelでの計算方法も習得します。

この章で学ぶこと

  • 平均値・中央値・最頻値の違いと使い分けを理解する
  • データのばらつき（分散・標準偏差）の意味を理解する
  • Excelで各種統計量を計算できるようになる
  • データの分布によって適切な代表値を選択できるようになる

3.1 記述統計とは - データの特徴を数値で表す

記述統計とは、手元にあるデータの特徴を整理し、要約する技術です。 大量のデータをそのまま見ても全体像は把握しにくいですが、代表的な数値（代表値）やグラフで表現することで、データの傾向が一目で分かるようになります。

記述統計の主な役割

  • • データの中心（平均的な値）を表す → 代表値
  • • データのばらつき（散らばり具合）を表す → 分散・標準偏差
  • • データの分布（形状）を表す → ヒストグラムなどのグラフ

この章では、特に代表値（データの中心）とばらつきに焦点を当てて学びます。

3.2 代表値(1): 平均値 - データの「真ん中」を求める

平均値（算術平均）は、データの合計をデータ数で割った値です。 最もよく使われる代表値で、「平均点」「平均年齢」など日常的にも頻繁に登場します。

平均値の計算式

平均値 = （データの合計） ÷ （データ数）

具体例: テストの点数

5人の生徒のテストの点数

80点、75点、90点、85点、70点

計算過程：

合計: 80 + 75 + 90 + 85 + 70 = 400点

データ数: 5人

平均値: 400 ÷ 5 = 80点

平均値の特徴

メリット

  • • 全てのデータが計算に使われる
  • • 計算が簡単で直感的
  • • 比較しやすい

デメリット

  • • 極端な値（外れ値）に影響されやすい
  • • データの分布が偏っている場合、実態と異なることがある

平均値が実態を表さない例

年収のデータ: 300万円、350万円、400万円、380万円、5000万円
→ 平均値は1286万円になるが、大多数の人は400万円以下

3.3 代表値(2): 中央値 - データを並べた時の真ん中

中央値（メディアン）は、データを小さい順に並べたとき、ちょうど真ん中にくる値です。 平均値と違い、極端な値の影響を受けにくいという特徴があります。

中央値の求め方

1. データを小さい順に並べる
2. データ数が奇数の場合 → 真ん中の値
3. データ数が偶数の場合 → 真ん中2つの値の平均

具体例: データ数が奇数の場合

5人の年齢: 25歳、30歳、22歳、35歳、28歳

計算過程：

並び替え: 22、25、28、30、35

中央値: 28歳（真ん中の値）

具体例: データ数が偶数の場合

6人の年齢: 25歳、30歳、22歳、35歳、28歳、32歳

計算過程：

並び替え: 22、25、28、30、32、35

真ん中の2つ: 28と30

中央値: (28 + 30) ÷ 2 = 29歳

平均値 vs 中央値 - どちらを使う？

年収データの比較

データ: 300万円、350万円、400万円、380万円、5000万円

平均値: 1286万円

→ 外れ値に引っ張られる

中央値: 380万円

→ 実態に近い

結論: このケースでは中央値のほうが「典型的な値」を表している

3.4 代表値(3): 最頻値 - 最も多く出現する値

最頻値（モード）は、データの中で最も頻繁に出現する値です。 カテゴリカルデータ（「男性/女性」「商品A/B/C」など）でよく使われます。

具体例: 靴のサイズ

10人の靴のサイズ（cm）

24.5、25.0、25.0、25.0、25.5、26.0、25.0、24.0、25.5、25.0

出現回数：

24.0cm: 1回、24.5cm: 1回、25.0cm: 5回、25.5cm: 2回、26.0cm: 1回

最頻値: 25.0cm（最も多く出現）

活用例: 靴屋さんが仕入れる際、最頻値（25.0cm）を多めに仕入れると効率的

最頻値が役立つ場面

  • • アンケートの選択肢（「満足/普通/不満」で最も多い回答）
  • • 商品のサイズ展開（最も売れるサイズを把握）
  • • 顧客の属性分析（最も多い年齢層、性別など）

3.5 代表値の使い分け - データに応じた選択

平均値・中央値・最頻値のどれを使うべきかは、データの性質や分布によって変わります。

代表値の選び方ガイド

1

平均値を使う場合

データが正規分布に近く、外れ値がない場合

例: テストの点数、身長、体重（通常範囲内）

2

中央値を使う場合

データに外れ値がある、または偏った分布の場合

例: 年収、不動産価格、Web訪問者の滞在時間

3

最頻値を使う場合

カテゴリカルデータや、「最も多い」値を知りたい場合

例: 靴のサイズ、アンケート結果、商品の色の人気順

3.6 データのばらつき(1): 分散 - データの散らばり具合

平均値だけではデータの全体像は分かりません。データが平均値からどれくらいばらついているかを知ることも重要です。 分散は、データのばらつきを数値化したものです。

なぜ「ばらつき」が重要なのか？

2つのクラスのテスト結果（どちらも平均70点）

Aクラス

68点、69点、70点、71点、72点

→ 全員が平均付近に集中

Bクラス

30点、50点、70点、90点、110点

→ 大きくばらついている

平均値は同じでも、ばらつきが全く違う

分散の計算手順（母集団の場合）

  1. 1. 平均値を求める
  2. 2. 各データと平均値の差を求める（偏差）
  3. 3. 偏差を2乗する
  4. 4. 2乗した値の平均を求める → これが分散

具体例: 分散の計算

データ: 2、4、6、8、10

計算過程：

1. 平均値: (2+4+6+8+10) ÷ 5 = 6

2. 偏差: (2-6)=-4、(4-6)=-2、(6-6)=0、(8-6)=2、(10-6)=4

3. 偏差の2乗: 16、4、0、4、16

4. 2乗の平均: (16+4+0+4+16) ÷ 5 = 8

分散 = 8

分散の読み方

分散が大きい → データのばらつきが大きい
分散が小さい → データが平均値の周りに集中している

3.7 データのばらつき(2): 標準偏差 - より直感的な指標

分散は偏差を2乗しているため、元のデータと単位が異なります（点数なら「点²」）。 標準偏差は、分散の平方根（ルート）を取ることで、元のデータと同じ単位に戻したものです。

標準偏差の計算

標準偏差 = √（分散）

先ほどの例の続き

データ: 2、4、6、8、10 （分散 = 8）

計算：

標準偏差 = √8 ≒ 2.83

標準偏差 ≒ 2.83

これは「データが平均値から平均して約2.83ずれている」ことを意味します

標準偏差の活用例

成績評価

テストの平均点と標準偏差から、自分の位置を把握できる（偏差値の計算に使用）

品質管理

製品の寸法のばらつきを管理（標準偏差が小さいほど品質が安定）

リスク評価

投資のリターンのばらつき（標準偏差が大きいほどリスクが高い）

健康診断

血圧や体重の変動を把握（標準偏差が大きいと不安定）

3.8 Excelでの計算方法 - 実践してみよう

手計算は理解のために重要ですが、実務ではExcelの関数を使うと簡単に計算できます。

実習 3-1: Excelで代表値とばらつきを計算する

以下のテストの点数データを使って、Excelで各種統計量を計算してみましょう。

手順

  1. 1 Excelを開き、A1セルから以下のデータを入力: 80、75、90、85、70、95、88、78、82、92
  2. 2 平均値を計算: 任意のセルに「=AVERAGE(A1:A10)」と入力
  3. 3 中央値を計算: 「=MEDIAN(A1:A10)」と入力
  4. 4 最頻値を計算: 「=MODE.SNGL(A1:A10)」と入力（データに複数回出現する値がない場合はエラーになります）
  5. 5 分散を計算（母集団）: 「=VAR.P(A1:A10)」と入力
  6. 6 標準偏差を計算（母集団）: 「=STDEV.P(A1:A10)」と入力

Excelの統計関数一覧

よく使う統計関数

関数名 計算内容 使用例
AVERAGE 平均値 =AVERAGE(A1:A10)
MEDIAN 中央値 =MEDIAN(A1:A10)
MODE.SNGL 最頻値 =MODE.SNGL(A1:A10)
VAR.P 分散（母集団） =VAR.P(A1:A10)
VAR.S 分散（標本） =VAR.S(A1:A10)
STDEV.P 標準偏差（母集団） =STDEV.P(A1:A10)
STDEV.S 標準偏差（標本） =STDEV.S(A1:A10)

母集団と標本の違い

• 母集団（.P）: 調べたい対象全体（例: クラス全員のテストの点数）
• 標本（.S）: 母集団の一部（例: 全国調査の際のサンプル）
通常、手元のデータが全体なら「.P」、一部なら「.S」を使います。

第3章のまとめ

平均値: 全データの合計÷データ数。外れ値に影響されやすい

中央値: データを並べた時の真ん中の値。外れ値に強い

最頻値: 最も多く出現する値。カテゴリカルデータで有効

分散: データの散らばり具合を数値化したもの（偏差の2乗の平均）

標準偏差: 分散の平方根。元のデータと同じ単位で解釈しやすい

Excelの活用: AVERAGE、MEDIAN、MODE.SNGL、VAR.P、STDEV.P関数で簡単に計算可能

理解度確認クイズ

  1. Q1. 次のデータの平均値と中央値を求めてください: 10、15、20、25、100

    回答を表示

    平均値: (10+15+20+25+100) ÷ 5 = 34
    中央値: 並べると10、15、20、25、100 → 20
    この例では、外れ値（100）の影響で平均値（34）が中央値（20）より大きくなっています。 実態を表すには中央値のほうが適切です。

  2. Q2. 「データのばらつきが小さい」とは、分散や標準偏差がどのような値になることを指しますか？

    回答を表示

    正解: 分散や標準偏差が小さい値になります。
    ばらつきが小さい = データが平均値の周りに集中している = 分散・標準偏差が小さい
    逆に、ばらつきが大きい場合は、分散・標準偏差が大きくなります。

  3. Q3. 靴のサイズの売れ筋を調べる場合、平均値・中央値・最頻値のどれが最も適切でしょうか？

    回答を表示

    正解: 最頻値が最も適切です。
    理由: 靴のサイズは離散的な値（24.5、25.0など）で、「最も多く売れるサイズ」を知りたいため、 出現頻度が高い値（最頻値）を調べることが実務的に役立ちます。 平均値や中央値では「25.3cm」のような中途半端な値になり、在庫管理には不向きです。

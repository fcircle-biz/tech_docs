推測統計入門 - 一部から全体を推測する

母集団と標本の関係を理解し、一部のデータから全体の傾向を推測する方法を学びます。点推定と区間推定（信頼区間）の考え方を、身近な例を通じて理解できます。

この章で学ぶこと

  • 母集団と標本の違いと関係を理解する
  • なぜ標本から全体を推測できるのかを知る
  • 点推定と区間推定の考え方を理解する
  • 信頼区間の意味と解釈方法を習得する

7.1 母集団と標本とは - 全体と一部の関係

統計学における「母集団」と「標本」は、推測統計の基本となる重要な概念です。

母集団（Population）

調査や分析の対象となる全体の集合のことです。

具体例：

  • • 日本国民全体（約1.2億人）
  • • 特定企業の全従業員
  • • 工場で生産された全製品

標本（Sample）

母集団から選び出された一部のデータのことです。

具体例：

  • • 世論調査の回答者1,000人
  • • 従業員満足度調査の回答者
  • • 品質検査のサンプル製品

わかりやすい例え

鍋で作ったスープの味を確認するとき、鍋全部を飲む必要はありませんよね。スプーン1杯で味見をすれば、鍋全体の味がわかります。

鍋全体 = 母集団
スプーン1杯 = 標本

7.2 なぜ標本から推測するのか

母集団全体を調査できれば最も正確ですが、実際には以下の理由で標本調査が必要になります。

コストの問題

日本国民全員に調査するには、膨大な費用と時間がかかります。標本調査なら低コストで実施可能です。

時間の問題

全数調査には数ヶ月〜数年かかることも。標本調査なら数日〜数週間で結果が得られます。

破壊検査の問題

電球の寿命検査など、検査すると製品が使えなくなる場合は、一部の標本で検査するしかありません。

無限母集団の問題

「未来のすべての顧客」など、母集団が無限または未確定の場合は、標本調査しか方法がありません。

標本調査の利点

適切に設計された標本調査は、少ないコストと時間で、母集団の特徴を十分な精度で推測できます。 世論調査で1,000人程度のサンプルで全国の傾向がわかるのは、この原理によるものです。

7.3 標本抽出の方法 - 無作為抽出の重要性

標本から正しく母集団を推測するには、無作為抽出（ランダムサンプリング）が極めて重要です。

無作為抽出とは

無作為抽出の定義

母集団のすべての個体が等しい確率で選ばれるように標本を抽出する方法です。 これにより、標本が母集団を代表する可能性が高まります。

例： くじ引き、乱数表、コンピュータによるランダム抽出

偏った標本抽出の危険性

偏った抽出の例

  • 駅前調査： 平日昼間に駅前で調査すると、主婦や高齢者が多くなり、 会社員や学生の意見が反映されない
  • 自主参加型調査： 興味のある人だけが回答するため、 無関心な人の意見が欠落する
  • 便宜的抽出： 「手近な人」だけを選ぶと、 特定の属性（年齢、地域など）に偏る

7.4 点推定 - 1つの値で推測する

点推定（Point Estimation）とは、標本のデータから計算した統計量（平均など）を使って、 母集団の特性を1つの値で推測する方法です。

点推定の具体例

例1: 平均身長の推定

日本人男性100人を無作為抽出して測定したところ、平均身長が171.5cmでした。
→ 日本人男性全体の平均身長を171.5cmと推定

例2: 製品不良率の推定

生産ライン製品500個を検査したところ、不良品が10個でした。
→ 全体の不良率を2%（10/500）と推定

点推定の限界

点推定は「1つの値」で推定するため、誤差の大きさがわからないという問題があります。 「平均171.5cm」と推定しても、実際の母集団平均が170cmなのか173cmなのか、 どれくらいのズレがあるかは点推定だけではわかりません。

7.5 区間推定（信頼区間） - 幅を持たせて推測する

区間推定（Interval Estimation）とは、母集団の特性を「○○から△△の間にある」という範囲で推測する方法です。 この範囲を信頼区間（Confidence Interval）と呼びます。

信頼区間の具体例

例: 平均身長の区間推定
標本100人の平均身長が171.5cm、標準偏差が5.0cmだったとします。

点推定の場合：

「母集団の平均は171.5cmです」

区間推定の場合（95%信頼区間）：

「母集団の平均は170.5cm～172.5cmの範囲にあると、95%の確信を持って言えます」

信頼水準とは

信頼水準（Confidence Level）は、「どれくらい自信を持って推定するか」を示す指標です。 一般的に95%または99%が使われます。

95%信頼区間

同じ方法で100回調査を繰り返したら、95回は真の母集団平均がこの範囲に含まれるという意味です。

例: 170.5cm～172.5cm（範囲が狭い）

99%信頼区間

より高い確信度を求める場合に使用します。ただし、範囲が広くなるというトレードオフがあります。

例: 169.8cm～173.2cm（範囲が広い）

わかりやすい例え

点推定は「的の中心を狙う」ようなもの。
区間推定は「この範囲なら当たる」と範囲を示すもの。

範囲を広くすれば当たる確率（信頼水準）は上がりますが、情報としての価値は下がります。 バランスが重要です。

7.6 信頼区間の解釈と活用

信頼区間を正しく理解し、実務で活用するためのポイントを学びます。

よくある誤解

間違った解釈

「95%信頼区間が170.5～172.5cmということは、母集団平均がこの範囲に入る確率が95%である」

なぜ間違いか？

母集団平均は固定された値です（確率的に変動するものではない）。 正しくは「この方法で区間を作ると、100回中95回は真の平均を含む区間ができる」という意味です。

ビジネスでの活用例

実務での信頼区間の使い方

1
顧客満足度調査

「顧客満足度の平均は3.8点（5点満点）、95%信頼区間は3.6～4.0点」
→ 真の満足度が3.5点以下である可能性は低いと判断できる

2
売上予測

「来月の売上予測は1,000万円、95%信頼区間は900～1,100万円」
→ 最悪900万円まで下がる可能性を考慮した資金計画が可能

3
品質管理

「不良率は2.1%、95%信頼区間は1.5～2.7%」
→ 目標不良率3%以下を満たしている可能性が高いと判断

実習: Excelで信頼区間を計算してみよう

以下のデータを使って、平均値の95%信頼区間を計算してみましょう。

データ: 10人の通勤時間（分）

25, 30, 28, 35, 32, 27, 29, 31, 26, 33

Excelでの計算手順

  1. 1. データをA1:A10に入力
  2. 2. 平均: =AVERAGE(A1:A10)
  3. 3. 標準誤差: =STDEV.S(A1:A10)/SQRT(10)
  4. 4. 95%信頼区間の幅: =CONFIDENCE.T(0.05, STDEV.S(A1:A10), 10)
  5. 5. 下限 = 平均 - 信頼区間の幅
  6. 6. 上限 = 平均 + 信頼区間の幅

第7章のまとめ

母集団と標本： 母集団は調査対象の全体、標本は選び出された一部

標本調査の理由： コスト、時間、破壊検査などの理由で全数調査が困難

無作為抽出： すべての個体が等しい確率で選ばれることが重要

点推定： 1つの値で母集団の特性を推測する方法（誤差不明が欠点）

区間推定： 範囲で推測し、信頼水準（通常95%）で確信度を示す

信頼区間の解釈： 「この方法で100回調査したら95回は真の値を含む」という意味

理解度確認クイズ

  1. Q1. 母集団と標本の説明として正しいものはどれでしょうか？

    回答を表示

    正解： 母集団は調査対象の全体（例: 日本国民全体）、標本は母集団から選び出された一部（例: 調査回答者1,000人）です。 標本から母集団の特性を推測することを推測統計と呼びます。

  2. Q2. 点推定と区間推定の違いは何でしょうか？

    回答を表示

    正解： 点推定は1つの値で推測する方法（例: 平均171.5cm）、区間推定は範囲で推測する方法（例: 170.5～172.5cm）です。 点推定はシンプルですが誤差が不明、区間推定は範囲が曖昧ですが信頼性が分かるという違いがあります。

  3. Q3. 「95%信頼区間が170～180cm」の正しい解釈はどれでしょうか？

    回答を表示

    正解： 「この方法で100回調査を繰り返したら、95回は真の母集団平均がこの範囲（170～180cm）に含まれる区間ができる」という意味です。 「母集団平均が170～180cmの範囲に入る確率が95%」ではありません（母集団平均は固定値で確率的に変動しない）。

# 統計解析入門 学習ガイドライン

このガイドラインでは、統計解析の基礎を入門者向けに段階的に学習するためのカリキュラムを提供しています。数学が苦手な方でも理解できるよう、具体例を多用し、IT・ビジネスでの実践的な活用を意識した内容となっています。

## 前提条件

### 必要な環境
- 表計算ソフト（Excel、Google スプレッドシートなど）
- 電卓（スマートフォンの計算機アプリでも可）
- インターネット接続環境（オンライン統計ツール利用時）

### 参考リソース
- [統計WEB - 統計学の基礎を学ぶ](https://bellcurve.jp/statistics/)
- [総務省統計局 - なるほど統計学園](https://www.stat.go.jp/naruhodo/)
- [Python公式 - statistics モジュール](https://docs.python.org/ja/3/library/statistics.html)

### 前提知識
- **必須**: 四則演算（足し算、引き算、掛け算、割り算）ができること
- **推奨**: 中学校レベルの数学（比率、割合、グラフの読み方）
- **推奨**: Excel等の表計算ソフトの基本操作

## 学習コンテンツ

### [1. 統計学とは何か - データで世界を理解する第一歩](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-01.html)
統計学がなぜ必要なのか、日常生活やビジネスでどのように活用されているかを具体例とともに学びます。「データに基づいた意思決定」の重要性を理解し、統計学の全体像を把握します。

### [2. データの種類と収集方法 - 正しいデータの見極め方](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-02.html)
量的データと質的データの違い、データの尺度（名義・順序・間隔・比率）を理解します。また、サンプリング手法（無作為抽出、層別抽出など）とバイアスの問題についても学びます。

### [3. 記述統計の基礎 - データの特徴をつかむ](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-03.html)
平均値、中央値、最頻値といった代表値の違いと使い分けを学びます。また、分散と標準偏差を通じてデータのばらつきを理解し、Excelでの計算方法も習得します。

### [4. データの可視化 - グラフで伝える技術](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-04.html)
ヒストグラム、棒グラフ、円グラフ、散布図、箱ひげ図など、各グラフの特徴と適切な使い分けを学びます。データの特性に合わせた効果的な可視化手法を身につけます。

### [5. 確率の基礎 - 不確実性を数値化する](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-05.html)
確率の基本概念（加法定理、乗法定理）をサイコロやトランプの例で直感的に理解します。条件付き確率やベイズの定理の入門的な内容も扱います。

### [6. 確率分布と正規分布 - データの「形」を知る](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-06.html)
二項分布、ポアソン分布の基礎と、統計学で最も重要な正規分布について学びます。標準化（z値への変換）と標準正規分布表の使い方を習得します。

### [7. 推測統計入門 - 一部から全体を推測する](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-07.html)
母集団と標本の関係、標本から母集団を推測する考え方を学びます。点推定と区間推定（信頼区間）の概念を身近な例で理解します。

### [8. 仮説検定の基礎 - データに基づいた判断の方法](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-08.html)
仮説検定の考え方（帰無仮説・対立仮説）、p値の意味、有意水準の設定方法を学びます。「統計的に有意」とはどういうことか、正しく理解できるようになります。

### [9. 実践的な検定手法 - よく使う統計テストをマスターする](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-09.html)
t検定（対応のある/ない場合）、カイ二乗検定、相関分析の基礎を学びます。各検定の使い分けと、Excelでの実施方法を習得します。

### [10. ビジネスでの統計活用 - データドリブンな意思決定](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/statistical-analysis/statistical-analysis-learning-material-10.html)
A/Bテスト、KPI分析、需要予測など、ビジネスの現場で統計がどのように活用されているかを学びます。データ分析プロジェクトの進め方と注意点も解説します。

## 学習の進め方

1. **順番に学習する**: 各章は前の章の知識を前提としているため、第1章から順番に進めてください
2. **具体例で理解する**: 各章で紹介される身近な例を通じて、抽象的な概念を具体的にイメージしましょう
3. **手を動かして計算する**: 電卓やExcelを使って実際に計算することで、理解が深まります
4. **練習問題に取り組む**: 各章末の練習問題で理解度を確認してください
5. **日常でデータを意識する**: ニュースや広告の数字を統計的な視点で見る習慣をつけましょう

## 学習目標

このガイドを完了すると、以下のスキルを身につけることができます：

- 統計学の基本用語（平均、分散、標準偏差、p値など）を正しく理解し説明できる
- データの種類に応じた適切な集計方法・可視化手法を選択できる
- 正規分布と確率の基礎を理解し、データの解釈に活用できる
- 仮説検定の考え方を理解し、統計的な主張の妥当性を判断できる
- Excelを使って基本的な統計分析（平均、標準偏差、t検定など）を実行できる
- ビジネスシーンでのデータ分析の基礎的なアプローチを理解できる

統計解析の実践：推測統計

記述統計から一歩進み、データの背後にある法則を探ります。仮説検定（t検定、カイ二乗検定：PROC TTEST, PROC FREQ）、分散分析（ANOVA）（PROC ANOVA / PROC GLM）、結果の解釈と統計的有意性の判断基準を学習します。

この章で学ぶこと

  • 仮説検定の基礎概念（帰無仮説・対立仮説、有意水準、p値）の理解
  • PROC TTESTによるt検定（1標本・2標本）の実施と結果の解釈
  • PROC FREQによるカイ二乗検定とクロス集計表の作成方法
  • PROC ANOVA / PROC GLMによる分散分析と多重比較の実行

8.1 仮説検定の基礎概念

仮説検定とは、データから得られた結果が偶然によるものなのか、それとも統計的に意味のある（有意な）差や関係性が存在するのかを判断する統計手法です。記述統計が「データの要約」であるのに対し、推測統計は「データの背後にある法則や傾向を推測」します。

仮説検定の基本用語

  • 帰無仮説（H0）: 差や関係性が「ない」という仮説（例: 平均値に差がない）
  • 対立仮説（H1）: 差や関係性が「ある」という仮説（例: 平均値に差がある）
  • 有意水準（α）: 誤って帰無仮説を棄却する確率の上限（通常0.05 = 5%）
  • p値: 帰無仮説が正しいと仮定した時に、観測された結果が偶然起こる確率
  • 判定基準: p値 ≤ α の場合、帰無仮説を棄却し対立仮説を採択

仮説検定の流れ

flowchart TD
    A["仮説設定
帰無仮説 H0 と対立仮説 H1"] --> B["有意水準の設定
通常 α=0.05"]
    B --> C["統計量の計算
t値、カイ二乗値など"]
    C --> D["p値の算出
観測された統計量が偶然起こる確率"]
    D --> E{"p値 ≤ α ?"}
    E -->|はい| F["帰無仮説を棄却
統計的に有意な差がある"]
    E -->|いいえ| G["帰無仮説を棄却できない
有意な差があるとは言えない"]

注意: 「有意でない」≠「差がない」

帰無仮説を棄却できない場合、「統計的に有意な差があるとは言えない」のであって、「差が存在しない」ことを証明したわけではありません。サンプルサイズが小さい場合など、検出力が不足している可能性があります。

8.2 t検定（PROC TTEST）

t検定は、連続変数の平均値の差を検定する最も基本的な手法です。SASではPROC TTESTを使用して、1標本t検定（特定の値との比較）や2標本t検定（2つのグループの平均比較）を実行します。

例1: 1標本t検定（平均値が特定の値と異なるか）

製品の重量が規格値100gと有意に異なるかを検定する例です。

1標本t検定.sas 
/* サンプルデータ: 製品重量 */
DATA product_weight;
    INPUT weight @@;
    DATALINES;
98.5 101.2 99.8 100.5 102.1 99.3 100.8 98.9 101.5 99.7
;
RUN;

/* 1標本t検定: 平均が100gと異なるか */
PROC TTEST DATA=product_weight H0=100;
    VAR weight;  /* 検定する変数 */
RUN;

出力結果の見方

                        t検定

変数: weight

            N      平均    標準偏差  標準誤差
           10   100.23      1.08      0.34

平均値   t値   自由度   Pr > |t|
100.23  0.67     9       0.5199

結果の解釈: p値（Pr > |t|）が0.5199と0.05より大きいため、帰無仮説「平均重量は100g」を棄却できません。つまり、製品重量は規格値100gと統計的に有意な差はないと判断されます。

例2: 2標本t検定（2つのグループの平均比較）

新薬と従来薬の効果（回復日数）に差があるかを検定します。

2標本t検定.sas 
/* サンプルデータ: 薬の効果 */
DATA drug_effect;
    INPUT drug $ recovery_days;
    DATALINES;
New 5
New 6
New 4
New 5
New 7
Old 8
Old 9
Old 7
Old 10
Old 8
;
RUN;

/* 2標本t検定: 新薬と従来薬の比較 */
PROC TTEST DATA=drug_effect;
    CLASS drug;          /* グループ分け変数 */
    VAR recovery_days;   /* 検定する変数 */
RUN;

t検定の前提条件

t検定は正規分布を仮定しています。サンプルサイズが小さい場合（n < 30）は、データが正規分布に従うかを確認すべきです（PROC UNIVARIATEの正規性検定、QQプロット等）。また、2標本t検定では等分散性も確認します（Levene検定）。

8.3 カイ二乗検定（PROC FREQ）

カイ二乗検定は、カテゴリカル変数（質的変数）間の関連性を検定します。例えば、性別と商品購入の有無の間に関係があるか、といった分析に使います。SASではPROC FREQのCHISQオプションを使用します。

例3: 独立性の検定（クロス集計表）

性別とキャンペーン応募の有無に関連があるかを検定します。

カイ二乗検定.sas 
/* サンプルデータ */
DATA campaign;
    INPUT gender $ applied $ count;
    DATALINES;
Male Yes 120
Male No 80
Female Yes 90
Female No 110
;
RUN;

/* カイ二乗検定 */
PROC FREQ DATA=campaign;
    TABLES gender * applied / CHISQ;  /* クロス集計とカイ二乗検定 */
    WEIGHT count;  /* 度数を指定 */
RUN;

出力結果の見方

カイ二乗検定

統計量                  値     自由度   Pr > ChiSq
カイ二乗              7.62      1        0.0058
尤度比カイ二乗        7.68      1        0.0056
Mantel-Haenszel       7.60      1        0.0058

結果の解釈: p値が0.0058と0.05より小さいため、帰無仮説「性別とキャンペーン応募に関連がない」を棄却します。つまり、性別とキャンペーン応募には統計的に有意な関連があると判断されます。

カイ二乗検定の前提条件

各セルの期待度数が5以上であることが望ましいです。期待度数が小さい場合は、Fisherの正確検定を使用します（PROC FREQのFISHERオプション）。

8.4 分散分析（ANOVA）

分散分析（ANOVA）は、3つ以上のグループの平均値を同時に比較する手法です。t検定は2群の比較ですが、ANOVAは複数群を一度に検定できます。SASではPROC ANOVAまたはPROC GLMを使用します。

PROC ANOVA vs PROC GLM

  • PROC ANOVA: バランス型データ（各群のサンプルサイズが同じ）に最適、計算が高速
  • PROC GLM: アンバランス型データにも対応、より柔軟（多重比較、共分散分析など）
  • 実務ではPROC GLMを使うことが多い（データがアンバランスであることが多いため）

例4: 一元配置分散分析

3つの異なる施肥方法（A, B, C）で作物の収量に差があるかを検定します。

分散分析.sas 
/* サンプルデータ: 施肥方法と収量 */
DATA fertilizer;
    INPUT method $ yield;
    DATALINES;
A 45
A 47
A 46
B 52
B 54
B 53
C 48
C 50
C 49
;
RUN;

/* 一元配置分散分析 */
PROC GLM DATA=fertilizer;
    CLASS method;        /* 要因（グループ分け変数） */
    MODEL yield = method;  /* 従属変数 = 要因 */
    MEANS method / TUKEY;  /* Tukeyの多重比較 */
RUN;

出力結果の見方

                    分散分析表

要因        自由度    平方和    平均平方    F値    Pr > F
method        2      48.00      24.00     8.73   0.0148
誤差          6      16.50       2.75
修正全体      8      64.50

結果の解釈: p値が0.0148と0.05より小さいため、帰無仮説「3つの施肥方法の収量平均は等しい」を棄却します。つまり、施肥方法によって収量に有意な差があると判断されます。

多重比較（どのグループ間に差があるか）

ANOVAで有意差が出た場合、次に「どのグループとどのグループの間に差があるのか」を調べる必要があります。これを多重比較といい、Tukeyの方法が広く使われます。 

Tukeyの多重比較 (有意水準0.05)

比較          平均値の差    95%信頼区間      有意差
B - A            7.0      [1.2, 12.8]        *
C - A            2.0      [-3.8, 7.8]
C - B           -5.0      [-10.8, 0.8]

結果の解釈: 施肥方法BはAよりも収量が有意に高いことがわかります（信頼区間が0を含まない）。

ANOVAの前提条件

ANOVAは、①各群が正規分布に従う、②各群の分散が等しい（等分散性）、③各観測値が独立、という前提条件があります。等分散性はLevene検定（HOVTEST=LEVENEオプション）で確認できます。

実習 8-1: 統計解析の総合演習

以下のシナリオに基づいて、適切な統計手法を選択し、仮説検定を実施してください。

シナリオ

新しい学習プログラム（方法A, B, C）の効果を検証します。各プログラムで学習した生徒のテストスコア（100点満点）を比較し、プログラム間に差があるかを調べます。

手順

  1. 1 サンプルデータを作成（各プログラムで5名ずつ、計15名のスコア）
  2. 2 PROC GLMを使って一元配置分散分析を実施
  3. 3 有意差が出た場合、Tukeyの多重比較でどのプログラム間に差があるかを特定
  4. 4 結果を解釈し、どのプログラムが最も効果的かを結論付ける

ヒント

/* サンプルデータ例 */
DATA learning;
    INPUT program $ score;
    DATALINES;
A 75
A 78
...
;

/* 分散分析と多重比較 */
PROC GLM DATA=learning;
    CLASS program;
    MODEL score = program;
    MEANS program / TUKEY;
RUN;

理解度確認クイズ

  1. Q1. p値が0.03の場合、有意水準0.05で帰無仮説をどのように判定しますか？

    回答を表示

    p値（0.03）が有意水準（0.05）より小さいため、帰無仮説を棄却します。つまり、統計的に有意な差があると判断されます。

  2. Q2. t検定とANOVAの使い分けは？

    回答を表示

    t検定は2つのグループの平均値を比較する場合に使用します。ANOVAは3つ以上のグループの平均値を同時に比較する場合に使用します。

  3. Q3. カイ二乗検定はどのようなデータに使用しますか？

    回答を表示

    カイ二乗検定はカテゴリカル変数（質的変数、例: 性別、商品カテゴリ）間の関連性を検定する場合に使用します。連続変数（身長、体重など）にはt検定やANOVAを使用します。

第8章のまとめ

  • 仮説検定により、観測されたデータの差が偶然なのか統計的に有意なのかを判断できる。
  • PROC TTESTで連続変数の平均値の差を検定（1標本、2標本）。
  • PROC FREQでカテゴリカル変数間の関連性をカイ二乗検定で確認。
  • PROC GLMで3つ以上のグループの平均を分散分析（ANOVA）で比較し、多重比較で詳細を調査。
  • 統計的有意性だけでなく、実務的な意味（効果の大きさ）も考慮して結果を解釈することが重要。

ODS (Output Delivery System) プログラミング

実務でそのまま使える、見栄えの良いレポートを自動出力します。ODS EXCELによる複数シートへの出力とセルの書式設定、ODS PDFでのレイアウト制御と目次作成、ODS SELECT / EXCLUDEによるプロシジャ結果の特定部分だけの抽出を習得します。

この章で学ぶこと

  • ODS EXCELによる複数シートへの分割出力とセルの書式設定方法
  • ODS PDFでのページレイアウト制御と自動目次生成の実装
  • ODS SELECT/EXCLUDEによるプロシジャ出力の選択的抽出テクニック

7.1 ODS（Output Delivery System）の概要

ODS（Output Delivery System）は、SASの分析結果を様々な形式（HTML、PDF、Excel、RTFなど）で出力するための強力な仕組みです。単なるデータ出力にとどまらず、書式設定、複数シート分割、レイアウト制御など、実務で即使えるレポート生成を自動化できます。

ODSの基本構文

ODS 出力先 FILE="ファイルパス" オプション;
（SASプロシジャの実行）
ODS 出力先 CLOSE;
出力先としてEXCEL、PDF、HTML、RTFなどを指定します。

ODSで出力できる形式

出力形式 主な用途 特徴
ODS EXCEL Excelレポート、データ分析 複数シート対応、セル書式設定可能
ODS PDF 印刷用レポート、プレゼン資料 目次自動生成、ページレイアウト制御
ODS HTML Webレポート、社内共有 ブラウザで閲覧可能、CSS適用可能
ODS RTF Word文書、報告書 Wordで編集可能

7.2 ODS EXCELによる複数シート出力と書式設定

ODS EXCELは、分析結果を直接Excelファイルに出力できます。複数のプロシジャ結果を異なるシートに分割したり、ヘッダーやセルの書式を指定したりすることで、実務でそのまま提出できるレベルのレポートを自動生成できます。

例1: 基本的なExcel出力

基本Excel出力.sas 
/* サンプルデータ */
DATA sales;
    INPUT region $ product $ sales;
    DATALINES;
North A 1000
North B 1500
South A 2000
South B 2500
;
RUN;

/* Excel出力の開始 */
ODS EXCEL FILE="/home/user/sales_report.xlsx";

/* 記述統計量 */
PROC MEANS DATA=sales N MEAN SUM;
    VAR sales;
    CLASS region product;
RUN;

/* Excel出力の終了 */
ODS EXCEL CLOSE;

例2: 複数シートに分割して出力

OPTIONS(SHEET_NAME='シート名')オプションで、出力するシート名を指定できます。

複数シート出力.sas 
/* Excel出力開始 */
ODS EXCEL FILE="/home/user/sales_multi_sheet.xlsx";

/* シート1: 地域別集計 */
ODS EXCEL OPTIONS(SHEET_NAME='地域別集計');
PROC MEANS DATA=sales N MEAN SUM;
    VAR sales;
    CLASS region;
RUN;

/* シート2: 商品別集計 */
ODS EXCEL OPTIONS(SHEET_NAME='商品別集計');
PROC MEANS DATA=sales N MEAN SUM;
    VAR sales;
    CLASS product;
RUN;

/* シート3: 詳細データ */
ODS EXCEL OPTIONS(SHEET_NAME='詳細データ');
PROC PRINT DATA=sales;
RUN;

ODS EXCEL CLOSE;

例3: セルの書式設定

STYLEオプションで、セルの背景色、フォント、枠線などを設定できます。

書式設定付きExcel.sas 
/* スタイル定義付きExcel出力 */
ODS EXCEL FILE="/home/user/sales_styled.xlsx"
    OPTIONS(SHEET_NAME='売上レポート'
            FROZEN_HEADERS='YES'        /* ヘッダー行を固定 */
            AUTOFILTER='YES');          /* 自動フィルタ有効化 */

PROC PRINT DATA=sales NOOBS
    STYLE(HEADER)={BACKGROUND=lightblue FONT_WEIGHT=bold}
    STYLE(DATA)={BACKGROUND=white};
RUN;

ODS EXCEL CLOSE;

便利なODS EXCELオプション

  • FROZEN_HEADERS='YES' - ヘッダー行を固定
  • AUTOFILTER='YES' - 自動フィルタを有効化
  • SHEET_INTERVAL='PROC' - プロシジャごとに自動でシート分割

7.3 ODS PDFでのレイアウト制御と目次作成

ODS PDFは、印刷やプレゼンテーションに適した高品質なPDFレポートを生成します。ページサイズ、余白、目次の自動生成など、細かいレイアウト制御が可能です。

例4: 基本的なPDF出力

基本PDF出力.sas 
/* PDF出力の開始 */
ODS PDF FILE="/home/user/sales_report.pdf";

PROC MEANS DATA=sales N MEAN SUM;
    TITLE "売上データ集計レポート";
    VAR sales;
    CLASS region product;
RUN;

PROC PRINT DATA=sales;
    TITLE "詳細データ一覧";
RUN;

ODS PDF CLOSE;

例5: 目次付きPDF出力

PDFTOC=1オプションで、TITLE文の内容を自動的に目次に反映できます。

目次付きPDF.sas 
/* 目次付きPDF出力 */
ODS PDF FILE="/home/user/sales_with_toc.pdf"
    PDFTOC=1                   /* 目次レベル1を有効化 */
    BOOKMARKLIST='YES';        /* ブックマークリストを表示 */

/* セクション1 */
ODS PROCLABEL "1. 地域別分析";
PROC MEANS DATA=sales N MEAN SUM;
    TITLE "1. 地域別売上分析";
    VAR sales;
    CLASS region;
RUN;

/* セクション2 */
ODS PROCLABEL "2. 商品別分析";
PROC MEANS DATA=sales N MEAN SUM;
    TITLE "2. 商品別売上分析";
    VAR sales;
    CLASS product;
RUN;

/* セクション3 */
ODS PROCLABEL "3. 詳細データ";
PROC PRINT DATA=sales;
    TITLE "3. 全売上データ詳細";
RUN;

ODS PDF CLOSE;

例6: ページレイアウトの制御

レイアウト制御PDF.sas 
/* ページ設定を細かく制御 */
ODS PDF FILE="/home/user/sales_layout.pdf"
    STARTPAGE=NO              /* プロシジャごとに改ページしない */
    STYLE=Journal;            /* スタイルテンプレートを指定 */

OPTIONS ORIENTATION=LANDSCAPE; /* 横向き */

PROC PRINT DATA=sales;
    TITLE "売上データ一覧（横向き）";
RUN;

OPTIONS ORIENTATION=PORTRAIT;  /* 縦向きに戻す */

ODS PDF CLOSE;

ODS PDF主要オプション

  • PDFTOC=1 - 目次を有効化（1〜4のレベル指定可能）
  • STARTPAGE=NO - プロシジャごとの改ページを無効化
  • STYLE=スタイル名 - スタイルテンプレート指定

7.4 ODS SELECT / EXCLUDEによる出力制御

SASのプロシジャは、通常複数の結果テーブルを出力します。ODS SELECTとODS EXCLUDEを使うことで、必要な結果だけを抽出したり、不要な結果を除外したりできます。

ODS SELECT/EXCLUDEの構文

ODS SELECT テーブル名1 テーブル名2; - 指定した結果のみ出力
ODS EXCLUDE テーブル名1 テーブル名2; - 指定した結果を除外
プロシジャが出力するテーブル名は、ODS TRACE ON;で確認できます。

例7: 出力テーブル名の確認

出力テーブル確認.sas 
/* どのテーブルが出力されるか確認 */
ODS TRACE ON;

PROC MEANS DATA=sales N MEAN SUM;
    VAR sales;
    CLASS region;
RUN;

ODS TRACE OFF;
Output Added:
-------------
Name:       Summary
Label:      Summary statistics
Template:   base.summary
Path:       Means.Summary
-------------

→ このプロシジャではSummaryというテーブルが出力されます。

例8: 特定の結果のみ出力

ODS SELECT使用.sas 
/* PROC REGでは複数の結果が出力されるが、FitStatisticsのみ抽出 */
DATA sample;
    INPUT x y;
    DATALINES;
1 2
2 4
3 5
4 7
5 8
;
RUN;

ODS SELECT FitStatistics ParameterEstimates;

PROC REG DATA=sample;
    MODEL y = x;
RUN;
QUIT;

ODS SELECT ALL; /* デフォルトに戻す */

例9: 特定の結果を除外

ODS EXCLUDE使用.sas 
/* PROC FREQで一部の結果を非表示 */
DATA survey;
    INPUT gender $ age_group $;
    DATALINES;
M 20s
F 30s
M 20s
F 20s
M 30s
;
RUN;

ODS EXCLUDE CrossTabFreqs; /* クロス集計表を非表示 */

PROC FREQ DATA=survey;
    TABLES gender * age_group / CHISQ;
RUN;

ODS EXCLUDE NONE; /* デフォルトに戻す */

実務での活用例

大量の分析を実行する際、ログが膨大になることがあります。ODS EXCLUDEで不要な出力を抑制することで、ログが見やすくなり、処理速度も向上します。

実習 7-1: 複数シートのExcelレポート作成

売上データから「地域別集計」「商品別集計」「詳細データ」の3つのシートを持つExcelファイルを生成してください。

手順

  1. 1 サンプルデータを作成します（地域、商品、売上金額）。
  2. 2 ODS EXCELでファイルを開き、シート名を指定して各集計を出力します。
  3. 3 FROZEN_HEADERS='YES'とAUTOFILTER='YES'を設定します。

ヒント

ODS EXCEL FILE="/home/user/report.xlsx";

ODS EXCEL OPTIONS(SHEET_NAME='地域別集計');
PROC MEANS DATA=sales;
    CLASS region;
    VAR sales;
RUN;

ODS EXCEL CLOSE;

動作確認 7-1: 目次付きPDFレポート生成

ODS PDFで目次（PDFTOC=1）とブックマークリスト（BOOKMARKLIST='YES'）を有効にしたレポートを作成し、PDFビューアで目次が正しく表示されることを確認してください。

理解度確認クイズ

  1. Q1. ODS EXCELで複数シートに分割するためのオプションは何ですか？

    回答を表示

    ODS EXCEL OPTIONS(SHEET_NAME='シート名');を使って、各プロシジャの出力先シートを指定します。

  2. Q2. ODS PDFで自動的に目次を生成するためのオプションは何ですか？

    回答を表示

    PDFTOC=1（または2、3、4）で目次レベルを指定し、BOOKMARKLIST='YES'でブックマークリストを表示します。

  3. Q3. プロシジャが出力するテーブル名を確認するコマンドは何ですか？

    回答を表示

    ODS TRACE ON;をプロシジャの前に記述すると、ログにテーブル名が出力されます。確認後はODS TRACE OFF;で無効化します。

よくあるエラーと対処法

Excelファイルが開けない

原因： ODS EXCEL CLOSEを実行していない

解決策： 必ずODS EXCEL CLOSE;でファイルを閉じてください。閉じていない場合、ファイルが破損します。

PDF目次が表示されない

原因： TITLE文が設定されていない、またはPDFTOCオプションが無効

解決策： PDFTOC=1を有効にし、各プロシジャにTITLE文を設定してください。

ODS SELECTで指定したテーブルが出力されない

原因： テーブル名のスペルミス、または大文字小文字の違い

解決策： ODS TRACE ON;で正確なテーブル名を確認してください。

第7章のまとめ

  • ODS EXCELにより、複数シートへの分割やセル書式設定を含む高品質なExcelレポートを自動生成できる。
  • ODS PDFで、目次付きのプレゼンテーション品質のPDFレポートを作成できる。
  • ODS SELECT/EXCLUDEにより、プロシジャ出力の必要な部分だけを抽出し、レポートを整理できる。
  • これらの機能を組み合わせることで、実務で即使える自動レポート生成システムを構築できる。

第9章

統計プロシジャの基礎

PROC MEANSやPROC FREQを使った基本統計量の算出、データの要約と集計方法を習得します。

この章で学ぶこと

  • PROC MEANSで基本統計量（平均、標準偏差など）を算出する
  • PROC FREQで度数分布表とクロス集計表を作成する
  • CLASS文を使ってグループ別に統計量を算出する
  • 統計結果をデータセットに出力して再利用する

9.1 統計プロシジャとは

統計プロシジャ（Statistical Procedure）は、データの集計や統計分析を行うための特別なプロシジャです。データステップでデータを加工した後、統計プロシジャで分析結果を出力することで、データの特性を理解し、意思決定に役立てることができます。

flowchart LR A[生データ] --> B[データステップで加工] B --> C[PROC MEANS
基本統計量] B --> D[PROC FREQ
度数分布] C --> E[分析レポート] D --> E

統計プロシジャの特徴

統計プロシジャはデータを読み込んで処理しますが、元のデータを変更することはありません。集計結果や統計量は、画面に表示されるか、OUTPUTオプションで別のデータセットに保存されます。

9.2 PROC MEANSの基本

PROC MEANSは、数値データの基本統計量（平均、標準偏差、最小値、最大値など）を計算するプロシジャです。データの全体像を把握するために最もよく使われます。

基本構文

基本構文 
PROC MEANS DATA=データセット名 統計量オプション;
    VAR 変数名リスト;  /* 分析対象の数値変数 */
RUN;

実習 9-1: 基本統計量の算出

売上データの基本統計量を計算してみましょう。

手順

  1. 1 サンプルの売上データを作成します
sales_data.sas 
/* サンプルデータ作成 */
DATA sales;
    INPUT product $ sales;
    DATALINES;
A 150
A 180
A 160
B 200
B 220
B 210
C 100
C 120
C 110
;
RUN;
  1. 2 PROC MEANSで基本統計量を計算します
means_basic.sas 
/* 基本統計量を計算 */
PROC MEANS DATA=sales;
    VAR sales;  /* salesカラムを分析 */
RUN;

期待される出力

結果ウィンドウに以下のような統計量が表示されます： 

The MEANS Procedure

Analysis Variable : sales

N          Mean      Std Dev      Minimum      Maximum
9      161.11       41.60         100.00       220.00

デフォルト統計量の説明

統計量 意味 説明
N 度数 欠損値を除いたデータ件数
Mean 平均値 データの合計÷件数
Std Dev 標準偏差 データのばらつきの度合い
Minimum 最小値 データの最小値
Maximum 最大値 データの最大値

9.3 PROC MEANSのオプション

PROC MEANSでは、必要な統計量だけを指定して出力することができます。主な統計量オプションを紹介します。

主な統計量オプション

N

データ件数

MEAN

平均値

MEDIAN

中央値

STD

標準偏差

SUM

合計値

MIN MAX

最小値・最大値

means_options.sas 
/* 必要な統計量だけを指定 */
PROC MEANS DATA=sales N MEAN MEDIAN SUM;
    VAR sales;
RUN;

9.4 グループ別集計（CLASS文）

CLASS文を使うと、カテゴリ変数ごとにグループ分けして統計量を計算できます。商品別売上、地域別売上など、グループ別の比較に便利です。

means_class.sas 
/* 商品別に統計量を計算 */
PROC MEANS DATA=sales N MEAN MIN MAX;
    CLASS product;  /* product変数でグループ化 */
    VAR sales;
RUN;

CLASS文のポイント

CLASS文を使うと、自動的にグループごとの統計量と全体の統計量の両方が出力されます。複数の変数を指定して多段階のグループ化も可能です（例：CLASS region product;）。

9.5 PROC FREQの基本

PROC FREQは、カテゴリデータの度数分布表（頻度表）を作成するプロシジャです。各カテゴリが何回出現したか、全体に占める割合は何%かを確認できます。

基本構文

基本構文 
PROC FREQ DATA=データセット名;
    TABLES 変数名;  /* 度数分布表を作成する変数 */
RUN;

実習 9-2: 度数分布表の作成

商品ごとの販売回数を集計してみましょう。

freq_basic.sas 
/* 商品の度数分布表を作成 */
PROC FREQ DATA=sales;
    TABLES product;  /* product変数の度数分布 */
RUN;

出力例：

product    Frequency    Percent    Cumulative    Cumulative
                                    Frequency     Percent
A              3          33.33          3           33.33
B              3          33.33          6           66.67
C              3          33.33          9          100.00

出力の見方

Frequency（度数）は各カテゴリの出現回数、Percent（パーセント）は全体に占める割合です。Cumulative（累積）は、その行までの合計を示します。

9.6 クロス集計表の作成

PROC FREQでは、2つの変数を掛け合わせたクロス集計表（2×2表など）を作成できます。2つの変数の関連性を調べるときに使います。

crosstab_data.sas 
/* クロス集計用サンプルデータ */
DATA survey;
    INPUT gender $ satisfaction $;
    DATALINES;
Male Good
Male Good
Male Average
Female Good
Female Excellent
Female Excellent
Male Average
Female Good
;
RUN;

/* クロス集計表を作成 */
PROC FREQ DATA=survey;
    TABLES gender * satisfaction;  /* * で2変数をクロス集計 */
RUN;

クロス集計のポイント

「変数A * 変数B」の形式で指定します。出力には各セルの度数、行パーセント、列パーセント、全体パーセントが表示されます。NOPCTオプションで不要なパーセント表示を省略できます。

9.7 統計量のデータセット出力

PROC MEANSやPROC FREQの結果は、画面表示だけでなくデータセットに保存して再利用できます。これにより、統計結果を別のプログラムで使ったり、グラフ化したりできます。

output_dataset.sas 
/* PROC MEANSの結果をデータセットに保存 */
PROC MEANS DATA=sales NOPRINT;
    CLASS product;
    VAR sales;
    OUTPUT OUT=sales_summary  /* 出力先データセット */
           MEAN=avg_sales     /* 平均値をavg_salesに */
           N=count;           /* 件数をcountに */
RUN;

/* 結果を確認 */
PROC PRINT DATA=sales_summary;
RUN;

NOPRINTオプション

NOPRINTオプションを指定すると、画面への出力が抑制され、OUTPUTオプションで指定したデータセットだけが作成されます。大量データの処理時に出力を省略したい場合に便利です。

理解度確認クイズ

  1. Q1. PROC MEANSがデフォルトで出力する統計量を3つ挙げてください。

    回答を表示

    N（度数）、Mean（平均値）、Std Dev（標準偏差）、Minimum（最小値）、Maximum（最大値）の5つです。この中から任意の3つを答えてください。

  2. Q2. グループ別に統計量を計算するときに使う文は何ですか？

    回答を表示

    CLASS文です。例えば「CLASS product;」と記述すると、product変数の各カテゴリごとに統計量が計算されます。

  3. Q3. PROC FREQで2つの変数のクロス集計表を作成するにはどう書きますか？

    回答を表示

    TABLES 変数A * 変数B; のように、アスタリスク（*）で2つの変数を繋げます。例：「TABLES gender * age_group;」

第9章のまとめ

  • PROC MEANSで基本統計量（平均、標準偏差、最小値、最大値など）を計算できる
  • 統計量オプションで必要な統計量だけを指定できる（N MEAN MEDIAN SUMなど）
  • CLASS文を使うとグループ別に統計量を算出できる
  • PROC FREQで度数分布表とクロス集計表を作成できる
  • OUTPUTオプションで統計結果をデータセットに保存して再利用できる

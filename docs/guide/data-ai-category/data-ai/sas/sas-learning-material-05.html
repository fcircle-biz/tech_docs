第5章

変数の操作と計算

数値変数と文字変数の違い、新しい変数の作成、基本的な計算処理を学びます。実務でよく使う変数操作のテクニックを習得します。

この章で学ぶこと

  • 数値変数と文字変数の違いを理解する
  • 新しい変数を作成し、計算処理を行う
  • SASの主要な数値関数と文字関数を使いこなす
  • 変数の型変換（数値⇔文字）を習得する

5.1 数値変数と文字変数

SASには大きく分けて2種類のデータ型があります：数値変数（Numeric）と文字変数（Character）です。この違いを理解することは、SASプログラミングの基本中の基本です。

flowchart LR A[SAS変数] --> B[数値変数] A --> C[文字変数] B --> B1[整数: 100] B --> B2[小数: 3.14] B --> B3[日付: 2025-01-01
内部は数値] C --> C1[文字列: 東京都] C --> C2[ID: A00123] C --> C3[カテゴリ: 男性]

数値変数の特徴

  • 計算が可能

    加減乗除などの演算ができます（例: 売上 = 単価 * 数量）

  • 固定長（8バイト）

    基本的に8バイトで格納され、約15桁の精度を持ちます

  • 日付・時刻も数値

    SASでは日付も内部的には数値として扱われます（1960年1月1日からの経過日数）

文字変数の特徴

  • 文字列を格納

    名前、住所、IDなどのテキスト情報を扱います

  • 長さ指定が必要

    文字変数は長さを明示的に指定します（例: LENGTH name $ 20;）

  • $マークで識別

    文字変数には$マークが付きます（例: INPUT name $ age;）

変数の種類は最初の出現で決まる

SASでは、変数が最初に登場したときに型が決定されます。一度決まった型は変更できないため、注意が必要です。

5.2 新しい変数の作成と計算

データステップ内で、既存の変数を使って新しい変数を作成できます。これは実務で非常によく使う重要な機能です。

変数作成の基本 
/* 売上データから売上金額を計算 */
DATA sales;
    INPUT product $ quantity price;

    /* 新しい変数を作成: 売上金額 */
    amount = quantity * price;

    /* 消費税（10%）を計算 */
    tax = amount * 0.10;

    /* 合計金額を計算 */
    total = amount + tax;

    DATALINES;
りんご 10 100
みかん 20 80
ぶどう 5 200
;
RUN;

/* 結果を確認 */
PROC PRINT DATA=sales;
RUN;

代入演算子は「=」

新しい変数を作成するときは、「変数名 = 式;」の形式で記述します。右辺の計算結果が左辺の変数に代入されます。

5.3 主要な数値関数

SASには数値計算を便利にする関数が多数用意されています。ここでは実務でよく使う関数を紹介します。

関数名 説明 使用例 結果
ROUND 四捨五入 ROUND(3.7) 4
CEIL 切り上げ CEIL(3.1) 4
FLOOR 切り捨て FLOOR(3.9) 3
ABS 絶対値 ABS(-5) 5
SQRT 平方根 SQRT(16) 4
MAX 最大値 MAX(3, 7, 2) 7
MIN 最小値 MIN(3, 7, 2) 2
SUM 合計（欠損値を無視） SUM(10, ., 5) 15
数値関数の使用例 
/* 数値関数の実践例 */
DATA numbers;
    INPUT value;

    /* 四捨五入 */
    round_val = ROUND(value);

    /* 小数点第1位で四捨五入 */
    round_1 = ROUND(value, 0.1);

    /* 絶対値 */
    abs_val = ABS(value);

    DATALINES;
3.14159
-2.7
10.555
;
RUN;

PROC PRINT DATA=numbers;
RUN;

5.4 主要な文字関数

文字変数を操作するための関数も豊富に用意されています。文字列の結合、抽出、変換などができます。

関数名 説明 使用例 結果
UPCASE 大文字に変換 UPCASE('Hello') HELLO
LOWCASE 小文字に変換 LOWCASE('Hello') hello
SUBSTR 部分文字列抽出 SUBSTR('ABCDE', 2, 3) BCD
LENGTH 文字列の長さ LENGTH('Tokyo') 5
TRIM 末尾の空白削除 TRIM('ABC ') ABC
STRIP 前後の空白削除 STRIP(' ABC ') ABC
COMPRESS 指定文字削除 COMPRESS('A-B-C', '-') ABC
CATS 文字列結合（空白除去） CATS('A ', ' B') AB
文字関数の使用例 
/* 文字関数の実践例 */
DATA names;
    INPUT first_name $ last_name $;

    /* 全て大文字に変換 */
    upper_first = UPCASE(first_name);

    /* フルネームを作成（姓 名） */
    full_name = CATS(last_name, ' ', first_name);

    /* 名前の長さを取得 */
    name_len = LENGTH(first_name);

    DATALINES;
Taro Yamada
Hanako Suzuki
Ken Tanaka
;
RUN;

PROC PRINT DATA=names;
RUN;

5.5 変数の型変換

実務では、数値を文字列に変換したり、文字列を数値に変換したりする必要がよくあります。SASには型変換のための関数が用意されています。

flowchart LR A[数値変数
123] -- PUT関数 --> B[文字変数
'123'] C[文字変数
'456'] -- INPUT関数 --> D[数値変数
456]

数値 → 文字（PUT関数）

PUT関数の使用例 
/* 数値を文字列に変換 */
DATA convert1;
    num_val = 123;

    /* PUT関数で数値を文字に変換 */
    char_val = PUT(num_val, 8.);

    /* 出力フォーマットを指定可能 */
    char_val2 = PUT(num_val, 5.);
RUN;

PROC PRINT DATA=convert1;
RUN;

文字 → 数値（INPUT関数）

INPUT関数の使用例 
/* 文字列を数値に変換 */
DATA convert2;
    char_val = '456';

    /* INPUT関数で文字を数値に変換 */
    num_val = INPUT(char_val, 8.);

    /* 計算が可能になる */
    calc = num_val * 2;
RUN;

PROC PRINT DATA=convert2;
RUN;

つまずきポイント

PUT関数とINPUT関数は逆の働きをします。混同しやすいので注意しましょう。覚え方：「PUTは出力（文字化）」「INPUTは入力（数値化）」

実習 5-1: 顧客データの加工

顧客名簿データに対して、変数の操作と計算を実践してみましょう。

課題

以下のプログラムを完成させてください：

実習課題 
DATA customers;
    INPUT cust_id $ name $ age purchase_amount;

    /* 1. 名前を大文字に変換 */
    name_upper = UPCASE(name);

    /* 2. ポイント付与（購入金額の5%） */
    points = ROUND(purchase_amount * 0.05);

    /* 3. 年代（10歳刻み）を計算 */
    generation = FLOOR(age / 10) * 10;

    /* 4. 顧客IDの最初の1文字を抽出 */
    id_prefix = SUBSTR(cust_id, 1, 1);

    DATALINES;
A001 yamada 35 12000
B002 suzuki 42 8500
C003 tanaka 28 15300
A004 sato 51 20000
;
RUN;

PROC PRINT DATA=customers;
RUN;

ヒント

実行すると、元の変数に加えて、name_upper、points、generation、id_prefixの4つの新しい変数が追加されます。

理解度確認クイズ

  1. Q1. 数値変数と文字変数を区別する際、文字変数に付く記号は何ですか？

    回答を表示

    $（ドルマーク）です。例えば「INPUT name $ age;」では、nameが文字変数、ageが数値変数として定義されます。

  2. Q2. 3.7という数値を四捨五入して4にするには、どの関数を使いますか？

    回答を表示

    ROUND関数を使います。「result = ROUND(3.7);」と記述すると、resultには4が格納されます。

  3. Q3. 文字列'ABCDE'から'BCD'を抽出するには、どの関数を使いますか？

    回答を表示

    SUBSTR関数を使います。「result = SUBSTR('ABCDE', 2, 3);」と記述すると、2文字目から3文字分が抽出されます。

  4. Q4. 数値を文字列に変換する関数と、文字列を数値に変換する関数の組み合わせで正しいものはどれですか？

    回答を表示

    PUT関数（数値→文字）とINPUT関数（文字→数値）です。覚え方は「PUTで出力（文字化）、INPUTで入力（数値化）」です。

第5章のまとめ

  • SASには数値変数と文字変数の2種類があり、文字変数には$マークが付く
  • 新しい変数は「変数名 = 式;」の形式で作成できる
  • ROUND、CEIL、FLOOR、ABS、MAX、MIN、SUMなどの数値関数が使える
  • UPCASE、LOWCASE、SUBSTR、LENGTH、TRIM、CATSなどの文字関数が使える
  • PUT関数で数値→文字、INPUT関数で文字→数値の型変換ができる

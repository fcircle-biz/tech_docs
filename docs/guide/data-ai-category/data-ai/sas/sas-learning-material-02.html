はじめてのSASプログラム

最初のSASプログラムを作成し、データの表示と簡単な計算を行います。プログラムの基本構造とコメントの書き方を学びます。

この章で学ぶこと

  • SASプログラムの基本構造を理解する
  • 最初のプログラムを作成し実行する
  • コメントの書き方をマスターする
  • 簡単な計算と出力の方法を習得する

2.1 SASプログラムの基本構造

SASプログラムは、ステップ（Step）と呼ばれる単位で構成されます。各ステップは特定の処理を実行し、セミコロン（;）で文を区切り、RUN;またはQUIT;でステップを終了します。

flowchart TD A[SASプログラム] --> B[DATAステップ] A --> C[PROCステップ] B --> D[データの作成・加工] C --> E[データの分析・出力] D --> F[RUN;で終了] E --> G[RUN; または QUIT;で終了]

2種類のステップ

1
DATAステップ

データの読み込み、作成、加工を行うステップ。DATA文で始まります。

2
PROCステップ

データの分析、集計、出力を行うステップ。PROC文で始まります。

重要ポイント

SASプログラムでは、すべての文をセミコロン（;）で終わらせる必要があります。セミコロンを忘れると、エラーの原因になります。

2.2 最初のプログラム

まずは最もシンプルなプログラムから始めましょう。ログウィンドウにメッセージを出力するプログラムを作成します。

hello.sas 
/* 最初のSASプログラム */
DATA _NULL_;
    PUT 'こんにちは、SAS!';
RUN;

プログラムの解説

1
/* ... */

コメント。プログラムの説明を書くための部分で、実行には影響しません。

2
DATA _NULL_;

DATAステップの開始。_NULL_は「データセットを作成しない」ことを意味します。

3
PUT 'こんにちは、SAS!';

ログウィンドウにメッセージを出力します。

4
RUN;

DATAステップの終了と実行を指示します。

2.3 コメントの書き方

コメントは、プログラムの説明や注釈を記述するために使います。SASでは2種類のコメント記法があります。

comments.sas 
/* ブロックコメント（複数行可） */
/* これは
   複数行に
   わたるコメントです */

* 行コメント（1行のみ）;
* この行全体がコメントとして扱われます;

DATA _NULL_;
    PUT 'プログラム実行中'; /* 文の後ろにも書ける */
RUN;
記法 書き方 特徴
ブロックコメント /* コメント */ 複数行可、文の途中でも使用可
行コメント * コメント; 1行のみ、セミコロン必須

コメントのベストプラクティス

プログラムの先頭に、作成日、作成者、目的を書くと、後で見返したときに理解しやすくなります。複雑な処理の前にも説明コメントを書く習慣をつけましょう。

2.4 PUTステートメントによる出力

PUT文は、ログウィンドウにメッセージや変数の値を出力するための命令です。デバッグやプログラムの動作確認に非常に便利です。

put_example.sas 
/* PUT文のいろいろな使い方 */
DATA _NULL_;
    /* 文字列の出力 */
    PUT '=== SASプログラム開始 ===';

    /* 変数の値を出力 */
    name = 'Tanaka';
    age = 25;
    PUT '名前: ' name;
    PUT '年齢: ' age;

    /* 複数の値を同時に出力 */
    PUT '名前=' name ' 年齢=' age;
RUN;

ログウィンドウでの出力例

=== SASプログラム開始 ===
名前: Tanaka
年齢: 25
名前=Tanaka 年齢=25

2.5 簡単な計算の実行

SASでは、変数に値を代入したり、四則演算を行ったりできます。計算結果をPUT文で確認してみましょう。

calculation.sas 
/* 四則演算の例 */
DATA _NULL_;
    /* 変数に値を代入 */
    price = 1000;      /* 単価 */
    quantity = 5;      /* 数量 */

    /* 計算 */
    total = price * quantity;           /* 合計 */
    tax = total * 0.1;                  /* 消費税（10%） */
    total_with_tax = total + tax;       /* 税込合計 */

    /* 結果を出力 */
    PUT '商品単価: ' price '円';
    PUT '数量: ' quantity '個';
    PUT '小計: ' total '円';
    PUT '消費税: ' tax '円';
    PUT '税込合計: ' total_with_tax '円';
RUN;

四則演算の記号

演算 記号 例
足し算 + a = 10 + 5; → 15
引き算 - a = 10 - 5; → 5
掛け算 * a = 10 * 5; → 50
割り算 / a = 10 / 5; → 2
べき乗 ** a = 2 ** 3; → 8

実習 2-1: 最初のプログラムを書いてみよう

自己紹介プログラムを作成し、名前、年齢、趣味をログに出力してみましょう。

手順

  1. 1 SAS Studioを開き、新しいプログラムファイルを作成します
  2. 2 以下のプログラムを入力します（名前、年齢、趣味は自分の情報に変更してください）
practice_2_1.sas 
/* 自己紹介プログラム */
DATA _NULL_;
    /* 変数に値を代入 */
    name = '山田太郎';
    age = 25;
    hobby = 'プログラミング';

    /* 自己紹介を出力 */
    PUT '========== 自己紹介 ==========';
    PUT '名前: ' name;
    PUT '年齢: ' age '歳';
    PUT '趣味: ' hobby;
    PUT '============================';
RUN;
  1. 3 実行ボタン（または F3 キー）を押してプログラムを実行します
  2. 4 ログウィンドウに自己紹介が表示されることを確認します

チャレンジ課題

プログラムに「出身地」の情報も追加してみましょう。変数名はbirthplaceを使います。

実習 2-2: 給与計算プログラム

基本給と手当から月給を計算し、さらに年収を算出するプログラムを作成します。

practice_2_2.sas 
/* 給与計算プログラム */
DATA _NULL_;
    /* 入力データ */
    base_salary = 250000;    /* 基本給 */
    allowance = 30000;       /* 手当 */

    /* 月給計算 */
    monthly_salary = base_salary + allowance;

    /* 年収計算（12ヶ月分） */
    annual_salary = monthly_salary * 12;

    /* 結果を出力 */
    PUT '===== 給与計算結果 =====';
    PUT '基本給: ' base_salary '円';
    PUT '手当: ' allowance '円';
    PUT '月給: ' monthly_salary '円';
    PUT '年収: ' annual_salary '円';
RUN;

このプログラムを実行して、計算結果がログに正しく表示されることを確認しましょう。

理解度確認クイズ

  1. Q1. SASプログラムの各文を区切る記号は何ですか？

    回答を表示

    セミコロン（;）です。SASプログラムでは、すべての文をセミコロンで終わらせる必要があります。

  2. Q2. ログウィンドウにメッセージを出力するために使う命令は何ですか？

    回答を表示

    PUT文です。PUT文を使うことで、文字列や変数の値をログウィンドウに表示できます。

  3. Q3. SASで掛け算を表す記号は何ですか？

    回答を表示

    アスタリスク（*）です。例: result = 10 * 5; は 50 となります。

  4. Q4. DATAステップを終了し実行するための命令は何ですか？

    回答を表示

    RUN;です。DATAステップはRUN;で終了し、実行されます。

第2章のまとめ

  • SASプログラムはDATAステップとPROCステップで構成される
  • すべての文はセミコロン（;）で終わり、ステップはRUN;またはQUIT;で終了する
  • コメントは /* ... */ または * ...; の形式で記述できる
  • PUT文でログウィンドウにメッセージや変数の値を出力できる
  • 四則演算（+、-、*、/）と変数代入の基本を習得した

第3章

データの入力と読み込み

手入力によるデータ作成、CSVやExcelファイルの読み込み方法を学習します。実務でよく使うデータ取り込みパターンを習得します。

この章で学ぶこと

  • SASでデータを作成する3つの方法を理解する
  • DATALINES文を使った手入力でのデータ作成を習得する
  • CSVファイルとExcelファイルの読み込み方法を学ぶ
  • データの内容を確認する基本的な方法を身につける

3.1 データ作成の3つの方法

SASでデータを扱う方法には、主に以下の3つがあります。それぞれの特徴と使い分けを理解しましょう。

手入力

DATALINES文を使ってプログラム内に直接データを記述

用途: 少量のテストデータ、サンプルデータ作成

CSVファイル

PROC IMPORTでCSV形式のファイルを読み込み

用途: 外部システムからの出力データ、大量データ

Excelファイル

PROC IMPORTでExcelファイルを読み込み

用途: Excelで管理されているデータ、業務データ

flowchart LR A[データソース] --> B[手入力
DATALINES] A --> C[CSVファイル
PROC IMPORT] A --> D[Excelファイル
PROC IMPORT] B --> E[SASデータセット] C --> E D --> E E --> F[データ分析]

3.2 DATAステップによる手入力

まずは基本的な手入力の方法を学びましょう。プログラム内に直接データを記述する方法です。

INPUT文の役割

INPUT文は、データの列（変数）名と、その型（数値または文字）を定義します。文字型の変数名には「$」記号を付けます。

基本的な手入力の例 
/* 生徒の成績データを作成 */
DATA seiseki;
    INPUT seito_id name $ kokugo sugaku eigo;
    DATALINES;
101 Tanaka 85 90 78
102 Suzuki 92 88 95
103 Sato 78 82 80
104 Yamada 88 75 92
105 Watanabe 95 98 89
;
RUN;

/* データの確認 */
PROC PRINT DATA=seiseki;
RUN;

プログラムのポイント

  • DATA seiseki; - 「seiseki」という名前のデータセットを作成
  • INPUT文 - 変数名を定義（name $は文字型を意味）
  • DATALINES; - この後にデータを記述することを宣言
  • データ行 - スペースまたはタブで区切ってデータを入力
  • セミコロン(;) - データの終わりを示す

3.3 DATALINES文の書き方のルール

DATALINES文を使う際の重要なルールを覚えましょう。

DATALINES使用時のルール

1
データの区切り

データ項目はスペースまたはタブで区切ります。カンマは使いません（標準入力の場合）。

2
文字データ

文字データにスペースが含まれる場合は、引用符で囲む必要があります。

3
データの終了

データの最後にセミコロン(;)だけの行を入れて終了を示します。

4
欠損値

欠損値（データがない項目）はピリオド(.)で表します。

欠損値とスペースを含むデータの例 
/* 欠損値やスペースを含むデータ */
DATA customer;
    INPUT customer_id name $ age address $20.;
    DATALINES;
1001 "山田太郎" 28 "東京都渋谷区"
1002 "鈴木花子" . "大阪府大阪市"
1003 "佐藤次郎" 35 "愛知県名古屋市"
;
RUN;

ヒント

$20.は「20文字以内の文字型変数」を意味します。スペースを含む長い文字データを扱う場合に指定します。

3.4 CSVファイルの読み込み

実務では、外部システムから出力されたCSVファイルを読み込むことが多くあります。PROC IMPORTプロシジャを使ってCSVファイルを読み込む方法を学びましょう。

CSVファイルの読み込み 
/* CSVファイルを読み込む */
PROC IMPORT DATAFILE="/home/mydata/sales.csv"
            OUT=sales_data
            DBMS=CSV
            REPLACE;
    GETNAMES=YES;  /* 1行目を変数名として使用 */
RUN;

/* データの確認 */
PROC PRINT DATA=sales_data (OBS=10);
RUN;

PROC IMPORTのオプション説明

オプション 説明
DATAFILE 読み込むファイルのパス（絶対パスまたは相対パス）
OUT 作成するSASデータセットの名前
DBMS ファイル形式（CSV、XLSX、TXTなど）
REPLACE 同名のデータセットがある場合、上書きする
GETNAMES YES: 1行目を変数名とする / NO: 1行目もデータとする

注意: ファイルパスの指定

SAS OnDemand for Academicsでは、まずファイルをアップロードしてから読み込みます。ナビゲーションペインでフォルダを確認し、正しいパスを指定してください。

3.5 Excelファイルの読み込み

Excelで管理されているデータも、PROC IMPORTで簡単に読み込むことができます。シート名の指定方法も覚えましょう。

Excelファイルの読み込み（基本） 
/* Excelファイルを読み込む（1つ目のシートを読み込み） */
PROC IMPORT DATAFILE="/home/mydata/employees.xlsx"
            OUT=emp_data
            DBMS=XLSX
            REPLACE;
    GETNAMES=YES;
RUN;
特定のシートを指定して読み込み 
/* 特定のシート名を指定 */
PROC IMPORT DATAFILE="/home/mydata/sales_report.xlsx"
            OUT=sales_2024
            DBMS=XLSX
            REPLACE;
    SHEET="2024年度売上";  /* シート名を指定 */
    GETNAMES=YES;
RUN;

ヒント

シート名を指定しない場合、Excelファイルの最初のシートが自動的に読み込まれます。複数のシートがある場合は、SHEET=オプションで明示的に指定しましょう。

3.6 データの確認方法

データを読み込んだ後は、必ず内容を確認する習慣をつけましょう。主な確認方法を紹介します。

PROC PRINT

データの内容を表形式で表示します。最も基本的な確認方法です。

PROC PRINT DATA=sales;
RUN;

PROC CONTENTS

データセットの構造（変数名、型、件数など）を確認します。

PROC CONTENTS DATA=sales;
RUN;
データ確認の実践例 
/* 最初の10行だけ表示 */
PROC PRINT DATA=sales_data (OBS=10);
RUN;

/* データセットの構造を確認 */
PROC CONTENTS DATA=sales_data;
RUN;

/* 基本統計量を確認 */
PROC MEANS DATA=sales_data;
    VAR price quantity;  /* 数値変数を指定 */
RUN;

実習 3-1: 商品マスタデータの作成

手入力で商品マスタを作成し、データの確認を行いましょう。

課題

以下の商品データをDATALINES文を使って作成してください。

商品ID 商品名 価格 在庫数
P001ノートPC9800015
P002マウス250050
P003キーボード850030
P004モニター3500020
解答例を表示 
/* 商品マスタの作成 */
DATA product_master;
    INPUT product_id $ product_name $ price stock;
    DATALINES;
P001 ノートPC 98000 15
P002 マウス 2500 50
P003 キーボード 8500 30
P004 モニター 35000 20
;
RUN;

/* データの確認 */
PROC PRINT DATA=product_master;
RUN;

/* データ構造の確認 */
PROC CONTENTS DATA=product_master;
RUN;

理解度確認クイズ

  1. Q1. DATALINES文でデータを入力する際、データの終わりを示す記号は何ですか？

    回答を表示

    セミコロン(;)です。データ行の後に、セミコロンだけの行を入れることでデータの終了を示します。

  2. Q2. CSVファイルを読み込むときに使うプロシジャは何ですか？

    回答を表示

    PROC IMPORTです。DBMS=CSVオプションを指定することでCSVファイルを読み込むことができます。

  3. Q3. データセットの構造（変数名、型、件数など）を確認するためのプロシジャは何ですか？

    回答を表示

    PROC CONTENTSです。データセットの変数情報やメタデータを確認できます。

第3章のまとめ

  • データ作成には「手入力（DATALINES）」「CSV読み込み」「Excel読み込み」の3つの方法がある
  • DATALINES文では、INPUT文で変数を定義し、セミコロン(;)でデータの終了を示す
  • PROC IMPORTを使ってCSVやExcelファイルを読み込むことができる
  • データの確認にはPROC PRINT、PROC CONTENTS、PROC MEANSなどを使用する
  • 欠損値はピリオド(.)で表し、文字型変数はINPUT文で「$」を付けて定義する

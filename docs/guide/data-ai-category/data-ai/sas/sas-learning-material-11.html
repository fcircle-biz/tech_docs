データの可視化（グラフ作成）

PROC SGPLOTを使った各種グラフ（棒グラフ、折れ線グラフ、散布図）の作成方法を学びます。見やすいグラフ作成のポイントも解説します。

この章で学ぶこと

  • PROC SGPLOTプロシジャの基本構文を理解する
  • 棒グラフを作成し、カテゴリ別のデータを視覚化する
  • 折れ線グラフで時系列データの推移を表現する
  • 散布図で2つの変数間の関係性を分析する
  • グラフのタイトル、軸ラベル、色などのカスタマイズ方法を習得する

11.1 データの可視化とは

データの可視化（グラフ化）は、数値の羅列では見えにくいパターンや傾向、異常値などを直感的に理解するための重要な手法です。SASでは、PROC SGPLOTプロシジャを使うことで、簡単に美しく効果的なグラフを作成できます。

SGPLOTとは

SGPLOT = Statistical Graphics Plot の略。統計グラフ作成専用のプロシジャで、1つのプログラムで複数のグラフを重ね描きすることもできます。

flowchart LR A[生データ] --> B[PROC SGPLOT] B --> C[棒グラフ] B --> D[折れ線グラフ] B --> E[散布図] B --> F[その他のグラフ] C --> G[データの理解] D --> G E --> G F --> G

グラフ作成の3つのメリット

直感的理解

数千行のデータでも、一目で傾向や特徴を把握できます。

異常値の発見

外れ値やデータの偏りを視覚的に発見できます。

効果的な報告

プレゼンテーション資料やレポートに最適です。

11.2 PROC SGPLOTの基本構文

PROC SGPLOTの基本的な構文は以下の通りです。 

/* PROC SGPLOTの基本構文 */
PROC SGPLOT DATA=データセット名;
    グラフ種類 X=X軸変数 Y=Y軸変数 / オプション;
    /* タイトルやラベルの設定 */
    TITLE "グラフタイトル";
    XAXIS LABEL="X軸ラベル";
    YAXIS LABEL="Y軸ラベル";
RUN;

主要なグラフ種類

  • 1
    VBAR / HBAR

    縦棒グラフ / 横棒グラフ（カテゴリ別の集計値を表示）

  • 2
    SERIES

    折れ線グラフ（時系列データの推移を表示）

  • 3
    SCATTER

    散布図（2つの変数の関係性を表示）

  • 4
    HISTOGRAM

    ヒストグラム（データの分布を表示）

11.3 棒グラフの作成

棒グラフは、カテゴリ別のデータを比較するのに最適です。まず、サンプルデータを作成してから棒グラフを描いてみましょう。

実習 11-1: 商品別売上の棒グラフ

月別の商品売上データを作成し、棒グラフで視覚化します。

サンプルデータの作成

/* 商品別売上データの作成 */
DATA sales;
    INPUT product $ sales;
    DATALINES;
製品A 450
製品B 380
製品C 520
製品D 290
製品E 410
;
RUN;

棒グラフの作成

/* 縦棒グラフの作成 */
PROC SGPLOT DATA=sales;
    VBAR product / RESPONSE=sales;  /* 商品別に売上を表示 */
    TITLE "商品別売上（万円）";
    XAXIS LABEL="商品名";
    YAXIS LABEL="売上（万円）";
RUN;

コードの説明

  • VBAR product: productをX軸とする縦棒グラフ
  • RESPONSE=sales: salesの値を棒の高さに使用
  • TITLE: グラフのタイトルを設定
  • XAXIS / YAXIS LABEL: 各軸のラベルを設定

棒グラフのカスタマイズ

/* 棒の色を指定したグラフ */
PROC SGPLOT DATA=sales;
    VBAR product / RESPONSE=sales
                   FILLATTRS=(COLOR=steelblue);  /* 棒の色を指定 */
    TITLE "商品別売上（万円）";
    XAXIS LABEL="商品名";
    YAXIS LABEL="売上（万円）";
RUN;

11.4 折れ線グラフの作成

折れ線グラフは、時系列データの推移を見るのに最適です。月別売上の推移をグラフ化してみましょう。

実習 11-2: 月別売上の推移グラフ

1年間の月別売上データを作成し、折れ線グラフで推移を確認します。

サンプルデータの作成

/* 月別売上データの作成 */
DATA monthly_sales;
    INPUT month sales;
    DATALINES;
1 320
2 340
3 380
4 420
5 450
6 480
7 510
8 490
9 470
10 500
11 530
12 580
;
RUN;

折れ線グラフの作成

/* 折れ線グラフの作成 */
PROC SGPLOT DATA=monthly_sales;
    SERIES X=month Y=sales / MARKERS;  /* マーカー付き折れ線 */
    TITLE "月別売上推移";
    XAXIS LABEL="月";
    YAXIS LABEL="売上（万円）";
RUN;

MARKERSオプション

MARKERSオプションを指定すると、データポイントに丸印などのマーカーが表示され、各データ点が見やすくなります。

複数の折れ線を描く

複数の製品の売上推移を1つのグラフに描くこともできます。 

/* 製品別月次売上データ */
DATA product_sales;
    INPUT month product $ sales;
    DATALINES;
1 製品A 120
2 製品A 130
3 製品A 140
1 製品B 100
2 製品B 110
3 製品B 105
;
RUN;

/* 製品別の折れ線グラフ */
PROC SGPLOT DATA=product_sales;
    SERIES X=month Y=sales / GROUP=product MARKERS;
    TITLE "製品別売上推移";
    XAXIS LABEL="月";
    YAXIS LABEL="売上（万円）";
RUN;

11.5 散布図の作成

散布図は、2つの変数の関係性を視覚的に確認するためのグラフです。例えば、広告費と売上の関係を調べる場合などに使います。

実習 11-3: 広告費と売上の散布図

広告費と売上の関係を散布図で確認します。

サンプルデータの作成

/* 広告費と売上のデータ */
DATA advertising;
    INPUT ad_cost sales;
    DATALINES;
10 120
15 140
20 160
25 190
30 210
35 230
40 250
45 270
50 290
;
RUN;

散布図の作成

/* 散布図の作成 */
PROC SGPLOT DATA=advertising;
    SCATTER X=ad_cost Y=sales;  /* X軸=広告費、Y軸=売上 */
    TITLE "広告費と売上の関係";
    XAXIS LABEL="広告費（万円）";
    YAXIS LABEL="売上（万円）";
RUN;

散布図から分かること

点の並びを見ることで、「広告費が増えると売上も増える」という正の相関関係があるかどうかを視覚的に確認できます。

回帰直線を追加する

REG文を追加することで、散布図に回帰直線（トレンドライン）を重ね描きできます。 

/* 回帰直線付き散布図 */
PROC SGPLOT DATA=advertising;
    SCATTER X=ad_cost Y=sales;      /* 散布図 */
    REG X=ad_cost Y=sales / CLM;    /* 回帰直線と信頼区間 */
    TITLE "広告費と売上の関係（回帰直線付き）";
    XAXIS LABEL="広告費（万円）";
    YAXIS LABEL="売上（万円）";
RUN;

CLMオプション

CLM（Confidence Limits for Mean）オプションを指定すると、回帰直線の周りに信頼区間が薄い色で表示されます。

10.6 グラフの見た目をカスタマイズする

PROC SGPLOTでは、様々なオプションでグラフの見た目を調整できます。

主なカスタマイズオプション

オプション 用途 使用例
FILLATTRS 塗りつぶし色の指定 FILLATTRS=(COLOR=blue)
LINEATTRS 線の色・太さ・種類 LINEATTRS=(COLOR=red THICKNESS=2)
MARKERATTRS マーカーの形・色・サイズ MARKERATTRS=(SYMBOL=circle SIZE=10)
TRANSPARENCY 透明度の指定 TRANSPARENCY=0.3（0～1）

カスタマイズ例

/* カスタマイズしたグラフ */
PROC SGPLOT DATA=sales;
    VBAR product / RESPONSE=sales
                   FILLATTRS=(COLOR=coral)
                   BARWIDTH=0.6;  /* 棒の幅を調整 */
    TITLE "商品別売上（万円）" BOLD HEIGHT=14pt;
    XAXIS LABEL="商品名" LABELATTRS=(SIZE=12pt);
    YAXIS LABEL="売上（万円）" LABELATTRS=(SIZE=12pt);
RUN;

10.7 見やすいグラフ作成のポイント

グラフデザインのベストプラクティス

1
明確なタイトルをつける

グラフが何を示しているのか一目で分かるタイトルにしましょう。

2
軸ラベルと単位を必ず記載

X軸・Y軸が何を表し、単位は何か（円、kg、%など）を明記します。

3
適切なグラフタイプを選ぶ

カテゴリ比較なら棒グラフ、時系列なら折れ線、関係性なら散布図が基本です。

4
色は3～4色以内に抑える

色を使いすぎるとかえって見づらくなります。シンプルが一番です。

5
フォントサイズは読みやすく

プレゼン資料にする場合、タイトルは14pt以上、ラベルは12pt以上が目安です。

よくある間違い

  • • 3Dグラフや派手なエフェクトの多用（データが読み取りにくくなる）
  • • Y軸の範囲を意図的に調整して誇張表現をする
  • • 凡例なしで複数の線や色を使う

理解度確認クイズ

  1. Q1. PROC SGPLOTで棒グラフを描く際に使用する文は何ですか？

    回答を表示

    VBAR文（縦棒グラフ）またはHBAR文（横棒グラフ）を使用します。
    例: VBAR product / RESPONSE=sales;

  2. Q2. 時系列データの推移を表示するのに最も適したグラフは何ですか？

    回答を表示

    折れ線グラフ（SERIES文）です。時間の経過とともに値がどう変化したかを直感的に理解できます。

  3. Q3. 散布図に回帰直線を追加する場合、どの文を使いますか？

    回答を表示

    REG文を使います。
    例: REG X=ad_cost Y=sales / CLM;
    CLMオプションを付けると信頼区間も表示されます。

第11章のまとめ

  • PROC SGPLOTを使うことで、簡単に様々なグラフを作成できる
  • 棒グラフ（VBAR）はカテゴリ別のデータ比較に最適
  • 折れ線グラフ（SERIES）は時系列データの推移表示に適している
  • 散布図（SCATTER）で2つの変数の関係性を視覚化できる
  • FILLATTRS、LINEATTRS、MARKERATTRSでグラフの見た目を調整可能
  • 明確なタイトル、軸ラベル、適切な色使いが見やすいグラフの鍵

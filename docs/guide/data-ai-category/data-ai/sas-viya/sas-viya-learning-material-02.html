SAS Viya環境のセットアップと基本操作

SAS Viya for Learnersの登録・ログイン方法、Web UIの使い方を習得します。SAS Studio（Viya版）、SAS Drive、Environment Managerなど主要コンポーネントの操作を学びます。

この章で学ぶこと

  • SAS Viya for Learnersの登録方法を理解する
  • SAS Viya Web UIの構成と基本操作を習得する
  • SAS Studio（Viya版）でコードを実行する方法を学ぶ
  • SAS Driveでファイルやレポートを管理する方法を理解する
  • Environment Managerで環境情報を確認する方法を学ぶ

2.1 SAS Viya for Learnersとは

SAS Viya for Learnersは、SAS Instituteが学習目的で提供する無料のクラウド版SAS Viya環境です。14日間のトライアル期間があり、本番環境と同等の機能を体験できます。

SAS Viya for Learnersの特徴

無料で14日間利用可能（再登録で延長可）、クラウド環境でインストール不要、Visual Analytics・Model Studioなど主要機能を含む、5GBのストレージとサンプルデータセット提供

利用可能な主要機能

  • SAS Studio: WebベースのSAS開発環境
  • Visual Analytics: インタラクティブなダッシュボード作成
  • Model Studio: 機械学習モデルの構築と管理
  • SAS Drive: コンテンツ管理とファイル共有
  • Environment Manager: 環境設定と監視

2.2 SAS Viya for Learnersの登録とログイン

SAS Viya for Learnersを使い始めるための手順を説明します。

Step 1: アカウント登録

実習: SAS Viya for Learnersに登録する

  1. 1

    公式サイトにアクセス

    ブラウザで https://www.sas.com/ja_jp/software/viya-for-learners.html にアクセスします。

  2. 2

    登録フォームに入力

    「今すぐ始める」ボタンをクリックし、名前・メールアドレス・会社名などを入力します。

  3. 3

    メール認証

    登録したメールアドレスに送信される確認リンクをクリックします。

  4. 4

    環境プロビジョニング完了を待つ

    環境の準備には5〜10分程度かかります。完了するとログイン情報がメールで送信されます。

Step 2: 初回ログイン

環境プロビジョニング完了後、メールに記載されたURL（https://<your-environment>.sas.com）にアクセスします。

ログインフロー

注意事項

14日間のトライアル期間終了後、環境は自動的に削除されます。重要なファイルは期限前にエクスポートしてください。再登録により新しい環境を取得できます。

SAS Studioは、WebベースのSAS開発環境です。従来のSAS Studio（SAS 9.x版）と似た操作感ですが、Viya版ではCASエンジンとの連携が強化されています。

最初のSASコードを実行する

実習: Hello Worldを実行する

  1. 1

    SAS Studioを起動

    ホーム画面から「SAS Studio」アイコンをクリックします。

  2. 2

    新しいプログラムを作成

    ツールバーの「新規作成」→「SASプログラム」を選択します。

  3. 3

    コードを入力して実行

    以下のコードを入力し、「実行」ボタン（▶）をクリックします。

hello_world.sas 
/* SAS Viya初回実行テスト */
data work.hello;
  message = "Hello, SAS Viya!";
  output;
run;

/* データを表示 */
proc print data=work.hello;
run;

動作確認

実行後、以下を確認してください：

  • ログタブ: エラーがなく、「NOTE: データセットWORK.HELLOが作成されました。」と表示される
  • 結果タブ: テーブルに「Hello, SAS Viya!」が表示される

ヒント

SAS Studio（Viya版）では、Ctrl+Enter（Windows）またはCmd+Enter（Mac）でカーソル行のコードを実行できます。全体を実行する場合はF3キーを使用します。

2.5 SAS Driveによるコンテンツ管理

SAS Driveは、SAS Viya内のコンテンツを一元管理するファイル管理システムです。Google DriveやDropboxのような操作感で、レポート・データセット・プログラムを管理できます。

SAS Driveの主な機能

  • フォルダ管理: 階層構造でコンテンツを整理
  • 共有機能: チームメンバーとの共有・権限管理
  • バージョン管理: ファイルの履歴管理と復元
  • メタデータ検索: タグ・キーワード・作成者などで検索
  • インポート/エクスポート: ローカルファイルとの連携

基本操作

実習: SAS Driveでフォルダを作成する

  1. 1

    SAS Driveを開く

    ホーム画面から「SAS Drive」を選択します。

  2. 2

    マイフォルダに移動

    左側のナビゲーションから「マイフォルダ」を選択します。

  3. 3

    新しいフォルダを作成

    ツールバーの「新規作成」→「フォルダ」を選択し、「Learning_Projects」という名前で作成します。

  4. 4

    SAS Studioから保存先に指定

    今後、SAS Studioでプログラムを保存する際、この「Learning_Projects」フォルダを使用します。

2.6 Environment Managerによる環境確認

Environment Managerは、SAS Viya環境の設定・監視・管理を行うツールです。学習者はここでCASサーバーの状態、ライブラリ、ユーザー権限などを確認できます。

確認できる主な情報

CASサーバーの状態を確認する

実習: CASサーバー情報を確認する

  1. 1

    Environment Managerを開く

    アプリケーションメニューから「Environment Manager」を選択します。

  2. 2

    サーバーセクションに移動

    左ナビゲーションから「Servers」→「CAS Servers」を選択します。

  3. 3

    サーバー詳細を確認

    CASサーバーの稼働状態（Running/Stopped）、使用中のメモリ、セッション数などを確認します。

重要ポイント

SAS Viya for Learners環境では、管理者権限が制限されているため、一部の設定変更はできません。CASサーバーの状態確認やライブラリの参照は可能です。

2.7 その他の主要コンポーネント概要

SAS Viya環境には、今後の章で詳しく学ぶ主要コンポーネントがあります。ここではそれぞれの役割を簡単に紹介します。

Visual Analytics

ノーコードでインタラクティブなダッシュボードを作成できます。ドラッグ＆ドロップ操作で、グラフ・テーブル・地図などを配置し、リアルタイムにデータ探索ができます。

📚 詳細: 第6-7章

Model Studio

機械学習モデルの構築・評価・デプロイを一元管理できます。AutoML機能により、自動的に最適なモデルを選択・チューニングできます。

📚 詳細: 第8-9章

Visual Text Analytics

テキストデータのマイニング・感情分析・トピック抽出などを視覚的に実行できます。大量のテキストから洞察を引き出します。

📚 詳細: 第11章

Visual Forecasting

時系列データの予測分析を視覚的に実行できます。需要予測・売上予測などのビジネスシナリオに対応します。

理解度確認クイズ

  1. Q1. SAS Viya for Learnersのトライアル期間は何日間ですか？

    回答を表示

    14日間です。期間終了後、再登録により新しい環境を取得できます。

  2. Q2. SAS Studio（Viya版）でコードを実行するショートカットキーは何ですか？（Windows環境）

    回答を表示

    カーソル行の実行はCtrl+Enter、全体実行はF3です。

  3. Q3. SAS Driveの主な役割を説明してください。

    回答を表示

    SAS Viya内のコンテンツ（レポート・データセット・プログラム）を一元管理するファイル管理システムです。共有機能やバージョン管理も提供します。

  4. Q4. Environment Managerで確認できる主な情報を2つ挙げてください。

    回答を表示

    CASサーバーの稼働状態、サービスの稼働状況、リソース使用量、ライブラリ管理、ユーザー権限などが確認できます（2つ以上挙げられればOK）。

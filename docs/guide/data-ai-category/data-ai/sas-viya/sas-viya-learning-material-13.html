13.1 なぜPython/Rと連携するのか

SAS Viyaは強力なSAS言語を備えていますが、PythonやRといったオープンソース言語との連携により、さらに幅広い分析手法とライブラリを活用できます。

Python/R連携のメリット

  • 豊富なライブラリ: scikit-learn、TensorFlow、PyTorch、tidyverseなどを活用
  • データサイエンティストの生産性: 慣れた言語とツールを継続使用
  • 既存資産の活用: 既存のPython/Rコードを再利用可能
  • CASの高速処理: Python/RコードでもCASエンジンを活用し高速化
  • エンタープライズ機能: ガバナンス、セキュリティ、スケーラビリティを維持

重要ポイント

SAS ViyaはオープンソースとSASの「ベスト・オブ・ボス・ワールド」を実現します。データサイエンティストは慣れた言語を使いながら、エンタープライズグレードのインフラを活用できます。

13.2 SWATライブラリによるPython連携

SWAT（SAS Wrapper for Analytics Transfer）は、PythonからCASエンジンにアクセスするための公式ライブラリです。pandasライクなAPIで直感的に操作できます。

SWATのインストール

Terminal 
# pipでSWATをインストール
pip install swat

# 追加の依存関係（推奨）
pip install pandas matplotlib seaborn

CASへの接続

connect_cas.py 
import swat

# CASサーバーに接続
conn = swat.CAS(
    'your-viya-server.com',  # SAS Viyaサーバーのホスト名
    port=5570,               # CASポート（通常5570）
    username='your_username',
    password='your_password'
)

print(f"CAS接続成功: {conn}")
print(f"サーバー情報: {conn.about()}")

# 接続終了時
# conn.close()

ヒント

認証情報を環境変数に保存すると安全です。また、SAS Viya for Learnersの場合、接続情報はダッシュボードで確認できます。

データのロードと基本操作

data_operations.py 
import pandas as pd

# pandasデータフレームを作成
df = pd.DataFrame({
    'id': [1, 2, 3, 4, 5],
    'name': ['Alice', 'Bob', 'Carol', 'Dave', 'Eve'],
    'score': [85, 92, 78, 88, 95]
})

# CASテーブルとしてアップロード
cas_table = conn.upload_frame(
    df,
    casout={'name': 'students', 'replace': True}
)

print(f"CASテーブル作成: {cas_table}")

# CASテーブルのサマリー
print(cas_table.head())
print(cas_table.summary())

13.3 実習: PythonでCASアクションを実行する

実習 13-1: PythonでCAS上のデータを集計・可視化

PythonからCASアクションを実行してデータ集計を行い、結果をmatplotlibで可視化します。

手順

  1. 1. サンプルデータをCASにロード
    load_data.py 
    # SASHELPライブラリのCARSデータをCASにロード
conn.loadactionset('simple')
cars = conn.CASTable('cars', caslib='casuser')

# サンプルデータがない場合はアップロード
# conn.upload('sashelp.cars', casout='cars')
  2. 2. CASアクションで集計
    aggregate.py 
    # メーカー別の平均価格を集計
result = cars.groupby('Make').agg({'MSRP': 'mean'})
df_result = result.to_frame()  # pandasに変換

print(df_result.head(10))
  3. 3. matplotlibで可視化
    visualize.py 
    import matplotlib.pyplot as plt

# 上位10メーカーをプロット
top10 = df_result.nlargest(10, 'MSRP')
plt.barh(top10.index, top10['MSRP'])
plt.xlabel('平均価格（USD）')
plt.ylabel('メーカー')
plt.title('メーカー別平均車両価格（上位10）')
plt.tight_layout()
plt.show()

13.5 Rからswatパッケージで連携

R言語ユーザーもswatパッケージを使ってCASエンジンにアクセスできます。RStudioから直接CASテーブルを操作し、dplyrやggplot2と組み合わせて分析できます。

Rパッケージのインストール

R Console 
# swatパッケージをインストール
install.packages("swat")

# 依存関係
install.packages(c("dplyr", "ggplot2"))

RからCASへの接続

connect_cas.R 
library(swat)

# CASサーバーに接続
conn <- CAS(
  hostname = "your-viya-server.com",
  port = 5570,
  username = "your_username",
  password = "your_password"
)

# セッション情報を確認
cas.sessionProp.getSessOpts(conn)

# 接続終了時
# cas.close(conn)

RとdplyrでCASテーブルを操作

dplyr_cas.R 
library(dplyr)

# CASテーブルをdefCasTableで参照
cars_tbl <- defCasTable(conn, "cars")

# dplyr構文でCAS上のデータを集計
result <- cars_tbl %>%
  group_by(Make) %>%
  summarise(avg_price = mean(MSRP)) %>%
  arrange(desc(avg_price)) %>%
  head(10)

# ローカルに取得して表示
print(to.casDataFrame(result))

13.6 オープンソースモデルのSAS Viya上での実行

SAS Viyaは、PythonやRで作成した機械学習モデルをCAS上で実行し、Model Managerで管理できます。これにより、オープンソースモデルもエンタープライズ環境で運用可能です。

scikit-learnモデルの登録

register_sklearn_model.py 
from sklearn.ensemble import RandomForestClassifier
import pickle

# モデルをトレーニング
clf = RandomForestClassifier(n_estimators=100)
clf.fit(X_train, y_train)

# モデルをpickle化
with open('rf_model.pkl', 'wb') as f:
    pickle.dump(clf, f)

# SAS Model Managerにアップロード
# （SAS Model Managerのpythonライブラリ sasctl を使用）
from sasctl import Session
sess = Session('your-viya-server.com', 'username', 'password')
from sasctl.tasks import register_model
register_model(clf, 'RandomForestModel', 'MyProject')

重要ポイント

オープンソースモデルをSAS Viyaに登録すると、バージョニング、A/Bテスト、モデルモニタリング、再学習などのMLOps機能をすべて利用できます。

13.7 Python/R連携のベストプラクティス

データ処理はCASで

大量データの集計・フィルタリングはCASエンジンで実行し、結果だけをPython/Rに転送することで高速化できます。

認証情報の管理

接続情報を環境変数や設定ファイルに保存し、Gitにコミットしないようにします。.envファイルを活用しましょう。

セッション管理

長時間利用しない場合はCAS接続を閉じましょう。with文を使うと自動的にクローズできます。

並列処理の活用

CASは自動的に並列処理を行います。ローカルのループ処理をCASアクションに置き換えることで大幅な高速化が可能です。

理解度確認クイズ

  1. Q1. PythonからSAS ViyaのCASエンジンにアクセスするための公式ライブラリ名は何ですか?

    回答を表示

    SWAT（SAS Wrapper for Analytics Transfer）です。pipでインストールでき、pandasライクなAPIを提供します。

  2. Q2. SAS ViyaとPython/Rを連携させる主なメリットを3つ挙げてください。

    回答を表示

    ①豊富なオープンソースライブラリの活用、②データサイエンティストの生産性向上（慣れた言語の継続使用）、③CASエンジンによる高速処理（他にも複数の正解があります）

  3. Q3. RからCASテーブルを操作する際に使うパッケージ名は何ですか?

    回答を表示

    swatパッケージです。dplyrと組み合わせることで、R上でCASテーブルを直感的に操作できます。

  4. Q4. オープンソースのscikit-learnモデルをSAS Viyaに登録すると、どのような機能が利用できますか?

    回答を表示

    モデルのバージョニング、A/Bテスト、モデルモニタリング、再学習などのMLOps機能がすべて利用できます。エンタープライズグレードのモデル管理が可能になります。

第6章

SAS Visual Analyticsによるデータ可視化（基礎）

ノーコードでインタラクティブなダッシュボードを作成する方法を学びます。基本的なグラフタイプ、フィルター、計算項目の作成を習得します。

この章で学ぶこと

  • SAS Visual Analyticsの概要と起動方法を理解する
  • データソースの選択とデータ項目の追加方法を学ぶ
  • 基本的なグラフタイプ（棒グラフ、折れ線、円グラフ等）の作成方法を習得する
  • フィルター機能と計算項目の作成方法を学ぶ
  • レポート全体のデザインとレイアウトのカスタマイズ方法を理解する

6.1 SAS Visual Analyticsとは

SAS Visual Analyticsは、SAS Viyaプラットフォームの中核を成すデータ可視化・ダッシュボード作成ツールです。コーディング不要で、直感的なドラッグ＆ドロップ操作により、インタラクティブで美しいダッシュボードやレポートを作成できます。

Visual Analyticsの主な特徴

  • ノーコード操作: プログラミング知識不要で高度な可視化が可能
  • 高速レスポンス: CASエンジンによるインメモリ処理で瞬時に結果を表示
  • 豊富なグラフタイプ: 30種類以上のビジュアライゼーションを提供
  • インタラクティブ: クリック、ドリルダウン、フィルタリングが可能
  • レスポンシブ: PC、タブレット、スマートフォンで自動最適化
  • 共有機能: レポートをチーム内で簡単に共有・公開可能

Visual AnalyticsとVisual Statistics

SAS Viyaには「Visual Analytics」と「Visual Statistics」という2つのビジュアルツールがあります。前者はビジネスアナリスト向けのダッシュボード作成、後者はデータサイエンティスト向けの統計モデリングに特化しています。

6.2 Visual Analyticsの起動とレポート新規作成

SAS Viya環境にログイン後、Visual Analyticsを起動してレポートを新規作成します。

起動手順

実習 6-1: Visual Analyticsの起動とレポート新規作成

Visual Analyticsを起動し、新しいレポートを作成します。

手順

  1. 1 SAS Viya環境にログインし、左上のハンバーガーメニュー（≡）をクリック
  2. 2 メニューから「分析とレポート」→「Visual Analytics」を選択
  3. 3 画面左上の「+新規」ボタンをクリックし、「レポート」を選択
  4. 4 「データソースの選択」ダイアログが表示されるので、使用するデータテーブルを選択
  5. 5 「OK」をクリックして、レポート編集画面が開けば成功

ヒント

初めてレポートを作成する場合は、サンプルデータとして「SASHELP.CARS」や「SASHELP.STOCKS」を使うと学習しやすいです。これらはSAS Viyaにプリインストールされています。

各エリアの役割

① データペイン

使用可能なデータ項目（カテゴリ、メジャー、階層）が表示されます。キャンバスやプロパティペインにドラッグして使用します。

② オブジェクトペイン

棒グラフ、折れ線グラフ、テーブル、フィルター等の可視化オブジェクトが一覧表示されます。キャンバスにドラッグして配置します。

③ キャンバス

レポート作成のメインエリア。オブジェクトを配置・サイズ変更・レイアウト調整を行います。

④ プロパティペイン

選択したオブジェクトの詳細設定が可能。データ割り当て、色、ラベル、フィルター等を設定します。

6.4 基本的なグラフの作成

Visual Analyticsでは30種類以上のグラフタイプが用意されています。ここでは最も基本的なグラフの作成方法を学びます。

主要なグラフタイプ

グラフタイプ 用途 必要なデータ項目
棒グラフ カテゴリ間の比較 カテゴリ + メジャー
折れ線グラフ 時系列データの推移 日付 + メジャー
円グラフ 構成比率の表示 カテゴリ + メジャー
散布図 2変数の相関分析 メジャー（X軸） + メジャー（Y軸）
ヒートマップ 2次元データの密度表示 カテゴリ（行） + カテゴリ（列） + メジャー
テーブル 詳細データの一覧表示 任意のデータ項目

実習 6-2: 棒グラフの作成

SASHELP.CARSデータを使って、メーカー別の平均価格を棒グラフで可視化します。

手順

  1. 1 左側のオブジェクトペインから「棒グラフ」をキャンバスにドラッグ
  2. 2 データペインから「Make」（メーカー）を右側プロパティペインの「カテゴリ」にドラッグ
  3. 3 データペインから「MSRP」（価格）を「メジャー」にドラッグ
  4. 4 プロパティペインのメジャー項目をクリックし、「集計」を「平均」に変更
  5. 5 メーカー別の平均価格が棒グラフで表示されることを確認

重要ポイント

グラフにデータ項目を割り当てる際、カテゴリは「軸」、メジャーは「値」として扱われます。この概念を理解することで、あらゆるグラフタイプを自在に作成できるようになります。

6.5 フィルター機能によるデータの絞り込み

フィルター機能を使うことで、レポート上で特定の条件に合致するデータのみを表示できます。ユーザーがインタラクティブに条件を変更できるため、柔軟な分析が可能です。

フィルターの種類

  • ページレベルフィルター: レポート全体のオブジェクトに影響
  • オブジェクトレベルフィルター: 特定のオブジェクトのみに影響
  • テキスト検索フィルター: キーワードで絞り込み
  • 日付範囲フィルター: 期間指定で絞り込み

実習 6-3: リストコントロールフィルターの追加

車種タイプ（Type）でフィルタリングできるリストコントロールを追加します。

手順

  1. 1 オブジェクトペインから「リストコントロール」をキャンバスの上部にドラッグ
  2. 2 データペインから「Type」をリストコントロールにドラッグ
  3. 3 リストコントロールが表示され、「Sedan」「SUV」「Sports」等の選択肢が表示されることを確認
  4. 4 リストから「Sports」を選択し、棒グラフが自動的にフィルタリングされることを確認

ヒント

複数のフィルターを組み合わせることで、より詳細な分析が可能です。例えば「車種タイプ」と「エンジンタイプ」の2つのフィルターを設定すれば、クロス条件での絞り込みができます。

6.6 計算項目の作成

既存のデータ項目を元に、新しい計算項目を作成できます。計算項目を使うことで、データセットに存在しない指標を可視化できます。

計算項目の作成方法

実習 6-4: 価格帯の計算項目を作成

車両価格を「低価格」「中価格」「高価格」の3つのカテゴリに分類する計算項目を作成します。

手順

  1. 1 データペインの上部にある「+」ボタンをクリックし、「計算項目」を選択
  2. 2 計算項目エディタで「カテゴリ」を選択し、名前を「価格帯」と入力
  3. 3 「関数」から「IF」を選択し、以下のような条件式を入力（簡易UI使用も可）
  4. 4 「OK」をクリックして保存すると、データペインに「価格帯」が追加される
  5. 5 新しく作成した「価格帯」を棒グラフのカテゴリに割り当てて動作確認

計算式の例（参考）

IF 'MSRP'n < 20000 THEN "低価格"
ELSE IF 'MSRP'n < 40000 THEN "中価格"
ELSE "高価格"

重要ポイント

計算項目はレポート内で作成され、元のデータテーブルには保存されません。複数のレポートで同じ計算項目を使いたい場合は、データ準備段階でCASテーブルに列を追加する方が効率的です。

6.7 レポートのデザインとレイアウト

Visual Analyticsでは、グラフの見た目やレポート全体のデザインを細かくカスタマイズできます。

主要なカスタマイズ項目

  • 色の変更: グラフの色を個別に設定、またはカラーテーマを適用
  • ラベル設定: 軸ラベル、データラベル、タイトルの表示・非表示
  • 並び替え: 昇順・降順、カスタム順序の設定
  • フォント設定: フォントサイズ、色、太字・斜体の設定
  • 背景と枠線: オブジェクトの背景色、枠線の設定

デザインのベストプラクティス

色は多用せず、2〜3色程度に抑えると見やすくなります。また、データラベルを全て表示するとグラフが見づらくなるため、重要なポイントのみ表示する工夫が大切です。

6.8 レポートの保存と共有

作成したレポートはSAS Driveに保存し、チームメンバーと共有できます。

保存と共有の手順

実習 6-5: レポートの保存と共有

作成したレポートを保存し、他のユーザーと共有します。

手順

  1. 1 画面右上の「保存」ボタンをクリック
  2. 2 レポート名（例: 「車両分析ダッシュボード」）と説明を入力
  3. 3 保存先フォルダを選択し、「保存」をクリック
  4. 4 画面右上の「共有」ボタンをクリックし、共有するユーザーまたはグループを追加
  5. 5 権限（表示のみ / 編集可能）を設定し、「OK」をクリック

ヒント

レポートはPDF、PowerPoint、画像ファイルとしてエクスポートすることも可能です。「エクスポート」メニューから形式を選択してください。

第6章のまとめ

  • SAS Visual Analyticsはノーコードでインタラクティブなダッシュボードを作成できる強力なツールである
  • 画面は「データペイン」「オブジェクトペイン」「キャンバス」「プロパティペイン」の4つのエリアで構成される
  • ドラッグ＆ドロップ操作で棒グラフ、折れ線グラフ、円グラフ等の基本的なグラフを簡単に作成できる
  • フィルター機能により、ユーザーがインタラクティブにデータを絞り込める
  • 計算項目を作成することで、元のデータに存在しない指標を可視化できる
  • 作成したレポートはSAS Driveに保存し、チームで共有・エクスポートが可能

理解度確認クイズ

  1. Q1. Visual Analyticsで棒グラフを作成する際、必要な最小のデータ項目は何ですか？

    回答を表示

    「カテゴリ項目」1つと「メジャー項目」1つです。カテゴリが横軸（または縦軸）、メジャーが棒の高さを表します。

  2. Q2. レポート全体のオブジェクトに影響を与えるフィルターの名称は何ですか？

    回答を表示

    「ページレベルフィルター」です。これに対し、特定のオブジェクトのみに影響するものを「オブジェクトレベルフィルター」と呼びます。

  3. Q3. 計算項目で使用できる関数のタイプを3つ挙げてください。

    回答を表示

    ①条件分岐関数（IF-THEN-ELSE）、②算術関数（四則演算）、③文字列関数（SUBSTR、UPCASE等）。他にも日付関数、集計関数など多数あります。

  4. Q4. Visual Analyticsで作成したレポートを外部ユーザーと共有する方法を1つ挙げてください。

    回答を表示

    PDF、PowerPoint、画像ファイルとしてエクスポートする方法が一般的です。また、SAS Viyaのアカウントを持つユーザーには、レポートのURLリンクを共有することも可能です。

予測モデリングと時系列分析

回帰分析、分類モデル、時系列予測の実践方法を学びます。SAS Visual Forecastingを使った需要予測、売上予測のハンズオンを行います。

この章で学ぶこと

  • 回帰分析と分類モデルの基本概念を理解する
  • 時系列データの特性と前処理手法を学ぶ
  • SAS Visual Forecastingの操作方法を習得する
  • 需要予測・売上予測モデルを実際に構築する
  • 予測モデルの評価指標と結果の解釈方法を学ぶ

10.1 予測モデリングの基礎

予測モデリングは、過去のデータから将来の値を予測する手法です。ビジネスにおいて、売上予測、需要予測、リスク評価など幅広く活用されています。

予測モデルの種類

予測モデルは主に以下の2つに分類されます。

回帰モデル

連続値を予測するモデル。売上金額、気温、株価など数値を予測します。

代表的な手法:

  • • 線形回帰
  • • 決定木回帰
  • • ランダムフォレスト回帰
  • • ニューラルネットワーク

分類モデル

カテゴリを予測するモデル。顧客の解約有無、購入有無、信用リスクなどを予測します。

代表的な手法:

  • • ロジスティック回帰
  • • 決定木
  • • ランダムフォレスト
  • • 勾配ブースティング

重要ポイント

SAS Model Studioでは、回帰・分類の両方に対応した自動モデル選択（AutoML）機能があり、最適なアルゴリズムを自動的に提案してくれます。

10.2 時系列データの特性と前処理

時系列データは、時間順に並んだデータで、売上、株価、気温などが該当します。時系列データには特有の性質があり、適切な前処理が必要です。

時系列データの主な特性

  • トレンド: 長期的な増加・減少傾向
  • 季節性: 周期的に繰り返されるパターン（年次・月次・週次など）
  • 周期性: 景気循環など不規則な周期のパターン
  • ランダムノイズ: 予測不可能な不規則変動

時系列データの前処理手法

欠損値処理

時系列データでは、前後の値から補間（線形補間、スプライン補間）するのが一般的です。

異常値検出・除外

移動平均や標準偏差を用いて、通常の変動範囲から大きく外れた値を検出・除外します。

平滑化

移動平均や指数平滑化により、ノイズを減らして傾向を把握しやすくします。

定常化

差分やlog変換により、トレンドや季節性を除去して統計的性質を安定させます。

10.3 SAS Visual Forecastingの操作

SAS Visual Forecastingは、時系列データの可視化、分析、予測を直感的に行えるノーコード/ローコードツールです。

Visual Forecastingの主な機能

  • データインポート: CSV、SASデータセット、データベースから時系列データを取り込み
  • データ探索: 時系列グラフ、トレンド、季節性の可視化
  • 自動モデル選択: ARIMA、ETS、UCMなど複数モデルを自動比較
  • 予測実行: 将来の値を予測し、信頼区間とともに可視化
  • 結果エクスポート: 予測結果をCSV、レポート、ダッシュボードとして出力

ヒント

Visual Forecastingは、複数の時系列を一括で予測する「階層予測」機能も提供しています。例えば、全店舗・全商品の売上を一度に予測できます。

10.4 実習：需要予測モデルの構築

実習: 月次需要予測モデルの構築

架空の小売店の月次売上データを使って、今後6ヶ月の需要を予測するモデルを構築します。

使用データ

  • • 期間: 2020年1月〜2024年12月（5年間、60ヶ月）
  • • 変数: 日付、売上金額、プロモーション有無、季節指標
  • • データセット名: retail_sales

手順

  1. ステップ1: データインポート

    Visual Forecastingを開き、retail_salesデータをCASにロードします。

  2. ステップ2: 時系列グラフの確認

    売上データをプロットし、トレンド・季節性を視覚的に確認します。

  3. ステップ3: モデル自動選択

    「Auto Model Selection」を実行し、ARIMA、ETS、UCMなどから最適モデルを選択します。

  4. ステップ4: 予測実行

    選択されたモデルで6ヶ月先（2025年1月〜6月）の売上を予測します。

  5. ステップ5: 結果評価

    MAPE（平均絶対パーセント誤差）やRMSE（二乗平均平方根誤差）で精度を評価します。

demand_forecast.sas 
/* CASセッションの開始 */
cas mysession;

/* データをCASにロード */
proc casutil;
  load data=work.retail_sales casout="sales";
quit;

/* 時系列予測（ARIMA自動選択） */
proc tsmodel data=sales outobj=(outfor=forecast);
  id date interval=month;
  var sales;
  require tsa;

  /* ARIMAモデルの自動選択 */
  submit;
    /* 季節性ARIMAモデル */
    declare object mymodel(arimax);
    rc = mymodel.setOption('criterion', 'aic');
    rc = mymodel.run();
    /* 6ヶ月先を予測 */
    rc = mymodel.forecast(6);
  endsubmit;
quit;

予測モデルの評価指標

指標 説明 特徴
MAPE 平均絶対パーセント誤差 パーセンテージで誤差を表現。異なるスケールのデータを比較可能
RMSE 二乗平均平方根誤差 大きな誤差にペナルティを与える。元のスケールで誤差を表現
MAE 平均絶対誤差 誤差の絶対値の平均。外れ値の影響を受けにくい
AIC/BIC 情報量規準 モデルの複雑さとフィット度のバランスを評価

10.6 高度な予測手法

基本的な時系列予測に加えて、SAS Viyaでは以下のような高度な手法も利用できます。

階層予測

複数の階層レベル（全体→地域→店舗）の時系列を同時に予測し、合計値の整合性を保ちます。

多変量予測

複数の変数間の相関を考慮した予測。VAR（ベクトル自己回帰）モデルなどを使用します。

因果モデル

外部変数（価格、プロモーション、天候など）を考慮した予測。回帰モデルと時系列を組み合わせます。

機械学習予測

ランダムフォレスト、勾配ブースティング、LSTMなどの機械学習モデルを時系列予測に適用します。

注意事項

高度な手法はデータ量や品質が十分な場合に有効ですが、単純なモデルの方が解釈性が高く、実務で使いやすい場合もあります。常に最もシンプルで効果的なモデルを選択しましょう。

第10章のまとめ

  • 予測モデリングには回帰（連続値予測）と分類（カテゴリ予測）の2種類がある
  • 時系列データはトレンド、季節性、周期性、ノイズの要素で構成される
  • SAS Visual Forecastingで直感的に時系列予測モデルを構築できる
  • MAPE、RMSE、MAEなどの評価指標で予測精度を測定する
  • 階層予測、多変量予測、機械学習など高度な手法も利用可能

理解度確認クイズ

  1. Q1. 回帰モデルと分類モデルの違いは何ですか？

    回答を表示

    回帰モデルは連続値（売上金額など）を予測するのに対し、分類モデルはカテゴリ（購入有無、リスクレベルなど）を予測します。

  2. Q2. 時系列データの4つの主な構成要素を挙げてください。

    回答を表示

    ①トレンド（長期傾向）、②季節性（周期的パターン）、③周期性（景気循環等）、④ランダムノイズ（不規則変動）です。

  3. Q3. MAPEとは何の略で、どのような指標ですか？

    回答を表示

    Mean Absolute Percentage Error（平均絶対パーセント誤差）の略で、予測誤差をパーセンテージで表現する指標です。異なるスケールのデータを比較する際に便利です。

  4. Q4. SAS Visual Forecastingの主な利点は何ですか？

    回答を表示

    ノーコード/ローコードで直感的に時系列予測ができること、自動モデル選択により最適なアルゴリズムを提案してくれること、複数の時系列を一括で予測できる階層予測機能があることなどが挙げられます。

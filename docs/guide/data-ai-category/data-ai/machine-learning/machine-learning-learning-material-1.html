1.1 機械学習とは何か

機械学習（Machine Learning）とは、コンピュータがデータから自動的にパターンやルールを学習し、 新しいデータに対して予測や判断を行う技術です。従来のプログラミングでは、人間が明示的にルールを記述していましたが、 機械学習では「データから自動的にルールを発見する」という点が大きな違いです。

プログラム（ルール）"] P1 --> O1["出力結果"] end subgraph ML["機械学習"] A2["入力データ +
正解ラベル"] --> P2["学習アルゴリズム"] P2 --> M["学習済みモデル

機械学習の特徴

  • データ駆動型: 大量のデータから自動的にパターンを発見
  • 適応性: 新しいデータに基づいてモデルを更新・改善可能
  • 複雑なパターン認識: 人間が明示的にルール化できない複雑な関係性を学習
  • 汎用性: 様々な分野や問題に適用可能

1.2 機械学習の応用分野

機械学習は私たちの日常生活の中で既に広く活用されています。以下は代表的な応用例です：

1. 画像認識・コンピュータビジョン

  • 顔認証システム: スマートフォンのロック解除、空港のセキュリティゲート
  • 医療画像診断: レントゲン写真やMRI画像からの病変検出
  • 自動運転: 道路標識の認識、歩行者の検出

2. 自然言語処理

  • 機械翻訳: Google翻訳、DeepLなどの自動翻訳サービス
  • 音声アシスタント: Siri、Alexa、Google Assistantなどの音声認識と応答
  • 感情分析: SNS投稿やレビューからの顧客満足度分析

3. レコメンデーションシステム

  • 動画配信: NetflixやYouTubeのおすすめ動画
  • ECサイト: Amazonの商品レコメンデーション
  • 音楽配信: Spotifyのプレイリスト提案

4. 金融・ビジネス

  • 信用スコアリング: ローン審査の自動化
  • 不正検知: クレジットカードの不正利用検出
  • 株価予測: 金融市場の動向予測

1.3 機械学習の種類

機械学習は学習方法によって主に3つのカテゴリに分類されます。それぞれの特徴と違いを理解することが重要です。

1.3.1 教師あり学習（Supervised Learning）

教師あり学習は、入力データとそれに対応する正解ラベル（答え）のペアから学習する手法です。 「教師」とは正解データのことを指し、モデルは正解を見ながら学習します。

具体例：スパムメール判定

入力データ: メールの本文（「無料」「今すぐクリック」などの単語）
正解ラベル: スパムかどうか（スパム / 通常メール）
学習プロセス: 過去のメールとそのラベルから、スパムの特徴を学習
予測: 新しいメールがスパムかどうかを判定

教師あり学習は主に2種類に分けられます：

  • 回帰（Regression）: 連続的な数値を予測
    • 例：住宅価格の予測、売上予測、気温予測
  • 分類（Classification）: カテゴリ（クラス）を予測
    • 例：画像の種類判別、病気の診断、顧客の離反予測

1.3.2 教師なし学習（Unsupervised Learning）

教師なし学習は、正解ラベルがないデータから、隠れたパターンや構造を発見する手法です。 データそのものの特徴や関係性を自動的に見つけ出します。

具体例：顧客セグメンテーション

入力データ: 顧客の購買履歴、年齢、居住地など
正解ラベル: なし（どのグループに属するかは未知）
学習プロセス: 似た特徴を持つ顧客を自動的にグループ化
発見: 「高所得で頻繁に購入する顧客層」などのセグメントを特定

教師なし学習の主な手法：

  • クラスタリング: 似たデータをグループ化
    • 例：顧客セグメント分析、画像のグループ化
  • 次元削減: データの特徴量を減らす
    • 例：高次元データの可視化、ノイズ除去
  • 異常検知: 通常とは異なるパターンを発見
    • 例：製造業での不良品検出、ネットワークの侵入検知

1.3.3 強化学習（Reinforcement Learning）

強化学習は、エージェント（学習する主体）が環境と相互作用しながら、報酬を最大化する行動を学習する手法です。 正解は明示的に与えられず、「良い行動」か「悪い行動」かを報酬という形でフィードバックを受けます。

具体例：ゲームAI

エージェント: ゲームをプレイするAI
環境: ゲーム世界（盤面、ステージなど）
行動: コマの移動、キャラクターの操作
報酬: ゲームで勝つと正の報酬、負けると負の報酬
学習プロセス: 試行錯誤を繰り返し、勝率を高める戦略を習得

強化学習の応用例：

  • ロボット制御: 歩行ロボット、マニピュレータの動作最適化
  • 自動運転: 運転戦略の学習
  • ゲームAI: 囲碁、チェス、ビデオゲーム
  • リソース管理: データセンターの電力最適化
sequenceDiagram participant E as 環境 participant A as エージェント Note over A,E: 強化学習のサイクル A->>E: 行動を実行 E->>A: 状態を観測 E->>A: 報酬を受け取る Note over A: 価値を評価し
行動を更新 A->>E: 次の行動を実行

1.4 開発環境の構築

機械学習を実践するために、Python環境とJupyter Notebookをセットアップします。 ここでは初心者でも理解しやすいように、各ステップを丁寧に説明します。

1.4.1 Pythonのインストール

Pythonは機械学習で最も広く使われているプログラミング言語です。豊富なライブラリと使いやすい文法が特徴です。

推奨環境
  • Python バージョン: 3.8以上（3.10または3.11を推奨）
  • メモリ: 8GB以上（大規模データ処理には16GB以上を推奨）
  • ストレージ: 10GB以上の空き容量

Anacondaを使用したインストール（推奨）

Anacondaは、Pythonと主要な科学計算ライブラリがパッケージ化されたディストリビューションです。 初心者にとって環境構築が簡単になるため、強く推奨します。

  1. Anacondaのダウンロード

    https://www.anaconda.com/download から、 ご使用のOS（Windows/Mac/Linux）に対応したインストーラーをダウンロード

  2. インストール実行

    ダウンロードしたインストーラーを実行し、画面の指示に従ってインストール

  3. インストール確認

    ターミナル（Windows: Anaconda Prompt、Mac/Linux: ターミナル）を開き、以下のコマンドで確認：

    # Pythonのバージョン確認
python --version

# condaのバージョン確認
conda --version

1.4.2 仮想環境の作成

仮想環境とは、プロジェクトごとに独立したPython環境を作成する仕組みです。 異なるプロジェクトで異なるバージョンのライブラリを使用する場合でも、競合を防ぐことができます。 

# 仮想環境の作成（名前: ml-env、Pythonバージョン: 3.10）
conda create -n ml-env python=3.10

# 仮想環境の有効化
conda activate ml-env

# 仮想環境の無効化（終了時）
conda deactivate
仮想環境のメリット
  • 依存関係の分離: プロジェクトごとに異なるライブラリバージョンを管理
  • クリーンな環境: 不要なライブラリが混在しない
  • 再現性: 環境を簡単に再構築できる

1.4.3 必要なライブラリのインストール

機械学習で使用する主要なライブラリをインストールします。各ライブラリの役割を理解しておきましょう。 

# 仮想環境を有効化した状態で実行
conda activate ml-env

# 機械学習ライブラリのインストール
pip install scikit-learn  # 機械学習アルゴリズム集
pip install pandas        # データ操作・分析
pip install numpy         # 数値計算
pip install matplotlib    # データ可視化（基本）
pip install seaborn       # データ可視化（高度）
pip install jupyter       # Jupyter Notebook環境

主要ライブラリの役割

  • scikit-learn: 機械学習のための統合ライブラリ
    • 回帰、分類、クラスタリングなど多様なアルゴリズムを提供
    • データの前処理、モデル評価の機能も充実
  • pandas: データ操作と分析のライブラリ
    • 表形式データ（DataFrame）を扱うための強力なツール
    • CSVファイルの読み込み、データのフィルタリング、集計などが簡単に実行可能
  • numpy: 数値計算の基礎ライブラリ
    • 高速な配列演算
    • 線形代数、統計関数などを提供
  • matplotlib / seaborn: データ可視化ライブラリ
    • グラフやチャートの作成
    • データの分布や傾向を視覚的に理解するために必須
  • jupyter: インタラクティブなプログラミング環境
    • コードとその実行結果、説明文を一つのノートブックで管理
    • 試行錯誤しながら学習するのに最適

1.4.4 Jupyter Notebookの起動と使い方

Jupyter Notebookは、コードをセル単位で実行できるインタラクティブな開発環境です。 機械学習の学習や実験に最適なツールです。 

# Jupyter Notebookの起動
jupyter notebook

# または、JupyterLabを使用する場合
jupyter lab

コマンドを実行すると、ブラウザが自動的に開き、Jupyter Notebookのインターフェースが表示されます。

Jupyter Notebookの基本操作
  • 新規ノートブック作成: 「New」→「Python 3」をクリック
  • セルの実行: Shift + Enterキー
  • セルの追加:
    • 下に追加: Bキー
    • 上に追加: Aキー
  • セルの削除: DDキー（Dを2回押す）
  • セルのタイプ変更:
    • コードセル: Yキー
    • Markdownセル（テキスト）: Mキー
実習 1-1: 環境動作確認

インストールした環境が正しく動作するか確認しましょう。Jupyter Notebookで新規ノートブックを作成し、 以下のコードを実行してください。

手順
  1. Jupyter Notebookを起動
  2. 新規ノートブック（Python 3）を作成
  3. 以下のコードをセルに入力して実行（Shift + Enter）
確認コード
# ライブラリのインポートとバージョン確認
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
import sklearn

print("環境確認:")

print(f"Scikit-learn version: {sklearn.__version__}")

# 簡単なグラフ表示テスト
x = np.linspace(0, 10, 100)
y = np.sin(x)

plt.figure(figsize=(8, 4))

plt.xlabel('x')
plt.ylabel('y')

plt.legend()

print("\n環境が正常にセットアップされました!")
期待される結果
  • 各ライブラリのバージョン番号が表示される
  • sin波のグラフが描画される
  • エラーが表示されない場合、環境構築は成功です
実習 1-2: データの読み込みと基本操作

機械学習の基本となるデータ操作を体験しましょう。pandasを使ってサンプルデータを作成し、基本的な操作を行います。

実習コード
# サンプルデータの作成
data = {
    '名前': ['太郎', '花子', '一郎', '二郎', '三郎'],
    '年齢': [25, 30, 35, 28, 32],
    '身長': [170, 160, 175, 168, 172],
    '体重': [65, 52, 70, 62, 68]
}

# DataFrameの作成
df = pd.DataFrame(data)

# データの確認
print("=== データの先頭5行 ===")
print(df.head())

print("\n=== データの基本情報 ===")
print(df.info())

print("\n=== 基本統計量 ===")
print(df.describe())

# BMIの計算（新しい列の追加）
df['BMI'] = df['体重'] / (df['身長'] / 100) ** 2
df['BMI'] = df['BMI'].round(2)

print("\n=== BMI追加後のデータ ===")
print(df)
学習ポイント
  • DataFrame: pandasの表形式データ構造
  • head(): データの先頭数行を表示
  • info(): データ型や欠損値の情報を確認
  • describe(): 数値列の統計量（平均、標準偏差など）を表示
  • 新しい列の追加: 既存の列から計算して新しい特徴量を作成

理解度確認クイズ
  1. 機械学習と従来のプログラミングの最も大きな違いは何ですか？
    解答を表示

    従来のプログラミングでは人間が明示的にルールを記述するのに対し、 機械学習ではデータから自動的にルールやパターンを学習する点が最大の違いです。

  2. 教師あり学習と教師なし学習の違いを説明してください。
    解答を表示

    教師あり学習は正解ラベル付きのデータから学習し、予測を行います（例：スパム判定）。 教師なし学習は正解ラベルがなく、データの構造やパターンを自動的に発見します（例：顧客セグメンテーション）。

  3. 回帰と分類の違いは何ですか？具体例を挙げて説明してください。
    解答を表示

    回帰は連続的な数値を予測する手法です（例：住宅価格の予測、明日の気温予測）。 分類はカテゴリやクラスを予測する手法です（例：画像が犬か猫かの判定、メールがスパムかどうかの判定）。

  4. 機械学習でPythonが広く使われる理由を3つ挙げてください。
    解答を表示

    1. 豊富な機械学習ライブラリ（scikit-learn、TensorFlow、PyTorchなど）が利用可能
    2. シンプルで読みやすい文法により、初心者でも学習しやすい
    3. 大規模なコミュニティとドキュメントが充実しており、問題解決がしやすい

  5. Jupyter Notebookを使うメリットを2つ挙げてください。
    解答を表示

    1. コードをセル単位で実行できるため、段階的にデバッグや動作確認が可能
    2. コード、実行結果、説明文を一つのノートブックで管理でき、学習や共有に最適

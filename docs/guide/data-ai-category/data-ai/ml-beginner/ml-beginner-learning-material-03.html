第3章

AIに「予測」させてみよう

「過去のデータから未来を予測する」という機械学習の基本的な考え方を学びます。天気予報や売上予測など、身近な予測の例を使って、AIがどのようにパターンを見つけるのかを直感的に理解します。

この章で学ぶこと

  • 予測（回帰）とは何かを理解する
  • 身近な予測の例から、AIがパターンを見つける仕組みを知る
  • 予測の精度と誤差の概念を理解する
  • 実際のデータを使った予測の例を体験する

3.1 予測とは何か？

予測（回帰）とは、過去のデータからパターンを見つけ出し、未来の数値を推測することです。天気予報で「明日の最高気温は25度」と予測したり、売上データから「来月の売上は500万円」と予測したりするのが、この予測です。

人間も日常的に予測をしています。例えば、過去の経験から「今日は晴れているから洗濯物が乾きそう」「この時間は道が混むから早めに出よう」といった判断をします。機械学習の予測も、同じように過去のデータから法則性を見つけて未来を推測します。

キーポイント

予測の本質は「過去の傾向から未来の数値を推測すること」です。完璧に当てることは難しいですが、過去のデータが多いほど、より正確な予測ができます。

3.2 身近な予測の例

私たちの生活には、様々な予測が使われています。以下のような場面で、機械学習による予測が活用されています。

天気予報

過去の気象データ（気温、湿度、気圧など）から、明日の天気や気温を予測します。数十年分のデータを学習することで、季節ごとの傾向やパターンを捉えます。

売上予測

過去の売上データ、季節、曜日、キャンペーン実施有無などから、来月や来週の売上を予測します。在庫管理や人員配置の計画に役立ちます。

不動産価格の予測

立地、広さ、築年数、周辺施設などの情報から、物件の適正価格を予測します。過去の取引データを学習することで、相場に近い価格を推測できます。

配車サービスの到着時間予測

過去の交通データ、時間帯、天候などから、タクシーやライドシェアの到着時間を予測します。ユーザーは待ち時間を事前に知ることができます。

3.3 AIがパターンを見つける仕組み

AIはどのようにして過去のデータから「パターン」を見つけるのでしょうか？簡単な例で見てみましょう。

例：アイスクリームの売上予測

あるアイスクリーム店の1週間のデータを見てみましょう。

曜日 最高気温（℃） 売上（個）
月曜日 20℃ 50個
火曜日 22℃ 60個
水曜日 25℃ 75個
木曜日 28℃ 90個
金曜日 30℃ 100個
土曜日 32℃ 110個
日曜日 27℃ 85個

このデータを見ると、「気温が高いほど売上が増える」という傾向が見えてきます。AIはこのパターンを数学的に学習し、「気温が26℃の日は約80個売れる」といった予測を行います。

補足：線形回帰とは

上記のような「気温と売上の関係」を直線で表現する手法を「線形回帰」と呼びます。中学校で習った「比例のグラフ（y = ax + b）」をイメージすると分かりやすいでしょう。AIは、データに最もフィットする直線を自動的に見つけ出します。

3.4 予測の精度と誤差

どんなに優れたAIでも、100%正確に未来を予測することはできません。必ず「誤差」が生じます。この誤差を理解し、許容範囲を決めることが重要です。

予測と実際の違い（誤差）の例

日付 予測値 実際の値 誤差
8/1 80個 85個 -5個（少し過小予測）
8/2 95個 90個 +5個（少し過大予測）
8/3 105個 110個 -5個（少し過小予測）

誤差は避けられない

予測には必ず誤差があります。重要なのは「誤差を小さくすること」と「誤差の範囲を理解すること」です。天気予報の「降水確率」のように、確率や幅を持たせた予測も有効です。

精度を高めるには

  • データを増やす： より多くの過去データを学習させる
  • 関連する要素を追加： 気温だけでなく、湿度や曜日なども考慮する
  • 異常値を除外： 特殊なイベント（祭りなど）のデータは別に扱う

3.5 予測を使うときの注意点

予測は便利ですが、盲信するのは危険です。以下の点に注意しましょう。

注意すべきポイント

  • 過去と未来が違う場合： 新型コロナのような未曾有の出来事が起きると、過去のパターンが通用しません
  • 因果関係の誤解： 「Aが増えるとBも増える」という相関関係があっても、AがBの原因とは限りません
  • データの偏り： 学習データに偏りがあると、予測も偏ります（例：夏のデータしかない場合、冬の予測は不正確）

失敗例：アイスの売上予測

「気温が高いほどアイスが売れる」というパターンを学習したAIに、真冬（気温5℃）の売上を予測させると、「10個」と予測してしまうかもしれません。しかし実際には、冬でもクリスマスやお正月など特別なイベントがあれば売上が増える可能性があります。

教訓： 気温だけでなく、季節イベントや曜日なども考慮した予測が必要です。

理解度確認クイズ

  1. Q1. 予測（回帰）とは何か、簡単に説明してください。

    回答を表示

    予測（回帰）とは、過去のデータからパターンを見つけ出し、未来の数値を推測することです。天気予報や売上予測など、数値を予測する場面で使われます。

  2. Q2. 身の回りで使われている予測の例を2つ挙げてください。

    回答を表示

    例：天気予報（気温予測）、売上予測、不動産価格の予測、配車サービスの到着時間予測など（2つ以上挙げられればOK）

  3. Q3. アイスクリームの売上予測の例で、AIはどのようなパターンを見つけましたか？

    回答を表示

    「気温が高いほど売上が増える」という傾向を発見しました。このパターンから、新しい気温データを入力すると、売上を予測できるようになります。

  4. Q4. 予測の精度を高めるためにできることを1つ挙げてください。

    回答を表示

    例：より多くの過去データを学習させる、関連する要素（湿度、曜日など）を追加する、異常値を除外する、など（1つ以上挙げられればOK）

  5. Q5. 予測を使うときに注意すべきことは何ですか？

    回答を表示

    例：過去と未来が大きく異なる場合は予測が外れる、相関関係と因果関係を混同しない、データに偏りがないか確認する、など（1つ以上挙げられればOK）

第3章のまとめ

  • 予測（回帰）は、過去のデータからパターンを見つけ出し、未来の数値を推測する技術です
  • 天気予報、売上予測、不動産価格予測など、私たちの生活の多くの場面で使われています
  • AIは過去のデータから傾向を発見し、関係性を数式化して予測を行います
  • 予測には必ず誤差があり、100%正確に未来を当てることはできません
  • データ量を増やす、関連要素を追加する、異常値を除外するなどの工夫で精度を高められます
  • 過去と未来が異なる場合、データに偏りがある場合は予測が外れる可能性があるため注意が必要です

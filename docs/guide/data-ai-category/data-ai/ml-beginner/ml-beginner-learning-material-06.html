第6章

ノーコードでAIを体験しよう

Google Teachable Machineを使って、プログラミングなしで自分だけのAIモデルを作成します。画像認識AIを実際に作って動かすことで、機械学習の仕組みを体感します。

この章で学ぶこと

  • ノーコードAIツール（Teachable Machine）の使い方を習得する
  • 実際に画像認識AIモデルを作成し、学習プロセスを体験する
  • AIが「学習する」とはどういうことかを実践を通じて理解する

6.1 Teachable Machineとは？

Teachable Machineは、Googleが提供する無料のノーコードAIツールです。プログラミングの知識がなくても、ブラウザ上で画像認識や音声認識のAIモデルを作成できます。

このツールを使うことで、「AIにデータを見せて学習させる」という機械学習の基本プロセスを、直感的に体験できます。

Teachable Machineの特徴

  • 無料で利用可能： Googleアカウントがあればすぐに始められます
  • プログラミング不要： マウス操作だけでAIモデルを作成できます
  • リアルタイム学習： データを追加するとすぐに結果が見られます
  • 3つのモード： 画像、音声、ポーズ（身体の動き）の認識が可能です

6.2 画像認識AIを作ってみよう

実際にTeachable Machineを使って、「身の回りの物体を認識するAI」を作ってみましょう。この実習では、ペン、スマートフォン、マグカップなど、3つの物体を区別するAIを作成します。

実習 6-1: 物体認識AIを作成する

身の回りの物体を認識するAIモデルを作成し、機械学習の学習プロセスを体験します。

準備するもの

  • パソコンとWebカメラ（またはノートPCの内蔵カメラ）
  • 認識させたい物体を3つ（例：ペン、スマートフォン、マグカップ）

手順

  1. 1 ブラウザで https://teachablemachine.withgoogle.com/ にアクセスします
  2. 2 「Get Started」ボタンをクリックし、「Image Project」→「Standard image model」を選択します
  3. 3 Class 1の名前を「ペン」に変更し、「Webcam」ボタンを押します。ペンをカメラに映して「Hold to Record」を長押しし、約50枚の画像を撮影します（角度を変えながら撮影すると精度が上がります）
  4. 4 「Add a class」ボタンをクリックし、Class 2を「スマートフォン」、Class 3を「マグカップ」として、同様に画像を撮影します
  5. 5 3つのクラスすべてのデータが揃ったら、画面中央の「Train Model」ボタンをクリックします。学習が始まるので、完了を待ちます（数十秒程度）
  6. 6 学習が完了したら、画面右側の「Preview」で実際に物体を認識してみましょう。カメラに物体を映すと、リアルタイムで認識結果が表示されます

ヒント

撮影時は物体を様々な角度から撮ると精度が向上します。また、背景を変えたり、明るさを変えたりすることで、より汎用的なモデルになります。

6.3 AIモデルを試してみよう

学習が完了したら、実際にAIの認識精度を試してみましょう。以下のような実験をしてみてください。

動作確認のポイント

  • 異なる角度で試す： 学習時とは違う角度から物体を映してみてください。認識できますか？
  • 似た物体で試す： 例えば、別のペンや別のマグカップを使ってみてください。認識できますか？
  • 全く違う物体で試す： 学習していない物体（例：本）を映すとどうなりますか？
  • 認識率を確認： 画面に表示されるパーセンテージ（確信度）を観察してください。100%に近いほど確信が高いです

認識結果の例

ペン 95%
スマートフォン 3%
マグカップ 2%

※ペンを映した場合の認識結果の例。95%の確信度で「ペン」と認識しています。

6.4 どうして認識できるの？

AIが物体を認識できるのは、たくさんの画像から「特徴」を学習しているからです。人間が「ペンは細長い形をしている」と認識するのと同じように、AIも画像のパターンを学習します。

図6.2: 画像認識AIの学習プロセス

具体的には、以下のような流れで学習が進みます。

1

画像を数値に変換

コンピュータは画像をピクセル（点）の集まりとして見ています。各ピクセルの色や明るさが数値として処理されます。

2

特徴を抽出

たくさんの画像から「ペンに共通する特徴」（例：細長い形、先端が尖っている）を自動的に見つけ出します。

3

パターンを記憶

抽出した特徴を「モデル」として記憶します。これが「学習」の本質です。

4

新しい画像を判定

新しい画像が入力されると、記憶したパターンと比較して「これはペンに一番似ている」と判定します。

キーポイント

機械学習では、プログラマーが「ペンの特徴はこれ」と明示的に教えるのではなく、AIがデータから自動的に特徴を見つけ出します。これが従来のプログラミングとの大きな違いです。

6.5 もっとデータを増やしてみよう

もし認識精度が低い場合は、データを追加してみましょう。機械学習では、データが多いほど精度が向上する傾向があります。

データを増やす際のコツ

  • 多様な角度から撮影： 正面だけでなく、横や斜めからも撮影します
  • 異なる明るさで撮影： 明るい場所と暗い場所で撮影します
  • 背景を変える： 机の上、手に持った状態、床の上など、様々な背景で撮影します
  • 距離を変える： カメラに近づけたり、遠ざけたりします

理解度確認クイズ

  1. Q1. Teachable Machineの特徴として正しいものはどれですか？

    回答を表示

    無料で利用可能、プログラミング不要、リアルタイム学習、画像・音声・ポーズの3つのモードがある、などが特徴です。誰でも簡単にAIモデルを作成できるノーコードツールです。

  2. Q2. AIが物体を認識できる理由を説明してください。

    回答を表示

    AIはたくさんの画像から「特徴」を自動的に学習しているからです。画像をピクセルの数値データとして処理し、パターンを抽出して記憶します。新しい画像が来たときに、記憶したパターンと比較して判定します。

  3. Q3. 認識精度を向上させるにはどうすればよいですか？

    回答を表示

    データ数を増やすことが効果的です。また、多様な角度、異なる明るさ、様々な背景、異なる距離で撮影することで、より汎用的なモデルを作成できます。

  4. Q4. 学習していない物体（例：本）をカメラに映すとどうなりますか？

    回答を表示

    学習済みの3つのクラス（ペン、スマートフォン、マグカップ）のいずれかに分類されますが、確信度（パーセンテージ）は低くなる傾向があります。AIは学習していないクラスを「分からない」と答えることはできず、既知のクラスの中から最も近いものを選びます。

第6章のまとめ

  • Teachable Machineを使えば、プログラミングなしでAIモデルを作成できます
  • AIは大量の画像から特徴を自動的に学習し、新しい画像を認識します
  • データ数を増やし、多様なパターンを学習させることで精度が向上します
  • 実際にAIを作って試すことで、「学習とは何か」を体験的に理解できました

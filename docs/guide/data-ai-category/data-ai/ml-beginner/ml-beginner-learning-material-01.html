機械学習って何だろう？

「AIが学ぶ」とはどういうことか、身近な例を使って解説します。機械学習が私たちの生活でどのように使われているかを理解し、学習のゴールを明確にします。

この章で学ぶこと

  • 機械学習・AIの基本概念を理解する
  • 身近な例から機械学習がどのように使われているかを知る
  • 機械学習の主な種類（予測、分類、グループ分け）の概要を理解する

1.1 機械学習とは何か？

機械学習（Machine Learning）とは、コンピュータが大量のデータから自動的にパターンやルールを見つけ出す技術のことです。人間が一つひとつ指示を与えなくても、データから学習することで賢くなっていきます。

例えば、赤ちゃんが成長する過程を想像してみてください。赤ちゃんは最初、犬や猫の区別がつきません。しかし、「これは犬だよ」「これは猫だよ」と何度も教えられることで、次第に自分で見分けられるようになります。機械学習も同じように、たくさんの例から学習していきます。

キーポイント

機械学習は「データから学ぶ」技術です。プログラマーが詳細なルールを書く代わりに、AIがデータのパターンを自動的に発見します。

1.2 身の回りにある機械学習の例

実は、私たちは毎日、意識せずに機械学習の恩恵を受けています。以下のような場面で機械学習が使われています。

スマートフォンの顔認証

顔の特徴（目の位置、鼻の形など）を学習し、あなたの顔を瞬時に見分けます。暗い場所でも、髪型が変わっても認識できるのは、学習したパターンがあるからです。

動画配信サービスのおすすめ機能

あなたの視聴履歴から、好みのジャンルや俳優を学習し、興味がありそうな作品を提案します。見れば見るほど精度が高くなります。

メールの迷惑メールフィルター

過去の迷惑メールの特徴（特定の単語、送信元など）を学習し、新しいメールが迷惑メールかどうかを自動判定します。

音声アシスタント（Siri、Google アシスタント）

たくさんの人の声を学習することで、あなたの話す言葉を理解し、質問に答えたり、操作を実行したりできます。

機械学習には大きく分けて3つのタイプがあります。それぞれ異なる場面で使われます。

図1.2: 機械学習の3つの主要なタイプと活用例

タイプ1: 予測（回帰）

過去のデータから「数値」を予測します。例えば、「明日の気温は何度か？」「来月の売上はいくらか？」といった問題です。

具体例： 過去10年間の天気データから、明日の最高気温を予測する

タイプ2: 分類

データを「カテゴリー」に分けます。例えば、「このメールは迷惑メールか、正常なメールか？」「この写真は犬か、猫か？」といった問題です。

具体例： メールの内容から、迷惑メールかどうかを判定する

タイプ3: グループ分け（クラスタリング）

正解がない状態で、似たデータをグループにまとめます。例えば、「顧客を購買行動でグループ分けする」「商品を似た特徴でまとめる」といった問題です。

具体例： 購買履歴から顧客を「頻繁に買う人」「たまに買う人」「最近買わない人」に自動分類する

図1.3: AIと機械学習の包含関係

AI、機械学習、深層学習の違い

  • AI（人工知能）： コンピュータが人間のように考えたり判断したりする技術全般を指します（一番広い概念）
  • 機械学習： AIの一部で、データから自動的に学習する技術です
  • 深層学習： 機械学習の一種で、人間の脳の仕組みを真似た「ニューラルネットワーク」を使う高度な手法です

  2. Q2. 身の回りで使われている機械学習の例を2つ挙げてください。

    回答を表示

    例：スマートフォンの顔認証、動画配信サービスのおすすめ機能、メールの迷惑メールフィルター、音声アシスタントなど（2つ以上挙げられればOK）

  3. Q3. 機械学習の3つの主要なタイプは何ですか？

    回答を表示

    予測（回帰）、分類、グループ分け（クラスタリング）の3つです。予測は数値を予測し、分類はカテゴリーに分け、グループ分けは似たデータをまとめます。

  4. Q4. AIと機械学習の違いを説明してください。

    回答を表示

    AIは人工知能全般を指す広い概念で、機械学習はAIの一部です。機械学習は、データから自動的に学習する技術に特化しています。つまり、「AI ＞ 機械学習」という包含関係になります。

第1章のまとめ

  • 機械学習は、コンピュータがデータから自動的にパターンを学習する技術です
  • スマートフォンの顔認証、おすすめ機能、迷惑メールフィルターなど、私たちの生活の多くの場面で使われています
  • 機械学習には「予測」「分類」「グループ分け」という3つの主要なタイプがあります

# 機械学習入門 学習ガイドライン

このガイドラインでは、機械学習の基礎を**プログラミング未経験者**向けに段階的に学習するためのカリキュラムを提供しています。専門用語を最小限に抑え、身近な例を使って「AIが学習するとはどういうことか」を直感的に理解できるよう構成しています。

## 前提条件
### 必要な環境
- インターネットに接続できるパソコン（Windows/Mac どちらでも可）
- Webブラウザ（Chrome、Edge、Safari など）
- Microsoft Excel または Google スプレッドシート（データ操作の練習用）
- Googleアカウント（無料のAIツールを使用するため）

### 参考リソース
- [Google Teachable Machine（無料・ノーコードAI体験）](https://teachablemachine.withgoogle.com/)
- [Google Colaboratory（無料・ブラウザでAI実行）](https://colab.research.google.com/)
- [Kaggle Learn（無料学習コース）](https://www.kaggle.com/learn)
- [総務省 ICTスキル総合習得プログラム](https://www.soumu.go.jp/ict_skill/)

### 前提知識
- **必須**: パソコンの基本操作（ファイルの保存、Webブラウザの使用など）
- **必須**: 小学校レベルの算数（足し算、引き算、平均の計算など）
- **推奨**: Excelやスプレッドシートの基本操作（セルへの入力、簡単な計算）
- **推奨**: 中学校レベルの数学（グラフの読み方、比例・反比例の概念）

## 学習コンテンツ
### [1. 機械学習って何だろう？](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-01.html)
「AIが学ぶ」とはどういうことか、身近な例（スマートフォンの顔認証、音声アシスタント、おすすめ機能など）を使って解説します。機械学習が私たちの生活でどのように使われているかを理解し、学習のゴールを明確にします。

### [2. データの見方を学ぼう](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-02.html)
機械学習の燃料となる「データ」について学びます。表形式のデータの読み方、平均や合計などの基本的な集計、グラフを使ったデータの可視化を、Excelやスプレッドシートを使って実践します。

### [3. AIに「予測」させてみよう](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-03.html)
「過去のデータから未来を予測する」という機械学習の基本的な考え方を学びます。天気予報や売上予測など、身近な予測の例を使って、AIがどのようにパターンを見つけるのかを直感的に理解します。

### [4. AIに「分類」させてみよう](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-04.html)
メールの迷惑メール判定、写真の自動分類など、「分類」の仕組みを学びます。人間がどのように物事を分類しているかを振り返り、AIも同様の考え方で学習していることを理解します。

### [5. AIに「グループ分け」させてみよう](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-05.html)
顧客セグメンテーションや商品カテゴリ分けなど、「似たものを集める」クラスタリングの考え方を学びます。正解がない状態からパターンを発見するAIの能力を理解します。

### [6. ノーコードでAIを体験しよう](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-06.html)
Google Teachable Machineを使って、プログラミングなしで自分だけのAIモデルを作成します。画像認識AIを実際に作って動かすことで、機械学習の仕組みを体感します。

### [7. 良いデータと悪いデータ](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-07.html)
機械学習の精度を左右する「データの品質」について学びます。欠損値、異常値、偏ったデータなど、実際のビジネスでよく遭遇する問題と、その対処法を分かりやすく解説します。

### [8. AIの予測は正しいの？](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-08.html)
AIの予測がどれくらい正確かを評価する方法を学びます。「正解率」「誤差」などの基本的な評価指標を、具体例を使って分かりやすく解説します。

### [9. ビジネスでの活用事例](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-09.html)
小売業、製造業、金融業、医療など、様々な業界での機械学習の活用事例を紹介します。自社の業務にどう活かせるかを考えるヒントを提供します。

### [10. AIプロジェクトの進め方](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-10.html)
機械学習プロジェクトを始める際の進め方を学びます。課題設定、データ収集、モデル構築、評価、運用という一連の流れを理解し、成功するプロジェクトのポイントを押さえます。

### [11. AIの限界と注意点](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-11.html)
AIが苦手なこと、AIを使う際の倫理的な注意点、データプライバシーの問題など、機械学習を活用する上で知っておくべき重要なポイントを学びます。

### [12. 次のステップへ](https://fcircle-biz.github.io/tech_docs/guide/data-ai-category/data-ai/ml-beginner/ml-beginner-learning-material-12.html)
このガイドで学んだことを振り返り、さらに学習を深めるための道筋を示します。プログラミング学習への橋渡しや、専門的なAI学習コースへの案内を行います。

## 学習の進め方
1. **概念を理解する**: 各章では、まず身近な例を使って「なぜそれが必要か」「どういう仕組みか」を理解することから始めます
2. **手を動かして体験する**: 可能な限り、実際にツールを使って体験することで理解を深めます
3. **ビジネスとの関連を考える**: 学んだことを自分の仕事や業務にどう活かせるかを常に意識します
4. **分からなくてもOK**: 最初は完璧に理解できなくても大丈夫です。繰り返し読み、実践することで徐々に理解が深まります
5. **質問メモを作る**: 疑問に思ったことはメモしておき、次章以降で解決するか確認します

## 学習目標
このガイドを完了すると、以下のスキルを身につけることができます：
- 機械学習・AIの基本的な仕組みと用語を理解し、他者に説明できる
- 「予測」「分類」「グループ分け」という3つの主要な機械学習タスクの違いを説明できる
- ノーコードツールを使って簡単なAIモデルを作成できる
- データの品質の重要性を理解し、良いデータの条件を説明できる
- AIの予測精度を評価する基本的な方法を理解している
- 自社の業務における機械学習の活用可能性を検討できる
- AIの限界や倫理的な注意点を理解し、適切に活用できる

第5章: リクエストとレスポンスの処理

リクエストとレスポンスの処理

この章で学ぶこと
  • HTTPリクエストとレスポンスの仕組み
  • requestオブジェクトからデータを取得する方法
  • responseオブジェクトを使って出力する方法
  • GETメソッドとPOSTメソッドの違いと適切な使い分け
  • フォームデータの処理方法
  • 文字エンコーディングの設定

5.1 HTTPリクエストとレスポンスの仕組み

Webアプリケーションでは、クライアント（ブラウザ）とサーバー間でHTTPプロトコルを使用してデータをやり取りします。 クライアントがサーバーにデータを送る際はHTTPリクエストを使用し、サーバーがクライアントにデータを返す際はHTTPレスポンスを使用します。

graph LR A[ブラウザ] -->|HTTPリクエスト| B[Webサーバー] B -->|HTTPレスポンス| A

JSPでは、以下の暗黙オブジェクトを使用してHTTPリクエストとレスポンスを処理します：

  • request - HTTPリクエストの情報を取得
  • response - HTTPレスポンスの設定と出力

5.2 requestオブジェクトの基本操作

5.2.1 パラメータの取得

フォームから送信されたデータや、URLに含まれるパラメータを取得するにはrequest.getParameter()メソッドを使用します。

実習 5-1: パラメータの取得

URLパラメータとフォームデータを取得するJSPページを作成してみましょう。

1. HTMLフォーム（form.html）を作成
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>フォームテスト</title>
</head>
<body>
    <h2>ユーザー情報入力フォーム</h2>
    <form action="param-test.jsp" method="post">
        <p>
            名前: <input type="text" name="name" value="田中太郎">
        </p>
        <p>
            年齢: <input type="number" name="age" value="25">
        </p>
        <p>
            趣味: 
            <input type="checkbox" name="hobby" value="reading"> 読書
            <input type="checkbox" name="hobby" value="sports"> スポーツ
            <input type="checkbox" name="hobby" value="music"> 音楽
        </p>
        <input type="submit" value="送信">
    </form>
</body>
</html>
2. パラメータ取得JSP（param-test.jsp）を作成
<%@ page contentType="text/html; charset=UTF-8" %>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>パラメータ取得結果</title>
</head>
<body>
    <h2>受信したデータ</h2>
    
    <p><strong>名前:</strong> <%= request.getParameter("name") %></p>
    <p><strong>年齢:</strong> <%= request.getParameter("age") %></p>
    
    <p><strong>趣味:</strong>
    <%
        String[] hobbies = request.getParameterValues("hobby");
        if (hobbies != null) {
            for (String hobby : hobbies) {
                out.print(hobby + " ");
            }
        } else {
            out.print("選択されていません");
        }
    %>
    </p>
    
    <p><strong>リクエストメソッド:</strong> <%= request.getMethod() %></p>
</body>
</html>
期待される結果

フォームを送信すると、入力したデータがJSPページに表示されます。複数選択された趣味も正しく表示されます。

5.2.2 その他のリクエスト情報

requestオブジェクトから取得できる主な情報：

メソッド 説明 例
getMethod() HTTPメソッドを取得 GET, POST
getRequestURI() リクエストURIを取得 /app/test.jsp
getRemoteAddr() クライアントIPアドレス 192.168.1.100
getHeader() HTTPヘッダーを取得 User-Agent情報など

5.3 GETメソッドとPOSTメソッドの使い分け

5.3.1 GETメソッドの特徴

  • パラメータがURLに含まれる
  • ブラウザの履歴に残る
  • ブックマークが可能
  • データ検索に適している

5.3.2 POSTメソッドの特徴

  • パラメータがリクエストボディに含まれる
  • URLにデータが表示されない
  • 大量のデータ送信が可能
  • データの更新・追加に適している
実習 5-2: GETとPOSTの比較

GETメソッドとPOSTメソッドの動作の違いを確認してみましょう。

比較用JSP（method-test.jsp）を作成
<%@ page contentType="text/html; charset=UTF-8" %>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>メソッド比較</title>
</head>
<body>
    <h2>GET vs POST メソッドの比較</h2>
    
    <h3>GETメソッドのフォーム</h3>
    <form action="method-result.jsp" method="get">
        メッセージ: <input type="text" name="message" value="GET送信">
        <input type="submit" value="GET送信">
    </form>
    
    <h3>POSTメソッドのフォーム</h3>
    <form action="method-result.jsp" method="post">
        メッセージ: <input type="text" name="message" value="POST送信">
        <input type="submit" value="POST送信">
    </form>
</body>
</html>
結果表示JSP（method-result.jsp）を作成
<%@ page contentType="text/html; charset=UTF-8" %>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>メソッド結果</title>
</head>
<body>
    <h2>受信結果</h2>
    <p><strong>HTTPメソッド:</strong> <%= request.getMethod() %></p>
    <p><strong>メッセージ:</strong> <%= request.getParameter("message") %></p>
    <p><strong>リクエストURI:</strong> <%= request.getRequestURI() %></p>
    <p><strong>クエリ文字列:</strong> <%= request.getQueryString() %></p>
    
    <p><a href="method-test.jsp">戻る</a></p>

</html>
期待される結果

GETメソッドの場合はURLにパラメータが表示され、POSTメソッドの場合はURLにパラメータが表示されないことを確認できます。

5.4 文字エンコーディングの設定

日本語を含むデータを正しく処理するには、文字エンコーディングの設定が重要です。

5.4.1 リクエストの文字エンコーディング

<%
    // リクエストの文字エンコーディングをUTF-8に設定
    request.setCharacterEncoding("UTF-8");
%>

5.4.2 レスポンスの文字エンコーディング

<%@ page contentType="text/html; charset=UTF-8" %>
重要： 文字エンコーディングの設定は、パラメータを取得する前に行う必要があります。

5.5 responseオブジェクトの基本操作

responseオブジェクトを使用して、HTTPレスポンスの設定やリダイレクトを行うことができます。

実習 5-3: リダイレクト処理

条件に応じてページをリダイレクトする処理を実装してみましょう。

ログイン処理JSP（login-check.jsp）を作成
<%@ page contentType="text/html; charset=UTF-8" %>
<%
    request.setCharacterEncoding("UTF-8");
    
    String username = request.getParameter("username");
    String password = request.getParameter("password");
    
    // 簡単な認証チェック
    if ("admin".equals(username) && "password123".equals(password)) {
        // ログイン成功：成功ページにリダイレクト
        response.sendRedirect("login-success.jsp");
    } else {
        // ログイン失敗：エラーページにリダイレクト
        response.sendRedirect("login-error.jsp?error=invalid");
    }
%>
ログイン成功ページ（login-success.jsp）を作成
<%@ page contentType="text/html; charset=UTF-8" %>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>ログイン成功</title>
</head>
<body>
    <h2>ログイン成功</h2>
    <p>管理者としてログインしました。</p>
    <p><a href="login-form.html">ログアウト</a></p>
</body>
</html>
期待される結果

正しい認証情報でログインすると成功ページに、間違った情報だとエラーページにリダイレクトされます。

5.6 実践演習：簡単な計算機

実習 5-4: Web計算機の作成

フォームから入力された数値を使って計算を行うWebアプリケーションを作成してみましょう。

計算機フォーム（calculator.html）を作成
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>Web計算機</title>
</head>
<body>
    <h2>Web計算機</h2>
    <form action="calculator.jsp" method="post">
        <p>
            数値1: <input type="number" name="num1" value="10">
        </p>
        <p>
            演算子: 
            <select name="operator">
                <option value="add">+（加算）</option>
                <option value="subtract">-（減算）</option>
                <option value="multiply">*（乗算）</option>
                <option value="divide">/（除算）</option>
            </select>
        </p>
        <p>
            数値2: <input type="number" name="num2" value="5">
        </p>
        <input type="submit" value="計算する">
    </form>
</body>
</html>
計算処理JSP（calculator.jsp）を作成
<%@ page contentType="text/html; charset=UTF-8" %>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>計算結果</title>
</head>
<body>
    <h2>計算結果</h2>
    
    <%
        String num1Str = request.getParameter("num1");
        String num2Str = request.getParameter("num2");
        String operator = request.getParameter("operator");
        
        if (num1Str != null && num2Str != null && operator != null) {
            try {
                double num1 = Double.parseDouble(num1Str);
                double num2 = Double.parseDouble(num2Str);
                double result = 0;
                String operatorSymbol = "";
                
                if ("add".equals(operator)) {
                    result = num1 + num2;
                    operatorSymbol = "+";
                } else if ("subtract".equals(operator)) {
                    result = num1 - num2;
                    operatorSymbol = "-";
                } else if ("multiply".equals(operator)) {
                    result = num1 * num2;
                    operatorSymbol = "*";
                } else if ("divide".equals(operator)) {
                    if (num2 != 0) {
                        result = num1 / num2;
                        operatorSymbol = "/";
                    } else {
                        out.println("<p style='color: red;'>エラー: ゼロ除算はできません</p>");
                    }
                }
                
                if (!"divide".equals(operator) || num2 != 0) {
    %>
                    <p><strong>計算式:</strong> <%= num1 %> <%= operatorSymbol %> <%= num2 %></p>
                    <p><strong>結果:</strong> <%= result %></p>
    <%
                }
            } catch (NumberFormatException e) {
    %>
                <p style="color: red;">エラー: 正しい数値を入力してください</p>
    <%
            }
        } else {
    %>
            <p style="color: red;">エラー: 必要なパラメータが不足しています</p>
    <%
        }
    %>
    
    <p><a href="calculator.html">戻る</a></p>
</body>
</html>
期待される結果

数値と演算子を選択して計算すると、結果が表示されます。ゼロ除算や不正な入力に対してはエラーメッセージが表示されます。

理解度確認クイズ
  1. 複数の値が選択可能なcheckboxの値を取得するために使用するメソッドは何ですか？
  2. GETメソッドとPOSTメソッドの主な違いを3つ挙げてください。
  3. 日本語の文字化けを防ぐために設定すべき2つの文字エンコーディングは何ですか？
  4. 別のページにリダイレクトするために使用するメソッドは何ですか？
  5. クライアントのIPアドレスを取得するメソッドは何ですか？
解答例
  1. request.getParameterValues()メソッド
  2. ・GETはURLにパラメータが表示される、POSTは表示されない
    ・GETはブックマーク可能、POSTは不可能
    ・GETは軽いデータ向け、POSTは大量データ向け
  3. request.setCharacterEncoding("UTF-8")とpage contentType="text/html; charset=UTF-8"
  4. response.sendRedirect()メソッド
  5. request.getRemoteAddr()メソッド

5.1 String クラスによる文字列操作

String クラス は、Javaで最も頻繁に使用されるクラスの一つです。文字列の作成、検索、置換、分割など、様々な操作を提供します。

String の重要な特徴
  • 不変（Immutable）：一度作成された String オブジェクトは変更できない
  • 文字列プール：同じ文字列リテラルは一つのオブジェクトを共有
  • 豊富なメソッド：検索、置換、分割、変換など多様な操作が可能
  • +演算子：文字列連結に使用可能（内部的には StringBuilder を使用）

主要な String メソッド

カテゴリ メソッド 説明 例
基本情報 length() 文字列の長さを取得 "Hello".length() → 5
isEmpty() 空文字列かどうか判定 "".isEmpty() → true
charAt(int) 指定位置の文字を取得 "Hello".charAt(1) → 'e'
検索 indexOf(String) 部分文字列の位置を検索 "Hello".indexOf("ll") → 2
contains(String) 部分文字列を含むか判定 "Hello".contains("ell") → true
startsWith(String) 指定文字列で始まるか判定 "Hello".startsWith("He") → true
endsWith(String) 指定文字列で終わるか判定 "Hello".endsWith("lo") → true
変換 toLowerCase() 小文字に変換 "Hello".toLowerCase() → "hello"
toUpperCase() 大文字に変換 "Hello".toUpperCase() → "HELLO"
trim() 前後の空白を除去 " Hello ".trim() → "Hello"
変更 substring(int, int) 部分文字列を抽出 "Hello".substring(1, 4) → "ell"
replace(String, String) 文字列を置換 "Hello".replace("ll", "rr") → "Herro"
split(String) 文字列を分割して配列に "a,b,c".split(",") → ["a", "b", "c"]
実習 5-1: String 操作の基本

文字列処理の基本的な操作を習得しましょう。

public class StringBasics {
    public static void main(String[] args) {
        // 基本的な文字列操作
        String text = "  Java Programming Language  ";
        
        System.out.println("=== 基本情報の取得 ===");
        System.out.println("元の文字列: '" + text + "'");
        System.out.println("文字列の長さ: " + text.length());
        System.out.println("5番目の文字: " + text.charAt(4));
        System.out.println("空文字列かどうか: " + text.isEmpty());
        
        // 前後の空白を除去
        String trimmed = text.trim();
        System.out.println("\n=== 前後空白除去 ===");
        System.out.println("trim後: '" + trimmed + "'");
        System.out.println("trim後の長さ: " + trimmed.length());
        
        // 検索操作
        System.out.println("\n=== 検索操作 ===");
        System.out.println("'Java'の位置: " + trimmed.indexOf("Java"));
        System.out.println("'Program'を含むか: " + trimmed.contains("Program"));
        System.out.println("'Java'で始まるか: " + trimmed.startsWith("Java"));
        System.out.println("'Language'で終わるか: " + trimmed.endsWith("Language"));
        
        // 大文字・小文字変換
        System.out.println("\n=== 大文字・小文字変換 ===");
        System.out.println("大文字: " + trimmed.toUpperCase());
        System.out.println("小文字: " + trimmed.toLowerCase());
        
        // 部分文字列の抽出
        System.out.println("\n=== 部分文字列の抽出 ===");
        System.out.println("0から4文字: " + trimmed.substring(0, 4));
        System.out.println("5文字目以降: " + trimmed.substring(5));
        
        // 文字列の置換
        System.out.println("\n=== 文字列の置換 ===");
        String replaced = trimmed.replace("Java", "Python");
        System.out.println("置換結果: " + replaced);
        
        // 文字列の分割
        System.out.println("\n=== 文字列の分割 ===");
        String languages = "Java,Python,C++,JavaScript,Go";
        String[] langArray = languages.split(",");
        System.out.println("分割前: " + languages);
        System.out.print("分割後: ");
        for (String lang : langArray) {
            System.out.print("[" + lang + "] ");
        }
        System.out.println();
        
        // 実践的な例：メールアドレスの検証
        System.out.println("\n=== 実践例：メールアドレスの検証 ===");
        String[] emails = {
            "user@example.com",
            "invalid-email",
            "test@domain.co.jp",
            "@invalid.com",
            "user@"
        };
        
        for (String email : emails) {
            boolean isValid = validateEmail(email);
            System.out.println(email + " → " + (isValid ? "有効" : "無効"));
        }
    }
    
    // 簡単なメールアドレス検証メソッド
    public static boolean validateEmail(String email) {
        if (email == null || email.trim().isEmpty()) {
            return false;
        }
        
        email = email.trim();
        
        // @マークが一つだけ含まれているか
        int atCount = 0;
        for (int i = 0; i < email.length(); i++) {
            if (email.charAt(i) == '@') {
                atCount++;
            }
        }
        
        if (atCount != 1) {
            return false;
        }
        
        // @マークで分割
        String[] parts = email.split("@");
        if (parts.length != 2) {
            return false;
        }
        
        String localPart = parts[0];  // @より前の部分
        String domainPart = parts[1]; // @より後の部分
        
        // 各部分が空でないか
        if (localPart.isEmpty() || domainPart.isEmpty()) {
            return false;
        }
        
        // ドメイン部分に.が含まれているか
        if (!domainPart.contains(".")) {
            return false;
        }
        
        return true;
    }
}

StringBuilder クラス

大量の文字列操作を行う場合、String の不変性により性能問題が発生することがあります。StringBuilder クラスは、可変な文字列バッファを提供し、効率的な文字列操作を可能にします。

String vs StringBuilder

String：少数の文字列操作に適している。不変なので安全だが、大量の連結では非効率。

StringBuilder：大量の文字列操作に適している。可変なので効率的だが、スレッドセーフではない。

実習 5-2: StringBuilder の活用

効率的な文字列構築を学びましょう。

public class StringBuilderExample {
    public static void main(String[] args) {
        // StringBuilder の基本操作
        StringBuilder sb = new StringBuilder();
        
        System.out.println("=== StringBuilder の基本操作 ===");
        
        // 文字列の追加
        sb.append("Hello");
        sb.append(" ");
        sb.append("World");
        System.out.println("追加後: " + sb.toString());
        
        // 文字列の挿入
        sb.insert(6, "Beautiful ");
        System.out.println("挿入後: " + sb.toString());
        
        // 文字列の削除
        sb.delete(6, 16);
        System.out.println("削除後: " + sb.toString());
        
        // 文字列の置換
        sb.replace(0, 5, "Hi");
        System.out.println("置換後: " + sb.toString());
        
        // 文字列の反転
        sb.reverse();
        System.out.println("反転後: " + sb.toString());
        
        // 実践例：HTMLテーブルの生成
        System.out.println("\n=== 実践例：HTMLテーブルの生成 ===");
        String[][] data = {
            {"名前", "年齢", "職業"},
            {"田中太郎", "25", "エンジニア"},
            {"佐藤花子", "30", "デザイナー"},
            {"山田次郎", "28", "営業"}
        };
        
        String htmlTable = generateHtmlTable(data);
        System.out.println(htmlTable);
        
        // 性能比較デモ
        System.out.println("\n=== 性能比較 ===");
        performanceComparison();
    }
    
    public static String generateHtmlTable(String[][] data) {
        StringBuilder html = new StringBuilder();
        
        html.append("<table border=\"1\">\n");
        
        for (int i = 0; i < data.length; i++) {
            html.append("  <tr>\n");
            
            for (int j = 0; j < data[i].length; j++) {
                if (i == 0) {
                    html.append("    <th>").append(data[i][j]).append("</th>\n");
                } else {
                    html.append("    <td>").append(data[i][j]).append("</td>\n");
                }
            }
            
            html.append("  </tr>\n");
        }
        
        html.append("</table>");
        
        return html.toString();
    }
    
    public static void performanceComparison() {
        int iterations = 10000;
        
        // String での連結（非効率）
        long startTime = System.currentTimeMillis();
        String result1 = "";
        for (int i = 0; i < iterations; i++) {
            result1 += "a";
        }
        long stringTime = System.currentTimeMillis() - startTime;
        
        // StringBuilder での連結（効率的）
        startTime = System.currentTimeMillis();
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < iterations; i++) {
            sb.append("a");
        }
        String result2 = sb.toString();
        long sbTime = System.currentTimeMillis() - startTime;
        
        System.out.println("文字列を" + iterations + "回連結:");
        System.out.println("String による連結: " + stringTime + "ms");
        System.out.println("StringBuilder による連結: " + sbTime + "ms");
        System.out.println("StringBuilder は約 " + (stringTime / (double)sbTime) + " 倍高速");
    }
}

5.3 Math クラスと Random クラス

Math クラス は数学的な計算を、Random クラス は乱数生成を提供します。これらは数値処理プログラムで頻繁に使用されます。

クラス メソッド 説明 例
Math abs(x) 絶対値 Math.abs(-5) → 5
max(a, b) 最大値 Math.max(3, 7) → 7
min(a, b) 最小値 Math.min(3, 7) → 3
pow(a, b) べき乗 Math.pow(2, 3) → 8.0
sqrt(x) 平方根 Math.sqrt(16) → 4.0
round(x) 四捨五入 Math.round(3.7) → 4
random() 0.0-1.0の乱数 Math.random() → 0.847291...
PI, E 数学定数 Math.PI → 3.141592...
Random nextInt(n) 0からn-1の整数乱数 random.nextInt(10) → 0-9
nextDouble() 0.0-1.0の小数乱数 random.nextDouble() → 0.234...
nextBoolean() 真偽値乱数 random.nextBoolean() → true/false
nextGaussian() 正規分布乱数 random.nextGaussian() → -0.234...
実習 5-3: 数学計算と乱数処理

Math と Random を使った実用的なプログラムを作成しましょう。

import java.util.Random;

public class MathAndRandomExample {
    public static void main(String[] args) {
        System.out.println("=== Math クラスの基本操作 ===");
        
        // 基本的な数学関数
        double x = -15.7;
        double y = 8.3;
        
        System.out.println("x = " + x + ", y = " + y);
        System.out.println("絶対値 |x| = " + Math.abs(x));
        System.out.println("最大値 max(x, y) = " + Math.max(x, y));
        System.out.println("最小値 min(x, y) = " + Math.min(x, y));
        System.out.println("べき乗 y^3 = " + Math.pow(y, 3));
        System.out.println("平方根 √y = " + Math.sqrt(y));
        
        // 四捨五入関数
        System.out.println("\n=== 四捨五入関数 ===");
        double value = 3.7;
        System.out.println("元の値: " + value);
        System.out.println("round(): " + Math.round(value));
        System.out.println("floor(): " + Math.floor(value));  // 切り下げ
        System.out.println("ceil(): " + Math.ceil(value));    // 切り上げ
        
        // 三角関数
        System.out.println("\n=== 三角関数 ===");
        double angle = Math.PI / 4;  // 45度（ラジアン）
        System.out.println("角度: " + angle + " ラジアン（45度）");
        System.out.println("sin: " + Math.sin(angle));
        System.out.println("cos: " + Math.cos(angle));
        System.out.println("tan: " + Math.tan(angle));
        
        // Random クラスの使用
        System.out.println("\n=== Random クラスの操作 ===");
        Random random = new Random();
        
        System.out.println("整数乱数（0-9）: " + random.nextInt(10));
        System.out.println("小数乱数（0.0-1.0）: " + random.nextDouble());
        System.out.println("真偽値乱数: " + random.nextBoolean());
        
        // 実践例1：サイコロシミュレーター
        System.out.println("\n=== サイコロシミュレーター ===");
        simulateDiceRolls(1000);
        
        // 実践例2：円周率の近似計算（モンテカルロ法）
        System.out.println("\n=== 円周率の近似計算 ===");
        approximatePi(100000);
        
        // 実践例3：パスワード生成器
        System.out.println("\n=== パスワード生成器 ===");
        for (int i = 0; i < 5; i++) {
            String password = generateRandomPassword(12);
            System.out.println("パスワード " + (i + 1) + ": " + password);
        }
    }
    
    public static void simulateDiceRolls(int trials) {
        Random random = new Random();
        int[] counts = new int[7];  // インデックス1-6を使用
        
        // サイコロを振る
        for (int i = 0; i < trials; i++) {
            int dice = random.nextInt(6) + 1;  // 1-6の乱数
            counts[dice]++;
        }
        
        // 結果を表示
        System.out.println(trials + "回サイコロを振った結果:");
        for (int i = 1; i <= 6; i++) {
            double percentage = (counts[i] * 100.0) / trials;
            System.out.printf("%d: %d回 (%.1f%%)\n", i, counts[i], percentage);
        }
    }
    
    public static void approximatePi(int trials) {
        Random random = new Random();
        int insideCircle = 0;
        
        for (int i = 0; i < trials; i++) {
            double x = random.nextDouble() * 2 - 1;  // -1.0 から 1.0
            double y = random.nextDouble() * 2 - 1;  // -1.0 から 1.0
            
            if (x * x + y * y <= 1.0) {
                insideCircle++;
            }
        }
        
        double piApprox = 4.0 * insideCircle / trials;
        System.out.println(trials + "回の試行による円周率の近似値:");
        System.out.println("計算値: " + piApprox);
        System.out.println("真の値: " + Math.PI);
        System.out.println("誤差: " + Math.abs(piApprox - Math.PI));
    }
    
    public static String generateRandomPassword(int length) {
        Random random = new Random();
        String chars = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789!@#$%^&*";
        StringBuilder password = new StringBuilder();
        
        for (int i = 0; i < length; i++) {
            int index = random.nextInt(chars.length());
            password.append(chars.charAt(index));
        }
        
        return password.toString();
    }
}

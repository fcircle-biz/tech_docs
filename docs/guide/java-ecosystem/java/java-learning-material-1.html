1.1 Java言語とは

Java は1995年にSun Microsystems（現在はOracle Corporation）によって開発されたオブジェクト指向プログラミング言語です。「Write Once, Run Anywhere」（一度書けばどこでも動く）をスローガンに、プラットフォームに依存しないプログラムを作成できることが特徴です。

Javaの主な特徴
  • プラットフォーム独立性：JVM（Java仮想マシン）上で動作するため、異なるOS間での移植性が高い
  • オブジェクト指向：再利用性と保守性に優れたプログラムが作成可能
  • メモリ管理の自動化：ガベージコレクション機能により、メモリリークを防止
  • セキュリティ：サンドボックス機能により安全な実行環境を提供

1.2 JDKのインストール

Javaプログラムを開発するには、JDK（Java Development Kit）をインストールする必要があります。JDKには、Javaコンパイラ（javac）、Java実行環境（JRE）、開発ツールが含まれています。

実習 1-1: JDKのインストール

Oracle JDK または OpenJDK をダウンロードしてインストールしましょう。

手順
  1. Oracle公式サイト（https://www.oracle.com/java/）またはOpenJDK（https://openjdk.java.net/）にアクセス
  2. 最新のJDK（推奨：JDK 11以上）をダウンロード
  3. ダウンロードしたファイルを実行してインストール
  4. 環境変数JAVA_HOMEとPATHを設定
  5. コマンドプロンプトで動作確認
動作確認コマンド
java -version
javac -version
期待される結果

Javaとjavacのバージョン情報が表示されれば、インストール成功です。

1.3 統合開発環境（IDE）の準備

効率的な開発のために、統合開発環境（IDE）を使用することをお勧めします。初学者にはEclipse、IntelliJ IDEA、Visual Studio Codeが人気です。

主要な Java IDE
  • Eclipse：無料で高機能。多くの企業で使用されている
  • IntelliJ IDEA：高度な補完機能とリファクタリング機能
  • Visual Studio Code：軽量で拡張性が高い
  • NetBeans：Oracle公式のIDE

1.4 最初のJavaプログラム

伝統的な「Hello, World!」プログラムを作成して、Javaプログラムの基本構造を理解しましょう。

実習 1-2: Hello Worldプログラムの作成

最初のJavaプログラムを作成し、コンパイル・実行してみましょう。

手順
  1. テキストエディタまたはIDEを開く
  2. HelloWorld.java という名前のファイルを作成
  3. 以下のコードを入力
  4. ファイルを保存
  5. コマンドラインでコンパイル・実行
HelloWorld.javaのコード
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        System.out.println("Javaプログラミングへようこそ！");
    }
}
コンパイルと実行
# コンパイル
javac HelloWorld.java

# 実行
java HelloWorld
期待される結果
Hello, World!
Javaプログラミングへようこそ！

1.5 Javaプログラムの基本構造

作成したHello Worldプログラムの各要素について詳しく解説します。

// クラス宣言
public class HelloWorld {
    
    // メインメソッド（プログラムの開始点）
    public static void main(String[] args) {
        
        // 文字列の出力
        System.out.println("Hello, World!");
        
    }
}
コード解説
  • public class HelloWorld: クラス宣言。ファイル名と同じ名前にする必要があります
  • public static void main(String[] args): メインメソッド。プログラムの開始点
  • System.out.println(): コンソールに文字列を出力するメソッド
  • { }: ブロックを表す中括弧
  • ;: 文の終わりを示すセミコロン

1.6 コンパイルと実行の仕組み

Javaプログラムが動作する仕組みを理解することは重要です。

実習 1-3: 複数のクラスファイル

複数のクラスファイルを持つプログラムを作成してみましょう。

手順
  1. Calculator.java ファイルを作成
  2. 以下のコードを入力
  3. コンパイル・実行して動作を確認
Calculator.javaのコード
public class Calculator {
    public static void main(String[] args) {
        int a = 10;
        int b = 5;
        
        System.out.println("a = " + a);
        System.out.println("b = " + b);
        System.out.println("a + b = " + (a + b));
        System.out.println("a - b = " + (a - b));
        System.out.println("a * b = " + (a * b));
        System.out.println("a / b = " + (a / b));
    }
}
期待される結果
a = 10
b = 5
a + b = 15
a - b = 5
a * b = 50
a / b = 2

  1. Javaプログラムが「Write Once, Run Anywhere」を実現できる理由は何ですか？

  3. Javaソースファイルの拡張子と、コンパイル後のファイルの拡張子をそれぞれ答えてください。
  4. public static void main(String[] args) の各要素の意味を説明してください。
  5. System.out.println() とSystem.out.print() の違いは何ですか？

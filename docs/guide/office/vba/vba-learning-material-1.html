第1章: VBAの基本とマクロ記録

VBAとは

この章で学ぶこと
  • VBAの概要と特徴
  • Excel VBAの開発環境設定
  • マクロの記録機能の活用
  • Visual Basic Editorの基本操作

1.1 VBAの概要

VBA（Visual Basic for Applications）は、Microsoft Officeアプリケーション（Excel、Word、Access、PowerPointなど）を 自動化・カスタマイズするためのプログラミング言語です。Excelの操作を自動化することで、 繰り返し作業の効率化や複雑なデータ処理を可能にします。

1.2 VBAの特徴

  • Office統合: Excel、Word、Accessなどと完全に統合
  • 自動化: 繰り返し作業を自動実行
  • カスタマイズ: 独自の機能やツールを作成
  • イベント駆動: セルの変更やボタンクリックに応答
  • 学習しやすさ: 比較的簡単な文法

1.3 開発環境の設定

Excel VBAを使用するための設定手順：

ステップ1: 開発タブの表示

  1. Excelを開く
  2. 「ファイル」→「オプション」をクリック
  3. 「リボンのユーザー設定」を選択
  4. 「開発」チェックボックスをオンにする
  5. 「OK」をクリック

ステップ2: マクロ有効ブックとして保存

  1. ファイル →「名前を付けて保存」をクリック
  2. 「ファイルの種類」で「Excelマクロ有効ブック（*.xlsm）」を選択
  3. ファイル名を入力して「保存」をクリック
  4. マクロ含むファイルを開く際は、「コンテンツの有効化」をクリック
💡 セキュリティのヒント

xlsm拡張子を使用することで、マクロの有無が明確になります。信頼できるソースからのファイルのみ「コンテンツの有効化」を行ってください。

1.4 マクロの記録

VBA学習の最初のステップとして、マクロ記録機能を使ってみましょう。

実習 1-1: 初めてのマクロ記録
  1. 新しいExcelブックを開く
  2. 「開発」タブ →「マクロの記録」をクリック
  3. マクロ名を「初めてのマクロ」に設定
  4. 「OK」をクリックして記録開始
  5. 以下の操作を実行：
    • セルA1に「Hello」と入力
    • セルB1に「VBA」と入力
    • セルA1とB1を選択して太字に設定
  6. 「記録終了」をクリック

1.5 Visual Basic Editorの起動

記録したマクロのコードを確認してみましょう。

  1. 「開発」タブ →「Visual Basic」をクリック
  2. または Alt + F11 を押す
  3. Visual Basic Editor（VBE）が起動します

1.6 記録されたコードの確認

VBEの左側のプロジェクトエクスプローラーで「標準モジュール」→「Module1」をダブルクリックすると、 記録されたVBAコードが表示されます。

Sub 初めてのマクロ() ' ' 初めてのマクロ Macro ' Range("A1").Select ActiveCell.FormulaR1C1 = "Hello" Range("B1").Select ActiveCell.FormulaR1C1 = "VBA" Range("A1:B1").Select Selection.Font.Bold = True End Sub
実習 1-2: 最初のVBAプログラム作成

新しいマクロを手動で作成してみましょう：

Sub 挨拶メッセージ() MsgBox "VBA学習へようこそ！" End Sub

このコードをModule1に入力し、F5キーを押して実行してみてください。

1.7 VBEの主要な機能

  • プロジェクトエクスプローラー: ワークブックとモジュールの管理
  • コードウィンドウ: VBAコードの編集
  • プロパティウィンドウ: オブジェクトのプロパティ表示
  • イミディエイトウィンドウ: コードのテスト実行

第3章: 制御構造

制御構造

この章で学ぶこと
  • If文による条件分岐
  • Select Case文による多分岐
  • For文による繰り返し処理
  • Do文による条件付き繰り返し
  • Exit文による制御の終了

3.1 If文による条件分岐

If文は条件に基づいて処理を分岐させるために使用します。

If 条件 Then ' 条件が真の場合の処理 ElseIf 条件2 Then ' 条件2が真の場合の処理 Else ' どの条件も真でない場合の処理 End If
実習 3-1: 成績評価システム
Sub 成績評価() Dim 点数 As Integer Dim 評価 As String 点数 = InputBox("点数を入力してください（0-100）:") If 点数 >= 90 Then 評価 = "A" ElseIf 点数 >= 80 Then 評価 = "B" ElseIf 点数 >= 70 Then 評価 = "C" ElseIf 点数 >= 60 Then 評価 = "D" Else 評価 = "F" End If MsgBox "点数: " & 点数 & "点" & vbCrLf & "評価: " & 評価 End Sub

3.2 Select Case文

複数の値を比較する場合は、Select Case文が便利です。

Select Case 変数 Case 値1 ' 処理1 Case 値2, 値3 ' 処理2 Case Is > 値4 ' 処理3 Case Else ' その他の処理 End Select
実習 3-2: 曜日判定
Sub 曜日判定() Dim 曜日番号 As Integer Dim 曜日名 As String 曜日番号 = Weekday(Now) Select Case 曜日番号 Case 1 曜日名 = "日曜日" Case 2 曜日名 = "月曜日" Case 3 曜日名 = "火曜日" Case 4 曜日名 = "水曜日" Case 5 曜日名 = "木曜日" Case 6 曜日名 = "金曜日" Case 7 曜日名 = "土曜日" End Select MsgBox "今日は" & 曜日名 & "です" End Sub

3.3 For文による繰り返し処理

決められた回数だけ処理を繰り返すにはFor文を使用します。

For カウンタ = 開始値 To 終了値 Step 増分値 ' 繰り返し処理 Next カウンタ
実習 3-3: 九九表の作成
Sub 九九表作成() Dim i As Integer, j As Integer Dim 結果 As String For i = 1 To 9 結果 = "" For j = 1 To 9 結果 = 結果 & i & "×" & j & "=" & i * j & vbTab Next j Debug.Print 結果 Next i MsgBox "九九表をイミディエイトウィンドウに出力しました" End Sub

3.4 Do文による条件付き繰り返し

条件が満たされている間、処理を繰り返します。

' 条件が真の間繰り返し Do While 条件 ' 繰り返し処理 Loop ' 条件が偽の間繰り返し Do Until 条件 ' 繰り返し処理 Loop
実習 3-4: 数値推測ゲーム
Sub 数値推測ゲーム() Dim 正解 As Integer Dim 推測 As Integer Dim 回数 As Integer 正解 = Int(Rnd() * 100) + 1 回数 = 0 Do 推測 = InputBox("1から100の数値を推測してください:") 回数 = 回数 + 1 If 推測 < 正解 Then MsgBox "もっと大きな数です" ElseIf 推測 > 正解 Then MsgBox "もっと小さな数です" Else MsgBox "正解！" & 回数 & "回目で当てました" Exit Do End If Loop End Sub

3.5 Exit文による制御の終了

  • Exit Sub: プロシージャの終了
  • Exit Function: ファンクションの終了
  • Exit For: For文の終了
  • Exit Do: Do文の終了

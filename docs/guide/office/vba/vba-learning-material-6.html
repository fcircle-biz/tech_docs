6.1 セルの値の読み書き

Excel VBAでは、セルの値を読み取ったり書き込んだりすることが基本的な操作です。

実習 6-1: セルの値操作
Sub セル値操作() Dim ws As Worksheet Set ws = ThisWorkbook.Worksheets("Sheet1") ' セルに値を設定 ws.Range("A1").Value = "商品名" ws.Range("B1").Value = "単価" ws.Range("C1").Value = "数量" ws.Range("D1").Value = "合計" ' データの入力 ws.Range("A2").Value = "りんご" ws.Range("B2").Value = 100 ws.Range("C2").Value = 5 ' 計算結果の設定 ws.Range("D2").Formula = "=B2*C2" ' セルの値を読み取り Dim 商品名 As String Dim 単価 As Integer Dim 数量 As Integer Dim 合計 As Integer 商品名 = ws.Range("A2").Value 単価 = ws.Range("B2").Value 数量 = ws.Range("C2").Value 合計 = ws.Range("D2").Value MsgBox 商品名 & "の合計: " & 合計 & "円" End Sub

6.2 セルの書式設定

セルの見た目を変更することで、データを見やすく整理できます。

実習 6-2: セルの書式設定
Sub セル書式設定() Dim ws As Worksheet Dim rng As Range Set ws = ThisWorkbook.Worksheets("Sheet1") ' ヘッダー行の書式設定 Set rng = ws.Range("A1:D1") With rng .Font.Bold = True .Font.Color = RGB(255, 255, 255) .Interior.Color = RGB(0, 100, 200) .HorizontalAlignment = xlCenter End With ' データ行の書式設定 Set rng = ws.Range("A2:D10") With rng .Font.Size = 10 .HorizontalAlignment = xlCenter .Borders.LineStyle = xlContinuous End With ' 数値列の書式設定 ws.Range("B2:D10").NumberFormat = "#,##0" ' 列幅の自動調整 ws.Columns("A:D").AutoFit MsgBox "書式設定が完了しました" End Sub

6.3 シートの操作

複数のシートを効率的に管理することで、データを整理できます。

実習 6-3: シートの管理
Sub シート管理() Dim ws As Worksheet Dim 月名 As String Dim i As Integer ' 月別シートの作成 For i = 1 To 12 月名 = Format(DateSerial(2024, i, 1), "mm月") Set ws = ThisWorkbook.Worksheets.Add ws.Name = 月名 ' シートにヘッダーを設定 ws.Range("A1").Value = "日付" ws.Range("B1").Value = "売上" ws.Range("C1").Value = "利益" ' ヘッダーの書式設定 With ws.Range("A1:C1") .Font.Bold = True .Interior.Color = RGB(200, 200, 200) End With Next i ' 集計シートの作成 Set ws = ThisWorkbook.Worksheets.Add ws.Name = "年間集計" ws.Move After:=ThisWorkbook.Worksheets(ThisWorkbook.Worksheets.Count) MsgBox "月別シートと集計シートを作成しました" End Sub

6.4 ブックの操作

複数のブックを操作して、データを統合・分析できます。

実習 6-4: ブックの操作
Sub ブック操作() Dim 元ブック As Workbook Dim 新ブック As Workbook Dim ファイル名 As String Set 元ブック = ThisWorkbook ' 新しいブックを作成 Set 新ブック = Workbooks.Add ' データのコピー 元ブック.Worksheets("Sheet1").Range("A1:D10").Copy 新ブック.Worksheets(1).Range("A1").PasteSpecial xlPasteValues ' ファイル名を設定して保存 ファイル名 = "データ集計_" & Format(Now, "yyyymmdd") & ".xlsx" 新ブック.SaveAs 元ブック.Path & "\" & ファイル名 ' ブックを閉じる 新ブック.Close ' クリップボードをクリア Application.CutCopyMode = False MsgBox "データを " & ファイル名 & " に保存しました" End Sub

6.5 データの検索・並べ替え・フィルタ

大量のデータを効率的に処理する方法を学びます。

実習 6-5: データ処理
Sub データ処理() Dim ws As Worksheet Dim rng As Range Dim 検索値 As String Dim 見つかったセル As Range Set ws = ThisWorkbook.Worksheets("Sheet1") Set rng = ws.Range("A1:D100") ' データの検索 検索値 = "りんご" Set 見つかったセル = rng.Find(検索値, LookIn:=xlValues) If Not 見つかったセル Is Nothing Then 見つかったセル.Interior.Color = RGB(255, 255, 0) MsgBox 検索値 & "が見つかりました: " & 見つかったセル.Address Else MsgBox 検索値 & "は見つかりませんでした" End If ' データの並べ替え rng.Sort Key1:=ws.Range("B1"), Order1:=xlDescending, Header:=xlYes ' オートフィルタの設定 If ws.AutoFilterMode = False Then rng.AutoFilter End If ' 条件でフィルタ ws.Range("A1:D100").AutoFilter Field:=2, Criteria1:=">100" MsgBox "データ処理が完了しました" End Sub

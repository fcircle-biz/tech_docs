クラウドって何？AWSを身近な例えで理解しよう

「クラウド」という言葉を聞いたことはあるけど、よくわからない...そんな方のために、レンタルサービスに例えながらクラウドの基本を解説します。

この章で学ぶこと

  • クラウドとは「インターネット上のレンタルサービス」
  • 自分でパソコンを買う（オンプレミス）との違い
  • AWSは「世界最大のITレンタルショップ」
  • なぜ今、多くの会社がクラウドを使うのか
  • 「リージョン」は倉庫の場所、「サービス」は商品の種類

1.1 クラウドとは何か？

クラウドを一言で表すと、「インターネット経由で使えるコンピュータ機能のレンタルサービス」です。

身近な例え

クラウド = 図書館（必要な本を必要なときだけ借りる）
オンプレミス = 自分で本を全部買って家に置く

普段、会社や家庭で使うパソコンは「手元にある機械」ですよね。これを「オンプレミス（自前で保有）」と呼びます。 一方、クラウドは「遠くにあるデータセンターのパソコンをインターネット経由で借りて使う」イメージです。

1.2 AWSとは何か？

AWS（Amazon Web Services）は、Amazonが提供する世界最大のクラウドサービスです。

重要ポイント

AWSは「世界中に支店がある超巨大図書館」のようなもので、200種類以上のサービス（本棚）を提供しています。

AWSの3つの魅力

1. 初期費用が安い

サーバーを買わなくてOK。使った分だけ後払い。

2. 柔軟に拡張できる

アクセスが増えたら数クリックで性能UP。減ったら縮小も可能。

3. 世界中で使える

日本、アメリカ、ヨーロッパなど世界中にデータセンター。

1.3 リージョンとサービス

AWSには2つの重要な概念があります。「リージョン」と「サービス」です。

リージョン = データセンターの場所

リージョンとは、AWSのデータセンターがある地域のことです。例えば「東京リージョン」「アメリカ東海岸リージョン」など。 日本から使う場合は、通信速度が速い「東京」を選ぶのが一般的です。

ヒント

リージョン = レンタルビデオ店の支店。近い支店（東京）で借りた方が受け取りが早いですよね。

サービス = 借りられる機能の種類

AWSには200種類以上の「サービス」があります。このガイドでは、初心者向けの主要サービスを学びます。

このガイドで学ぶ主要サービス

EC2

レンタルパソコン（サーバー）

S3

ファイル保存庫（無制限）

RDS

データベース（整理されたデータ保存）

IAM

アクセス権限管理

CloudWatch

監視ツール（サーバーの健康診断）

Lambda

サーバーレス実行環境

1.4 なぜ今、クラウドが注目されるのか？

クラウドが登場する前、企業はサーバーを自社で購入して管理していました（オンプレミス）。 しかし、これにはいくつかの課題がありました。

オンプレミスの課題

  • 初期費用が高い（数百万円〜）
  • 設置場所が必要（サーバールーム）
  • 故障したら自分で修理
  • 使わなくても維持費がかかる
  • 急にアクセスが増えると対応できない

クラウドはこれらの課題を解決します。

クラウドのメリット

  • 初期費用ゼロ - 使った分だけ後払い
  • スケーラブル - アクセスが増えても数分で対応
  • メンテナンス不要 - AWSが機械を管理してくれる
  • 世界展開が簡単 - 海外進出もすぐできる

1.1 AWS Certified Developer - Associate（DVA-C02）とは

AWS Certified Developer - Associate（DVA-C02）は、AWS上でアプリケーションを開発、デプロイ、デバッグできる能力を証明する中級レベルの認定資格です。この資格は、インフラ設計に重点を置くSolutions Architect Associateとは異なり、**開発者としての実装スキル**に焦点を当てています。

重要ポイント

DVA-C02試験では、AWSサービスの「使い方」だけでなく、「なぜそのサービスを選ぶべきか」「どのように組み合わせるか」といった**設計判断**も問われます。

試験の基本情報

  • 試験時間: 130分
  • 問題数: 65問（選択問題と複数選択問題の組み合わせ）
  • 合格ライン: 1000点満点中720点以上
  • 試験形式: オンライン試験またはテストセンター
  • 有効期間: 3年間
  • 受験料: $150 USD（税別）

1.2 試験の4つのドメイン

DVA-C02試験は以下の4つのドメイン（分野）で構成されています。各ドメインの出題割合を理解し、重点的に学習する分野を決めましょう。

pie title DVA-C02試験のドメイン別出題割合 "ドメイン1: AWSサービスを使用した開発" : 32 "ドメイン2: セキュリティ" : 26 "ドメイン3: デプロイ" : 24 "ドメイン4: トラブルシューティングと最適化" : 18

ドメイン1: AWSサービスを使用した開発（32%）

このドメインでは、サーバーレスアーキテクチャ、コンテナ技術、マイクロサービスの実装に関する知識が問われます。

  • AWS Lambda関数の開発とデプロイ
  • Amazon API Gatewayを使用したRESTful API設計
  • Amazon DynamoDBによるNoSQLデータベース設計
  • AWS Step Functionsによるワークフローオーケストレーション
  • EventBridge、SQS、SNSによるイベント駆動アーキテクチャ

ドメイン2: セキュリティ（26%）

アプリケーションレベルのセキュリティ実装、認証・認可、データ保護に関する知識が必要です。

  • IAMロールとポリシーの実装
  • Amazon CognitoによるユーザープールとIDプール
  • AWS Secrets ManagerとParameter Storeによるシークレット管理
  • AWS KMSによる暗号化キー管理
  • 署名付きURLとCookie

ドメイン3: デプロイ（24%）

CI/CDパイプライン、Infrastructure as Code（IaC）、デプロイ戦略に関する知識が求められます。

  • AWS CodePipeline、CodeBuild、CodeDeployによるCI/CD
  • AWS SAM、AWS CDKによるIaC
  • ブルー/グリーンデプロイとカナリアデプロイ
  • Elastic Beanstalkによるアプリケーションデプロイ

ドメイン4: トラブルシューティングと最適化（18%）

CloudWatch、X-Rayを使用した監視、ログ分析、パフォーマンス最適化に関する知識が必要です。

  • CloudWatch LogsとLogsInsightsによるログ分析
  • AWS X-Rayによる分散トレーシング
  • Lambda関数のパフォーマンスチューニング
  • DynamoDBのスループット最適化

1.3 SAA-C03との違い - 開発者視点とアーキテクト視点

AWS Certified Solutions Architect - Associate（SAA-C03）とDVA-C02の主な違いは、**視点の違い**です。SAAはインフラ設計者、DVA-C02は開発者の視点で問題を解決します。

flowchart LR subgraph SAA["SAA-C03（Solutions Architect）"] direction TB S1["インフラ設計"] S2["高可用性・スケーラビリティ"] S3["コスト最適化"] S4["マルチリージョン/マルチAZ"] end subgraph DVA["DVA-C02（Developer）"] direction TB D1["アプリケーション開発"] D2["サーバーレス実装"] D3["CI/CDパイプライン"] D4["監視・デバッグ"] end SAA -.インフラ視点.- DVA DVA -.開発視点.- SAA

問題の違いの具体例

SAA-C03の典型的な問題

問: 複数リージョンにまたがる高可用性Webアプリケーションを設計する必要があります。最もコスト効率的なアーキテクチャはどれですか？

焦点: インフラの冗長性、Route 53、CloudFront、マルチリージョンRDS

DVA-C02の典型的な問題

問: Lambda関数がDynamoDBにアクセスする際、環境変数に保存されたデータベース接続情報を暗号化する必要があります。最も安全な方法はどれですか？

焦点: Lambda実装、KMS暗号化、環境変数の管理、Secrets Manager

学習期間の目安

  • AWS初心者: 3〜4ヶ月（週10〜15時間）
  • AWS基礎知識あり: 2〜3ヶ月（週10〜12時間）
  • SAA-C03合格済み: 1.5〜2ヶ月（週8〜10時間）

ヒント: ハンズオンの重要性

DVA-C02試験では、実際にコードを書いてデプロイした経験が問われます。AWS無料利用枠を活用して、必ず実際にLambda関数を作成し、API Gatewayと連携させるハンズオンを行いましょう。

理解度確認クイズ

  1. Q1. DVA-C02試験で最も出題割合が高いドメインはどれですか？

    回答を表示

    ドメイン1: AWSサービスを使用した開発（32%）
    このドメインでは、Lambda、API Gateway、DynamoDB、Step Functionsなどのサーバーレスサービスの実装が中心となります。

  2. Q2. SAA-C03とDVA-C02の最も大きな違いは何ですか？

    回答を表示

    SAA-C03はインフラ設計者（アーキテクト）視点、DVA-C02は開発者視点という違いがあります。SAAでは高可用性やマルチリージョン設計が重視されるのに対し、DVA-C02ではアプリケーションの実装、デプロイ、デバッグが重視されます。

  3. Q3. DVA-C02試験対策で最も重要な学習方法は何ですか？

    回答を表示

    理論学習とハンズオン実践の両立が最も重要です。公式ドキュメントで理論を学び、AWS無料利用枠を活用して実際にLambda関数を作成し、API Gatewayと連携させるなどの実践を行うことで、試験で問われる実装の理解が深まります。

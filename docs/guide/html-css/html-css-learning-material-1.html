1.1 インターネットとWebサイトの仕組み

インターネットとは？

インターネットは、世界中のコンピュータがつながった巨大なネットワークです。 家庭や会社、学校など、様々な場所のコンピュータが相互に通信できるようになっています。

身近な例で考えてみよう

インターネットは「情報の道路網」のようなものです。道路で車が行き来するように、 インターネット上ではデータが行き来しています。あなたがスマホでWebサイトを見るとき、 そのデータはこの「情報の道路」を通って届いているのです。

Webサイトとは？

Webサイトは、インターネット上で公開されている情報のまとまりです。 ニュースサイト、SNS、オンラインショップなど、様々なWebサイトが存在します。

Webサイトは複数の「Webページ」から構成されており、それぞれのページがHTMLという言語で書かれています。

1.2 ブラウザとサーバーの関係

ブラウザとは？

ブラウザは、Webサイトを見るためのソフトウェアです。 Chrome、Firefox、Safari、Edgeなどが代表的なブラウザです。

サーバーとは？

サーバーは、Webページのデータを保管して、ブラウザからのリクエストに応じてデータを送り返すコンピュータです。 世界中のどこかにあるサーバーが、24時間365日休まず動いています。

クライアント・サーバーモデル

あなたのパソコンやスマホ（クライアント）がブラウザでWebサイトにアクセスすると、 サーバーがそのページのデータを送り返してくれます。これを「クライアント・サーバーモデル」と呼びます。

1.3 HTMLとCSSの役割

HTMLとは？

HTML（HyperText Markup Language）は、Webページの「構造」と「内容」を定義する言語です。 見出し、段落、画像、リンクなど、ページに何があるかを記述します。

HTMLの例え

HTMLは家の「骨組み」のようなものです。どこに壁があって、どこにドアがあって、 どこに窓があるかを決める設計図のイメージです。

CSSとは？

CSS（Cascading Style Sheets）は、Webページの「見た目」や「デザイン」を定義する言語です。 色、フォント、レイアウト、余白など、ページの装飾を記述します。

CSSの例え

CSSは家の「内装」や「外装」のようなものです。壁の色を塗ったり、 カーテンをつけたり、家具を配置したりするイメージです。

見出し、段落、画像など] C --> E[見た目・デザイン
色、フォント、レイアウト]

HTMLとCSSの関係

HTMLとCSSは密接に連携して動作します：

  • HTML: 「これは見出しです」「これは段落です」と宣言
  • CSS: 「見出しは赤色にしよう」「段落は14ピクセルの文字にしよう」と装飾

1.4 エディタの準備とセットアップ

テキストエディタとは？

テキストエディタは、HTMLやCSSのコードを書くためのソフトウェアです。 Windowsの「メモ帳」でも書けますが、コードを書くのに便利な専用エディタを使うことをお勧めします。

推奨エディタ：Visual Studio Code

Visual Studio Code（VS Code）は、Microsoftが提供する無料のテキストエディタです。 初心者から上級者まで幅広く使われており、以下のような特徴があります：

  • 完全無料で使える
  • Windows、Mac、Linuxすべてで動作
  • コードの色分け表示（シンタックスハイライト）
  • 自動補完機能でタイピングが楽
  • 拡張機能で機能追加が可能
実習 1-1: Visual Studio Codeのインストール

Visual Studio Codeをインストールしてみましょう。

手順
  1. Webブラウザで https://code.visualstudio.com/ にアクセス
  2. 「Download for Windows」（またはMac/Linux）ボタンをクリック
  3. ダウンロードしたファイルを実行
  4. インストーラーの指示に従ってインストール
  5. インストール完了後、VS Codeを起動
確認

VS Codeが正常に起動すれば成功です！

1.5 最初のHTMLファイルを作成

HTMLファイルの基本構造

すべてのHTMLファイルには、決まった基本構造があります。 まずはこの構造を覚えましょう。 

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>ページのタイトル</title>
</head>
<body>
    <!-- ここにページの内容を書きます -->
</body>
</html>

各部分の説明

  • <!DOCTYPE html>: このファイルがHTML5であることを宣言
  • <html lang="ja">: HTML文書の開始、日本語ページであることを示す
  • <head>: ページの情報（タイトル、文字コードなど）を記述
  • <meta charset="UTF-8">: 文字コードをUTF-8に設定（日本語対応）
  • <title>: ブラウザのタブに表示されるページタイトル
  • <body>: ページに表示される内容を記述
実習 1-2: Hello Worldページを作成

最初のHTMLファイルを作成して、ブラウザで表示してみましょう。

手順
  1. VS Codeを起動
  2. 「ファイル」→「新規ファイル」を選択
  3. 以下のコードを入力（コピー＆ペーストでもOK）
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>はじめてのWebページ</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>これが私の最初のWebページです。</p>
</body>
</html>
  1. 「ファイル」→「名前を付けて保存」を選択
  2. ファイル名を hello.html として保存（好きな場所に保存してOK）
  3. 保存したファイルをダブルクリック、またはブラウザにドラッグ＆ドロップ
期待される結果

ブラウザに「Hello, World!」という見出しと「これが私の最初のWebページです。」という文章が表示されます。

おめでとうございます！

これであなたも立派なWeb開発者の仲間入りです！ たった数行のコードで、ブラウザに文字を表示できました。

1.6 自己紹介ページを作成しよう

次は、もう少し情報を追加して、自己紹介ページを作ってみましょう。

実習 1-3: 自己紹介ページの作成

自分の名前、好きなこと、趣味などを含む自己紹介ページを作りましょう。

手順
  1. VS Codeで新しいファイルを作成
  2. 以下のテンプレートを参考に、自分の情報に書き換える
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>自己紹介ページ</title>
</head>
<body>
    <h1>自己紹介</h1>

    <h2>私について</h2>
    <p>はじめまして！私の名前は○○です。</p>
    <p>Web制作に興味を持ち、HTMLとCSSの学習を始めました。</p>

    <h2>好きなこと</h2>
    <p>読書、音楽鑑賞、散歩が好きです。</p>

    <h2>学習の目標</h2>
    <p>自分のポートフォリオサイトを作れるようになることが目標です。</p>
</body>
</html>
  1. ファイル名を profile.html として保存
  2. ブラウザで開いて確認
カスタマイズのポイント
  • <h1>は最も大きな見出し、<h2>はその次のレベルの見出し
  • <p>は段落（paragraph）を表すタグ
  • 自分の情報に書き換えて、オリジナリティを出してみましょう

  • HTMLファイルの基本構造

重要なポイント
  • HTMLは構造と内容を定義する（家の骨組み）
  • CSSは見た目とデザインを定義する（家の内装・外装）
  • すべてのHTMLファイルには基本構造がある
  • <h1>は見出し、<p>は段落を表す
理解度確認クイズ
  1. 問題1: HTMLとCSSの役割の違いを説明してください。
    解答例

    HTMLはWebページの構造と内容を定義し、CSSは見た目やデザインを定義します。 HTMLが「何を表示するか」を決め、CSSが「どのように表示するか」を決めます。

  2. 問題2: ブラウザとサーバーの関係を説明してください。
    解答例

    ブラウザ（クライアント）がWebページをリクエストすると、サーバーがそのページのデータを送り返します。 ブラウザは受け取ったHTMLを解釈して、画面に表示します。

  3. 問題3: HTMLファイルの基本構造に必ず含まれる3つの主要タグは何ですか？
    解答例

    <html>、<head>、<body>の3つです。

  4. 問題4: <h1>タグと<p>タグの違いは何ですか？
    解答例

    <h1>は見出しを表すタグで、ページの主要なタイトルに使用します。 <p>は段落を表すタグで、通常の文章を書くときに使用します。

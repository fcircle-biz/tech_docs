8.1 CSSフレームワークとは？

CSSフレームワークは、予め用意されたCSSクラスやコンポーネントの集合体です。 ゼロから書く手間を省き、美しいWebサイトを素早く構築できます。

CSSフレームワークの利点
  • 開発速度の向上（既製のコンポーネント利用）
  • 一貫したデザイン（統一されたスタイル）
  • レスポンシブ対応が容易
  • ブラウザ互換性の問題が少ない
  • 豊富なドキュメントとコミュニティ

主要なCSSフレームワーク

  • Bootstrap: 最も人気、コンポーネント豊富
  • Tailwind CSS: ユーティリティファースト、カスタマイズ性高い
  • Foundation: プロフェッショナル向け
  • Bulma: シンプルで軽量

8.2 Bootstrap 5の基礎

Bootstrap 5の導入

CDN経由で簡単に導入できます。

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Example</title>

    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- コンテンツ -->

    <!-- Bootstrap JavaScript（オプション） -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

8.3 Bootstrapグリッドシステム

Bootstrapのグリッドシステムは、12カラムのレイアウトシステムです。 

<div class="container">
    <div class="row">
        <!-- 3カラム（1/4） -->
        <div class="col-md-3">サイドバー</div>

        <!-- 9カラム（3/4） -->
        <div class="col-md-9">メインコンテンツ</div>
    </div>
</div>
グリッドのブレークポイント
  • col-: 全てのデバイス
  • col-sm-: スマホ（576px以上）
  • col-md-: タブレット（768px以上）
  • col-lg-: PC（992px以上）
  • col-xl-: 大画面（1200px以上）

レスポンシブグリッド

<div class="row">
    <!-- スマホ: 12/12、タブレット: 6/12、PC: 4/12 -->
    <div class="col-12 col-md-6 col-lg-4">カラム1</div>
    <div class="col-12 col-md-6 col-lg-4">カラム2</div>
    <div class="col-12 col-md-6 col-lg-4">カラム3</div>
</div>

8.5 Tailwind CSS入門

Tailwind CSSは、ユーティリティファーストのCSSフレームワークです。 HTMLに直接クラスを書いてスタイリングします。

Tailwind CSSの導入

<script src="https://cdn.tailwindcss.com"></script>

Tailwind CSSの使用例

<!-- ボタン -->
<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">
    ボタン
</button>

<!-- カード -->
<div class="max-w-sm rounded overflow-hidden shadow-lg">
    <img class="w-full" src="image.jpg" alt="...">
    <div class="px-6 py-4">
        <div class="font-bold text-xl mb-2">タイトル</div>
        <p class="text-gray-700 text-base">説明文</p>
    </div>
</div>
BootstrapとTailwind CSSの比較
特徴 Bootstrap Tailwind CSS
アプローチ コンポーネントベース ユーティリティファースト
学習曲線 緩やか やや急
カスタマイズ 限定的 高い
ファイルサイズ 大きい 小さい（PurgeCSS使用時）

理解度確認クイズ
  1. 問題1: CSSフレームワークを使う利点は？
    解答例

    開発速度の向上、一貫したデザイン、レスポンシブ対応の容易さ、ブラウザ互換性の問題が少ないことなどです。

  2. 問題2: Bootstrapのグリッドシステムは何カラム？
    解答例

    12カラムシステムです。

  3. 問題3: BootstrapとTailwind CSSの最大の違いは？

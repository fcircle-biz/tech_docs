第2章: 変数とデータ型

データを格納し、操作するための基本要素

この章で学ぶこと
  • 変数とは何か、なぜ必要なのかを理解する
  • PHPでの変数の定義方法と命名規則
  • PHPの主要なデータ型（文字列、数値、真偽値、null）
  • 文字列と数値の基本的な操作方法
  • 変数の型変換とvar_dump()による型確認

2.1 変数とは

変数は、データを一時的に保存するための「箱」のようなものです。この箱には名前（変数名）をつけて、後でその名前を使ってデータを取り出したり、新しいデータに置き換えたりできます。

PHPでの変数定義

PHPでは、変数名の前に$記号を付けます。変数の宣言は不要で、最初に値を代入したときに自動的に作成されます。

<?php
// 文字列を格納
$name = "田中太郎";

// 数値を格納
$age = 25;

// 変数の値を表示
echo "名前: " . $name;
echo "年齢: " . $age;

2.2 変数の命名規則

PHPの変数名には以下のルールがあります：

  • $記号で始まる
  • 2文字目は英字またはアンダースコア（_）で始まる
  • 3文字目以降は英数字とアンダースコア
  • 大文字と小文字は区別される（$nameと$Nameは別の変数）
  • 日本語の変数名も使用可能だが推奨されない

良い変数名の例

<?php
$userName = "user123";     // キャメルケース（推奨）
$user_name = "user123";    // スネークケース
$total_price = 1000;       // 意味が明確
$_private_var = "secret";  // アンダースコア開始（特殊用途）

悪い変数名の例

<?php
$1name = "error";      // 数字から始まっている（エラー）
$user-name = "error";  // ハイフンは使用不可（エラー）
$a = "不明確";          // 意味が不明確
$あ = "日本語";         // 日本語（動作するが推奨されない）
実習 2-1: 変数の基本操作

様々な変数を定義して、その値を表示してみましょう。

手順
  1. 「variables.php」ファイルを作成する
  2. 以下のコードを入力する
  3. ブラウザで実行して結果を確認する
実行例
<?php
// 個人情報の変数
$firstName = "田中";
$lastName = "太郎";
$age = 30;
$height = 175.5;

// 変数の値を表示

期待される結果

プロフィール情報が整理されて表示され、年齢が1歳増加して表示されます。

2.3 PHPのデータ型

PHPは動的型付け言語ですが、内部的には以下のデータ型を持っています：

基本的なデータ型

データ型 説明 例
string（文字列） 文字の集合 "Hello World"
integer（整数） 小数点のない数値 42, -100
float（浮動小数点） 小数点のある数値 3.14, -2.5
boolean（真偽値） true または false true, false
null 値が存在しない状態 null

2.4 文字列の操作

文字列は単一引用符（'）または二重引用符（"）で囲んで定義します。

文字列の定義方法

<?php
// 単一引用符
$message1 = 'Hello World';

// 二重引用符
$message2 = "Hello World";

// 変数の展開（二重引用符のみ）
$name = "太郎";
$greeting = "こんにちは、{$name}さん";  // こんにちは、太郎さん

// 文字列の連結
$fullMessage = $message1 . " " . $name;  // Hello World 太郎

単一引用符と二重引用符の違い

<?php
$name = "田中";

// 二重引用符：変数が展開される
echo "名前は $name です";      // 結果: 名前は 田中 です

// 単一引用符：変数は展開されない
echo '名前は $name です';      // 結果: 名前は $name です

2.5 数値の操作

PHPでは整数と浮動小数点数を扱うことができます。

基本的な算術演算

<?php
$a = 10;
$b = 3;

echo $a + $b;  // 加算: 13
echo $a - $b;  // 減算: 7
echo $a * $b;  // 乗算: 30
echo $a / $b;  // 除算: 3.3333...
echo $a % $b;  // 剰余（余り）: 1

// 指数演算（PHP 5.6以降）
echo $a ** $b; // べき乗: 1000 (10の3乗)
実習 2-2: 計算機プログラム

基本的な計算機能を実装してみましょう。

手順
  1. 「calculator.php」ファイルを作成する
  2. 以下のコードを入力する
  3. 様々な値で実行して結果を確認する
実行例
<?php
// 商品の価格計算
$price = 1200;        // 商品単価
$quantity = 5;        // 数量
$taxRate = 0.1;       // 消費税率（10%）

// 計算
$subtotal = $price * $quantity;           // 小計
$tax = $subtotal * $taxRate;              // 消費税
$total = $subtotal + $tax;                // 総計

// 結果表示
echo "<h2>お会計</h2>";
echo "<p>単価: ¥" . number_format($price) . "</p>";
echo "<p>数量: " . $quantity . "個</p>";
echo "<p>小計: ¥" . number_format($subtotal) . "</p>";
echo "<p>消費税: ¥" . number_format($tax) . "</p>";
echo "<p><strong>総計: ¥" . number_format($total) . "</strong></p>";
期待される結果

商品価格、数量、税込み価格が整理されて表示されます。number_format()関数により、数値にカンマが挿入されます。

2.6 真偽値とnull

真偽値（boolean）はtrueまたはfalseの2つの値のみを持ちます。

<?php
$isLoggedIn = true;
$isAdmin = false;

// 条件によって真偽値を設定
$age = 20;
$isAdult = ($age >= 18);  // true

// null値
$undefinedValue = null;

// var_dump()で型と値を確認
var_dump($isLoggedIn);    // bool(true)
var_dump($isAdmin);       // bool(false)
var_dump($undefinedValue); // NULL

2.7 型の確認と変換

PHPではvar_dump()関数やgettype()関数で変数の型を確認できます。

型の確認

<?php
$text = "Hello";
$number = 42;
$decimal = 3.14;
$flag = true;

echo gettype($text);    // string
echo gettype($number);  // integer
echo gettype($decimal); // double
echo gettype($flag);    // boolean

// より詳細な情報
var_dump($text);        // string(5) "Hello"

型変換（キャスト）

<?php
$stringNumber = "123";
$intNumber = (int)$stringNumber;      // 文字列を整数に変換
$floatNumber = (float)$stringNumber;   // 文字列を浮動小数点に変換

$number = 123;
$stringFromNumber = (string)$number;   // 数値を文字列に変換

// 自動型変換の例
$result = "10" + 5;  // 結果: 15（文字列"10"が数値10に自動変換）
実習 2-3: データ型の確認

様々なデータ型の変数を作成して、その型を確認してみましょう。

手順
  1. 「datatypes.php」ファイルを作成する
  2. 以下のコードを入力する
  3. 実行してデータ型の違いを確認する
実行例
<?php
// 様々なデータ型の変数
$name = "PHP太郎";
$age = 25;
$height = 170.5;
$isStudent = true;
$graduationDate = null;

echo "<h2>変数の型確認</h2>";

// gettype()で型を確認
echo "<p>\$name の型: " . gettype($name) . " 値: $name</p>";
echo "<p>\$age の型: " . gettype($age) . " 値: $age</p>";
echo "<p>\$height の型: " . gettype($height) . " 値: $height</p>";
echo "<p>\$isStudent の型: " . gettype($isStudent) . " 値: " . ($isStudent ? 'true' : 'false') . "</p>";
echo "<p>\$graduationDate の型: " . gettype($graduationDate) . "</p>";

echo "<h3>var_dump()による詳細情報</h3>";
echo "<pre>";
var_dump($name, $age, $height, $isStudent, $graduationDate);
echo "</pre>";
期待される結果

各変数のデータ型が表示され、var_dump()により詳細な型情報（文字列の長さなど）が確認できます。

理解度確認クイズ
  1. PHPで変数を定義するときに必要な記号は何ですか？
  2. 単一引用符（'）と二重引用符（"）の文字列定義の違いを説明してください。
  3. 以下のコードの実行結果を予想してください：echo "5" + 3;
  4. 変数の型を確認するための2つの関数名を答えてください。
  5. 適切な変数名を3つ、不適切な変数名を2つ挙げてください。

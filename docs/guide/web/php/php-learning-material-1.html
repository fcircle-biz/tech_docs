第1章: PHP基礎とセットアップ

PHPとは何か、そして開発環境を構築しよう

この章で学ぶこと
  • PHPというプログラミング言語の特徴と用途
  • PHP開発環境の構築方法
  • PHPの基本的な文法とコードの書き方
  • はじめてのPHPプログラムの実行

1.1 PHPとは

PHP（PHP: Hypertext Preprocessor）は、特にWeb開発に適したサーバーサイドプログラミング言語です。1994年にRasmus Lerdorfによって開発され、現在世界中の多くのWebサイトで使用されています。

PHPの特徴

  • サーバーサイド実行: Webサーバー上で実行され、結果がHTMLとしてブラウザに送信される
  • 動的なWebページ作成: データベースと連携して動的なコンテンツを生成できる
  • 学習コストが低い: シンプルな文法で初心者にも学びやすい
  • 豊富なライブラリ: データベース、画像処理、XML処理など多様な機能を提供
  • オープンソース: 無料で使用でき、活発なコミュニティがサポート

1.2 開発環境のセットアップ

PHP開発を始めるために必要な環境を整えましょう。複数の方法がありますが、初心者に適した方法を紹介します。

方法1: XAMPP/MAMPを使用する（推奨）

XAMPP（Windows/Linux）やMAMP（Mac）は、Apache、MySQL/MariaDB、PHPが一括でインストールできるパッケージです。

注意

本ガイドではPostgreSQLを使用しますが、まずはPHPの基礎を学ぶためにXAMPPの標準構成から始めることをお勧めします。データベース学習時にPostgreSQLを別途インストールします。

方法2: 個別インストール

  • PHP 8.0以上
  • Webサーバー（Apache または Nginx）
  • PostgreSQL 13以上（第8章で使用）

方法3: 開発用内蔵サーバー（学習用）

PHPには開発用の内蔵Webサーバーが含まれており、簡単なテストに使用できます。

1.3 PHPの基本文法

PHPタグ

PHPコードは<?phpタグで開始し、?>タグで終了します。ファイルの最後がPHPコードの場合、終了タグは省略できます（推奨）。

<?php
// これはPHPコードです
echo "Hello, World!";
// ファイル末尾では終了タグを省略

コメント

PHPでは以下の方法でコメントを書くことができます：

<?php
// 単行コメント
# これも単行コメント

/*
  複数行
  コメント
*/
実習 1-1: Hello World プログラム

最初のPHPプログラムを作成して実行してみましょう。

手順
  1. テキストエディタを開く
  2. 以下のコードを入力する
  3. ファイルを「hello.php」として保存する
  4. Webサーバーのドキュメントルート（htdocs等）に配置する
  5. ブラウザで「http://localhost/hello.php」にアクセスする
実行例
<?php
echo "<h1>Hello, PHP World!</h1>";
echo "<p>PHPプログラミングの世界へようこそ！</p>";
echo "<p>現在の日時: " . date('Y-m-d H:i:s') . "</p>";
期待される結果

ブラウザに以下のようなHTMLが表示されます：

  • 「Hello, PHP World!」が大きな見出しで表示
  • 「PHPプログラミングの世界へようこそ！」が段落で表示
  • 現在の日時が表示

1.4 PHPとHTMLの統合

PHPの大きな特徴の一つは、HTMLと組み合わせて使用できることです。

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>PHP + HTML</title>
</head>
<body>
    <h1>今日の情報</h1>
    <p>今日は<?php echo date('Y年m月d日'); ?>です。</p>
    <p>現在時刻：<?php echo date('H:i:s'); ?></p>
</body>
</html>
実習 1-2: 内蔵サーバーでの実行

PHP内蔵サーバーを使って、開発環境を体験してみましょう。

手順
  1. コマンドプロンプト/ターミナルを開く
  2. PHPファイルがあるディレクトリに移動する
  3. php -S localhost:8000 を実行する
  4. ブラウザで「http://localhost:8000/hello.php」にアクセスする
コマンド例
# ディレクトリ移動
cd /path/to/your/php/files

# 内蔵サーバー起動
php -S localhost:8000

# 停止するにはCtrl+Cを押す
期待される結果

コマンドラインに「PHP Development Server started」と表示され、Webサーバーが起動します。

1.5 PHPの実行フロー

PHPがどのように動作するかを理解しましょう。

  1. リクエスト受信: ブラウザがPHPファイルをリクエスト
  2. 解析・実行: WebサーバーがPHPエンジンにファイルを渡し、PHPコードを実行
  3. HTML生成: PHPコードの実行結果としてHTMLが生成される
  4. レスポンス送信: 生成されたHTMLがブラウザに送信される
  5. 表示: ブラウザがHTMLを表示
重要なポイント

PHPコードはサーバー側で実行され、ブラウザにはHTMLのみが送信されます。ブラウザの「ソース表示」ではPHPコードは見えません。

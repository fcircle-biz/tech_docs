第3章

テキストとリンク - 文章を構造化する

見出し、段落、強調、リンクなど、文章を構成する基本的なタグを学びます。Webページ間のリンクや、ページ内リンクの作成方法も習得します。

この章で学ぶこと

  • 見出しタグ（h1〜h6）の使い方と階層構造の理解
  • 段落（p）と改行（br）の使い分け
  • テキストの強調（strong、em）と装飾（b、i）の違い
  • リンク（a）の基本的な使い方と様々な形式
  • ページ内リンクと外部リンクの作成方法

3.1 見出しタグの使い方

見出しタグは、文章の構造を明確にするために使用します。HTMLでは、<h1>から<h6>までの6段階の見出しタグがあり、数字が小さいほど重要度が高くなります。

headings.html 
<!-- 最も重要な見出し -->
<h1>Webページのメインタイトル</h1>

<!-- サブセクションの見出し -->
<h2>大見出し</h2>
<h3>中見出し</h3>
<h4>小見出し</h4>

<!-- さらに細かい分類 -->
<h5>最小見出し</h5>
<h6>最も小さい見出し</h6>

見出しタグの使い方のルール

  • <h1>は1ページに1つだけ使う（ページのメインタイトル）
  • 階層を飛ばさず順番に使う（h2の次にh4を使わない）
  • 見出しの大きさは後からCSSで調整できる

3.2 段落と改行

文章を読みやすく表示するには、段落と改行を適切に使い分ける必要があります。<p>タグは段落を作り、<br>タグは改行を表します。

paragraph.html 
<!-- 段落タグ -->
<p>これは最初の段落です。段落と段落の間には自動的に余白が入ります。</p>
<p>これは2つ目の段落です。</p>

<!-- 改行タグ -->
<p>
    これは1つの段落ですが、<br>
    途中で改行しています。<br>
    段落は変わりません。
</p>

段落と改行の使い分け

  • <p>: 意味のまとまりを持つ文章のブロックに使用
  • <br>: 住所や詩など、段落を変えずに改行したい場合に使用
  • 余白を作るためだけに<br>を連続で使うのは避ける（CSSで調整すべき）

3.3 テキストの強調と装飾

文章の中で重要な部分を強調したり、装飾したりするためのタグがあります。見た目が似ていても、意味が異なるタグがあるので注意が必要です。

emphasis.html 
<!-- 意味的な強調（推奨） -->
<p>この文章には<strong>重要な内容</strong>が含まれています。</p>
<p>ここは<em>強調したい部分</em>です。</p>

<!-- 見た目の装飾（非推奨） -->
<p>これは<b>太字</b>で表示されます。</p>
<p>これは<i>斜体</i>で表示されます。</p>

強調タグと装飾タグの違い

タグ 意味 表示 推奨
<strong> 重要性が高い 太字 ◎
<em> 強調したい 斜体 ◎
<b> 見た目だけ太字 太字 △
<i> 見た目だけ斜体 斜体 △

セマンティックHTML

<strong>や<em>のように、見た目だけでなく「意味」を持つタグを使うことを「セマンティックHTML」と呼びます。検索エンジンや音声読み上げソフトが内容を正しく理解できるため、アクセシビリティの向上につながります。

3.4 リンクの基本

<a>タグ（アンカータグ）は、他のWebページや同じページ内の別の場所にリンクを作成します。href属性でリンク先のURLを指定します。

link.html 
<!-- 外部サイトへのリンク -->
<a href="https://www.google.com">Googleへ移動</a>

<!-- 同じサイト内の別ページへのリンク -->
<a href="about.html">会社概要</a>

<!-- 新しいタブで開く -->
<a href="https://www.google.com" target="_blank">
    Googleを新しいタブで開く
</a>

セキュリティの注意点

target="_blank"を使用する場合は、セキュリティのためにrel="noopener noreferrer"も一緒に指定することが推奨されます。

<a href="..." target="_blank" rel="noopener noreferrer">リンク</a>

3.5 様々なリンクの形式

リンクには、外部サイトへのリンクだけでなく、メールアドレスへのリンク、電話番号へのリンク、ページ内リンクなど、様々な形式があります。

various-links.html 
<!-- メールアドレスへのリンク -->
<a href="mailto:info@example.com">お問い合わせ</a>

<!-- 電話番号へのリンク（スマートフォンで発信） -->
<a href="tel:03-1234-5678">03-1234-5678</a>

<!-- ページ内リンク（id属性を指定した要素へ移動） -->
<a href="#section1">セクション1へ移動</a>

<!-- ジャンプ先の要素 -->
<h2 id="section1">セクション1</h2>

実習: 自己紹介ページを作成してみよう

学んだタグを使って、簡単な自己紹介ページを作成してみましょう。

profile.html 
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>自己紹介</title>
</head>
<body>
    <h1>自己紹介</h1>

    <h2>基本情報</h2>
    <p><strong>名前:</strong> 山田太郎</p>
    <p><strong>趣味:</strong> <em>プログラミング</em>と読書</p>

    <h2>好きなWebサイト</h2>
    <p>
        私がよく見るサイトは
        <a href="https://www.google.com" target="_blank" rel="noopener">Google</a>
        です。
    </p>

    <h2>お問い合わせ</h2>
    <p>
        メール: <a href="mailto:taro@example.com">taro@example.com</a><br>
        電話: <a href="tel:090-1234-5678">090-1234-5678</a>
    </p>

    <p><a href="#top">ページの先頭へ戻る</a></p>
</body>
</html>

理解度確認クイズ

  1. Q1. <h1>タグの使用に関する正しい説明はどれですか？

    回答を表示

    <h1>は1ページに1つだけ使用し、ページのメインタイトルを表します。これはSEOやアクセシビリティの観点からも重要です。

  2. Q2. <strong>と<b>の違いは何ですか？

    回答を表示

    <strong>は「重要性が高い」という意味を持つセマンティックタグです。一方、<b>は単に見た目を太字にするだけで、特別な意味を持ちません。検索エンジンや音声読み上げソフトは<strong>を重要な情報として認識します。

  3. Q3. ページ内リンクを作成する方法を説明してください。

    回答を表示

    ジャンプ先の要素にid属性を設定し、リンク元の<a>タグのhref属性に「#id名」を指定します。例: <a href="#section1">セクション1へ</a>と<h2 id="section1">セクション1</h2>

第3章のまとめ

  • 見出しタグ（h1〜h6）は階層構造を作り、h1は1ページに1つだけ使用する
  • <p>は段落を作り、<br>は改行を表す
  • <strong>と<em>は意味を持つセマンティックタグで、<b>と<i>より推奨される
  • <a>タグでリンクを作成し、href属性でリンク先を指定する
  • 外部リンク、メールリンク、電話リンク、ページ内リンクなど様々な形式がある

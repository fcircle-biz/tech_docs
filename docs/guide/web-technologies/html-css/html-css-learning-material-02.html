第2章

HTMLの基本構造 - はじめてのWebページ作成

HTMLファイルの基本的な構造（、<html>、<head>、<body>）について学び、実際に簡単なWebページを作成します。タグとは何か、要素と属性の違いなど、HTMLの基礎概念を丁寧に説明します。

この章で学ぶこと

  • HTMLファイルの基本構造を理解し、各部分の役割を説明できる
  • タグ、要素、属性の違いを理解し、正しく使い分けられる
  • テキストエディタを使って最初のHTMLファイルを作成できる
  • 作成したHTMLファイルをブラウザで表示して確認できる

2.1 HTMLファイルの基本構造

すべてのHTMLファイルは、決まった基本構造を持っています。この構造は、どんなWebページでも共通で、ブラウザがHTMLを正しく解釈するために必要不可欠です。

HTMLファイルの基本構造は、主に以下の4つの部分から構成されています：

index.html 
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>私の最初のWebページ</title>
</head>
<body>
    <h1>ようこそ！</h1>
    <p>これは私の最初のHTMLページです。</p>
</body>
</html>

1 DOCTYPE宣言

<!DOCTYPE html>は、これがHTML5文書であることをブラウザに伝える宣言です。必ずファイルの先頭に記述します。

2 html要素

<html>は、HTML文書全体を包む「ルート要素」です。すべてのHTML要素は、この中に配置されます。lang="ja"は、このページが日本語で書かれていることを示す属性です。

3 head要素

<head>には、ページのメタ情報（文字コード、タイトル、CSSファイルのリンクなど）を記述します。ここに書かれた内容は、基本的に画面には表示されません。

4 body要素

<body>には、実際に画面に表示される内容（見出し、段落、画像、リンクなど）を記述します。ユーザーが見る部分はすべてここに書きます。

2.2 タグとは何か

タグは、HTMLの基本的な構成要素です。タグは山括弧< >で囲まれた記号で、ブラウザに「これは見出しです」「これは段落です」といった指示を与えます。

ほとんどのタグは、開始タグと終了タグのペアで使用されます：

タグの基本構造 
<!-- 開始タグと終了タグのペア -->
<p>これは段落です。</p>

<!-- 開始タグ: <p> -->
<!-- 内容: これは段落です。 -->
<!-- 終了タグ: </p> -->

終了タグの特徴

終了タグには、タグ名の前にスラッシュ/が付きます。例えば、<p>の終了タグは</p>となります。

空要素（自己完結タグ）

一部のタグは、終了タグが不要な「空要素」として機能します。代表的な空要素には以下があります：

空要素の例 
<!-- 画像タグ（終了タグ不要） -->
<img src="photo.jpg" alt="写真">

<!-- 改行タグ（終了タグ不要） -->
<br>

<!-- 水平線タグ（終了タグ不要） -->
<hr>

<!-- メタ情報タグ（終了タグ不要） -->
<meta charset="UTF-8">

2.3 要素と属性

要素とは

要素は、開始タグ、内容、終了タグを含めた全体のことを指します。HTMLは、さまざまな要素を組み合わせてWebページを構築します。

要素の構成

</p>"] D["p要素全体"] -.-> A D -.-> B D -.-> C style A fill:#dbeafe,stroke:#3b82f6 style B fill:#fef3c7,stroke:#f59e0b style C fill:#dbeafe,stroke:#3b82f6 style D fill:#d1fae5,stroke:#10b981

属性とは

属性は、要素に追加情報を与えるための記述です。属性は開始タグの中に書き、「属性名="値"」の形式で指定します。

属性の例 
<!-- href属性: リンク先URLを指定 -->
<a href="https://example.com">リンクテキスト</a>

<!-- src属性: 画像ファイルのパスを指定 -->
<!-- alt属性: 画像の代替テキストを指定 -->
<img src="logo.png" alt="会社ロゴ">

<!-- class属性: CSSでスタイルを適用するための名前 -->
<div class="container">コンテナ</div>

<!-- id属性: 要素を一意に識別するための名前 -->
<p id="introduction">紹介文</p>

重要ポイント

属性の値は、必ずダブルクォーテーション"..."またはシングルクォーテーション'...'で囲みます。どちらを使っても構いませんが、通常はダブルクォーテーションを使用します。

よく使う属性

属性名 用途 使用例
href リンク先のURL <a href="...">
src 画像やスクリプトのパス <img src="...">
alt 画像の代替テキスト <img alt="...">
class CSSクラス名 <div class="...">
id 一意の識別子 <p id="...">
title ツールチップテキスト <a title="...">

2.4 最初のHTMLファイルを作成しよう

それでは、実際に最初のHTMLファイルを作成してみましょう。この実習では、テキストエディタを使ってHTMLファイルを書き、ブラウザで表示するまでの流れを体験します。

実習 2-1: 最初のHTMLファイルを作成

自己紹介ページを作成して、HTMLの基本構造を理解しましょう。

手順

  1. 1 デスクトップに「my-first-webpage」という名前のフォルダを作成します
  2. 2 テキストエディタ（Visual Studio Codeなど）を開き、新しいファイルを作成します
  3. 3 以下のHTMLコードを入力します（コピー&ペーストでもOK）
index.html 
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>私の自己紹介ページ</title>
</head>
<body>
    <h1>はじめまして！</h1>
    <p>私の名前は山田太郎です。</p>
    <p>趣味はプログラミングと読書です。</p>
    <p>よろしくお願いします！</p>
</body>
</html>
  1. 4 ファイルを「index.html」という名前で「my-first-webpage」フォルダに保存します
  2. 5 保存したファイルをダブルクリックするか、ブラウザにドラッグ&ドロップして開きます

動作確認

ブラウザで正しく表示されているか確認しましょう。

期待される結果

  • ブラウザのタブに「私の自己紹介ページ」と表示される
  • 「はじめまして！」が大きな見出しとして表示される
  • 3つの段落が順番に表示される

ヒント

Visual Studio Codeを使っている場合、「html」と入力して「Tab」キーを押すと、基本的なHTML構造が自動的に挿入されます（Emmet機能）。これを使うと、タイピングの手間が省けます。

注意

ファイル名は必ず「.html」という拡張子を付けて保存してください。拡張子がないと、ブラウザがHTMLファイルとして認識できません。また、ファイル名には日本語や空白文字を使わず、英数字とハイフン「-」のみを使うことを推奨します。

トラブルシューティング

ブラウザで開いても文字化けする

原因: 文字コードの設定が正しくないか、ファイルの保存時に文字コードがUTF-8になっていない可能性があります。

解決方法: head要素内に<meta charset="UTF-8">が記述されているか確認してください。また、エディタの設定でファイルをUTF-8で保存し直してください。

タグがそのまま画面に表示されてしまう

原因: ファイルが.htmlという拡張子で保存されていないか、ブラウザがHTMLファイルとして認識していない可能性があります。

解決方法: ファイル名が「index.html」のように.htmlで終わっているか確認してください。拡張子が表示されていない場合は、Windowsでは「表示」→「ファイル名拡張子」にチェックを入れると確認できます。

ファイルをブラウザで開けない

原因: ファイルが正しい場所に保存されていないか、ブラウザとの関連付けがされていない可能性があります。

解決方法: ファイルを右クリックして「プログラムから開く」→お使いのブラウザを選択してください。または、ブラウザを開いて、ファイルをブラウザのウィンドウにドラッグ&ドロップしてください。

理解度確認クイズ

  1. Q1. HTMLファイルの基本構造で、画面に表示される内容を記述するのはどの要素ですか？

    回答を表示

    body要素です。<body>要素には、実際に画面に表示される内容（見出し、段落、画像、リンクなど）を記述します。一方、<head>要素には、画面に表示されないメタ情報を記述します。

  2. Q2. タグと要素の違いを説明してください。

    回答を表示

    タグは、<p>や</p>のような山括弧で囲まれた記号のことです。要素は、開始タグ、内容、終了タグを含めた全体を指します。例えば、<p>これは段落です。</p>全体がp要素です。

  3. Q3. 属性の役割は何ですか？また、属性はどこに記述しますか？

    回答を表示

    属性は、要素に追加情報を与えるための記述です。例えば、リンク先のURLや画像のパス、要素の識別名などを指定します。属性は開始タグの中に記述し、「属性名="値"」の形式で指定します。例：<a href="https://example.com">

  4. Q4. 終了タグが不要な要素（空要素）の例を3つ挙げてください。

    回答を表示

    代表的な空要素は以下の通りです：
    1. <img> - 画像を表示する
    2. <br> - 改行する
    3. <hr> - 水平線を引く
    4. <meta> - メタ情報を定義する

第2章のまとめ

  • HTMLファイルは、DOCTYPE宣言、html要素、head要素、body要素の4つの基本構造で構成される
  • タグは山括弧< >で囲まれた記号で、開始タグと終了タグのペアで使用される
  • 要素は、開始タグ、内容、終了タグを含めた全体のことを指す
  • 属性は、要素に追加情報を与えるために開始タグの中に記述する
  • img、br、hr、metaなどは終了タグが不要な空要素である
  • テキストエディタでHTMLファイルを作成し、.html拡張子で保存することでブラウザで表示できる

第2章: HTMLの基本タグを学ぼう

HTMLタグでWebページを構造化する

この章で学ぶこと

  • 見出しタグ（h1〜h6）の使い方とその重要性

2.1 HTML文書の基本構造

HTML文書の骨格

すべてのHTML文書には決まった構造があります。これは建物の設計図と同じで、 正しい構造に従うことで、ブラウザがページを正しく表示できます。 

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ページタイトル</title>
</head>
<body>
    <!-- ページの内容をここに書く -->
</body>
</html>
各部分の詳細説明
タグ 説明
<!DOCTYPE html> HTML5文書であることを宣言。必ず最初に書く
<html lang="ja"> HTML文書全体を囲む。lang="ja"で日本語ページであることを示す
<head> ページの情報（メタ情報）を記述する領域
<meta charset="UTF-8"> 文字コードをUTF-8に設定（日本語を表示するために必須）
<meta name="viewport"...> スマホやタブレットでの表示を最適化
<title> ブラウザのタブに表示されるページタイトル
<body> ブラウザに実際に表示される内容を記述する領域

2.2 見出しタグ（h1〜h6）

見出しとは？

見出しはページの構造を示す重要な要素です。本や新聞の見出しと同じように、 HTMLでも見出しを使ってコンテンツを整理します。

見出しタグは <h1> から <h6> まで6種類あり、 数字が小さいほど重要度が高くなります。 

<h1>最も重要な見出し（ページタイトル）</h1>
<h2>大見出し（セクションタイトル）</h2>
<h3>中見出し（サブセクション）</h3>
<h4>小見出し</h4>
<h5>より小さい見出し</h5>
<h6>最も小さい見出し</h6>
見出しの使い方のルール
  • h1は1ページに1つ: ページの最も重要なタイトルに使用
  • 階層を飛ばさない: h1の次にh3を使うのではなく、h2→h3と順番に使う
  • 見た目ではなく意味で選ぶ: 大きく表示したいからh1を使う、ではなく重要度で選ぶ
flowchart TD A[h1: ページタイトル] --> B[h2: セクション1] A --> C[h2: セクション2] B --> D[h3: サブセクション1-1] B --> E[h3: サブセクション1-2] C --> F[h3: サブセクション2-1]
実習 2-1: 見出しを使った構造化されたページ

見出しを使って、自己紹介ページを構造化してみましょう。

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>私の自己紹介</title>
</head>
<body>
    <h1>山田太郎の自己紹介</h1>

    <h2>基本情報</h2>
    <h3>名前と年齢</h3>
    <h3>出身地</h3>

    <h2>趣味</h2>
    <h3>読書</h3>
    <h3>音楽鑑賞</h3>

    <h2>学習目標</h2>
</body>
</html>

2.3 段落と改行（p、br）

段落タグ（p）

<p>タグは段落（paragraph）を表します。 通常の文章を書くときに使用します。 

<p>これは最初の段落です。</p>
<p>これは2番目の段落です。段落と段落の間には自動的に余白が入ります。</p>

改行タグ（br）

<br>タグは改行を表します。段落内で改行したいときに使用します。 

<p>
    東京都渋谷区〇〇1-2-3<br>
    〒150-0000<br>
    電話: 03-1234-5678
</p>
注意点

<br>を連続して使って余白を作るのは避けましょう。 余白の調整はCSSで行うのが正しい方法です。

2.4 テキストの強調（strong、em）

強い強調（strong）

<strong>タグは、特に重要なテキストを強調します。 ブラウザではデフォルトで太字で表示されます。 

<p>この商品は<strong>本日限り</strong>の特別価格です。</p>

通常の強調（em）

<em>タグは、テキストを強調します。 ブラウザではデフォルトで斜体（イタリック）で表示されます。 

<p>このレストランの料理は<em>本当に</em>美味しいです。</p>

2.5 リンクの作成（a）

リンクとは？

リンク（aタグ）は、他のページや別の場所へ移動するための仕組みです。 Webの「ハイパーテキスト」の核となる機能です。 

<a href="https://www.google.com">Googleへ移動</a>
リンクの属性
  • href: リンク先のURL（必須）
  • target="_blank": 新しいタブで開く
  • title: リンクの説明（マウスホバー時に表示）
<!-- 同じタブで開く -->
<a href="page2.html">次のページ</a>

<!-- 新しいタブで開く -->
<a href="https://www.example.com" target="_blank">外部サイト</a>

<!-- タイトル付きリンク -->
<a href="contact.html" title="お問い合わせページへ">お問い合わせ</a>

2.6 画像の挿入（img）

画像タグの基本

<img>タグは、ページに画像を表示します。 

<img src="photo.jpg" alt="風景写真">
画像タグの属性
  • src: 画像ファイルのパス（必須）
  • alt: 画像の説明文（必須）。画像が表示されない場合や、視覚障害者向けの読み上げに使用
  • width: 画像の幅
  • height: 画像の高さ
<!-- サイズ指定あり -->
<img src="logo.png" alt="会社ロゴ" width="200" height="100">

<!-- 画像にリンクを設定 -->
<a href="home.html">
    <img src="logo.png" alt="ホームに戻る">
</a>

2.7 リストの作成（ul、ol、li）

順序なしリスト（ul）

<ul>（Unordered List）は、順序のない項目リストを作成します。 各項目には箇条書きのマーカーが表示されます。 

<h3>好きな食べ物</h3>
<ul>
    <li>ラーメン</li>
    <li>寿司</li>
    <li>カレー</li>
</ul>

順序ありリスト（ol）

<ol>（Ordered List）は、順序のある項目リストを作成します。 各項目には番号が自動的に付きます。 

<h3>学習の手順</h3>
<ol>
    <li>HTMLの基礎を学ぶ</li>
    <li>CSSでデザインを学ぶ</li>
    <li>JavaScriptで動きを追加</li>
</ol>

入れ子のリスト

リストの中にリストを入れることもできます。

<ul>
    <li>フロントエンド
        <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
        </ul>
    </li>
    <li>バックエンド
        <ul>
            <li>Python</li>
            <li>Java</li>
        </ul>
    </li>
</ul>

2.8 テーブルの作成（table、tr、td、th）

テーブルとは？

テーブルは、データを行と列で整理して表示する表です。

テーブルの構成要素
  • <table>: テーブル全体を囲む
  • <tr>: 行（Table Row）
  • <th>: 見出しセル（Table Header）
  • <td>: データセル（Table Data）
<table>
    <tr>
        <th>名前</th>
        <th>年齢</th>
        <th>職業</th>
    </tr>
    <tr>
        <td>山田太郎</td>
        <td>25歳</td>
        <td>エンジニア</td>
    </tr>
    <tr>
        <td>佐藤花子</td>
        <td>28歳</td>
        <td>デザイナー</td>
    </tr>
</table>

2.9 フォームの基礎

フォームとは？

フォームは、ユーザーからの入力を受け取るための仕組みです。 お問い合わせフォーム、検索ボックス、ログイン画面などで使用します。

基本的なフォーム

<form>
    <label>名前：</label>
    <input type="text" name="username">

    <label>メールアドレス：</label>
    <input type="email" name="email">

    <button type="submit">送信</button>
</form>
主なinputタイプ
  • text: テキスト入力
  • email: メールアドレス入力
  • password: パスワード入力（文字が隠れる）
  • number: 数値入力
  • date: 日付入力
  • checkbox: チェックボックス
  • radio: ラジオボタン
<form>
    <h3>お問い合わせフォーム</h3>

    <label>お名前：</label>
    <input type="text" name="name" required>
    <br>

    <label>メールアドレス：</label>
    <input type="email" name="email" required>
    <br>

    <label>性別：</label>
    <input type="radio" name="gender" value="male"> 男性
    <input type="radio" name="gender" value="female"> 女性
    <br>

    <label>お問い合わせ内容：</label>
    <textarea name="message" rows="5" cols="30"></textarea>
    <br>

    <button type="submit">送信</button>
</form>

2.10 まとめ

この章では、HTMLの主要なタグを学習しました：

学習した主なタグ
カテゴリ タグ 用途
見出し <h1>〜<h6> ページの構造を示す見出し
テキスト <p>, <br>, <strong>, <em> 段落、改行、強調
リンク・画像 <a>, <img> リンク作成、画像挿入
リスト <ul>, <ol>, <li> 順序なし/順序ありリスト

理解度確認クイズ
  1. 問題1: 見出しタグh1とh2の違いは何ですか？使い分けの基準を説明してください。
    解答例

    h1はページで最も重要な見出し（通常は1ページに1つ）で、h2はその下のセクションタイトルに使います。 h1はページ全体のタイトル、h2は各セクションのタイトルという階層構造で使い分けます。

  2. 問題2: <strong>と<em>の違いは何ですか？
    解答例

    <strong>は特に重要な強調（太字表示）、<em>は通常の強調（斜体表示）です。 意味的には、strongの方がより強い重要性を示します。

  3. 問題3: 画像タグのalt属性はなぜ必要ですか？
    解答例

    alt属性は、画像が表示できない場合の代替テキストや、視覚障害者向けの読み上げ機能で使用されます。 アクセシビリティの観点から非常に重要です。

  4. 問題4: <ul>と<ol>の使い分けを説明してください。
    解答例

    <ul>は順序が重要でないリスト（買い物リスト、趣味リストなど）、 <ol>は順序が重要なリスト（手順、ランキングなど）に使用します。

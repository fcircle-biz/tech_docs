ボックスモデル - レイアウトの基礎

CSSボックスモデル（margin、border、padding、content）の概念を理解し、要素の配置や間隔の調整方法を学びます。displayプロパティによる要素の表示形式の変更も習得します。

この章で学ぶこと

  • CSSボックスモデルの4つの領域（content、padding、border、margin）を理解する
  • paddingとmarginの違いを説明できるようになる
  • box-sizingプロパティの役割を理解する
  • displayプロパティ（block、inline、inline-block）の違いを理解する

9.1 ボックスモデルとは

CSSでは、すべてのHTML要素は「ボックス」として扱われます。このボックスは、中心から外側に向かって、content（内容）、padding（内側の余白）、border（枠線）、margin（外側の余白）の4つの領域で構成されています。

この構造を「CSSボックスモデル」と呼びます。ボックスモデルを理解することは、Webページのレイアウトを正確にコントロールするための基礎となります。

ボックスモデルの視覚化

ブラウザの開発者ツール（F12キー）を開くと、選択した要素のボックスモデルが色分けされて表示されます。これを見ながら、margin、border、padding、contentの各領域を確認することができます。

9.2 content（コンテンツ領域）

content（コンテンツ）は、ボックスの中心部分で、実際のテキストや画像などの内容が表示される領域です。widthとheightプロパティで、この領域のサイズを指定します。

content-example.css 
/* コンテンツ領域のサイズ指定 */
.box {
    width: 300px;   /* 幅300px */
    height: 200px;  /* 高さ200px */
    background-color: #dbeafe;
}

この例では、要素のコンテンツ領域が幅300px、高さ200pxのボックスとして表示されます。

9.3 padding（パディング/内側の余白）

padding（パディング）は、コンテンツとボーダーの間の「内側の余白」です。paddingを指定すると、コンテンツの周りに余白が追加され、ボックスが大きくなります。

padding-example.css 
/* パディングの指定 */
.box {
    padding: 20px;  /* 上下左右すべて20px */
}

/* 方向別に指定 */
.box2 {
    padding-top: 10px;     /* 上 */
    padding-right: 20px;   /* 右 */
    padding-bottom: 10px;  /* 下 */
    padding-left: 20px;    /* 左 */
}

/* 短縮記法 */
.box3 {
    padding: 10px 20px;  /* 上下10px、左右20px */
}

paddingの短縮記法

paddingプロパティは、値の個数によって意味が変わります：
• 1つ：すべての方向
• 2つ：上下、左右
• 3つ：上、左右、下
• 4つ：上、右、下、左（時計回り）

9.4 border（ボーダー/枠線）

border（ボーダー）は、paddingとmarginの間にある「枠線」です。線の太さ、スタイル、色を指定できます。

border-example.css 
/* ボーダーの基本指定 */
.box {
    border: 2px solid #3b82f6;
    /* 太さ スタイル 色 */
}

/* 個別に指定 */
.box2 {
    border-width: 2px;     /* 太さ */
    border-style: solid;   /* スタイル */
    border-color: #3b82f6; /* 色 */
}

/* 方向別に指定 */
.box3 {
    border-top: 3px dotted #f59e0b;
    border-bottom: 1px dashed #10b981;
}

主なborder-styleの種類

スタイル 説明
solid 実線
dashed 破線
dotted 点線
double 二重線
none 枠線なし

9.5 margin（マージン/外側の余白）

margin（マージン）は、ボーダーの外側にある「外側の余白」です。他の要素との間隔を調整するために使います。

margin-example.css 
/* マージンの指定 */
.box {
    margin: 20px;  /* 上下左右すべて20px */
}

/* 中央揃え */
.center-box {
    width: 500px;
    margin: 0 auto;  /* 左右auto で中央揃え */
}

/* 個別指定 */
.box2 {
    margin-top: 10px;
    margin-bottom: 20px;
}

paddingとmarginの違い

paddingは「要素の内側」の余白で、背景色が適用されます。marginは「要素の外側」の余白で、背景色は適用されません。要素間の間隔を開けたいときはmargin、要素内のコンテンツに余裕を持たせたいときはpaddingを使います。

マージンの相殺（margin collapse）

上下に隣接する要素のmarginは、大きい方のmarginに「相殺」されます。例えば、上の要素のmargin-bottom: 20pxと、下の要素のmargin-top: 30pxが隣接すると、間隔は50pxではなく30pxになります。

9.6 box-sizing プロパティ

デフォルトでは、widthとheightはコンテンツ領域のサイズのみを指定します。つまり、paddingとborderを追加すると、要素全体のサイズが大きくなってしまいます。

box-sizing: border-boxを指定すると、widthとheightにpaddingとborderを含めることができ、サイズ計算が簡単になります。

box-sizing-example.css 
/* デフォルト（content-box） */
.box1 {
    width: 300px;
    padding: 20px;
    border: 2px solid #000;
    /* 実際の幅 = 300 + 20×2 + 2×2 = 344px */
}

/* border-boxを使用 */
.box2 {
    box-sizing: border-box;
    width: 300px;
    padding: 20px;
    border: 2px solid #000;
    /* 実際の幅 = 300px（paddingとborder含む） */
}

全要素にborder-boxを適用

多くの開発者は、すべての要素にbox-sizing: border-boxを適用することを推奨しています。CSSファイルの最初に以下を記述すると便利です： 

* {
    box-sizing: border-box;
}

9.7 displayプロパティ

displayプロパティは、要素がどのように表示されるかを制御します。主な値にはblock、inline、inline-blockがあります。

display-example.css 
/* ブロックレベル要素 */
.block-element {
    display: block;
    /* 幅いっぱいに広がる、前後で改行される */
}

/* インライン要素 */
.inline-element {
    display: inline;
    /* 横並び、width/heightは無効 */
}

/* インラインブロック要素 */
.inline-block-element {
    display: inline-block;
    /* 横並びだが、width/heightが有効 */
}

displayプロパティの比較

値 特徴 width/height 改行
block 幅いっぱいに広がる 有効 前後で改行
inline コンテンツに合わせる 無効 改行なし
inline-block 横並びだがサイズ指定可 有効 改行なし

デフォルトのdisplay値

<div>、<p>、<h1>などはblock、<span>、<a>などはinlineがデフォルトです。

実習 9-1: ボックスモデルを理解する

ボックスモデルの各領域を視覚的に確認し、理解を深めましょう。

手順

  1. 1 新しいHTMLファイルを作成し、divタグで3つのボックスを作成します
  2. 2 CSSで各ボックスに異なるpadding、border、marginを指定します
  3. 3 ブラウザの開発者ツールで、各ボックスのボックスモデルを確認します
  4. 4 box-sizing: border-box の有無で、要素のサイズがどう変わるか比較します

理解度確認クイズ

  1. Q1. CSSボックスモデルの4つの領域を、内側から順に答えてください。

    回答を表示

    内側から順に、content（コンテンツ）、padding（内側の余白）、border（枠線）、margin（外側の余白）です。

  2. Q2. paddingとmarginの違いを説明してください。

    回答を表示

    paddingは要素の内側の余白で、背景色が適用されます。marginは要素の外側の余白で、背景色は適用されません。paddingはコンテンツとボーダーの間、marginはボーダーと他の要素の間に配置されます。

  3. Q3. box-sizing: border-box を使用する利点は何ですか？

    回答を表示

    widthとheightの値に、paddingとborderを含めることができます。これにより、要素の実際のサイズが計算しやすくなり、レイアウトの管理が簡単になります。デフォルトのcontent-boxでは、widthにpaddingとborderが加算されるため、意図しないサイズになることがあります。

  4. Q4. display: inline と display: inline-block の違いは何ですか？

    回答を表示

    display: inlineは横並びになりますが、widthとheightが無効です。display: inline-blockも横並びになりますが、widthとheightを指定できます。inline-blockは、横並びレイアウトでサイズを制御したいときに便利です。

第9章のまとめ

  • CSSボックスモデルは、content、padding、border、marginの4層構造
  • paddingは内側の余白、marginは外側の余白を指定する
  • borderで枠線の太さ、スタイル、色を設定できる
  • box-sizing: border-box でサイズ計算を簡単にできる
  • displayプロパティで要素の表示形式（block、inline、inline-block）を制御できる

フォーム作成 - ユーザーからの入力を受け取る

入力フォーム（<form>）、テキスト入力（<input>）、選択ボックス（<select>）、ボタンなど、ユーザーとやり取りするための要素を学びます。お問い合わせフォームや登録フォームの基礎を作成します。

この章で学ぶこと

  • HTMLフォームの基本構造と役割を理解する
  • 様々な入力タイプ（text、email、password、numberなど）を使い分けられるようになる
  • ラジオボタン、チェックボックス、プルダウンメニューを作成できる
  • 実際にお問い合わせフォームを作成できるようになる

6.1 フォームとは何か

フォームとは、Webページ上でユーザーから情報を入力してもらうための仕組みです。お問い合わせ、会員登録、ログイン、アンケートなど、ユーザーとWebサイトがやり取りする場面で使われます。

フォームは<form>タグで作成し、その中にテキストボックス、ボタン、チェックボックスなどの部品を配置します。

フォームでできること

お問い合わせの送信、ログイン認証、検索機能、アンケート回答、商品注文、ファイルアップロードなど、Webサイトのほとんどのインタラクティブな機能にフォームが使われています。

6.2 基本的なフォームの構造

フォームは<form>タグで囲み、その中に入力部品を配置します。基本的な構造を見てみましょう。

basic-form.html 
<form action="/submit" method="POST">
    <!-- 名前入力欄 -->
    <label for="name">お名前:</label>
    <input type="text" id="name" name="name">

    <!-- 送信ボタン -->
    <button type="submit">送信</button>
</form>

<form>タグには、重要な属性が2つあります：

formタグの主な属性

属性 説明 例
action フォームデータの送信先URL action="/contact"
method 送信方法（GETまたはPOST） method="POST"

覚えておきたいポイント

<label>タグは入力欄のラベル（説明）を表示するために使います。for属性でinputのid属性と紐付けると、ラベルをクリックしても入力欄にフォーカスが当たるようになり、使いやすくなります。

6.3 テキスト入力フィールド

<input>タグは、様々な種類の入力欄を作成できます。type属性を変えることで、異なる入力形式を指定できます。

input-types.html 
<!-- テキスト入力 -->
<input type="text" name="username" placeholder="ユーザー名">

<!-- メールアドレス入力 -->
<input type="email" name="email" placeholder="example@example.com">

<!-- パスワード入力（入力内容が隠される） -->
<input type="password" name="password" placeholder="パスワード">

<!-- 数値入力 -->
<input type="number" name="age" min="0" max="120">

<!-- 電話番号入力 -->
<input type="tel" name="phone" placeholder="090-1234-5678">

主なinputタイプ

type属性 説明 用途
text 1行テキスト 名前、住所など
email メールアドレス メールアドレス入力（自動検証）
password パスワード（隠し文字） パスワード入力
number 数値 年齢、数量など
tel 電話番号 電話番号入力
date 日付 日付選択（カレンダー表示）

placeholder属性

placeholder属性を使うと、入力欄に薄く表示される例示文（ヒント）を設定できます。ユーザーが入力を開始すると自動的に消えます。

6.4 ラジオボタンとチェックボックス

ラジオボタンは複数の選択肢から1つだけを選ぶ場合、チェックボックスは複数の選択肢から複数を選べる場合に使います。

radio-checkbox.html 
<!-- ラジオボタン（1つだけ選択） -->
<p>性別を選択してください:</p>
<input type="radio" id="male" name="gender" value="male">
<label for="male">男性</label>

<input type="radio" id="female" name="gender" value="female">
<label for="female">女性</label>

<input type="radio" id="other" name="gender" value="other">
<label for="other">その他</label>

<!-- チェックボックス（複数選択可） -->
<p>興味のある分野を選択してください:</p>
<input type="checkbox" id="web" name="interest" value="web">
<label for="web">Web開発</label>

<input type="checkbox" id="design" name="interest" value="design">
<label for="design">デザイン</label>

<input type="checkbox" id="marketing" name="interest" value="marketing">
<label for="marketing">マーケティング</label>

覚えておきたいポイント

ラジオボタンでは、同じグループの選択肢は同じname属性を持つ必要があります。こうすることで、1つだけ選択できるようになります。一方、チェックボックスは複数選択可能なので、同じname属性でも複数チェックできます。

6.5 プルダウンメニュー（セレクトボックス）

<select>タグを使うと、ドロップダウン形式の選択メニューを作成できます。選択肢が多い場合に便利です。

select-box.html 
<label for="prefecture">都道府県を選択:</label>
<select id="prefecture" name="prefecture">
    <option value="">選択してください</option>
    <option value="tokyo">東京都</option>
    <option value="osaka">大阪府</option>
    <option value="kyoto">京都府</option>
    <option value="hokkaido">北海道</option>
</select>

<option>タグで各選択肢を定義します。value属性にサーバーに送信される値を、タグの中身にユーザーに表示される文字列を書きます。

optgroupでグループ化

<optgroup>タグを使うと、選択肢をカテゴリーごとにグループ化できます。 

<select name="country">
    <optgroup label="アジア">
        <option value="jp">日本</option>
        <option value="cn">中国</option>
    </optgroup>
    <optgroup label="ヨーロッパ">
        <option value="uk">イギリス</option>
        <option value="fr">フランス</option>
    </optgroup>
</select>

6.6 テキストエリア（複数行入力）

<textarea>タグを使うと、複数行のテキストを入力できる欄を作成できます。お問い合わせ内容やコメントなど、長い文章を入力する場合に使います。

textarea.html 
<label for="message">お問い合わせ内容:</label>
<textarea id="message" name="message" rows="5" cols="40"></textarea>

rows属性で行数、cols属性で列数（幅）を指定できます。ただし、実際のレイアウトではCSSで調整することが一般的です。

6.7 ボタンの種類

フォームには様々な種類のボタンがあります。<button>タグまたは<input>タグでボタンを作成できます。

buttons.html 
<!-- 送信ボタン -->
<button type="submit">送信</button>

<!-- リセットボタン -->
<button type="reset">リセット</button>

<!-- 汎用ボタン -->
<button type="button">キャンセル</button>

<!-- inputタグでも作成可能 -->
<input type="submit" value="送信">
<input type="reset" value="クリア">

ボタンタイプの違い

type属性 動作 用途
submit フォームを送信する 送信、登録、ログイン
reset 入力内容をクリア 入力のやり直し
button 何もしない（JavaScriptで制御） カスタム動作、キャンセル

6.8 実習：お問い合わせフォームの作成

実習 6-1: お問い合わせフォームの作成

これまで学んだフォーム要素を組み合わせて、実際のお問い合わせフォームを作成してみましょう。

手順

  1. 1 contact.htmlというファイルを作成します
  2. 2 以下のHTMLコードを記述してください
contact.html 
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>お問い合わせフォーム</title>
</head>
<body>
    <h1>お問い合わせフォーム</h1>
    <form action="/submit" method="POST">
        <!-- 名前 -->
        <label for="name">お名前*</label>
        <input type="text" id="name" name="name" required>

        <!-- メールアドレス -->
        <label for="email">メールアドレス*</label>
        <input type="email" id="email" name="email" required>

        <!-- お問い合わせ種別 -->
        <label for="category">お問い合わせ種別*</label>
        <select id="category" name="category" required>
            <option value="">選択してください</option>
            <option value="product">製品について</option>
            <option value="support">サポート</option>
            <option value="other">その他</option>
        </select>

        <!-- お問い合わせ内容 -->
        <label for="message">お問い合わせ内容*</label>
        <textarea id="message" name="message" rows="5" required></textarea>

        <!-- ボタン -->
        <button type="submit">送信</button>
        <button type="reset">クリア</button>
    </form>
</body>
</html>

動作確認

ブラウザでcontact.htmlを開いて、以下を確認してみましょう：

  • 各入力欄が正しく表示されているか
  • 必須項目を空欄にして送信ボタンを押すとエラーメッセージが表示されるか
  • リセットボタンを押すと入力内容がクリアされるか

required属性

required属性を付けると、その項目が必須入力になります。空欄のまま送信しようとすると、ブラウザが自動的にエラーメッセージを表示してくれます。

理解度確認クイズ

  1. Q1. ラジオボタンとチェックボックスの違いを説明してください。

    回答を表示

    ラジオボタンは複数の選択肢から1つだけを選ぶ場合に使います（例：性別選択）。一方、チェックボックスは複数の選択肢から複数を選べる場合に使います（例：興味のある分野を複数選択）。

  2. Q2. required属性は何のために使いますか？

    回答を表示

    required属性は、その入力項目を必須にするために使います。この属性が付いている項目を空欄のまま送信しようとすると、ブラウザが自動的にエラーメッセージを表示し、送信をブロックします。

  3. Q3. <input type="email">と<input type="text">の違いは何ですか？

    回答を表示

    type="email"は、メールアドレス専用の入力欄で、ブラウザが自動的にメールアドレスの形式（@が含まれているかなど）を検証してくれます。また、スマートフォンでは@記号が入力しやすいキーボードが表示されます。type="text"は通常のテキスト入力で、特別な検証はありません。

第6章のまとめ

  • フォームは<form>タグで作成し、ユーザーから情報を入力してもらうための仕組み
  • <input>タグのtype属性を変えることで、様々な種類の入力欄を作成できる
  • ラジオボタンは1つだけ選択、チェックボックスは複数選択が可能
  • <select>タグでプルダウンメニュー、<textarea>で複数行テキスト入力が可能
  • required属性で必須項目を設定でき、ブラウザが自動検証してくれる
  • これらの要素を組み合わせて、お問い合わせフォームや登録フォームが作成できる

第7章

CSSの基礎 - Webページにスタイルを追加

CSSとは何か、CSSの記述方法（インライン、内部、外部スタイルシート）、セレクタの基本（要素、クラス、ID）について学びます。HTMLにスタイルを適用する仕組みを理解します。

この章で学ぶこと

  • CSSの役割と、HTMLとの関係を理解する
  • CSSの3つの記述方法（インライン、内部、外部）を使い分けられるようになる
  • 基本的なCSSセレクタ（要素、クラス、ID）を理解し、適切に使用できるようになる
  • セレクタの優先順位（詳細度）の基本を理解する

7.1 CSSとは何か

前章までに、HTMLを使ってWebページの構造を作る方法を学びました。しかし、HTMLだけで作ったページは、装飾のないシンプルな見た目になります。CSS（Cascading Style Sheets）は、HTMLで作った構造に「見た目」を追加するための言語です。

CSSを使うことで、文字の色やサイズ、背景色、余白、配置など、さまざまなデザイン要素を指定できます。HTMLが「家の骨組み」なら、CSSは「壁紙、ペンキ、インテリア」に例えられます。

HTMLとCSSの役割分担

構造・内容] --> C[Webブラウザ] B[CSS
デザイン・見た目] --> C C --> D[美しいWebページ] style A fill:#dbeafe,stroke:#3b82f6 style B fill:#fef3c7,stroke:#f59e0b style C fill:#d1fae5,stroke:#10b981 style D fill:#e0e7ff,stroke:#6366f1

CSSの名前の由来

「Cascading」は「滝が流れるように」という意味で、複数のスタイル指定がある場合に、優先順位に従って適用されていく様子を表しています。「Style Sheets」は「スタイルの設定をまとめた書類」という意味です。

7.2 CSSの3つの記述方法

CSSには、HTMLに適用する方法が3つあります。それぞれ特徴があり、状況に応じて使い分けることが大切です。

7.2.1 インラインスタイル

HTML要素のstyle属性に直接CSSを書く方法です。その要素だけに適用されます。

inline-style.html 
<!-- インラインスタイルの例 -->
<p style="color: blue; font-size: 18px;">
    この段落は青色で、文字サイズは18pxです。
</p>

インラインスタイルの注意点

インラインスタイルは手軽ですが、HTMLとCSSが混在して管理が難しくなります。また、同じスタイルを複数の要素に適用したい場合に、すべての要素にコピーする必要があり非効率です。基本的には使用を避け、特別な理由がある場合のみ使用しましょう。

7.2.2 内部スタイルシート

HTMLファイルの<head>セクション内に<style>タグを使ってCSSを書く方法です。そのHTMLファイル全体に適用されます。

internal-style.html 
<!DOCTYPE html>
<html lang="ja">
<head>
    <title>内部スタイルシートの例</title>
    <style>
        /* すべてのh1要素に適用 */
        h1 {
            color: #2563eb;
            text-align: center;
        }
        /* すべてのp要素に適用 */
        p {
            font-size: 16px;
            line-height: 1.6;
        }
    </style>
</head>
<body>
    <h1>見出し</h1>
    <p>段落のテキスト</p>
</body>
</html>

内部スタイルシートの用途

1ページだけの簡単なWebサイトや、練習・テスト用のページを作る場合に便利です。ただし、複数のHTMLファイルで同じスタイルを使いたい場合は、次に紹介する外部スタイルシートを使う方が効率的です。

7.2.3 外部スタイルシート（推奨）

CSSを別ファイル（.css）に保存し、HTMLから読み込む方法です。最も推奨される方法で、プロの現場でも標準的に使われています。

style.css 
/* CSSファイルの内容 */
h1 {
    color: #2563eb;
    text-align: center;
}

p {
    font-size: 16px;
    line-height: 1.6;
}
index.html 
<!DOCTYPE html>
<html lang="ja">
<head>
    <title>外部スタイルシートの例</title>
    <!-- 外部CSSファイルを読み込む -->
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>見出し</h1>
    <p>段落のテキスト</p>
</body>
</html>

外部スタイルシートの利点

  • 複数のHTMLファイルで同じCSSを共有できる
  • HTMLとCSSが分離され、管理しやすい
  • デザイン変更時にCSSファイルだけ修正すれば、全ページに反映される
  • ブラウザにキャッシュされるため、ページ読み込みが速くなる

7.3 CSSの基本構文

CSSは「セレクタ」と「宣言ブロック」で構成されます。この基本構文を理解することが、CSS習得の第一歩です。

CSSの基本構文

セレクタ {
  プロパティ: 値;
  プロパティ: 値;
}
  • セレクタ: どのHTML要素にスタイルを適用するかを指定
  • プロパティ: 何を変更するか（例: color、font-size）
  • 値: どのように変更するか（例: red、16px）
基本構文の例 
/* h1要素の文字色を青にする */
h1 {
    color: blue;
}

/* p要素に複数のプロパティを指定 */
p {
    color: #333333;      /* 文字色 */
    font-size: 16px;     /* 文字サイズ */
    line-height: 1.6;    /* 行間 */
    margin-bottom: 20px; /* 下余白 */
}

CSSのコメント

CSSでは/* */でコメントを書きます。コメントはブラウザに無視され、コードの説明を書くために使います。HTMLのコメント（<!-- -->）とは記法が異なるので注意しましょう。

7.4 基本的なセレクタ

CSSセレクタは、「どのHTML要素にスタイルを適用するか」を指定するための記法です。基本的なセレクタには、要素セレクタ、クラスセレクタ、IDセレクタの3種類があります。

7.4.1 要素セレクタ（タグセレクタ）

HTML要素のタグ名を使ってスタイルを適用します。該当するタグすべてに適用されます。

要素セレクタの例 
/* すべてのh1要素に適用 */
h1 {
    color: #1e40af;
    font-size: 32px;
}

/* すべてのp要素に適用 */
p {
    color: #475569;
    line-height: 1.8;
}

7.4.2 クラスセレクタ

HTML要素にclass属性を付けて、その名前で指定します。クラス名の前にピリオド（.）を付けます。同じクラス名は複数の要素に使えます。

HTML（クラスを使用） 
<p class="highlight">この段落は強調されます。</p>
<p>この段落は通常のスタイルです。</p>
<p class="highlight">この段落も強調されます。</p>
CSS（クラスセレクタ） 
/* highlightクラスを持つ要素に適用 */
.highlight {
    background-color: #fef3c7;
    padding: 10px;
    border-left: 4px solid #f59e0b;
}

クラスの使い方のコツ

クラス名は、見た目ではなく「役割」を表す名前にすると良いでしょう。例えば、red-text（赤い文字）ではなく、error（エラー）やwarning（警告）のように、その要素の意味を表す名前をつけると、後で色を変更したくなった時に困りません。

7.4.3 IDセレクタ

HTML要素にid属性を付けて、その名前で指定します。IDの前にハッシュ（#）を付けます。同じIDは1つのページ内で1つの要素にのみ使えます。

HTML（IDを使用） 
<header id="main-header">
    <h1>私のWebサイト</h1>
</header>
CSS（IDセレクタ） 
/* main-headerというIDを持つ要素に適用 */
#main-header {
    background-color: #1e40af;
    color: white;
    padding: 20px;
    text-align: center;
}

セレクタの比較

セレクタ 記号 例 使用頻度
要素セレクタ なし h1 { ... } 高い
クラスセレクタ . .highlight { ... } 非常に高い
IDセレクタ # #main-header { ... } 中程度

7.5 セレクタの優先順位（詳細度）

同じHTML要素に対して複数のCSSルールが適用される場合、どのスタイルが優先されるかは「詳細度（Specificity）」によって決まります。

セレクタの優先順位（高い順）

flowchart TD A["インラインスタイル
(style属性)"] --> B["IDセレクタ
(#id)"] B --> C["クラスセレクタ
(.class)"] C --> D["要素セレクタ
(h1, p)"] style A fill:#fef3c7,stroke:#f59e0b style B fill:#fce7f3,stroke:#ec4899 style C fill:#dbeafe,stroke:#3b82f6 style D fill:#d1fae5,stroke:#10b981
優先順位の例 
<!DOCTYPE html>
<html lang="ja">
<head>
    <style>
        /* 要素セレクタ（優先度: 低） */
        p {
            color: blue;
        }
        /* クラスセレクタ（優先度: 中） */
        .important {
            color: green;
        }
        /* IDセレクタ（優先度: 高） */
        #special {
            color: red;
        }
    </style>
</head>
<body>
    <p>青色（要素セレクタのみ適用）</p>
    <p class="important">緑色（クラスが要素より優先）</p>
    <p id="special" class="important">赤色（IDが最優先）</p>
</body>
</html>

同じ優先度の場合

同じ詳細度のセレクタが複数ある場合は、後に書かれたスタイルが優先されます。これが「Cascading（カスケード）」の意味です。CSSファイルの下の方に書いたスタイルが、上のスタイルを上書きします。

実習: 3つのセレクタを使ってみよう

HTMLファイルを作成し、要素セレクタ、クラスセレクタ、IDセレクタの3種類を実際に使ってみましょう。

手順:

  1. 新しいフォルダを作り、index.htmlとstyle.cssを作成
  2. HTMLに見出し、段落、リンクなどの要素を追加
  3. いくつかの要素にクラス名やIDを付ける
  4. CSSファイルで、要素セレクタ、クラスセレクタ、IDセレクタを使ってスタイルを適用
  5. ブラウザで開いて、スタイルが正しく適用されているか確認

試してみること:

  • 同じクラスを複数の要素に付けて、まとめてスタイルを適用してみる
  • セレクタの優先順位を確認するため、同じ要素に複数のセレクタを適用してみる
  • CSSのコメントを使って、各ルールの説明を書いてみる

理解度確認クイズ

  1. Q1. CSSの3つの記述方法を挙げ、それぞれの特徴を説明してください。

    回答を表示

    1. インラインスタイル: HTML要素のstyle属性に直接記述。その要素のみに適用されるが、HTMLとCSSが混在するため推奨されない。
    2. 内部スタイルシート: HTMLのheadセクション内にstyleタグで記述。そのHTMLファイル全体に適用される。
    3. 外部スタイルシート: 別ファイル(.css)に記述し、linkタグで読み込む。複数のHTMLで共有でき、最も推奨される方法。

  2. Q2. クラスセレクタとIDセレクタの違いは何ですか？

    回答を表示

    クラスセレクタ（.）: 同じクラス名を複数の要素に使用できる。繰り返し使うスタイルに適している。
    IDセレクタ（#）: 1つのページ内で1つの要素にのみ使用できる。ページ内で一意の要素（ヘッダー、フッターなど）に適している。IDの方が優先順位が高い。

  3. Q3. 次のCSSで、p要素は何色になりますか？理由も説明してください。

    p { color: blue; }
.text { color: green; }
#main { color: red; }

<p id="main" class="text">この文字は何色？</p>
    回答を表示

    答え: 赤色（red）
    理由: セレクタの優先順位は「IDセレクタ > クラスセレクタ > 要素セレクタ」の順です。この例では、p要素に対して3つのセレクタが適用されていますが、最も優先順位が高いIDセレクタ（#main）の赤色が適用されます。

第7章のまとめ

  • CSSはWebページの見た目を整えるための言語で、HTMLと組み合わせて使う
  • CSSの記述方法には、インライン、内部、外部の3種類があり、外部スタイルシートが推奨される
  • CSSは「セレクタ { プロパティ: 値; }」という構文で記述する
  • 基本的なセレクタには、要素セレクタ、クラスセレクタ（.）、IDセレクタ（#）がある
  • セレクタの優先順位は「インライン > ID > クラス > 要素」の順で高くなる

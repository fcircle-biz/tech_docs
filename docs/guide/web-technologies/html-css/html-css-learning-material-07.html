7.1 transitionプロパティ

transitionは、CSSプロパティの変化を滑らかにアニメーションさせる機能です。 

/* 基本的なtransition */
.button {
    background-color: #E44D26;
    color: white;
    padding: 10px 20px;
    transition: all 0.3s ease;
}

.button:hover {
    background-color: #c43d1e;
    transform: scale(1.05);
}
transitionの構文

transition: プロパティ 時間 イージング関数;

  • プロパティ: 変化させるプロパティ（allで全て）
  • 時間: アニメーションの長さ（秒またはミリ秒）
  • イージング関数: 変化の速度カーブ

主なイージング関数

  • ease: ゆっくり開始→速く→ゆっくり終了（デフォルト）
  • linear: 一定速度
  • ease-in: ゆっくり開始
  • ease-out: ゆっくり終了
  • ease-in-out: ゆっくり開始＆終了

7.2 transformプロパティ

transformは、要素を移動、回転、拡大縮小、傾斜させることができます。 

/* 移動 */
.move {
    transform: translate(50px, 100px); /* X方向50px、Y方向100px */
}

/* 回転 */
.rotate {
    transform: rotate(45deg); /* 45度回転 */
}

/* 拡大縮小 */
.scale {
    transform: scale(1.5); /* 1.5倍に拡大 */
}

/* 傾斜 */
.skew {
    transform: skew(10deg, 5deg);
}

/* 複数の変形を組み合わせ */
.combined {
    transform: translate(50px, 0) rotate(45deg) scale(1.2);
}

7.3 @keyframesアニメーション

@keyframesを使うと、複雑なアニメーションを定義できます。 

/* アニメーションの定義 */
@keyframes fadeIn {
    from {
        opacity: 0;
    }
    to {
        opacity: 1;
    }
}

/* アニメーションの適用 */
.fade-in {
    animation: fadeIn 2s ease;
}

複数ステップのアニメーション

@keyframes bounce {
    0% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-30px);
    }
    100% {
        transform: translateY(0);
    }
}

.bounce-animation {
    animation: bounce 1s infinite;
}
animationプロパティ

animation: 名前 時間 イージング 繰り返し;

  • 名前: @keyframesで定義した名前
  • 時間: アニメーションの長さ
  • 繰り返し: 回数またはinfinite（無限）

7.4 実践的なアニメーション例

ローディングアニメーション

@keyframes spin {
    from {
        transform: rotate(0deg);
    }
    to {
        transform: rotate(360deg);
    }
}

.loader {
    width: 50px;
    height: 50px;
    border: 5px solid #f3f3f3;
    border-top: 5px solid #E44D26;
    border-radius: 50%;
    animation: spin 1s linear infinite;
}

フェードインアニメーション

@keyframes slideInFromLeft {
    from {
        opacity: 0;
        transform: translateX(-100%);
    }
    to {
        opacity: 1;
        transform: translateX(0);
    }
}

.slide-in {
    animation: slideInFromLeft 1s ease-out;
}
実習 7-2: パルスアニメーション
<style>
@keyframes pulse {
    0%, 100% {
        transform: scale(1);
        opacity: 1;
    }
    50% {
        transform: scale(1.1);
        opacity: 0.7;
    }
}

.pulse-button {
    padding: 15px 30px;
    background-color: #E44D26;
    color: white;
    border: none;
    border-radius: 5px;
    font-size: 18px;
    cursor: pointer;
    animation: pulse 2s infinite;
}
</style>

<button class="pulse-button">クリックしてね！</button>

理解度確認クイズ
  1. 問題1: transitionとanimationの違いは？
    解答例

    transitionは状態変化（ホバーなど）に応じて自動的にアニメーションし、animationは@keyframesで定義した独立したアニメーションを実行します。

  2. 問題2: transformで要素を45度回転させるコードは？
    解答例

    transform: rotate(45deg);

      2. 5.1 リストとは何か

      Webページで情報を整理して表示する方法として、「リスト」と「テーブル」があります。リストは箇条書きで情報を並べる方法で、テーブルは行と列で構造化されたデータを表示する方法です。

      リストは主に3種類あります：

      • 順序なしリスト（Unordered List）：項目の順序が重要でない箇条書き（・で表示）
      • 順序ありリスト（Ordered List）：項目の順序が重要な箇条書き（1, 2, 3で表示）
      • 定義リスト（Definition List）：用語とその説明をセットで表示

      リストを使うメリット

      リストを使うことで、情報を視覚的に整理し、読みやすくできます。また、スクリーンリーダー（視覚障害者向けの読み上げソフト）がリストを認識しやすくなり、アクセシビリティが向上します。

      5.5 テーブルの基本 - `

      `の構造

      テーブルは、行と列で構成された表を作成するためのHTML要素です。データを整理して表示するのに非常に便利です。

      テーブルを作成するための主なタグは以下の通りです：

      • <table>：テーブル全体を囲む
      • <tr>（Table Row）：行を定義する
      • <th>（Table Header）：見出しセルを定義する
      • <td>（Table Data）：データセルを定義する
      basic-table.html 
      <!-- 成績表 -->
<table>
    <tr>
        <th>科目</th>
        <th>点数</th>
    </tr>
    <tr>
        <td>国語</td>
        <td>85点</td>
    </tr>
    <tr>
        <td>数学</td>
        <td>92点</td>
    </tr>
    <tr>
        <td>英語</td>
        <td>78点</td>
    </tr>
</table>

      このコードは、見出し行（科目と点数）とデータ行（各科目の点数）からなる簡単な表を作成します。

      5.6 テーブルの構造 - ヘッダー、ボディ、フッター

      大きなテーブルでは、見出し部分、データ部分、まとめ部分を明確に分けることで、より構造化された表を作成できます。そのために以下のタグを使います：

      • <thead>：テーブルのヘッダー部分
      • <tbody>：テーブルのデータ部分
      • <tfoot>：テーブルのフッター部分（合計など）
      structured-table.html 
      <!-- 売上表 -->
<table>
    <thead>
        <tr>
            <th>商品名</th>
            <th>単価</th>
            <th>数量</th>
            <th>小計</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>りんご</td>
            <td>100円</td>
            <td>5個</td>
            <td>500円</td>
        </tr>
        <tr>
            <td>バナナ</td>
            <td>80円</td>
            <td>3個</td>
            <td>240円</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <th colspan="3">合計</th>
            <th>740円</th>
        </tr>
    </tfoot>
</table>

      この例では、`colspan="3"`属性を使って、「合計」のセルを3列分結合しています。`colspan`は列方向の結合、`rowspan`は行方向の結合に使います。

      セルの結合

      `colspan`属性で横方向（列）、`rowspan`属性で縦方向（行）のセルを結合できます。例えば、`

      		`とすると、そのセルが2列分の幅を持つようになります。

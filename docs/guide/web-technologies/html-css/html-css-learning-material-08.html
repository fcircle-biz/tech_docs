第8章

色とテキストのスタイリング - 見た目を整える

文字色・背景色の設定、フォントの変更、文字サイズ・行間の調整など、テキストの見た目を整える方法を学びます。色の指定方法（カラーネーム、HEX、RGB）についても理解します。

この章で学ぶこと

  • CSSでの色の指定方法（カラーネーム、HEX、RGB、RGBA）を理解する
  • 文字色と背景色を変更し、読みやすいデザインを作成する
  • フォントの種類、サイズ、太さを適切に設定する
  • 行間や文字間隔を調整して、読みやすいテキストレイアウトを実現する

8.1 色の指定方法

CSSで色を指定する方法には、主に3つあります。それぞれの方法を理解して、適切に使い分けましょう。

カラーネーム（色名）

英語の色名で指定する方法です。red、blue、greenなど、約140種類の色名が使えます。シンプルですが、細かい色の調整には向いていません。

カラーネームの例 
/* カラーネームで色を指定 */
h1 {
    color: blue;  /* 青色 */
}

p {
    color: darkgray;  /* ダークグレー */
    background-color: lightblue;  /* 薄い青色 */
}

HEX（16進数カラーコード）

#RRGGBBの形式で、赤(R)・緑(G)・青(B)の3色を16進数（00〜FF）で指定します。Web制作で最もよく使われる方法です。

HEXカラーコードの例 
/* HEXカラーコードで色を指定 */
h1 {
    color: #3b82f6;  /* 青色 */
}

p {
    color: #475569;  /* グレー */
    background-color: #f1f5f9;  /* 薄いグレー */
}

HEXカラーコードの読み方

#RRGGBB形式で、RR（赤）、GG（緑）、BB（青）の3つの要素がそれぞれ00（最小）〜FF（最大）の範囲で色の強さを表します。例えば、#FF0000は純粋な赤色（赤が最大、緑と青が0）になります。

RGB / RGBA

rgb(赤, 緑, 青)の形式で、0〜255の範囲で色を指定します。RGBA形式では、最後に透明度（0.0〜1.0）を追加できます。

RGB/RGBAの例 
/* RGBで色を指定 */
h1 {
    color: rgb(59, 130, 246);  /* 青色 */
}

/* RGBAで透明度も指定 */
.overlay {
    background-color: rgba(0, 0, 0, 0.5);  /* 半透明の黒 */
}

色の指定方法の比較

指定方法 形式 例 用途
カラーネーム 色名 red, blue 簡単な色指定
HEX #RRGGBB #3b82f6 一般的な色指定
RGB rgb(R, G, B) rgb(59, 130, 246) 数値で細かく指定
RGBA rgba(R, G, B, A) rgba(0, 0, 0, 0.5) 透明度が必要な場合

8.2 文字色と背景色の設定

colorプロパティで文字の色を、background-colorプロパティで背景色を変更できます。

文字色と背景色の設定 
/* 文字色の設定 */
h1 {
    color: #1e40af;  /* 濃い青色 */
}

/* 背景色の設定 */
.highlight {
    background-color: #fef3c7;  /* 薄い黄色 */
}

/* 文字色と背景色を組み合わせる */
.button {
    color: white;  /* 白い文字 */
    background-color: #3b82f6;  /* 青い背景 */
}

重要ポイント

文字色と背景色のコントラスト（明暗の差）を十分に確保しましょう。薄い色の文字を薄い背景に表示すると、読みにくくなってしまいます。基本的には、明るい背景には濃い文字、濃い背景には明るい文字を使います。

実習 8-1: 色を使ってみよう

実際に文字色と背景色を設定して、見た目を変えてみましょう。

手順

  1. 1 HTMLファイルを作成し、見出しといくつかの段落を追加します
  2. 2 CSSで見出しの文字色を青色（#3b82f6）に設定します
  3. 3 段落の背景色を薄いグレー（#f1f5f9）に、文字色を濃いグレー（#334155）に設定します
  4. 4 ブラウザで開いて、色が正しく表示されているか確認します

8.3 フォントの設定

フォント（書体）を変更することで、Webページの印象を大きく変えることができます。CSSでは、フォントの種類、サイズ、太さなどを細かく指定できます。

font-family - フォントの種類

font-familyプロパティで、使用するフォントを指定します。複数のフォントをカンマ区切りで指定すると、最初のフォントがない場合に次のフォントが使われます。

フォントの種類を指定 
/* ゴシック体（サンセリフ体） */
body {
    font-family: 'Noto Sans JP', sans-serif;
}

/* 明朝体（セリフ体） */
.serif-text {
    font-family: 'Noto Serif JP', serif;
}

/* 等幅フォント（コード表示用） */
code {
    font-family: 'Courier New', monospace;
}

フォントのフォールバック

指定したフォントがユーザーのパソコンにインストールされていない場合に備えて、代替フォントを指定しておくことが重要です。最後には必ずsans-serifやserifなどの一般的なフォントファミリー名を書きます。

font-size - 文字サイズ

font-sizeプロパティで文字のサイズを変更します。単位にはpx（ピクセル）、em、remなどが使えます。

文字サイズの指定 
/* ピクセルで指定（絶対サイズ） */
h1 {
    font-size: 32px;
}

p {
    font-size: 16px;
}

/* remで指定（ルート要素を基準） */
h2 {
    font-size: 2rem;  /* ルート要素の2倍 */
}

font-weight - 文字の太さ

font-weightプロパティで文字の太さを変更します。normal（通常）、bold（太字）のほか、100〜900の数値でも指定できます。

文字の太さを指定 
/* 通常の太さ */
p {
    font-weight: normal;  /* または 400 */
}

/* 太字 */
strong {
    font-weight: bold;  /* または 700 */
}

/* 細字 */
.light-text {
    font-weight: 300;
}

8.4 文字サイズと行間の調整

文字サイズと行間（行の高さ）を適切に設定することで、読みやすいテキストレイアウトを実現できます。

line-height - 行間

line-heightプロパティで、行の高さ（行間）を指定します。数値だけ（単位なし）で指定すると、文字サイズに対する倍率として扱われます。

行間の設定 
/* 倍率で指定（推奨） */
p {
    font-size: 16px;
    line-height: 1.6;  /* 文字サイズの1.6倍 */
}

/* ピクセルで指定 */
h1 {
    font-size: 32px;
    line-height: 48px;  /* 固定値 */
}

ヒント

一般的に、本文テキストの行間は1.5〜1.8倍程度が読みやすいと言われています。行間が狭すぎると文章が詰まって見えますし、広すぎると文章のまとまりが失われてしまいます。

text-align - 文字揃え

text-alignプロパティで、テキストの揃え方を指定します。

文字揃えの設定 
/* 左揃え（デフォルト） */
.left-align {
    text-align: left;
}

/* 中央揃え */
.center-align {
    text-align: center;
}

/* 右揃え */
.right-align {
    text-align: right;
}

8.5 テキストの装飾

テキストに下線を引いたり、大文字に変換したり、文字間隔を調整したりすることで、デザインのアクセントを加えることができます。

text-decoration - テキスト装飾

text-decorationプロパティで、下線、上線、打ち消し線などを追加できます。

テキスト装飾の設定 
/* 下線 */
.underline {
    text-decoration: underline;
}

/* 打ち消し線 */
.line-through {
    text-decoration: line-through;
}

/* 装飾なし（リンクの下線を消す） */
a {
    text-decoration: none;
}

text-transform - 大文字・小文字の変換

text-transformプロパティで、テキストを自動的に大文字や小文字に変換できます。

大文字・小文字の変換 
/* すべて大文字に */
.uppercase {
    text-transform: uppercase;
}

/* すべて小文字に */
.lowercase {
    text-transform: lowercase;
}

/* 各単語の最初を大文字に */
.capitalize {
    text-transform: capitalize;
}

letter-spacing - 文字間隔

letter-spacingプロパティで、文字と文字の間隔を調整できます。

文字間隔の調整 
/* 文字間隔を広げる */
h1 {
    letter-spacing: 2px;
}

/* 文字間隔を狭める */
.tight {
    letter-spacing: -0.5px;
}

/* 通常の間隔 */
p {
    letter-spacing: normal;
}

実習 8-2: 総合演習

学んだスタイルを組み合わせて、読みやすいカード型コンポーネントを作成してみましょう。

手順

  1. 1 HTMLでカード要素（div.card）を作成し、見出し、段落、ボタンを含めます
  2. 2 カードの背景色を白（#ffffff）、枠線を薄いグレーに設定します
  3. 3 見出しを青色（#3b82f6）、太字（bold）、サイズ24pxに設定します
  4. 4 段落の文字サイズを16px、行間を1.6、色を濃いグレー（#475569）に設定します
  5. 5 ボタンに白い文字、青い背景、下線なしを設定します

理解度確認クイズ

  1. Q1. CSSで色を指定する3つの主な方法を説明してください。

    回答を表示

    1. カラーネーム: 英語の色名（red、blueなど）で指定する方法。シンプルですが細かい調整には向きません。
    2. HEX（16進数）: #RRGGBBの形式で指定する方法。Web制作で最もよく使われます。
    3. RGB/RGBA: rgb(赤, 緑, 青)の形式で指定する方法。RGBAでは透明度も指定できます。

  2. Q2. 読みやすいテキストを作るために、文字色と背景色を設定する際の注意点は何ですか？

    回答を表示

    文字色と背景色のコントラスト（明暗の差）を十分に確保することが重要です。明るい背景には濃い文字、濃い背景には明るい文字を使います。コントラストが低いと、テキストが読みにくくなってしまいます。

  3. Q3. line-heightプロパティの値を単位なしの数値で指定すると、どのように扱われますか？

    回答を表示

    文字サイズに対する倍率として扱われます。例えば、font-sizeが16pxでline-heightが1.6の場合、行の高さは16px × 1.6 = 25.6pxになります。単位なしで指定する方法が推奨されています。

第8章のまとめ

  • CSSでの色指定には、カラーネーム、HEX、RGB/RGBAの3つの方法がある
  • colorプロパティで文字色、background-colorプロパティで背景色を設定する
  • フォントはfont-family、font-size、font-weightで細かく制御できる
  • line-heightで行間を、text-alignで文字揃えを設定する
  • text-decoration、text-transform、letter-spacingでテキストに装飾を加えられる
  • 読みやすさを考慮して、適切なコントラスト、行間、文字サイズを設定することが重要

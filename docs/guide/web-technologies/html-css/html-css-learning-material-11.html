第11章

フレックスボックス - 柔軟なレイアウト作成

Flexboxレイアウトの基本概念、主軸と交差軸、アイテムの配置と整列（justify-content、align-items）について学びます。現代的で柔軟なレイアウトの作成方法を習得します。

この章で学ぶこと

  • Flexboxの基本概念と利点を理解する
  • 主軸（main axis）と交差軸（cross axis）の違いを説明できるようになる
  • justify-contentとalign-itemsを使った配置方法を習得する
  • flex-direction、flex-wrap、gapなどのプロパティを理解する
  • 実践的なナビゲーションメニューを作成できるようになる

11.1 Flexboxとは何か

Flexbox（Flexible Box Layout）は、要素を柔軟に配置するためのCSSレイアウトモジュールです。従来のレイアウト手法（floatやpositionなど）と比べて、簡単かつ直感的に、複雑なレイアウトを実現できます。

Flexboxは、特に1次元のレイアウト（横一列、または縦一列）に強みがあり、ナビゲーションメニュー、カードの並び、フォームの配置など、現代的なWebデザインで広く使われています。

Flexboxの利点

  • • 要素を簡単に中央揃えできる
  • • 要素の順序を自由に変更できる
  • • 要素のサイズを自動調整できる
  • • レスポンシブデザインに適している

11.2 Flexboxの基本構造

Flexboxは、フレックスコンテナ（Flex Container）とフレックスアイテム（Flex Items）の2つの要素で構成されます。

flexbox-basic.html 
<!-- HTML -->
<div class="container">
    <div class="item">アイテム1</div>
    <div class="item">アイテム2</div>
    <div class="item">アイテム3</div>
</div>
flexbox-basic.css 
/* フレックスコンテナ */
.container {
    display: flex;  /* Flexboxを有効化 */
    background-color: #f1f5f9;
    padding: 20px;
}

/* フレックスアイテム */
.item {
    background-color: #3b82f6;
    color: white;
    padding: 20px;
    margin: 10px;
}

display: flex の効果

親要素にdisplay: flexを指定すると、その直下の子要素が自動的にフレックスアイテムになり、横一列に並びます。

11.3 主軸と交差軸の理解

Flexboxには、主軸（Main Axis）と交差軸（Cross Axis）という2つの軸があります。この概念を理解することが、Flexboxを使いこなす鍵です。

主軸と交差軸の方向

主軸と交差軸の違い

軸 説明 制御プロパティ
主軸 アイテムが並ぶ方向の軸 justify-content
交差軸 主軸に対して垂直な軸 align-items、align-content

覚え方のコツ

flex-direction: rowの場合、主軸は横方向、交差軸は縦方向。flex-direction: columnの場合は逆になります。

11.4 justify-content - 主軸方向の配置

justify-contentプロパティは、フレックスアイテムを主軸方向にどのように配置するかを指定します。

justify-content.css 
.container {
    display: flex;
    justify-content: center;  /* 中央揃え */
}

/* その他の値 */
justify-content: flex-start;    /* 左揃え（デフォルト） */
justify-content: flex-end;      /* 右揃え */
justify-content: space-between; /* 両端揃え */
justify-content: space-around;  /* 均等配置（両端に半分のスペース） */
justify-content: space-evenly;  /* 完全均等配置 */

justify-contentの主な値

値 説明
flex-start 主軸の開始位置に配置（横方向なら左）
flex-end 主軸の終了位置に配置（横方向なら右）
center 主軸の中央に配置
space-between 両端に配置し、残りを等間隔で配置
space-around 各アイテムの周囲に均等なスペース
space-evenly 完全に均等なスペース

11.5 align-items - 交差軸方向の配置

align-itemsプロパティは、フレックスアイテムを交差軸方向にどのように配置するかを指定します。

align-items.css 
.container {
    display: flex;
    align-items: center;  /* 交差軸の中央揃え */
    height: 200px;        /* 高さを指定 */
}

/* その他の値 */
align-items: flex-start;  /* 上揃え */
align-items: flex-end;    /* 下揃え */
align-items: stretch;     /* 伸縮（デフォルト） */
align-items: baseline;    /* ベースライン揃え */

align-itemsの主な値

値 説明
stretch コンテナの高さいっぱいに伸ばす（デフォルト）
flex-start 交差軸の開始位置に配置（横方向なら上）
flex-end 交差軸の終了位置に配置（横方向なら下）
center 交差軸の中央に配置
baseline テキストのベースラインで揃える

中央揃えの実現

justify-content: centerとalign-items: centerを組み合わせることで、要素を完全に中央に配置できます。従来の方法よりも簡単です。

11.6 flex-directionとflex-wrap

flex-directionは主軸の方向を、flex-wrapはアイテムの折り返しを制御します。

flex-direction.css 
/* flex-directionの例 */
.container {
    display: flex;
    flex-direction: row;  /* 横方向（デフォルト） */
}

/* その他の値 */
flex-direction: row-reverse;    /* 横方向（逆順） */
flex-direction: column;         /* 縦方向 */
flex-direction: column-reverse; /* 縦方向（逆順） */
flex-wrap.css 
/* flex-wrapの例 */
.container {
    display: flex;
    flex-wrap: wrap;  /* 折り返しを許可 */
}

/* その他の値 */
flex-wrap: nowrap;      /* 折り返さない（デフォルト） */
flex-wrap: wrap-reverse; /* 逆方向に折り返す */

レスポンシブ対応

flex-wrap: wrapを使うと、画面幅に応じてアイテムが自動的に折り返されるため、レスポンシブデザインに最適です。

11.7 gapプロパティ - アイテム間の余白

gapプロパティを使うと、フレックスアイテム間の余白を簡単に設定できます。従来のmarginを使う方法よりも直感的です。

gap.css 
.container {
    display: flex;
    gap: 20px;  /* 全方向に20pxの余白 */
}

/* 行と列で異なる余白を設定 */
.container {
    display: flex;
    flex-wrap: wrap;
    row-gap: 30px;     /* 行間の余白 */
    column-gap: 20px;  /* 列間の余白 */
}

実習: ナビゲーションメニューの作成

Flexboxを使って、レスポンシブなナビゲーションメニューを作成してみましょう。

navigation.html 
<nav class="navbar">
    <div class="logo">MyWebsite</div>
    <ul class="nav-menu">
        <li><a href="#">ホーム</a></li>
        <li><a href="#">サービス</a></li>
        <li><a href="#">お問い合わせ</a></li>
    </ul>
</nav>
navigation.css 
/* ナビゲーションバー全体 */
.navbar {
    display: flex;
    justify-content: space-between;  /* 左右に配置 */
    align-items: center;             /* 垂直方向に中央揃え */
    background-color: #1e40af;
    padding: 15px 30px;
}

/* ロゴ */
.logo {
    color: white;
    font-size: 24px;
    font-weight: bold;
}

/* メニューリスト */
.nav-menu {
    display: flex;
    gap: 30px;  /* メニュー間の余白 */
    list-style: none;
    margin: 0;
    padding: 0;
}

/* メニューリンク */
.nav-menu a {
    color: white;
    text-decoration: none;
}

理解度確認クイズ

  1. Q1. Flexboxの主軸と交差軸の違いを説明してください。

    回答を表示

    主軸はフレックスアイテムが並ぶ方向の軸で、flex-directionによって決まります。rowの場合は横方向、columnの場合は縦方向です。交差軸は主軸に対して垂直な軸です。主軸の配置はjustify-content、交差軸の配置はalign-itemsで制御します。

  2. Q2. 要素を完全に中央配置するには、どのプロパティを組み合わせますか？

    回答を表示

    display: flex、justify-content: center、align-items: centerを組み合わせます。これにより、主軸方向と交差軸方向の両方で中央に配置されます。

  3. Q3. flex-wrap: wrapの役割を説明してください。

    回答を表示

    flex-wrap: wrapは、フレックスアイテムがコンテナの幅を超える場合に、自動的に次の行に折り返すようにします。これにより、レスポンシブデザインが容易になり、画面幅が小さくなってもアイテムが適切に表示されます。

第11章のまとめ

  • Flexboxは柔軟なレイアウトを実現する現代的なCSS機能
  • display: flexを指定すると、直下の子要素がフレックスアイテムになる
  • 主軸はアイテムが並ぶ方向、交差軸は主軸に垂直な方向
  • justify-contentで主軸方向、align-itemsで交差軸方向の配置を制御
  • flex-directionで並びの方向、flex-wrapで折り返しを制御できる
  • gapプロパティでアイテム間の余白を簡単に設定できる

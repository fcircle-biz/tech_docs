インターネットとWebの仕組み

インターネットとWebの違いを理解し、私たちが普段使っているWebブラウザがどのような役割を果たしているのかを学びます。

この章で学ぶこと

  • インターネットとWebの違いを説明できるようになる
  • Webブラウザの役割を理解する
  • Webページが表示される基本的な仕組みを把握する

1.1 インターネットとは何か

インターネットは、世界中のコンピュータを繋ぐ巨大なネットワークです。 家庭のパソコン、会社のサーバー、スマートフォンなど、あらゆる機器がこのネットワークに接続されています。

身近な例で考えてみよう

インターネットは「道路網」のようなものです。 日本中、世界中に道路があり、その道路を使って車（データ）が行き来しています。 道路自体は何かを運ぶためのインフラ（基盤）であり、 その上を走る車や荷物は別のものです。

インターネットができたのは1960年代のことです。もともとはアメリカの軍事・研究目的で作られました。 その後、大学や研究機関で広まり、1990年代から一般の人々も使えるようになりました。

1.2 Webとは何か

Web（ウェブ）、正式にはWorld Wide Web（ワールド・ワイド・ウェブ）は、 インターネット上で動く「サービス」の一つです。

ポイント: インターネット と Web の違い

インターネット = 世界中のコンピュータを繋ぐ「道路網」（インフラ）

Web = その道路網の上を走る「車」（サービス）の一つ

インターネットの上では、Web以外にも様々なサービスが動いています。

Web

Webサイトの閲覧、Webアプリの利用など

電子メール

GmailやOutlookなどのメールサービス

FTP

ファイルの転送（サーバーへのアップロードなど）

ストリーミング

YouTubeやNetflixなどの動画配信

私たちが普段「インターネットを使う」と言うとき、多くの場合は「Webを使う」ことを指しています。 Webサイトを見たり、SNSを使ったり、ネットショッピングをしたりするのは、すべてWebの利用です。

1.3 Webブラウザの役割

Webブラウザ（ブラウザ）は、Webページを表示するためのソフトウェアです。 Google Chrome、Safari、Firefox、Microsoft Edgeなどが代表的なブラウザです。

ブラウザの主な役割

1
Webサーバーにデータを要求する

URLを入力すると、ブラウザはそのWebサーバーに「このページのデータをください」と要求を送ります。

2
受け取ったデータを画面に表示する

サーバーから受け取ったHTML、CSS、画像などのデータを解釈し、人間が見やすい形に変換して画面に表示します。

Webページが表示されるまでの流れ

sequenceDiagram participant U as あなた participant B as ブラウザ participant S as Webサーバー U->>B: URLを入力 B->>S: このページをください Note over S: ページのデータを準備 S->>B: HTML/CSS/画像などを送信 Note over B: データを解釈して表示 B->>U: Webページを表示

ブラウザはあなたとWebサーバーの間の「仲介役」

身近な例で考えてみよう

ブラウザは「通訳」のようなものです。 Webサーバーが送ってくるデータ（HTML）は、コンピュータ向けの言葉で書かれています。 ブラウザはそれを人間が理解できる形（きれいなWebページ）に翻訳してくれます。

1.4 Webページの正体: HTML

WebページはHTML（HyperText Markup Language）という言語で書かれています。 HTMLは、文章の構造や意味を伝えるための「印」をつける言語です。

HTMLの例 
<!DOCTYPE html>
<html>
<head>
    <title>はじめてのWebページ</title>
</head>
<body>
    <h1>こんにちは！</h1>
    <p>これは私のはじめてのWebページです。</p>
</body>
</html>

上のコードを見ると、<h1>や<p>のような記号（タグ）が使われています。 これがHTMLの「印」です。ブラウザはこれを読み取り、画面に表示します。

コンピュータが見ているもの
<h1>こんにちは！</h1>
<p>これは私のはじめての
Webページです。</p>
あなたが見ているもの

こんにちは！

これは私のはじめてのWebページです。

ブラウザがHTMLを「翻訳」してくれるおかげで、私たちは読みやすいWebページを見ることができます。

理解度確認クイズ

  1. Q1. インターネットとWebの関係として正しいものはどれですか？

    回答を表示

    Webはインターネット上で動くサービスの一つです。インターネットが「道路網」だとすれば、Webは「その上を走る車」のようなものです。

  2. Q2. Webブラウザの主な役割を2つ挙げてください。

    回答を表示

    (1) Webサーバーにデータを要求する
    (2) 受け取ったデータ（HTML等）を画面に表示する

  3. Q3. Webページを作るために使われる言語は何ですか？

    回答を表示

    HTML（HyperText Markup Language）です。HTMLはWebページの構造を記述するための言語で、ブラウザがこれを解釈して画面に表示します。

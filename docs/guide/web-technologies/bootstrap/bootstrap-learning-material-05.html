第5章: フォームとインプット

美しく使いやすいフォームの作成

この章で学ぶこと
  • Bootstrapのフォームコントロールの基本的な使い方を理解する
  • インプットグループとフィードバックメッセージの実装方法を学ぶ
  • チェックボックスとラジオボタンのスタイリング方法を習得する
  • フォームのバリデーション（検証）スタイルの適用方法を学ぶ

5.1 フォームの基礎

フォームは、ユーザーからの入力を受け取るための重要なUI要素です。 Bootstrapは、見た目が美しく、使いやすいフォームを簡単に作成できるスタイルを提供しています。

基本的なフォーム要素

<form>
    <!-- テキスト入力 -->
    <div class="mb-3">
        <label for="username" class="form-label">ユーザー名</label>
        <input type="text" class="form-control" id="username" placeholder="例: yamada_taro">
    </div>

    <!-- メール入力 -->
    <div class="mb-3">
        <label for="email" class="form-label">メールアドレス</label>
        <input type="email" class="form-control" id="email" placeholder="name@example.com">
    </div>

    <!-- パスワード入力 -->
    <div class="mb-3">
        <label for="password" class="form-label">パスワード</label>
        <input type="password" class="form-control" id="password">
    </div>

    <button type="submit" class="btn btn-primary">送信</button>
</form>
フォームの構造
  • .mb-3: 各入力項目の下に適切なマージンを追加
  • .form-label: ラベルのスタイル
  • .form-control: テキストボックスなどの入力フィールドのスタイル

5.2 様々な入力タイプ

テキストエリア（複数行入力）

<div class="mb-3">
    <label for="message" class="form-label">メッセージ</label>
    <textarea class="form-control" id="message" rows="3"></textarea>
</div>

セレクトボックス（ドロップダウン）

<div class="mb-3">
    <label for="country" class="form-label">国を選択</label>
    <select class="form-select" id="country">
        <option selected>選択してください...</option>
        <option value="1">日本</option>
        <option value="2">アメリカ</option>
        <option value="3">イギリス</option>
    </select>
</div>

ファイル選択

<div class="mb-3">
    <label for="formFile" class="form-label">ファイル選択</label>
    <input class="form-control" type="file" id="formFile">
</div>

5.3 チェックボックスとラジオボタン

チェックボックス

複数の選択肢から複数を選べる場合に使用します。

<div class="form-check">
    <input class="form-check-input" type="checkbox" id="check1">
    <label class="form-check-label" for="check1">
        ニュースレターを受け取る
    </label>
</div>

<div class="form-check">
    <input class="form-check-input" type="checkbox" id="check2">
    <label class="form-check-label" for="check2">
        利用規約に同意する
    </label>
</div>

ラジオボタン

複数の選択肢から1つだけ選べる場合に使用します。

<div class="form-check">
    <input class="form-check-input" type="radio" name="gender" id="male" checked>
    <label class="form-check-label" for="male">
        男性
    </label>
</div>

<div class="form-check">
    <input class="form-check-input" type="radio" name="gender" id="female">
    <label class="form-check-label" for="female">
        女性
    </label>
</div>
ラジオボタンの注意点

ラジオボタンでグループ化するには、すべてのラジオボタンに同じname属性を指定します。 上記の例ではname="gender"で統一しています。

5.4 インプットグループ

インプットグループは、入力フィールドにテキストやボタンを前後に追加できる機能です。

テキストの追加

<!-- 前にテキストを追加 -->
<div class="input-group mb-3">
    <span class="input-group-text">@</span>
    <input type="text" class="form-control" placeholder="ユーザー名">
</div>

<!-- 後ろにテキストを追加 -->
<div class="input-group mb-3">
    <input type="text" class="form-control" placeholder="価格">
    <span class="input-group-text">円</span>
</div>

<!-- 前後両方に追加 -->
<div class="input-group mb-3">
    <span class="input-group-text">￥</span>
    <input type="text" class="form-control" placeholder="金額">
    <span class="input-group-text">.00</span>
</div>

ボタンの追加

<div class="input-group mb-3">
    <input type="text" class="form-control" placeholder="検索キーワード">
    <button class="btn btn-primary" type="button">検索</button>
</div>

5.5 フォームのバリデーション

バリデーション（検証）は、ユーザーの入力が正しいかをチェックし、 適切なフィードバックを表示する機能です。

成功・エラーのフィードバック

<form>
    <!-- 正常な入力 -->
    <div class="mb-3">
        <label for="validInput" class="form-label">正常な入力</label>
        <input type="text" class="form-control is-valid" id="validInput" value="正しい値">
        <div class="valid-feedback">
            入力内容は正しいです！
        </div>
    </div>

    <!-- エラーのある入力 -->
    <div class="mb-3">
        <label for="invalidInput" class="form-label">エラーのある入力</label>
        <input type="text" class="form-control is-invalid" id="invalidInput">
        <div class="invalid-feedback">
            この項目は必須です。
        </div>
    </div>

    <button type="submit" class="btn btn-primary">送信</button>
</form>
バリデーションクラスの使い分け
  • .is-valid: 入力が正しい場合（緑色の枠線）
  • .is-invalid: 入力にエラーがある場合（赤色の枠線）
  • .valid-feedback: 成功メッセージ
  • .invalid-feedback: エラーメッセージ

5.6 フォームレイアウト

横並びフォーム

グリッドシステムを使って、ラベルと入力フィールドを横並びにできます。

<form>
    <div class="row mb-3">
        <label for="inputEmail" class="col-sm-2 col-form-label">メール</label>
        <div class="col-sm-10">
            <input type="email" class="form-control" id="inputEmail">
        </div>
    </div>
    
    <div class="row mb-3">
        <label for="inputPassword" class="col-sm-2 col-form-label">パスワード</label>
        <div class="col-sm-10">
            <input type="password" class="form-control" id="inputPassword">
        </div>
    </div>
</form>

インラインフォーム

複数の入力フィールドを1行に配置できます。

<form class="row g-3">
    <div class="col-auto">
        <label for="name" class="visually-hidden">名前</label>
        <input type="text" class="form-control" id="name" placeholder="名前">
    </div>
    <div class="col-auto">
        <label for="email" class="visually-hidden">メール</label>
        <input type="email" class="form-control" id="email" placeholder="メール">
    </div>
    <div class="col-auto">
        <button type="submit" class="btn btn-primary">送信</button>
    </div>
</form>
実習 5-1: お問い合わせフォームの作成

ユーザー登録フォームを作成し、バリデーションメッセージを表示してみましょう。

実装例
<div class="container mt-5">
    <h2>お問い合わせフォーム</h2>
    <form>
        <div class="mb-3">
            <label for="name" class="form-label">お名前</label>
            <input type="text" class="form-control" id="name" required>
        </div>

        <div class="mb-3">
            <label for="email" class="form-label">メールアドレス</label>
            <input type="email" class="form-control" id="email" required>
        </div>

        <div class="mb-3">
            <label for="category" class="form-label">お問い合わせ種類</label>
            <select class="form-select" id="category">
                <option selected>選択してください...</option>
                <option value="1">商品について</option>
                <option value="2">サポート</option>
                <option value="3">その他</option>
            </select>
        </div>

        <div class="mb-3">
            <label for="message" class="form-label">お問い合わせ内容</label>
            <textarea class="form-control" id="message" rows="5"></textarea>
        </div>

        <div class="form-check mb-3">
            <input class="form-check-input" type="checkbox" id="agree">
            <label class="form-check-label" for="agree">
                個人情報の取り扱いに同意します
            </label>
        </div>

        <button type="submit" class="btn btn-primary">送信</button>
    </form>
</div>
理解度確認クイズ
  1. チェックボックスとラジオボタンの違いは何ですか？
    解答例
    • チェックボックス: 複数の選択肢から複数を選べる
    • ラジオボタン: 複数の選択肢から1つだけ選べる（同じname属性でグループ化）
  2. .is-invalidクラスの役割は何ですか？
    解答例

    入力フィールドにエラーがあることを視覚的に示すクラスです。フィールドが赤色の枠線で表示されます。

  3. インプットグループの用途を説明してください。
    解答例

    入力フィールドの前後にテキストやボタンを追加して、入力の意味を明確にしたり、関連する操作を提供したりするために使用します。例：「@」マークの後にユーザー名を入力、価格の後に「円」を表示など。

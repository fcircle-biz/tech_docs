第4章: 基本コンポーネント

よく使用されるUIコンポーネント

この章で学ぶこと
  • Bootstrapのボタンコンポーネントの種類と使い方を理解する
  • カードコンポーネントの構造と活用方法を学ぶ
  • バッジとアラートの実装方法を習得する
  • パンくずリストとページネーションの作成方法を学ぶ

4.1 ボタン（Button）

ボタンは、ユーザーのアクション（クリック、タップ）を受け付けるUI要素です。 Bootstrapには、用途に応じた様々なスタイルのボタンが用意されています。

基本的なボタン

<button type="button" class="btn btn-primary">プライマリ</button>
<button type="button" class="btn btn-secondary">セカンダリ</button>
<button type="button" class="btn btn-success">成功</button>
<button type="button" class="btn btn-danger">危険</button>
<button type="button" class="btn btn-warning">警告</button>
<button type="button" class="btn btn-info">情報</button>
ボタンの色の使い分け
  • btn-primary: 主要なアクション（送信、保存など）
  • btn-success: 成功や確認のアクション
  • btn-danger: 削除や重要な操作
  • btn-secondary: 副次的なアクション（キャンセルなど）

アウトラインボタン

背景色なしで、境界線だけのボタンも作成できます。

<button type="button" class="btn btn-outline-primary">プライマリ</button>
<button type="button" class="btn btn-outline-success">成功</button>

ボタンのサイズ

<!-- 大きいボタン -->
<button type="button" class="btn btn-primary btn-lg">大きいボタン</button>

<!-- 通常サイズ -->
<button type="button" class="btn btn-primary">通常ボタン</button>

<!-- 小さいボタン -->
<button type="button" class="btn btn-primary btn-sm">小さいボタン</button>

ブロックレベルボタン（全幅）

<div class="d-grid">
    <button class="btn btn-primary" type="button">全幅ボタン</button>
</div>

4.2 カード（Card）

カードは、関連する情報をまとめて表示するための柔軟なコンテナです。 画像、テキスト、ボタンなどを組み合わせて使用できます。

基本的なカード

<div class="card" style="width: 18rem;">
    <img src="image.jpg" class="card-img-top" alt="カード画像">
    <div class="card-body">
        <h5 class="card-title">カードタイトル</h5>
        <p class="card-text">
            カードの説明文です。ここに詳細な情報を記載します。
        </p>
        <a href="#" class="btn btn-primary">詳細を見る</a>
    </div>
</div>

ヘッダーとフッター付きカード

<div class="card">
    <div class="card-header">
        特集記事
    </div>
    <div class="card-body">
        <h5 class="card-title">記事タイトル</h5>
        <p class="card-text">記事の内容...</p>
    </div>
    <div class="card-footer text-muted">
        2日前に投稿
    </div>
</div>

4.3 バッジ（Badge）

バッジは、数値やステータスを示す小さなラベルです。 通知の件数や、項目の状態を表示する際に使用します。 

<h1>見出し <span class="badge bg-secondary">新着</span></h1>

<button type="button" class="btn btn-primary">
    通知 <span class="badge bg-danger">5</span>
</button>

<span class="badge bg-success">成功</span>
<span class="badge bg-warning text-dark">警告</span>
<span class="badge bg-info">情報</span>

ピルバッジ（丸みのあるバッジ）

<span class="badge rounded-pill bg-primary">プライマリ</span>
<span class="badge rounded-pill bg-success">成功</span>

4.4 アラート（Alert）

アラートは、ユーザーに重要な情報を伝えるためのメッセージボックスです。 

<div class="alert alert-primary" role="alert">
    これはプライマリアラートです。
</div>

<div class="alert alert-success" role="alert">
    操作が正常に完了しました！
</div>

<div class="alert alert-danger" role="alert">
    エラーが発生しました。再度お試しください。
</div>

<div class="alert alert-warning" role="alert">
    この操作は取り消せません。ご注意ください。
</div>

リンク付きアラート

<div class="alert alert-success" role="alert">
    登録が完了しました。<a href="#" class="alert-link">詳細はこちら</a>
</div>

閉じるボタン付きアラート

<div class="alert alert-warning alert-dismissible fade show" role="alert">
    この通知は閉じることができます。
    <button type="button" class="btn-close" data-bs-dismiss="alert"></button>
</div>

4.5 パンくずリスト（Breadcrumb）

パンくずリストは、現在のページ位置を示すナビゲーションです。 ユーザーがサイト内のどこにいるかを把握しやすくします。 

<nav aria-label="breadcrumb">
    <ol class="breadcrumb">
        <li class="breadcrumb-item"><a href="#">ホーム</a></li>
        <li class="breadcrumb-item"><a href="#">カテゴリ</a></li>
        <li class="breadcrumb-item active" aria-current="page">現在のページ</li>
    </ol>
</nav>
パンくずリストの由来

童話「ヘンゼルとグレーテル」で、主人公が森で迷わないようにパンくずを落としながら進んだことに由来します。 Webサイトでも、訪問者が迷わないための道しるべとして機能します。

4.6 ページネーション（Pagination）

ページネーションは、複数ページにわたるコンテンツをナビゲートするためのコンポーネントです。 

<nav aria-label="ページネーション">
    <ul class="pagination">
        <li class="page-item disabled">
            <a class="page-link" href="#">前へ</a>
        </li>
        <li class="page-item"><a class="page-link" href="#">1</a></li>
        <li class="page-item active"><a class="page-link" href="#">2</a></li>
        <li class="page-item"><a class="page-link" href="#">3</a></li>
        <li class="page-item">
            <a class="page-link" href="#">次へ</a>
        </li>
    </ul>
</nav>

ページネーションのサイズ

<!-- 大きいサイズ -->
<ul class="pagination pagination-lg">...</ul>

<!-- 小さいサイズ -->
<ul class="pagination pagination-sm">...</ul>

理解度確認クイズ
  1. ボタンの用途別の色の使い分けで、削除ボタンに適しているのはどれですか？
    解答例

    btn-danger（赤）が適しています。削除など重要で取り消しが難しい操作に使用します。

  2. カードコンポーネントの基本構造を説明してください。
    解答例
    • .card: カード全体のコンテナ
    • .card-body: カードの本文エリア
    • .card-title: カードのタイトル
    • .card-text: カードの説明文
  3. パンくずリストの役割は何ですか？
    解答例

    ユーザーが現在のページ位置を把握し、上位階層に戻りやすくするためのナビゲーションです。

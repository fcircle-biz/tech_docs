# Bootstrap 学習ガイドライン

このガイドラインでは、Bootstrapの基礎を入門者向けに段階的に学習するためのカリキュラムを提供しています。

## 前提条件
### 必要な環境
- モダンなWebブラウザ（Chrome、Firefox、Safari、Edge等）
- テキストエディタ（Visual Studio Code、Sublime Text等）
- Node.js（オプション: npmを使用する場合）

### 参考リソース
- [Bootstrap公式ドキュメント](https://getbootstrap.com/docs/)
- [Bootstrap日本語ドキュメント](https://getbootstrap.jp/)
- [Bootstrap GitHub](https://github.com/twbs/bootstrap)
- [W3Schools Bootstrap Tutorial](https://www.w3schools.com/bootstrap5/)

### 前提知識
- **必須**: HTML/CSSの基礎知識（タグ、セレクタ、ボックスモデル、レイアウト）
- **推奨**: JavaScriptの基礎知識（DOM操作、イベント処理の基本概念）

## 学習コンテンツ
### [1. Bootstrap入門](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-01.html)
Bootstrapとは何か、なぜ使うのか、フレームワークの歴史と特徴を学びます。CDNとnpmを使った環境構築方法、基本的なテンプレート構成を習得します。

### [2. グリッドシステム基礎](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-02.html)
Bootstrapの中核機能であるグリッドシステムを学びます。12カラムシステムの概念、コンテナとrow/colの使い方、レスポンシブブレークポイント、カラムの入れ子とオフセットを習得します。

### [3. タイポグラフィとユーティリティ](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-03.html)
テキストスタイリングとスペーシングの制御を学びます。見出し、段落、リストのスタイリング、マージン・パディングのユーティリティクラス、テキスト配置と色の指定方法を習得します。

### [4. 基本コンポーネント](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-04.html)
よく使用されるUIコンポーネントを学びます。ボタンのスタイルとサイズ、カードコンポーネント、バッジとアラート、パンくずリストとページネーションの実装方法を習得します。

### [5. フォームとインプット](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-05.html)
美しく使いやすいフォームの作成方法を学びます。フォームコントロールのスタイリング、インプットグループとフィードバック、チェックボックスとラジオボタン、カスタムバリデーションスタイルを習得します。

### [6. ナビゲーション](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-06.html)
サイトナビゲーションの実装方法を学びます。ナビゲーションバー（Navbar）の作成、レスポンシブメニュー、タブとピル、パンくずナビゲーションの実装方法を習得します。

### [7. モーダルとポップオーバー](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-07.html)
インタラクティブなオーバーレイ要素を学びます。モーダルダイアログの作成と制御、ツールチップとポップオーバー、トーストメッセージ、オフキャンバスメニューの実装方法を習得します。

### [8. カルーセルとアコーディオン](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-08.html)
動的なコンテンツ表示コンポーネントを学びます。カルーセル（スライダー）の作成と制御、アコーディオンとコラプス、スクロールスパイ、プログレスバーの実装方法を習得します。

### [9. カスタマイズとSass](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-09.html)
Bootstrapのカスタマイズ方法を学びます。カスタムCSSでの上書き方法、Sass変数を使ったテーマのカスタマイズ、カラーパレットとタイポグラフィの変更、カスタムビルドの作成方法を習得します。

### [10. 実践プロジェクト](https://fcircle-biz.github.io/tech_docs/guide/web-technologies/bootstrap/bootstrap-learning-material-10.html)
実際のWebサイトを構築します。ランディングページの作成、ダッシュボード画面の実装、Eコマースサイトのレイアウト、パフォーマンス最適化とベストプラクティスを習得します。

## 学習の進め方
1. **各章を順番に学習する**: 章は難易度順に構成されているため、1章から順に進めることを推奨します。
2. **実際にコードを書いて確認する**: 読むだけでなく、サンプルコードを自分で書いて動作を確認しましょう。
3. **公式ドキュメントを参照する**: 各機能の詳細は公式ドキュメントで確認する習慣をつけましょう。
4. **実践的なプロジェクトを作る**: 学んだ内容を使って小さなプロジェクトを作成し、理解を深めましょう。

## 推奨学習期間
- **基礎習得コース** (1-5章): 2-3週間
  - グリッドシステム、基本コンポーネント、フォームの作成ができるようになります
- **実践活用コース** (6-10章): 3-4週間
  - 複雑なレイアウト、インタラクティブな要素、カスタマイズまで習得できます

第6章: ナビゲーション

サイトナビゲーションの実装

この章で学ぶこと
  • ナビゲーションバー（Navbar）の基本構造と作成方法を理解する
  • レスポンシブメニューの実装方法を学ぶ
  • タブとピルナビゲーションの使い方を習得する
  • ドロップダウンメニューの実装方法を学ぶ

6.1 ナビゲーションバー（Navbar）

ナビゲーションバーは、Webサイトの主要なナビゲーションを提供する重要なコンポーネントです。

基本的なNavbar

<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">サイト名</a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarNav">
            <ul class="navbar-nav">
                <li class="nav-item">
                    <a class="nav-link active" href="#">ホーム</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">特徴</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">価格</a>
                </li>
            </ul>
        </div>
    </div>
</nav>
Navbarの主要な要素
  • .navbar: ナビゲーションバーのコンテナ
  • .navbar-brand: サイト名やロゴ
  • .navbar-toggler: モバイル用のメニュー開閉ボタン
  • .navbar-collapse: メニュー項目を含む領域
  • .navbar-nav: ナビゲーションリンクのリスト

Navbarの色

<!-- ダークテーマ -->
<nav class="navbar navbar-dark bg-dark">...</nav>
<nav class="navbar navbar-dark bg-primary">...</nav>

<!-- ライトテーマ -->
<nav class="navbar navbar-light bg-light">...</nav>

6.2 レスポンシブNavbar

画面サイズに応じて、メニューを折りたたんだり展開したりする機能を持ったNavbarを作成します。

<nav class="navbar navbar-expand-lg navbar-dark bg-primary">
    <div class="container">
        <a class="navbar-brand" href="#">マイサイト</a>
        
        <!-- ハンバーガーメニューボタン -->
        <button class="navbar-toggler" type="button" 
                data-bs-toggle="collapse" 
                data-bs-target="#navbarContent">
            <span class="navbar-toggler-icon"></span>
        </button>

        <div class="collapse navbar-collapse" id="navbarContent">
            <ul class="navbar-nav ms-auto">
                <li class="nav-item"><a class="nav-link" href="#">ホーム</a></li>
                <li class="nav-item"><a class="nav-link" href="#">サービス</a></li>
                <li class="nav-item"><a class="nav-link" href="#">お問い合わせ</a></li>
            </ul>
        </div>
    </div>
</nav>
レスポンシブNavbarのポイント
  • .navbar-expand-lg: lgサイズ（992px）以上で横並び、それ以下で縦並び
  • .ms-auto: メニューを右寄せ
  • data-bs-toggleとdata-bs-target: ボタンとメニューを連動

6.3 ドロップダウンメニュー

メニュー項目にサブメニューを追加できます。

<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <div class="container">
        <a class="navbar-brand" href="#">サイト名</a>
        <div class="navbar-collapse">
            <ul class="navbar-nav">
                <li class="nav-item">
                    <a class="nav-link" href="#">ホーム</a>
                </li>
                <!-- ドロップダウン -->
                <li class="nav-item dropdown">
                    <a class="nav-link dropdown-toggle" href="#" 
                       data-bs-toggle="dropdown">
                        サービス
                    </a>
                    <ul class="dropdown-menu">
                        <li><a class="dropdown-item" href="#">Web制作</a></li>
                        <li><a class="dropdown-item" href="#">アプリ開発</a></li>
                        <li><hr class="dropdown-divider"></li>
                        <li><a class="dropdown-item" href="#">コンサルティング</a></li>
                    </ul>
                </li>
            </ul>
        </div>
    </div>
</nav>

6.4 タブナビゲーション

コンテンツを切り替えるためのタブを作成できます。

<ul class="nav nav-tabs">
    <li class="nav-item">
        <a class="nav-link active" href="#">アクティブ</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">リンク</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">別のリンク</a>
    </li>
</ul>

ピルナビゲーション

丸みのあるタブスタイルです。

<ul class="nav nav-pills">
    <li class="nav-item">
        <a class="nav-link active" href="#">アクティブ</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">リンク</a>
    </li>
</ul>
実習 6-1: レスポンシブNavbarの作成

ドロップダウンメニューを含むレスポンシブなナビゲーションバーを作成しましょう。

実装例
<nav class="navbar navbar-expand-lg navbar-dark bg-primary">
    <div class="container">
        <a class="navbar-brand" href="#">My Website</a>
        <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#nav">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="nav">
            <ul class="navbar-nav ms-auto">
                <li class="nav-item"><a class="nav-link" href="#">ホーム</a></li>
                <li class="nav-item dropdown">
                    <a class="nav-link dropdown-toggle" href="#" data-bs-toggle="dropdown">製品</a>
                    <ul class="dropdown-menu">
                        <li><a class="dropdown-item" href="#">製品A</a></li>
                        <li><a class="dropdown-item" href="#">製品B</a></li>
                    </ul>
                </li>
                <li class="nav-item"><a class="nav-link" href="#">お問い合わせ</a></li>
            </ul>
        </div>
    </div>
</nav>
理解度確認クイズ
  1. Navbarでメニューを右寄せするクラスは？
    解答例

    .ms-auto（margin-startをautoに設定）

  2. タブとピルの違いは？
    解答例

    タブ（.nav-tabs）は角張った形、ピル（.nav-pills）は丸みのある形です。

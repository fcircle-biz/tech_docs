第7章

ボタンとフォームコンポーネント

様々なスタイルのボタンの作成と、美しく機能的なフォームの実装方法を学びます。HTML/CSS学習ガイド第6章で学んだフォームの知識を活かし、より洗練されたユーザーインターフェースを作成します。

この章で学ぶこと

  • Bootstrapボタンの基本的な使い方と、カラーバリエーションの適用方法
  • アウトラインボタン、サイズ調整、ボタングループなどの応用テクニック
  • Bootstrapフォームコントロールの基本と、美しいフォームレイアウトの作り方
  • フォームバリデーションスタイルの適用と、実践的なフォームの作成

7.1 Bootstrapボタンの基本

Bootstrapでは、.btnクラスを基本として、様々なスタイルのボタンを簡単に作成できます。HTMLのボタン要素に加えて、リンク（<a>）や入力要素（<input>）にもボタンスタイルを適用できます。

基本のボタン

ボタンを作成するには、.btnクラスに加えて、カラークラスを組み合わせます。

button-example.html 
<!-- 基本のボタン -->
<button type="button" class="btn btn-primary">Primary</button>
<button type="button" class="btn btn-secondary">Secondary</button>
<button type="button" class="btn btn-success">Success</button>
<button type="button" class="btn btn-danger">Danger</button>
<button type="button" class="btn btn-warning">Warning</button>
<button type="button" class="btn btn-info">Info</button>

ボタンカラーの意味

  • Primary（主要）: メインアクション（送信、保存など）
  • Secondary（補助）: サブアクション（キャンセルなど）
  • Success（成功）: 成功や完了を示すアクション
  • Danger（危険）: 削除など注意が必要なアクション
  • Warning（警告）: 注意喚起が必要なアクション
  • Info（情報）: 情報表示やヘルプ

リンクやinput要素にボタンスタイルを適用

button-link.html 
<!-- リンクをボタンスタイルに -->
<a href="#" class="btn btn-primary" role="button">リンクボタン</a>

<!-- input要素をボタンスタイルに -->
<input type="button" class="btn btn-primary" value="Input Button">
<input type="submit" class="btn btn-success" value="送信">

7.2 ボタンのバリエーション

アウトラインボタン

.btn-outline-*クラスを使うと、背景が透明で枠線のみのボタンが作成できます。

outline-button.html 
<button type="button" class="btn btn-outline-primary">Primary</button>
<button type="button" class="btn btn-outline-secondary">Secondary</button>
<button type="button" class="btn btn-outline-success">Success</button>
<button type="button" class="btn btn-outline-danger">Danger</button>

ボタンサイズ

.btn-lg（大）、.btn-sm（小）クラスでサイズを変更できます。

button-sizes.html 
<button type="button" class="btn btn-primary btn-lg">大きいボタン</button>
<button type="button" class="btn btn-primary">通常サイズ</button>
<button type="button" class="btn btn-primary btn-sm">小さいボタン</button>

ブロックボタン

親要素の幅いっぱいに広がるボタンを作成するには、.w-100クラス（Bootstrapのユーティリティクラス）を使用します。

block-button.html 
<button type="button" class="btn btn-primary w-100">幅いっぱいのボタン</button>

ボタングループ

複数のボタンをグループ化して並べることができます。.btn-groupクラスを使用します。

button-group.html 
<div class="btn-group" role="group">
  <button type="button" class="btn btn-primary">左</button>
  <button type="button" class="btn btn-primary">中央</button>
  <button type="button" class="btn btn-primary">右</button>
</div>

7.3 Bootstrapフォームの基本

Bootstrapでは、フォーム要素に.form-controlクラスを追加することで、統一されたスタイルを適用できます。ラベルには.form-label、フォーム全体のグループ化には.mb-3（マージン下部）を使用します。

基本のテキスト入力

basic-form.html 
<form>
  <div class="mb-3">
    <label for="name" class="form-label">お名前</label>
    <input type="text" class="form-control" id="name" placeholder="山田 太郎">
  </div>
  <div class="mb-3">
    <label for="email" class="form-label">メールアドレス</label>
    <input type="email" class="form-control" id="email" placeholder="name@example.com">
  </div>
  <div class="mb-3">
    <label for="message" class="form-label">メッセージ</label>
    <textarea class="form-control" id="message" rows="3"></textarea>
  </div>
</form>

チェックボックスとラジオボタン

チェックボックスとラジオボタンには、.form-checkクラスを使用します。

checkbox-radio.html 
<!-- チェックボックス -->
<div class="form-check">
  <input class="form-check-input" type="checkbox" value="" id="check1">
  <label class="form-check-label" for="check1">
    メールマガジンを受け取る
  </label>
</div>

<!-- ラジオボタン -->
<div class="form-check">
  <input class="form-check-input" type="radio" name="plan" id="basic" checked>
  <label class="form-check-label" for="basic">ベーシックプラン</label>
</div>
<div class="form-check">
  <input class="form-check-input" type="radio" name="plan" id="premium">
  <label class="form-check-label" for="premium">プレミアムプラン</label>
</div>

セレクトボックス

select-box.html 
<div class="mb-3">
  <label for="country" class="form-label">国を選択</label>
  <select class="form-select" id="country">
    <option selected>選択してください</option>
    <option value="1">日本</option>
    <option value="2">アメリカ</option>
    <option value="3">イギリス</option>
  </select>
</div>

重要ポイント

セレクトボックスには.form-selectクラスを使用します（.form-controlではありません）。これにより、ドロップダウンアイコンなど、セレクトボックス専用のスタイルが適用されます。

7.4 フォームレイアウト

横並びフォーム（Row形式）

グリッドシステムを組み合わせることで、ラベルと入力欄を横並びにできます。

horizontal-form.html 
<form>
  <div class="row mb-3">
    <label for="inputEmail" class="col-sm-2 col-form-label">メール</label>
    <div class="col-sm-10">
      <input type="email" class="form-control" id="inputEmail">
    </div>
  </div>
  <div class="row mb-3">
    <label for="inputPassword" class="col-sm-2 col-form-label">パスワード</label>
    <div class="col-sm-10">
      <input type="password" class="form-control" id="inputPassword">
    </div>
  </div>
</form>

インラインフォーム

複数のフォーム要素を一行に並べるには、Flexboxのクラスを使います。

inline-form.html 
<form class="row row-cols-lg-auto g-3 align-items-center">
  <div class="col-12">
    <input type="text" class="form-control" placeholder="ユーザー名">
  </div>
  <div class="col-12">
    <input type="password" class="form-control" placeholder="パスワード">
  </div>
  <div class="col-12">
    <button type="submit" class="btn btn-primary">ログイン</button>
  </div>
</form>

フォームグリッド

グリッドシステムを使って、フォーム要素を柔軟にレイアウトできます。

form-grid.html 
<form>
  <div class="row mb-3">
    <div class="col-md-6">
      <label for="firstName" class="form-label">名</label>
      <input type="text" class="form-control" id="firstName">
    </div>
    <div class="col-md-6">
      <label for="lastName" class="form-label">姓</label>
      <input type="text" class="form-control" id="lastName">
    </div>
  </div>
</form>

7.5 フォームのバリデーション

Bootstrapでは、フォームの入力チェック（バリデーション）を視覚的にわかりやすく表示できます。.is-valid（正常）と.is-invalid（エラー）のクラスを使用します。

バリデーションの基本

validation-form.html 
<form>
  <!-- 正常な入力 -->
  <div class="mb-3">
    <label for="validInput" class="form-label">正常な入力</label>
    <input type="text" class="form-control is-valid" id="validInput" value="正しい入力">
    <div class="valid-feedback">
      正しく入力されています！
    </div>
  </div>

  <!-- エラーのある入力 -->
  <div class="mb-3">
    <label for="invalidInput" class="form-label">エラーのある入力</label>
    <input type="text" class="form-control is-invalid" id="invalidInput">
    <div class="invalid-feedback">
      この項目は必須です。
    </div>
  </div>
</form>

JavaScriptとの組み合わせ

実際のWebサイトでは、JavaScriptを使って入力内容をチェックし、動的に.is-validや.is-invalidクラスを追加・削除します。この章ではBootstrapのスタイルを学ぶため、HTMLに直接クラスを記述していますが、実践ではJavaScriptで制御します。

7.6 実践：お問い合わせフォームの作成

これまで学んだボタンとフォームの知識を使って、実用的なお問い合わせフォームを作成してみましょう。

contact-form.html 
<div class="container mt-5">
  <div class="row justify-content-center">
    <div class="col-md-8">
      <h2 class="mb-4">お問い合わせフォーム</h2>
      <form>
        <div class="row mb-3">
          <div class="col-md-6">
            <label for="lastName" class="form-label">姓</label>
            <input type="text" class="form-control" id="lastName" required>
          </div>
          <div class="col-md-6">
            <label for="firstName" class="form-label">名</label>
            <input type="text" class="form-control" id="firstName" required>
          </div>
        </div>

        <div class="mb-3">
          <label for="email" class="form-label">メールアドレス</label>
          <input type="email" class="form-control" id="email" required>
        </div>

        <div class="mb-3">
          <label for="subject" class="form-label">件名</label>
          <select class="form-select" id="subject">
            <option selected>選択してください</option>
            <option value="1">製品について</option>
            <option value="2">サービスについて</option>
            <option value="3">その他</option>
          </select>
        </div>

        <div class="mb-3">
          <label for="message" class="form-label">お問い合わせ内容</label>
          <textarea class="form-control" id="message" rows="5" required></textarea>
        </div>

        <div class="form-check mb-3">
          <input class="form-check-input" type="checkbox" id="agree" required>
          <label class="form-check-label" for="agree">
            プライバシーポリシーに同意する
          </label>
        </div>

        <div class="d-grid gap-2">
          <button type="submit" class="btn btn-primary">送信する</button>
          <button type="reset" class="btn btn-outline-secondary">リセット</button>
        </div>
      </form>
    </div>
  </div>
</div>

コードのポイント

  • .containerでページ中央にコンテンツを配置
  • .rowと.col-md-8でレスポンシブレイアウト
  • 姓名は.col-md-6で横並び（モバイルでは縦）
  • .d-gridでボタンを縦並び配置
  • required属性でHTML5の入力チェック

実習 7-1: ログインフォームを作成しよう

Bootstrapのボタンとフォームコンポーネントを使って、シンプルなログインフォームを作成してみましょう。

要件

  1. 1 メールアドレス入力欄（type="email"）
  2. 2 パスワード入力欄（type="password"）
  3. 3 「ログイン状態を保持する」チェックボックス
  4. 4 「ログイン」ボタン（.btn-primary、幅100%）
  5. 5 中央寄せで最大幅400pxのカード内に配置

第7章のまとめ

  • Bootstrapボタンは.btnクラスを基本とし、カラークラスを組み合わせて多彩なスタイルを実現できる
  • アウトライン、サイズ変更、グループ化など、ボタンのバリエーションが豊富に用意されている
  • フォーム要素には.form-controlや.form-selectを使って統一されたスタイルを適用できる
  • グリッドシステムを活用することで、横並び、インライン、複雑なレイアウトのフォームを簡単に作成できる
  • .is-validと.is-invalidクラスでフォームバリデーションを視覚的に表現できる

理解度確認クイズ

  1. Q1. 赤色の削除ボタンを作成するために必要なクラスの組み合わせは？

    回答を表示

    class="btn btn-danger"
    解説: 削除など注意が必要なアクションには、危険を示す.btn-dangerクラスを使用します。これにより赤色のボタンが作成されます。

  2. Q2. セレクトボックス（ドロップダウンメニュー）に適用するべきBootstrapクラスは？

    回答を表示

    .form-select
    解説: セレクトボックスには.form-selectクラスを使用します。.form-controlはテキスト入力やテキストエリアに使用するクラスです。

  3. Q3. フォームの入力エラーを視覚的に表示するために使用するクラスは？

    回答を表示

    .is-invalid（入力欄に適用）と.invalid-feedback（エラーメッセージに適用）
    解説: 入力欄に.is-invalidを追加すると赤色のボーダーが表示され、.invalid-feedbackでエラーメッセージを赤色で表示できます。

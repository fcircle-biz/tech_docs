第7章: モーダルとポップオーバー

インタラクティブなオーバーレイ要素

この章で学ぶこと
  • モーダルダイアログの基本構造と制御方法を理解する
  • ツールチップとポップオーバーの使い分けを学ぶ
  • トーストメッセージの実装方法を習得する
  • オフキャンバスメニューの作成方法を学ぶ

7.1 モーダル（Modal）

モーダルは、ページの上に重なって表示されるダイアログボックスです。重要な情報を表示したり、ユーザーの確認を求める際に使用します。

基本的なモーダル

<!-- モーダルを開くボタン -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
    モーダルを開く
</button>

<!-- モーダル本体 -->
<div class="modal fade" id="exampleModal" tabindex="-1">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">モーダルタイトル</h5>
                <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body">
                <p>モーダルの本文がここに入ります。</p>
            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">閉じる</button>
                <button type="button" class="btn btn-primary">保存</button>
            </div>
        </div>
    </div>
</div>
モーダルの構造
  • .modal: モーダル全体のコンテナ
  • .modal-dialog: モーダルの配置を制御
  • .modal-content: モーダルの内容
  • .modal-header: ヘッダー（タイトルと閉じるボタン）
  • .modal-body: 本文
  • .modal-footer: フッター（アクションボタン）

モーダルのサイズ

<!-- 小さいモーダル -->
<div class="modal-dialog modal-sm">...</div>

<!-- 大きいモーダル -->
<div class="modal-dialog modal-lg">...</div>

<!-- 特大モーダル -->
<div class="modal-dialog modal-xl">...</div>

7.2 ツールチップ（Tooltip）

ツールチップは、要素にマウスを乗せた際に表示される小さな説明テキストです。

<button type="button" class="btn btn-secondary" 
        data-bs-toggle="tooltip" 
        data-bs-placement="top" 
        data-bs-title="上に表示されるツールチップ">
    上
</button>

<!-- JavaScriptで初期化が必要 -->
<script>
    var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))
    var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
        return new bootstrap.Tooltip(tooltipTriggerEl)
    })
</script>

7.3 ポップオーバー（Popover）

ポップオーバーは、ツールチップよりも多くの情報を表示できるオーバーレイです。

<button type="button" class="btn btn-danger" 
        data-bs-toggle="popover" 
        data-bs-title="ポップオーバータイトル" 
        data-bs-content="ここに詳しい説明文を入れることができます。">
    クリックしてポップオーバーを表示
</button>

<script>
    var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'))
    var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
        return new bootstrap.Popover(popoverTriggerEl)
    })
</script>

7.4 トースト（Toast）

トーストは、画面の端に一時的に表示される通知メッセージです。

<div class="toast" role="alert">
    <div class="toast-header">
        <strong class="me-auto">通知</strong>
        <small>たった今</small>
        <button type="button" class="btn-close" data-bs-dismiss="toast"></button>
    </div>
    <div class="toast-body">
        操作が正常に完了しました。
    </div>
</div>

7.5 オフキャンバス（Offcanvas）

オフキャンバスは、画面の端からスライドして表示されるサイドバーメニューです。

<button class="btn btn-primary" type="button" data-bs-toggle="offcanvas" data-bs-target="#offcanvasExample">
    メニューを開く
</button>

<div class="offcanvas offcanvas-start" tabindex="-1" id="offcanvasExample">
    <div class="offcanvas-header">
        <h5 class="offcanvas-title">メニュー</h5>
        <button type="button" class="btn-close" data-bs-dismiss="offcanvas"></button>
    </div>
    <div class="offcanvas-body">
        <p>メニューの内容がここに表示されます。</p>
    </div>
</div>
実習 7-1: モーダルを使った確認ダイアログ

削除確認用のモーダルを作成してみましょう。

実装例
<button class="btn btn-danger" data-bs-toggle="modal" data-bs-target="#deleteModal">
    削除
</button>

<div class="modal fade" id="deleteModal" tabindex="-1">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header bg-danger text-white">
                <h5 class="modal-title">削除の確認</h5>
                <button type="button" class="btn-close btn-close-white" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body">
                <p>本当に削除してもよろしいですか？この操作は取り消せません。</p>
            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">キャンセル</button>
                <button type="button" class="btn btn-danger">削除する</button>
            </div>
        </div>
    </div>
</div>
理解度確認クイズ
  1. ツールチップとポップオーバーの違いは？
    解答例

    ツールチップは簡単な説明文のみ、ポップオーバーはタイトルと本文を含むより詳しい情報を表示できます。

  2. モーダルを開くボタンに必要な属性は？
    解答例

    data-bs-toggle="modal"とdata-bs-target="#モーダルのID"が必要です。

第8章

カードとリストグループ

カードコンポーネントを使った情報の整理方法と、リストグループによるデータの表示方法を学び、実用的なコンポーネントを作成します。

この章で学ぶこと

  • カードコンポーネントの基本構造とスタイリング方法を理解する
  • ブログ記事、商品カード、プロフィールカードなど実用的なカードを作成する
  • リストグループを使ったデータ表示とインタラクティブな要素を学ぶ
  • カードとリストグループを組み合わせた複雑なレイアウトを実装する

8.1 カードコンポーネントの基本

カード（Card）は、関連する情報をまとめて表示するための最も汎用的なコンポーネントです。画像、テキスト、リンク、ボタンなど、さまざまな要素を柔軟に配置できます。ブログ記事、商品情報、ユーザープロフィールなど、多くの場面で使われます。

カードコンポーネントの特徴

カードは「枠で囲まれた独立した情報のまとまり」です。Webサイトでよく見かける「記事カード」「商品カード」「プロフィールカード」などは、すべてこのカードコンポーネントで作成されています。レスポンシブに対応しており、グリッドシステムと組み合わせることで美しいレイアウトを実現できます。

カードの基本構造

カードは以下の主要な部分で構成されます：

基本的なカードの構造 
<div class="card">
  <!-- カードヘッダー（オプション） -->
  <div class="card-header">特集記事</div>

  <!-- カード画像（オプション） -->
  <img src="image.jpg" class="card-img-top" alt="画像の説明">

  <!-- カードボディ（メインコンテンツ） -->
  <div class="card-body">
    <h5 class="card-title">カードのタイトル</h5>
    <p class="card-text">カードの本文です。</p>
    <a href="#" class="btn btn-primary">詳細を見る</a>
  </div>

  <!-- カードフッター（オプション） -->
  <div class="card-footer">2日前に更新</div>
</div>
graph TD A[.card カード全体のコンテナ] --> B[.card-header ヘッダー部分] A --> C[.card-img-top 上部画像] A --> D[.card-body メインコンテンツ] A --> E[.card-footer フッター部分] D --> F[.card-title タイトル] D --> G[.card-text 本文テキスト] D --> H[.btn ボタンやリンク]

8.2 実用的なカードの作成

例1：ブログ記事カード

画像、タイトル、本文、メタ情報を含む典型的なブログ記事カードを作成します。

ブログ記事カード 
<div class="card">
  <img src="article.jpg" class="card-img-top" alt="記事のサムネイル">
  <div class="card-body">
    <!-- カテゴリバッジ -->
    <span class="badge bg-primary mb-2">テクノロジー</span>

    <!-- タイトル -->
    <h5 class="card-title">Web開発の最新トレンド2025</h5>

    <!-- 本文抜粋 -->
    <p class="card-text">
      2025年のWeb開発トレンドを徹底解説。
      注目すべき技術とツールを紹介します。
    </p>

    <!-- 続きを読むボタン -->
    <a href="#" class="btn btn-outline-primary">続きを読む</a>
  </div>
  <div class="card-footer text-muted">
    <i class="fas fa-calendar"></i> 2025年1月15日
  </div>
</div>

例2：商品カード

ECサイトなどで使われる商品情報を表示するカードです。

商品カード 
<div class="card">
  <img src="product.jpg" class="card-img-top" alt="商品画像">
  <div class="card-body">
    <h5 class="card-title">ワイヤレスヘッドホン</h5>
    <p class="card-text">
      高音質・長時間バッテリー・快適な装着感を実現した
      プレミアムワイヤレスヘッドホン。
    </p>
    <div class="d-flex justify-content-between align-items-center">
      <span class="h4 mb-0 text-primary">¥15,800</span>
      <button class="btn btn-primary">カートに追加</button>
    </div>
  </div>
</div>

重要ポイント

d-flex、justify-content-between、align-items-centerなどのユーティリティクラスを組み合わせることで、カード内の要素を柔軟に配置できます。

8.3 カードのレイアウトとグリッド配置

グリッドシステムと組み合わせることで、複数のカードを美しく配置できます。

3カラムのカードレイアウト

レスポンシブなカードグリッド 
<div class="container">
  <div class="row g-4">
    <!-- カード1 -->
    <div class="col-12 col-md-6 col-lg-4">
      <div class="card h-100">
        <img src="card1.jpg" class="card-img-top" alt="カード1">
        <div class="card-body">
          <h5 class="card-title">カード1</h5>
          <p class="card-text">カードの説明文です。</p>
        </div>
      </div>
    </div>

    <!-- カード2、カード3も同様 -->
  </div>
</div>

カード配置のポイント

  • g-4 - カード間に適切な間隔（gutter）を設定
  • h-100 - カードの高さを揃える（同じ行のカードを同じ高さにする）
  • col-12 col-md-6 col-lg-4 - レスポンシブに1列→2列→3列と変化

8.4 リストグループの基本

リストグループ（List Group）は、関連する項目をリスト形式で表示するコンポーネントです。メニュー、設定項目、タスクリスト、お知らせなど、さまざまな場面で使用できます。

基本的なリストグループ

シンプルなリストグループ 
<ul class="list-group">
  <li class="list-group-item">ホーム</li>
  <li class="list-group-item">プロフィール</li>
  <li class="list-group-item">メッセージ</li>
  <li class="list-group-item">設定</li>
</ul>

アクティブ状態とバッジ付きリスト

アクティブ状態とバッジ 
<ul class="list-group">
  <li class="list-group-item active">受信トレイ
    <span class="badge bg-light text-dark rounded-pill float-end">14</span>
  </li>
  <li class="list-group-item">下書き
    <span class="badge bg-secondary rounded-pill float-end">2</span>
  </li>
  <li class="list-group-item">送信済み</li>
  <li class="list-group-item">ゴミ箱
    <span class="badge bg-warning text-dark rounded-pill float-end">3</span>
  </li>
</ul>

クリック可能なリストグループ

リンク付きリストグループ 
<div class="list-group">
  <a href="#" class="list-group-item list-group-item-action active">
    ダッシュボード
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    ユーザー管理
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    レポート
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    設定
  </a>
</div>

list-group-item-actionクラス

list-group-item-actionクラスを追加すると、ホバー時に背景色が変わり、クリック可能であることがユーザーに伝わります。

8.5 カードとリストグループの組み合わせ

カードとリストグループを組み合わせることで、より複雑で実用的なUIを作成できます。

例：お知らせカード

カード内にリストグループ 
<div class="card">
  <div class="card-header">
    <h5 class="mb-0">最新のお知らせ</h5>
  </div>
  <div class="list-group list-group-flush">
    <a href="#" class="list-group-item list-group-item-action">
      <div class="d-flex w-100 justify-content-between">
        <h6 class="mb-1">システムメンテナンスのお知らせ</h6>
        <small>3日前</small>
      </div>
      <p class="mb-1">1月20日午前2時〜4時にメンテナンスを実施します。</p>
    </a>
    <a href="#" class="list-group-item list-group-item-action">
      <div class="d-flex w-100 justify-content-between">
        <h6 class="mb-1">新機能のリリース</h6>
        <small>1週間前</small>
      </div>
      <p class="mb-1">レポート機能が強化されました。</p>
    </a>
  </div>
</div>

list-group-flushクラス

list-group-flushクラスを使うと、リストグループの境界線とパディングが削除され、カード内に自然に収まるデザインになります。

実習課題：プロフィールカードの作成

以下の要件を満たすプロフィールカードを作成してください。

  1. 1. 上部に円形のプロフィール画像（rounded-circleクラスを使用）
  2. 2. 名前とキャッチフレーズを表示
  3. 3. フォローボタンとメッセージボタンを配置
  4. 4. 下部にスキルや興味をリストグループで表示

第8章のまとめ

  • カードコンポーネントは、画像、テキスト、ボタンなどを柔軟に配置できる汎用的なコンポーネントです
  • カードはcard-header、card-body、card-footerなどの部分で構成されます
  • リストグループは、関連する項目をリスト形式で表示し、クリック可能なメニューとしても使えます
  • カードとグリッドシステムを組み合わせることで、レスポンシブな複数カラムレイアウトを実現できます
  • カードとリストグループを組み合わせることで、より複雑で実用的なUIコンポーネントを作成できます

理解度確認クイズ

  1. Q1. カードのメインコンテンツを含む要素に使用するクラス名は何ですか？

    回答を表示

    card-bodyクラスです。カード内のタイトル、本文、ボタンなど、主要なコンテンツを含む部分に使用します。

  2. Q2. 複数のカードの高さを揃えるために使用するユーティリティクラスは何ですか？

    回答を表示

    h-100クラスです。これをカード要素に追加することで、同じ行にある全てのカードの高さが揃います。

  3. Q3. リストグループの項目をクリック可能にし、ホバー効果を追加するクラスは何ですか？

    回答を表示

    list-group-item-actionクラスです。このクラスを追加すると、ホバー時に背景色が変わり、クリック可能であることが視覚的に示されます。

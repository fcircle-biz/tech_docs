第9章

ナビゲーションバーの作成

レスポンシブなナビゲーションバー（Navbar）の作成方法を学び、モバイルメニューやドロップダウン、固定ヘッダーを実装します。

この章で学ぶこと

  • Bootstrapのnavbarコンポーネントの基本構造を理解する
  • レスポンシブなナビゲーションバーの実装方法を学ぶ
  • モバイルメニューのハンバーガーボタンとトグル機能を実装する
  • ドロップダウンメニューと固定ヘッダーの作成方法を習得する

9.1 ナビゲーションバー（Navbar）とは？

ナビゲーションバー（Navbar）とは、Webサイトの上部に配置される、サイト内の主要なページへのリンクを集めたメニューのことです。多くのWebサイトで見られる、ロゴとメニューが横並びになっている部分がナビゲーションバーです。

Bootstrapのnavbarコンポーネントは、レスポンシブデザインに対応しており、画面サイズに応じて自動的にレイアウトが変わります。PC画面では横並びのメニュー、スマートフォンでは「ハンバーガーメニュー」（≡のアイコン）として表示され、タップすると展開するメニューになります。

用語解説：ハンバーガーメニュー

「ハンバーガーメニュー」とは、3本の横線（≡）で表現されるメニューアイコンのことです。スマートフォンなど画面が小さいデバイスで、メニューを折りたたんで表示するために使われます。アイコンをタップすることでメニューが展開します。

Navbarの主要な要素

ブランド／ロゴ

サイト名やロゴを配置するエリア。通常はトップページへのリンクとして機能します。

ナビゲーションリンク

サイト内の主要なページへのリンク集。ホーム、製品、会社概要などのメニュー項目です。

トグルボタン

モバイル画面でメニューを開閉するためのハンバーガーアイコンボタンです。

追加要素

検索ボックス、ログインボタン、カート、通知アイコンなどを配置できます。

9.2 基本的なナビゲーションバーの作成

まずは最もシンプルなNavbarを作成してみましょう。ロゴとメニューリンクだけのシンプルな構成です。

simple-navbar.html 
<!-- シンプルなNavbar -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <!-- ブランド/ロゴ -->
    <a class="navbar-brand" href="#">MySite</a>

    <!-- ナビゲーションリンク -->
    <div class="navbar-nav">
      <a class="nav-link active" href="#">ホーム</a>
      <a class="nav-link" href="#">製品</a>
      <a class="nav-link" href="#">会社概要</a>
      <a class="nav-link" href="#">お問い合わせ</a>
    </div>
  </div>
</nav>

重要ポイント

navbarクラスは必須の基本クラスです。navbar-expand-lgは「large画面以上で横並びに展開する」という意味で、navbar-light bg-lightは明るい背景色と文字色を指定しています。

主要なクラスの説明

クラス名 説明
navbar ナビゲーションバーの基本クラス（必須）
navbar-expand-{size} 指定サイズ以上で横並び展開（sm/md/lg/xl/xxl）
navbar-light 明るい背景用の文字色（暗い文字）
navbar-dark 暗い背景用の文字色（明るい文字）
navbar-brand ブランド／ロゴ部分のスタイル
nav-link 各メニューリンクのスタイル

9.3 レスポンシブなNavbarの実装

次に、モバイル画面でハンバーガーメニューになるレスポンシブなNavbarを作成します。これがBootstrap Navbarの真骨頂です。

responsive-navbar.html 
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  <div class="container-fluid">
    <!-- ブランド -->
    <a class="navbar-brand" href="#">MySite</a>

    <!-- トグルボタン（モバイル用） -->
    <button class="navbar-toggler" type="button"
            data-bs-toggle="collapse" data-bs-target="#navbarNav">
      <span class="navbar-toggler-icon"></span>
    </button>

    <!-- 折りたたみ可能なメニュー -->
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item">
          <a class="nav-link active" href="#">ホーム</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">製品</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">会社概要</a>
        </li>
      </ul>
    </div>
  </div>
</nav>

仕組みの解説

navbar-togglerはモバイル画面でのみ表示されるハンバーガーボタンです。data-bs-toggle="collapse"とdata-bs-target="#navbarNav"により、ボタンをクリックするとid="navbarNav"の要素が開閉します。JavaScriptを書く必要はありません！

レスポンシブの動作イメージ

flowchart LR A["PC画面
（lg以上）"] --> B["メニューが横並び
トグルボタン非表示"] C["モバイル画面
（lg未満）"] --> D["メニュー折りたたみ
トグルボタン表示"] D --> E["ボタンクリック"] --> F["メニュー展開"]

便利なユーティリティクラス

ms-auto（margin-start: auto）を使うと、メニューを右寄せできます。左寄せの場合はme-auto、中央寄せはmx-autoを使います。

9.4 ドロップダウンメニューの追加

Navbarには、サブメニューを持つドロップダウンメニューを簡単に追加できます。「製品」メニューをクリックすると、製品カテゴリーのサブメニューが表示されるような仕組みです。

navbar-with-dropdown.html 
<ul class="navbar-nav">
  <li class="nav-item">
    <a class="nav-link" href="#">ホーム</a>
  </li>

  <!-- ドロップダウンメニュー -->
  <li class="nav-item dropdown">
    <a class="nav-link dropdown-toggle" href="#"
       data-bs-toggle="dropdown">
      製品
    </a>
    <ul class="dropdown-menu">
      <li><a class="dropdown-item" href="#">スマートフォン</a></li>
      <li><a class="dropdown-item" href="#">タブレット</a></li>
      <li><hr class="dropdown-divider"></li>
      <li><a class="dropdown-item" href="#">アクセサリー</a></li>
    </ul>
  </li>

  <li class="nav-item">
    <a class="nav-link" href="#">会社概要</a>
  </li>
</ul>

ドロップダウンの構造

nav-item dropdownで囲み、dropdown-toggleクラスでクリックできるメニュー項目を作ります。dropdown-menu内にdropdown-itemでサブメニュー項目を配置します。

ドロップダウンの追加要素

要素 説明
dropdown-divider メニュー項目を区切る水平線
dropdown-header サブメニューの見出し
disabled クリックできない無効なメニュー項目

9.5 固定ヘッダーの実装

ページをスクロールしても常に画面上部に表示される「固定ヘッダー」は、ユーザビリティの向上に役立ちます。Bootstrapでは簡単に実装できます。

fixed-navbar.html 
<!-- 上部に固定 -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark fixed-top">
  <!-- Navbar内容 -->
</nav>

<!-- 下部に固定 -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark fixed-bottom">
  <!-- Navbar内容 -->
</nav>

<!-- スクロール時のみ固定（Sticky） -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark sticky-top">
  <!-- Navbar内容 -->
</nav>

重要な注意点

fixed-topを使うと、Navbarがページの上に浮いた状態になり、その下のコンテンツが隠れてしまいます。必ず<body>タグにpadding-topを追加して、コンテンツが隠れないようにしましょう（例: style="padding-top: 70px;"）。

固定位置クラスの比較

クラス 動作 用途
fixed-top 常に画面上部に固定 常に表示したいメニュー
fixed-bottom 常に画面下部に固定 フッターメニュー、アクション
sticky-top スクロールすると上部に固定 省スペースで固定表示

9.6 Navbarのカラーバリエーション

Bootstrapの標準カラーを使って、Navbarの見た目を簡単に変更できます。背景色に合わせて文字色も自動調整されます。

navbar-colors.html 
<!-- プライマリー色（青） -->
<nav class="navbar navbar-dark bg-primary">...</nav>

<!-- 成功色（緑） -->
<nav class="navbar navbar-dark bg-success">...</nav>

<!-- 警告色（黄） -->
<nav class="navbar navbar-light bg-warning">...</nav>

<!-- 白背景 -->
<nav class="navbar navbar-light bg-white shadow-sm">...</nav>

暗い背景の場合

navbar-darkを使用します。文字が白色になります。

例: bg-dark, bg-primary, bg-success

明るい背景の場合

navbar-lightを使用します。文字が暗色になります。

例: bg-light, bg-white, bg-warning

実習: プロフェッショナルなNavbarを作成しよう

以下の要件を満たすNavbarを作成してみましょう。

  1. 1. 暗い背景色（bg-dark）で固定ヘッダー（fixed-top）を使用
  2. 2. ブランド名「TechStore」を左側に配置
  3. 3. メニュー項目: ホーム、製品（ドロップダウン）、サポート、ログイン
  4. 4. 製品ドロップダウンには「ノートPC」「デスクトップ」「アクセサリー」を含める
  5. 5. lg画面以上で横並び、それ以下でハンバーガーメニュー
実装例を表示
practice-navbar.html 
<nav class="navbar navbar-expand-lg navbar-dark bg-dark fixed-top">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">TechStore</a>
    <button class="navbar-toggler" type="button"
            data-bs-toggle="collapse" data-bs-target="#mainNav">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="mainNav">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item">
          <a class="nav-link active" href="#">ホーム</a>
        </li>
        <li class="nav-item dropdown">
          <a class="nav-link dropdown-toggle" href="#"
             data-bs-toggle="dropdown">製品</a>
          <ul class="dropdown-menu">
            <li><a class="dropdown-item" href="#">ノートPC</a></li>
            <li><a class="dropdown-item" href="#">デスクトップ</a></li>
            <li><hr class="dropdown-divider"></li>
            <li><a class="dropdown-item" href="#">アクセサリー</a></li>
          </ul>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">サポート</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">ログイン</a>
        </li>
      </ul>
    </div>
  </div>
</nav>

<!-- コンテンツがNavbarに隠れないようにパディング追加 -->
<div style="padding-top: 70px;">
  <!-- メインコンテンツ -->
</div>

動作確認のポイント

  • PC画面でメニューが横並びに表示されるか
  • モバイル画面でハンバーガーアイコンが表示されるか
  • ハンバーガーアイコンをクリックするとメニューが展開されるか
  • ドロップダウンメニューが正しく表示されるか
  • 固定ヘッダーの場合、スクロールしても上部に表示されたままか

よくあるトラブルと解決方法

ハンバーガーボタンをクリックしてもメニューが開かない

原因：Bootstrap JavaScriptが読み込まれていない
解決：<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js">を</body>の直前に追加

固定ヘッダーの下にコンテンツが隠れてしまう

原因：コンテンツが上部に詰まっている
解決：<body>またはメインコンテンツにpadding-topを追加（例: 70px）

ドロップダウンが表示されない

原因：data-bs-toggle="dropdown"が抜けている
解決：ドロップダウンのトリガー要素に必ずdata-bs-toggle="dropdown"を追加

第9章のまとめ

  • BootstrapのNavbarコンポーネントを使うと、レスポンシブなナビゲーションバーを簡単に実装できます
  • navbar-togglerとcollapseクラスを組み合わせることで、モバイルメニューを実現します
  • ドロップダウンメニューはdropdownクラスを使い、JavaScriptを書かずに実装できます
  • fixed-topやsticky-topで固定ヘッダーを実装できます
  • 背景色に合わせてnavbar-lightまたはnavbar-darkを使い分けます

理解度確認クイズ

  1. Q1. モバイル画面でメニューを折りたたむために必要なクラスを2つ挙げてください。

    回答を表示

    navbar-toggler（トグルボタン用）とcollapse navbar-collapse（折りたたみ可能なメニュー用）の2つです。これらを組み合わせることで、モバイルでのハンバーガーメニューを実現できます。

  2. Q2. 暗い背景のNavbarで文字を見やすくするために使うクラスは何ですか？

    回答を表示

    navbar-darkクラスを使います。これにより、暗い背景に対して明るい文字色（白色）が適用され、視認性が向上します。逆に明るい背景の場合はnavbar-lightを使います。

  3. Q3. 常に画面上部に固定されるヘッダーを作るためのクラスは何ですか？また、その際に注意すべきことは何ですか？

    回答を表示

    fixed-topクラスを使います。注意点として、固定ヘッダーの下にコンテンツが隠れてしまうため、bodyまたはメインコンテンツに適切なpadding-topを追加する必要があります（例: 70px程度）。

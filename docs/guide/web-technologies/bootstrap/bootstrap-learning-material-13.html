第13章

アイコンと装飾要素

Bootstrap Iconsの使い方と、バッジ、プログレスバー、スピナーなどの装飾要素を学び、UIをより直感的で使いやすくします。

この章で学ぶこと

  • Bootstrap Iconsの導入方法と基本的な使い方を習得する
  • バッジを使った通知やステータス表示の方法を学ぶ
  • プログレスバーで進捗状況を視覚化する技術を理解する
  • スピナーを使った読み込み中の表現方法を身につける

13.1 Bootstrap Iconsとは？

Bootstrap Iconsは、Bootstrapの公式アイコンライブラリで、1,800種類以上の高品質なアイコンを無料で使用できます。Webサイトのボタンやナビゲーション、情報表示など、さまざまな場面でアイコンを活用することで、視覚的に分かりやすいUIを実現できます。

アイコンの役割

アイコンは「視覚的な記号」として、言葉だけでは伝わりにくい情報を瞬時に理解させる役割を果たします。例えば、ゴミ箱アイコンは「削除」、鉛筆アイコンは「編集」を表すなど、直感的な操作を可能にします。

Bootstrap Iconsの導入方法

Bootstrap IconsをCDN経由で導入するのが最も簡単です。HTMLの<head>セクションに以下のリンクを追加します。

index.html 
<!-- Bootstrap Icons CDN -->
<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css">

アイコンの基本的な使い方

Bootstrap Iconsは、<i>タグにbiクラスとアイコン名を指定して使用します。

アイコンの使用例 
<!-- ハートアイコン -->
<i class="bi bi-heart"></i>

<!-- ハートアイコン（塗りつぶし） -->
<i class="bi bi-heart-fill"></i>

<!-- スターアイコン -->
<i class="bi bi-star"></i>

<!-- スターアイコン（塗りつぶし） -->
<i class="bi bi-star-fill"></i>

プレビュー

アイコンのサイズ変更

アイコンのサイズは、font-sizeを変更することで調整できます。Bootstrapのユーティリティクラスを使うと便利です。

サイズ調整の例 
<!-- 通常サイズ -->
<i class="bi bi-heart-fill"></i>

<!-- 大きいサイズ（fs-1からfs-6まで利用可能） -->
<i class="bi bi-heart-fill fs-1"></i>
<i class="bi bi-heart-fill fs-3"></i>
<i class="bi bi-heart-fill fs-5"></i>

13.2 バッジ（Badges）

バッジは、通知やステータス、カウント表示など、小さな情報を目立たせるための装飾要素です。メールアプリの未読数表示や、オンラインショップの「NEW」ラベルなどに使用されます。

基本的なバッジ

バッジの基本 
<!-- 基本的なバッジ -->
<span class="badge bg-primary">Primary</span>
<span class="badge bg-success">Success</span>
<span class="badge bg-danger">Danger</span>
<span class="badge bg-warning text-dark">Warning</span>
<span class="badge bg-info text-dark">Info</span>

プレビュー

Primary Success Danger Warning Info

ボタンにバッジを追加

ボタンにバッジを組み合わせることで、通知数などを表示できます。

ボタンとバッジの組み合わせ 
<!-- 通知付きボタン -->
<button class="btn btn-primary">
  通知 <span class="badge bg-danger">9+</span>
</button>

<!-- メッセージ付きボタン -->
<button class="btn btn-success">
  メッセージ <span class="badge bg-light text-dark">4</span>
</button>

プレビュー

ピルバッジ（丸いバッジ）

rounded-pillクラスを追加することで、より丸みのあるバッジが作成できます。

ピルバッジ 
<span class="badge rounded-pill bg-primary">Pill Badge</span>
<span class="badge rounded-pill bg-success">New</span>
<span class="badge rounded-pill bg-danger">Sale</span>

プレビュー

Pill Badge New Sale

13.3 プログレスバー（Progress Bars）

プログレスバーは、タスクの進捗状況や完了度を視覚的に表示するための要素です。ファイルのアップロード進捗、学習の進行状況、プロジェクトの完了率などを表現できます。

基本的なプログレスバー

プログレスバーの基本 
<!-- 25%進捗のプログレスバー -->
<div class="progress">
  <div class="progress-bar" style="width: 25%"></div>
</div>

<!-- 50%進捗のプログレスバー -->
<div class="progress">
  <div class="progress-bar" style="width: 50%"></div>
</div>

<!-- 完了（100%） -->
<div class="progress">
  <div class="progress-bar" style="width: 100%"></div>
</div>

プレビュー

25%完了
50%完了
100%完了

ラベル付きプログレスバー

プログレスバー内にパーセンテージを表示することで、より詳細な進捗状況を伝えられます。

ラベル付きプログレスバー 
<div class="progress">
  <div class="progress-bar" style="width: 65%">65%</div>
</div>

プレビュー

65%

色付きプログレスバー

プログレスバーの色を変更することで、状態を視覚的に表現できます。

色付きプログレスバー 
<!-- 成功（緑） -->
<div class="progress">
  <div class="progress-bar bg-success" style="width: 80%"></div>
</div>

<!-- 警告（黄） -->
<div class="progress">
  <div class="progress-bar bg-warning" style="width: 60%"></div>
</div>

<!-- 危険（赤） -->
<div class="progress">
  <div class="progress-bar bg-danger" style="width: 30%"></div>
</div>

プレビュー

成功（80%）
警告（60%）
危険（30%）

ストライプ模様のプログレスバー

progress-bar-stripedクラスを追加すると、ストライプ模様が表示されます。さらにprogress-bar-animatedを追加すると、アニメーション効果が加わります。

ストライプ＆アニメーション 
<!-- ストライプ模様 -->
<div class="progress">
  <div class="progress-bar progress-bar-striped"
       style="width: 75%"></div>
</div>

<!-- ストライプ＋アニメーション -->
<div class="progress">
  <div class="progress-bar progress-bar-striped progress-bar-animated"
       style="width: 75%"></div>
</div>

実用例

アニメーション付きプログレスバーは、ファイルアップロード中やデータ処理中など、「現在進行中」であることを示すのに効果的です。

13.4 スピナー（Spinners）

スピナーは、ローディング中やデータ取得中など、「処理が進行中である」ことをユーザーに伝えるためのアニメーション要素です。ユーザーが待機していることを明確に示すことで、より良いUX（ユーザー体験）を提供できます。

ボーダースピナー（Border Spinner）

ボーダースピナー 
<!-- 基本的なスピナー -->
<div class="spinner-border" role="status">
  <span class="visually-hidden">読み込み中...</span>
</div>

<!-- 色付きスピナー -->
<div class="spinner-border text-primary"></div>
<div class="spinner-border text-success"></div>
<div class="spinner-border text-danger"></div>

プレビュー

成長スピナー（Growing Spinner）

成長するアニメーションのスピナーも利用できます。

成長スピナー 
<div class="spinner-grow text-primary"></div>
<div class="spinner-grow text-success"></div>
<div class="spinner-grow text-danger"></div>

プレビュー（成長スピナー風）

ボタン内のスピナー

スピナーをボタンに組み込むことで、送信処理中や保存中などの状態を表現できます。

ボタン内スピナー 
<!-- 送信処理中のボタン -->
<button class="btn btn-primary" disabled>
  <span class="spinner-border spinner-border-sm"
        role="status"></span>
  送信中...
</button>

<!-- 保存処理中のボタン -->
<button class="btn btn-success" disabled>
  <span class="spinner-grow spinner-grow-sm"></span>
  保存中...
</button>

プレビュー

アクセシビリティへの配慮

スピナーにはvisually-hiddenクラスで「読み込み中」などのテキストを含めることで、スクリーンリーダーを使用するユーザーにも状態が伝わります。

実習 13-1: 装飾要素を使ったプロフィールカードの作成

アイコン、バッジ、プログレスバーを組み合わせて、ユーザープロフィールカードを作成してみましょう。

手順

  1. 1 新しいHTMLファイルを作成し、Bootstrap 5とBootstrap Iconsを読み込みます
  2. 2 カードコンポーネントを作成し、ユーザーアイコン（bi-person-circle）と名前を表示します
  3. 3 「初級者」「中級者」などのバッジを追加します
  4. 4 スキル習得度をプログレスバーで表示します（HTML: 90%, CSS: 75%, JavaScript: 60%など）
  5. 5 ブラウザで表示を確認し、バッジやプログレスバーが正しく表示されているか確認します

第13章のまとめ

  • Bootstrap Iconsは1,800種類以上のアイコンを提供し、CDN経由で簡単に導入できます
  • バッジは通知数やステータス表示に使用し、色や形状を変えることで様々な情報を表現できます
  • プログレスバーはタスクの進捗状況を視覚的に表現し、色やアニメーションで状態を明確に伝えられます
  • スピナーはローディング中を示すアニメーションで、ユーザーに待機状態を明確に伝えます
  • これらの装飾要素を適切に組み合わせることで、直感的で使いやすいUIを実現できます

理解度確認クイズ

  1. Q1. Bootstrap Iconsを使用する際、HTMLの<head>セクションに追加すべきCDNリンクの役割は何ですか？

    回答を表示

    Bootstrap IconsのCDNリンクは、1,800種類以上のアイコンのスタイル定義（CSSファイル）を読み込む役割を果たします。これにより、アイコンのクラス名を指定するだけで、簡単にアイコンを表示できるようになります。

  2. Q2. バッジを使って通知数を表示する際、ピルバッジ（丸いバッジ）を作成するために追加するクラス名は何ですか？

    回答を表示

    rounded-pill クラスを追加します。これにより、バッジの角が完全に丸くなり、ピル（カプセル）のような形状になります。
    例：<span class="badge rounded-pill bg-primary">5</span>

  3. Q3. プログレスバーとスピナーの使い分けについて説明してください。

    回答を表示

    プログレスバー： 進捗状況が明確で、完了までの割合が分かる場合に使用します。例：ファイルアップロード（50%完了）、学習進捗（80%完了）など。

    スピナー： 進捗状況が不明確で、処理が進行中であることだけを示したい場合に使用します。例：データ読み込み中、サーバーとの通信中など。

10.1 モーダル（Modal）とは？

モーダルとは、Webページの上に重なって表示されるポップアップダイアログのことです。ユーザーに重要な情報を伝えたり、確認を求めたり、フォーム入力を促したりする際に使われます。モーダルが開いている間は、背景のページ操作ができなくなるため、ユーザーの注意を集中させることができます。

用語解説：モーダル

「モーダル」という名前は、「モード」（状態）を変更するという意味に由来します。モーダルが開くと、通常のページ操作モードから、モーダル内での操作モードに切り替わります。閉じるか操作を完了するまで、背景ページには戻れません。

モーダルの主な用途

ログイン・登録フォーム

ページ遷移なしでユーザー認証を行う

確認ダイアログ

削除などの重要な操作の確認を求める

画像・動画の拡大表示

サムネイル画像をクリックして拡大

詳細情報の表示

利用規約やプライバシーポリシーなど

10.2 基本的なモーダルの実装

Bootstrapのモーダルは、JavaScriptを書かずにHTML属性だけで実装できます。基本的な構造を見ていきましょう。

モーダルの基本構造

modal-basic.html 
<!-- モーダルを開くボタン -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">
  モーダルを開く
</button>

<!-- モーダル本体 -->
<div class="modal fade" id="myModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <!-- ヘッダー -->
      <div class="modal-header">
        <h5 class="modal-title">モーダルのタイトル</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <!-- 本文 -->
      <div class="modal-body">
        <p>ここにモーダルの内容が入ります。</p>
      </div>
      <!-- フッター -->
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">閉じる</button>
        <button type="button" class="btn btn-primary">保存</button>
      </div>
    </div>
  </div>
</div>

重要な属性の説明

  • data-bs-toggle="modal" - ボタンをクリックするとモーダルが開くことを指定
  • data-bs-target="#myModal" - 開くモーダルのIDを指定
  • data-bs-dismiss="modal" - モーダルを閉じるボタンを指定
  • class="modal fade" - フェードイン・アウトのアニメーション効果

10.3 モーダルのサイズバリエーション

Bootstrapのモーダルは、内容に応じて4つのサイズから選択できます。modal-dialogにクラスを追加することでサイズを変更できます。

サイズの種類

サイズ クラス名 最大幅 用途
小 modal-sm 300px 簡単な確認メッセージ
標準 クラス指定なし 500px 一般的な情報表示
大 modal-lg 800px フォームや詳細情報
特大 modal-xl 1140px 画像ギャラリーや複雑な内容

サイズ指定の例

modal-sizes.html 
<!-- 小サイズのモーダル -->
<div class="modal fade" id="smallModal">
  <div class="modal-dialog modal-sm">
    <!-- ... -->
  </div>
</div>

<!-- 大サイズのモーダル -->
<div class="modal fade" id="largeModal">
  <div class="modal-dialog modal-lg">
    <!-- ... -->
  </div>
</div>

10.4 アラートメッセージの基本

アラートは、ユーザーに重要な情報を伝えるための通知メッセージです。成功、情報、警告、エラーなど、状況に応じた色分けで表示できます。

アラートの種類

Primary - 主要な情報メッセージ（青色）
Success - 成功メッセージ（緑色）- 保存完了、送信成功など
Info - お知らせメッセージ（水色）- ヒントや追加情報
Warning - 警告メッセージ（黄色）- 注意が必要な情報
Danger - エラーメッセージ（赤色）- 失敗や重大なエラー

基本的なアラートの記述

alert-basic.html 
<!-- 成功メッセージ -->
<div class="alert alert-success" role="alert">
  データが正常に保存されました。
</div>

<!-- エラーメッセージ -->
<div class="alert alert-danger" role="alert">
  エラーが発生しました。もう一度お試しください。
</div>

10.5 閉じるボタン付きアラート

アラートに閉じるボタンを追加すると、ユーザーが不要になったメッセージを自分で消すことができます。

閉じるボタン付きアラートの実装

alert-dismissible.html 
<div class="alert alert-warning alert-dismissible fade show" role="alert">
  <strong>注意！</strong> この操作は取り消せません。
  <button type="button" class="btn-close" data-bs-dismiss="alert"></button>
</div>

重要なクラスとポイント

  • alert-dismissible - 閉じるボタンを表示するために必要
  • fade show - フェードアウトアニメーションを有効化
  • data-bs-dismiss="alert" - アラートを閉じる機能を追加

10.6 アラート内のリンクとアイコン

アラートにリンクやアイコンを追加して、より情報量の多いメッセージを作成できます。

リンク付きアラート

alert-with-link.html

  新しいアップデートがあります。
  <a href="#" class="alert-link">詳細を確認</a>
</div>

理解度確認クイズ

  1. Q1. モーダルを開くボタンに必要な属性は何ですか？

    回答を表示

    data-bs-toggle="modal" と data-bs-target="#モーダルのID" の2つの属性が必要です。これらの属性により、JavaScriptを書かずにモーダルを開くことができます。

  2. Q2. 大サイズのモーダルを作成するには、どのクラスを使いますか？

    回答を表示

    modal-lg クラスを modal-dialog に追加します。例: <div class="modal-dialog modal-lg">

  3. Q3. 閉じるボタン付きアラートを作成するために必要なクラスを2つ挙げてください。

    回答を表示

    必要なクラス：
    1. alert-dismissible - 閉じるボタンを表示
    2. fade show - フェードアウトアニメーションを有効化
    さらに、閉じるボタンには data-bs-dismiss="alert" 属性が必要です。

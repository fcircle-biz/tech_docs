カルーセルと画像ギャラリー

画像スライダー（カルーセル）の作成と、レスポンシブな画像ギャラリーの実装方法を学び、視覚的に魅力的なコンテンツ表示を実現します。

この章で学ぶこと

  • Bootstrapのカルーセルコンポーネントの基本構造と使い方を理解する
  • カルーセルのカスタマイズオプション（インジケーター、コントロール、自動再生など）を学ぶ
  • レスポンシブな画像ギャラリーの実装方法を習得する
  • 画像の最適化とアクセシビリティを考慮した実装方法を学ぶ

11.1 カルーセル（スライダー）とは？

カルーセル（Carousel）は、複数の画像やコンテンツをスライドショー形式で表示するUI部品です。「スライダー」とも呼ばれ、限られたスペースで多くの情報を効果的に表示できるため、Webサイトのトップページや商品紹介などで広く使われています。

スペースの有効活用

限られた画面領域で複数のコンテンツを順番に表示できます。

視覚的な魅力

アニメーション効果で訪問者の注目を集められます。

モバイル対応

スワイプ操作で直感的に操作できます。

自動再生

自動でスライドを切り替えることができます。

用語解説：カルーセル

「カルーセル（Carousel）」は元々「メリーゴーラウンド（回転木馬）」を意味する言葉です。コンテンツが順番に回転して表示される様子から、このUI部品をカルーセルと呼ぶようになりました。

11.2 基本的なカルーセルの実装

Bootstrapのカルーセルは、.carouselクラスを持つコンテナ内に、スライド画像とコントロール要素を配置することで作成できます。

実装例：シンプルなカルーセル

carousel.html 
<!-- カルーセル -->
<div id="carouselExample" class="carousel slide" data-bs-ride="carousel">
  <!-- スライド内容 -->
  <div class="carousel-inner">
    <div class="carousel-item active">
      <img src="image1.jpg" class="d-block w-100" alt="スライド1">
    </div>
    <div class="carousel-item">
      <img src="image2.jpg" class="d-block w-100" alt="スライド2">
    </div>
    <div class="carousel-item">
      <img src="image3.jpg" class="d-block w-100" alt="スライド3">
    </div>
  </div>

  <!-- 前へボタン -->
  <button class="carousel-control-prev" type="button"
          data-bs-target="#carouselExample" data-bs-slide="prev">
    <span class="carousel-control-prev-icon"></span>
  </button>

  <!-- 次へボタン -->
  <button class="carousel-control-next" type="button"
          data-bs-target="#carouselExample" data-bs-slide="next">
    <span class="carousel-control-next-icon"></span>
  </button>
</div>

重要ポイント

  • 最初のスライドには.activeクラスが必要
  • data-bs-ride="carousel"で自動再生を有効化
  • 各スライドには.carousel-itemクラスを指定
  • 画像には.d-block .w-100で幅100%表示

11.3 インジケーター付きカルーセル

インジケーターは、現在何番目のスライドを表示しているかを示すドット（点）です。ユーザーがクリックすることで、直接そのスライドにジャンプできます。

carousel-indicators.html 
<div id="carouselIndicators" class="carousel slide" data-bs-ride="carousel">
  <!-- インジケーター（スライドの位置を示すドット） -->
  <div class="carousel-indicators">
    <button type="button" data-bs-target="#carouselIndicators"
            data-bs-slide-to="0" class="active"></button>
    <button type="button" data-bs-target="#carouselIndicators"
            data-bs-slide-to="1"></button>
    <button type="button" data-bs-target="#carouselIndicators"
            data-bs-slide-to="2"></button>
  </div>

  <div class="carousel-inner">
    <div class="carousel-item active">
      <img src="slide1.jpg" class="d-block w-100" alt="スライド1">
    </div>
    <div class="carousel-item">
      <img src="slide2.jpg" class="d-block w-100" alt="スライド2">
    </div>
    <div class="carousel-item">
      <img src="slide3.jpg" class="d-block w-100" alt="スライド3">
    </div>
  </div>

  <button class="carousel-control-prev" type="button"
          data-bs-target="#carouselIndicators" data-bs-slide="prev">
    <span class="carousel-control-prev-icon"></span>
  </button>
  <button class="carousel-control-next" type="button"
          data-bs-target="#carouselIndicators" data-bs-slide="next">
    <span class="carousel-control-next-icon"></span>
  </button>
</div>

コードのポイント

  • data-bs-slide-toでスライド番号を指定（0から始まる）
  • 最初のインジケーターには.activeクラスが必要
  • インジケーターの数はスライドの数と一致させる

11.4 キャプション付きカルーセル

各スライドにテキストの説明（キャプション）を追加することができます。.carousel-captionクラスを使います。

carousel-caption.html 
<div class="carousel-item active">
  <img src="product1.jpg" class="d-block w-100" alt="商品1">

  <!-- キャプション（画像の下部に表示されるテキスト） -->
  <div class="carousel-caption d-none d-md-block">
    <h5>最新モデル発売</h5>
    <p>革新的なデザインと高性能を両立した新製品をご紹介します。</p>
  </div>
</div>

<div class="carousel-item">
  <img src="product2.jpg" class="d-block w-100" alt="商品2">
  <div class="carousel-caption d-none d-md-block">
    <h5>期間限定セール</h5>
    <p>人気商品が今だけ特別価格でお求めいただけます。</p>
  </div>
</div>

デザインのコツ

  • .d-none .d-md-blockでスマホでは非表示、タブレット以上で表示
  • キャプションのテキストは簡潔に（1～2行程度）
  • 背景が暗い画像では白文字が見やすい

11.5 カルーセルのカスタマイズオプション

Bootstrapのカルーセルは、data属性を使って様々なカスタマイズができます。

主なカスタマイズオプション

属性 説明 値
data-bs-ride 自動再生の有効化 carousel
data-bs-interval スライド切り替え間隔（ミリ秒） 5000（5秒）
data-bs-pause ホバー時に一時停止するか hover / false
data-bs-wrap 最後のスライド後に最初に戻るか true / false

実装例：カスタマイズしたカルーセル

carousel-custom.html 
<!-- 10秒間隔で自動再生、ホバー時一時停止なし、ループあり -->
<div id="carouselCustom" class="carousel slide carousel-fade"
     data-bs-ride="carousel"
     data-bs-interval="10000"
     data-bs-pause="false"
     data-bs-wrap="true">
  <!-- スライド内容 -->
  <div class="carousel-inner">
    <!-- スライド省略 -->
  </div>
</div>

フェードエフェクト

.carousel-fadeクラスを追加すると、スライド効果ではなくフェードイン/フェードアウト効果になります。

11.6 レスポンシブ画像ギャラリー

Bootstrapのグリッドシステムを使って、画面サイズに応じて自動調整される画像ギャラリーを作成できます。

実装例：レスポンシブギャラリー

gallery.html 
<!-- 画像ギャラリー（スマホ: 1列、タブレット: 2列、PC: 3列） -->
<div class="container">
  <div class="row g-3">
    <!-- 画像1 -->
    <div class="col-12 col-md-6 col-lg-4">
      <img src="gallery1.jpg" class="img-fluid rounded" alt="ギャラリー画像1">
    </div>

    <!-- 画像2 -->
    <div class="col-12 col-md-6 col-lg-4">
      <img src="gallery2.jpg" class="img-fluid rounded" alt="ギャラリー画像2">
    </div>

    <!-- 画像3 -->
    <div class="col-12 col-md-6 col-lg-4">
      <img src="gallery3.jpg" class="img-fluid rounded" alt="ギャラリー画像3">
    </div>

    <!-- 画像4～6も同様 -->
  </div>
</div>

ホバーエフェクト付きギャラリー

gallery-hover.html 
<style>
.gallery-item {
  position: relative;
  overflow: hidden;
  border-radius: 8px;
}

.gallery-item img {
  transition: transform 0.3s ease;
}

.gallery-item:hover img {
  transform: scale(1.1);
}
</style>

<div class="col-12 col-md-6 col-lg-4">
  <div class="gallery-item">
    <img src="photo.jpg" class="img-fluid" alt="写真">
  </div>
</div>

ギャラリーのポイント

  • .img-fluidで画像を親要素の幅に合わせて自動調整
  • .g-3で画像間に適切な余白を設定
  • ホバーエフェクトで画像を少し拡大すると視覚的に魅力的

実習: カルーセルとギャラリーの作成

課題1: インジケーター付きカルーセル

  1. 3枚のスライドを持つカルーセルを作成
  2. インジケーター（ドット）を追加
  3. 自動再生間隔を7秒に設定
  4. フェードエフェクトを適用

課題2: レスポンシブギャラリー

  1. 6枚の画像を配置したギャラリーを作成
  2. スマホで1列、タブレットで2列、PCで3列表示
  3. 画像に.roundedで角丸を適用
  4. ホバー時に画像を拡大する効果を追加

11.7 画像の最適化とアクセシビリティ

カルーセルやギャラリーを実装する際は、画像の最適化とアクセシビリティに配慮することが重要です。

画像最適化のベストプラクティス

1. 画像サイズの最適化

Web用に適切なサイズに圧縮しましょう。大きすぎる画像はページ読み込みを遅くします。目安としてJPEG画像は70～85%の品質で圧縮。

2. 適切なフォーマットの選択

写真はJPEG、アイコンやイラストはPNG、より高品質が必要な場合はWebP形式を使用しましょう。

3. alt属性の指定

全ての画像に意味のあるalt属性を指定して、スクリーンリーダーユーザーにも内容を伝えましょう。

アクセシビリティ対応の実装例

accessible-carousel.html 
<div id="carouselA11y" class="carousel slide"
     data-bs-ride="carousel" aria-label="商品スライドショー">
  <div class="carousel-inner">
    <div class="carousel-item active">
      <!-- alt属性で画像の内容を説明 -->
      <img src="product1.jpg" class="d-block w-100"
           alt="新製品ノートパソコン、シルバー色、開いた状態">
    </div>
  </div>

  <!-- スクリーンリーダー用のラベル -->
  <button class="carousel-control-prev" type="button"
          data-bs-target="#carouselA11y" data-bs-slide="prev">
    <span class="carousel-control-prev-icon" aria-hidden="true"></span>
    <span class="visually-hidden">前へ</span>
  </button>
  <button class="carousel-control-next" type="button"
          data-bs-target="#carouselA11y" data-bs-slide="next">
    <span class="carousel-control-next-icon" aria-hidden="true"></span>
    <span class="visually-hidden">次へ</span>
  </button>
</div>

カルーセル使用時の注意点

  • 自動再生は一部のユーザーにとって読みづらい場合があります
  • 重要な情報はカルーセルだけに頼らず、別途表示することを検討
  • スライド切り替え速度は適切に設定（速すぎない）

第11章のまとめ

  • カルーセルは複数の画像やコンテンツをスライドショー形式で表示するUI部品で、限られたスペースで多くの情報を効果的に表示できます
  • Bootstrapの.carouselクラスを使うことで、簡単にカルーセルを実装できます
  • インジケーターやキャプション、カスタムオプションを使ってカルーセルを豊かにできます
  • グリッドシステムを活用することで、レスポンシブな画像ギャラリーを簡単に作成できます
  • 画像の最適化とアクセシビリティ（alt属性、スクリーンリーダー対応）を考慮することが重要です

理解度確認クイズ

  1. Q1. カルーセルで最初に表示するスライドには、どのクラスを追加する必要がありますか？

    回答を表示

    .activeクラスを追加します。.carousel-item.activeのようにマークアップすることで、そのスライドが最初に表示されます。activeクラスがないとカルーセルが正しく動作しません。

  2. Q2. カルーセルのスライド切り替え間隔を8秒に設定するには、どの属性を使いますか？

    回答を表示

    data-bs-interval="8000"を使います。単位はミリ秒なので、8秒 = 8000ミリ秒と指定します。例：<div class="carousel slide" data-bs-interval="8000">

  3. Q3. 画像ギャラリーで、スマホで1列、タブレットで2列、PCで4列表示するには、どのようなクラスを使いますか？

    回答を表示

    col-12 col-md-6 col-lg-3を使います。
    ・col-12: スマホで12カラム全て使用（1列）
    ・col-md-6: タブレットで6カラム使用（12÷6=2列）
    ・col-lg-3: PCで3カラム使用（12÷3=4列）

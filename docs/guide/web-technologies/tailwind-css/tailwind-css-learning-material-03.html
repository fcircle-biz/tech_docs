第3章: レイアウト構築の基礎

レイアウト構築の基礎

この章で学ぶこと
  • FlexboxとGridを使用したレイアウトの基本概念を理解する
  • コンテナとグリッドシステムの使い方を習得する
  • 要素の配置とアライメントの方法を学ぶ
  • 実用的なレイアウトパターン（カード、ナビゲーション）を作成する
  • レイアウトに関連するディスプレイプロパティを理解する

3.1 Flexboxレイアウトの基礎

Flexboxは、1次元のレイアウト（行または列）を作成するための強力なCSSレイアウトモデルです。Tailwind CSSでは、Flexboxを簡単に使用できるユーティリティクラスが用意されています。

Flexコンテナの作成

flexクラスを適用すると、要素がFlexコンテナになります。

<div class="flex">
  <div>アイテム1</div>
  <div>アイテム2</div>
  <div>アイテム3</div>
</div>

Flex方向の指定

flowchart LR A["Flex Direction"] --> B["flex-row - 横並び（デフォルト）"] A --> C["flex-col - 縦並び"] A --> D["flex-row-reverse - 横並び逆順"] A --> E["flex-col-reverse - 縦並び逆順"] 
<!-- 横並び（デフォルト） -->
<div class="flex flex-row">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>

<!-- 縦並び -->
<div class="flex flex-col">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>

主軸方向の配置（Justify Content）

主軸（通常は水平方向）に沿った要素の配置を制御します。

<div class="flex justify-start">左寄せ（デフォルト）</div>
<div class="flex justify-center">中央寄せ</div>
<div class="flex justify-end">右寄せ</div>
<div class="flex justify-between">両端揃え（間隔均等）</div>
<div class="flex justify-around">周囲に余白を均等配置</div>
<div class="flex justify-evenly">完全均等配置</div>

交差軸方向の配置（Align Items）

交差軸（通常は垂直方向）に沿った要素の配置を制御します。

<div class="flex items-start">上揃え</div>
<div class="flex items-center">中央揃え</div>
<div class="flex items-end">下揃え</div>
<div class="flex items-stretch">伸縮（デフォルト）</div>
<div class="flex items-baseline">ベースライン揃え</div>

Flexアイテムの伸縮

<div class="flex">
  <div class="flex-1">均等に伸縮</div>
  <div class="flex-1">均等に伸縮</div>
  <div class="flex-none">固定サイズ</div>
</div>

3.2 Gridレイアウトの基礎

CSS Gridは、2次元のレイアウト（行と列の両方）を作成するための強力なシステムです。複雑なレイアウトを簡潔に表現できます。

Gridコンテナの作成

<!-- 3カラムのグリッド -->
<div class="grid grid-cols-3 gap-4">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
  <div>6</div>
</div>

グリッドカラムの指定

flowchart TD A["Grid Columns"] --> B["grid-cols-1 - 1カラム"] A --> C["grid-cols-2 - 2カラム"] A --> D["grid-cols-3 - 3カラム"] A --> E["grid-cols-4 - 4カラム"] A --> F["grid-cols-12 - 12カラム"] 
<!-- 2カラムレイアウト -->
<div class="grid grid-cols-2 gap-4">
  <div class="bg-blue-200 p-4">左側</div>
  <div class="bg-green-200 p-4">右側</div>
</div>

<!-- 4カラムレイアウト -->
<div class="grid grid-cols-4 gap-6">
  <div class="bg-purple-200 p-4">1</div>
  <div class="bg-purple-200 p-4">2</div>
  <div class="bg-purple-200 p-4">3</div>
  <div class="bg-purple-200 p-4">4</div>
</div>

Grid Gap（グリッド間隔）

<div class="grid grid-cols-3 gap-2">小さい間隔（8px）</div>
<div class="grid grid-cols-3 gap-4">通常の間隔（16px）</div>
<div class="grid grid-cols-3 gap-8">大きい間隔（32px）</div>
<div class="grid grid-cols-3 gap-x-4 gap-y-8">水平と垂直で異なる間隔</div>

グリッドアイテムのスパン

複数のカラムや行にまたがる要素を作成できます。

<div class="grid grid-cols-3 gap-4">
  <div class="col-span-2 bg-blue-200 p-4">2カラムにまたがる</div>
  <div class="bg-green-200 p-4">1カラム</div>
  <div class="bg-yellow-200 p-4">1カラム</div>
  <div class="col-span-2 bg-red-200 p-4">2カラムにまたがる</div>
</div>
実習 3-1: カードグリッドレイアウトの作成

GridとFlexboxを組み合わせて、実用的なカードグリッドレイアウトを作成します。

実装例
<div class="grid grid-cols-3 gap-6 p-8">
  <!-- カード1 -->
  <div class="bg-white rounded-lg shadow-md overflow-hidden">
    <div class="h-48 bg-blue-300"></div>
    <div class="p-6">
      <h3 class="text-xl font-bold mb-2">カードタイトル1</h3>
      <p class="text-gray-600">カードの説明文がここに入ります。</p>
    </div>
  </div>

  <!-- カード2 -->
  <div class="bg-white rounded-lg shadow-md overflow-hidden">
    <div class="h-48 bg-green-300"></div>
    <div class="p-6">
      <h3 class="text-xl font-bold mb-2">カードタイトル2</h3>
      <p class="text-gray-600">カードの説明文がここに入ります。</p>
    </div>
  </div>

  <!-- カード3 -->
  <div class="bg-white rounded-lg shadow-md overflow-hidden">
    <div class="h-48 bg-purple-300"></div>
    <div class="p-6">
      <h3 class="text-xl font-bold mb-2">カードタイトル3</h3>
      <p class="text-gray-600">カードの説明文がここに入ります。</p>
    </div>
  </div>
</div>
使用したクラスの解説
  • grid grid-cols-3 - 3カラムのグリッドレイアウト
  • gap-6 - カード間の間隔（1.5rem / 24px）
  • overflow-hidden - 角丸の外側にはみ出る要素を隠す
  • h-48 - 画像エリアの高さ（12rem / 192px）

3.3 コンテナとセンタリング

コンテナクラス

containerクラスは、コンテンツを中央に配置し、画面サイズに応じた最大幅を設定します。

<div class="container mx-auto px-4">
  <!-- コンテンツ -->
</div>

要素の中央寄せ

様々な方法で要素を中央に配置できます。

<!-- 水平方向の中央寄せ（ブロック要素） -->
<div class="mx-auto w-64">中央寄せ</div>

<!-- Flexboxで完全中央寄せ -->
<div class="flex items-center justify-center h-screen">
  <div>画面中央に配置</div>
</div>

<!-- Gridで完全中央寄せ -->
<div class="grid place-items-center h-screen">
  <div>画面中央に配置</div>
</div>

3.4 ナビゲーションバーの作成

実習 3-2: レスポンシブナビゲーションバーの構築
<nav class="bg-gray-800 text-white">
  <div class="container mx-auto px-4">
    <div class="flex justify-between items-center h-16">
      <!-- ロゴ -->
      <div class="text-xl font-bold">
        MyWebsite
      </div>

      <!-- ナビゲーションリンク -->
      <div class="flex space-x-4">
        <a href="#" class="hover:text-gray-300">ホーム</a>
        <a href="#" class="hover:text-gray-300">サービス</a>
        <a href="#" class="hover:text-gray-300">会社情報</a>
        <a href="#" class="hover:text-gray-300">お問い合わせ</a>
      </div>
    </div>
  </div>
</nav>
ポイント解説
  • justify-between - ロゴとリンクを両端に配置
  • items-center - 垂直方向に中央揃え
  • space-x-4 - 子要素間に水平スペースを追加
  • h-16 - ナビゲーションバーの高さを統一

3.5 ポジショニング

Position プロパティ

<div class="relative">相対位置指定</div>
<div class="absolute">絶対位置指定</div>
<div class="fixed">固定位置指定</div>
<div class="sticky">粘着位置指定</div>

位置の微調整

<div class="relative">
  <div class="absolute top-0 right-0">右上に配置</div>
  <div class="absolute bottom-0 left-0">左下に配置</div>
  <div class="absolute top-1/2 left-1/2">中央近くに配置</div>
</div>
理解度確認クイズ
  1. FlexboxとGrid、それぞれどのような場合に適していますか？

    解答例：Flexboxは1次元のレイアウト（行または列のいずれか）に適しており、ナビゲーションバーやボタングループなどに便利です。Gridは2次元のレイアウト（行と列の両方）に適しており、カードグリッドやページ全体のレイアウトに適しています。

  2. justify-betweenとjustify-evenlyの違いは何ですか？

    解答：justify-betweenは最初と最後のアイテムを両端に配置し、間隔を均等にします。justify-evenlyは全てのアイテムと両端の間隔を完全に均等にします。

  3. col-span-2は何を意味しますか？

    解答：グリッドレイアウトで、その要素が2カラム分の幅を占めることを意味します。

3.6 まとめ

  • Flexboxは1次元レイアウトに、Gridは2次元レイアウトに適している
  • justify-*クラスで主軸方向、items-*クラスで交差軸方向の配置を制御
  • Gridのgrid-cols-*とgap-*で柔軟なグリッドレイアウトを作成
  • 実用的なレイアウトパターンを組み合わせることで、複雑なUIを構築可能

基本的なスタイリング - テキストと色の装飾

文字サイズ、文字色、背景色、フォント、文字の装飾など、基本的なテキストスタイリングのクラスを学びます。Tailwindの色システムと、よく使われるテキスト関連のユーティリティクラスを習得します。

この章で学ぶこと

  • Tailwind CSSの色システムとカラーパレットの理解
  • 文字サイズと文字色のユーティリティクラスの使い方
  • 背景色の設定とグラデーション表現
  • フォントウェイトやテキスト装飾などの文字関連クラスの使い方

2.1 Tailwindの色システム

Tailwind CSSには、11色の基本カラーがあり、それぞれ50〜900の濃淡（合計10段階）が用意されています。 この統一された色システムにより、一貫性のあるデザインを簡単に作成できます。

色の濃淡システム

flowchart LR A[色名] --> B[濃淡レベル] B --> C[50 - 最も薄い] B --> D[100] B --> E[...] B --> F[500 - 標準] B --> G[...] B --> H[900 - 最も濃い]

カラーパレットの仕組み

Tailwindのカラーは「色名-濃淡」の形式で指定します。 例えば、blue-500は青色の標準的な濃さ、 blue-900は最も濃い青色を表します。 数字が大きいほど濃くなります。

2.2 文字サイズ（Font Size）

Tailwind CSSでは、text-{size}クラスで文字サイズを指定します。 xs（最小）から9xl（最大）まで、14段階のサイズが用意されています。

text-size-example.html 
<p class="text-xs">とても小さい文字（12px）</p>
<p class="text-sm">小さい文字（14px）</p>
<p class="text-base">標準サイズ（16px）</p>
<p class="text-lg">大きめの文字（18px）</p>
<p class="text-xl">より大きい文字（20px）</p>
<p class="text-2xl">さらに大きい文字（24px）</p>
<p class="text-4xl">見出しサイズ（36px）</p>

より大きい文字（text-xl）

見出しサイズ（text-4xl）

使い分けのヒント

  • text-xs, text-sm: 注釈、キャプション、メタ情報
  • text-base, text-lg: 本文テキスト
  • text-xl, text-2xl, text-3xl: 小見出し
  • text-4xl, text-5xl, text-6xl: 大見出し、ヒーローセクション

2.3 文字色（Text Color）

文字色はtext-{color}-{shade}クラスで指定します。 前述のカラーパレットと組み合わせて、様々な色を表現できます。

text-color-example.html 
<p class="text-slate-600">グレー系の文字（通常の本文）</p>
<p class="text-blue-500">青色の文字（リンクなど）</p>
<p class="text-red-600">赤色の文字（エラーメッセージ）</p>
<p class="text-green-600">緑色の文字（成功メッセージ）</p>
<p class="text-yellow-700">黄色の文字（警告メッセージ）</p>

2.4 背景色（Background Color）

背景色はbg-{color}-{shade}クラスで指定します。 また、bg-gradient-to-{direction}を使うと、 グラデーション背景も簡単に作成できます。

bg-color-example.html 
<!-- 単色背景 -->
<div class="bg-blue-100 p-4 rounded">薄い青色の背景</div>
<div class="bg-red-500 text-white p-4 rounded">赤色の背景</div>

<!-- グラデーション背景 -->
<div class="bg-gradient-to-r from-blue-500 to-purple-600 text-white p-4 rounded">
    左から右へのグラデーション
</div>

表示例

薄い青色の背景（bg-blue-100）
赤色の背景（bg-red-500）
薄い緑色の背景（bg-green-200）
左から右へのグラデーション（bg-gradient-to-r from-blue-500 to-purple-600）
左上から右下へのグラデーション（bg-gradient-to-br from-pink-400 to-orange-500）

グラデーション方向の指定

  • bg-gradient-to-r: 左から右
  • bg-gradient-to-l: 右から左
  • bg-gradient-to-t: 下から上
  • bg-gradient-to-b: 上から下
  • bg-gradient-to-br: 左上から右下

2.5 フォントウェイト（Font Weight）

文字の太さはfont-{weight}クラスで指定します。 細い文字から極太まで、9段階の太さが用意されています。

font-weight-example.html 
<p class="font-light">細い文字（300）</p>
<p class="font-normal">通常の太さ（400）</p>
<p class="font-medium">少し太め（500）</p>
<p class="font-semibold">半太字（600）</p>
<p class="font-bold">太字（700）</p>

実習 2-1: カラフルな自己紹介カードを作成

学んだテキストスタイリングのクラスを使って、カラフルな自己紹介カードを作成してみましょう。

手順

  1. 1 新しいHTMLファイル（profile-card.html）を作成し、Tailwind CSS CDNを読み込みます
  2. 2 bg-gradient-to-rでグラデーション背景のカードを作成（色は自由に選択）
  3. 3 名前をtext-3xl font-boldで表示し、白色の文字にします
  4. 4 職業や趣味をtext-lg font-mediumで追加します
  5. 5 自己紹介文をtext-baseで記述し、適切な余白を設定します

期待される結果

グラデーション背景のカードに、異なるサイズと太さの文字で構成された自己紹介が表示されます。

理解度確認クイズ

  1. Q1. Tailwind CSSの色システムで、数字が大きくなるほど色はどうなりますか？

    回答を表示

    数字が大きくなるほど色が濃くなります。 例えば、blue-100は薄い青色ですが、 blue-900は非常に濃い青色になります。 50が最も薄く、900が最も濃い色です。

  2. Q2. 本文テキストに最適な文字サイズのクラスはどれですか？

    回答を表示

    text-base（16px）またはtext-lg（18px）が本文に適しています。 text-baseは標準的な読みやすさを提供し、 text-lgは少し大きめで快適な読書体験を提供します。

  3. Q3. 左から右へのグラデーション背景を作成するクラスの組み合わせは何ですか？

    回答を表示

    bg-gradient-to-r from-{color}-{shade} to-{color}-{shade}です。
    例：bg-gradient-to-r from-blue-500 to-purple-600は、 青色から紫色へ左から右にグラデーションします。
    from-で開始色、to-で終了色を指定します。

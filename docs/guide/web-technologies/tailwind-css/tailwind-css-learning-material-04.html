ボーダーと装飾 - 要素の見た目を整える

ボーダー、角丸、影（shadow）、透明度など、要素を装飾するためのクラスを学びます。カードやボタンなどのUIコンポーネントに必要な装飾技術を習得します。

この章で学ぶこと

  • ボーダーの太さ、色、スタイルを制御できる
  • 角丸（border-radius）を使った柔らかいデザインの実装
  • 影（shadow）を使った立体的なデザインの作成
  • 透明度（opacity）を使った視覚効果の理解

4.1 ボーダー（Border）

Tailwind CSSでは、ボーダーの太さ、色、表示位置を簡単に制御できます。 borderクラスを基本として、数値やカラー名を組み合わせて使用します。

ボーダーの種類と使い方

flowchart TD Border[ボーダークラス] --> Width[太さ指定] Border --> Color[色指定] Border --> Position[位置指定] Width --> W1["border（1px）"] Width --> W2["border-2（2px）"] Width --> W3["border-4（4px）"] Color --> C1["border-blue-500"] Color --> C2["border-red-500"] Color --> C3["border-slate-300"] Position --> P1["border-t（上のみ）"] Position --> P2["border-r（右のみ）"] Position --> P3["border-b（下のみ）"] Position --> P4["border-l（左のみ）"]

border-example.html 
<!-- 基本的なボーダー -->
<div class="border border-slate-300 p-4">
    標準のボーダー

4.2 角丸（Border Radius）

roundedクラスを使うことで、要素の角を丸くすることができます。 丸みを調整することで、柔らかい印象のデザインや、完全な円形の要素を作成できます。

角丸のサイズバリエーション

rounded-none

角丸なし

rounded

小（4px）

rounded-lg

大（8px）

rounded-full

完全な円

角の位置指定

rounded-tl-3xl

左上のみ

rounded-tr-3xl

右上のみ

rounded-bl-3xl

左下のみ

rounded-br-3xl

右下のみ

ヒント

rounded-fullを正方形の要素に適用すると完全な円形になります。 アバター画像やアイコンボタンでよく使われる手法です。

rounded-example.html 
<!-- 基本的な角丸 -->
<div class="bg-blue-500 text-white p-4 rounded">
    小さな角丸
</div>

<!-- 大きめの角丸 -->
<div class="bg-emerald-500 text-white p-4 rounded-xl">
    大きな角丸
</div>

<!-- 円形アバター -->
<div class="w-20 h-20 bg-purple-500 rounded-full flex items-center justify-center text-white">
    A
</div>

4.3 影（Shadow）

shadowクラスを使うと、要素に影をつけることができます。 影を使うことで、要素に立体感を与え、階層構造を視覚的に表現できます。

影のサイズバリエーション

shadow-sm

微細な影

shadow

標準の影

shadow-md

中程度の影

shadow-lg

大きな影

shadow-xl

特大の影

shadow-2xl

最大の影

実用例：カードデザイン

サンプルカード

shadow-lgを使った立体的なカードデザイン

shadow-card.html 
<div class="bg-white rounded-xl shadow-lg p-6">
    <h3 class="text-xl font-bold mb-2">タイトル</h3>
    <p class="text-slate-600">
        立体感のあるカードコンポーネント
    </p>
</div>

重要ポイント

影は控えめに使うことが重要です。大きすぎる影を多用すると、デザインが重たく見えてしまいます。 通常はshadowまたはshadow-mdで十分です。

4.4 透明度（Opacity）

opacityクラスを使うと、要素全体の透明度を調整できます。 オーバーレイや無効化状態の表現、視覚的な階層づけに便利です。

実用例：オーバーレイ

オーバーレイ効果

背景に半透明の黒い層を重ねています

overlay-example.html 
<!-- オーバーレイ効果 -->
<div class="relative">
    <!-- 背景画像 -->
    <div class="bg-gradient-to-r from-purple-400 to-pink-400 h-64"></div>

    <!-- 半透明のオーバーレイ -->
    <div class="absolute inset-0 bg-black opacity-50"></div>

    <!-- コンテンツ -->
    <div class="relative z-10 text-white p-8">
        <h3>見やすいテキスト</h3>
    </div>
</div>

実習 4-1: 価格カードの作成

ボーダー、角丸、影を組み合わせて、プロフェッショナルな価格カードを作成しましょう。

要件

  1. 1 白い背景（bg-white）で角丸（rounded-xl）のカードを作成
  2. 2 立体感を出すために影（shadow-lg）を追加
  3. 3 上部に色付きのボーダー（border-t-4 border-blue-500）でアクセントを追加
  4. 4 カード内にプラン名、価格、機能リストを含める

期待される結果

プロフェッショナルな見た目の価格カードが作成され、 ボーダー、角丸、影が適切に組み合わされて立体感のあるデザインになっています。

理解度確認クイズ

  1. Q1. 要素の左側だけに太さ4pxの赤いボーダーを追加するには、どのクラスを使用しますか？

    回答を表示

    border-l-4 border-red-500
    border-l-4で左側のボーダー太さを指定し、 border-red-500で赤色を指定します。

  2. Q2. 正方形の要素を完全な円形にするには、どのクラスを使用しますか？

    回答を表示

    rounded-full
    このクラスはborder-radius: 9999pxを適用し、 正方形の要素に使用すると完全な円形になります。

  3. Q3. 影のサイズを大きくしたい場合、どのクラスを使用すべきですか？

    回答を表示

    影のサイズは小さい順に：
    shadow-sm → shadow → shadow-md → shadow-lg → shadow-xl → shadow-2xl
    通常はshadow-lgまたはshadow-xlが適切です。

  4. Q4. 要素の透明度を50%にするには、どのクラスを使用しますか？

    回答を表示

    opacity-50
    このクラスは要素全体の透明度を50%（0.5）に設定します。 オーバーレイや無効化状態の表現によく使われます。

この章のまとめ

  • borderクラスでボーダーの太さ、色、位置を制御できます
  • roundedクラスで角丸を作成でき、rounded-fullで円形にできます
  • shadowクラスで要素に立体感を与えることができます
  • opacityクラスで要素の透明度を調整できます
  • これらの装飾を組み合わせることで、プロフェッショナルなUIコンポーネントを作成できます

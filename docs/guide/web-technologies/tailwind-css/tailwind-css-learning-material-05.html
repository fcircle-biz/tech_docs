Flexboxレイアウト - 柔軟な配置の実現

Tailwind CSSのFlexboxユーティリティクラスを使って、要素の配置と整列を制御する方法を学びます。ヘッダー、サイドバー、カードレイアウトなど実用的な例を作成します。

この章で学ぶこと

  • Flexboxの基本概念とTailwindのFlexクラスを理解する
  • 要素の配置と整列（justify、align）を制御できる
  • 方向制御（flex-direction）と折り返し（flex-wrap）を使える
  • ヘッダー、カード、サイドバーなどの実用的なレイアウトを作成できる

5.1 Flexboxの基本概念

Flexbox（Flexible Box Layout）は、要素を柔軟に配置・整列させるためのCSSレイアウトモデルです。 従来のfloatやpositionよりも直感的で、 レスポンシブデザインに最適です。

Flexboxの構造

Flexboxの基本用語

  • Flexコンテナ: display: flexを適用した親要素
  • Flexアイテム: Flexコンテナの直接の子要素
  • 主軸 (Main Axis): アイテムが並ぶ方向（デフォルトは横方向）
  • 交差軸 (Cross Axis): 主軸に対して垂直な方向

5.2 Tailwind CSSのFlexユーティリティクラス

Tailwind CSSでは、Flexboxの機能をシンプルなクラス名で使えます。 最も基本的なクラスはflexで、これを親要素に適用することで Flexコンテナになります。

基本クラス

  • flex - Flexコンテナを作成
  • inline-flex - インラインFlexコンテナ

方向制御

  • flex-row - 横並び（デフォルト）
  • flex-col - 縦並び
  • flex-row-reverse - 横並び逆順
  • flex-col-reverse - 縦並び逆順

折り返し

  • flex-wrap - 折り返しあり
  • flex-nowrap - 折り返しなし
  • flex-wrap-reverse - 逆順折り返し

アイテム制御

  • flex-1 - 均等に伸縮
  • flex-auto - 自動サイズ調整
  • flex-none - 伸縮なし
  • flex-grow - 拡張許可
  • flex-shrink - 縮小許可

5.3 配置と整列（justify-content、align-items）

Flexboxの最も重要な機能の一つが、要素の配置と整列です。 justify-contentは主軸方向、align-itemsは交差軸方向の配置を制御します。

justify-content（主軸方向の配置）

Tailwindクラス

  • justify-start - 先頭に配置（デフォルト）
  • justify-center - 中央に配置
  • justify-end - 末尾に配置
  • justify-between - 両端配置（間隔均等）
  • justify-around - 周囲均等配置
  • justify-evenly - 完全均等配置

視覚例: justify-between

align-items（交差軸方向の配置）

Tailwindクラス

  • items-start - 上端に配置
  • items-center - 中央に配置
  • items-end - 下端に配置
  • items-baseline - ベースライン揃え
  • items-stretch - 引き伸ばし（デフォルト）

視覚例: items-center

center-box.html 
<!-- 完全中央配置の例 -->
<div class="flex justify-center items-center h-64 bg-gray-100">
    <div class="bg-blue-500 text-white px-6 py-4 rounded-lg">
        中央に配置されたボックス
    </div>
</div>

5.4 間隔とギャップ（gap）

Flexアイテム間の間隔は、gapクラスで簡単に設定できます。 従来のようにマージンを個別に指定する必要がなく、一貫性のある間隔を保てます。

gap（全方向）

  • gap-1 - 0.25rem (4px)
  • gap-2 - 0.5rem (8px)
  • gap-4 - 1rem (16px)
  • gap-6 - 1.5rem (24px)
  • gap-8 - 2rem (32px)

個別指定

  • gap-x-4 - 横方向のみ
  • gap-y-2 - 縦方向のみ

※ gap-x と gap-y を組み合わせて、横と縦で異なる間隔を設定できます

gap-example.html 
<!-- gapを使った間隔制御 -->
<div class="flex gap-4 flex-wrap">
    <div class="bg-purple-500 text-white px-4 py-2 rounded">アイテム1</div>
    <div class="bg-purple-500 text-white px-4 py-2 rounded">アイテム2</div>
    <div class="bg-purple-500 text-white px-4 py-2 rounded">アイテム3</div>
</div>

5.5 実用例1: ヘッダーレイアウト

Flexboxを使えば、ロゴとナビゲーションメニューを両端に配置するヘッダーを簡単に作成できます。

header.html 
<!-- シンプルなヘッダーレイアウト -->
<header class="flex justify-between items-center bg-blue-600 text-white px-6 py-4">
    <!-- ロゴ -->
    <div class="flex items-center gap-2">
        <i class="fas fa-rocket text-2xl"></i>
        <span class="text-xl font-bold">MyApp</span>
    </div>

    <!-- ナビゲーション -->
    <nav class="flex gap-6">
        <a href="#" class="hover:text-blue-200">ホーム</a>
        <a href="#" class="hover:text-blue-200">サービス</a>
        <a href="#" class="hover:text-blue-200">お問い合わせ</a>
    </nav>
</header>

5.7 実用例3: サイドバーレイアウト

サイドバーとメインコンテンツを横並びにする2カラムレイアウトもFlexboxで簡単に実装できます。

sidebar-layout.html 
<!-- サイドバーレイアウト -->
<div class="flex gap-4">
    <!-- サイドバー（固定幅） -->
    <aside class="w-64 bg-gray-800 text-white p-4 rounded-lg">
        <h2 class="text-lg font-bold mb-4">メニュー</h2>
        <nav class="flex flex-col gap-2">
            <a href="#" class="hover:bg-gray-700 px-3 py-2 rounded">ダッシュボード</a>
            <a href="#" class="hover:bg-gray-700 px-3 py-2 rounded">プロジェクト</a>
            <a href="#" class="hover:bg-gray-700 px-3 py-2 rounded">設定</a>
        </nav>
    </aside>

    <!-- メインコンテンツ（可変幅） -->
    <main class="flex-1 bg-white p-6 rounded-lg shadow-lg">
        <h1 class="text-2xl font-bold mb-4">メインコンテンツ</h1>
        <p class="text-gray-600">ここにメインコンテンツが表示されます。</p>
    </main>
</div>

実習 5-1: Flexboxでレスポンシブナビゲーションを作成

Flexboxを使って、画面サイズに応じて変化するナビゲーションバーを作成してみましょう。

手順

  1. 1 新しいHTMLファイルを作成し、CDN版Tailwind CSSを読み込みます
  2. 2 flex justify-between items-centerを使ってヘッダーを作成
  3. 3 左側にロゴ、右側にナビゲーションメニュー（3つ以上のリンク）を配置
  4. 4 ナビゲーションメニューにはgap-4以上の間隔を設定
  5. 5 ブラウザで開いて、両端配置と中央揃えが正しく機能しているか確認

期待される結果

ロゴとナビゲーションが両端に配置され、垂直方向は中央揃えになったヘッダーが表示されます。 ナビゲーションリンク間には均等な間隔があり、ホバー時の色変化も機能します。

理解度確認クイズ

  1. Q1. Flexboxで要素を横並びにするために必要な最小限のクラスは何ですか？

    回答を表示

    flexクラスを親要素に適用するだけで、子要素は横並び（flex-row）になります。 Tailwindではflex-rowがデフォルトなので、flexだけで十分です。

  2. Q2. justify-contentとalign-itemsの違いは何ですか？

    回答を表示

    justify-contentは主軸方向（デフォルトは横方向）の配置を制御します。 例：justify-centerで横方向の中央配置。
    align-itemsは交差軸方向（デフォルトは縦方向）の配置を制御します。 例：items-centerで縦方向の中央配置。
    両方を組み合わせることで完全な中央配置が実現できます。

  3. Q3. flex-1クラスは何をするものですか？

    回答を表示

    flex-1は、Flexアイテムに対して 「残りの空間を使って伸縮する」ことを指定します。 複数のアイテムにflex-1を指定すると、 それらは均等な幅になります。サイドバーレイアウトでメインコンテンツに使うことで、 サイドバー以外の空間をすべて使用できます。

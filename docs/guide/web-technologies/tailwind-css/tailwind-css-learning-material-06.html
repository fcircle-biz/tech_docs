Gridレイアウト - 格子状の配置を簡単に

CSS Gridに対応したTailwindクラスを使って、複雑なグリッドレイアウトを簡単に作成する方法を学びます。ギャラリー、ダッシュボード、マガジンレイアウトなどを実装します。

この章で学ぶこと

  • CSS Gridの基本概念とTailwindのGridクラスの理解
  • グリッドコンテナと列数（grid-cols）の指定方法
  • グリッドアイテムの配置とスパン（列・行の結合）
  • ギャラリー、ダッシュボード、マガジンレイアウトの実装

6.1 CSS Gridとは

CSS Gridは、2次元のレイアウトシステムで、行（row）と列（column）の両方を同時に制御できます。 Flexboxは1次元（行または列のどちらか）のレイアウトですが、Gridは行と列を組み合わせた格子状のレイアウトを簡単に作成できます。

FlexboxとGridの使い分け

Flexbox: ナビゲーション、ボタン配置など、1方向の並びに最適
Grid: カードギャラリー、ダッシュボード、ページ全体のレイアウトなど、格子状の配置に最適

FlexboxとGridの違い

flowchart LR subgraph Flexbox["Flexbox（1次元）"] F1[アイテム1] --> F2[アイテム2] F2 --> F3[アイテム3] F3 --> F4[アイテム4] end subgraph Grid["Grid（2次元）"] G1[アイテム1] G2[アイテム2] G3[アイテム3] G4[アイテム4] G5[アイテム5] G6[アイテム6] end

Tailwind CSSでは、gridクラスでGridコンテナを作成し、 grid-cols-{n}で列数を指定します。

6.2 基本的なグリッドレイアウト

まずは、シンプルな3列グリッドを作成してみましょう。

grid-basic.html 
<!-- 3列グリッド -->
<div class="grid grid-cols-3 gap-4">
    <div class="bg-blue-500 text-white p-4 rounded">アイテム1</div>
    <div class="bg-blue-500 text-white p-4 rounded">アイテム2</div>
    <div class="bg-blue-500 text-white p-4 rounded">アイテム3</div>
    <div class="bg-blue-500 text-white p-4 rounded">アイテム4</div>
    <div class="bg-blue-500 text-white p-4 rounded">アイテム5</div>
    <div class="bg-blue-500 text-white p-4 rounded">アイテム6</div>
</div>

6.3 レスポンシブグリッド

画面サイズに応じて列数を変更することで、レスポンシブなグリッドレイアウトを作成できます。

responsive-grid.html 
<!-- レスポンシブグリッド: モバイル1列、タブレット2列、PC4列 -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-4">
    <div class="bg-purple-500 text-white p-4 rounded">カード1</div>
    <div class="bg-purple-500 text-white p-4 rounded">カード2</div>
    <div class="bg-purple-500 text-white p-4 rounded">カード3</div>
    <div class="bg-purple-500 text-white p-4 rounded">カード4</div>
</div>

レスポンシブクラスの適用

grid-cols-1: モバイル（デフォルト）で1列
md:grid-cols-2: タブレット（768px以上）で2列
lg:grid-cols-4: PC（1024px以上）で4列

6.4 グリッドスパン（列・行の結合）

col-span-{n}クラスを使うと、 特定のアイテムを複数列にまたがって配置できます。行のスパンにはrow-span-{n}を使います。

grid-span.html 
<!-- マガジンレイアウト風 -->
<div class="grid grid-cols-3 gap-4">
    <!-- メイン記事（2列分） -->
    <div class="col-span-2 bg-cyan-500 text-white p-6 rounded">
        <h3 class="text-xl font-bold mb-2">メイン記事</h3>
        <p>大きく表示する重要な記事</p>
    </div>
    <!-- サイド記事（1列） -->
    <div class="bg-cyan-400 text-white p-4 rounded">サイド記事1</div>
    <div class="bg-cyan-400 text-white p-4 rounded">記事2</div>
    <div class="bg-cyan-400 text-white p-4 rounded">記事3</div>
    <div class="bg-cyan-400 text-white p-4 rounded">記事4</div>
</div>

スパンの応用

col-span-full: 全列にまたがる（幅いっぱい）
row-span-2: 2行分の高さを占める

実習 6-1: ブログカードグリッドの作成

レスポンシブなブログカードギャラリーを作成しましょう。

手順

  1. 1 grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6でグリッドコンテナを作成
  2. 2 各ブログカードに画像、タイトル、要約、日付を含める
  3. 3 最初のカード（注目記事）をmd:col-span-2で2列分にする
  4. 4 ホバー時に影が濃くなる効果（hover:shadow-xl）を追加

期待される結果

モバイルでは1列、タブレットでは2列（最初のカードは2列分）、PCでは3列で表示され、 レスポンシブに切り替わるブログカードグリッドが完成します。

理解度確認クイズ

  1. Q1. CSS GridとFlexboxの違いは何ですか？

    回答を表示

    Flexboxは1次元レイアウト（行または列のどちらか）に適しており、ナビゲーションやボタン配置などに使われます。 一方、CSS Gridは2次元レイアウト（行と列の両方）を同時に制御でき、ギャラリー、ダッシュボード、ページ全体のレイアウトなど、 格子状の配置に最適です。

  2. Q2. col-span-2クラスの役割は何ですか？

    回答を表示

    グリッドアイテムを2列分にまたがって配置します。 例えば、3列グリッドでcol-span-2を指定すると、 そのアイテムは2列分の幅を占め、より目立つレイアウトを作れます。

  3. Q3. レスポンシブグリッドでgrid-cols-1 md:grid-cols-3と指定した場合、どのように表示されますか？

    回答を表示

    モバイル（デフォルト）では1列で縦に並び、タブレット以上（768px以上、md:ブレークポイント）では3列の横並びグリッドになります。 画面幅に応じて自動的にレイアウトが切り替わります。

この章のまとめ

  • CSS Gridは2次元レイアウトで、行と列を同時に制御できます
  • grid grid-cols-{n}でグリッドコンテナを作成し、列数を指定します
  • レスポンシブプレフィックス（md:など）で画面サイズごとに列数を変更できます
  • col-span-{n}で複数列にまたがるレイアウトを作成できます
  • ギャラリー、ダッシュボード、マガジンレイアウトなど、実用的なレイアウトが簡単に実装できます

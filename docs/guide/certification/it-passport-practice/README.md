# ITパスポート試験 練習問題集

このガイドラインでは、ITパスポート試験の合格を目指す方向けに、各章・各節ごとの練習問題を4択形式で提供しています。

## 前提条件
### 必要な事前学習
- [ITパスポート入門スライド教材](https://fcircle-biz.github.io/tech_docs/slide/certification/it-passport-intro/index.html) 履修済み

### 必要な環境
- Webブラウザ（Chrome、Firefox、Safari等）
- インターネット接続環境

### 参考リソース
- [ITパスポート試験公式サイト（IPA）](https://www3.jitec.ipa.go.jp/JitesCbt/index.html)
- [ITパスポート試験シラバス（最新版）](https://www.ipa.go.jp/shiken/syllabus/ps6vr70000011kdk-att/syllabus_ip_ver6_0.pdf)
- [過去問題・解答例（IPA公式）](https://www3.jitec.ipa.go.jp/JitesCbt/html/openinfo/questions.html)

### 前提知識
- **必須**: ITパスポート入門スライド教材の内容を一通り学習済み
- **推奨**: 過去問題を数問解いた経験があること

## 学習コンテンツ

### [第1章 コンピュータ](https://fcircle-biz.github.io/tech_docs/guide/certification/it-passport-practice/it-passport-practice-01.html)
コンピュータの基本構成、記憶装置、入出力装置、ソフトウェア、表計算ソフト、2進数、マルチメディアに関する練習問題（約40問）

**収録内容:**
- 1-1 コンピュータの基本：五大装置、CPU、GPU
- 1-2 記憶装置：主記憶、補助記憶、キャッシュメモリ
- 1-3 入出力装置：USB、HDMI、Bluetooth、デバイスドライバ、センサー、アクチュエーター
- 1-4 ソフトウェア：OS、OSS
- 1-5 表計算ソフト - 複写：絶対参照と相対参照
- 1-6 表計算ソフト - 関数：合計、平均、最大・最小、IF、VLOOKUP等
- 1-7 2進数：基数変換、論理演算
- 1-8 マルチメディア：静止画形式、動画形式、圧縮・伸長

### [第2章 データベース](https://fcircle-biz.github.io/tech_docs/guide/certification/it-passport-practice/it-passport-practice-02.html)
データベースの基礎、設計、操作に関する練習問題（約20問）

**収録内容:**
- 2-1 データベース：DBMS、RDBMS、NoSQL
- 2-2 データベースの作成：E-R図、正規化
- 2-3 データベースの操作：主キー、外部キー
- 2-4 抽出条件の指定：関係演算（選択、射影、結合）

### [第3章 ネットワーク](https://fcircle-biz.github.io/tech_docs/guide/certification/it-passport-practice/it-passport-practice-03.html)
ネットワークの基礎、無線LAN、プロトコル、IPアドレス、IoTに関する練習問題（約35問）

**収録内容:**
- 3-1 ネットワーク：LAN、WAN、ルーター、スイッチ
- 3-2 無線LAN：Wi-Fi規格、ESSID、WPA2/WPA3
- 3-3 プロトコル：TCP/IP、HTTP/HTTPS、SMTP/POP/IMAP
- 3-4 IPアドレス：IPv4、IPv6、DNS
- 3-5 インターネットへの接続：ISP、テザリング
- 3-6 インターネットに関する技術：Cookie、URL
- 3-7 IoT：LPWA、エッジコンピューティング

### [第4章 セキュリティ](https://fcircle-biz.github.io/tech_docs/guide/certification/it-passport-practice/it-passport-practice-04.html)
セキュリティの脅威、攻撃手法、対策、暗号化技術に関する練習問題（約30問）

**収録内容:**
- 4-1 情報資産が抱える脅威：マルウェア、脆弱性
- 4-2 脅威となる攻撃手法：フィッシング、SQLインジェクション、ブルートフォース攻撃、プロンプトインジェクション
- 4-3 脅威への対策：ファイアウォール、WAF、IDS/IPS
- 4-4 ネットワークセキュリティ：DMZ、VPN
- 4-5 暗号化技術：共通鍵・公開鍵暗号、デジタル署名、PKI
- 4-6 情報セキュリティマネジメント：CIA、ISMS、リスクアセスメント

### [第5章 システムの導入](https://fcircle-biz.github.io/tech_docs/guide/certification/it-passport-practice/it-passport-practice-05.html)
システム構成、導入プロセス、障害対策、監査に関する練習問題（約20問）

**収録内容:**
- 5-1 システムの構成：デュアルシステム、デュプレックスシステム、仮想化
- 5-2 システム導入の流れ：RFI、RFP
- 5-3 システムの障害対策：稼働率、MTBF、MTTR、バックアップ
- 5-4 システム監査：システム監査人、フォローアップ

### [第6章 システム開発](https://fcircle-biz.github.io/tech_docs/guide/certification/it-passport-practice/it-passport-practice-06.html)
システム開発プロセス、設計、プログラミング、テスト、開発モデルに関する練習問題（約30問）

**収録内容:**

- 7-2 アローダイアグラム：クリティカルパス

- 8-2 経営戦略：SWOT分析、PPM、DX
- 8-3 生産戦略：JIT、かんばん方式
- 8-4 業績の評価：BSC、KPI、KGI

**収録内容:**
- 10-1 法務：コンプライアンス、内部統制
- 10-2 ITに関する法律：個人情報保護法、不正アクセス禁止法
- 10-3 知的財産権：著作権法、産業財産権
- 10-4 標準化：ISO/IEC、JIS、QRコード
- 10-5 財務：損益分岐点、変動費と固定費
- 10-6 財務諸表：貸借対照表、損益計算書、キャッシュフロー計算書

## 学習の進め方
1. **スライド教材を復習**: 各章の問題を解く前に、対応するスライド教材を一読しましょう
2. **節ごとに問題を解く**: 各節の5問程度を解いて理解度を確認しましょう
3. **解答・解説を確認**: 間違えた問題は解説をよく読み、理解を深めましょう
4. **繰り返し学習**: 苦手な分野は繰り返し問題を解いて定着させましょう

- 4択問題形式の出題に慣れることができる
- 自分の弱点分野を把握し、効率的に復習できる
- 本番試験への自信をつけることができる

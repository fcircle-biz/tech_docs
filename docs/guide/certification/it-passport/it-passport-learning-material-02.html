第2章: ストラテジ系（企業活動・法務）

企業活動と法務の基礎知識

この章で学ぶこと
  • 企業組織の種類と経営管理の基本概念を理解する
  • 会計・財務の基礎知識と財務諸表の読み方を習得する
  • 知的財産権の種類と保護制度を理解する
  • 情報セキュリティとプライバシー関連の法規制を学ぶ
  • 労働関連法規と取引関連法規の基本を理解する

2.1 企業活動の基礎

経営組織の種類

企業にはさまざまな組織形態があり、それぞれ特徴と目的が異なります。

flowchart TB A[企業組織の種類] A --> B[機能別組織] A --> C[事業部制組織] A --> D[マトリックス組織] A --> E[プロジェクト組織] B --> B1[営業部、開発部、
製造部など機能別に分割] C --> C1[製品や地域ごとに
独立した事業部] D --> D1[機能と事業の
二元的管理] E --> E1[プロジェクトごとに
一時的に編成]
1. 機能別組織

機能別組織は、営業、開発、製造、人事などの業務機能ごとに部門を分ける最も基本的な組織形態です。

  • メリット：専門性が高まる、効率的な業務遂行
  • デメリット：部門間の連携が難しい、意思決定が遅い
2. 事業部制組織

事業部制組織は、製品や地域ごとに独立した事業部を設置し、各事業部が独自に経営判断を行う形態です。

  • メリット：迅速な意思決定、責任の明確化
  • デメリット：重複投資、全社的な調整が難しい
3. マトリックス組織

マトリックス組織は、機能別組織と事業部制組織を組み合わせた形態で、縦と横の二元的な管理を行います。

  • メリット：柔軟な資源配分、専門性と事業性の両立
  • デメリット：指揮命令系統の複雑化、権限の曖昧さ

業務分析とOR・IE

企業活動を効率化するために、業務分析手法が用いられます。

OR（Operations Research：オペレーションズリサーチ）

数学的手法を用いて、最適な意思決定を支援する科学的アプローチです。

  • 線形計画法：資源配分の最適化
  • 待ち行列理論：サービスの待ち時間分析
  • 在庫理論：適正在庫量の決定
  • ゲーム理論：競合関係の分析
IE（Industrial Engineering：インダストリアルエンジニアリング）

作業の効率化と生産性向上を目指す工学的アプローチです。

  • ワークサンプリング：作業時間の分析
  • 時間研究：標準作業時間の設定
  • 動作研究：無駄な動作の削減
実習 2-1: 組織形態の理解

身近な企業や組織を例に、組織形態の特徴を理解しましょう。

手順
  1. 自分が所属する（または知っている）組織の構造を考える
  2. どの組織形態に当てはまるか分類する
  3. その組織形態のメリット・デメリットを実例で確認する
  4. 他の組織形態と比較して、なぜその形態が採用されているか考える
例

大手家電メーカー：事業部制組織（白物家電事業部、AV機器事業部など）
→ 各事業部が独立して製品開発・販売を行い、市場変化に迅速に対応できる

2.2 会計と財務

財務諸表の基礎

企業の経営状況を把握するために、3つの主要な財務諸表があります。

flowchart LR A[財務3表] --> B[貸借対照表
B/S] A --> C[損益計算書
P/L] A --> D[キャッシュフロー計算書
C/F] B --> B1[一定時点の
財政状態] C --> C1[一定期間の
経営成績] D --> D1[一定期間の
資金の流れ]
1. 貸借対照表（B/S：Balance Sheet）

企業がある時点で保有する資産、負債、純資産を示す財務諸表です。

貸借対照表の構造
資産の部
- 流動資産（現金、売掛金など）
- 固定資産（建物、機械など）		 負債の部
- 流動負債（買掛金、短期借入金など）
- 固定負債（長期借入金など）

純資産の部
- 資本金、利益剰余金など
重要な等式

資産 = 負債 + 純資産

この等式は常に成り立ちます。企業が持っている資産は、他人資本（負債）と自己資本（純資産）によって調達されたものです。

2. 損益計算書（P/L：Profit and Loss Statement）

企業が一定期間にどれだけの利益（または損失）を上げたかを示す財務諸表です。

項目 内容
売上高 商品やサービスの販売による収入
売上原価 商品の仕入れや製造にかかった費用
売上総利益 売上高 - 売上原価
販売費及び一般管理費 人件費、広告費、減価償却費など
営業利益 売上総利益 - 販売費及び一般管理費
営業外収益・費用 受取利息、支払利息など
経常利益 営業利益 + 営業外収益 - 営業外費用
特別損益 固定資産売却益、災害損失など
当期純利益 経常利益 + 特別損益 - 法人税等
3. キャッシュフロー計算書（C/F）

企業の現金の流れを示す財務諸表で、以下の3つに分類されます：

  • 営業活動によるキャッシュフロー：本業による現金の増減
  • 投資活動によるキャッシュフロー：設備投資や有価証券の売買による現金の増減
  • 財務活動によるキャッシュフロー：借入金や株式発行による現金の増減
利益と現金の違い

損益計算書で利益が出ていても、現金が不足していることがあります（黒字倒産）。キャッシュフロー計算書で実際の現金の流れを確認することが重要です。

経営指標

財務諸表から企業の経営状態を分析するための主要な指標：

指標 計算式 意味
自己資本比率 純資産 ÷ 総資産 × 100 財務の安全性（高いほど安定）
流動比率 流動資産 ÷ 流動負債 × 100 短期的な支払能力（100%以上が望ましい）
売上高営業利益率 営業利益 ÷ 売上高 × 100 本業での収益性
ROE（自己資本利益率） 当期純利益 ÷ 純資産 × 100 株主資本の効率性
ROA（総資産利益率） 当期純利益 ÷ 総資産 × 100 資産全体の効率性

2.3 知的財産権

知的財産権とは

知的財産権は、人間の創造的活動によって生み出されたアイデアや表現を法的に保護する権利です。

flowchart TB A[知的財産権] A --> B[産業財産権] A --> C[著作権] A --> D[その他] B --> B1[特許権] B --> B2[実用新案権] B --> B3[意匠権] B --> B4[商標権] C --> C1[著作者人格権] C --> C2[著作財産権] D --> D1[営業秘密
ノウハウ]

産業財産権

1. 特許権

発明を保護する権利です。

  • 対象：技術的思想の創作のうち高度なもの
  • 保護期間：出願日から20年
  • 権利：独占的に実施できる権利
  • 例：新しい製造方法、新素材、医薬品
2. 実用新案権

物品の形状、構造、組合せに関する考案を保護する権利です。

  • 対象：特許ほど高度でない技術的アイデア
  • 保護期間：出願日から10年
  • 例：日用品の改良、便利グッズ
3. 意匠権

物品のデザインを保護する権利です。

  • 対象：視覚を通じて美感を起こさせる形状・模様・色彩
  • 保護期間：出願日から25年
  • 例：製品の外観デザイン、GUI（画面デザイン）
4. 商標権

商品やサービスを識別するためのマークを保護する権利です。

  • 対象：文字、図形、記号、立体的形状、音など
  • 保護期間：登録日から10年（更新可能）
  • 例：企業ロゴ、ブランド名、サウンドロゴ

著作権

著作権は、文芸、学術、美術、音楽などの創作物を保護する権利です。

著作権の特徴
  • 無方式主義：登録不要で、創作した時点で自動的に権利が発生
  • 保護期間：原則として著作者の死後70年（法人は公表後70年）
  • 著作物の例：小説、音楽、絵画、写真、映画、プログラム
著作者人格権と著作財産権
権利の種類 内容 譲渡可否
著作者人格権 公表権、氏名表示権、同一性保持権 譲渡不可
著作財産権 複製権、公衆送信権、翻訳権、上演権など 譲渡可能

ソフトウェアの知的財産権

コンピュータプログラムは、以下の権利で保護されます：

  • 著作権：プログラムのソースコードや構造
  • 特許権：プログラムが実現する技術的アイデア
実習 2-2: 知的財産権の識別

身の回りの製品やサービスに、どのような知的財産権が関係しているか考えてみましょう。

例題
  1. スマートフォン → 特許権（通信技術）、意匠権（外観デザイン）、商標権（ブランド名）
  2. 音楽配信サービス → 著作権（楽曲）、商標権（サービス名）
  3. ゲームソフト → 著作権（プログラム、グラフィック）、特許権（ゲームシステム）

2.4 セキュリティ関連法規

個人情報保護法

個人情報保護法は、個人情報の適切な取り扱いを定めた法律です。

個人情報とは

生存する個人に関する情報で、以下のいずれかに該当するもの：

  • 氏名、生年月日、住所、電話番号など特定の個人を識別できる情報
  • マイナンバー、運転免許証番号など個人識別符号を含む情報
要配慮個人情報

特に慎重な取り扱いが必要な個人情報：

  • 人種、信条、社会的身分
  • 病歴、障害、犯罪歴
  • 健康診断結果、遺伝情報
個人情報取扱事業者の義務
義務 内容
利用目的の特定・通知 個人情報を取得する際、利用目的を明示する
適正な取得 不正な手段で個人情報を取得しない
安全管理措置 個人情報の漏えい、滅失を防止する対策を講じる
第三者提供の制限 本人の同意なく第三者に提供しない
開示・訂正・削除 本人から請求があれば対応する

不正アクセス禁止法

正当な権限なくコンピュータに侵入する行為を禁止する法律です。

  • 不正アクセス行為：他人のIDとパスワードを無断使用してシステムにログインする
  • 助長行為の禁止：他人のIDやパスワードを不正に取得・保管・提供する
  • 罰則：3年以下の懲役または100万円以下の罰金

不正競争防止法

公正な競争を阻害する行為を禁止する法律です。

  • 営業秘密の保護：企業の技術情報や顧客リストなどの不正取得・使用を禁止
  • ドメイン名の不正取得：他人の商標と同じドメイン名を不正に取得する行為を禁止
  • 技術的制限手段の回避：コピープロテクトの解除を禁止

2.5 労働・取引関連法規

労働関連法規

労働基準法

労働者の権利を守るための基本的な法律です。

  • 労働時間：1日8時間、週40時間が原則
  • 休憩・休日：労働時間6時間超で45分、8時間超で1時間の休憩、週1日以上の休日
  • 時間外労働：36協定を締結すれば時間外労働可能（割増賃金必要）
  • 年次有給休暇：6か月継続勤務で10日付与
労働者派遣法

派遣労働者の保護と適正な派遣事業の運営を定めた法律です。

  • 派遣期間の制限：原則3年（一部業務を除く）
  • 派遣先の責任：労働条件の明示、安全配慮義務

取引関連法規

下請代金支払遅延等防止法（下請法）

下請事業者を保護するための法律です。

  • 書面の交付義務：発注時に契約内容を明記した書面を交付
  • 支払期日：納品後60日以内に代金を支払う
  • 禁止事項：受領拒否、返品、買いたたき、代金減額など
製造物責任法（PL法）

製品の欠陥による損害について、製造業者の責任を定めた法律です。

  • 対象：製造または加工された動産（ソフトウェアは原則対象外）
  • 製造業者の責任：製品の欠陥が原因で損害が生じた場合、過失がなくても責任を負う（無過失責任）
ソフトウェアとPL法

ソフトウェア単体はPL法の対象外ですが、ハードウェアに組み込まれたソフトウェア（組込みソフト）の欠陥は対象となる場合があります。

理解度確認クイズ
  1. 貸借対照表の等式として正しいものは？
    • A. 資産 = 負債 - 純資産
    • B. 資産 = 負債 + 純資産
    • C. 資産 + 負債 = 純資産
    • D. 資産 - 負債 = 資本金
    解答を見る

    正解：B
    資産 = 負債 + 純資産の等式が常に成り立ちます。

  2. 特許権の保護期間は？
    • A. 出願日から10年
    • B. 出願日から15年
    • C. 出願日から20年
    • D. 出願日から25年
    解答を見る

    正解：C
    特許権の保護期間は出願日から20年です。

  3. 個人情報保護法における要配慮個人情報でないものは？
    • A. 病歴
    • B. 犯罪歴
    • C. 年齢
    • D. 人種
    解答を見る

    正解：C
    年齢は個人情報ですが、要配慮個人情報ではありません。

  4. 労働基準法における1日の法定労働時間は？
    • A. 6時間
    • B. 7時間
    • C. 8時間
    • D. 10時間
    解答を見る

    正解：C
    労働基準法では1日8時間、週40時間が原則です。

  5. 著作権の保護期間（原則）は？
    • A. 著作者の死後50年
    • B. 著作者の死後70年
    • C. 公表後50年
    • D. 公表後30年
    解答を見る

    正解：B
    著作権の保護期間は原則として著作者の死後70年です。

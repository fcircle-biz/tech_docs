企業活動

企業の基礎、経営戦略、生産戦略、業績評価、マーケティング、分析ツール、組織、業務について学習します。

この章で学ぶこと

  • 企業の基礎（経営理念MVV、CSR、SDGs）を理解する
  • 経営戦略（SWOT分析、PPM、DX）を理解する
  • 生産戦略（JIT、かんばん方式）を学ぶ
  • 業績評価（BSC、KPI/KGI）を理解する
  • マーケティング（4P/4C、RFM分析）を習得する
  • 分析ツール（パレート図、特性要因図）を理解する
  • 組織（職能別・事業部制・マトリックス組織、持株会社）を学ぶ
  • 業務（BPR、PDCA）の基礎知識を身につける

8.1 企業の基礎

企業は、利益を追求するだけでなく、社会的責任や持続可能性を考慮した経営が求められています。

経営理念MVV

MVVは、企業の方向性を示す3つの要素です。

要素 内容
Mission（ミッション） 企業の使命や存在意義
Vision（ビジョン） 企業が目指す将来の姿
Value（バリュー） 企業の価値観や行動指針

CSR（企業の社会的責任）

CSR（Corporate Social Responsibility）は、企業が利益追求だけでなく、環境保護、法令遵守、地域貢献など、社会的責任を果たすことです。

SDGs（持続可能な開発目標）

SDGs（Sustainable Development Goals）は、2030年までに達成すべき17の国際目標です。企業は、貧困・環境・教育など様々な分野でSDGsに貢献することが求められています。

重要ポイント

CSRとSDGsは、企業の持続可能性（サステナビリティ）を高めるための重要な取り組みです。

8.2 経営戦略

企業が市場で競争優位を築くための戦略について学びます。

SWOT分析

SWOT分析は、企業の内部環境（強み・弱み）と外部環境（機会・脅威）を分析するフレームワークです。

graph LR A[SWOT分析] --> B["内部環境"] A --> C["外部環境"] B --> D["Strength（強み）"] B --> E["Weakness（弱み）"] C --> F["Opportunity（機会）"] C --> G["Threat（脅威）"]

PPM（プロダクトポートフォリオマネジメント）

PPMは、市場成長率と市場シェアの2軸で事業を4つに分類し、経営資源の配分を最適化する手法です。

分類 特徴 戦略
花形（Star） 高成長・高シェア 積極投資
金のなる木（Cash Cow） 低成長・高シェア 収益確保
問題児（Question Mark） 高成長・低シェア 選択的投資
負け犬（Dog） 低成長・低シェア 撤退検討

DX（デジタルトランスフォーメーション）

DXは、デジタル技術を活用して、ビジネスモデルや業務プロセスを変革することです。単なるIT化ではなく、企業文化や組織全体の変革を伴います。

8.3 生産戦略

効率的な生産を実現するための手法について学びます。

JIT（ジャストインタイム）

JITは、必要なものを、必要なときに、必要な量だけ生産する方式です。在庫を最小限に抑え、ムダを削減します。トヨタ生産方式の中核概念です。

かんばん方式

かんばん方式は、JITを実現するための具体的な手法です。「かんばん」と呼ばれる指示カードを使って、必要な部品を必要な分だけ後工程から前工程に引き取る仕組みです。

JITのメリット

  • 在庫保管コストの削減
  • 製品の品質向上（不良品の早期発見）
  • 市場変化への柔軟な対応

8.4 業績評価

企業の業績を多面的に評価するための手法について学びます。

BSC（バランスト・スコアカード）

BSCは、財務だけでなく、顧客、業務プロセス、学習と成長の4つの視点から企業を評価する手法です。

graph TD A[BSC] --> B["財務の視点"] A --> C["顧客の視点"] A --> D["業務プロセスの視点"] A --> E["学習と成長の視点"]

KPI / KGI

KGI（Key Goal Indicator）は、最終目標の達成度を測る指標です。KPI（Key Performance Indicator）は、KGIを達成するための中間目標の指標です。

指標 例
KGI 年間売上高、利益率
KPI Webサイトアクセス数、顧客満足度

8.5 マーケティング

市場で商品やサービスを販売するための戦略について学びます。

4P / 4C

4Pは企業視点のマーケティングミックス、4Cは顧客視点のフレームワークです。

4P（企業視点） 4C（顧客視点）
Product（製品） Customer Value（顧客価値）
Price（価格） Cost（顧客の負担）
Place（流通） Convenience（利便性）
Promotion（販促） Communication（対話）

RFM分析

RFM分析は、顧客を3つの指標で評価し、優良顧客を特定する手法です。

指標 意味
Recency（最終購買日） 最近購入したか
Frequency（購買頻度） どのくらいの頻度で購入するか
Monetary（購買金額） いくら購入したか

8.6 分析ツール

データを視覚的に分析し、問題解決に役立てるツールについて学びます。

パレート図

パレート図は、項目を大きい順に並べた棒グラフと、累積比率を示す折れ線グラフを組み合わせた図です。重点的に改善すべき項目を特定するのに役立ちます。

パレートの法則（80:20の法則）

全体の結果の80%は、20%の要因によって生み出されるという経験則。例えば、売上の80%は20%の顧客から生まれるなど。

特性要因図（フィッシュボーンチャート）

特性要因図は、問題の原因を魚の骨のような形で整理する図です。原因を「人」「方法」「設備」「材料」などのカテゴリに分類し、体系的に分析します。

8.7 組織

企業の組織構造について学びます。

組織形態

組織形態 特徴
職能別組織 営業、製造、人事など機能ごとに部門を編成
事業部制組織 製品や地域ごとに独立した事業部を編成
マトリックス組織 職能別組織と事業部制組織を組み合わせた形態

持株会社（ホールディングカンパニー）

持株会社は、他社の株式を保有することで、複数の子会社を統括する企業です。グループ全体の経営戦略を立案し、各子会社は事業運営に専念します。

8.8 業務

業務プロセスの改善と管理について学びます。

BPR（ビジネスプロセスリエンジニアリング）

BPRは、既存の業務プロセスを根本から見直し、抜本的に再設計することです。IT技術を活用して、業務効率やサービス品質を劇的に向上させます。

PDCA（Plan-Do-Check-Act）

PDCAは、継続的な改善を実現するサイクルです。

graph LR A["Plan（計画）"] --> B["Do（実行）"] B --> C["Check（評価）"] C --> D["Act（改善）"] D --> A
フェーズ 内容
Plan（計画） 目標を設定し、実施計画を立てる
Do（実行） 計画に基づいて実行する
Check（評価） 実行結果を評価・分析する
Act（改善） 評価に基づいて改善策を講じる

重要ポイント

PDCAサイクルは、1回で終わりではなく、継続的に繰り返すことで業務品質が向上します。

理解度確認クイズ

  1. Q1. SWOT分析における「内部環境」に該当するものはどれか。

    回答を表示

    解答: Strength（強み）、Weakness（弱み）
    解説: 内部環境は、企業が自らコントロールできる要素です。外部環境は、Opportunity（機会）とThreat（脅威）です。

  2. Q2. PPMで「高成長・高シェア」に分類されるものはどれか。

    回答を表示

    解答: 花形（Star）
    解説: 花形は、積極的に投資して市場シェアを維持・拡大すべき事業です。

  3. Q3. RFM分析の「F」は何を表すか。

    回答を表示

    解答: Frequency（購買頻度）
    解説: RFM分析は、Recency（最終購買日）、Frequency（購買頻度）、Monetary（購買金額）の3つの指標で顧客を評価します。

  4. Q4. PDCAサイクルの「C」は何を表すか。

    回答を表示

    解答: Check（評価）
    解説: PDCAは、Plan（計画）、Do（実行）、Check（評価）、Act（改善）のサイクルです。

  5. Q5. BSC（バランスト・スコアカード）に含まれない視点はどれか。

    回答を表示

    解答: 環境の視点（含まれません）
    解説: BSCの4つの視点は、財務、顧客、業務プロセス、学習と成長です。

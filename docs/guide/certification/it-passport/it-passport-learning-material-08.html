8.1 データベース

関係データベース（RDB）

関係データベースは、データを表（テーブル）形式で管理するデータベースです。

基本用語
  • テーブル（表）：データを格納する表
  • 行（レコード、タプル）：1件のデータ
  • 列（フィールド、属性）：データ項目
  • 主キー：レコードを一意に識別する列
  • 外部キー：他のテーブルの主キーを参照する列

SQL（Structured Query Language）

SQLは、データベースを操作するための言語です。

主なSQL文
分類 命令 用途
データ操作 SELECT、INSERT、UPDATE、DELETE データの検索・追加・更新・削除
データ定義 CREATE、ALTER、DROP テーブルの作成・変更・削除
データ制御 GRANT、REVOKE 権限の付与・剥奪

トランザクション処理

トランザクションは、複数の処理をまとめた作業単位です。

ACID特性
  • 原子性（Atomicity）：全て成功するか、全て失敗するか
  • 一貫性（Consistency）：データの整合性が保たれる
  • 独立性（Isolation）：他のトランザクションの影響を受けない
  • 永続性（Durability）：確定したデータは永続的に保存される
トランザクション制御
  • COMMIT：変更を確定
  • ROLLBACK：変更を取り消し

データベースの正規化

正規化は、データの冗長性を排除し、整合性を保つためにテーブルを分割する手法です。

  • 第1正規形：繰り返し項目をなくす
  • 第2正規形：部分関数従属をなくす
  • 第3正規形：推移的関数従属をなくす

8.2 ネットワーク

ネットワークの種類

  • LAN（Local Area Network）：建物内などの狭い範囲のネットワーク
  • WAN（Wide Area Network）：広範囲のネットワーク
  • インターネット：世界規模のネットワーク

ネットワークトポロジ

  • スター型：中心のハブに全端末を接続
  • バス型：1本のケーブルに複数の端末を接続
  • リング型：端末を環状に接続

IPアドレス

IPアドレスは、ネットワーク上の機器を識別するための番号です。

IPv4

32ビット（4バイト）、ドット区切りの10進数で表記（例：192.168.1.1）

IPv6

128ビット（16バイト）、コロン区切りの16進数で表記（例：2001:0db8:85a3:0000:0000:8a2e:0370:7334）

プライベートIPアドレス
  • 10.0.0.0 ～ 10.255.255.255
  • 172.16.0.0 ～ 172.31.255.255
  • 192.168.0.0 ～ 192.168.255.255

サブネットマスク

サブネットマスクは、IPアドレスのネットワーク部とホスト部を区別するためのマスクです。

例：255.255.255.0（ネットワーク部24ビット、ホスト部8ビット）

プロトコル

プロトコルは、通信の規約です。

プロトコル 用途
HTTP/HTTPS Webページの閲覧
FTP ファイル転送
SMTP メール送信
POP3/IMAP メール受信
DNS ドメイン名とIPアドレスの変換
DHCP IPアドレスの自動割り当て

ネットワーク機器

  • リピータ：信号を増幅
  • ハブ：複数の機器を接続（全ポートに転送）
  • スイッチ：MACアドレスで宛先を判断（該当ポートのみ転送）
  • ルータ：異なるネットワークを接続、IPアドレスで経路制御
  • ゲートウェイ：異なるプロトコル間の変換

無線LAN

  • Wi-Fi：無線LAN規格（IEEE 802.11a/b/g/n/ac/ax）
  • SSID：無線LANのネットワーク識別子
  • 暗号化：WPA2、WPA3（WEPは脆弱で非推奨）

8.3 ヒューマンインターフェース

GUI（Graphical User Interface）

GUIは、アイコンやウィンドウを使った直感的な操作画面です。

GUIの要素
  • ウィンドウ：作業領域
  • アイコン：機能を表す小さな絵
  • メニュー：機能の一覧
  • ダイアログボックス：ユーザーとの対話

ユニバーサルデザイン

ユニバーサルデザインは、すべての人が使いやすいデザインです。

7原則
  1. 公平な利用
  2. 利用における柔軟性
  3. 簡単で直感的な利用
  4. 分かりやすい情報
  5. 間違いに対する寛大さ
  6. 少ない身体的負担
  7. 接近や使用のためのサイズと空間

アクセシビリティ

アクセシビリティは、高齢者や障がい者も利用しやすくする配慮です。

  • 音声読み上げ：視覚障がい者向け
  • 字幕表示：聴覚障がい者向け
  • 配色の工夫：色覚特性への配慮
  • 拡大表示：視力低下への対応

マルチメディア

画像データ
  • ビットマップ（ラスタ）：ピクセルの集合（JPEG、PNG、GIF）
  • ベクタ：図形の数式で表現（SVG）
画像の解像度

解像度は、画像の細かさを表す（dpi：dots per inch）

音声データ
  • サンプリング：アナログ音声をデジタル化
  • サンプリング周波数：1秒間に何回測定するか（Hz）
  • 量子化ビット数：音の大きさの精度
理解度確認クイズ
  1. SQLでデータを検索する命令は？
    • A. INSERT
    • B. SELECT
    • C. UPDATE
    • D. DELETE
    解答を見る

    正解：B
    SELECTはデータを検索する命令です。

  2. プライベートIPアドレスの範囲は？
    • A. 192.168.0.0 ～ 192.168.255.255
    • B. 8.8.8.0 ～ 8.8.8.255
    • C. 1.1.1.0 ～ 1.1.1.255
    • D. 127.0.0.0 ～ 127.255.255.255
    解答を見る

    正解：A
    192.168.0.0 ～ 192.168.255.255はプライベートIPアドレスです。

  3. HTTPSで使用されるポート番号は？
    • A. 80
    • B. 443
    • C. 21
    • D. 25
    解答を見る

    正解：B
    HTTPSはポート443を使用します（HTTPは80）。

  4. トランザクションの確定命令は？
    • A. COMMIT
    • B. ROLLBACK
    • C. SELECT
    • D. UPDATE
    解答を見る

    正解：A
    COMMITはトランザクションを確定する命令です。

  5. ルータの役割は？
    • A. 信号の増幅
    • B. 異なるネットワークの接続と経路制御
    • C. MACアドレスによる転送
    • D. ドメイン名の解決
    解答を見る

    正解：B
    ルータは異なるネットワークを接続し、経路制御を行います。

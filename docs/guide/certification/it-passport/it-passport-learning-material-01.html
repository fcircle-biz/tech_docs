1.1 コンピュータの五大装置

コンピュータは、五大装置と呼ばれる5つの基本構成要素から成り立っています。これらの装置が連携することで、様々な処理を実行できます。

（キーボード、マウス）"] Memory["記憶装置
（RAM、HDD、SSD）"] Control["制御装置
（CPU内部）"] Calc["演算装置
（CPU内部）"] Output["出力装置
（ディスプレイ、プリンタ）"] Input --> Memory Memory --> Control Control --> Calc Calc --> Memory Memory --> Output Control -.制御信号.-> Input Control -.制御信号.-> Output

五大装置とは

  • 入力装置: データや命令をコンピュータに入力（キーボード、マウス、スキャナ等）
  • 記憶装置: データやプログラムを保存（RAM、HDD、SSD等）
  • 制御装置: 各装置の動作を制御（CPUの一部）
  • 演算装置: 計算や論理演算を実行（CPUの一部）
  • 出力装置: 処理結果を出力（ディスプレイ、プリンタ等）

CPU（中央処理装置）

CPU（Central Processing Unit）は、コンピュータの「頭脳」に相当する装置です。制御装置と演算装置を含み、プログラムの命令を解釈・実行します。CPUの性能はクロック周波数（GHz単位）やコア数で表されます。

GPU（画像処理装置）

GPU（Graphics Processing Unit）は、グラフィックス処理に特化した装置です。3D描画や動画編集、最近では機械学習やAI処理にも広く使われています。CPUと比較して、大量の並列計算を高速に実行できる特徴があります。

1.2 記憶装置

コンピュータには様々な種類の記憶装置があり、それぞれ用途と特性が異なります。

主記憶装置（メインメモリ）

主記憶装置は、CPUが直接アクセスできる記憶装置で、主にRAM（Random Access Memory）が使われます。データの読み書きが高速ですが、電源を切るとデータが消える揮発性メモリです。

重要ポイント

主記憶装置（RAM）の容量が大きいほど、多くのプログラムを同時に実行できます。

補助記憶装置

補助記憶装置は、データを長期間保存するための記憶装置です。電源を切ってもデータが消えない不揮発性メモリです。代表的なものに、HDD（ハードディスクドライブ）やSSD（ソリッドステートドライブ）があります。

種類 特徴 速度 価格
HDD 磁気ディスクにデータを記録。大容量で安価。 遅い 安い
SSD フラッシュメモリにデータを記録。高速で衝撃に強い。 速い 高い

キャッシュメモリ

キャッシュメモリは、CPUと主記憶装置の間に配置される超高速な記憶装置です。頻繁にアクセスされるデータを一時的に保存することで、CPUの処理速度を向上させます。L1、L2、L3の3階層があり、L1が最も高速です。

1.3 入出力装置とインタフェース

コンピュータと周辺機器を接続するためのインタフェースには、様々な規格があります。

USB（Universal Serial Bus）

USBは、最も広く使われている汎用的な接続規格です。キーボード、マウス、外付けストレージなど多くの機器に対応しています。USB 2.0、USB 3.0、USB 3.1、USB 3.2、USB4など、バージョンによって転送速度が異なります。

HDMI（High-Definition Multimedia Interface）

HDMIは、映像と音声を1本のケーブルで伝送できるデジタル接続規格です。テレビやディスプレイ、プロジェクターとの接続に使われます。

Bluetooth

Bluetoothは、近距離の無線通信規格です。ワイヤレスマウス、ワイヤレスヘッドホン、スマートフォンとの連携などに使われます。

その他の主なインタフェース

  • DisplayPort: 高解像度映像伝送に対応したデジタル接続規格
  • Thunderbolt: 高速データ転送と映像出力に対応（Intel開発）
  • Wi-Fi: 無線LAN規格（詳細は第3章で学習）

1.4 ソフトウェアの基礎

ソフトウェアは、コンピュータを動作させるためのプログラムの総称です。大きく分けてシステムソフトウェアとアプリケーションソフトウェアがあります。

OS（オペレーティングシステム）

OSは、コンピュータのハードウェアとアプリケーションソフトウェアの間を取り持つ基本ソフトウェアです。代表的なOSには、Windows、macOS、Linux、Android、iOSなどがあります。

OSの主な役割

  • ハードウェア資源の管理（CPU、メモリ、ストレージ）
  • ファイルシステムの管理
  • アプリケーションの実行環境提供
  • ユーザーインタフェース（GUI、CLI）の提供

OSS（オープンソースソフトウェア）

OSSは、ソースコードが公開されており、誰でも自由に使用・改変・再配布できるソフトウェアです。代表的なOSSには、Linux、Apache、MySQL、Firefoxなどがあります。

OSSのメリット

無償で利用でき、ソースコードを自由にカスタマイズできるため、企業のシステム構築や教育現場で広く活用されています。

1.5 表計算ソフトの操作

表計算ソフト（Excel、Google スプレッドシート等）は、ビジネスや学習で広く使われるツールです。ITパスポート試験では、相対参照・絶対参照と基本関数が頻出です。

相対参照と絶対参照

セルに数式を入力してコピーすると、参照先が自動的に変わる相対参照と、参照先が固定される絶対参照があります。

参照形式 記述例 説明
相対参照 A1 コピーすると参照先が自動的に変わる
絶対参照 $A$1 コピーしても参照先が固定される
複合参照 $A1 または A$1 列または行のみを固定

基本関数

表計算ソフトでよく使われる基本関数を理解しましょう。

関数 機能 例
SUM 合計値を計算 =SUM(A1:A10)
AVERAGE 平均値を計算 =AVERAGE(A1:A10)
MAX 最大値を取得 =MAX(A1:A10)
MIN 最小値を取得 =MIN(A1:A10)
COUNT 数値データの個数を数える =COUNT(A1:A10)
IF 条件によって値を変える =IF(A1>=60,"合格","不合格")

1.6 2進数と論理演算

コンピュータは、すべてのデータを2進数（0と1）で表現します。また、論理演算はコンピュータの基本的な演算処理です。

2進数の基本

2進数は、0と1の2つの数字だけを使って数を表現します。10進数の「5」は、2進数では「101」となります。

10進数と2進数の対応例

  • 10進数: 0 → 2進数: 0
  • 10進数: 1 → 2進数: 1
  • 10進数: 2 → 2進数: 10
  • 10進数: 3 → 2進数: 11
  • 10進数: 4 → 2進数: 100
  • 10進数: 5 → 2進数: 101

論理演算

論理演算は、真（True）と偽（False）の2つの値を扱う演算です。主な論理演算には、AND、OR、NOT、XORがあります。

演算 意味 例（1=真、0=偽）
AND 両方が真のとき真 1 AND 1 = 1、1 AND 0 = 0
OR どちらかが真のとき真 1 OR 0 = 1、0 OR 0 = 0
NOT 真偽を反転 NOT 1 = 0、NOT 0 = 1
XOR どちらか一方のみ真のとき真 1 XOR 1 = 0、1 XOR 0 = 1

1.7 マルチメディア

マルチメディアとは、テキスト、静止画、動画、音声などの複数のメディアを統合した情報形式です。

静止画形式

形式 特徴 用途
JPEG 非可逆圧縮。写真に適している。 デジタルカメラ、Web画像
PNG 可逆圧縮。透明度をサポート。 Webグラフィックス、アイコン
GIF 可逆圧縮。256色まで。アニメーション対応。 簡易アニメーション

動画形式

形式 特徴
MP4 最も広く使われる動画形式。高圧縮率で高画質。
AVI Windows標準の動画形式。
MOV Apple QuickTime形式。Mac環境で広く使われる。

圧縮技術

データの容量を削減するための技術です。可逆圧縮は元のデータを完全に復元できる方式（ZIP、PNG等）、非可逆圧縮は元のデータを完全には復元できないが高い圧縮率を実現する方式（JPEG、MP3等）です。

理解度確認クイズ

  1. Q1. コンピュータの五大装置に含まれないものはどれか。

    回答を表示

    解答: GPU（画像処理装置）
    解説: 五大装置は、入力装置、記憶装置、制御装置、演算装置、出力装置です。GPUは演算処理を行う装置の一種ですが、五大装置には含まれません。

  2. Q2. 電源を切るとデータが消える記憶装置はどれか。

    回答を表示

    解答: RAM（主記憶装置）
    解説: RAMは揮発性メモリで、電源を切るとデータが消えます。HDD、SSDは不揮発性メモリで、電源を切ってもデータは保持されます。

  3. Q3. 表計算ソフトで、セルA1を絶対参照するにはどう記述するか。

    回答を表示

    解答: $A$1
    解説: 絶対参照は、列と行の両方に「$」記号を付けます。これにより、数式をコピーしても参照先が固定されます。

  4. Q4. 2進数「1010」は10進数でいくつか。

    回答を表示

    解答: 10
    解説: 2進数「1010」は、1×2³ + 0×2² + 1×2¹ + 0×2⁰ = 8 + 0 + 2 + 0 = 10です。

  5. Q5. 写真を保存するのに適した静止画形式はどれか。

    回答を表示

    解答: JPEG
    解説: JPEGは非可逆圧縮で、写真など色数が多い画像に適しています。PNGやGIFは、イラストやロゴなどに適しています。

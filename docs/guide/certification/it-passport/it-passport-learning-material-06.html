6.1 2進数と基数変換

基数（n進数）

コンピュータは2進数で情報を表現します。

基数 使用する数字 用途
2進数 0, 1 コンピュータ内部の表現
8進数 0～7 2進数の簡略表記
10進数 0～9 人間が日常使う数
16進数 0～9, A～F メモリアドレス、色コード

2進数から10進数への変換

例：2進数 1011 を10進数に変換

1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 0 + 2 + 1 = 11

10進数から2進数への変換

例：10進数 13 を2進数に変換（2で割り続ける）

  • 13 ÷ 2 = 6 余り 1
  • 6 ÷ 2 = 3 余り 0
  • 3 ÷ 2 = 1 余り 1
  • 1 ÷ 2 = 0 余り 1
  • 余りを逆順に並べる：1101

16進数

16進数では、10～15をA～Fで表現します。

  • A = 10, B = 11, C = 12, D = 13, E = 14, F = 15
  • 例：16進数 2F = 2×16¹ + 15×16⁰ = 32 + 15 = 47

ビットとバイト

  • ビット（bit）：情報の最小単位、0または1
  • バイト（byte）：8ビット
  • キロバイト（KB）：1,024バイト（2¹⁰）
  • メガバイト（MB）：1,024KB（2²⁰）
  • ギガバイト（GB）：1,024MB（2³⁰）
  • テラバイト（TB）：1,024GB（2⁴⁰）

6.3 データ構造とアルゴリズム

基本的なデータ構造

配列（Array）

同じ型のデータを連続したメモリ領域に格納するデータ構造です。

  • 特徴：インデックスで要素に直接アクセス可能
  • 用途：固定サイズのデータ管理
リスト（List）

要素を順番に連結したデータ構造です。

  • 特徴：要素の追加・削除が容易
  • 用途：可変長のデータ管理
スタック（Stack）

LIFO（Last In, First Out：後入れ先出し）のデータ構造です。

  • 操作：プッシュ（追加）、ポップ（取り出し）
  • 用途：関数呼び出しの管理、戻るボタンの履歴
キュー（Queue）

FIFO（First In, First Out：先入れ先出し）のデータ構造です。

  • 操作：エンキュー（追加）、デキュー（取り出し）
  • 用途：印刷待ち行列、メッセージキュー

代表的なアルゴリズム

線形探索

データを先頭から順番に探索する方法です。

  • 計算量：O(n)
  • 特徴：データがソートされていなくても使用可能
2分探索

ソート済みデータを半分ずつ絞り込んで探索する方法です。

  • 計算量：O(log n)
  • 特徴：高速だが、データのソートが必要
バブルソート

隣接する要素を比較して交換を繰り返すソート方法です。

  • 計算量：O(n²)
  • 特徴：実装は簡単だが効率は悪い

プログラミング言語

言語の分類
分類 特徴 例
機械語 コンピュータが直接実行できる言語 0と1の羅列
アセンブリ言語 機械語を人間が読みやすくした言語 LOAD、STORE、ADD など
高級言語 人間が理解しやすい言語 Python、Java、C、JavaScript
コンパイラとインタプリタ
  • コンパイラ：ソースコードを機械語に一括変換（C、Java）
  • インタプリタ：ソースコードを1行ずつ実行（Python、JavaScript）

マークアップ言語

  • HTML：Webページの構造を記述
  • XML：データの構造を記述
  • JSON：データ交換フォーマット
実習 6-1: 基数変換の練習
問題
  1. 2進数 10110 を10進数に変換しなさい
  2. 10進数 25 を2進数に変換しなさい
  3. 16進数 1A を10進数に変換しなさい
解答
  1. 1×16 + 0×8 + 1×4 + 1×2 + 0×1 = 22
  2. 11001（25÷2=12余1、12÷2=6余0、6÷2=3余0、3÷2=1余1、1÷2=0余1）
  3. 1×16 + 10×1 = 26
理解度確認クイズ
  1. 1バイトは何ビット？
    • A. 4ビット
    • B. 8ビット
    • C. 16ビット
    • D. 32ビット
    解答を見る

    正解：B
    1バイトは8ビットです。

  2. AND演算で1 AND 0の結果は？
    • A. 0
    • B. 1
    • C. 2
    • D. エラー
    解答を見る

    正解：A
    AND演算は両方が1の場合のみ1になります。

  3. スタックのデータ取り出し順は？
    • A. FIFO
    • B. LIFO
    • C. ランダム
    • D. ソート順
    解答を見る

    正解：B
    スタックはLIFO（後入れ先出し）です。

  4. ソースコードを機械語に一括変換するのは？
    • A. インタプリタ
    • B. コンパイラ
    • C. アセンブラ
    • D. リンカ
    解答を見る

    正解：B
    コンパイラはソースコードを機械語に一括変換します。

  5. 2分探索の前提条件は？
    • A. データがソートされている
    • B. データが配列である
    • C. データが整数である
    • D. データが100個以下である
    解答を見る

    正解：A
    2分探索はデータがソートされている必要があります。

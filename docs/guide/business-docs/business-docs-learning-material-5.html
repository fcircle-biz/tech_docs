第5章: 提案書・企画書の作成

説得力のある提案で承認を獲得する

この章で学ぶこと
  • 提案書・企画書の目的と構成を理解する
  • 現状分析と課題の明確化手法を習得する
  • 説得力のある解決策の提示方法を学ぶ
  • 実施計画とスケジュールの作成技術を身につける
  • ビジュアル要素を効果的に活用する方法を理解する

5.1 提案書・企画書の目的と構成

提案書・企画書は、新しいアイデアや改善案を相手に提示し、承認や実施を求めるための文書です。論理性と説得力が重要です。

flowchart TD A[提案書・企画書の目的] --> B[問題解決] A --> C[新規事業提案] A --> D[業務改善] A --> E[承認獲得] B --> B1["現状の課題を
解決する"] C --> C1["新しいビジネス
チャンスを提示"] D --> D1["効率化・
コスト削減を提案"] E --> E1["上司・経営層の
承認を得る"]

5.2 提案書の基本構成

提案書には、読み手を説得するための論理的な構成が必要です。以下の構成が一般的です。

構成要素内容ポイント
表紙提案タイトル、提案者、日付魅力的で分かりやすいタイトルをつける
エグゼクティブサマリー提案の要約1ページで全体像を把握できるように
背景・目的なぜこの提案が必要か現状の問題点を明確にする
現状分析データに基づく現状把握具体的な数値やグラフを使用
課題の明確化解決すべき問題点優先度をつけて整理する
解決策具体的な提案内容実現可能で効果的な方法を提示
実施計画スケジュールとマイルストーンいつ、誰が、何をするか明示
予算必要な費用と費用対効果ROI（投資対効果）を示す
リスクと対策想定されるリスクと対応策事前にリスクを把握していることを示す
期待される効果提案実施後の成果定量的・定性的効果を明示

5.3 現状分析と課題の明確化

説得力のある提案には、データに基づく現状分析が不可欠です。「なんとなく問題がある」ではなく、具体的な数値やファクトで示します。

現状分析のフレームワーク

  • SWOT分析: 強み・弱み・機会・脅威を分析
  • As-Is / To-Be分析: 現状と理想の状態を比較
  • 問題の定量化: 数値で問題の大きさを示す
  • 原因分析: なぜその問題が発生しているか
実習 5-1: 課題の明確化

テーマ: 社内コミュニケーション改善の提案

分析例
【現状分析】
・部門間の情報共有が不十分
・社員アンケート: 70%が「他部署の状況が分からない」と回答
・会議時間: 月平均40時間（前年比+25%）
・メール処理時間: 1日平均2時間

【課題の明確化】
1. 情報が散在し、必要な情報にアクセスしにくい
2. 会議が多く、本来業務に集中できない
3. メールでのやり取りが非効率

【問題の影響】
・業務効率の低下（推定で年間500万円のコスト増）
・社員の不満増加（離職率+3%）
・意思決定の遅延（平均5日→8日に悪化）

5.4 提案内容の具体化

課題を明確にしたら、それを解決するための具体的な提案を示します。抽象的な提案ではなく、実行可能な内容にします。

提案の3要素

flowchart TD A[効果的な提案] --> B["What
何をするか"] A --> C["Why
なぜ有効か"] A --> D["How
どう実現するか"] B --> B1[具体的な施策] C --> C1[効果の根拠] D --> D1[実現方法]

5.5 実施計画とスケジュールの作成

提案を実行するための具体的なスケジュールを示すことで、提案の実現可能性を高めます。

フェーズ期間主な活動成果物
準備1ヶ月要件定義、体制構築要件定義書
設計2ヶ月詳細設計、ツール選定設計書、契約書
構築3ヶ月システム構築、テスト稼働システム
導入1ヶ月社員教育、移行作業教育資料
運用継続運用・保守、改善運用報告書

5.6 予算と費用対効果の算出

提案には必ず予算と費用対効果（ROI）を示します。投資に見合う効果があることを証明します。

費用対効果の計算例

【投資額】
・システム導入費用: 300万円
・運用費用(年間): 50万円
・教育費用: 30万円
合計: 380万円（初年度）

【期待効果（年間）】
・業務時間削減: 500時間 × 3,000円/時間 = 150万円
・会議時間削減: 300時間 × 4,000円/時間 = 120万円
・ペーパーレス化: 30万円
合計効果: 300万円/年

【ROI】
初年度: (300万円 - 380万円) / 380万円 = -21%
2年目以降: (300万円 - 50万円) / 50万円 = 500%
投資回収期間: 約1.3年

5.7 リスクと対策の記述

提案を実行する際のリスクを事前に洗い出し、対策を示すことで、提案の信頼性が高まります。

リスク影響度発生確率対策
システム不具合高中十分なテスト期間を確保、段階的導入
社員の抵抗中高事前説明会の実施、サポート体制構築
予算超過中低予備費10%を確保、進捗管理の徹底
スケジュール遅延低中バッファ期間の設定、優先度管理

5.8 ビジュアル要素の効果的な活用

提案書では、図解、グラフ、イラストなどのビジュアル要素を活用すると、理解しやすく印象に残ります。

効果的なビジュアル活用例

  • 比較グラフ: 現状と提案後の効果を視覚的に比較
  • フローチャート: プロセスや業務フローを図解
  • タイムライン: スケジュールを時系列で表示
  • アイコン: 重要なポイントを視覚的に強調
  • イメージ写真: 完成イメージを視覚化

5.9 説得力のある提案書の書き方

提案を承認してもらうには、読み手（意思決定者）の視点に立つことが重要です。

意思決定者が知りたいこと

  • 問題の重要性: なぜ今やる必要があるのか
  • 実現可能性: 本当にできるのか
  • 費用対効果: 投資に見合う効果があるか
  • リスク: どんな問題が起こりうるか
  • 代替案: 他の選択肢は検討したか
提案書作成の注意点
  • 専門用語を多用しない（相手の理解度に合わせる）
  • 楽観的すぎる見通しは避ける（現実的な計画を）
  • 一方的な主張ではなく、客観的なデータに基づく
  • 代替案も提示し、推奨案を明確にする

5.10 PowerPointを活用した企画書作成

企画書は、PowerPointで作成することも多いです。視覚的に訴求力のあるスライドを作成しましょう。

スライド作成のポイント

  • 1スライド1メッセージ: 情報を詰め込みすぎない
  • 見出しで結論を示す: スライドタイトルに要点を
  • フォントは大きく: 最小でも18pt以上
  • 余白を確保: ゆとりのあるレイアウト
  • 色は3色まで: 統一感のある配色
実習 5-2: 提案書の作成

テーマ: テレワーク制度導入の提案

構成例
【提案書: テレワーク制度導入について】

1. エグゼクティブサマリー
   テレワーク制度を導入し、社員の働きやすさ向上と
   優秀な人材の確保を実現する。

2. 背景・目的
   ・働き方改革の推進
   ・通勤時間の削減
   ・多様な人材の採用促進

3. 現状と課題
   ・平均通勤時間: 片道1時間
   ・育児・介護との両立が困難
   ・競合他社は既に導入済み

4. 提案内容
   ・週2日までのテレワークを認める
   ・対象: 全正社員（試用期間終了後）
   ・勤務時間: フレックスタイム制と併用

5. 実施計画
   準備期間: 3ヶ月
   試験導入: 3ヶ月（一部部署で）
   本格導入: 試験結果を踏まえて全社展開

6. 必要な投資
   ・ノートPC: 100台 × 10万円 = 1,000万円
   ・セキュリティツール: 年間300万円
   ・通信費補助: 月5,000円 × 200名 = 年間1,200万円
   合計: 初年度2,500万円

7. 期待される効果
   ・社員満足度向上
   ・通勤費削減: 年間800万円
   ・オフィススペース削減: 年間500万円
   ・離職率低下: 年間1,000万円相当

8. リスクと対策
   【リスク】業務管理の難しさ
   【対策】勤怠管理システムの導入、定期的な1on1実施

9. 結論
   初年度の投資は必要だが、2年目以降は年間2,300万円の
   コスト削減が見込まれ、社員満足度向上も期待できる。
理解度確認クイズ
  1. 提案書の基本構成要素を5つ以上挙げてください。
  2. 現状分析で使用できるフレームワークを2つ挙げてください。
  3. 費用対効果（ROI）とは何ですか？なぜ重要ですか？
  4. 提案書でリスクを記載する目的は何ですか？
  5. PowerPointスライド作成で重要なポイントを3つ挙げてください。
解答例
  1. 表紙、エグゼクティブサマリー、背景・目的、現状分析、課題、解決策、実施計画、予算、リスクと対策、期待される効果
  2. SWOT分析（強み・弱み・機会・脅威）、As-Is / To-Be分析（現状と理想の比較）
  3. 投資に対してどれだけの効果が得られるかを示す指標。投資判断の重要な根拠となる。
  4. リスクを事前に把握し対策を示すことで、提案の信頼性を高め、意思決定者の不安を軽減する
  5. 1スライド1メッセージ、見出しで結論を示す、フォントは大きく（18pt以上）
この章のまとめ
  • 提案書は問題解決や新規事業を提案し、承認を得るための文書
  • 背景、現状分析、課題、解決策、実施計画、予算、リスク、効果の構成が基本
  • データに基づく現状分析で問題を定量化する
  • 具体的で実現可能な提案内容を示す
  • 費用対効果（ROI）を明示し、投資の妥当性を証明する
  • リスクと対策を示すことで提案の信頼性を高める
  • ビジュアル要素を活用して視覚的に分かりやすく表現する

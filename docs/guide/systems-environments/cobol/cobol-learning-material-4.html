第4章: 基本的な計算と演算

COBOLにおける数値計算の基本

この章で学ぶこと
  • 四則演算の基本的な使い方を理解する
  • ADD、SUBTRACT、MULTIPLY、DIVIDE文の使い方を習得する
  • COMPUTE文による複雑な計算を学ぶ
  • 数値編集による表示形式の制御を理解する
  • 丸め処理とエラーハンドリングを学ぶ

4.1 四則演算の基本

COBOLでは、専用の動詞を使用して四則演算を行います。

演算 動詞 基本構文 例
加算 ADD ADD A TO B ADD 10 TO WS-TOTAL
減算 SUBTRACT SUBTRACT A FROM B SUBTRACT 5 FROM WS-COUNT
乗算 MULTIPLY MULTIPLY A BY B MULTIPLY WS-PRICE BY 1.08
除算 DIVIDE DIVIDE A INTO B DIVIDE 2 INTO WS-AMOUNT

4.2 算術動詞の使い方

各算術動詞の詳細な使い方を学習します。

ADD文

基本形式
ADD A TO B * BにAを加算
複数項目の加算
ADD A B C TO D * DにA+B+Cを加算

SUBTRACT文

基本形式
SUBTRACT A FROM B * BからAを減算
複数項目の減算
SUBTRACT A B FROM C * CからA+Bを減算

MULTIPLY文とDIVIDE文

MULTIPLY文
MULTIPLY A BY B * BにAを掛算
DIVIDE文
DIVIDE A INTO B * BをAで除算
IDENTIFICATION DIVISION. PROGRAM-ID. ARITHMETIC-EXAMPLE. DATA DIVISION. WORKING-STORAGE SECTION. 01 WS-NUM1 PIC 9(3) VALUE 100. 01 WS-NUM2 PIC 9(3) VALUE 50. 01 WS-RESULT PIC 9(5). PROCEDURE DIVISION. MAIN-PARA. * 加算 ADD WS-NUM1 TO WS-NUM2. DISPLAY "加算結果: " WS-NUM2. * 減算 SUBTRACT 25 FROM WS-NUM2. DISPLAY "減算結果: " WS-NUM2. * 乗算 MULTIPLY WS-NUM1 BY WS-NUM2. DISPLAY "乗算結果: " WS-NUM2. * 除算 DIVIDE 2 INTO WS-NUM2. DISPLAY "除算結果: " WS-NUM2. STOP RUN.

4.3 COMPUTE文

COMPUTE文を使用すると、複雑な計算式を一度に記述できます。

COMPUTE文の利点
  • 複雑な計算式を一行で記述可能
  • 数学的な演算子を使用（+、-、*、/）
  • 括弧を使用した優先順位の指定
  • 数学関数の利用（SQRT、SIN、COS等）

COMPUTE文の基本構文

COMPUTE 結果変数 = 計算式

使用例

DATA DIVISION. WORKING-STORAGE SECTION. 01 WS-PRICE PIC 9(5)V9(2). 01 WS-QUANTITY PIC 9(3). 01 WS-TAX-RATE PIC 9V9(3) VALUE 0.08. 01 WS-TOTAL PIC 9(7)V9(2). 01 WS-RADIUS PIC 9(3)V9(2). 01 WS-AREA PIC 9(5)V9(4). PROCEDURE DIVISION. MAIN-PARA. MOVE 1250.00 TO WS-PRICE. MOVE 3 TO WS-QUANTITY. * 税込み金額の計算 COMPUTE WS-TOTAL = WS-PRICE * WS-QUANTITY * (1 + WS-TAX-RATE). * 円の面積計算 MOVE 5.5 TO WS-RADIUS. COMPUTE WS-AREA = 3.14159 * WS-RADIUS ** 2. DISPLAY "税込み総額: " WS-TOTAL. DISPLAY "円の面積: " WS-AREA.

4.4 数値編集

数値編集を使用して、数値を見やすい形式で表示できます。

編集用PICTURE文字

文字 意味 例 結果
Z ゼロ抑制 ZZ9 5（005の場合）
, カンマ区切り Z,ZZ9 1,234
. 小数点 ZZ9.99 123.45
+/- 符号表示 +ZZ9 +123
¥ 通貨記号 ¥ZZ,ZZ9 ¥1,234
DATA DIVISION. WORKING-STORAGE SECTION. 01 WS-AMOUNT PIC 9(6)V9(2) VALUE 123456.78. 01 WS-FORMATTED PIC ¥ZZ,ZZ9.99. PROCEDURE DIVISION. MAIN-PARA. MOVE WS-AMOUNT TO WS-FORMATTED. DISPLAY "編集前: " WS-AMOUNT. DISPLAY "編集後: " WS-FORMATTED.

4.5 丸め処理とエラーハンドリング

計算結果の丸め処理とエラーハンドリングについて学習します。

ROUNDED句

ROUNDED句を使用すると、計算結果を四捨五入できます。

COMPUTE WS-RESULT ROUNDED = WS-NUM1 / WS-NUM2

ON SIZE ERROR句

計算結果が変数の範囲を超えた場合のエラーハンドリング

DIVIDE WS-NUM1 BY WS-NUM2 GIVING WS-RESULT ON SIZE ERROR DISPLAY "エラー: 計算結果が範囲を超えています"

2.1 変数の概念と役割

変数とは、プログラムの実行中にデータを一時的に保管する「入れ物」のような存在です。変数を使うことで、計算結果を保存したり、ユーザーからの入力を記録したり、プログラム内でデータを操作することができます。

変数を使う理由

変数がないプログラムを想像してみてください。毎回同じ値を直接書かなければならず、値を変更したいときには、すべての場所を手動で書き換える必要があります。変数を使うことで：

  • 再利用性: 同じデータを複数の場所で使用できる
  • 保守性: 値の変更が一箇所で済む
  • 可読性: コードが理解しやすくなる
  • 動的性: 実行時にデータを変更できる

変数の宣言方法

VB.NETでは、Dimキーワードを使って変数を宣言します。 

' 基本的な変数宣言の構文
Dim 変数名 As データ型

' 具体例
Dim userName As String      ' 文字列を格納する変数
Dim age As Integer         ' 整数を格納する変数
Dim price As Decimal       ' 小数を格納する変数

データ型は、変数が格納できるデータの種類を定義します。適切なデータ型を選ぶことで、メモリ使用量を最適化し、プログラムの安全性を向上させることができます。

実習 2-1: 様々なデータ型を使った変数宣言

異なるデータ型の変数を宣言し、値を代入して表示するプログラムを作成しましょう。

サンプルコード
Module Program
    Sub Main(args As String())
        ' 様々なデータ型の変数宣言と初期化
        Dim studentName As String = "田中太郎"
        Dim age As Integer = 20
        Dim height As Double = 170.5
        Dim isStudent As Boolean = True
        Dim birthday As Date = #1/15/2003#
        Dim balance As Decimal = 50000.99D
        
        ' 変数の内容を表示
        Console.WriteLine("=== 学生情報 ===")
        Console.WriteLine("名前: " + studentName)
        Console.WriteLine("年齢: " + age.ToString() + "歳")
        Console.WriteLine("身長: " + height.ToString() + "cm")
        Console.WriteLine("学生: " + isStudent.ToString())
        Console.WriteLine("生年月日: " + birthday.ToString("yyyy年M月d日"))
        Console.WriteLine("残高: " + balance.ToString("C"))
        
        Console.WriteLine("何かキーを押してください...")
        Console.ReadKey()
    End Sub
End Module

2.3 定数の使用

定数は、プログラム実行中に値が変わらないデータを格納するためのものです。変数とは異なり、一度値を設定すると変更できません。

定数を使う利点

  • 安全性: 意図しない値の変更を防げる
  • 保守性: 定数値の変更が一箇所で済む
  • 可読性: 意味のある名前で値を表現できる
  • パフォーマンス: コンパイル時に値が確定される
' 定数の宣言方法
Const TAX_RATE As Double = 0.1         ' 消費税率
Const MAX_USERS As Integer = 1000       ' 最大ユーザー数
Const COMPANY_NAME As String = "ABC商事" ' 会社名

' 使用例
Dim price As Decimal = 1000
Dim taxAmount As Decimal = price * TAX_RATE
Dim totalPrice As Decimal = price + taxAmount

2.4 演算子の種類と使い方

演算子は、値に対して計算や比較などの操作を行うための記号です。VB.NETでは様々な種類の演算子が提供されています。

算術演算子

演算子 意味 例 結果
+ 加算 5 + 3 8
- 減算 5 - 3 2
* 乗算 5 * 3 15
/ 除算 10 / 3 3.333...
\ 整数除算 10 \ 3 3
Mod 剰余 10 Mod 3 1
^ 累乗 2 ^ 3 8

比較演算子

演算子 意味 例 結果
= 等しい 5 = 5 True
<> 等しくない 5 <> 3 True
< より小さい 3 < 5 True
> より大きい 5 > 3 True
<= 以下 3 <= 5 True
>= 以上 5 >= 5 True

論理演算子

演算子 意味 説明
And 論理積 両方がTrueの場合のみTrue
Or 論理和 どちらかがTrueならTrue
Not 論理否定 TrueをFalse、FalseをTrueに変換
AndAlso 短絡評価And 最初がFalseなら2番目を評価しない
OrElse 短絡評価Or 最初がTrueなら2番目を評価しない
実習 2-2: 様々な演算子を使った計算

算術演算子、比較演算子、論理演算子を組み合わせて実用的な計算プログラムを作成しましょう。

サンプルコード
Module Program
    Sub Main(args As String())
        ' 商品購入の計算例
        Const TAX_RATE As Double = 0.1
        Const DISCOUNT_RATE As Double = 0.05
        Const FREE_SHIPPING_THRESHOLD As Integer = 3000
        
        Dim itemPrice As Integer = 2500
        Dim quantity As Integer = 2
        Dim isMember As Boolean = True
        
        ' 基本計算
        Dim subtotal As Integer = itemPrice * quantity
        Dim discount As Double = If(isMember, subtotal * DISCOUNT_RATE, 0)
        Dim afterDiscount As Double = subtotal - discount
        Dim tax As Double = afterDiscount * TAX_RATE
        Dim total As Double = afterDiscount + tax
        
        ' 送料の計算
        Dim shippingFee As Integer = If(total >= FREE_SHIPPING_THRESHOLD, 0, 500)
        Dim finalTotal As Double = total + shippingFee
        
        ' 結果の表示
        Console.WriteLine("=== 購入計算結果 ===")
        Console.WriteLine("商品価格: " + itemPrice.ToString() + "円")
        Console.WriteLine("数量: " + quantity.ToString() + "個")
        Console.WriteLine("小計: " + subtotal.ToString() + "円")
        Console.WriteLine("会員割引: " + discount.ToString("F0") + "円")
        Console.WriteLine("割引後: " + afterDiscount.ToString("F0") + "円")
        Console.WriteLine("税金: " + tax.ToString("F0") + "円")
        Console.WriteLine("送料: " + shippingFee.ToString() + "円")
        Console.WriteLine("合計: " + finalTotal.ToString("F0") + "円")
        
        ' 条件判定の表示
        Console.WriteLine()
        Console.WriteLine("=== 判定結果 ===")
        Console.WriteLine("会員: " + isMember.ToString())
        Console.WriteLine("送料無料対象: " + (total >= FREE_SHIPPING_THRESHOLD).ToString())
        Console.WriteLine("高額購入: " + (total > 5000).ToString())
        
        Console.ReadKey()
    End Sub
End Module

2.5 コメントの書き方

コメントは、プログラムの動作には影響しない説明文です。コードの目的や動作を記録し、後でコードを読み返したときに理解しやすくするために使用します。

コメントの種類

  • 単行コメント: '（アポストロフィ）で始まる
  • 複数行コメント: VB.NETには専用構文はないが、各行に'を付けて記述
' これは単行コメントです
Dim userName As String = "田中"  ' 行末コメントも可能

' 複数行にわたるコメントは
' 各行の先頭にアポストロフィを付けます
' このように長い説明を書くことができます

'==================================================
' プログラムのメイン処理開始
'==================================================

良いコメントの書き方

コメントのベストプラクティス
  • 「なぜ」を説明する: コードが「何を」しているかより「なぜ」そうしているかを記録
  • 複雑なロジックに注釈: 理解に時間がかかる処理には詳細な説明を
  • 仮定や制約を記録: コードが前提としている条件を明記
  • TODOや修正予定の記録: 後で対応すべき項目をマーク
  • 簡潔で正確に: 長すぎず、間違った情報は書かない

2.6 VB.NET特有の構文ルール

VB.NETには、他のプログラミング言語とは異なる独特の構文ルールがあります。これらのルールを理解することで、正しくて読みやすいコードを書けるようになります。

主な構文ルール

1. ステートメント区切り
  • 通常は 改行 でステートメントを区切る
  • 1行に複数のステートメントを書く場合は : で区切る
  • 長いステートメントは _ で行継続できる
' 通常の書き方（推奨）
Dim name As String = "田中"
Dim age As Integer = 25

' 1行に複数ステートメント（非推奨）
Dim x As Integer = 10 : Dim y As Integer = 20

' 行継続（長い行を分割）
Dim longMessage As String = "これは非常に長いメッセージで、" + _
                           "複数行にわたって記述されています。"
2. 大文字・小文字の区別

VB.NETは 大文字・小文字を区別しません。userName、UserName、USERNAME はすべて同じ変数として扱われます。 

' これらはすべて同じ変数
Dim userName As String
USERNAME = "田中"
UserName = "佐藤"
Console.WriteLine(username)  ' "佐藤" が表示される
3. ブロック構造

VB.NETでは明示的な開始・終了キーワードを使ってブロックを定義します：

構造 開始 終了
Module Module End Module
Sub/Function Sub/Function End Sub/End Function
If文 If End If
For文 For Next

2.7 文字列の基本操作

文字列は、プログラムで最も頻繁に使用されるデータ型の一つです。VB.NETでは文字列に対して様々な操作を行うことができます。

文字列の連結

' 文字列連結の方法
Dim firstName As String = "田中"
Dim lastName As String = "太郎"

' + 演算子による連結
Dim fullName1 As String = firstName + lastName

' & 演算子による連結（推奨）
Dim fullName2 As String = firstName & lastName

' 複数の値を連結
Dim age As Integer = 25
Dim message As String = "私の名前は" & fullName2 & "で、" & age & "歳です。"

文字列の基本メソッド

メソッド 機能 例
Length 文字列の長さ "Hello".Length → 5
ToUpper() 大文字に変換 "hello".ToUpper() → "HELLO"
ToLower() 小文字に変換 "HELLO".ToLower() → "hello"
Trim() 前後の空白除去 " Hello ".Trim() → "Hello"
Substring() 部分文字列の取得 "Hello".Substring(1, 3) → "ell"
実習 2-3: 文字列操作の実践

様々な文字列操作を組み合わせて、実用的なプログラムを作成しましょう。

サンプルコード
Module Program
    Sub Main(args As String())
        ' ユーザー情報の処理例
        Dim inputName As String = "  田中 太郎  "
        Dim email As String = "TANAKA@example.com"
        Dim phoneNumber As String = "090-1234-5678"
        
        ' 文字列の整形
        Dim cleanName As String = inputName.Trim()
        Dim normalizedEmail As String = email.ToLower()
        Dim phoneDigits As String = phoneNumber.Replace("-", "")
        
        ' 文字列の分析
        Dim nameLength As Integer = cleanName.Length
        Dim containsAt As Boolean = normalizedEmail.Contains("@")
        Dim phoneLength As Integer = phoneDigits.Length
        
        ' 結果の表示
        Console.WriteLine("=== 文字列処理結果 ===")
        Console.WriteLine("元の名前: '" & inputName & "'")
        Console.WriteLine("整形後名前: '" & cleanName & "'")
        Console.WriteLine("名前の長さ: " & nameLength & "文字")
        Console.WriteLine()
        
        Console.WriteLine("元のメール: " & email)
        Console.WriteLine("正規化メール: " & normalizedEmail)
        Console.WriteLine("@マークを含む: " & containsAt)
        Console.WriteLine()
        
        Console.WriteLine("元の電話番号: " & phoneNumber)
        Console.WriteLine("数字のみ: " & phoneDigits)
        Console.WriteLine("電話番号の桁数: " & phoneLength)
        
        ' 条件判定
        If nameLength > 0 AndAlso containsAt AndAlso phoneLength = 11 Then
            Console.WriteLine()
            Console.WriteLine("✓ すべての入力が有効です")
        Else
            Console.WriteLine()
            Console.WriteLine("✗ 入力に問題があります")
        End If
        
        Console.ReadKey()
    End Sub
End Module
理解度確認クイズ
  1. VB.NETで変数を宣言するキーワードは？
    a) Var
    b) Dim
    c) Let
    d) Declare
  2. 整数除算を行う演算子は？
    a) /
    b) \
    c) //
    d) div
  3. VB.NETは大文字・小文字を区別しますか？
    a) 区別する
    b) 区別しない
    c) キーワードのみ区別する
    d) 変数名のみ区別する
  4. 文字列連結に推奨される演算子は？
    a) +
    b) &
    c) *
    d) |
  5. コメントを書くための記号は？
    a) //
    b) /* */
    c) '
    d) #
解答を表示
  1. b) Dim - "Dimension"の略で、VB.NETの変数宣言に使用します。
  2. b) \ - バックスラッシュは整数除算を行い、結果は整数になります。
  3. b) 区別しない - VB.NETは大文字・小文字を区別しないため、userNameとUSERNAMEは同じ変数です。
  4. b) & - アンパサンド（&）は明示的に文字列連結を示し、+演算子より推奨されます。
  5. c) ' - アポストロフィ（'）でコメントを開始します。

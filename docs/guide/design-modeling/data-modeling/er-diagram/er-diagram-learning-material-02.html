draw.io の基本操作 - ER図を描く準備をしよう

無料ツールdraw.ioを使って、ER図を描くための基本操作を実践的に学びましょう。

この章で学ぶこと

  • draw.ioの起動方法と画面構成を理解する
  • 図形の配置・移動・サイズ変更などの基本操作を習得する
  • 図形の接続と線の編集方法を学ぶ
  • テキスト入力、色の変更、図の保存方法を覚える
  • ER図専用のテンプレートとシェイプライブラリの使い方を知る

2.1 draw.ioとは - 無料で使える作図ツール

draw.io（diagrams.net）は、ブラウザ上で無料で使える作図ツールです。 インストール不要で、アカウント登録なしでもすぐに使い始めることができます。

draw.ioの特徴

  • 完全無料：すべての機能を無料で利用可能
  • インストール不要：ブラウザがあればすぐに使える
  • 豊富なテンプレート：フローチャート、ER図、UMLなど多彩
  • クラウド保存対応：Google Drive、OneDriveなどと連携可能
  • 多言語対応：日本語インターフェースあり

アクセス方法

ブラウザで以下のURLにアクセスします：

URL 
https://app.diagrams.net/

または「draw.io」で検索して公式サイトにアクセスしてください。

2.2 draw.ioの起動と画面の見方

draw.ioにアクセスすると、まず保存場所の選択画面が表示されます。 初めて使う場合は「デバイス」を選択すると、ローカルにファイルを保存できます。

実習 2-1: draw.ioを起動してみよう

  1. 1. ブラウザで https://app.diagrams.net/ にアクセスする
  2. 2. 保存場所の選択画面で「デバイス」を選択する
  3. 3. 「新しい図を作成」をクリックする
  4. 4. テンプレート選択画面で「空白の図」を選択し、「作成」をクリック

draw.ioの画面構成

作図画面は以下の4つの主要なエリアで構成されています。

graph TB A[draw.io画面] --> B[左側：シェイプパネル] A --> C[中央：キャンバス] A --> D[右側：書式設定パネル] A --> E[上部：ツールバー] B --> B1["図形ライブラリ
基本図形、ER図、UMLなど"] C --> C1["作図エリア
ここに図形を配置"] D --> D1["選択図形の設定
色、線、テキストなど"] E --> E1["ファイル操作
保存、エクスポートなど"]

左側：シェイプパネル

使用できる図形の一覧が表示されます。基本図形、フローチャート、ER図など、用途に応じたライブラリを選択できます。

右側：書式設定パネル

選択した図形の色、線の太さ、テキストのフォントなど、詳細な設定を行えます。

中央：キャンバス

実際に図を描くエリアです。左側から図形をドラッグ＆ドロップして配置します。

上部：ツールバー

ファイルの保存、元に戻す/やり直す、拡大/縮小など、よく使う機能にすぐアクセスできます。

2.3 新しい図の作成とテンプレートの選択

draw.ioでは、空白の図から作成する方法と、テンプレートを使う方法があります。 ER図を描く場合は、専用のテンプレートを使うと効率的です。

実習 2-2: ER図テンプレートを使ってみよう

  1. 1. メニューバーの「ファイル」→「新規」→「テンプレートから」を選択
  2. 2. 左側のカテゴリから「Software」→「Entity Relation」を選択
  3. 3. 表示されたテンプレートの中から気に入ったものを選んで「作成」をクリック
  4. 4. テンプレートの図形を参考にしながら、自分の図を作成できます

ヒント：テンプレートの活用法

テンプレートをそのまま使う必要はありません。参考として表示させておき、必要な図形の種類や配色を確認しながら、 自分で一から作成する方が学習効果が高まります。

2.4 基本操作1: 図形の配置と移動

ER図の最も基本的な要素である「四角形（エンティティを表す）」を配置してみましょう。

実習 2-3: 四角形を配置して操作してみよう

【配置】

  1. 1. 左側のシェイプパネルで「一般」カテゴリを選択
  2. 2. 「四角形（Rectangle）」をキャンバスにドラッグ＆ドロップ

【移動】

  • 図形をクリックして選択し、そのままドラッグして移動

【サイズ変更】

  • 図形を選択すると四隅と辺の中央に青い点（ハンドル）が表示される
  • ハンドルをドラッグしてサイズを変更

【削除】

  • 図形を選択してDeleteキー、またはBackspaceキーを押す

便利なショートカット

  • Ctrl + Z（Mac: Cmd + Z）：元に戻す
  • Ctrl + Y（Mac: Cmd + Shift + Z）：やり直す
  • Ctrl + C / Ctrl + V：コピー＆ペースト
  • Ctrl + D：選択図形を複製

2.5 基本操作2: 図形の接続と線の編集

ER図では、エンティティ同士の関係を線で表現します。draw.ioでは、図形を簡単に接続できます。

実習 2-4: 2つの図形を線で接続してみよう

  1. 1. キャンバスに2つの四角形を配置する
  2. 2. 一つ目の図形をクリックすると、周囲に小さな矢印アイコンが表示される
  3. 3. 矢印アイコンをクリックし、そのまま二つ目の図形までドラッグ
  4. 4. 二つ目の図形の上でマウスを離すと、自動的に接続される

線の編集

接続した線は、様々な方法で編集できます。

線のスタイル変更

線を選択して、右側の書式設定パネルで「線」セクションを開きます。 実線、点線、太さ、色などを変更できます。

線の曲げ方

線を選択すると、途中に青い点が表示されます。 この点をドラッグすることで、線を曲げたり経路を変更できます。

矢印の追加

線を選択し、書式設定パネルの「線」セクションで、始点と終点の矢印スタイルを選択できます。

線の削除

線を選択してDeleteキーを押すと削除されます。図形を移動しても、線は自動的に追従します。

2.6 基本操作3: テキスト入力と書式設定

図形にテキストを追加して、エンティティ名や属性名を記入しましょう。

実習 2-5: 図形にテキストを入力してみよう

  1. 1. テキストを入力したい図形をダブルクリック
  2. 2. 編集モードになるので、「顧客」と入力
  3. 3. 図形の外をクリックするか、Escキーを押して編集を完了
  4. 4. 同様に、もう一つの図形に「注文」と入力

テキストの書式設定

テキストのフォント、サイズ、色、配置などを変更できます。

書式設定の手順

  1. 1. テキストを含む図形を選択
  2. 2. 右側の書式設定パネルで「テキスト」セクションを開く
  3. 3. フォントサイズ、太字、斜体、色などを設定
  4. 4. 水平配置（左寄せ、中央、右寄せ）や垂直配置も調整可能

ER図での推奨フォント設定

  • エンティティ名：14pt、太字、中央揃え
  • 属性名：11pt、標準、左揃え
  • 色：黒または濃いグレー（見やすさ優先）

2.7 基本操作4: 色の変更とスタイルの適用

図形の色を変えることで、ER図をより見やすく、分かりやすくできます。

実習 2-6: 図形の色を変更してみよう

  1. 1. 色を変更したい図形を選択
  2. 2. 右側の書式設定パネルで「スタイル」セクションを開く
  3. 3. 「塗りつぶし」のカラーボックスをクリックして、好きな色を選択
  4. 4. 「線」の色や太さも同様に変更できます

ER図での色の使い方

主要エンティティ

青系の色で塗りつぶし
（例：顧客、商品、注文など）

補助エンティティ

緑系の色で塗りつぶし
（例：カテゴリ、タグなど）

中間テーブル

黄色系の色で塗りつぶし
（例：注文明細、タグ付けなど）

注意：色の使いすぎに注意

あまり多くの色を使うと、かえって見づらくなります。 基本は2〜3色に抑え、グループや重要度を表現する程度にとどめましょう。

2.8 ER図専用のシェイプライブラリ

draw.ioには、ER図専用の図形ライブラリが用意されています。 エンティティや属性を表現する専用の図形を使うと、より本格的なER図を作成できます。

実習 2-7: ER図シェイプライブラリを追加しよう

  1. 1. 左側のシェイプパネル下部にある「その他の図形...」をクリック
  2. 2. 「Software」カテゴリを展開
  3. 3. 「Entity Relation」にチェックを入れる
  4. 4. 「適用」をクリック
  5. 5. シェイプパネルに「Entity Relation」セクションが追加される

主なER図シェイプ

1

Table（テーブル）

エンティティを表す四角形。上部にエンティティ名、下部に属性を一覧表示できる形式

2

Table with Header（ヘッダー付きテーブル）

主キーを上部に、通常の属性を下部に分けて表示できる形式

3

Relationship（リレーションシップ）

エンティティ間の関連を表す菱形。IE記法では使わないことも多い

4

Crow's Foot（鳥の足記法）

カーディナリティを表現する特殊な線。1対多、多対多などを視覚的に表現

ヒント：記法の選択

ER図には複数の記法（IE記法、Chen記法など）があります。 このガイドでは、実務でよく使われるIE記法（鳥の足記法）を中心に学習します。 draw.ioの「Entity Relation」ライブラリには、両方の記法の図形が含まれています。

2.9 図の保存とエクスポート

作成した図は、様々な形式で保存・エクスポートできます。

保存方法

実習 2-8: 図を保存してみよう

  1. 1. メニューバーの「ファイル」→「名前を付けて保存」を選択
  2. 2. ファイル名を入力（例：「顧客管理ER図」）
  3. 3. 保存場所を選択（デバイス、Google Drive、OneDriveなど）
  4. 4. 「保存」をクリック

ファイル形式は.drawioで保存されます。このファイルは、次回draw.ioで開いて編集を続けることができます。

エクスポート（画像として保存）

作成したER図を、ドキュメントやプレゼンテーションで使用する場合は、画像形式でエクスポートします。

エクスポート手順

  1. 1. メニューバーの「ファイル」→「エクスポート形式」を選択
  2. 2. PNG、JPEG、SVG、PDFなどの形式を選択（推奨：PNG）
  3. 3. 解像度やズームレベルを設定（デフォルトのままでOK）
  4. 4. 「エクスポート」をクリックして保存先を指定

PNG形式

用途：Word、PowerPoint、Webページなど
特徴：背景透過可能、汎用性が高い

SVG形式

用途：Webページ、技術ドキュメント
特徴：拡大しても劣化しないベクター形式

PDF形式

用途：印刷、正式文書
特徴：レイアウト固定、印刷に最適

JPEG形式

用途：メール添付、簡易共有
特徴：ファイルサイズが小さい

推奨：二重保存

.drawioファイル（編集用）とPNG画像（共有用）の両方を保存しておくと便利です。 後から修正が必要になったときに、.drawioファイルを開いて編集できます。

動作確認：簡単なER図を作成してみよう

これまで学んだ操作を総合して、簡単なER図を作成してみましょう。

課題：「顧客」と「注文」のER図

  1. 1. 2つの四角形を配置し、それぞれに「顧客」「注文」と入力
  2. 2. 2つの図形を線で接続する
  3. 3. 「顧客」の図形を青色、「注文」の図形を緑色に変更
  4. 4. 図を「my-first-er-diagram.drawio」という名前で保存
  5. 5. PNG形式でもエクスポート

  2. Q2. draw.ioで2つの図形を線で接続する手順を説明してください。

    回答を表示

  3. Q3. ER図を作成する際に推奨されるエクスポート形式とその理由を2つ挙げてください。

第2章のまとめ

  • draw.ioは、ブラウザで無料で使える作図ツールで、インストール不要
  • 画面はシェイプパネル、キャンバス、書式設定パネル、ツールバーの4つのエリアで構成
  • 図形の配置、移動、サイズ変更、接続、テキスト入力などの基本操作をマスターした
  • ER図専用のシェイプライブラリを追加することで、より本格的なER図を作成できる
  • 図は.drawioファイルで保存し、PNG、SVG、PDFなどの形式でエクスポート可能
  • 実務では、編集用の.drawioファイルと共有用のPNG画像の両方を保存するのが推奨

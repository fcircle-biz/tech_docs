画像を使ってみよう

写真やイラストなどの画像をデザインに取り込む方法を学びます。サイズ変更、トリミング、配置の仕方を覚えましょう。

この章で学ぶこと

  • 画像をFigmaに取り込む方法（ドラッグ&ドロップ、Fillで配置）
  • 画像のサイズを変更し、トリミング（切り抜き）する方法
  • マスク機能を使って画像を丸く切り抜く方法
  • 画像の明るさやコントラストを調整する方法
  • 無料で使える画像素材サイトと著作権の基礎知識

7.1 画像を取り込む方法

Figmaには、画像を取り込む方法が大きく分けて2つあります。どちらも簡単なので、好きな方法を使ってください。

ドラッグ&ドロップ"] A --> C["方法2:
図形のFillに設定"] B --> B1["パソコンの画像を
キャンバスに直接ドラッグ"] C --> C1["四角形を描いて
Fill設定で画像を配置"]

方法1: ドラッグ&ドロップで取り込む

最もかんたんな方法です。パソコン内の画像ファイルを、Figmaのキャンバスにドラッグ&ドロップするだけです。

実習 7-1: ドラッグ&ドロップで画像を取り込む

パソコン内の画像をFigmaに取り込んでみましょう。

手順

  1. 1 パソコン内の任意の画像ファイル（JPG、PNG、SVGなど）を用意します。
  2. 2 Figmaのキャンバス（デザイン画面）を開きます。
  3. 3 画像ファイルをキャンバスにドラッグ&ドロップします。
  4. 4 画像がキャンバス上に配置されます。

期待される結果

✅ 画像がキャンバス上に表示され、選択状態（青い枠線）になっていればOKです。

方法2: 図形のFillに画像を設定する

あらかじめ四角形などの図形を描いておき、その塗りつぶし（Fill）に画像を設定する方法です。画像のサイズや配置を細かくコントロールしたい場合に便利です。

実習 7-2: FillプロパティーのImageで画像を配置する

四角形の塗りつぶしに画像を設定してみましょう。

手順

  1. 1 四角形ツール（R）で、適当な大きさの四角形を描きます。
  2. 2 四角形を選択した状態で、右側のプロパティパネルの「Fill」セクションを見ます。
  3. 3 塗りつぶしの色の四角いボックスをクリックします。
  4. 4 カラーピッカーの上部にある「Image」ボタンをクリックします。
  5. 5 「Choose image」から画像を選択するか、画像をドラッグ&ドロップします。
  6. 6 四角形の塗りつぶしが画像に変わります。

期待される結果

✅ 四角形の中に画像が表示されていればOKです。

ヒント

Fillに画像を設定する方法は、画像の表示方法（Fill、Fit、Crop、Tile）を細かく調整できるため、デザインの自由度が高まります。

7.2 画像のサイズを変える

画像のサイズを変更する方法は、通常の図形と同じです。画像を選択すると、周りにハンドル（四隅と辺の中央にある小さな四角）が表示されます。

サイズ変更の基本操作

  • 四隅のハンドルをドラッグ: 縦横比を保ったままサイズ変更（推奨）
  • 辺のハンドルをドラッグ: 横幅または高さのみ変更（画像が歪むことがあります）
  • Shift + 四隅をドラッグ: 中心を基準にサイズ変更

重要ポイント

画像のサイズを変更するときは、四隅のハンドルを使いましょう。辺の中央のハンドルを使うと、画像が横や縦に伸びて歪んでしまいます。

実習 7-3: 画像のサイズを変更する

画像を選択して、四隅のハンドルでサイズを変えてみましょう。

手順

  1. 1 実習7-1または7-2で取り込んだ画像を選択します。
  2. 2 四隅にある小さな四角（ハンドル）のいずれかをドラッグします。
  3. 3 画像のサイズが変わります。縦横比は自動的に保たれます。

期待される結果

✅ 画像が歪まずに、大きくなったり小さくなったりすればOKです。

7.3 画像を切り抜く（トリミング）

画像の一部分だけを表示したいときは、トリミング（切り抜き）機能を使います。画像の不要な部分を隠して、必要な部分だけを表示できます。

実習 7-4: 画像をトリミングする

画像の一部分だけを表示するように切り抜いてみましょう。

手順

  1. 1 トリミングしたい画像を選択します。
  2. 2 画像をダブルクリックするか、画像を選択した状態で「Enter」キーを押します。
  3. 3 トリミングモードに入ると、画像全体が表示され、中央に枠が表示されます。
  4. 4 画像をドラッグして、表示したい部分を調整します。
  5. 5 枠の四隅をドラッグすると、枠のサイズを変更できます。
  6. 6 調整が終わったら「Enter」キーを押すか、キャンバスの空白部分をクリックして確定します。

期待される結果

✅ 画像の一部分だけが表示され、トリミングされた状態になればOKです。

ヒント：トリミングは元の画像を削除しない

トリミングしても元の画像データは削除されません。再度トリミングモードに入れば、いつでも調整し直すことができます。

7.4 画像を丸く切り抜く（マスク機能）

プロフィール画像などでよく見る、丸く切り抜かれた画像を作るには、マスク機能を使います。マスクとは、図形の形で画像を切り抜く機能のことです。

実習 7-5: 画像を丸く切り抜く

円形の図形を使って、画像を丸く切り抜いてみましょう。

手順

  1. 1 楕円ツール（O）で、Shiftキーを押しながらドラッグして、正円を描きます。
  2. 2 画像をキャンバスにドラッグ&ドロップするか、Fillで画像を配置します。
  3. 3 画像を円の上に重ねます（画像が円の上に来るように配置）。
  4. 4 画像と円の両方を選択します（Shiftキーを押しながらクリック）。
  5. 5 上部メニューの「Use as mask」（マスクとして使用）ボタンをクリックするか、Ctrl + Alt + M（Mac: Cmd + Option + M）を押します。
  6. 6 画像が円の形に切り抜かれます。

期待される結果

✅ 画像が丸く切り抜かれて表示されていればOKです。レイヤーパネル（左側）を見ると、円の中に画像が入った階層構造になっています。

flowchart TD A["マスク機能の流れ"] --> B["1. 図形を描く
（円、四角形など）"] B --> C["2. 画像を配置"] C --> D["3. 両方を選択"] D --> E["4. Use as mask"] E --> F["完成！
図形の形に切り抜かれる"]

重要ポイント：マスクの仕組み

マスクは、「下にある図形の形」で「上にある画像」を切り抜く仕組みです。円だけでなく、四角形やハート型など、どんな図形でもマスクとして使えます。

7.5 画像の明るさ・コントラストを調整する

画像が暗すぎたり明るすぎたりする場合、Figmaのエフェクト機能を使って、明るさやコントラストを調整できます。

実習 7-6: 画像の明るさを調整する

画像の明るさを調整してみましょう。

手順

  1. 1 調整したい画像を選択します。
  2. 2 右側のプロパティパネルで「Effects」セクションを見つけます。
  3. 3 「+」ボタンをクリックし、「Adjust」を選択します（最下部にあります）。
  4. 4 Adjustパネルが表示されるので、以下のパラメーターを調整してみましょう。
パラメーター 効果
Exposure 明るさ（プラスで明るく、マイナスで暗く）
Contrast コントラスト（明暗の差）
Saturation 彩度（色の鮮やかさ）
Temperature 色温度（暖色寄り/寒色寄り）

期待される結果

✅ スライダーを動かすと、リアルタイムで画像の明るさや色が変化すればOKです。

ヒント：元の画像は変更されない

Adjustエフェクトは、元の画像データを変更せずに見た目だけを調整します。いつでも元に戻したり、調整し直したりできます。

7.6 無料で使える画像素材サイトの紹介

デザインを作るとき、自分で撮影した写真がない場合は、無料の画像素材サイトを活用しましょう。以下は、商用利用可能で高品質な画像が手に入るサイトです。

Unsplash

高品質な写真が豊富。商用利用可能で、クレジット表記も不要です。

https://unsplash.com/

Pexels

写真だけでなく動画素材も豊富。すべて無料で商用利用可能です。

https://www.pexels.com/

Pixabay

写真、イラスト、ベクター画像まで揃っています。日本語検索も可能。

https://pixabay.com/

unDraw

ビジネス向けのイラスト素材。色をカスタマイズできるのが特徴です。

https://undraw.co/

ヒント：Figmaプラグインを活用しよう

FigmaにはUnsplashやPexelsの画像を直接検索・挿入できるプラグインがあります。「Resources」→「Plugins」から検索してみましょう。

7.7 著作権について知っておこう

画像を使う際に最も重要なのが著作権の理解です。他人が撮影した写真やイラストには著作権があり、勝手に使うと法律違反になることがあります。

著作権の基本ルール

  • 他人が作った画像は、許可なく使えません
  • 商用利用可能と明記されている画像のみ、ビジネス用途で使えます
  • クレジット表記が必要な画像もあります（サイトの規約を確認しましょう）
  • GoogleやSNSの画像検索で見つけた画像は、原則使えません
flowchart TD A["画像を使いたい"] --> B{どこから入手？} B -->|無料素材サイト| C["利用規約を確認"] B -->|検索エンジン| D["基本的に使用NG"] B -->|自分で撮影| E["自由に使える"] C --> F{商用利用可能？} F -->|はい| G["クレジット表記
必要か確認"] F -->|いいえ| H["個人利用のみ"] G --> I["OK！使える"]

注意：やってはいけないこと

  • Google画像検索で見つけた写真を無断で使う
  • SNS（Instagram、Twitterなど）の他人の写真を無断で使う
  • 有名キャラクターやロゴを無断で使う
  • 商用利用不可の画像を仕事で使う

安全な画像の使い方

無料素材サイト（Unsplash、Pexels、Pixabayなど）から入手した、商用利用可能な画像を使いましょう。サイトの利用規約を必ず確認してください。

理解度確認クイズ

  1. Q1. Figmaに画像を取り込む方法として正しいのはどれですか？

    回答を表示

    正解：ドラッグ&ドロップ、または図形のFillプロパティで設定
    Figmaには画像を取り込む方法が2つあります。キャンバスに直接ドラッグ&ドロップする方法と、図形を描いてFillプロパティで画像を設定する方法です。

  2. Q2. 画像のサイズを変更するとき、縦横比を保つにはどうすればよいですか？

    回答を表示

    正解：四隅のハンドルをドラッグする
    画像の四隅にあるハンドル（小さな四角）をドラッグすると、縦横比が自動的に保たれます。辺の中央のハンドルを使うと、画像が歪んでしまいます。

  3. Q3. 画像を丸く切り抜くために使う機能は何ですか？

    回答を表示

    正解：マスク機能（Use as mask）
    円形の図形と画像を重ね、両方を選択してから「Use as mask」を実行すると、画像が円の形に切り抜かれます。

  4. Q4. 画像の著作権について、正しい記述はどれですか？

    回答を表示

    正解：商用利用可能な無料素材サイトから入手した画像は安全に使える
    Unsplash、Pexels、Pixabayなどの無料素材サイトで「商用利用可能」と明記されている画像は、仕事でも安心して使えます。ただし、サイトの利用規約は必ず確認しましょう。Google検索で見つけた画像やSNSの他人の写真を無断で使うのは著作権違反です。

第7章のまとめ

  • 画像を取り込む方法はドラッグ&ドロップとFillプロパティのImage設定の2つ
  • 画像のサイズ変更は四隅のハンドルを使うと縦横比が保たれる
  • トリミングは画像をダブルクリックして調整できる（元の画像は削除されない）
  • マスク機能（Use as mask）で画像を図形の形に切り抜ける
  • Adjustエフェクトで明るさ、コントラスト、彩度を調整できる
  • 無料素材サイト（Unsplash、Pexels、Pixabayなど）から商用利用可能な画像を入手する
  • Google検索やSNSの画像を無断で使うのは著作権違反になる可能性がある

# Figma入門 学習ガイドライン

このガイドラインでは、デザインツールを使ったことがない方を対象に、Figmaの基本的な使い方をゼロからやさしく学べるカリキュラムを提供しています。「デザインはやったことがない」「絵を描くのは苦手」という方でも安心して取り組めるよう、一つひとつの操作を丁寧に解説しています。プログラミングの知識は一切必要ありません。

## 前提条件
### 必要な環境
- Webブラウザ（Google Chrome推奨、またはFirefox、Safari、Microsoft Edge）
- インターネット接続環境
- Figmaアカウント（無料プランで十分です）
- パソコン（Windows 10/11、またはMac）
- マウスまたはトラックパッド

### 参考リソース
- [Figma公式サイト](https://www.figma.com/)
- [Figma日本語ヘルプセンター](https://help.figma.com/hc/ja)
- [Figma Community（テンプレート集）](https://www.figma.com/community)

### 前提知識
- **必須**: パソコンの基本操作ができる方（マウス操作、キーボード入力）
- **必須**: Webブラウザを使ったことがある方
- **推奨**: 特になし（プログラミングやデザインの経験は不要です）

## 学習コンテンツ

### [1. Figmaを始めよう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-01.html)
Figmaとは何か、どんなことができるツールなのかを紹介します。アカウントの作り方から画面の見方まで、はじめの一歩をやさしく解説します。

**学習内容:**
- Figmaとは？ - デザインツールってなに？
- Figmaでできること（画面デザイン、イラスト、プレゼン資料など）
- 無料でどこまでできる？ - 料金プランの説明
- アカウントを作ろう
- Figmaの画面の見方（メニュー、ツールバー、キャンバス）
- 最初のファイルを作ってみよう

### [2. 図形を描いてみよう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-02.html)
Figmaの基本となる「図形ツール」の使い方を学びます。四角形、丸、線などのかんたんな図形を描きながら、基本操作を身につけます。

**学習内容:**
- 図形ツールの場所と種類
- 四角形を描いてみよう
- 丸（円・楕円）を描いてみよう
- 線を引いてみよう
- 図形の大きさを変える
- 図形を移動する
- 図形を削除する
- 元に戻す・やり直す

### [3. 色を付けてみよう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-03.html)
図形に色を付ける方法を学びます。塗りつぶしの色、線（枠）の色、透明度など、色に関する基本的な設定を覚えましょう。

**学習内容:**
- 塗りつぶしの色を変える（Fill）
- 線の色を変える（Stroke）
- 色の選び方（カラーピッカーの使い方）
- よく使う色を保存する（カラースタイル）
- 透明度を調整する
- グラデーションを付けてみよう
- 色のコードってなに？（16進数カラーコード）

### [4. 文字を入れてみよう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-04.html)
デザインに欠かせない「文字」の入力方法を学びます。フォントの変え方、サイズの調整、色の変更など、テキストの基本操作を身につけます。

**学習内容:**
- テキストツールの使い方
- 文字を入力してみよう
- フォント（書体）を変える
- 文字の大きさを変える
- 文字の色を変える
- 太字・斜体にする
- 文字の配置（左寄せ、中央、右寄せ）
- 行間・文字間隔を調整する

### [5. 図形を整理・配置しよう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-05.html)
複数の図形をきれいに並べる方法を学びます。整列、均等配置、グループ化など、デザインを整えるための重要な機能を覚えましょう。

**学習内容:**
- 複数の図形を選択する
- 図形を整列させる（上・下・左・右・中央）
- 均等に配置する
- 図形をグループ化する
- グループを解除する
- 図形の重なり順を変える（前面・背面）
- ロック機能で誤操作を防ぐ

### [6. フレームを使いこなそう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-06.html)
Figmaの重要な機能「フレーム」について学びます。スマートフォンやパソコンの画面サイズに合わせたデザインの作り方を覚えましょう。

**学習内容:**
- フレームってなに？ - デザインの枠組み
- フレームの作り方
- スマートフォン用のフレームを作る
- パソコン画面用のフレームを作る
- フレームのサイズを変更する
- フレームに名前を付ける
- フレームとグループの違い
- オートレイアウト入門

### [7. 画像を使ってみよう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-07.html)
写真やイラストなどの画像をデザインに取り込む方法を学びます。サイズ変更、トリミング、配置の仕方を覚えましょう。

**学習内容:**
- 画像を取り込む方法
- 画像のサイズを変える
- 画像を切り抜く（トリミング）
- 画像を丸く切り抜く（マスク機能）
- 画像の明るさ・コントラストを調整する
- 無料で使える画像素材サイトの紹介
- 著作権について知っておこう

### [8. コンポーネントを作ろう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-08.html)
同じパーツを何度も使い回せる「コンポーネント」機能を学びます。ボタンやアイコンなど、繰り返し使う要素を効率よく管理できます。

**学習内容:**
- コンポーネントってなに？ - 再利用できるパーツ
- コンポーネントを作る
- コンポーネントを使う（インスタンス）
- コンポーネントを編集すると全部変わる
- インスタンスだけ変更する（オーバーライド）
- バリアント（バリエーション）を作る
- コンポーネントを整理する

### [9. プロトタイプを作ろう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-09.html)
画面遷移を再現する「プロトタイプ」機能を学びます。ボタンを押すと次の画面に移動する、といった動きのあるデザインが作れます。

**学習内容:**
- プロトタイプってなに？ - 動くデザインの見本
- プロトタイプモードに切り替える
- 画面と画面をつなげる
- クリックで画面を移動させる
- アニメーション効果を付ける
- プレビューで動作を確認する
- スマートフォンで確認する方法
- プロトタイプを共有する

### [10. 他の人と一緒に作業しよう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-10.html)
Figmaの強みである「共同編集」機能を学びます。ファイルの共有方法、コメント機能、リアルタイムでの共同作業について覚えましょう。

**学習内容:**
- ファイルを他の人と共有する
- 閲覧のみ・編集可能の違い
- リアルタイムで一緒に編集する
- コメントを付ける
- コメントに返信する
- 変更履歴を確認する
- デザインをエクスポート（書き出し）する

### [11. 実践！SNS投稿画像を作ろう](https://fcircle-biz.github.io/tech_docs/guide/design-modeling/ui-ux-design/figma/figma-learning-material-11.html)
これまで学んだことを活かして、実際に使えるSNS投稿用の画像を作ってみましょう。Instagram、X（Twitter）など、各SNSに合わせたサイズで制作します。

**学習内容:**
- 学んだことのおさらい
- Instagram投稿画像を作ってみよう
- X（Twitter）ヘッダー画像を作ってみよう
- YouTubeサムネイルを作ってみよう
- テンプレートを活用しよう
- よく使うショートカットキーまとめ
- 困ったときのQ&A
- さらにFigmaを学ぶには

## 学習の進め方

1. **順番通りに進めましょう**: 各章は前の章で学んだことをもとに進みます。はじめから順番に学習してください
2. **実際に手を動かしましょう**: 読むだけでなく、必ず自分でFigmaを開いて同じ操作をしてみてください
3. **何度でも練習しましょう**: 1回でできなくても大丈夫です。わからなければ戻って何度も練習しましょう
4. **保存は自動です**: Figmaはクラウド上で動くので、作業内容は自動的に保存されます。安心して操作してください
5. **休憩を取りながら進めましょう**: 疲れたら休憩！無理せず自分のペースで進めてください

## 学習目標

このガイドを完了すると、以下のことができるようになります：

- Figmaのアカウントを作成し、基本的な画面操作ができる
- 四角形、丸、線などの基本図形を描ける
- 図形や文字に色を付け、見た目を整えられる
- テキストを入力し、フォントやサイズを変更できる
- 複数の要素をきれいに整列・配置できる
- スマートフォンやパソコン用のフレームを作成できる
- 画像を取り込み、サイズ調整やトリミングができる
- コンポーネントを作成し、効率的にデザインを管理できる
- プロトタイプを作成し、画面遷移を再現できる
- ファイルを共有し、他の人と協力して作業できる
- SNS投稿用の画像を自分で作成できる

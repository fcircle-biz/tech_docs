図形を整理・配置しよう

複数の図形をきれいに並べる方法を学びます。整列、均等配置、グループ化など、デザインを整えるための重要な機能を覚えましょう。

この章で学ぶこと

  • 複数の図形を選択する方法を理解する
  • 図形をきれいに整列させる（上・下・左・右・中央）
  • 図形を均等に配置し、グループ化する
  • 図形の重なり順を変え、ロック機能で誤操作を防ぐ

5.1 複数の図形を選択する

デザインをきれいに整えるには、まず複数の図形を選択する方法を知る必要があります。Figmaでは主に3つの選択方法があります。

選択方法1: クリックしながら追加選択

  • 1つ目の図形をクリックして選択
  • Shiftキーを押しながら他の図形をクリック
  • 複数の図形がまとめて選択されます

選択方法2: ドラッグで範囲選択

  • キャンバスの空白部分からドラッグ（マウスを押したまま引っ張る）
  • 青い選択枠に含まれた図形がすべて選択されます

選択方法3: レイヤーパネルで選択

  • 画面左側のレイヤーパネルから図形を選択
  • Shiftキーを押しながらクリックで複数選択
  • Cmdキー（Mac）/ Ctrlキー（Windows）を押しながらクリックで飛び飛びの選択も可能
flowchart TD A["複数選択の方法"] --> B["Shift + クリック"] A --> C["ドラッグで範囲選択"] A --> D["レイヤーパネルで選択"] B --> B1["1つずつ追加していく"] C --> C1["まとめて選択"] D --> D1["正確に選択できる"]

ヒント

選択を解除したいときは、キャンバスの空白部分をクリックするか、Escキーを押してください。

5.2 図形を整列させる（上・下・左・右・中央）

複数の図形を選択したら、整列機能を使ってきれいに並べることができます。プロパティパネル（画面右側）に整列ボタンが表示されます。

実習 5-1: 図形を左揃えにする

バラバラに配置された図形を、左端できれいに揃えてみましょう。

手順

  1. 1 3つの四角形を適当な位置に描きます。
  2. 2 3つすべてを選択します（Shiftキーを押しながらクリック、またはドラッグ）。
  3. 3 画面右側のプロパティパネルで「Align left」（左揃え）ボタンをクリック。
  4. 4 3つの四角形の左端がきれいに揃います。

期待される結果

✅ 3つの四角形の左端が一直線に揃っていればOKです。

主な整列機能

整列の種類 アイコン 説明
左揃え 図形の左端を揃える
水平中央揃え 図形を水平方向の中央に揃える
右揃え 図形の右端を揃える
上揃え 図形の上端を揃える
垂直中央揃え 図形を垂直方向の中央に揃える
下揃え 図形の下端を揃える

重要ポイント

整列機能は2つ以上の図形を選択していないと表示されません。1つだけ選んでいると灰色で使えなくなります。

5.3 均等に配置する

図形の間隔を均等にしたいときは、分布（Distribute）機能を使います。これを使うと、図形の間隔を自動的に同じ幅に揃えてくれます。

実習 5-2: 図形を均等に配置する

3つの丸を横一列に並べ、間隔を均等にしてみましょう。

手順

  1. 1 3つの丸（円）を横に並べます（間隔はバラバラでOK）。
  2. 2 3つすべてを選択します。
  3. 3 プロパティパネルの「Tidy up」（整頓）セクションで「Distribute horizontal spacing」（水平方向に均等配置）をクリック。
  4. 4 3つの丸の間隔が自動的に均等になります。

期待される結果

✅ 丸と丸の間隔が同じ幅になっていればOKです。

分布機能の種類

  • Distribute horizontal spacing: 横方向の間隔を均等にする
  • Distribute vertical spacing: 縦方向の間隔を均等にする
flowchart LR A["バラバラな配置"] --> B["均等配置を実行"] B --> C["間隔が揃った配置"] style A fill:#fecaca style C fill:#bbf7d0

ヒント

均等配置は、3つ以上の図形を選択していないと使えません。2つだけでは動作しないので注意しましょう。

5.4 図形をグループ化する

複数の図形を1つのまとまりとして扱いたいときは、グループ化機能を使います。グループ化すると、まとめて移動したり、コピーしたりできます。

実習 5-3: 図形をグループ化する

簡単なアイコン（例: 家の形）を作って、グループ化してみましょう。

手順

  1. 1 四角形を1つ描きます（家の壁部分）。
  2. 2 三角形を1つ描きます（屋根部分）。四角形の上に配置します。
  3. 3 2つの図形を選択します（Shiftキーを押しながらクリック）。
  4. 4 右クリックして「Group selection」（選択範囲をグループ化）を選択。または、Cmd+G（Mac）/ Ctrl+G（Windows）を押します。
  5. 5 2つの図形が1つのグループになり、まとめて移動できるようになります。

期待される結果

✅ 家の形をクリックすると、四角形と三角形がまとめて選択され、一緒に移動できればOKです。

グループ化のメリット

  • 複数の図形を1つのまとまりとして移動・コピーできる
  • 誤って図形の位置関係が崩れるのを防げる
  • デザインを整理しやすくなる

重要ポイント

グループ化した図形の中の1つだけを編集したいときは、グループをダブルクリックすると中に入れます。編集が終わったら、Escキーを押すか、外側をクリックすると抜けられます。

5.5 グループを解除する

グループ化を解除したいときは、「Ungroup」（グループ解除）機能を使います。

グループ解除の手順

  1. 解除したいグループを選択します
  2. 右クリックして「Ungroup」を選択
  3. または、Shift+Cmd+G（Mac）/ Shift+Ctrl+G（Windows）を押します
  4. グループが解除され、個別の図形に戻ります

5.6 図形の重なり順を変える（前面・背面）

図形が重なっているとき、どちらを上に表示するかを変更できます。これを重なり順（レイヤー順）と呼びます。

実習 5-4: 図形の重なり順を変更する

2つの図形を重ねて、前面・背面の順番を入れ替えてみましょう。

手順

  1. 1 青い四角形を1つ描きます。
  2. 2 赤い丸を1つ描き、青い四角形に少し重ねます。
  3. 3 赤い丸を選択します。
  4. 4 右クリックして「Send to back」（最背面へ移動）を選択。または、Cmd+[ （Mac）/ Ctrl+[ （Windows）を押します。
  5. 5 赤い丸が青い四角形の後ろに移動します。

期待される結果

✅ 赤い丸が青い四角形の下に隠れる形になっていればOKです。

重なり順の主な操作

操作 ショートカット（Mac） ショートカット（Win） 説明
最前面へ移動 Cmd+] Ctrl+] 一番前に移動
最背面へ移動 Cmd+[ Ctrl+[ 一番後ろに移動
1つ前へ移動 Cmd+Opt+] Ctrl+Alt+] 1段階前に移動
1つ後ろへ移動 Cmd+Opt+[ Ctrl+Alt+[ 1段階後ろに移動

ヒント

レイヤーパネル（画面左側）でも、図形をドラッグして重なり順を変更できます。上にあるものほど前面に表示されます。

5.7 ロック機能で誤操作を防ぐ

完成した図形をロックすることで、誤って移動したり削除したりするのを防ぐことができます。

ロックのかけ方

  1. ロックしたい図形を選択します
  2. 右クリックして「Lock」（ロック）を選択
  3. または、Cmd+Shift+L（Mac）/ Ctrl+Shift+L（Windows）を押します
  4. ロックされた図形は選択できなくなります（灰色の鍵マークが表示されます）

ロックの解除方法

  1. レイヤーパネル（画面左側）でロックされた図形を探します
  2. 図形の横にある鍵マークをクリック
  3. または、右クリックして「Unlock」を選択
  4. ロックが解除され、再び編集できるようになります
flowchart TD A["図形を選択"] --> B["ロック実行"] B --> C["編集不可"] C --> D["ロック解除"] D --> A style C fill:#fef3c7 style A fill:#bbf7d0

重要ポイント

背景や装飾など、変更したくない要素は積極的にロックしておきましょう。作業中の誤操作を防ぎ、効率的にデザインを進められます。

理解度確認クイズ

  1. Q1. 複数の図形を追加で選択するとき、押すキーはどれですか？

    回答を表示

    正解：Shiftキー
    Shiftキーを押しながら図形をクリックすると、すでに選択している図形に追加で選択できます。Cmdキー（Mac）/ Ctrlキー（Windows）を使うと、飛び飛びの選択も可能です。

  2. Q2. 図形をグループ化するショートカットキーは何ですか？

    回答を表示

    正解：Cmd+G（Mac）/ Ctrl+G（Windows）
    複数の図形を選択した状態で Cmd+G（Mac）または Ctrl+G（Windows）を押すと、グループ化できます。グループ解除は Shift+Cmd+G / Shift+Ctrl+G です。

  3. Q3. 図形を最背面に移動するショートカットキーは何ですか？

    回答を表示

    正解：Cmd+[（Mac）/ Ctrl+[（Windows）
    選択した図形を最背面に移動するには、Cmd+[ または Ctrl+[ を押します。最前面に移動する場合は Cmd+] または Ctrl+] です。

  4. Q4. ロックされた図形のロックを解除する方法として正しいものはどれですか？

    回答を表示

    正解：レイヤーパネルで鍵マークをクリック
    ロックされた図形は直接選択できないため、レイヤーパネル（画面左側）で図形を見つけ、横にある鍵マークをクリックしてロックを解除します。

第5章のまとめ

  • 複数の図形を選択するには、Shiftキー+クリック、ドラッグ、またはレイヤーパネルを使う
  • 整列機能を使うと、図形を上・下・左・右・中央などにきれいに揃えられる
  • 分布機能を使うと、図形の間隔を均等にできる（3つ以上必要）
  • グループ化（Cmd/Ctrl+G）で複数の図形をまとめて扱える
  • 重なり順は、Cmd/Ctrl+[ または Cmd/Ctrl+] で変更できる
  • ロック機能を使うと、誤操作を防げる

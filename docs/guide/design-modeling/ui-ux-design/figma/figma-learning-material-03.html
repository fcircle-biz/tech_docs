色を付けてみよう

図形に色を付ける方法を学びます。塗りつぶしの色、線（枠）の色、透明度など、色に関する基本的な設定を覚えましょう。

この章で学ぶこと

  • 図形の塗りつぶし色と線の色を変更する方法を理解する
  • カラーピッカーを使った直感的な色選択方法を習得する
  • カラースタイルを保存し、よく使う色を管理する
  • 透明度の調整とグラデーションの適用方法を学ぶ

3.1 塗りつぶしの色を変える（Fill）

Figmaで図形に色を付ける最も基本的な方法は、Fill（フィル）という機能を使うことです。Fillは「塗りつぶし」を意味し、図形の内側の色を変更できます。

実習 3-1: 四角形の塗りつぶし色を変える

前の章で作成した四角形に色を付けてみましょう。

手順

  1. 1 左側のツールバーから「四角形ツール（R）」を選択し、キャンバスに四角形を1つ描きます。
  2. 2 四角形を選択した状態で、画面右側のプロパティパネルを確認します。
  3. 3 プロパティパネル内の「Fill」という項目を探します。デフォルトではグレーの小さな四角（カラーチップ）が表示されています。
  4. 4 このカラーチップをクリックすると、カラーピッカー（色を選ぶパネル）が開きます。
  5. 5 カラーピッカー内の色のエリアをクリックして、好きな色を選びます。青、赤、緑など、いろいろな色を試してみましょう。

期待される結果

✅ 四角形の内側の色が、選んだ色に変わっていればOKです！

Fill（塗りつぶし）とは？

Fillは図形の「内側」を塗りつぶす色です。四角形、丸、多角形など、すべての図形に適用できます。デフォルトでは薄いグレーに設定されていますが、カラーピッカーで自由に変更できます。

3.2 線の色を変える（Stroke）

図形の「内側」だけでなく、「枠（アウトライン）」の色も変更できます。これをStroke（ストローク）と呼びます。

実習 3-2: 図形に枠線を追加して色を変える

四角形に枠線を追加し、その色を変更してみましょう。

手順

  1. 1 四角形を選択した状態で、右側のプロパティパネルを確認します。
  2. 2 「Fill」の下に「Stroke」という項目があります。まだ何も設定されていない場合は、「＋」マークをクリックして追加します。
  3. 3 Strokeのカラーチップをクリックして、カラーピッカーから枠線の色を選びます。
  4. 4 枠線の太さも変更できます。Strokeの右側にある数字（デフォルトは1）をクリックして、「3」や「5」など大きな数字にしてみましょう。

期待される結果

✅ 四角形の周りに色のついた枠線が表示されればOKです！

3.3 色の選び方（カラーピッカーの使い方）

Figmaのカラーピッカーには、色を選ぶための便利な機能がいくつかあります。詳しく見ていきましょう。

カラーピッカーの構成

  • ① 色相バー（縦長のカラフルなバー） - 赤、青、緑などの基本色を選びます
  • ② 明度・彩度エリア（大きな四角いエリア） - 選んだ色の明るさや鮮やかさを調整します
  • ③ 不透明度スライダー（Opacity） - 色の透明度を0%〜100%で調整します
  • ④ カラーコード入力欄 - 16進数カラーコードを直接入力できます

ヒント：色の選び方のコツ

まず縦長の色相バーで「赤系」「青系」など大まかな色を選び、次に四角いエリアで明るさや鮮やかさを調整すると、きれいに色を選べます。

動作確認: いろいろな色を試してみよう

カラーピッカーを使って、以下の色を試してみましょう。

  • 明るい青色
  • 濃い赤色
  • 淡い黄色
  • 紫色

✅ いろいろな色を試すことで、カラーピッカーの使い方に慣れましょう！

3.4 よく使う色を保存する（カラースタイル）

デザインでは、同じ色を何度も使うことがよくあります。そんなときに便利なのがカラースタイル機能です。よく使う色を保存しておけば、毎回カラーピッカーで探す手間が省けます。

実習 3-3: カラースタイルを作成する

自分の好きな色をカラースタイルとして保存してみましょう。

手順

  1. 1 四角形を選択し、Fillのカラーチップをクリックしてカラーピッカーを開きます。
  2. 2 好きな色を選んだら、カラーピッカーの右上にある「スタイルアイコン（4つの丸が並んだアイコン）」をクリックします。
  3. 3 「＋」マークをクリックして「Create style」（スタイルを作成）を選びます。
  4. 4 スタイルに名前を付けます（例: 「メインブルー」「アクセントレッド」など）。
  5. 5 「Create style」ボタンを押して保存します。

期待される結果

✅ 次回から、カラーピッカーのスタイル一覧に保存した色が表示されます！

重要ポイント

カラースタイルを使うと、デザイン全体で色を統一しやすくなります。また、後から色を変更したいときも、スタイルを編集すれば、そのスタイルを使っているすべての図形の色が一括で変わります。

3.5 透明度を調整する

図形の色を半透明にすることもできます。これを不透明度（Opacity）の調整と呼びます。

実習 3-4: 図形を半透明にする

図形の透明度を調整して、下の図形が透けて見えるようにしてみましょう。

手順

  1. 1 キャンバスに2つの四角形を描き、少し重なるように配置します。
  2. 2 上側の四角形を選択します。
  3. 3 右側のプロパティパネルで「Opacity」（不透明度）の数値を探します。デフォルトは100%です。
  4. 4 Opacityの数値をクリックして、「50」と入力してみましょう（50%透明になります）。
  5. 5 上の図形が半透明になり、下の図形が透けて見えます。

期待される結果

✅ 上側の四角形が半透明になり、下の図形が透けて見えればOKです！

不透明度の数値について

不透明度は0%〜100%の範囲で指定できます。
100% = 完全に不透明（透けない）
50% = 半透明
0% = 完全に透明（見えなくなる）

3.6 グラデーションを付けてみよう

単色ではなく、色が徐々に変化するグラデーションを適用することもできます。グラデーションを使うと、デザインに奥行きや華やかさが生まれます。

実習 3-5: 図形にグラデーションを適用する

四角形に2色のグラデーションを付けてみましょう。

手順

  1. 1 四角形を選択し、右側のプロパティパネルでFillのカラーチップをクリックします。
  2. 2 カラーピッカーの上部に「Solid」（単色）というドロップダウンがあります。これをクリックします。
  3. 3 「Linear」（線形グラデーション）を選択します。
  4. 4 グラデーションバーが表示されます。左右にある「●」をクリックして、それぞれの色を変更できます。
  5. 5 左側を青、右側を紫にしてみましょう。

期待される結果

✅ 四角形が青から紫へ徐々に変化するグラデーションになっていればOKです！

グラデーションの種類

種類 説明
Linear（線形） 色が直線的に変化します。左から右、上から下など、方向を変更できます。
Radial（円形） 中心から外側に向かって円状に色が変化します。
Angular（角度） 中心を軸に回転するように色が変化します。
Diamond（ひし形） ひし形状に色が広がります。

ヒント：グラデーションの方向変更

グラデーションを適用した図形には、キャンバス上に「グラデーションハンドル」が表示されます。これをドラッグすると、グラデーションの方向や範囲を視覚的に調整できます。

3.7 色のコードってなに？（16進数カラーコード）

デザインの世界では、色を「コード」で表現することがよくあります。これを16進数カラーコードと呼びます。

カラーコードとは？

カラーコードは、色を「#」で始まる6桁の英数字で表現したものです。例えば：

  • #FF0000 = 赤
  • #0000FF = 青
  • #00FF00 = 緑
  • #FFFFFF = 白
  • #000000 = 黒
  • #808080 = グレー

カラーコードの使い方

Figmaのカラーピッカーでは、カラーコード入力欄があり、ここに直接コードを入力することで、正確な色を指定できます。

カラーコードを覚える必要はありません

初心者のうちは、カラーピッカーで直感的に色を選べば十分です。ただし、デザイナーや開発者と協力するときに「この色は #3B82F6 を使ってください」と指定されることがあるので、「色にはコードがある」ということだけ覚えておきましょう。

理解度確認クイズ

  1. Q1. 図形の内側の色を変更するための機能は何ですか？

    回答を表示

    正解：Fill（塗りつぶし）
    Fillは図形の内側を塗りつぶす色を設定する機能です。右側のプロパティパネルから変更できます。

  2. Q2. 図形の枠線の色や太さを変更する機能は何ですか？

    回答を表示

    正解：Stroke（ストローク）
    Strokeは図形の周りの枠線を設定する機能です。色だけでなく、線の太さも調整できます。

  3. Q3. よく使う色を保存しておく機能を何と呼びますか？

    回答を表示

    正解：カラースタイル
    カラースタイルを作成すると、同じ色を何度も使いたいときに便利です。スタイルを編集すれば、その色を使っているすべての図形が一括で変更されます。

  4. Q4. 図形を半透明にするには、何の数値を変更すればよいですか？

    回答を表示

    正解：Opacity（不透明度）
    Opacityを100%から下げると、図形が半透明になります。0%にすると完全に透明になります。

第3章のまとめ

  • Fill（塗りつぶし）で図形の内側の色を変更できる
  • Stroke（枠線）で図形の周りの線の色と太さを変更できる
  • カラーピッカーを使って色相、明度、彩度を調整して色を選べる
  • カラースタイルを作成すると、よく使う色を保存して再利用できる
  • Opacity（不透明度）を調整すると、図形を半透明にできる
  • グラデーション（Linear、Radial、Angular、Diamond）を適用できる
  • 色を16進数カラーコード（例: #FF0000）で指定できる

文字を入れてみよう

デザインに欠かせない「文字」の入力方法を学びます。フォントの変え方、サイズの調整、色の変更など、テキストの基本操作を身につけます。

この章で学ぶこと

  • テキストツールの使い方を理解し、文字を入力できるようになる
  • フォント（書体）の変更方法を習得する
  • 文字の大きさ、色、スタイル（太字・斜体）を調整できるようになる
  • 文字の配置（左寄せ、中央、右寄せ）や行間・文字間隔を調整できるようになる

4.1 テキストツールの使い方

デザインに文字を入れるには、テキストツールを使います。左側のツールバーから「T」のアイコンを探してください。

flowchart LR A["左側ツールバー"] --> B["Tアイコンを
クリック"] B --> C["キャンバスを
クリック"] C --> D["文字入力
開始！"]

ショートカットキー

キーボードの「T」キーを押すだけでも、テキストツールに切り替わります。慣れてきたらこちらの方が速いです。

4.2 文字を入力してみよう

それでは実際に文字を入力してみましょう。

実習 4-1: 初めての文字入力

Figmaで初めて文字を入力してみましょう。

手順

  1. 1 左側のツールバーから「T」アイコンをクリックするか、キーボードの「T」キーを押します。
  2. 2 キャンバスの好きな場所をクリックします。
  3. 3 「こんにちは」と入力してみてください。
  4. 4 入力が終わったら、キャンバスの別の場所をクリックするか、左上の移動ツール（矢印アイコン）をクリックします。

期待される結果

✅ キャンバス上に「こんにちは」という文字が表示されればOKです！

ヒント：テキスト編集モードを終了する方法

文字の入力が終わったら、Escキーを押すか、移動ツール（V）をクリックすることで、テキスト編集モードを終了できます。

4.3 フォント（書体）を変える

入力した文字のフォント（書体）を変更してみましょう。フォントを変えると、文字の印象がガラッと変わります。

フォントの変更は、右側のプロパティパネルにある「Text」セクションで行います。

実習 4-2: フォントを変更する

先ほど入力した「こんにちは」のフォントを変更してみましょう。

手順

  1. 1 移動ツール（V）で、先ほど入力した「こんにちは」のテキストをクリックして選択します。
  2. 2 右側のプロパティパネルの「Text」セクションを確認します。
  3. 3 フォント名が表示されているドロップダウンをクリックします（初期設定では「Inter」などになっています）。
  4. 4 検索ボックスに「Noto Sans JP」と入力して、日本語フォントを選択してみましょう。

期待される結果

✅ 文字の書体が変わって、より読みやすい日本語フォントになります！

日本語フォントのおすすめ

Figmaには最初から多くのフォントが入っていますが、日本語デザインには以下のフォントがおすすめです：

  • Noto Sans JP: 読みやすいゴシック体
  • Noto Serif JP: 落ち着いた明朝体
  • M PLUS Rounded 1c: かわいい丸ゴシック体

4.4 文字の大きさを変える

文字の大きさ（フォントサイズ）を調整する方法を学びましょう。タイトルには大きな文字、本文には小さめの文字を使うと、デザインにメリハリが出ます。

flowchart TD A["文字サイズの使い分け"] --> B["大きい文字
24〜48px"] A --> C["中くらい文字
16〜20px"] A --> D["小さい文字
10〜14px"] B --> B1["タイトル・見出し"] C --> C1["本文・説明文"] D --> D1["注釈・補足情報"]

実習 4-3: 文字サイズを変更する

テキストを選択して、文字サイズを大きくしたり小さくしたりしてみましょう。

手順

  1. 1 テキストを選択します。
  2. 2 右側のプロパティパネルで、フォント名の隣にある数字（フォントサイズ）を確認します。
  3. 3 数字をクリックして「48」と入力し、Enterキーを押します。
  4. 4 文字が大きくなったことを確認してください。

期待される結果

✅ 文字が48pxサイズに拡大されます。数字を変えれば自由にサイズを調整できます！

ヒント：数値の増減ボタン

フォントサイズの数字にカーソルを合わせ、左右にドラッグすることでも、サイズを調整できます。細かい調整に便利です。

4.5 文字の色を変える

文字の色を変える方法は、図形に色を付けるときと同じです。右側のプロパティパネルにあるFill（塗りつぶし）セクションを使います。

実習 4-4: 文字の色を変更する

テキストの色を青色に変更してみましょう。

手順

  1. 1 テキストを選択します。
  2. 2 右側のプロパティパネルで「Fill」セクションを探します。
  3. 3 色の四角いアイコンをクリックして、カラーピッカーを開きます。
  4. 4 青色の範囲をクリックして色を選びます。

期待される結果

✅ 文字の色が青色に変わります！図形と同じように、好きな色に変更できます。

4.6 太字・斜体にする

文字を太字（ボールド）や斜体（イタリック）にすることで、重要な部分を強調できます。

太字や斜体の設定は、右側のプロパティパネルの「Text」セクションにあるスタイルボタンで行います。

実習 4-5: 太字と斜体を試す

テキストを太字にしたり、斜体にしたりしてみましょう。

手順

  1. 1 テキストを選択します。
  2. 2 右側のプロパティパネルの「Text」セクションで、フォント名の右側にあるウェイト（Weight）のドロップダウンを探します。
  3. 3 「Bold」を選択すると太字になります。「Black」を選ぶとさらに太くなります。
  4. 4 元に戻すには「Regular」を選びます。

期待される結果

✅ 文字が太字になります。フォントによって選べるウェイト（太さ）の種類が異なります。

フォントウェイトについて

フォントの太さは通常、Light（細い）→ Regular（通常）→ Medium（やや太い）→ Bold（太字）→ Black（極太）のように段階があります。選べるウェイトはフォントによって異なります。

4.7 文字の配置（左寄せ、中央、右寄せ）

文字を左寄せ、中央揃え、右寄せにする方法を学びましょう。文字の配置を変えることで、デザインの印象が変わります。

flowchart TD A["テキスト配置"] --> B["左寄せ"] A --> C["中央揃え"] A --> D["右寄せ"] B --> B1["一般的な本文に使う"] C --> C1["タイトルや見出しに使う"] D --> D1["日付や署名に使う"]

実習 4-6: テキストを中央揃えにする

テキストを中央揃えにして、タイトルっぽく見せてみましょう。

手順

  1. 1 テキストを選択します。
  2. 2 右側のプロパティパネルの「Text」セクションを見ます。
  3. 3 3つの横線アイコンが並んでいる部分を探します（左寄せ、中央、右寄せのボタン）。
  4. 4 真ん中のアイコン（中央揃え）をクリックします。

期待される結果

✅ 文字が中央に配置されます。タイトルや見出しっぽくなります！

4.8 行間・文字間隔を調整する

文字が詰まりすぎていたり、逆に離れすぎていたりすると読みにくくなります。行間（Line height）と文字間隔（Letter spacing）を調整して、読みやすくしましょう。

行間（Line height）

行間とは、文字の行と行の間のスペースのことです。行間を広げると文章が読みやすくなります。

文字間隔（Letter spacing）

文字間隔とは、文字と文字の間のスペースのことです。文字間隔を調整することで、見た目の印象を変えることができます。

実習 4-7: 行間を調整する

複数行のテキストを作成して、行間を調整してみましょう。

手順

  1. 1 テキストツール（T）で、複数行の文章を入力します（例：「これは1行目です」Enterキー「これは2行目です」）。
  2. 2 テキストを選択して、右側のプロパティパネルを確認します。
  3. 3 「Line height」という項目を探します（Autoと表示されている場合が多いです）。
  4. 4 数値をクリックして「150%」と入力し、Enterキーを押します。

期待される結果

✅ 行と行の間が広がって、文章が読みやすくなります！

ヒント：行間の目安

一般的に、本文の行間は140%〜160%くらいが読みやすいとされています。タイトルは120%〜140%くらいが適切です。

文字間隔の調整

文字間隔（Letter spacing）も同じプロパティパネルで調整できます。数値を大きくすると文字が離れ、小さくすると詰まります。タイトルなどで文字を少し離して見せたいときに使います。

理解度確認クイズ

  1. Q1. テキストツールを起動するショートカットキーは何ですか？

    回答を表示

    正解：Tキー
    キーボードの「T」キーを押すことで、すぐにテキストツールに切り替わります。Text（テキスト）の頭文字ですね。

  2. Q2. 日本語デザインに適したフォントとして紹介されたものはどれですか？

    回答を表示

    正解：Noto Sans JP、Noto Serif JP、M PLUS Rounded 1c
    これらはGoogleが提供している無料の日本語フォントで、Figmaでも使えます。Noto Sans JPはゴシック体、Noto Serif JPは明朝体、M PLUS Rounded 1cは丸ゴシック体です。

  3. Q3. 文字を太字にするには、どの設定を変更しますか？

    回答を表示

    正解：フォントウェイト（Weight）をBoldに変更する
    右側のプロパティパネルの「Text」セクションにあるウェイト設定で「Bold」を選ぶと、文字が太字になります。

  4. Q4. 本文の行間として読みやすい目安は何％ですか？

    回答を表示

    正解：140%〜160%
    本文の行間は140%〜160%くらいが読みやすいとされています。行間が狭すぎると窮屈で読みにくく、広すぎると文章のまとまりが感じられなくなります。

  • フォントサイズを調整して、タイトルと本文でメリハリをつけることができる
  • 文字の色は、Fillセクションで変更できる（図形と同じ方法）
  • フォントウェイトを変更することで、太字や細字にできる
  • テキストの配置（左寄せ、中央、右寄せ）を変更できる
  • 行間（Line height）と文字間隔（Letter spacing）を調整して、読みやすさを改善できる

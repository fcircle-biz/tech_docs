Figmaを始めよう

Figmaとは何か、どんなことができるツールなのかを紹介します。アカウントの作り方から画面の見方まで、はじめの一歩をやさしく解説します。

この章で学ぶこと

  • Figmaがどんなツールで、何ができるかを理解する
  • 無料プランでできることと、料金プランの違いを知る
  • Figmaアカウントを作成し、初めてのファイルを作成する
  • Figmaの画面構成（ツールバー、キャンバス、パネル）の基本を理解する

1.1 Figmaとは？ - デザインツールってなに?

Figma（フィグマ）は、Webブラウザで動くデザインツールです。デザインツールとは、画面のレイアウトやイラスト、プレゼン資料などを作るためのソフトウェアのことです。

従来のデザインツールは、パソコンにソフトウェアをインストールする必要がありましたが、Figmaはブラウザ上で動作するため、インストール不要ですぐに使い始めることができます。

ブラウザで動くとは？

Google ChromeやSafari、Firefoxなど、普段Webサイトを見るときに使っているソフトウェア（ブラウザ）で直接Figmaを開いて使えるという意味です。ソフトウェアのインストールが不要なので、どのパソコンからでもアクセスできます。

1.2 Figmaでできること

Figmaは主に以下のようなことに使われています。初心者の方は「こんなこともできるんだ」という感覚でまずは眺めてみてください。

Webサイト・アプリの画面デザイン

スマートフォンアプリやWebサイトの画面レイアウトを作成できます。ボタンの配置、色、文字の大きさなど、細かく調整できます。

SNS投稿画像・バナー制作

InstagramやTwitterの投稿画像、YouTubeのサムネイル、Webサイトのバナー広告などを作ることができます。

イラスト・アイコン制作

シンプルなイラストやアイコンを描くこともできます。ベクター形式なので、拡大しても画質が劣化しません。

プレゼン資料作成

PowerPointのようなスライド資料も作成できます。視覚的に美しい資料を簡単に作れます。

重要ポイント

Figmaは「デザインツール」ですが、プログラミングの知識は一切不要です。マウスで図形を描いたり、色を選んだりするだけで、誰でもデザインを作ることができます。

1.3 無料でどこまでできる？ - 料金プランの説明

Figmaには無料プランと有料プランがありますが、初心者の学習では無料プランで十分です。

1.4 アカウントを作ろう

それでは実際にFigmaのアカウントを作成してみましょう。手順はとても簡単です。

実習 1-1: Figmaアカウントの作成

Figmaの公式サイトにアクセスして、無料アカウントを作成します。

手順

  1. 1 Webブラウザで https://www.figma.com/ にアクセスします。
  2. 2 右上にある「Sign up」（サインアップ）ボタンをクリックします。
  3. 3 メールアドレスを入力するか、GoogleアカウントまたはApple IDでサインアップできます。（Googleアカウント連携が簡単でおすすめです）
  4. 4 メールアドレスで登録した場合は、届いた確認メール内のリンクをクリックして認証します。
  5. 5 名前や職種などのプロフィール情報を入力します（スキップしても大丈夫です）。
  6. 6 アカウント作成完了！Figmaのホーム画面（ダッシュボード）が表示されます。

期待される結果

✅ Figmaのホーム画面が表示され、「Drafts」というセクションが見えていればOKです。

注意

Figmaは英語表示がデフォルトですが、ほとんどの機能はアイコンで理解できるので、英語が苦手でも問題ありません。このガイドでは各機能を日本語で丁寧に説明していきます。

1.5 Figmaの画面の見方（メニュー、ツールバー、キャンバス）

Figmaの画面は大きく3つのエリアに分かれています。それぞれの役割を理解しましょう。

flowchart TD A["Figma画面"] --> B["左側: ツールバー"] A --> C["中央: キャンバス"] A --> D["右側: プロパティパネル"] B --> B1["図形ツール"] B --> B2["ペンツール"] B --> B3["テキストツール"] C --> C1["実際にデザインを
作る場所"] D --> D1["色・サイズ調整"] D --> D2["フォント設定"]

① 左側：ツールバー

画面左側には、図形を描いたり文字を入力したりするためのツールが並んでいます。主なツールは次の通りです：

  • 移動ツール（V）: オブジェクトを選択・移動します
  • フレームツール（F）: スマホやパソコンの画面サイズを設定します
  • 図形ツール（R、O、L）: 四角形、丸、線などを描きます
  • テキストツール（T）: 文字を入力します
  • ペンツール（P）: 自由な形を描きます（少し難しいので後半で学びます）

② 中央：キャンバス

画面中央の広い白いエリアがキャンバスです。ここが実際にデザインを作る場所です。自由に図形を置いたり、文字を入力したりできます。

③ 右側：プロパティパネル

画面右側には、選択した図形やテキストの色、サイズ、フォントなどを細かく調整するパネルがあります。何かを選択すると、ここに設定項目が表示されます。

ショートカットキーについて

ツールの隣に表示されている「V」「F」「T」などは、キーボードのショートカットキーです。例えば「T」キーを押すとテキストツールに切り替わります。最初は覚えなくても大丈夫ですが、慣れてくると作業が速くなります。

1.6 最初のファイルを作ってみよう

それでは実際に、Figmaで初めてのデザインファイルを作成してみましょう。

実習 1-2: 新しいデザインファイルを作成する

Figmaのホーム画面から新しいデザインファイルを作成してみましょう。

手順

  1. 1 Figmaのホーム画面（ダッシュボード）で、「Drafts」セクションを確認します。
  2. 2 画面右上の「+ New design file」（新しいデザインファイル）ボタンをクリックします。
  3. 3 Figmaのエディター画面（デザイン編集画面）が開きます。真っ白なキャンバスが表示されればOKです。
  4. 4 画面上部のファイル名（初期状態では「Untitled」など）をクリックして、「はじめてのFigma」など好きな名前に変更してみましょう。

期待される結果

✅ 真っ白なキャンバスが表示され、左にツールバー、右にプロパティパネルが見えていればOKです。ファイル名も変更できていれば完璧です！

重要ポイント：自動保存機能

Figmaは自動的に保存されます。「保存」ボタンを押す必要はありません。デザインを変更するたびに、クラウド上に自動的に保存されるので、データが消える心配もありません。

理解度確認クイズ

  1. Q1. Figmaの特徴として正しいものはどれですか？

    回答を表示

    正解：Webブラウザで動作し、インストール不要で使える
    Figmaの最大の特徴は、Webブラウザ上で動作するため、ソフトウェアのインストールが不要な点です。インターネット接続があれば、どのパソコンからでもアクセスできます。

  2. Q2. 無料プラン（Starter）でできないことは何ですか？

    回答を表示

    正解：Draftファイルを4つ以上作ること
    無料プランでは、Draftファイルは3つまでしか作成できません。ただし、学習には十分な制限であり、他の主要機能（共同編集、プラグイン利用など）は無料でも使えます。

  3. Q3. Figma画面の中央にある白いエリアを何と呼びますか？

    回答を表示

    正解：キャンバス
    画面中央の広い白いエリアは「キャンバス」と呼ばれ、実際にデザインを作成する場所です。図形を配置したり、文字を入力したりします。

  4. Q4. Figmaで作業中、データを保存するためにどうすればよいですか？

    回答を表示

    正解：何もしなくてOK（自動保存される）
    Figmaは変更内容を自動的にクラウド上に保存します。「保存」ボタンを押す必要はありません。安心して作業を進めてください。

第1章のまとめ

  • Figmaは、Webブラウザで動作するデザインツールで、インストール不要で使える
  • 画面デザイン、SNS投稿画像、イラスト、プレゼン資料など、幅広い用途に使える
  • 無料プランで学習には十分な機能が使える（Draftファイル3つまで）
  • 画面は左：ツールバー、中央：キャンバス、右：プロパティパネルの3つで構成される
  • データは自動保存されるので、保存ボタンを押す必要はない

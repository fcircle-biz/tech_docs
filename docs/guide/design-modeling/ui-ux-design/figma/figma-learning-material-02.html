図形を描いてみよう

Figmaの基本となる「図形ツール」の使い方を学びます。四角形、丸、線などのかんたんな図形を描きながら、基本操作を身につけます。

この章で学ぶこと

  • 図形ツールの場所と種類を理解する
  • 四角形、丸（円・楕円）、線を描けるようになる
  • 図形の大きさを変更したり、移動・削除できるようになる
  • 元に戻す・やり直す操作を理解する

2.1 図形ツールの場所と種類

Figmaで図形を描くには、画面左側のツールバーにある図形ツールを使います。

図形ツールには、以下のような種類があります：

四角形ツール（R）

長方形や正方形を描くツールです。ボタンやカードなど、デザインの基本となる形を作れます。

円形ツール（O）

円や楕円を描くツールです。アイコン、ボタン、装飾などに使います。

線ツール（L）

まっすぐな線を引くツールです。区切り線や矢印などに使えます。

移動ツール（V）

図形を選択したり移動したりするツールです。描いた図形を選ぶときに使います。

ショートカットキーについて

ツール名の横にある（R）や（O）などは、キーボードのショートカットキーです。例えば「R」キーを押すと四角形ツールに切り替わります。最初は覚えなくても大丈夫ですが、慣れると作業が速くなります。

2.2 四角形を描いてみよう

それでは実際に四角形を描いてみましょう。四角形はデザインの基本となる図形です。

実習 2-1: 四角形を描く

四角形ツールを使って、キャンバスに長方形を描いてみましょう。

手順

  1. 1 ツールバーの四角形アイコンをクリックするか、キーボードの「R」キーを押します。
  2. 2 キャンバス上の好きな場所でマウスをクリックしたままにします。
  3. 3 マウスを押したまま斜め下にドラッグします。四角形が表示されます。
  4. 4 好きな大きさになったらマウスから手を離すと、四角形が確定します。

期待される結果

✅ キャンバス上にグレー色の長方形が表示されます。図形の周りには小さな四角い点（ハンドル）が表示されています。

ヒント：正方形を描く方法

四角形を描くとき、キーボードのShiftキーを押しながらドラッグすると、縦と横が同じ長さの正方形になります。

2.3 丸（円・楕円）を描いてみよう

次は円形ツールを使って、丸を描いてみましょう。円もデザインでよく使う図形です。

実習 2-2: 円を描く

円形ツールを使って、キャンバスに円を描いてみましょう。

手順

  1. 1 ツールバーの円形アイコンをクリックするか、キーボードの「O」キーを押します。
  2. 2 四角形のときと同じように、クリック＆ドラッグで円を描きます。
  3. 3 Shiftキーを押しながらドラッグすると、縦横が同じ長さの真円になります。
  4. 4 マウスから手を離すと円が確定します。

期待される結果

✅ キャンバス上にグレー色の円（または楕円）が表示されます。

円と楕円の違い

円は縦横が同じ長さの丸（Shiftキーを押しながら描く）、楕円は縦横の長さが違う丸（Shiftキーなしで自由に描く）です。

2.4 線を引いてみよう

線ツールを使って、まっすぐな線を引いてみましょう。区切り線や矢印として使えます。

実習 2-3: 線を引く

線ツールを使って、水平線や斜めの線を引いてみましょう。

手順

  1. 1 ツールバーの線アイコンをクリックするか、キーボードの「L」キーを押します。
  2. 2 線の始点でクリックします。
  3. 3 マウスを押したまま線の終点までドラッグします。
  4. 4 マウスから手を離すと線が引かれます。

期待される結果

✅ キャンバス上にグレー色の線が引かれます。

ヒント：まっすぐな線を引く方法

線を引くとき、Shiftキーを押しながらドラッグすると、水平・垂直・45度のまっすぐな線が引けます。

2.5 図形の大きさを変える

描いた図形の大きさを後から変更できます。これはリサイズと呼ばれる操作です。

実習 2-4: 図形の大きさを変更する

先ほど描いた四角形の大きさを変更してみましょう。

手順

  1. 1 まず、移動ツール（V）に切り替えます。キーボードの「V」キーを押すか、ツールバーの矢印アイコンをクリックします。
  2. 2 大きさを変えたい図形をクリックして選択します。図形の周りに8つの小さな四角（ハンドル）が表示されます。
  3. 3 四隅にあるハンドルのどれか1つにマウスカーソルを合わせます。
  4. 4 ハンドルをクリックしたままドラッグすると、図形の大きさが変わります。
  5. 5 好きな大きさになったらマウスから手を離します。

期待される結果

✅ 図形の大きさが変わります。比率を保ちたい場合は、Shiftキーを押しながらハンドルをドラッグしてください。

2.6 図形を移動する

描いた図形を別の場所に移動させることができます。

実習 2-5: 図形を移動する

描いた図形を別の場所に移動させてみましょう。

手順

  1. 1 移動ツール（V）が選択されていることを確認します。
  2. 2 移動したい図形をクリックして選択します。
  3. 3 図形の上にマウスカーソルを置き、クリックしたままドラッグします。
  4. 4 好きな場所でマウスから手を離すと、そこに図形が移動します。

期待される結果

✅ 図形が新しい場所に移動します。

ヒント：細かい移動はキーボードで

図形を選択した状態で矢印キー（↑↓←→）を押すと、1ピクセルずつ細かく移動できます。Shiftキーを押しながら矢印キーを押すと、10ピクセルずつ移動します。

2.7 図形を削除する

不要な図形を削除する方法を学びましょう。

実習 2-6: 図形を削除する

不要な図形を削除してみましょう。

手順

  1. 1 削除したい図形をクリックして選択します（移動ツールを使います）。
  2. 2 キーボードのDeleteキーまたはBackspaceキーを押します。
  3. 3 図形がキャンバスから消えます。

期待される結果

✅ 選択した図形がキャンバスから削除されます。

注意：削除は慎重に

間違って削除してしまった場合でも大丈夫です。次のセクションで学ぶ「元に戻す」機能で復元できます。

2.8 元に戻す・やり直す

Figmaには元に戻す（アンドゥ）とやり直す（リドゥ）機能があります。間違えても安心して作業できます。

操作 Windows Mac
元に戻す Ctrl + Z ⌘ + Z
やり直す Ctrl + Shift + Z ⌘ + Shift + Z

実習 2-7: 元に戻す・やり直すを試す

元に戻す・やり直す機能を実際に使ってみましょう。

手順

  1. 1 適当に図形を1つ描いてください。
  2. 2 Ctrl + Z（Macは⌘ + Z）を押してみてください。図形が消えます。
  3. 3 Ctrl + Shift + Z（Macは⌘ + Shift + Z）を押してみてください。図形が戻ってきます。
  4. 4 何度かCtrl + Zを押して、過去の操作を遡ってみましょう。

期待される結果

✅ 操作を1つずつ戻したり、やり直したりできることを確認できます。

重要ポイント

元に戻す機能は、Figmaで最もよく使う機能の1つです。間違いを恐れず、どんどん試してみましょう。失敗してもCtrl + Zで元に戻せます！

理解度確認クイズ

  1. Q1. 四角形ツールのショートカットキーは何ですか？

    回答を表示

    正解：R（アール）キー
    キーボードの「R」キーを押すと、四角形ツールに切り替わります。Rectangleの頭文字です。

  2. Q2. 真円を描くにはどうすればよいですか？

    回答を表示

    正解：Shiftキーを押しながら円形ツールでドラッグする
    Shiftキーを押しながらドラッグすると、縦横の比率が1:1に固定され、真円（正円）が描けます。

  3. Q3. 図形を細かく移動させるにはどうすればよいですか？

    回答を表示

    正解：図形を選択して矢印キー（↑↓←→）を押す
    矢印キーを押すと1ピクセルずつ移動できます。Shiftキーを押しながら矢印キーを押すと、10ピクセルずつ移動します。

  4. Q4. 操作を元に戻すショートカットキーは何ですか（Windows）？

    回答を表示

    正解：Ctrl + Z
    Ctrl + Zで1つ前の操作に戻れます。何度も押すと、さらに過去の操作まで遡れます。Macの場合は⌘ + Zです。

第2章のまとめ

  • Figmaには四角形（R）、円形（O）、線（L）などの図形ツールがある
  • 図形はクリック＆ドラッグで描ける。Shiftキーを押しながら描くと正方形や真円になる
  • 図形の大きさはハンドルをドラッグすることで変更できる
  • 図形の移動は移動ツール（V）で選択してドラッグ。矢印キーでも移動できる
  • 図形の削除はDeleteキーまたはBackspaceキーで行う
  • Ctrl + Zで元に戻す、Ctrl + Shift + Zでやり直すことができる

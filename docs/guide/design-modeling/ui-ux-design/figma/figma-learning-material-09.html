プロトタイプを作ろう

画面遷移を再現する「プロトタイプ」機能を学びます。ボタンを押すと次の画面に移動する、といった動きのあるデザインが作れます。

この章で学ぶこと

  • プロトタイプとは何か、どんなときに使うのかを理解する
  • プロトタイプモードに切り替え、画面と画面を繋げる
  • クリック・タップで画面遷移するインタラクションを作成する
  • アニメーション効果を付け、プレビューで動作確認する
  • プロトタイプを共有し、他の人に見てもらう方法を覚える

9.1 プロトタイプってなに？ - 動くデザインの見本

プロトタイプとは、Figmaで作ったデザインに「動き」を付けることができる機能です。例えば、ボタンをクリックしたら次の画面に移動する、といった画面遷移を再現できます。

通常の静止画デザインでは、「この画面からこの画面に移動します」と説明する必要がありますが、プロトタイプを作れば、実際に動かして見せることができます。

プロトタイプのメリット

プロトタイプを作ることで、以下のようなメリットがあります：

  • 実際にアプリを使っているような感覚で確認できる
  • クライアントやチームメンバーにイメージを共有しやすい
  • 開発前にユーザーテストを実施できる
  • 画面遷移の設計ミスを早期発見できる

9.2 プロトタイプモードに切り替える

Figmaにはデザインモードとプロトタイプモードの2つのモードがあります。プロトタイプを作るには、まず右上のタブで切り替える必要があります。

実習 9-1: プロトタイプモードに切り替える

デザインモードからプロトタイプモードに切り替えてみましょう。

手順

  1. 1 Figmaのファイルを開きます。複数のフレームがある画面を用意してください（なければ第6章を参考に2つ以上のフレームを作成してください）。
  2. 2 画面右上の「Prototype」タブをクリックします（Designタブの隣にあります）。
  3. 3 右側のパネルが「Prototype」設定パネルに切り替わります。

期待される結果

✅ 右側パネルに「Prototype」と表示され、「Interactions」「Flow starting point」などの項目が見えればOKです。

ヒント

デザインモードとプロトタイプモードは、画面右上のタブで簡単に切り替えられます。デザインを修正したいときはDesignタブ、画面遷移を設定したいときはPrototypeタブを使います。

9.3 画面と画面をつなげる

プロトタイプでは、フレーム同士を線で繋ぐことで画面遷移を設定します。例えば「ホーム画面」から「詳細画面」への遷移を設定するには、ホーム画面のボタンから詳細画面に向かって線を引きます。

実習 9-2: 画面遷移を設定する

2つのフレームを繋いで、画面遷移を設定してみましょう。

手順

  1. 1 プロトタイプモードになっていることを確認します。
  2. 2 1つ目のフレーム内にあるボタンや図形（遷移の起点となる要素）をクリックして選択します。
  3. 3 選択した要素の右側に「+」マーク（青い丸）が表示されます。この「+」をクリックします。
  4. 4 そのままマウスをドラッグして、2つ目のフレームまで線を引きます。
  5. 5 2つ目のフレーム上でマウスを離すと、青い矢印で2つのフレームが繋がります。
  6. 6 右側パネルの「Interactions」に「On click → Navigate to」と表示されます。

期待される結果

✅ 青い矢印が2つのフレームを繋ぎ、右側パネルに「On click → Navigate to [フレーム名]」と表示されればOKです。

9.4 クリックで画面を移動させる

画面遷移には様々なトリガー（きっかけ）を設定できます。最も一般的なのが「On click」（クリック時）です。他にも、ホバー（マウスを乗せたとき）や時間経過での自動遷移なども設定可能です。

インタラクションの種類

トリガー 説明 用途例
On click 要素をクリック・タップしたとき ボタンを押して次の画面へ
While hovering マウスカーソルを乗せたとき メニューの展開表示
After delay 指定時間後に自動的に スプラッシュ画面の自動遷移
While pressing 要素を押している間 ボタンのアクティブ状態表示

アクションの種類

トリガーが発動したときに実行されるアクションも複数あります。

アクション 説明
Navigate to 指定したフレームに移動する（最も一般的）
Back 前の画面に戻る
Open overlay 元の画面の上にポップアップを表示する
Close overlay 表示中のポップアップを閉じる
Open URL 外部URLを開く

重要ポイント

最初は「On click」と「Navigate to」の組み合わせを覚えましょう。これが最も基本的で、よく使う設定です。

9.5 アニメーション効果を付ける

画面遷移の際に、スライドやフェードなどのアニメーション効果を付けることができます。これにより、より自然で洗練された動きを再現できます。

主なアニメーション効果

Instant（瞬時）

アニメーションなしで瞬時に切り替わります。最もシンプルな遷移です。

Dissolve（溶解）

画面がフェードインフェードアウトしながら切り替わります。柔らかい印象を与えます。

Move in / Move out（スライド）

画面がスライドして入ってくる・出ていくアニメーションです。方向を指定できます。

Smart Animate（スマートアニメート）

同じ名前の要素間で滑らかにアニメーションします。高度な表現に使います。

実習 9-3: アニメーション効果を設定する

画面遷移にスライドアニメーションを付けてみましょう。

手順

  1. 1 先ほど設定した画面遷移（青い矢印）をクリックして選択します。
  2. 2 右側パネルの「Interaction details」で、「Animation」の項目を見つけます。
  3. 3 「Animation」のドロップダウンをクリックし、「Move in」を選択します。
  4. 4 「Direction」で「From right」（右から）を選択します。
  5. 5 「Duration」（アニメーション時間）を「300 ms」（0.3秒）に設定します。
  6. 6 「Easing」（イージング）は「Ease out」のままでOKです。

期待される結果

✅ 右側パネルに「Move in, From right, 300ms, Ease out」と表示されていればOKです。

イージング（Easing）とは？

イージングは、アニメーションの速度変化の仕方を決める設定です。「Ease out」は最初は速く、徐々に遅くなります。「Ease in」は最初は遅く、徐々に速くなります。「Linear」は一定速度です。自然な動きには「Ease out」や「Ease in-out」がおすすめです。

9.6 プレビューで動作を確認する

プロトタイプの設定ができたら、実際に動かして確認してみましょう。Figmaにはプレビューモードがあり、ボタンをクリックして画面遷移を体験できます。

実習 9-4: プレビューで確認する

作成したプロトタイプをプレビューで動かしてみましょう。

手順

  1. 1 画面右上の再生ボタン（▶）をクリックします（「Present」と書いてあります）。
  2. 2 新しいウィンドウまたはタブでプレビュー画面が開きます。
  3. 3 最初のフレームが表示されるので、設定したボタンをクリックしてみます。
  4. 4 アニメーション付きで次の画面に遷移すれば成功です！
  5. 5 プレビューを閉じるには、ウィンドウを閉じるか、左上の「×」ボタンを押します。

期待される結果

✅ ボタンをクリックすると、スライドアニメーション付きで次の画面に遷移すればOKです。

プレビューのショートカット

  • Shift + Space: プレビューモードに入る（ショートカット）
  • R キー: プレビューをリスタート（最初のフレームに戻る）
  • Esc キー: プレビューモードを終了する

ヒント：フローの開始点を設定する

複数のフレームがある場合、どこから始めるかを設定する必要があります。最初に表示したいフレームをクリックし、右パネルの「Flow starting point」の「+」をクリックすると、そのフレームが開始点になります。

9.7 スマートフォンで確認する方法

スマートフォン向けのデザインの場合、実際のスマホで動作を確認したいですよね。FigmaのFigmaミラーアプリを使えば、スマホでプロトタイプを確認できます。

Figmaミラーの使い方

  1. スマートフォンのアプリストアで「Figma」アプリをインストールします
  2. Figmaアプリを開き、同じアカウントでログインします
  3. パソコンでプレビューモードに入ります
  4. スマホアプリでファイルを開くと、パソコンで開いているプレビューが同期表示されます
  5. スマホ上で実際にタップして動作確認できます

QRコードで共有する方法

プレビュー画面の右上にある「Share」ボタンを押すと、QRコードが表示されます。このQRコードをスマホで読み取ると、ブラウザでプロトタイプを開けます。アプリをインストールしなくても確認できるので便利です。

9.8 プロトタイプを共有する

作成したプロトタイプは、リンクを共有することで他の人に見てもらえます。クライアントやチームメンバーにフィードバックをもらうときに便利です。

実習 9-5: プロトタイプリンクを共有する

プロトタイプを他の人が見られるように共有設定をしてみましょう。

手順

  1. 1 Figmaのエディター画面（編集画面）に戻ります。
  2. 2 画面右上の「Share」（共有）ボタンをクリックします。
  3. 3 ダイアログが開くので、「Copy link」（リンクをコピー）タブを選択します。
  4. 4 「Link to」のドロップダウンで「Prototype」を選択します（これでプロトタイプのプレビューリンクになります）。
  5. 5 「Anyone with the link can view」（リンクを知っている人は誰でも閲覧可能）になっていることを確認します。
  6. 6 「Copy link」ボタンを押すと、リンクがクリップボードにコピーされます。
  7. 7 このリンクをメールやSlack、チャットなどで共有すれば、相手はプロトタイプを見ることができます。

期待される結果

✅ コピーしたリンクをブラウザで開くと、プロトタイプのプレビュー画面が表示されればOKです。

共有設定の種類

設定 説明
Anyone with the link can view リンクを知っている人は誰でも閲覧可能（最も一般的）
Only people invited to this file 招待した人のみ閲覧可能（セキュリティ重視）
Password protection パスワードを設定して保護（有料プランのみ）

重要ポイント：編集権限は別

「View」権限でリンクを共有すると、相手は閲覧とコメントはできますが、デザインの編集はできません。編集も許可したい場合は「Can edit」権限で招待する必要があります。

動作確認チェックリスト

以下の項目を実際に試して、プロトタイプが正しく動作するか確認しましょう。

  • プロトタイプモードに切り替えられる
  • フレーム間を矢印で繋げられる
  • アニメーション効果を設定できる
  • プレビューモードでボタンをクリックして画面遷移できる
  • プロトタイプリンクをコピーして共有できる

    プロトタイプは、静的なデザイン画面に「動き」を付けることで、実際にアプリを使っているような体験を再現する機能です。ボタンを押すと次の画面に移動する、といった画面遷移を視覚的に確認できます。

  2. Q2. 画面遷移を設定するには、どのタブに切り替える必要がありますか?

    回答を表示

    正解：Prototypeタブ
    画面右上の「Prototype」タブに切り替えることで、プロトタイプモードになり、画面遷移の設定ができるようになります。通常のデザイン編集は「Design」タブで行います。

  3. Q3. 最も基本的なインタラクションの組み合わせは何ですか?

    回答を表示

    正解：On click（クリック時）+ Navigate to（指定フレームに移動）
    「On click」と「Navigate to」の組み合わせが最も基本的で、よく使われるインタラクション設定です。ボタンをクリックしたときに、指定した画面に遷移します。

  4. Q4. プロトタイプをプレビューするためのショートカットキーは何ですか?

    回答を表示

    正解：Shift + Space
    「Shift + Space」キーを押すと、すぐにプレビューモードに入り、プロトタイプの動作を確認できます。また、右上の再生ボタン（Present）をクリックしても同じです。

よくあるトラブルと解決方法

プレビューでボタンを押しても何も起こらない

原因：インタラクションが正しく設定されていないか、フローの開始点が設定されていない可能性があります。
解決策：プロトタイプモードで青い矢印が表示されているか確認し、最初のフレームをクリックして「Flow starting point」を設定してください。

アニメーションがカクカクする

原因：フレーム内の要素が複雑すぎる、または画像サイズが大きすぎる可能性があります。
解決策：画像を圧縮する、または不要なレイヤーを削除してシンプルにしましょう。また、Duration（アニメーション時間）を短くすると改善することもあります。

共有リンクを開いてもプレビューが表示されない

原因：共有設定が「Prototype」になっていない、または閲覧権限が設定されていない可能性があります。
解決策：Shareボタンから「Link to」を「Prototype」に設定し、「Anyone with the link can view」になっているか確認してください。

  • 要素の「+」マークをドラッグして、フレーム間を青い矢印で繋ぐ
  • 基本は「On click」+「Navigate to」の組み合わせ
  • Move in、Dissolveなどのアニメーション効果を設定できる
  • 再生ボタン（Present）またはShift+Spaceでプレビュー表示
  • Shareボタンからプロトタイプリンクを作成し、他の人と共有できる

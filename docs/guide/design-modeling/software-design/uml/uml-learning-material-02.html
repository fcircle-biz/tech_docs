UML作図ツールの基本操作

無料で使えるUMLツール（draw.io）の基本操作を実践的に学びます。図形の配置、線の引き方、文字の入力、保存と共有の方法など、実際に手を動かしながら操作に慣れていきます。

この章で学ぶこと

  • draw.ioの基本的な使い方を習得する
  • 図形の配置、編集、接続の基本操作を学ぶ
  • 図の保存と共有の方法を理解する
  • PlantUMLによるテキストベースの作図を体験する

2.1 draw.ioとは

draw.io（diagrams.net）は、無料で使えるオンライン作図ツールです。 インストール不要でWebブラウザから即座に利用でき、UMLをはじめとするさまざまな図を直感的に作成できます。

無料＆インストール不要

Webブラウザがあればすぐに使い始められます

豊富な図形ライブラリ

UML図形が標準で用意されています

多様な保存形式

PNG、PDF、XMLなど多様な形式で保存可能

クラウド連携

Google Drive、OneDriveなどに保存可能

アクセス方法

Webブラウザでhttps://app.diagrams.net/にアクセスするだけで、すぐに使い始められます。

2.2 draw.ioの起動と初期画面

draw.ioを起動すると、最初に保存場所を選択する画面が表示されます。 初めて使う場合は、「デバイス」を選択するのが簡単です。

実習：draw.ioを起動してみよう

  1. 1 Webブラウザでhttps://app.diagrams.net/にアクセス
  2. 2 「デバイス」を選択（ローカルPCに保存）
  3. 3 「新しい図を作成」をクリック
  4. 4 「空白の図」を選択して「作成」をクリック

初期画面では、以下の3つの主要エリアが表示されます：

flowchart LR A[左側パネル
図形ライブラリ] --> B[中央キャンバス
作図エリア] B --> C[右側パネル
書式設定] style A fill:#ddd6fe,stroke:#a78bfa style B fill:#f5f3ff,stroke:#c4b5fd style C fill:#ddd6fe,stroke:#a78bfa
  • 左側パネル：図形ライブラリ（UML図形などを選択）
  • 中央キャンバス：実際に図を描く作業エリア
  • 右側パネル：選択した図形の書式設定（色、サイズなど）

2.3 基本的な図形の配置と編集

UML図を描くには、まず左側パネルからUML図形ライブラリを追加します。

実習：UML図形を配置してみよう

  1. 1 左側パネル下部の「その他の図形」をクリック
  2. 2 「UML」カテゴリにチェックを入れて「適用」
  3. 3 左側に「UML」セクションが表示されることを確認
  4. 4 「Class」図形をドラッグ＆ドロップでキャンバスに配置
  5. 5 図形をダブルクリックして「User」と入力

図形の基本操作

操作 方法
図形の選択 図形をクリック
図形の移動 図形をドラッグ
サイズ変更 図形の角や辺をドラッグ
テキスト編集 図形をダブルクリック
削除 図形を選択してDeleteキー
複製 Ctrl+C（コピー）→ Ctrl+V（貼り付け）

2.4 線（関連線）の引き方

UML図では、図形同士を線でつないで関係性を表現します。 draw.ioでは、複数の方法で線を引くことができます。

方法1：接続点から線を引く（推奨）

実習：図形同士を線でつなぐ

  1. 1 2つのClass図形をキャンバスに配置（「User」と「Order」）
  2. 2 「User」図形にマウスを近づける（青い×印が表示される）
  3. 3 青い×印からドラッグして「Order」図形まで引く
  4. 4 2つの図形が線でつながることを確認

方法2：矢印図形を使う

左側パネルの「一般」カテゴリから矢印を選んで配置することもできます。

ヒント

線の種類を変更するには、線を選択して右側パネルの「線」セクションで変更できます。 実線、破線、矢印の種類など、さまざまなスタイルが選べます。

2.5 テキストの入力と書式設定

図形内のテキストは、ダブルクリックで編集できます。 また、右側パネルから詳細な書式設定が可能です。

主な書式設定項目

フォント設定

  • フォントサイズ
  • 太字、斜体
  • 文字色
  • 配置（左揃え、中央揃え、右揃え）

図形の書式

  • 塗りつぶしの色
  • 線の色と太さ
  • 角の丸み
  • 影の設定

実習：テキストを装飾してみよう

  1. 1 Class図形を選択し、ダブルクリックでテキスト編集モードに
  2. 2 クラス名を入力後、右側パネルでフォントサイズを14ptに変更
  3. 3 「塗りつぶし」で図形の背景色を薄い青色に変更
  4. 4 変更が反映されることを確認

2.6 図の保存と共有

作成した図は、さまざまな形式で保存・共有できます。

保存形式

形式 用途 特徴
.drawio 編集可能な形式で保存 後で編集を続けられる
.png 画像として共有 メールやドキュメントに挿入
.pdf 印刷・プレゼン資料 高品質で印刷可能
.svg Webページに埋め込み 拡大しても画質が劣化しない

実習：図を保存してみよう

  1. 1 メニューから「ファイル」→「名前を付けて保存」を選択
  2. 2 ファイル名を「my-first-uml.drawio」と入力
  3. 3 「保存」をクリック（PCのダウンロードフォルダに保存される）
  4. 4 「ファイル」→「エクスポート」→「PNG」で画像としても保存

重要ポイント

編集可能な.drawio形式で保存しておけば、後で修正や追加ができます。 画像形式（PNG、PDF）は共有用として別途エクスポートしましょう。

2.7 PlantUMLによるテキストベースの作図

PlantUMLは、テキストで記述してUML図を自動生成するツールです。 マウス操作ではなく、コードのように記述することで図を作成します。

PlantUMLのメリット

バージョン管理がしやすい

テキストファイルなのでGitで管理できる

大量の図を効率的に作成

コピー＆ペーストで効率的に作成

プログラムで自動生成可能

スクリプトで図を自動生成できる

配置を自動調整

図形の配置が自動で最適化される

PlantUMLの基本記法

簡単なクラス図の例を見てみましょう：

PlantUML クラス図の例 
@startuml
' クラス定義
class User {
  - id: int
  - name: String
  + login()
}

class Order {
  - orderId: int
  - orderDate: Date
  + placeOrder()
}

' 関連を定義
User "1" -- "*" Order : 注文する
@enduml

このテキストをPlantUML Web版に貼り付けると、 自動的にクラス図が生成されます。

実習：PlantUMLで図を作成してみよう

  1. 1 PlantUML Web版にアクセス
  2. 2 上記のコードをコピーして、テキストエリアに貼り付け
  3. 3 右側にクラス図が自動生成されることを確認
  4. 4 クラス名や属性を変更して、図が更新されることを試してみる

draw.io vs PlantUML

draw.io：直感的で視覚的な配置が自由。初心者向け。
PlantUML：テキストベースで効率的。バージョン管理に向いている。

両方使えるようになると、場面に応じて使い分けができるようになります。

理解度確認クイズ

  1. Q1. draw.ioで図形同士を線でつなぐ最も簡単な方法は何ですか？

    回答を表示

    図形にマウスを近づけると表示される青い×印（接続点）から、もう一方の図形にドラッグする方法です。 この方法なら、図形を移動しても線が自動的に追従します。

  2. Q2. draw.ioで作成した図を後で編集できる形式で保存するには、どの形式を選べばよいですか？

    回答を表示

    .drawio形式で保存します。この形式なら、後で開いて編集を続けることができます。 PNG、PDF、SVGは共有・表示用で、編集はできません。

  3. Q3. PlantUMLを使うメリットを2つ挙げてください。

    回答を表示

    主なメリットは以下です：
    1. バージョン管理がしやすい：テキストファイルなのでGitなどで管理できる
    2. 大量の図を効率的に作成できる：コピー＆ペーストやスクリプトで自動生成できる
    3. 配置が自動調整される：図形の配置を手動で調整する手間が省ける

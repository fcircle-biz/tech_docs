基本文法

制御構文

条件分岐 (if文)
if (age >= 18) {
  System.out.println("成人");
}
繰り返し (for文)
for (int i = 0; i < 5; i++) {
  System.out.println(i);
}
選択 (switch文)
switch (day) {
  case 1: name = "月"; break;
  default: name = "他";
}

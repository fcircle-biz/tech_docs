環境準備

セットアップ手順

1
JDKのインストール
Oracle JDK 11以降またはOpenJDKをダウンロード・インストール
2
環境変数の設定
JAVA_HOME、PATHを設定してコマンドラインから実行可能にする
3
IDEの選択・設定
統合開発環境をインストールして初期設定を完了
4
Hello World実行
最初のJavaプログラムを作成して動作確認
バージョン確認コマンド
java -version
javac -version

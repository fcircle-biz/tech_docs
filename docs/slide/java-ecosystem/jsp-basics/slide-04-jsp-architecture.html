JSPアーキテクチャ

JSPライフサイクル

1
変換（Translation）
JSPファイルをサーブレットのJavaソースコードに変換
2
コンパイル（Compilation）
Javaソースコードをクラスファイルにコンパイル
3
初期化（Initialization）
jspInit()メソッドを実行してリソース初期化
4
実行（Execution）
_jspService()メソッドでリクエスト処理・レスポンス生成
5
破棄（Destruction）
jspDestroy()メソッドでリソース解放

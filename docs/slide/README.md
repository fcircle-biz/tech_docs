## 📖 概要

このディレクトリには、スライド形式の技術教材が含まれています。各教材は以下の特徴を持っています：

- **視覚的学習**: 図解とコード例を豊富に使用
- **標準サイズ**: 1280×720pxの統一フォーマット
- **PDF対応**: HTML版とPDF版を併用提供
- **プレゼン可能**: 研修や勉強会での使用に最適

## 🎨 スライド形式について

### HTML版の特徴
- **ブラウザで閲覧**: 特別なソフトウェア不要
- **カラー対応**: 技術別の色分けで視認性向上
- **リンク対応**: 外部リソースへのリンクが機能
- **レスポンシブ**: 様々な画面サイズに対応

### PDF版の特徴
- **印刷最適化**: A4サイズでの印刷に対応
- **オフライン閲覧**: ネット環境不要
- **配布しやすい**: メール添付やファイル共有が容易
- **安定性**: レイアウト崩れの心配なし

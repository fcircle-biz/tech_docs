実践例：学生管理アプリ

アプリケーション概要

学生情報を管理するWebFormsアプリケーション。学生の登録、検索、履修登録、成績管理などの機能を実装。

データモデル設計

Entity Frameworkを使用したCode Firstアプローチ。学生、コース、登録情報を関連付け。

データモデルクラス例

public class Student {
  public int StudentID { get; set; }
  public string LastName { get; set; }
  public string FirstName { get; set; }
  public AcademicYear Year { get; set; }
  public virtual ICollection<Enrollment> Enrollments { get; set; }
}

学生管理アプリの全体アーキテクチャ図

GridView実装例

<!-- 学生一覧表示用GridView -->
<asp:GridView ID="studentsGrid" runat="server"
  ItemType="StudentApp.Models.Student"
  SelectMethod="GetStudents"
  UpdateMethod="UpdateStudent"
  DeleteMethod="DeleteStudent"
  AutoGenerateColumns="false"
  DataKeyNames="StudentID">
  <Columns>
    <asp:DynamicField DataField="StudentID" />
    <asp:DynamicField DataField="LastName" />
    <asp:DynamicField DataField="FirstName" />
    <asp:DynamicField DataField="Year" />
    <asp:CommandField ShowEditButton="true" />
  </Columns>
</asp:GridView>

開発環境セットアップ

1

Visual Studio

ASP.NET WebForms開発に最適な統合開発環境

  • Community Edition（無料）、Professional、Enterpriseから選択
  • インストール時に「ASP.NET と Web開発」ワークロードを選択
  • 「.NET デスクトップ開発」も推奨
2

.NET Framework

ASP.NET WebFormsの実行基盤

  • 通常はVisual Studioと共にインストール
  • 最新版は4.8（2019リリース）
  • WebFormsでは.NET Framework 4.5以上を推奨
3

SQL Server

データベース連携のために必要

  • SQL Server Express（無料版）でも十分
  • Management Studio (SSMS)でデータベース管理
  • LocalDBも開発用に便利（軽量版）
